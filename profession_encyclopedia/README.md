## 使用说明

- **费曼学习法**：用最简单的话解释专业概念，假设读者完全没有相关背景
- **最新趋势**：包含2024-2025年新兴职业和传统职业的新变化
- **实用性**：每个职业包含核心技能要求和发展方向
- **分类精准**：参考ISC国际标准职业分类和中国职业分类大典
