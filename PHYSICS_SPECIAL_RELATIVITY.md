## 1. 狭义相对论基础

### 两个基本假设

**费曼解释**：爱因斯坦从两个"简单"假设出发，彻底改变了我们对时空的认识。

**假设1：相对性原理**
```
物理定律在所有惯性参考系中形式相同
不存在"绝对静止"的参考系
```

**假设2：光速不变原理**
```
真空中的光速c在所有惯性参考系中相同
与光源和观察者的运动无关
c = 299,792,458 m/s（定义值）
```

**费曼直觉**：
> 如果你以0.9c追光，光还是以c远离你。这完全违背常识，但这是实验事实。

**推论**：
- 同时性是相对的
- 时间不是绝对的
- 空间不是绝对的
- 质量和能量等价

---

### 光速不变的意义

**费曼解释**：光速c是宇宙的"速度上限"和"转换因子"。

**为什么光速不变**：
```
麦克斯韦方程：c = 1/√(ε₀μ₀)
这个值不涉及参考系！
```

**实验验证**：
- 迈克尔逊-莫雷实验（1887）
- 肯尼迪-索尼克实验
- 双星观测
- 粒子加速器

**光速作为极限**：
```
任何有质量物体不能达到c
需要无限大能量
信息不能超光速传播（因果律）
```

**费曼困惑**：
> 为什么大自然设定这个速度限制？我们不知道，但它就在那里。

---

### 伽利略变换 vs 洛伦兹变换

**费曼解释**：伽利略变换是"低速近似"，洛伦兹变换是"精确形式"。

**伽利略变换**（经典）：
```
x' = x - vt
t' = t
```
时间绝对，空间相对

**洛伦兹变换**（相对论）：
```
x' = γ(x - vt)
t' = γ(t - vx/c²)
y' = y
z' = z

γ = 1/√(1 - v²/c²)
```
时间和空间都相对

**低速极限**：
```
当 v << c 时：
γ → 1
洛伦兹变换 → 伽利略变换
```

**费曼直觉**：
> 在日常生活中，v/c ≈ 10⁻⁸，洛伦兹因子γ几乎等于1，所以我们感觉不到相对论效应。

---

### 洛伦兹因子

**费曼解释**：γ是相对论效应的"放大因子"。

**定义**：
```
γ = 1/√(1 - v²/c²)
```

**数值示例**：

| v/c | γ |
|-----|---|
| 0.1 | 1.005 |
| 0.5 | 1.15 |
| 0.9 | 2.29 |
| 0.99 | 7.09 |
| 0.999 | 22.4 |
| 0.9999 | 70.7 |

**物理意义**：
```
时间膨胀因子：Δt' = γΔt
长度收缩因子：L' = L/γ
质量增加因子：m' = γm
```

**v → c时**：
```
γ → ∞
需要无限大能量加速到c
```

## 2. 时间膨胀

### 时间膨胀效应

**费曼解释**：运动的时钟走得慢。不是时钟出故障，是时间本身变慢了。

**公式**：
```
Δt' = γΔt_0

Δt'：运动参考系中的时间
Δt_0：静止参考系中的时间（固有时）
γ > 1
```

**固有时**：
```
与事件一起运动的时钟测量的时间
最短的时间
```

**例子**：
```
飞船以0.8c飞行
γ = 1.67

飞船上过了1小时
地球上过了1.67小时
```

**费曼直觉**：
> 不是时钟坏了。如果飞船上有两件事相隔1小时，地球观察者看到它们相隔1.67小时。时间本身被"拉伸"了。

---

### 双生子佯谬

**费曼解释**：双胞胎，一个留在地球，一个坐飞船。飞船回来，谁更年轻？

**情景**：
```
Alice留在地球
Bob乘飞船以0.9c去半人马座（4光年远）再返回

Bob认为：
- 往返8光年，速度0.9c
- 我的钟：8/(0.9×γ) ≈ 3.9年

Alice认为：
- Bob往返：8/0.9 ≈ 8.9年
```

**Bob比Alice年轻！**

**佯谬在哪里？**
```
Bob也可以说自己在静止，地球在运动
为什么不是Alice更年轻？
```

**解答**：
```
Bob不是惯性参考系！
- 出发时加速
- 到达时减速
- 转向时加速
- 返回时减速

非惯性运动打破了对称性
Bob的路径在时空中不同
```

**费曼解释**：
> 这不是佯谬。Bob经历的"时空距离"更短，所以他的固有时更短。就像两个城市之间，直线路径最短。

---

### 宇宙线μ子

**费曼解释**：μ子寿命很短，按理说到不了地面。但因为时间膨胀，它们能到达。

**μ子性质**：
```
产生于高空（10-15km）
寿命：τ = 2.2 μs
速度：≈ 0.995c
```

**经典计算**：
```
飞行距离 = 0.995c × 2.2 μs ≈ 0.66 km
到不了地面！
```

**相对论计算**（地球参考系）：
```
γ ≈ 10
寿命延长：τ' = γτ ≈ 22 μs
飞行距离 ≈ 6.6 km
（仍不够？实际上μ子产生更高）
```

**μ子参考系**：
```
μ子看地球大气层：
长度收缩：L' = L/γ
距离变短，能到达！
```

**实验验证**：海平面观测到大量μ子 ✓

## 3. 长度收缩

### 长度收缩效应

**费曼解释**：运动的物体在运动方向上变短。

**公式**：
```
L' = L_0/γ

L'：运动参考系测量的长度
L_0：静止参考系的长度（固有长度）
γ > 1，所以 L' < L_0
```

**固有长度**：
```
物体静止时测量的长度
最长的长度
```

**注意**：
- 只在运动方向收缩
- 垂直方向不变
- 不是物体"真的"被压缩
- 是测量的结果

**例子**：
```
飞船固有长度100m
以0.8c飞过空间站
空间站测量：L = 100/γ = 100/1.67 ≈ 60m
```

**费曼直觉**：
> 这不是物理压缩。如果飞船上的人测量，长度还是100m。是不同参考系的测量结果不同。

---

### 梯子佯谬

**费曼解释**：梯子比车库长，但高速运动时能"放进"车库？

**情景**：
```
车库长度：L = 10m
梯子长度：L_0 = 15m
梯子速度：0.8c（γ = 1.67）
```

**车库参考系**：
```
梯子收缩：L' = 15/1.67 ≈ 9m
9m < 10m，能放进！
```

**梯子参考系**：
```
车库收缩：L' = 10/1.67 ≈ 6m
6m << 15m，放不进！
```

**矛盾？**

**解答**：
```
"放进去"是同时性问题
- 车库系：前后门同时关闭
- 梯子系：后门先关，前门后开
（不同时！）

没有矛盾，只是"同时"的定义不同
```

## 4. 同时性的相对性

**费曼解释**："同时"不是绝对的。对一个人同时的两件事，对另一个人可能不同时。

**爱因斯坦火车实验**：
```
火车以速度v运动
火车中央闪光
- 火车观察者：光同时到达前后
- 地面观察者：光先到后端，后到前端
```

**原因**：
```
光速对所有人都相同
但观察者相对运动
```

**结论**：
```
没有绝对的"现在"
同时性依赖于参考系
```

**费曼直觉**：
> 你以为"现在"全世界都是同一时刻？错了。你的"现在"和我的"现在"可能不一样。

---

### 时空间隔

**费曼解释**：虽然时间和空间各自相对，但它们的"组合"是不变的。

**时空间隔**：
```
s² = c²t² - x² - y² - z²

或用固有单位：
s² = t² - x²/c²（忽略y,z）
```

**不变性**：
```
所有惯性参考系测量相同的s²
就像三维空间中的距离
```

**分类**：

| s² | 类型 | 物理意义 |
|----|------|----------|
| > 0 | 类时 | 可以有因果关系 |
| = 0 | 类光 | 光的路径 |
| < 0 | 类空 | 不能有因果关系 |

**因果结构**：
```
类时间隔：A可以影响B
类光间隔：光从A到B
类空间隔：A和B无因果联系
```

## 5. 相对论动力学

### 相对论质量

**费曼解释**：物体运动越快，"有效质量"越大。

**相对论质量**：
```
m = γm_0

m_0：静止质量（固有质量）
m：运动质量（相对论质量）
```

**例子**：
```
质子静止质量：m_0 = 1.67 × 10⁻²⁷ kg
在LHC中：v = 0.999999991c
γ ≈ 7500
相对论质量 ≈ 7500 × m_0
```

**注意**：
```
现代物理倾向只用静止质量
"相对论质量"概念逐渐废弃
更好的表述：动量和能量
```

---

### 相对论动量

**费曼解释**：动量公式需要修正，否则动量守恒不成立。

**相对论动量**：
```
p = γm_0v

而不是 p = mv（经典）
```

**低速极限**：
```
当 v << c：
γ → 1
p → m_0v（经典动量）
```

**动量守恒**：
```
在所有惯性参考系中成立
（用相对论动量）
```

---

### 相对论能量 ⭐⭐⭐⭐⭐

**费曼解释**：E = mc² 是最著名的公式。质量和能量是同一东西的不同形式。

**总能量**：
```
E = γm_0c²
```

**静止能量**：
```
E_0 = m_0c²

物体静止时的能量
质量本身就是能量！
```

**动能**：
```
K = E - E_0 = (γ - 1)m_0c²
```

**能-动关系**：
```
E² = (pc)² + (m_0c²)²

或：
E² - (pc)² = (m_0c²)²
```

**无质量粒子**（m_0 = 0）：
```
E = pc
p = E/c
例如：光子
```

**费曼直觉**：
> 质量就是"凝结"的能量。1克物质完全转化为能量 = 2万吨TNT。

---

### E = mc² 的意义

**费曼解释**：这个公式改变了世界。

**数值**：
```
1 kg 物质 = c² × 1 kg 能量
         = (3×10⁸)² J
         = 9×10¹⁶ J
         ≈ 2000万吨TNT
```

**应用**：
1. **核裂变**：原子弹、核电站
   ```
   铀235裂变：质量亏损 ≈ 0.1%
   E = Δm·c²
   ```

2. **核聚变**：氢弹、太阳
   ```
   氢聚变：质量亏损 ≈ 0.7%
   ```

3. **正反物质湮灭**：100%转化
   ```
   电子+正电子 → 2γ
   所有质量转化为能量
   ```

**费曼故事**：
> 一点点物质，巨大的能量。太阳每秒把400万吨质量转化为能量，已经燃烧了46亿年。

---

### 相对论力

**费曼解释**：F = ma 不再成立。需要用相对论动量的变化率。

**相对论力**：
```
F = dp/dt = d(γm_0v)/dt
```

**当 v → c**：
```
F 需要趋于 ∞
才能继续加速
永远达不到c
```

## 6. 闵可夫斯基时空

### 四维时空

**费曼解释**：时间和空间不是分开的，而是统一的"四维时空"。

**四维位置矢量**：
```
x^μ = (ct, x, y, z)

μ = 0, 1, 2, 3
x^0 = ct（时间维度）
x^1, x^2, x^3 = x, y, z（空间维度）
```

**时空度规**：
```
ds² = c²dt² - dx² - dy² - dz²
    = η_μν dx^μ dx^ν

η_μν = diag(1, -1, -1, -1)
```

**费曼直觉**：
> 在时空中，时间是"正的"维度，空间是"负的"维度。这种"符号差"是相对论的核心。

---

### 世界线

**费曼解释**：物体在时空中的轨迹，就像飞机在天空留下的轨迹。

**静止物体**：
```
只有时间变化，空间不变
世界线是竖直线
```

**运动物体**：
```
时间和空间都变化
世界线是斜线
速度越快，越平
```

**光**：
```
ds² = 0
世界线与时间轴成45°（时空图上）
```

**费曼直觉**：
> 一生就是一条世界线。出生是一个点，死亡是另一个点，中间是你的轨迹。

---

### 光锥

**费曼解释**：光锥划分"因果可达"和"不可达"的区域。

**光锥结构**：
```
        未来光锥
           /|\
          / | \
         /  |  \
        /   |   \
    现在 ----P----
        \   |   /
         \  |  /
          \ | /
           \|/
        过去光锥

P：当前事件
```

**区域**：
1. **未来光锥**：可以影响的区域（类时）
2. **过去光锥**：可以影响的来源（类时）
3. **光锥表面**：光可达的区域（类光）
4. **光锥外**：无因果联系的区域（类空）

**因果律**：
```
信息传播 ≤ c
只能在光锥内传播
```

---

### 四维动量

**费曼解释**：能量和动量统一为四维矢量。

**四维动量**：
```
p^μ = (E/c, p_x, p_y, p_z)
    = (γm_0c, γm_0v_x, γm_0v_y, γm_0v_z)
```

**不变量**：
```
p^μp_μ = (E/c)² - p² = m_0²c²

所有参考系相同！
```

**零质量粒子**：
```
m_0 = 0
(E/c)² = p²
E = pc
```

## 7. 相对论电磁学

### 电磁场的统一

**费曼解释**：电场和磁场是同一事物的不同表现。不同参考系看到不同的"混合"。

**场变换**：
```
在参考系S中只有E
参考系S'运动 → 看到E和B

电场和磁场会互相转化！
```

**例子**：
```
静止电荷：只有电场E
运动观察者：看到电场E'和磁场B'
```

**四维电流**：
```
J^μ = (cρ, J)

ρ：电荷密度
J：电流密度
```

---

### 麦克斯韦方程的协变性

**费曼解释**：麦克斯韦方程在洛伦兹变换下形式不变（协变）。

**协变形式**：
```
∂_μF^μν = μ₀J^ν
∂_λF_μν + ∂_μF_νλ + ∂_νF_λμ = 0

F^μν：电磁场张量
```

**意义**：
- 电磁学天然符合相对论
- 这也是为什么光速c在麦克斯韦方程中出现

## 8. 实验验证

```
地球绕太阳运动（30 km/s）
如果存在以太，光速应该不同
```

```
光速各方向相同
没有以太
```

**意义**：
- 光速不变
- 以太不存在

## 9. 狭义相对论的局限

### 不能处理的问题

**1. 引力**：
```
牛顿引力：瞬时传播（超光速）
狭义相对论：信息不能超光速
→ 需要广义相对论
```

**2. 加速运动**：
```
狭义相对论：惯性系
非惯性运动：需要推广
→ 广义相对论
```

**3. 量子力学结合**：
```
狭义相对论 + 量子力学
→ 量子场论
→ Dirac方程
→ QED
```
