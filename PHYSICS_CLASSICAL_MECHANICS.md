## 1. 牛顿力学

### 牛顿第一定律（惯性定律）

**费曼解释**：物体会保持它当前的状态——静止的继续静止，运动的继续运动，除非有外力推它或拉它。就像溜冰场上的人，没人推就不会停，也没人会无缘无故开始滑。

**数学表述**：
```
如果 F_net = 0，则 v = 常数
```

**深层含义**：
- 定义了惯性参考系
- 质量是惯性大小的量度
- 自然状态不是"静止"而是"匀速直线运动"

**实验验证**：
- 气垫导轨实验
- 空间站中的物体运动

---

### 牛顿第二定律

**费曼解释**：力等于质量乘以加速度。要改变物体的运动状态（产生加速度），需要施加力。物体越重（质量越大），同样的力产生的加速越小；力越大，加速越快。

**数学表述**：
```
F = ma = m(dv/dt) = dp/dt
```

其中 p = mv 是动量。

**费曼直觉**：
```
推超市购物车：
- 空车（质量小）→ 轻轻一推就加速
- 装满的车（质量大）→ 用同样的力，加速很慢
```

**适用范围**：
- 惯性参考系
- 低速（v << c）
- 宏观物体

**局限性**：
- 不适用于高速（需狭义相对论）
- 不适用于微观（需量子力学）
- 非惯性系需要修正

---

### 牛顿第三定律（作用-反作用定律）

**费曼解释**：力总是成对出现的。你推墙，墙也在推你，大小相等、方向相反。就像两个人互推，都会向后退。

**数学表述**：
```
F_12 = -F_21
```

**关键点**：
- 作用力和反作用力：
  - 大小相等
  - 方向相反
  - 作用在不同物体上
  - 同时产生、同时消失
  - 性质相同

**常见误区**：
❌ "作用力和反作用力相互抵消"
✅ 它们作用在不同物体上，不会抵消

**应用**：
- 火箭推进
- 游泳
- 行走

---

### 万有引力定律

**费曼解释**：任何两个有质量的东西都会相互吸引。地球吸引苹果，苹果也在吸引地球，只是地球太重，看不出动。引力的大小与质量乘积成正比，与距离平方成反比。

**数学表述**：
```
F = G(m₁m₂)/r²

G = 6.674 × 10⁻¹¹ N·m²/kg²
```

**方向**：沿两物体连线，相互吸引

**费曼直觉**：
```
距离翻倍 → 引力变为1/4
质量翻倍 → 引力翻倍
```

**历史**：
- 牛顿受苹果启发（传说）
- 解释了开普勒定律
- 统一了天上和地上的力学

**应用**：
- 行星轨道
- 卫星发射
- 潮汐

**局限**：
- 不能解释水星近日点进动（需广义相对论）
- 不适用于强引力场

---

### 质点运动学

#### 位移、速度、加速度

**费曼解释**：
- **位移**：从A点到B点的直线距离和方向
- **速度**：位置变化有多快（位移/时间）
- **加速度**：速度变化有多快（速度变化/时间）

**数学关系**：
```
位移: Δr = r(t₂) - r(t₁)
速度: v = dr/dt
加速度: a = dv/dt = d²r/dt²
```

**一维运动公式**：
```
v = v₀ + at
x = x₀ + v₀t + ½at²
v² = v₀² + 2a(x - x₀)
```

---

#### 抛体运动

**费曼解释**：扔出去的东西走的是抛物线。水平方向匀速（忽略空气阻力），竖直方向匀加速（重力）。

**运动分解**：
```
水平: x = v₀ₓt
竖直: y = v₀ᵧt - ½gt²
```

**轨迹**：
```
y = x·tan(θ) - (gx²)/(2v₀²cos²θ)
```

**最大射程**：45°角发射

---

#### 圆周运动

**费曼解释**：做圆周运动需要持续"转弯"，这需要向心力指向圆心。速度大小可能不变，但方向一直在变。

**关键量**：
```
角速度: ω = dθ/dt
线速度: v = ωr
向心加速度: a_c = v²/r = ω²r
向心力: F_c = ma_c = mv²/r
```

**例子**：
- 绳子系着小球旋转
- 卫星绕地球
- 汽车转弯

---

### 功与能量

#### 功（Work）

**费曼解释**：功是力在位移方向上的"有效作用"。用力推箱子走了一段距离，就做了功。如果用力但没动，或力垂直于运动方向，不做功。

**数学定义**：
```
W = F·s = Fscosθ

单位：焦耳（J）= 牛顿·米（N·m）
```

**关键点**：
- 功是标量（有正负）
- 正功：力帮助运动
- 负功：力阻碍运动

---

#### 动能

**费曼解释**：运动物体"携带"的能量。速度越快、质量越大，动能越大。

**数学定义**：
```
K = ½mv²

单位：焦耳（J）
```

**动能定理**：
```
W_net = ΔK = K_f - K_i
```

合外力做的功等于动能的变化。

---

#### 势能

**费曼解释**：储存在"位置"或"结构"中的能量，可以释放出来做功。水坝的水、拉伸的弹簧、高处的石头都有势能。

**常见势能**：

1. **重力势能**：
```
U = mgh
```

2. **弹性势能**（弹簧）：
```
U = ½kx²
```

3. **引力势能**（通用）：
```
U = -GMm/r
```

**特点**：
- 只有保守力才有对应的势能
- 势能零点可任意选取（差值才有意义）

---

#### 机械能守恒

**费曼解释**：如果只有"保守力"（重力、弹力等）做功，动能和势能可以互相转化，但总和不变。就像过山车，下坡时重力势能转化为动能，上坡时又转化回去。

**数学表述**：
```
如果只有保守力做功：
K + U = 常数

或：
K_i + U_i = K_f + U_f
```

**条件**：
- 无摩擦力
- 无空气阻力
- 无非保守力做功

**应用**：
- 过山车设计
- 单摆
- 行星轨道

---

### 动量与冲量

#### 动量

**费曼解释**：动量是物体"运动的量"，既看质量也看速度。一辆卡车和一辆自行车以同样速度运动，卡车的动量大得多，更难停下来。

**数学定义**：
```
p = mv

单位：kg·m/s
```

**动量守恒定律**：
```
如果系统不受外力：
p_total = 常数
```

**费曼直觉**：
- 质量越大 → 动量越大
- 速度越快 → 动量越大
- 动量是矢量

---

#### 冲量

**费曼解释**：冲量是力作用一段时间的效果。短时间大力的效果可以等于长时间小力。球拍击球、汽车碰撞都涉及冲量。

**数学定义**：
```
J = FΔt = Δp

冲量-动量定理：J = p_f - p_i
```

**应用**：
- 汽车安全气囊（延长作用时间，减小力）
- 接球缓冲
- 火箭推进

---

#### 碰撞

**费曼解释**：碰撞时动量守恒，但能量要看情况。完全弹性碰撞像台球，能量不损失；完全非弹性碰撞像车祸，两车撞在一起。

**类型**：

1. **完全弹性碰撞**：
```
动量守恒：m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂'
能量守恒：½m₁v₁² + ½m₂v₂² = ½m₁v₁'² + ½m₂v₂'²
```

2. **完全非弹性碰撞**：
```
碰后两物体一起运动
动量守恒：m₁v₁ + m₂v₂ = (m₁+m₂)v'
```

3. **非完全弹性碰撞**：
```
介于两者之间
用恢复系数描述：e = (v₂' - v₁')/(v₁ - v₂)
```

---

### 转动

#### 转动运动学

**费曼解释**：转动是"旋转版的平动"。角度对应位移，角速度对应速度，角加速度对应加速度。

**对应关系**：

| 平动 | 转动 |
|------|------|
| 位移 s | 角位移 θ |
| 速度 v = ds/dt | 角速度 ω = dθ/dt |
| 加速度 a = dv/dt | 角加速度 α = dω/dt |
| 质量 m | 转动惯量 I |
| 力 F | 力矩 τ |
| F = ma | τ = Iα |
| 动能 ½mv² | 动能 ½Iω² |
| 动量 p = mv | 角动量 L = Iω |

---

#### 转动惯量

**费曼解释**：转动惯量是"旋转版的惯性"。质量分布离转轴越远，越难转动。花样滑冰运动员收拢手臂后转得更快，因为转动惯量减小了。

**数学定义**：
```
I = Σmᵢrᵢ²

连续分布：I = ∫r²dm
```

**常见物体**：

| 物体 | 转轴 | 转动惯量 |
|------|------|----------|
| 细圆环 | 中心轴 | I = MR² |
| 圆盘 | 中心轴 | I = ½MR² |
| 实心球 | 直径 | I = ⅖MR² |
| 细杆 | 中心 | I = ¹⁄₁₂ML² |
| 细杆 | 端点 | I = ⅓ML² |

**平行轴定理**：
```
I = I_cm + Md²
```

---

#### 力矩

**费曼解释**：力矩是"旋转版的力"。不只是力的大小，还看力到转轴的距离。用扳手拧螺丝，手离螺丝越远越省力。

**数学定义**：
```
τ = r × F

大小：τ = rFsinθ
方向：右手定则
```

**转动定律**：
```
τ_net = Iα
```

---

#### 角动量

**费曼解释**：角动量是"旋转版的动量"。一个物体在转动，它的角动量表示旋转的"量"。

**数学定义**：
```
L = Iω
或 L = r × p
```

**角动量守恒**：
```
如果 τ_net = 0：
L = 常数
```

**应用**：
- 花样滑冰（收臂加速）
- 跳水（抱膝加速）
- 行星轨道（开普勒第二定律）
- 陀螺仪

## 2. 分析力学

### 约束与广义坐标

**费曼解释**：很多时候物体运动受限制（如小球在圆环上运动）。用"广义坐标"描述这种受约束的运动，可以简化问题。

**约束类型**：
1. **完整约束**：可用方程表示
   ```
   f(r₁, r₂, ..., t) = 0
   ```
   例如：单摆的约束 x² + y² = L²

2. **非完整约束**：不可积分
   例如：滚动无滑动

**广义坐标**：
- 独立坐标数量 = 自由度
- 单摆：1个（角度θ）
- 双摆：2个（角度θ₁, θ₂）
- 刚体：6个（3平动 + 3转动）

---

### 虚功原理

**费曼解释**：想象给系统一个"虚拟"的小位移（符合约束），如果系统平衡，这个虚位移做的总功为零。

**数学表述**：
```
平衡条件：δW = ΣFᵢ·δrᵢ = 0
```

**虚位移**：
- 想象的、无限小的位移
- 必须符合约束
- 瞬时发生（时间不变）

**意义**：
- 无需分析约束力
- 简化复杂系统分析

---

### 拉格朗日力学

**费曼解释**：拉格朗日用"能量"代替"力"来描述运动。定义拉格朗日量L = T - U（动能减势能），然后让系统"选择"最优路径。

**拉格朗日量**：
```
L = T - U
```

**拉格朗日方程**：
```
d/dt(∂L/∂q̇ᵢ) - ∂L/∂qᵢ = 0
```

其中：
- qᵢ：广义坐标
- q̇ᵢ：广义速度

**为什么用L = T - U而不是T + U？**
- 欧拉-拉格朗日方程推导的自然结果
- 最小作用量原理的要求

**优势**：
1. 标量形式（无需处理矢量）
2. 自动处理约束
3. 广义坐标自由选择
4. 对称性分析方便
5. 便于推广到量子力学和场论

**示例：单摆**
```python
# 广义坐标：θ
# 动能：T = ½mL²θ̇²
# 势能：U = -mgL cos(θ)
# 拉格朗日量：L = ½mL²θ̇² + mgL cos(θ)

# 拉格朗日方程：
# d/dt(∂L/∂θ̇) - ∂L/∂θ = 0
# mL²θ̈ + mgL sin(θ) = 0
# θ̈ + (g/L)sin(θ) = 0  ← 单摆方程
```

---

### 哈密顿力学

**费曼解释**：哈密顿把拉格朗日再变换一次，用"广义动量"代替"广义速度"。这让物理量之间的关系更对称，也更利于量子化。

**广义动量**：
```
pᵢ = ∂L/∂q̇ᵢ
```

**勒让德变换**：
```
H = Σpᵢq̇ᵢ - L
```

**哈密顿量**（通常）：
```
H = T + U（总能量）
```

**哈密顿方程**（正则方程）：
```
q̇ᵢ = ∂H/∂pᵢ
ṗᵢ = -∂H/∂qᵢ
```

**相空间**：
- 坐标：(q₁, q₂, ..., p₁, p₂, ...)
- 每个点代表系统的一个状态
- 运动是相空间中的轨迹

**优势**：
1. 更对称的形式
2. 便于量子化（q, p → 算符）
3. 相空间分析
4. 统计力学基础
5. 混沌理论

---

### 最小作用量原理

**费曼解释**：自然界"懒惰"，总是选择作用量最小的路径。光走直线、物体走最短时间路径、行星走椭圆轨道，都是这个原理。

**作用量**：
```
S = ∫L dt = ∫(T - U) dt
```

**最小作用量原理**：
```
δS = 0
```

**费曼讲的故事**：
> 自然界似乎在尝试所有可能的路径，但只"选择"作用量最小的那条。就像光似乎"知道"哪条路径最快。

**意义**：
- 统一各种物理定律
- 量子力学的路径积分表述
- 场论的基础

---

### 守恒律与对称性（诺特定理）

**费曼解释**：每一个"对称性"对应一个"守恒量"。时间对称（物理定律不随时间变）→ 能量守恒；空间平移对称 → 动量守恒；空间旋转对称 → 角动量守恒。

**诺特定理**（1915）：
```
连续对称性 ↔ 守恒量

时间平移对称 → 能量守恒
空间平移对称 → 动量守恒
空间旋转对称 → 角动量守恒
规范对称性 → 电荷守恒
```

**深刻意义**：
- 守恒律不是偶然
- 对称性决定物理定律
- 现代物理学的核心思想

---

### 泊松括号

**费曼解释**：泊松括号是一种"运算"，描述两个量随时间变化的关系。量子力学中它变成了"对易子"。

**定义**：
```
{f, g} = Σ(∂f/∂qᵢ)(∂g/∂pᵢ) - (∂f/∂pᵢ)(∂g/∂qᵢ)
```

**重要性质**：
```
{qᵢ, pⱼ} = δᵢⱼ
{qᵢ, qⱼ} = 0
{pᵢ, pⱼ} = 0
```

**时间演化**：
```
df/dt = {f, H} + ∂f/∂t
```

**量子力学对应**：
```
{A, B} → (1/iℏ)[Â, B̂]
```

## 3. 刚体力学

### 刚体运动

**费曼解释**：刚体是"不会变形的物体"。刚体运动只有两种：平动和转动。任何复杂运动都可以分解为：质心的平动 + 绕质心的转动。

**自由度**：
- 3维空间中的刚体：6个自由度
  - 3个平动
  - 3个转动

---

### 欧拉角

**费曼解释**：用三个角度描述刚体的取向。就像描述飞机的姿态：俯仰、翻滚、偏航。

**三个角度**：
1. **进动角 φ**：绕z轴旋转
2. **章动角 θ**：绕新x轴旋转
3. **自转角 ψ**：绕新z轴旋转

**问题**：
- 万向锁（Gimbal Lock）
- 奇点

**现代替代**：四元数

---

### 欧拉方程

**费曼解释**：描述刚体转动的动力学方程。

**欧拉方程**：
```
I₁ω̇₁ + (I₃ - I₂)ω₂ω₃ = τ₁
I₂ω̇₂ + (I₁ - I₃)ω₃ω₁ = τ₂
I₃ω̇₃ + (I₂ - I₁)ω₁ω₂ = τ₃
```

**自由刚体**（τ = 0）：
- 绕主轴转动稳定（最大和最小惯量主轴）
- 绕中间主轴不稳定

**应用**：
- 陀螺
- 卫星姿态控制
- 地球自转

---

### 陀螺运动

**费曼解释**：陀螺不倒的原因是角动量守恒。重力力矩使角动量方向改变（进动），而不是减小。

**进动**：
```
Ω = τ/(Iω) = Mgr/(Iω)
```

**章动**：叠加在进动上的摆动

**应用**：
- 陀螺仪（导航）
- 自行车稳定性
- 地球岁差

---

## 4. 流体力学基础

### 流体静力学

**费曼解释**：静止流体的压力只与深度有关，与形状无关。潜水越深，压力越大。

**流体静压强**：
```
P = P₀ + ρgh
```

**帕斯卡定律**：
```
施加在封闭流体上的压强会等值传递到各个方向
```

**阿基米德原理**：
```
浮力 = 排开流体的重量
F_B = ρ_fluid · V_submerged · g
```

**应用**：
- 液压机
- 船舶浮力
- 气压计

---

### 流体运动学

**费曼解释**：流体流动有两种描述方法：
1. **拉格朗日法**：跟踪每个流体质点
2. **欧拉法**：描述每点的流速场

**连续性方程**：
```
∂ρ/∂t + ∇·(ρv) = 0

不可压缩流体：∇·v = 0
```

**流线与迹线**：
- 流线：同一时刻各点的切线方向
- 迹线：一个质点随时间的轨迹

---

### 伯努利方程

**费曼解释**：流体速度越快，压强越小。飞机升力、喷雾器原理都基于此。

**伯努利方程**（理想流体、定常流动）：
```
P + ½ρv² + ρgh = 常数
```

或：
```
P₁ + ½ρv₁² + ρgh₁ = P₂ + ½ρv₂² + ρgh₂
```

**应用**：
- 飞机升力
- 文丘里效应
- 空吸作用
- 球类运动（弧线球）

**局限性**：
- 理想流体（无粘性）
- 不可压缩
- 定常流动
- 沿一条流线

---

### 纳维-斯托克斯方程

**费曼解释**：牛顿第二定律的流体版本。描述粘性流体如何运动。这是物理学中最难的方程之一，至今没有完全解决。

**方程**：
```
ρ(∂v/∂t + v·∇v) = -∇P + μ∇²v + f
```

各项含义：
- 左边：质量×加速度
- -∇P：压力梯度力
- μ∇²v：粘性力
- f：外力（如重力）

**千禧年大奖问题**：
- 证明光滑解的存在性和唯一性
- 奖金：100万美元

**应用**：
- 空气动力学
- 海洋学
- 血液流动
- 天气预报

---

### 雷诺数

**费曼解释**：雷诺数区分"层流"和"湍流"。小雷诺数是平滑的层流，大雷诺数是混乱的湍流。

**定义**：
```
Re = ρvL/μ = vL/ν
```

其中：
- ρ：密度
- v：特征速度
- L：特征长度
- μ：动力粘度
- ν = μ/ρ：运动粘度

**流动类型**：
```
Re < 2300（圆管）：层流
Re > 4000：湍流
2300 < Re < 4000：过渡区
```

**应用**：
- 管道设计
- 飞机设计
- 船舶设计

---

## 5. 振动与波

### 简谐振动

**费曼解释**：简谐振动是最简单、最基本的振动。回复力与位移成正比。像弹簧、单摆（小角度）都是简谐振动。

**回复力**：
```
F = -kx
```

**运动方程**：
```
x(t) = A cos(ωt + φ)

ω = √(k/m)
```

**能量**：
```
总能量 E = ½kA²
动能 K = ½mv²
势能 U = ½kx²
```

**周期**：
```
T = 2π/ω = 2π√(m/k)
```

---

### 阻尼振动

**费曼解释**：真实振动都有阻力，振幅会逐渐减小。

**阻尼力**：
```
F_d = -bv
```

**运动方程**：
```
mẍ + bẋ + kx = 0
```

**三种情况**：
1. **欠阻尼**（b² < 4mk）：
   ```
   x(t) = Ae^(-γt)cos(ω't + φ)
   有振荡，振幅衰减
   ```

2. **过阻尼**（b² > 4mk）：
   ```
   无振荡，缓慢回到平衡
   ```

3. **临界阻尼**（b² = 4mk）：
   ```
   最快回到平衡，无振荡
   ```

**应用**：
- 汽车悬挂系统
- 门闭合器
- 仪表指针

---

### 受迫振动与共振

**费曼解释**：用周期性外力驱动振动系统。当驱动频率接近固有频率时，振幅急剧增大——这就是共振。

**受迫振动方程**：
```
mẍ + bẋ + kx = F₀cos(ωt)
```

**稳态解**：
```
x(t) = A cos(ωt - φ)

振幅：A = F₀/√[(k - mω²)² + (bω)²]
```

**共振**：
```
共振频率：ω_res = √(ω₀² - 2γ²)
         其中 ω₀ = √(k/m), γ = b/(2m)

当 ω → ω₀ 且 b 很小时，A → 很大
```

**应用与危害**：
- 应用：收音机调谐、MRI
- 危害：桥梁共振（塔科马海峡大桥）
- 防护：阻尼设计

---

### 耦合振动

**费曼解释**：两个振子通过某种方式连接，能量会在它们之间来回传递。

**例子**：两个单摆用弹簧连接

**简正模式**：
- 同相模式：两摆同向振动
- 反相模式：两摆反向振动

**一般运动**：简正模式的叠加

**应用**：
- 分子振动
- 固体晶格（声子）
- 耦合谐振子

---

### 波动

**费曼解释**：波是振动的传播。不是物质在传播，而是"扰动"在传播。

**波动方程**（一维）：
```
∂²y/∂x² = (1/v²)∂²y/∂t²
```

**行波解**：
```
y(x,t) = f(x - vt) + g(x + vt)

正弦波：y = A sin(kx - ωt + φ)
```

**关键量**：
```
波长：λ = 2π/k
频率：f = ω/(2π)
波速：v = fλ = ω/k
```

**波的叠加**：
```
y = y₁ + y₂
```

**干涉**：
- 相长干涉：相位差 0, 2π, 4π...
- 相消干涉：相位差 π, 3π, 5π...

---

### 驻波

**费曼解释**：两列相同的波相向传播，叠加后形成"不走的波"。有些点永远不动（波节），有些点振动最大（波腹）。

**驻波**：
```
y(x,t) = 2A sin(kx) cos(ωt)

波节：sin(kx) = 0 → x = nλ/2
波腹：sin(kx) = ±1 → x = (n + ½)λ/2
```

**边界条件**：
- 固定端：波节
- 自由端：波腹

**应用**：
- 弦乐器
- 管乐器
- 微波炉

---

### 多普勒效应

**费曼解释**：波源或观察者运动时，接收到的频率会改变。救护车向你驶来时声音变高，驶离时变低。

**公式**：
```
f' = f(v ± v_o)/(v ∓ v_s)

v：波速
v_o：观察者速度
v_s：波源速度
```

**约定**：
- 观察者朝向波源：+v_o
- 波源朝向观察者：-v_s

**应用**：
- 雷达测速
- 超声波血流检测
- 天文学（红移/蓝移）

---

## 6. 经典力学的局限性

### 经典力学的适用范围

**费曼解释**：经典力学在"日常世界"完美工作，但在极端情况下失效。

**适用**：
- 宏观物体（远大于原子）
- 低速（v << c）
- 弱引力场
- 常温常压

**失效**：
- 微观：量子力学
- 高速：狭义相对论
- 强引力：广义相对论

---

### 经典力学的问题

**1. 紫外灾难**
- 黑体辐射在短波无限大
- 需要量子力学解决

**2. 原子稳定性**
- 电子绕核旋转会辐射能量并坍缩
- 需要量子力学

**3. 光速不变**
- 麦克斯韦方程与伽利略变换矛盾
- 需要狭义相对论

**4. 水星近日点进动**
- 牛顿引力预测偏差
- 需要广义相对论

---

### 向现代物理过渡

**费曼解释**：经典力学不是"错"了，而是"不够精确"。它是一个近似，在适用范围内仍然正确。

**对应原理**（玻尔）：
```
在经典极限下（h → 0 或 v/c → 0）
量子力学/相对论 → 经典力学
```

**例子**：
```
量子力学 + ℏ → 0 → 经典力学
狭义相对论 + v << c → 经典力学
```
