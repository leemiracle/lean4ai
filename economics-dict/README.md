## 🎯 使用说明

### 费曼学习法原则

每个概念的解释遵循以下原则：

1. **说人话** - 避免术语堆砌，用日常语言解释
2. **打比方** - 用生活中的例子类比抽象概念
3. **抓本质** - 一句话概括核心要点
4. **连知识** - 说明与其他概念的联系

### 条目格式

```
## 概念名称

> 一句话概括

### 通俗解释
[用简单语言解释]

### 生活类比
[用生活例子类比]

### 核心要点
- 要点1
- 要点2

### 相关概念
[列出相关概念]
```
