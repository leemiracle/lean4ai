## 🎓 核心知识点速览

### 民法核心（★★★★★）
- 民事法律行为（有效、无效、可撤销）
- 物权变动（登记、交付）
- 合同履行（抗辩权、保全）
- 侵权责任（归责原则）

### 刑法核心（★★★★★）
- 犯罪构成（四要件）
- 正当防卫（一般、特殊）
- 共同犯罪（主从犯）
- 重点罪名（抢劫、盗窃、诈骗）

### 行政法核心（★★★★）
- 行政处罚（种类、程序）
- 行政许可（设定权限）
- 行政复议（机关、期限）
- 行政诉讼（管辖、举证）

### 诉讼法核心（★★★★）
- 管辖（级别、地域）
- 举证责任（谁主张谁举证）
- 起诉期限（民事3年、行政6个月）
- 执行程序（强制措施）
