## 📖 民法重点（★★★★★）

### 一、总则编 ★★★★★

#### 1. 民事法律行为

**★ 重点**:
- **有效要件**: 行为能力 + 意思真实 + 合法
- **无效情形**: 无行为能力 + 虚假意思 + 强制性规定 + 公序良俗
- **可撤销**: 重大误解 + 欺诈 + 胁迫 + 显失公平
- **效力待定**: 限制行为能力 + 无权代理

**★ 高频考点**:
1. 无效vs可撤销的区分
2. 撤销权的行使期限（1年/5年）
3. 无权代理vs表见代理

#### 2. 代理

**★ 重点**:
- **代理种类**: 委托代理 + 法定代理
- **无权代理**: 效力待定
- **表见代理**: 有效（被代理人担责）

**★ 高频考点**:
1. 表见代理的构成要件
2. 无权代理的法律后果

#### 3. 诉讼时效

**★ 重点**:
- **普通时效**: 3年
- **最长保护**: 20年
- **起算点**: 知道或应当知道
- **中止**: 最后6个月内
- **中断**: 重新计算

---

### 二、物权编 ★★★★

#### 1. 物权变动

**★ 重点**:
- **不动产物权**: 登记生效
- **动产物权**: 交付生效
- **例外**: 继承、法院判决等（事实行为）

**★ 高频考点**:
1. 一房二卖的处理
2. 物权变动vs合同效力

#### 2. 所有权

**★ 重点**:
- **业主的建筑物区分所有权**: 专有部分 + 共有部分 + 共同管理
- **善意取得**: 善意 + 合理价格 + 登记/交付

**★ 高频考点**:
1. 善意取得的构成要件
2. 业主大会表决规则

#### 3. 担保物权

**★ 重点**:
- **抵押权**: 不转移占有
- **质权**: 转移占有
- **留置权**: 合法占有 + 牵连关系

**★ 高频考点**:
1. 抵押vs质押的区别
2. 担保物权的竞合
3. 动产浮动抵押

---

### 三、合同编 ★★★★★

#### 1. 合同订立

**★ 重点**:
- **要约**: 内容具体 + 愿受约束
- **承诺**: 同意要约
- **格式条款**: 提示说明义务 + 无效情形

**★ 高频考点**:
1. 要约vs要约邀请
2. 格式条款的效力

#### 2. 合同履行

**★ 重点**:
- **同时履行抗辩权**: 你不给我也不给你
- **先履行抗辩权**: 你先该给我
- **不安抗辩权**: 你快破产了

**★ 高频考点**:
1. 三大抗辩权的适用条件
2. 代位权vs撤销权

#### 3. 违约责任

**★ 重点**:
- **继续履行**
- **赔偿损失**
- **违约金**: 可调整
- **定金**: ≤20% + 双倍返还

**★ 高频考点**:
1. 违约金vs定金的选择
2. 预期违约

---

### 四、侵权责任编 ★★★★

#### 1. 归责原则

**★ 重点**:
- **过错责任**: 一般原则
- **过错推定**: 法律规定
- **无过错责任**: 法律规定（产品、环境、饲养动物等）

**★ 高频考点**:
1. 不同侵权适用不同归责原则
2. 举证责任分配

#### 2. 特殊侵权

**★ 重点**:
- **产品责任**: 无过错
- **机动车交通事故**: 过错
- **医疗损害**: 过错
- **环境污染**: 无过错
- **高空抛物**: 连带补偿

**★ 高频考点**:
1. 归责原则
2. 责任主体
3. 免责事由

## 📖 刑法重点（★★★★★）

### 一、犯罪论 ★★★★★

#### 1. 犯罪构成

**★ 四要件**:
- **客体**: 侵害的利益
- **客观方面**: 行为 + 结果 + 因果关系
- **主体**: 自然人/单位
- **主观方面**: 故意/过失

**★ 高频考点**:
1. 故意vs过失的区分
2. 不作为犯罪的成立条件
3. 因果关系的认定

#### 2. 排除犯罪事由

**★ 重点**:
- **正当防卫**: 正在进行 + 针对侵害人 + 不过当
- **紧急避险**: 危险正在发生 + 不得已 + 不过当

**★ 高频考点**:
1. 一般防卫vs特殊防卫
2. 防卫过当的认定
3. 防卫时间点

#### 3. 故意犯罪形态

**★ 重点**:
- **犯罪预备**: 准备工具 + 制造条件
- **犯罪未遂**: 已经着手 + 未得逞
- **犯罪中止**: 自动放弃 + 有效地防止结果发生

**★ 高频考点**:
1. 预备vs未遂的区分（是否着手）
2. 未遂vs中止的区分（自动vs被动）

#### 4. 共同犯罪

**★ 重点**:
- **条件**: 二人以上 + 共同故意 + 共同行为
- **分类**: 主犯 + 从犯 + 胁从犯 + 教唆犯

**★ 高频考点**:
1. 共同犯罪的成立条件
2. 主犯vs从犯的认定
3. 教唆犯的处罚

---

### 二、刑罚论 ★★★★

#### 1. 刑罚种类

**★ 重点**:
- **主刑**: 管制 + 拘役 + 有期徒刑 + 无期徒刑 + 死刑
- **附加刑**: 罚金 + 剥夺政治权利 + 没收财产 + 驱逐出境

**★ 高频考点**:
1. 死刑的适用限制
2. 剥夺政治权利的内容

#### 2. 量刑情节

**★ 重点**:
- **自首**: 自动投案 + 如实供述
- **立功**: 揭发他人犯罪
- **累犯**: 5年内再犯罪

**★ 高频考点**:
1. 自首的认定
2. 累犯的后果（从重 + 不缓刑 + 不假释）

---

### 三、重点罪名 ★★★★★

#### 1. 侵犯人身权利罪

**★ 重点罪名**:
- 故意杀人罪
- 故意伤害罪
- 强奸罪
- 绑架罪
- 非法拘禁罪

**★ 高频考点**:
1. 杀人vs伤害致死的区分
2. 绑架罪vs抢劫罪的区分

#### 2. 侵犯财产罪

**★ 重点罪名**:
- 抢劫罪（暴力 + 当场）
- 盗窃罪（秘密 + 数额较大）
- 诈骗罪（欺骗 + 自愿交付）
- 侵占罪（代为保管 + 拒不退还）

**★ 高频考点**:
1. 抢劫vs抢夺vs敲诈勒索的区分
2. 盗窃vs诈骗的区分
3. 盗窃vs侵占的区分

#### 3. 贪污贿赂罪

**★ 重点罪名**:
- 贪污罪
- 受贿罪
- 行贿罪

**★ 高频考点**:
1. 主体（国家工作人员）
2. 行为方式
3. 数额标准

## 📖 行政法重点（★★★★）

### 一、行政行为 ★★★★★

#### 1. 行政处罚

**★ 重点**:
- **种类**: 警告 + 罚款 + 没收 + 暂扣吊销 + 拘留
- **设定权限**: 法律 > 行政法规 > 地方性法规 > 规章
- **程序**: 简易程序 + 一般程序 + 听证程序

**★ 高频考点**:
1. 处罚种类与设定权限的对应
2. 听证程序的适用条件
3. 一事不再罚

#### 2. 行政许可

**★ 重点**:
- **设定权限**: 法律 + 行政法规 + 地方性法规 + 省级规章（临时）
- **程序**: 申请 + 审查 + 决定

**★ 高频考点**:
1. 设定权限
2. 许可期限（20日 + 延长10日）

#### 3. 行政强制

**★ 重点**:
- **强制措施**: 查封 + 扣押 + 冻结（暂时性）
- **强制执行**: 拍卖 + 划拨 + 代履行（终局性）

**★ 高频考点**:
1. 强制措施vs强制执行的区分
2. 催告程序

---

### 二、行政复议与诉讼 ★★★★★

#### 1. 行政复议

**★ 重点**:
- **期限**: 60日
- **机关**: 上级政府/主管部门
- **结果**: 维持 + 撤销 + 变更 + 确认违法

**★ 高频考点**:
1. 复议机关的确定
2. 复议前置的情形

#### 2. 行政诉讼

**★ 重点**:
- **受案范围**: 具体行政行为
- **管辖**: 被告所在地 + 不动产所在地
- **举证责任**: 被告举证
- **起诉期限**: 6个月（一般）

**★ 高频考点**:
1. 受案范围（哪些可以告）
2. 管辖（到哪个法院）
3. 举证责任分配
4. 判决类型

## 📖 诉讼法重点（★★★★）

**★ 重点**:

- **当事人**: 原告 + 被告 + 第三人
- **证据**: 种类 + 举证责任
- **程序**: 一审 + 二审 + 执行

**★ 高频考点**:
1. 地域管辖的一般原则（被告所在地）
2. 举证责任分配
3. 二审的处理结果

### 二、刑事诉讼法 ★★★★★

**★ 重点**:
- **管辖**: 职能 + 级别 + 地域
- **强制措施**: 拘传 + 取保候审 + 监视居住 + 拘留 + 逮捕
- **辩护**: 辩护人 + 法律援助
- **程序**: 侦查 + 起诉 + 审判

**★ 高频考点**:
1. 强制措施的适用条件
2. 辩护人的权利
3. 证据规则
4. 审判程序

## 📖 理论法重点（★★★）

### 一、法理学 ★★★

**★ 重点**:
- **法的要素**: 规则 + 原则 + 概念
- **法的渊源**: 宪法 > 法律 > 行政法规 > 地方性法规
- **法的效力**: 时间 + 空间 + 对人
- **法律关系**: 主体 + 客体 + 内容

### 二、宪法 ★★★

**★ 重点**:
- **国家机构**: 人大 + 政府 + 法院 + 检察院
- **公民权利**: 平等权 + 政治权利 + 人身自由 + 社会经济权利

### 三、法制史 ★★

**★ 重点**:
- **西方法制**: 罗马法 + 英美法系 + 大陆法系
- **中国法制**: 唐律 + 清末修律

## 💡 备考建议

### 时间分配

| 阶段 | 时间 | 内容 |
|------|------|------|
| 基础阶段 | 3-4个月 | 系统学习各科目 |
| 强化阶段 | 2-3个月 | 重点突破难点 |
| 冲刺阶段 | 1-2个月 | 真题 + 模拟 |
| 考前 | 1-2周 | 回顾重点 |

### 学习方法

1. **理解为主，记忆为辅**
2. **先总则后分则**（民法、刑法）
3. **多做题**（真题 > 模拟题）
4. **归纳总结**（错题本 + 笔记）
5. **案例分析**（理论联系实际）
