## 📑 一、民事类文书

---

### 1. 民事起诉状

```
民 事 起 诉 状

原告：姓名、性别、出生年月日、民族、职业、住址、联系电话
      （法人的，写明名称、住所、法定代表人姓名和职务）

被告：姓名、性别、出生年月日、民族、职业、住址、联系电话
      （法人的，写明名称、住所、法定代表人姓名和职务）

诉讼请求：

1. 请求判令被告__________________________；（具体请求）
2. 请求判令被告承担本案诉讼费用。

事实和理由：

___________________________________________________________________
___________________________________________________________________
___________________________________________________________________
（详细陈述案件事实，包括时间、地点、经过、结果）

综上所述，被告的行为已经侵害了原告的合法权益，根据《中华人民共和国
______法》第__条的规定，原告特向贵院提起诉讼，请求依法判决。

此致
________人民法院

原告：________（签名或盖章）
     年  月  日

附：
1. 起诉状副本__份
2. 证据清单及相关证据
```

**填写要点**：
1. **原被告信息**：准确完整，特别是被告的准确信息（影响管辖）
2. **诉讼请求**：具体明确，可执行
3. **事实和理由**：客观真实，有证据支持
4. **管辖法院**：一般原则是被告所在地法院

### 2. 借条

```
借  条

借款人：__________，身份证号：____________________

出借人：__________，身份证号：____________________

借款人于____年__月__日向出借人借款人民币（大写）________元整
（小写：￥________元），借款期限自____年__月__日至____年__月__日。

借款利率：年利率____%（不得超过LPR的4倍）

还款方式：____________________

借款人承诺按时还款，逾期不还的，借款人愿意承担违约责任，并承担
出借人为实现债权而产生的律师费、诉讼费等全部费用。

借款人（签名按手印）：__________
借款日期：____年__月__日

身份证复印件粘贴处：

[粘贴身份证正面]
[粘贴身份证反面]
```

**填写要点**：
1. **借款金额**：大写小写都要写
2. **借款期限**：明确起止日期
3. **利率**：不得超过法律保护上限
4. **签名按手印**：必须在名字上按手印
5. **身份证复印件**：附上借款人身份证复印件

### 3. 收条

```
收  条

收款人：__________，身份证号：____________________

付款人：__________，身份证号：____________________

今收到付款人支付的款项人民币（大写）________元整（小写：￥________元）。

款项性质：____________________（如：货款、借款、租金等）

收款方式：现金/银行转账/支付宝/微信

收款人（签名按手印）：__________
收款日期：____年__月__日
```

**填写要点**：
1. 明确收款金额
2. 注明款项性质
3. 保留转账凭证

### 4. 简易买卖合同

```
买 卖 合 同

甲方（买方）：__________，身份证号/统一社会信用代码：__________
乙方（卖方）：__________，身份证号/统一社会信用代码：__________

甲乙双方经协商一致，就下列买卖事宜达成协议：

一、标的物
名称：__________  规格型号：__________
数量：__________  单价：__________元
总价款：人民币（大写）__________元整（小写：￥________元）

二、交付
1. 交付时间：____年__月__日前
2. 交付地点：____________________
3. 交付方式：甲方自提/乙方送货

三、付款
1. 付款时间：____年__月__日前
2. 付款方式：现金/银行转账/支票

四、验收
甲方应在收到货物后__日内验收，如有质量问题应在验收期内书面通知乙方。

五、违约责任
1. 甲方逾期付款的，按日支付未付款项__%的违约金
2. 乙方逾期交货的，按日支付合同价款__%的违约金
3. 货物质量不符合约定的，乙方应更换或退货

六、争议解决
双方发生争议，协商解决；协商不成的，提交________人民法院诉讼解决。

甲方（签字/盖章）：__________  乙方（签字/盖章）：__________
日期：____年__月__日          日期：____年__月__日
```

### 5. 房屋租赁合同（简易版）

```
房 屋 租 赁 合 同

出租方（甲方）：__________，身份证号：__________
承租方（乙方）：__________，身份证号：__________

甲方将自有房屋出租给乙方使用，双方经协商一致，达成以下协议：

一、房屋基本情况
房屋地址：____________________
房屋面积：____平方米
房屋用途：居住/商用

二、租赁期限
租赁期限自____年__月__日至____年__月__日

三、租金及支付方式
1. 月租金：人民币____元整
2. 押金：人民币____元整（合同期满无违约退还）
3. 支付方式：押__付__（押金__个月，每__个月付一次）
4. 支付时间：每期租金应提前__日支付

四、费用承担
1. 水费、电费、燃气费、网络费：由乙方承担
2. 物业费、取暖费：甲方承担/乙方承担
3. 房屋维修：因房屋质量问题由甲方承担，因乙方使用不当由乙方承担

五、甲方权利义务
1. 按约交付房屋
2. 保证房屋符合出租条件
3. 协助乙方办理居住证等相关手续

六、乙方权利义务
1. 按约支付租金
2. 合法使用房屋
3. 不得擅自转租
4. 合同期满归还房屋

七、违约责任
1. 乙方逾期支付租金超过__日的，甲方可以解除合同
2. 乙方提前退租的，应提前__日通知甲方，押金不予退还
3. 甲方提前收回房屋的，应提前__日通知乙方，退还押金并赔偿__个月租金

八、其他约定
____________________________________

九、争议解决
双方发生争议，协商解决；协商不成的，提交房屋所在地人民法院诉讼解决。

甲方（签字）：__________  乙方（签字）：__________
联系电话：__________      联系电话：__________
日期：____年__月__日      日期：____年__月__日
```

### 6. 劳动仲裁申请书

```
劳动争议仲裁申请书

申请人：姓名、性别、出生年月日、民族、身份证号、住址、联系电话

被申请人：单位名称、住所地、法定代表人姓名和职务、联系电话

仲裁请求：

1. 请求裁决被申请人支付申请人____年__月至____年__月的工资，
   共计人民币________元；
2. 请求裁决被申请人支付申请人违法解除劳动合同赔偿金人民币
   ________元；
3. 请求裁决被申请人支付申请人加班费人民币________元；
4. 请求裁决被申请人为申请人补缴____年__月至____年__月的
   社会保险。

事实和理由：

申请人于____年__月入职被申请人处，担任________职务，
月工资________元。____年__月__日，被申请人无故辞退申请人，
未支付经济补偿。申请人工作期间，被申请人经常安排申请人加班，
但未支付加班费。

根据《中华人民共和国劳动合同法》的相关规定，被申请人的行为
已经侵犯了申请人的合法权益。为此，申请人特向贵委申请仲裁，
请求依法裁决。

此致
________劳动人事争议仲裁委员会

申请人：________（签名）
     年  月  日

附：
1. 仲裁申请书副本__份
2. 证据清单及相关证据
   （劳动合同、工资条、考勤记录、解除劳动合同通知书等）
```

**填写要点**：
1. **仲裁请求**：具体明确，有计算依据
2. **事实和理由**：简明扼要，突出重点
3. **证据**：劳动关系证明、工资证明、违法解除证明等

### 7. 行政复议申请书

```
行政复议申请书

申请人：姓名、性别、出生年月日、身份证号、住址、联系电话
        （法人的，写明名称、住所、法定代表人姓名和职务）

被申请人：行政机关名称、住所地、法定代表人姓名和职务

复议请求：

1. 请求撤销被申请人于____年__月__日作出的________（行政行为）；
2. 请求责令被申请人退还已收取的________元；
3. 请求确认被申请人的行政行为违法。

事实和理由：

____年__月__日，被申请人以________为由，对申请人作出
________的行政行为（具体内容）。

申请人认为，被申请人的上述行政行为存在以下违法之处：
1. ________________；
2. ________________；
3. ________________。

根据《中华人民共和国行政复议法》的规定，申请人特向贵机关
申请行政复议，请求依法处理。

此致
________人民政府/________厅（局）

申请人：________（签名或盖章）
     年  月  日

附：
1. 行政复议申请书副本__份
2. 被申请人作出的行政行为文书
3. 证据材料
```

**填写要点**：
1. **复议请求**：撤销/变更/确认违法
2. **60日内**申请
3. **复议机关**：作出行政行为的机关的上级机关

### 8. 授权委托书

```
授 权 委 托 书

委托人：姓名、性别、出生年月日、身份证号、住址、联系电话
受托人：姓名、性别、出生年月日、身份证号、住址、联系电话

委托事项：

委托人因________纠纷一案，委托受托人作为委托人的代理人，
代理权限如下：

□ 一般代理：代为起诉/应诉，参加庭审，陈述事实，提供证据，
  进行辩论，签收法律文书。

□ 特别授权：
  1. 代为承认、放弃、变更诉讼请求；
  2. 代为和解、调解；
  3. 代为上诉；
  4. 代为申请执行；
  5. 代为领取执行款物。

委托期限：自____年__月__日起至____年__月__日止。

委托人：________（签名）
受托人：________（签名）
     年  月  日
```

### 10. 财产保全申请书

```
财产保全申请书

申请人：姓名、性别、出生年月日、身份证号、住址、联系电话
被申请人：姓名、性别、出生年月日、身份证号、住址、联系电话

请求事项：

请求贵院依法对被申请人的以下财产采取保全措施：
1. 查封/冻结被申请人银行存款人民币________元；
2. 查封被申请人所有的位于________________的房产。

事实和理由：

申请人与被申请人因________纠纷一案，已向贵院起诉（或准备起诉）。
为防止被申请人转移财产，致使判决难以执行，申请人特根据《中华
人民共和国民事诉讼法》第100条的规定，申请财产保全。

申请人愿意提供担保，如申请错误，愿意赔偿被申请人因保全所受
的损失。

此致
________人民法院

申请人：________（签名）
     年  月  日

附：
1. 申请人身份证明
2. 被申请人财产线索
3. 担保财产证明
```

## 📊 法律文书清单

| 文书类型 | 用途 | 注意事项 |
|----------|------|----------|
| 起诉状 | 向法院提起诉讼 | 管辖法院、诉讼请求明确 |
| 答辩状 | 被告回应原告 | 在法定期限内提交 |
| 上诉状 | 不服一审判决 | 15日内（民事）、10日内（刑事） |
| 申请执行书 | 申请强制执行 | 判决生效后2年内 |
| 劳动仲裁申请书 | 劳动争议 | 先仲裁后诉讼 |
| 行政复议申请书 | 不服行政行为 | 60日内申请 |
| 借条 | 借款凭证 | 大写小写、按手印 |
| 收条 | 收款凭证 | 款项性质明确 |
| 合同 | 设立权利义务 | 主要条款齐全 |
| 授权委托书 | 委托他人代理 | 代理权限明确 |

## 💡 文书写作要点

### 1. 基本要求

- **真实**：事实真实，证据确实
- **完整**：要素齐全，内容完整
- **明确**：请求明确，意思清楚
- **规范**：格式规范，语言准确

### 2. 常见错误

1. **当事人信息不准确**
   - 后果：管辖错误、送达不能

2. **诉讼请求不明确**
   - 后果：法院无法审理

3. **事实陈述不清**
   - 后果：举证困难

4. **法律依据错误**
   - 后果：请求不能支持

5. **期限延误**
   - 后果：丧失权利

### 3. 注意事项

1. **保留原件**：证据原件自己保留
2. **留存副本**：提交材料留存副本
3. **签收回执**：要求签收并保留回执
4. **及时跟进**：关注案件进展
