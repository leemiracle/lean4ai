## 1. 静电学

### 电荷

**费曼解释**：电荷是物质的一种基本属性，就像质量一样。电荷有两种：正和负。同种相斥，异种相吸。

**基本性质**：
1. **量子化**：所有电荷都是e的整数倍
   ```
   q = ne, n = 0, ±1, ±2, ...
   e = 1.602 × 10⁻¹⁹ C
   ```

2. **守恒**：孤立系统电荷总量不变

3. **相对论不变**：所有参考系测量相同

**基本电荷**：
- 质子：+e
- 电子：-e
- 夸克：±⅓e 或 ±⅔e

---

### 库仑定律

**费曼解释**：两个电荷之间的力，与电荷乘积成正比，与距离平方成反比。很像万有引力，但电力比引力强得多。

**数学表述**：
```
F = k(q₁q₂)/r² = (1/4πε₀)(q₁q₂)/r²

k = 8.99 × 10⁹ N·m²/C²
ε₀ = 8.85 × 10⁻¹² C²/(N·m²)
```

**方向**：
- 同号：相斥
- 异号：相吸

**与万有引力对比**：
```
F_electric/F_gravity ≈ 10³⁶

电子-质子间：
电力远大于引力（约10³⁹倍）
```

**适用范围**：
- 静止电荷
- 点电荷或球形对称
- 真空或均匀介质

---

### 电场

**费曼解释**：电荷在空间中产生一种"影响"，这种影响就是电场。把试探电荷放在某点，感受到的力除以电荷量，就是那点的电场。

**定义**：
```
E = F/q_test

单位：N/C 或 V/m
```

**点电荷电场**：
```
E = kq/r² = q/(4πε₀r²)

方向：正电荷向外，负电荷向内
```

**电场叠加**：
```
E_total = E₁ + E₂ + E₃ + ...
```

**电场线**：
- 从正电荷出发，到负电荷终止
- 密集处电场强
- 不相交
- 切线方向为场方向

**费曼直觉**：
> 电场就像一种"幽灵力量"，充斥在电荷周围的空间。即使没有试探电荷，电场也在那里。

---

### 高斯定律

**费曼解释**：通过一个闭合面的电通量，只与面内的电荷有关。就像围起来的网，鱼的流出量只与网里的鱼有关。

**积分形式**：
```
∮E·dA = Q_enclosed/ε₀
```

**微分形式**：
```
∇·E = ρ/ε₀
```

**应用**：
1. **均匀带电球壳**：
   ```
   内部：E = 0
   外部：E = Q/(4πε₀r²)
   ```

2. **无限长带电直线**：
   ```
   E = λ/(2πε₀r)
   ```

3. **无限大带电平面**：
   ```
   E = σ/(2ε₀)
   ```

**意义**：
- 库仑定律的另一种表述
- 麦克斯韦方程组之一
- 对称性问题简化

---

### 电势

**费曼解释**：电势是单位电荷的势能。就像山的高度，越高的地方势能越大。两点电势差就是"电压"。

**定义**：
```
V = U/q = W/q

单位：伏特（V）= J/C
```

**电势与电场**：
```
E = -∇V
V_b - V_a = -∫E·dl
```

**点电荷电势**：
```
V = kq/r = q/(4πε₀r)
（取无穷远为零点）
```

**等势面**：
- 电势相等的点构成的面
- 与电场线垂直
- 等势面密 → 电场强

**费曼直觉**：
> 电势就像"电的高度"。电荷从高电势流向低电势，就像水往低处流。

---

### 电容

**费曼解释**：电容是"存储电荷的能力"。就像水桶能装水，电容器能装电荷。容量大，同样的电压能装更多电荷。

**定义**：
```
C = Q/V

单位：法拉（F）= C/V
```

**平行板电容器**：
```
C = ε₀A/d

A：极板面积
d：极板间距
```

**电介质**：
```
C = κε₀A/d

κ：介电常数（相对电容率）
```

**电容器能量**：
```
U = ½CV² = ½QV = Q²/(2C)
```

**电容器组合**：
```
并联：C_eq = C₁ + C₂ + C₃ + ...
串联：1/C_eq = 1/C₁ + 1/C₂ + 1/C₃ + ...
```

---

### 电介质

**费曼解释**：电介质是"绝缘体"。在电场中，内部电荷会"极化"（微小位移），减弱总电场。

**极化**：
```
P = χₑε₀E

χₑ：电极化率
```

**电介质中的电场**：
```
E = E₀/κ

κ：相对介电常数
E₀：真空中的电场
```

**常见介电常数**：
| 材料 | κ |
|------|---|
| 真空 | 1 |
| 空气 | 1.0006 |
| 水 | 80 |
| 玻璃 | 5-10 |

## 2. 静磁学

### 磁场

**费曼解释**：磁场是电流或磁铁产生的一种"场"。移动的电荷在磁场中受力。磁场不像电场那样"从正到负"，而是"环绕"电流。

**磁感应强度B**：
```
单位：特斯拉（T）= N/(A·m)
      或高斯（G），1 T = 10⁴ G
```

**磁感线**：
- 闭合曲线（无磁单极子）
- 从N极出，从S极入
- 密集处磁场强
- 不相交

**费曼困惑**：
> 为什么没有磁单极子？电有正负电荷，磁却只有偶极子？这是物理学的一个未解之谜。

---

### 毕奥-萨伐尔定律

**费曼解释**：电流元在空间中产生的磁场，与电流成正比，与距离平方成反比，方向由右手定则确定。

**数学表述**：
```
dB = (μ₀/4π)(Idl × r̂)/r²

μ₀ = 4π × 10⁻⁷ T·m/A
```

**应用**：
1. **长直导线**：
   ```
   B = μ₀I/(2πr)
   ```

2. **圆形电流中心**：
   ```
   B = μ₀I/(2R)
   ```

3. **螺线管内部**：
   ```
   B = μ₀nI
   n：单位长度匝数
   ```

---

### 安培定律

**费曼解释**：磁场沿闭合路径的积分，等于路径包围的电流。像"数"穿过环的电流线。

**积分形式**：
```
∮B·dl = μ₀I_enclosed
```

**微分形式**：
```
∇×B = μ₀J
```

**适用**：
- 稳恒电流
- 对称性好的情况

**安培-麦克斯韦定律**（完整版）：
```
∮B·dl = μ₀(I_enclosed + ε₀dΦ_E/dt)

加入了位移电流项
```

---

### 洛伦兹力

**费曼解释**：运动电荷在磁场中受到的力。力的大小与速度、磁场、电荷都成正比，方向垂直于速度和磁场。

**数学表述**：
```
F = qv × B

大小：F = qvB sin θ
方向：右手定则（正电荷）
```

**特点**：
- 力 ⊥ v：不做功，只改变方向
- 力 ⊥ B：垂直于磁场
- v = 0 时 F = 0

**应用**：
1. **回旋运动**：
   ```
   r = mv/(qB)
   ω = qB/m
   ```

2. **速度选择器**：
   ```
   E ⊥ B
   v = E/B 的粒子直线通过
   ```

3. **质谱仪**：
   ```
   根据r测定质量
   ```

---

### 磁矩

**费曼解释**：电流圈相当于一个"小磁铁"，有磁矩。磁矩在外磁场中受力矩，会试图对齐磁场。

**定义**：
```
μ = IAn̂

I：电流
A：面积
n̂：法向（右手定则）
```

**力矩**：
```
τ = μ × B
```

**势能**：
```
U = -μ·B
```

**应用**：
- 电流计
- 电动机
- 磁共振

---

## 3. 电磁感应

### 法拉第定律

**费曼解释**：变化的磁场产生电场。磁通量变化，闭合回路中产生感应电动势。

**数学表述**：
```
ε = -dΦ_B/dt

Φ_B = ∫B·dA
```

**负号意义**（楞次定律）：
```
感应电流方向：阻止磁通量变化
```

**产生感应电动势的方式**：
1. 磁场B变化
2. 面积A变化
3. 角度变化

**费曼直觉**：
> 大自然"讨厌"变化。磁通量想变，感应电流就产生，试图阻止这种变化。

---

### 楞次定律

**费曼解释**：感应电流总是"反抗"引起它的原因。磁铁接近，感应电流产生磁场推开它；磁铁远离，感应电流产生磁场吸引它。

**判断步骤**：
1. 确定原磁通量如何变化
2. 感应电流产生"相反"的磁通量
3. 用右手定则确定电流方向

**费曼比喻**：
> 感应电流像"保守派"，试图维持现状。

---

### 涡电流

**费曼解释**：导体在变化的磁场中，内部会产生涡旋状的感应电流。

**应用**：
- 电磁炉
- 电磁刹车
- 金属探测器
- 感应加热

**能量损耗**：
```
P = I²R（焦耳热）
```

**减少涡流**：
- 变压器铁芯分层
- 使用绝缘层

---

### 自感与互感

#### 自感

**费曼解释**：线圈中电流变化，产生变化的磁场，这个变化磁场又在同一线圈中产生感应电动势。

**自感系数**：
```
L = Φ/I

单位：亨利（H）= Wb/A
```

**自感电动势**：
```
ε = -L(dI/dt)
```

**螺线管自感**：
```
L = μ₀n²lA
```

**能量**：
```
U = ½LI²
```

---

#### 互感

**费曼解释**：一个线圈的电流变化，在另一个线圈中产生感应电动势。这就是变压器的原理。

**互感系数**：
```
M = Φ₂₁/I₁ = Φ₁₂/I₂
```

**互感电动势**：
```
ε₂ = -M(dI₁/dt)
```

**变压器**：
```
V₂/V₁ = N₂/N₁
I₂/I₁ = N₁/N₂
```

---

## 4. 麦克斯韦方程组 ⭐⭐⭐⭐⭐

### 完整方程组

**费曼解释**：麦克斯韦方程组是电磁学的"宪法"，四个方程描述了所有电磁现象。电场和磁场不再分开，而是统一的"电磁场"。

**积分形式**：

1. **高斯定律**（电场）：
```
∮E·dA = Q/ε₀
```
电荷产生电场

2. **高斯定律**（磁场）：
```
∮B·dA = 0
```
无磁单极子

3. **法拉第定律**：
```
∮E·dl = -dΦ_B/dt
```
变化磁场产生电场

4. **安培-麦克斯韦定律**：
```
∮B·dl = μ₀I + μ₀ε₀(dΦ_E/dt)
```
电流和变化电场产生磁场

---

**微分形式**：

1. **∇·E = ρ/ε₀**

2. **∇·B = 0**

3. **∇×E = -∂B/∂t**

4. **∇×B = μ₀J + μ₀ε₀(∂E/∂t)**

---

### 位移电流

**费曼解释**：麦克斯韦的伟大发现——变化的电场等效于电流，也能产生磁场。这补全了安培定律。

**位移电流**：
```
I_d = ε₀(dΦ_E/dt)
```

**为什么需要**：
- 电容器充电时，导线有电流，但极板间无真实电流
- 为使安培定律连续，引入"位移电流"

**意义**：
- 预言电磁波
- 统一电和磁

---

### 电磁波

**费曼解释**：变化的电场产生磁场，变化的磁场又产生电场，互相激发，向前传播——这就是电磁波。

**波动方程**：
```
∇²E = μ₀ε₀(∂²E/∂t²)
∇²B = μ₀ε₀(∂²B/∂t²)
```

**波速**：
```
c = 1/√(μ₀ε₀) = 3 × 10⁸ m/s

（恰好等于光速！）
```

**麦克斯韦的洞察**：
> 光就是电磁波！

**电磁波特性**：
- 横波
- E ⊥ B ⊥ 传播方向
- E和B同相位
- 真空中速度c
- 携带能量和动量

---

### 电磁波谱

**费曼解释**：所有电磁波本质相同，只是频率/波长不同。

| 类型 | 波长 | 频率 | 来源 |
|------|------|------|------|
| 无线电波 | >1m | <300MHz | 振荡电路 |
| 微波 | 1mm-1m | 300MHz-300GHz | 磁控管 |
| 红外线 | 700nm-1mm | 300GHz-430THz | 热辐射 |
| 可见光 | 400-700nm | 430-750THz | 原子跃迁 |
| 紫外线 | 10-400nm | 750THz-30PHz | 原子跃迁 |
| X射线 | 0.01-10nm | 30PHz-30EHz | 电子跃迁 |
| γ射线 | <0.01nm | >30EHz | 核衰变 |

## 5. 电路基础

### 欧姆定律

**费曼解释**：电压、电流、电阻的关系。电压推动电荷流动，电阻阻碍流动。

**数学表述**：
```
V = IR

I = V/R
```

**电阻**：
```
R = ρl/A

ρ：电阻率
l：长度
A：截面积
```

**功率**：
```
P = IV = I²R = V²/R
```

---

### 基尔霍夫定律

**费曼解释**：电路的两个基本规则。

**1. 节点定律**（电流守恒）：
```
ΣI_in = ΣI_out
```

**2. 回路定律**（能量守恒）：
```
ΣV = 0（回路中）
```

---

### RC电路

**费曼解释**：电阻和电容的组合，充电放电有延迟。

**充电**：
```
Q(t) = Q_∞(1 - e^(-t/RC))
V(t) = V_0(1 - e^(-t/RC))
```

**放电**：
```
Q(t) = Q₀e^(-t/RC)
V(t) = V₀e^(-t/RC)
```

**时间常数**：
```
τ = RC
```

---

### RL电路

**费曼解释**：电阻和电感的组合，电流变化有延迟。

**电流增长**：
```
I(t) = I_∞(1 - e^(-tR/L))
```

**电流衰减**：
```
I(t) = I₀e^(-tR/L)
```

**时间常数**：
```
τ = L/R
```

---

### RLC电路

**费曼解释**：电阻、电感、电容的组合，可以振荡。

**方程**：
```
L(d²Q/dt²) + R(dQ/dt) + Q/C = 0
```

**谐振频率**：
```
ω₀ = 1/√(LC)
```

**阻尼振荡**：
```
Q(t) = Q₀e^(-Rt/2L)cos(ω't + φ)
```

## 6. 电磁场的能量与动量

### 电场能量密度

**费曼解释**：电场存储能量，能量存在于电场中。

**能量密度**：
```
u_E = ½ε₀E²
```

**总能量**：
```
U = ∫u_E dV = ½∫ε₀E² dV
```

**例子**：电容器
```
U = ½CV² = ½ε₀E²(Ad)
```

---

### 磁场能量密度

**费曼解释**：磁场也存储能量。

**能量密度**：
```
u_B = B²/(2μ₀)
```

**总能量**：
```
U = ∫u_B dV = ½∫(B²/μ₀) dV
```

**例子**：电感
```
U = ½LI² = (B²/2μ₀)(Al)
```

---

### 电磁场能量密度

**费曼解释**：电磁场的总能量密度。

**公式**：
```
u = u_E + u_B = ½ε₀E² + B²/(2μ₀)
```

**电磁波**：
```
u = ε₀E² = B²/μ₀
（因为 E = cB）
```

---

### 坡印廷矢量

**费曼解释**：电磁波携带能量流动，坡印廷矢量描述能量流动的方向和大小。

**定义**：
```
S = (1/μ₀)E × B

单位：W/m²
```

**意义**：
- 方向：能量传播方向
- 大小：单位时间通过单位面积的能量

**电磁波**：
```
S = (1/μ₀)EB = cε₀E²
```

**平均功率**：
```
<S> = ½cε₀E₀²
```

---

### 电磁动量

**费曼解释**：电磁场也有动量。

**动量密度**：
```
g = S/c² = (1/μ₀c²)E × B
```

**辐射压**：
```
P = S/c（完全吸收）
P = 2S/c（完全反射）
```

**应用**：
- 太阳帆
- 激光冷却

## 7. 物质中的电磁场

### 电介质中的电场

**费曼解释**：电介质削弱电场。

**有效电场**：
```
E = E₀/κ
```

**高斯定律（介质）**：
```
∮D·dA = Q_free

D = ε₀E + P = εE
ε = κε₀
```

---

### 磁介质中的磁场

**费曼解释**：不同磁介质对磁场有不同影响。

**磁化强度**：
```
M = χ_m H
```

**磁场强度**：
```
H = B/μ₀ - M
```

**磁导率**：
```
μ = μ₀(1 + χ_m) = μ₀μ_r
```

**磁介质类型**：

| 类型 | χ_m | 例子 |
|------|-----|------|
| 抗磁质 | <0（小）| 铜、铋 |
| 顺磁质 | >0（小）| 铝、氧 |
| 铁磁质 | >>0 | 铁、钴、镍 |

---

### 边界条件

**费曼解释**：电磁场在介质界面处的连续性条件。

**电场**：
```
E_t连续（切向）
D_n连续（无自由电荷）
```

**磁场**：
```
H_t连续（无表面电流）
B_n连续（法向）
```

## 8. 电磁学的应用

### 电动机

**费曼解释**：通电线圈在磁场中受力矩转动。

**原理**：
```
τ = μ × B = NIAB sin θ
```

**类型**：
- 直流电机
- 交流电机
- 步进电机

---

### 发电机

**费曼解释**：线圈在磁场中转动，产生感应电动势。

**原理**：
```
ε = -dΦ/dt = NBAω sin(ωt)
```

**交流电**：
```
ε(t) = ε₀ sin(ωt)
```

---

### 变压器

**费曼解释**：利用互感改变电压。

**原理**：
```
V₂/V₁ = N₂/N₁
I₂/I₁ = N₁/N₂
P₂ ≈ P₁（理想）
```

**应用**：
- 电力传输（升压减少损耗）
- 电子设备（降压）

---

### 无线通信

**费曼解释**：发射天线产生电磁波，接收天线感应电流。

**原理**：
- LC振荡电路产生高频信号
- 天线辐射电磁波
- 接收天线感应
- 调谐电路选择频率

**调制**：
- AM：调幅
- FM：调频
