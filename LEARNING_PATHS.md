# Lean4 学习路线图

根据你的目标，选择适合的学习路径：

## 🎓 路径 1: 数学研究者

**目标**: 使用 Lean 形式化数学定理

### 时间线（8-12周）

```
第 1-2 周: 基础
├── NNG4: 自然数游戏
├── TPIL: 第 1-5 章
└── 基本战术练习

第 3-4 周: 进阶
├── TPIL: 第 6-10 章
├── MIL: 前 3 章
└── mathlib 探索

第 5-6 周: 实践
├── MIL: 完整阅读
├── mathlib 文档熟悉
└── 小型形式化项目

第 7-8 周: 深入
├── 选择数学领域
├── 阅读相关 mathlib 源码
└── 尝试贡献

第 9-12 周: 研究
├── 独立项目
├── 社区协作
└── 论文/博客
```

### 重点项目
- **mathlib4**: 必须熟悉
- **FLT**: 参考前沿数学
- **特定领域库**: 拓扑/代数/分析

### 关键技能
- [ ] 依赖类型理论
- [ ] 常用战术（simp, grind, induction）
- [ ] 类型类
- [ ] 结构体和类
- [ ] mathlib 组织结构
