## 一、新能源汽车概述

### 1.1 定义与分类

**新能源汽车（New Energy Vehicle, NEV）**

**简单解释**：用电、氢等清洁能源驱动的汽车，减少或无尾气排放。

**分类**：

| 类型 | 英文 | 特点 | 例子 |
|------|------|------|------|
| 纯电动 | BEV | 仅电池 | Tesla Model 3、比亚迪海豚 |
| 插电混动 | PHEV | 电池+燃油 | 比亚迪DM-i |
| 增程式 | EREV | 电池+发电 | 理想L系列 |
| 燃料电池 | FCEV | 氢燃料 | 丰田Mirai |
| 混合动力 | HEV | 油电混合（不可外充）| 丰田普锐斯 |

### 1.2 发展历程

```
第一阶段（1990-2010）：技术探索
├── GM EV1（1996）
├── 丰田普锐斯（1997）

第二阶段（2010-2020）：初步商业化
├── 日产Leaf（2010）

├── 比亚迪DM-i爆发
├── 中国占全球60%+
└── 智能化竞争
```

## 二、动力电池

### 2.1 锂离子电池原理

**简单解释**：锂离子在正负极之间来回移动，像摇椅一样，充电存储能量。

```
正极材料：LiMO2（M=Co、Ni、Mn、Fe）
负极材料：石墨（C）

充电：Li⁺从正极→负极（嵌入）
放电：Li⁺从负极→正极（脱嵌）

反应：
正极：LiMO2 ↔ Li1-xMO2 + xLi⁺ + xe⁻
负极：C + xLi⁺ + xe⁻ ↔ LiC₆
```

### 2.2 电池参数

**能量密度**：
```
质量能量密度（Wh/kg）= 容量 × 电压 / 质量
体积能量密度（Wh/L）= 容量 × 电压 / 体积

三元电池：250-300 Wh/kg（单体）
磷酸铁锂：160-180 Wh/kg（单体）
固态电池：400+ Wh/kg（目标）
```

**关键指标**：

| 参数 | 单位 | 三元 | 磷酸铁锂 |
|------|------|------|---------|
| 标称电压 | V | 3.6-3.7 | 3.2 |
| 能量密度 | Wh/kg | 250-300 | 160-180 |
| 循环寿命 | 次 | 1000-1500 | 3000-5000 |
| 安全性 | - | 中 | 高 |
| 成本 | ¥/Wh | 0.6-0.8 | 0.4-0.5 |

### 2.3 正极材料

| 材料 | 化学式 | 特点 | 应用 |
|------|--------|------|------|
| 钴酸锂（LCO）| LiCoO₂ | 能量密度高、贵 | 手机 |
| 三元（NCM）| LiNiCoMnO₂ | 平衡 | 主流EV |
| 三元（NCA）| LiNiCoAlO₂ | 高能量 | Tesla |
| 磷酸铁锂（LFP）| LiFePO₄ | 安全、便宜 | 比亚迪、Tesla |
| 锰酸锂（LMO）| LiMn₂O₄ | 便宜 | 低端EV |

**三元材料演进**：
```
NCM111 → NCM523 → NCM622 → NCM811 → NCMA

高镍化趋势：
- 提高能量密度
- 降低钴用量
- 降低成本
- 稳定性挑战
```

### 2.4 负极材料

| 材料 | 容量(mAh/g) | 电压(V) | 应用 |
|------|-----------|---------|------|
| 石墨 | 350-370 | 0.1 | 主流 |
| 硅碳 | 450-600 | 0.4 | 高端EV |
| 钛酸锂 | 175 | 1.5 | 快充 |

**硅碳负极**：
```
优势：
- 容量高（理论4200 mAh/g）
- 能量密度提升10-20%

挑战：
- 体积膨胀（300%）
- 循环寿命
- 成本

应用：
- 特斯拉4680
- 宁德时代麒麟电池
```

### 2.5 电解质

**液态电解质**：
```
成分：LiPF₆ + 有机溶剂（EC、DMC、EMC）

特点：
- 离子电导率高
- 成本低
- 易燃

改进：
- 添加剂（FEC、VC）
- 阻燃剂
```

**固态电解质**：

**简单解释**：用固体替代液体电解质，更安全、能量密度更高。

| 类型 | 离子电导率(S/cm) | 稳定性 | 进展 |
|------|----------------|--------|------|
| 氧化物 | 10⁻³-10⁻⁴ | 好 | 半固态量产 |
| 硫化物 | 10⁻²-10⁻³ | 差 | 研发中 |
| 聚合物 | 10⁻⁴-10⁻⁵ | 好 | 已量产 |

### 2.6 电池管理系统（BMS）

**功能**：
```
├── 状态估算
│   ├── SOC（荷电状态）
│   ├── SOH（健康状态）
│   └── SOP（功率状态）
├── 均衡管理
│   ├── 被动均衡
│   └── 主动均衡
├── 热管理
│   ├── 加热
│   └── 冷却
├── 安全保护
│   ├── 过充/过放
│   ├── 过温
│   └── 短路
└── 通信
    └── CAN、CAN-FD
```

**SOC估算**：
```
方法：
├── 安时积分（累积误差）
├── 开路电压（OCV，需静置）
└── 卡尔曼滤波（主流）

精度要求：±3-5%
```

## 三、电机与电控

### 3.1 电机类型

| 类型 | 特点 | 效率 | 应用 |
|------|------|------|------|
| 永磁同步（PMSM）| 效率高、体积小 | 95-97% | 主流 |
| 感应电机（IM）| 无稀土、成本低 | 90-94% | Tesla |
| 开关磁阻（SRM）| 结构简单 | 85-90% | 低端 |
| 电励磁同步 | 无稀土 | 92-95% | BMW |

### 3.2 永磁同步电机

```
结构：
├── 定子（绕组）
├── 转子（永磁体）
└── 控制器

永磁体：钕铁硼（NdFeB）

参数：
| 功率 | 100-300 kW |
| 扭矩 | 200-600 Nm |
| 转速 | 0-18000 rpm |
| 效率 | 95-97% |

优势：
- 效率高
- 功率密度高
- 响应快

劣势：
- 稀土成本
- 高温退磁
```

### 3.3 电机控制器

**功率器件**：

| 器件 | 开关频率 | 效率 | 应用 |
|------|---------|------|------|
| IGBT | 10-20 kHz | 97% | 成熟EV |
| SiC MOSFET | 50-100 kHz | 99% | 高端EV |

**SiC优势**：
```
- 开关损耗低50%
- 工作温度高（200°C）
- 体积小30%
- 续航+5-10%
```

**控制算法**：
```
矢量控制（FOC）：
- id=0控制
- MTPA（最大转矩/电流）
- 弱磁控制

效率MAP优化：
- 不同工况不同控制策略
- 效率>95%区域最大化
```

### 4.3 超充技术

**800V平台**：

**简单解释**：提高电压（800V vs 400V），降低电流，减少线损，充电更快。

```
优势：
├── 充电功率↑（350 kW+）
├── 线缆轻（电流↓）
├── 效率↑（损耗↓）
└── 充电时间↓（10-80%: 15min）

代表：
├── 保时捷Taycan（800V）
├── 小鹏G9（800V）
├── 极氪001（800V）
├── 比亚迪汉EV（部分800V）
└── 理想MEGA（800V）
```

**充电曲线**：
```
10-80%：
- 250 kW：20-30 min
- 350 kW：15-20 min
- 500 kW（规划）：10-15 min

80%后降功率（保护电池）
```
