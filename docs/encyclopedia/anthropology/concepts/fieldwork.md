## 1. 摘要

田野工作（Fieldwork）是人类学核心的研究方法，指研究者长期深入研究对象的生活环境，通过参与观察、访谈、记录等方式收集资料，以理解当地人的文化、社会和生活方式。田野工作不只是数据收集技术，更是一种认识论立场：相信只有通过亲身经历和深度参与，才能真正理解另一种文化。

田野工作的基本特征包括：(1) **长期性**：通常持续12-24个月；(2) **参与观察**：研究者既参与又观察当地生活；(3) **语言学习**：学习当地语言进行交流；(4) **建立关系**：与当地人建立信任和友谊；(5) **整体视角**：关注文化的各个方面。

田野工作方法的确立归功于**马林诺夫斯基**（Bronislaw Malinowski, 1884-1942）。由于第一次世界大战，他被迫在特罗布里恩群岛停留两年（1915-1917），期间发展出了系统的田野工作方法。在《西太平洋的航海者》（1922）中，他提出了田野工作的三项原则：(1) 必须有真正的科学目标；(2) 必须把自己置于好的条件下工作；(3) 必须使用特殊的方法收集资料。

当代田野工作呈现出多元化趋势。传统"单点、长期、孤立"的田野工作模式受到挑战，出现了**多点民族志**（multi-sited ethnography）、**虚拟民族志**（virtual ethnography）、**短期田野**（short-term fieldwork）等新形式。全球化、人口流动、数字技术改变了田野工作的性质和可能性。

田野工作的伦理和反思性也日益受到重视。研究者必须考虑**知情同意**、**匿名保护**、**互利研究**等伦理问题。同时，研究者自身的身份、立场、偏见如何影响研究，也成为田野工作反思的核心议题。

**关键词**：参与观察、民族志、田野、马林诺夫斯基、反身性、伦理

## 2. 历史发展

### 2.1 前史：旅行者与传教士

**早期的文化记录**：
在人类学成为专业学科之前，关于其他文化的信息来自：
- **旅行者**：马可·波罗、伊本·白图泰等
- **传教士**：耶稣会士对美洲、亚洲的记录
- **殖民官员**：为殖民统治收集信息
- **商人**：贸易接触的副产品

**特点**：
- 非系统的观察
- 欧洲中心视角
- "风俗"记录而非深度理解
- 服务于殖民、宗教或商业目的

**早期的"田野"**：
- **摩尔根**（Lewis Henry Morgan）：与易洛魁人接触（1840s）
- **泰勒**（Edward Tylor）：墨西哥旅行
- **鲍德默**（Karl von den Steinen）：巴西探险

### 2.2 博厄斯与早期田野工作

**博厄斯**（Franz Boas, 1858-1942）：
- 在巴芬岛与因纽特人一起生活（1883-1884）
- 强调语言学习和深度接触
- 批判进化论的推测方法
- 倡导历史特殊论

**早期田野工作的特征**：
- 时间较短（几周到几个月）
- 关注物质文化和语言
- 记录正在消失的文化（salvage ethnography）
- 研究者与研究对象有明显距离

**博厄斯学派**：
- **克鲁伯**（Alfred Kroeber）：加州印第安人研究
- **萨丕尔**（Edward Sapir）：语言人类学
- **本尼迪克特**（Ruth Benedict）：但她的田野工作有限
- **米德**（Margaret Mead）：萨摩亚、新几内亚

### 2.3 马林诺夫斯基革命

**马林诺夫斯基**（Bronislaw Malinowski, 1884-1942）：

**背景**：
- 波兰裔英国人类学家
- 1914年前往美拉尼西亚
- 因一战被困在澳大利亚，1915-1918年在特罗布里恩群岛

**田野工作创新**：

1. **长期居住**：两年以上持续田野
2. **学习语言**：使用当地语言而非翻译
3. **参与观察**：参与日常生活
4. **整体视角**：关注文化各方面

**《西太平洋的航海者》（1922）导言**：

马林诺夫斯基提出田野工作的三个目标：

1. **部落的组织和文化架构**：
   "必须给出文化现象的骨架，社群的固定框架"

2. **日常生活的具体行为**：
   "日常生活的非仪式性行为"

3. **本土人的视角**：
   "本土人的世界观、他们的生活观"

**田野工作的方法**：
- **统计文献法**：记录人口、社会组织
- **日常生活观察**：关注日常活动
- **获取本土心理状态**：通过对话理解

**影响**：
- 确立田野工作为人类学的核心方法
- 成为人类学训练的必要环节
- 影响了一代人类学家

### 2.4 英国功能主义传统

**拉德克利夫-布朗**（A.R. Radcliffe-Brown, 1881-1955）：
- 安达曼岛人研究（1906-1908）
- 强调社会结构而非文化
- 结构功能主义：研究制度如何维持社会秩序

**英国人类学训练**：
- 严格的田野工作要求
- 通常在英属殖民地做田野
- 关注社会结构、政治组织、亲属制度

**代表人物**：
- **埃文斯-普里查德**（E.E. Evans-Pritchard）：努尔人、阿赞德人
- **迈耶·福蒂斯**（Meyer Fortes）：塔伦西人
- **格卢克曼**（Max Gluckman）：罗得西亚

**田野工作传统**：
- 强调长期田野（2年）
- 关注社会组织
- 详细记录亲属关系

### 2.5 美国文化人类学

**博厄斯传统的延续**：
- **米德**（Margaret Mead, 1901-1978）：
  - 萨摩亚（1925-1926）：青春期研究
  - 新几内亚：性别角色研究
  - 巴厘岛：儿童发展

- **贝特森**（Gregory Bateson, 1904-1980）：
  - 新几内亚：与米德合作
  - 巴厘岛：与米德合作
  - 发展系统理论

**美国田野工作特点**：
- 关注文化心理
- 强调文化相对主义
- 重视个人和经验

### 2.6 反思性转向 (1980s-)

**《写文化》（Writing Culture, 1986）**：

**Clifford & Marcus** 提出的问题：
1. **民族志的权威性**：民族志如何建构"文化"？
2. **研究者的位置**：研究者的身份如何影响研究？
3. **权力关系**：田野工作中的权力不平等
4. **文本性**：民族志写作的文学性

**关键议题**：

**1. 反身性（Reflexivity）**：
- 研究者必须反思自己的立场
- 承认研究是位置性的
- 研究者也是被研究的对象

**2. 权力与知识**：
- 田野工作与殖民主义的关系
- 知识生产中的权力
- 谁有权代表谁？

**3. 伦理**：
- 知情同意
- 匿名保护
- 互利研究

**4. 表征危机**：
- 如何公正地代表他人？
- 谁有权说话？

**代表人物**：
- **James Clifford**：《文化的困境》
- **George Marcus**：多点民族志
- **Lila Abu-Lughod**：反文化写作
- **Ruth Behar**：脆弱的观察者

### 2.7 当代发展 (2000-2026)

**新形式的田野工作**：

**1. 多点民族志（Multi-sited Ethnography）**：
- **Marcus (1995)**：《民族志在/在世界体系》
- 追踪人、物、观念的流动
- 不局限于单一地点
- 全球化的田野

**2. 虚拟/数字民族志（Virtual/Digital Ethnography）**：
- **Hine (2000)**：《虚拟民族志》
- **Boellstorff (2008)**：《第二人生中的到来》
- 研究在线社区
- 虚拟田野工作

**3. 家乡人类学（Native Anthropology）**：
- 在自己社会做田野
- 挑战"他者"研究范式
- 本土视角

**4. 行动/应用田野工作**：
- **Fals Borda**：参与行动研究
- 田野工作促进社会变革
- 应用人类学

**5. 短期田野（Short-term Fieldwork）**：
- 资金和时间限制
- 多次短期访问
- 快速评估

**2020-2026年最新发展**：

**1. 疫情与田野工作（2020-2023）**：
- 疫情限制了面对面田野
- 发展远程田野方法
- 在线访谈
- 数字田野工作

**2. AI与田野工作（2022-2026）**：
- AI辅助田野记录
- 自动转录和翻译
- 大数据分析与田野结合

**3. 气候变化田野**：
- 环境人类学的田野创新
- 跟踪气候影响的田野

**4. 社交媒体田野**：
- 平台民族志
- 算法民族志

## 3. 核心理论与概念

### 3.1 参与观察

**定义**：
研究者既参与当地生活，又观察和记录的田野工作方法。

**参与 vs 观察**：

| 维度 | 参与 | 观察 |
|------|------|------|
| 角色 | 局内人 | 局外人 |
| 活动 | 参与活动 | 观看活动 |
| 知识 | 体验知识 | 观察知识 |
| 风险 | 入戏太深 | 理解不足 |

**参与的谱系**：

```
完全观察 ←—————————————→ 完全参与
(旁观者)     (观察参与)     (成员)
```

**参与观察的优势**：
- 获得"局内人"视角
- 理解非言语行为
- 体验而非只是描述

**参与观察的挑战**：
- "Going native"：过度同化
- 情感卷入
- 距离的保持

### 3.2 田野笔记

**定义**：
田野工作者在田野中记录观察、经历、思考的笔记。

**田野笔记的类型**：

| 类型 | 内容 | 时间 |
|------|------|------|
| **速记** | 简短关键词 | 活动进行时 |
| **详细笔记** | 完整描述 | 每日整理 |
| **日记** | 个人感受 | 每日 |
| **分析笔记** | 初步分析 | 田野期间 |
| **方法笔记** | 方法反思 | 田野期间 |

**田野笔记的内容**：
1. **观察**：看到了什么
2. **对话**：访谈和日常对话
3. **事件**：参与的活动
4. **反思**：研究者的思考
5. **问题**：待研究的问题

**田野笔记写作**：
- 每日写、及时写
- 详尽具体
- 区分观察和解释
- 使用直接引语

### 3.3 关键信息人

**定义**：
田野中与研究者关系密切、提供大量信息的当地人。

**信息人的类型**：

| 类型 | 特征 | 作用 |
|------|------|------|
| **关键信息人** | 深度合作、长期关系 | 主要信息来源 |
| **专家信息人** | 特殊知识 | 技术性知识 |
| **普通信息人** | 一般信息 | 常规信息 |
| **偶然信息人** | 短暂接触 | 补充信息 |

**与信息人的关系**：
- 建立信任
- 互利互惠
- 明确角色
- 伦理考量

**信息人问题**：
- 信息人是否代表社区？
- 信息人的偏见
- 过度依赖信息人

### 3.4 文化休克与适应

**文化休克（Culture Shock）**：
进入陌生文化时的焦虑、困惑和不适感。

**阶段理论**（Oberg 1960）：

1. **蜜月期**：兴奋、好奇
2. **危机期**：焦虑、挫败、思乡
3. **调整期**：逐渐适应
4. **适应期**：舒适、理解

**田野工作者的应对**：
- 学习语言和文化
- 建立日常习惯
- 寻找支持
- 写田野笔记作为出口
- 与家乡保持联系

**反向文化休克**：
- 田野结束回家的不适应
- 田野经历改变了研究者
- 难以与家乡的人分享经历

### 3.5 反身性

**定义**：
研究者反思自己在研究中的位置、影响和偏见。

**反身性的层次**：

1. **方法论反身性**：
   - 我的方法如何影响发现？

2. **个人反身性**：
   - 我的身份、背景如何影响研究？

3. **政治反身性**：
   - 研究的权力关系是什么？

4. **认识论反身性**：
   - 我如何知道我所知道的？

**反身性实践**：
- 写田野日记
- 记录自己的感受和反应
- 与同行讨论
- 在民族志中呈现自己的角色

### 3.6 伦理

**田野工作伦理的核心原则**：

1. **知情同意（Informed Consent）**：
   - 解释研究目的
   - 解释参与方式
   - 解释风险和收益
   - 自愿参与

2. **匿名保护（Anonymity/Confidentiality）**：
   - 不透露真实姓名
   - 不透露可识别信息
   - 保护弱势群体

3. **不伤害原则**：
   - 研究不应伤害参与者
   - 考虑长期影响

4. **互利研究**：
   - 研究应惠及参与者
   - 回馈社区

5. **诚实与诚信**：
   - 不伪造数据
   - 诚实报告发现

**伦理审查**：
- 机构审查委员会（IRB）
- 伦理审查流程
- 知情同意书

**伦理困境**：
- 知情同意 vs 自然观察
- 匿名 vs 可识别性
- 忠诚于谁：参与者 vs 学术？
- 发现非法活动怎么办？

## 4. 主要方法与立场

### 4.1 传统民族志田野

**特征**：
- 长期（12-24个月）
- 单一地点
- 深度参与
- 整体视角

**经典案例**：
- 马林诺夫斯基：特罗布里恩群岛
- 埃文斯-普里查德：努尔人
- 格尔茨：巴厘岛、摩洛哥

**程序**：
1. 准备：文献回顾、语言学习
2. 进入：获得许可、建立关系
3. 主要田野期：参与观察、访谈、记录
4. 离开：告别、后续联系
5. 写作：民族志生产

### 4.2 多点民族志

**定义**：
在多个地点进行田野工作，追踪人、物、观念的流动。

**Marcus (1995) 的提出**：
- 全球化改变了田野的边界
- 需要跟随对象移动
- 不局限于"一个村庄"

**方法**：
1. **跟随人**：追踪移民、散居者
2. **跟随物**：追踪商品、技术
3. **跟随符号**：追踪观念、话语
4. **跟随隐喻**：追踪类比、叙事
5. **跟随情节**：追踪故事、冲突
6. **跟随生命**：追踪传记

**例子**：
- Appadurai：追踪全球文化流动
- Ong：追踪华人散居
- Tsing：追踪蘑菇供应链

### 4.3 虚拟/数字民族志

**定义**：
在在线环境中进行田野工作。

**方法**：
- 参与在线社区
- 在线访谈
- 数字内容分析
- 网络追踪

**类型**：

| 类型 | 场域 | 方法 |
|------|------|------|
| **MMORPG田野** | 虚拟世界 | 创建角色参与 |
| **社交媒体田野** | Facebook, Twitter等 | 参与观察、内容分析 |
| **在线社区田野** | Reddit, 论坛 | 长期参与 |
| **混合田野** | 线上+线下 | 结合方法 |

**代表研究**：
- Boellstorff (2008)：《第二人生中的到来》
- Hine (2000)：虚拟民族志
- Miller (2011)：Facebook研究

### 4.4 快速评估/短期田野

**背景**：
- 发展项目需要快速评估
- 资金和时间限制
- 应用人类学需求

**方法**：
- 结构化访谈
- 调查问卷
- 快速 rural appraisal (RRA)
- 参与 rural appraisal (PRA)

**争议**：
- 是否牺牲深度？
- 数据质量如何？
- 与传统田野的关系？

### 4.5 行动研究/参与式田野

**定义**：
田野工作不只是研究，还促进社会变革。

**方法**：
- 参与者作为合作者
- 共同定义研究问题
- 行动导向
- 回馈社区

**代表**：
- **Fals Borda**：参与行动研究
- **Freire**：批判教育学
- **应用人类学**：解决实际问题

## 5. 当代争论

### 5.1 田野工作的危机？

**问题提出**：
- 传统田野工作是否仍然可能？
- 全球化是否改变了田野的性质？
- 数字时代是否需要新方法？

**批评**：
1. **"原始"田野消失**：没有"孤立"的文化
2. **权力不平等**：田野工作延续殖民关系
3. **表征危机**：如何公正代表？
4. **伦理问题**：知情同意的困难

**回应**：
1. **适应而非放弃**：田野工作在变化
2. **多点田野**：适应全球化
3. **数字田野**：适应技术变化
4. **反思性实践**：承认权力关系

### 5.2 本土人类学

**定义**：
