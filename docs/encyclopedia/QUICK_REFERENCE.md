### 2. 如何阅读词条

**词条结构**（每个词条遵循相同格式）：

```
1. 摘要 (500-1000字)
   └─ 快速了解词条核心内容

2. 历史发展 (1500-2500字)
   ├─ 古典起源
   ├─ 中世纪发展
   ├─ 近代转型
   ├─ 现代形成
   └─ 当代发展 (2000-2026)

3. 核心理论与概念 (2000-3000字)
   ├─ 基本定义
   ├─ 核心要素
   ├─ 类型学
   └─ 关键命题

4. 主要理论立场 (1500-2000字)
   └─ 对比表格 + 详细阐述

5. 当代争论 (1500-2500字)
   ├─ 核心争议问题
   └─ 最新研究进展 (2020-2026)

6. 跨学科应用 (1000-1500字)
   └─ 在其他学科的应用

7. 非西方视角 (1000-1500字)
   ├─ 中国传统观点
   ├─ 印度哲学
   ├─ 阿拉伯传统
   └─ 其他视角

8. 中国学术贡献 (800-1200字)
   ├─ 古代思想
   ├─ 近现代研究
   └─ 当代学者 (2000-2026)

9. 方法论与工具 (800-1000字)

10. 未来发展方向 (500-800字)

11. 批判性分析 (800-1000字)
    ├─ 理论局限
    ├─ 方法论问题
    └─ 意识形态批判

12. 术语表 (核心术语)

13. 参考文献 (30-80条)

14. 相关词条 (10-20个)
```

**阅读建议**：
- **快速了解**：只读摘要和核心概念
- **系统学习**：从头到尾完整阅读
- **研究参考**：重点看当代争论和参考文献
- **跨学科**：查看跨学科应用部分

## ❓ 常见问题

**Q1：词条为什么这么长？**
A：遵循 Stanford Encyclopedia of Philosophy 的标准，每个词条是完整的学术文章，需要详尽覆盖历史、理论、争论、应用等各方面。

**Q2：如何找到最新研究？**
A：每个词条都有"5.2 最新研究进展 (2020-2026)"部分，列出最新的重要研究。

**Q3：为什么强调中国视角？**
A：传统百科全书往往以西方为中心，本百科全书特别强调非西方（包括中国）的学术贡献，以提供更全面的视角。

**Q4：如何贡献词条？**
A：参见 [FRAMEWORK_v3.md](./FRAMEWORK_v3.md) 中的"协作指南"部分。

**Q5：词条如何更新？**
A：每个词条都有"最后更新"日期，定期修订以反映最新研究进展。
