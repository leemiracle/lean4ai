## 一、半导体物理基础

### 1.1 半导体定义与特性

**半导体（Semiconductor）**

**简单解释**：介于导体和绝缘体之间的材料，导电能力可以通过掺杂、温度、光照等方式调节，是现代电子器件的基础。

**基本特性**：
- 电阻率：10⁻³ - 10⁸ Ω·cm（介于导体和绝缘体之间）
- 负温度系数：温度升高，电阻降低
- 光电导效应：光照增加导电性
- 掺杂效应：微量杂质大幅改变导电性

**主要半导体材料**：

| 材料 | 禁带宽度(eV) | 迁移率(cm²/V·s) | 应用 |
|------|-------------|----------------|------|
| 硅（Si）| 1.12 | 电子1400，空穴450 | 主流 |
| 锗（Ge）| 0.66 | 电子3900，空穴1900 | 早期、红外 |
| 砷化镓（GaAs）| 1.43 | 电子8500 | 射频、LED |
| 磷化铟（InP）| 1.35 | 电子5400 | 高速、光通信 |
| 碳化硅（SiC）| 3.26 | 电子800 | 功率器件 |
| 氮化镓（GaN）| 3.40 | 电子2000 | 射频、功率 |
| 氧化镓（Ga₂O₃）| 4.9 | - | 超高功率 |

### 1.2 能带理论

**能带结构**

**简单解释**：电子在原子中的能量是分层排列的，形成能带。能带之间的间隙叫禁带，决定了材料的导电性。

```
能带结构
├── 导带（Conduction Band）
│   └── 电子可以自由移动
├── 禁带（Band Gap，Eg）
│   └── 电子不能存在的能量区间
└── 价带（Valence Band）
    └── 束缚电子

材料分类：
- 导体：导带与价带重叠（无禁带）
- 半导体：禁带小（0.1-4 eV）
- 绝缘体：禁带大（>4 eV）
```

**禁带宽度（Band Gap）**：

| 材料 | Eg (eV) | 波长(nm) | 光电应用 |
|------|---------|----------|---------|
| Ge | 0.66 | 1880 | 红外探测器 |
| Si | 1.12 | 1107 | 可见光-近红外 |
| GaAs | 1.43 | 867 | LED、太阳电池 |
| InP | 1.35 | 919 | 光通信 |
| GaN | 3.40 | 365 | 蓝光LED、紫外 |
| SiC | 3.26 | 380 | 紫外 |

### 1.3 载流子

**本征载流子浓度（Intrinsic Carrier Concentration）**

**简单解释**：纯净半导体中，电子和空穴数量相等，由热激发产生。

```
ni = √(Nc × Nv) × exp(-Eg/2kT)

ni = 本征载流子浓度 [cm⁻³]
Nc = 导带有效态密度 [cm⁻³]
Nv = 价带有效态密度 [cm⁻³]
Eg = 禁带宽度 [eV]
k = 玻尔兹曼常数 = 8.617×10⁻⁵ eV/K
T = 绝对温度 [K]

Si (300K)：
ni = 1.5×10¹⁰ cm⁻³
Nc = 2.8×10¹⁹ cm⁻³
Nv = 1.04×10¹⁹ cm⁻³
```

**载流子迁移率（Mobility）**

**简单解释**：电子或空穴在电场作用下移动的快慢。

```
v = μE

v = 漂移速度 [cm/s]
μ = 迁移率 [cm²/V·s]
E = 电场强度 [V/cm]

电子迁移率 > 空穴迁移率（同种材料）

Si：μn = 1400，μp = 450 cm²/V·s
GaAs：μn = 8500，μp = 400 cm²/V·s
```

**迁移率影响因素**：
- 温度：μ ∝ T^(-m)（m = 1.5-2.5）
- 掺杂浓度：高掺杂降低迁移率
- 晶格散射
- 杂质散射

### 1.4 掺杂（Doping）

**简单解释**：在纯净半导体中加入少量杂质原子，改变导电性和载流子类型。

**掺杂类型**：

| 类型 | 掺杂元素 | 载流子 | 半导体类型 |
|------|---------|--------|-----------|
| N型 | P, As, Sb（V族）| 电子 | 电子导电 |
| P型 | B, Al, Ga（III族）| 空穴 | 空穴导电 |

**掺杂浓度**：

| 级别 | 浓度范围(cm⁻³) | 电阻率(Ω·cm) | 应用 |
|------|---------------|-------------|------|
| 轻掺杂 | 10¹⁴-10¹⁶ | 10-100 | 衬底 |
| 中掺杂 | 10¹⁶-10¹⁸ | 0.1-10 | 有源区 |
| 重掺杂 | 10¹⁸-10²⁰ | 0.001-0.1 | 欧姆接触 |
| 超重掺杂 | >10²⁰ | <0.001 | 源漏区 |

**费米能级（Fermi Level）**

**简单解释**：电子占据概率为50%的能量水平。

```
本征半导体：Ef = Ei（禁带中央）
N型半导体：Ef > Ei（靠近导带）
P型半导体：Ef < Ei（靠近价带）

N型：Ef - Ei = kT ln(Nd/ni)
P型：Ei - Ef = kT ln(Na/ni)

Nd = 施主浓度
Na = 受主浓度
```

### 1.5 PN结

**PN结形成**

**简单解释**：P型和N型半导体接触，载流子扩散，形成耗尽区和内建电场。

```
PN结结构
├── P区
│   └── 多子：空穴，少子：电子
├── 耗尽区（空间电荷区）
│   ├── 无自由载流子
│   ├── 内建电场（N→P）
│   └── 势垒电压
└── N区
    └── 多子：电子，少子：空穴
```

**内建电势（Built-in Potential）**：

```
Vbi = (kT/q) × ln(Na×Nd/ni²)

Vbi = 内建电势 [V]
kT/q = 0.026 V (300K)

Si PN结：
Vbi ≈ 0.6-0.7 V（室温）
```

**耗尽区宽度**：

```
W = √(2ε×Vbi/(q) × (Na+Nd)/(Na×Nd))

W = 总耗尽区宽度
ε = 介电常数 = εr×ε0
εr(Si) = 11.7
ε0 = 8.85×10⁻¹⁴ F/cm

不对称结（Na >> Nd）：
WN >> WP
W ≈ √(2ε×Vbi/(qNd))
```

**PN结电容**：

```
Cj = εA/W

Cj = 结电容 [F]
A = 结面积 [cm²]
W = 耗尽区宽度 [cm]

反向偏置：W增大，Cj减小
正向偏置：W减小，Cj增大（扩散电容主导）
```

### 1.6 载流子输运

**漂移电流（Drift Current）**：

```
Jdrift = q(nμn + pμp)E

J = 电流密度 [A/cm²]
n, p = 电子、空穴浓度 [cm⁻³]
μn, μp = 电子、空穴迁移率 [cm²/V·s]
E = 电场强度 [V/cm]
```

**扩散电流（Diffusion Current）**：

```
Jdiff = qDn(dn/dx) + qDp(dp/dx)

D = 扩散系数 [cm²/s]
D = (kT/q)μ（爱因斯坦关系）

Si (300K)：
Dn = 35 cm²/s
Dp = 12 cm²/s
```

**电流连续性方程**：

```
∂n/∂t = (1/q)∂Jn/∂x + Gn - Rn
∂p/∂t = -(1/q)∂Jp/∂x + Gp - Rp

G = 产生率 [cm⁻³s⁻¹]
R = 复合率 [cm⁻³s⁻¹]
```

**载流子寿命（Carrier Lifetime）**：

```
R = Δn/τ（过剩载流子复合）

τ = 载流子寿命 [s]

Si：τ = 10⁻⁶ - 10⁻³ s（取决于掺杂和缺陷）
GaAs：τ = 10⁻⁹ - 10⁻⁶ s
```

**GaN-on-Si**：
- 在硅衬底上生长GaN

**GaN-on-SiC**：

### 2.4 材料制备

**硅提纯**：

```
冶金级硅（MGS，98-99%）
    ↓ Siemens法
多晶硅（EGS，9N-11N）
    ↓ 直拉法/区熔法
单晶硅锭
    ↓ 切片、抛光
硅晶圆
```

**直拉法（CZ，Czochralski）**：

```
工艺流程：
1. 多晶硅+掺杂剂装入石英坩埚
2. 加热熔化（~1420°C）
3. 籽晶接触熔体
4. 缓慢提拉并旋转
5. 单晶生长
6. 冷却

参数：
- 提拉速度：1-2 mm/min
- 转速：10-40 rpm
- 晶锭直径：150-300 mm
- 晶锭长度：1-2 m

特点：
- 氧含量高（10¹⁷-10¹⁸ cm⁻³）
- 成本低
- 主流工艺
```

**区熔法（FZ，Float Zone）**：

```
工艺流程：
1. 多晶硅棒
2. 高频感应加热局部熔化
3. 熔区从下向上移动
4. 杂质随熔区移动
5. 单晶生长

特点：
- 氧含量低（<10¹⁶ cm⁻³）
- 纯度高
- 电阻率高
- 成本高
- 应用：功率器件
```

**外延生长（Epitaxy）**：

```
外延：在单晶衬底上生长单晶层

类型：
├── 同质外延：Si on Si
└── 异质外延：GaAs on Si，GaN on SiC

方法：
├── CVD（化学气相沉积）
│   └── SiH₄ + heat → Si + 2H₂
├── MBE（分子束外延）
│   └── 超高真空，精确控制
└── MOCVD（金属有机化学气相沉积）
    └── 化合物半导体主流

外延层厚度：0.1-100 μm
掺杂浓度：10¹⁴-10²⁰ cm⁻³
```

## 三、PN结与二极管

### 3.1 二极管工作原理

**简单解释**：PN结加正向电压导通，加反向电压截止，像单向阀门。

**正向偏置**：
```
VF > 0：势垒降低，载流子注入，大电流

IF = IS(exp(qVF/kT) - 1)

IF = 正向电流
IS = 反向饱和电流
VF = 正向电压

当 VF > 3kT/q ≈ 0.078 V (300K)：
IF ≈ IS × exp(qVF/kT)

VF = (kT/q) × ln(IF/IS)
```

**反向偏置**：
```
VR < 0：势垒增大，仅少数载流子漂移

IR ≈ -IS

IS = A × ni² × (Dn/(LnNa) + Dp/(LpNd))

A = 结面积
L = 扩散长度 = √(Dτ)
```

**击穿机制**：

| 机制 | 条件 | 原理 |
|------|------|------|
| 齐纳击穿 | BV < 4V | 量子隧穿 |
| 雪崩击穿 | BV > 6V | 碰撞电离 |
| 混合 | 4-6V | 两者都有 |

### 3.2 二极管类型

**整流二极管**：

| 参数 | 典型值 |
|------|--------|
| 正向压降 | 0.6-0.7 V (Si) |
| 反向击穿电压 | 50-1000 V |
| 反向恢复时间 | 1-10 μs |
| 电流范围 | 1 mA-100 A |

**肖特基二极管（Schottky）**：

**简单解释**：金属-半导体结，无少子存储，开关速度快，正向压降低。

| 参数 | 典型值 |
|------|--------|
| 正向压降 | 0.2-0.4 V |
| 反向漏电 | 较大 |
| 反向恢复时间 | <100 ps |
| 应用 | 高频整流、钳位 |

**稳压二极管（Zener）**：

| 击穿电压 | 应用 |
|---------|------|
| 2.4-200 V | 稳压、保护 |
| <5V | 齐纳击穿 |
| >5V | 雪崩击穿 |

**变容二极管（Varactor）**：

```
Cj = Cj0 / (1 + VR/Vbi)^m

m = 0.33-0.5（突变结）
m = 0.5（线性缓变结）

应用：压控振荡器（VCO）、调谐
```

**发光二极管（LED）**：

| 颜色 | 材料 | 波长(nm) |
|------|------|---------|
| 红 | GaAsP | 650 |
| 橙 | GaAsP | 610 |
| 黄 | GaAsP | 590 |
| 绿 | GaP | 555 |
| 蓝 | GaN | 470 |
| 白 | GaN+荧光粉 | - |

**光电二极管**：

```
光生电流：
Iph = q × η × P/(hν) = q × η × Pλ/(hc)

η = 量子效率
P = 光功率 [W]
λ = 波长 [m]

响应率：
R = Iph/P = ηqλ/(hc) [A/W]

Si光电二极管：R ≈ 0.5 A/W (λ=900 nm)
```

### 3.3 二极管模型

**大信号模型**：

```
理想模型：
VF < 0：IF = 0
VF ≥ 0：VF = 0

恒压降模型：
VF < Vd：IF = 0
VF ≥ Vd：VF = Vd (≈0.7 V)

分段线性模型：
VF < Vd：IF = 0
VF ≥ Vd：IF = (VF - Vd)/rd

rd = 正向电阻（≈10-20 Ω）
```

**小信号模型**：

```
rd = nVT/ID = nkT/(qID)

n = 理想因子（1-2）
VT = kT/q = 26 mV (300K)
ID = 直流工作点电流

rd (ID=1mA) = 26 Ω
rd (ID=10mA) = 2.6 Ω

结电容：
Cd = τ/rd + Cj

τ = 渡越时间
```

### 3.4 特殊二极管

**隧道二极管（Tunnel Diode）**：

**简单解释**：重掺杂PN结，量子隧穿效应，负阻区，超高频振荡。

| 参数 | 数值 |
|------|------|
| 峰值电流 | 1-100 mA |
| 负阻区 | 0.1-1 V |
| 频率 | >100 GHz |
| 应用 | 振荡器、放大器 |

**反向二极管（Backward Diode）**：

- 重掺杂，但不如隧道二极管
- 反向导通优于正向
- 应用：小信号检波

**PIN二极管**：

```
结构：P+ - I（本征）- N+

特点：
- I层：高电阻率，低掺杂
- 正向：I层注入载流子，低阻
- 反向：I层耗尽，高阻

应用：
- 射频开关
- 衰减器
- 光电探测器
```

## 四、晶体管基础

### 4.1 双极型晶体管（BJT）

**简单解释**：由三个掺杂区（发射极、基极、集电极）组成，小电流控制大电流，是电流控制器件。

**结构**：

```
NPN晶体管
├── 发射极（Emitter，N+）
│   └── 高掺杂，注入电子
├── 基极（Base，P）
│   └── 薄（<1 μm），轻掺杂
└── 集电极（Collector，N）
    └── 中等掺杂

PNP晶体管：极性相反
```

**工作原理**：

```
放大模式：
1. EB结正偏：电子注入基区
2. 基区薄：大部分电子到达CB结
3. CB结反偏：电子被集电极收集

电流关系：
IC = βIB + ICEO ≈ βIB
IE = IC + IB = (1+β)IB

β = IC/IB = 电流增益 = 50-300
α = IC/IE = β/(1+β) ≈ 0.99
```

**Ebers-Moll模型**：

```
IC = IS(exp(VBE/VT) - exp(VBC/VT)) - ICBO

IS = 饱和电流
VT = kT/q = 26 mV

正向有源区（VBE > 0, VBC < 0）：
IC ≈ IS exp(VBE/VT)
IB ≈ IS/βF exp(VBE/VT)
```

**工作区域**：

| 区域 | 偏置条件 | 特点 |
|------|---------|------|
| 截止区 | VBE < 0, VBC < 0 | IC ≈ 0 |
| 放大区 | VBE > 0, VBC < 0 | IC = βIB |
| 饱和区 | VBE > 0, VBC > 0 | VCE(sat) ≈ 0.2 V |
| 反向有源 | VBE < 0, VBC > 0 | βR << βF |

**小信号参数**：

```
跨导：
gm = dIC/dVBE = IC/VT = IC/26mV

gm (IC=1mA) = 38.5 mS
gm (IC=10mA) = 385 mS

输入电阻：
rπ = β/gm = βVT/IC

输出电阻：
ro = VA/IC

VA = Early电压 = 50-200 V

结电容：
Cπ = Cje + gmτF
Cμ = Cjc
```

### 4.2 MOSFET

**简单解释**：金属-氧化物-半导体场效应晶体管，电压控制器件，栅极电压控制沟道导电性。

**结构**：

```
N沟道MOSFET（NMOS）
├── 衬底（P-type）
├── 源极（Source，N+）
├── 漏极（Drain，N+）
├── 栅极（Gate）
│   └── 金属/多晶硅
├── 栅氧化层（SiO₂）
│   └── 厚度：1-10 nm
└── 沟道：栅极下方

P沟道MOSFET（PMOS）：掺杂相反
```

**工作原理**：

```
1. VGS < VTH：无沟道，ID ≈ 0（截止）
2. VGS ≥ VTH：反型层形成，沟道导通
3. VDS小：线性区（电阻）
4. VDS大：饱和区（恒流）

阈值电压：
VTH = VFB + 2φF + √(2εsiqNA(2φF))/Cox

VFB = 平带电压
φF = (kT/q)ln(NA/ni)
Cox = 栅氧化层电容/面积
```

**电流-电压特性**：

```
线性区（VDS < VGS - VTH）：
ID = μnCox(W/L)[(VGS-VTH)VDS - VDS²/2]

饱和区（VDS ≥ VGS - VTH）：
ID = (μnCox/2)(W/L)(VGS-VTH)²(1+λVDS)

λ = 沟道长度调制系数
μn = 电子迁移率
Cox = εox/tox
W/L = 宽长比
```

**MOSFET参数**：

| 参数 | NMOS | PMOS |
|------|------|------|
| 阈值电压 | 0.3-0.7 V | -0.3 to -0.7 V |
| 迁移率 | μn = 1400 | μp = 450 |
| 跨导 | gm = μnCox(W/L)(VGS-VTH) | - |
| 输出电阻 | ro = 1/(λID) | - |

**小信号模型**：

```
gm = dID/dVGS = √(2μCox(W/L)ID)（饱和区）

gds = 1/ro = dID/dVDS = λID

栅电容：
Cgs = (2/3)WL Cox（饱和区）
Cgd = WLCov（覆盖电容）
```

**体效应（Body Effect）**：

```
VTH = VTH0 + γ(√(2φF + VSB) - √(2φF))

γ = 体效应系数 = √(2εsiqNA)/Cox

VSB = 源-衬底电压
```

### 4.3 CMOS技术

**简单解释**：NMOS和PMOS配对使用，静态功耗低，是数字集成电路主流技术。

**CMOS反相器**：

```
电路：
VIN → [PMOS上拉] → VOUT
    → [NMOS下拉]

工作：
- VIN = 0：PMOS导通，NMOS截止，VOUT = VDD
- VIN = VDD：PMOS截止，NMOS导通，VOUT = 0

特点：
- 静态功耗≈0（无直流通路）
- 动态功耗：CLVDD²f
- 噪声容限大
```

**转移特性**：

| 区域 | 条件 | 输出 |
|------|------|------|
| PMOS导通，NMOS截止 | VIN < VTHN | VOUT = VDD |
| PMOS线性，NMOS饱和 | VTHN < VIN < VINV | VOUT下降 |
| 转换点 | VIN = VINV | VOUT = VINV ≈ VDD/2 |
| PMOS饱和，NMOS线性 | VINV < VIN < VDD-|VTHP| | VOUT下降 |
| PMOS截止，NMOS导通 | VIN > VDD-|VTHP| | VOUT = 0 |

**开关特性**：

```
上升时间：
tLH = CL × VDD / (μpCox(W/L)p × (VDD/2)²)

下降时间：
tHL = CL × VDD / (μnCox(W/L)n × (VDD/2)²)

传播延迟：
tp = (tLH + tHL)/4

优化：(W/L)p = 2-3 × (W/L)n（补偿μp < μn）
```

**功耗**：

```
静态功耗：
Pstatic = VDD × Ileak

Ileak：亚阈值漏电、栅漏电、结漏电

动态功耗：
Pdynamic = CL × VDD² × f

f：开关频率

短路功耗：
Pshort = β/12 × (VDD - 2VT)³ × τ × f

总功耗：
Ptotal = Pstatic + Pdynamic + Pshort
```

### 4.4 其他晶体管

**JFET（结型场效应晶体管）**：

```
结构：P-N结栅极

工作：
- VGS < 0：沟道夹断
- VGS = 0：最大ID

特点：
- 输入阻抗高（反偏PN结）
- 噪声低
- 应用：射频、音频
```

**HEMT（高电子迁移率晶体管）**：

**简单解释**：利用异质结形成二维电子气（2DEG），迁移率极高。

```
结构：AlGaAs/GaAs异质结

特点：
- 2DEG迁移率：>10⁴ cm²/V·s
- 高频性能优异
- 噪声低

应用：
- 5G基站
- 卫星通信
- 雷达
```

**FinFET（鳍式场效应晶体管）**：

**简单解释**：3D结构，栅极包裹沟道，更好的栅极控制，是14nm以下主流技术。

```
结构：
- 鳍状沟道（Fin）
- 三面栅极包围
- 更好的沟道控制

优势：
- 抑制短沟道效应
- 降低漏电
- 22nm以下主流
```

**GAA（Gate-All-Around）**：

```
结构：纳米线/纳米片完全被栅极包围

优势：
- 最佳栅极控制
- 进一步缩小
- 3nm以下主流
```

## 六、光刻技术

### 6.1 光刻原理

**简单解释**：用光在硅片上"画"图案，类似照相技术，是芯片制造最关键、最昂贵的工艺。

**基本流程**：

```
光刻流程
├── 1. 清洗
├── 2. 底胶（HMDS）
├── 3. 旋涂光刻胶
├── 4. 前烘（软烘）
├── 5. 对准曝光
├── 6. 后烘（PEB）
├── 7. 显影
├── 8. 硬烘
└── 9. 检验
```

**光刻胶（Photoresist）**：

| 类型 | 反应 | 应用 |
|------|------|------|
| 正胶（Positive）| 曝光部分溶解 | 主流 |
| 负胶（Negative）| 曝光部分不溶 | 特殊应用 |

**光刻胶参数**：

```
厚度：0.1-5 μm
- 先进工艺：30-100 nm
- 传统工艺：1-3 μm

旋涂：
t = k × ω^(-1/2)

t = 厚度
ω = 转速（rpm）
k = 常数

前烘：
- 温度：90-120°C
- 时间：60-90 s
- 目的：去除溶剂

分辨率：
R = k₁ × λ/NA

k₁ = 工艺因子（0.25-0.8）
λ = 波长
NA = 数值孔径

极限分辨率：
- i线(365nm)：0.35 μm
- KrF(248nm)：0.25 μm
- ArF(193nm)：0.15 μm
- EUV(13.5nm)：<10 nm
```

### 6.2 曝光系统

**投影式光刻（主流）**：

```
系统组成：
├── 光源
│   ├── 汞灯（g/h/i线）
│   ├── 准分子激光（KrF, ArF）
│   └── EUV源
├── 照明系统
│   └── 均匀化、整形
├── 掩膜版（Mask/Reticle）
│   └── 4×缩小
├── 投影镜头
│   └── NA=0.75-1.35
└── 硅片台
    └── 精确定位

曝光方式：
├── 步进式（Stepper）
│   └── 逐场曝光
└── 步进扫描（Scanner）
    └── 硅片和掩膜版同步移动
```

**数值孔径（NA）**：

```
NA = n × sin(θ)

n = 介质折射率
θ = 半孔径角

提高NA方法：
1. 大孔径角（复杂镜头）
2. 浸没式（n>1）

浸没式光刻：
- 介质：去离子水（n=1.44）
- NAmax = 1.44 × sin(θ)
- 193nm浸没：NA = 1.35
```

**分辨率增强技术（RET）**：

| 技术 | 原理 | 效果 |
|------|------|------|
| OPC（光学邻近效应校正）| 掩膜图形预畸变 | 改善图形保真 |
| PSM（相移掩膜）| 利用相位相消 | 提高对比度 |
| OAI（离轴照明）| 倾斜照明 | 提高分辨率 |
| 多重曝光 | 图形分解 | 突破分辨率限制 |

**多重曝光技术**：

```
双重曝光（LELE）：
1. 曝光图形1
2. 刻蚀图形1
3. 曝光图形2
4. 刻蚀图形2
- 节距减半

自对准双重曝光（SADP）：
1. 曝光图形
2. 沉积侧墙
3. 去除核心图形
4. 刻蚀
- 节距减半，精度高

自对准四重曝光（SAQP）：
- 节距减为1/4
- 7nm、5nm主流
```

### 6.3 EUV光刻

**EUV光源**：

**简单解释**：极紫外光（13.5nm波长），比传统光刻波长短一个数量级，是7nm以下工艺必需技术。

```
产生原理：
CO₂激光 → Sn液滴 → 等离子体 → EUV光

参数：
- 波长：13.5 nm
- 带宽：2% (13.3-13.7 nm)
- 功率：>250 W（输出）
- 重复频率：50 kHz

Sn液滴：
- 直径：20-30 μm
```

## 七、刻蚀工艺

### 7.1 刻蚀原理

**简单解释**：用化学或物理方法去除材料，在硅片上形成图案，是光刻后的关键工艺。

**刻蚀类型**：

| 类型 | 原理 | 特点 |
|------|------|------|
| 湿法刻蚀 | 化学溶液腐蚀 | 各向同性、选择比高 |
| 干法刻蚀 | 等离子体 | 各向异性、精度高 |

**刻蚀参数**：

```
刻蚀速率（ER）：
ER = 厚度/时间 [nm/min]

选择比（Selectivity）：
S = ER材料1/ER材料2

各向异性（Anisotropy）：
A = 1 - ER横向/ER纵向

A = 1：完全各向异性（垂直）
A = 0：各向同性

刻蚀均匀性：
U = (ERmax - ERmin)/(2 × ERavg) × 100%

目标：U < ±5%
```

### 7.2 湿法刻蚀

**常用刻蚀液**：

| 材料 | 刻蚀液 | 速率(nm/min) | 应用 |
|------|--------|------------|------|
| Si | KOH、TMAH | 500-2000 | MEMS |
| SiO₂ | HF、BOE | 10-1000 | 牺牲层 |
| Si₃N₄ | H₃PO₄ | 5-10 | 释放 |
| Al | H₃PO₄+HNO₃ | 100-1000 | 互连 |
| Cu | H₂SO₄+H₂O₂ | 500-2000 | 互连 |

**各向同性刻蚀**：

```
Si湿法刻蚀（各向同性）：
HNA：HF + HNO₃ + CH₃COOH

反应：
Si + HNO₃ → SiO₂（氧化）
SiO₂ + HF → H₂SiF₆（溶解）

刻蚀速率：1-50 μm/min（取决于配比）
```

**各向异性刻蚀**：

```
Si各向异性刻蚀：
KOH、TMAH、EDP

晶面刻蚀速率：
(110) >> (100) > (111)

KOH (80°C, 30%)：
(100)：1.4 μm/min
(110)：3.0 μm/min
(111)：0.007 μm/min

应用：
- MEMS结构
- 硅通孔（TSV）
- 背腔刻蚀
```

### 7.3 干法刻蚀（RIE）

**反应离子刻蚀（RIE）原理**：

**简单解释**：等离子体产生的离子轰击+化学反应，实现各向异性刻蚀。

```
RIE系统组成：
├── 真空腔体
├── RF电源（13.56 MHz）
├── 上电极（接地）
├── 下电极（RF，衬底）
├── 气体供应
└── 真空系统

等离子体参数：
- 功率密度：0.1-5 W/cm²
- 压力：1-100 mTorr
- 偏压：100-800 V
- 电子温度：2-5 eV
- 离子密度：10⁹-10¹¹ cm⁻³
```

**Si刻蚀**：

```
气体：SF₆、CF₄、CHF₃、Cl₂、HBr

反应：
SF₆ → SFₓ + F*
Si + 4F* → SiF₄↑

氟基刻蚀：
- 速率快（100-500 nm/min）
- 各向同性（加聚合物实现各向异性）

氯基刻蚀：
- 各向异性
- 速率慢（10-100 nm/min）
- 用于栅极、浅槽

Bosch工艺（深反应离子刻蚀，DRIE）：
1. 刻蚀：SF₆（各向同性）
2. 钝化：C₄F₈（聚合物侧壁）
3. 重复1-2

特点：
- 高深宽比（>20:1）
- 垂直侧壁
- 刻蚀速率：1-10 μm/min
- 应用：TSV、MEMS
```

**SiO₂刻蚀**：

```
气体：CF₄、CHF₃、C₄F₈

反应：
CF₄ → CF₃* + F*
SiO₂ + 4F* → SiF₄↑ + O*
O* + C → CO、CO₂

选择比：
SiO₂:Si = 10:1 - 50:1（CHF₃）
SiO₂:光刻胶 = 5:1 - 20:1

刻蚀速率：50-200 nm/min
```

**Si₃N₄刻蚀**：

```
气体：CHF₃、CH₂F₂

反应：类似SiO₂

选择比：
Si₃N₄:SiO₂ = 2:1 - 5:1
Si₃N₄:Si = 5:1 - 10:1

应用：接触孔刻蚀停止
```

**金属刻蚀**：

```
Al刻蚀：
气体：Cl₂、BCl₃、CHCl₃
反应：Al + 3Cl* → AlCl₃↑
速率：200-500 nm/min

W刻蚀：
气体：SF₆、CF₄
反应：W + 6F* → WF₆↑
速率：100-300 nm/min

Cu刻蚀：
- 干法刻蚀困难（CuCl₂不挥发）
- 主流：CMP（化学机械抛光）
```

### 7.4 高级刻蚀技术

**电感耦合等离子体（ICP）**：

```
特点：
- 双RF源
- 高密度等离子体（>10¹¹ cm⁻³）
- 独立控制离子能量和密度

优势：
- 刻蚀速率高
- 损伤小
- 选择比高
```

**深反应离子刻蚀（DRIE）**：

```
Bosch工艺（如前述）

Cryogenic刻蚀：
- 温度：-100°C
- 气体：SF₆ + O₂
- 侧壁钝化：SiOxFy
- 一次刻蚀，无需循环

性能：
- 深宽比：>30:1
- 垂直度：>89.5°
- 速率：1-10 μm/min
```

**原子层刻蚀（ALE）**：

**简单解释**：逐层刻蚀，精度达到原子级别，是先进工艺必需技术。

```
流程：
1. 自限制吸附（氯气）
2. 等离子体活化
3. 去除

特点：
- 自限制反应
- 原子级精度
- 选择比高
- 速率慢

应用：
- 7nm以下
- 栅极、沟道
```

## 八、薄膜沉积

### 8.1 沉积技术概述

**简单解释**：在硅片表面"生长"一层薄膜，材料包括绝缘层、金属、半导体。

**技术分类**：

| 类型 | 原理 | 典型厚度 | 应用 |
|------|------|---------|------|
| CVD（化学气相沉积）| 化学反应 | 10 nm-10 μm | 绝缘层、多晶硅 |
| PVD（物理气相沉积）| 物理过程 | 5 nm-1 μm | 金属 |
| ALD（原子层沉积）| 自限制反应 | 0.1-50 nm | 超薄、高保形 |
| ECD（电化学沉积）| 电镀 | 0.5-10 μm | Cu互连 |

### 8.2 化学气相沉积（CVD）

**CVD原理**：

```
反应过程：
1. 气体输入
2. 扩散到表面
3. 吸附
4. 表面反应
5. 副产物脱附
6. 排出

沉积速率：
R = k × Preactant^n

k = 速率常数
P = 反应物分压
n = 反应级数
```

**CVD类型**：

| 类型 | 压力 | 温度 | 特点 |
|------|------|------|------|
| APCVD | 常压 | 400-800°C | 产量高 |
| LPCVD | 0.1-10 Torr | 600-900°C | 均匀性好 |
| PECVD | 0.1-10 Torr | 200-400°C | 低温 |
| HDPCVD | 1-10 mTorr | 300-500°C | 高密度等离子体 |

**SiO₂沉积**：

```
APCVD：
SiH₄ + O₂ → SiO₂ + 2H₂
温度：400-500°C
速率：100-1000 nm/min

LPCVD：
SiH₄ + 2N₂O → SiO₂ + 2N₂ + 2H₂
温度：800-900°C
速率：10-20 nm/min
台阶覆盖：差

PECVD：
SiH₄ + N₂O → SiO₂ + ...
温度：300-400°C
速率：50-300 nm/min
台阶覆盖：中等
应力：可调

HDPCVD：
SiH₄ + O₂/He → SiO₂ + ...
温度：300-500°C
台阶覆盖：好
应用：STI、侧墙
```

**Si₃N₄沉积**：

```
LPCVD：
3SiH₂Cl₂ + 4NH₃ → Si₃N₄ + 6HCl + 6H₂
温度：700-800°C
速率：3-10 nm/min
应力：拉应力（~1 GPa）

PECVD：
SiH₄ + NH₃ → Si₃N₄ + ...
温度：300-400°C
速率：20-100 nm/min
应力：可调（-1 to +1 GPa）
应用：钝化层、侧墙
```

**多晶硅沉积**：

```
LPCVD：
SiH₄ → Si + 2H₂
温度：600-650°C
压力：100-300 mTorr
速率：5-20 nm/min
晶粒：细晶到粗晶

掺杂：PH₃、B₂H₆（in-situ）

应用：
- 栅极
- 电阻
- 提升源漏
```

### 8.3 物理气相沉积（PVD）

**溅射（Sputtering）原理**：

**简单解释**：用离子"轰击"靶材，把原子打出来沉积到硅片上。

```
DC溅射：
Ar⁺（轰击）→ 靶材原子 → 硅片

RF溅射：
- 用于绝缘靶材
- 频率：13.56 MHz

磁控溅射：
- 磁场约束等离子体
- 沉积速率高（5-50 nm/min）

参数：
- 功率密度：1-10 W/cm²
- Ar压力：1-10 mTorr
- 靶材-衬底距离：5-15 cm
```

**台阶覆盖**：

```
PVD台阶覆盖差：
- 遮蔽效应
- 再溅射
- 侧壁薄、底部厚

改善方法：
- 多源
- 衬底旋转
- 离化金属等离子体（IMP）
- 准直器

台阶覆盖比：侧壁/平面
- 溅射：10-50%
- ALD：>95%
```

**金属PVD**：

| 金属 | 应用 | 厚度 | 方法 |
|------|------|------|------|
| Al | 互连 | 0.3-1 μm | 溅射 |
| Ti | 粘附层、势垒 | 10-50 nm | 溅射 |
| TiN | 势垒、硬掩膜 | 20-100 nm | 反应溅射 |
| Ta/TaN | Cu势垒 | 5-20 nm | 溅射 |
| Co | 硅化物 | 5-20 nm | 溅射 |

### 8.4 原子层沉积（ALD）

**ALD原理**：

**简单解释**：一层一层沉积，每层反应自限制，厚度精确到原子级别。

```
ALD循环（以Al₂O₃为例）：
1. TMA脉冲（三甲基铝）
   - TMA吸附：Al(CH₃)₃* → Al(CH₃)₂* + CH₄
   - 吹扫：去除多余TMA
2. H₂O脉冲
   - 反应：Al(CH₃)₂* + H₂O → AlO* + 2CH₄
   - 吹扫：去除CH₄、多余H₂O
3. 重复1-2

每个循环：0.05-0.15 nm
速率：0.5-5 nm/min（取决于循环时间）
```

**ALD特点**：

```
优势：
- 厚度精确控制（单原子层）
- 极佳的台阶覆盖（>95%）
- 大面积均匀（±1%）
- 低温（100-400°C）

劣势：
- 沉积速率慢
- 设备成本高

应用：
- 高k栅介质（HfO₂）
- 势垒层（TaN）
- 侧墙
- 3D NAND
```

**ALD材料**：

| 材料 | 前驱体 | 温度(°C) | 应用 |
|------|--------|---------|------|
| Al₂O₃ | TMA + H₂O | 150-300 | 钝化、侧墙 |
| HfO₂ | TEMAH + H₂O | 200-400 | 高k栅介质 |
| ZrO₂ | TEMAZ + H₂O | 200-300 | 高k |
| TiN | TiCl₄ + NH₃ | 350-450 | 势垒 |
| TaN | TaCl₅ + NH₃ | 300-400 | Cu势垒 |

### 8.5 电化学沉积（ECD）

**铜电镀（Cu ECD）**：

**简单解释**：通过电化学反应在硅片表面沉积铜，是互连金属主流技术。

```
原理：
阴极（硅片）：Cu²⁺ + 2e⁻ → Cu
阳极（铜球）：Cu → Cu²⁺ + 2e⁻

电解液：
- CuSO₄：50-100 g/L
- H₂SO₄：50-100 g/L
- 添加剂：
  - 加速剂（PEG、SPS）
  - 抑制剂（PEG）
  - 平整剂（Janus Green B）

工艺参数：
- 电流密度：5-50 mA/cm²
- 温度：20-30°C
- 时间：30-120 s
- 厚度：0.5-5 μm

特点：
- 填孔能力（superfilling）
- 速率快（0.5-1 μm/min）
- 成本低
```

**铜互连流程（大马士革）**：

```
1. 刻蚀沟槽/通孔
2. 沉积势垒层（TaN，5-10 nm）
3. 沉积Cu种子层（PVD，20-50 nm）
4. Cu电镀（ECD，填孔）
5. CMP（去除多余Cu）
```

## 九、离子注入与掺杂

### 9.1 离子注入原理

**简单解释**：用电场加速离子，打入硅片，改变导电性，是最精确的掺杂方法。

```
离子注入系统
├── 离子源
│   └── 产生离子（BF₃、PH₃、AsH₃、B₂H₆）
├── 质量分析器
│   └── 磁场分离，选择离子
├── 加速系统
│   └── 电场加速（10 keV - 10 MeV）
├── 聚焦系统
│   └── 束流整形
├── 扫描系统
│   └── 均匀注入
└── 靶室
    └── 晶圆、真空
```

**注入参数**：

```
剂量（Dose）：
D = Q/A [ions/cm²]

Q = 总电荷 [C]
A = 注入面积 [cm²]

典型值：
- 沟道：10¹²-10¹³ cm⁻²
- 源漏：10¹⁵-10¹⁶ cm⁻²

能量（Energy）：
E = qV [keV]

q = 电子电荷
V = 加速电压

射程（Range）：
Rp ≈ (E^1.5)/k（LSS理论）

k = 常数

典型射程：
- 10 keV B: 0.1 μm
- 100 keV B: 1 μm
- 1 MeV B: 3 μm
```

### 9.2 注入分布

**高斯分布**：

```
n(x) = (D/√(2π)ΔRp) × exp(-(x-Rp)²/(2ΔRp²))

Rp = 投影射程
ΔRp = 标准偏差（straggle）

峰值浓度：
np = D/(√(2π)ΔRp)
```

**掺杂剂特性**：

| 掺杂剂 | 质量 | Rp（100keV）| 应用 |
|--------|------|------------|------|
| B（硼）| 11 | 0.3 μm | P型、浅结 |
| BF₂ | 49 | 0.1 μm | P型、超浅结 |
| P（磷）| 31 | 0.12 μm | N型 |
| As（砷）| 75 | 0.06 μm | N型、源漏 |
| Sb（锑）| 122 | 0.04 μm | N型、低扩散 |

**沟道效应**：

```
问题：离子沿晶格沟道穿透过深

解决方法：
1. 倾斜注入（7°倾角）
2. 损伤层（预非晶化）
3. 氧屏蔽数
```

### 9.3 退火

**快速热退火（RTA）**：

```
目的：
1. 修复晶格损伤
2. 激活掺杂剂

工艺：
- 温度：900-1100°C
- 时间：1-60 s
- 升温速率：50-200°C/s
- 气氛：N₂、Ar

激活率：
- B：>90%
- P：>90%
- As：>90%

缺点：
- 瞬态增强扩散（TED）
```

**高级退火技术**：

| 技术 | 时间 | 特点 | 应用 |
|------|------|------|------|
| RTA | 1-60 s | 传统 | 成熟工艺 |
| Spike退火 | <1 s | 快速升降温 | 先进工艺 |
| 激光退火 | ms | 超快、浅结 | 45nm+ |
| 闪光灯退火 | ms | 大面积 | 28nm+ |
| 微波退火 | min | 低温激活 | 特殊应用 |

## 十、化学机械抛光（CMP）

### 10.1 CMP原理

**简单解释**：用化学腐蚀+机械研磨，使硅片表面平坦化，是多层互连必需工艺。

```
CMP系统组成：
├── 抛光垫（Pad）
│   └── 聚氨酯材料
├── 抛光液（Slurry）
│   ├── 磨料（SiO₂、Al₂O₃、CeO₂）
│   └── 化学添加剂
├── 抛光头（Head）
│   └── 晶圆载体
├── 抛光台（Platen）
│   └── 抛光垫载体
└── 终点检测
    └── 电机电流、光学
```

### 10.2 CMP工艺参数

```
压力：
P = 1-5 psi

相对速度：
v = 20-100 rpm

去除速率（RR）：
RR = Kp × P × v

Kp = Preston系数

典型去除速率：
- SiO₂：100-300 nm/min
- Cu：200-500 nm/min
- W：100-200 nm/min

选择比：
SiO₂:Si₃N₄ = 1:1 - 10:1
Cu:势垒 = 10:1 - 100:1
```

### 10.3 CMP应用

| 应用 | 抛光材料 | 抛光液 | 目的 |
|------|---------|--------|------|
| STI | SiO₂ | SiO₂磨料 | 浅槽隔离 |
| ILD | SiO₂ | SiO₂磨料 | 层间介质 |
| Cu互连 | Cu | Al₂O₃+氧化剂 | 大马士革 |
| W栓塞 | W | Al₂O₃ | 通孔填充 |
| 高k | HfO₂ | CeO₂ | 栅介质 |

**碟形凹陷（Dishing）**：

```
问题：宽线条抛光过多

度量：
Dishing = hcenter - hedge

典型值：
- Cu线宽10μm：<100 nm
- Cu线宽100μm：<300 nm

减少方法：
- 优化抛光液
- 多步抛光
- 增加图形密度
```

## 十一、清洗与表面处理

### 11.1 清洗原理

**简单解释**：去除颗粒、金属污染、有机物、自然氧化层，保证器件性能。

**污染类型**：

| 污染 | 来源 | 影响 |
|------|------|------|

| 有机物 | 光刻胶、人体 | 附着、污染 |
| 自然氧化层 | 空气 | 界面态、接触电阻 |

### 11.2 清洗工艺

**RCA清洗**：

```
SC-1（Standard Clean 1）：
- 成分：NH₄OH:H₂O₂:H₂O = 1:1:5
- 温度：70-80°C
- 时间：10-20 min
- 作用：去除有机物、颗粒

SC-2（Standard Clean 2）：
- 成分：HCl:H₂O₂:H₂O = 1:1:6
- 温度：70-80°C
- 时间：10-20 min
- 作用：去除金属

HF清洗（DHF）：
- 成分：HF:H₂O = 1:100 - 1:50
- 温度：室温
- 时间：30-60 s
- 作用：去除自然氧化层

稀释HF（DHF）：
- 成分：HF:H₂O = 1:50 - 1:200
- 作用：去除薄氧化层（1-2 nm）
```

## 十二、先进封装技术

### 12.1 传统封装

**封装功能**：
- 保护芯片
- 电气连接
- 散热
- 机械支撑

**封装类型**：

| 类型 | 引脚 | 应用 |
|------|------|------|
| DIP | 8-64 | 传统 |
| QFP | 32-304 | 消费电子 |
| BGA | 100-2000+ | 高密度 |
| QFN | 8-200 | 小型化 |

### 12.2 先进封装

**倒装芯片（Flip Chip）**：

**简单解释**：芯片正面朝下，凸点直接连接基板，比打线短、性能好。

```
工艺：
1. 凸点下金属层（UBM）
2. 焊料凸点（SnAg、SnAgCu）
3. 倒装对准
4. 回流焊
5. 底部填充（Underfill）

凸点参数：
- 直径：50-200 μm
- 节距：100-500 μm
- 高度：50-150 μm

优势：
- 互连短、性能好
- I/O密度高
- 尺寸小
```

**晶圆级封装（WLP）**：

```
特点：
- 在晶圆上完成封装
- 尺寸≈芯片尺寸
- 成本低（批量）

类型：
├── Fan-in WLP
│   └── RDL（重布线层）
└── Fan-out WLP
    ├── 晶圆重构
    └── 模塑（Molding）

RDL：
- 金属：Cu（2-5 μm）
- 介质：PI、BCB
- 线宽/间距：5-20 μm
```

**2.5D/3D封装**：

```
2.5D封装：
├── 硅中介层（Silicon Interposer）
│   ├── TSV（硅通孔）
│   ├── RDL
│   └── 凸点
└── 多个芯片并排

3D封装：
├── 芯片堆叠
│   ├── TSV
│   └── 微凸点（Micro-bump）
└── 晶圆键合

TSV参数：
- 直径：5-50 μm
- 深度：50-500 μm
- 深宽比：5:1 - 20:1
- 节距：20-200 μm
```

**Chiplet**：

**简单解释**：把大芯片拆成小芯片，用先进封装组装，降低成本、提高良率。

```
优势：
- 小芯片良率高
- 灵活组合
- 不同工艺混合
- IP复用

互连标准：
- UCIe（Universal Chiplet Interconnect Express）
- 线宽：2-10 μm
- 节距：10-50 μm
- 带宽：4-32 GT/s

案例：
- Intel Ponte Vecchio：47个Chiplet
- AMD MI300：多个Chiplet 3D堆叠
- Apple M1 Ultra：两个M1 Max互连
```

### 13.2 RTL设计

**HDL语言**：

| 语言 | 特点 | 应用 |
|------|------|------|
| Verilog | 简洁、C风格 | 主流 |
| VHDL | 严格、Ada风格 | 欧洲、军用 |
| SystemVerilog | 增强、面向对象 | 验证 |

**同步设计原则**：

```
1. 全同步设计
   - 单一/多时钟域
   - 寄存器间组合逻辑

2. 时钟约束
   create_clock -period 10 [get_ports clk]

3. 复位策略
   - 同步复位：稳定
   - 异步复位：快速
   - 异步复位同步释放

4. 跨时钟域（CDC）
   - 单比特：打两拍
   - 多比特：握手、FIFO
```

### 13.3 逻辑综合

**综合流程**：

```
RTL → 综合 → 门级网表

输入：
- RTL代码
- 约束文件（SDC）
- 标准单元库（.lib）

约束：
- 时钟：create_clock
- 输入延迟：set_input_delay
- 输出延迟：set_output_delay
- 负载：set_load
- 面积：set_max_area

优化目标：
- 时序优先（-effort high）
- 面积优先
- 功耗优先

输出：
- 门级网表（.v）
- 时序报告
- 面积报告
```

### 13.4 物理设计（P&R）

**布局规划（Floorplan）**：

```
考虑因素：
├── I/O位置
├── 宏单元（Memory、IP）位置
├── 电源网络
├── 时钟树
└── 标准单元区域

约束：
- 核心利用率：70-85%
- 宏单元到边界距离：>10 μm
- 宏单元间通道：>5 μm
```

**时钟树综合（CTS）**：

```
目标：
- 时钟偏斜（Skew）< 100 ps
- 插入延迟（Latency）最小
- 功耗最小

方法：
├── 平衡树（Zero Skew）
├── 有用偏斜（Useful Skew）
└── 多源时钟树（Multi-source）

时钟缓冲器：
- 驱动能力：2X、4X、8X
- 插入：平衡延迟
```

**布线（Routing）**：

```
布线层次：
- 金属1-6：标准单元内部
- 金属7-12：全局布线
- 金属13+：电源、时钟

设计规则：
- 线宽：0.1-1 μm
- 线间距：0.1-0.5 μm
- 通孔：单孔、多孔

信号完整性：
- 串扰：屏蔽线
- 压降（IR Drop）：<5%
- 电迁移（EM）：满足规则
```

### 14.2 基本电路模块

**运算放大器（Op-Amp）**：

```
理想参数：
- 增益：无穷大
- 带宽：无穷大
- 输入阻抗：无穷大
- 输出阻抗：0
- 失调：0

实际参数（典型）：
- 开环增益：80-120 dB
- 单位增益带宽：10-1000 MHz
- 输入阻抗：10⁶-10¹² Ω
- 输出阻抗：10-100 Ω
- 输入失调：10 μV-10 mV

性能指标：
- 相位裕度：>45°
- 增益裕度：>10 dB
- 转换速率（SR）：10-1000 V/μs
```

**基准电压源**：

```
带隙基准（Bandgap）：
Vref = VBE + kVT

VBE：负温度系数（-2 mV/K）
VT：正温度系数（+0.085 mV/K）
k：调整系数

输出：~1.25 V
温度系数：<10 ppm/°C

应用：
- LDO
- ADC/DAC基准
- 振荡器
```

**LDO（低压差线性稳压器）**：

```
参数：
- 输入电压：2.5-5.5 V
- 输出电压：0.8-3.3 V（可调）
- 压差：100-300 mV
- 负载调整率：<1%
- 线性调整率：<0.1%
- PSRR：>60 dB（低频）

功耗：
P = (Vin - Vout) × Iload

效率：
η = Vout/Vin（低效）
```

### 14.3 数据转换器

**ADC（模数转换器）**：

| 架构 | 分辨率 | 速度 | 应用 |
|------|--------|------|------|
| Flash | 4-8 bit | 极快（Gsps）| 高速 |
| SAR | 8-18 bit | 中（Msps）| 主流 |
| Sigma-Delta | 16-24 bit | 慢（ksps）| 高精度 |
| Pipeline | 10-14 bit | 快（100s Msps）| 通信 |

**ADC性能参数**：

```
分辨率：N bits
量化单位：LSB = Vref/2^N

SNR（信噪比）：
SNR = 6.02N + 1.76 dB

ENOB（有效位数）：
ENOB = (SNR - 1.76)/6.02

DNL（微分非线性）：<0.5 LSB
INL（积分非线性）：<1 LSB
```

**DAC（数模转换器）**：

| 架构 | 特点 | 应用 |
|------|------|------|
| 电阻串 | 简单、低速 | 低速 |
| R-2R | 精度高 | 通用 |
| 电流舵 | 高速 | 通信 |
| Sigma-Delta | 高精度 | 音频 |

### 41.3 常用公式

```
t = RC
R = ρL/A
C = εA/d
```

**功耗**：
```
P = CV²f（动态）

```

```
ER = Δd/Δt (nm/min)
```

```
S = ER₁/ER₂
```
