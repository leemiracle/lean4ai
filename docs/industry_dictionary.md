## 一、工业基础理论

### 1.1 工业定义

**Industry** /ˈɪndəstri/

**定义**：从事自然资源开采、加工以及对工农业产品再加工的社会物质生产部门。

**简单解释**：把原材料变成产品的行业，是一切现代经济的基础。

### 1.2 工业分类

**按行业分类**：

| 类别 | 内容 | 典型产品 |
|------|------|----------|
| 采矿业 | 开采自然资源 | 煤炭、石油、矿石 |
| 制造业 | 加工制造 | 机械、电子、汽车 |
| 电力/能源 | 能源生产与供应 | 电力、燃气 |
| 化工 | 化学加工 | 化肥、塑料、药品 |
| 冶金 | 金属冶炼 | 钢铁、有色金属 |
- 建材
  | 建筑材料 | 水泥、玻璃、陶瓷 |

**按生产方式分类**：

| 类型 | 特点 | 例子 |
|------|------|------|
| 流程工业 | 连续生产 | 石化、电力、水泥 |
| 离散工业 | 批量生产 | 汽车、电子、机械 |
| 混合型 | 两者结合 | 食品、制药 |

### 1.3 工业革命历程

**四次工业革命**：

```
第一次工业革命（1760-1840）
├── 标志：蒸汽机
├── 特点：机械化
└── 代表：纺织业机械化

第二次工业革命（1870-1914）
├── 标志：电力、内燃机
├── 特点：电气化、大规模生产
└── 代表：流水线生产

第三次工业革命（1960-2000）
├── 标志：计算机、互联网
├── 特点：自动化、信息化
└── 代表：数控机床、PLC

第四次工业革命（2010-）
├── 标志：AI、物联网、大数据
├── 特点：智能化、互联化
└── 代表：工业4.0、智能制造
```

### 1.4 工业体系架构

```
工业体系层次
├── 战略层：企业战略、市场定位
├── 管理层：生产计划、质量管理
├── 执行层：MES、APS、WMS
├── 控制层：PLC、DCS、SCADA
└── 设备层：传感器、执行器、设备
```

### 1.5 关键指标

**生产效率指标**：

| 指标 | 英文 | 计算公式 | 意义 |
|------|------|----------|------|
| OEE | Overall Equipment Effectiveness | 可用率×性能率×质量率 | 设备综合效率 |
| 产能利用率 | Capacity Utilization | 实际产量/设计产能 | 产能利用程度 |
| 良品率 | First Pass Yield | 良品数/总产量 | 质量水平 |
- 生产周期
  | Cycle Time | 完成一个产品的时间 | 生产效率 |
| 节拍时间 | Takt Time | 可用时间/需求量 | 生产节奏 |

## 二、制造业

### 2.1 制造工艺分类

**减材制造**：
- 切削加工：车、铣、刨、磨
- 电火花加工：EDM
- 激光切割
- 水刀切割

**等材制造**：
- 铸造
- 锻造
- 冲压
- 焊接

**增材制造（3D打印）**：
- SLA：光固化
- SLS：选择性激光烧结
- FDM：熔融沉积
- SLM：选择性激光熔化
- EBM：电子束熔化

### 2.2 切削加工

**车削（Turning）**：

**简单解释**：工件旋转，刀具移动，像削铅笔一样把材料削下来。

**应用**：轴类、盘类、套类零件

**铣削（Milling）**：

**简单解释**：刀具旋转，工件移动，像钻头一样在材料上"铣"出形状。

**应用**：平面、沟槽、齿轮、复杂曲面

**磨削（Grinding）**：

**简单解释**：用高速旋转的砂轮"磨"掉材料，精度极高。

**应用**：高精度表面、硬质材料

**切削参数**：

| 参数 | 符号 | 单位 | 影响 |
|------|------|------|------|
| 切削速度 | Vc | m/min | 刀具寿命、表面质量 |
| 进给量 | f | mm/rev | 表面粗糙度、效率 |
| 切削深度 | ap | mm | 材料去除率 |
- 主轴转速
  | n | rpm | 与切削速度相关 |

**切削速度计算**：
```
Vc = π × D × n / 1000

其中：
Vc = 切削速度(m/min)
D = 工件直径(mm)
n = 主轴转速(rpm)
```

### 2.3 铸造

**定义**：将熔融金属浇入铸型，冷却后获得铸件。

**简单解释**：把金属化成水，倒进模具，冷却后就是零件。

**主要方法**：

| 方法 | 特点 | 应用 |
|------|------|------|
| 砂型铸造 | 成本低、适用广 | 大型铸件 |
- 熔模铸造（失蜡铸造）
  | 精度高 | 复杂零件 |
| 压铸 | 效率高、精度好 | 批量生产 |
| 金属型铸造 | 表面质量好 | 有色金属 |
- 离心铸造
  | 致密性好 | 管类、套类 |

**铸造缺陷**：

| 缺陷 | 原因 | 预防 |
|------|------|------|
| 气孔 | 气体未排出 | 改善排气 |
- 缩孔
  | 补缩不足 | 设计冒口 |
| 夹渣 | 杂质混入 | 过滤、除渣 |
| 裂纹 | 应力过大 | 控制冷却 |

### 2.4 锻造

**定义**：通过外力使金属产生塑性变形。

**简单解释**：像打铁一样，用锤子把金属敲成想要的样子。

**分类**：

| 类型 | 特点 | 应用 |
|------|------|------|
| 自由锻 | 工具简单、灵活性高 | 单件小批 |
- 模锻
  | 精度高、效率高 | 批量生产 |
| 辊锻 | 连续生产 | 板簧、连杆 |
| 精密锻造 | 少无切削 | 精密零件 |

### 2.5 焊接

**定义**：通过加热或加压，使材料永久连接。

**主要方法**：

| 方法 | 原理 | 应用 |
|------|------|------|
| 电弧焊 | 电弧熔化金属 | 通用 |
| 氩弧焊（TIG/MIG） | 惰性气体保护 | 不锈钢、铝 |
| CO2焊 | CO2气体保护 | 碳钢 |
- 电阻焊
  | 电流电阻热 | 薄板 |
| 激光焊 | 激光热源 | 精密焊接 |
| 摩擦焊 | 摩擦生热 | 异种金属 |

### 2.6 表面处理

**目的**：提高耐磨性、耐腐蚀性、美观性

**主要方法**：

| 方法 | 原理 | 应用 |
|------|------|------|
| 电镀 | 电化学沉积 | 防腐、装饰 |
- 化学镀
  | 化学还原沉积 | 复杂形状 |
| 喷涂 | 热喷涂涂层 | 修复、耐磨 |
| 阳极氧化 | 电化学氧化 | 铝合金 |
- 渗碳/渗氮
  | 扩散强化 | 齿轮、轴 |
| PVD/CVD | 气相沉积 | 刀具涂层 |

## 三、工业自动化

### 3.1 自动化基础

**定义**：用机器设备代替人工，实现生产过程的自动控制。

**简单解释**：让机器自己干活，人只负责监督和维护。

**自动化层次**：

```
Level 4: 企业层（ERP）
Level 3: 管理层（MES）
Level 2: 监控层（SCADA）
Level 1: 控制层（PLC/DCS）
Level 0: 现场层（传感器/执行器）
```

### 3.2 控制器

**PLC（可编程逻辑控制器）**：

**定义**：专为工业环境设计的数字控制设备。

**简单解释**：工业版的"电脑"，按照程序控制机器运行。

**主要品牌**：
| 品牌 | 国家 | 市场地位 |
|------|------|----------|
| 西门子 | 德国 | 全球第一 |
| 罗克韦尔 | 美国 | 北美领先 |
| 三菱 | 日本 | 亚洲领先 |
| 欧姆龙 | 日本 | 中小系统 |
- 台达
  | 中国台湾 | 性价比 |

**DCS（分布式控制系统）**：

**定义**：用于大型流程工业的控制系统。

**简单解释**：多个小控制器分工协作，适合化工厂、电厂等大型设施。

**PLC vs DCS**：

| 特性 | PLC | DCS |
|------|-----|-----|
| 适用场合 | 离散工业、中小系统 | 流程工业、大型系统 |
| 控制方式 | 集中控制 | 分散控制 |
- 响应速度
  | 快（ms级） | 较慢 |
| 编程方式 | 梯形图为主 | 功能块图为主 |

### 3.3 传感器

**温度传感器**：

| 类型 | 原理 | 范围 | 精度 |
|------|------|------|------|
| 热电偶 | 热电效应 | -200~1800°C | ±0.5~2°C |
- 热电阻（RTD）
  | 电阻变化 | -200~850°C | ±0.1~0.5°C |
| 红外测温 | 热辐射 | -50~3000°C | ±1% |

**压力传感器**：

| 类型 | 原理 | 应用 |
|------|------|------|
| 应变式 | 电阻变化 | 通用 |
- 压电式
  | 压电效应 | 动态压力 |
| 电容式 | 电容变化 | 高精度 |

**位置/位移传感器**：

| 类型 | 原理 | 精度 |
|------|------|------|
| 编码器 | 光电/磁电 | 高 |
- 光栅尺
  | 光学 | 极高（μm级） |
| LVDT | 电磁感应 | 高 |
| 激光测距 | 激光飞行时间 | 极高 |

**流量传感器**：

| 类型 | 原理 | 适用介质 |
|------|------|----------|
| 电磁式 | 电磁感应 | 导电液体 |
- 涡街式
  | 漩涡频率 | 气体/液体 |
| 超声波式 | 声波传播 | 多种介质 |
| 质量流量计 | 科氏力 | 精确计量 |

### 3.4 执行器

**电机**：

| 类型 | 特点 | 应用 |
|------|------|------|

- 直流电机
  | 调速方便 | 特殊场合 |

| 组件 | 功能 |
|------|------|
- 液压泵
  | 提供压力油 |
| 液压缸 | 直线运动 |
| 液压马达 | 旋转运动 |

**气动系统**：

| 组件 | 功能 |

### 3.6 SCADA系统

**功能**：
- 数据采集
- 过程监视
- 报警管理
- 历史数据存储
- 报表生成
- 远程控制

**主要软件**：

  | Wonderware | 易用 |

## 四、工业互联网与智能制造

### 4.1 工业4.0

**定义**：以智能制造为主导的第四次工业革命。

**简单解释**：工厂变聪明，设备会沟通，生产更灵活。

**核心特征**：

| 特征 | 英文 | 说明 |
|------|------|------|
| 互联 | Interconnection | 设备、人、数据互联 |
| 数据 | Information | 大数据驱动 |
| 集成 | Integration | 纵向、横向、端到端 |
- 创新
  | Innovation | 技术与模式创新 |
| 转型 | Transformation | 服务化转型 |

### 4.2 工业 internet of things（IIoT）

**定义**：工业领域的物联网应用。

**简单解释**：给工厂里的每台设备装上"大脑"和"嘴巴"，让它们能联网说话。

**架构层次**：

```
IIoT架构
├── 感知层：传感器、RFID、摄像头
├── 网络层：工业以太网、5G、LPWAN
├── 平台层：云平台、边缘计算
└── 应用层：工业APP、数据分析
```

**关键技术**：

| 技术 | 应用 |
|------|------|
| 5G | 低延迟、高带宽通信 |
| 边缘计算 | 本地实时处理 |
| 数字孪生 | 虚拟仿真与优化 |
- 工业大数据
  | 预测性维护、质量分析 |
| AI/ML | 智能决策 |

### 4.3 数字化工厂

**定义**：利用数字化技术实现工厂全生命周期的数字化管理。

**核心要素**：

| 要素 | 内容 |
|------|------|
| 产品数字化 | 3D模型、BOM |
| 工艺数字化 | 工艺路线、参数 |
| 资源数字化 | 设备、人员、物料 |
- 生产数字化
  | 计划、执行、质量 |

### 4.4 智能制造

**定义**：利用信息技术实现制造过程的智能化。

**核心能力**：

```
智能制造能力
├── 智能设计：AI辅助设计
├── 智能生产：自适应生产
├── 智能物流：AGV、智能仓储
├── 智能服务：远程运维
└── 智能决策：大数据分析
```

**智能工厂特征**：
- 设备互联
- 数据透明
- 技术辅助
- 自主决策
- 柔性生产

### 4.5 数字孪生

**定义**：物理实体的虚拟映射。

**简单解释**：在电脑里造一个一模一样的工厂，先在虚拟世界里试运行。

**应用场景**：

| 场景 | 作用 |
|------|------|
| 产品设计 | 虚拟验证 |
| 工艺优化 | 仿真优化 |
| 生产监控 | 实时映射 |
- 预测维护
  | 状态预测 |
| 培训 | 虚拟培训 |

### 4.6 制造执行系统（MES）

**定义**：车间级生产管理系统。

**简单解释**：管理车间里每天生产什么、怎么生产、生产得怎么样的系统。

**核心功能**：

| 模块 | 功能 |
|------|------|
| 生产调度 | 计划排程 |
| 生产追踪 | 工单执行 |
| 资源管理 | 人、机、料 |
- 质量管理
  | 过程质量控制 |
| 绩效分析 | OEE计算 |
| 文档管理 | 工艺文档 |

### 4.7 企业资源计划（ERP）

**定义**：企业级综合管理系统。

**主要模块**：

| 模块 | 功能 |
|------|------|
| 财务 | 会计、成本 |
- 采购
  | 供应商管理、采购订单 |
| 库存 | 物料管理 |
| 生产 | 计划、BOM |
| 销售 | 订单、客户 |
| 人力资源 | 人事、薪资 |

**主要软件**：
| 软件 | 厂商 | 定位 |
|------|------|------|
| SAP ERP | SAP | 大型跨国企业 |
- Oracle ERP
  | Oracle | 大中型企业 |
| 用友 | 用友 | 中国企业 |
| 金蝶 | 金蝶 | 中小企业 |

## 五、能源工业

### 5.1 能源分类

**一次能源**：
- 化石能源：煤炭、石油、天然气
- 可再生能源：水能、风能、太阳能、生物质能
- 核能

**二次能源**：
- 电力
- 氢能
- 蒸汽
- 成品油

### 5.2 电力系统

**发电方式**：

| 方式 | 原理 | 效率 | 特点 |
|------|------|------|------|
| 火电 | 燃烧发电 | 35-45% | 主力电源 |
| 水电 | 水流发电 | 80-90% | 清洁、可调峰 |
| 核电 | 核裂变发电 | 30-35% | 基荷、清洁 |
- 风电
  | 风力发电 | 30-50% | 间歇性 |
| 光伏 | 光电转换 | 15-25% | 间歇性 |

**电网结构**：

```

## 七、冶金工业

### 7.1 钢铁冶炼

**炼铁**：

**简单解释**：把铁矿石中的铁"解放"出来，在高炉里用焦炭把铁还原。

**高炉炼铁**：

```
原料：铁矿石 + 焦炭 + 石灰石
    ↓
高炉（反应温度：1500°C）
    ↓
产物：铁水 + 炉渣 + 高炉煤气
```

**主要反应**：
```
Fe2O3 + 3CO → 2Fe + 3CO2
（铁矿石 + 一氧化碳 → 铁 + 二氧化碳）
```

**炼钢**：

**简单解释**：把铁水中的杂质（碳、硅、硫、磷）"烧掉"，调整成分得到钢。

**方法**：

| 方法 | 特点 | 应用 |
|------|------|------|
| 转炉 | 效率高、成本低 | 主流 |
- 电炉
  | 灵活、可加废钢 | 特种钢 |
| 平炉 | 已淘汰 | - |

**转炉炼钢过程**：
```
铁水 → 装炉 → 吹氧 → 脱碳、脱磷 → 出钢 → 连铸 → 钢坯
```

### 7.2 轧钢

**定义**：通过轧制使钢材成型。

**简单解释**：像压面条一样，把钢坯压成不同形状的钢材。

**分类**：

| 类型 | 产品 | 应用 |

- 冷轧
  | 薄钢板 | 汽车、家电 |
| 钢管 | 无缝管、焊管 | 管道、结构 |

**轧制工艺**：
```
加热 → 除鳞 → 粗轧 → 精轧 → 冷却 → 矫直 → 剪切 → 检验
```

CL（中心线）= μ

```

## 附录F：化工工艺详解

### F.1 乙烯生产

**蒸汽裂解工艺**：

```

    ↓

    ↓

天然气 → 脱硫 → 一段转化 → 二段转化 →

```
