# 蓝海战略

## 概述

蓝海战略是由 W. Chan Kim 和 Renée Mauborgne 提出的战略理论，主张通过开创无人竞争的市场空间（蓝海），而非在竞争激烈的市场（红海）中厮杀。

## 红海 vs 蓝海

### 对比

| 维度 | 红海战略 | 蓝海战略 |
|------|---------|---------|
| 市场 | 已有市场 | 创造新市场 |
| 竞争 | 击败竞争者 | 摆脱竞争者 |
| 需求 | 挖掘现有需求 | 创造新需求 |
| 战略 | 差异化或低成本 | 差异化和低成本 |
| 价值 | 价值-成本权衡 | 打破价值-成本权衡 |

### 示例

```
红海                          蓝海
传统马戏团                    太阳马戏团
├── 动物表演                  ├── 去除动物
├── 明星演员                  ├── 去除明星
├── 多个舞台                  ├── 单一舞台
└── 瞄准儿童                  └── 瞄准成人
                               → 创造全新体验
```

## 核心概念

### 价值创新

```
价值创新 = 创新 + 价值
    │
    ├── 创新：改变传统做法
    └── 价值：为用户创造价值

同时追求：
- 差异化（独特价值）
- 低成本（成本优势）
```

## 四步动作框架

### 四个动作

```
         增加/创造
              ↑
              │
   削减/消除 ←┼→ 提升/增加
              │
              ↓
         降低/减少
```

### 详细说明

#### 1. 剔除（Eliminate）

**问题**：行业中哪些要素应该被剔除？

**目的**：
- 去除用户不关心的
- 降低成本

**示例**：
- 太阳马戏团剔除动物表演
- 西南航空剔除餐饮服务
- 宜家剔除送货上门

#### 2. 减少（Reduce）

**问题**：哪些要素应该减少到行业标准以下？

**目的**：
- 降低成本
- 减少过度设计

**示例**：
- 经济型酒店减少房间面积
- 瑞安航空减少服务项目
- 戴尔减少中间商

#### 3. 增加（Raise）

**问题**：哪些要素应该增加到行业标准以上？

**目的**：
- 提升用户价值
- 差异化竞争

**示例**：
- 苹果增加设计美感
- 星巴克增加社交空间
- 特斯拉增加科技感

#### 4. 创造（Create）

**问题**：哪些新要素应该被创造？

**目的**：
- 创造新价值
- 开创新市场

**示例**：
- 任天堂 Wii 创造体感游戏
- Airbnb 创造共享住宿
- Uber 创造网约车模式

### 应用示例：太阳马戏团

```markdown
## 太阳马戏团四步动作

### 剔除
- 动物表演
- 明星演员
- 多个舞台表演

### 减少
- 刺激和危险
- 幽默搞笑

### 增加
- 帐篷的神秘感和艺术氛围
- 更舒适的环境

### 创造
- 主题表演
- 艺术音乐和舞蹈
- 多种制作
```

## 蓝海战略制定

### 六步路径

#### 1. 重建市场边界

**方法**：
- 跨越替代性行业
- 跨越战略集团
- 跨越买方链
- 跨越互补性产品和服务
- 跨越功能-情感导向
- 跨越时间

**示例**：

| 路径 | 示例 |
|------|------|
| 替代性行业 | 电影院 vs 餐厅 |
| 战略集团 | 经济型 vs 豪华型 |
| 买方链 | 购买者 vs 使用者 |
| 互补产品 | 咖啡 vs 书 |
| 功能-情感 | 功能型 vs 情感型 |
| 时间 | 现在趋势 vs 未来趋势 |

#### 2. 注重全局而非数字

**原则**：使用战略布局图而非数字报表

**方法**：
- 绘制价值曲线
- 一图胜千言
- 便于理解和沟通

#### 3. 超越现有需求

**策略**：关注非客户而非现有客户

**非客户层次**：

```
第一层：即将成为客户
第二层：拒绝型非客户
第三层：未探索型非客户

从内到外：
现有客户 → 第一层 → 第二层 → 第三层
```

**示例**：
- 健身房的非客户：在家锻炼的人
- 传统银行的非客户：不使用银行的人

#### 4. 遵循合理的战略顺序

**顺序**：

```
1. 买方效用 → 用户是否认可价值？
        ↓
2. 价格 → 价格是否可接受？
        ↓
3. 成本 → 能否以成本价生产？
        ↓
4. 接受 → 是否有采用障碍？
```

**买方效用地图**：

```
                购买  使用  补给  维护  处置
                ───  ───  ───  ───  ───
客户生产率       │    │    │    │    │
简单性           │    │    │    │    │
方便性           │    │    │    │    │
风险             │    │    │    │    │
趣味与形象       │    │    │    │    │
环保             │    │    │    │    │
```

#### 5. 克服关键组织障碍

**四大障碍**：

| 障碍 | 应对策略 |
|------|---------|
| 认知障碍 | 让人们看到现实 |
| 资源障碍 | 将资源重新分配 |
| 动力障碍 | 将关键人物放在核心位置 |
| 政治障碍 | 邀请魔鬼代言人 |

#### 6. 将战略执行建成战略的一部分

**原则**：执行过程本身就是战略的一部分

**方法**：
- 公平过程
- 参与决策
- 解释原因
- 明确期望

## 蓝海战略案例

### 案例1：任天堂 Wii

```markdown
## 蓝海战略分析

### 行业传统竞争要素
- 高性能图形
- 复杂操控
- 独占游戏
- 高价格

### 四步动作

**剔除**：
- 高性能图形追求
- 复杂操控方式

**减少**：
- 硬件性能
- 游戏开发成本

**增加**：
- 游戏互动性
- 社交属性

**创造**：
- 体感操控
- 家庭娱乐定位
- 简单易上手

### 结果
- 开创休闲游戏市场
- 非传统游戏玩家成为客户
- 销量远超竞争对手
```

### 案例2：黄尾葡萄酒

```markdown
## 蓝海战略分析

### 行业传统竞争要素
- 复杂口味
- 年份等级
- 传统营销
- 高价格

### 四步动作

**剔除**：
- 复杂的品酒术语
- 年份和等级
- 传统营销

**减少**：
- 产品种类
- 酒品种类

**增加**：
- 选择的简便性
- 趣味和冒险感

**创造**：
- 简单易饮的口感
- 时尚的包装
- 亲民的价格

### 结果
- 成为美国进口葡萄酒第一品牌
- 非传统葡萄酒消费者成为客户
```

### 案例3：iTunes + iPod

```markdown
## 蓝海战略分析

### 传统音乐行业
- CD 专辑销售
- 单曲捆绑销售
- 实体店购买
- 盗版猖獗

### 四步动作

**剔除**：
- 实体媒介
- 专辑捆绑
- 物理分销

**减少**：
- 拥有成本
- 选择复杂性

**增加**：
- 便携性
- 选择自由
- 搜索便利

**创造**：
- 单曲购买模式
- 数字音乐库
- 合法数字音乐平台

### 结果
- 改变音乐消费方式
- 开创数字音乐市场
- 成为行业领导者
```

## 蓝海战略实施

### 实施步骤

```
1. 战略分析
   - 识别红海竞争要素
   - 分析客户和非客户
   - 应用六步路径

2. 战略制定
   - 应用四步动作
   - 绘制战略布局图
   - 定义价值创新

3. 战略验证
   - 市场测试
   - 用户验证
   - 成本分析

4. 战略执行
   - 资源配置
   - 组织变革
   - 市场推广

5. 战略迭代
   - 监控反馈
   - 持续优化
```

### 风险管理

| 风险 | 应对策略 |
|------|---------|
| 模仿风险 | 持续创新 |
| 市场风险 | 小规模测试 |
| 组织风险 | 文化建设 |
| 执行风险 | 分步实施 |

## 蓝海与红海的转换

### 蓝海终将变红海

```
蓝海 → 竞争者进入 → 红海

应对：
1. 持续创新
2. 建立壁垒
3. 寻找新蓝海
```

### 红海中寻找蓝海

**方法**：
- 细分市场
- 差异化定位
- 创造新需求

## 总结

蓝海战略的核心：

1. **价值创新**：同时追求差异化和低成本
2. **创造新需求**：关注非客户
3. **四步动作**：剔除、减少、增加、创造
4. **战略布局**：可视化战略
5. **系统执行**：六步路径
6. **持续创新**：不断寻找新蓝海
