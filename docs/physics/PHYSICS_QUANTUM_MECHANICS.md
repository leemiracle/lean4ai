## 1. 量子力学基础

### 波粒二象性

**费曼解释**：光和物质既是波又是粒子。你问"是波还是粒子"，答案是"都是，也都不完全是"。就像圆柱体，从一边看是圆，从另一边看是方形。

**光的粒子性**：
- 光电效应（爱因斯坦，1905）
- 康普顿散射
- 光子能量：E = hν = ℏω

**物质的波动性**：
- 电子衍射（戴维森-革末实验，1927）
- 德布罗意波长：λ = h/p

**费曼名言**：
> "我想我可以有把握地说，没有人真正理解量子力学。"

**双缝实验**（核心）：
```
单个粒子 → 通过双缝 → 干涉图案
即使一个个发射，也会形成干涉
粒子似乎"同时"通过两条缝
但一旦测量，只通过一条缝
```

---

### 普朗克常数与约化普朗克常数

**费曼解释**：这是量子世界的"基本单位"。所有量子效应的尺度都由它决定。

**普朗克常数**：
```
h = 6.626 × 10⁻³⁴ J·s
```

**约化普朗克常数**：
```
ℏ = h/(2π) = 1.055 × 10⁻³⁴ J·s
```

**意义**：
- 量子化的最小单位
- 角动量的量子单位
- 作用量的量子单位

**对应原理**：
```
ℏ → 0：量子力学 → 经典力学
宏观世界：ℏ可忽略
微观世界：ℏ不可忽略
```

---

### 波函数

**费曼解释**：波函数描述量子系统的"状态"。它本身不是物理量，但它的模平方给出找到粒子的概率。

**波函数**：ψ(x, t)

**概率解释**（玻恩，1926）：
```
|ψ(x,t)|² = 概率密度

在x到x+dx间找到粒子的概率 = |ψ|²dx
```

**归一化**：
```
∫|ψ|²dx = 1
（总概率为1）
```

**性质**：
- 复数值
- 模平方为实数（概率密度）
- 包含系统全部信息

**费曼直觉**：
> 波函数不是"物理波"，而是"概率幅"。它描述的不是粒子在哪，而是粒子"可能"在哪。

---

### 薛定谔方程 ⭐⭐⭐⭐⭐

**费曼解释**：薛定谔方程是量子力学的"牛顿第二定律"，描述波函数如何随时间演化。

**含时薛定谔方程**：
```
iℏ(∂ψ/∂t) = Ĥψ

Ĥ：哈密顿算符（能量算符）
```

**不含时薛定谔方程**：
```
Ĥψ = Eψ

E：能量本征值
ψ：能量本征态
```

**哈密顿算符**：
```
Ĥ = -ℏ²/(2m)∇² + V(x)
   ↑动能        ↑势能
```

**一维形式**：
```
iℏ(∂ψ/∂t) = -ℏ²/(2m)(∂²ψ/∂x²) + V(x)ψ
```

**意义**：
- 决定波函数演化
- 能量守恒的量子形式
- 线性方程（可叠加）

---

### 算符与可观测量

**费曼解释**：在量子力学中，物理量（位置、动量、能量）用"算符"表示。算符对波函数"操作"，提取物理信息。

**常用算符**：

| 物理量 | 算符 | 作用 |
|--------|------|------|
| 位置 | x̂ | x̂ψ = xψ |
| 动量 | p̂ = -iℏ(∂/∂x) | p̂ψ = -iℏ(∂ψ/∂x) |
| 能量 | Ĥ | Ĥψ = Eψ（本征值问题） |
| 角动量 | L̂ = r̂ × p̂ | L̂ψ = l(l+1)ℏ²ψ |

**本征值方程**：
```
Âψ = aψ

Â：算符
a：本征值（测量结果）
ψ：本征态
```

**厄米算符**：
- 本征值为实数（可观测）
- 本征态正交

---

### 测量与波函数坍缩

**费曼解释**：测量前，粒子处于叠加态。测量时，波函数"坍缩"到某个本征态，测量结果是本征值之一。

**测量过程**：
```
1. 系统处于叠加态：|ψ⟩ = c₁|a₁⟩ + c₂|a₂⟩ + ...

2. 测量可观测量A

3. 波函数坍缩到某个本征态|aᵢ⟩

4. 测量结果为aᵢ

5. 概率 = |cᵢ|²
```

**坍缩的谜团**：
- 什么算"测量"？
- 坍缩是瞬间发生的吗？
- 这是物理过程还是知识更新？

**哥本哈根诠释**：
```
测量导致波函数坍缩
波函数描述的是"我们的知识"
```

**费曼困惑**：
> 谁也不知道它怎么会那样，这个谜至今无人能解。

---

### 不确定性原理

**费曼解释**：你不能同时精确知道粒子的位置和动量。这不是测量技术的限制，而是自然的根本特性。

**海森堡不确定性原理**（1927）：
```
Δx·Δp ≥ ℏ/2

Δx：位置不确定度
Δp：动量不确定度
```

**一般形式**：
```
ΔA·ΔB ≥ |⟨[Â,B̂]⟩|/2

[Â,B̂] = ÂB̂ - B̂Â：对易子
```

**能量-时间不确定性**：
```
ΔE·Δt ≥ ℏ/2
```

**物理图像**：
- 位置精确 → 动量完全不确定
- 动量精确 → 位置完全不确定
- 粒子没有确定的轨道

**不是测量误差**：
- 即使理想测量，也无法同时确定
- 源于波的本质

**费曼解释**：
> 想象你要测量海浪的位置。波长确定（动量），位置就不确定；位置确定，波长就不确定。粒子也是如此。

---

### 叠加原理

**费曼解释**：量子系统可以同时处于多个状态的"叠加"。就像薛定谔的猫，既死又活。

**叠加态**：
```
|ψ⟩ = c₁|ψ₁⟩ + c₂|ψ₂⟩

c₁, c₂：复数系数
|ψ₁⟩, |ψ₂⟩：基态
```

**例子**：
```
|→⟩ = (1/√2)(|↑⟩ + |↓⟩)

自旋既向上又向下
```

**干涉**：
```
叠加态各分量可以干涉
|ψ₁ + ψ₂|² ≠ |ψ₁|² + |ψ₂|²
```

**量子计算基础**：
```
|0⟩ 和 |1⟩ 的叠加
量子比特可以同时是0和1
```

---

### 量子态与狄拉克符号

**费曼解释**：狄拉克发明了一套优雅的符号，用"左矢"和"右矢"表示量子态。

**右矢（ket）**：
```
|ψ⟩：表示量子态
```

**左矢（bra）**：
```
⟨ψ|：|ψ⟩的共轭转置
```

**内积**：
```
⟨φ|ψ⟩：复数
|⟨φ|ψ⟩|²：|ψ⟩在|φ⟩上的投影概率
```

**归一化**：
```
⟨ψ|ψ⟩ = 1
```

**基矢展开**：
```
|ψ⟩ = Σcₙ|n⟩
cₙ = ⟨n|ψ⟩
```

## 2. 一维势场问题

### 无限深势阱

**费曼解释**：粒子被限制在一个"盒子"里，出不去。这是最简单的量子束缚态问题。

**势能**：
```
V(x) = 0,    0 < x < L
V(x) = ∞,    其他
```

**波函数**：
```
ψₙ(x) = √(2/L) sin(nπx/L)
n = 1, 2, 3, ...
```

**能级**（量子化！）：
```
Eₙ = n²π²ℏ²/(2mL²)
```

**关键特点**：
1. **能量量子化**：n = 1, 2, 3...
2. **零点能**：E₁ > 0（不能静止）
3. **驻波**：波函数是驻波
4. **节点**：n越大，节点越多

**费曼直觉**：
> 粒子像被囚禁的波，只能形成特定频率的驻波。

---

### 有限深势阱

**费曼解释**：墙没那么高了，能量高的粒子能"穿透"墙壁。

**特点**：
- 束缚态能级有限
- 波函数在墙外指数衰减
- 量子隧穿的雏形

---

### 谐振子

**费曼解释**：弹簧振子的量子版本。这是最重要的量子模型之一，几乎处处出现。

**势能**：
```
V(x) = ½kx² = ½mω²x²
```

**能级**：
```
Eₙ = ℏω(n + ½), n = 0, 1, 2, ...

注意：n从0开始，且E₀ = ½ℏω
```

**零点能**：
```
E₀ = ½ℏω

不能低于这个能量！
```

**波函数**：
```
ψₙ(x) = NₙHₙ(αx)e^(-α²x²/2)

Hₙ：厄米多项式
```

**重要性**：
- 分子振动
- 固体晶格（声子）
- 量子场论（场量子化）
- 化学键

**升降算符**：
```
â⁺|n⟩ = √(n+1)|n+1⟩  （升）
â|n⟩ = √n|n-1⟩        （降）
```

---

### 量子隧穿 ⭐⭐⭐⭐

**费曼解释**：粒子可以"穿墙"！能量不足以翻越势垒，却能以一定概率出现在另一边。

**经典 vs 量子**：
```
经典：E < V₀ → 完全反射
量子：E < V₀ → 有概率穿透
```

**隧穿概率**（粗略）：
```
T ≈ e^(-2κd)

κ = √(2m(V₀-E))/ℏ
d：势垒宽度
```

**应用**：
1. **α衰变**：α粒子隧穿逃离原子核
2. **扫描隧道显微镜（STM）**：电子隧穿探测表面
3. **核聚变**：太阳中心质子隧穿
4. **闪存**：电子隧穿进出浮栅

**费曼直觉**：
> 波函数渗透到经典禁区，就像水波绕过障碍物。

## 3. 三维问题与氢原子

### 三维薛定谔方程

**费曼解释**：把一维推广到三维，引入球坐标处理球对称问题。

**方程**：
```
iℏ(∂ψ/∂t) = -ℏ²/(2m)∇²ψ + Vψ

∇² = ∂²/∂x² + ∂²/∂y² + ∂²/∂z²
```

**球坐标**：
```
∇² = (1/r²)(∂/∂r)(r²∂/∂r) 
     + (1/(r²sin θ))(∂/∂θ)(sin θ ∂/∂θ)
     + (1/(r²sin²θ))∂²/∂φ²
```

---

### 角动量量子化

**费曼解释**：角动量也是量子化的，不能取任意值。

**轨道角动量**：
```
|L|² = l(l+1)ℏ²
L_z = mℏ

l = 0, 1, 2, ...（轨道量子数）
m = -l, -l+1, ..., l-1, l（磁量子数）
```

**费曼直觉**：
> 角动量大小固定，方向只能取特定角度（空间量子化）。

---

### 氢原子

**费曼解释**：量子力学对氢原子的完美解释，是理论物理的巅峰成就之一。

**能级**（玻尔公式，现从量子力学推导）：
```
Eₙ = -13.6 eV/n²

n = 1, 2, 3, ...（主量子数）
```

**波函数**：
```
ψₙₗₘ(r, θ, φ) = Rₙₗ(r)Yₗᵐ(θ, φ)

Rₙₗ：径向部分
Yₗᵐ：球谐函数
```

**量子数**：
```
n = 1, 2, 3, ...
l = 0, 1, 2, ..., n-1
m = -l, ..., l
```

**轨道符号**：
```
l = 0: s轨道
l = 1: p轨道
l = 2: d轨道
l = 3: f轨道
```

**简并度**：
```
n能级的简并度 = n²
（无外场时）
```

---

### 电子自旋

**费曼解释**：电子有个"内部陀螺"，即使不运动也有角动量——这就是自旋。

**自旋量子数**：
```
s = ½（电子）

|S|² = s(s+1)ℏ² = ¾ℏ²
S_z = mₛℏ, mₛ = ±½
```

**自旋本征态**：
```
|↑⟩：自旋向上（mₛ = +½）
|↓⟩：自旋向下（mₛ = -½）
```

**自旋算符**：
```
S = (ℏ/2)σ

σ：泡利矩阵
```

**泡利矩阵**：
```
σₓ = [0  1]    σᵧ = [0 -i]    σ_z = [1  0]
     [1  0]         [i  0]         [0 -1]
```

**自旋的奇特**：
- 不是"真的在转"
- 内禀属性，不能停止
- 半整数自旋（费米子）

---

### 泡利不相容原理

**费曼解释**：两个电子不能处于完全相同的状态。这决定了元素周期表。

**原理**：
```
在全同费米子系统中，
不能有两个粒子处于相同量子态。
```

**对氢原子的意义**：
```
每个轨道最多2个电子（自旋相反）
n, l, m, mₛ 四个量子数完全确定一个状态
```

**费米子 vs 玻色子**：
```
费米子：半整数自旋（电子、质子、中子）
       遵循费米-狄拉克统计
       泡利不相容原理

玻色子：整数自旋（光子、介子）
       遵循玻色-爱因斯坦统计
       可以在同一态（激光、BEC）
```

## 4. 量子力学诠释

### 哥本哈根诠释

**费曼解释**：主流诠释，由玻尔等人提出。测量导致波函数坍缩。

**核心观点**：
1. 波函数描述"我们的知识"
2. 测量导致坍缩
3. 量子世界本质上概率性
4. 观测者扮演特殊角色

**费曼态度**：
> "别问它怎么会那样，没人知道。"

---

### 多世界诠释

**费曼解释**：没有坍缩！每次测量，宇宙分裂成多个平行宇宙。

**核心观点**（Everett, 1957）：
1. 波函数永不坍缩
2. 所有可能都实际发生
3. 在不同"分支"中
4. 观测者也被分裂

**优势**：
- 理论简洁（无坍缩）
- 确定性演化

**问题**：
- 平行宇宙的存在无法验证
- 概率的起源不明

---

### 量子退相干

**费曼解释**：量子系统与环境的相互作用导致叠加态"消失"。

**退相干**：
```
|ψ⟩ = c₁|a⟩|E₁⟩ + c₂|b⟩|E₂⟩

|E₁⟩, |E₂⟩：环境态
当环境态正交时，干涉消失
```

**意义**：
- 解释量子到经典的过渡
- 测量问题的部分解答
- 量子计算的主要障碍

---

## 5. 量子纠缠 ⭐⭐⭐⭐⭐

### EPR佯谬

**费曼解释**：爱因斯坦不相信量子力学的"鬼魅超距作用"。

**EPR论证**（1935）：
```
1. 产生纠缠粒子对
2. 分开很远
3. 测量粒子A的自旋
4. 粒子B的自旋"瞬间"确定

爱因斯坦：这违反相对论！
```

**玻尔的回应**：
```
粒子在测量前没有确定状态
不存在"超距作用"
```

---

### 贝尔不等式

**费曼解释**：贝尔找到了判据，可以区分"隐变量"和"量子力学"。

**贝尔不等式**（1964）：
```
经典理论：|P(a,b) - P(a,c)| ≤ 1 + P(b,c)

量子力学：可以违反这个不等式
```

**实验验证**（Aspect, 1982；2022诺奖）：
```
量子力学正确
局域隐变量理论错误
```

**意义**：
- 量子纠缠是真实的
- 量子力学非局域
- 挑战直觉

---

### 纠缠态

**费曼解释**：两个粒子的状态"纠缠"在一起，无法分离描述。

**纠缠态例子**（贝尔态）：
```
|Φ⁺⟩ = (1/√2)(|↑⟩|↑⟩ + |↓⟩|↓⟩)

|Ψ⁺⟩ = (1/√2)(|↑⟩|↓⟩ + |↓⟩|↑⟩)
```

**特点**：
- 不能写成两个独立态的乘积
- 测量一个，另一个"瞬间"确定
- 距离无关（非局域）

**应用**：
- 量子通信
- 量子计算
- 量子隐形传态

---

### 量子隐形传态

**费曼解释**：不是传送物质，而是传送"量子状态"。

**过程**：
```
1. 共享纠缠对（Alice和Bob）
2. Alice有要传送的态|ψ⟩
3. Alice对|ψ⟩和她的纠缠粒子做测量
4. 经典通信告知测量结果
5. Bob根据结果操作，恢复|ψ⟩
```

**关键**：
- 原始态被破坏
- 需要经典通信（不超光速）
- 已实验验证（光子、原子）

## 6. 量子力学应用

### 激光

**费曼解释**：激光是"受激辐射"的放大。

**原理**：
```
1. 粒子数反转（上能级粒子多）
2. 光子诱导跃迁
3. 受激辐射：产生相同光子
4. 链式反应：放大
```

**相干光**：
- 所有光子相位相同
- 方向相同
- 频率相同

---

### 核磁共振（NMR）

**费曼解释**：原子核自旋在磁场中能级分裂，用射频激发。

**原理**：
```
1. 核自旋在外磁场中分裂（塞曼效应）
2. 射频激发跃迁
3. 检测共振信号
4. 分析分子结构
```

**应用**：
- MRI（医学成像）
- 化学分析
- 量子计算（自旋量子比特）

---

### 扫描隧道显微镜（STM）

**费曼解释**：利用量子隧穿效应，"触摸"原子。

**原理**：
```
1. 探针接近表面（几个埃）
2. 电子隧穿产生电流
3. 电流 ∝ e^(-d)
4. 扫描成像
```

**应用**：
- 原子分辨率成像
- 单原子操纵
- 表面科学
