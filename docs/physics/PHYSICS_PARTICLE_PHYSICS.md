## 1. 标准模型概述

### 什么是标准模型？

**费曼解释**：标准模型是描述基本粒子和它们相互作用的"元素周期表"。它告诉我们宇宙最基本的组成是什么。

**组成部分**：
```
1. 物质粒子（费米子）
   - 夸克（6种）
   - 轻子（6种）

2. 力载体（玻色子）
   - 光子（电磁力）
   - 胶子（强核力）
   - W/Z玻色子（弱核力）
   - 希格斯玻色子（质量起源）

3. 相互作用
   - 强相互作用
   - 弱相互作用
   - 电磁相互作用
```

**缺失**：
```
引力（太弱，粒子物理可忽略）
暗物质
暗能量
中微子质量
```

---

### 基本粒子一览

**费曼解释**：只有17种基本粒子（不是"基本"的更小组成）。

**费米子**（自旋½，物质）：

| 类型 | 第一代 | 第二代 | 第三代 |
|------|--------|--------|--------|
| **夸克** | 上u、下d | 粲c、奇s | 顶t、底b |
| **轻子** | 电子e、νe | μ子、νμ | τ子、ντ |

**玻色子**（自旋0或1，力载体）：

| 玻色子 | 作用 | 质量 |
|--------|------|------|
| 光子γ | 电磁 | 0 |
| 胶子g | 强 | 0 |
| W± | 弱 | 80.4 GeV |
| Z⁰ | 弱 | 91.2 GeV |
| 希格斯H | 质量 | 125 GeV |

**费曼直觉**：
> 12种费米子是"物质"，5种玻色子是"胶水"（希格斯比较特殊）。

## 2. 夸克

### 夸克种类

**费曼解释**：夸克是构成质子、中子的基本粒子。共有6种（味），带分数电荷。

**六种夸克**：

| 夸克 | 符号 | 电荷 | 质量 |
|------|------|------|------|
| 上 | u | +⅔ | ~2 MeV |
| 下 | d | -⅓ | ~5 MeV |
| 粲 | c | +⅔ | ~1.3 GeV |
| 奇 | s | -⅓ | ~95 MeV |
| 顶 | t | +⅔ | ~173 GeV |
| 底 | b | -⅓ | ~4.2 GeV |

**代**：
```
第一代：(u, d) - 构成普通物质
第二代：(c, s) - 较重，不稳定
第三代：(t, b) - 最重，极不稳定
```

---

### 夸克禁闭

**费曼解释**：夸克不能单独存在，总是被"关"在强子里面。

**原因**：
```
强相互作用随距离增强
（不像电磁力随距离减弱）

拉夸克 → 势能增加 → 产生新夸克对
永远分离不出来！
```

**证据**：
```
从未观察到自由夸克
只有强子（qqq或qq̄）
```

---

### 颜色荷

**费曼解释**：夸克有"颜色"（不是真颜色，是量子数）。

**三色**：
```
红(R)、绿(G)、蓝(B)
加上反颜色
```

**色荷**：
```
夸克带色
胶子带色荷
强相互作用 = 色相互作用
```

**无色**：
```
强子必须无色
重子：RGB组合（白）
介子：色+反色（白）
```

---

## 3. 轻子

### 轻子种类

**费曼解释**：轻子是不参与强相互作用的费米子。

**六种轻子**：

| 轻子 | 符号 | 电荷 | 质量 |
|------|------|------|------|
| 电子 | e⁻ | -1 | 0.511 MeV |
| 电子中微子 | νe | 0 | < 0.1 eV |
| μ子 | μ⁻ | -1 | 105.7 MeV |
| μ中微子 | νμ | 0 | < 0.2 eV |
| τ子 | τ⁻ | -1 | 1.78 GeV |
| τ中微子 | ντ | 0 | < 18 MeV |

---

### 电子

**费曼解释**：电子是最轻的带电轻子，构成原子的外层。

**性质**：
```
质量：0.511 MeV/c²
电荷：-e
自旋：½
稳定（不衰变）
```

**应用**：
```
电力
化学
电子显微镜
```

---

### 中微子

**费曼解释**：中微子是最神秘的粒子，几乎不与任何东西相互作用。

**性质**：
```
无电荷
极轻（原以为无质量）
只参与弱相互作用
穿透地球如入无物
```

**来源**：
```
太阳（太阳中微子）
核反应堆
宇宙线
超新星
```

**探测困难**：
```
需要巨大探测器
数吨甚至千吨物质
等待稀有相互作用
```

**中微子振荡**：
```
νe ↔ νμ ↔ ντ
证明中微子有质量！
（超出原始标准模型）
```

---

## 4. 强相互作用（QCD）

### 量子色动力学

**费曼解释**：QCD描述夸克和胶子的相互作用，是强相互作用的量子场论。

**特点**：
```
1. 夸克带"颜色"
2. 胶子传递强相互作用
3. 渐近自由（近距离弱）
4. 夸克禁闭（远距离强）
```

**渐近自由**：
```
高能量/短距离：α_s 很小
夸克几乎自由

低能量/长距离：α_s 很大
夸克被禁闭
```

---

### 胶子

**费曼解释**：胶子是强相互作用的载体，把夸克"粘"在一起。

**性质**：
```
自旋：1
质量：0（无质量）
带颜色荷
8种（颜色组合）
```

**与光子对比**：
```
光子：不带电荷，不与光子相互作用
胶子：带色荷，胶子间相互作用
```

---

### 强子

**费曼解释**：强子是由夸克组成的复合粒子。

**两类**：

**1. 重子（3夸克）**：
```
质子：uud
中子：udd
Ω⁻：sss
```

**2. 介子（夸克-反夸克）**：
```
π⁺：u d̄
π⁻：d ū
K⁰：d s̄
```

**夸克模型**：
```
Gell-Mann (1964)
解释了"粒子动物园"
预言了Ω⁻
```

---

## 5. 弱相互作用

### 弱相互作用特点

**费曼解释**：弱力是"改变味道"的力，可以改变夸克和轻子的类型。

**特点**：
```
1. 短程（W/Z重）
2. 弱（耦合常数小）
3. 唯一违反宇称的相互作用
4. 改变粒子类型
```

---

### W/Z玻色子

**费曼解释**：W和Z是弱相互作用的载体。

**W玻色子**：
```
W⁺, W⁻
质量：80.4 GeV
电荷：±1
带电弱流
```

**Z玻色子**：
```
Z⁰
质量：91.2 GeV
电荷：0
中性弱流
```

**发现**（1983）：
```
CERN，Rubbia领导
诺贝尔奖1984
```

---

### β衰变

**费曼解释**：β衰变是弱相互作用的典型例子。

**中子衰变**：
```
n → p + e⁻ + ν̄e

实际上：
d → u + W⁻
W⁻ → e⁻ + ν̄e

下夸克变上夸克
W⁰衰变为电子和反中微子
```

**费曼图**：
```
n ──→ p
      |
      W⁻
     / \
    e⁻  ν̄e
```

---

### 宇称不守恒

**费曼解释**：弱相互作用分左右，自然界有"手性"。

**发现**（吴健雄，1957）：
```
钴-60 β衰变
电子发射方向不对称
```

**意义**：
```
宇称(P)不守恒！
自然区分左和右
震惊物理学界
```

**CP破坏**：
```
联合宇称(CP)也不完全守恒
解释物质-反物质不对称？
```

## 6. 电磁相互作用（QED）

### 量子电动力学

**费曼解释**：QED描述电子和光子的相互作用，是最成功的量子场论。

**特点**：
```
精确度极高
实验符合到10⁻¹²
费曼图工具
```

**费曼图**：
```
电子-电子散射：

e⁻ ────\    /──── e⁻
         γ
e⁻ ────/    \──── e⁻

光子交换
```

**重整化**：
```
处理无穷大
可重整化理论
朗道：QED的完美
```

---

### 光子

**费曼解释**：光子是光的量子，电磁相互作用的载体。

**性质**：
```
自旋：1
质量：0
电荷：0
速度：c
能量：E = hν
```

**相互作用**：
```
与带电粒子耦合
耦合常数：α ≈ 1/137
```

## 7. 希格斯机制 ⭐⭐⭐⭐⭐

### 希格斯机制

**费曼解释**：希格斯场像"糖浆"，粒子在里面"游泳"获得质量。

**自发对称性破缺**：
```
规范对称性不允许有质量
但希格斯场打破对称性
粒子获得质量
```

**类比**：
```
房间里的派对
人们均匀分布（对称）
名人进来
人们围过来（对称打破）
名人移动困难（获得"质量"）
```

---

### 希格斯玻色子

**费曼解释**：希格斯粒子是希格斯场的"涟漪"。

**性质**：
```
自旋：0
质量：125 GeV
无电荷
不稳定（衰变快）
```

**发现**（2012.7.4）：
```
LHC，ATLAS和CMS
125 GeV
标准模型最后一块拼图
诺贝尔奖2013（Englert, Higgs）
```

**意义**：
```
验证希格斯机制
解释质量起源
标准模型完成
```

## 8. 对撞机

### LHC

**费曼解释**：大型强子对撞机是人类最大的机器，探索最小的事物。

**参数**：
```
周长：27 km
能量：13-14 TeV
地点：CERN，日内瓦
地下：100m深
```

**探测器**：
```
ATLAS：通用
CMS：通用
ALICE：重离子
LHCb：底夸克物理
```

**成就**：
```
2012：希格斯发现
夸克-胶子等离子体
超出标准模型搜寻
```

## 9. 超出标准模型

### 标准模型的问题

**费曼解释**：标准模型很成功，但不是最终理论。

**未解问题**：
```
1. 引力未包含
2. 暗物质（不是标准模型粒子）
3. 暗能量
4. 中微子质量
5. 物质-反物质不对称
6. 等级问题
7. 为什么三代？
8. 参数太多（19+）
```

---

### 超对称（SUSY）

**费曼解释**：每个粒子都有一个"超对称伙伴"。

**预言**：
```
费米子 ↔ 玻色子
电子 ↔ 标量电子
光子 ↔ 光微子
```

**动机**：
```
解决等级问题
暗物质候选
大统一理论
```

**现状**：
```
LHC未发现
越来越紧的限制
可能不存在，或在更高能标
```

---

### 大统一理论

**费曼解释**：把强、弱、电磁力统一为一个力。

**思想**：
```
高能量：耦合常数会合
单一规范群
质子衰变
```

**预测**：
```
质子衰变：τ > 10³⁴年
实验未观测到
```
