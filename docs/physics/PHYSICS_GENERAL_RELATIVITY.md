## 1. 广义相对论基础

### 等效原理

**费曼解释**：在自由下落的电梯里，你感觉不到引力。引力和加速度不可区分。

**爱因斯坦的"最快乐的想法"**（1907）：
```
"如果一个人自由下落，他不会感觉到自己的重量。"
```

**弱等效原理**：
```
惯性质量 = 引力质量

m_惯性 · a = m_引力 · g
```

**强等效原理**：
```
在足够小的区域内，
引力效应与加速运动不可区分
```

**例子**：
```
1. 地面上的观察者：感觉重量
2. 自由下落的电梯：失重
3. 加速的火箭（无引力）：感觉"重量"

1和3的感觉完全相同！
```

**费曼直觉**：
> 引力不是"力"，而是时空的弯曲。物体只是在弯曲时空中走"直线"（测地线）。

---

### 时空弯曲

**费曼解释**：质量告诉时空如何弯曲，时空告诉质量如何运动。

**比喻**：
```
在平坦床单上放重球
床单凹陷
轻球滚向重球
不是被"拉"，而是沿着弯曲表面走
```

**数学描述**：
```
时空度规 g_μν 描述弯曲

平直时空（闵可夫斯基）：
ds² = c²dt² - dx² - dy² - dz²

弯曲时空：
ds² = g_μν dx^μ dx^ν
```

---

### 广义协变原理

**费曼解释**：物理定律在所有参考系中形式相同（不仅是惯性系）。

**原理**：
```
物理方程在任意坐标变换下协变
必须用张量形式表达
```

**与狭义相对论区别**：
```
狭义相对论：惯性系
广义相对论：任意参考系（包括加速）
```

## 2. 爱因斯坦场方程 ⭐⭐⭐⭐⭐

### 场方程

**费曼解释**：这个方程描述物质如何产生时空弯曲。

**爱因斯坦场方程**：
```
G_μν = (8πG/c⁴)T_μν

G_μν：爱因斯坦张量（时空弯曲）
T_μν：能动张量（物质能量）
G：牛顿引力常数
c：光速
```

**展开形式**：
```
R_μν - ½Rg_μν + Λg_μν = (8πG/c⁴)T_μν

R_μν：里奇张量
R：曲率标量
Λ：宇宙学常数
```

**费曼直觉**：
```
左边：时空的几何（舞台）
右边：物质和能量（演员）
物质告诉时空如何弯曲
```

**历史趣闻**：
```
爱因斯坦先写错了，后来修正
1915年11月25日最终形式
"我激动得睡不着觉"
```

---

### 宇宙学常数

**费曼解释**：爱因斯坦引入的"反引力"项，为了得到静态宇宙。

**引入原因**（1917）：
```
没有Λ：宇宙必须膨胀或收缩
爱因斯坦想要静态宇宙
加入Λ平衡引力
```

**爱因斯坦的"最大错误"**：
```
1929哈勃发现宇宙膨胀
静态宇宙不需要了
爱因斯坦称Λ为"最大错误"
```

**现代复活**（1998）：
```
宇宙加速膨胀
暗能量 = 宇宙学常数？
Λ ≠ 0，占宇宙能量约70%
```

## 3. 经典解

### 施瓦西解

**费曼解释**：球对称质量外的时空几何。描述黑洞和行星轨道。

**施瓦西度规**（1916）：
```
ds² = (1 - r_s/r)c²dt² - (1 - r_s/r)⁻¹dr² 
      - r²dθ² - r²sin²θdφ²

r_s = 2GM/c²（施瓦西半径）
```

**施瓦西半径**：
```
r_s = 2GM/c²

地球：r_s ≈ 9 mm
太阳：r_s ≈ 3 km
```

**意义**：
- 描述球对称质量外的引力场
- 行星轨道（比牛顿更精确）
- 黑洞

---

### 克尔解

**费曼解释**：旋转黑洞的度规。

**克尔度规**（1963）：
```
更复杂的表达式
包含角动量参数a
```

**特点**：
- 旋转黑洞
- 能层（可提取能量）
- 拖拽效应

---

### 弗里德曼-勒梅特-罗伯逊-沃克度规（FLRW）

**费曼解释**：宇宙的大尺度几何。

**FLRW度规**：
```
ds² = -c²dt² + a(t)²[dr²/(1-kr²) + r²dΩ²]

a(t)：尺度因子（宇宙膨胀）
k：曲率（+1闭，0平，-1开）
```

**宇宙学**：
- 大爆炸模型
- 宇宙膨胀
- 暗能量

## 4. 黑洞 ⭐⭐⭐⭐⭐

### 什么是黑洞？

**费曼解释**：引力太强，连光都逃不掉的区域。

**定义**：
```
事件视界内的区域
r < r_s = 2GM/c²
任何东西不能逃出
```

**形成**：
```
大质量恒星（> 3M_☉）坍缩
核燃料耗尽
引力坍缩
```

**类型**：
1. **恒星质量黑洞**（3-100 M_☉）
2. **超大质量黑洞**（10⁶-10¹⁰ M_☉）
3. **中等质量黑洞**（100-10⁶ M_☉）
4. **原初黑洞**（大爆炸产生？）

---

### 事件视界

**费曼解释**：不归点。一旦越过，永远回不来。

**定义**：
```
r = r_s = 2GM/c²

内部：任何信号不能到达外部
外部：可以接收内部（如果已有信号发出）
```

**特性**：
- 不是物理表面
- 穿越时没有特别感觉
- 外部观察者看物体永远不能到达

---

### 奇点

**费曼解释**：时空的"终点"，物理定律失效。

**性质**：
```
r = 0
曲率无限大
密度无限大
已知物理失效
```

**彭罗斯-霍金奇点定理**：
```
在合理条件下，
黑洞内部必然存在奇点
```

**问题**：
```
需要量子引力？
弦论？圈量子引力？
```

---

### 霍金辐射

**费曼解释**：黑洞不是完全黑的，它会"蒸发"。

**霍金辐射**（1974）：
```
黑洞温度：
T = ℏc³/(8πGMk_B)

质量越小，温度越高
黑洞会蒸发
```

**机制**：
```
量子真空涨落
虚粒子对
一个落入，一个逃逸
黑洞失去质量
```

**结果**：
```
黑洞蒸发
10¹⁰⁰年后消失？
信息丢失问题
```

---

### 黑洞热力学

**费曼解释**：黑洞有熵，遵循热力学定律。

**贝肯斯坦-霍金熵**：
```
S = (kc³A)/(4Gℏ)

A：事件视界面积
熵正比于表面积，不是体积！
```

**黑洞四定律**：
```
0. 温度趋于平衡
1. dM = TdS + ...（能量守恒）
2. 熵永不减少
3. 不能达到T = 0
```

**全息原理**：
```
三维空间信息编码在二维表面？
黑洞熵暗示深层物理
```

## 5. 引力波 ⭐⭐⭐⭐⭐

### 什么是引力波？

**费曼解释**：时空的涟漪。像池塘里的水波，但在时空中传播。

**预言**（爱因斯坦，1916）：
```
时空弯曲以波的形式传播
速度 = c
```

**特性**：
- 横波
- 两种偏振（+和×）
- 极其微弱
- 携带能量

---

### 探测

**费曼解释**：探测时空的微小"拉伸"和"压缩"。

**LIGO原理**：
```
激光干涉仪
两条4km臂
引力波经过：一臂伸长，一臂缩短
测量相位差
```

**精度**：
```
测量变化：ΔL/L ≈ 10⁻²¹
相当于：地球到太阳距离变化一个原子大小
```

**首次探测**（2015.9.14）：
```
GW150914
两个黑洞合并（36M_☉ + 29M_☉ → 62M_☉）
3M_☉转化为引力波能量
距地球13亿光年
```

**诺贝尔奖**（2017）：
```
Weiss, Barish, Thorne
```

---

### 引力波源

**类型**：
1. **致密双星合并**
   - 黑洞-黑洞
   - 中子星-中子星
   - 黑洞-中子星

2. **连续波**
   - 旋转中子星

3. **随机背景**
   - 早期宇宙
   - 大量叠加源

4. **爆发**
   - 超新星

## 6. 经典验证

### 水星近日点进动

**费曼解释**：水星轨道的"摆动"，牛顿力学解释不了。

**观测**：
```
每世纪进动：574角秒
牛顿力学：531角秒
差值：43角秒
```

**广义相对论计算**：
```
43角秒/世纪 ✓
爱因斯坦："我好几天高兴得发狂"
```

---

### 光线弯曲

**费曼解释**：光线经过太阳附近会弯曲。

**预言**：
```
经过太阳边缘：
偏转角 = 1.75角秒
```

**验证**（爱丁顿，1919）：
```
日食时测量恒星位置
偏移符合预言
爱因斯坦成为世界名人
```

---

### 引力红移

**费曼解释**：光从引力场中"爬出来"会损失能量，频率降低。

**公式**：
```
ν_observed = ν_emitted × √(1 - r_s/r)

或近似：
Δν/ν ≈ -GM/(rc²)
```

**验证**：
```
1959: Pound-Rebka实验
哈佛塔，高度差22.5m
符合预言
```

**应用**：
```
GPS必须修正引力红移
否则误差10km/天
```

---

### 雷达回波延迟

**费曼解释**：雷达信号经过太阳附近会有延迟。

**预言**（夏皮罗，1964）：
```
经过太阳附近
延迟约200 μs
```

**验证**：
```
水星、金星雷达回波
符合预言
```

## 7. 宇宙学

### 大爆炸模型

**费曼解释**：宇宙从一个极热极密的状态膨胀而来。

**证据**：
1. **宇宙膨胀**（哈勃，1929）
2. **宇宙微波背景**（1965）
3. **轻元素丰度**

**时间线**：
```
t = 0：奇点
t ~ 10⁻³⁶s：暴胀
t ~ 3min：核合成
t ~ 38万年：复合，CMB发射
t ~ 138亿年：现在
```

---

### 宇宙微波背景

**费曼解释**：大爆炸的"余晖"，宇宙早期的光。

**发现**（Penzias & Wilson，1965）：
```
各向同性的微波辐射
T ≈ 2.7 K
诺贝尔奖1978
```

**特性**：
```
温度：2.725 K
黑体谱
各向异性：ΔT/T ~ 10⁻⁵
```

**意义**：
- 大爆炸的证据
- 早期宇宙的"照片"
- 宇宙学参数测量

---

### 暗物质与暗能量

**费曼解释**：宇宙的95%我们看不到。

**暗物质**（~27%）：
```
不发光、不吸收光
只有引力效应
证据：星系旋转曲线、引力透镜
候选：WIMPs、轴子？
```

**暗能量**（~68%）：
```
导致宇宙加速膨胀
可能是宇宙学常数Λ
本质未知
```

**普通物质**（~5%）：
```
我们熟悉的物质
恒星、行星、气体
```
