### 【道德心理学】(Moral Psychology)

**词源**：希腊语 *ethos* (习俗)、*psyche* (心灵)、*logos* (学问)

**定义**：研究道德判断、动机、情感和行为的心理机制。

**费曼解释**：
当你做出道德判断时：
- "这是错的！"
- 你的大脑发生了什么？
- 你为什么这么判断？
- 你会怎么做？

道德心理学研究"道德的心理机制"。

**道德心理学的核心问题**：

| 问题 | 描述 |
|------|------|
| **道德判断** | 道德判断是如何形成的？ |
| **道德动机** | 人们为什么做道德的事？ |
| **道德情感** | 情感在道德中的作用？ |
| **道德发展** | 道德能力如何发展？ |
| **道德差异** | 道德的个体和文化差异？ |

**道德心理学的主要议题**：

| 议题 | 描述 |
|------|------|
| **直觉 vs 推理** | 道德判断来自直觉还是推理？ |
| **情感的作用** | 情感如何影响道德判断？ |
| **意志软弱** | 为什么知道应该做却不做？ |
| **道德责任** | 什么条件下我们负道德责任？ |
| **道德运气** | 运气如何影响道德评价？ |

**代表人物**：
- **Haidt**：社会直觉主义
- **Kohlberg**：道德发展阶段
- **Greene**：道德双加工
- **当代**：道德心理学研究者

**经典文献**：
- Haidt, "The Emotional Dog and Its Rational Tail" (2001)
- Kohlberg, *The Psychology of Moral Development* (1981)
- Greene, *Moral Tribes* (2013)

**当代发展 (2020-2026)**：

### 【道德判断】(Moral Judgment)

**定义**：对行为、品格或事态的道德评价。

**费曼解释**：
看到一个人偷东西：
- 你立即判断："这是错的"
- 这就是道德判断

道德判断是"对事物说对或错"。

**道德判断的理论**：

| 理论 | 核心主张 | 代表 |
|------|----------|------|
| **理性主义** | 道德判断来自推理 | 康德、Kohlberg |
| **情感主义** | 道德判断来自情感 | 休谟、Haidt |
| **双加工理论** | 两种加工都重要 | Greene |
| **社会直觉主义** | 直觉为主，推理为辅 | Haidt |

**理性主义模型**：

**核心主张**：
- 道德判断是推理的结果
- 情感可能影响但不是核心
- 道德推理遵循逻辑规则

**Kohlberg的道德推理阶段**：

| 阶段 | 描述 | 推理方式 |
|------|------|----------|
| **前习俗** | 奖惩导向 | 避免惩罚 |
| **习俗** | 社会规范导向 | 遵循规则 |
| **后习俗** | 原则导向 | 普遍原则 |

**情感主义模型**：

**核心主张**：
- 道德判断首先是情感反应
- 推理是事后合理化
- "理性是情感的奴隶"（休谟）

**Haidt的社会直觉主义**：

**核心主张**：
1. 道德判断通常是直觉的
2. 推理主要是事后的
3. 社会影响塑造道德直觉

```
直觉 → 判断 → 推理（事后合理化）
         ↑
      社会影响
```

**双加工理论** (Dual-Process Theory)：

**Greene的观点**：
- 系统1：快速、自动、情感
- 系统2：缓慢、受控、推理
- 不同情境激活不同系统

**电车难题的证据**：

| 情境 | 激活系统 | 脑区 |
|------|----------|------|
| **开关** | 系统2（功利） | 背外侧前额叶 |
| **推胖子** | 系统1（情感） | 情感相关区域 |

**道德判断的影响因素**：

| 因素 | 影响 |
|------|------|
| **情感** | 厌恶、同情影响判断 |
| **框架** | 同样问题的不同表述 |
| **群体** | 内群体偏好 |
| **权威** | 权威的影响 |
| **文化** | 文化差异 |

**代表人物**：
- **Kohlberg**：道德推理阶段
- **Haidt**：社会直觉主义
- **Greene**：双加工理论
- **当代**：道德心理学研究者

**经典文献**：
- Kohlberg, *The Psychology of Moral Development* (1981)
- Haidt, "The Emotional Dog and Its Rational Tail" (2001)
- Greene et al., "An fMRI Investigation of Emotional vs. Moral Judgment" (2001)

**当代发展 (2020-2026)**：
- **神经影像**：道德判断的神经基础

- **跨文化研究**：道德判断的文化差异

**跨学科联系**：
- **认知心理学**：认知加工
- **神经科学**：神经基础
- **社会心理学**：社会影响
- **哲学**：道德认识论

**中国视角**：

- **四端**：四端说
- **当代**：道德判断的中国视角

**争议问题**：
1. 道德判断主要是直觉还是推理？

### 【道德直觉】(Moral Intuition)

**定义**：快速、自动的道德评价，不需要有意推理。

**费曼解释**：
看到有人虐待动物：
- 你立即感到"这是错的"
- 不需要思考为什么
- 这就是道德直觉

**道德直觉的特征**：

| 特征 | 描述 |
|------|------|
| **快速** | 立即出现 |
| **自动** | 不需要有意努力 |
| **情感性** | 常伴随情感 |
| **非反思** | 不经过深思 |
| **可错** | 可能是错的 |

**Haidt的道德基础理论**：

| 道德基础 | 描述 | 例子 |
|----------|------|------|
| **关怀/伤害** | 关心他人，避免伤害 | 同情弱者 |
| **公平/欺骗** | 公平对待，不欺骗 | 反对不公 |
| **忠诚/背叛** | 对群体忠诚 | 爱国主义 |
| **权威/颠覆** | 尊重权威 | 尊老 |
| **圣洁/堕落** | 追求圣洁 | 厌恶乱伦 |
| **自由/压迫** | 追求自由 | 反对奴役 |

**道德直觉的来源**：

| 来源 | 描述 |
|------|------|
| **进化** | 进化塑造的道德直觉 |
| **文化** | 文化塑造的道德直觉 |
| **经验** | 个人经验形成直觉 |
| **学习** | 学习获得的直觉 |

**对道德直觉的批评**：

| 批评 | 描述 | 回应 |
|------|------|------|
| **不可靠** | 直觉经常错误 | 区分可靠与不可靠直觉 |
| **文化偏见** | 直觉因文化而异 | 寻找普遍直觉 |
| **可操纵** | 直觉可以被操纵 | 提高直觉意识 |
| **冲突** | 直觉可能冲突 | 反思平衡 |

**道德直觉与道德推理的关系**：

```
传统观点：推理 → 直觉（推理导致直觉）
Haidt：直觉 → 推理（直觉导致推理）
综合：直觉 ↔ 推理（双向影响）
```

**代表人物**：
- **Haidt**：社会直觉主义
- **Ross**：道德直觉主义
- **当代**：道德心理学研究者

**经典文献**：
- Haidt, "The Emotional Dog and Its Rational Tail" (2001)
- Haidt, *The Righteous Mind* (2012)

**当代发展 (2020-2026)**：
- **道德基础扩展**：新的道德基础
- **直觉训练**：培养道德直觉
- **AI直觉**：AI道德直觉
- **跨文化直觉**：道德直觉的文化差异

**跨学科联系**：
- **心理学**：直觉心理学
- **神经科学**：直觉的神经基础
- **进化心理学**：道德进化
- **哲学**：直觉主义

**中国视角**：
- **良知**：良知的直觉性
- **恻隐**：恻隐之心
- **直觉**：直觉与道德
- **当代**：道德直觉的中国视角

**争议问题**：
1. 道德直觉可靠吗？
2. 道德直觉是先天的还是后天的？
3. 如何处理直觉冲突？

**相关词条**：道德判断、道德推理、道德基础

### 【道德动机】(Moral Motivation)

**定义**：推动人们做出道德行为的心理力量。

**费曼解释**：
你看到一个老人摔倒：
- 你扶他起来
- 你为什么这么做？
- 这是道德动机

**道德动机的理论**：

| 理论 | 核心主张 |
|------|----------|
| **内在主义** | 道德判断内在地激发行动 |
| **外在主义** | 道德判断需要外在欲望 |
| **情感理论** | 情感驱动道德行为 |
| **理性理论** | 理性驱动道德行为 |

**道德动机的来源**：

| 来源 | 描述 | 例子 |
|------|------|------|
| **情感** | 同情、内疚、羞耻 | 同情弱者 |
| **理性** | 理性认识到义务 | 康德的义务 |
| **品格** | 德性驱动行为 | 诚实的人诚实 |
| **社会** | 社会规范和期望 | 遵循规范 |

**意志软弱** (Akrasia)：

**定义**：知道应该做却不做

**例子**：
- 知道应该锻炼但不锻炼
- 知道不应该吃太多但吃太多

**解释**：

| 解释 | 描述 |
|------|------|
| **情感压倒** | 情感压倒理性 |
| **意志失败** | 意志力不足 |
| **信念改变** | 在当下信念改变 |
| **欲望结构** | 欲望的结构问题 |

**道德动机的发展**：

| 阶段 | 动机来源 | 描述 |
|------|----------|------|
| **儿童** | 奖惩 | 避免惩罚，获得奖励 |
| **青少年** | 社会认可 | 寻求认可 |
| **成人** | 内在原则 | 内化的原则 |

**代表人物**：
- **康德**：理性动机
- **休谟**：情感动机
- **当代**：道德动机研究者

**经典文献**：
- Hume, *A Treatise of Human Nature*
- Kant, *Groundwork of the Metaphysics of Morals*
- Mele, *Motivation and Agency* (2003)

**当代发展 (2020-2026)**：

- **动机干预**：如何增强道德动机
- **AI动机**：AI道德动机
- **跨文化动机**：道德动机的文化差异

**跨学科联系**：

### 【道德情感】(Moral Emotions)

**定义**：与道德判断和行为相关的情感。

**费曼解释**：
- 看到不公，你愤怒
- 做错事，你内疚
- 帮助他人，你感到温暖
- 这些是道德情感

**主要道德情感**：

| 情感 | 触发 | 功能 |
|------|------|------|
| **内疚** | 违背道德标准 | 促进补偿 |
| **羞耻** | 损害社会形象 | 促进社会遵守 |
| **愤怒** | 感知不公 | 促进正义 |
| **同情** | 他人受苦 | 促进帮助 |
| **厌恶** | 道德违反 | 避免污染 |
| **骄傲** | 道德成就 | 强化行为 |
| **感激** | 受益于他人 | 促进互惠 |

**内疚 vs 羞耻**：

| 特征 | 内疚 | 羞耻 |
|------|------|------|
| **焦点** | 行为 | 自我 |
| **内容** | "我做了坏事" | "我是坏人" |
| **结果** | 补偿 | 逃避 |
| **适应性** | 更适应 | 较不适应 |

**同情** (Empathy/Sympathy)：

**定义**：
- **同情**：感受他人的感受
- **恻隐**：对他人苦难的关切

**同情的作用**：
- 促进利他行为
- 是道德判断的基础
- 道德发展的起点

**对同情的批评**：

| 批评 | 描述 |
|------|------|
| **偏袒** | 同情偏向亲近的人 |
| **有限** | 同情是有限的 |
| **不可靠** | 同情可能误导 |
| **压迫** | 同情可能延续压迫 |

**道德情感的功能**：

| 功能 | 描述 |
|------|------|
| **评价** | 评价道德行为 |
| **动机** | 激发道德行为 |
| **沟通** | 沟通道德态度 |
| **惩罚** | 社会惩罚机制 |

**代表人物**：
- **休谟**：道德情感论
- **Prinz**：情感主义
- **Nussbaum**：情感与正义
- **当代**：道德情感研究者

**经典文献**：

- Prinz, *The Emotional Construction of Morals* (2007)
- Nussbaum, *Upheavals of Thought* (2001)

**当代发展 (2020-2026)**：
- **神经情感**：道德情感的神经基础
- **AI情感**：AI道德情感
- **情感干预**：培养道德情感
- **跨文化情感**：道德情感的文化差异

**跨学科联系**：
- **心理学**：情感心理学
- **神经科学**：情感的神经基础

**中国视角**：
- **恻隐**：恻隐之心

### 【道德发展】(Moral Development)

**定义**：道德推理、情感和行为能力的发展过程。

**费曼解释**：
儿童如何变成道德的成人？
- 小时候：不做坏事因为怕惩罚
- 长大后：不做坏事因为知道是错的
- 这就是道德发展

**Kohlberg的道德发展阶段**：

| 水平 | 阶段 | 描述 |
|------|------|------|
| **前习俗** | 1. 惩罚服从 | 避免惩罚 |
| | 2. 利己主义 | 自己的利益 |
| **习俗** | 3. 好孩子 | 社会认可 |
| | 4. 法律秩序 | 遵守法律 |
| **后习俗** | 5. 社会契约 | 社会契约 |
| | 6. 普遍原则 | 普遍道德原则 |

**Kohlberg的批评者**：

| 批评 | 描述 | 批评者 |
|------|------|--------|
| **性别偏见** | 基于男性样本 | Gilligan |
| **文化偏见** | 基于西方样本 | Shweder |
| **推理偏见** | 忽视情感 | Haidt |
| **阶段问题** | 不是严格阶段 | 批评者 |

**Gilligan的女性道德发展**：

**核心批评**：
- Kohlberg的研究主要基于男性
- 女性可能倾向于"关怀伦理"
- 不同的声音

**关怀伦理**：

| 特征 | 关怀伦理 | 正义伦理 |
|------|----------|----------|
| **焦点** | 关系 | 规则 |
| **方法** | 情境判断 | 抽象原则 |
| **价值** | 关怀、责任 | 权利、义务 |
| **关联** | 女性（更常见） | 男性（更常见） |

**Turiel的道德领域理论**：

| 领域 | 内容 | 例子 |
|------|------|------|
| **道德** | 伤害、公平、权利 | 不打人 |
| **习俗** | 社会规范 | 穿着得体 |
| **个人** | 个人选择 | 爱好 |

**道德发展的因素**：

| 因素 | 作用 |
|------|------|
| **家庭** | 早期道德社会化 |
| **同伴** | 同伴影响 |
| **学校** | 道德教育 |
| **文化** | 文化价值观 |

### 【道德责任】(Moral Responsibility)

**定义**：个体对其行为和后果承担道德义务的状态。

**费曼解释**：
- 你故意伤害某人 → 你负责任
- 你意外伤害某人 → 责任较轻
- 你被强迫伤害某人 → 可能不负责任

**道德责任的条件**：

| 条件 | 描述 |
|------|------|
| **控制** | 对行为有控制 |
| **知识** | 知道或应该知道 |
| **自由** | 自由选择 |
| **能力** | 有道德能力 |

**道德责任的理论**：

| 理论 | 核心主张 | 问题 |
|------|----------|------|
| **自由意志论** | 自由意志是责任基础 | 自由意志存在吗？ |
| **相容论** | 自由意志与决定论相容 | 什么是"自由"？ |
| **硬决定论** | 没有自由意志，没有责任 | 社会如何运作？ |
| **溯源论** | 责任来自品格溯源 | 如何溯源？ |

**道德责任与自由意志**：

| 立场 | 描述 |
|------|------|
| **自由意志** | 我们有自由选择的能力 |
| **决定论** | 所有事件都被因果决定 |
| **相容论** | 自由意志与决定论可以相容 |
| **不相容论** | 自由意志与决定论不相容 |

**影响责任的因素**：

| 因素 | 影响 |
|------|------|
| **强迫** | 减少或消除责任 |
| **无知** | 可能减少责任 |
| **精神疾病** | 可能减少责任 |
| **童年** | 可能减少责任 |

**道德运气** (Moral Luck)：

**定义**：超出我们控制的因素影响道德评价

**类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **因果运气** | 因果链上的运气 | 事故的随机性 |
| **结果运气** | 行为结果的运气 | 酒驾是否撞人 |
| **构成运气** | 品格的运气 | 天生倾向 |
| **环境运气** | 所处环境的运气 | 成长环境 |

**道德运气的问题**：
- 我们应该为超出控制的事负责吗？
- 如果不应该，什么可以负责？

**代表人物**：

- Strawson, "Freedom and Resentment" (1962)

**当代发展 (2020-2026)**：

- **集体责任**：集体道德责任

### 【利他主义】(Altruism)

**定义**：为了他人利益而行动的动机和行为。

**费曼解释**：
你看到一个孩子掉进河里，你跳下去救他——不考虑自己的危险。
这就是利他主义——为了他人而行动。

**核心要素**：
1. **他人导向**：动机指向他人利益
2. **自我牺牲**：可能需要自我牺牲
3. **无偿性**：不期望回报
4. **真诚性**：真正的利他动机

**利他主义的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **进化利他** | 有利于群体的基因 | 蜜蜂牺牲 |
| **心理利他** | 心理上的利他动机 | 帮助陌生人 |
| **伦理利他** | 应该利他的道德要求 | 功利主义 |

**利他主义的争议**：

| 问题 | 描述 |
|------|------|
| **真正的利他** | 是否存在真正的利他？ |
| **利己解释** | 利他是否隐藏利己动机？ |
| **利他范围** | 利他应该延伸多远？ |

**代表人物**：
- **Batson**：利他的心理学研究
- **Singer**：有效的利他主义

**相关词条**：道德动机、同情、有效的利他主义

### 【道德伪善】(Moral Hypocrisy)

**定义**：声称坚持道德标准但自己不遵守的行为。

**费曼解释**：
"不要说谎"——但你自己说谎。
这就是道德伪善——对己一套，对人一套。

**核心要素**：
1. **双重标准**：对己对人不同标准
2. **声称vs实践**：说一套做一套
3. **欺骗性**：试图维持道德形象
4. **普遍性**：伪善是普遍的？

**道德伪善的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **行为伪善** | 行为与声称不一致 | 说不要贪污但贪污 |
| **态度伪善** | 态度与行为不一致 | 说关心穷人不帮助 |
| **无意识伪善** | 不自知的伪善 | 真诚但不一致 |

**对道德伪善的解释**：

| 解释 | 描述 |
|------|------|
| **心理防御** | 伪善保护自我形象 |
| **社会压力** | 社会期望导致伪善 |
| **有限资源** | 有限意志力导致伪善 |

**代表人物**：
- **Batson**：伪善的心理学研究

**相关词条**：道德品格、道德动机、一致性

### 【道德认同】(Moral Identity)

**定义**：个人将道德价值作为自我概念核心的程度。

**费曼解释**：
"我是一个诚实的人"——如果你的自我认同与道德价值紧密相连，
你更可能按照道德价值行动。

**核心要素**：
1. **自我概念**：道德是自我的一部分
2. **中心性**：道德在自我中的中心程度
3. **自我一致性**：行为与道德自我一致
4. **情境敏感性**：认同可被情境激活

**道德认同的维度**：

| 维度 | 描述 |
|------|------|
| **内化** | 道德价值成为自我的一部分 |
| **象征化** | 道德价值在行动中体现 |

**道德认同与行为**：

| 研究 | 描述 |
|------|------|
| **预测行为** | 道德认同预测道德行为 |
| **志愿活动** | 高道德认同更多志愿 |
| **欺骗行为** | 低道德认同更多欺骗 |

**代表人物**：
- **Blasi**：道德认同理论
- **Aquino & Reed**：道德认同的测量

**相关词条**：自我概念、道德发展、道德品格

### 【道德脱钩】(Moral Disengagement)

**定义**：通过认知机制使不道德行为与道德自我分离。

**费曼解释**：
你做了不道德的事，但你告诉自己"这不是我的错"。
你用各种理由让不道德行为与你的道德自我"脱钩"。

**核心要素**：
1. **认知重构**：重新解释不道德行为
2. **责任转移**：将责任归咎于他人
3. **受害者贬低**：贬低受害者
4. **自我保护**：保护道德自我形象

**道德脱钩的机制**：

| 机制 | 描述 | 例子 |
|------|------|------|
| **道德正当化** | 重新定义为道德的 | "这是为他们好" |
| **委婉标签** | 使用中性语言 | "附带损害" |
| **优势比较** | 与更坏的比较 | "我比某人好" |
| **责任分散** | 大家都这样做 | "不是我一个人的错" |
| **责任转移** | 权威让我做的 | "我只是执行命令" |
| **结果忽视** | 忽视后果 | "没有真正伤害" |
| **非人化** | 受害者不是人 | "他们是动物" |
| **责备受害者** | 受害者活该 | "他们自己造成的" |

**代表人物**：
- **Bandura**：道德脱钩理论

**经典文献**：
- Bandura, *Moral Disengagement* (2016)

**相关词条**：道德伪善、道德品格、认知失调
