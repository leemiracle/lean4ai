### 【应用伦理学】(Applied Ethics)

**词源**：拉丁语 *applicare* (应用)、希腊语 *ethos* (习俗)

**定义**：将伦理理论和原则应用于具体实践领域和问题的学科。

**费曼解释**：
- 规范伦理学讨论一般原则（"不要杀人"）
- 元伦理学讨论道德本质（"错误"是什么意思？）
- 应用伦理学问：堕胎是杀人吗？AI决策公正吗？

应用伦理学是把伦理"应用"到具体问题。

**应用伦理学的核心问题**：

| 问题 | 描述 |
|------|------|
| **如何应用？** | 一般原则如何应用于具体情况？ |
| **理论选择** | 哪种伦理理论更适合？ |
| **情境判断** | 情境如何影响道德判断？ |
| **专业责任** | 专业人士的特殊道德责任？ |

**主要应用领域**：

| 领域 | 核心问题 |
|------|----------|
| **生命伦理** | 堕胎、安乐死、基因编辑 |
| **医学伦理** | 医患关系、知情同意、资源分配 |
| **商业伦理** | 企业责任、公平交易、广告 |
| **环境伦理** | 气候变化、动物权利、污染 |
| **AI伦理** | 算法公平、隐私、价值对齐 |
| **职业伦理** | 专业责任、利益冲突 |

**应用伦理学的方法**：

| 方法 | 描述 |
|------|------|
| **原则主义** | 应用一般原则 |
| **案例分析** | 分析具体案例 |
| **反思平衡** | 原则与直觉平衡 |
| **情境分析** | 情境敏感判断 |

**代表人物**：
- **Beauchamp & Childress**：生命伦理原则
- **Singer**：应用伦理学
- **当代**：各领域应用伦理学者

**经典文献**：
- Beauchamp & Childress, *Principles of Biomedical Ethics* (1979)
- Singer, *Practical Ethics* (1979)

**当代发展 (2020-2026)**：
- **AI伦理**：人工智能的道德问题
- **气候伦理**：气候变化伦理
- **全球健康**：全球健康伦理

**跨学科联系**：
- **医学**：医学伦理
- **法律**：法律伦理
- **商业**：商业伦理
- **科技**：科技伦理

**中国视角**：
- **应用**：理论与实践结合
- **情境**：情境伦理
- **当代**：应用伦理学的中国视角

**争议问题**：

### 【安乐死】(Euthanasia)

**词源**：希腊语 *eu* (好)、*thanatos* (死亡)——"好的死亡"

**定义**：在患者请求下，有意结束其生命以解除痛苦的行为。

**费曼解释**：
一个绝症患者：
- 极度痛苦，无法治愈
- 他请求医生结束他的生命
- 这是安乐死

安乐死是"主动结束痛苦的生命"。

**安乐死的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **主动安乐死** | 主动结束生命 | 注射致死药物 |
| **被动安乐死** | 停止治疗 | 撤除生命支持 |
| **自愿安乐死** | 患者同意 | 患者明确请求 |
| **非自愿安乐死** | 患者无法同意 | 昏迷患者 |
| **协助自杀** | 提供手段让患者自杀 | 开具致死药物 |

**安乐死的伦理争论**：

| 支持论证 | 反对论证 |
|----------|----------|
| **自主权** | 患者有权决定 | **生命神圣** | 生命不可侵犯 |
| **解除痛苦** | 避免无意义痛苦 | **滑坡论证** | 可能滥用 |
| **尊严** | 有尊严地死亡 | **医生角色** | 医生应该救人 |
| **质量** | 生命质量重要 | **替代方案** | 姑息治疗 |

**各理论立场**：

| 理论 | 立场 |
|------|------|
| **功利主义** | 如果减少痛苦，可能支持 |
| **义务论** | 主动杀人违背道德法则 |
| **德性伦理** | 关注动机和情境 |
| **权利论** | 患者有自主权 |

**各国实践**：

| 国家/地区 | 立场 |
|-----------|------|
| **荷兰** | 主动安乐死合法（2002） |
| **比利时** | 主动安乐死合法（2002） |
| **加拿大** | 医疗协助死亡合法（2016） |
| **美国部分州** | 协助自杀合法 |
| **大多数国家** | 安乐死非法 |

**滑坡论证**：

**论证**：
1. 如果允许安乐死
2. 就可能扩展到非自愿安乐死
3. 最终导致滥用
4. 因此，不应该允许安乐死

**批评**：
- 荷兰经验表明可以控制
- 任何政策都可能滥用
- 应该加强监管，而不是禁止

**代表人物**：
- **Rachels**：支持安乐死
- **Singer**：支持安乐死
- **当代**：生命伦理学者

**经典文献**：
- Rachels, "Active and Passive Euthanasia" (1975)
- Singer, *Rethinking Life and Death* (1994)

**当代发展 (2020-2026)**：
- **协助死亡扩展**：更多国家合法化
- **精神疾病**：精神疾病的协助死亡
- **AI决策**：AI在安乐死决策中的角色
- **全球趋势**：全球安乐死合法化趋势

**跨学科联系**：
- **医学**：临床实践
- **法律**：合法性问题
- **心理学**：心理影响
- **社会学**：社会态度

**中国视角**：
- **善终**：善终的观念
- **孝道**：孝道与安乐死
- **当代**：安乐死的中国讨论

**争议问题**：
1. 主动和被动安乐死有道德区别吗？
2. 安乐死会导致滑坡吗？
3. 患者真的能"自由"选择吗？

**相关词条**：生命伦理、自主权、死亡定义

### 【堕胎】(Abortion)

**定义**：故意终止妊娠的行为。

**费曼解释**：
一个意外怀孕的女性：
- 她可以选择堕胎
- 但胎儿也有"生命"
- 如何平衡？

堕胎是"女性权利 vs 胎儿生命"的问题。

**堕胎的伦理争论**：

| 支持堕胎权 | 反对堕胎 |
|------------|----------|
| **女性自主权** | 女性有权决定身体 | **胎儿生命权** | 胎儿有人权 |
| **隐私权** | 是私人决定 | **道德地位** | 胎儿有道德地位 |
| **社会后果** | 禁止堕胎有害 | **潜在人** | 胎儿是潜在的人 |
| **特殊情况** | 强奸、健康威胁 | **无辜生命** | 胎儿无辜 |

**胎儿道德地位问题**：

| 阶段 | 道德地位 | 支持者 |
|------|----------|--------|
| **受精** | 完整人权 | 保守派 |
| **心跳** | 开始有人权 | 一些保守派 |
| **存活期** | 有独立生存能力 | 温和派 |
| **出生** | 完整人权 | 自由派 |

**Thomson的小提琴家论证**：

**情景**：
- 你醒来发现一个小提琴家连在你身上
- 他的肾脏坏了，需要你的9个月
- 如果你断开，他会死
- 你必须继续吗？

**类比**：
- 你有权断开（即使他会死）
- 同样，女性有权堕胎（即使胎儿会死）
- 即使胎儿有人权，女性也有身体自主权

**对Thomson论证的批评**：

| 批评 | 描述 |
|------|------|
| **责任论证** | 怀孕通常是自愿行为 |
| **杀人 vs 让死** | 堕胎是主动杀人 |
| **父母义务** | 父母对孩子有特殊义务 |
| **不完美类比** | 小提琴家不是你的孩子 |

**代表人物**：
- **Thomson**：支持堕胎权
- **Marquis**：反对堕胎
- **当代**：生命伦理学者

**经典文献**：
- Thomson, "A Defense of Abortion" (1971)
- Marquis, "Why Abortion is Immoral" (1989)

**当代发展 (2020-2026)**：
- **美国最高法院**：推翻罗诉韦德案（2022）
- **全球趋势**：各国堕胎法变化
- **药物堕胎**：药物堕胎的伦理
- **技术发展**：胚胎研究

**跨学科联系**：
- **医学**：医疗实践
- **法律**：合法性问题
- **女性主义**：女性权利
- **宗教**：宗教观点

**中国视角**：
- **计划生育**：计划生育历史
- **胎儿地位**：中国传统的胎儿观
- **当代**：堕胎的中国讨论

**争议问题**：
1. 胎儿有道德地位吗？从什么时候开始？
2. 女性自主权与胎儿权利如何平衡？
3. 堕胎法应该如何制定？

### 【基因编辑】(Gene Editing)

**定义**：使用技术修改生物体基因组的做法。

**费曼解释**：
你可以修改DNA：
- 治疗遗传疾病（治疗性）
- 增强"设计婴儿"（增强性）
- 这是基因编辑

基因编辑让我们能"设计生命"。

**基因编辑的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **体细胞编辑** | 修改身体细胞 | 治疗癌症 |
| **生殖系编辑** | 修改生殖细胞 | 遗传给后代 |
| **治疗性** | 治疗疾病 | 治疗镰刀细胞病 |
| **增强性** | 增强能力 | 增加智力 |

**基因编辑的伦理问题**：

| 问题 | 描述 |
|------|------|
| **安全性** | 是否安全？脱靶效应？ |
| **公平性** | 只有富人能用？ |
| **身份认同** | 改变人的本质？ |
| **后代权利** | 后代没有同意 |
| **滑坡** | 从治疗到增强？ |

**治疗 vs 增强**：

| 区分 | 描述 | 问题 |
|------|------|------|
| **治疗** | 恢复正常功能 | 什么是"正常"？ |
| **增强** | 超越正常功能 | 增强是错的吗？ |
| **预防** | 预防疾病 | 与治疗的界限？ |

**生殖系编辑的特殊问题**：

1. **不可逆性**：改变会遗传给后代
2. **同意问题**：后代无法同意
3. **未知的长期影响**：我们不知道后果
4. **社会影响**：可能加剧不平等

**贺建奎事件** (2018)：

**事件**：
- 中国科学家贺建奎创造了基因编辑婴儿
- 修改CCR5基因以抵抗艾滋病
- 引发全球争议

**伦理问题**：
- 没有充分的安全保证
- 没有医疗必要性
- 违反科学伦理规范

**基因编辑的伦理框架**：

| 原则 | 描述 |
|------|------|
| **安全优先** | 确保安全 |
| **必要性** | 有医疗必要 |
| **知情同意** | 充分知情同意 |
| **公平获取** | 确保公平获取 |
| **透明** | 公开透明 |

**代表人物**：
- **Doudna**：CRISPR技术先驱
- **Savulescu**：增强伦理
- **当代**：生物伦理学者

**经典文献**：
- Doudna & Sternberg, *A Crack in Creation* (2017)
- Savulescu, "Procreative Beneficence" (2001)

**当代发展 (2020-2026)**：
- **CRISPR应用**：CRISPR的临床应用
- **国际共识**：基因编辑的国际规范
- **增强辩论**：人类增强的辩论
- **监管发展**：各国监管发展

**跨学科联系**：
- **生物学**：基因技术
- **医学**：临床应用

### 【算法公平】(Algorithmic Fairness)

**定义**：确保算法决策不歧视特定群体的原则。

**费曼解释**：
一个AI招聘系统：
- 它筛选简历
- 但可能对女性或少数族裔有偏见
- 算法公平就是确保它公平对待所有人

**算法偏见的来源**：

| 来源 | 描述 | 例子 |
|------|------|------|
| **数据偏见** | 训练数据有偏见 | 历史招聘数据偏向男性 |
| **设计偏见** | 设计者的偏见 | 选择有偏见的特征 |
| **使用偏见** | 使用方式有偏见 | 在不适当的情境使用 |

**算法公平的定义问题**：

| 定义 | 描述 | 问题 |
|------|------|------|
| **人口统计均等** | 不同群体被批准比例相同 | 可能牺牲准确性 |
| **均等机会** | 不同群体真正阳性被正确识别率相同 | 需要知道真实标签 |
| **预测均等** | 不同群体的阳性预测值相同 | 可能不兼容 |
| **个体公平** | 相似个体被相似对待 | 什么是"相似"？ |

**公平定义的不兼容性**：

**不可能定理**：
- 在某些条件下
- 不同公平定义不能同时满足
- 必须做出权衡

**算法公平的伦理原则**：

| 原则 | 描述 |
|------|------|
| **透明性** | 算法决策可解释 |
| **问责性** | 有人对结果负责 |
| **可审查性** | 可以审查和挑战 |
| **包容性** | 考虑所有群体 |

**算法偏见的案例**：

| 案例 | 问题 |
|------|------|
| **COMPAS** | 种族偏见（累犯预测） |
| **亚马逊招聘** | 性别偏见 |
| **人脸识别** | 种族偏见（深色皮肤） |
| **信贷评分** | 社会经济偏见 |

**代表人物**：
- **O'Neil**：《算法武器》
- **Eubanks**：《自动化的不平等》
- **当代**：AI伦理学者

**经典文献**：
- O'Neil, *Weapons of Math Destruction* (2016)
- Eubanks, *Automating Inequality* (2018)

**当代发展 (2020-2026)**：
- **公平AI**：公平AI算法发展
- **监管发展**：AI公平监管
- **工具发展**：公平检测工具
- **跨文化公平**：不同文化的公平概念

**跨学科联系**：
- **计算机科学**：算法设计
- **统计学**：公平度量
- **法律**：反歧视法
- **社会学**：社会公平

**中国视角**：
- **算法推荐**：算法推荐管理
- **公平观念**：中国公平观念
- **当代**：算法公平的中国视角

**争议问题**：

### 【自动驾驶伦理】(Autonomous Vehicle Ethics)

**定义**：自动驾驶汽车面临的道德决策问题。

**费曼解释**：
一辆自动驾驶汽车：
- 刹车失灵
- 前面是5个行人
- 转向会撞墙，杀死乘客
- 汽车应该怎么做？

这就是自动驾驶的"电车难题"。

**自动驾驶的伦理问题**：

| 问题 | 描述 |
|------|------|
| **电车难题** | 撞谁？乘客还是行人？ |
| **责任分配** | 谁负责？制造商？程序员？ |
| **决策透明** | 决策逻辑应该公开吗？ |
| **公众偏好** | 应该遵循公众偏好吗？ |

**道德机器实验** (Moral Machine)：

**结果**：
- 全球数百万人参与
- 不同文化有不同偏好
- 普遍偏好：救人多于动物，救年轻多于老年

**自动驾驶的伦理框架**：

| 框架 | 核心主张 | 问题 |
|------|----------|------|
| **功利主义** | 最大化总福利 | 如何量化？ |
| **义务论** | 遵循道德规则 | 什么规则？ |
| **公众偏好** | 遵循公众选择 | 公众可能不一致 |
| **法律合规** | 遵守法律 | 法律可能不明确 |

**责任问题**：

|------|------|

| **失业** | 司机失业 |
| **城市规划** | 城市需要重新设计 |
| **隐私** | 行踪数据隐私 |

**代表人物**：
- **Awad et al.**：道德机器实验
- **Lin**：机器人伦理
- **当代**：AI伦理学者

### 【数据隐私】(Data Privacy)

**定义**：个人对其数据如何被收集、使用和共享的控制权。

**费曼解释**：
你在网上的每个行为：
- 搜索记录、购物历史、位置数据
- 都被公司收集
- 数据隐私问：你应该有权控制吗？

**数据隐私的核心问题**：

| 问题 | 描述 |
|------|------|
| **收集** | 谁可以收集什么数据？ |
| **使用** | 数据如何被使用？ |
| **共享** | 数据可以共享给谁？ |
| **删除** | 可以要求删除吗？ |
| **知情** | 用户知道吗？ |

**隐私的伦理基础**：

| 基础 | 描述 |
|------|------|
| **自主权** | 隐私是自主权的一部分 |
| **尊严** | 隐私保护人格尊严 |
| **信任** | 隐私是信任的基础 |
| **民主** | 隐私是民主社会的条件 |

**隐私与效用的权衡**：

| 场景 | 隐私 | 效用 | 权衡 |

| **安全** | 监控 | 防恐 | 界限、监督 |
| **商业** | 个性化广告 | 免费服务 | 知情同意、选择权 |

**数据隐私的保护框架**：

| 框架 | 核心 | 代表 |

| **GDPR** | 欧盟数据保护 | 欧盟 |
| **知情同意** | 用户同意 | 各国 |
| **最小收集** | 只收集必要数据 | 隐私设计 |
| **去标识化** | 匿名化数据 | 数据处理 |

**隐私技术的伦理**：

| 技术 | 隐私保护 | 问题 |
|------|----------|------|
| **加密** | 保护数据 | 可能阻碍执法 |

### 【干细胞研究】(Stem Cell Research)

**定义**：使用干细胞进行医学研究的伦理问题。

**费曼解释**：
干细胞可以变成任何类型的细胞，有望治愈很多疾病。
但胚胎干细胞研究需要破坏胚胎——这是在杀死潜在的人吗？

**核心要素**：
1. **干细胞类型**：胚胎vs成体干细胞
2. **胚胎地位**：胚胎有道德地位吗？
3. **治疗潜力**：研究的医学价值
4. **替代方案**：是否有替代方法

**伦理争议**：

| 问题 | 支持 | 反对 |
|------|------|------|
| **胚胎地位** | 胚胎不是人 | 胚胎有道德地位 |
| **研究价值** | 重大医学价值 | 价值不确定 |
| **替代方案** | 成体干细胞不够 | 应该用成体干细胞 |

**主要立场**：

| 立场 | 描述 |
|------|------|
| **支持** | 研究的医学价值超过胚胎损失 |
| **反对** | 胚胎有完全的道德地位 |
| **中间** | 允许有限制的研究 |

**相关词条**：生命伦理、胚胎地位、医学研究

### 【器官交易】(Organ Trade)

**定义**：买卖人体器官的伦理问题。

**费曼解释**：
有人需要器官救命，有人愿意卖器官。
应该允许器官交易吗？
还是说人体部分不应该商品化？

**核心要素**：
1. **供需失衡**：器官需求远大于供应
2. **商业化问题**：身体能否商品化？
3. **剥削问题**：穷人是否被剥削？
4. **自愿性**：卖器官是否真正自愿？

**支持与反对**：

| 理由 | 支持交易 | 反对交易 |
|------|----------|----------|
| **效率** | 增加器官供应 | 黑市问题 |
| **自主权** | 应该允许选择 | 选择可能不自由 |
| **尊严** | 可以帮助他人 | 身体商品化贬低尊严 |
| **公平** | 自愿交易 | 穷人被剥削 |

**代表人物**：
- **Singer**：支持有规制的市场
- **Sandel**：反对身体商品化

**相关词条**：生命伦理、商品化、自主权

### 【人体实验】(Human Experimentation)

**定义**：在人身上进行科学实验的伦理问题。

**费曼解释**：
医学进步需要人体实验，但如何保护实验对象？
不道德的人体实验历史（如纳粹实验）警示我们。

**核心要素**：
1. **知情同意**：实验对象必须知情并同意
2. **风险评估**：评估实验风险
3. **利益平衡**：科学利益vs个人风险
4. **脆弱群体**：保护弱势群体

**伦理原则**：

| 原则 | 描述 |
|------|------|
| **尊重人** | 尊重自主权，知情同意 |
| **善行** | 最大化利益，最小化伤害 |
| **正义** | 公平分配利益和负担 |

**历史教训**：

| 事件 | 描述 |
|------|------|
| **纳粹实验** | 不道德的人体实验 |
| **塔斯基吉梅毒研究** | 隐瞒治疗的实验 |
| **纽伦堡法典** | 人体实验伦理准则 |

**相关词条**：医学伦理、知情同意、研究伦理

### 【知情同意】(Informed Consent)

**定义**：医疗或研究对象在充分了解信息后自愿同意。

**费曼解释**：
医生要给你做手术，必须先告诉你风险和选择。
你了解后自愿同意——这就是知情同意。

**核心要素**：
1. **信息揭示**：提供充分信息
2. **理解能力**：对象能理解信息
3. **自愿性**：同意是自愿的
4. **决策能力**：对象有决策能力

**知情同意的条件**：

| 条件 | 描述 |
|------|------|
| **披露** | 揭示相关信息 |
| **理解** | 患者理解信息 |
| **自愿** | 没有强迫 |
| **能力** | 有决策能力 |

**知情同意的问题**：

| 问题 | 描述 |
|------|------|
| **披露标准** | 应该披露多少信息？ |
| **理解差距** | 患者可能不理解 |
| **文化差异** | 不同文化对同意的理解不同 |

**相关词条**：医学伦理、自主权、患者权利

### 【医患关系】(Doctor-Patient Relationship)

**定义**：医生与患者之间的专业关系及其伦理维度。

**费曼解释**：
医生和患者是什么关系？
医生是权威？还是合作伙伴？
医患关系如何影响医疗伦理？

**核心要素**：
1. **信任**：患者信任医生
2. **保密**：医生保护患者隐私
3. **责任**：医生对患者有责任
4. **权力不对称**：医生有更多知识和权力

**医患关系模型**：

| 模型 | 描述 |
|------|------|
| **家长式** | 医生决定，患者听从 |
| **信息式** | 医生提供信息，患者决定 |
| **解释式** | 医生帮助患者理解价值 |
| **商议式** | 医生和患者共同决策 |

**医患关系的伦理**：

| 伦理要求 | 描述 |
|----------|------|
| **信任** | 建立和维护信任 |
| **沟通** | 有效沟通 |
| **尊重** | 尊重患者自主权 |
| **关怀** | 关怀患者的福祉 |

**相关词条**：医学伦理、知情同意、信任
