### 【元伦理学】(Metaethics)

**词源**：希腊语 *meta* (之后、超越)、*ethos* (习俗)

**定义**：研究道德语言、道德知识和道德事实本质的哲学分支。

**费曼解释**：
当你问"杀人是错的吗？"：
- 规范伦理学回答："是"或"否"
- 元伦理学回答："错误"是什么意思？"错误"是真的吗？我们怎么知道？

元伦理学研究"道德本身"，而不是"具体道德问题"。

**元伦理学的核心问题**：

| 问题 | 描述 |
|------|------|
| **语义问题** | 道德语言是什么意思？ |
| **形而上学问题** | 道德事实存在吗？ |
| **认识论问题** | 我们如何获得道德知识？ |
| **心理学问题** | 道德判断的心理本质是什么？ |
| **客观性问题** | 道德判断是客观的吗？ |

**元伦理学的主要立场**：

| 立场 | 核心主张 |
|------|----------|
| **道德实在论** | 道德事实客观存在 |
| **道德反实在论** | 道德事实不存在或非客观 |
| **表达主义** | 道德判断表达态度 |
| **错误论** | 所有道德判断都是假的 |
| **相对主义** | 道德真理相对于文化/个人 |

**元伦理学的发展**：

| 时期 | 主导观点 |
|------|----------|
| **古代-近代** | 道德实在论主导 |
| **20世纪初** | 情感主义兴起 |
| **20世纪中** | 规定主义发展 |
| **20世纪末至今** | 道德实在论复兴 |

**代表人物**：
- **摩尔**：《伦理学原理》——元伦理学先驱
- **艾耶尔**：情感主义
- **史蒂文森**：情感主义
- **布莱克本**：准实在论
- **当代**：元伦理学研究者

**经典文献**：
- Moore, *Principia Ethica* (1903)
- Ayer, *Language, Truth and Logic* (1936)
- Mackie, *Ethics: Inventing Right and Wrong* (1977)

**当代发展 (2020-2026)**：
- **实验元伦理学**：实证研究道德判断
- **AI元伦理学**：AI的道德客观性
- **认知科学**：道德认知的神经基础
- **跨文化研究**：道德的普遍性与差异

**跨学科联系**：
- **语言哲学**：道德语言
- **认识论**：道德知识
- **心理学**：道德心理学
- **人类学**：道德文化

**中国视角**：
- **道**：道与道德
- **理**：理与道德真理
- **心**：心与道德判断
- **当代**：元伦理学的中国视角

**争议问题**：
1. 道德事实存在吗？
2. 道德判断是认知的还是非认知的？
3. 道德是客观的还是主观的？

**相关词条**：道德实在论、表达主义、道德认识论

### 【道德实在论】(Moral Realism)

**定义**：认为道德事实客观存在，道德判断可以是真的或假的。

**费曼解释**：
- "雪是白的"是真的，因为雪真的是白的
- "杀人是错的"是真的，因为杀人真的是错的
- 道德实在论说：道德事实就像物理事实一样真实

**道德实在论的核心主张**：

| 要素 | 描述 |
|------|------|
| **道德事实** | 道德事实客观存在 |
| **道德真理** | 道德判断有真假 |
| **独立性** | 道德事实独立于我们的信念 |
| **认知性** | 道德判断表达信念 |

**道德实在论的类型**：

| 类型 | 核心主张 | 代表 |
|------|----------|------|
| **自然主义实在论** | 道德性质是自然性质 | Boyd, Brink |
| **非自然主义实在论** | 道德性质是非自然性质 | 摩尔 |
| **建构主义实在论** | 道德真理是建构的 | Korsgaard |

**自然主义道德实在论**：

**核心主张**：
- 道德性质就是自然性质
- 例如："好" = "促进幸福"
- 道德判断可以通过经验研究证实

**支持论证**：
- **最佳解释论证**：道德事实最好解释道德现象
- **因果论证**：道德事实有因果作用

**非自然主义道德实在论**：

**摩尔的观点**：
- "好"是简单的、不可分析的性质
- 好不同于任何自然性质
- 自然主义谬误：把好等同于自然性质

**开放问题论证**：
- "X是好的"和"X是令人愉快的"意思不同
- 因此"好"≠"令人愉快"
- 类似地，"好"≠任何自然性质

**对道德实在论的批评**：

| 批评 | 描述 | 回应 |
|------|------|------|
| **本体论问题** | 道德事实在哪里？ | 道德事实是抽象的 |
| **认识论问题** | 我们如何知道道德事实？ | 道德直觉、推理 |
| **差异问题** | 道德判断差异巨大 | 错误理论解释 |
| **动机问题** | 为什么道德判断能激发行动？ | 内在主义解释 |

**代表人物**：
- **摩尔**：非自然主义实在论
- **Boyd**：自然主义实在论
- **Brink**：道德实在论
- **当代**：道德实在论者

**经典文献**：
- Moore, *Principia Ethica* (1903)
- Boyd, "How to Be a Moral Realist" (1988)
- Brink, *Moral Realism and the Foundations of Ethics* (1989)

**当代发展 (2020-2026)**：
- **神经实在论**：道德的神经基础
- **进化实在论**：道德的进化解释
- **建构主义实在论**：建构的客观性
- **AI实在论**：AI道德的客观性

**跨学科联系**：
- **形而上学**：道德事实
- **认识论**：道德知识
- **科学哲学**：道德与科学
- **认知科学**：道德认知

**中国视角**：
- **道**：道与道德实在
- **理**：理与道德真理
- **天理**：天理与人理
- **当代**：道德实在论的中国视角

**争议问题**：
1. 道德事实如何存在？
2. 我们如何认识道德事实？
3. 自然主义谬误真的存在吗？

**相关词条**：道德反实在论、道德认识论、开放问题论证

### 【表达主义】(Expressivism)

**定义**：认为道德判断表达态度而非信念，没有真假可言。

**费曼解释**：
- "雪是白的"描述事实，有真假
- "杀人是错的"表达态度，没有真假
- 表达主义说：道德判断就像"嘘！"或"好！"，是态度表达

**表达主义的核心主张**：

| 要素 | 描述 |
|------|------|
| **非认知主义** | 道德判断不表达信念 |
| **态度表达** | 道德判断表达态度 |
| **无真假** | 道德判断没有真假 |
| **激发行动** | 道德判断与动机相连 |

**表达主义的发展**：

| 阶段 | 理论 | 代表 |
|------|------|------|
| **情感主义** | 道德判断表达情感 | 艾耶尔、史蒂文森 |
| **规定主义** | 道德判断是命令 | 黑尔 |
| **准实在论** | 表达主义可以"准实在论" | 布莱克本 |
| **建构表达主义** | 表达主义的新发展 | Gibbard |

**情感主义** (Emotivism)：

**艾耶尔的观点**：
- "偷窃是错的" = "偷窃！" + "不要偷窃！"
- 道德语言表达情感、发出命令
- 没有认知内容

**史蒂文森的观点**：
- 道德判断有两个要素：
  1. 信念要素（描述性的）
  2. 态度要素（情感性的）
- 道德分歧是态度分歧

**规定主义** (Prescriptivism)：

**黑尔的观点**：
- 道德判断是可普遍化的命令
- "你应该做X" = "让每个人都做X"
- 道德推理是逻辑推理

**准实在论** (Quasi-Realism)：

**布莱克本的观点**：
- 表达主义可以"赢得"实在论的话语权
- 我们可以说"道德真理"但不是实在论意义
- 道德语言有"投射"的性质

**对表达主义的批评**：

| 批评 | 描述 | 回应 |
|------|------|------|
| **弗雷格-吉奇问题** | 嵌入问题 | 复杂表达主义 |
| **道德分歧** | 真的只是态度分歧？ | 信念与态度交织 |
| **道德推理** | 道德推理如何可能？ | 表达主义逻辑 |
| **道德真理** | 道德真理概念如何解释？ | 准实在论 |

**弗雷格-吉奇问题**：

**问题**：
- "偷窃是错的"（表达态度）
- "如果偷窃是错的，那么让弟弟偷窃也是错的"
- 在条件句中，"偷窃是错的"不表达态度
- 表达主义如何解释？

**回应**：
- 发展"复杂表达主义"
- 解释嵌入的道德语言

**代表人物**：
- **艾耶尔**：情感主义
- **史蒂文森**：情感主义
- **黑尔**：规定主义
- **布莱克本**：准实在论
- **Gibbard**：当代表达主义

**经典文献**：
- Ayer, *Language, Truth and Logic* (1936)
- Stevenson, *Ethics and Language* (1944)
- Hare, *The Language of Morals* (1952)
- Blackburn, *Spreading the Word* (1984)

**当代发展 (2020-2026)**：
- **认知表达主义**：信念与态度混合
- **实验表达主义**：实证研究
- **AI表达主义**：AI道德判断
- **社会表达主义**：社会道德态度

**跨学科联系**：
- **语言哲学**：道德语言
- **心灵哲学**：态度与信念
- **心理学**：道德情感
- **认知科学**：道德认知

**中国视角**：
- **情**：情与道德
- **意**：意与道德判断
- **心**：心与道德态度
- **当代**：表达主义的中国视角

**争议问题**：
1. 弗雷格-吉奇问题能解决吗？
2. 道德推理如何可能？
3. 表达主义能解释道德真理吗？

**相关词条**：情感主义、规定主义、准实在论

### 【道德相对主义】(Moral Relativism)

**定义**：认为道德真理相对于文化、社会或个人的观点。

**费曼解释**：
- 古希腊人认为奴隶制是正确的
- 现代人认为奴隶制是错误的
- 道德相对主义问：谁对？
- 相对主义者说：都对，只是不同文化

**道德相对主义的核心主张**：

| 要素 | 描述 |
|------|------|
| **相对性** | 道德真理相对于框架 |
| **无绝对真理** | 没有普遍道德真理 |
| **差异承认** | 承认道德差异 |
| **宽容** | 应该宽容不同道德 |

**相对主义的类型**：

| 类型 | 相对什么 | 核心主张 |
|------|----------|----------|
| **文化相对主义** | 文化 | 道德真理相对于文化 |
| **社会相对主义** | 社会 | 道德真理相对于社会 |
| **个人相对主义** | 个人 | 道德真理相对于个人 |

**支持相对主义的论证**：

**差异论证**：
1. 不同文化有不同道德
2. 这种差异没有客观标准来解决
3. 因此，道德是相对的

**宽容论证**：
1. 我们应该宽容其他文化
2. 道德绝对主义导致不宽容
3. 因此，我们应该接受相对主义

**对相对主义的批评**：

| 批评 | 描述 | 回应 |
|------|------|------|
| **自我反驳** | "相对主义是真的"本身是绝对主张？ | 这是元层面主张 |
| **宽容悖论** | "应该宽容"本身是绝对主张？ | 这是文化规范 |
| **道德进步** | 如何说社会"进步"？ | 相对的进步 |
| **道德批评** | 如何批评其他文化？ | 从自己文化出发 |

**道德多样性的事实**：

| 现象 | 描述 |
|------|------|
| **表面差异** | 道德实践不同 |
| **深层一致** | 基本价值相似 |
| **不同条件** | 不同环境导致不同规范 |
| **共同核心** | 可能有普遍道德核心 |

**代表人物**：
- **人类学家**：文化相对主义
- **Wong**：温和相对主义
- **当代**：相对主义者

**经典文献**：
- Benedict, "Anthropology and the Abnormal" (1934)
- Wong, *Moral Relativity* (1984)

**当代发展 (2020-2026)**：
- **全球化相对主义**：全球伦理与相对主义
- **多元文化主义**：文化多元与道德
- **AI相对主义**：AI道德的相对性
- **温和相对主义**：新相对主义理论

**跨学科联系**：
- **人类学**：文化差异
- **社会学**：社会规范
- **政治学**：多元文化
- **国际关系**：文化冲突

**中国视角**：
- **和而不同**：和而不同
- **求同存异**：求同存异
- **各美其美**：各美其美
- **当代**：相对主义的中国视角

**争议问题**：
1. 道德相对主义是自我反驳的吗？
2. 如何解释道德进步？
3. 相对主义与宽容的关系？

**相关词条**：文化相对主义、道德普遍主义、道德差异

### 【道德错误论】(Moral Error Theory)

**定义**：认为所有道德判断都是假的，因为不存在道德事实。

**费曼解释**：
- "独角兽存在"是假的，因为独角兽不存在
- "杀人是错的"也是假的，因为"错误"性质不存在
- 错误论说：所有道德判断都错了

**错误论的核心主张**：

| 要素 | 描述 |
|------|------|
| **道德语言** | 道德语言声称描述客观事实 |
| **道德事实不存在** | 没有客观道德事实 |
| **所有道德判断都是假的** | 因为没有道德事实 |
| **我们应该放弃道德话语** | 或修改道德语言 |

**Mackie的错误论**：

**两个论证**：

**1. 相对性论证**：
- 道德判断差异巨大
- 最好的解释是没有客观道德事实

**2.奇怪性论证**：
- 客观道德价值会非常奇怪
- 它们必须是"内在地"激发行动的
- 这不符合我们对世界的理解

**道德的"奇怪"特征**：

| 特征 | 描述 |
|------|------|
| **内在激发性** | 道德判断自动激发行动 |
| **客观规范性** | 道德要求独立于欲望 |
| **不可错性** | 道德直觉声称不可错 |

**对错误论的批评**：

| 批评 | 描述 | 回应 |
|------|------|------|
| **道德实践** | 我们不能放弃道德 | 可以使用"虚构主义" |
| **反直觉** | 与直觉冲突 | 直觉可能是错的 |
| **道德进步** | 如何解释道德进步？ | 实践进步，非真理进步 |
| **科学类比** | 科学也曾被认为奇怪 | 道德更奇怪 |

**对错误论的回应**：

**道德虚构主义**：
- 我们继续使用道德语言
- 但把它当作有用的虚构
- 就像谈论"太阳升起"

**道德革命主义**：
- 我们应该放弃道德话语
- 发展新的规范性语言
- 避免错误论的问题

**代表人物**：
- **Mackie**：错误论创始人
- **Joyce**：当代错误论
- **当代**：错误论者

**经典文献**：
- Mackie, *Ethics: Inventing Right and Wrong* (1977)
- Joyce, *The Myth of Morality* (2001)

**当代发展 (2020-2026)**：
- **虚构主义**：道德虚构主义
- **革命主义**：道德革命主义
- **AI错误论**：AI道德的错误
- **进化解释**：道德的进化解释

**跨学科联系**：
- **语言哲学**：道德语言
- **心理学**：道德信念
- **进化论**：道德进化
- **认知科学**：道德认知

**中国视角**：
- **无善恶**：道家无善恶论
- **名实**：名与实
- **当代**：错误论的中国视角

**争议问题**：
1. 道德实践真的能放弃吗？
2. 奇怪性论证有效吗？
3. 虚构主义是否可行？

**相关词条**：道德反实在论、道德虚构主义、奇怪性论证

### 【道德认识论】(Moral Epistemology)

**定义**：研究道德知识的来源、本质和辩护的分支。

**费曼解释**：
- 你怎么知道"2+2=4"？（数学直觉）
- 你怎么知道"雪是白的"？（感知）
- 你怎么知道"杀人是错的"？（道德认识论问）

**道德认识论的核心问题**：

| 问题 | 描述 |
|------|------|
| **道德知识可能吗？** | 我们能有道德知识吗？ |
| **道德知识的来源** | 道德知识从哪里来？ |
| **道德辩护** | 道德信念如何被辩护？ |
| **道德直觉** | 道德直觉可靠吗？ |

**道德知识的主要理论**：

| 理论 | 核心主张 | 代表 |
|------|----------|------|
| **道德直觉主义** | 道德知识来自直觉 | 摩尔、罗斯 |
| **道德感知理论** | 道德知识来自感知 | McDowell |
| **道德理性主义** | 道德知识来自理性 | 康德 |
| **道德经验主义** | 道德知识来自经验 | 休谟 |

**道德直觉主义**：

**核心主张**：
- 我们通过直觉认识道德真理
- 道德直觉是直接的、不依赖推理的
- 类似于数学直觉

**道德直觉的特征**：

| 特征 | 描述 |
|------|------|
| **直接性** | 不需要推理 |
| **自明性** | 看起来明显正确 |
| **可错性** | 可能是错的 |
| **普遍性** | 广泛共享 |

**对直觉主义的批评**：

| 批评 | 描述 | 回应 |
|------|------|------|
| **直觉差异** | 人们直觉不同 | 区分表面与深层直觉 |
| **直觉不可靠** | 直觉经常错误 | 发展可靠直觉 |
| **循环论证** | 用直觉支持直觉 | 基础主义辩护 |
| **文化偏见** | 直觉受文化影响 | 寻找普遍直觉 |

**道德感知理论**：

**核心主张**：
- 我们可以"感知"道德性质
- 类似于感知颜色
- 道德感知需要道德敏感性

**道德感知的条件**：

| **经验** | 需要道德经验 |
| **概念能力** | 需要道德概念 |
| **可靠过程** | 感知过程要可靠 |

### 【自然主义谬误】(Naturalistic Fallacy)

**词源**：摩尔在《伦理学原理》(1903)中提出

**定义**：试图用自然性质来定义"好"的错误。

**费曼解释**：
- 有人说："好 = 令人愉快"
- 摩尔说：这是谬误
- 因为"X是好的"和"X是令人愉快的"是两个不同的问题
- 你总可以问："但令人愉快就是好的吗？"

**摩尔的核心主张**：

| 主张 | 描述 |
|------|------|
| **"好"不可分析** | "好"是简单的、不可分析的性质 |
| **自然主义谬误** | 把"好"等同于自然性质是错误的 |
| **开放问题** | "X是好的"总是开放问题 |

**开放问题论证**：

**步骤**：
1. 假设"好" = "令人愉快"
2. 问："令人愉快的东西是好吗？"
3. 这个问题仍然是"开放的"（有意义）
4. 如果"好" = "令人愉快"，问题应该是琐碎的
5. 因此，"好" ≠ "令人愉快"

**开放问题的特征**：

| 特征 | 描述 |
|------|------|
| **非琐碎** | 问题不是显然为真或假 |
| **有意义** | 问题有实际意义 |
| **开放性** | 任何定义都会留下开放问题 |

**对自然主义谬误的批评**：

| 批评 | 描述 | 回应 |
|------|------|------|
| **语言混淆** | 混淆了意义与指称 | 摩尔的区分是对的 |
| **分析失败** | 不是所有概念都能分析 | 摩尔自己也分析"好" |
| **循环** | 摩尔假设了自己要证明的 | 开放问题不只是直觉 |
| **无意义** | 可能"好"就是自然性质 | 开放问题另有原因 |

**自然主义的回应**：

**回应1：意义与指称**：
- "水" = "H2O"（指称相同）
- 但"水是H2O吗？"是开放问题
- 所以开放问题不能证明概念不同

**回应2：无谓的"好"**：
- 也许"好"是自然性质
- 开放问题来自我们的无知

**代表人物**：
- **摩尔**：自然主义谬误提出者
- **当代**：伦理学研究者

**经典文献**：
- Moore, *Principia Ethica* (1903)

**当代发展 (2020-2026)**：
- **新摩尔主义**：摩尔观点的复兴
- **自然主义复兴**：自然主义的辩护
- **分析哲学**：概念分析的新发展
- **实验哲学**：开放问题的实证研究

**跨学科联系**：
- **语言哲学**：意义与指称
- **逻辑学**：定义与分析
- **心理学**：概念认知
- **认知科学**：概念结构

**中国视角**：
- **名实**：名与实
- **善**：善的定义
- **当代**：自然主义谬误的中国视角

**争议问题**：
1. 自然主义谬误真的存在吗？
2. "好"能被定义吗？
3. 开放问题论证有效吗？

**相关词条**：摩尔、开放问题论证、道德非自然主义

### 【道德动机】(Moral Motivation)

**定义**：道德判断与行动动机之间的关系。

**费曼解释**：
- 你判断"应该捐款"
- 你会捐款吗？
- 道德判断和行动动机是什么关系？

**内在主义 vs 外在主义**：

| 问题 | 内在主义 | 外在主义 |
|------|----------|----------|
| 道德判断能激发行动吗？ | 能，内在地 | 需要外在欲望 |
| 道德判断与动机的关系 | 必然 | 偶然 |
| 不道德者的道德判断 | 不可能 | 可能但不起作用 |

**内在主义** (Internalism)：

**核心主张**：
- 道德判断内在地激发行动
- 如果某人真心判断"应该做X"，就有动力做X
- 不道德者不是真正理解道德判断

**内在主义的类型**：

| 类型 | 核心主张 |
|------|----------|
| **存在内在主义** | 道德判断必然带来动机 |
| **强内在主义** | 真诚判断就有充分动机 |
| **弱内在主义** | 真诚判断有某种动机 |

**外在主义** (Externalism)：

**核心主张**：
- 道德判断需要外在欲望才能激发行动
- 不道德者可以理解道德判断但没动力
- 道德判断与动机是偶然关系

**支持外在主义的现象**：

| 现象 | 描述 |
|------|------|
| **意志软弱** | 知道应该做但不做 |
| **抑郁** | 知道应该做但没动力 |
| **不道德** | 理解道德但不被激发 |

**对内在主义的辩护**：

**回应1：条件限制**：
- 内在主义要求理性正常、没有抑郁等
- 在正常条件下，道德判断激发行动

**回应2：理解问题**：
- 不道德者没有真正理解道德判断
- 完全理解道德的人会被激发

**代表人物**：
- **内在主义者**：Hare, Blackburn
- **外在主义者**：Brink, Foot
- **当代**：道德动机研究者

**经典文献**：
- Hare, *The Language of Morals* (1952)
- Foot, "Morality as a System of Hypothetical Imperatives" (1972)

**当代发展 (2020-2026)**：
- **神经道德动机**：道德动机的神经基础
- **AI道德动机**：AI的道德动机
- **实验研究**：道德判断与行动的实证研究
- **社会道德动机**：社会情境与道德动机

**跨学科联系**：
- **心理学**：动机心理学
- **神经科学**：动机的神经基础
- **经济学**：行为经济学
- **人工智能**：AI动机

**中国视角**：
- **知行**：知行合一
- **意**：意与动机
- **志**：志与动机
- **当代**：道德动机的中国视角

**争议问题**：
1. 道德判断内在地激发行动吗？
2. 如何解释意志软弱？
3. 不道德者能理解道德吗？

**相关词条**：内在主义、外在主义、意志软弱

## G-Z

### 【善的概念】(The Concept of Good)

**定义**：伦理学中最基本、最核心的概念。

**费曼解释**：
"好"是什么意思？
摩尔说："好"是简单的、不可分析的概念。
就像"黄色"不能被定义，"好"也不能。

**核心要素**：
1. **简单性**：善是简单概念（摩尔）
2. **不可分析性**：不能用更基本概念分析
3. **规范性**：善提供行动理由
4. **开放问题**："X是善的"总是可追问的

**善的开放问题论证**：
```
"快乐是好的吗？"——这个问题有意义
"快乐是快乐吗？"——这个问题无意义
因此，"善"≠"快乐"
```

**善的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **道德善** | 道德上的好 | 正义、诚实 |
| **非道德善** | 非道德的好 | 快乐、知识 |
| **工具善** | 作为手段的好 | 钱财 |
| **内在善** | 本身就是好的 | 幸福 |

**代表人物**：
- **Moore**：善的不可分析性
- **Aristotle**：善=幸福

**相关词条**：摩尔、善、开放问题论证

### 【道德真理】(Moral Truth)

**定义**：道德命题是否可以为真，以及什么使它们为真。

**费曼解释**：
"杀人是错的"是真的吗？
实在论者说：是的，这是一个客观真理。
反实在论者说：这不是真或假的问题。

**核心要素**：
1. **真理地位**：道德命题可以为真吗？
2. **真理制造者**：什么使道德命题为真？
3. **真理标准**：如何知道道德真理？
4. **真理类型**：道德真理是什么类型的真理？

**道德真理的理论**：

| 理论 | 描述 | 代表 |
|------|------|------|
| **道德实在论** | 存在客观道德真理 | 不同版本 |
| **道德反实在论** | 没有客观道德真理 | Mackie |
| **相对主义** | 真理相对于文化/个人 | 不同版本 |
| **表达主义** | 道德语句不表达真理 | Blackburn |

**道德真理的问题**：

| 问题 | 描述 |
|------|------|
| **存在性** | 道德真理存在吗？ |
| **可知性** | 我们能知道道德真理吗？ |
| **多样性** | 道德真理是唯一的吗？ |

**代表人物**：
- **Moore**：道德真理存在
- **Mackie**：没有客观道德真理

**相关词条**：道德实在论、真理、元伦理学

### 【道德属性】(Moral Properties)

**定义**：道德判断所归赋的属性，如"善"、"恶"、"对"、"错"。

**费曼解释**：
"这个行为是错的"——"错"是这个行为的属性吗？
像"红色"是物体的属性？
道德属性存在吗？它们是什么？

**核心要素**：
1. **属性地位**：道德属性是真正的属性吗？
2. **本体论**：道德属性是什么类型的存在？
3. **因果性**：道德属性有因果力吗？
4. **可还原性**：道德属性能还原为自然属性吗？

**道德属性的理论**：

| 理论 | 描述 | 代表 |
|------|------|------|
| **道德实在论** | 道德属性存在 | 不同版本 |
| **错误论** | 道德属性不存在 | Mackie |
| **投射论** | 道德属性是投射 | 不同版本 |

**道德属性的问题**：

| 问题 | 描述 |
|------|------|
| **排队问题** | 道德属性在哪里？ |
| **认识问题** | 如何认识道德属性？ |
| **怪异性** | 道德属性很"奇怪"？ |

**代表人物**：
- **Mackie**：道德属性的怪异性论证
- **Boyd**：道德属性的自然主义

**相关词条**：道德实在论、属性、元伦理学
