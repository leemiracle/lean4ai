### 【规范伦理学】(Normative Ethics)

**词源**：希腊语 *nomos* (规范)、*ethos* (习俗)

**定义**：研究道德行为规范的学科，探讨什么行为是对的、什么是错的。

**费曼解释**：
你在路上捡到一个钱包：
- 功利主义：归还它能让更多人幸福
- 义务论：归还它是你的道德义务
- 德性伦理：诚实的人会归还它

三种理论都指向同一行为，但理由不同。

**三大主要理论**：

| 理论 | 核心问题 | 判断标准 |
|------|----------|----------|
| **功利主义** | 什么行为产生最多幸福？ | 后果 |
| **义务论** | 什么行为是道德义务？ | 动机/规则 |
| **德性伦理** | 什么是好品格？ | 品格 |

**规范伦理学的核心问题**：

1. **道德判断的标准**：什么使行为正确或错误？
2. **道德冲突**：当道德规则冲突时如何选择？
3. **道德动机**：为什么要做道德的事？

**代表人物**：
- **边沁、密尔**：功利主义
- **康德**：义务论
- **亚里士多德**：德性伦理
- **当代**：规范伦理学研究者

**经典文献**：
- Bentham, *An Introduction to the Principles of Morals and Legislation* (1789)
- Kant, *Groundwork of the Metaphysics of Morals* (1785)
- Aristotle, *Nicomachean Ethics*

**当代发展 (2020-2026)**：
- **道德人工智能**：AI伦理决策
- **混合理论**：结合多种理论
- **实验伦理学**：实证研究道德判断
- **全球化伦理**：跨文化道德规范

**跨学科联系**：
- **心理学**：道德心理学
- **经济学**：行为经济学
- **法律**：法哲学
- **人工智能**：机器伦理

**中国视角**：
- **仁**：仁者爱人
- **义**：义利之辨
- **礼**：礼与规范
- **当代**：规范伦理学的中国视角

**争议问题**：
1. 哪种理论是正确的？
2. 后果与动机哪个更重要？
3. 道德规范是普遍的还是相对的？

**相关词条**：功利主义、义务论、德性伦理

### 【功利主义】(Utilitarianism)

**词源**：拉丁语 *utilitas* (有用、利益)

**定义**：认为正确的行为是最大化整体幸福或效用的伦理理论。

**费曼解释**：
你在分配10个苹果给5个人：
- 每个人都想要苹果
- 功利主义问：怎样分配能让整体满足感最大？
- 可能的答案：每人2个？还是给最喜欢苹果的人更多？

功利主义就是计算"最大幸福"。

**功利主义的核心主张**：

| 要素 | 描述 |
|------|------|
| **后果主义** | 行为的道德性取决于后果 |
| **福利主义** | 福利/幸福是唯一内在善 |
| **最大化** | 应该最大化整体福利 |
| **中立性** | 每个人的福利同等重要 |

**功利主义的类型**：

| 类型 | 核心主张 | 代表 |
|------|----------|------|
| **行动功利主义** | 每个行为都要最大化效用 | 边沁 |
| **规则功利主义** | 遵循最大化效用的规则 | 密尔 |
| **偏好功利主义** | 满足偏好而非幸福 | Singer |
| **行为功利主义** | 实际行为结果决定 | 当代 |

**边沁的快乐计算** (Hedonic Calculus)：

| 要素 | 描述 |
|------|------|
| **强度** | 快乐的强度 |
| **持续时间** | 快乐持续多久 |
| **确定性** | 快乐的确定性 |
| **邻近性** | 快乐有多近 |
| **繁殖性** | 产生更多快乐的可能性 |
| **纯粹性** | 不伴随痛苦 |
| **范围** | 影响多少人 |

**密尔的快乐质量**：

**高级快乐 vs 低级快乐**：
- "做一个不满足的苏格拉底，好过做一个满足的傻瓜"
- 不同类型的快乐有质的区别
- 知性快乐高于感官快乐

**对功利主义的批评**：

| 批评 | 描述 | 回应 |
|------|------|------|
| **要求过高** | 要求太多牺牲 | 区分道德与超义务 |
| **忽视权利** | 可能侵犯少数人权利 | 规则功利主义 |
| **难以计算** | 无法精确计算效用 | 使用近似值 |
| **忽视正义** | 可能不公正 | 加入正义约束 |

**电车难题**：

```
情景：一辆失控电车即将撞死5人
选项：拉杆让电车转向，撞死1人
功利主义：拉杆（5 > 1）
义务论：不能主动杀人
```

**代表人物**：
- **边沁**：功利主义创始人
- **密尔**：快乐质量论
- **Singer**：当代偏好功利主义
- **当代**：功利主义者

**经典文献**：
- Bentham, *An Introduction to the Principles of Morals and Legislation* (1789)
- Mill, *Utilitarianism* (1863)
- Singer, *Practical Ethics* (1979)

**当代发展 (2020-2026)**：
- **有效利他主义**：最大化行善效果
- **AI功利主义**：AI决策的效用计算
- **全球功利主义**：全球贫困、气候变化
- **动物福利**：动物的幸福

**跨学科联系**：
- **经济学**：效用理论
- **公共政策**：成本收益分析
- **心理学**：幸福研究
- **人工智能**：效用函数

**中国视角**：
- **利**：义利之辨
- **百姓**：民本思想
- **兼爱**：墨家功利思想
- **当代**：功利主义的中国视角

**争议问题**：
1. 幸福能被量化吗？
2. 少数人的权利如何保护？
3. 高级快乐和低级快乐如何比较？

**相关词条**：后果主义、效用、快乐主义

### 【义务论】(Deontology)

**词源**：希腊语 *deon* (义务、责任)

**定义**：认为某些行为本质上是对或错的伦理理论，独立于其后果。

**费曼解释**：
你在医院探病：
- 你看到一个健康的人
- 医生说：我们可以杀了他，用他的器官救5个病人
- 功利主义说：可以（5 > 1）
- 义务论说：不行！杀人是错的，无论后果如何

义务论关注"行为本身的对错"，不是"后果的好坏"。

**康德义务论的核心**：

| 概念 | 描述 |
|------|------|
| **善良意志** | 唯一无条件善的东西 |
| **义务** | 出于对道德法则的尊重而行动 |
| **定言命令** | 无条件的道德命令 |
| **人格尊严** | 人是目的，不是手段 |

**定言命令** (Categorical Imperative)：

**第一公式：普遍化原则**
- "只按照你能同时意愿它成为普遍法则的准则行动"
- 例子：能承诺还钱但不打算还？不能普遍化！

**第二公式：目的公式**
- "永远把人当作目的，而不是手段"
- 例子：欺骗是把他人当作手段

**第三公式：自律公式**
- "道德法则来自理性的自我立法"
- 我们是自己给自己立 法

**义务的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **完美义务** | 必须总是遵守 | 不杀人 |
| **不完美义务** | 有时可以不遵守 | 行善 |
| **对己义务** | 对自己的义务 | 不自杀 |
| **对人义务** | 对他人的义务 | 不欺骗 |

**康德的四个例子**：

1. **自杀**：出于自爱而自杀不能普遍化
2. **虚假承诺**：借钱不还不能普遍化
3. **浪费才能**：不发展才能不能普遍化
4. **不帮助他人**：不帮助不能普遍化

**对义务论的批评**：

| 批评 | 描述 | 回应 |
|------|------|------|
| **过于严格** | 不允许任何例外 | 区分准则与行为 |
| **义务冲突** | 义务冲突时怎么办？ | 优先级排序 |
| **缺乏情感** | 忽视道德情感 | 情感是辅助的 |
| **普遍化问题** | 普遍化可能被操纵 | 更精确的准则 |

**罗斯的初步义务** (Prima Facie Duties)：

| 义务类型 | 描述 |
|----------|------|
| **信守承诺** | 保持承诺 |
| **补偿** | 弥补过去的错误 |
| **感恩** | 回报恩惠 |
| **正义** | 公平分配 |
| **行善** | 帮助他人 |
| **自我完善** | 发展自己 |
| **不伤害** | 不造成伤害 |

**代表人物**：
- **康德**：义务论创始人
- **罗斯**：初步义务
- **当代**：义务论者

**经典文献**：
- Kant, *Groundwork of the Metaphysics of Morals* (1785)
- Ross, *The Right and the Good* (1930)

**当代发展 (2020-2026)**：
- **契约论**：基于协议的义务
- **权利论**：基于权利的义务
- **康德主义AI**：AI的道德规则
- **医学伦理**：医学义务

**跨学科联系**：
- **法律**：法律义务
- **政治哲学**：公民义务
- **医学**：医学伦理
- **商业**：商业伦理

**中国视角**：
- **义**：义与利
- **忠**：忠与诚
- **信**：信与诺
- **当代**：义务论的中国视角

**争议问题**：
1. 定言命令是否有效？
2. 义务冲突时如何选择？
3. 后果真的不重要吗？

**相关词条**：康德伦理学、定言命令、道德规则

### 【德性伦理】(Virtue Ethics)

**词源**：拉丁语 *virtus* (德性、优秀)

**定义**：以品格和德性为核心的伦理理论，关注"成为什么样的人"。

**费曼解释**：
你看到一个小孩落水：
- 功利主义：救他（最大化幸福）
- 义务论：救他（这是你的义务）
- 德性伦理：救他（勇敢的人会这样做）

德性伦理关注"品格"而不是"规则"或"后果"。

**德性伦理的核心主张**：

| 要素 | 描述 |
|------|------|
| **品格优先** | 品格比行为更重要 |
| **德性** | 品格的优良特质 |
| **实践智慧** | 正确判断的智慧 |
| **幸福** | 德性生活带来幸福 |

**亚里士多德的德性理论**：

**德性的结构**：
- 德性是两个极端之间的中道
- 勇敢 = 怯懦与鲁莽之间
- 慷慨 = 吝啬与挥霍之间

**主要德性**：

| 德性 | 过度 | 不及 | 中道 |
|------|------|------|------|
| **勇敢** | 鲁莽 | 怯懦 | 勇敢 |
| **节制** | 放纵 | 麻木 | 节制 |
| **慷慨** | 挥霍 | 吝啬 | 慷慨 |
| **诚实** | 夸张 | 虚伪 | 诚实 |
| **机智** | 滑稽 | 粗鄙 | 机智 |

**理智德性与道德德性**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **理智德性** | 理性的优秀 | 智慧、理解力 |
| **道德德性** | 品格的优秀 | 勇敢、正义 |

**实践智慧** (Phronesis)：

- 知道在特定情境中如何行动
- 不是抽象的知识
- 需要经验和判断

**幸福** (Eudaimonia)：

- 德性生活的结果
- 不仅是主观快乐
- 是"人的繁荣"

**现代德性伦理**：

| 学者 | 核心主张 |
|------|----------|
| **Anscombe** | 回到亚里士多德 |
| **Foot** | 自然主义德性 |
| **MacIntyre** | 德性的社会背景 |
| **Nussbaum** | 能力与德性 |

**对德性伦理的批评**：

| 批评 | 描述 | 回应 |
|------|------|------|
| **行动指导不足** | 没有明确规则 | 实践智慧指导行动 |
| **文化相对主义** | 德性因文化而异 | 普遍德性 |
| **循环论证** | 德性是做对的事的倾向 | 德性可以独立定义 |
| **冲突问题** | 德性冲突时怎么办？ | 实践智慧判断 |

**代表人物**：
- **亚里士多德**：德性伦理创始人
- **Anscombe**：现代德性伦理复兴
- **MacIntyre**：德性传统
- **当代**：德性伦理学者

**经典文献**：
- Aristotle, *Nicomachean Ethics*
- Anscombe, "Modern Moral Philosophy" (1958)
- MacIntyre, *After Virtue* (1981)

**当代发展 (2020-2026)**：
- **德性认识论**：认知德性
- **职业德性**：专业伦理
- **AI德性**：机器德性
- **教育德性**：品格教育

**跨学科联系**：
- **心理学**：积极心理学
- **教育学**：品格教育
- **政治学**：公民德性
- **组织研究**：组织德性

**中国视角**：
- **仁**：仁者爱人
- **义**：义与利
- **礼**：礼与德
- **智**：智与德
- **信**：信与德
- **当代**：德性伦理的中国视角

**争议问题**：
1. 德性能给出行动指导吗？
2. 德性是普遍的还是文化的？
3. 德性与规则如何协调？

### 【后果主义】(Consequentialism)

**词源**：拉丁语 *consequentia* (后果、结果)

**定义**：认为行为的道德性完全由其后果决定的伦理理论。

**费曼解释**：
你在判断一个行为是否道德：
- 后果主义问：这个行为产生了什么结果？
- 如果结果是好的，行为就是对的
- 如果结果不好，行为就是错的

只看结果，不看动机。

**后果主义的核心结构**：

```
行为 → 后果 → 评价
（行为本身不重要，后果决定一切）
```

**后果主义 vs 义务论**：

| 问题 | 后果主义 | 义务论 |
|------|----------|--------|
| 评价标准 | 后果 | 行为本身 |
| 说谎 | 如果后果好，可以 | 总是错的 |
| 杀一人救五人 | 可以 | 不可以 |
| 动机 | 不重要 | 重要 |

**后果主义的类型**：

| 类型 | 善的定义 | 代表 |
|------|----------|------|
| **功利主义** | 幸福/效用 | 边沁、密尔 |
| **享乐主义** | 快乐 | 古代享乐主义 |
| **偏好满足** | 偏好被满足 | Singer |
| **客观列表** | 客观价值列表 | Parfit |
| **规则后果主义** | 规则的后果 | Brandt |

**行为后果主义 vs 规则后果主义**：

| 类型 | 核心主张 | 问题 |
|------|----------|------|
| **行为后果主义** | 每个行为都要最大化善 | 要求过高 |
| **规则后果主义** | 遵循能最大化善的规则 | 与义务论相似？ |

**对后果主义的批评**：

| 批评 | 描述 | 回应 |
|------|------|------|
| **要求过高** | 要求太多牺牲 | 区分道德与超义务 |
| **正义问题** | 可能不公正 | 加入正义约束 |
| **计算问题** | 无法预测后果 | 使用最佳估计 |
| **行为正当化** | 可能正当化不道德行为 | 规则后果主义 |

**电车难题**：

```
情景1：拉杆让电车转向（杀1救5）
- 后果主义：拉杆（5 > 1）

情景2：把胖子推下桥（杀1救5）
- 后果主义：推？（结果相同）
- 但直觉不同
```

**代表人物**：
- **边沁**：功利主义先驱
- **密尔**：功利主义
- **Singer**：当代后果主义
- **当代**：后果主义者

**经典文献**：
- Bentham, *An Introduction to the Principles of Morals and Legislation* (1789)
- Mill, *Utilitarianism* (1863)
- Singer, *Practical Ethics* (1979)

**当代发展 (2020-2026)**：
- **AI后果主义**：AI的后果计算
- **全球后果主义**：全球问题
- **证据主义后果主义**：基于证据的判断
- **混合理论**：结合后果主义和其他理论

**跨学科联系**：
- **经济学**：成本收益分析
- **公共政策**：政策评估
- **医学**：医疗决策
- **环境**：环境影响评估

**中国视角**：
- **后果**：后果与动机
- **利**：利与义
- **当代**：后果主义的中国视角

**争议问题**：
1. 后果能被准确预测吗？
2. 如何定义"善"？
3. 后果主义会正当化不公正吗？

**相关词条**：功利主义、效用、道德计算

### 【道德规则】(Moral Rules)

**定义**：指导道德行为的一般原则或规范。

**费曼解释**：
道德规则就像交通规则：
- "不要闯红灯" → "不要说谎"
- "让行人先行" → "帮助有需要的人"
- 规则告诉我们什么能做、什么不能做

**道德规则的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **绝对规则** | 没有例外 | 不折磨无辜者 |
| **初步规则** | 可以被推翻 | 不说谎 |
| **理想规则** | 最佳规则 | 功利主义规则 |
| **实际规则** | 社会实际遵循 | 文化规范 |

**规则功利主义**：

**核心主张**：
- 遵循那些"如果被普遍遵循会产生最佳后果"的规则
- 不是每个行为都要计算后果

**例子**：
- "不说谎"是规则功利主义的规则
- 如果每个人都诚实，社会更好
- 所以我们应该遵循"不说谎"

**规则与行为**：

| 问题 | 规则功利主义 | 行为功利主义 |
|------|--------------|--------------|
| 说谎 | 通常不应该 | 如果后果好，可以 |
| 承诺 | 应该遵守 | 如果后果好，可以违背 |
| 规则 | 重要 | 不重要 |

**对道德规则的批评**：

| 批评 | 描述 |
|------|------|
| **规则崇拜** | 盲目遵循规则 |
| **例外问题** | 规则总是有例外 |
| **冲突问题** | 规则之间可能冲突 |
| **情境问题** | 规则可能不适合特定情境 |

**代表人物**：
- **密尔**：规则功利主义
- **Brandt**：规则功利主义
- **当代**：规则伦理学者

**经典文献**：
- Mill, *Utilitarianism* (1863)
- Brandt, *Morality and the Moral Sentiments* (1967)

**当代发展 (2020-2026)**：
- **AI规则**：AI的道德规则
- **职业伦理**：职业规则
- **全球规则**：全球道德规则
- **情境伦理**：情境与规则

**跨学科联系**：
- **法律**：法律规则
- **社会学**：社会规范
- **心理学**：道德发展
- **人工智能**：规则系统

**中国视角**：
- **礼**：礼与规则
- **法**：法与规则
- **当代**：道德规则的中国视角

**争议问题**：

2. 规则冲突时如何选择？
3. 规则与情境如何平衡？

### 【道德困境】(Moral Dilemma)

**定义**：无论选择什么都会违反某种道德义务的情况。

**费曼解释**：
你在战争中：
- 你可以救5个战友，但必须牺牲1个
- 无论你做什么，都有人死
- 这就是道德困境

**道德困境的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **义务冲突** | 两个义务冲突 | 救A还是救B？ |
| **规则冲突** | 两条规则冲突 | 说真话还是保护朋友？ |
| **价值冲突** | 两个价值冲突 | 自由 vs 安全？ |

**经典案例**：

**电车难题**：
```
电车失控，即将撞死5人
你可以拉杆让电车转向，但会撞死1人
你拉不拉？
```

**萨特的学生的困境**：
```
二战期间，学生想加入抵抗运动
但他的母亲需要他照顾
加入抵抗 vs 照顾母亲
```

**电车难题变体**：

| 变体 | 情景 | 大多数人反应 |
|------|------|--------------|
| **开关** | 拉杆让电车转向 | 拉杆（80%） |
| **胖子** | 推胖子下桥挡住电车 | 不推（80%） |
| **环形轨道** | 胖子在环上 | 推？ |
| **移植** | 杀1人取器官救5人 | 不杀（90%） |

**对道德困境的回应**：

| 理论 | 回应 |
|------|------|
| **义务论** | 有优先级排序 |
| **功利主义** | 选择后果最好的 |
| **德性伦理** | 德性的人会怎么做？ |
| **存在主义** | 自由选择并承担责任 |

**道德困境的哲学意义**：

1. **道德理论的检验**：困境测试道德理论
2. **道德直觉的冲突**：揭示我们直觉的复杂性
3. **道德规则的局限**：规则可能不足以解决困境

**代表人物**：
- **萨特**：存在主义困境
- **Williams**：道德运气
- **当代**：道德困境研究者

**经典文献**：
- Sartre, "Existentialism is a Humanism"
- Foot, "The Problem of Abortion and the Doctrine of the Double Effect" (1967)
- Thomson, "Killing, Letting Die, and the Trolley Problem" (1976)

**当代发展 (2020-2026)**：
- **AI困境**：自动驾驶困境
- **医学困境**：医疗分配困境
- **全球困境**：气候困境
- **实验研究**：困境的实证研究

**跨学科联系**：
- **心理学**：道德判断
- **认知科学**：道德认知
- **法律**：法律困境
- **人工智能**：AI决策

**中国视角**：
- **义利**：义利困境
- **忠孝**：忠孝困境
- **当代**：道德困境的中国视角

**争议问题**：

2. 困境是否有正确答案？

### 【超义务行为】(Supererogation)

**词源**：拉丁语 *supererogare* (做得更多)

**定义**：超出道德要求的行为，做了值得赞扬，不做也不受谴责。

**费曼解释**：
- 你必须帮助一个溺水的人 → 这是义务
- 你跳入冰冷的水中救他，差点丧命 → 这是超义务

超义务是"超出义务"的行为，是道德英雄主义。

**超义务行为的特征**：

| 特征 | 描述 |
|------|------|
| **超出义务** | 不是必须做的 |
| **值得赞扬** | 做了是好的 |
| **不做不受谴责** | 不做也不算错 |
| **风险或代价** | 通常有风险或代价 |

**超义务行为的例子**：

| 例子 | 描述 |
|------|------|
| **牺牲救人** | 冒生命危险救陌生人 |
| **捐赠大部分财产** | 捐出大部分财富 |
| **极限利他** | 把肾脏捐给陌生人 |
| **英雄行为** | 战场上的英勇行为 |

**功利主义的问题**：

- 功利主义可能说：你应该最大化效用
- 这意味着超义务行为变成义务
- 这是"要求过高"问题

**超义务与道德要求**：

```
        道德谱系
    ──────────────────
    禁止 ← 义务 → 超义务
    （错误）（必须）（超出）
```

**对超义务概念的批评**：

| 批评 | 描述 | 回应 |
|------|------|------|
| **功利主义批评** | 没有超义务，只有义务 | 区分理想与现实 |
| **德性批评** | 德性不存在超义务 | 德性是连续谱 |
| **相对主义批评** | 超义务因文化而异 | 有普遍共识 |

**代表人物**：
- **Urmson**：超义务概念
- **Milo**：超义务研究
- **当代**：超义务研究者

**经典文献**：
- Urmson, "Saints and Heroes" (1958)
- Milo, *Immorality* (1984)

**当代发展 (2020-2026)**：
- **有效利他主义**：超义务还是义务？
- **职业超义务**：职业中的超义务
- **AI超义务**：AI的超义务行为
- **全球超义务**：全球贫困问题

**跨学科联系**：
- **心理学**：利他主义
- **社会学**：英雄主义
- **政治学**：公民超义务
- **组织研究**：组织公民行为

**中国视角**：
- **仁**：仁与超义务
- **义**：义与义务
- **君子**：君子与超义务
- **当代**：超义务的中国视角

**争议问题**：
1. 超义务行为存在吗？
2. 功利主义要求过高吗？
3. 超义务与义务的界限在哪里？

**相关词条**：义务、利他主义、道德英雄

## G-Z

### 【善】(Good)

**定义**：伦理学的核心概念，指值得追求、有价值的东西。

**费曼解释**：
"善"是伦理学最基本的概念。
什么善？善是什么？这是伦理学的核心问题。

**核心要素**：
1. **价值性**：善是有价值的
2. **可追求性**：善值得追求
3. **规范性**：善提供行动理由
4. **多元性**：可能不止一种善

**善的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **内在善** | 本身就有价值 | 快乐、友谊 |
| **工具善** | 作为手段有价值 | 钱、工具 |
| **完善善** | 内在和工具价值都有 | 知识、健康 |

**善的理论**：

| 理论 | 描述 | 代表 |
|------|------|------|
| **享乐主义** | 善=快乐 | Bentham |
| **偏好论** | 善=偏好满足 | 不同作者 |
| **客观清单** | 善=客观价值列表 | 不同作者 |

**代表人物**：
- **Moore**：善不可分析
- **Aristotle**：善=幸福

**相关词条**：价值、功利主义、幸福

---

### 【幸福】(Happiness/Eudaimonia)

**定义**：人类生活的完满状态，德性伦理学的核心概念。

**费曼解释**：
幸福不只是感觉快乐。
亚里士多德说：幸福是实现你的潜能，活出人的本质。

**核心要素**：
1. **完满性**：生活的完满状态
2. **活动性**：幸福是活动，不是状态
3. **德性**：德性活动构成幸福
4. **外在善**：也需要外在条件

**Happiness vs Eudaimonia**：

| 概念 | 描述 | 内容 |
|------|------|------|
| **Happiness** | 主观感觉 | 快乐、满足 |
| **Eudaimonia** | 客观完满 | 德性活动+外在善 |

**幸福主义**：
- 幸福是终极善
- 其他东西因其对幸福的贡献而有价值
- 伦理学应该研究如何实现幸福

**代表人物**：
- **Aristotle**：幸福作为终极善
- **当代幸福主义**：不同版本

**相关词条**：德性伦理、善、亚里士多德

### 【道德冲突】(Moral Conflict)

**定义**：两个或多个道德要求之间发生冲突的情况。

**费曼解释**：
你说"不要说谎"，也说"不要伤害他人"。
但如果说实话会伤害他人，怎么办？
这就是道德冲突——两个道德要求不能同时满足。

**核心要素**：
1. **多个道德要求**：有两个或多个道德要求
2. **不可同时满足**：不能同时满足所有要求
3. **困境**：选择困难
4. **道德残余**：无论选哪个都有遗憾

**道德冲突的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **价值冲突** | 不同价值的冲突 | 自由vs平等 |
| **义务冲突** | 不同义务的冲突 | 说真话vs不伤害 |
| **角色冲突** | 不同角色的冲突 | 作为医生vs作为父母 |

**对道德冲突的立场**：

| 立场 | 描述 |
|------|------|
| **可解决论** | 所有冲突都能解决 |
| **不可解决论** | 有些冲突真正不可解决 |
| **悲剧论** | 冲突是道德生活的悲剧性特征 |

**代表人物**：
- **Ross**：初步义务的冲突
- **Williams**：道德残余

**相关词条**：道德困境、义务论、价值

### 【道德直觉】(Moral Intuition)

**定义**：对道德真理的直接、非推理性把握。

**费曼解释**：
"杀人是错的"——你需要推理才知道吗？
不需要，你直接"感觉"到这是错的。
这就是道德直觉。

**核心要素**：
1. **直接性**：不需要推理
2. **自明性**：直觉上明显为真
3. **可靠性**：直觉是否可靠？
4. **文化差异**：直觉可能因文化而异

**直觉主义**：
- 道德真理通过直觉被认识
- 有些道德原则是自明的
- 直觉是道德认识的基础

**对直觉的挑战**：

| 挑战 | 描述 |
|------|------|
| **文化差异** | 不同文化有不同直觉 |
| **偏见** | 直觉可能受偏见影响 |
| **不可靠** | 直觉可能不可靠 |

**代表人物**：
- **Ross**：道德直觉主义
- **Audi**：当代直觉主义

**相关词条**：直觉主义、道德认识论、元伦理学

---

### 【道德推理】(Moral Reasoning)

**定义**：通过推理过程得出道德结论的思维方式。

**费曼解释**：
道德直觉是直接"感觉"，道德推理是"思考"。
"我不应该说谎，因为..."
道德推理给出道德判断的理由。

**核心要素**：
1. **推理过程**：从理由到结论
2. **原则应用**：应用道德原则
3. **一致性**：追求判断一致
4. **可辩护性**：推理使判断可辩护

**道德推理的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **演绎推理** | 从原则推出结论 | 所有杀人是错的→这是杀人→这是错的 |
| **类比推理** | 通过类比判断 | 案例A与案例B相似 |
| **反思平衡** | 调整原则与直觉 | 使原则与直觉一致 |

**推理 vs 直觉**：

| 问题 | 直觉 | 推理 |
|------|------|------|
| **速度快** | 是 | 否 |
| **可辩护** | 否 | 是 |
| **可靠** | 争议 | 争议 |

**代表人物**：
- **Kant**：道德推理的重要性
- **Rawls**：反思平衡

**相关词条**：道德直觉、道德判断、反思平衡
