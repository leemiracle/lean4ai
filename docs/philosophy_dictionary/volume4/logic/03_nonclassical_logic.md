## A-F

### 【非经典逻辑】(Non-Classical Logic)

**定义**：不同于或扩展经典逻辑的逻辑系统。

**费曼解释**：
经典逻辑假设：
- 每个命题或真或假
- 排中律成立
- 矛盾导致爆炸

非经典逻辑质疑这些假设。

**非经典逻辑的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **扩展逻辑** | 添加经典逻辑 | 模态逻辑 |
| **偏离逻辑** | 改变经典逻辑 | 多值逻辑 |
| **不一致逻辑** | 处理矛盾 | 次协调逻辑 |

**为什么要非经典逻辑？**

| 理由 | 描述 |
|------|------|
| **悖论** | 处理经典逻辑的悖论 |
| **模糊性** | 处理模糊概念 |
| **矛盾** | 容忍矛盾而不爆炸 |
| **直觉主义** | 构造性要求 |

**主要非经典逻辑**：

| 逻辑 | 核心改变 |
|------|----------|
| **直觉主义逻辑** | 拒绝排中律 |
| **多值逻辑** | 多于两个真值 |
| **模糊逻辑** | 连续真值 |
| **次协调逻辑** | 容忍矛盾 |
| **相关逻辑** | 要求相关性 |
| **自由逻辑** | 允许空词项 |

**代表人物**：
- **布劳威尔**：直觉主义
- **乌卡谢维奇**：多值逻辑
- **达科斯塔**：次协调逻辑
- **当代**：非经典逻辑学家

**相关词条**：直觉主义逻辑、多值逻辑、模糊逻辑

### 【直觉主义】(Intuitionism)

**定义**：布劳威尔的数学哲学：数学是心智的构造活动。

**费曼解释**：
直觉主义问："数学真理是什么？"
- 经典观点：数学真理独立于我们
- 直觉主义：数学是我们心智的构造
- 只有我们能构造的，才是存在的

**直觉主义的核心原则**：

| 原则 | 描述 |
|------|------|
| **构造性** | 证明 = 构造 |
| **心智依赖** | 数学对象是心智构造 |
| **反排中律** | 不接受 p∨¬p |

**直觉主义 vs 经典数学**：

| 问题 | 经典 | 直觉主义 |
|------|------|----------|
| **排中律** | 接受 | 不接受 |
| **存在证明** | 非构造性可以 | 必须构造 |
| **无穷** | 实无穷 | 潜无穷 |
| **双否定** | ¬¬p→p | 不成立 |

**直觉主义逻辑**：

| 原则 | 经典 | 直觉主义 |
|------|------|----------|
| **排中律** p∨¬p | 有效 | 无效 |
| **双否定消去** ¬¬p→p | 有效 | 无效 |
| **皮尔士律** | 有效 | 无效 |

**BHK解释**：

| 联结词 | 解释 |
|--------|------|
| **p∧q** | p的证明和q的证明 |
| **p∨q** | p的证明或q的证明（知道哪个） |
| **p→q** | 把p的证明转换为q的证明的方法 |
| **¬p** | 从p的证明推出矛盾的方法 |
| **∃x P(x)** | 对象a和P(a)的证明 |
| **∀x P(x)** | 对任意x构造P(x)的证明的方法 |

**代表人物**：
- **布劳威尔**：直觉主义创始人
- **海廷**：直觉主义逻辑
- **海廷**：直觉主义数学

**经典文献**：
- Brouwer, "Intuitionism and Formalism" (1912)
- Heyting, *Intuitionism* (1956)

**相关词条**：直觉主义逻辑、构造主义、排中律

### 【直觉主义逻辑】(Intuitionistic Logic)

**定义**：不接受排中律的逻辑系统。

**费曼解释**：
直觉主义逻辑问："为什么一定要p或非p？"
- 我不能证明p
- 我也不能证明¬p
- 凭什么说"p或非p"一定成立？

**与经典逻辑的区别**：

| 原则 | 经典 | 直觉主义 |
|------|------|----------|
| **排中律** p∨¬p | ✓ | ✗ |
| **双否定消去** ¬¬p→p | ✓ | ✗ |
| **德摩根** ¬(p∧q)→¬p∨¬q | ✓ | ✗ |
| **蕴涵** ¬p∨q | ✓ | ✗ |

**直觉主义逻辑的公理**：

| 公理 | 形式 |
|------|------|
| **蕴涵引入** | p→(q→p) |
| **蕴涵分配** | (p→(q→r))→((p→q)→(p→r)) |
| **合取消去** | (p∧q)→p, (p∧q)→q |
| **合取引入** | p→(q→(p∧q)) |
| **析取引入** | p→(p∨q), q→(p∨q) |

**克里普克语义**：

| 条件 | 描述 |
|------|------|
| **可能世界** | W集合 |
| **偏序** | ≤关系 |
| **单调性** | 如果w≤w'且w⊩p，则w'⊩p |

**直觉主义逻辑的应用**：

| 领域 | 应用 |
|------|------|
| **计算机科学** | 类型论、程序验证 |
| **数学** | 构造性数学 |
| **拓扑学** | 拓扑语义 |

**代表人物**：
- **布劳威尔**：直觉主义创始人
- **海廷**：形式化
- **克里普克**：语义

**相关词条**：直觉主义、构造主义、排中律

### 【排中律】(Law of Excluded Middle)

**定义**：经典逻辑原则：p∨¬p 对任何命题p都成立。

**费曼解释**：
排中律说："要么p，要么非p，没有第三种可能"
- "明天会下雨"：要么下雨，要么不下雨？
- 直觉主义：我不知道！可能两个都不能证明

**排中律的形式**：

```
p ∨ ¬p
```

**排中律的地位**：

| 逻辑 | 接受？ |
|------|--------|
| **经典逻辑** | 接受 |
| **直觉主义逻辑** | 不接受 |
| **多值逻辑** | 可能不接受 |

**为什么拒绝排中律？**

| 理由 | 描述 |
|------|------|
| **构造性** | 不能构造p或¬p的证明 |
| **数学实例** | 某些数学命题既不能证明也不能否证 |
| **开放问题** | 哥德巴赫猜想尚未解决 |

**排中律的等价形式**：

| 形式 | 描述 |
|------|------|
| p∨¬p | 直接形式 |
| ¬¬p→p | 双否定消去 |
| (p→q)→(¬p∨q) | 蕴涵定义 |

**对排中律的辩护**：

| 辩护 | 描述 |
|------|------|
| **真值间隙论** | 命题有真值间隙 |
| **二值性** | 每个命题有确定真值 |

**代表人物**：
- **亚里士多德**：排中律先驱
- **布劳威尔**：批评者

**相关词条**：直觉主义、矛盾律、经典逻辑

### 【矛盾律】(Law of Non-Contradiction)

**定义**：经典逻辑原则：¬(p∧¬p)，没有命题同时为真和假。

**费曼解释**：
矛盾律说："不能同时是p又不是p"
- 这是亚里士多德的第一原则
- 但次协调逻辑质疑它

**矛盾律的形式**：

```
¬(p ∧ ¬p)
```

**矛盾律的地位**：

| 逻辑 | 接受？ |
|------|--------|
| **经典逻辑** | 接受 |
| **直觉主义逻辑** | 接受 |
| **次协调逻辑** | 部分接受或不接受 |

**对矛盾律的挑战**：

| 挑战 | 描述 |
|------|------|
| **悖论** | 某些命题似乎既真又假 |
| **辩证法** | 矛盾是发展的动力 |
| **不完整信息** | 矛盾信息可能存在 |

**亚里士多德的观点**：

| 要点 | 描述 |
|------|------|
| **最确定的** | 矛盾律是最确定的原则 |
| **不可证明** | 不能用更基本的原则证明 |
| **一切证明的基础** | 所有证明都依赖它 |

**代表人物**：
- **亚里士多德**：矛盾律
- **赫格尔**：辩证矛盾
- **次协调逻辑学家**：质疑者

**相关词条**：排中律、次协调逻辑、悖论

### 【多值逻辑】(Many-Valued Logic)

**定义**：允许命题有超过两个真值的逻辑系统。

**费曼解释**：
经典逻辑：真、假
多值逻辑：真、假、还有其他

"明天会下雨"可能既不真也不假——不确定。

**多值逻辑的类型**：

| 类型 | 真值数 | 真值 |
|------|--------|------|
| **三值逻辑** | 3 | 真、假、不确定 |
| **有限多值** | n | n个真值 |
| **无限多值** | 无限 | 如[0,1]区间 |

**三值逻辑**：

| 真值 | 符号 | 含义 |
|------|------|------|
| **T** | 1 | 真 |
| **I** | 1/2 | 不确定/未知/无定义 |
| **F** | 0 | 假 |

**主要的三值逻辑**：

| 系统 | 创始人 | 第三真值含义 |
|------|--------|--------------|
| **乌卡谢维奇** | Łukasiewicz | 可能 |
| **克莱尼** | Kleene | 未定义 |
| **博奇瓦尔** | Bochvar | 无意义 |
| **普里斯特** | Priest | 既真又假 |

**乌卡谢维奇三值逻辑**：

| 联结词 | 定义 |
|--------|------|
| **¬p** | 1-p |
| **p∧q** | min(p,q) |
| **p∨q** | max(p,q) |
| **p→q** | min(1,1-p+q) |

**多值逻辑的应用**：

| 领域 | 应用 |
|------|------|
| **计算机科学** | 程序语义 |
| **人工智能** | 不确定推理 |
| **语言学** | 预设 |
| **哲学** | 悖论 |

**代表人物**：
- **乌卡谢维奇**：多值逻辑创始人
- **克莱尼**：三值逻辑
- **当代**：多值逻辑学家

**经典文献**：
- Łukasiewicz, "On Three-Valued Logic" (1920)

**相关词条**：模糊逻辑、直觉主义逻辑、真值间隙

### 【乌卡谢维奇逻辑】(Łukasiewicz Logic)

**定义**：乌卡谢维奇提出的多值逻辑系统。

**费曼解释**：
乌卡谢维奇逻辑：
- 真值可以是0到1之间的任何数
- ¬p = 1-p
- p→q = min(1, 1-p+q)

为"可能"提供逻辑基础。

**乌卡谢维奇逻辑的特点**：

| 特点 | 描述 |
|------|------|
| **真值** | [0,1]区间 |
| **否定** | ¬p = 1-p |
| **蕴涵** | p→q = min(1, 1-p+q) |

**三值乌卡谢维奇逻辑**：

| p | q | ¬p | p→q |
|---|---|-----|------|
| T | T | F | T |
| T | I | F | I |
| T | F | F | F |
| I | T | I | T |
| I | I | I | T |
| I | F | I | I |
| F | T | T | T |
| F | I | T | T |
| F | F | T | T |

**与经典逻辑的关系**：

| 原则 | 乌卡谢维奇 | 经典 |
|------|------------|------|
| **排中律** p∨¬p | 不是重言式 | 重言式 |
| **矛盾律** ¬(p∧¬p) | 不是重言式 | 重言式 |

**代表人物**：
- **乌卡谢维奇**：创始人

**相关词条**：多值逻辑、模糊逻辑

### 【模糊逻辑】(Fuzzy Logic)

**定义**：处理模糊概念和部分真值的逻辑系统。

**费曼解释**：
模糊逻辑问："'高'是什么？"
- 170cm高吗？不一定
- 180cm高吗？有点
- 190cm高吗？很高
- 真值是程度，不是非黑即白

**模糊逻辑的特点**：

| 特点 | 描述 |
|------|------|
| **连续真值** | [0,1]区间 |
| **隶属度** | 对象对集合的隶属程度 |
| **模糊谓词** | 如"高"、"年轻"、"热" |

**模糊集合**：

| 概念 | 描述 |
|------|------|
| **经典集合** | 元素要么属于要么不属于 |
| **模糊集合** | 元素以某种程度属于 |

**隶属函数**：

```
μ_高(身高)
  1 |         _____
    |        /
    |       /
  0 |______/______
        150 170 190
```

**模糊逻辑的运算**：

| 运算 | 定义 |
|------|------|
| **否定** | ¬p = 1-p |
| **合取** | p∧q = min(p,q) |
| **析取** | p∨q = max(p,q) |
| **蕴涵** | 多种定义 |

**模糊逻辑的应用**：

| 领域 | 应用 |
|------|------|
| **控制系统** | 洗衣机、空调 |
| **人工智能** | 模糊推理 |
| **决策** | 模糊决策 |
| **语言学** | 模糊语言 |

**对模糊逻辑的批评**：

| 批评 | 描述 |
|------|------|
| **精确性问题** | 隶属度本身是精确的 |
| **边界问题** | 隶属度如何确定？ |
| **可用性** | 是否比概率更好？ |

**代表人物**：
- **扎德**：模糊逻辑创始人
- **当代**：模糊逻辑学家

**经典文献**：
- Zadeh, "Fuzzy Sets" (1965)

**相关词条**：多值逻辑、模糊性、精确化悖论
