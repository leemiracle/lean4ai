## A-F

### 【模态】(Modality)

**词源**：拉丁语 *modalis* (方式的)

**定义**：命题的"方式"或"样态"，如必然、可能、偶然等。

**费曼解释**：
模态是关于"如何为真"的：
- "2+2=4"是**必然**真的
- "下雨了"是**偶然**真的（可能不下雨）
- "1=2"是**不可能**真的

模态逻辑研究这些"方式"。

**模态的类型**：

| 类型 | 符号 | 含义 | 例子 |
|------|------|------|------|
| **必然** | □ | 必须为真 | □(2+2=4) |
| **可能** | ◇ | 可以为真 | ◇(下雨) |
| **偶然** | - | 可能真也可能假 | 偶然下雨 |
| **不可能** | - | 不可能为真 | 不可能1=2 |

**模态词之间的关系**：

| 关系 | 公式 |
|------|------|
| **必然 = 不可能不** | □p ↔ ¬◇¬p |
| **可能 = 不必然不** | ◇p ↔ ¬□¬p |
| **不可能 = 必然不** | ¬◇p ↔ □¬p |

**模态的多样性**：

| 模态类型 | 描述 | 应用 |
|----------|------|------|
| **真势模态** | 必然/可能 | 形而上学 |
| **认识模态** | 知道/相信 | 认识论 |
| **道义模态** | 必须/允许 | 伦理学 |
| **时间模态** | 总是/有时 | 时态逻辑 |

**代表人物**：
- **亚里士多德**：模态逻辑先驱
- **刘易斯**：现代模态逻辑
- **克里普克**：可能世界语义

**经典文献**：
- Lewis, *A Survey of Symbolic Logic* (1918)
- Kripke, "Semantical Considerations on Modal Logic" (1963)

**相关词条**：必然性、可能性、可能世界

### 【必然性】(Necessity)

**定义**：在所有情况下都为真的性质。

**费曼解释**：
什么是"必然"？
- 数学真理：2+2=4 是必然的
- 逻辑真理：p或非p 是必然的
- 物理定律：是必然的吗？（争议）

**必然性的类型**：

| 类型 | 定义 | 例子 |
|------|------|------|
| **逻辑必然** | 逻辑上不可能假 | p∨¬p |
| **数学必然** | 数学上不可能假 | 2+2=4 |
| **形而上必然** | 本质上不可能假 | 水是H₂O |
| **物理必然** | 物理定律 | 能量守恒？ |

**必然性与可能世界**：

| 类型 | 描述 |
|------|------|
| **逻辑必然** | 在所有可能世界中真 |
| **物理必然** | 在所有具有我们物理定律的可能世界中真 |

**必然性的符号**：

| 符号 | 读法 |
|------|------|
| □p | 必然p |
| □ | 必然算子 |
| ◇p | 可能p |

**必然性的问题**：

| 问题 | 描述 |
|------|------|
| **休谟问题** | 因果是必然的吗？ |
| **先验偶然** | 有先验但偶然的真理吗？ |
| **后验必然** | 有后验但必然的真理吗？ |

**克里普克的后验必然**：

- "晨星 = 暮星"是必然的但后验可知
- "水 = H₂O"是必然的但后验可知

**代表人物**：
- **亚里士多德**：必然性分析
- **克里普克**：后验必然
- **当代**：形而上学家

**经典文献**：
- Kripke, *Naming and Necessity* (1972)

**相关词条**：可能性、偶然性、后验必然

### 【可能性】(Possibility)

**定义**：在某些情况下可以为真的性质。

**费曼解释**：
什么是"可能"？
- "明天可能下雨"：有些可能世界下雨
- "猪可能会飞"：在有些可能世界猪会飞？
- 可能性的范围是什么？

**可能性的类型**：

| 类型 | 定义 | 例子 |
|------|------|------|
| **逻辑可能** | 不矛盾 | 圆的方？ |
| **形而上可能** | 本质上可能 | 猪会飞？ |
| **物理可能** | 符合物理定律 | 超光速？ |
| **技术可能** | 技术上可行 | 火星殖民？ |

**可能性与必然性的关系**：

| 关系 | 公式 |
|------|------|
| **可能 = 不必然不** | ◇p ↔ ¬□¬p |
| **必然 = 不可能不** | □p ↔ ¬◇¬p |

**可能性的等级**：

| 等级 | 描述 |
|------|------|
| **逻辑可能** | 最宽：不矛盾即可 |
| **形而上可能** | 较宽：符合事物本质 |
| **物理可能** | 较窄：符合物理定律 |
| **实际可能** | 最窄：考虑实际限制 |

**可能性的哲学问题**：

| 问题 | 描述 |
|------|------|
| **什么是可能世界？** | 可能世界的本体论地位 |
| **可能性从何而来？** | 可能性的来源 |
| **有多少可能性？** | 可能性的范围 |

**代表人物**：
- **莱布尼茨**：可能世界
- **刘易斯**：模态实在论
- **当代**：形而上学家

**相关词条**：必然性、可能世界、模态实在论

---

### 【偶然性】(Contingency)

**定义**：既非必然也非不可能的性质。

**费曼解释**：
什么是"偶然"？
- "下雨了"：可能下也可能不下
- "我在这里"：可能在这里也可能不在
- 偶然 = 可能真也可能假

**偶然性的定义**：

| 公式 | 含义 |
|------|------|
| 偶然(p) | ◇p ∧ ◇¬p |
| | 可能p且可能非p |

**偶然性的例子**：

| 命题 | 类型 |
|------|------|
| "2+2=4" | 必然 |
| "1=2" | 不可能 |
| "正在下雨" | 偶然 |
| "地球有月球" | 偶然？ |

**偶然性与科学**：

| 问题 | 描述 |
|------|------|
| **物理定律是偶然的吗？** | 物理定律可能在其他可能世界不同？ |
| **宇宙是偶然的吗？** | 宇宙可能不存在？ |
| **进化是偶然的吗？** | 进化路径可能不同？ |

**代表人物**：
- **亚里士多德**：偶然性分析
- **当代**：形而上学家

**相关词条**：必然性、可能性

### 【模态算子】(Modal Operator)

**定义**：表示模态的逻辑符号。

**费曼解释**：
模态算子是"模态逻辑的工具"：
- □ (方框)：必然
- ◇ (菱形)：可能

它们可以加到任何命题前面。

**基本模态算子**：

| 算子 | 符号 | 读法 | 等价 |
|------|------|------|------|
| **必然** | □ | 必然 | ¬◇¬ |
| **可能** | ◇ | 可能 | ¬□¬ |

**模态算子的组合**：

| 组合 | 含义 |
|------|------|
| □□p | 必然必然p |
| ◇◇p | 可能可能p |
| □◇p | 必然可能p |
| ◇□p | 可能必然p |

**不同模态的算子**：

| 模态类型 | 必然算子 | 可能算子 |
|----------|----------|----------|
| **真势** | □ 必然 | ◇ 可能 |
| **认识** | K 知道 | B 相信 |
| **道义** | O 必须 | P 允许 |
| **时间** | G 总是 | F 有时 |

**模态算子的公理**：

| 公理 | 形式 | 描述 |
|------|------|------|
| **K** | □(p→q)→(□p→□q) | 分配律 |
| **T** | □p→p | 必然蕴涵实然 |
| **4** | □p→□□p | 正内省 |
| **B** | p→□◇p | 对称性 |
| **5** | ◇p→□◇p | 欧几里得性 |

**代表人物**：
- **刘易斯**：模态算子
- **克里普克**：模态算子语义
- **当代**：逻辑学家

**相关词条**：模态、模态系统、可及关系

### 【可能世界】(Possible World)

**定义**：事物可能存在的完整方式。

**费曼解释**：
可能世界是"事物可能的样子"：
- 现实世界：我们所在的世界
- 其他可能世界：事物可能的其他方式
- "在某个可能世界，猪会飞"

**可能世界的直觉**：

| 表达 | 含义 |
|------|------|
| "可能下雨" | 在某个可能世界，下雨了 |
| "必然2+2=4" | 在所有可能世界，2+2=4 |
| "不可能1=2" | 在没有可能世界，1=2 |

**可能世界的本体论地位**：

| 立场 | 描述 | 代表 |
|------|------|------|
| **模态实在论** | 可能世界真实存在 | D. Lewis |
| **抽象论** | 可能世界是抽象对象 | Plantinga |
| **组合论** | 可能世界是组合 | C. I. Lewis |
| **虚构论** | 可能世界是虚构 | Rosen |

**刘易斯的模态实在论**：

| 主张 | 描述 |
|------|------|
| **存在性** | 所有可能世界都存在 |
| **具体性** | 可能世界是具体的 |
| **隔离性** | 世界之间没有时空联系 |
| **平等性** | 现实世界不特殊 |

**对模态实在论的批评**：

| 批评 | 描述 |
|------|------|
| **本体论膨胀** | 太多存在物 |
| **不可知** | 如何知道其他世界？ |
| **不可接受** | 违反直觉 |

**替代方案**：

| 方案 | 描述 |
|------|------|
| **抽象论** | 可能世界是最大一致的性质集 |
| **组合论** | 可能世界是现有元素的组合 |
| **还原论** | 可能世界还原为其他概念 |

**代表人物**：
- **莱布尼茨**：可能世界先驱
- **大卫·刘易斯**：模态实在论
- **普兰丁格**：抽象论

**经典文献**：
- Lewis, *On the Plurality of Worlds* (1986)
- Plantinga, *The Nature of Necessity* (1974)

**相关词条**：模态实在论、必然性、可能性

### 【模态实在论】(Modal Realism)

**定义**：大卫·刘易斯的观点：所有可能世界都像现实世界一样真实存在。

**费曼解释**：
刘易斯说："其他可能世界真的存在"
- 不只是"思维工具"
- 不只是"抽象概念"
- 真的、具体的世界

只是我们恰好在这个世界。

**刘易斯模态实在论的要点**：

| 要点 | 描述 |
|------|------|
| **存在** | 所有可能世界都存在 |
| **具体** | 可能世界是具体的，不是抽象的 |
| **隔离** | 世界之间没有时空联系 |
| **索引性** | "现实"只是索引词 |

**为什么相信模态实在论？**

| 理由 | 描述 |
|------|------|
| **简化** | 简化模态语义 |
| **统一** | 统一不同类型的量化 |
| **解释力** | 解释模态概念 |

**对模态实在论的反对**：

| 反对 | 描述 |
|------|------|
| **本体论膨胀** | 太多存在物 |
| **不可接受** | 违反直觉 |
| **不可知** | 如何认识其他世界？ |
| **无益** | 不增加解释力 |

**替代方案**：

| 方案 | 描述 |
|------|------|
| **抽象论** | 可能世界是抽象对象 |
| **组合论** | 可能世界是组合 |
| **虚构主义** | 可能世界是有用的虚构 |

**代表人物**：
- **大卫·刘易斯**：模态实在论创始人
- **批评者**：许多哲学家

**经典文献**：
- Lewis, *On the Plurality of Worlds* (1986)

**相关词条**：可能世界、抽象论

### 【可及关系】(Accessibility Relation)

**定义**：可能世界之间的关系，决定一个世界"能看见"哪些世界。

**费曼解释**：
可及关系是"世界之间的桥梁"：
- 世界w1可以"到达"世界w2
- 意味着：从w1的角度看，w2是可能的

**可及关系的形式**：

| 符号 | 含义 |
|------|------|
| wRw' | w可以到达w' |
| w'从w可及 | w'相对于w是可能的 |

**可及关系的直观**：

| 语境 | 可及关系 |
|------|----------|
| **真势模态** | w'符合w的物理/逻辑限制 |
| **认识模态** | w'与w所知一致 |
| **道义模态** | w'符合w的道德规范 |
| **时间模态** | w'在w的未来 |

**可及关系的性质**：

| 性质 | 条件 | 对应的公理 |
|------|------|------------|
| **自反** | ∀w: wRw | T: □p→p |
| **传递** | wRw' ∧ w'Rw'' → wRw'' | 4: □p→□□p |
| **对称** | wRw' → w'Rw | B: p→□◇p |
| **欧几里得** | wRw' ∧ wRw'' → w'Rw'' | 5: ◇p→□◇p |

**可及关系与模态系统**：

| 关系性质 | 模态系统 |
|----------|----------|
| 无限制 | K |
| 自反 | T |
| 自反+传递 | S4 |
| 自反+对称+传递 | S5 |
| 自反+欧几里得 | S5 |

**代表人物**：
- **克里普克**：可及关系语义
- **当代**：逻辑学家

**相关词条**：可能世界、模态系统、克里普克语义

### 【克里普克语义】(Kripke Semantics)

**定义**：模态逻辑的可能世界语义学。

**费曼解释**：
克里普克语义是"模态逻辑的数学模型"：
- 可能世界集合W
- 可及关系R
- 赋值函数V

这三者构成了模态逻辑的"模型"。

**克里普克模型**：

| 组成 | 描述 |
|------|------|
| **W** | 可能世界集合 |
| **R** | 可及关系 |
| **V** | 真值指派 |

**模态公式的语义**：

| 公式 | 语义条件 |
|------|----------|
| □p | 在所有可及世界中p为真 |
| ◇p | 在某个可及世界中p为真 |

**形式定义**：

```
M, w ⊨ □p  当且仅当  对所有w'使得wRw'，M, w' ⊨ p
M, w ⊨ ◇p  当且仅当  存在w'使得wRw'，M, w' ⊨ p
```

**克里普克语义的重要性**：

| 贡献 | 描述 |
|------|------|
| **完备性** | 证明了模态系统的完备性 |
| **直观** | 给出模态逻辑的直观解释 |
| **统一** | 统一不同模态系统的语义 |

**代表人物**：
- **克里普克**：创始人
- **当代**：逻辑学家

**经典文献**：
- Kripke, "Semantical Considerations on Modal Logic" (1963)

**相关词条**：可及关系、可能世界、模态系统

### 【模态系统】(Modal System)

**定义**：不同强度的模态逻辑公理系统。

**费曼解释**：
模态系统像"不同版本的模态逻辑"：
- 有些强：很多必然性成立
- 有些弱：较少必然性成立
- 根据可及关系的性质区分

**主要模态系统**：

| 系统 | 公理 | 可及关系 |
|------|------|----------|
| **K** | K公理 | 无限制 |
| **T** | K + T | 自反 |
| **S4** | K + T + 4 | 自反+传递 |
| **S5** | K + T + 5 | 等价关系 |

**公理模式**：

| 公理 | 形式 | 名称 |
|------|------|------|
| **K** | □(p→q)→(□p→□q) | 分配 |
| **T** | □p→p | 必然蕴涵实然 |
| **4** | □p→□□p | 正内省 |
| **B** | p→□◇p | 对称 |
| **5** | ◇p→□◇p | 欧几里得 |

**系统的强度**：

```
K ⊂ T ⊂ S4 ⊂ S5
    ⊂ S4.2 ⊂ S4.3 ⊂ S5
```

**常见系统的关系**：

| 系统 | 包含 | 被包含 |
|------|------|--------|
| K | - | T |
| T | K | S4, B |
| S4 | T | S5 |
| B | T | S5 |
| S5 | S4, B | - |

**系统的选择**：

| 问题 | 考虑 |
|------|------|
| **认识模态** | S4或S5？ |
| **真势模态** | S5？ |
| **道义模态** | T？ |

**代表人物**：
- **刘易斯**：早期模态系统
- **克里普克**：语义
- **当代**：逻辑学家

**相关词条**：模态算子、可及关系、克里普克语义

### 【K公理】(K Axiom)

**定义**：最基本的模态公理：□(p→q)→(□p→□q)。

**费曼解释**：
K公理说：
- 如果"如果p则q"是必然的
- 且p是必然的
- 那么q也是必然的

这是"必然性的分配律"。

**K公理的形式**：

```
□(p→q) → (□p → □q)
```

**K公理的直观**：

| 前提 | 结论 |
|------|------|
| 必然：如果p则q | |
| 必然：p | |
| | 所以：必然q |

**K公理的重要性**：

| 重要性 | 描述 |
|--------|------|
| **最小** | 最小的模态系统K的核心 |
| **分配** | 必然性分配到蕴涵 |
| **普遍** | 几乎所有模态系统都接受 |

**K系统的性质**：

| 性质 | 描述 |
|------|------|
| **最小** | 只有K公理和命题逻辑 |
| **无限制** | 可及关系无限制 |
| **基础** | 其他系统的基础 |

**代表人物**：
- **克里普克**：K系统
- **当代**：逻辑学家

**相关词条**：模态系统、T公理

---

### 【T公理】(T Axiom)

**定义**：模态公理：□p→p。

**费曼解释**：
T公理说：
- 如果p是必然的
- 那么p是真的

必然蕴涵实然。

**T公理的形式**：

```
□p → p
```

**T公理的直观**：

| 前提 | 结论 |
|------|------|
| 必然：p | |
| | 所以：p（实际上） |

**可及关系条件**：

- T公理对应可及关系的**自反性**
- ∀w: wRw（每个世界可及自己）

**T公理的问题**：

| 模态类型 | 是否接受T？ |
|----------|-------------|
| **真势模态** | 接受 |
| **认识模态** | 接受（知道p蕴涵p） |
| **道义模态** | 不接受（应该p不蕴涵p） |

**道义系统的处理**：

- 道义逻辑不使用T公理
- 使用D公理：□p→◇p（应该蕴涵可能）

**代表人物**：
- **当代**：逻辑学家

**相关词条**：K公理、模态系统、自反性

---

### 【S4系统】(S4 System)

**定义**：包含K、T和4公理的模态系统。

**费曼解释**：
S4是"知道你知道"的系统：
- K：分配
- T：知道蕴涵真
- 4：知道蕴涵知道你知道

**S4的公理**：

| 公理 | 形式 |
|------|------|
| **K** | □(p→q)→(□p→□q) |
| **T** | □p→p |
| **4** | □p→□□p |

**4公理的意义**：

- □p→□□p
- 如果必然p，那么必然"必然p"
- 正内省：如果你知道p，你知道你知道p

**可及关系**：

- S4对应**自反且传递**的可及关系
- 自反：每个世界可及自己
- 传递：如果w可达w'，w'可达w''，则w可达w''

**S4的模态词简化**：

| 公式 | S4中等价于 |
|------|------------|
| □□...□p | □p |
| ◇◇...◇p | ◇p |
| □◇□◇p | □◇p |
| ◇□◇□p | ◇□p |

**S4的应用**：

| 领域 | 应用 |
|------|------|
| **认识逻辑** | 知道（正内省） |
| **直觉主义逻辑** | S4嵌入 |
| **拓扑学** | 拓扑语义 |

**代表人物**：
- **刘易斯**：S4系统
- **当代**：逻辑学家

**相关词条**：S5系统、模态系统、传递性

### 【S5系统】(S5 System)

**定义**：最强的标准模态系统，包含K、T和5公理。

**费曼解释**：
S5是"最简单的模态逻辑"：
- 所有模态词可以简化
- 可能世界互相可及
- 必然 = 在所有可能世界真

**S5的公理**：

| 公理 | 形式 |
|------|------|
| **K** | □(p→q)→(□p→□q) |
| **T** | □p→p |
| **5** | ◇p→□◇p |

**5公理的意义**：

- ◇p→□◇p
- 如果可能p，那么必然"可能p"
- 负内省：如果你不知道p，你知道你不知道

**可及关系**：

- S5对应**等价关系**
- 自反：每个世界可及自己
- 对称：如果w可及w'，则w'可及w
- 传递：可及性是传递的

**S5的模态词简化**：

| 公式 | S5中等价于 |
|------|------------|
| □□...□p | □p |
| ◇◇...◇p | ◇p |
| □◇□◇p | ◇p 或 □p |
| 任意嵌套 | 简化为□或◇ |

**S5 vs S4**：

| 特点 | S4 | S5 |
|------|-----|-----|
| **可及关系** | 自反+传递 | 等价关系 |
| **模态简化** | 部分 | 完全 |
| **应用** | 知识逻辑（部分） | 知识逻辑（完全） |

**S5的应用**：

| 领域 | 应用 |
|------|------|
| **认识逻辑** | 完全内省的知识 |
| **真势模态** | 形而上学必然 |
| **简化证明** | 最简单的模态逻辑 |

**代表人物**：
- **刘易斯**：S5系统
- **当代**：逻辑学家

**相关词条**：S4系统、模态系统、等价关系

### 【严格蕴涵】(Strict Implication)

**定义**：刘易斯引入的蕴涵概念：□(p→q)，即"必然（如果p则q）"。

**费曼解释**：
实质蕴涵有问题：
- "如果1=2，则雪是白的"为真？（假蕴涵任何命题）

严格蕴涵解决：
- p严格蕴涵q = 必然（如果p则q）
- 需要有"必然联系"

**严格蕴涵 vs 实质蕴涵**：

| 类型 | 定义 | 问题 |
|------|------|------|
| **实质蕴涵** | p→q = ¬p∨q | 悖论 |
| **严格蕴涵** | p⇒q = □(p→q) | 仍有悖论 |

**严格蕴涵的符号**：

| 符号 | 含义 |
|------|------|
| p⇒q | p严格蕴涵q |
| p⤳q | p严格蕴涵q |
| □(p→q) | 必然（如果p则q） |

**严格蕴涵的悖论**：

| 悖论 | 形式 |
|------|------|
| **必然命题被任何命题严格蕴涵** | □q ⇒ (p⇒q) |
| **不可能命题严格蕴涵任何命题** | ¬◇p ⇒ (p⇒q) |

**对严格蕴涵的回应**：

| 方案 | 描述 |
|------|------|
| **相关逻辑** | 要求前件和后件相关 |
| **接受悖论** | 认为不是悖论 |
| **修改定义** | 其他条件 |

**代表人物**：
- **刘易斯**：严格蕴涵
- **当代**：相关逻辑学家

**经典文献**：
- Lewis, *A Survey of Symbolic Logic* (1918)

**相关词条**：实质蕴涵、相关逻辑

### 【认识逻辑】(Epistemic Logic)

**定义**：研究知识、信念等认识论概念的模态逻辑。

**费曼解释**：
认识逻辑问："知道"和"相信"的逻辑是什么？
- Kp：知道p
- Bp：相信p
- 可以分析知识悖论、信念逻辑

**认识算子**：

| 算子 | 符号 | 含义 |
|------|------|------|
| **知道** | Kp | 知道p |
| **相信** | Bp | 相信p |
| **对每个人** | □_a p | 每个人知道p |

**认识逻辑的公理**：

| 公理 | 形式 | 含义 |
|------|------|------|
| **K** | K(p→q)→(Kp→Kq) | 知道分配到蕴涵 |
| **T** | Kp→p | 知道蕴涵真 |
| **4** | Kp→KKp | 正内省 |
| **5** | ¬Kp→K¬Kp | 负内省 |

**认识逻辑的问题**：

| 问题 | 描述 |
|------|------|
| **逻辑全知** | 知道所有逻辑后承？ |
| **内省** | 知道自己知道？ |
| **信念悖论** | 相信悖论 |

**逻辑全知问题**：

- 如果知道p，且p蕴涵q
- 是否知道q？
- 现实中：不一定！

**公共知识**：

| 概念 | 符号 | 含义 |
|------|------|------|
| **公共知识** | Cp | 每个人知道p，每个人知道每个人知道p，... |

**代表人物**：

**相关词条**：模态逻辑、知识、信念

### 【道义逻辑】(Deontic Logic)

**定义**：研究义务、许可等道义概念的模态逻辑。

**费曼解释**：
道义逻辑问："应该"和"允许"的逻辑是什么？
- Op：应该p
- Pp：允许p
- 可以分析道德推理

**道义算子**：

| 算子 | 符号 | 含义 |
|------|------|------|
| **应该** | Op | 应该p |
| **允许** | Pp | 允许p |
| **禁止** | Fp | 禁止p |

**道义算子的关系**：

| 关系 | 公式 |
|------|------|
| **允许 = 不禁止** | Pp ↔ ¬Fp |
| **应该 = 不允许不** | Op ↔ ¬P¬p |
| **禁止 = 应该不** | Fp ↔ O¬p |

**道义逻辑的公理**：

| 公理 | 形式 | 含义 |
|------|------|------|
| **K** | O(p→q)→(Op→Oq) | 应该分配 |
| **D** | Op→Pp | 应该蕴涵允许 |

**道义悖论**：

| 悖论 | 描述 |
|------|------|
| **罗斯悖论** | 应该寄信 → 应该（寄信或烧信）？ |
| **善意悖论** | 应该帮助 → 应该（帮助或杀人）？ |
| **齐硕姆悖论** | 条件义务的问题 |

**齐硕姆悖论**：

| 前提 | 问题 |
|------|------|
| 应该去帮助 | |
| 如果去帮助，应该告诉 | |
| 不去帮助 | |
| 如果不去帮助，不应该告诉 | |
| | 应该告诉 且 不应该告诉？ |

**代表人物**：
- **冯·赖特**：道义逻辑创始人
- **当代**：逻辑学家

**经典文献**：
- von Wright, "Deontic Logic" (1951)

**相关词条**：模态逻辑、义务、伦理学

### 【后验必然】(A Posteriori Necessity)

**定义**：克里普克概念：需要经验才能知道，但必然为真的命题。

**费曼解释**：
后验必然是"违反直觉的组合"：
- 后验：需要经验才能知道
- 必然：在所有可能世界都真

例如："晨星 = 暮星"

**后验必然的例子**：

| 例子 | 描述 |
|------|------|
| **晨星 = 暮星** | 都指金星，但需要天文观测才知道 |
| **水 = H₂O** | 水的本质是H₂O，但需要化学发现 |
| **热 = 分子运动** | 热的本质，但需要物理学发现 |
| **猫是动物** | 猫的本质，但需要经验确认 |

**为什么是后验必然？**

| 特征 | 解释 |
|------|------|
| **后验** | 需要经验发现两个词项指同一事物 |
| **必然** | 一旦发现，知道在所有可能世界都真 |
| **严格指示词** | 专名在所有可能世界指同一对象 |

**传统观点的挑战**：

| 传统观点 | 克里普克的挑战 |
|----------|----------------|
| 先验 ↔ 必然 | 可以分离 |
| 后验 ↔ 偶然 | 可以分离 |

**四种组合**：

| 类型 | 例子 |
|------|------|
| **先验必然** | 2+2=4 |
| **后验必然** | 晨星=暮星 |
| **先验偶然** | 巴黎标准米是一米 |
| **后验偶然** | 地球有月球 |

**代表人物**：
- **克里普克**：后验必然概念

**经典文献**：
- Kripke, *Naming and Necessity* (1972)

**相关词条**：先验偶然、必然性、严格指示词

### 【先验偶然】(A Priori Contingency)

**定义**：克里普克概念：可以独立于经验知道，但偶然为真的命题。

**费曼解释**：
先验偶然也是"违反直觉的组合"：
- 先验：不需要经验就能知道
- 偶然：可能在其他可能世界不同

例如："巴黎标准米是一米"

**先验偶然的例子**：

| 例子 | 描述 |
|------|------|
| **巴黎标准米是一米** | 根据定义，但棒的长度是偶然的 |
| **我存在（现在）** | 先验可知，但我的存在是偶然的 |
| **如果p，则p** | 先验可知，但p的真值是偶然的 |

**巴黎标准米案例**：

| 分析 | 描述 |
|------|------|
| **定义** | "一米"定义为巴黎标准米的长度 |
| **先验** | 知道"巴黎标准米是一米"不需要测量 |
| **偶然** | 巴黎标准棒的长度可能是不同的 |

**争议**：

| 问题 | 描述 |
|------|------|
| **是否真有先验偶然？** | 有些哲学家质疑 |
| **例子是否恰当？** | 对例子的不同分析 |

**代表人物**：
- **克里普克**：先验偶然概念

**相关词条**：后验必然、偶然性、先验

### 【严格指示词】(Rigid Designator)

**定义**：克里普克概念：在所有可能世界指称同一对象的词项。

**费曼解释**：
严格指示词是"总是指向同一对象"：
- "亚里士多德"：在所有可能世界都指那个人
- 不是"亚历山大的老师"（非严格）
- 不是"柏拉图的学生"（非严格）

**严格 vs 非严格**：

| 类型 | 定义 | 例子 |
|------|------|------|
| **严格指示词** | 在所有可能世界指同一对象 | "亚里士多德" |
| **非严格指示词** | 在不同可能世界可能指不同对象 | "亚历山大的老师" |

**严格指示词的类型**：

| 类型 | 例子 |
|------|------|
| **专名** | "亚里士多德"、"金星" |
| **自然种类词** | "水"、"黄金"、"老虎" |

**严格指示词与必然性**：

| 关系 | 描述 |
|------|------|
| **如果a=b且a和b都是严格指示词** | 则"必然a=b" |
| **例子** | 必然（晨星=暮星） |

**对摹状词理论的批评**：

| 传统观点 | 克里普克的批评 |
|----------|----------------|
| 专名=缩写的摹状词 | 专名不是摹状词 |
| "亚里士多德"="亚历山大的老师" | 亚里士多德可能不是亚历山大的老师 |

**因果历史理论**：

| 要点 | 描述 |
|------|------|
| **命名仪式** | 对象最初被命名 |
| **因果链** | 名称通过因果链传递 |
| **不需要描述** | 不需要知道描述就能使用名称 |

**代表人物**：
- **克里普克**：严格指示词概念

**相关词条**：后验必然、命名理论、因果历史理论

### 【反事实条件句】(Counterfactual Conditional)

**定义**：关于非实际情况的条件句："如果...（但实际上不...），那么..."。

**费曼解释**：
反事实条件句问"如果情况不同会怎样"：
- "如果我是你，我会接受这份工作"
- 但我不是你
- 这句话有意义吗？如何判断真假？

**反事实条件句的形式**：

| 符号 | 含义 |
|------|------|
| p □→ q | 如果p（反事实地），则q |

**反事实的问题**：

| 问题 | 描述 |
|------|------|
| **真值条件** | 反事实条件句何时为真？ |
| **相似性** | 哪些可能世界是"最相似"的？ |
| **前提固定** | 哪些事实应该保持不变？ |

**刘易斯的语义**：

| 条件 | 描述 |
|------|------|
| p □→ q 真 | 当且仅当在p为真的最相似世界中，q为真 |

**反事实的应用**：

| 领域 | 应用 |
|------|------|
| **因果** | 因果推理常使用反事实 |
| **历史** | "如果希特勒死于1939年..." |
| **决策** | "如果我选择不同..." |
| **道德** | "如果你做了不同的事..." |

**反事实与因果**：

| 关系 | 描述 |
|------|------|
| **因果分析** | C导致E = 如果C不发生，E不会发生 |
| **反事实依赖** | E反事实依赖于C |

**代表人物**：
- **大卫·刘易斯**：反事实语义
- **当代**：形而上学家

**经典文献**：
- Lewis, *Counterfactuals* (1973)

**相关词条**：可能世界、因果关系

### 【本质主义】(Essentialism)

**定义**：认为事物具有本质属性的观点。

**费曼解释**：
本质主义问："什么东西使得这个东西是这个东西？"
- 水的本质：H₂O
- 人的本质：理性？（争议）
- 亚里士多德的本质：他是那个人

**本质主义的核心**：

| 概念 | 描述 |
|------|------|
| **本质属性** | 在所有可能世界都有的属性 |
| **偶有属性** | 可能在某些可能世界没有的属性 |

**本质 vs 偶有**：

| 类型 | 定义 | 例子（对亚里士多德） |
|------|------|----------------------|
| **本质** | 必然具有 | 是人？是亚里士多德？ |
| **偶有** | 可能没有 | 是亚历山大的老师 |

**本质属性的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **种类本质** | 属于某一种类 | 水是H₂O |
| **个体本质** | 个体的本质 | 亚里士多德是那个人 |
| **起源本质** | 起源决定本质 | 来自这个受精卵 |

**对本质主义的批评**：

| 批评 | 描述 |
|------|------|
| **蒯因** | 本质主义是可疑的 |
| **语境依赖** | 本质是相对的 |
| **模糊性** | 本质边界不清楚 |

**克里普克的辩护**：

- 自然种类词是严格指示词
- 因此有本质属性
- 水的本质是H₂O

**代表人物**：
- **亚里士多德**：本质主义先驱
- **克里普克**：现代本质主义
- **蒯因**：批评者

**相关词条**：必然性、模态实在论、自然种类

### 【跨界同一性】(Transworld Identity)

**定义**：同一对象在不同可能世界中的同一性问题。

**费曼解释**：
跨界同一性问："不同可能世界中的'亚里士多德'是同一个人吗？"
- 这个世界的亚里士多德：亚历山大的老师
- 那个世界的亚里士多德：不是亚历山大的老师
- 仍然是同一个人吗？

**跨界同一性的问题**：

| 问题 | 描述 |
|------|------|
| **如何识别？** | 如何判断不同世界的对象是同一的？ |
| **本质属性** | 需要共享哪些属性？ |
| **多少属性？** | 需要共享多少属性？ |

**解决方案**：

| 方案 | 描述 |
|------|------|
| **本质主义** | 共享本质属性 |
| **跨界个体** | 使用跨界个体概念 |
| **反实在论** | 拒绝跨界同一性 |

**刘易斯的对应体理论**：

| 概念 | 描述 |
|------|------|
| **对应体** | 不同世界的对象不是同一的，而是"对应体" |
| **相似性** | 对应体关系基于相似性 |

**对对应体理论的批评**：

| 批评 | 描述 |
|------|------|
| **违反直觉** | 不同世界的"我"不是我？ |
| **问题转移** | 问题变成"什么是对应体？" |

**代表人物**：
- **刘易斯**：对应体理论
- **克里普克**：跨界同一性
- **当代**：形而上学家

**相关词条**：可能世界、本质主义、同一性

### 【对应体理论】(Counterpart Theory)

**定义**：大卫·刘易斯的理论：不同世界的对象不是同一的，而是对应体关系。

**费曼解释**：
刘易斯说："其他世界的'你'不是你"
- 每个对象只存在于一个世界
- 其他世界的对象是"对应体"
- 非常相似，但不是同一

**对应体理论的核心**：

| 主张 | 描述 |
|------|------|
| **世界限制** | 每个对象只存在于一个世界 |
| **对应体** | 其他世界的"相似对象" |
| **相似性** | 对应体关系基于相似性 |

**为什么需要对应体理论？**

| 理由 | 描述 |
|------|------|
| **避免跨界同一问题** | 不需要跨界同一 |
| **符合模态实在论** | 世界是完全隔离的 |
| **简化** | 没有跨界实体 |

**对应体理论的问题**：

| 问题 | 描述 |
|------|------|
| **违反直觉** | "其他世界的我"不是我？ |
| **相似性模糊** | 如何定义相似性？ |
| **本质问题** | 本质属性如何确定？ |

**对对应体理论的批评**：

| 批评者 | 批评 |
|--------|------|
| **克里普克** | 违反直觉，"汉弗雷可能赢"应该是关于汉弗雷的 |
| **其他** | 对应体关系本身需要解释 |

**代表人物**：
- **大卫·刘易斯**：对应体理论创始人

**相关词条**：跨界同一性、模态实在论、可能世界

### 【模态坍缩】(Modal Collapse)

**定义**：在某些模态系统中，模态词失去区分力的问题。

**费曼解释**：
模态坍缩是"模态词变成了平凡"：
- 必然p ↔ 可能p ↔ p
- 失去了区分必然和偶然的能力

**模态坍缩的形式**：

| 问题 | 描述 |
|------|------|
| **必然=可能** | □p ↔ ◇p |
| **必然=实然** | □p ↔ p |
| **模态消失** | 模态词没有意义 |

**导致模态坍缩的原因**：

| 原因 | 描述 |
|------|------|
| **神学应用** | 上帝存在导致所有事必然 |
| **公理太强** | 某些公理组合导致坍缩 |
| **系统问题** | 某些模态系统有问题 |

**相关词条**：模态系统、模态逻辑

---

### 【模态谓词逻辑】(Modal Predicate Logic)

**定义**：结合模态算子和量词的逻辑系统。

**费曼解释**：
模态谓词逻辑是"模态+量词"：
- □∀x P(x)：必然，所有x都是P
- ∀x □P(x)：对所有x，x必然是P
- 这两个不同！

**模态谓词逻辑的问题**：

| 问题 | 描述 |
|------|------|
| **量词辖域** | □∀x vs ∀x□ |
| **跨界量化** | 量词跨可能世界？ |
| **Barcan公式** | ◇∃x → ∃x◇ 的有效性 |

**Barcan公式**：

| 公式 | 形式 | 方向 |
|------|------|------|
| **Barcan公式** | ◇∃x P(x) → ∃x ◇P(x) | 从左到右 |
| **逆Barcan** | ∃x ◇P(x) → ◇∃x P(x) | 从右到左 |

**Barcan公式的语义条件**：

| 公式 | 条件 |
|------|------|
| **Barcan** | 递增域 |
| **逆Barcan** | 递减域 |

**存在域问题**：

| 立场 | 描述 |
|------|------|
| **常数域** | 所有可能世界有相同个体 |
| **变元域** | 不同世界可有不同个体 |
| **递增域** | 如果w可及w'，则w的域⊆w'的域 |

**代表人物**：
- **巴坎**：Barcan公式
- **当代**：逻辑学家

**相关词条**：谓词逻辑、模态逻辑、量词

---

### 【巴坎公式】(Barcan Formula)

**定义**：模态谓词逻辑中的公式：◇∃x P(x) → ∃x ◇P(x)。

**形式**：

| 公式 | 形式 |
|------|------|
| **Barcan公式** | ◇∃x Px → ∃x ◇Px |
| **逆Barcan** | ∃x ◇Px → ◇∃x Px |

**语义含义**：

| 公式 | 含义 |
|------|------|
| **Barcan** | 如果可能存在x使得Px，则存在x使得可能Px |
| **逆Barcan** | 如果存在x使得可能Px，则可能存在x使得Px |

**有效性条件**：

| 公式 | 语义条件 |
|------|----------|
| **Barcan** | 递增域（可及世界的域不减少） |
| **逆Barcan** | 递减域（可及世界的域不增加） |

**哲学意义**：

| 问题 | 描述 |
|------|------|
| **存在跨世界** | 同一个体可以存在于不同世界？ |
| **可能对象** | 是否有"可能存在但实际不存在"的对象？ |

**代表人物**：
- **露丝·巴坎·马库斯**：Barcan公式

**相关词条**：模态谓词逻辑、量词
