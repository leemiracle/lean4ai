## A-F

### 【事件】(Events)

**定义**：在特定时间、地点发生的具体发生的事情或状态变化。

**费曼解释**：
- 你看书（活动/过程）
- 你翻了一页（事件）
- 你读完了一章（事件）

事件是"发生的"——有明确的开始和结束。

**事件的哲学问题**：

| 问题 | 描述 |
|------|------|
| **事件是什么？** | 事件的本质？ |
| **事件存在吗？** | 还是只是语言现象？ |
| **事件vs物体** | 区别是什么？ |
| **事件的同一性** | 两个事件何时是同一个？ |

**事件理论**：

| 理论 | 核心主张 | 代表 |
|------|----------|------|
| **性质例示** | 事件是对象在某时具有某性质 | Kim |
| **事实理论** | 事件是事实 | Davidson |
| **变化理论** | 事件是变化 | 广泛接受 |
| **四维部分** | 事件是时空部分 | 四维论者 |

**金在权的性质例示理论**：

```
事件 = (对象, 性质, 时间)
```

例子：
- (苏格拉底, 喝毒酒, 公元前399年)
- (这张桌子, 是棕色的, 2026年)

**事件的特征**：

| 特征 | 描述 |
|------|------|
| **时间性** | 发生在特定时间 |
| **空间性** | 发生在特定地点 |
| **具体性** | 是具体的，不是抽象的 |
| **唯一性** | 不可重复（类似事件≠同一事件） |

**事件 vs 物体**：

| 特征 | 物体 | 事件 |
|------|------|------|
| 存在方式 | 持续存在 | 发生/出现 |
| 时间性 | 在时间中延伸 | 占据时间段 |
| 变化 | 可以变化 | 本身是变化 |
| 例子 | 桌子 | 桌子的移动 |

**事件的同一性**：

**问题**：两个事件何时是同一个事件？

| 标准 | 描述 | 问题 |
|------|------|------|
| **同一时间地点** | 同时间地点的事件是同一的 | 同一时间地点可以有多个事件 |
| **同一原因结果** | 因果链相同的事件是同一的 | 循环？ |
| **同一描述** | 描述相同的事件是同一的 | 描述依赖语言 |

**代表人物**：
- **Donald Davidson**：事件理论
- **Jaegwon Kim**：性质例示理论
- **当代**：事件本体论研究者

**经典文献**：
- Davidson, "Events as Particulars" (1970)
- Kim, "Events as Property Exemplifications" (1976)

**当代发展 (2020-2026)**：
- **事件与量子力学**：量子事件
- **事件与社会**：社会事件
- **事件与AI**：AI识别事件
- **事件与历史**：历史事件

**跨学科联系**：
- **语言学**：事件语义学
- **计算机科学**：事件驱动编程
- **历史学**：历史事件
- **心理学**：事件记忆

**中国视角**：
- **事**：中国哲学的事
- **物**：物与事
- **当代**：事件哲学的中国视角

**争议问题**：
1. 事件的本质是什么？
2. 事件如何个体化？
3. 事件与物体的关系？

**相关词条**：过程、变化、物体、时间

### 【过程】(Processes)

**定义**：在时间中持续进行的活动或变化。

**费曼解释**：
- 你走路（过程）vs 你走了一步（事件）
- 水流动（过程）vs 水流到了某处（事件）
- 思考（过程）vs 想到了一个想法（事件）

过程是"进行的"——在时间中持续展开。

**过程的特征**：

| 特征 | 描述 |
|------|------|
| **持续性** | 在时间中持续 |
| **动态性** | 不断变化 |
| **内在结构** | 有阶段和部分 |
| **开放性** | 可以被中断 |

**过程 vs 事件**：

| 特征 | 过程 | 事件 |
|------|------|------|
| 完成性 | 可以未完成 | 有完成点 |
| 内在结构 | 有丰富结构 | 相对简单 |
| 语言 | "在V-ing" | "V-ed" |
| 例子 | 在跑步 | 跑了一步 |

**过程哲学** (Process Philosophy)：

**怀特海的过程哲学**：
- 一切都是过程
- 物体是"慢过程"
- 没有静态的存在

**核心主张**：
1. 过程是基本的，物体是派生的
2. 变化是根本的，静态是错觉
3. 关系优先于实体

**过程本体论**：

| 类型 | 描述 | 代表 |
|------|------|------|
| **激进过程论** | 一切都是过程 | 怀特海 |
| **温和过程论** | 过程和物体都存在 | 广泛 |
| **实体优先** | 过程是实体的变化 | 亚里士多德传统 |

**代表人物**：
- **怀特海 (Whitehead)**：过程哲学创始人
- **柏格森 (Bergson)**：时间与过程
- **当代**：过程本体论研究者

**经典文献**：
- Whitehead, *Process and Reality* (1929)
- Bergson, *Creative Evolution* (1907)

**当代发展 (2020-2026)**：
- **过程与量子力学**：量子过程
- **过程与生物**：生命过程
- **过程与AI**：AI的过程理解
- **过程与社会**：社会过程

**跨学科联系**：
- **物理学**：物理过程
- **生物学**：生命过程
- **心理学**：心理过程
- **社会学**：社会过程

**中国视角**：
- **易**：变化与过程
- **生生**：生生之谓易
- **气**：气的流动过程
- **当代**：过程哲学的中国视角

**争议问题**：
1. 过程是基本的吗？
2. 过程与事件的关系？
3. 过程如何个体化？

**相关词条**：事件、变化、怀特海、时间

### 【变化】(Change)

**定义**：事物在不同时间具有不同性质或状态。

**费曼解释**：
- 苹果从绿色变成红色
- 你从儿童变成成人
- 水从液体变成气体

变化是"不同时间，不同状态"。

**变化的哲学问题**：

| 问题 | 描述 |
|------|------|
| **变化如何可能？** | 同一事物怎能既A又非A？ |
| **变化是真实的吗？** | 巴门尼德：变化是错觉 |
| **变化的主体是什么？** | 什么在变化？ |

**巴门尼德的挑战**：

> 变化是不可能的：
> - 如果某物变为存在，它之前不存在
> - 但不存在者不能变为存在
> - 因此，变化不可能

**亚里士多德的回应**：

**潜能与现实**：
- 橡子不是树，但"潜能地"是树
- 变化是潜能变为现实
- 不是"无中生有"

**变化的类型**：

| 类型 | 定义 | 例子 |
|------|------|------|
| **性质变化** | 性质改变 | 苹果变红 |
| **位置变化** | 位置移动 | 汽车移动 |
| **数量变化** | 数量增减 | 长大 |
| **生成/毁灭** | 开始/停止存在 | 出生/死亡 |

**变化与时间**：

| 理论 | 变化与时间的关系 |
|------|------------------|
| **A-理论** | 变化是真实的，时间流逝 |
| **B-理论** | 变化只是不同时间部分不同 |
| **麦克塔格特** | 变化需要A-序列，但A-序列矛盾 |

**四维论的变化观**：

- 没有真正的"变化"
- 只有不同时间部分有不同性质
- "变化"只是语言现象

**代表人物**：
- **巴门尼德**：变化不可能
- **亚里士多德**：潜能与现实
- **当代**：变化的理论研究者

**经典文献**：
- Aristotle, *Physics*
- Parmenides fragments

**当代发展 (2020-2026)**：
- **量子变化**：量子态的变化
- **社会变化**：社会变迁
- **AI变化**：AI系统的变化
- **变化与时间**：持续研究

**跨学科联系**：
- **物理学**：物理变化
- **生物学**：进化与变化
- **历史学**：历史变迁
- **经济学**：经济变化

**中国视角**：
- **易**：变化哲学
- **变**：变与不变
- **化**：化生万物
- **当代**：变化哲学的中国视角

**争议问题**：
1. 变化如何可能？
2. 变化是真实的吗？
3. 变化与时间的关系？

**相关词条**：过程、事件、时间、持存

### 【事实】(Facts)

**定义**：使命题为真的东西，世界中的"真相"。

**费曼解释**：
- "雪是白的"是真的，因为雪确实是白的
- "雪确实是白的"就是一个事实
- 事实是"使真理为真"的东西

**事实理论**：

| 理论 | 核心主张 | 代表 |
|------|----------|------|
| **真理制造者** | 事实使命题为真 | Mulligan, Simons |
| **事实实在论** | 事实是世界的组成部分 | Armstrong |
| **事实取消论** | 不需要事实，只需对象和性质 | 某些唯名论者 |

**事实的类型**：

| 类型 | 定义 | 例子 |
|------|------|------|
| **原子事实** | 最基本的事实 | a是F |
| **分子事实** | 复合事实 | a是F且b是G |
| **普遍事实** | 关于所有的事实 | 所有F都是G |
| **否定事实** | 否定性事实 | a不是F |
| **模态事实** | 关于模态的事实 | a可能存在 |

**维特根斯坦的事实观**：

> "世界是一切发生的事情"（事实的总和）
> "发生的事情——即事实——是诸事态的存在"

**罗素的事实观**：

- 事实是使命题为真的东西
- 事实不是对象，而是对象的组合
- 例如："苏格拉底是智慧的"这个事实 = 苏格拉底 + 智慧

**否定事实的问题**：

- "独角兽不存在"是真的
- 什么使它为真？
- 需要"否定事实"吗？还是"没有肯定事实"？

**代表人物**：
- **维特根斯坦**：逻辑哲学论
- **罗素**：逻辑原子论
- **Armstrong**：事实实在论

**经典文献**：
- Wittgenstein, *Tractatus Logico-Philosophicus* (1921)
- Russell, *The Philosophy of Logical Atomism* (1918)

**当代发展 (2020-2026)**：
- **真理制造者理论**：当代发展
- **事实与科学**：科学事实
- **事实与社会**：社会事实
- **事实与AI**：AI的事实理解

**跨学科联系**：
- **逻辑学**：事实与命题
- **语言哲学**：真理与意义
- **科学哲学**：科学事实
- **法律**：法律事实

**中国视角**：
- **实**：实事求是
- **真**：真理与事实
- **当代**：事实哲学的中国视角

**争议问题**：
1. 事实存在吗？
2. 事实是什么？
3. 否定事实存在吗？

**相关词条**：真理、命题、事件、状态

### 【状态】(States)

**定义**：事物在一定时间内的持续状况，不同于事件的变化性质。

**费曼解释**：
"杯子在桌上"是一个状态——持续的、静态的。
"杯子掉落"是一个事件——变化的、动态的。
状态是"静止的画面"，事件是"变化的画面"。

**核心要素**：
1. **持续性**：状态持续一段时间
2. **静态性**：状态本身不蕴含变化
3. **可终止**：状态可以开始和结束
4. **关系性**：状态可以是关系的（x在y上）

**状态 vs 事件**：

| 特征 | 状态 | 事件 |
|------|------|------|
| **动态性** | 静态 | 动态 |
| **变化** | 不蕴含变化 | 蕴含变化 |
| **时间** | 持续 | 发生 |
| **例子** | 在桌上、知道、相信 | 掉落、学习、决定 |

**状态的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **物理状态** | 物理系统的状态 | 静止、匀速运动 |
| **心理状态** | 心理系统的状态 | 相信、欲望 |
| **社会状态** | 社会关系的状态 | 已婚、雇佣 |

**状态的形而上学**：

| 问题 | 描述 |
|------|------|
| **状态存在吗？** | 状态是独立实体还是事件的派生物？ |
| **状态与属性** | 状态就是属性吗？ |
| **静态世界** | 能否有只有状态没有事件的世界？ |

**代表人物**：
- **Vendler**：语言中的状态分析
- **Mourelatos**：事件本体论中的状态

**经典文献**：
- Vendler, "Verbs and Times" (1957)
- Mourelatos, "Events, Processes, and States" (1978)

**当代发展 (2020-2026)**：
- **认知科学**：状态的心理表征
- **语言学**：状态与事件的语义
- **AI**：状态表示与推理

**跨学科联系**：
- **语言学**：时态与体态
- **认知科学**：状态的概念化
- **计算机科学**：状态机

**中国视角**：
- **静与动**：中国哲学中的动静之辩
- **状态**：阴阳状态

**争议问题**：
1. 状态是独立实体还是派生概念？
2. 状态如何与事件相关？
3. 所有状态都能事件化吗？

**相关词条**：事件、过程、变化

### 【发生】(Occurrences)

**定义**：在时间中发生的事件类型，包括瞬时发生和持续发生。

**费曼解释**：
"发生"是事件在时间中的方式：
- 闪电：瞬间发生
- 跑步：持续发生
- 到达：瞬间的边界

**核心要素**：
1. **时间性**：发生在时间中
2. **边界**：有开始和结束
3. **完整性**：可以完成或进行中
4. **体态**：完成体vs未完成体

**Vendler的分类**：

| 类型 | 描述 | 例子 | 体态特征 |
|------|------|------|----------|
| **状态 (States)** | 持续、静态 | 知道 | 无边界 |
| **活动 (Activities)** | 持续、动态 | 跑步 | 无自然终点 |
| **成就 (Achievements)** | 瞬时、有终点 | 到达 | 瞬间完成 |
| **完成 (Accomplishments)** | 持续、有终点 | 画一幅画 | 有过程有终点 |

**体态 (Aspect)**：

| 体态 | 描述 | 例子 |
|------|------|------|
| **完成体** | 已完成的视角 | 他跑完了 |
| **进行体** | 进行中的视角 | 他正在跑 |
| **未完成体** | 未完成的视角 | 他没跑完 |

**发生的本体论**：

| 问题 | 描述 |
|------|------|
| **发生是实体吗？** | 发生是独立的事件实体？ |
| **时间位置** | 发生如何占据时间？ |
| **部分关系** | 长发生由短发生构成？ |

**代表人物**：
- **Vendler**：动词与时间类型学
- **Pianesi & Varzi**：事件本体论

**经典文献**：
- Vendler, "Verbs and Times" (1957)
- Pianesi & Varzi, "Events and Event Talk" (1996)

**当代发展 (2020-2026)**：
- **形式语义学**：事件语义学
- **计算语言学**：事件抽取
- **认知科学**：事件概念化

**跨学科联系**：
- **语言学**：时态与体态
- **计算机科学**：事件处理
- **心理学**：事件感知

**中国视角**：
- **时态**：汉语的时态系统

**争议问题**：
1. 发生的四分类型足够吗？

### 【活动】(Activities)

**定义**：持续进行但没有自然终点的动态事件。

**费曼解释**：
跑步是活动——可以一直跑下去，没有"跑完"的内在终点。
画一幅画不是活动——有"画完"的自然终点。

**核心要素**：
1. **持续性**：在时间中持续
2. **动态性**：不同于静态状态
3. **无内在终点**：没有自然的"完成"点
4. **同质性**：各部分性质相同

**活动 vs 完成**：

| 特征 | 活动 | 完成 |
|------|------|------|
| **内在终点** | 无 | 有 |
| **完成标准** | 无内在标准 | 有内在标准 |
| **例子** | 跑、游泳 | 跑一英里、游过河 |

**活动的同质性**：
- 活动的任何部分都是同一活动
- 跑步的一分钟还是跑步
- 完成的部分不是完成（画一半不是"画了一幅画"）

**活动的语言特征**：

| 特征 | 例子 |
|------|------|
| **可带"正在"** | 正在跑步 ✓ |
| **不可带"完成"** | 完成跑步 ✗（语义怪异） |
| **可带时长** | 跑了一小时 ✓ |

**代表人物**：
- **Vendler**：活动作为事件类型
- **Dowty**：词汇语义学中的活动

**经典文献**：
- Vendler, "Verbs and Times" (1957)
- Dowty, *Word Meaning and Montague Grammar* (1979)

**当代发展 (2020-2026)**：
- **词汇语义学**：动词分类
- **事件语义学**：活动的形式分析
- **AI**：活动识别

**跨学科联系**：
- **语言学**：动词体态
- **认知科学**：活动概念
- **AI**：行为识别

**中国视角**：
- **动补结构**：汉语的动补结构与体态
- **过程动词**：汉语过程动词

**争议问题**：
1. 活动与状态的区别是什么？

### 【成就】(Achievements)

**定义**：瞬时完成的事件类型，有明确的终点。

**费曼解释**：
到达是成就——你到达的那一刻，事件就完成了。
没有"正在到达"——要么到达了，要么没到达。

**核心要素**：
1. **瞬时性**：在时间点上发生
2. **有终点**：有明确的完成点
3. **二值性**：要么完成要么未完成
4. **无过程**：没有持续的过程

**成就的例子**：

| 成就 | 描述 |
|------|------|
| **到达** | 瞬时完成 |
| **认出** | 瞬时识别 |
| **死亡** | 瞬时发生 |
| **赢** | 瞬时达成 |

**成就 vs 完成**：

| 特征 | 成就 | 完成 |
|------|------|------|
| **过程** | 无过程 | 有过程 |
| **瞬时性** | 瞬时 | 持续 |
| **例子** | 到达、死亡 | 画一幅画、建房子 |

**成就的悖论**：

| 悖论 | 描述 |
|------|------|
| **准备悖论** | 到达有准备过程，但到达本身瞬时 |
| **进行体悖论** | "正在到达"似乎是可能的？ |

**代表人物**：
- **Vendler**：成就作为事件类型
- **Kenny**：行动类型学

**经典文献**：
- Vendler, "Verbs and Times" (1957)
- Kenny, *Action, Emotion and Will* (1963)

**当代发展 (2020-2026)**：
- **词汇语义学**：成就动词的分析
