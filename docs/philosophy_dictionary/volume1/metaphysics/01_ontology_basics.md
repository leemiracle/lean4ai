### 【本体论】(Ontology)

**词源**：希腊语 *on* (存在) + *logos* (学问)

**定义**：形而上学的核心分支，研究存在的本质、存在的种类、存在的基本范畴。

**费曼解释**：
想象你在整理宇宙中所有的东西：
- 苹果、椅子、人（物理对象）
- 数字、命题、性质（抽象对象）
- 时间、空间（时空实体）
- 可能世界、虚构人物（模态实体）

本体论问：这些东西"存在"吗？它们是什么类型的"存在"？它们之间有什么关系？

**核心要素**：
1. **存在 (Being/Existence)**：什么存在？
2. **范畴 (Categories)**：存在的类型有哪些？
3. **基本实体 (Fundamental Entities)**：什么是实在的基础？
4. **存在优先性 (Ontological Priority)**：什么是最根本的？

**主要立场**：

| 立场 | 核心主张 | 代表人物 |
|------|----------|----------|
| **本体论实在论** | 存在客观的范畴结构 | Armstrong, Lowe |
| **本体论反实在论** | 本体论是概念框架的产物 | Carnap, Putnam |
| **基础论 (Grounding)** | 存在有层级结构 | Schaffer, Fine |
| **存在一元论** | 只有一种根本存在 | Jonathan Schaffer |

**代表人物**：
- **亚里士多德**：《范畴篇》- 十范畴
- **海德格尔**：《存在与时间》- 存在问题
- **Quine**："On What There Is" - 存在量词
- **当代**：E.J. Lowe, David Oderberg, Kathrin Koslicki

**经典文献**：
- Aristotle, *Categories*
- Heidegger, *Being and Time* (1927)
- Quine, "On What There Is" (1948)
- Lowe, *The Possibility of Metaphysics* (1998)

**当代发展 (2020-2026)**：
- **形而上学基础 (Metaphysical Grounding)**：Fine, Schaffer的基础关系理论
- **本体论简约原则**：简化本体论承诺
- **社会本体论**：Searle的社会实在建构
- **量子本体论**：量子力学的本体论解释

**跨学科联系**：
- **信息科学**：本体论在AI和知识表示中的应用（OWL、RDF）
- **数据库设计**：本体论建模
- **认知科学**：概念范畴的心理基础
- **语言学**：语义范畴

**中国视角**：
- **冯契**：《中国古代哲学的逻辑发展》- 中国本体论传统
- **张世英**：《天人之际》- 中西本体论比较
- **牟宗三**：智的直觉与本体论
- **当代**：杨国荣、陈少明

**争议问题**：
1. 本体论问题是真问题还是语言问题？（Carnap vs Quine）
2. 是否有唯一的正确本体论？
3. 本体论依赖科学吗？

**相关词条**：存在、实在、形而上学、范畴、基础

### 【存在】(Being/Existence)

**词源**：拉丁语 *existere* (站出来)

**定义**：最普遍的哲学概念，指一切实在的东西所共有的性质——"有"或"是"。

**费曼解释**：
"存在"是最基本的"有"：
- 苹果存在 ✓
- 数字7存在？(抽象对象)
- 独角兽存在？(虚构对象)
- 过去存在？(时间)
- 可能世界存在？(模态)

就像"东西"是最一般的概念，"存在"是最一般的"属性"——但它是属性吗？

**存在的理论**：

| 理论 | 核心主张 | 主要支持者 |
|------|----------|------------|
| **存在是谓词** | "存在"是属性 | Meinong |
| **存在不是谓词** | "存在"是量词 | Kant, Frege, Russell |
| **存在即被感知** | 存在依赖于心灵 | Berkeley |
| **存在先于本质** | 人先存在，后定义自己 | Sartre |

**Quine的标准**：
"存在就是成为约束变元的值" (To be is to be the value of a bound variable)
- ∃x (x 是一只猫) = 有猫存在
- 存在就是被量词 ∃ 所涵盖

**存在的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **实际存在** | 现实世界中的存在 | 你、我、地球 |
| **可能存在** | 可能世界中的存在 | 可能的你（另一种职业） |
| **抽象存在** | 非时空的存在 | 数字、性质、命题 |
| **虚构存在** | 文学作品中的存在 | 哈利·波特 |

**经典论证**：

**本体论证明 (Anselm)**：
1. 上帝是"最伟大的可设想者"
2. 存在于现实中比只存在于心灵中更伟大
3. 因此上帝必然存在于现实中

**康德的批判**：
- "存在"不是真实的谓词
- 说"上帝存在"不增加任何性质
- 就像"有100塔勒"不比"可能的100塔勒"多什么

**当代发展 (2020-2026)**：
- **存在量化理论**：形式语义学的新发展
- **存在与模态**：模态实在论中的存在问题
- **社会存在**：社会建构物的存在模式
- **数字存在**：虚拟对象的存在论

**跨学科联系**：
- **逻辑学**：存在量词
- **语言学**：存在句（There be...）
- **神学**：上帝存在的证明
- **存在主义文学**：加缪、萨特

**中国视角**：
- **道家**："无"与"有"的辩证（老子）
- **佛家**："空"与"有"（中观、唯识）
- **儒家**：天人合一的存在观
- **熊十力**：体用不二

**争议问题**：
1. 存在是谓词还是量词？
2. 虚构对象存在吗？
3. 是否有不同的存在模式？

**相关词条**：本体论、实在、本质、虚无

### 【实在】(Reality/The Real)

**词源**：拉丁语 *realis* (与事物相关)

**定义**：独立于心灵、语言、表象而存在的事物总体；真实存在的东西。

**费曼解释**：
"实在"就像"真实的电影vs特效"：
- 你看到电影里的龙（表象）
- 但电影是特效制作的（不实在）
- 真实世界呢？
- 是我们看到的样子，还是别的什么？

**实在的类型**：

| 类型 | 定义 | 理论 |
|------|------|------|
| **朴素实在论** | 世界就是我们看到的样子 | 常识 |
| **科学实在论** | 科学描述真实实在 | 科学实在论 |
| **现象实在** | 我们经验到的实在 | 康德的现象 |
| **绝对实在** | 物自体 | 康德的本体 |

**实在论 vs 反实在论**：

| 问题 | 实在论 | 反实在论 |
|------|--------|----------|
| 外部世界 | 独立存在 | 依赖心灵 |
| 科学理论实体 | 真实存在 | 只是工具 |
| 数学对象 | 客观存在 | 人造物 |
| 道德事实 | 客观存在 | 建构的 |

**代表人物**：
- **柏拉图**：理念是真正的实在
- **亚里士多德**：个别事物是实在
- **康德**：现象是实在，本体不可知
- **当代**：John Searle (External Realism)

**经典论证**：

**笛卡尔的蜡块论证**：
- 一块蜡：有形状、气味、硬度
- 靠近火后：形状变了、气味没了、软了
- 但它还是"那块蜡"
- 因此，我们认识的不是感官性质，而是"广延的实体"

**当代发展 (2020-2026)**：
- **量子实在**：量子力学挑战经典实在观
- **虚拟实在**：元宇宙是"实在"吗？
- **社会实在**：Searle的制度性事实
- **生态实在**：人类世的实在性

**跨学科联系**：
- **物理学**：量子力学的实在观
- **认知科学**：感知与实在
- **虚拟现实**：模拟的实在
- **心理学**：现实检验

**中国视角**：
- **庄子**：梦蝶与实在
- **佛家**：如梦幻泡影（金刚经）
- **王阳明**：心外无物
- **当代**：金岳霖的实在论

**争议问题**：
1. 我们能认识"实在本身"吗？
2. 量子力学是否证伪实在论？
3. 虚拟实在是否"实在"？

**相关词条**：存在、现象、本体、表象

### 【本体论范畴】(Ontological Categories)

**定义**：存在的最基本类型，用来对一切存在物进行分类的最高层级概念。

**费曼解释**：
想象一个超级分类系统：
- 生物界：界、门、纲、目、科、属、种
- 存在界：？？？

本体论范畴就是"存在界"的最高分类：
- 实体、性质、关系、事件、过程...

**亚里士多德的十范畴**：

| 范畴 | 定义 | 例子 |
|------|------|------|
| 实体 (Substance) | 独立存在 | 苏格拉底、马 |
| 数量 (Quantity) | 多少 | 2米、3公斤 |
| 性质 (Quality) | 怎样 | 白色、智慧 |
| 关系 (Relation) | 与他物的关系 | 父亲、两倍 |
| 地点 (Place) | 在哪里 | 在市场 |
| 时间 (Time) | 何时 | 昨天 |
| 姿态 (Position) | 如何安置 | 躺着 |
| 状态 (State) | 有什么 | 有鞋子 |
| 动作 (Action) | 做什么 | 切割 |
| 被动 (Passion) | 承受什么 | 被切 |

**当代范畴系统**：

| 哲学家 | 范畴系统 | 特点 |
|--------|----------|------|
| **亚里士多德** | 十范畴 | 古典系统 |
| **康德** | 十二范畴 | 认知形式 |
| **怀特海** | 过程范畴 | 过程哲学 |
| **Ingarden** | 存在模式 | 现象学 |
| **Lowe** | 四范畴系统 | 当代形而上学 |

**Lowe的四范畴 (2006)**：

```
         对象 (Objects)
        /              \
    实体              模式
(Substances)       (Modes)
      \                /
         属性 (Attributes)
            |
         种类 (Kinds)
```

**范畴争论**：

| 问题 | 立场A | 立场B |
|------|-------|-------|
| 范畴是发现的还是发明的？ | 发现（客观） | 发明（概念框架） |
| 有唯一正确的范畴系统吗？ | 是 | 否，多元论 |
| 范畴是实在的反映吗？ | 是（实在论） | 否（约定论） |

**当代发展 (2020-2026)**：
- **形式本体论 (Formal Ontology)**：用逻辑方法研究范畴
- **应用本体论**：信息科学中的本体工程
- **神经哲学**：范畴的认知基础
- **跨文化范畴**：不同文化的本体论差异

**跨学科联系**：
- **计算机科学**：本体工程、OWL、RDF
- **认知语言学**：Lakoff的理想化认知模型
- **发展心理学**：儿童如何习得范畴
- **人类学**：跨文化范畴系统

**中国视角**：
- **五行**：金木水火土（中国的"范畴"？）
- **阴阳**：二元范畴
- **理气**：朱熹的形而上学范畴
- **体用**：中国哲学的基本范畴

**争议问题**：
1. 是否有普遍的本体论范畴？
2. 范畴是语言依赖的吗？
3. 科学发现新范畴吗？

**相关词条**：本体论、实体、性质、关系

### 【实体】(Substance)

**词源**：拉丁语 *substantia* (站在下面)

**定义**：能够独立存在、支撑性质、是变化的基底的东西。

**费曼解释**：
想象一个苹果：
- 它是红色的（性质）
- 它是圆的（性质）
- 它是甜的（性质）
- 它很重（性质）

但是：什么是"它"？
- 是性质的集合？
- 还是性质依附的某个"东西"？

实体就是那个"东西"——性质依附其上的基底。

**实体理论的历史演变**：

| 时期 | 理论 | 代表人物 |
|------|------|----------|
| **古希腊** | 实体是形式+质料 | 亚里士多德 |
| **中世纪** | 实体有本质 | 阿奎那 |
| **近代** | 实体是支撑性质的基底 | 洛克 |
| **近代** | 实体是能思/广延的东西 | 笛卡尔 |
| **当代** | 取消实体，只保留性质 | 罗素、斯特劳森 |

**三种实体观**：

| 观点 | 核心主张 | 问题 |
|------|----------|------|
| **基底论 (Substratum Theory)** | 实体是性质的赤裸载体 | 载体无性质，如何理解？ |
| **束理论 (Bundle Theory)** | 实体只是性质的集合 | 性质如何捆绑？ |
| **实体论 (Substance Theory)** | 实体是基本，性质从属 | 实体是什么？ |

**基底 (Substratum) 理论**：

```
实体 = 基底 + 性质
      (载体)   (依附于载体)
```

**问题**：
- 基底没有任何性质？（"赤裸的"）
- 这不是自相矛盾吗？

**束理论 (Bundle Theory)**：

```
实体 = 性质的集合
苹果 = {红色, 圆形, 甜味, ...}
```

**问题**：
- 是什么使得这些性质"捆绑"在一起？
- 两个东西有完全相同的性质怎么办？

**当代发展 (2020-2026)**：
- **新实体论**：实体是性质束的"粘合剂"
- **功能实体论**：实体是功能角色
- **量子实体**：粒子是实体还是场的激发？
- **社会实体**：公司、国家是实体吗？

**跨学科联系**：
- **物理学**：基本粒子是实体吗？
- **化学**：分子是实体吗？
- **生物学**：有机体是实体吗？
- **计算机科学**：面向对象编程中的"对象"

**中国视角**：
- **气**：中国哲学的"实体"？（张载）
- **理**：朱熹的形而上学实体
- **体用**：实体与功能的区分
- **当代**：冯友兰的"理"与"气"

**争议问题**：
1. 实体存在吗？还是只是性质的集合？
2. 如果实体存在，它是什么？
3. 基本粒子是实体吗？

**相关词条**：本体论、性质、基底、束理论

### 【性质】(Property)

**定义**：事物具有的特征、属性或特质。

**费曼解释**：
苹果的"红色"、人的"智慧"、三角形的"三边性"都是性质。
问题：
- 性质是真实的吗？
- 性质存在于哪里？
- 多个东西可以"共享"同一个性质吗？

**性质的类型**：

| 类型 | 定义 | 例子 |
|------|------|------|
| **第一性质** | 对象本身的性质 | 形状、大小 |
| **第二性质** | 依赖于感知者的性质 | 颜色、味道 |
| **本质性质** | 必要的性质 | 三角形有三边 |
| **偶然性质** | 可能改变的性质 | 桌子是红色的 |
| **倾向性质** | 倾向性行为 | 易碎、可溶 |

**性质理论**：

| 理论 | 核心主张 | 问题 |
|------|----------|------|
| **实在论** | 性质（共相）真实存在 | 存在太多抽象实体？ |
| **唯名论** | 性质只是名称 | 如何解释相似性？ |
| **概念论** | 性质是心灵概念 | 客观性如何保证？ |
| **倾向论** | 性质是倾向性 | 倾向性如何理解？ |

**性质的语义学**：

| 分析 | 例子 |
|------|------|
| **一元谓词** | "是红色的" |
| **集合** | 红色事物集合 {苹果, 玫瑰, ...} |
| **函数** | 颜色函数：对象 → 颜色值 |

**性质与共相**：

问题：多个红色物体如何"共享"同一个"红色"？

| 回答 | 理论 |
|------|------|
| 它们都"分有"同一个红色 | 柏拉图实在论 |
| 它们的"红性"是共相的实例 | 亚里士多德实在论 |
| 它们只是相似，没有共享的东西 | 唯名论 |

**当代发展 (2020-2026)**：
- **倾向性 (Dispositions)**：倾向性质的形而上学
- **性质实在论 vs 唯名论**：持续争论
- **量子性质**：不确定性是性质吗？
- **认知科学**：性质如何被表征

**跨学科联系**：
- **语言学**：形容词与性质
- **认知科学**：概念表征
- **计算机科学**：属性-值对
- **物理学**：物理性质

**中国视角**：
- **性**：中国哲学中的"性质"概念
- **理**：朱熹：理即性
- **气**：张载：气质之性

**争议问题**：
1. 性质是共相还是殊相？
2. 性质可以独立于对象存在吗？
3. 所有的性质都是倾向性吗？

**相关词条**：共相、实体、本质、偶然

### 【本质】(Essence)

**词源**：拉丁语 *essentia* (是/存在)

**定义**：使事物成为其所是的东西，失去它事物就不再是该事物的性质。

**费曼解释**：
- 三角形的本质是"有三条边"——没有三条边就不是三角形
- 人的本质是...？（亚里士多德：理性动物）
- 你的本质是什么？什么东西失去后，你就不是你了？

**本质主义**：

| 类型 | 核心主张 | 例子 |
|------|----------|------|
| **传统本质主义** | 每个种类都有本质 | 人=理性动物 |
| **个体本质主义** | 每个个体有本质 | 苏格拉底的本质 |
| **温和本质主义** | 某些事物有本质 | 自然种类 |
| **反本质主义** | 没有本质 | Quine |

**本质与偶然**：

| 类型 | 定义 | 例子 |
|------|------|------|
| **本质性质** | 必要的性质 | 三角形有三边 |
| **偶然性质** | 可能改变的性质 | 这只天鹅是白色的 |

**克里普克 (Kripke) 的贡献**：

**后验必然性**：
- "水=H₂O"是偶然的吗？不！
- 我们通过经验发现水的本质
- 但一旦发现，就是必然的

**形而上学本质主义**：
- 自然种类（水、金、老虎）有本质
- 本质是内在的、科学的
- 不是语言的或概念的

**本质主义争论**：

| 问题 | 本质主义 | 反本质主义 |
|------|----------|------------|
| 种类有本质吗？ | 有（内在性质） | 无（只是相似性） |
| 本质是什么？ | 科学的微观结构 | 不存在 |
| 分类是发现还是发明？ | 发现 | 发明 |

**代表人物**：
- **亚里士多德**：本质主义创始人
- **Locke**：区分本质/名义本质
- **Kripke**：形而上学本质主义
- **反本质主义**：Quine, Wittgenstein

**当代发展 (2020-2026)**：
- **生物本质主义争议**：物种有本质吗？
- **基因本质主义**：基因是生命本质吗？
- **社会本质主义**：种族、性别有本质吗？
- **认知本质主义**：概念本质主义

**跨学科联系**：
- **生物学**：物种本质主义
- **认知科学**：心理本质主义
- **性别研究**：社会建构 vs 本质
- **法律**：权利的本质

**中国视角**：
- **性**：孟子性善论 vs 荀子性恶论
- **理**：朱熹：理即性，性即理
- **情**：性与情的关系
- **当代**：新儒家的本性论

**争议问题**：
1. 自然种类有本质吗？
2. 个体有本质吗？
3. 社会种类（种族、性别）有本质吗？

**相关词条**：性质、共相、种类、本质主义

### 【本质主义】(Essentialism)

**定义**：认为事物、种类或个体具有固定不变的本质属性的哲学观点。

**费曼解释**：
本质主义说：
- 每种东西都有一个"核心定义"
- 知道这个核心，你就完全理解了它
- 就像：老虎的本质是什么？DNA+某种生物学特征

反本质主义说：
- 没有固定的核心
- 只有家族相似性（维特根斯坦）
- "游戏"没有共同本质，只有相似性网络

**本质主义的类型**：

| 类型 | 对象 | 核心主张 | 代表 |
|------|------|----------|------|
| **种类本质主义** | 自然种类 | 物种有本质结构 | Kripke, Putnam |
| **个体本质主义** | 个体 | 个体有起源本质 | Kripke |
| **生物本质主义** | 生物 | 物种有基因本质 | 传统生物学 |
| **社会本质主义** | 社会群体 | 种族/性别有本质 | 有争议 |
| **心理本质主义** | 心理 | 人天然倾向于本质主义 | 认知科学 |

**本质主义的历史**：

| 时期 | 理论 | 人物 |
|------|------|------|
| **古代** | 理念论 | 柏拉图 |
| **古代** | 本质/偶性区分 | 亚里士多德 |
| **中世纪** | 本质主义确立 | 经院哲学 |
| **近代** | 实在本质vs名义本质 | Locke |
| **20世纪** | 反本质主义 | Quine, Wittgenstein |
| **当代** | 形而上学本质主义 | Kripke, Putnam |

**Quine的反本质主义**：

1. **同一性标准**：没有同一性就没有实体
2. **模糊边界**：物种边界模糊，没有明确本质
3. **变异**：变异是进化的基础，没有固定本质

**Kripke的辩护**：

1. **后验必然**："水=H₂O"是必然的，虽然后验
2. **内在结构**：微观结构是本质
3. **起源本质**：个体起源于特定配子

**当代发展 (2020-2026)**：
- **生物反本质主义**：物种是历史个体（Hull, Ghiselin）
- **社会建构论**：种族、性别是社会建构
- **认知本质主义**：人类天然是本质主义者
- **基因本质主义批判**：基因不是生命本质

**跨学科联系**：
- **生物学**：物种本质主义争议
- **心理学**：心理本质主义（Gelman）
- **性别研究**：社会建构论
- **认知科学**：概念的心理学

**中国视角**：
- **性善/性恶**：孟荀之争
- **天命之性**：朱熹
- **气质之性**：张载
- **当代**：牟宗三的本质观

**争议问题**：
1. 本质主义是科学还是偏见？
2. 生物物种有本质吗？
3. 社会种类有本质吗？

**相关词条**：本质、共相、种类、自然种类

### 【偶性/偶然性】(Accident/Contingency)

**定义**：事物的非本质属性，失去它们事物仍然是其所是。

**费曼解释**：
- 你是"理性的"（本质？）
- 你是"高个子"（偶然）
- 你是"黑头发"（偶然）
- 你可以改变身高或发色，但仍然是你

偶然性质就像"服装"，本质性质是"身体"。

**本质 vs 偶然**：

| 特征 | 本质性质 | 偶然性质 |
|------|----------|----------|
| 可失去性 | 不可失去 | 可失去 |
| 变化 | 变化=不再存在 | 变化=仍然存在 |
| 必然性 | 必然的 | 偶然的 |
| 定义 | 进入定义 | 不进入定义 |

**模态意义上的偶然**：

| 概念 | 定义 | 例子 |
|------|------|------|
| **必然** | 在所有可能世界为真 | 2+2=4 |
| **可能** | 在至少一个可能世界为真 | 这张桌子可能是绿色的 |
| **偶然** | 在某些世界为真，某些为假 | 这张桌子是棕色的 |

**偶性理论**：

| 理论 | 核心主张 | 问题 |
|------|----------|------|
| **偶性实在论** | 偶性是真实的性质 | 如何与实体相关？ |
| **偶性束理论** | 偶性构成实体 | 本质vs偶性区分？ |
| **偶性取消论** | 偶性是语言现象 | 如何解释变化？ |

**当代发展 (2020-2026)**：
- **模态偶然**：偶然性与可能世界
- **因果偶然**：因果关系是偶然的吗？
- **自然律的偶然**：物理定律是必然还是偶然？
- **存在的偶然**：为什么有东西而不是虚无？

**跨学科联系**：
- **逻辑学**：模态逻辑
- **科学哲学**：自然律的必然性
- **神学**：世界的偶然性（莱布尼茨）
- **文学**：偶然与命运

**中国视角**：
- **命**：命运与偶然
- **遇**：际遇与偶然
- **势**：形势与偶然
- **当代**：命运与偶然的哲学

**争议问题**：
1. 本质/偶然的区分是客观的吗？
2. 所有性质都是偶然的吗？
3. 自然律是必然还是偶然？

**相关词条**：本质、模态、必然性、可能性

### 【基础/根基】(Grounding)

**定义**：一种形而上学关系，解释为什么某些事实或实体依赖于其他事实或实体。

**费曼解释**：
- 原子构成了分子（分子"基于"原子）
- 规则导致比赛结果（结果"基于"规则）
- 基础关系问：什么"支撑"什么？

就像建筑：
- 地基支撑楼房
- 楼房基于地基存在

**基础的形式**：

| 概念 | 关系 | 例子 |
|------|------|------|
| **构成基础** | 部分构成整体 | 原子→分子 |
| **因果基础** | 原因导致结果 | 推→倒 |
| **逻辑基础** | 前提蕴含结论 | A∧B→A |
| **形而上学基础** | 更基本的支撑 | 物理→心理？ |

**基础 vs 其他关系**：

| 关系 | 类型 | 方向 | 例子 |
|------|------|------|------|
| **因果** | 时间性 | 原因→结果 | 推→倒 |
| **构成** | 空间性 | 部分→整体 | 砖→墙 |
| **基础** | 形而上学 | 更基本→更不基本 | 物理→化学 |

**基础的核心特征**：

1. **方向性**：A基础B ≠ B基础A
2. **传递性**：A基础B，B基础C → A基础C
3. **非对称性**：A基础B → B不基础A
4. **不可逆性**：基础的方向固定

**代表人物**：
- **Kit Fine**：基础的逻辑研究
- **Jonathan Schaffer**：基础一元论
- **Gideon Rosen**：基础与还原

**当代发展 (2020-2026)**：
- **基础逻辑**：基础的形式化
- **基础一元论**：Schaffer：只有一种根本实体
- **基础与科学**：科学理论的基础结构
- **社会基础**：社会事实如何基于个人事实

**跨学科联系**：
- **逻辑学**：基础逻辑
- **科学哲学**：理论还原
- **心灵哲学**：心理/物理基础
- **数学哲学**：数学基础

**中国视角**：
- **本末**：本为基础，末为衍生
- **体用**：体为基础，用为功能
- **道器**：道为形而上基础
- **当代**：基础问题的中国视角

**争议问题**：
1. 基础是客观关系吗？
2. 基础链有终点吗？
3. 基础与因果有何区别？

**相关词条**：本体论、因果性、构成、还原

### 【依赖性】(Dependence)

**定义**：一个实体的存在或性质依赖于另一个实体的关系。

**费曼解释**：
- 微笑依赖于脸（没有脸就没有微笑）
- 影子依赖于物体（没有物体就没有影子）
- 你依赖于父母（你的存在依赖于他们的存在）

**依赖的类型**：

| 类型 | 定义 | 例子 |
|------|------|------|
| **存在依赖** | A存在需要B存在 | 微笑依赖脸 |
| **身份依赖** | A的身份由B决定 | 丈夫依赖妻子 |
| **因果依赖** | A的存在由B导致 | 孩子依赖父母 |
| **概念依赖** | A的概念需要B的概念 | 父母依赖子女 |

**依赖 vs 基础**：

| 特征 | 依赖 | 基础 |
|------|------|------|
| 方向 | 依赖的 → 被依赖的 | 更基本 → 更不基本 |
| 强度 | 可以是对称的 | 总是非对称的 |
| 应用 | 一般关系 | 形而上学解释 |

**存在依赖的模式**：

| 模式 | 公式 | 例子 |
|------|------|------|
| **刚性依赖** | □(∃x → ∃y) | 微笑依赖脸 |
| **一般依赖** | ◇(∃x → ∃y) | 父母依赖某个孩子 |

**当代发展 (2020-2026)**：
- **本体论依赖**：实体的存在依赖
- **概念依赖**：概念的关系
- **社会依赖**：社会实体的依赖
- **模态依赖**：可能世界的依赖

**跨学科联系**：
- **逻辑学**：依赖逻辑
- **计算机科学**：依赖图
- **经济学**：经济依赖
- **社会学**：社会依赖

**中国视角**：
- **缘起**：佛教的相互依赖
- **关系**：儒家的人际依赖
- **天人合一**：人与自然的依赖
- **当代**：依赖哲学的中国视角

**争议问题**：
1. 是否有完全不依赖他物的东西？
2. 依赖是客观关系吗？
3. 万物互依还是某些东西独立？

**相关词条**：基础、关系、整体论、还原

### 【存在量化】(Existential Quantification)

**定义**：逻辑中表示"存在"的量词，符号为 ∃，表示"至少存在一个"。

**费曼解释**：
∃x P(x) 读作："存在某个x，使得P(x)为真"
- ∃x (x是猫) = 有猫存在
- ∃x (x是中国人) = 有中国人存在

**Quine的名言**："存在就是成为约束变元的值"

**相关词条**：存在、量词、逻辑

### 【虚构对象】(Fictional Objects)

**定义**：文学、艺术作品中想象出来的对象，如哈利·波特、福尔摩斯。

**费曼解释**：
哈利·波特存在吗？
- 不存在于现实世界 ✗
- 但我们可以说他"有黑头发" ✓
- 他是什么类型的存在？

**虚构对象的争论**：

| 立场 | 主张 | 代表 |
|------|------|------|
| **迈农主义** | 虚构对象存在（但不实在） | Meinong |
| **实在论** | 虚构对象是抽象实体 | Thomasson |
| **消除论** | 虚构对象不存在 | Russell |
| **假装理论** | 我们只是"假装"它们存在 | Walton |

**当代发展 (2020-2026)**：
- **数字虚构**：游戏、元宇宙中的对象
- **AI生成虚构**：AI创造的虚构人物
- **虚拟偶像**：数字虚构的新形式

**相关词条**：存在、抽象对象、可能世界

### 【四维论】(Four-Dimensionalism)

**定义**：认为物体是四维时空中的"虫"，在时间中延伸，有"时间部分"。

**费曼解释**：
想象你的一生是一条长蛇：
- 头：出生
- 身体：一生中的你
- 尾：死亡
- 你不只是在空间中延伸，也在时间中延伸

**三维论 vs 四维论**：

| 特征 | 三维论 | 四维论 |
|------|--------|--------|
| 存在模式 | 完全存在于每个时刻 | 在时间中延伸 |
| 持存方式 | 持续同一 | 历时部分 |
| 时间部分 | 否认 | 承认 |
| 类比 | 点在时间移动 | 虫在时空 |

**代表人物**：
- **David Lewis**：四维论的支持者
- **Theodore Sider**：《Four-Dimensionalism》

**当代发展 (2020-2026)**：
- **时空物理学**：相对论支持四维观
- **人格同一性**：四维论的应用
- **时间旅行**：四维论的后果

**相关词条**：时间、持存、同一性
