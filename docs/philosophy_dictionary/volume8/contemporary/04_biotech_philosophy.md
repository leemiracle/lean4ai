### 【生物技术哲学】(Philosophy of Biotechnology)

**词源**：希腊语 *bios* (生命) + *techne* (技术) + *philosophia* (哲学)

**定义**：研究与生物技术相关的哲学问题的交叉学科。

**费曼解释**：
生物技术哲学问：
- 我们应该编辑人类基因吗？
- 人类增强是道德的吗？
- 生物技术会改变人类本质吗？

**生物技术哲学的核心问题**：

| 问题 | 描述 |
|------|------|
| **伦理问题** | 生物技术是道德的吗？ |
| **身份问题** | 生物技术改变人的身份？ |
| **正义问题** | 生物技术如何公平分配？ |
| **安全问题** | 生物技术安全吗？ |

**主要研究领域**：

| 领域 | 描述 |
|------|------|
| **基因编辑** | CRISPR等技术伦理 |
| **人类增强** | 增强人类能力伦理 |
| **合成生物学** | 合成生命伦理 |
| **生物信息** | 生物数据伦理 |
| **神经技术** | 脑机接口伦理 |

**代表人物**：

### 【基因编辑】(Gene Editing)

**定义**：精确修改生物基因组的技术。

**费曼解释**：
基因编辑像文字处理：
- 找到特定基因
- 删除、添加或修改
- CRISPR让这变得容易

**基因编辑技术**：

| 技术 | 描述 |
|------|------|
| **CRISPR-Cas9** | 最常用的基因编辑技术 |
| **碱基编辑** | 精确编辑单个碱基 |
| **先导编辑** | 更精确的编辑 |
| **TALEN** | 早期编辑技术 |

| 领域 | 应用 |

| **农业** | 改良作物 |
