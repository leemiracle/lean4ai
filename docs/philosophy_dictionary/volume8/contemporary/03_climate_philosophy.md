### 【气候哲学】(Philosophy of Climate Change)

**词源**：希腊语 *klima* (气候) + *philosophia* (哲学)

**定义**：研究气候变化相关哲学问题的交叉学科。

**费曼解释**：
气候哲学问：
- 气候变化是真的吗？我们怎么知道？

### 【代际伦理】(Intergenerational Ethics)

**定义**：当代人与后代之间的伦理关系。

**费曼解释**：
我们今天的行为影响后代：
- 气候变化
- 核废料
- 资源耗竭

我们对后代有什么义务？

**代际伦理的核心问题**：

| 问题 | 描述 |
|------|------|
| **后代存在吗？** | 后代尚未存在 |
| **后代有权利吗？** | 不存在者能有权？ |
| **我们有什么义务？** | 对后代有什么义务？ |
| **如何权衡？** | 当代vs后代利益 |

**后代的问题**：

**问题1：非存在性**
- 后代尚未存在
- 不存在者能有权利吗？

**问题2：人数问题**
- 我们决定生育多少人
- 这影响谁存在
- 如何比较不同人群的福利？

**问题3：不确定性**
- 我们不知道后代会面对什么
- 如何为不确定的未来规划？

**代际伦理理论**：

| 理论 | 描述 |
|------|------|
| **权利论** | 后代有权利 |
| **义务论** | 我们对后代有义务 |
| **功利主义** | 应该考虑后代福利 |
| **契约论** | 假设的代际契约 |
| **信任论** | 我们是地球的受托人 |

**折扣问题**：

| 问题 | 描述 |
|------|------|
| **未来折扣** | 未来价值应该折扣吗？ |
| **纯时间偏好** | 仅仅因为时间就更低？ |
| **经济折扣** | 经济效率要求折扣？ |

**立场**：

| 立场 | 描述 |
|------|------|
| **支持折扣** | 经济效率、不确定性 |
| **反对折扣** | 时间中立、道德平等 |
| **温和折扣** | 有限度折扣 |

**代表人物**：

- **Parfit**：人数问题
- **当代**：代际伦理学者

**经典文献**：

### 【预防原则】(Precautionary Principle)

**定义**：在面临严重或不可逆伤害风险时应采取预防措施的原则。

**费曼解释**：
预防原则说：
- 如果可能有严重伤害
- 即使科学不完全确定
- 也应该采取预防措施

**预防原则的核心要素**：

| 要素 | 描述 |
|------|------|
| **潜在伤害** | 有严重或不可逆伤害的可能 |
| **科学不确定性** | 科学认识不完全确定 |
| **预防行动** | 应该采取预防措施 |
| **举证责任** | 举证责任转移 |

**预防原则的表述**：

| 表述 | 描述 |
|------|------|
| **弱版本** | 不确定性不阻止行动 |
| **强版本** | 举证责任在行动者 |
| **中间版本** | 平衡风险和收益 |

**预防原则的应用**：

| 领域 | 应用 |
|------|------|
| **气候变化** | 气候行动 |
| **公共卫生** | 公共卫生政策 |
| **环境** | 环境保护 |
| **新技术** | 新技术监管 |

**对预防原则的批评**：

| 批评 | 回应 |
|------|------|
| **过于保守** | 阻碍创新 | 平衡风险 |
| **定义模糊** | 不清楚什么是"严重" | 可以具体化 |
| **不可能应用** | 一切都有风险 | 关注严重风险 |

**预防与进步**：

|------|------|

| **风险评估** | 如何评估风险？ |
