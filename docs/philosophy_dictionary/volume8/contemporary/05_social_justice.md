### 【社会正义】(Social Justice)

**词源**：拉丁语 *societas* (社会) + *justitia* (正义)

**定义**：社会中利益和负担公平分配的原则。

**费曼解释**：
社会正义问：
- 社会的利益和负担如何分配？
- 不平等何时是不正义的？
| 如何实现公正社会？

**社会正义的核心问题**：

| 问题 | 描述 |
|------|------|
| **分配正义** | 如何分配社会资源？ |
| **程序正义** | 决策过程是否公正？ |
| **承认正义** | 是否承认所有群体？ |
| **矫正正义** | 如何纠正历史不正义？ |

**主要的正义理论**：

| 理论 | 描述 | 代表 |
|------|------|------|
| **平等主义** | 强调平等 | Rawls |
| **自由至上主义** | 强调自由 | Nozick |
| **能力理论** | 强调能力 | Sen, Nussbaum |
| **马克思主义** | 强调阶级 | Marx |

**分配原则**：

| 原则 | 描述 |
|------|------|
| **平等** | 平等分配 |
| **按需** | 按需要分配 |
| **按功** | 按贡献分配 |
| **优先性** | 优先最差者 |

**代表人物**：
- **Rawls**：《正义论》
- **Sen**：能力理论
- **Fraser**：承认理论
- **当代**：正义理论家

**经典文献**：
- Rawls, *A Theory of Justice* (1971)
- Sen, *The Idea of Justice* (2009)

**当代发展 (2020-2026)**：
- **BLM**：种族正义运动
- **#MeToo**：性别正义运动
- **气候正义**：气候分配正义
- **数字正义**：数字不平等

**跨学科联系**：
- **政治哲学**：正义理论
- **经济学**：分配经济学
- **社会学**：社会不平等
- **法学**：社会法

**中国视角**：
- **社会正义**：中国社会正义
- **共同富裕**：共同富裕目标
- **当代**：社会正义的中国视角

**争议问题**：
1. 什么是正义的社会？
2. 不平等何时是不正义的？
3. 如何实现社会正义？

**相关词条**：分配正义、平等、不平等

### 【经济不平等】(Economic Inequality)

**定义**：收入和财富分配的不平等。

**费曼解释**：
经济不平等：
- 有些人很富有
- 有些人很贫穷
| 这种差距越来越大

**经济不平等的测量**：

| 指标 | 描述 |
|------|------|
| **基尼系数** | 收入不平等指数 |
| **收入份额** | 最富有人群收入份额 |
| **财富差距** | 财富分配差距 |
| **代际流动性** | 代际收入流动 |

**经济不平等的趋势**：

| 趋势 | 描述 |
|------|------|
| **收入差距扩大** | 近几十年收入差距扩大 |
| **财富集中** | 财富越来越集中 |
| **全球不平等** | 国家间差距缩小，国内差距扩大 |

**经济不平等的原因**：

| 原因 | 描述 |
|------|------|
| **技术变革** | 技能偏向的技术进步 |
| **全球化** | 全球分工 |
| **政策** | 税收和福利政策 |
| **制度** | 劳动市场制度 |

**不平等是否不公正？**

| 观点 | 描述 |
|------|------|
| **不平等本身不公正** | 不平等本身就是问题 |
| **结果不平等不公正** | 机会不平等导致的问题 |
| **不平等可接受** | 一定不平等可接受 |
| **不平等是应得的** | 市场结果反映贡献 |

**应对不平等的政策**：

| 政策 | 描述 |
|------|------|
| **累进税** | 累进税收 |
| **社会保障** | 社会保障体系 |
| **教育** | 教育机会平等 |
| **最低工资** | 最低工资标准 |

**代表人物**：
- **Piketty**：《21世纪资本论》
- **Stiglitz**：《不平等的代价》
- **当代**：不平等研究者

**经典文献**：
- Piketty, *Capital in the Twenty-First Century* (2013)
- Stiglitz, *The Price of Inequality* (2012)

**当代发展 (2020-2026)**：

### 【交叉性】(Intersectionality)

**词源**：Intersect (交叉) + -ality (性质)

**定义**：多种社会身份交叉产生独特压迫体验的概念。

**费曼解释**：
交叉性说：
- 你不是只有一种身份
- 你同时是女性、黑人、工人等
- 这些身份交叉产生独特体验

**交叉性的核心概念**：

| 概念 | 描述 |
|------|------|
| **多重身份** | 每个人有多种社会身份 |

| **独特体验** | 交叉产生独特体验 |

| 身份 | 例子 |

**交叉性的应用**：

### 【分配正义】(Distributive Justice)

**定义**：社会利益和负担公平分配的原则。

**费曼解释**：
分配正义问：
- 社会的好东西怎么分？
- 谁应该得到什么？
- 什么是公平的分配？

**分配正义的原则**：

| 原则 | 描述 |
|------|------|
| **平等** | 平等分配 |
| **按需** | 按需要分配 |
| **按功** | 按贡献分配 |
| **按劳** | 按劳动分配 |
| **优先性** | 优先最差者 |

**主要的分配正义理论**：

| 理论 | 核心主张 | 代表 |
|------|----------|------|
| **严格平等主义** | 应该平等分配 | 有些平等主义者 |
| **自由至上主义** | 市场分配 | Nozick |
| **充足主义** | 确保人人足够 | Frankfurt |
| **优先主义** | 优先最差者 | Rawls, Parfit |
| **能力理论** | 扩展能力 | Sen, Nussbaum |

**Rawls的分配正义**：

| 原则 | 描述 |
|------|------|
| **自由原则** | 基本自由平等 |
| **差异原则** | 不平等应有利于最差者 |
| **公平机会** | 机会公平开放 |

**差异原则**：

**核心**：社会和经济不平等应该有利于最不利者

**解读**：
- 不是追求平等
- 而是允许有利于最差者的不平等
- 关注最差者的绝对地位

**Nozick的分配正义**：

| 原则 | 描述 |
|------|------|
| **获取正义** | 初始获取公正 |
| **转让正义** | 转让过程公正 |
| **矫正正义** | 纠正不正义获取 |

**核心**：只要过程公正，结果就公正

**代表人物**：
- **Rawls**：《正义论》
- **Nozick**：《无政府、国家和乌托邦》
- **Sen**：能力理论
- **当代**：分配正义理论家

**经典文献**：
- Rawls, *A Theory of Justice* (1971)
- Nozick, *Anarchy, State, and Utopia* (1974)

**当代发展 (2020-2026)**：
- **运气平等主义** | 运气与选择
- **全球分配正义** | 全球分配
- **代际分配** | 代际分配
- **AI分配** | AI收益分配

**跨学科联系**：
- **政治哲学** | 正义理论
- **经济学** | 分配经济学
- **法学** | 分配法
- **社会学** | 社会分层

**中国视角**：
- **分配正义** | 中国分配正义
- **共同富裕** | 共同富裕
- **当代** | 分配正义的中国视角

**争议问题**：
1. 什么是公平的分配？
