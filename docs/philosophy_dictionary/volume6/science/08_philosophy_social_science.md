### 【社会科学哲学】(Philosophy of Social Science)

**定义**：研究社会科学基础概念和方法的哲学分支。

**费曼解释**：
社会科学哲学问：
- 社会科学能像自然科学一样吗？
- 如何解释人的行为？
- 社会规律存在吗？

**核心问题**：

| 问题 | 描述 |
|------|------|
| **解释vs理解** | 解释还是理解？ |
| **方法论** | 社会科学用什么方法？ |
| **还原** | 社会科学能还原为心理学？ |
| **客观性** | 社会科学能客观吗？ |

**主要领域**：

| 领域 | 描述 |
|------|------|
| **方法论** | 社会科学的方法问题 |
| **解释** | 社会科学解释的本质 |
| **因果关系** | 社会因果关系 |
| **价值中立** | 价值中立问题 |

**相关词条**：解释vs理解、方法论、社会科学

### 【解释vs理解】(Explanation vs Understanding)

**定义**：自然科学方法（解释）与人文科学方法（理解）的区别。

**费曼解释**：
解释vs理解是"两种方法"：
- **解释**（Erklären）：自然科学，找因果规律
- **理解**（Verstehen）：人文科学，理解意义和意图

**两种方法的区别**：

| 问题 | 解释（自然科学） | 理解（人文科学） |
|------|------------------|------------------|
| **目标** | 发现因果规律 | 理解意义 |
| **方法** | 观察、实验 | 解释学、共情 |
| **对象** | 自然现象 | 人类行为和意义 |

**狄尔泰的区别**：

**核心主张**：
- 自然科学：解释（Erklären）
- 精神科学：理解（Verstehen）
- 理解需要"内部视角"

**理解的方法**：

| 方法 | 描述 |
|------|------|
| **共情** | 设身处地 |
| **解释学** | 解释文本和意义 |
| **意向性** | 理解意图 |

**争论**：

| 立场 | 观点 |
|------|------|
| **统一论** | 社会科学应该用自然科学方法 |
| **分离论** | 社会科学有独特方法 |
| **互补论** | 两种方法互补 |

**相关词条**：狄尔泰、方法论、社会科学

**生平**：1833-1911，德国哲学家

**贡献**：解释学、精神科学方法论

**主要理论**：

| 理论 | 描述 |
|------|------|
| **解释vs理解** | 自然科学与精神科学的区别 |
| **理解** | 理解是人类科学的方法 |
| **生命哲学** | 生命哲学 |

**核心主张**：

| 主张 | 描述 |
|------|------|
| **精神科学** | 精神科学有独特方法 |
| **理解** | 理解是核心方法 |
| **生命表达** | 研究生命表达 |

**经典文献**：
- Dilthey, *Introduction to the Human Sciences* (1883)

**相关词条**：解释vs理解、解释学、社会科学

### 【解释学】(Hermeneutics)

**定义**：解释文本和意义的理论与方法。

**费曼解释**：
解释学是"理解的艺术"：
- 如何理解文本？
- 如何理解他人？
- 理解的方法是什么？

**核心概念**：

| 概念 | 描述 |
|------|------|
| **解释循环** | 部分与整体的循环 |
| **前理解** | 解释前的预设 |
| **视域融合** | 解释者与文本的视域融合 |

**解释循环**：

**描述**：
- 要理解整体，需要理解部分
- 要理解部分，需要理解整体
- 解释是循环的

**在社会科学中的应用**：

| 应用 | 描述 |
|------|------|
| **理解文化** | 理解文化意义 |
| **理解行为** | 理解行为意图 |
| **理解历史** | 理解历史意义 |

**代表人物**：
- **狄尔泰**：解释学发展
- **伽达默尔**：哲学解释学

**相关词条**：理解、狄尔泰、伽达默尔

### 【韦伯】(Max Weber)

**生平**：1864-1920，德国社会学家

**贡献**：理解社会学、理性化、权威类型

**主要理论**：

| 理论 | 描述 |
|------|------|
| **理解社会学** | 理解社会行动的意义 |
| **理性化** | 现代社会的理性化 |
| **权威类型** | 传统、魅力、法理权威 |
| **价值中立** | 社会科学应该价值中立 |

**理解社会学**：

**核心主张**：
- 社会学应该理解社会行动的意义
- 关注行动者的主观意义
- 使用"理想类型"

**理性化**：

| 方面 | 描述 |
|------|------|
| **工具理性** | 计算效率 |
| **科层制** | 科层组织 |
| **去魅** | 世界去魅 |

**权威类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **传统** | 习俗和传统 | 世袭君主 |
| **魅力** | 个人魅力 | 革命领袖 |
| **法理** | 法律规则 | 现代国家 |

**经典文献**：
- Weber, *Economy and Society* (1922)
- Weber, *The Protestant Ethic and the Spirit of Capitalism* (1905)

**相关词条**：理解、价值中立、理性化

### 【理解】(Verstehen)

**定义**：韦伯提出的理解社会行动主观意义的方法。

**费曼解释**：
理解是"进入他人心中"：
- 理解他人为什么这样做
- 理解行为的主观意义
- 这是社会科学的核心方法

**核心主张**：

| 主张 | 描述 |
|------|------|
| **主观意义** | 理解行动的主观意义 |
| **动机** | 理解行动者的动机 |
| **内部视角** | 从内部视角理解 |

**类型**：

| 类型 | 描述 |
|------|------|
| **直接理解** | 直接观察理解 |
| **解释性理解** | 解释动机和意义 |

**方法**：

| 方法 | 描述 |
|------|------|
| **共情** | 设身处地 |
| **理想类型** | 使用理想类型 |
| **历史比较** | 历史比较方法 |

**相关词条**：韦伯、解释学、社会科学方法

### 【理想类型】(Ideal Type)

**定义**：韦伯提出的概念工具，强调某些特征的理想化模型。

**费曼解释**：
理想类型是"理想化的模型"：
- 不是描述现实
- 而是强调某些特征
- 用于比较和分析

**特征**：

| 特征 | 描述 |
|------|------|
| **理想化** | 强调某些特征 |
| **分析工具** | 用于分析 |
| **比较标准** | 用于比较 |

**例子**：

| 理想类型 | 描述 |
|----------|------|
| **科层制** | 理想的科层组织 |
| **资本主义** | 理想的资本主义 |
| **权威** | 三种权威类型 |

**作用**：

| 作用 | 描述 |
|------|------|
| **分析** | 分析社会现象 |
| **比较** | 比较不同社会 |
| **测量** | 测量与理想的偏差 |

**相关词条**：韦伯、概念形成、社会科学方法

### 【价值中立】(Value Neutrality)

**定义**：社会科学应该避免价值判断的主张。

**费曼解释**：
价值中立说：
- 社会科学应该是中立的
- 科学家不应该把自己的价值观强加
- 区分事实和价值

**韦伯的主张**：

| 主张 | 描述 |
|------|------|
| **事实vs价值** | 区分事实判断和价值判断 |
| **教学中立** | 教学应该中立 |
| **研究客观** | 研究应该客观 |

**问题**：

| 问题 | 描述 |
|------|------|
| **可能吗？** | 价值中立可能吗？ |
| **可取吗？** | 价值中立可取吗？ |
| **价值选择** | 研究主题选择涉及价值 |

**批评**：

| 批评 | 描述 |
|------|------|
| **不可能** | 不可能完全价值中立 |
| **不道德** | 某些情况下不道德 |
| **伪装** | 可能伪装价值判断 |

**相关词条**：韦伯、社会科学客观性、事实vs价值

### 【方法论个人主义】(Methodological Individualism)

**定义**：认为社会现象应该还原为个人行为的观点。

**费曼解释**：
方法论个人主义说：
- 社会由个人组成
- 要理解社会，就要理解个人
- 社会现象可以从个人行为解释

**核心主张**：

| 主张 | 描述 |
|------|------|
| **个人是基础** | 个人是社会的基础 |
| **还原解释** | 社会现象还原为个人行为 |
| **拒绝整体主义** | 拒绝社会整体实体 |

**vs 方法论整体主义**：

| 问题 | 个人主义 | 整体主义 |
|------|----------|----------|
| **分析单位** | 个人 | 社会整体 |
| **解释** | 从个人解释社会 | 从社会解释个人 |
| **本体论** | 只有个人存在 | 社会实体存在 |

**支持理由**：

| 理由 | 描述 |
|------|------|
| **个人行动** | 只有个人行动 |
| **避免神秘** | 避免社会神秘实体 |
| **清晰** | 更清晰的解释 |

**批评**：

| 批评 | 描述 |
|------|------|
| **忽视社会结构** | 忽视社会结构的影响 |
| **过度简化** | 过度简化社会现象 |
| **涌现** | 社会有涌现性质 |

**相关词条**：还原论、整体主义、社会本体论

### 【方法论整体主义】(Methodological Holism)

**定义**：认为社会整体有不能还原为个人的性质的观点。

**费曼解释**：
方法论整体主义说：
- 社会不只是一群人
- 社会结构、制度有自己的因果力
- 不能只从个人理解社会

**核心主张**：

| 主张 | 描述 |
|------|------|
| **整体存在** | 社会整体存在 |
| **不可还原** | 社会性质不可还原 |
| **结构因果** | 社会结构有因果力 |

**支持理由**：

| 理由 | 描述 |
|------|------|
| **涌现性质** | 社会有涌现性质 |
| **结构约束** | 社会结构约束个人 |
| **不可预测** | 不能从个人预测社会 |

**例子**：

| 社会现象 | 整体主义解释 |
|----------|-------------|
| **经济周期** | 不能从个人行为预测 |
| **社会规范** | 规范约束个人 |
| **制度** | 制度有因果力 |

**相关词条**：方法论个人主义、社会结构、涌现

### 【社会规律】(Social Laws)

**定义**：描述社会现象规律性的一般性陈述。

**费曼解释**：
社会规律问：
- 社会有规律吗？
- 社会规律像自然规律吗？
- 社会科学能发现规律吗？

**问题**：

| 问题 | 描述 |
|------|------|
| **存在性** | 社会规律存在吗？ |
| **普遍性** | 社会规律普遍吗？ |
| **预测力** | 能预测社会吗？ |

**与自然规律的区别**：

| 问题 | 自然规律 | 社会规律 |
|------|----------|----------|
| **普遍性** | 普遍 | 历史特定 |
| **必然性** | 必然 | 可能有例外 |
| **稳定性** | 稳定 | 可能变化 |

**例子**：

| 可能的社会规律 | 描述 |
|----------------|------|
| **经济规律** | 供需规律 |
| **人口规律** | 人口转型 |
| **社会演化** | 社会演化规律 |

**争议**：

| 立场 | 观点 |
|------|------|
| **存在规律** | 社会规律存在 |
| **否认规律** | 没有真正的社会规律 |
| **弱规律** | 只有弱规律 |

**相关词条**：科学规律、社会科学、预测

### 【理性选择理论】(Rational Choice Theory)

**定义**：假设个人理性地最大化效用的理论。

**费曼解释**：
理性选择理论说：
- 人是理性的
- 人最大化自己的效用
- 可以用数学模型预测行为

**核心假设**：

| 假设 | 描述 |
|------|------|
| **理性** | 人是理性的 |
| **效用最大化** | 人最大化效用 |
| **偏好一致** | 偏好是一致的 |

**应用**：

| 领域 | 应用 |
|------|------|
| **经济学** | 消费者行为 |
| **政治学** | 投票行为 |
| **社会学** | 社会互动 |

**形式化**：

| 要素 | 描述 |
|------|------|
| **偏好** | 对结果的排序 |
| **信念** | 对可能性的估计 |
| **选择** | 选择最大化期望效用 |

**批评**：

| 批评 | 描述 |
|------|------|
| **有限理性** | 人的理性有限 |
| **非理性行为** | 人经常非理性 |
| **过度简化** | 过度简化人类行为 |

**相关词条**：经济学、理性、效用

### 【有限理性】(Bounded Rationality)

**定义**：人的理性受认知限制约束的观点。

**费曼解释**：
有限理性说：
- 人不是完全理性的
- 认知能力有限
- 信息不完整

**vs 完全理性**：

| 问题 | 完全理性 | 有限理性 |
|------|----------|----------|
| **信息** | 完全信息 | 不完全信息 |
| **计算** | 无限计算 | 有限计算 |
| **决策** | 最优决策 | 满意决策 |

**西蒙的贡献**：

| 概念 | 描述 |
|------|------|
| **满意原则** | 满意即可，不必最优 |
| **启发法** | 使用启发法决策 |
| **认知限制** | 认知能力有限 |

**启发法和偏见**：

| 启发法 | 描述 | 可能的偏见 |
|--------|------|-----------|
| **代表性** | 根据相似性判断 | 忽视基础率 |
| **可得性** | 根据容易想起判断 | 近期事件权重高 |
| **锚定** | 根据初始值调整 | 调整不充分 |

**代表人物**：
- **赫伯特·西蒙**：有限理性
- **卡尼曼和特沃斯基**：启发法和偏见

**相关词条**：理性选择、决策理论、认知偏见

### 【博弈论】(Game Theory)

**定义**：研究策略互动的数学理论。

**费曼解释**：
博弈论是"策略思维"：
- 你的选择影响我的结果
- 我的选择影响你的结果
- 如何做出最好的选择？

**核心概念**：

| 概念 | 描述 |
|------|------|
| **参与者** | 博弈的参与者 |
| **策略** | 可选的行动方案 |
| **收益** | 每种结果的收益 |
| **均衡** | 稳定的策略组合 |

**经典博弈**：

| 博弈 | 描述 | 寓意 |
|------|------|------|
| **囚徒困境** | 合作vs背叛 | 集体行动困境 |
| **公共地悲剧** | 共享资源 | 公共资源过度使用 |
| **协调博弈** | 协调选择 | 需要协调 |

**囚徒困境**：

| | 合作 | 背叛 |
|--|------|------|
| **合作** | 3,3 | 0,5 |
| **背叛** | 5,0 | 1,1 |

**问题**：
- 理性个体选择背叛
- 但集体最优是合作
- 个体理性导致集体非理性

**纳什均衡**：

**定义**：给定他人策略，没有人有动力单方面改变策略。

**相关词条**：理性选择、策略、均衡

### 【囚徒困境】(Prisoner's Dilemma)

**定义**：个体理性导致集体非理性的经典博弈。

**费曼解释**：
囚徒困境是"合作难题"：
- 两个人被捕
- 如果都沉默，各判1年
- 如果都揭发，各判5年
- 如果一个揭发一个沉默，揭发者释放，沉默者判10年

**收益矩阵**：

| | 沉默 | 揭发 |
|--|------|------|
| **沉默** | -1,-1 | -10,0 |
| **揭发** | 0,-10 | -5,-5 |

**困境**：
- 理性选择是揭发
- 但都沉默更好
- 个体理性≠集体理性

**解决方案**：

| 方案 | 描述 |
|------|------|
| **重复博弈** | 重复可以促进合作 |
| **声誉** | 声誉机制 |
| **惩罚** | 惩罚背叛者 |
| **规范** | 社会规范促进合作 |

**应用**：

| 领域 | 应用 |
|------|------|
| **国际关系** | 军备竞赛 |
| **环境** | 公共地悲剧 |
| **经济** | 卡特尔 |

**相关词条**：博弈论、合作、集体行动

### 【集体行动问题】(Collective Action Problem)

**定义**：个人利益与集体利益冲突的问题。

**费曼解释**：
集体行动问题是"合作难题"：
- 每个人都想搭便车
- 如果都不贡献，集体失败
- 如何解决搭便车？

**类型**：

| 类型 | 描述 |
|------|------|
| **公共物品** | 公共物品供给不足 |
| **公共地悲剧** | 公共资源过度使用 |
| **投票悖论** | 为什么投票？ |

**公共物品**：

**特征**：
- **非排他性**：不能排除他人使用
- **非竞争性**：一人使用不影响他人

**问题**：
- 每个人都想搭便车
- 公共物品供给不足

**解决方案**：

| 方案 | 描述 |
|------|------|
| **强制** | 政府强制（税收） |
| **激励** | 激励贡献 |
| **规范** | 社会规范 |
| **小群体** | 小群体更容易合作 |

**奥尔森的《集体行动的逻辑》**：

**核心主张**：
- 大群体难以组织集体行动
- 小群体更容易
- 需要选择性激励

**相关词条**：囚徒困境、公共物品、合作

### 【社会因果】(Social Causation)

**定义**：社会现象之间的因果关系。

**费曼解释**：
社会因果问：
- 社会现象有因果关系吗？
- 如何确定社会因果？
- 社会因果与物理因果有何不同？

**类型**：

| 类型 | 描述 |
|------|------|
| **制度因果** | 制度导致结果 |
| **结构因果** | 社会结构导致结果 |

### 【社会本体论】(Social Ontology)

**定义**：研究社会存在物本质的哲学分支。

**费曼解释**：
社会本体论问：
- 社会是什么？
- 制度、组织是真实的吗？
- 社会实体与物理实体有何不同？

**核心问题**：

| 问题 | 描述 |
|------|------|
| **社会实体** | 社会实体存在吗？ |
| **制度** | 制度是什么？ |
| **集体意图** | 集体如何有意向？ |
| **社会建构** | 社会是建构的吗？ |

**立场**：

| 立场 | 观点 |
|------|------|
| **实在论** | 社会实体真实存在 |
| **反实在论** | 社会实体只是概念 |
| **建构论** | 社会是建构的 |

**社会实体**：

| 实体 | 本体论问题 |
|------|-----------|
| **制度** | 制度是什么？ |
| **组织** | 组织是什么？ |
| **货币** | 货币是什么？ |
| **法律** | 法律是什么？ |

**相关词条**：本体论、社会建构、制度

### 【社会建构论】(Social Constructivism)

**定义**：认为社会现实是被社会建构的观点。

**费曼解释**：
社会建构论说：
- 很多"事实"是社会建构的
- 不是自然界给定的
- 是社会互动创造的

**核心主张**：

| 主张 | 描述 |
|------|------|
| **社会建构** | 社会现实被建构 |
| **历史变化** | 建构随历史变化 |
| **权力关系** | 建构涉及权力 |

**例子**：

| 被建构的"事实" | 描述 |
|----------------|------|
| **性别** | 性别是社会建构 |
| **种族** | 种族是社会建构 |
| **疾病** | 某些疾病是社会建构 |
| **知识** | 知识是社会建构 |

**强纲领**：

**布鲁尔的四条原则**：
1. **因果性**：相信是社会原因导致的
2. **公正性**：对真和假一视同仁
3. **对称性**：用同样的原因解释真和假
4. **反身性**：理论也适用于自己

**批评**：

| 批评 | 描述 |
|------|------|
| **相对主义** | 可能导致相对主义 |
| **否认实在** | 可能否认客观实在 |
| **自我反驳** | 可能自我反驳 |

**相关词条**：社会本体论、知识社会学、建构主义

### 【塞尔的社会本体论】(Searle's Social Ontology)

**定义**：塞尔提出的基于集体意向性和制度性事实的社会本体论。

**费曼解释**：
塞尔说：
- 社会现实基于"我们相信"
- 货币是货币，因为我们相信它是
- 制度性事实由集体意向性构成

**核心概念**：

| 概念 | 描述 |
|------|------|
| **集体意向性** | "我们相信"或"我们打算" |
| **功能归属** | 赋予事物功能 |
| **制度性事实** | 依赖人类制度的事实 |
| **构成性规则** | "X算作Y"的规则 |

**制度性事实**：

**定义**：只有因为人类制度才存在的事实。

**结构**：X在语境C中算作Y

**例子**：
- 这张纸算作货币
- 这个人算作总统
- 这些建筑算作大学

**vs 物理性事实**：

| 问题 | 物理性事实 | 制度性事实 |
|------|-----------|-----------|
| **独立性** | 独立于人类 | 依赖人类制度 |
| **存在** | 自在存在 | 因为集体承认而存在 |

**代表人物**：
- **约翰·塞尔**：社会本体论

**经典文献**：
- Searle, *The Construction of Social Reality* (1995)
- Searle, *Making the Social World* (2010)

**相关词条**：社会本体论、集体意向性、制度

---

### 【制度性事实】(Institutional Facts)

**定义**：依赖人类制度才存在的事实。

**费曼解释**：
制度性事实是"因为相信而存在"：
- 货币是货币，因为我们都承认
- 婚姻是婚姻，因为社会承认
- 这些事实依赖制度

**特征**：

| 特征 | 描述 |
|------|------|
| **依赖制度** | 依赖人类制度 |
| **集体承认** | 需要集体承认 |
| **语言依赖** | 依赖语言 |

**vs 原始事实**：

| 类型 | 例子 | 独立性 |
|------|------|--------|
| **原始事实** | 山的存在 | 独立于人类 |
| **制度性事实** | 货币的价值 | 依赖人类制度 |

**构成性规则**：

**形式**：X在语境C中算作Y

**例子**：

| X | C | Y |
|---|---|---|
| 这张纸 | 美国发行 | 美元 |
| 这个人 | 当选 | 总统 |
| 这些声音 | 棒球比赛中 | 好球 |

**相关词条**：塞尔、社会本体论、制度

### 【集体意向性】(Collective Intentionality)

**定义**：群体的"我们打算"或"我们相信"。

**费曼解释**：
集体意向性是"我们一起"：
- 不只是个人意向的加总
- 而是"我们"的意向
- 合作、共享信念的基础

**核心问题**：

| 问题 | 描述 |
|------|------|
| **存在性** | 集体意向性存在吗？ |
| **还原** | 能还原为个人意向吗？ |
| **形式** | 集体意向性的形式是什么？ |

**塞尔的观点**：

**核心主张**：
- 集体意向性是原始的
- 不能还原为个人意向
- "我们打算"是基本的

**吉尔伯特的观点**：

**核心主张**：
- 集体意向性基于" plural subject "
- "我们打算"= 我们形成一个plural subject

**应用**：

| 应用 | 描述 |
|------|------|
| **合作行动** | 合作需要集体意向 |
| **共享信念** | 共享信念需要集体意向 |
| **制度** | 制度需要集体承认 |

**相关词条**：塞尔、社会本体论、意向性

### 【社会科学客观性】(Objectivity in Social Science)

**定义**：社会科学能否达到客观性的问题。

**费曼解释**：
社会科学客观性问：

| **自我实现预言** | 研究可能影响现实 |

**韦伯的价值中立**：

**主张**：
- 区分事实判断和价值判断

- 研究者应该尽量客观

**回应**：

| 回应 | 策略 |

**相关词条**：价值中立、社会科学、客观性

### 【反身性】(Reflexivity)

**定义**：社会科学研究可能影响被研究对象的现象。

**费曼解释**：
反身性是"研究改变对象"：
- 社会科学研究社会
- 研究本身是社会的一部分
- 研究可能改变被研究的对象

**类型**：

| 类型 | 描述 |
|------|------|
| **理论反身性** | 理论影响现实 |
| **方法反身性** | 方法影响结果 |
| **知识反身性** | 知识改变行为 |

**例子**：

| 例子 | 描述 |
|------|------|
| **股票市场** | 预测影响市场 |
| **霍桑效应** | 被观察者改变行为 |
| **自我实现预言** | 预言导致预言成真 |

**挑战**：

| 挑战 | 描述 |
|------|------|
| **预测困难** | 预测可能改变结果 |
| **客观性** | 影响客观性 |
| **方法论** | 需要考虑反身性 |

**相关词条**：社会科学客观性、自证预言、霍桑效应

### 【质性研究】(Qualitative Research)

**定义**：使用非数值方法的社会研究。

**费曼解释**：
质性研究是"深入理解"：
- 不是用数字
- 而是用文字、图像
- 深入理解社会现象

**方法**：

| 方法 | 描述 |

| **深度访谈** | 深度访谈 |
| **参与观察** | 参与式观察 |
| **焦点小组** | 焦点小组讨论 |
| **文本分析** | 分析文本 |

**vs 量化研究**：

| 问题 | 质性研究 | 量化研究 |
|------|----------|----------|
| **数据** | 文字、图像 | 数字 |
| **方法** | 访谈、观察 | 统计、实验 |
| **深度vs广度** | 深度 | 广度 |
| **普遍性** | 有限 | 较高 |

**优势**：

| 优势 | 描述 |
|------|------|
| **深度** | 深入理解 |
| **灵活性** | 方法灵活 |
| **语境** | 保留语境 |

**挑战**：

| 挑战 | 描述 |
|------|------|
| **主观性** | 可能有主观性 |
| **普遍性** | 难以推广 |
| **可重复性** | 难以重复 |

**相关词条**：量化研究、民族志、解释性研究

### 【量化研究】(Quantitative Research)

**vs 质性研究**：

| 问题 | 量化研究 | 质性研究 |

| **广度vs深度** | 广度 | 深度 |
| **普遍性** | 较高 | 有限 |

**优势**：

| 优势 | 描述 |
|------|------|
| **客观性** | 较客观 |
| **可重复** | 可重复 |

**挑战**：

| 挑战 | 描述 |

| **表面化** | 可能表面化 |

### 【混合方法】(Mixed Methods)

**定义**：结合质性和量化方法的研究方法。

**费曼解释**：
混合方法说：
- 质性和量化各有优缺点
- 为什么不结合？
- 两种方法互补

**设计**：

| 设计 | 描述 |
|------|------|
| **解释性序列** | 先量化，后质性解释 |
| **探索性序列** | 先质性，后量化验证 |
| **并行** | 同时进行 |
| **嵌入** | 一种方法嵌入另一种 |

**优势**：

| 优势 | 描述 |
|------|------|
| **互补** | 两种方法互补 |
| **三角验证** | 三角验证 |
| **全面** | 更全面的了解 |

**挑战**：

| 挑战 | 描述 |
|------|------|
| **复杂性** | 更复杂 |
| **资源** | 需要更多资源 |
| **整合** | 难以整合 |

**相关词条**：质性研究、量化研究、三角验证
