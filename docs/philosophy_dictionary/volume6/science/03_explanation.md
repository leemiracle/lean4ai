## A-D

### 【科学解释】(Scientific Explanation)

**定义**：用科学原理来回答"为什么"问题的活动。

**费曼解释**：
科学解释回答"为什么"：
- 为什么天是蓝色的？
- 为什么物体会下落？
- 为什么水沸腾？

科学解释用定律、机制和原因来回答。

**解释的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **因果解释** | 解释原因 | 因为引力，物体下落 |
| **定律解释** | 用定律解释 | 根据热力学定律... |
| **机制解释** | 解释机制 | 神经元放电导致... |
| **统一解释** | 统一多种现象 | 牛顿定律统一天体和地面 |
| **目的论解释** | 解释目的 | 心脏为了泵血 |

**解释的模型**：

| 模型 | 核心主张 | 代表 |
|------|----------|------|
| **D-N模型** | 解释是演绎论证 | 亨普尔 |
| **I-S模型** | 解释是统计论证 | 亨普尔 |
| **因果模型** | 解释是提供原因 | 萨尔蒙 |
| **统一模型** | 解释是统一 | 基切尔 |
| **语用模型** | 解释依赖语境 | 范弗拉森 |

**解释的规范要求**：

| 要求 | 描述 |
|------|------|
| **相关性** | 解释必须相关 |
| **真** | 解释前提必须为真 |
| **深刻** | 解释应该深刻 |

**相关词条**：D-N模型、因果解释、统一性

### 【D-N模型】(Deductive-Nomological Model)

**定义**：亨普尔提出的科学解释模型——解释是从定律和初始条件演绎出被解释项的论证。

**费曼解释**：
D-N模型说：
- 解释是"论证"
- 从定律 + 条件演绎出结果
- 像数学证明一样

**结构**：

```
定律：L1, L2, ...
条件：C1, C2, ...
───────────────
现象：E
```

**例子：旗杆影子**：

| 组成部分 | 内容 |
|----------|------|
| **定律** | 光沿直线传播 |
| **条件** | 太阳在45°角，旗杆高10米 |
| **结论** | 影子长10米 |

**解释和预测的对称性**：

亨普尔主张：
- 解释和预测结构相同
- 如果你能解释，你就能预测
- 如果你能预测，你就能解释

**D-N模型的问题**：

| 问题 | 描述 |
|------|------|
| **对称性问题** | 旗杆和影子都能"解释"对方？ |
| **不相关问题** | 不相关因素可能被包含 |
| **因果方向** | 没有区分因果方向 |

**对称性问题**：

**旗杆悖论**：
- 正常解释：太阳角度+旗杆高度 → 影子长度
- 反向"解释"：太阳角度+影子长度 → 旗杆高度

问题：反向似乎不是"解释"，但D-N模型允许它。

**代表人物**：
- **亨普尔**：D-N模型创始人

**经典文献**：
- Hempel, "Aspects of Scientific Explanation" (1965)
- Hempel & Oppenheim, "Studies in the Logic of Explanation" (1948)

**相关词条**：科学解释、I-S模型、对称性问题

### 【I-S模型】(Inductive-Statistical Model)

**定义**：亨普尔提出的用统计定律解释的模型。

**费曼解释**：
I-S模型说：
- 不是所有定律都是决定论的
- 有些定律是统计的
- 解释可以是归纳的

**结构**：

```
统计定律：P(O|C) = r (高概率)
条件：C
───────────────
现象：O（高概率）
```

**例子：青霉素治愈**：

| 组成部分 | 内容 |
|----------|------|
| **统计定律** | 青霉素治愈链球菌感染的概率是95% |
| **条件** | 患者服用青霉素 |
| **结论** | 患者被治愈（高概率） |

**与D-N的区别**：

| 问题 | D-N模型 | I-S模型 |
|------|---------|---------|
| **定律类型** | 决定论定律 | 统计定律 |
| **演绎关系** | 演绎必然 | 归纳支持 |
| **确定性** | 确定结论 | 概率结论 |

**I-S模型的问题**：

| 问题 | 描述 |
|------|------|
| **最大特异性** | 需要最大特异性的参考类 |
| **概率阈值** | 多高的概率才算解释？ |
| **因果关系** | 统计相关不是因果关系 |

**最大特异性要求**：

**问题**：同一个事件可能有不同的统计概率。

**例子**：
- 服用青霉素 → 治愈概率95%
- 服用青霉素+吸烟 → 治愈概率90%

应该用哪个？亨普尔要求使用"最大特异性"的参考类。

**相关词条**：D-N模型、统计解释、亨普尔

### 【对称性问题】(Symmetry Problem)

**定义**：D-N模型允许对称的解释，但直觉上解释应该是非对称的问题。

**费曼解释**：
对称性问题说：
- D-N模型允许A解释B，也允许B解释A
- 但直觉上解释有方向
- 旗杆高度解释影子长度，不是反过来

**旗杆悖论**：

**正常解释**：
- 定律：三角函数
- 条件：太阳角度、旗杆高度
- 结论：影子长度

**反向"解释"**：
- 定律：三角函数
- 条件：太阳角度、影子长度
- 结论：旗杆高度

问题：反向似乎不是"解释"，但D-N模型允许它。

**其他例子**：

| 正常解释 | 反向"解释" | 问题 |
|----------|-----------|------|
| 气压下降 → 风暴来临 | 风暴来临 → 气压下降 | 反向不是解释 |
| 压力增加 → 体积减少 | 体积减少 → 压力增加 | 反向不是解释 |

**解决方案**：

| 方案 | 策略 |
|------|------|
| **因果方向** | 引入因果方向 |
| **时间方向** | 引入时间方向 |
| **语用方向** | 解释依赖语境 |
| **反事实依赖** | 引入反事实依赖 |

**代表人物**：
- **亨普尔**：D-N模型
- **萨尔蒙**：因果解释

**相关词条**：D-N模型、因果解释、解释方向

### 【不相关问题】(Irrelevance Problem)

**定义**：D-N模型允许包含不相关因素的解释的问题。

**费曼解释**：
不相关问题说：
- D-N模型允许任何"论证"
- 即使包含不相关因素
- 但真正的解释应该只包含相关因素

**例子：魔法咒语**：

**"解释"**：
- 定律：服用青霉素 → 治愈
- 条件：服用青霉素 + 念咒语
- 结论：被治愈

问题：咒语是不相关的，但D-N模型允许它。

**例子：男避孕药**：

**"解释"**：
- 条件：服用避孕药 + 是男性
- 结论：没有怀孕

问题：是男性才是真正原因，避孕药不相关。

**解决方案**：

| 方案 | 策略 |
|------|------|
| **因果相关性** | 只包含因果相关因素 |
| **反事实依赖** | 只包含反事实依赖因素 |
| **解释相关性** | 定义解释相关性条件 |

**相关词条**：D-N模型、因果解释、相关性

### 【因果解释】(Causal Explanation)

**定义**：通过提供原因来解释现象。

**费曼解释**：
因果解释说：
- 解释就是找原因
- 为什么窗户碎了？因为球击中了它
- 原因导致结果

**核心主张**：

| 要素 | 描述 |
|------|------|
| **原因** | 解释提供原因 |
| **因果链** | 描述因果链 |
| **因果机制** | 揭示因果机制 |

**萨尔蒙的因果模型**：

**核心概念**：
- **因果过程**：传递因果影响的过程
- **因果交互**：因果过程之间的交互
- **共同原因**：导致多个结果的原因

**因果过程**：

| 特征 | 描述 |
|------|------|
| **传递性** | 传递因果影响 |
| **持续性** | 在时间和空间中持续 |
| **能量传递** | 传递能量/动量 |

**例子：子弹**：

- 子弹飞行是因果过程
- 击中目标是因果交互
- 导致目标损伤

**因果解释的优势**：

| 优势 | 描述 |
|------|------|
| **解决对称性** | 因果有方向 |
| **解决不相关性** | 只包含因果相关因素 |
| **符合直觉** | 符合解释直觉 |

**对因果解释的批评**：

| 批评 | 描述 |
|------|------|
| **因果概念** | 因果本身需要解释 |
| **非因果解释** | 有些解释似乎不是因果的 |
| **缺失原因** | 量子力学中可能没有因果 |

**代表人物**：
- **萨尔蒙**：因果解释创始人

**经典文献**：
- Salmon, *Scientific Explanation and the Causal Structure of the World* (1984)

**相关词条**：因果关系、萨尔蒙、科学解释

### 【统一性解释】(Unificationist Explanation)

**定义**：通过统一多种现象来解释的解释模型。

**费曼解释**：
统一性解释说：
- 解释就是统一
- 牛顿定律统一了天体运动和地面运动
- 用少数原理解释多种现象

**核心主张**：

| 要素 | 描述 |
|------|------|
| **统一** | 解释是统一现象 |
| **简化** | 用少数原理解释多种现象 |
| **论证模式** | 使用少数论证模式 |

**基切尔的模型**：

**核心概念**：
- **论证模式**：重复使用的论证模式
- **统一**：用少数模式解释多种现象
- **接受**：接受的论证模式

**例子：牛顿力学**：

| 现象 | 统一原理 |
|------|----------|
| 行星运动 | 牛顿定律 |
| 落体运动 | 牛顿定律 |
| 潮汐 | 牛顿定律 |

牛顿定律统一了这些现象。

**统一性的优势**：

| 优势 | 描述 |
|------|------|
| **简洁** | 用少数原理 |
| **深刻** | 揭示深层联系 |
| **预测** | 允许新预测 |

**对统一性解释的批评**：

| 批评 | 描述 |
|------|------|
| **统一≠解释** | 统一不一定解释 |
| **因果缺失** | 可能忽略因果 |
| **标准模糊** | 什么是"好的统一"？ |

**代表人物**：
- **基切尔**：统一性解释创始人

**经典文献**：
- Kitcher, "Explanatory Unification" (1989)

**相关词条**：科学解释、基切尔、统一性

### 【语用解释论】(Pragmatic Theory of Explanation)

**定义**：范弗拉森提出的强调解释依赖语境的理论。

**费曼解释**：
语用解释论说：
- 解释不是理论的内在属性
- 解释是"为什么"问题的回答
- 依赖于语境和提问者

**核心主张**：

| 要素 | 描述 |
|------|------|
| **语用性** | 解释依赖语境 |
| **问答** | 解释是回答"为什么"问题 |
| **对比类** | 为什么是P而不是Q？ |
| **相关性** | 相关性依赖语境 |

**为什么-问题**：

**结构**：
- **话题**：被解释的现象
- **对比类**：为什么是P而不是Q？
- **相关性关系**：什么算相关

**例子**：

| 问题 | 对比类 | 相关答案 |
|------|--------|----------|
| 为什么塔倒了？ | 为什么塔倒了而不是立着？ | 地震 |
| 为什么塔倒了？ | 为什么是这座塔倒了？ | 建筑缺陷 |

同样的问题，不同的对比类，不同的答案。

**语用解释论的优势**：

| 优势 | 描述 |
|------|------|
| **灵活性** | 解释依赖语境 |
| **避免问题** | 避免D-N模型的问题 |
| **符合实践** | 符合科学实践 |

**对语用解释论的批评**：

| 批评 | 描述 |
|------|------|
| **主观性** | 解释变得主观 |
| **客观性** | 失去客观标准 |
| **因果忽视** | 可能忽视因果 |

**代表人物**：
- **范弗拉森**：语用解释论创始人

**经典文献**：
- van Fraassen, *The Scientific Image* (1980)

**相关词条**：科学解释、范弗拉森、语境

### 【因果机制】(Causal Mechanism)

**定义**：产生因果关系的底层过程或结构。

**费曼解释**：
因果机制是"幕后过程"：
- 为什么药物有效？→ 与受体结合
- 为什么汽车会动？→ 发动机燃烧汽油
- 机制揭示"如何工作"

**核心要素**：

| 要素 | 描述 |
|------|------|
| **实体** | 机制中的实体 |
| **活动** | 实体的活动 |
| **组织** | 如何组织 |
| **结果** | 产生的结果 |

**例子**：

| 现象 | 机制 |
|------|------|
| **神经传导** | 离子通道开关 |
| **蛋白质合成** | 转录和翻译 |
| **肌肉收缩** | 肌动蛋白-肌球蛋白滑动 |

**机制解释的特点**：

| 特点 | 描述 |
|------|------|
| **描述过程** | 描述因果过程 |
| **揭示结构** | 揭示底层结构 |
| **连续性** | 连续的因果链 |

**机制 vs 定律**：

| 问题 | 定律解释 | 机制解释 |
|------|----------|----------|
| **形式** | 数学定律 | 过程描述 |
| **抽象度** | 抽象 | 具体 |
| **领域** | 物理学 | 生物学/社会科学 |

**相关词条**：因果解释、机制发现、科学解释

### 【解释深度】(Explanatory Depth)

**定义**：解释揭示现象本质的程度。

**费曼解释**：
解释深度问：
- 有些解释更"深刻"
- "因为上帝想要" vs "因为引力"
- 什么是"深刻"的解释？

**深度的标准**：

| 标准 | 描述 |
|------|------|
| **普遍性** | 解释更普遍的现象 |
| **统一性** | 统一更多现象 |
| **因果链** | 揭示更深的因果链 |
| **机制** | 揭示更深的机制 |

**例子**：

| 解释 | 深度 |
|------|------|
| "苹果下落因为它重" | 较浅 |
| "苹果下落因为引力" | 较深 |
| "引力因为时空弯曲" | 更深 |

**深度的比较**：

| 理论 | 解释深度 |
|------|----------|
| **托勒密** | 只描述运动 |
| **哥白尼** | 部分解释 |
| **牛顿** | 统一解释 |
| **爱因斯坦** | 更深层解释 |

**相关词条**：科学解释、统一性、深刻性

### 【目的论解释】(Teleological Explanation)

**定义**：用目的或功能来解释现象的解释方式。

**费曼解释**：
目的论解释用"为了"来解释：
- 心脏跳动是为了泵血
- 眼睛是为了看
- 为什么有这种行为？为了生存

**目的论解释的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **内在目的论** | 内在目的 | 生物功能 |
| **外在目的论** | 外在设计 | 人造物 |
| **自然目的论** | 自然目的 | 亚里士多德 |
| **进化目的论** | 进化选择 | 适应 |

**目的论解释 vs 因果解释**：

| 问题 | 因果解释 | 目的论解释 |
|------|----------|-----------|
| **心脏跳动** | 因为神经信号 | 为了泵血 |
| **叶子朝向阳光** | 因为生长素分布 | 为了光合作用 |

**目的论解释的问题**：

| 问题 | 描述 |
|------|------|
| **倒退问题** | 如何解释目的？ |
| **自然目的** | 自然界有目的吗？ |
| **循环** | 目的本身需要解释 |

**进化论的解释**：

进化论提供自然化的目的论解释：
- "功能" = 被自然选择的功能
- 不需要意识目的

**相关词条**：功能解释、进化解释、亚里士多德

### 【功能解释】(Functional Explanation)

**定义**：用功能来解释现象的解释方式。

**费曼解释**：
功能解释用"功能"来解释：
- 心脏的功能是泵血
- 肾脏的功能是过滤
- 解释为什么有这个器官

**核心概念**：

| 概念 | 描述 |
|------|------|
| **功能** | 某物做什么 |
| **角色** | 在系统中的角色 |
| **贡献** | 对系统的贡献 |

**功能解释的类型**：

| 类型 | 描述 |
|------|------|
| **因果角色功能** | 在因果网络中的角色 |
| **选择功能** | 被选择的功能 |
| **设计功能** | 设计的目的 |

**功能解释的问题**：

| 问题 | 描述 |
|------|------|
| **功能的因果力** | 功能如何有因果力？ |
| **多功能性** | 同一事物有多种功能 |
| **副功能** | 哪些是主要功能？ |

**相关词条**：目的论解释、功能主义、生物学哲学

### 【进化解释】(Evolutionary Explanation)

**定义**：用进化机制来解释生物特征的解释方式。

**费曼解释**：
进化解释说：
- 为什么有这个特征？
- 因为它在进化中被选择
- 它提供了适应优势

**核心机制**：

| 机制 | 描述 |
|------|------|
| **自然选择** | 适者生存 |
| **遗传漂变** | 随机变化 |
| **基因流** | 基因迁移 |
| **突变** | 随机变异 |

**进化解释的例子**：

| 特征 | 进化解释 |
|------|----------|
| **眼睛** | 有视觉优势，被选择 |
| **伪装** | 逃避捕食者，被选择 |
| **亲代照顾** | 增加后代存活，被选择 |

**适应主义**：

**核心主张**：大多数特征是自然选择的适应。

**批评**：有些特征可能不是适应（如：漂变、副作用）。

**相关词条**：自然选择、适应、生物学哲学

### 【机制解释】(Mechanistic Explanation)

**定义**：通过描述机制来解释现象的解释方式。

**费曼解释**：
机制解释描述"如何工作"：
- 神经如何传导？
- 蛋白质如何合成？
- 揭示底层机制

**核心要素**：

| 要素 | 描述 |
|------|------|
| **实体** | 机制中的组件 |
| **活动** | 实体的活动 |
| **组织** | 如何组织 |
| **层次** | 不同层次 |

**机制解释的例子**：

| 现象 | 机制 |
|------|------|
| **神经传导** | 离子通道开关 |
| **肌肉收缩** | 肌动蛋白滑动 |
| **基因表达** | 转录和翻译 |

**与定律解释的比较**：

| 问题 | 定律解释 | 机制解释 |
|------|----------|----------|
| **形式** | 数学定律 | 过程描述 |
| **抽象度** | 抽象 | 具体 |

### 【反事实解释】(Counterfactual Explanation)

**定义**：用反事实条件来解释的解释方式。

**费曼解释**：
反事实解释问：
- "如果X没发生，Y会怎样？"
- "如果没有引力，苹果不会下落"
- 解释用反事实来表述

**核心概念**：

| 概念 | 描述 |
|------|------|
| **反事实条件** | "如果...会怎样？" |
| **依赖** | 结果依赖原因 |
| **可能世界** | 其他可能世界 |

**反事实依赖**：

**定义**：如果原因不发生，结果也不发生。

**例子**：
- 如果没击中窗户，窗户不会碎
- 窗户碎依赖球击中

**在解释中的角色**：

| 角色 | 描述 |
|------|------|
| **区分因果** | 区分因果和非因果 |
| **解释方向** | 确定解释方向 |
| **解释深度** | 评估解释深度 |

**相关词条**：反事实条件、因果关系、解释

### 【解释鸿沟】(Explanatory Gap)

**定义**：物理事实与意识经验之间的解释鸿沟。

**费曼解释**：
解释鸿沟说：
- 我们可以解释水 = H₂O
- 但无法解释"为什么C纤维激活是疼痛的感觉"
- 物理事实和意识之间有"鸿沟"

**核心问题**：

| 问题 | 描述 |
|------|------|
| **为什么** | 为什么某些物理过程有意识？ |
| **必然性** | 为什么是必然的？ |
| **可理解性** | 物理解释能解释意识吗？ |

** Levine的论证**：

1. 物理事实不能先验地解释意识
2. 存在解释鸿沟
3. 这挑战物理主义

**回应**：

| 回应 | 策略 |
|------|------|
| **认知鸿沟** | 只是认知鸿沟，不是本体论鸿沟 |
| **现象概念** | 概念鸿沟，不是事实鸿沟 |
| **接受鸿沟** | 但否认本体论后果 |

**代表人物**：
- **Joseph Levine**：解释鸿沟提出者

**相关词条**：困难问题、意识、物理主义

覆盖律模型是D-N和I-S的统称：

| **演绎/归纳** | 演绎或归纳论证 |
| **统一** | 定律统一现象 |

**组成部分**：

| 模型 | 定律类型 | 关系 |
|------|----------|------|
| **D-N模型** | 决定论定律 | 演绎 |

### 【解释还原】(Explanatory Reduction)

**定义**：将高层解释还原为低层解释的过程。

**费曼解释**：
解释还原说：
- 生物学解释可以还原为化学解释
- 化学解释可以还原为物理解释
- 高层解释最终还原为物理

**层次结构**：

```
社会科学
  ↓
心理学
  ↓
生物学
  ↓
化学
  ↓
物理学
```

**还原的条件**：

| 条件 | 描述 |
|------|------|
| **术语连接** | 高层术语可定义为低层术语 |
| **定律推导** | 高层定律可从低层定律推导 |
| **桥接原理** | 存在桥接原理 |

**对解释还原的批评**：

| 批评 | 描述 |
|------|------|
| **涌现** | 有些性质不可还原 |
| **多重实现** | 高层可被多种低层实现 |
| **自主性** | 高层解释有自主性 |

**相关词条**：还原论、涌现、科学统一

### 【多层次解释】(Multi-Level Explanation)

### 【解释性排除】(Explanatory Exclusion)

**定义**：如果一个现象有充分的原因，另一个潜在原因被排除的原则。

**费曼解释**：
解释性排除说：
- 如果A充分解释了C
- 那么B不能独立地解释C
- 除非A和B有某种关系

**排除论证**：

**金的形式**：
1. 如果心理状态M导致行为B
2. 且物理状态P也充分导致B
3. 那么M和P必须有某种关系（同一、实现等）

**应用于心身问题**：

| 问题 | 排除论证 |
|------|----------|
| **心理因果** | 如果物理原因充分，心理因果被排除？ |
| **非还原物理主义** | 非还原的心理状态有因果力吗？ |

**回应**：

| 回应 | 策略 |
|------|------|
| **同一** | 心理状态 = 物理状态 |
| **实现** | 心理状态被物理状态实现 |
| **否认排除** | 不同层次的因果不排除 |

**代表人物**：
- **金在权**：排除论证提出者

**相关词条**：因果排除、心理因果、心身问题

---

### 【因果排除论证】(Causal Exclusion Argument)

**定义**：金在权提出的论证非还原心理状态无因果力的论证。

**费曼解释**：
因果排除论证说：
- 物理原因充分解释行为
- 如果心理状态独立于物理状态
- 那么心理状态无因果力

**论证结构**：

1. 物理世界因果封闭
2. 心理状态M导致物理效果P
3. 但P已有充分物理原因P*
4. 如果M ≠ P*，则P被过度决定
5. 过度决定不是普遍现象
6. 因此，M = P*（心理状态是物理状态）

**对非还原物理主义的挑战**：

| 问题 | 描述 |
|------|------|
| **独立性** | 如果心理属性独立，因果力在哪？ |
| **副现象论** | 非还原导致副现象论？ |
| **选择** | 要么还原，要么接受副现象论 |

**回应**：

| 回应 | 策略 |
|------|------|
| **否认排除** | 不同层次的因果不排除 |
| **接受副现象论** | 心理无因果，但直觉是错的 |
| **功能解释** | 心理因果是功能因果 |

**代表人物**：
- **金在权**：排除论证提出者

**经典文献**：
- Kim, *Mind in a Physical World* (1998)

**相关词条**：解释性排除、心理因果、因果封闭

### 【解释的语用学】(Pragmatics of Explanation)

**定义**：解释如何依赖于语境和使用的研究。

**费曼解释**：
解释的语用学问：
- 解释如何依赖于语境？
- 什么因素影响解释的适当性？
- 解释的社会维度

**语用因素**：

| 因素 | 描述 |
|------|------|
| **听众** | 解释给谁听 |
| **语境** | 解释发生在什么语境 |
| **背景知识** | 听众的背景知识 |
| **目的** | 解释的目的是什么 |

**范弗拉森的观点**：

解释是"为什么-问题"的回答，依赖于：
- 话题
- 对比类
- 相关性关系

**例子**：

| 语境 | 问题 | 适当解释 |
|------|------|----------|
| **医学** | 为什么生病？ | 病毒感染 |
| **社会学** | 为什么生病？ | 社会经济因素 |
| **心理学** | 为什么生病？ | 心理压力 |

**相关词条**：语用解释论、语境、范弗拉森

**主观主义的论证**：

| 论证 | 描述 |
|------|------|

| **兴趣依赖** | 解释依赖兴趣 |

### 【解释性无知】(Explanatory Ignorance)

**定义**：缺乏对现象的解释。

**费曼解释**：
解释性无知是"不知道为什么"：
- 为什么有意识？不知道
- 为什么有量子纠缠？不知道
- 有些现象我们无法解释

**类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **暂时无知** | 暂时不知道，将来可能知道 | 中世纪不知道传染病 |
| **本质无知** | 本质上无法知道 | 量子不确定性？ |
| **概念无知** | 概念上无法解释 | 意识？ |

**对科学的意义**：

| 意义 | 描述 |
|------|------|
| **谦逊** | 承认无知 |
| **动力** | 无知驱动研究 |
| **边界** | 标记科学边界 |

**相关词条**：科学解释、不可知论、科学边界

### 【解释的可理解性】(Intelligibility of Explanation)

**定义**：解释能够被理解的程度。

**费曼解释**：
解释的可理解性问：
- 解释能被理解吗？
- 什么使解释可理解？
- 不可理解的解释是解释吗？

**可理解性的要素**：

| 要素 | 描述 |
|------|------|
| **概念清晰** | 使用清晰的概念 |
| **逻辑连贯** | 逻辑连贯 |
| **经验联系** | 与经验联系 |
| **类比** | 使用类比 |

**不可理解的解释**：

| 情况 | 描述 |
|------|------|
| **概念不清晰** | 概念不清楚 |
| **逻辑不连贯** | 逻辑问题 |
| **与经验脱离** | 无法与经验联系 |
| **太抽象** | 过于抽象 |

**量子力学的例子**：

量子力学的解释被认为是"不可理解"的：
- 波粒二象性
- 测量问题
- 纠缠

**相关词条**：科学解释、量子力学哲学、可理解性

### 【解释的完整性】(Completeness of Explanation)

**定义**：解释覆盖现象所有方面的程度。

**费曼解释**：
解释的完整性问：
- 解释是否完整？
- 是否遗漏了重要因素？
- 完整的解释可能吗？

**完整性的维度**：

| 维度 | 描述 |
|------|------|
| **因素完整** | 包含所有相关因素 |
| **机制完整** | 描述完整机制 |
| **因果链完整** | 描述完整因果链 |
| **层次完整** | 包含所有相关层次 |

**完整性 vs 简单性**：

| 问题 | 完整性 | 简单性 |
|------|--------|--------|
| **权衡** | 完整可能复杂 | 简单可能不完整 |
| **理想** | 完整+简单 | |

**相关词条**：科学解释、简单性、完整性
