## A-D

### 【因果关系】(Causation)

**定义**：一个事件（原因）导致另一个事件（结果）发生的关系。

**费曼解释**：
因果关系是"为什么"的核心：
- 为什么窗户碎了？因为球击中了它
- 球击中是原因，窗户碎是结果
- 原因导致结果

**因果关系的特征**：

| 特征 | 描述 |
|------|------|
| **时间优先** | 原因在结果之前 |
| **必然联系** | 原因和结果有必然联系 |
| **反事实依赖** | 如果原因不发生，结果也不发生 |
| **可操纵性** | 原因可以被操纵 |

**因果理论**：

| 理论 | 核心主张 | 代表 |
|------|----------|------|
| **规律性理论** | 原因是结果的充分条件 | 休谟 |
| **反事实理论** | 原因是反事实依赖 | 刘易斯 |
| **过程理论** | 因果是过程传递 | 萨尔蒙 |
| **操纵理论** | 原因是可操纵的 | 伍德沃德 |
| **概率理论** | 原因提高结果的概率 | Suppes |

**休谟的因果分析**：

**三种定义**：
1. 时间优先：原因在结果之前
2. 空间邻近：原因和结果邻近
3. 恒常连接：类似原因总是伴随类似结果

**休谟问题**：
- 我们只能观察到"恒常连接"
- 观察不到"必然联系"
- 因果必然性是心理投射？

**相关词条**：休谟问题、因果推断、反事实

### 【因果推断】(Causal Inference)

**定义**：从数据中推断因果关系的方法。

**费曼解释**：
因果推断问：
- 如何从数据推断因果？
- 相关不等于因果
- 如何区分相关和因果？

**因果推断的方法**：

| 方法 | 描述 | 代表 |
|------|------|------|
| **实验法** | 随机对照实验 | 费希尔 |
| **准实验** | 自然实验 | 坎贝尔 |
| **统计方法** | 统计控制 | Pearl |
| **机制方法** | 揭示机制 | 萨尔蒙 |

**相关 vs 因果**：

| 情况 | 相关 | 因果 |
|------|------|------|
| **冰淇淋销量和溺水** | 正相关 | 无因果（共同原因：夏天） |
| **吸烟和肺癌** | 正相关 | 因果关系 |

**因果推断的困难**：

| 困难 | 描述 |
|------|------|
| **混淆变量** | 混淆变量导致虚假相关 |
| **选择偏倚** | 选择偏倚影响推断 |
| **反向因果** | 可能是结果导致原因 |
| **共同原因** | 共同原因导致相关 |

**珍珠的因果图**：

**有向无环图（DAG）**：
- 节点：变量
- 边：因果方向
- 用于表示因果结构

```
A → B → C
    ↑
    D
```

**代表人物**：
- **朱迪亚·珍珠**：因果图模型
- **坎贝尔**：准实验

**经典文献**：
- Pearl, *Causality* (2009)
- Campbell & Stanley, *Experimental and Quasi-Experimental Designs* (1963)

**相关词条**：因果关系、因果图、随机对照试验

### 【休谟问题】(Hume's Problem)

**定义**：休谟提出的关于因果关系本质的哲学问题。

**费曼解释**：
休谟问题问：
- 我们如何知道因果关系？
- 我们只能观察到"恒常连接"
- 观察不到"必然联系"

**休谟的分析**：

**三个要素**：
1. **时间优先**：原因在结果之前
2. **空间邻近**：原因和结果在空间上邻近
3. **恒常连接**：类似原因总是伴随类似结果

**核心问题**：

| 问题 | 描述 |
|------|------|
| **必然联系** | 我们看不到"必然联系" |
| **归纳问题** | 过去的恒常连接不保证未来 |
| **因果定义** | 因果关系如何定义？ |

**休谟的怀疑结论**：

- 因果必然性可能是心理投射
- 习惯/期望让我们相信因果
- 因果推理不是理性推理

**对休谟问题的回应**：

| 回应 | 策略 | 代表 |
|------|------|------|
| **康德** | 因果是先验范畴 | 康德 |
| **规律性理论** | 接受休谟的结论 | 逻辑经验主义 |
| **反事实理论** | 用反事实定义因果 | 刘易斯 |
| **实在论** | 因果是实在的 | 萨尔蒙 |

**相关词条**：因果关系、归纳问题、休谟

### 【反事实因果理论】(Counterfactual Theory of Causation)

**定义**：刘易斯提出的用反事实条件定义因果的理论。

**费曼解释**：
反事实因果理论说：
- "如果原因不发生，结果也不会发生"
- 球击中窗户，窗户碎了
- 如果球没有击中，窗户不会碎

**核心定义**：

**因果的反事实定义**：
- C是E的原因，当且仅当：
- 如果C不发生，E也不会发生

**反事实条件**：

| 类型 | 形式 | 例子 |
|------|------|------|
| **真反事实** | 如果P，会Q | 如果下雨，地会湿 |
| **假反事实** | 如果非P（事实是P），会Q | 如果没下雨，地不会湿 |

**刘易斯的可能世界语义**：

**核心概念**：
- 考虑"最相似的可能的非C世界"
- 在那个世界中，E是否发生？
- 如果不发生，则C是E的原因

**问题**：

| 问题 | 描述 |
|------|------|
| **过度决定** | 多个充分原因 |
| **先发制人** | 预先阻止 |
| **相似性** | 如何定义世界相似性？ |
| **传递性** | 因果是否传递？ |

**先发制人问题**：

**例子**：
- A和B都想杀C
- A先开枪，C死了
- 如果A不开枪，B会开枪，C还是会死
- 所以反事实条件说A不是原因？
- 直觉上A是原因

**代表人物**：
- **大卫·刘易斯**：反事实因果理论创始人

**经典文献**：
- Lewis, "Causation" (1973)
- Lewis, "Causation as Influence" (2000)

**相关词条**：因果关系、反事实条件、刘易斯

### 【因果过程】(Causal Process)

**定义**：萨尔蒙提出的传递因果影响的过程。

**费曼解释**：
因果过程是"传递因果的线"：
- 子弹飞行是因果过程
- 光传播是因果过程
- 它们传递因果影响

**萨尔蒙的标记理论**：

**核心概念**：
- **因果过程**：能传递标记的过程
- **标记**：对过程的修改
- **伪过程**：不能传递标记的过程

**例子**：

| 过程 | 类型 | 原因 |
|------|------|------|
| **子弹飞行** | 因果过程 | 传递动量 |
| **影子移动** | 伪过程 | 不传递能量 |

**标记测试**：

- 修改因果过程，标记会传播
- 修改影子，标记不会传播

**因果交互**：

**定义**：两个因果过程的交互，导致两者都被修改。

**例子**：
- 子弹击中目标
- 两车碰撞

**代表人物**：
- **韦斯利·萨尔蒙**：因果过程理论创始人

**经典文献**：
- Salmon, *Scientific Explanation and the Causal Structure of the World* (1984)

**相关词条**：因果解释、萨尔蒙、因果结构

### 【因果操纵理论】(Manipulationist Theory)

**定义**：伍德沃德提出的以可操纵性为核心的因果理论。

**费曼解释**：
因果操纵理论说：
- 原因是"可以被操纵以改变结果的变量"
- 如果改变X会导致Y改变
- 则X是Y的原因

**核心主张**：

| 要素 | 描述 |
|------|------|
| **可操纵性** | 原因可被操纵 |
| **变化关系** | 改变原因导致结果变化 |
| **干预** | 干预是操纵的方式 |

**干预主义**：

**定义**：干预是对变量的操纵，切断变量与其他原因的联系。

**条件**：
1. 干预改变目标变量
2. 干预不通过其他路径影响结果
3. 干涉与结果的其他原因独立

**因果关系的定义**：

X是Y的原因，当且仅当：存在可能的干预，改变X会导致Y改变。

**优势**：

| 优势 | 描述 |
|------|------|
| **解释因果推断** | 解释如何推断因果 |
| **解释实验** | 解释实验的作用 |
| **处理反事实** | 自然处理反事实 |

**代表人物**：
- **詹姆斯·伍德沃德**：操纵理论创始人

**经典文献**：
- Woodward, *Making Things Happen* (2003)

**相关词条**：因果关系、干预、因果推断

### 【概率因果】(Probabilistic Causation)

**定义**：用概率关系定义因果的理论。

**费曼解释**：
概率因果说：
- 不是所有原因都必然导致结果
- 吸烟增加肺癌概率，但不一定导致肺癌
- 原因提高结果的概率

**核心定义**：

**概率提升**：
- C是E的原因，当且仅当：
- P(E|C) > P(E|非C)

**问题**：

| 问题 | 描述 |
|------|------|
| **辛普森悖论** | 分组和整体概率可能相反 |
| **混淆变量** | 混淆变量导致虚假概率提升 |
| **共同原因** | 共同原因导致概率相关 |

**辛普森悖论**：

**例子**：
- 整体上，治疗A成功率更高
- 但分性别看，治疗B在男女中都更高
- 这就是辛普森悖论

**条件化**：

**解决**：需要条件化适当的背景因素。

P(E|C, K) > P(E|非C, K)

其中K是适当的背景条件。

**代表人物**：
- **Patrick Suppes**：概率因果理论

**经典文献**：
- Suppes, *A Probabilistic Theory of Causality* (1970)

**相关词条**：因果关系、概率、辛普森悖论

### 【因果图】(Causal Diagram)

**定义**：用图形表示因果关系的工具。

**费曼解释**：
因果图是"因果地图"：
- 变量是节点
- 箭头表示因果方向
- 用于推理因果

**基本元素**：

| 元素 | 描述 |
|------|------|
| **节点** | 变量 |
| **边** | 因果关系 |
| **箭头** | 因果方向 |

**基本结构**：

| 结构 | 图形 | 含义 |
|------|------|------|
| **链** | A → B → C | A通过B影响C |
| **叉** | A ← B → C | B是A和C的共同原因 |
| **碰撞** | A → B ← C | B是A和C的共同结果 |

**d-分离**：

**定义**：如果两个变量被一组变量d-分离，则它们在该条件下独立。

**应用**：
- 识别混淆变量
- 设计控制策略
- 推断因果效应

**代表人物**：
- **朱迪亚·珍珠**：因果图创始人

**经典文献**：
- Pearl, *Causality* (2009)
- Spirtes, Glymour & Scheines, *Causation, Prediction, and Search* (2000)

**相关词条**：因果推断、珍珠、贝叶斯网络

### 【混淆变量】(Confounder)

**定义**：同时影响原因和结果的变量，导致虚假因果推断。

**费曼解释**：
混淆变量是"幕后黑手"：
- 吃冰淇淋和溺水相关
- 不是冰淇淋导致溺水
- 混淆变量：夏天

**特征**：

| 特征 | 描述 |
|------|------|
| **影响原因** | 与原因相关 |
| **影响结果** | 与结果相关 |
| **非中介** | 不是因果路径的一部分 |

**例子**：

| 相关 | 混淆变量 |
|------|----------|
| 冰淇淋销量和溺水 | 夏天 |
| 教育和收入 | 家庭背景 |
| 咖啡和心脏病 | 吸烟 |

**控制方法**：

| 方法 | 描述 |
|------|------|
| **随机化** | 随机分配打破混淆 |
| **统计控制** | 控制混淆变量 |
| **匹配** | 匹配混淆变量 |
| **工具变量** | 使用工具变量 |

**相关词条**：因果推断、随机对照试验、因果图

### 【随机对照试验】(Randomized Controlled Trial)

**定义**：通过随机分配来建立因果关系的实验方法。

**费曼解释**：
随机对照试验是"因果的金标准"：
- 随机分配到实验组或对照组
- 随机化打破所有混淆
- 可以建立因果关系

**核心要素**：

| 要素 | 描述 |
|------|------|
| **随机化** | 随机分配 |
| **对照组** | 设立对照组 |
| **盲法** | 尽可能双盲 |
| **意向性分析** | 按分配分析 |

**为什么随机化有效**：

| 理由 | 描述 |
|------|------|
| **打破混淆** | 随机化打破混淆变量 |
| **平均可比** | 组间平均可比 |
| **因果推断** | 允许因果推断 |

**局限**：

| 局限 | 描述 |
|------|------|
| **伦理问题** | 可能不道德 |
| **可行性** | 有时不可行 |
| **外部效度** | 结果可能不普适 |

**相关词条**：因果推断、实验、混淆变量

### 【工具变量】(Instrumental Variable)

**定义**：用于处理内生性和混淆的统计方法。

**费曼解释**：
工具变量是"第三方帮助者"：
- 混淆变量影响X和Y
- 找到工具Z
- Z只通过X影响Y

**条件**：

| 条件 | 描述 |
|------|------|
| **相关性** | Z与X相关 |
| **排他性** | Z只通过X影响Y |
| **外生性** | Z不受混淆影响 |

**例子**：

| 情况 | 工具变量 |
|------|----------|
| **教育和收入** | 义务教育法的变化 |
| **吸烟和健康** | 烟草税 |

**因果图**：

```
混淆 → X → Y
       ↑
       Z
```

**相关词条**：因果推断、混淆变量、计量经济学

### 【因果机制】(Causal Mechanism)

**定义**：产生因果关系的底层过程或结构。

**费曼解释**：
因果机制是"因果的内部运作"：
- 药物为什么有效？与受体结合
- 汽车为什么会动？发动机燃烧
- 机制揭示"如何工作"

**核心要素**：

| 要素 | 描述 |
|------|------|
| **实体** | 机制中的实体 |
| **活动** | 实体的活动 |
| **组织** | 如何组织 |
| **结果** | 产生的结果 |

**机制发现**：

| 方法 | 描述 |
|------|------|
| **干预** | 干预揭示机制 |
| **观察** | 观察机制运作 |
| **模拟** | 模拟机制 |
| **分解** | 分解系统 |

**例子**：

| 因果关系 | 机制 |
|----------|------|
| **药物效果** | 与受体结合 |
| **神经传导** | 离子通道开关 |
| **肌肉收缩** | 肌动蛋白滑动 |

**相关词条**：机制解释、因果解释、发现

机制发现是"寻找幕后过程"：
- 科学家发现DNA结构

**发现的挑战**：

### 【因果解释】(Causal Explanation)

**定义**：通过提供原因来解释现象。

**费曼解释**：
因果解释说：
- 解释就是找原因
- 为什么窗户碎了？因为球击中了它
- 因果链：原因→结果

**核心主张**：

| 要素 | 描述 |
|------|------|
| **原因** | 解释提供原因 |
| **因果链** | 描述因果链 |
| **因果机制** | 揭示因果机制 |

**vs D-N模型**：

| 问题 | D-N模型 | 因果解释 |
|------|---------|----------|
| **解释** | 演绎论证 | 提供原因 |
| **方向** | 无方向 | 有因果方向 |
| **相关性** | 不保证相关 | 只包含因果相关 |

**萨尔蒙的因果模型**：

- 解释是提供因果信息
- 描述因果过程和交互
- 揭示因果结构

**相关词条**：科学解释、萨尔蒙、因果过程

### 【因果涌现】(Causal Emergence)

**定义**：宏观层次有微观层次不具备的因果力的现象。

**费曼解释**：
因果涌现说：
- 宏观层次有自己的因果力
- 不能完全还原为微观因果
- 整体大于部分

**类型**：

| 类型 | 描述 |
|------|------|
| **弱涌现** | 宏观因果可从微观推导（但复杂） |
| **强涌现** | 宏观因果不可从微观推导 |
| **下向因果** | 宏观影响微观 |

**争议**：

| 立场 | 观点 |
|------|------|
| **还原论** | 否认因果涌现 |
| **涌现论** | 接受因果涌现 |
| **弱涌现论** | 接受弱涌现 |

**例子**：

| 系统 | 涌现因果？ |
|------|-----------|
| **意识** | 意识的因果力？ |
| **生命** | 生命的因果力？ |
| **社会** | 社会的因果力？ |

**相关词条**：涌现、下向因果、还原论

### 【下向因果】(Downward Causation)

**定义**：高层次对低层次产生因果影响的现象。

**费曼解释**：
下向因果说：
- 不只是原子影响分子
- 分子也可能影响原子
- 整体可以影响部分

**在心身问题中**：

### 【因果封闭】(Causal Closure of the Physical)

**定义**：每个物理事件都有充分的物理原因的原则。

**费曼解释**：
因果封闭说：
- 如果台球移动
- 它移动的原因（物理上）是充分的
- 不需要"心灵"来解释

**核心主张**：

| 要素 | 描述 |
|------|------|
| **物理原因充分** | 每个物理事件有物理原因 |
| **不需要超物理** | 不需要非物理原因 |
| **能量守恒** | 符合能量守恒定律 |

**对二元论的挑战**：

1. 如果物理世界因果封闭
2. 且心理有因果力
3. 那么心理必须是物理的
4. 否则违反封闭性

**回应**：

| 回应 | 策略 |
|------|------|
| **否认封闭** | 量子不确定性为非物理因果留下空间 |
| **非过度决定** | 心理和物理原因是同一原因 |
| **非竞争性** | 不同层面的描述 |

**相关词条**：心身问题、下向因果、物理主义

### 【因果排除】(Causal Exclusion)

**定义**：如果一个现象有充分的原因，另一个潜在原因被排除的原则。

**费曼解释**：
因果排除说：
- 如果A充分解释了C
- 那么B不能独立地解释C
- 除非A和B有某种关系

**金在权的排除论证**：

1. 物理世界因果封闭
2. 心理状态M导致行为B
3. 但B已有充分物理原因P
4. 如果M ≠ P，则B被过度决定
5. 过度决定不是普遍现象
6. 因此，M = P（心理状态是物理状态）

**对非还原物理主义的挑战**：

| 问题 | 描述 |
|------|------|
| **独立性** | 如果心理属性独立，因果力在哪？ |
| **副现象论** | 非还原导致副现象论？ |
| **选择** | 要么还原，要么接受副现象论 |

**相关词条**：心理因果、因果封闭、金在权

### 【过度决定】(Overdetermination)

**定义**：一个事件有多个独立充分原因的现象。

**费曼解释**：
过度决定是"多个杀手"：
- 两个人同时开枪杀死受害者
- 每一枪都足以致死
- 死亡被过度决定

**在因果排除中**：

| 问题 | 描述 |
|------|------|
| **心理过度决定** | 如果心理和物理都是原因，行为被过度决定 |
| **普遍性** | 过度决定应该是罕见的 |

**回应**：

| 回应 | 策略 |
|------|------|
| **同一** | 心理状态 = 物理状态 |
| **实现** | 心理状态被物理状态实现 |

### 【共同原因】(Common Cause)

**定义**：同时导致多个结果的单个原因。

**费曼解释**：
共同原因是"一因多果"：
- 气压下降导致温度计下降和风暴来临
- 温度计和风暴相关，但没有因果
- 共同原因：气压下降

**赖辛巴哈的共同原因原则**：

**核心主张**：如果A和B相关，要么A导致B，要么B导致A，要么有共同原因C。

**图示**：

```
    C
   ↙ ↘
  A   B
```

**例子**：

| 相关 | 共同原因 |
|------|----------|
| 冰淇淋销量和溺水 | 夏天 |
| 温度计和气压计 | 气压 |
| 教育和健康 | 社会经济地位 |

**意义**：

- 解释虚假相关
- 指导因果推断
- 设计控制策略

**相关词条**：因果推断、混淆变量、赖辛巴哈

### 【赖辛巴哈的共同原因原则】(Reichenbach's Common Cause Principle)

**定义**：赖辛巴哈提出的解释统计相关的原则。

**费曼解释**：
共同原因原则说：
- 如果A和B统计相关
- 那么要么A导致B，要么B导致A，要么有共同原因C
- 相关需要因果解释

**形式表述**：

如果 P(A∧B) > P(A)P(B)，那么：
- A导致B，或
- B导致A，或
- 存在共同原因C

**条件**：

**共同原因C的屏蔽数**：
- P(A∧B|C) = P(A|C)P(B|C)
- P(A∧B|非C) = P(A|非C)P(B|非C)

**意义**：

| 意义 | 描述 |
|------|------|
| **解释相关** | 相关需要因果解释 |
| **指导推断** | 指导因果推断 |
| **量子纠缠** | 量子纠缠挑战原则？ |

**相关词条**：共同原因、因果推断、赖辛巴哈

### 【因果排除论证】(Causal Exclusion Argument)

**定义**：金在权提出的论证非还原心理状态无因果力的论证。

**费曼解释**：
排除论证说：
- 物理原因充分解释行为
- 如果心理状态独立于物理状态
- 那么心理状态无因果力

**论证结构**：

1. 物理世界因果封闭
2. 心理状态M导致物理效果P
3. 但P已有充分物理原因P*
4. 如果M ≠ P*，则P被过度决定
5. 过度决定不是普遍现象
6. 因此，M = P*（心理状态是物理状态）

**对非还原物理主义的挑战**：

| 问题 | 描述 |
|------|------|
| **独立性** | 如果心理属性独立，因果力在哪？ |
| **副现象论** | 非还原导致副现象论？ |
| **选择** | 要么还原，要么接受副现象论 |

**回应**：

| 回应 | 策略 |
|------|------|
| **否认排除** | 不同层次的因果不排除 |
| **接受副现象论** | 心理无因果，但直觉是错的 |
| **功能解释** | 心理因果是功能因果 |

**代表人物**：
- **金在权**：排除论证提出者

**经典文献**：
- Kim, *Mind in a Physical World* (1998)

**相关词条**：心理因果、因果封闭、金在权
