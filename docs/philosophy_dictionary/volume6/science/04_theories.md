## A-D

### 【科学理论】(Scientific Theory)

**定义**：用于解释和预测自然现象的系统化知识体系。

**费曼解释**：
科学理论是"解释框架"：
- 进化论解释生物多样性
- 相对论解释时空
- 理论是科学的核心

**科学理论的要素**：

| 要素 | 描述 |
|------|------|
| **概念** | 基本概念 |
| **定律** | 描述规律的定律 |
| **原理** | 基本原理 |
| **模型** | 理论模型 |

**理论 vs 定律 vs 假说**：

| 概念 | 描述 |
|------|------|
| **假说** | 尝试性解释 |
| **定律** | 描述规律性 |
| **理论** | 系统解释框架 |

**理论的评价标准**：

| 标准 | 描述 |
|------|------|
| **经验适当** | 与观察一致 |
| **预测成功** | 成功预测 |
| **简单性** | 理论简单 |
| **统一性** | 统一现象 |

**相关词条**：范式、研究纲领、科学定律

### 【范式】(Paradigm)

**定义**：库恩提出的科学共同体的共享信念、价值观和技术。

**费曼解释**：
范式是"世界观框架"：
- 牛顿范式 vs 爱因斯坦范式
- 范式决定什么问题重要
- 范式内工作 = 常规科学

**范式的要素**：

| 要素 | 描述 |
|------|------|
| **形而上学信念** | 关于世界的基本信念 |
| **价值观** | 什么理论是好的 |
| **技术方法** | 实验和理论方法 |
| **经典案例** | 范例（exemplars） |

**范式的功能**：

| 功能 | 描述 |
|------|------|
| **指导研究** | 指导科学实践 |
| **定义问题** | 定义哪些问题重要 |
| **提供标准** | 提供评价标准 |
| **凝聚共同体** | 凝聚科学共同体 |

**范式转换**：

| 阶段 | 描述 |
|------|------|
| **常规科学** | 在范式内解谜 |
| **反常积累** | 发现反常 |
| **危机** | 反常导致危机 |
| **革命** | 新范式的出现 |
| **新常规科学** | 在新范式内工作 |

**例子**：

| 范式转换 | 旧范式 | 新范式 |
|----------|--------|--------|
| **天文学** | 地心说 | 日心说 |
| **光学** | 微粒说 | 波动说 |
| **力学** | 牛顿力学 | 相对论/量子力学 |

**代表人物**：
- **库恩**：范式概念创始人

**经典文献**：
- Kuhn, *The Structure of Scientific Revolutions* (1962)

**相关词条**：常规科学、科学革命、不可通约性

### 【常规科学】(Normal Science)

**定义**：库恩提出的在既定范式内进行的科学活动。

**费曼解释**：
常规科学是"解谜"：
- 范式给定规则
- 科学家解范式定义的谜
- 不质疑范式本身

**特征**：

| 特征 | 描述 |
|------|------|
| **范式主导** | 在范式框架内工作 |
| **解谜活动** | 解决范式定义的问题 |
| **不质疑基础** | 不质疑范式本身 |
| **累积性** | 知识累积增长 |

**常规科学的任务**：

| 任务 | 描述 |
|------|------|
| **确定重要事实** | 测量范式预测的量 |
| **事实与理论匹配** | 使观察与理论一致 |
| **阐明理论** | 精细化理论 |

**解谜**：

**定义**：在范式规则内解决的问题。

**特征**：
- 有确定的解
- 有规则指导
- 有标准检验

**对常规科学的批评**：

| 批评 | 描述 |
|------|------|
| **保守性** | 常规科学太保守 |
| **不创新** | 不鼓励创新 |
| **教条** | 范式变成教条 |

**相关词条**：范式、科学革命、库恩

---

### 【科学革命】(Scientific Revolution)

**定义**：范式转换的过程，科学世界观的根本变化。

**费曼解释**：
科学革命是"世界观转换"：
- 地心说 → 日心说
- 牛顿力学 → 相对论
- 革命是根本性的变化

**革命的特征**：

| 特征 | 描述 |
|------|------|
| **范式转换** | 从旧范式到新范式 |
| **格式塔转换** | 世界观的根本变化 |
| **不可通约** | 新旧范式不可通约 |
| **整体性** | 整体的变化 |

**革命的过程**：

```
常规科学 → 反常 → 危机 → 革命 → 新常规科学
```

| 阶段 | 描述 |
|------|------|
| **常规科学** | 范式内解谜 |
| **反常** | 发现与范式矛盾的现象 |
| **危机** | 反常积累导致紧张 |
| **革命** | 新范式的出现 |
| **新常规科学** | 在新范式内工作 |

**经典例子**：

| 革命 | 旧范式 | 新范式 |
|------|--------|--------|
| **哥白尼革命** | 地心说 | 日心说 |
| **化学革命** | 燃素说 | 氧化说 |
| **达尔文革命** | 神创论 | 进化论 |
| **爱因斯坦革命** | 牛顿力学 | 相对论 |
| **量子革命** | 经典物理 | 量子力学 |

**对革命模型的批评**：

| 批评 | 描述 |
|------|------|
| **连续性** | 革命也有连续性 |
| **渐进性** | 变化可能更渐进 |
| **多元性** | 多种变化模式 |

**相关词条**：范式、常规科学、不可通约性

---

### 【不可通约性】(Incommensurability)

**定义**：不同范式之间无法用共同标准比较的性质。

**费曼解释**：
不可通约性说：
- 牛顿范式和爱因斯坦范式
- 没有共同的标准来比较
- 它们是"不同世界"的居民

**三种不可通约性**：

| 类型 | 描述 |
|------|------|
| **方法论不可通约** | 不同方法标准 |
| **语义不可通约** | 术语意义不同 |
| **本体论不可通约** | 不同的世界观 |

**费耶阿本德的例子**：

| 术语 | 前伽利略 | 伽利略后 |
|------|----------|----------|
| "质量" | 不同概念 | 不同概念 |
| "运动" | 亚里士多德概念 | 惯性概念 |

**意义**：

- 挑战了科学的累积进步观
- 支持了科学革命的激进性
- 引发了相对主义的担忧

**对不可通约性的回应**：

| 回应 | 策略 |
|------|------|
| **否认** | 范式之间可以比较 |
| **局部化** | 只是局部不可通约 |
| **翻译** | 可以翻译（虽然不完全） |
| **接受** | 接受但否认导致相对主义 |

**代表人物**：
- **库恩**：不可通约性概念
- **费耶阿本德**：不可通约性发展

**相关词条**：范式、科学革命、理论比较

---

### 【反常】(Anomaly)

**定义**：与范式预期不符的现象。

**费曼解释**：
反常是"意外"：
- 范式预测X
- 但观察到Y
- 这就是反常

**反常的处理**：

| 阶段 | 描述 |
|------|------|
| **忽视** | 最初可能被忽视 |
| **解释** | 尝试在范式内解释 |
| **危机** | 积累到一定程度导致危机 |
| **革命** | 可能触发范式转换 |

**经典例子**：

| 反常 | 范式 | 结果 |
|------|------|------|
| **黑体辐射** | 经典物理 | 量子革命 |
| **水星近日点** | 牛顿力学 | 广义相对论 |
| **以太漂移** | 以太理论 | 相对论 |

**反常的作用**：

| 作用 | 描述 |
|------|------|
| **挑战** | 挑战现有范式 |
| **动力** | 推动理论发展 |
| **筛选** | 筛选更好的理论 |

**相关词条**：范式、危机、科学革命

---

### 【危机】(Crisis)

**定义**：反常积累导致的范式动摇状态。

**费曼解释**：
危机是"范式的动摇"：
- 反常不断积累
- 科学家开始怀疑范式
- 为革命创造条件

**危机的特征**：

| 特征 | 描述 |
|------|------|
| **反常积累** | 反常不断出现 |
| **信心动摇** | 对范式的信心动摇 |
| **尝试多样** | 尝试多种解决方案 |
| **开放变革** | 对变革开放 |

**危机的发展**：

```
反常 → 尝试解释 → 失败 → 更多反常 → 危机
```

**经典例子**：

| 危机 | 反常 | 结果 |
|------|------|------|
| **19世纪末物理** | 黑体辐射、以太漂移 | 量子论、相对论 |
| **化学危机** | 燃烧现象 | 氧化说 |
| **地质学危机** | 大陆分布 | 板块构造 |

**相关词条**：范式、反常、科学革命

### 【研究纲领】(Research Programme)

**定义**：拉卡托斯提出的科学理论发展单位，包含硬核和保护带。

**费曼解释**：
研究纲领是"理论系列"：
- 不是单个理论，而是一系列理论
- 有"硬核"（不可放弃）
- 有"保护带"（可调整）

**研究纲领的结构**：

| 组成部分 | 描述 |
|----------|------|
| **硬核** | 不可放弃的核心假设 |
| **保护带** | 可调整的辅助假设 |
| **正面启发法** | 如何发展纲领 |
| **反面启发法** | 如何保护硬核 |

**例子**：

| 研究纲领 | 硬核 | 保护带 |
|----------|------|--------|
| **牛顿力学** | 运动定律、引力定律 | 初始条件、辅助假设 |
| **弗洛伊德精神分析** | 无意识理论 | 具体案例解释 |

**进步 vs 退化纲领**：

| 类型 | 描述 |
|------|------|
| **理论进步** | 预测新事实 |
| **经验进步** | 新预测被确认 |
| **进步纲领** | 理论+经验进步 |
| **退化纲领** | 只能事后解释 |

**对波普尔的改进**：

| 问题 | 波普尔 | 拉卡托斯 |
|------|--------|----------|
| **检验单位** | 单个理论 | 研究纲领 |
| **证伪** | 立即证伪 | 需要时间 |
| **科学合理性** | 证伪规则 | 进步评价 |

**代表人物**：
- **拉卡托斯**：研究纲领概念创始人

**经典文献**：
- Lakatos, "Falsification and the Methodology of Scientific Research Programmes" (1970)

**相关词条**：硬核、保护带、科学进步

---

### 【硬核】(Hard Core)

**定义**：研究纲领中不可放弃的核心假设。

**费曼解释**：
硬核是"不可动摇的核心"：
- 牛顿力学的硬核：运动定律+引力定律
- 放弃硬核 = 放弃整个纲领
- 必须保护硬核

**硬核的特征**：

| 特征 | 描述 |
|------|------|
| **不可放弃** | 不能放弃 |
| **纲领定义** | 定义纲领 |
| **被保护** | 由保护带保护 |

**例子**：

| 研究纲领 | 硬核 |
|----------|------|
| **牛顿力学** | 运动定律、引力定律 |
| **相对论** | 相对性原理、光速不变 |
| **达尔文进化论** | 自然选择、共同祖先 |

**硬核的作用**：

| 作用 | 描述 |
|------|------|
| **定义纲领** | 定义什么是该纲领 |
| **指导研究** | 指导研究方向 |
| **限制调整** | 限制可接受的调整 |

**相关词条**：研究纲领、保护带、拉卡托斯

---

### 【保护带】(Protective Belt)

**定义**：研究纲领中可调整的辅助假设。

**费曼解释**：
保护带是"缓冲区"：
- 当反常出现时
- 调整保护带保护硬核
- 硬核不动，保护带动

**保护带的内容**：

| 内容 | 描述 |
|------|------|
| **辅助假说** | 辅助性假设 |
| **初始条件** | 初始条件假设 |
| **观察理论** | 观察相关假设 |

**例子**：

| 研究纲领 | 保护带调整 |
|----------|-----------|
| **牛顿力学** | 添加新行星解释天王星轨道 |
| **弗洛伊德理论** | 修改案例解释 |

**保护带的作用**：

| 作用 | 描述 |
|------|------|
| **保护硬核** | 保护硬核免受证伪 |
| **吸收反常** | 吸收反常现象 |
| **允许调整** | 允许理论调整 |

**相关词条**：研究纲领、硬核、辅助假说

---

### 【正面启发法】(Positive Heuristic)

**定义**：研究纲领中指导如何发展纲领的原则。

**费曼解释**：
正面启发法是"发展指南"：
- 告诉科学家应该做什么
- 定义了研究纲领的"研究纲领"
- 指导理论建构

**内容**：

| 内容 | 描述 |
|------|------|
| **研究问题** | 哪些问题重要 |
| **解决策略** | 如何解决问题 |
| **理论建构** | 如何建构理论 |

**例子**：

| 研究纲领 | 正面启发法 |
|----------|-----------|
| **牛顿力学** | 用力学解释所有现象 |
| **达尔文进化论** | 用自然选择解释适应 |

**正面启发法的作用**：

| 作用 | 描述 |
|------|------|
| **指导研究** | 指导研究方向 |
| **忽略反常** | 暂时忽略某些反常 |
| **自主发展** | 允许纲领自主发展 |

**相关词条**：研究纲领、反面启发法、拉卡托斯

---

### 【反面启发法】(Negative Heuristic)

**定义**：研究纲领中指导如何保护硬核的原则。

**费曼解释**：
反面启发法是"保护指南"：
- 告诉科学家不应该做什么
- 禁止修改硬核
- 指导如何处理反常

**内容**：

| 内容 | 描述 |
|------|------|
| **禁止修改硬核** | 不能放弃硬核 |
| **调整保护带** | 通过调整保护带处理反常 |
| **保护策略** | 如何保护硬核 |

**反面启发法的作用**：

| 作用 | 描述 |
|------|------|
| **保护硬核** | 保护纲领核心 |
| **处理反常** | 处理反常的方式 |
| **维持纲领** | 维持纲领连续性 |

**相关词条**：研究纲领、正面启发法、硬核

---

### 【进步纲领】(Progressive Programme)

**定义**：能够预测新事实并得到确认的研究纲领。

**费曼解释**：
进步纲领是"前进的纲领"：
- 能预测新现象
- 预测被确认
- 纲领在进步

**进步的标准**：

| 标准 | 描述 |
|------|------|
| **理论进步** | 预测新事实 |
| **经验进步** | 新预测被确认 |
| **启发力** | 有强的启发力 |

**vs 退化纲领**：

| 问题 | 进步纲领 | 退化纲领 |
|------|----------|----------|
| **预测** | 预测新事实 | 只事后解释 |
| **确认** | 预测被确认 | 预测失败 |
| **发展** | 不断发展 | 停滞不前 |

**例子**：

| 纲领 | 类型 | 原因 |
|------|------|------|
| **相对论** | 进步 | 预测并确认新效应 |
| **燃素说** | 退化 | 只能事后解释 |

**相关词条**：研究纲领、退化纲领、科学进步

---

### 【退化纲领】(Degenerating Programme)

**定义**：只能事后解释，无法预测新事实的研究纲领。

**费曼解释**：
退化纲领是"后退的纲领"：
- 不能预测新现象
- 只能事后解释
- 纲领在退化

**退化的标志**：

| 标志 | 描述 |
|------|------|
| **无新预测** | 不能预测新事实 |
| **事后解释** | 只能事后解释 |
| **反常积累** | 反常不断积累 |
| **启发力下降** | 启发力下降 |

**拉卡托斯的观点**：

- 科学家应该选择进步纲领
- 放弃退化纲领
- 但需要时间判断

**相关词条**：研究纲领、进步纲领、拉卡托斯

### 【库恩】(Thomas Kuhn)

**生平**：1922-1996，美国科学史家和科学哲学家

**贡献**：范式、科学革命、不可通约性

**主要理论**：

| 理论 | 描述 |
|------|------|
| **范式** | 科学共同体的共享框架 |
| **科学革命** | 范式转换 |
| **不可通约性** | 范式间不可比较 |
| **常规科学** | 范式内的科学 |

**经典文献**：
- Kuhn, *The Structure of Scientific Revolutions* (1962)
- Kuhn, *The Essential Tension* (1977)

**影响**：

| 领域 | 影响 |
|------|------|
| **科学哲学** | 挑战累积进步观 |
| **科学史** | 提供新视角 |
| **科学社会学** | 影响社会学发展 |

**相关词条**：范式、科学革命、不可通约性

---

### 【拉卡托斯】(Imre Lakatos)

**生平**：1922-1974，匈牙利裔英国科学哲学家

**贡献**：研究纲领、科学方法论

**主要理论**：

| 理论 | 描述 |
|------|------|

| **确认偏见** | 只寻找支持证据 |
| **模糊性** | 概念模糊 |
| **不进步** | 不进步发展 |

**例子**：

| 伪科学 | 描述 |
|--------|------|
| **占星术** | 星象预测命运 |
| **创世论** | 神创论 |
| **UFO学** | 外星人研究 |
| **灵学** | 超自然研究 |

**与科学的区别**：

| 问题 | 科学 | 伪科学 |
|------|------|--------|
| **可检验** | 可检验 | 不可检验 |
| **可证伪** | 可证伪 | 不可证伪 |
| **进步** | 进步 | 不进步 |
| **方法** | 严格方法 | 方法不严格 |

**相关词条**：科学划界、可证伪性、科学方法

---

### 【边缘科学】(Fringe Science)

**定义**：处于主流科学边缘的研究领域。

**费曼解释**：
边缘科学是"非主流科学"：
- 不被主流接受
- 可能是革命性的
- 也可能是错误的

**特征**：

| 特征 | 描述 |
|------|------|
| **非主流** | 不被主流接受 |
| **争议性** | 有争议 |
| **潜在革命性** | 可能是革命性的 |

**例子**：

| 边缘科学 | 描述 |
|----------|------|
| **冷聚变** | 室温核聚变 |
| **心灵感应** | 超感知 |
| **自由能** | 永动机 |

**评价**：

| 评价 | 描述 |
|------|------|
| **可能是革命** | 可能是新范式 |
| **可能是错误** | 可能是错的 |
| **需要时间** | 需要时间检验 |

**相关词条**：伪科学、范式、科学革命

---

### 【科学中的隐喻】(Metaphor in Science)

**定义**：科学理论和解释中使用的隐喻。

**费曼解释**：
科学隐喻是"类比和图像"：
- "原子是太阳系"
- "基因是蓝图"
- 隐喻帮助理解

**类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **结构隐喻** | 结构类比 | 原子-太阳系 |
| **功能隐喻** | 功能类比 | 心-计算机 |
| **解释隐喻** | 解释类比 | 自然选择 |

**作用**：

| 作用 | 描述 |
|------|------|
| **理解** | 帮助理解 |
| **发现** | 促进发现 |
| **沟通** | 促进沟通 |

**问题**：

| 问题 | 描述 |
|------|------|
| **误导** | 可能误导 |
| **限制** | 限制思维 |
| **字面化** | 可能被字面理解 |

**相关词条**：科学模型、类比、科学语言
