# 6.2 科学实在论 (Scientific Realism)

> **词条数**：50+
> **难度**：基础到进阶
> **更新**：2026-03-26

---

## A-D

### 【科学实在论】(Scientific Realism)

**定义**：认为成熟的科学理论是真的或近似真的，理论实体真实存在的哲学立场。

**费曼解释**：
科学实在论说：
- 科学理论描述真实的世界
- 电子、基因、黑洞真的存在
- 科学会越来越接近真理

**核心主张**：

| 要素 | 描述 |
|------|------|
| **本体论主张** | 理论实体真实存在 |
| **语义主张** | 理论是真的或假的 |
| **认识论主张** | 成熟理论近似为真 |

**科学实在论 vs 反实在论**：

| 问题 | 实在论 | 反实在论 |
|------|--------|----------|
| 理论实体 | 真实存在 | 只是工具 |
| 理论真理 | 理论是真的 | 只是经验适当 |
| 科学目标 | 发现真理 | 预测成功 |

**支持实在论的论证**：

| 论证 | 描述 |
|------|------|
| **无奇迹论证** | 如果科学不真，成功就是奇迹 |
| **操作论证** | 我们能操作理论实体 |
| **最佳解释** | 实在论最佳解释科学成功 |

**反对实在论的论证**：

| 论证 | 描述 |
|------|------|
| **悲观归纳** | 过去成功的理论后来被证明是假的 |
| **不充分决定** | 数据不能唯一决定理论 |
| **不可观察性** | 不可观察实体如何知道存在？ |

**代表人物**：
- **普特南**：无奇迹论证
- **波义德**：实在论辩护
- **塞拉斯**：科学实在论

**经典文献**：
- Putnam, "What Is Mathematical Truth?" (1975)
- Boyd, "The Current Status of Scientific Realism" (1983)

**相关词条**：反实在论、结构实在论、无奇迹论证

### 【反实在论】(Anti-Realism)

**定义**：否认科学理论描述独立实在的哲学立场。

**费曼解释**：
反实在论说：
- 理论只是工具，不必为真
- 电子可能不存在，但理论好用就行
- 科学的目标是经验成功，不是真理

**反实在论的类型**：

| 类型 | 核心主张 | 代表 |
|------|----------|------|
| **工具主义** | 理论只是预测工具 | 马赫 |
| **经验主义** | 只相信可观察的 | 范弗拉森 |
| **建构经验主义** | 理论只需经验适当 | 范弗拉森 |
| **社会建构主义** | 科学是社会建构 | 某些社会学家 |

**工具主义**：

**核心主张**：
- 理论是预测的"工具"
- 不必假设理论实体存在
- 问"理论是否为真"是错误的

**建构经验主义**：

**范弗拉森的主张**：
- 科学的目标是经验适当性，不是真理
- 接受理论 = 相信它经验适当
- 不必相信不可观察实体存在

**可观察性标准**：

| 可观察 | 不可观察 |
|--------|----------|
| 桌子 | 电子 |
| 行星 | 夸克 |
| 细胞 | 量子场 |

**相关词条**：科学实在论、建构经验主义、工具主义

### 【建构经验主义】(Constructive Empiricism)

**定义**：范弗拉森提出的反实在论立场——科学的目标是经验适当性。

**费曼解释**：
建构经验主义说：
- 科学家不试图发现真理
- 他们试图构建经验适当的理论
- 接受理论 ≠ 相信它为真

**核心主张**：

| 要素 | 描述 |
|------|------|
| **科学目标** | 经验适当性，不是真理 |
| **接受理论** | 相信经验适当 |
| **可观察性** | 区分可观察/不可观察 |
| **反形而上学** | 不问理论是否为真 |

**经验适当性**：

**定义**：理论关于可观察事物的断言都是真的。

**例子**：
- 理论说"存在电子"→ 不必相信
- 理论说"云室中有轨迹"→ 这是可观察的，应该相信

**可观察性**：

**范弗拉森的标准**：
- 可观察 = 人类感官能检测（包括仪器辅助）
- 不可观察 = 超出人类感官能力

**边界案例**：
- 细菌？可观察（显微镜）
- 电子？不可观察

**对建构经验主义的批评**：

| 批评 | 描述 |
|------|------|
| **可观察性模糊** | 可观察/不可观察边界不清 |
| **依赖理论** | 观察依赖理论 |
| **接受概念** | "接受"到底是什么意思？ |
| **真理概念** | 为什么要区分经验适当和真理？ |

**代表人物**：
- **范弗拉森**：建构经验主义创始人

**经典文献**：
- van Fraassen, *The Scientific Image* (1980)

**相关词条**：反实在论、经验主义、可观察性

### 【无奇迹论证】(No Miracles Argument)

**定义**：支持科学实在论的论证——科学的成功若是奇迹，不如相信科学是真的。

**费曼解释**：
无奇迹论证说：
- 科学预测如此准确
- 如果理论不是真的，这就像奇迹
- 所以理论应该是真的

**论证结构**：

```
前提1：科学理论在预测和应用上非常成功
前提2：如果理论不是（近似）真的，成功就是奇迹
前提3：奇迹不存在（或应该避免）
结论：因此，科学理论是（近似）真的
```

**普特南的表述**：

"实在论的积极论证是，它是唯一不使科学的成功成为奇迹的哲学。"

**例子**：

| 成功 | 实在论解释 | 反实在论解释 |
|------|-----------|-------------|
| **量子力学** | 量子是真的 | 工具好用 |
| **DNA双螺旋** | DNA真的那样 | 模型好用 |
| **疫苗** | 病毒理论为真 | 预测成功 |

**对无奇迹论证的批评**：

| 批评 | 描述 |
|------|------|
| **悲观归纳** | 过去的成功理论后来被证明是假的 |
| **成功选择性** | 只看到成功的理论，忽略失败的 |
| **其他解释** | 可能有其他解释成功的原因 |

**代表人物**：
- **普特南**：论证提出者
- **波义德**：论证发展

**相关词条**：科学实在论、悲观归纳、科学成功

---

### 【悲观归纳】(Pessimistic Induction)

**定义**：反对科学实在论的论证——过去成功的理论后来被证明是假的。

**费曼解释**：
悲观归纳说：
- 历史上有很多成功的理论
- 它们后来都被证明是假的
- 所以，今天的成功理论可能也是假的

**论证结构**：

```
前提1：过去有很多成功的理论（如地心说、热质说、以太论）
前提2：这些理论后来被证明是假的
前提3：我们没有理由认为今天不同
结论：因此，今天的成功理论可能也是假的
```

**历史例子**：

| 理论 | 时期 | 成功 | 结果 |
|------|------|------|------|
| **地心说** | 古代-文艺复兴 | 解释行星运动 | 被日心说取代 |
| **燃素说** | 18世纪 | 解释燃烧 | 被氧化说取代 |
| **热质说** | 18世纪 | 解释热现象 | 被热力学取代 |
| **以太论** | 19世纪 | 解释光传播 | 被相对论取代 |

**劳丹的论证**：

劳丹列举了历史上成功但为假的理论，质疑实在论。

**实在论的回应**：

| 回应 | 策略 |
|------|------|
| **选择性悲观归纳** | 只选择部分理论 |
| **结构保留** | 旧理论的部分被保留 |
| **近似真理** | 旧理论是近似真的 |
| **分界** | 区分成熟和不成熟科学 |

**结构实在论**：

**沃勒尔的方案**：
- 保留理论的结构
- 不是理论实体的存在
- 如：菲涅尔的光学方程被保留

**代表人物**：
- **劳丹**：悲观归纳提出者
- **沃勒尔**：结构实在论

**经典文献**：
- Laudan, "A Confutation of Convergent Realism" (1981)
- Worrall, "Structural Realism" (1989)

**相关词条**：无奇迹论证、科学实在论、理论变迁

### 【结构实在论】(Structural Realism)

**定义**：认为科学理论捕获实在的结构，而不是实体本身的立场。

**费曼解释**：
结构实在论说：
- 我们不能说电子"真的"存在
- 但我们可以说电子之间的"关系"是真的
- 科学捕获的是结构，不是实体

**核心主张**：

| 要素 | 描述 |
|------|------|
| **结构优先** | 结构是知识的核心 |
| **关系真实** | 关系是真实的 |
| **实体怀疑** | 对实体存在保持怀疑 |
| **连续性** | 理论变迁中结构有连续性 |

**类型**：

| 类型 | 核心主张 | 代表 |
|------|----------|------|
| **认识论结构实在论** | 我们只能知道结构 | 沃勒尔 |
| **本体论结构实在论** | 只有结构存在 | 弗伦奇、Ladyman |
| **部分结构实在论** | 部分结构是真实的 | 查克拉瓦蒂 |

**历史案例：菲涅尔-麦克斯韦**：

| 理论 | 实体 | 结构 |
|------|------|------|
| **菲涅尔光学** | 以太 | 光的方程 |
| **麦克斯韦电磁学** | 电磁场 | 同样的方程 |

**观察**：
- 实体变了（以太→电磁场）
- 但结构保留（方程相同）
- 因此结构是真实的

**对结构实在论的批评**：

| 批评 | 描述 |
|------|------|
| **结构需要载体** | 结构不能没有实体 |
| **如何区分结构** | 什么是"结构"？ |
| **不够强** | 似乎放弃了太多 |

**代表人物**：
- **沃勒尔**：认识论结构实在论
- **弗伦奇**：本体论结构实在论
- **Ladyman**：本体论结构实在论

**经典文献**：
- Worrall, "Structural Realism" (1989)
- Ladyman, "What is Structural Realism?" (2007)

**相关词条**：科学实在论、理论连续性、结构主义

### 【实体实在论】(Entity Realism)

**定义**：相信我们可以因果操作的理论实体存在的立场。

**费曼解释**：
实体实在论说：
- 我不关心理论是否为真
- 但如果我能"操作"电子
- 那电子应该存在

**核心主张**：

| 要素 | 描述 |
|------|------|
| **因果操作** | 能因果操作的实体存在 |
| **理论中立** | 不必相信整个理论 |
| **实践基础** | 基于科学实践 |

**哈金的论证**：

**核心主张**：
- 如果我们能用电子做事情
- 如果我们能干预电子的行为
- 那电子必须存在

**例子**：

| 实体 | 如何操作 |
|------|----------|
| **电子** | 在加速器中偏转电子 |
| **基因** | 修改基因改变性状 |
| **病毒** | 用病毒感染细胞 |

**实体实在论 vs 理论实在论**：

| 问题 | 实体实在论 | 理论实在论 |
|------|-----------|-----------|
| 理论真理 | 不必相信 | 应该相信 |
| 实体存在 | 基于操作相信 | 基于理论相信 |
| 范围 | 更窄 | 更宽 |

**对实体实在论的批评**：

| 批评 | 描述 |
|------|------|
| **操作概念** | "操作"概念模糊 |
| **理论依赖** | 操作依赖理论 |
| **不够一致** | 为什么只相信部分？ |

**代表人物**：
- **哈金**：实体实在论创始人

**经典文献**：
- Hacking, *Representing and Intervening* (1983)

**相关词条**：科学实在论、因果操作、科学实践

### 【工具主义】(Instrumentalism)

**定义**：认为科学理论只是预测工具，不必描述实在的立场。

**费曼解释**：
工具主义说：
- 理论像地图，好用就行
- 不必问地图是否"真实"
- 电子只是计算工具

**核心主张**：

| 要素 | 描述 |
|------|------|
| **工具性** | 理论是工具 |
| **预测导向** | 成功 = 预测成功 |
| **反实在论** | 不问理论是否为真 |
| **实用主义** | 实用即好 |

**工具主义 vs 实在论**：

| 问题 | 工具主义 | 实在论 |
|------|----------|--------|
| 理论 | 预测工具 | 描述实在 |
| 电子 | 计算工具 | 真实存在 |
| 真理 | 不重要 | 重要目标 |

**支持工具主义的理由**：

| 理由 | 描述 |
|------|------|
| **实用成功** | 工具主义也能解释成功 |
| **避免形而上学** | 不需要形而上学承诺 |
| **历史教训** | 成功理论也会被取代 |

**对工具主义的批评**：

| 批评 | 描述 |
|------|------|
| **无法解释** | 无法解释为什么工具好用 |
| **科学实践** | 科学家确实相信实体存在 |
| **理论选择** | 没有标准选择工具 |

**代表人物**：
- **马赫**：现象主义
- **杜恒**：工具主义先驱

**相关词条**：反实在论、建构经验主义、实用主义

### 【理论实体】(Theoretical Entity)

**定义**：科学理论假设存在的、不可直接观察的实体。

**费曼解释**：
理论实体是"看不见的东西"：
- 电子、质子
- 基因、DNA
- 夸克、玻色子

**理论实体的例子**：

| 领域 | 理论实体 |
|------|----------|
| **物理学** | 电子、夸克、光子、引力波 |
| **化学** | 原子、分子、化学键 |
| **生物学** | 基因、病毒、蛋白质 |
| **地质学** | 板块、地幔对流 |

**理论实体的存在问题**：

| 立场 | 对理论实体的看法 |
|------|-----------------|
| **实在论** | 真实存在 |
| **反实在论** | 只是工具 |
| **结构实在论** | 结构真实，实体存疑 |
| **实体实在论** | 可操作的实体存在 |

**指称问题**：

- 理论术语真的"指称"理论实体吗？
- 如果理论是假的，术语还指称吗？

**相关词条**：科学实在论、不可观察性、理论术语

### 【可观察性】(Observability)

**定义**：某物能否被人类感官（包括仪器辅助）感知的性质。

**费曼解释**：
可观察性问：
- 什么是可观察的？
- 电子可观察吗？
- 可观察和不可观察的边界在哪里？

**可观察性的标准**：

| 立场 | 标准 |
|------|------|
| **范弗拉森** | 人类感官能力（包括仪器辅助） |
| **实在论者** | 可观察性是连续的 |
| **哈金** | 因果可干预性 |

**边界案例**：

| 实体 | 可观察性 |
|------|----------|
| **桌子** | 直接可观察 |
| **细菌** | 可观察（显微镜） |
| **病毒** | 边界案例 |
| **电子** | 不可观察？ |

**对可观察性标准的批评**：

| 批评 | 描述 |
|------|------|
| **模糊边界** | 边界不清晰 |
| **理论依赖** | 观察依赖理论 |
| **连续性** | 可观察性是连续的 |
| **人类中心** | 为什么以人类感官为标准？ |

**相关词条**：建构经验主义、观察、理论实体

### 【近似真理】(Approximate Truth)

**定义**：理论虽然不是完全为真，但足够接近真理的概念。

**费曼解释**：
近似真理说：
- 牛顿力学不是完全正确的
- 但它是"近似"正确的
- 在低速情况下，它足够好

**近似真理的问题**：

| 问题 | 描述 |
|------|------|
| **定义** | 什么是"近似真理"？ |
| **度量** | 如何度量近似程度？ |
| **比较** | 如何比较两个理论的近似程度？ |

**理论序列**：

| 理论 | 近似真理程度 |
|------|-------------|
| **地心说** | 较低 |
| **日心说** | 较高 |
| **广义相对论** | 更高 |

**波普尔的逼真度**：

**定义**：理论的真理内容减去虚假内容。

**问题**：如何形式化？

**相关词条**：科学实在论、真理、理论进步

---

### 【逼真度】(Verisimilitude)

**定义**：理论接近真理的程度。

**费曼解释**：
逼真度问：
- 如何度量理论"接近真理"？
- 牛顿力学比地心说更接近真理吗？
- 如何定义"更接近"？

**波普尔的定义**：

```
逼真度 = 真理内容 - 虚假内容
```

**问题**：

| 问题 | 描述 |
|------|------|
| **定义困难** | 形式定义有问题 |
| **比较困难** | 难以比较两个理论 |
| **应用困难** | 难以应用于实际理论 |

**米勒的问题**：

米勒证明波普尔的定义有问题：一个理论可能比另一个更接近真理，但在波普尔的意义上却不是。

**相关词条**：近似真理、科学进步、波普尔

### 【科学成功】(Success of Science)

**定义**：科学在预测、解释和技术应用上的成就。

**费曼解释**：
科学成功是显而易见的：
- 预测日食
- 制造药物
- 发送火箭到火星

**成功的方面**：

| 方面 | 例子 |
|------|------|
| **预测** | 预测行星位置、粒子存在 |
| **解释** | 解释引力、进化 |
| **技术** | 计算机、疫苗、核能 |
| **统一** | 统一电磁力 |

**对成功的解释**：

| 立场 | 解释 |
|------|------|
| **实在论** | 成功因为理论为真 |
| **反实在论** | 成功因为选择过程 |
| **自然主义** | 进化解释成功 |

**选择性成功**：

- 很多理论失败了
- 我们只看到成功的
- 这是选择偏倚

**相关词条**：无奇迹论证、悲观归纳、科学实在论

### 【经验适当性】(Empirical Adequacy)

**定义**：理论关于可观察现象的断言全部为真的性质。

**费曼解释**：
经验适当性说：
- 理论可能不是完全为真
- 但关于可观察事物的断言是真的
- 这就足够了

**核心主张**：

| 要素 | 描述 |
|------|------|
| **可观察** | 只关于可观察事物 |
| **真断言** | 所有可观察断言为真 |
| **理论实体** | 不必相信 |

**与真理的区别**：

| 概念 | 要求 |
|------|------|
| **真理** | 所有断言为真（包括不可观察） |
| **经验适当** | 只有可观察断言为真 |

**例子**：

- 理论说"电子存在"→ 不必为真
- 理论说"云室中有轨迹"→ 必须为真

**相关词条**：建构经验主义、可观察性、真理

### 【指称】(Reference)

**定义**：语言表达式与世界中的对象之间的关系。

**费曼解释**：
指称问：
- "电子"这个词指称什么？
- 如果理论是假的，"电子"还指称吗？
- 理论术语如何获得指称？

**指称理论**：

| 理论 | 描述 | 代表 |
|------|------|------|
| **描述论** | 指称由描述确定 | 弗雷格、罗素 |
| **因果论** | 指称由因果链确定 | 克里普克、普特南 |

**描述论**：

**核心主张**：
- "电子"指称那个满足电子描述的东西
- 如果描述错误，可能没有指称

**问题**：
- 如果描述错误，术语可能没有指称

**因果理论**：

**核心主张**：
- 指称由命名时的因果链确定
- 描述可以错误，但指称保持

**例子**：
- "电子"在汤姆逊时代被命名
- 即使汤姆逊的描述错了，"电子"仍然指称电子

**对实在论的意义**：

| 问题 | 描述论 | 因果理论 |
|------|--------|----------|
| **理论变迁** | 旧术语可能没指称 | 旧术语仍有指称 |
| **实在论支持** | 不利 | 有利 |

**相关词条**：理论术语、意义、因果指称论

### 【因果指称论】(Causal Theory of Reference)

**定义**：指称由命名时的因果链确定，而非由描述确定的理论。

**费曼解释**：
因果指称论说：
- "电子"指称什么，不取决于我们如何描述电子
- 而是取决于命名时的因果链
- 即使描述错了，指称保持

**核心主张**：

| 要素 | 描述 |
|------|------|
| **初始命名** | 某时刻被命名 |
| **因果链** | 指称通过因果链传播 |
| **描述不决定** | 描述可以错误 |
| **严格性** | 专有名词是严格指示词 |

**克里普克的论证**：

**哥德尔案例**：
- 假设哥德尔不是证明不完全性定理的人
- 而是史密斯证明的，哥德尔窃取了
- "哥德尔"仍然指称哥德尔，不是史密斯

**结论**：描述不决定指称。

**对科学实在论的意义**：

| 问题 | 描述论 | 因果理论 |
|------|--------|----------|
| **悲观归纳** | 旧理论术语没指称 | 旧理论术语仍有指称 |
| **理论连续性** | 无连续性 | 有连续性 |

**例子**：

| 术语 | 最初描述 | 因果链保留指称 |
|------|----------|---------------|
| **电子** | 汤姆逊的描述 | 仍指称电子 |
| **原子** | 德谟克利特的描述 | 仍指称原子 |

**代表人物**：
- **克里普克**：因果指称论创始人
- **普特南**：自然种类词

**经典文献**：
- Kripke, *Naming and Necessity* (1980)
- Putnam, "The Meaning of 'Meaning'" (1975)

**相关词条**：指称、描述论、自然种类

### 【描述论】(Descriptivism)

**定义**：指称由描述确定的理论。

**费曼解释**：
描述论说：
- "电子"指称那个满足电子描述的东西
- 如果描述是"带负电的最小粒子"
- 那么"电子"指称带负电的最小粒子

**核心主张**：

| 要素 | 描述 |
|------|------|
| **描述决定指称** | 满足描述的对象是所指 |
| **描述是意义** | 描述构成词的意义 |
| **认知价值** | 认知上可理解 |

**弗雷格-罗素理论**：

**弗雷格**：指称由"涵义"确定
**罗素**：专有名词是伪装的描述

**问题**：

| 问题 | 描述 |
|------|------|
| **认知问题** | 不同人有不同描述 |
| **空名问题** | 描述无对象时怎么办？ |
| **模态问题** | 描述可能不必然为真 |

**对科学的意义**：

如果描述论正确：
- 旧理论的描述可能错误
- 旧理论术语可能没有指称
- 支持悲观归纳

**相关词条**：指称、因果指称论、意义

### 【自然种类】(Natural Kinds)

### 【自然主义】(Naturalism)

**定义**：认为一切都可以用自然原因解释，不需要超自然解释的哲学立场。

**费曼解释**：
自然主义说：
- 世界是自然的
- 不需要神或灵魂来解释
- 科学是理解世界的途径

**类型**：

| 类型 | 描述 |
|------|------|
| **本体论自然主义** | 只有自然存在 |
| **方法论自然主义** | 科学方法假设自然主义 |
| **认识论自然主义** | 知识来自自然方法 |

**核心主张**：

| 要素 | 描述 |
|------|------|
| **无超自然** | 不需要超自然解释 |
| **因果封闭** | 自然因果封闭 |
| **科学方法** | 科学是认识方法 |

**在科学哲学中的角色**：

| 问题 | 自然主义观点 |
|------|-------------|
| **方法论** | 科学方法论是自然的一部分 |
| **认识论** | 认识论应该自然化 |
| **心身问题** | 心灵是自然现象 |

**对自然主义的批评**：

| 批评 | 描述 |
|------|------|
| **循环问题** | 自然主义假设自然主义？ |
| **规范问题** | 自然主义难以处理规范 |
| **意识** | 意识难以自然化？ |

**代表人物**：
- **奎因**：自然化认识论
- **杜威**：实用主义自然主义

**相关词条**：科学主义、科学方法、认识论

### 【真理】(Truth)

**定义**：陈述与世界之间符合的关系。

**费曼解释**：
真理是"符合事实"：
- "雪是白色的"是真的，因为雪确实是白色的
- 科学追求真理

**真理理论**：

| 理论 | 描述 | 代表 |
|------|------|------|
| **符合论** | 真理是符合事实 | 罗素、早期维特根斯坦 |
| **融贯论** | 真理是与信念系统融贯 | 布兰夏德 |
| **实用主义** | 真理是有用的 | 詹姆斯、杜威 |
| **紧缩论** | "真"只是逻辑工具 | 拉姆齐 |

**符合论**：

**核心主张**：陈述为真当且仅当它符合事实。

**问题**：
- 什么是"符合"？
- 什么是"事实"？

**紧缩论**：

**核心主张**："真"只是逻辑工具，"雪是白色的"是真的 = 雪是白色的。

**科学中的真理**：

| 问题 | 描述 |
|------|------|
| **科学目标** | 科学追求真理？ |
| **近似真理** | 理论是近似真的？ |
| **实在论** | 实在论关于真理？ |

**相关词条**：科学实在论、符合论、意义

### 【符合论】(Correspondence Theory)

**定义**：真理是陈述与世界符合关系的理论。

**费曼解释**：
符合论说：
- "雪是白色的"是真的
- 因为雪确实是白色的
- 陈述与世界"符合"

**核心主张**：

| 要素 | 描述 |
|------|------|
| **符合关系** | 真理是符合关系 |
| **事实** | 存在客观事实 |
| **独立于信念** | 真理独立于信念 |

**形式化**：

```
命题P为真 ↔ P
"雪是白色的"为真 ↔ 雪是白色的
```

**问题**：

| 问题 | 描述 |
|------|------|
| **什么是符合** | 如何定义"符合"？ |
| **事实问题** | 什么是"事实"？ |
| **不可观察** | 不可观察的真理如何符合？ |

**对科学实在论的意义**：

- 实在论通常接受符合论
- 理论为真 = 理论符合世界

**相关词条**：真理、科学实在论、融贯论

---

### 【融贯论】(Coherence Theory)

**定义**：真理是信念系统内部融贯的理论。

**费曼解释**：
融贯论说：
- 单个信念不真不假
- 只有整个信念系统融贯
- 真理是融贯性

**核心主张**：

| 要素 | 描述 |
|------|------|
| **系统** | 真理属于信念系统 |
| **融贯** | 信念互相支持 |
| **整体** | 整体评价 |

**融贯的标准**：

| 标准 | 描述 |
|------|------|
| **一致性** | 不矛盾 |
| **联系** | 信念互相联系 |
| **解释力** | 能解释多 |

**问题**：

| 问题 | 描述 |
|------|------|
| **隔离系统** | 多个融贯系统可能隔离 |
| **经验输入** | 如何处理经验输入？ |
| **循环** | 融贯可能循环 |

**相关词条**：真理、符合论、整体论

---

### 【实用主义真理观】(Pragmatic Theory of Truth)

**定义**：真理是有用的或成功的信念的理论。

**费曼解释**：
实用主义说：
- "真"的就是"有用的"
- 真理在实践中验证
- 真理是成功的指导

**核心主张**：

| 要素 | 描述 |
|------|------|
| **实用性** | 真理是有用的 |
| **验证** | 在实践中验证 |
| **成功** | 真理导致成功 |

**詹姆斯的表述**：

"真理是观念的一种，它成为真的，是被事件使之成为真的。"

**问题**：

| 问题 | 描述 |
|------|------|
| **有用≠真** | 有用不一定是真 |
| **主观性** | "有用"可能主观 |
| **长期vs短期** | 短期有用可能长期有害 |

**相关词条**：真理、实用主义、工具主义

---

### 【紧缩论】(Deflationism)

**定义**：认为"真"只是逻辑工具，没有深层形而上学意义的理论。

**费曼解释**：
紧缩论说：
- "雪是白色的"是真的
- 就是"雪是白色的"
- "真的"没增加任何内容

**核心主张**：

| 要素 | 描述 |
|------|------|
| **逻辑工具** | "真"只是逻辑工具 |
| **等价** | "P是真的"等价于"P" |
| **无本质** | 真理没有本质 |

**等价图式**：

```
"P"是真的 ↔ P
```

**类型**：

| 类型 | 描述 |
|------|------|
| **冗余论** | "真"是冗余的 |
| **去引号论** | 真理是去引号 |
| **代语句论** | 真理是代语句 |

**对科学哲学的意义**：

| 问题 | 紧缩论观点 |
|------|-----------|
| **科学真理** | 没有特别的"科学真理" |
| **实在论争论** | 争论可能没意义 |

**相关词条**：真理、冗余论、科学哲学

### 【科学实在论争论】(Scientific Realism Debate)

**定义**：关于科学理论是否描述实在的哲学争论。

**费曼解释**：
科学实在论争论问：
- 科学描述真实世界吗？
- 电子真的存在吗？
- 理论是真的还是只是工具？

**争论双方**：

| 立场 | 核心主张 |
|------|----------|
| **实在论** | 理论描述真实世界 |
| **反实在论** | 理论只是工具 |
| **结构实在论** | 结构真实 |
| **实体实在论** | 可操作实体真实 |

**主要论证**：

| 论证 | 支持方 |
|------|--------|
| **无奇迹论证** | 实在论 |
| **悲观归纳** | 反实在论 |
| **不充分决定** | 反实在论 |
| **操作论证** | 实体实在论 |

**当代发展 (2020-2026)**：

| 发展 | 描述 |
|------|------|
| **具体化** | 更具体的实在论版本 |
| **自然化** | 自然化实在论 |
| **语境化** | 语境敏感的实在论 |
| **多元论** | 实在论多元论 |

**相关词条**：科学实在论、反实在论、无奇迹论证
