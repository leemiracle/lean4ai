### 【科学方法】(Scientific Method)

**定义**：科学家用于获取、验证和发展科学知识的一套系统性程序和原则。

**费曼解释**：
科学方法就像一张寻宝地图：
- **观察**：发现有趣的现象
- **假说**：提出可能的解释
- **实验**：设计测试来验证
- **结论**：根据结果修正理解

它不是僵死的规则，而是不断修正的过程。

**科学方法的核心要素**：

| 要素 | 描述 | 例子 |
|------|------|------|
| **观察** | 系统地观察现象 | 观察行星运动 |
| **假说** | 提出可检验的解释 | 引力假说 |
| **预测** | 从假说推导可观察的后果 | 预测行星位置 |
| **实验** | 控制条件下测试预测 | 观测验证 |
| **修正** | 根据结果修正假说 | 调整理论 |

**科学方法的模型**：

| 模型 | 核心主张 | 代表 |
|------|----------|------|
| **归纳主义** | 从观察归纳出定律 | 培根 |
| **假说演绎** | 从假说演绎预测 | 波普尔 |
| **贝叶斯主义** | 概率更新信念 | 当代哲学家 |

**相关词条**：归纳、假说演绎、贝叶斯主义

### 【归纳】(Induction)

**词源**：拉丁语 *inductio* (引入、诱导)

**定义**：从特殊观察推出一般结论的推理方法。

**费曼解释**：
归纳就像这样：
- 观察到1000只天鹅都是白色的
- 得出结论：所有天鹅都是白色的

问题：第1001只天鹅可能是黑色的！

**归纳的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **简单枚举归纳** | 从多个例子推出一般规律 | 所有观察到的A都是B |
| **排除归纳** | 排除其他可能性 | 密尔排除法 |
| **统计归纳** | 从样本推断总体 | 民意调查 |
| **类比归纳** | 从相似性推断 | 类比推理 |

**休谟的归纳问题**：

**核心问题**：归纳推理的合理性如何证明？

**休谟的分析**：
1. 归纳不能从逻辑上证明（归纳结论不是必然的）
2. 归纳不能从经验上证明（会循环论证）
3. 归纳只是"习惯"或"心理预期"

**休谟问题的结构**：

```
前提1：过去A总是伴随着B
前提2：未来类似于过去
结论：未来A将伴随着B
```

**问题**：前提2如何证明？

**对休谟问题的回应**：

| 回应 | 策略 | 代表 |
|------|------|------|
| **先验辩护** | 归纳是理性的先验原则 | 康德 |
| **实用辩护** | 归纳是成功的方法 | Reichenbach |
| **可靠性辩护** | 如果过程可靠，就是合理的 | Goldman |
| **接受问题** | 归纳问题是无解的 | 某些哲学家 |

**古德曼的"绿蓝"问题**：

**定义**：
- "绿蓝" = 在时间t之前是绿色的，之后是蓝色的

**问题**：
- 所有观察到的祖母绿都是绿色的
- 所有观察到的祖母绿也都是"绿蓝"的
- 为什么我们预测"绿色"而不是"绿蓝"？

**意义**：揭示了归纳的投射问题。

**代表人物**：
- **休谟**：归纳问题提出者
- **古德曼**：绿蓝问题
- **Reichenbach**：实用辩护

**经典文献**：
- Hume, *An Enquiry Concerning Human Understanding* (1748)
- Goodman, *Fact, Fiction, and Forecast* (1955)
- Reichenbach, *The Theory of Probability* (1949)

**相关词条**：休谟问题、假说演绎、科学方法

### 【假说演绎法】(Hypothetico-Deductive Method)

**定义**：从假说演绎出可观察的预测，然后通过实验检验的科学方法。

**费曼解释**：
假说演绎法就像破案：
1. 提出假说："凶手是管家"
2. 演绎预测："管家的衣服上应该有血迹"
3. 检验：检查管家的衣服
4. 结论：如果预测被证实，假说得到支持；如果被证伪，假说被拒绝

**基本结构**：

```
假说 H
H → 预测 P（演绎）
观察到 P（或非P）
结论：H 被支持（或被拒绝）
```

**形式化表述**：

**确认**：
```
H → P
P
──────────
H（被确认）
```

**证伪**：
```
H → P
¬P
──────────
¬H（被证伪）
```

**H-D方法的问题**：

| 问题 | 描述 |
|------|------|
| **确认的统计问题** | 多个假说可以预测同一观察 |
| **迪昂-奎因问题** | 假说不单独被检验 |
| **理论负荷** | 观察依赖理论 |
| **归纳问题** | 确认仍是归纳问题 |

**迪昂-奎因论题**：

**核心主张**：假说不能单独被检验。

**原因**：
- 预测是从"假说 + 辅助假说"推导的
- 如果预测失败，可能是假说错，也可能是辅助假说错

**例子**：
- 预测行星位置失败
- 可能是引力理论错
- 也可能是望远镜校准错

**当代发展 (2020-2026)**：
- **贝叶斯H-D**：贝叶斯框架下的H-D方法
- **证据理论**：新证据理论
- **模型检验**：计算机模拟与假说检验

**代表人物**：
- **波普尔**：证伪主义
- **迪昂**：整体论
- **奎因**：整体论发展

**经典文献**：
- Popper, *The Logic of Scientific Discovery* (1959)
- Duhem, *The Aim and Structure of Physical Theory* (1906)
- Quine, "Two Dogmas of Empiricism" (1951)

**相关词条**：证伪、迪昂-奎因论题、科学方法

### 【贝叶斯主义】(Bayesianism)

**定义**：用贝叶斯定理来理解科学推理，特别是证据如何更新假说概率的观点。

**费曼解释**：
贝叶斯主义说：
- 我们对假说的相信程度是概率
- 新证据会更新这个概率
- 更新规则就是贝叶斯定理

**贝叶斯定理**：

$$P(H|E) = \frac{P(E|H) \cdot P(H)}{P(E)}$$

| 符号 | 含义 |
|------|------|
| $P(H)$ | 先验概率（证据前对H的相信程度） |
| $P(H|E)$ | 后验概率（证据后对H的相信程度） |
| $P(E|H)$ | 似然性（如果H为真，E的概率） |
| $P(E)$ | 证据的边际概率 |

**贝叶斯更新的例子**：

1. **先验**：医生认为病人有50%可能患病
2. **证据**：检测结果为阳性
3. **更新**：
   - 如果患病，90%概率阳性
   - 如果不患病，10%概率假阳性
   - 后验概率 = ?

**贝叶斯科学哲学的优势**：

| 优势 | 描述 |
|------|------|
| **量化证据** | 证据对假说的支持程度可量化 |
| **解释确认** | 解释为什么证据确认假说 |
| **处理不确定性** | 自然处理不确定性 |
| **主观概率** | 允许主观先验 |

**贝叶斯主义的问题**：

| 问题 | 描述 |
|------|------|
| **先验问题** | 先验概率如何确定？ |
| **旧证据问题** | 旧证据如何确认假说？ |
| **问题** | 如何处理新异假说？ |
| **计算复杂性** | 实际计算可能很复杂 |

**先验问题**：

**核心问题**：先验概率从哪来？

| 立场 | 主张 |
|------|------|
| **主观贝叶斯** | 先验是主观的，最终会收敛 |
| **客观贝叶斯** | 先验应该是客观的（如最大熵） |
| **经验贝叶斯** | 先验应该基于经验 |

**旧证据问题**：

**问题**：如果证据E在提出假说H之前就已知，P(E)=1，贝叶斯更新不产生变化。

**回应**：
- 使用"预测性"解释
- 考虑"如果当时不知道E"

**当代发展 (2020-2026)**：
- **因果贝叶斯**：因果推理与贝叶斯
- **深度学习贝叶斯**：贝叶斯神经网络
- **科学贝叶斯**：科学实践的贝叶斯模型

**代表人物**：
- **贝叶斯**：贝叶斯定理提出者
- **Ramsey**：主观概率
- **de Finetti**：主观概率发展
- **Earman**：贝叶斯科学哲学

**经典文献**：
- Bayes, "An Essay towards solving a Problem in the Doctrine of Chances" (1763)
- Earman, *Bayes or Bust?* (1992)
- Howson & Urbach, *Scientific Reasoning: The Bayesian Approach* (2006)

**相关词条**：科学方法、概率、证据

### 【证伪主义】(Falsificationism)

**定义**：波普尔提出的科学方法论，强调科学理论必须可证伪，科学通过不断证伪而进步。

**费曼解释**：
证伪主义说：
- 你不能证明一个理论是正确的
- 但你可以证明它是错误的
- 科学的标志是"可证伪性"

就像黑天鹅：看到1000只白天鹅不能证明"所有天鹅都是白色的"，但看到1只黑天鹅就能证伪它。

**核心主张**：

| 要素 | 描述 |
|------|------|
| **可证伪性** | 科学理论必须是可证伪的 |
| **大胆 conjecture** | 科学家大胆提出假说 |
| **严格检验** | 努力证伪假说 |
| **淘汰错误** | 被证伪的理论被淘汰 |

**科学与非科学的划界**：

| 标准 | 波普尔的判断 |
|------|-------------|
| **可证伪性** | 科学的标志 |
| **弗洛伊德精神分析** | 不可证伪，不是科学 |
| **马克思主义（教条化后）** | 不可证伪，不是科学 |
| **爱因斯坦相对论** | 可证伪，是科学 |

**波普尔的问题**：

1. **迪昂-奎因问题**：不能单独证伪一个假说
2. **实际科学实践**：科学家不会轻易放弃理论
3. **概率理论**：概率陈述如何证伪？

**拉卡托斯的改进**：

**科学研究纲领**：
- 科学不是单个理论，而是"研究纲领"
- 研究纲领有"硬核"（不可放弃）和"保护带"（可调整）
- 评价研究纲领的"进步"或"退化"

**代表人物**：
- **波普尔**：证伪主义创始人
- **拉卡托斯**：科学研究纲领

**经典文献**：
- Popper, *The Logic of Scientific Discovery* (1959)
- Popper, *Conjectures and Refutations* (1963)
- Lakatos, "Falsification and the Methodology of Scientific Research Programmes" (1970)

**相关词条**：假说演绎、科学划界、研究纲领

### 【确证】(Confirmation)

**定义**：证据支持或确证假说的关系。

**费曼解释**：
确证问的是：
- 什么情况下证据"支持"假说？
- 观察到E，H变得"更可能"了吗？

**确证理论**：

| 理论 | 核心标准 | 问题 |
|------|----------|------|
| **演绎确证** | E从H演绎出来 | 太宽松（任何E确证任何H？） |
| **尼科德标准** | E是H的正面实例 | 乌鸦悖论 |
| **贝叶斯确证** | P(H\|E) > P(H) | 先验问题 |
| **似然性确证** | P(E\|H) > P(E\|¬H) | 依赖比较 |

**乌鸦悖论** (Hempel's Raven Paradox)：

**背景**：
- 假说H："所有乌鸦都是黑色的"
- 等价形式：¬B → ¬R（非黑的都是非乌鸦）

**悖论**：
- 观察到一只白鞋
- 白鞋是非黑的，也是非乌鸦
- 根据尼科德标准，白鞋确证"所有乌鸦都是黑色的"
- 这似乎荒谬！

**解决方案**：

| 方案 | 策略 |
|------|------|
| **接受悖论** | 白鞋确实（轻微）确证假说 |
| **贝叶斯解** | 白鞋提供的信息量很小 |
| **选择性确证** | 只有相关证据才确证 |
| **拒斥尼科德标准** | 尼科德标准是错的 |

**古德曼的"绿蓝"问题**：

**定义**：
- "绿蓝"(grue) = 在时间t之前是绿色的，之后是蓝色的

**问题**：
- 所有观察到的祖母绿都确证"祖母绿是绿色的"
- 也确证"祖母绿是绿蓝的"
- 为什么我们只接受前者？

**投射性谓词**：
- 古德曼区分"投射性"和"非投射性"谓词
- 只有投射性谓词可以用于归纳

**相关词条**：证据、归纳、乌鸦悖论

### 【观察渗透理论】(Theory-Ladenness of Observation)

**定义**：观察不是中立的，而是受到理论影响的主张。

**费曼解释**：
传统观点：观察是客观的、中立的"给定"。

理论负荷说：
- 你看到什么取决于你的理论
- 科学家和普通人看同一张X光片，看到不同的东西
- 观察是"理论渗透的"

**汉森的论证**：

**例子**：
- 开普勒和第谷看日出
- 开普勒看到"地球转动"
- 第谷看到"太阳升起"
- 他们的视觉经验不同？

**核心主张**：
- 观察的"看到"包含解释成分
- 不同理论背景的人"看到"不同的东西

**库恩的论证**：

**范式转换**：
- 不同范式的科学家"生活在不同的世界"
- 格式塔转换：看鸭兔图
- 革命后，科学家"看到"不同的世界

**科学含义**：

| 含义 | 描述 |
|------|------|
| **没有中性观察** | 不能用观察独立地检验理论 |
| **比较困难** | 不同理论如何比较？ |
| **相对主义威胁** | 是否导致相对主义？ |

**回应**：

| 回应 | 策略 |
|------|------|
| **弱解释** | 只是心理现象，不威胁客观性 |
| **区分观察/知觉** | 观察是知觉+解释 |
| **因果理论** | 观察的因果链保证客观性 |
| **接受** | 科学确实如此，但不导致相对主义 |

**代表人物**：
- **汉森**：观察渗透理论提出者
- **库恩**：范式与观察
- **费耶阿本德**：理论多元论

**经典文献**：
- Hanson, *Patterns of Discovery* (1958)
- Kuhn, *The Structure of Scientific Revolutions* (1962)

**相关词条**：范式、不可通约性、中性观察

### 【实验】(Experiment)

**定义**：在控制条件下检验假说的系统性观察。

**费曼解释**：
实验是科学的核心：
- 不是随便观察
- 而是精心设计、控制变量
- 目的是检验假说

**实验的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **探索性实验** | 发现新现象 | 法拉第电磁实验 |
| **检验性实验** | 检验假说 | 迈克尔逊-莫雷实验 |
| **测量实验** | 精确测量 | 测量光速 |
| **思想实验** | 想象中的实验 | 薛定谔的猫 |

**实验设计**：

| 要素 | 描述 |
|------|------|
| **控制变量** | 保持其他变量不变 |
| **操作化** | 将概念转化为可测量 |
| **重复性** | 实验可重复 |
| **盲法** | 避免偏见 |

**实验的作用**：

| 作用 | 描述 |
|------|------|
| **检验理论** | 验证或证伪假说 |
| **发现现象** | 发现新现象 |
| **测量常数** | 确定物理常数 |
| **技术应用** | 开发新技术 |

**当代发展 (2020-2026)**：
- **计算实验**：计算机模拟
- **大数据实验**：数据驱动的发现
- **公民科学**：公众参与实验

**相关词条**：假说检验、控制实验、思想实验

### 【可证伪性】(Falsifiability)

**定义**：波普尔提出的科学与非科学的划界标准——科学理论必须是可证伪的。

**费曼解释**：
可证伪性说：
- 一个理论如果不能被任何可能的观察证明是错的
- 那它就不是科学

"所有天鹅都是白色的"是可证伪的（找到一只黑天鹅就证伪）。
"或者有天鹅或者没有天鹅"是不可证伪的（没有什么能证伪它）。

**可证伪性的程度**：

| 理论 | 可证伪性 | 评价 |
|------|----------|------|
| "明天可能下雨可能不下雨" | 低 | 没有预测力 |
| "明天会下雨" | 中 | 有一定预测力 |
| "明天下午2点在北京会下10mm雨" | 高 | 高预测力，高风险 |

**科学与伪科学**：

| 理论 | 波普尔的判断 | 原因 |
|------|-------------|------|
| **弗洛伊德精神分析** | 伪科学 | 任何行为都能解释，不可证伪 |
| **阿德勒个体心理学** | 伪科学 | 同上 |
| **爱因斯坦相对论** | 科学 | 有明确预测，可证伪 |
| **马克思主义（教条化）** | 伪科学 | 任何结果都能解释 |

**对可证伪性的批评**：

| 批评 | 描述 |
|------|------|
| **太严格** | 一些科学理论似乎不可证伪 |
| **太宽松** | 一些伪科学也可证伪 |
| **迪昂-奎因** | 不能单独证伪理论 |
| **历史不准确** | 科学家不这样工作 |

**相关词条**：证伪主义、科学划界、波普尔

### 【可检验性】(Testability)

**定义**：理论能够被经验检验的性质。

**费曼解释**：
可检验性说：
- 科学理论必须能被检验
- 必须能做出可观察的预测
- 必须能被实验或观察支持或反驳

**可检验性的要素**：

| 要素 | 描述 |
|------|------|
| **经验内容** | 理论涉及可观察的现象 |
| **可预测** | 理论能做出预测 |
| **可验证/证伪** | 预测可被检验 |

**可检验性的问题**：

| 问题 | 描述 |
|------|------|
| **理论负荷** | 检验依赖其他理论 |
| **间接检验** | 有些理论只能间接检验 |
| **概率陈述** | 概率陈述如何检验？ |
| **理想化** | 理论通常是理想化的 |

**相关词条**：可证伪性、假说检验、经验主义

---

### 【对照实验】(Controlled Experiment)

**定义**：通过控制变量来检验因果关系的实验方法。

**费曼解释**：
对照实验是科学的金标准：
- 设置实验组和对照组
- 只改变一个变量
- 观察结果差异

**核心要素**：

| 要素 | 描述 |
|------|------|
| **实验组** | 接受处理的组 |
| **对照组** | 不接受处理的组 |
| **随机化** | 随机分配到各组 |
| **盲法** | 参与者不知道组别 |

**医学实验**：

| 类型 | 描述 |
|------|------|
| **随机对照试验(RCT)** | 随机分配+对照 |
| **单盲** | 受试者不知道 |
| **双盲** | 受试者和研究者都不知道 |
| **安慰剂对照** | 对照组接受安慰剂 |

**因果推断**：

对照实验是建立因果关系的最好方法：
- 如果其他变量都相同
- 只有处理不同
- 结果不同
- 那么处理导致了结果

**相关词条**：实验、因果推断、随机对照试验

---

### 【双盲】(Double-Blind)

**定义**：受试者和研究者都不知道谁接受处理的实验设计。

**费曼解释**：
双盲实验：
- 病人不知道自己吃的是真药还是假药
- 医生也不知道
- 这样可以排除安慰剂效应和研究者偏见

**为什么需要双盲**：

| 偏见类型 | 描述 | 双盲如何解决 |
|----------|------|-------------|
| **安慰剂效应** | 病人相信有效就会感觉有效 | 对照组也给予安慰剂 |
| **研究者偏见** | 研究者可能无意识地影响结果 | 研究者也不知道 |
| **期望效应** | 期望影响观察 | 双方都不知道 |

**双盲的局限**：

| 局限 | 描述 |
|------|------|
| **不总是可行** | 有些处理无法隐藏（如手术） |
| **伦理问题** | 某些情况下不道德 |
| **解盲** | 副作用可能暴露组别 |

**相关词条**：对照实验、安慰剂效应、随机对照试验

---

### 【安慰剂效应】(Placebo Effect)

**定义**：接受无活性治疗后产生的生理或心理改善。

**费曼解释**：
安慰剂效应很神奇：
- 病人吃糖丸（没有药物成分）
- 但病情真的改善了
- 因为病人"相信"它有效

**机制**：

| 机制 | 描述 |
|------|------|
| **期望效应** | 期望产生改善 |
| **条件反射** | 过去的治疗经验形成条件反射 |
| **神经化学** | 释放内啡肽等物质 |
| **心理因素** | 减少焦虑，改善感觉 |

**对科学的意义**：

| 意义 | 描述 |
|------|------|
| **需要对照** | 必须排除安慰剂效应 |
| **自愈** | 有些病会自然好转 |
| **心理-身体联系** | 心理影响身体 |

**相关词条**：双盲、对照实验、医学哲学

---

### 【随机对照试验】(Randomized Controlled Trial, RCT)

**定义**：通过随机分配和对照来检验治疗效果的金标准方法。

**费曼解释**：
RCT是医学研究的金标准：
1. 随机分配病人到实验组或对照组
2. 实验组接受新治疗
3. 对照组接受标准治疗或安慰剂
4. 比较两组结果

**RCT的要素**：

| 要素 | 描述 |
|------|------|
| **随机化** | 随机分配，避免选择偏倚 |
| **对照** | 设立对照组 |
| **盲法** | 尽可能双盲 |
| **意向性分析** | 按分配组分析 |

**RCT的问题**：

| 问题 | 描述 |
|------|------|
| **伦理问题** | 可能剥夺对照组的有效治疗 |
| **外部效度** | 结果可能不适用于真实世界 |
| **成本** | 昂贵、耗时 |
| **选择偏倚** | 参与者可能不代表一般人群 |

**相关词条**：对照实验、因果推断、医学哲学

---

### 【可重复性】(Reproducibility)

**定义**：实验结果能够被独立研究者重复验证的性质。

**费曼解释**：
可重复性是科学的核心：
- 如果你的实验只能做一次
- 那它不是科学
- 科学必须是可重复的

**可重复性的类型**：

| 类型 | 描述 |
|------|------|
| **精确重复** | 完全相同的条件重复 |
| **概念重复** | 用不同方法检验同一假说 |
| **直接重复** | 同一研究者重复 |
| **独立重复** | 不同研究者重复 |

**可重复性危机**：

| 领域 | 问题 |
|------|------|
| **心理学** | 许多经典实验无法重复 |
| **医学** | 许多临床试验无法重复 |
| **经济学** | 许多研究结果不可重复 |
| **生物学** | 许多实验难以重复 |

**原因**：

| 原因 | 描述 |
|------|------|
| **发表偏倚** | 只发表正面结果 |
| **p值操纵** | 选择性报告 |
| **样本量小** | 统计检验力不足 |
| **方法不透明** | 方法描述不完整 |

**解决方案**：

| 方案 | 描述 |
|------|------|
| **预注册** | 实验前注册假设和方法 |
| **开放数据** | 公开原始数据 |
| **大样本** | 增加样本量 |
| **重复研究** | 资助重复研究 |

**相关词条**：实验、科学方法、发表偏倚

### 【思想实验】(Thought Experiment)

**定义**：在想象中进行的、用于检验概念或论证的实验。

**费曼解释**：
思想实验是在脑海中做实验：
- 不需要真实设备

**经典思想实验**：

**思想实验的结构**：

| 要素 | 描述 |

### 【科学解释】(Scientific Explanation)

**定义**：用科学理论来解释为什么现象发生。

**费曼解释**：
科学解释回答"为什么"：
- 为什么天是蓝色的？
- 为什么物体会下落？
- 为什么水沸腾？

科学解释用定律和机制来回答这些问题。

**解释的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **因果解释** | 解释原因 | 因为引力，物体下落 |
| **定律解释** | 用定律解释 | 根据热力学定律... |
| **机制解释** | 解释机制 | 神经元放电导致... |
| **统一解释** | 统一多种现象 | 牛顿定律统一天体和地面 |

**D-N模型**：

**核心主张**：解释是论证——从定律和初始条件演绎出被解释项。

**结构**：
```
定律：L1, L2, ...
条件：C1, C2, ...
───────────────
现象：E
```

**问题**：对称性问题、不相关问题。

**因果解释**：

**核心主张**：解释是提供原因。

**代表**：Salmon

**统一性解释**：

**核心主张**：解释是统一多种现象。

**代表**：Kitcher

**相关词条**：D-N模型、因果解释、统一性

### 【迪昂-奎因论题】(Duhem-Quine Thesis)

**定义**：假说不能单独被检验，因为预测是从假说群推导的主张。

**费曼解释**：
传统观点：检验单个假说。

迪昂-奎因说：
- 预测来自"假说 + 辅助假说"
- 如果预测失败
- 可能是假说错，也可能是辅助假说错

**结构**：

```
假说H
辅助假说A1, A2, ...
初始条件C
───────────────
预测P
```

如果P假，只能推出：(H ∧ A1 ∧ A2 ∧ C) 假

**例子**：

| 情况 | 假说 | 辅助假说 | 预测失败的可能原因 |
|------|------|----------|-------------------|
| **行星位置** | 牛顿引力 | 望远镜准确 | 引力理论错或望远镜错 |
| **粒子实验** | 标准模型 | 探测器正常 | 理论错或探测器错 |

**后果**：

| 后果 | 描述 |
|------|------|
| **整体论** | 检验是整体的 |
| **理论的韧性** | 理论可以通过调整辅助假说来保护 |
| **经验的不充分决定** | 数据不能唯一决定理论 |

**对波普尔的挑战**：

- 波普尔说科学通过证伪进步
- 但迪昂-奎因说不能单独证伪假说
- 科学家可以选择放弃辅助假说

**代表人物**：
- **迪昂**：论题提出者
- **奎因**：论题发展

**经典文献**：
- Duhem, *The Aim and Structure of Physical Theory* (1906)
- Quine, "Two Dogmas of Empiricism" (1951)

**相关词条**：整体论、不充分决定、证伪主义

---

### 【经验不充分决定】(Underdetermination of Theory by Data)

**定义**：可用的经验证据不足以唯一决定理论的主张。

**费曼解释**：
不充分决定说：
- 有限的证据可以支持多个不同理论
- 数据不能"充分决定"理论
- 理论选择需要其他标准

**类型**：

| 类型 | 描述 |
|------|------|
| **局部不充分决定** | 某时刻的证据不充分 |
| **全局不充分决定** | 所有可能的证据都不充分 |
| **暂时不充分决定** | 暂时不充分，将来可能充分 |

**例子**：

| 例子 | 描述 |
|------|------|
| **托勒密 vs 哥白尼** | 两者都能解释行星运动 |
| **以太 vs 相对论** | 迈克尔逊-莫雷实验前两者都兼容 |
| **多个量子诠释** | 量子力学有多种诠释 |

**后果**：

| 后果 | 描述 |
|------|------|
| **理论选择** | 需要额外标准（简单性等） |
| **科学实在论** | 挑战实在论（我们怎么知道哪个理论是真的？） |
| **相对主义** | 可能导致相对主义 |

**回应**：

| 回应 | 策略 |
|------|------|
| **简单性** | 选择最简单的理论 |
| **经验等价** | 经验等价的理论实际上是同一理论 |
| **收敛** | 长期看理论会收敛 |
| **接受** | 不充分决定是事实，但不导致相对主义 |

**相关词条**：迪昂-奎因论题、科学实在论、简单性

### 【简单性】(Simplicity)

**定义**：理论选择中偏好简单理论的原则（奥卡姆剃刀）。

**费曼解释**：
简单性说：
- 如果两个理论都能解释数据
- 选择更简单的那个
- "如无必要，勿增实体"（奥卡姆剃刀）

**简单性的类型**：

| 类型 | 描述 |
|------|------|
| **本体论简单性** | 假设更少的实体 |
| **句法简单性** | 更少的公理或定律 |
| **概念简单性** | 更少的基本概念 |
| **数学简单性** | 更简单的数学形式 |

**为什么偏好简单性？**

| 理由 | 描述 |
|------|------|
| **美学** | 简单的理论更美 |
| **实用** | 简单的理论更容易使用 |
| **概率** | 简单的理论先验概率更高？ |
| **真理** | 世界可能是简单的？ |

**问题**：

| 问题 | 描述 |
|------|------|
| **简单性定义** | 什么是"简单"？ |
| **世界是简单的吗？** | 世界可能很复杂 |
| **主观性** | 简单性可能是主观的 |

**相关词条**：奥卡姆剃刀、理论选择、不充分决定

### 【奥卡姆剃刀】(Occam's Razor)

**定义**：在解释现象时，不应假设比必要更多的实体的原则。

**费曼解释**：
奥卡姆剃刀说：
- "如无必要，勿增实体"
- 能用一个原因解释，就不要用两个
- 能用简单理论解释，就不要用复杂理论

**历史**：

| 时期 | 人物 | 贡献 |
|------|------|------|
| **中世纪** | 奥卡姆的威廉 | 提出原则 |
| **近代** | 牛顿 | 应用于科学 |
| **当代** | 科学家 | 广泛使用 |

**科学应用**：

| 领域 | 应用 |
|------|------|
| **天文学** | 日心说比地心说更简单 |
| **生物学** | 进化论比神创论更简单 |
| **物理学** | 爱因斯坦比洛伦兹更简单 |

**问题**：

| 问题 | 描述 |
|------|------|
| **简单性是真理指示吗？** | 简单的理论更可能是真的吗？ |
| **定义问题** | 什么是"简单"？ |
| **反例** | 有些简单理论是错的 |

**相关词条**：简单性、理论选择、最佳解释推理

### 【最佳解释推理】(Inference to the Best Explanation)

**定义**：从多个可能解释中选择最佳解释的推理方法。

**费曼解释**：
最佳解释推理说：
- 观察到现象
- 有多个可能解释
- 选择能最好解释现象的那个

**结构**：

```
观察O
假说H1, H2, H3都能解释O
H1是最好的解释
───────────────
因此，H1可能是真的
```

**评价标准**：

| 标准 | 描述 |
|------|------|
| **解释力** | 解释多少现象 |
| **简单性** | 理论是否简单 |
| **一致性** | 与已知知识一致 |
| **新颖预测** | 能做出新颖预测 |

**与归纳的关系**：

| 问题 | IBE | 归纳 |
|------|-----|------|
| **方向** | 从证据到最佳假说 | 从实例到一般规律 |
| **形式** | 选择最佳解释 | 概率更新 |
| **风险** | 可能选择错误解释 | 可能归纳错误 |

**对IBE的批评**：

| 批评 | 描述 |
|------|------|
| **循环** | 什么是"最好"需要标准 |
| **不充分决定** | 可能有多个同样好的解释 |
| **简单性问题** | 为什么简单的是更好的？ |

**代表人物**：
- **Harman**：IBE提出者
- **Lipton**：IBE发展

**经典文献**：
- Harman, "The Inference to the Best Explanation" (1965)
- Lipton, *Inference to the Best Explanation* (2004)

**相关词条**：简单性、理论选择、归纳

### 【科学发现的逻辑】(Logic of Scientific Discovery)

**定义**：波普尔提出的关于科学方法和知识增长的哲学理论。

**费曼解释**：
波普尔说科学发现的逻辑是：
1. 大胆提出假说（猜想）
2. 严格检验假说（反驳）
3. 被证伪的放弃，没被证伪的暂时保留
4. 科学通过不断试错进步

**核心主张**：

| 要素 | 描述 |
|------|------|
| **可证伪性** | 科学的划界标准 |
| **猜想与反驳** | 科学方法 |
| **批判理性主义** | 科学态度 |
| **淘汰错误** | 科学进步方式 |

**与归纳主义的关系**：

| 问题 | 归纳主义 | 波普尔 |
|------|----------|--------|
| **科学方法** | 从观察归纳 | 猜想与反驳 |
| **验证** | 确证增加概率 | 只有证伪有意义 |
| **进步** | 确认积累 | 不断证伪进步 |

**代表人物**：
- **波普尔**：理论创始人

**经典文献**：
- Popper, *The Logic of Scientific Discovery* (1959)

**相关词条**：证伪主义、可证伪性、科学方法

### 【科学革命的结构】(Structure of Scientific Revolutions)

**定义**：库恩描述的科学发展的历史模型。

**费曼解释**：
库恩说科学发展不是渐进的：
- 常规科学：在范式内解决谜题
- 危机：出现异常，范式动摇
- 革命：范式转换
- 新常规科学：在新范式内工作

**库恩的模型**：

```
前科学 → 常规科学 → 危机 → 革命 → 新常规科学 → ...
```

| 阶段 | 描述 |
|------|------|
| **前科学** | 没有统一范式 |
| **常规科学** | 在范式内解谜 |
| **危机** | 异常积累 |
| **革命** | 范式转换 |
| **新常规科学** | 新范式内工作 |

**核心概念**：

| 概念 | 描述 |

### 【判决性实验】(Crucial Experiment)

### 【辅助假说】(Auxiliary Hypothesis)

**定义**：与主要假说一起用于推导预测的额外假说。

**费曼解释**：
辅助假说是"帮手"：
- 主要假说：引力定律
- 辅助假说：望远镜准确
- 一起才能做出预测

**在迪昂-奎因论题中的作用**：

```
主要假说H
辅助假说A1, A2, ...
初始条件C
───────────────
预测P
```

如果P假，不能确定是H错还是某个A错。

**例子**：

| 情况 | 主要假说 | 辅助假说 |
|------|----------|----------|
| **天文观测** | 引力理论 | 望远镜校准 |
| **医学实验** | 药物有效 | 测量准确 |
| **物理实验** | 理论预测 | 仪器正常 |

**意义**：

- 解释了为什么科学家可以坚持理论
- 理论可以通过调整辅助假说来保护
- 证伪不是简单的

**相关词条**：迪昂-奎因论题、证伪主义、假说检验

### 【特设性假说】(Ad Hoc Hypothesis)

**定义**：专门为挽救理论免受反例反驳而引入的假说。

**费曼解释**：
特设性假说是"事后补救"：
- 理论预测失败
- 科学家说"啊，那是因为..."
- 这个"因为"就是特设性假说

**特征**：

| 特征 | 描述 |
|------|------|
| **事后** | 在反例后引入 |
| **保护性** | 保护理论免受证伪 |
| **无独立检验** | 不能独立检验 |
| **不可预测** | 不能做出新预测 |

**例子**：

| 原假说 | 反例 | 特设性假说 |
|--------|------|-----------|
| 所有天鹅都是白色 | 发现黑天鹅 | "那不是真正的天鹅" |
| 以太存在 | 迈克尔逊-莫雷实验 | "以太被拖着走" |

**波普尔的批评**：

波普尔认为特设性假说是"不科学的"：
- 科学应该大胆预测，而不是事后解释
- 好的假说应该能独立检验

**相关词条**：辅助假说、证伪主义、科学方法

### 【对应规则】(Correspondence Rule)

**定义**：连接理论术语与观察术语的规则。

对应规则是"桥梁"：
- 理论说"有电子"
- 对应规则说"如果电子存在，你会看到..."
- 这样理论术语与观察联系起来

**功能**：

| 功能 | 描述 |
|------|------|
| **连接** | 连接理论与观察 |
| **意义** | 给理论术语意义 |
| **检验** | 使理论可检验 |

**例子**：

| 理论术语 | 对应规则 | 观察术语 |
|----------|----------|----------|
| "温度" | 温度计读数 | "37度" |
| "压力" | 压力表读数 | "100帕" |

**问题**：

| 问题 | 描述 |
|------|------|
| **模糊性** | 对应规则可能模糊 |
| **理论负荷** | 观察术语不中立 |
| **复杂性** | 对应可能很复杂 |

**相关词条**：理论术语、观察术语、逻辑经验主义

---

### 【意义证实原则】(Verificationism)

**定义**：一个陈述的意义在于其证实方法的哲学原则。

**费曼解释**：
证实原则说：
- "明天会下雨"有意义吗？
- 有！因为你可以等待明天来证实
- "形而上学是无意义的"无法证实，所以无意义

**核心主张**：

| 要素 | 描述 |
|------|------|
| **意义=证实方法** | 陈述的意义就是其证实方法 |
| **可证实性** | 有意义的陈述必须可证实 |
| **拒斥形而上学** | 不可证实的陈述无意义 |

**类型**：

| 类型 | 描述 |
|------|------|
| **强证实** | 必须能被完全证实 |
| **弱证实** | 能被经验支持或反驳 |
| **原则证实** | 能被原则上证实的观察 |

**问题**：

| 问题 | 描述 |
|------|------|
| **自身问题** | 证实原则本身可证实吗？ |
| **普遍陈述** | "所有乌鸦都是黑色"不能完全证实 |
| **反事实** | 反事实条件句如何证实？ |

**相关词条**：逻辑经验主义、可检验性、科学语言

---

### 【科学统一】(Unity of Science)

**定义**：所有科学最终可以统一为一个连贯体系的理念。

**费曼解释**：
科学统一说：
- 生物学可以还原为化学
- 化学可以还原为物理学
- 最终所有科学统一为物理学

**统一的模型**：

```
社会学
  ↓
心理学
  ↓
生物学
  ↓
化学
  ↓
物理学
```

**统一的类型**：

| 类型 | 描述 |
|------|------|
| **还原统一** | 高层科学还原为低层科学 |
| **理论统一** | 理论间的统一 |
| **方法统一** | 方法的统一 |
| **本体论统一** | 存在什么的统一 |

**争议**：

| 争议 | 描述 |
|------|------|
| **还原成功** | 还原真的能成功吗？ |
| **涌现** | 是否有涌现属性？ |
| **自主性** | 高层科学是否自主？ |

**相关词条**：还原论、涌现、科学层次

---

### 【还原论】(Reductionism)

**定义**：复杂系统可以还原为简单组成部分的观点。

**费曼解释**：
还原论说：
- 要理解生命，就研究细胞
- 要理解细胞，就研究分子
- 要理解分子，就研究原子

**还原的类型**：

| 类型 | 描述 |
|------|------|
| **本体论还原** | 存在什么可以还原 |
| **理论还原** | 理论可以还原 |
| **解释还原** | 解释可以还原 |
| **方法还原** | 方法可以还原 |

**理论还原**：

Nagel的条件：
1. **可连接性**：高层科学的术语可以用低层科学的术语定义
2. **可推导性**：高层科学的定律可以从低层科学的定律推导

**对还原论的批评**：

| 批评 | 描述 |
|------|------|
| **涌现** | 有些属性是涌现的，不可还原 |
| **多重实现** | 高层属性可被多种低层实现 |
| **自主性** | 高层科学有自己的规律 |
| **复杂性** | 还原可能太复杂而不可行 |

**相关词条**：涌现、科学统一、理论还原

---

### 【涌现】(Emergence)

**词源**：拉丁语 *emergere* (出现、浮现)

**定义**：复杂系统具有其组成部分所不具备的属性的现象。

**费曼解释**：
涌现说：
- 水是湿润的，但氢和氧不是
- 意识是有意识的，但神经元不是
- 整体大于部分之和

**涌现的类型**：

| 类型 | 描述 |
|------|------|
| **弱涌现** | 可从部分预测（但复杂） |
| **强涌现** | 不可从部分预测，有下向因果 |
| **认识论涌现** | 我们不知道如何预测 |
| **本体论涌现** | 本体上不可还原 |

**涌现 vs 还原**：

| 问题 | 还原论 | 涌现论 |
|------|--------|--------|
| 整体与部分 | 整体可还原为部分 | 整体有新属性 |
| 预测 | 可从部分预测 | 有不可预测的属性 |
| 解释 | 还原解释 | 多层次解释 |
| 因果 | 只有微观因果 | 有宏观因果 |

**例子**：

| 系统 | 部分 | 涌现属性 |
|------|------|----------|
| **水** | H₂O分子 | 湿润、流动性 |
| **生命** | 分子 | 代谢、繁殖 |
| **意识** | 神经元 | 主观体验 |
| **社会** | 个人 | 文化、制度 |

**相关词条**：还原论、科学层次、下向因果
