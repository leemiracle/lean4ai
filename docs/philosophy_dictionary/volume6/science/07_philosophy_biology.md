### 【生物学哲学】(Philosophy of Biology)

**定义**：研究生物学基础概念和理论的哲学分支。

**费曼解释**：
生物学哲学问：
- 什么是生命？
- 进化论告诉我们什么？
- 基因决定论正确吗？

**核心问题**：

| 问题 | 描述 |
|------|------|
| **生命本质** | 什么是生命？ |
| **进化机制** | 进化如何工作？ |
| **基因vs环境** | 天性vs教养？ |
| **还原论** | 生物学能还原为物理/化学？ |

**主要领域**：

| 领域 | 描述 |
|------|------|
| **进化哲学** | 进化论的哲学问题 |
| **遗传学哲学** | 遗传学的哲学问题 |
| **发育哲学** | 发育的哲学问题 |
| **生态学哲学** | 生态学的哲学问题 |
| **神经科学哲学** | 神经科学的哲学问题 |

**相关词条**：进化论、基因、生命

### 【进化论】(Evolution)

**定义**：达尔文提出的解释生物多样性的理论。

**费曼解释**：
进化论说：
- 所有生物有共同祖先
- 自然选择塑造适应
- 随机变异提供原料

**核心机制**：

| 机制 | 描述 |
|------|------|
| **自然选择** | 适者生存 |
| **变异** | 随机变异 |
| **遗传** | 性状遗传 |
| **漂变** | 随机漂变 |

**达尔文的贡献**：

| 贡献 | 描述 |
|------|------|
| **共同祖先** | 所有生物有共同祖先 |
| **自然选择** | 自然选择是主要机制 |
| **渐进变化** | 进化是渐进的 |

**现代综合**：

| 组成部分 | 描述 |
|----------|------|
| **达尔文选择** | 自然选择 |
| **孟德尔遗传** | 遗传学 |
| **群体遗传** | 群体层次的进化 |

**哲学问题**：

| 问题 | 描述 |
|------|------|
| **选择层次** | 选择在哪个层次工作？ |
| **适应主义** | 所有特征都是适应吗？ |
| **进步** | 进化是进步的吗？ |

**相关词条**：自然选择、适应、达尔文

### 【自然选择】(Natural Selection)

**定义**：适者生存的进化机制。

**费曼解释**：
自然选择是"进化的引擎"：
- 有变异
- 变异影响生存和繁殖
- 有利变异被保留

**三个要素**：

| 要素 | 描述 |
|------|------|
| **变异** | 个体间有变异 |
| **遗传** | 变异可以遗传 |
| **适合度差异** | 变异影响适合度 |

**适合度**：

**定义**：个体的繁殖成功。

**组成**：
- 生存能力
- 繁殖能力
- 后代存活

**类型**：

| 类型 | 描述 |
|------|------|
| **定向选择** | 选择某一方向 |
| **稳定选择** | 选择中间值 |
| **分裂选择** | 选择两个极端 |

**哲学问题**：

| 问题 | 描述 |
|------|------|
| **同义反复** | "适者生存"是同义反复？ |
| **选择单位** | 选择在什么层次？ |
| **目的论** | 自然选择是目的论的？ |

**相关词条**：进化论、适应、适合度

### 【适应】(Adaptation)

**定义**：由自然选择形成的特征。

**费曼解释**：
适应是"进化的产物"：
- 眼睛是为了看
- 翅膀是为了飞
- 适应是自然选择的结果

**定义问题**：

| 问题 | 描述 |
|------|------|
| **历史定义** | 由自然选择形成的特征 |
| **功能定义** | 现在有功能的特征 |

**适应主义争论**：

| 立场 | 观点 |
|------|------|
| **强适应主义** | 所有特征都是适应 |
| **弱适应主义** | 大多数特征是适应 |
| **反适应主义** | 很多特征不是适应 |

**非适应解释**：

| 解释 | 描述 |
|------|------|
| **漂变** | 随机漂变 |
| **约束** | 发育约束 |
| **副作用** | 其他特征的副作用 |
| **历史遗产** | 历史遗留 |

**相关词条**：自然选择、适应主义、功能

### 【适应主义】(Adaptationism)

**定义**：认为大多数生物特征是自然选择产生的适应的观点。

**费曼解释**：
适应主义说：
- 几乎每个特征都有适应功能
- 为什么有这个特征？因为它有生存价值
- 这是最常见的解释

**类型**：

| 类型 | 描述 |
|------|------|
| **经验适应主义** | 大多数特征是适应 |
| **方法论适应主义** | 先假设适应，再检验 |
| **解释适应主义** | 适应是最好的解释 |

**支持**：

| 支持 | 描述 |
|------|------|
| **成功预测** | 适应主义预测成功 |
| **发现功能** | 发现了很多功能 |
| **启发价值** | 有启发价值 |

**批评**：

| 批评 | 描述 |
|------|------|
| **潘恩和莱文廷** | "只是如此故事" |
| **忽视约束** | 忽视发育约束 |
| **忽视历史** | 忽视历史因素 |
| **不可证伪** | 可能不可证伪 |

** Gould和Lewontin的批评**：

"圣马可大教堂的拱肩"类比：
- 拱肩不是"为了"装饰
- 而是结构的副产品
- 同样，很多特征是副作用

**相关词条**：适应、自然选择、拱肩

### 【基因】(Gene)

**定义**：遗传的基本单位。

**费曼解释**：
基因是"遗传的字母"：
- DNA上的序列
- 编码蛋白质
- 从父母传给后代

**定义的历史**：

| 时期 | 定义 |
|------|------|
| **孟德尔** | 遗传因子 |
| **摩尔根** | 染色体上的单位 |
| **分子生物学** | DNA序列 |
| **现代** | 功能性定义 |

**哲学问题**：

| 问题 | 描述 |
|------|------|
| **基因概念** | 基因如何定义？ |
| **基因同一性** | 什么是同一个基因？ |
| **基因范畴** | 基因是发现的还是发明的？ |

**基因概念**：

| 概念 | 描述 |
|------|------|
| **分子基因** | DNA序列 |
| **进化基因** | 选择单位 |
| **发育基因** | 发育中的功能单位 |

**相关词条**：遗传、DNA、基因组

### 【基因型-表型区别】(Genotype-Phenotype Distinction)

**定义**：遗传构成（基因型）与可观察特征（表型）的区别。

**费曼解释**：
基因型-表型区别说：
- 基因型：基因组成
- 表型：可观察的特征
- 同一基因型可以有不同表型（环境影响）

**关系**：

| 关系 | 描述 |
|------|------|
| **基因型→表型** | 基因型影响表型 |
| **环境影响** | 环境也影响表型 |
| **交互** | 基因和环境交互 |

**哲学意义**：

| 意义 | 描述 |
|------|------|
| **反对遗传决定论** | 表型不只是基因决定 |
| **复杂性** | 关系很复杂 |
| **可塑性** | 表型有可塑性 |

**相关词条**：基因、表型、环境

### 【基因决定论】(Genetic Determinism)

**定义**：认为基因决定所有生物特征的观点。

**费曼解释**：
基因决定论说：
- 你是什么由基因决定
- 行为、智力、性格都是遗传的
- 环境不重要

**类型**：

| 类型 | 描述 |
|------|------|
| **强决定论** | 完全由基因决定 |
| **弱决定论** | 基因有强影响 |

**批评**：

| 批评 | 描述 |
|------|------|
| **环境影响** | 环境有重要影响 |
| **复杂性** | 发展很复杂 |
| **交互** | 基因和环境交互 |
| **简化论** | 过度简化 |

**例子**：

| 特征 | 基因影响 | 环境影响 |
|------|----------|----------|
| **身高** | 强 | 营养 |
| **智力** | 中等 | 教育、环境 |
| **性格** | 中等 | 环境、文化 |

**相关词条**：天性vs教养、基因、环境

### 【天性vs教养】(Nature vs Nurture)

**定义**：关于遗传（天性）和环境（教养）哪个更重要的争论。

**费曼解释**：
天性vs教养问：
- 你的性格是天生的还是后天的？
- 基因重要还是环境重要？
- 这是一个经典的争论

**历史**：

| 时期 | 主导观点 |
|------|----------|
| **早期** | 天性主导 |
| **行为主义** | 教养主导 |
| **现代** | 交互主义 |

**现代观点**：

| 观点 | 描述 |
|------|------|
| **交互主义** | 天性和教养交互 |
| **量化遗传学** | 可以量化贡献 |
| **表观遗传学** | 环境可以改变基因表达 |

**量化**：

| 方法 | 描述 |
|------|------|
| **双胞胎研究** | 比较同卵和异卵双胞胎 |
| **领养研究** | 比较亲生和领养 |
| **遗传率** | 量化遗传贡献 |

**哲学意义**：

| 意义 | 描述 |
|------|------|
| **反决定论** | 不是单一因素决定 |
| **复杂性** | 发展很复杂 |
| **责任** | 对道德责任的影响 |

**相关词条**：基因决定论、环境、遗传率

### 【物种概念】(Species Concept)

**定义**：定义生物学物种的标准。

**费曼解释**：
物种概念问：
- 什么是"物种"？
- 如何划分物种？
- 不同物种之间有什么区别？

**主要概念**：

| 概念 | 定义 | 问题 |
|------|------|------|
| **生物学物种** | 可以交配产生可育后代 | 不能用于无性繁殖 |
| **形态学物种** | 形态相似 | 主观性 |
| **系统发育物种** | 单系群 | 可能划分太细 |
| **生态学物种** | 占据相同生态位 | 难以定义生态位 |

**生物学物种概念**（BSC）：

**Mayr的定义**：
- 物种是实际上或潜在可以交配的群体
- 与其他这样的群体生殖隔离

**问题**：
- 无性繁殖物种
- 化石物种
- 杂交

**哲学意义**：

| 意义 | 描述 |
|------|------|
| **实在性** | 物种是实在的吗？ |
| **个体vs类** | 物种是个体还是类？ |
| **划界问题** | 如何划界？ |

**相关词条**：分类学、进化论、Mayr

### 【分类学】(Taxonomy)

**定义**：对生物进行分类的科学。

**费曼解释**：
分类学是"给生命分类"：
- 界、门、纲、目、科、属、种
- 把相似的生物放在一起
- 反映进化关系

**分类等级**：

| 等级 | 例子（人类） |
|------|-------------|
| **界** | 动物界 |
| **门** | 脊索动物门 |
| **纲** | 哺乳纲 |
| **目** | 灵长目 |
| **科** | 人科 |
| **属** | 人属 |
| **种** | 智人 |

**方法**：

| 方法 | 描述 |
|------|------|
| **表征分类** | 基于相似性 |
| **支序分类** | 基于共同祖先 |
| **进化分类** | 结合两者 |

**哲学问题**：

| 问题 | 描述 |
|------|------|
| **自然性** | 分类是自然的吗？ |
| **本质主义** | 有物种本质吗？ |
| **约定性** | 分类是约定的吗？ |

**相关词条**：物种概念、支序分类、进化

### 【支序分类学】(Cladistics)

**定义**：基于共同祖先的分类方法。

**费曼解释**：
支序分类学是"按祖先分类"：
- 只考虑共同祖先
- 不考虑相似程度
- 分支点决定分类

**核心概念**：

| 概念 | 描述 |
|------|------|
| **支序** | 共同祖先的所有后代 |
| **单系群** | 共同祖先的所有后代 |
| **并系群** | 不完全的单系群 |
| **多系群** | 多个祖先的后代 |

**例子**：

| 群体 | 类型 |
|------|------|
| **哺乳动物** | 单系群 |
| **爬行动物** | 并系群（不含鸟类） |
| **温血动物** | 多系群 |

**哲学意义**：

| 意义 | 描述 |
|------|------|
| **进化反映** | 反映进化历史 |
| **客观性** | 更客观的分类 |
| **问题** | 可能不实用 |

**相关词条**：分类学、物种、进化

### 【选择单位】(Unit of Selection)

**定义**：自然选择作用的层次或单位。

**费曼解释**：
选择单位问：
- 自然选择在哪个层次工作？
- 个体？基因？群体？
- 这是生物学的大争论

**候选单位**：

| 单位 | 描述 | 支持 |
|------|------|------|
| **基因** | 基因是选择单位 | 道金斯 |
| **个体** | 个体是选择单位 | 传统观点 |
| **群体** | 群体是选择单位 | 群体选择论者 |
| **物种** | 物种是选择单位 | 物种选择 |

**道金斯的基因选择**：

**核心主张**：
- 基因是"自私的"
- 个体只是基因的"载体"
- 选择在基因层次工作

**自私的基因**：

**比喻**：
- 基因是"复制子"
- 个体是"载体"
- 基因利用个体复制自己

**群体选择**：

**核心主张**：
- 选择可以在群体层次工作
- 解释利他行为
- 有争议

**多层次选择**：

**现代观点**：
- 选择可以在多个层次工作
- 不同层次有时冲突
- 需要具体情况具体分析

**相关词条**：自然选择、自私的基因、道金斯

### 【自私的基因】(The Selfish Gene)

**定义**：道金斯提出的以基因为中心的进化观点。

**费曼解释**：
自私的基因说：
- 进化是基因的竞争
- 个体只是基因的载体
- 基因"想"复制自己

**核心主张**：

| 主张 | 描述 |
|------|------|
| **基因中心** | 进化是基因层次的选择 |
| **载体** | 个体是基因的载体 |
| **复制子** | 基因是复制子 |

**解释利他行为**：

| 行为 | 基因解释 |
|------|----------|
| **亲缘利他** | 帮助携带相同基因的亲属 |
| **互惠利他** | 期望未来回报 |

**批评**：

| 批评 | 描述 |
|------|------|
| **基因概念** | 基因定义不清 |
| **个体能动性** | 忽视个体能动性 |
| **隐喻** | 只是隐喻？ |

**代表人物**：
- **理查德·道金斯**：自私的基因作者

**经典文献**：
- Dawkins, *The Selfish Gene* (1976)

**相关词条**：选择单位、道金斯、自然选择

### 【利他行为】(Altruism)

**定义**：牺牲自己适合度来帮助他人的行为。

**费曼解释**：
利他行为问：
- 为什么会帮助他人？
- 自然选择应该淘汰利他
- 如何解释利他？

**进化解释**：

| 解释 | 描述 |
|------|------|
| **亲缘选择** | 帮助亲属传播基因 |
| **互惠利他** | 期望回报 |
| **群体选择** | 有利群体的特征 |

**亲缘选择**：

**汉密尔顿规则**：
rB > C

- r = 亲缘系数
- B = 受益者的收益
- C = 利他者的代价

**例子**：

| 行为 | 解释 |
|------|------|
| **蜜蜂蜇人** | 保护蜂巢（亲属） |
| **鸟报警** | 警告亲属 |

**互惠利他**：

**Trivers的理论**：
- 帮助他人，期望回报
- 需要识别和惩罚欺骗者
- 在稳定社会中可行

**例子**：

| 行为 | 解释 |
|------|------|
| **吸血蝙蝠分享血** | 期望未来回报 |
| **灵长类互助** | 互惠交换 |

**相关词条**：亲缘选择、汉密尔顿、社会行为

### 【亲缘选择】(Kin Selection)

**定义**：通过帮助亲属来传播自己基因的进化机制。

**费曼解释**：
亲缘选择说：
- 帮助亲属 = 帮助自己的基因
- 因为亲属有相同的基因
- 这解释了利他行为

**汉密尔顿规则**：

rB > C

| 变量 | 含义 |
|------|------|
| r | 亲缘系数（共享基因比例） |
| B | 受益者获得的适合度收益 |
| C | 利他者付出的适合度代价 |

**亲缘系数**：

| 关系 | r值 |
|------|-----|
| **同卵双胞胎** | 1.0 |
| **亲子** | 0.5 |
| **兄弟姐妹** | 0.5 |
| **祖孙** | 0.25 |
| **表兄妹** | 0.125 |

**例子**：

| 行为 | 计算 |
|------|------|
| **为2个兄弟牺牲** | 2 × 0.5 = 1.0，值得 |
| **为1个表妹牺牲** | 0.125 < 1，不值得 |

**代表人物**：
- **威廉·汉密尔顿**：亲缘选择理论

**相关词条**：利他行为、适合度、社会进化

### 【发育】(Development)

**定义**：从受精卵到成熟个体的过程。

**费曼解释**：
发育是"生命的故事"：
- 从一个细胞变成复杂的生物
- 基因和环境交互
- 发育是理解生命的关键

**核心问题**：

| 问题 | 描述 |
|------|------|
| **分化** | 细胞如何分化？ |
| **模式形成** | 身体模式如何形成？ |
| **基因控制** | 基因如何控制发育？ |

**发育约束**：

**定义**：发育过程对进化的限制。

| 类型 | 描述 |
|------|------|
| **物理约束** | 物理限制 |
| **遗传约束** | 遗传网络限制 |
| **历史约束** | 历史遗留 |

**哲学问题**：

| 问题 | 描述 |
|------|------|
| **基因中心** | 发育挑战基因中心观？ |
| **表观遗传** | 表观遗传的作用？ |
| **模块性** | 发育是模块的吗？ |

**相关词条**：表观遗传、发育约束、基因调控

### 【表观遗传学】(Epigenetics)

**定义**：研究基因表达的可遗传变化，不涉及DNA序列变化。

**费曼解释**：
表观遗传学是"基因之上的遗传"：
- 不改变DNA序列
- 但改变基因表达
- 有时可以遗传

**机制**：

| 机制 | 描述 |
|------|------|
| **DNA甲基化** | DNA上添加甲基 |
| **组蛋白修饰** | 组蛋白化学修饰 |
| **非编码RNA** | 非编码RNA调控 |

**意义**：

| 意义 | 描述 |
|------|------|
| **环境效应** | 环境可以影响基因表达 |
| **跨代遗传** | 某些效应可以遗传 |
| **挑战遗传学** | 挑战传统遗传学 |

**哲学问题**：

| 问题 | 描述 |
|------|------|
| **遗传概念** | 什么是遗传？ |
| **拉马克主义** | 支持拉马克主义？ |
| **还原论** | 挑战基因还原论？ |

**相关词条**：发育、遗传、环境效应

## E-Z

### 【生命】(Life)

**定义**：区分生物和非生物的特征。

**费曼解释**：
生命问：
- 什么是活的？
- 病毒是活的吗？
- 如何定义生命？

**生命的特征**：

| 特征 | 描述 |
|------|------|
| **代谢** | 能量转换 |
| **繁殖** | 产生后代 |
| **生长** | 生长和发育 |
| **响应** | 响应环境 |
| **稳态** | 维持内部稳定 |
| **进化** | 进化适应 |

**定义问题**：

| 问题 | 描述 |
|------|------|
| **边界案例** | 病毒是活的吗？ |
| **人工生命** | 人工智能是活的吗？ |
| **本质主义** | 有生命的本质吗？ |

**生命定义**：

| 定义 | 描述 |
|------|------|
| **热力学** | 负熵系统 |
| **信息** | 信息处理系统 |
| **进化** | 达尔文进化系统 |

**相关词条**：生物学哲学、生命起源、人工生命

---

### 【目的论】(Teleology)

**定义**：用目的或目标来解释现象的思维方式。

**费曼解释**：
目的论是"为了..."的解释：
- 心脏是为了泵血
- 眼睛是为了看
- 这种解释在生物学中常见

**类型**：

| 类型 | 描述 |
|------|------|
| **外在目的论** | 外部目的 |
| **内在目的论** | 内在目的 |
| **自然目的论** | 自然选择的目的 |

**与进化论的关系**：

| 问题 | 回答 |
|------|------|
| **有目的吗？** | 自然选择没有目的 |
| **目的论解释** | 可以有"好像"目的的解释 |
| **功能解释** | 功能解释是目的论的 |

**自然化的目的论**：

**核心**：用自然选择解释"功能"和"目的"。

| 概念 | 自然选择解释 |
|------|-------------|
| **功能** | 被选择的功能 |
| **目的** | 选择的历史原因 |

**相关词条**：功能解释、适应、自然选择

---

### 【功能解释】(Functional Explanation)

**定义**：用功能来解释生物特征。

**费曼解释**：
功能解释是"它做什么"的解释：
- 心脏的功能是泵血
- 肾脏的功能是过滤
- 功能解释回答"为什么有这个器官"

**功能的概念**：

| 概念 | 描述 |
|------|------|
| **因果角色功能** | 在系统中的因果角色 |
| **选择功能** | 被选择的功能 |
| **设计功能** | 设计的目的 |

**进化功能**：

**定义**：一个特征的功能是它被选择来做的事情。

**例子**：

| 特征 | 功能 |
|------|------|
| **心脏** | 泵血（被选择来泵血） |
| **眼睛** | 感光（被选择来感光） |

**功能 vs 效果**：

| 区别 | 功能 | 效果 |
|------|------|------|
| **历史** | 被选择 | 不一定被选择 |
| **解释** | 解释为什么存在 | 只是描述 |

**相关词条**：目的论、适应、解释

---

### 【还原论】(Reductionism)

**定义**：认为复杂系统可以还原为简单部分的观点。

**费曼解释**：
还原论说：
- 要理解生命，就研究分子
- 生物学可以还原为化学
- 化学可以还原为物理

**类型**：

| 类型 | 描述 |
|------|------|
| **本体论还原** | 生物就是物理化学 |
| **理论还原** | 生物学理论可还原为物理学理论 |
| **方法还原** | 用还原方法研究 |

**支持**：

| 支持 | 描述 |
|------|------|
| **分子生物学成功** | 分子生物学很成功 |
| **科学统一** | 支持科学统一 |
| **解释力** | 还原解释有力量 |

**批评**：

| 批评 | 描述 |
|------|------|
| **涌现** | 有涌现性质 |
| **多重实现** | 生物特征可被多种物理实现 |
| **自主性** | 生物学有自主性 |
| **复杂性** | 还原太复杂 |

**反还原论**：

| 观点 | 描述 |
|------|------|
| **涌现论** | 有涌现性质不可还原 |
| **自主论** | 生物学有自主性 |
| **多层次** | 需要多层次解释 |

**相关词条**：涌现、科学统一、复杂性

---

### 【涌现】(Emergence)

**定义**：复杂系统具有其组成部分所不具备的属性。

**费曼解释**：
涌现说：
- 生命是分子涌现的
- 意识是神经元涌现的
- 整体大于部分之和

**类型**：

| 类型 | 描述 |
|------|------|
| **弱涌现** | 可从部分推导（但复杂） |
| **强涌现** | 不可从部分推导 |
| **认识论涌现** | 我们不知道如何推导 |
| **本体论涌现** | 本体上不可还原 |

**在生物学中的例子**：

| 系统 | 部分 | 涌现属性 |
|------|------|----------|
| **生命** | 分子 | 代谢、繁殖 |
| **意识** | 神经元 | 主观体验 |
| **社会** | 个体 | 文化 |

**对还原论的挑战**：

| 挑战 | 描述 |
|------|------|
| **不可还原** | 有些性质不可还原 |
| **下向因果** | 整体可以影响部分 |
| **自主性** | 高层科学有自主性 |

**相关词条**：还原论、多层次、复杂性

| 理论 | 描述 |

| **自然选择** | 自然选择是进化机制 |

**经典文献**：

**哲学影响**：

**相关词条**：进化论、自然选择、适应

### 【古尔德】(Stephen Jay Gould)

**生平**：1941-2002，美国古生物学家

**贡献**：间断平衡、反对适应主义

**主要理论**：

| 理论 | 描述 |
|------|------|

- Gould & Eldredge, "Punctuated Equilibria" (1972)

### 【间断平衡】(Punctuated Equilibrium)

**定义**：进化以快速变化和长期稳定交替进行的理论。

**费曼解释**：
间断平衡说：
- 进化不是渐进的
- 而是快速变化后长期稳定
- 化石记录支持这种模式

**核心主张**：

| 主张 | 描述 |
|------|------|
| **稳定期** | 长期稳定 |
| **快速变化** | 短期快速变化 |
| **物种形成** | 变化发生在物种形成时 |

**vs 渐进主义**：

| 问题 | 渐进主义 | 间断平衡 |
|------|----------|----------|
| **变化速度** | 缓慢均匀 | 快慢交替 |
| **化石记录** | 不完整 | 反映真实 |

**代表人物**：
- **古尔德**：间断平衡
- **艾尔德里奇**：间断平衡

**相关词条**：进化论、古尔德、化石记录

---

### 【扩展表型】(Extended Phenotype)

**定义**：基因效应可以超出个体身体的观点。

**费曼解释**：
扩展表型说：
- 基因的效应不只是身体
- 还包括行为、环境
- 如：蜘蛛网、鸟巢、寄生虫行为

**例子**：

| 基因效应 | 描述 |
|----------|------|
| **蜘蛛网** | 蜘蛛基因的表达 |
| **海狸坝** | 海狸基因的表达 |
| **寄生虫行为** | 寄生虫基因改变宿主行为 |

**意义**：

| 意义 | 描述 |
|------|------|
| **基因效应范围** | 基因效应超出个体 |
| **选择单位** | 支持基因选择 |
| **进化理解** | 更全面的进化理解 |

**代表人物**：
- **道金斯**：扩展表型

**相关词条**：道金斯、表型、基因效应

### 【模因】(Meme)

**定义**：文化传播的单位，类似基因。

**费曼解释**：
模因是"文化的基因"：
- 思想、观念、习俗像基因一样传播
- 有变异、选择、遗传
- 文化进化类似生物进化

**特征**：

| 特征 | 描述 |
|------|------|
| **复制** | 可以被复制 |
| **变异** | 可以变化 |
| **选择** | 有些更成功 |

**例子**：

| 模因类型 | 例子 |
|----------|------|
| **观念** | 民主、自由 |
| **技能** | 语言、工具使用 |
| **艺术** | 歌曲、故事 |
| **技术** | 发明、方法 |

**批评**：

| 批评 | 描述 |
|------|------|
| **类比限制** | 与基因类比有限 |
| **选择机制** | 选择机制不清楚 |
| **不可证伪** | 可能不可证伪 |

**代表人物**：
- **道金斯**：模因概念提出者

**相关词条**：道金斯、文化进化、自私的基因

### 【进化心理学】(Evolutionary Psychology)

**定义**：用进化论解释心理特征的学科。

**费曼解释**：
进化心理学说：
- 心理特征是进化的产物
- 大脑是适应的集合
- 用进化解释行为

**核心主张**：

| 主张 | 描述 |
|------|------|
| **心理适应** | 心理是适应的产物 |
| **模块性** | 大脑是模块化的 |
| **石器时代大脑** | 现代大脑适应石器时代 |

**例子**：

| 心理特征 | 进化解释 |
|----------|----------|
| **恐惧** | 避免危险 |
| **择偶偏好** | 繁殖成功 |
| **利他** | 亲缘选择 |

**批评**：

| 批评 | 描述 |
|------|------|
| **适应主义** | 过度适应主义 |
| **不可证伪** | 故事不可证伪 |
| **忽视文化** | 忽视文化影响 |
| **政治化** | 被政治化 |

**相关词条**：进化论、心理学、适应主义

### 【生物伦理学】(Bioethics)

**定义**：研究生物学和医学中的伦理问题的学科。

**费曼解释**：
生物伦理学问：
- 基因编辑应该被允许吗？
- 动物实验是道德的吗？

**核心问题**：

| 问题 | 描述 |

| **动物权利** | 动物有什么权利？ |
| **资源分配** | 医疗资源如何分配？ |

**原则**：

| 原则 | 描述 |
|------|------|

| **有益性** | 有益于患者 |
| **无害性** | 不造成伤害 |
| **公正** | 公平分配 |

**相关词条**：伦理学、基因编辑、医学伦理
