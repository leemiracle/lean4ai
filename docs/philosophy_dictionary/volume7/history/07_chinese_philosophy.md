## A-F

### 【中国哲学】(Chinese Philosophy)

**定义**：中国传统的哲学思想体系，包括儒家、道家、佛家等。

**费曼解释**：
中国哲学关注"如何生活"，而不是"世界是什么"。
核心问题：如何成为君子？如何与道合一？

**中国哲学的特点**：

1. **实践性**：关注如何生活
2. **整体性**：天人合一
3. **关系性**：强调关系而非个体
4. **历史性**：重视传统和历史

**主要流派**：

| 流派 | 代表人物 | 核心关注 |
|------|----------|----------|
| **儒家** | 孔子、孟子、荀子 | 仁、礼、君子 |
| **道家** | 老子、庄子 | 道、无为、自然 |
| **佛家** | 禅宗、华严、天台 | 空、觉悟 |
| **墨家** | 墨子 | 兼爱、非攻 |
| **法家** | 韩非 | 法、术、势 |

**中国哲学的核心概念**：

| 概念 | 含义 |
|------|------|
| **道** | 宇宙的本原和规律 |
| **仁** | 仁爱、人性 |
| **礼** | 社会规范和仪式 |
| **德** | 德性、道德力量 |
| **气** | 生命能量、物质 |
| **理** | 道理、原则 |

**历史分期**：

| 时期 | 时间 | 特点 |
|------|------|------|
| **先秦** | 前770-前221 | 百家争鸣 |
| **汉唐** | 前206-907 | 儒术独尊、佛学传入 |
| **宋明** | 960-1644 | 理学、心学 |
| **近现代** | 1840- | 中西融合 |

**相关词条**：儒家、道家、佛家

### 【儒家】(Confucianism)

**定义**：以孔子为创始人的中国哲学传统。

**费曼解释**：
儒家问：如何成为君子？
答案：修身、齐家、治国、平天下。

**儒家的核心概念**：

1. **仁**：仁爱，人性的核心
2. **礼**：社会规范和仪式
3. **义**：正义、道义
4. **智**：智慧
5. **信**：诚信

**仁 (Ren)**：

- "仁者爱人"
- 人性的核心
- 推己及人："己所不欲，勿施于人"

**礼 (Li)**：

- 社会规范
- 行为准则
- 礼是仁的外在表现

**君子**：

| 特质 | 描述 |
|------|------|
| **仁** | 有仁爱之心 |
| **义** | 行义 |
| **礼** | 守礼 |
| **智** | 有智慧 |
| **信** | 有诚信 |

**五伦**：

| 关系 | 规范 |
|------|------|
| **君臣** | 君仁臣忠 |
| **父子** | 父慈子孝 |
| **夫妇** | 夫义妇听 |
| **兄弟** | 兄友弟恭 |
| **朋友** | 朋友有信 |

**代表人物**：

| 哲学家 | 核心贡献 |
|--------|----------|
| **孔子** | 创始人，仁与礼 |
| **孟子** | 性善论、仁政 |
| **荀子** | 性恶论、礼治 |
| **朱熹** | 理学、格物致知 |
| **王阳明** | 心学、致良知 |

**代表著作**：
- *《论语》*
- *《孟子》*
- *《荀子》*

**相关词条**：孔子、孟子、仁、礼

### 【道家】(Daoism)

**定义**：以老子、庄子为代表的中国哲学传统。

**费曼解释**：
道家说：无为而无不为。
不要强求，顺其自然，就像水一样。

**道家的核心概念**：

1. **道**：宇宙的本原和规律
2. **无为**：不强为，顺其自然
3. **自然**：本来的样子
4. **柔弱**：柔弱胜刚强

**道 (Dao)**：

- "道可道，非常道"
- 宇宙的本原
- 无法用语言完全描述

**无为**：

- 不是"什么都不做"
- 而是"不强为"
- 顺势而为

**老子 vs 庄子**：

| 问题 | 老子 | 庄子 |
|------|------|------|
| 关注 | 政治哲学 | 人生哲学 |
| 方法 | 无为而治 | 逍遥游 |
| 风格 | 简练 | 隐喻丰富 |

**老子的政治哲学**：

- 小国寡民
- 无为而治
- 柔弱胜刚强

**庄子的人生哲学**：

- 逍遥游
- 齐物论
- 蝴蝶梦

**代表著作**：
- *《道德经》*（老子）
- *《庄子》*

**相关词条**：老子、庄子、道、无为

### 【佛家】(Chinese Buddhism)

**定义**：佛教在中国的本土化发展，包括禅宗、华严、天台等。

**费曼解释**：
佛家说：一切皆空。
不是说什么都没有，而是说没有固定不变的自性。

**中国佛教的核心概念**：

1. **空**：无自性
2. **缘起**：因缘和合
3. **觉悟**：成佛
4. **涅槃**：解脱

**主要宗派**：

| 宗派 | 特点 | 代表 |
|------|------|------|
| **禅宗** | 直指人心，见性成佛 | 惠能 |
| **华严** | 法界缘起 | 法藏 |
| **天台** | 一念三千 | 智者大师 |
| **净土** | 念佛往生 | 慧远 |

**禅宗**：

- 不立文字
- 直指人心
- 见性成佛

**公案**：

| 公案 | 问题 |
|------|------|
| **赵州狗子** | 狗子有佛性也无？ |
| **柏树子** | 如何是祖师西来意？ |

**空 (Śūnyatā)**：

- 不是虚无
- 而是无自性
- 缘起故空

**中国佛教的贡献**：

- 佛教中国化
- 三教合一（儒释道）
- 对文学艺术的影响

**相关词条**：禅宗、空、缘起

## G-Z

### 【墨家】(Mohism)

**定义**：墨子创立的哲学流派，主张兼爱、非攻。

**费曼解释**：
墨家说：为什么只爱自己的家人？应该"兼爱"——平等地爱所有人。
战争是不对的——"非攻"。

**核心思想**：
1. **兼爱**：平等地爱所有人
2. **非攻**：反对侵略战争
3. **尚贤**：任人唯贤
4. **节用**：节约使用资源

**相关词条**：中国哲学、兼爱

---

### 【法家】(Legalism)

**定义**：强调法治和政治权力的中国哲学流派。

**费曼解释**：
法家说：不要相信人的道德——人趋利避害。
需要法律和惩罚来维持秩序。

**核心思想**：
1. **法**：公开的法律
2. **术**：统治的技术
3. **势**：权力的威势

**相关词条**：中国哲学、韩非

---

### 【理学】(Neo-Confucianism/Lixue)

**定义**：宋明时期儒家哲学的新发展。

**费曼解释**：
理学融合了儒、释、道三家。
"理"是宇宙的本原和规律——万物有"理"。

**核心思想**：
1. **理**：宇宙的本原和规律
2. **气**：物质基础
3. **格物致知**：通过研究事物获得知识
4. **存天理，灭人欲**：道德修养

**代表人物**：
- **朱熹**：理学集大成者
- **程颢、程颐**：理学先驱

**相关词条**：中国哲学、儒家

---

### 【心学】(School of Mind/Xinxue)

**定义**：强调"心"的儒家流派，由王阳明发展。

**费曼解释**：
心学说：道理不在外面，在心里。
"心即理"——心就是理。
"知行合一"——知和行是合一的。

**核心思想**：
1. **心即理**：心就是理
2. **知行合一**：知和行不可分
3. **致良知**：发现内心的良知

**代表人物**：
- **王阳明**：心学创始人

**相关词条**：中国哲学、理学
