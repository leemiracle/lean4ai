### 【近代哲学】(Modern Philosophy)

**定义**：17-18世纪欧洲哲学，以理性主义和经验主义的对立与综合为特征。

**费曼解释**：
近代哲学问：我们如何获得知识？
- 理性主义说：通过理性，不需要经验
- 经验主义说：通过经验，没有先天知识
- 康德说：两者都对，也都错

**近代哲学的分期**：

| 时期 | 时间 | 特点 |
|------|------|------|
| **早期近代** | 17世纪 | 理性主义兴起 |
| **中期近代** | 17-18世纪 | 理性主义与经验主义 |
| **晚期近代** | 18世纪末 | 康德的批判哲学 |

**两大传统**：

| 传统 | 核心主张 | 代表人物 |
|------|----------|----------|
| **理性主义** | 知识源于理性 | 笛卡尔、斯宾诺莎、莱布尼茨 |
| **经验主义** | 知识源于经验 | 洛克、贝克莱、休谟 |

**核心问题**：

| 问题 | 描述 |
|------|------|
| **知识来源** | 知识从哪里来？ |
| **方法论怀疑** | 如何找到确定的知识？ |
| **心身问题** | 心灵与身体的关系？ |
| **自由意志** | 人有自由意志吗？ |

**代表人物**：
- **笛卡尔**：近代哲学之父
- **斯宾诺莎**：泛神论
- **莱布尼茨**：单子论
- **洛克**：经验主义创始人
- **贝克莱**：主观唯心主义
- **休谟**：怀疑论
- **康德**：批判哲学

**经典文献**：
- Descartes, *Meditations on First Philosophy*
- Locke, *An Essay Concerning Human Understanding*
- Hume, *A Treatise of Human Nature*
- Kant, *Critique of Pure Reason*

**当代发展 (2020-2026)**：
- **近代哲学研究**：新的解读
- **分析哲学史**：用分析哲学方法研究
- **比较研究**：与早期现代思想比较

**中国视角**：

- **格物**：与格物致知比较

3. 康德的综合成功了吗？

### 【笛卡尔】(René Descartes)

**生卒**：1596-1650

**定义**：法国哲学家、数学家、科学家，被称为"近代哲学之父"。

**费曼解释**：
笛卡尔做了一件激进的事：
- 他怀疑一切：感官、数学、甚至上帝
- 他发现有一件事不能怀疑：我在怀疑
- 所以"我思故我在"
- 从这里重建所有知识

**笛卡尔的核心思想**：

| 思想 | 内容 |
|------|------|
| **方法论怀疑** | 怀疑一切可怀疑的 |
| **我思故我在** | 确定的起点 |
| **心身二元** | 心灵与身体是两种实体 |
| **天赋观念** | 某些观念是天赋的 |

**方法论怀疑**：

| 怀疑对象 | 怀疑理由 |
|----------|----------|
| **感官** | 感官可能欺骗我们 |
| **梦** | 可能是在做梦 |
| **恶魔** | 可能被恶魔欺骗 |
| **数学** | 恶魔可能让数学错误 |

**我思故我在**：

```
我在怀疑 → 我在思考 → 我存在
（Cogito, ergo sum）
```

为什么这确定？
- 即使怀疑，也在思考
- 即使被欺骗，也在思考
- 思考不能没有思考者

**清楚明白原则**：

| 原则 | 内容 |
|------|------|
| **标准** | 清楚明白的观念是真的 |
| **保证** | 上帝保证这个标准 |
| **应用** | 用于重建知识 |

**心身二元**：

| 实体 | 属性 | 模式 |
|------|------|------|
| **心灵** | 思维 | 思想、意愿、怀疑 |
| **身体** | 广延 | 形状、大小、运动 |

**心身问题**：

- 问题：心灵（思维）和身体（广延）如何相互作用？
- 笛卡尔的回答：松果体
- 后来的批评：这不解决问题

**上帝存在的证明**：

| 证明 | 内容 |
|------|------|
| **上帝观念** | 我有无限完满的观念 |
| **因果原则** | 原因必须大于结果 |
| **结论** | 无限完满的上帝存在 |

**主要著作**：

| 著作 | 内容 |
|------|------|
| 《第一哲学沉思集》 | 形而上学 |
| 《方法论》 | 方法论 |
| 《哲学原理》 | 系统哲学 |
| 《几何》 | 数学 |

**经典文献**：
- Descartes, *Meditations on First Philosophy*
- Descartes, *Discourse on the Method*

**当代发展 (2020-2026)**：
- **笛卡尔研究**：新的解读
- **心身问题**：心身问题的当代讨论
- **怀疑论**：笛卡尔与怀疑论

**中国视角**：
- **怀疑**：与怀疑思想比较
- **心身**：与心身思想比较
- **理性**：与理性思想比较

**争议问题**：
1. "我思故我在"是有效的论证吗？
2. 心身二元论是否正确？

### 【我思故我在】(Cogito, ergo sum)

**定义**：笛卡尔的第一确定性，"我在怀疑"证明"我存在"。

**费曼解释**：
笛卡尔问：有什么是我绝对确定的？
- 我看见一棵树？不，可能是幻觉
- 1+1=2？不，恶魔可能让我搞错
- 但我在怀疑 → 我在思考 → 我存在
- 这是不可以怀疑的

**论证的结构**：

| 步骤 | 内容 |
|------|------|
| 1 | 我在怀疑 |
| 2 | 怀疑是一种思考 |
| 3 | 思考必须有思考者 |
| 4 | 所以我存在 |

**为什么确定？**：

| 可能性 | 为什么不能否定我思 |
|----------|---------------------|
| 感官欺骗 | 即使被骗，也在思考 |
| 做梦 | 即使做梦，也在思考 |
| 恶魔欺骗 | 即使被欺骗，也在思考 |
| 否认 | 否认也在思考 |

**不同解释**：

| 解释 | 内容 |
|------|------|
| **推论** | 从"我思考"推出"我存在" |
| **直观** | "我存在"是直接被把握的 |
| **表演性** | 思考的行为预设了存在 |

**对Cogito的批评**：

| 批评 | 内容 |
|------|------|
| **"我"的问题** | 思考不预设"我" |
| **循环** | 预设了主谓结构 |
| **瞬间存在** | 只证明现在的存在 |
| **尼采** | "思考"不是"我思考" |

**经典文献**：
- Descartes, *Meditations* II
- Descartes, *Discourse on the Method* IV

**当代发展 (2020-2026)**：
- **自我意识**：Cogito与自我意识
- **第一人称**：第一人称视角的哲学
- **基础主义**：基础信念

**中国视角**：
- **自知**：自知之明
- **反思**：反思与自我意识
- **诚意**：诚意与自欺

**争议问题**：
1. Cogito是推论还是直观？
2. "我"是什么？
3. Cogito建立了什么？

**相关词条**：笛卡尔、方法论怀疑、自我意识

### 【心身问题】(Mind-Body Problem)

**定义**：心灵与身体的关系问题，由笛卡尔的二元论明确化。

**费曼解释**：
心身问题问：
- 你有身体（物质的）
- 你有心灵（精神的）
- 它们是什么关系？
- 它们如何相互作用？

**笛卡尔的二元论**：

| 实体 | 属性 | 特点 |
|------|------|------|
| **心灵** | 思维 | 不可分、非空间 |
| **身体** | 广延 | 可分、空间性 |

**相互作用问题**：

| 问题 | 描述 |
|------|------|
| **如何互动？** | 非物质的心灵如何影响物质的身体？ |
| **在哪里互动？** | 笛卡尔：松果体 |
| **因果封闭性** | 物理世界是因果封闭的 |

**主要的解决方案**：

| 理论 | 主张 | 代表 |
|------|------|------|
| **互动论** | 心灵和身体相互作用 | 笛卡尔 |
| **偶因论** | 上帝导致心灵和身体协调 | 马勒伯朗士 |
| **平行论** | 心灵和身体平行运行 | 莱布尼茨 |
| **副现象论** | 心灵是身体的副产品 | 休谟 |
| **同一论** | 心灵=大脑 | 当代物理主义 |
| **功能主义** | 心灵是功能状态 | 当代功能主义 |

**偶因论**（马勒伯朗士）：

```
心灵想动胳膊 → 上帝导致胳膊动
     ↓              ↓
   不直接因果     上帝作为真正原因
```

**平行论**（莱布尼茨）：

```
心灵 ←── 预定和谐 ──→ 身体
     （像两个同步的钟）
```

**当代发展**：

| 理论 | 主张 |
|------|------|
| **物理主义** | 只存在物理事物 |
| **功能主义** | 心灵是功能状态 |
| **性质二元论** | 只有一种实体，但有心理和物理性质 |
| **取消主义** | 心灵概念将被取消 |

**经典文献**：
- Descartes, *Meditations* VI
- Leibniz, *Monadology*

**当代发展 (2020-2026)**：
- **神经科学**：心身问题的新视角
- **意识研究**：意识的难问题
- **心灵哲学**：心灵哲学的新发展

**中国视角**：
- **心身**：与心身关系比较
- **形神**：形神关系
- **心物**：心物关系

**争议问题**：
1. 二元论是否正确？
2. 心灵如何影响身体？
3. 意识能否被还原？

**相关词条**：笛卡尔、莱布尼茨、心灵哲学、意识

### 【天赋观念】(Innate Ideas)

**定义**：理性主义的主张，认为某些观念是天赋的，不依赖于经验。

**费曼解释**：
天赋观念说：
- 有些知识我们"天生"就知道
- 不是从经验学来的
- 比如：数学真理、逻辑法则、上帝观念
- 经验只是"唤醒"这些天赋观念

**理性主义与经验主义之争**：

| 问题 | 理性主义 | 经验主义 |
|------|----------|----------|
| **知识来源** | 理性 | 经验 |
| **天赋观念** | 存在 | 不存在 |
| **知识确定性** | 理性知识最确定 | 只有经验知识 |

**笛卡尔的天赋观念**：

| 观念 | 来源 |
|------|------|
| **上帝** | 天赋的 |
| **自我** | 天赋的 |
| **数学真理** | 天赋的 |
| **感官观念** | 外来的或造作的 |

**莱布尼茨的天赋观念**：

| 比喻 | 内容 |
|------|------|
| **大理石纹路** | 天赋观念像大理石的纹路 |
| **不是白板** | 心灵不是洛克说的白板 |
| **潜能** | 天赋观念是潜能，需要经验触发 |

**洛克的反驳**：

| 批评 | 内容 |
|------|------|
| **没有普遍同意** | 如果天赋，为什么不是所有人都知道？ |
| **白板说** | 心灵是白板，所有知识来自经验 |
| **经验解释** | 经验可以解释所谓"天赋"知识 |

**莱布尼茨的回应**：

| 回应 | 内容 |
|------|------|
| **潜能** | 天赋观念是潜能，需要被唤醒 |
| **必然真理** | 必然真理不能从经验得出 |
| **逻辑原则** | 同一律、矛盾律是天赋的 |

**康德的先天综合**：

| 概念 | 内容 |
|------|------|
| **先天** | 独立于经验 |
| **综合** | 增加知识内容 |
| **先天综合** | 既是先天的又是综合的 |

**经典文献**：
- Descartes, *Meditations* III
- Leibniz, *New Essays on Human Understanding*
- Locke, *Essay* I

**当代发展 (2020-2026)**：
- **乔姆斯基**：语言能力是天赋的
- **认知科学**：核心知识
- **进化心理学**：心理模块是天赋的

**中国视角**：
- **良知**：与良知比较
- **性善**：与性善论比较
- **先验**：与先验比较

**争议问题**：
1. 天赋观念存在吗？
2. 如果存在，它们是什么？
3. 天赋观念如何被"唤醒"？

**相关词条**：笛卡尔、莱布尼茨、理性主义、经验主义

### 【斯宾诺莎】(Baruch Spinoza)

**生卒**：
