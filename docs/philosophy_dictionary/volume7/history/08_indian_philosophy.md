## A-F

### 【印度哲学】(Indian Philosophy)

**定义**：印度次大陆的哲学传统，包括印度教、佛教、耆那教等。

**费曼解释**：
印度哲学问：如何从轮回中解脱？
答案：通过知识、行动或奉献，达到解脱。

**印度哲学的特点**：

1. **解脱导向**：目标是从轮回中解脱
2. **整体性**：哲学与宗教、生活结合
3. **多元性**：多种哲学传统并存
4. **辩证性**：通过辩论发展

**主要流派**：

| 流派 | 核心关注 | 代表 |
|------|----------|------|
| **吠檀多** | 梵我一如 | 商羯罗 |
| **数论** | 二元论 | 自性、神我 |
| **瑜伽** | 修行方法 | 帕坦伽利 |
| **佛教** | 缘起、空 | 龙树 |
| **耆那教** | 非暴力 | 大雄 |

**印度哲学的核心概念**：

| 概念 | 含义 |
|------|------|
| **梵 (Brahman)** | 宇宙的终极实在 |
| **我 (Atman)** | 个体的真实自我 |
| **法 (Dharma)** | 职责、真理、法则 |
| **业 (Karma)** | 因果报应 |
| **轮回 (Samsara)** | 生死循环 |
| **解脱 (Moksha)** | 从轮回中解脱 |

**印度哲学的目标**：

- 从轮回中解脱
- 认识梵我一如
- 达到涅槃

**相关词条**：吠檀多、佛教、业

### 【吠檀多】(Vedanta)

**定义**：印度教哲学的主流传统，基于《奥义书》。

**费曼解释**：
吠檀多说：你本质上就是梵。
你只是忘记了。一旦认识梵我一如，就解脱了。

**吠檀多的核心概念**：

1. **梵我一如**：个体我和宇宙梵是同一的
2. **摩耶**：幻象，世界是幻象
3. **解脱**：通过知识解脱

**商羯罗的不二论**：

| 概念 | 描述 |
|------|------|
| **梵** | 唯一的实在 |
| **世界** | 幻象（摩耶） |
| **我** | 与梵同一 |

**梵我一如**：

- 个体的自我（我）与宇宙的实在（梵）是同一的
- 我们只是因为无明而不知道
- 通过知识，认识梵我一如

**代表著作**：
- *《奥义书》*
- *《梵经》*
- 商羯罗注释

**相关词条**：印度哲学、梵、解脱

### 【业/因果】(Karma)

**定义**：印度哲学中的因果报应法则。

**费曼解释**：
业说：你现在的状态是过去行为的结果。
你未来的状态取决于现在的行为。

**业的特点**：

1. **因果性**：善恶有报
2. **累积性**：业力累积
3. **道德性**：与道德相关

**业的类型**：

| 类型 | 描述 |
|------|------|
| **善业** | 好的行为，好的结果 |
| **恶业** | 坏的行为，坏的结果 |
| **无记业** | 中性的行为 |

**业与轮回**：

- 业决定来世
- 善业升天，恶业堕地狱
- 解脱需要消除业

**佛教对业的看法**：

- 业是缘起的一部分
- 不是固定的命运
- 可以改变

**相关词条**：轮回、解脱、佛教

### 【佛教哲学】(Buddhist Philosophy)

**定义**：佛教的哲学思想，包括缘起、空、中观等。

**费曼解释**：
佛教说：一切皆空，缘起故空。
没有固定不变的东西，一切都是因缘和合。

**佛教的核心概念**：

1. **四谛**：苦、集、灭、道
2. **缘起**：因缘和合
3. **空**：无自性
4. **涅槃**：解脱

**四谛**：

| 谛 | 描述 |
|------|------|
| **苦谛** | 生命是苦 |
| **集谛** | 苦的原因（执着） |
| **灭谛** | 苦的止息（涅槃） |
| **道谛** | 八正道 |

**缘起**：

- 此有故彼有
- 一切都是因缘和合
- 没有独立自性

**空 (Śūnyatā)**：

- 不是虚无
- 而是无自性
- 龙树：缘起故空

**中观**：

- 不落两边
- 非有非无
- 空亦复空

**代表人物**：
- **释迦牟尼**：创始人
- **龙树**：中观创始人
- **无著/世亲**：唯识宗

**代表著作**：
- *《中论》*
- *《唯识三十颂》*

**相关词条**：空、缘起、涅槃

## G-Z

### 【耆那教】(Jainism)

**定义**：起源于印度的古老宗教，由大雄创立。

**费曼解释**：
耆那教说：不伤害任何生命——"不杀生"是最高原则。
通过苦行和禁欲，解脱灵魂。

**核心思想**：
1. **不杀生**：不伤害任何生命
2. **不妄语**：不撒谎
3. **不偷盗**：不偷窃
4. **不邪淫**：禁欲
5. **不执著**：不执着于物质

**相关词条**：印度哲学、非暴力

---

### 【数论】(Sankhya)

**定义**：印度哲学六派之一，二元论哲学。

**费曼解释**：
数论说：世界有两种实在——神（意识）和自性（物质）。
解脱在于认识神和自性的区别。

**核心概念**：
1. **神**：纯粹的意识
2. **自性**：物质基础
3. **三德**：自性的三种性质

**相关词条**：印度哲学、二元论

---

### 【瑜伽】(Yoga)

**定义**：印度哲学六派之一，实践性的解脱方法。

**费曼解释**：
瑜伽不只是体式——它是"心的止息"。
通过修行，控制心念，达到解脱。

**核心概念**：
1. **八支瑜伽**：八个修行阶段
2. **三昧**：深度冥想状态
3. **独存**：最终解脱状态

**相关词条**：印度哲学、冥想
