# 7.5 20世纪分析哲学 (20th Century Analytic Philosophy)

> **难度**：基础到进阶
> **更新**：2026-03-26

## A-F

### 【分析哲学】(Analytic Philosophy)

**定义**：20世纪英美哲学的主流传统，强调逻辑分析和语言分析。

**费曼解释**：
分析哲学说：哲学问题往往是语言问题。
通过分析语言，我们可以澄清甚至解决哲学问题。

**分析哲学的特点**：

1. **逻辑分析**：使用现代逻辑
2. **语言分析**：分析日常语言
3. **科学取向**：与科学密切关联
4. **清晰性**：追求清晰、严谨

**主要阶段**：

| 阶段 | 时间 | 特点 | 代表 |
|------|------|------|------|
| **逻辑分析** | 1900-1930 | 逻辑、理想语言 | 罗素、早期维特根斯坦 |
| **逻辑实证主义** | 1920-1950 | 证实原则 | 维也纳学派 |
| **日常语言哲学** | 1945-1970 | 日常语言分析 | 后期维特根斯坦、奥斯汀 |
| **当代分析哲学** | 1970- | 多元化 | 克里普克、刘易斯等 |

**核心人物**：

- **弗雷格**：现代逻辑创始人
- **罗素**：逻辑原子论
- **维特根斯坦**：语言哲学革命
- **卡尔纳普**：逻辑实证主义
- **蒯因**：自然化认识论
- **克里普克**：可能世界语义学

**相关词条**：罗素、维特根斯坦、逻辑实证主义

---

### 【罗素】(Russell)

**定义**：英国哲学家、逻辑学家，分析哲学的奠基人之一。

**费曼解释**：
罗素说：哲学应该像科学一样严谨。
用逻辑分析来解决哲学问题。

**罗素的核心贡献**：

1. **逻辑学**：与怀特海合著《数学原理》
2. **类型论**：解决罗素悖论
3. **逻辑原子论**：世界由原子事实构成
4. **摹状词理论**：分析限定摹状词

**罗素悖论**：

```
所有不包含自己的集合的集合
  → 如果包含自己，则不包含
  → 如果不包含自己，则包含
```

**摹状词理论**：

- "当今法国国王是秃头"如何分析？
- 罗素：当今存在一个法国国王，他是秃头
- 因为没有法国国王，所以这句话是假的

**逻辑原子论**：

- 世界由原子事实构成
- 语言反映世界结构
- 复杂命题由原子命题构成

**代表著作**：
- *《数学原理》*（与怀特海）
- *《哲学问题》*
- *《逻辑原子论》*

**相关词条**：分析哲学、弗雷格、维特根斯坦

### 【维特根斯坦】(Wittgenstein)

**定义**：奥地利裔英国哲学家，语言哲学的革命者。

**费曼解释**：
早期维特根斯坦：语言描画世界，不能说的必须沉默。
晚期维特根斯坦：语言是游戏，意义在于使用。

**两个维特根斯坦**：

| 时期 | 著作 | 核心观点 |
|------|------|----------|
| **早期** | 《逻辑哲学论》 | 图像论、不可说 |
| **晚期** | 《哲学研究》 | 语言游戏、家族相似 |

**早期：图像论**：

- 命题是事实的图像
- 语言与世界有相同的逻辑结构
- "凡是可说的都能说清楚，凡是不可说的必须沉默"

**晚期：语言游戏**：

- 语言的意义在于使用
- 语言有多种"游戏"
- 没有共同本质，只有"家族相似"

**家族相似**：

- "游戏"没有共同特征
- 只有重叠交叉的相似性
- 就像一个家族的成员

**私人语言论证**：

- 私人语言（只有我理解的语言）是不可能的
- 意义需要公共规则
- "盒中甲虫"：即使我有盒中甲虫，也无法有意义地谈论

**代表著作**：
- *《逻辑哲学论》* (1921)
- *《哲学研究》* (1953)

**相关词条**：分析哲学、语言游戏、家族相似

### 【逻辑实证主义】(Logical Positivism)

**定义**：20世纪20-50年代的哲学运动，强调证实原则。

**费曼解释**：
逻辑实证主义说：一个陈述要有意义，必须能被证实。
"上帝存在"不能被证实，所以是无意义的。

**证实原则**：

```
陈述有意义的条件：
1. 分析命题（逻辑/数学真理）
2. 综合命题（可被经验证实）
否则：无意义（形而上学）
```

**维也纳学派**：

| 成员 | 贡献 |
|------|------|
| **石里克** | 学派领袖 |
| **卡尔纳普** | 逻辑分析 |
| **纽拉特** | 物理主义 |
| **哥德尔** | 不完全性定理（批判） |

**核心主张**：

1. **反形而上学**：形而上学陈述无意义
2. **统一科学**：所有科学可还原为物理学
3. **逻辑分析**：用逻辑澄清哲学问题
4. **经验主义**：知识来自经验

**对逻辑实证主义的批评**：

1. **证实原则自驳**：
   - 证实原则本身不能被证实
   - 所以它是无意义的？

2. **整体论**：
   - 蒯因：信念是整体面对经验
   - 不能逐个证实

3. **历史失败**：
   - 20世纪50年代后衰落
   - 但对分析哲学影响深远

**代表著作**：
- Carnap, *The Logical Structure of the World*
- Ayer, *Language, Truth and Logic*

**相关词条**：分析哲学、卡尔纳普、蒯因

## G-Z

### 【蒯因】(Willard Van Orman Quine, 1908-2000)

**定义**：美国哲学家，分析哲学的重要人物。

**费曼解释**：
蒯因说：我们的知识是一个"网络"——边缘可以修改，但核心很难改。
没有"必然真理"——一切都可以修正。
翻译是不确定的。

**核心思想**：
1. **自然化认识论**：认识论应该自然科学化
2. **翻译不确定性**：翻译没有唯一正确答案
3. **本体论承诺**：存在就是成为约束变元的值
4. **整体论**：知识是一个整体

**整体论**：

| 观点 | 描述 |
|------|------|
| **信念网络** | 知识是相互关联的网络 |
| **边缘vs核心** | 边缘可以修改，核心稳定 |
| **经验法庭** | 整体面对经验检验 |

**翻译不确定性**：
- 可以有多个同样好的翻译手册
- 没有事实决定哪个翻译是"正确"的
- 意义不只是行为

**代表著作**：
- *《从逻辑的观点看》* (1953)
- *《词与物》* (1960)

**相关词条**：分析哲学、整体论、自然主义

### 【克里普克】(Saul Kripke, 1940-2022)

**定义**：美国哲学家，模态逻辑和语言哲学的重要贡献者。

**费曼解释**：
克里普克改变了很多事情：
- 名字没有"意义"，只有"指称"
- 有些真理是必然的但需要经验发现
- 语言有"严格指示词"

**核心贡献**：

| 贡献 | 描述 |
|------|------|
| **严格指示词** | 名字在所有可能世界指称同一对象 |
| **后验必然** | 水=H₂O是必然的但需要经验发现 |
| **先验偶然** | 有些先验真理是偶然的 |
| **专名理论** | 专名没有描述性内容 |

**后验必然**：
- "水是H₂O"是必然的
- 但我们需要经验才能知道
- 区分认识论和形而上学模态

**克里普克语义学**：
- 可能世界语义学
- 为模态逻辑提供模型
- 影响整个哲学

**代表著作**：
- *《命名与必然性》* (1980)

**相关词条**：模态逻辑、指称理论、后验必然

---

### 【戴维森】(Donald Davidson, 1917-2003)

**定义**：美国哲学家，语言哲学和心灵哲学的重要贡献者。

**费曼解释**：
戴维森说：解释他人语言，你需要假设他们大多是理性的。
这就是"宽容原则"——给对方最大的合理性。

**核心贡献**：

| 贡献 | 描述 |
|------|------|
| **彻底解释** | 如何在没有任何预设的情况下解释语言 |
| **宽容原则** | 假设对方信念大多是真和理性的 |
| **异常一元论** | 心理事件是物理事件，但没有心理定律 |
| **真理语义学** | 真理理论作为意义理论 |

**彻底解释**：
- 想象你遇到一个完全陌生的语言
- 如何开始理解？
- 通过观察行为和假设理性

**异常一元论**：
- 心理事件 = 物理事件
- 但没有严格的心理定律
- 心理描述是"异常的"

**代表著作**：
- *《真理与解释的研究》* (1984)

**相关词条**：语言哲学、心灵哲学、解释理论

---

### 【普特南】(Hilary Putnam, 1926-2016)

**定义**：美国哲学家，心灵哲学、语言哲学和数学哲学的重要贡献者。

**费曼解释**：
普特南提出很多重要思想：
- "意义不在头脑里"——外在论
- "缸中之脑"——怀疑论论证
- 功能主义——心灵是功能

**核心贡献**：

| 贡献 | 描述 |
|------|------|
| **语义外在论** | 意义部分在于环境 |
| **缸中之脑** | 怀疑论的思想实验 |
| **功能主义** | 心灵是功能状态 |
| **内在实在论** | 真理是内在的 |

**孪生地球论证**：
- 孪生地球上有"水"（XYZ不是H₂O）
- 地球人和孪生地球人心理状态相同
- 但"水"指称不同
- 因此意义不在头脑里

**缸中之脑**：
- 你怎么知道你不是缸中之脑？
- 普特南后来论证：你不能是缸中之脑
- 因为"缸"指称需要外在联系

**代表著作**：
- *《理性、真理与历史》* (1981)

**相关词条**：外在论、功能主义、实在论
