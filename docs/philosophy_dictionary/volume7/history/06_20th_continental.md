## A-F

### 【欧陆哲学】(Continental Philosophy)

**定义**：20世纪欧洲大陆的哲学传统，包括现象学、存在主义、后结构主义等。

**费曼解释**：
分析哲学关注语言和逻辑，欧陆哲学关注存在、历史、社会。
就像两个不同的望远镜，看不同的方向。

**欧陆哲学的流派**：

| 流派 | 代表人物 | 核心关注 |
|------|----------|----------|
| **现象学** | 胡塞尔、海德格尔 | 意识、存在 |
| **存在主义** | 萨特、加缪 | 自由、荒谬 |
| **解释学** | 伽达默尔、利科 | 理解、解释 |
| **批判理论** | 法兰克福学派 | 社会批判 |
| **后结构主义** | 福柯、德里达 | 权力、解构 |

**欧陆哲学的特点**：

1. **历史性**：关注历史和传统
2. **系统性**：构建宏大体系
3. **文学性**：与文学密切关联
4. **批判性**：批判现代性

**与英美分析哲学的对比**：

| 特征 | 分析哲学 | 欧陆哲学 |
|------|----------|----------|
| 方法 | 逻辑分析 | 现象学描述 |
| 风格 | 清晰、严谨 | 文学、隐喻 |
| 关注 | 语言、逻辑 | 存在、社会 |
| 代表 | 罗素、维特根斯坦 | 胡塞尔、海德格尔 |

**相关词条**：现象学、存在主义、海德格尔

### 【现象学】(Phenomenology)

**定义**：研究意识及其对象的哲学方法，由胡塞尔创立。

**费曼解释**：
现象学说：回到事物本身。
不要预设，直接描述意识中呈现的东西。

**胡塞尔的现象学**：

1. **回到事物本身**：悬置预设
2. **意向性**：意识总是关于某物的意识
3. **悬置/加括号**：暂停存在判断
4. **本质直观**：直观本质

**意向性**：

```
意识 → 对象
（意识总是指向某个对象）
```

**悬置 (Epoché)**：

- 暂停"自然态度"
- 不判断对象是否存在
- 只关注意识中呈现的

**现象学还原**：

| 步骤 | 描述 |
|------|------|
| **悬置** | 暂停存在判断 |
| **本质还原** | 把握本质 |
| **先验还原** | 达到先验主体 |

**海德格尔的存在论**：

- 从意识转向存在
- 此在：人的存在方式
- 在世存在：人总是在世界中

**代表著作**：
- Husserl, *Logical Investigations*
- Husserl, *Ideas I*
- Heidegger, *Being and Time*

**相关词条**：胡塞尔、海德格尔、意向性

### 【海德格尔】(Heidegger)

**定义**：德国哲学家，存在主义和现象学的关键人物。

**费曼解释**：
海德格尔问：存在的意义是什么？
不是问"这是什么"，而是问"存在本身意味着什么"。

**海德格尔的核心概念**：

1. **存在与存在者**：区分存在本身和具体存在者
2. **此在**：人的存在方式
3. **在世存在**：人总是在世界中存在
4. **操心**：此在的基本结构

**此在**：

- 此在 = Dasein = "在这里存在"
- 人的独特存在方式
- 存在对其是个问题

**在世存在**：

| 结构 | 描述 |
|------|------|
| **理解** | 对存在的理解 |
| **情态** | 情感状态（焦虑等） |
| **话语** | 语言性 |

**本真与非本真**：

| 类型 | 描述 |
|------|------|
| **非本真** | 沉沦于"常人"（das Man） |
| **本真** | 面向死亡，真实地存在 |

**向死存在**：

- 死亡是此在最本己的可能性
- 面向死亡，本真地存在
- "提前到死亡中去"

**技术批判**：

- 现代技术是"座架" (Gestell)
- 把一切都变成"持存物"
- 需要另一种思维方式

**代表著作**：
- *《存在与时间》* (1927)
- *《技术的追问》*

**相关词条**：现象学、存在主义、萨特

### 【萨特】(Sartre)

**定义**：法国哲学家、作家，存在主义的代表人物。

**费曼解释**：
萨特说："存在先于本质"
你不是被定义的，你必须定义自己。
你是自由的，"被判自由"。

**萨特的核心概念**：

1. **存在先于本质**：人先存在，后定义自己
2. **绝对自由**：人是绝对自由的
3. **自欺**：逃避自由的责任
4. **他人即地狱**：他人的凝视客体化我

**存在先于本质**：

- 人造物品：本质先于存在（剪刀的本质先于它被制造）
- 人：存在先于本质（人先存在，然后定义自己）

**自由与责任**：

- 人是"被判自由"的
- 自由带来责任
- 不能借口"我就是这样"

**自欺 (Bad Faith)**：

| 例子 | 自欺的表现 |
|------|-----------|
| **服务员** | 我只是一个服务员 |
| **懦夫** | 我天生是懦夫 |
| **恋人** | 我身不由己 |

**他人即地狱**：

- 他人的凝视将我客体化
- 我成为他人的对象
- 冲突是不可避免的

**代表著作**：
- *《存在与虚无》* (1943)
- *《存在主义是一种人道主义》* (1946)
- *《恶心》* (小说)

**相关词条**：存在主义、海德格尔、加缪

## G-Z

### 【福柯】(Michel Foucault, 1926-1984)

**定义**：法国哲学家，权力/知识理论和社会批判的代表。

**费曼解释**：
福柯说：权力不只是压制，也是生产。
知识不是中立的——它和权力纠缠在一起。
"权力/知识"是一个整体。

**核心思想**：
1. **权力/知识**：权力和知识不可分割
2. **话语**：话语构成现实
3. **规训**：现代社会是规训社会
4. **生命政治**：对生命的管理

**权力理论**：

| 传统观点 | 福柯观点 |
|----------|----------|
| 权力是压制 | 权力也是生产 |
| 权力是集中的 | 权力是弥散的 |
| 权力是自上而下 | 权力无处不在 |

**规训与惩罚**：
- 监狱的诞生
- 从肉体惩罚到规训
- "全景敞视主义"

**代表著作**：
- *《规训与惩罚》* (1975)
- *《疯癫与文明》* (1961)

**相关词条**：权力、后结构主义、社会批判

---

### 【德里达】(Jacques Derrida, 1930-2004)

**定义**：法国哲学家，解构主义的创始人。

**费曼解释**：
德里达说：文本没有固定意义。
意义总是"延异"(différance)——推迟和不同。
"解构"不是破坏，而是揭示文本的矛盾。

**核心概念**：

| 概念 | 描述 |
|------|------|
| **延异** | 意义总是推迟的 |
| **解构** | 揭示文本的内在矛盾 |
| **逻各斯中心主义** | 西方对在场/真理的迷恋 |
| **书写** | 书写不是派生的 |

**解构**：
- 不是简单的否定
- 而是揭示文本的自我矛盾
- 文本"解构"自己

**对逻各斯中心主义的批判**：
- 西方哲学追求"在场"
- 贬低书写，抬高言语
- 但言语也有"书写"特征

**代表著作**：
- *《论书写学》* (1967)
- *《书写与差异》* (1967)

**相关词条**：后结构主义、解构、后现代主义

---

### 【利奥塔】(Jean-François Lyotard, 1924-1998)

**定义**：法国哲学家，后现代主义的重要理论家。

**费曼解释**：
利奥塔说：后现代就是"对元叙事的怀疑"。
没有大故事能统一一切——科学、进步、解放的故事都不可靠。

**核心思想**：
1. **后现代状况**：元叙事的衰落
2. **元叙事**：统一性的大故事
3. **语言游戏**：多元化的语言游戏
4. **合法性危机**：科学知识的合法性危机

**元叙事 vs 小叙事**：

| 元叙事 | 小叙事 |
|--------|--------|
| 统一性 | 多元性 |
| 普遍真理 | 局部真理 |
| 进步故事 | 异质故事 |

**后现代定义**：
"后现代就是对元叙事的怀疑。"

**代表著作**：
- *《后现代状况》* (1979)

**相关词条**：后现代主义、元叙事、知识合法性

### 【法兰克福学派】(Frankfurt School)

**定义**：以法兰克福大学社会研究所为中心的批判理论学派。

**费曼解释**：
法兰克福学派说：启蒙变成了它的反面——神话。
理性变成了统治的工具——工具理性。
批判理论就是要批判这个"启蒙辩证法"。

**核心人物**：

| 人物 | 贡献 |
|------|------|
| **霍克海默** | 批判理论创始人 |
| **阿多诺** | 启蒙辩证法 |
| **马尔库塞** | 单向度的人 |
| **哈贝马斯** | 交往理性 |

**核心概念**：

| 概念 | 描述 |
|------|------|
| **批判理论** | 批判社会的理论 |
| **工具理性** | 只关注目的的理性 |
| **启蒙辩证法** | 启蒙变成神话 |
| **单向度的人** | 失去批判能力的人 |

**代表著作**：
- 霍克海默 & 阿多诺, *《启蒙辩证法》* (1947)
- 马尔库塞, *《单向度的人》* (1964)

**相关词条**：批判理论、马克思主义、社会哲学
