## A-F

### 【19世纪哲学】(19th Century Philosophy)

**定义**：19世纪欧洲哲学，包括德国唯心主义、马克思主义、存在主义先驱等。

**费曼解释**：
19世纪哲学像一场大戏：
- 德国唯心主义：绝对精神
- 马克思主义：历史唯物主义
- 叔本华/尼采：意志与生命

**主要流派**：

| 流派 | 代表人物 | 核心观点 |
|------|----------|----------|
| **德国唯心主义** | 黑格尔 | 绝对精神自我展开 |
| **马克思主义** | 马克思 | 历史唯物主义 |
| **意志哲学** | 叔本华、尼采 | 意志/权力意志 |
| **实证主义** | 孔德 | 科学主义 |
| **实用主义先驱** | 皮尔士 | 实用主义原则 |

**核心主题**：

1. **历史**：历史的意义和规律
2. **自由**：自由与必然的关系
3. **异化**：现代人的异化
4. **意志**：意志作为世界的本质

**相关词条**：黑格尔、马克思、尼采

---

### 【黑格尔】(Hegel)

**定义**：德国哲学家，唯心主义的集大成者。

**费曼解释**：
黑格尔说：历史是"绝对精神"的自我展开。
就像种子长成树，精神在历史中实现自己。

**黑格尔的核心概念**：

1. **绝对精神**：宇宙的本质是精神
2. **辩证法**：正-反-合的运动
3. **历史理性**：历史是理性的展开
4. **主奴辩证法**：承认的斗争

**辩证法**：

```
正 (Thesis) → 反 (Antithesis) → 合 (Synthesis)
  → 新的正 → 新的反 → 新的合 ...
```

**精神现象学**：

| 阶段 | 描述 |
|------|------|
| **意识** | 感性确定性 |
| **自我意识** | 主奴关系 |
| **理性** | 观察、实践 |
| **精神** | 伦理、文化 |
| **绝对精神** | 艺术、宗教、哲学 |

**历史哲学**：

- 历史是理性的
- 历史有目的：自由
- 普鲁士国家是历史的顶点

**代表著作**：
- *《精神现象学》*
- *《逻辑学》*
- *《法哲学原理》*

**相关词条**：辩证法、马克思、德国唯心主义

---

### 【马克思】(Marx)

**定义**：德国哲学家、经济学家，马克思主义创始人。

**费曼解释**：
马克思说：不是意识决定存在，而是存在决定意识。
经济基础决定上层建筑。

**马克思主义的核心概念**：

1. **历史唯物主义**：物质生产决定历史
2. **阶级斗争**：历史是阶级斗争的历史
3. **异化**：资本主义下人的异化
4. **剩余价值**：剥削的秘密

**历史唯物主义**：

```
生产力 → 生产关系 → 上层建筑
  ↓          ↓          ↓
技术      经济制度    政治、法律、意识形态
```

**异化**：

| 类型 | 描述 |
|------|------|
| **与劳动产品异化** | 产品不属于工人 |
| **与劳动过程异化** | 劳动成为强迫 |
| **与类本质异化** | 失去人的本质 |
| **与他人异化** | 人际关系异化 |

**阶级斗争**：

- 资产阶级 vs 无产阶级
- 无产阶级革命
- 共产主义社会

**代表著作**：
- *《资本论》*
- *《共产党宣言》*
- *《德意志意识形态》*

**相关词条**：历史唯物主义、阶级斗争、异化

### 【尼采】(Nietzsche)

**定义**：德国哲学家，存在主义和后现代主义先驱。

**费曼解释**：
尼采宣布："上帝死了！"
不是说他杀死了上帝，而是说现代世界不再相信上帝。
那我们如何生活？创造自己的价值！

**尼采的核心概念**：

1. **上帝死了**：传统价值的崩溃
2. **权力意志**：生命的本质是权力扩张
3. **超人**：创造新价值的人
4. **永恒轮回**：一切都是永恒的重复

**上帝死了**：

- 传统价值（基督教道德）失效
- 虚无主义危机
- 需要重新评估一切价值

**权力意志**：

- 不是"想要权力的意志"
- 而是生命的本质：扩张、创造、超越
- 一切生命都是权力意志

**超人**：

- 不是"优越种族"
- 而是能创造新价值的人
- 超越传统道德，成为自己

**永恒轮回**：

- 想象一切都会无限重复
- 你愿意这样活吗？
- 这是生命的最高肯定

**代表著作**：
- *《查拉图斯特拉如是说》*
- *《善恶的彼岸》*
- *《悲剧的诞生》*

**相关词条**：存在主义、权力意志、虚无主义

## G-Z

### 【叔本华】(Arthur Schopenhauer, 1788-1860)

**定义**：德国哲学家，意志哲学的创始人。

**费曼解释**：
叔本华说：世界的本质是"意志"——盲目的、无理性的追求。
我们都是意志的奴隶，人生是痛苦的。
出路？艺术和禁欲。

**核心思想**：
1. **意志**：世界的本质是意志
2. **表象**：世界是我们的表象
3. **痛苦**：人生是痛苦的
4. **解脱**：通过艺术和禁欲解脱

**意志哲学**：

| 概念 | 描述 |
|------|------|
| **意志** | 世界的本质，盲目的追求 |
| **表象** | 康德的"现象" |
| **理念** | 意志的客观化 |

**人生是痛苦的**：
- 意志是无穷的欲望
- 欲望满足→无聊
- 欲望不满足→痛苦
- 人生在痛苦和无聊之间摇摆

**解脱之路**：

| 路径 | 描述 |
|------|------|
| **艺术** | 暂时摆脱意志 |
| **禁欲** | 否定意志 |
| **同情** | 认识到众生同一 |

**代表著作**：
- *《作为意志和表象的世界》* (1818)

**相关词条**：意志哲学、尼采、悲观主义

### 【克尔凯郭尔】(Søren Kierkegaard, 1813-1855)

**定义**：丹麦哲学家，存在主义的先驱。

**费曼解释**：
克尔凯郭尔说：真理是主观的。
重要的不是"客观真理"，而是"你如何活"。
你面临选择——你必须选择。

**核心思想**：
1. **主观性**：真理是主观的
2. **选择**：你必须选择
3. **焦虑**：选择的焦虑
4. **信仰飞跃**：信仰的飞跃

**人生三阶段**：

| 阶段 | 描述 | 例子 |
|------|------|------|
| **审美** | 追求快乐 | 唐璜 |
| **伦理** | 遵循道德 | 婚姻 |
| **宗教** | 信仰飞跃 | 亚伯拉罕 |

**焦虑概念**：
- 焦虑来自自由
- 在选择面前，我们感到焦虑
- 焦虑是"自由的眩晕"

**亚伯拉罕的困境**：
- 上帝命令杀子
- 伦理说不能杀
- 信仰超越伦理
- "信仰的飞跃"

**代表著作**：
- *《非此即彼》* (1843)
- *《恐惧与战栗》* (1843)

**相关词条**：存在主义、焦虑、信仰

### 【孔德】(Auguste Comte, 1798-1857)

**定义**：法国哲学家，实证主义和社会学的创始人。

**费曼解释**：
孔德说：科学是唯一可靠的知识。
人类经历了三个阶段：神学、形而上学、实证。
现在是实证时代——科学时代。

**核心思想**：
1. **实证主义**：只有科学知识是可靠的
2. **三阶段定律**：人类发展的三阶段
3. **社会学**：创立"社会学"
4. **人道教**：科学宗教

**三阶段定律**：

| 阶段 | 描述 | 特点 |
|------|------|------|
| **神学** | 用神解释一切 | 万物有灵 |
| **形而上学** | 用抽象力解释 | 本质、实体 |
| **实证** | 用科学解释 | 观察、定律 |

**实证主义原则**：
- 只承认可观察的事实
- 寻求规律，不是本质
- 科学是渐进的

**社会学**：
- 孔德创造"社会学"一词
- 社会是可科学研究的对象
- 社会学是"女王科学"

**代表著作**：
- *《实证哲学教程》* (1830-1842)

**相关词条**：实证主义、社会学、科学主义
