## A-F

### 【德性认识论】(Virtue Epistemology)

**定义**：从认知德性角度研究知识和辩护的认识论方法。

**费曼解释**：
传统认识论关注"信念的辩护"（规则）
德性认识论关注"认知者"（人）

就像：
- 传统：关注"这招棋对不对"
- 德性：关注"这个棋手的品格和技能"

**德性认识论的核心主张**：

1. **知识是认知德性的成果**
2. **辩护来自认知德性的发挥**
3. **关注认知者而非信念**
4. **借鉴德性伦理学方法**

**德性认识论的类型**：

| 类型 | 核心主张 | 代表 |
|------|----------|------|
| **可靠主义德性论** | 认知德性=可靠能力 | Sosa |
| **责任德性论** | 认知德性=认知责任 | Zagzebski |
| **激进德性论** | 所有认识概念基于德性 | Baehr |

**认知德性**：

| 德性 | 描述 | 例子 |
|------|------|------|
| **理智勇气** | 敢于质疑传统 | 提出新理论 |
| **理智谦逊** | 承认自己的局限 | 知道自己的无知 |
| **开放心智** | 愿意考虑反例 | 公平对待反对意见 |
| **理智严谨** | 仔细检查推理 | 避免草率结论 |
| **好奇心** | 求知欲 | 主动探索 |

**代表人物**：
- **Ernest Sosa**：德性可靠主义
- **Linda Zagzebski**：德性责任论
- **当代**：德性认识论者

**经典文献**：
- Sosa, *Knowledge in Perspective* (1991)
- Zagzebski, *Virtues of the Mind* (1996)

**当代发展 (2020-2026)**：
- **德性与教育**：认知德性培养
- **德性与AI**：AI的认知德性？
- **集体德性**：群体认知德性
- **德性与偏见**：克服认知偏见

**相关词条**：可靠性理论、德性伦理学、认知责任

### 【认知德性】(Intellectual Virtues)

**定义**：导致真理的认知能力或品格特质。

**费曼解释**：
好的认知者有好的"认知品格"：
- 好奇心：想知道
- 谦逊：知道自己不知道
- 勇气：敢于质疑
- 严谨：仔细思考

就像好人有好的道德品格，好的认知者有好的认知品格。

**认知德性的类型**：

| 类型 | 定义 | 例子 |
|------|------|------|
| **能力德性** | 可靠的认知能力 | 视觉、记忆、推理 |
| **品格德性** | 认知品格特质 | 谦逊、勇气、好奇心 |
| **实践德性** | 实践中的认知习惯 | 批判性思维习惯 |

**主要认知德性**：

**1. 理智性 (Intellectuality)**

| 特质 | 描述 |
|------|------|
| **好奇心** | 求知欲、探索欲 |
| **智识诚信** | 诚实对待证据 |
| **思想开放** | 愿意考虑不同观点 |

**2. 批判性 (Critical Thinking)**

| 特质 | 描述 |
|------|------|
| **智识勇气** | 敢于质疑权威 |
| **智识严谨** | 仔细评估论证 |
| **智识自主** | 独立思考 |

**3. 谦逊性 (Intellectual Humility)**

| 特质 | 描述 |
|------|------|
| **承认无知** | 知道自己不知道 |
| **接受批评** | 愿意被纠正 |
| **认识局限** | 知道自己的局限 |

**认知德性的培养**：

| 方法 | 描述 |
|------|------|
| **教育** | 批判性思维教育 |
| **实践** | 在实践中培养 |
| **榜样** | 向好的认知者学习 |
| **反思** | 反思自己的认知习惯 |

**认知德性与道德德性**：

| 特征 | 认知德性 | 道德德性 |
|------|----------|----------|
| 目标 | 真理 | 善 |
| 例子 | 好奇心、谦逊 | 勇气、诚实 |
| 关系 | 可能重叠 | 可能重叠 |

**代表人物**：
- **Sosa**：能力德性
- **Zagzebski**：品格德性
- **Roberts & Wood**：《智力德性》

**经典文献**：
- Sosa, *A Virtue Epistemology* (2007)
- Zagzebski, *Virtues of the Mind* (1996)
- Roberts & Wood, *Intellectual Virtues* (2007)

**当代发展 (2020-2026)**：
- **测量认知德性**：心理测量
- **认知德性教育**：教育应用
- **AI认知德性**：AI的认知品格
- **集体认知德性**：群体德性

**相关词条**：德性认识论、认知责任

### 【认知责任】(Epistemic Responsibility)

**定义**：认知者在信念形成中应承担的责任。

**费曼解释**：
我们对自己的信念有责任：
- 你应该只相信有证据的东西
- 你应该避免偏见
- 你应该批判性思考

但问题是：信念是自愿的吗？我们能选择相信什么吗？

**认知责任的问题**：

| 问题 | 描述 |
|------|------|
| **信念是自愿的吗？** | 我们能选择相信吗？ |
| **认知义务** | 我们应该相信什么？ |
| **认知责备** | 错误信念应被责备吗？ |

**信念的自愿性**：

| 观点 | 描述 | 问题 |
|------|------|------|
| **直接自愿论** | 可以直接选择相信 | 似乎不对 |
| **间接自愿论** | 可以间接影响信念 | 如何影响？ |
| **非自愿论** | 信念是非自愿的 | 如何有责任？ |

**认知义务**：

**Clifford的义务论**：
> 在证据不足的情况下相信任何事情都是错误的。

**James的回应**：
> 有时我们有权利在证据不足的情况下相信（"信仰意志"）。

**认知责备**：

| 情况 | 应被责备？ |
|------|-----------|
| **故意忽视证据** | 是 |
| **认知懒惰** | 是 |
| **认知偏见** | 可能 |
| **合理错误** | 否 |

**认知责任的标准**：

| 标准 | 描述 |
|------|------|
| **证据主义** | 应该根据证据相信 |
| **实用主义** | 应该考虑实践后果 |
| **德性论** | 应该发挥认知德性 |

**代表人物**：
- **Clifford**：认知义务
- **James**：信仰意志
- **Code**：认知责任
- **当代**：责任德性论

**经典文献**：
- Clifford, "The Ethics of Belief" (1877)
- James, "The Will to Believe" (1896)
- Code, *Epistemic Responsibility* (1987)

**当代发展 (2020-2026)**：
- **认知责任与AI**：AI的认知责任
- **集体认知责任**：群体责任
- **认知责任教育**：培养认知责任
- **认知偏见与责任**：偏见与责任

**相关词条**：信念伦理学、德性认识论

## G-Z

### 【索萨的德性理论】(Sosa's Virtue Theory)

**定义**：Ernest Sosa提出的德性认识论，以认知德性为核心理解知识。

**费曼解释**：
射箭能命中靶心，是因为射手有技能。
信念能为真，是因为认知者有认知德性。
知识就是"因为认知德性而为真的信念"。

**核心要素**：
1. **认知德性**：可靠产生真信念的能力
2. **因果性**：德性因果地导致真信念
3. **层级结构**：知识有三个层次
4. **理智之眼**：对认知能力的元认知

**Sosa的知识三层结构**：

| 层次 | 描述 | 条件 |
|------|------|------|
| **动物知识** | 一阶德性的成功表现 | 信念因德性为真 |
| **反思知识** | 有元认知的知识 | 知道自己知道 |
| **完全知识** | 最高层次 | 反思+安全+适当 |

**Sosa的AAA模型**：

| 维度 | 描述 |
|------|------|
| **准确(Accuracy)** | 信念为真 |
| **熟练(Adroitness)** | 表现体现德性 |
| **适当(Aptness)** | 因为熟练而准确 |

**代表人物**：
- **Ernest Sosa**：德性认识论的创立者

**经典文献**：
- Sosa, *Knowledge in Perspective* (1991)
- Sosa, *A Virtue Epistemology* (2007)

**相关词条**：德性认识论、认知德性、知识

### 【认知成就】(Cognitive Achievement)

**定义**：通过认知能力和努力获得的认识成果。

**费曼解释**：
你解决了一道难题——这是认知成就。
成就不同于运气：成就是能力的体现。

**核心要素**：
1. **能力**：成就依赖认知能力
2. **努力**：成就需要努力
3. **成功**：成就达成目标
4. **因果性**：能力因果地导致成功

**成就 vs 运气**：

| 特征 | 成就 | 运气 |
|------|------|------|
| **能力** | 依赖能力 | 不依赖能力 |
| **控制** | 可控制 | 不可控制 |
| **可重复** | 可重复 | 不可重复 |

**认知成就与知识**：
- 知识是认知成就吗？
- Sosa：知识是认知成就的一种
- 有争议：简单感知知识是成就吗？

**代表人物**：
- **Sosa**：认知成就与知识
- **Pritchard**：认知成就与运气

**相关词条**：德性认识论、认知德性、知识
