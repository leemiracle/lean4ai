### 【意向性】(Intentionality)

**词源**：拉丁语 *intentio* (指向)

**定义**：心理状态的"关于性"——心理状态总是关于某物的特性。

**费曼解释**：
意向性说：
- 你的信念是关于巴黎的
- 你的恐惧是关于蛇的
- 你的希望是关于天气的

心理状态总是"指向"某物。

**布伦塔诺论题**：

**核心主张**：意向性是心理现象的标志。

| 要素 | 描述 |
|------|------|
| **心理标志** | 意向性区分心理和物理 |
| **内在对象** | 心理状态有内在对象 |
| **意向性内存在** | 对象"内在于"心理状态 |

**意向性的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **原始意向性** | 原始的关于性 | 人类思想 |
| **派生意向性** | 派生的关于性 | 文字、符号 |
| **"好像"意向性** | 模拟的关于性 | AI？ |

**意向性 vs 意图**：

| 概念 | 含义 | 例子 |
|------|------|------|
| **意向性** | 关于性 | 信念（关于巴黎） |
| **意图** | 有目的的行动 | 意图去买东西 |

**意向性的特征**：

| 特征 | 描述 |
|------|------|
| **指向性** | 指向对象 |
| **内容** | 有命题内容 |
| **满足条件** | 有真值条件 |
| **可能不存在** | 对象可能不存在 |

**意向性问题**：

| 问题 | 描述 |
|------|------|
| **如何可能？** | 心理状态如何"指向"对象？ |
| **内容确定** | 什么决定心理内容？ |
| **虚假对象** | 虚假对象如何被指向？ |
| **AI意向性** | AI有意向性吗？ |

**代表人物**：
- **Franz Brentano**：意向性理论创始人
- **John Searle**：意向性理论发展
- **Jerry Fodor**：心理表征理论

**经典文献**：
- Brentano, *Psychology from an Empirical Standpoint* (1874)
- Searle, *Intentionality* (1983)
- Fodor, *Psychosemantics* (1987)

**当代发展 (2020-2026)**：
- **外在主义**：内容外在决定
- **诠释主义**：Dennett的意向立场
- **AI意向性**：AI的意向性
- **社会意向性**：集体意向性

**相关词条**：心理内容、外在主义、表征

### 【心理内容】(Mental Content)

**定义**：心理状态所具有的"关于性"内容。

**费曼解释**：
心理内容是：
- 你信念的内容（"巴黎在法国"）
- 你欲望的内容（"想要去巴黎"）
- 心理状态"说什么"

**内容的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **命题内容** | 可真可假 | "天在下雨" |
| **对象内容** | 关于某对象 | 想到巴黎 |
| **概念内容** | 概念性 | 狗的概念 |
| **非概念内容** | 非概念性 | 感知经验 |

**内容问题**：

| 问题 | 描述 |
|------|------|
| **内容确定** | 什么决定内容？ |
| **内容个体化** | 什么使内容不同？ |
| **内容共享** | 不同人能共享内容吗？ |
| **动物内容** | 动物有心理内容吗？ |

**内在主义 vs 外在主义**：

| 问题 | 内在主义 | 外在主义 |
|------|----------|----------|
| 内容位置 | 在头脑内 | 依赖环境 |
| 双胞胎案例 | 内容相同 | 内容不同 |
| 代表 | Fodor | Putnam, Burge |

**相关词条**：意向性、外在主义、语义外在主义

### 【外在主义】(Externalism)

**定义**：认为心理内容由外部环境决定的理论。

**费曼解释**：
外在主义说：
- 你想"水"
- 这个"水"的内容是什么？
- 不是由你头脑决定的
- 而是由你环境中的水决定的

**外在主义的类型**：

| 类型 | 描述 | 代表 |
|------|------|------|
| **语义外在主义** | 意义由环境决定 | Putnam |
| **心理外在主义** | 心理内容由环境决定 | Burge |
| **载体外在主义** | 认知延伸到环境 | Clark & Chalmers |

**Putnam的双胞胎地球**：

**思想实验**：
- 地球上有水（H₂O）
- 双胞胎地球上有"水"（XYZ，外表相同）
- 奥斯卡（地球人）和双胞胎奥斯卡（双胞胎地球人）
- 他们头脑内部完全相同
- 但"水"的意义不同

**结论**：
- 意义不在头脑中
- 环境决定意义
- "意义的含义不是在头脑中"

**Burge的关节炎案例**：

**思想实验**：
- 某人错误地相信"关节炎"影响大腿
- 在我们的社会，这个信念是假的
- 在一个"关节炎"指任何炎症的社会，这个信念是真的
- 因此，信念内容由社会环境决定

**外在主义的影响**：

| 领域 | 影响 |
|------|------|
| **认识论** | 知识可能依赖环境 |
| **心灵哲学** | 心理可能不在头脑中 |
| **语言哲学** | 意义不在头脑中 |
| **自我知识** | 自我知识如何可能？ |

**对外在主义的批评**：

| 批评 | 描述 |
|------|------|
| **自我知识** | 我们如何知道自己的思想？ |
| **狭窄内容** | 是否存在狭窄内容？ |
| **因果问题** | 外在因素如何因果影响？ |

**代表人物**：
- **Hilary Putnam**：语义外在主义
- **Tyler Burge**：心理外在主义
- **Andy Clark**：延伸心灵

**经典文献**：
- Putnam, "The Meaning of 'Meaning'" (1975)
- Burge, "Individualism and the Mental" (1979)
- Clark & Chalmers, "The Extended Mind" (1998)

**相关词条**：心理内容、内在主义、延伸心灵

### 【内在主义】(Internalism)

**定义**：认为心理内容由内在因素决定的理论。

**费曼解释**：
内在主义说：
- 你想"水"
- 这个"水"的内容是什么？
- 由你头脑内部的状态决定
- 不依赖外部环境

**内在主义的类型**：

| 类型 | 描述 | 代表 |
|------|------|------|
| **语义内在主义** | 意义在头脑中 | Fodor |
| **心理内在主义** | 心理内容在头脑中 | 传统观点 |
| **认识内在主义** | 辩护在头脑中 | Chisholm |

**对双胞胎地球的回应**：

| 回应 | 描述 |
|------|------|
| **狭窄内容** | 存在狭窄内容（两个奥斯卡相同） |
| **概念 vs 延伸** | 概念相同，延伸不同 |
| **否认直觉** | 双胞胎地球直觉不可靠 |

**支持内在主义的论证**：

| 论证 | 描述 |
|------|------|
| **自我知识** | 我们直接知道自己的思想 |
| **因果效能** | 内在状态才能因果解释行为 |
| **认知科学** | 认知科学假设内在状态 |

**相关词条**：外在主义、心理内容

### 【双胞胎地球】(Twin Earth)

**定义**：Putnam提出的思想实验，用于支持语义外在主义。

**费曼解释**：
双胞胎地球说：
- 想象一个和地球几乎相同的星球
- 唯一区别："水"是XYZ而不是H₂O
- 奥斯卡和双胞胎奥斯卡头脑内部相同
- 但他们"水"的意义不同

**实验设置**：

| 特征 | 地球 | 双胞胎地球 |
|------|------|------------|
| 水 | H₂O | XYZ |
| 居民 | 奥斯卡 | 双胞胎奥斯卡 |
| 头脑内部 | 相同 | 相同 |
| "水"的意义 | H₂O | XYZ |

**结论**：
- 意义不在头脑中
- 环境决定意义
- 支持外在主义

**哲学意义**：

| 领域 | 意义 |
|------|------|
| **语言哲学** | 意义外在决定 |
| **心灵哲学** | 心理内容外在决定 |
| **认识论** | 知识可能外在依赖 |

**相关词条**：外在主义、Putnam、意义

### 【狭窄内容】(Narrow Content)

**定义**：仅由内在因素决定的心理内容。

**费曼解释**：
狭窄内容说：
- 虽然外在主义可能是对的
- 但还存在"狭窄内容"
- 两个奥斯卡的"水"概念在狭窄意义上相同

**狭窄内容的特征**：

| 特征 | 描述 |
|------|------|
| **内在性** | 仅依赖内在状态 |
| **环境独立** | 不依赖外部环境 |
| **因果相关** | 与行为因果相关 |

**支持狭窄内容的论证**：

| 论证 | 描述 |
|------|------|
| **因果论证** | 狭窄内容解释行为 |
| **自我知识** | 我们知道狭窄内容 |
| **认知科学** | 认知科学需要狭窄内容 |

**对狭窄内容的批评**：

| 批评 | 描述 |
|------|------|
| **空洞** | 狭窄内容可能太抽象 |
| **不存在** | 也许不存在狭窄内容 |
| **作用不明** | 狭窄内容有什么用？ |

**相关词条**：外在主义、内在主义、心理内容

### 【意向立场】(Intentional Stance)

**定义**：Dennett提出的理解系统行为的三种立场之一。

**费曼解释**：
意向立场说：
- 要理解一个系统
- 把它当作有信念和欲望的"理性行动者"
- 预测它的行为

**三种立场**：

| 立场 | 描述 | 例子 |
|------|------|------|
| **物理立场** | 用物理定律预测 | 石头下落 |
| **设计立场** | 用设计预测 | 闹钟响铃 |
| **意向立场** | 用信念/欲望预测 | 人买东西 |

**意向立场的应用**：

| 对象 | 适用性 |
|------|--------|
| 人类 | 非常适用 |
| 动物 | 部分适用 |
| 计算机 | 可能适用 |
| 简单机器 | 不适用 |

**实在论 vs 工具主义**：

| 问题 | 实在论 | 工具主义 |
|------|--------|----------|
| 信念存在吗？ | 真实存在 | 有用工具 |
| 意向性是真实的吗？ | 是 | 只是立场 |

**代表人物**：
- **Daniel Dennett**：意向立场提出者

**经典文献**：
- Dennett, *The Intentional Stance* (1987)

**相关词条**：意向性、Dennett

### 【命题态度】(Propositional Attitude)

**定义**：对命题的心理态度，如相信、希望、恐惧等。

**费曼解释**：
命题态度是：
- "相信天会下雨"（相信）
- "希望天会下雨"（希望）
- "恐惧天会下雨"（恐惧）

同一命题，不同态度。

**命题态度的结构**：

| 要素 | 描述 | 例子 |
|------|------|------|
| **态度** | 心理关系 | 相信 |
| **命题** | 内容 | 天会下雨 |
| **主体** | 持有态度者 | 我 |

**常见的命题态度**：

| 态度类型 | 例子 |
|----------|------|
| **认知** | 相信、知道、怀疑 |
| **情感** | 恐惧、希望、后悔 |
| **意志** | 想要、意图、决定 |

**命题态度的问题**：

| 问题 | 描述 |
|------|------|
| **态度类型** | 有多少种态度？ |
| **态度同一性** | 什么使态度相同？ |
| **态度归赋** | 如何知道他人态度？ |

**相关词条**：信念、意向性

### 【信念】(Belief)

**定义**：认为某命题为真的命题态度。

**费曼解释**：
信念是：
- "我相信巴黎在法国"
- 你认为这个命题是真的
- 这是"相信"的态度

**信念的特征**：

| 特征 | 描述 |
|------|------|
| **命题性** | 信念有命题内容 |
| **真值可评估** | 可真可假 |
| **影响行为** | 信念影响行为 |
| **非自愿** | 不能选择相信 |

**信念的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **显明信念** | 有意识的 | 我相信现在在看书 |
| **隐含信念** | 无意识的 | 隐含的偏见 |
| **基础信念** | 不需要支持 | 我存在 |
| **派生信念** | 基于其他信念 | 基于推理的信念 |

**信念的问题**：

| 问题 | 描述 |
|------|------|
| **信念存在吗？** | 取消主义的挑战 |
| **信念内容** | 什么决定信念内容？ |
| **信念归赋** | 如何知道他人信念？ |

**相关词条**：命题态度、知识、辩护

### 【欲望】(Desire)

**定义**：想要某事发生的心理状态。

**费曼解释**：
欲望是：
- "我想要吃冰淇淋"
- 你希望某事发生
- 这是"想要"的状态

**欲望的特征**：

| 特征 | 描述 |
|------|------|
| **目标导向** | 指向目标 |
| **动机性** | 激发行动 |
| **满足条件** | 有满足条件 |
| **非真值评估** | 不能说真假 |

**欲望 vs 信念**：

| 特征 | 信念 | 欲望 |
|------|------|------|
| 方向 | 相信 → 世界 | 世界 → 欲望 |
| 真值 | 可真可假 | 无真假 |
| 作用 | 描述世界 | 推动行动 |

**欲望的理论**：

| 理论 | 描述 |
|------|------|
| **享乐主义** | 欲望追求快乐 |
| **偏好理论** | 欲望是偏好 |
| **功能主义** | 欲望由功能定义 |

**相关词条**：信念、动机、命题态度

### 【意向对象】(Intentional Object)

**定义**：心理状态所指向的对象。

**费曼解释**：
意向对象是：
- 你想"独角兽"
- 虽然独角兽不存在
- 但"意向对象"存在
- 它是你心理状态指向的东西

**意向对象 vs 真实对象**：

| 概念 | 描述 |
|------|------|
| **意向对象** | 心理状态指向的对象 |
| **真实对象** | 实际存在的对象 |

**问题**：

| 问题 | 描述 |
|------|------|
| **不存在对象** | 如何指向不存在的对象？ |
| **意向对象的本体论** | 意向对象是什么？ |
| **对象确定** | 什么决定意向对象？ |

**相关词条**：意向性、心理内容

### 【意向性内存在】(Inexistence)

**定义**：Brentano的概念，指对象"内在于"心理状态。

**费曼解释**：
意向性内存在说：
- 你想"独角兽"
- 独角兽不存在于现实中
- 但"内在于"你的心理状态
- 这是"意向性内存在"

**Brentano的原意**：
- 对象可以"内在于"心理状态
- 即使对象不存在于现实中
- 这是心理现象的特征

**解释争议**：

| 解释 | 描述 |
|------|------|
| **对象内存在** | 对象确实存在于心理状态中 |
| **内容存在** | 不是对象存在，而是内容存在 |
| **比喻说法** | 这只是比喻说法 |

**相关词条**：意向性、Brentano

### 【满足条件】(Conditions of Satisfaction)

**定义**：Searle的概念，指心理状态被满足的条件。

**费曼解释**：
满足条件是：
- 信念"天在下雨"的满足条件是天确实在下雨
- 欲望"想吃冰淇淋"的满足条件是吃到冰淇淋
- 每个意向状态都有满足条件

**满足条件的类型**：

| 心理状态 | 满足条件 |
|----------|----------|
| 信念 | 命题为真 |
| 欲望 | 目标实现 |
| 意图 | 行为完成 |
| 恐惧 | 危险存在 |

**意义**：
- 满足条件统一了不同心理状态
- 信念和欲望有"相反方向"的满足条件
- 这是意向性的核心

**相关词条**：意向性、Searle

### 【适切方向】(Direction of Fit)

**定义**：心理状态与世界之间适应方向的概念。

**费曼解释**：
适切方向说：
- 信念：让心灵适应世界（心灵→世界）
- 欲望：让世界适应心灵（世界→心灵）

**两种适切方向**：

| 方向 | 描述 | 例子 |
|------|------|------|
| **心灵→世界** | 心灵适应世界 | 信念 |
| **世界→心灵** | 世界适应心灵 | 欲望 |

**Anscombe的例子**：

**购物清单**：
- 购物清单：世界→心灵（希望买这些）
- 收据清单：心灵→世界（记录买了什么）

**相关词条**：信念、欲望

### 【心理表征】(Mental Representation)

**定义**：心理状态通过表征外部世界来具有内容。

**费曼解释**：
心理表征说：
- 你想"猫"
- 你的心理状态"表征"猫
- 就像图片表征事物

**表征理论**：

| 要素 | 描述 |
|------|------|
| **载体** | 表征的物理基础（神经状态） |
| **内容** | 表征所关于的 |
| **解释** | 如何解释表征 |

**表征的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **类似表征** | 类似于对象 | 视觉图像 |
| **符号表征** | 约定关系 | 语言 |
| **结构表征** | 结构相似 | 地图 |

**表征问题**：

| 问题 | 描述 |
|------|------|
| **内容问题** | 什么决定表征内容？ |
| **因果问题** | 表征如何因果相关？ |
| **AI表征** | AI有表征吗？ |

**相关词条**：意向性、心理内容

### 【思想语言】(Language of Thought)

**定义**：Fodor提出的心理表征具有语言结构的假设。

**费曼解释**：
思想语言说：
- 思考不是用英语或中文
- 而是用"思想语言"（Mentalese）
- 大脑中的"语言"

**核心主张**：

| 要素 | 描述 |
|------|------|
| **命题结构** | 思想有命题结构 |
| **组合性** | 思想是组合的 |
| **因果效力** | 思想语言有因果效力 |
| **先天** | 思想语言是先天的 |

**支持论证**：

| 论证 | 描述 |
|------|------|
| **生产性** | 我们能产生无限新思想 |
| **系统性** | 思想是系统的 |
| **推理** | 推理需要结构化表征 |

**对思想语言的批评**：

| 批评 | 描述 |
|------|------|
| **循环论证** | 思想语言预设了它要解释的 |
| **神经不现实** | 与神经科学不兼容 |
| **联结主义** | 联结主义模型不需要 |

**代表人物**：
- **Jerry Fodor**：思想语言假设提出者

**经典文献**：
- Fodor, *The Language of Thought* (1975)

**相关词条**：心理表征、Fodor

---

### 【语义外在主义】(Semantic Externalism)

**定义**：认为意义由外部环境决定的理论。

**费曼解释**：
语义外在主义说：
- "水"这个词的意义
- 不是由你头脑决定的
- 而是由环境中的水决定的

**支持论证**：

| 论证 | 描述 |
|------|------|
| **双胞胎地球** | 环境不同→意义不同 |
| **劳动分工** | 意义依赖专家知识 |
| **指称理论** | 指称由因果链决定 |

**影响**：

| 领域 | 影响 |
|------|------|
| **语言哲学** | 意义外在决定 |
| **心灵哲学** | 心理内容外在决定 |
| **认识论** | 知识可能外在依赖 |

**相关词条**：外在主义、双胞胎地球

---

### 【心理外在主义】(Mental Externalism)

**定义**：认为心理内容由外部环境决定的理论。

**费曼解释**：
心理外在主义说：
- 不只是语言意义外在
- 心理内容也外在
- 你的思想内容由环境决定

**Burge的论证**：

**关节炎案例**：
- 某人错误相信"关节炎"影响大腿
- 在我们的社会，这是假信念
- 在"关节炎"指任何炎症的社会，这是真信念
- 因此，信念内容依赖社会环境

**与语义外在主义的关系**：
- 语义外在主义：意义外在
- 心理外在主义：心理内容外在
- 心理外在主义更强

**相关词条**：外在主义、语义外在主义

---

### 【延伸心灵】(Extended Mind)

**定义**：Clark和Chalmers提出的认知延伸到环境中的理论。

**费曼解释**：
延伸心灵说：
- 认知不只是在大脑中
- 可以延伸到环境中
- 如笔记本可能是你"记忆"的一部分

**Otto案例**：

**情境**：
- Otto有记忆障碍，用笔记本记东西
- 他把"MoMA在53街"记在笔记本
- 这和普通人的记忆有相同功能
- 因此，笔记本是Otto"心灵"的一部分

**Parity原则**：
- 如果外部过程完成同样的功能
- 且内部完成会被认为是认知
- 那外部过程也应被认为是认知

**支持论证**：

| 论证 | 描述 |
|------|------|
| **功能等同** | 外部工具功能等同内部过程 |
| **认知整合** | 外部工具被整合到认知中 |
| **动态交互** | 与外部工具动态交互 |

**对延伸心灵的批评**：

| 批评 | 描述 |
|------|------|
| **认知膨胀** | 把什么都算认知？ |
| **个体边界** | 个体边界在哪里？ |
| **神经依赖** | 认知最终依赖大脑 |

**代表人物**：
- **Andy Clark**：延伸心灵提出者
- **David Chalmers**：共同提出者

**经典文献**：
- Clark & Chalmers, "The Extended Mind" (1998)

**相关词条**：外在主义、4E认知

---

### 【集体意向性】(Collective Intentionality)

**定义**：群体或集体层面上的意向性。

**费曼解释**：
集体意向性说：
- "我们一起做某事"
- 这不只是"我做"和"你做"
- 而是我们作为群体的意向

**Searle的区别**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **我意向** | 个体意向 | 我打算去 |
| **我们意向** | 集体意向 | 我们打算去 |

**集体意向性的特征**：

| 特征 | 描述 |
|------|------|
| **不可还原** | 不能还原为个体意向 |
| **协作性** | 涉及协作 |
| **共享性** | 共享的意向 |

**应用**：

| 领域 | 应用 |
|------|------|
| **社会本体论** | 社会事实的基础 |
| **合作行为** | 合作如何可能 |
| **制度事实** | 制度的意向基础 |

**相关词条**：意向性、Searle

---

### 【背景能力】(Background Capacities)

**定义**：Searle的概念，指支持意向性运作的非意向性能力。

**费曼解释**：
背景能力说：
- 你的意向性依赖"背景"
- 如：要理解"开门"
- 你需要知道门是什么、如何开
- 这些是"背景能力"

**核心主张**：

| 要素 | 描述 |
|------|------|
| **非意向性** | 背景本身不是意向的 |
| **使能** | 使意向性可能 |
| **实践性** | 主要是实践能力 |
| **前概念** | 前概念的能力 |

**背景的类型**：

| 类型 | 描述 |
|------|------|
| **深度背景** | 一般生物能力 |
| **局部背景** | 文化特定能力 |

**相关词条**：意向性、Searle

---

### 【网络】(Network)

**定义**：Searle的概念，指意向状态相互关联的网络。

**费曼解释**：
网络说：
- 一个信念依赖其他信念
- "我想去巴黎"依赖"巴黎存在"、"我可以旅行"等
- 信念形成"网络"

**核心主张**：

| 要素 | 描述 |
|------|------|
| **相互关联** | 意向状态相互关联 |
| **依赖性** | 状态依赖其他状态 |
| **整体性** | 不能孤立理解状态 |

**与背景的区别**：

| 概念 | 描述 |
|------|------|
| **网络** | 意向状态的相互关联 |
| **背景** | 非意向性能力 |

**相关词条**：意向性、Searle

---

### 【内容个体化】(Content Individuation)

**定义**：什么使心理内容成为独特的个体的原则。

**费曼解释**：
内容个体化问：
- 两个信念什么时候是"同一个信念"？
- 什么时候是"不同信念"？
- 什么原则区分内容？

**个体化原则**：

| 原则 | 描述 |
|------|------|
| **精细个体化** | 最小差异→不同内容 |
| **粗糙个体化** | 重大差异→不同内容 |
| **功能个体化** | 功能相同→相同内容 |

**与外在主义的关系**：

| 立场 | 个体化原则 |
|------|------------|
| **外在主义** | 环境差异→内容差异 |
| **内在主义** | 环境相同/不同不影响 |

**相关词条**：心理内容、外在主义

---

### 【存在量词问题】(Problem of Non-Existents）

**定义**：如何解释对不存在对象的思想。

**费曼解释**：
存在量词问题问：
- 我可以想"独角兽"
- 但独角兽不存在
- 我在"想什么"？

**解决方案**：

| 方案 | 描述 |
|------|------|
| **迈农主义** | 不存在对象有某种存在 |
| **罗素理论** | 描述理论，不承诺对象 |
| **可能世界** | 对象存在于可能世界 |
| **取消论** | 问题本身是错误的 |

**相关词条**：意向对象、意向性

---

### 【Frege问题】(Frege's Problem)

**定义**：同一陈述的认知价值问题。

**费曼解释**：
Frege问题问：
- "晨星=暮星"是有信息的
- "晨星=晨星"是没信息的
- 但它们指称相同！
- 认知价值从何而来？

**Frege的解答**：

| 概念 | 描述 |
|------|------|
| **指称 (Bedeutung)** | 词语指称的对象 |
| **意义 (Sinn)** | 词语呈现对象的方式 |

**晨星/暮星**：
- 指称相同：金星
- 意义不同：晨星意义≠暮星意义

**在心灵哲学中的应用**：
- 心理内容可能是"意义"而非"指称"
- 不同信念可以有相同指称

**相关词条**：心理内容、意义

---

### 【内容的因果理论】(Causal Theory of Content)

**定义**：认为心理内容由因果关系决定的理论。

**费曼解释**：
因果理论说：
- "猫"这个概念的内容是什么？
- 它由"猫"这个概念与猫的因果联系决定
- 猫导致你有"猫"的概念

**核心主张**：

| 要素 | 描述 |
|------|------|
| **因果链** | 内容由因果链决定 |
| **协变** | 概念与对象协变 |
| **历史** | 依赖历史因果链 |

**问题**：

| 问题 | 描述 |
|------|------|
| **错误表征** | 如何解释错误表征？ |
| **析取问题** | 多种原因怎么办？ |
| **新表征** | 新概念如何形成？ |

**相关词条**：心理表征、外在主义

---

### 【目的语义学】(Teleosemantics)

**定义**：Millikan和Papineau提出的基于生物功能的语义理论。

**费曼解释**：
目的语义学说：
- 心理表征的内容
- 由它的"生物功能"决定
- 蜜蜂舞蹈的"功能"是什么，决定了它的内容

**核心主张**：

| 要素 | 描述 |
|------|------|
| **生物功能** | 内容由生物功能决定 |
| **进化历史** | 功能由进化历史决定 |
| **适当条件** | 功能有适当运作条件 |

**与因果理论的区别**：

| 理论 | 内容来源 |
|------|----------|
| **因果理论** | 因果历史 |
| **目的语义学** | 生物功能/进化历史 |

**问题**：

| 问题 | 描述 |
|------|------|
| **非生物系统** | 非生物系统如何？ |
| **功能不确定性** | 功能如何确定？ |
| **新表征** | 新功能如何产生？ |

**代表人物**：
- **Ruth Millikan**：目的语义学创始人
- **David Papineau**：目的语义学发展

**经典文献**：
- Millikan, *Language, Thought, and Other Biological Categories* (1984)

**相关词条**：心理表征、生物功能

---

### 【诠释主义】(Interpretivism)

**定义**：Dennett的观点，认为意向性是诠释的产物。

**费曼解释**：
诠释主义说：
- 信念和欲望不是"内在实体"
- 而是我们"诠释"系统的产物
- 我们用意向立场诠释系统

**核心主张**：

| 要素 | 描述 |
|------|------|
| **意向立场** | 用信念/欲望诠释 |
| **工具主义** | 意向性是工具 |
| **实在模式** | 某些模式比其他更真实 |

**与实在论的对立**：

| 问题 | 实在论 | 诠释主义 |
|------|--------|----------|
| 信念存在吗？ | 真实存在 | 诠释产物 |
| 意向性是真实的吗？ | 是 | 是诠释 |

**相关词条**：意向立场、Dennett

---

### 【意向性因果】(Intentional Causation)

**定义**：意向状态作为原因产生效果的因果关系。

**费曼解释**：
意向性因果问：
- 我的"想去巴黎"导致我去巴黎
- 这是因果吗？
- 意向状态如何因果？

**核心问题**：

| 问题 | 描述 |
|------|------|
| **心理因果** | 意向状态有因果力吗？ |
| **因果排除** | 物理原因是否排除意向原因？ |
| **因果描述** | 如何描述意向因果？ |

**相关词条**：心理因果、意向性

---

### 【意向性继承】(Derived Intentionality）

**定义**：非原始意向性的意向性来源。

**费曼解释**：
意向性继承说：
- 文字有意向性（"猫"指猫）
- 但这不是原始意向性
- 它从我们的原始意向性"继承"

**意向性类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **原始意向性** | 原始的 | 人类思想 |
| **派生意向性** | 从原始派生 | 文字、符号 |

**派生机制**：

| 问题 | 描述 |
|------|------|
| **如何派生？** | 意向性如何从原始派生？ |
| **AI意向性** | AI意向性是派生的吗？ |
| **原始来源** | 原始意向性从何而来？ |

**相关词条**：意向性、原始意向性

---

### 【原始意向性】(Original/Intrinsic Intentionality)

**定义**：非派生的、原始的意向性。

**费曼解释**：
原始意向性说：
- 人类思想有原始意向性
- 不是从其他东西派生
- 是意向性的"源头"

**特征**：

| 特征 | 描述 |
|------|------|
| **非派生** | 不是从其他意向性派生 |
| **源头** | 是派生意向性的源头 |
| **内在** | 内在于有意识的存在 |

**谁有原始意向性？**

| 存在 | 有原始意向性？ |
|------|----------------|
| 人类 | 是 |
| 动物 | 有争议 |
| AI | 通常认为没有 |

**相关词条**：意向性、派生意向性
