### 【心理因果】(Mental Causation)

**定义**：心理状态作为原因产生物理或心理效果的现象。

**费曼解释**：
心理因果问：
- 我的"想要举手"导致我举手吗？
- 心理状态真的有因果力吗？
- 还是只是物理过程在起作用？

**核心问题**：

| 问题 | 描述 |
|------|------|
| **心理因果存在吗？** | 心理状态能导致物理事件吗？ |
| **排除问题** | 物理原因是否排除了心理原因？ |
| **过度决定** | 同一效果有两个原因？ |
| **副现象论** | 心理只是副产品？ |

**心理因果的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **心理→物理** | 心理导致物理 | 想举→举手 |
| **心理→心理** | 心理导致心理 | 信念→欲望 |
| **物理→心理** | 物理导致心理 | 刺激→感觉 |
| **物理→物理** | 物理导致物理 | 神经→肌肉 |

**心理因果的重要性**：

| 领域 | 重要性 |
|------|--------|
| **道德责任** | 心理因果是责任前提 |
| **科学解释** | 心理学解释需要因果 |
| **自我理解** | 我们如何理解自己 |
| **行动理论** | 行动由心理状态导致 |

**主要立场**：

| 立场 | 核心主张 | 代表 |
|------|----------|------|
| **还原论** | 心理状态=物理状态 | Smart |
| **非还原论** | 心理状态≠物理状态但有因果力 | Fodor |
| **副现象论** | 心理状态无因果力 | Huxley |
| **取消论** | 心理状态不存在 | Churchlands |

**代表人物**：
- **Jaegwon Kim**：排除论证
- **Jerry Fodor**：非还原心理因果
- **Donald Davidson**：异常一元论

**经典文献**：
- Kim, *Mind in a Physical World* (1998)
- Davidson, "Mental Events" (1970)

**当代发展 (2020-2026)**：
- **涌现因果**：涌现属性的因果力
- **信息因果**：信息作为因果中介
- **神经因果**：神经层面的因果
- **AI因果**：AI的因果能力

**相关词条**：因果封闭、排除论证、副现象论

### 【因果封闭性】(Causal Closure of the Physical)

**定义**：每个物理事件都有充分的物理原因的原则。

**费曼解释**：
因果封闭性说：
- 如果一个台球移动
- 它移动的原因（物理上）是充分的
- 不需要"心灵"来解释

**核心主张**：

| 要素 | 描述 |
|------|------|
| **物理原因充分** | 每个物理事件有物理原因 |
| **不需要超物理** | 不需要非物理原因 |
| **能量守恒** | 符合能量守恒定律 |

**形式表述**：
- 对于每个物理事件P
- 如果P有原因，那么P有充分的物理原因

**支持因果封闭的论证**：

| 论证 | 描述 |
|------|------|
| **科学成功** | 物理学不需要非物理原因 |
| **能量守恒** | 非物理因果会违反能量守恒 |
| **解释充分** | 物理原因解释充分 |

**对二元论的挑战**：

1. 如果物理世界因果封闭
2. 且心理有因果力
3. 那么心理必须是物理的
4. 否则违反封闭性

**二元论的回应**：

| 回应 | 策略 |
|------|------|
| **否认封闭** | 量子不确定性为非物理因果留下空间 |
| **非overdetermination** | 心理和物理原因是同一原因 |
| **非竞争性** | 不同层面的描述 |

**当代发展 (2020-2026)**：
- **量子因果**：量子力学与因果封闭
- **涌现因果**：涌现属性的因果力
- **信息因果**：信息的因果角色

**相关词条**：心理因果、排除论证

### 【排除论证】(Exclusion Argument)

**定义**：Jaegwon Kim提出的论证，认为非还原物理主义无法解释心理因果。

**费曼解释**：
排除论证说：
- 你的手臂移动有物理原因（神经激活）
- 如果心理状态是独立的，它只是"旁观者"
- 因此，要么心理=物理，要么心理无因果

**论证结构**：

1. **因果封闭**：每个物理事件有充分的物理原因
2. **因果排除**：没有事件被两个独立原因过度决定
3. **心理因果**：心理事件有因果力
4. **结论**：心理事件是物理事件

**排除原则**：
- 如果P*是P的充分原因
- 且M不是P*
- 那么M不能是P的原因（除非overdetermination）

**对非还原物理主义的挑战**：

| 问题 | 描述 |
|------|------|
| **独立性** | 如果心理属性独立于物理，因果力在哪？ |
| **副现象论** | 非还原导致副现象论？ |
| **选择** | 要么还原，要么接受副现象论 |

**对排除论证的回应**：

| 回应 | 策略 |
|------|------|
| **否认排除** | 不同层面的因果不排除 |
| **接受副现象论** | 心理无因果，但直觉是错的 |
| **功能解释** | 心理因果是功能因果 |
| **同一策略** | 心理和物理是同一原因 |

**代表人物**：
- **Jaegwon Kim**：排除论证提出者

**经典文献**：
- Kim, *Mind in a Physical World* (1998)

**相关词条**：心理因果、因果封闭、副现象论

### 【副现象论】(Epiphenomenalism)

**词源**：希腊语 *epi* (在...之上) + *phenomenon* (现象)

**定义**：认为心理状态是物理过程的副产品，没有独立的因果力量的理论。

**费曼解释**：
副现象论说：
- 意识像"烟囱的烟"
- 火产生烟，但烟不产生任何东西
- 大脑产生意识，但意识不产生任何东西

**核心主张**：

| 要素 | 描述 |
|------|------|
| **因果无效性** | 心理状态没有因果力量 |
| **物理因果封闭** | 物理世界因果封闭 |
| **副产品地位** | 意识是"副现象" |

**支持副现象论的论证**：

| 论证 | 描述 |
|------|------|
| **因果封闭论证** | 物理世界因果封闭 |
| **进化论证** | 如果意识无因果力，为何进化出意识？ |
| **神经科学证据** | Libet实验：大脑活动先于有意识决定 |

**副现象论的问题**：

| 问题 | 描述 |
|------|------|
| **直觉问题** | 我们感觉意识有因果力 |
| **进化问题** | 意识为何进化？ |
| **知识问题** | 如果意识无因果，我们如何知道它？ |
| **适应性** | 意识只是"乘客"？ |

**知识问题**：

**Huxley的困境**：
- 如果意识无因果力
- 我对你说"我有意识"这句话
- 不是由意识导致的
- 那你怎么知道我有意识？

**进化问题**：

**核心困难**：
- 意识似乎是复杂的
- 复杂特征通常有进化功能
- 但副现象论说意识无功能
- 那意识为何进化？

**副现象论者的回应**：
- 意识可能是其他功能的副产品
- 如：复杂神经处理的副产品
- 不是被选择的特征

**当代发展 (2020-2026)**：
- **新副现象论**：意识的因果局限
- **进化副现象论**：意识的进化解释
- **功能副现象论**：功能解释与意识

**代表人物**：
- **T.H. Huxley**：副现象论创始人
- **Frank Jackson**：知识论证（后转变）
- **Jaegwon Kim**：心理因果研究

**经典文献**：
- Huxley, "On the Hypothesis that Animals are Automata" (1874)
- Jackson, "Epiphenomenal Qualia" (1982)

**相关词条**：心理因果、排除论证

### 【过度决定】(Overdetermination)

**定义**：一个事件有两个充分原因的情况。

**费曼解释**：
过度决定是：
- 一个人被枪击和毒杀同时杀死
- 枪击足以杀死，毒杀也足以杀死
- 死亡被"过度决定"

**在心理因果中的角色**：

| 情境 | 描述 |
|------|------|
| **心理原因** | "想举手"导致举手 |
| **物理原因** | 神经激活导致举手 |
| **两者都充分？** | 如果是，就是过度决定 |

**排除论证中使用**：

| 步骤 | 描述 |
|------|------|
| 1 | 物理原因P*是物理效果P的充分原因 |
| 2 | 如果心理原因M也是P的原因 |
| 3 | 那么P被过度决定 |
| 4 | 但系统过度决定不是普遍现象 |
| 5 | 因此，M不是P的独立原因 |

**对过度决定的回应**：

| 回应 | 策略 |
|------|------|
| **接受** | 也许心理因果就是过度决定 |
| **同一** | 心理和物理是同一原因 |
| **否认独立** | 心理原因不是独立的 |

**相关词条**：排除论证、心理因果

### 【还原论】(Reductionism)

**定义**：认为心理状态可以还原为物理状态的观点。

**费曼解释**：
还原论说：
- 疼痛就是C纤维激活
- 不是"疼痛导致C纤维激活"
- 而是它们是同一回事

**还原的类型**：

| 类型 | 描述 |
|------|------|
| **类型还原** | 心理类型=物理类型 |
| **个例还原** | 心理个例=物理个例 |
| **理论还原** | 心理理论还原为物理理论 |

**支持还原论的论证**：

| 论证 | 描述 |
|------|------|
| **科学统一** | 科学应该统一 |
| **简单性** | 奥卡姆剃刀 |
| **因果问题** | 还原解决因果问题 |

**对还原论的批评**：

| 批评 | 描述 |
|------|------|
| **多重实现** | 心理状态可被多种物理状态实现 |
| **现象意识** | 感质不能还原 |
| **自主性** | 心理学有自主性 |

**代表人物**：
- **J.J.C. Smart**：同一理论
- **David Lewis**：还原论

**经典文献**：
- Smart, "Sensations and Brain Processes" (1959)

**相关词条**：同一理论、物理主义

### 【非还原物理主义】(Non-Reductive Physicalism)

**定义**：认为心理状态是物理的但不能还原为物理状态的观点。

**费曼解释**：
非还原物理主义说：
- 心灵是物理的（依赖物理）
- 但不能还原为物理（不可还原）
- 就像生物不能完全还原为物理

**核心主张**：

| 要素 | 描述 |
|------|------|
| **依赖物理** | 心理依赖物理 |
| **不可还原** | 不能还原为物理 |
| **随附性** | 心理随附于物理 |

**支持论证**：

| 论证 | 描述 |
|------|------|
| **多重实现** | 心理可被多种物理实现 |
| **自主性** | 心理学有自主性 |
| **直觉** | 心理似乎不可还原 |

**问题**：

| 问题 | 描述 |
|------|------|
| **排除论证** | 如何解释心理因果？ |
| **副现象论威胁** | 是否导致副现象论？ |
| **随附性太弱** | 随附性不能保证因果 |

**对排除论证的回应**：

| 回应 | 策略 |
|------|------|
| **不同层面** | 心理和物理是不同层面的描述 |
| **功能因果** | 心理因果是功能因果 |
| **涌现因果** | 涌现属性有因果力 |

**代表人物**：
- **Jerry Fodor**：非还原物理主义
- **Donald Davidson**：异常一元论

**经典文献**：
- Fodor, "Special Sciences" (1974)
- Davidson, "Mental Events" (1970)

**相关词条**：物理主义、排除论证

### 【异常一元论】(Anomalous Monism)

**定义**：Davidson提出的认为心理事件是物理事件，但心理定律不存在的理论。

**费曼解释**：
异常一元论说：
- 每个心理事件都是物理事件（一元论）
- 但没有严格的心理定律（异常）
- 心理描述不遵循严格定律

**核心三原则**：

| 原则 | 描述 |
|------|------|
| **因果交互原则** | 心理事件与物理事件因果交互 |
| **因果法则原则** | 因果关系需要严格定律 |
| **异常原则** | 不存在严格的心理定律 |

**如何一致？**

- 心理事件是物理事件
- 作为物理事件，它们遵循物理定律
- 但作为心理事件，没有严格定律

**对异常一元论的批评**：

| 批评 | 描述 |
|------|------|
| **副现象论威胁** | 心理描述似乎无因果力 |
| **法则性** | 心理科学可能吗？ |
| **解释** | 心理解释的本质是什么？ |

**代表人物**：
- **Donald Davidson**：异常一元论创始人

**经典文献**：
- Davidson, "Mental Events" (1970)

**相关词条**：非还原物理主义、心理因果

### 【实现】(Realization)

**定义**：一个系统或状态被物理基质实现的关系。

**费曼解释**：
实现说：
- 同一软件可以在不同硬件上运行
- 同一心理状态可以在不同大脑中实现
- 这就是"多重实现"

**实现 vs 同一**：

| 关系 | 描述 |
|------|------|
| **同一** | A就是B |
| **实现** | A在物理基质上实现 |

**在心理因果中的角色**：

| 问题 | 描述 |
|------|------|
| **实现关系** | 心理如何被物理实现？ |
| **实现与因果** | 实现关系如何支持因果？ |
| **多重实现** | 多重实现如何影响因果？ |

**相关词条**：多重实现、功能主义

### 【涌现因果】(Emergent Causation)

**定义**：涌现属性的因果力。

**费曼解释**：
涌现因果说：
- 涌现属性（如意识）可能有因果力
- 不仅仅是物理属性的因果
- 涌现属性是"新"的因果来源

**核心主张**：

| 要素 | 描述 |
|------|------|
| **涌现属性** | 涌现属性存在 |
| **因果力** | 涌现属性有因果力 |
| **下向因果** | 涌现属性影响底层 |

**对排除论证的回应**：

- 如果涌现属性有因果力
- 那心理因果是可能的
- 不需要还原

**问题**：

| 问题 | 描述 |
|------|------|
| **涌现定义** | 什么是涌现？ |
| **因果力来源** | 涌现因果力从哪来？ |
| **与封闭兼容** | 与因果封闭兼容吗？ |

**相关词条**：涌现、下向因果

### 【下向因果】(Downward Causation)

**定义**：高层次属性对低层次属性产生因果影响的现象。

**费曼解释**：
下向因果说：
- 不只是原子影响分子
- 分子也可能影响原子
- 整体可以影响部分

**在心理因果中**：

| 层面 | 因果方向 |
|------|----------|
| **上向因果** | 大脑 → 心灵 |
| **下向因果** | 心灵 → 大脑 |

**问题**：

| 问题 | 描述 |
|------|------|
| **与封闭兼容** | 下向因果与因果封闭兼容吗？ |
| **因果力来源** | 下向因果力从哪来？ |
| **循环风险** | 是否导致因果循环？ |

**相关词条**：涌现因果、因果封闭

### 【因果继承原则】(Causal Inheritance Principle)

**定义**：实现属性的因果力继承自实现它的基础属性的因果力。

**费曼解释**：
因果继承说：
- 心理属性被物理属性实现
- 心理属性的因果力来自物理属性
- 是"继承"的因果力

**形式表述**：
- 如果属性M被属性P实现
- 且M导致效果E
- 那是因为P导致E

**问题**：

| 问题 | 描述 |
|------|------|
| **独立性** | M有独立的因果力吗？ |
| **副现象论** | M是否只是旁观者？ |
| **多重实现** | 多重实现如何影响继承？ |

**相关词条**：实现、排除论证

### 【因果排除原则】(Causal Exclusion Principle）

**定义**：没有事件被两个独立原因过度决定的原则。

**费曼解释**：
因果排除说：
- 如果P*是P的充分原因
- 那么没有其他独立原因M可以导致P
- 除非是过度决定

**形式表述**：
- 如果P*是P的充分原因
- 且M ≠ P*
- 且M和P*是独立的
- 那么M不是P的原因

**在排除论证中的角色**：

| 角色 | 描述 |
|------|------|
| **关键前提** | 是排除论证的关键前提 |
| **争议点** | 是主要争议点之一 |

**对排除原则的批评**：

| 批评 | 描述 |
|------|------|
| **不同层面** | 不同层面的因果不排除 |
| **过度决定常见** | 过度决定可能常见 |
| **原则不可靠** | 原则本身不可靠 |

**相关词条**：排除论证、过度决定

### 【同一策略】(Identity Strategy)

**定义**：通过同一心理和物理来解释心理因果的策略。

**费曼解释**：
同一策略说：
- 如果心理状态 = 物理状态
- 那心理因果 = 物理因果
- 问题解决了

**核心主张**：

| 要素 | 描述 |
|------|------|
| **同一** | 心理状态与物理状态同一 |
| **因果同一** | 心理因果与物理因果同一 |
| **避免排除** | 避免排除问题 |

**问题**：

| 问题 | 描述 |
|------|------|
| **多重实现** | 心理可被多种物理实现 |
| **现象意识** | 感质似乎不能同一 |
| **自主性** | 失去心理学自主性 |

**相关词条**：同一理论、还原论

### 【功能因果】(Functional Causation)

**定义**：功能属性通过其因果角色产生效果。

**费曼解释**：
功能因果说：
- 心理状态由其功能角色定义
- 功能角色本身就是因果角色
- 因此心理状态有因果力

**核心主张**：

| 要素 | 描述 |
|------|------|
| **功能角色** | 心理状态由功能角色定义 |
| **因果角色** | 功能角色是因果角色 |
| **因果力** | 因此心理状态有因果力 |

**对排除论证的回应**：
- 心理属性的功能角色就是因果角色
- 不需要与物理原因竞争
- 功能实现不影响因果

**相关词条**：功能主义、心理因果

---

### 【因果描述】(Causal Description)

**定义**：对同一因果关系的不同层面描述。

**费曼解释**：
因果描述说：
- 同一事件可以从不同层面描述
- "他因为愤怒而打人"（心理层面）
- "他因为神经激活而打人"（物理层面）
- 这是同一原因的不同描述

**核心主张**：

| 要素 | 描述 |
|------|------|
| **同一原因** | 心理和物理是同一原因 |
| **不同描述** | 从不同层面描述 |
| **不排除** | 不同描述不相互排除 |

**相关词条**：同一策略、心理因果

---

### 【自律论】(Autonomism)

**定义**：认为心理领域相对自主的观点。

**费曼解释**：
自律论说：
- 心理学有自己的规律
- 不需要还原为物理学
- 心理领域是自主的

**核心主张**：

| 要素 | 描述 |
|------|------|
| **心理自主** | 心理领域自主 |
| **心理定律** | 心理有自己的定律 |
| **不可还原** | 不能还原为物理 |

**问题**：

| 问题 | 描述 |
|------|------|
| **心理因果** | 如何解释心理因果？ |
| **物理基础** | 如何与物理基础协调？ |
| **科学统一** | 如何与科学统一协调？ |

**相关词条**：非还原物理主义、特殊科学

---

### 【特殊科学】(Special Sciences)

**定义**：研究特定领域现象的科学，如心理学、生物学等。

**费曼解释**：
特殊科学说：
- 物理学是基础科学
- 但心理学、生物学等是"特殊科学"
- 它们有自己的定律和自主性

**Fodor的论证**：

1. 特殊科学有自己的定律
2. 这些定律不能还原为物理定律
3. 因此，特殊科学是自主的

**与心理因果的关系**：

| 关系 | 描述 |
|------|------|
| **心理定律** | 心理学有自己的定律 |
| **因果解释** | 心理学提供因果解释 |
| **自主性** | 心理学有自主性 |

**代表人物**：
- **Jerry Fodor**：特殊科学自主性

**经典文献**：
- Fodor, "Special Sciences" (1974)

**相关词条**：自律论、非还原物理主义

---

### 【因果相关 vs 因果充分】(Causally Relevant vs Causally Sufficient)

**定义**：因果相关性与因果充分性的区别。

**费曼解释**：
因果相关vs充分：
- 相关：M与E有因果相关
- 充分：M足以导致E
- 相关不等于充分

**在心理因果中的角色**：

| 问题 | 描述 |
|------|------|
| **心理相关** | 心理状态因果相关？ |
| **物理充分** | 物理状态因果充分？ |
| **区分意义** | 区分有何意义？ |

**Yablo的成比例因果**：

**核心主张**：
- 因果需要"成比例"
- 太细或太粗都不合适
- 心理描述可能比物理描述更"成比例"

**相关词条**：心理因果、排除论证

---

### 【成比例因果】(Proportional Causation)

**定义**：Yablo提出的因果概念，强调因果描述的"成比例性"。

**费曼解释**：
成比例因果说：
- 因果描述要"刚好"
- 不要太细，不要太粗
- 心理描述可能比物理描述更"成比例"

**核心主张**：

| 要素 | 描述 |
|------|------|
| **成比例** | 因果描述要成比例 |
| **不过细** | 不要太细节 |
| **不过粗** | 不要太宽泛 |

**例子**：

| 描述 | 是否成比例 |
|------|------------|
| "鸟飞是因为有翅膀" | ✓ |
| "鸟飞是因为有羽毛细胞" | ✗（太细） |
| "鸟飞是因为是动物" | ✗（太粗） |

**在心理因果中的应用**：
- 心理描述可能比神经描述更"成比例"
- 因此心理因果是真实的

**代表人物**：
- **Stephen Yablo**：成比例因果

**经典文献**：
- Yablo, "Mental Causation" (1992)

**相关词条**：心理因果、因果相关

---

### 【倾向性】(Dispositions)

**定义**：事物在特定条件下表现出某种行为的倾向。

**费曼解释**：
倾向性是：
- 玻璃是易碎的（倾向性）
- 盐是可溶的（倾向性）
- 在适当条件下表现出某种行为

**与心理因果的关系**：

| 关系 | 描述 |
|------|------|
| **心理倾向** | 心理状态可能是倾向性 |
| **因果来源** | 倾向性提供因果来源 |
| **实现** | 倾向性被物理属性实现 |

**倾向性理论**：

| 理论 | 描述 |
|------|------|
| **倾向一元论** | 所有属性都是倾向性 |
| **双重方面** | 属性有范畴和倾向方面 |

**相关词条**：因果力、心理属性

---

### 【实现多元论】(Realization Multiple）

**定义**：同一功能属性可被多种物理属性实现。

**费曼解释**：
实现多元论说：
- 疼痛可以被人类神经实现
- 也可以被章鱼神经实现
- 同一功能，多种实现

**与心理因果的关系**：

| 关系 | 描述 |
|------|------|
| **类型因果** | 心理类型有类型因果？ |
| **实现差异** | 实现差异影响因果？ |
| **因果统一** | 如何保证因果统一？ |

**相关词条**：多重实现、功能主义

---

### 【因果竞争】(Causal Competition）

**定义**：不同层面的原因相互竞争的观点。

**费曼解释**：
因果竞争说：
- 心理原因和物理原因"竞争"
- 只有一个能成为真正原因
- 这导致排除问题

**反对因果竞争的观点**：

| 观点 | 描述 |
|------|------|
| **同一** | 心理和物理是同一原因 |
| **不同层面** | 不同层面不竞争 |
| **功能实现** | 功能因果不与实现因果竞争 |

**相关词条**：排除论证、心理因果

---

### 【实现者因果】(Realizer Causation）

**定义**：实现者的因果，而非被实现属性的因果。

**费曼解释**：
实现者因果说：
- 心理属性M被物理属性P实现
- M的因果实际是P的因果
- 是实现者的因果

**问题**：

| 问题 | 描述 |
|------|------|
| **M有因果吗？** | M有独立的因果吗？ |
| **副现象论** | M是否只是旁观者？ |
| **功能因果** | 功能因果如何解释？ |

**相关词条**：实现、因果继承

### 【属性因果 vs 事件因果】(Property Causation vs Event Causation）

**定义**：属性作为原因与事件作为原因的区别。

**费曼解释**：
属性因果vs事件因果：
- 事件因果：事件1导致事件2
- 属性因果：属性A（作为属性）导致事件

**在心理因果中的角色**：

| **区别意义** | 区别有何意义？ |

**相关词条**：心理因果

### 【因果效力的传递性】(Transitivity of Causal Efficacy）

**定义**：因果效力是否可以传递。

**费曼解释**：
因果效力传递说：
- 如果A导致B，B导致C
- 那么A间接导致C
- 因果效力是传递的

**相关词条**：心理因果

### 【因果基础主义】(Causal Fundamentalism）

**定义**：认为只有基础物理因果是真实的观点。

**费曼解释**：
因果基础主义说：
- 只有物理层面的因果是"真实"的
- 高层面的因果只是"描述"
- 基础物理因果是根本的

**挑战**：

| 挑战 | 描述 |

| **心理因果** | 心理因果的现实性 |

**相关词条**：还原论、物理主义

### 【因果排除与自由意志】(Exclusion and Free Will）

**定义**：因果排除论证对自由意志的挑战。

**费曼解释**：
排除与自由意志说：
- 如果心理状态被物理状态排除
- 那自由意志是否被排除？
- 自由意志如何与排除兼容？

**核心问题**：

| 问题 | 描述 |
|------|------|
| **心理排除** | 心理是否被排除？ |
| **自由排除** | 自由意志是否被排除？ |
| **责任基础** | 道德责任的基础在哪？ |

**回应**：

| 回应 | 策略 |
|------|------|
| **相容论** | 自由意志与决定论兼容 |
| **功能因果** | 自由意志是功能因果 |
| **非排除** | 不接受排除原则 |

**相关词条**：排除论证、自由意志

### 【神经因果】(Neural Causation）

神经因果是：
