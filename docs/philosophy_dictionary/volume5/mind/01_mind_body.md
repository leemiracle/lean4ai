## A-D

### 【心身问题】(Mind-Body Problem)

**定义**：心灵与身体之间关系的哲学问题——心灵是什么？它与物理身体有何关系？

**费曼解释**：
想象你吃了一个苹果：
- 物理过程：咀嚼、消化、化学反应
- 心理过程：感觉甜、想起童年、决定再吃一个

问题是：这两个过程有什么关系？
- 心灵只是大脑的活动吗？
- 还是心灵是某种独立的东西？

**核心问题**：

| 问题 | 描述 |
|------|------|
| **本体论问题** | 心灵是什么类型的实在？ |
| **因果问题** | 心灵如何影响身体？ |
| **知识问题** | 我们如何知道他人的心灵？ |
| **地位问题** | 心灵在自然界中的位置是什么？ |

**主要立场**：

| 立场 | 核心主张 | 代表人物 |
|------|----------|----------|
| **二元论** | 心灵和身体是两种不同的实在 | 笛卡尔 |
| **物理主义** | 心灵是物理过程 | Smart, Armstrong |
| **功能主义** | 心灵是功能角色 | Putnam, Lewis |
| **泛心论** | 意识是根本的、普遍的 | Chalmers |
| **取消主义** | "心灵"是错误的理论概念 | Churchlands |

**历史发展**：

| 时期 | 主要理论 | 人物 |
|------|----------|------|
| **古代** | 灵魂是身体的形式 | 亚里士多德 |
| **近代** | 心身二元论 | 笛卡尔 |
| **20世纪中期** | 行为主义 | Ryle |
| **20世纪后期** | 功能主义 | Putnam |
| **当代** | 物理主义主导 | 当代哲学家 |

**相关词条**：二元论、物理主义、功能主义、意识

### 【二元论】(Dualism)

**词源**：拉丁语 *duo* (二)

**定义**：认为心灵和身体是两种根本不同类型的实在的哲学立场。

**费曼解释**：
二元论说：
- 身体是物质（有重量、占空间、可测量）
- 心灵是非物质（无重量、不占空间、私密）
- 它们是两种完全不同的东西

就像：软件和硬件——它们相关，但本质不同。

**二元论的类型**：

| 类型 | 核心主张 | 代表 |
|------|----------|------|
| **实体二元论** | 心灵和身体是两种实体 | 笛卡尔 |
| **属性二元论** | 同一实体有两种属性 | Jackson |
| **自然主义二元论** | 意识是自然的但不可还原 | Chalmers |

**笛卡尔的实体二元论**：

**核心主张**：
1. 心灵是"思考的实体"（res cogitans）
2. 身体是"广延的实体"（res extensa）
3. 两者相互作用（通过松果体）

**笛卡尔的论证**：

**可设想性论证**：
1. 我可以设想我的心灵没有身体
2. 凡是可设想的都是可能的
3. 因此，心灵可以没有身体
4. 因此，心灵不是身体

**怀疑论证**：
1. 我可以怀疑我的身体存在
2. 但我不能怀疑我在思考
3. 因此，心灵和身体是不同的

**二元论的问题**：

| 问题 | 描述 |
|------|------|
| **交互问题** | 非物质的心灵如何影响物质的身体？ |
| **能量守恒** | 心灵因果会违反能量守恒吗？ |
| **科学问题** | 神经科学似乎不需要二元论 |
| **进化问题** | 非物质心灵如何进化？ |

**交互问题**：

**核心困难**：
- 如果心灵是非物质的
- 它如何"推动"物质？
- 这不违反物理定律吗？

**二元论的回应**：

| 回应 | 策略 | 代表 |
|------|------|------|
| **偶因论** | 上帝协调两者 | 马勒伯朗士 |
| **平行论** | 两者平行但无因果 | 莱布尼茨 |
| **属性论** | 不是两种实体 | 属性二元论 |

**当代发展 (2020-2026)**：
- **自然主义二元论**：Chalmers的"强意识"理论
- **涌现二元论**：心灵作为涌现属性
- **量子意识**：量子力学与意识
- **神经二元论**：神经科学的新证据

**代表人物**：
- **笛卡尔**：实体二元论创始人
- **杰克逊**：属性二元论
- **Chalmers**：自然主义二元论

**经典文献**：
- Descartes, *Meditations on First Philosophy* (1641)
- Jackson, "Epiphenomenal Qualia" (1982)
- Chalmers, *The Conscious Mind* (1996)

**相关词条**：心身问题、物理主义、交互问题

### 【属性二元论】(Property Dualism)

**定义**：认为心灵和身体是同一实体的两种不同属性的理论。

**费曼解释**：
属性二元论说：
- 不是"两个东西"（实体二元论）
- 而是"一个东西的两面"
- 就像一枚硬币有正反两面

**属性二元论 vs 实体二元论**：

| 问题 | 实体二元论 | 属性二元论 |
|------|------------|------------|
| 实体数量 | 两个实体 | 一个实体 |
| 属性类型 | 两种属性 | 两种属性 |
| 独立性 | 心灵可独立存在 | 心灵依赖身体 |

**属性二元论的形式**：

| 类型 | 核心主张 | 代表 |
|------|----------|------|
| **涌现属性论** | 心灵从大脑涌现 | Broad |
| **非还原属性论** | 心灵不可还原但依赖身体 | Jackson |
| **罗尔斯属性论** | 心灵是"罗尔斯"属性 | Lovejoy |

**涌现属性论**：

**核心主张**：
- 当物质组织达到一定复杂度
- 新的属性"涌现"出来
- 涌现属性不可还原为组成部分

**类比**：
- 水分子的属性不能从氢和氧预测
- 意识的属性不能从神经元预测

**杰克逊的知识论证**：

**玛丽思想实验**：
- 玛丽是色彩科学家，知道所有关于颜色的物理事实
- 但她一直生活在黑白房间
- 当她第一次看到红色时，她学到了新东西

**结论**：
- 存在非物理的事实（感质）
- 因此物理主义是错误的

**当代发展 (2020-2026)**：
- **涌现论复兴**：复杂系统与意识
- **非还原物理主义**：依赖但不可还原
- **泛属性论**：所有事物都有心理属性

**相关词条**：二元论、涌现、感质、知识论证

### 【物理主义】(Physicalism)

**词源**：希腊语 *physis* (自然)

**定义**：认为一切都是物理的，心灵可以还原为物理过程的哲学立场。

**费曼解释**：
物理主义说：
- 世界只有物理的东西
- 心灵只是大脑的活动
- 就像"软件只是硬件的状态"

**物理主义的类型**：

| 类型 | 核心主张 | 代表 |
|------|----------|------|
| **还原物理主义** | 心灵可还原为物理 | Smart, Lewis |
| **非还原物理主义** | 心灵依赖但不可还原 | Putnam |
| **取消物理主义** | "心灵"概念应该取消 | Churchlands |
| **先验物理主义** | 物理主义是先验真的 | Jackson (早期) |

**还原物理主义**：

**核心主张**：
- 心理状态 = 大脑状态
- "疼痛" = "C纤维激活"
- 这是类型同一，不是个例同一

**支持论证**：

**因果论证**：
1. 物理世界因果封闭
2. 心理状态有因果作用
3. 因此心理状态是物理的

**科学统一论证**：
1. 科学最终统一为物理学
2. 心理学应该能还原为神经科学
3. 因此心理状态是物理状态

**类型同一理论**：

**核心主张**：疼痛 = C纤维激活（对所有生物）

**问题**：
- 章鱼也能感到疼痛（但没有C纤维）
- 外星人可能有疼痛（但大脑完全不同）

**个例同一理论**：

**核心主张**：我的疼痛 = 我的C纤维激活（只对个体）

**问题**：这放弃了类型同一，似乎不够强

**功能主义批评**：
- 心理状态应该用功能定义
- 不是用物理实现定义
- 同一理论太"硬件导向"

**非还原物理主义**：

**核心主张**：
- 心灵是物理的（依赖物理）
- 但不能还原为物理（不可还原）
- 就像生物不能完全还原为物理

**实现问题**：
- 心理状态可以被不同物理系统实现
- 这支持功能主义，反对同一理论

**当代发展 (2020-2026)**：
- **先验物理主义争论**：Jackson的转变
- **涌现物理主义**：涌现与物理主义
- **量子物理主义**：量子力学与心灵
- **AI物理主义**：AI意识问题

**代表人物**：
- **J.J.C. Smart**：同一理论
- **David Lewis**：分析功能主义
- **Churchlands**：取消主义
- **Frank Jackson**：从二元论转向物理主义

**经典文献**：
- Smart, "Sensations and Brain Processes" (1959)
- Lewis, "An Argument for the Identity Theory" (1966)
- Churchland, *Matter and Consciousness* (1984)

**相关词条**：心身问题、同一理论、还原、取消主义

### 【功能主义】(Functionalism)

**定义**：认为心理状态由其功能角色定义，而非由其物理构成决定的哲学立场。

**费曼解释**：
功能主义说：
- 心灵不是"什么材料做的"
- 而是"做什么的"
- 就像"钟表"不取决于材料，而取决于功能

**核心主张**：

| 要素 | 描述 |
|------|------|
| **功能定义** | 心理状态 = 功能角色 |
| **多重实现** | 同一功能可被不同物理实现 |
| **因果角色** | 心理状态由其因果作用定义 |
| **计算类比** | 心灵类似计算机程序 |

**功能主义的历史**：

| 时期 | 发展 | 人物 |
|------|------|------|
| **1960s** | 机器功能主义 | Putnam |
| **1970s** | 分析功能主义 | Lewis |
| **1980s** | 功能主义批评 | Block, Searle |
| **当代** | 功能主义与AI | 当代哲学家 |

**Putnam的机器功能主义**：

**核心主张**：
- 心灵是图灵机
- 心理状态 = 机器状态
- 心理状态由其因果角色定义

**多重实现论证**：

1. 同一心理状态可被不同物理系统实现
2. 人类、章鱼、外星人都可以有疼痛
3. 因此，心理状态不能等同于特定物理状态
4. 因此，功能主义优于同一理论

**Lewis的分析功能主义**：

**核心主张**：
- 心理状态由"常识心理学"定义
- 普通人如何理解心理概念
- 分析"疼痛"的因果角色

**方法**：
1. 收集常识心理学的概括
2. 用拉姆齐句定义心理概念
3. 填充物理描述

**功能主义的优势**：

| 优势 | 描述 |
|------|------|
| **多重实现** | 解释不同物种的心理共性 |
| **科学兼容** | 与认知科学兼容 |
| **自主性** | 心理学有自主性 |
| **计算模型** | 支持心灵的计算模型 |

**功能主义的问题**：

| 问题 | 描述 | 提出者 |
|------|------|--------|
| **中国脑论证** | 功能主义允许荒谬的意识 | Block |
| **中文屋论证** | 句法不产生语义 | Searle |
| **感质问题** | 功能主义忽视感质 | Jackson |
| **外在主义** | 功能角色可能外在 | Putnam |

**中国脑论证** (Block)：

**思想实验**：
- 整个中国人口模拟大脑
- 每个人用无线电通信
- 这整个系统有意识吗？

**Block的直觉**：
- 功能主义者应该说"有"
- 但这似乎荒谬
- 因此功能主义有问题

**功能主义者的回应**：
- 我们的直觉不可靠
- 也许这样的系统真的有意识？
- 或者需要更复杂的功能条件

**当代发展 (2020-2026)**：
- **计算功能主义**：AI与功能主义
- **生物功能主义**：生物约束
- **4E功能主义**：具身、嵌入、延伸、生成
- **量子功能主义**：量子计算与心灵

**代表人物**：
- **Hilary Putnam**：机器功能主义创始人
- **David Lewis**：分析功能主义
- **Jerry Fodor**：心灵模块性
- **Ned Block**：功能主义批评者

**经典文献**：
- Putnam, "The Nature of Mental States" (1967)
- Lewis, "Psychophysical and Theoretical Identifications" (1970)
- Block, "Troubles with Functionalism" (1978)

**相关词条**：心身问题、多重实现、计算主义、中文屋

### 【同一理论】(Identity Theory)

**定义**：认为心理状态与大脑状态同一的哲学立场。

**费曼解释**：
同一理论说：
- 疼痛就是C纤维激活
- 不是"疼痛导致C纤维激活"
- 而是它们是同一回事

就像：
- 水就是H₂O
- 闪电就是放电
- 疼痛就是神经激活

**同一理论的类型**：

| 类型 | 核心主张 | 问题 |
|------|----------|------|
| **类型同一** | 疼痛 = C纤维激活（类型） | 多重实现问题 |
| **个例同一** | 我的疼痛 = 我的C纤维激活（个例） | 太弱？ |
| **功能同一** | 疼痛 = 疼痛功能（= 某物理实现） | 还是功能主义？ |

**类型同一理论**：

**Smart的论证**：
1. 奥卡姆剃刀：不要假设多余的实体
2. 如果心理和物理相关联，为什么不说是同一的？
3. 科学支持同一性

**类型同一的问题**：

**多重实现问题**：
- 人类有疼痛 = C纤维激活
- 章鱼有疼痛 = ？？？（没有C纤维）
- 外星人可能有完全不同的"疼痛"
- 因此"疼痛 = C纤维激活"是假的

**个例同一理论**：

**核心主张**：
- 我的疼痛 = 我的C纤维激活（个别同一）
- 你的疼痛 = 你的神经激活
- 类型同一不成立，但个例同一成立

**问题**：
- 这太弱了——几乎任何物理主义都接受
- 不能解释心理类型的同一性

**功能同一理论**：

**核心主张**：
- 疼痛 = 疼痛功能（在人类中由C纤维实现）
- 这结合了同一理论和功能主义

**当代发展 (2020-2026)**：
- **神经同一理论**：精细的神经同一
- **动力系统同一**：动力系统与意识
- **量子同一**：量子过程与意识

**代表人物**：
- **J.J.C. Smart**：同一理论创始人
- **David Lewis**：功能同一
- **Ullin Place**：同一理论先驱

**经典文献**：
- Place, "Is Consciousness a Brain Process?" (1956)
- Smart, "Sensations and Brain Processes" (1959)

**相关词条**：物理主义、功能主义、还原

### 【交互问题】(Interaction Problem)

**定义**：如果心灵是非物质的，它如何与物质身体相互作用的问题。

**费曼解释**：
二元论者说：
- 心灵是非物质的
- 身体是物质的
- 但心灵可以"控制"身体

问题是：
- 非物质的东西怎么"推动"物质？
- 这不违反物理定律吗？

**问题的核心**：

| 问题 | 描述 |
|------|------|
| **因果问题** | 非物质如何导致物质变化？ |
| **能量问题** | 心灵因果会违反能量守恒吗？ |
| **位置问题** | 心灵在哪里与身体交互？ |
| **时间问题** | 心灵因果有时间延迟吗？ |

**笛卡尔的解决方案**：
- 心灵通过松果体与身体交互
- 松果体是"灵魂的座位"

**问题**：
- 这只是推迟了问题
- 松果体还是物质的
- 非物质如何影响松果体？

**二元论者的回应**：

| 回应 | 策略 | 问题 |
|------|------|------|
| **偶因论** | 上帝协调两者 | 引入神 |
| **平行论** | 两者平行但无因果 | 否认心理因果 |
| **量子解决** | 利用量子不确定性 | 科学上有争议 |

**偶因论** (Occasionalism)：

**马勒伯朗士的主张**：
- 心灵和身体不直接交互
- 当我想举手时，上帝让我举手
- 表面交互只是"时机"

**问题**：
- 上帝要不断干预
- 违反上帝的自然秩序观

**平行论** (Parallelism)：

**莱布尼茨的主张**：
- 心灵和身体像两个钟表
- 它们同步但不交互
- 上帝预先设定了和谐

**问题**：
- 否认了心理因果
- 我们感觉我们在控制身体

**量子解决方案**：

**核心主张**：
- 量子力学有不确定性
- 心灵可以在量子层面影响大脑
- 不违反能量守恒

**问题**：
- 量子效应在大脑尺度太小
- 量子退相干快速发生
- 科学上有争议

**物理主义者的批评**：
- 交互问题是二元论的致命缺陷
- 物理主义没有这个问题
- 应该放弃二元论

**当代发展 (2020-2026)**：
- **量子意识理论**：Penrose, Hameroff
- **涌现交互**：涌现属性的因果
- **信息因果**：信息作为因果中介

**相关词条**：二元论、心理因果、偶因论、平行论

### 【副现象论】(Epiphenomenalism)

**词源**：希腊语 *epi* (在...之上) + *phenomenon* (现象)

**定义**：认为心理状态是物理过程的副产品，没有独立的因果力量的理论。

**费曼解释**：
副现象论说：
- 意识像"烟囱的烟"
- 火产生烟，但烟不产生任何东西
- 大脑产生意识，但意识不产生任何东西

**核心主张**：

| 要素 | 描述 |
|------|------|
| **因果无效性** | 心理状态没有因果力量 |
| **物理因果封闭** | 物理世界因果封闭 |
| **副产品地位** | 意识是"副现象" |

**支持副现象论的论证**：

**因果封闭论证**：
1. 物理世界因果封闭（每个物理事件有物理原因）
2. 如果心理有因果力，会违反封闭
3. 因此，心理没有因果力

**进化论证**：
1. 如果意识无因果力，为何进化出意识？
2. 这不违反自然选择吗？
3. 也许意识是进化的副产品

**神经科学证据**：
- Libet实验：大脑活动先于有意识决定
- 似乎意识在决策之后产生

**副现象论的问题**：

| 问题 | 描述 |
|------|------|
| **直觉问题** | 我们感觉意识有因果力 |
| **进化问题** | 意识为何进化？ |
| **知识问题** | 如果意识无因果，我们如何知道它？ |
| **适应性 | 意识只是"乘客"？ |

**知识问题**：

**Huxley的困境**：
- 如果意识无因果力
- 我对你说"我有意识"这句话
- 不是由意识导致的
- 那你怎么知道我有意识？

**进化问题**：

**核心困难**：
- 意识似乎是复杂的
- 复杂特征通常有进化功能
- 但副现象论说意识无功能
- 那意识为何进化？

**副现象论者的回应**：
- 意识可能是其他功能的副产品
- 如：复杂神经处理的副产品
- 不是被选择的特征

**杰克逊的转变**：

**早期**：支持属性二元论（知识论证）
**晚期**：转向物理主义，放弃知识论证

**当代发展 (2020-2026)**：
- **新副现象论**：意识的因果局限
- **进化副现象论**：意识的进化解释
- **功能副现象论**：功能解释与意识

**代表人物**：
- **T.H. Huxley**：副现象论创始人
- **Frank Jackson**：知识论证（后转变）
- **Jaegwon Kim**：心理因果研究

**经典文献**：
- Huxley, "On the Hypothesis that Animals are Automata" (1874)
- Jackson, "Epiphenomenal Qualia" (1982)

**相关词条**：心理因果、二元论、知识论证

### 【取消主义】(Eliminativism)

**定义**：认为"信念"、"欲望"等常识心理学概念是错误的理论术语，应该被取消的哲学立场。

**费曼解释**：
取消主义说：
- "信念"、"欲望"像"燃素"、"以太"
- 是错误的理论概念
- 神经科学将取代常识心理学

就像：
- "燃素"被"氧气"取代
- "以太"被"电磁场"取代
- "信念"将被"神经状态"取代

**核心主张**：

| 要素 | 描述 |
|------|------|
| **常识心理学是理论** | 民间心理学是经验理论 |
| **常识心理学是错的** | 这个理论是错误的 |
| **应该取消** | 应该取消其概念 |
| **神经科学替代** | 神经科学将取代它 |

**取消主义 vs 还原主义**：

| 问题 | 还原主义 | 取消主义 |
|------|----------|----------|
| 常识心理学 | 真的（可还原） | 假的（应取消） |
| "信念" | 真实存在 | 不存在 |
| 未来 | 保留概念 | 取消概念 |

**支持取消主义的论证**：

**理论失败论证**：
1. 常识心理学是理论
2. 这个理论在解释精神疾病、创造力等方面失败
3. 失败的理论应该被取代
4. 因此，常识心理学应该被取代

**科学发展论证**：
1. 科学史上有概念被取消（燃素、以太）
2. 常识心理学概念可能类似
3. 神经科学将提供更好的理论

**取消主义要取消什么？**

| 概念 | 取消主义观点 |
|------|--------------|
| **信念** | 不存在，应用神经状态取代 |
| **欲望** | 不存在，应用神经状态取代 |
| **意向** | 不存在，应用神经状态取代 |
| **感质** | 不存在，应用神经状态取代 |

**对取消主义的批评**：

| 批评 | 描述 |
|------|------|
| **自我反驳** | 取消主义者"相信"取消主义吗？ |
| **直觉问题** | 我们确实有信念！ |
| **实践问题** | 取消后如何理解人类行为？ |
| **科学问题** | 神经科学还不够发达 |

**自我反驳问题**：

**论证**：
1. 取消主义说"信念不存在"
2. 但说这句话就是表达一个信念
3. 因此，取消主义自我矛盾

**回应**：
- "信念不存在"是神经状态的描述
- 不是在"表达信念"
- 常识心理学是错误的理论框架

**当代发展 (2020-2026)**：
- **温和取消主义**：部分取消，部分保留
- **神经取消主义**：基于神经科学的取消
- **AI取消主义**：AI视角的取消

**代表人物**：
- **Paul Churchland**：取消主义创始人
- **Patricia Churchland**：神经哲学
- **Stephen Stich**：认知科学取消主义

**经典文献**：
- P. Churchland, "Eliminative Materialism and the Propositional Attitudes" (1981)
- Churchlands, *Matter and Consciousness* (1984)

**相关词条**：物理主义、还原主义、常识心理学

### 【行为主义】(Behaviorism)

**定义**：认为心理状态应该用行为倾向来定义的哲学立场。

**费曼解释**：
行为主义说：
- 不要问"疼痛是什么内在状态"
- 问"疼痛时人会做什么"
- "疼痛" = "呻吟、退缩、避免刺激..."

**行为主义的类型**：

| 类型 | 核心主张 | 代表 |
|------|----------|------|
| **方法论行为主义** | 心理学应该只研究行为 | Watson |
| **逻辑行为主义** | 心理概念 = 行为概念 | Ryle |
| **心理行为主义** | 心理状态 = 行为倾向 | Skinner |

**逻辑行为主义**：

**Ryle的批判**：
- "机器中的幽灵"是错误观念
- 心理概念不是指内在实体
- 而是指行为倾向

**分析**：
- "他相信天会下雨" = "他带了伞"
- 不是指内在状态，而是行为倾向

**支持行为主义的论证**：

| 论证 | 描述 |
|------|------|
| **可观察性** | 行为是可观察的，内心不是 |
| **可验证性** | 行为概念可验证 |
| **避免二元论** | 不需要"机器中的幽灵" |

**对行为主义的批评**：

| 批评 | 描述 | 提出者 |
|------|------|--------|
| **多重可实现** | 同一行为可由不同心理导致 | Putnam |
| **倒置感质** | 行为相同但感质不同 | 倒置光谱 |
| **假装** | 可以假装有心理状态 | 假装疼痛 |
| **循环问题** | 行为定义依赖心理概念 | 循环 |

**倒置光谱问题**：

**思想实验**：
- 你看红色，我看绿色
- 但我们行为完全相同
- 行为主义无法区分

**假装问题**：

**思想实验**：
- 演员假装疼痛
- 行为和真疼痛一样
- 但心理状态不同

**行为主义的衰落**：
- 1950s-60s：认知革命
- 心理过程被重新重视
- 行为主义被功能主义取代

**当代发展 (2020-2026)**：
- **新行为主义**：与认知科学结合
- **外显行为主义**：强调外显行为
- **应用行为分析**：临床应用

**代表人物**：
- **John B. Watson**：行为主义创始人
- **B.F. Skinner**：心理行为主义
- **Gilbert Ryle**：《心的概念》

**经典文献**：
- Watson, "Psychology as the Behaviorist Views It" (1913)
- Ryle, *The Concept of Mind* (1949)
- Skinner, *Verbal Behavior* (1957)

**相关词条**：功能主义、心理状态、行为

### 【多重实现】(Multiple Realization)

**定义**：同一心理状态可以被不同物理系统实现的现象。

**费曼解释**：
多重实现说：
- "疼痛"不是特定的神经状态
- 人类、章鱼、外星人都可以有疼痛
- 但它们的神经完全不同
- 因此，疼痛是"多重可实现的"

就像：
- "钟表"可以被发条、电池、水力实现
- "钟表"不等于某种特定机制

**多重实现论证**：

**Putnam的论证**：
1. 人类有疼痛（C纤维激活）
2. 章鱼有疼痛（但没有C纤维）
3. 外星人可能有疼痛（完全不同的大脑）
4. 因此，疼痛 ≠ C纤维激活
5. 因此，同一理论是错的
6. 因此，功能主义更好

**论证结构**：

```
前提1：疼痛是多重重实现的
前提2：如果M是多重可实现的，那么M ≠ 任何物理类型
结论：因此，疼痛 ≠ 任何物理类型
```

**多重实现的例子**：

| 心理状态 | 人类实现 | 其他实现 |
|----------|----------|----------|
| **疼痛** | C纤维激活 | 章鱼的神经 |
| **视觉** | 视觉皮层 | 章鱼的相机眼 |
| **学习** | 人脑 | AI神经网络 |
| **记忆** | 海马体 | 计算机存储 |

**对多重实现论证的批评**：

| 批评 | 描述 | 回应 |
|------|------|------|
| **人限制** | 我们只关心人类 | 但理论上可能 |
| **同一限制** | 可以用"异类同一" | 失去简单性 |
| **科学实践** | 神经科学似乎有类型同一 | 还在研究中 |

**异类同一**：

**核心主张**：
- 疼痛 = (人类：C纤维激活) ∨ (章鱼：...) ∨ (外星人：...)
- 这是析取的物理类型
- 仍然是物理主义

**功能主义的反驳**：
- 这种析取类型太人为
- 失去了科学的统一性
- 功能类型更自然

**当代发展 (2020-2026)**：
- **AI多重实现**：AI作为心理的多重实现
- **生物限制**：生物对实现的限制
- **计算多重实现**：计算的多重实现

**相关词条**：功能主义、同一理论、物理主义

### 【涌现】(Emergence)

**定义**：复杂系统具有其组成部分所不具备的属性的现象。

**费曼解释**：
涌现说：
- 水是湿润的，但氢和氧不是

**涌现的类型**：

### 【泛心论】(Panpsychism)

**词源**：希腊语 *pan* (全) + *psyche* (灵魂)

**定义**：认为意识是宇宙的根本特征，存在于所有事物中的哲学立场。

**费曼解释**：
泛心论说：
- 不是只有复杂的脑才有意识
- 意识是基本的，像质量、电荷一样
- 电子、原子都有某种"原始意识"

**核心主张**：

| 要素 | 描述 |
|------|------|
| **普遍性** | 意识存在于所有事物 |
| **根本性** | 意识是根本特征 |
| **组合问题** | 微观意识如何组合成宏观意识 |

**泛心论的类型**：

| 类型 | 核心主张 | 代表 |
|------|----------|------|
| **构成泛心论** | 所有事物都有意识 | Strawson |
| **泛原心论** | 所有事物都有意识基础 | Chalmers |
| **宇宙泛心论** | 宇宙本身有意识 | Mathews |

**支持泛心论的论证**：

**组合论证**：
1. 物理主义不能解释意识
2. 二元论有交互问题
3. 泛心论避免了这些问题
4. 因此，泛心论是最好的选择

**罗素的论证**：
1. 物理学只描述结构
2. 物质的内在本质未知
3. 意识可能是物质的内在本质

**组合问题**：

**核心困难**：
- 如果电子有"微意识"
- 它们如何组合成人类意识？
- 这是泛心论的主要问题

**当代发展 (2020-2026)**：
- **意识组合理论**：解决组合问题
- **量子泛心论**：量子力学与泛心论
- **信息泛心论**：信息与意识

**代表人物**：
- **Thomas Nagel**："做一只蝙蝠是什么感觉"
- **David Chalmers**：自然主义二元论
- **Galen Strawson**：实在的一元论

**经典文献**：
- Nagel, "What Is It Like to Be a Bat?" (1974)
- Chalmers, *The Conscious Mind* (1996)
- Strawson, "Realistic Monism" (2006)

**相关词条**：意识、困难问题、物质内在性

### 【机器状态功能主义】(Machine State Functionalism)

**定义**：Putnam提出的用图灵机状态来定义心理状态的功能主义版本。

**费曼解释**：
机器状态功能主义说：
- 心灵是图灵机
- 心理状态 = 机器状态
- 心理状态由其因果角色定义

**核心主张**：

| 要素 | 描述 |
|------|------|
| **图灵机类比** | 心灵像图灵机 |
| **机器状态** | 心理状态 = 机器状态 |
| **因果角色** | 由输入-输出-状态关系定义 |

**Putnam的论证**：

1. 心理状态由其因果角色定义
2. 图灵机状态由其因果角色定义
3. 因此，心理状态可以定义为机器状态

**形式描述**：

对于一个图灵机M，心理状态S定义为：
- 给定输入I和前一状态S'，转移到状态S
- 给定当前状态S和输入I，产生输出O

**问题**：

| 问题 | 描述 |
|------|------|
| **太抽象** | 忽略了生物细节 |
| **太弱** | 任何系统都可能是图灵机？ |
| **感质问题** | 无法解释感质 |

**相关词条**：功能主义、图灵机、计算主义

### 【分析功能主义】(Analytic Functionalism)

**定义**：Lewis提出的通过分析常识心理学来定义心理状态的功能主义版本。

**费曼解释**：
分析功能主义说：
- 不要假设心灵是图灵机
- 而是分析普通人如何理解心理概念
- 用"常识心理学"定义心理状态

**方法**：

1. **收集常识概括**：普通人如何描述心理关系
2. **构建拉姆齐句**：用逻辑形式化
3. **填充物理描述**：用物理实现替换

**拉姆齐句方法**：

**步骤1**：收集常识概括
- 信念 + 欲望 → 行动
- 疼痛 → 避免行为

**步骤2**：构建拉姆齐句
- ∃X∃Y (X + Y → 行动 ∧ ...)

**步骤3**：填充物理描述
- X = 神经状态N₁
- Y = 神经状态N₂

**优势与问题**：

| 优势 | 问题 |
|------|------|
| 基于常识理解 | 常识可能是错的 |
| 不预设计算主义 | 常识不一致 |
| 灵活 | 难以形式化 |

**相关词条**：功能主义、拉姆齐句、常识心理学

**意义**：
- 区分真正的变化和"罗尔斯变化"
- 心理属性可能是罗尔斯属性？

### 【内格尔论题】(Nagel's Thesis)

**定义**：Thomas Nagel关于意识主观性的核心论题——存在"做...是什么感觉"。

**费曼解释**：
内格尔说：
- 做一只蝙蝠是什么感觉？
- 我们永远无法知道
- 因为意识有"主观性"

**核心论点**：

1. 意识存在"主观性"
2. 主观性无法被客观科学完全捕捉
3. 因此，物理主义可能是不完整的

**蝙蝠论证**：

**前提**：
- 蝙蝠用回声定位感知世界
- 这是我们无法想象的感知方式
- 我们无法知道"做一只蝙蝠是什么感觉"

**结论**：
- 存在主观事实
- 这些事实无法从客观角度把握
- 物理主义（客观科学）无法捕捉所有事实

**相关词条**：意识、主观性、困难问题

### 【知识论证】(Knowledge Argument)

**定义**：Jackson提出反对物理主义的论证——玛丽可以从物理事实中学到新东西。

**费曼解释**：
玛丽思想实验：
- 玛丽是色彩科学家
- 知道所有关于颜色的物理事实
- 但她住在黑白房间
- 当她第一次看到红色...

**她学到了新东西吗？**
- 如果是，那存在非物理事实
- 因此物理主义是错的

**论证结构**：

1. 玛丽知道所有物理事实
2. 玛丽不知道"看到红色是什么感觉"
3. 因此，"看到红色是什么感觉"不是物理事实
4. 因此，物理主义是错的

**物理主义的回应**：

| 回应 | 描述 |
|------|------|
| **能力假说** | 玛丽获得的是能力，不是知识 |
| **亲知知识** | 玛丽获得的是亲知，不是命题知识 |
| **现象概念** | 玛丽获得新概念，不是新事实 |

**杰克逊的转变**：
- 早期：知识论证支持二元论
- 晚期：接受物理主义，放弃知识论证

**当代发展 (2020-2026)**：
- **新知识论证**：改进版论证
- **现象概念策略**：物理主义回应
- **AI知识论证**：AI能学到感质吗？

**相关词条**：感质、物理主义、现象概念

### 【现象概念】(Phenomenal Concepts)

**定义**：我们用来指称意识经验的特殊概念。

**费曼解释**：
现象概念说：
- 我们有两种方式指称疼痛
- 物理概念："C纤维激活"
- 现象概念："这种痛感！"
- 它们指称同一事物，但概念不同

**现象概念的特性**：

| 特性 | 描述 |
|------|------|
| **亲知要求** | 必须亲身体验才能获得 |
| **索引性** | "这种体验"（指向性） |
| **不可传达** | 无法用语言完全传达 |
| **直接指称** | 直接指称经验本身 |

**现象概念策略**：

**核心主张**：
- 知识论证的问题在于混淆概念和指称
- 玛丽获得了新概念（现象概念）
- 但没有新事实（仍指称同一物理状态）

**支持物理主义**：
- 解释了知识论证的直觉
- 但不需要接受二元论
- 因为没有新事实，只有新概念

**对现象概念策略的批评**：

| 批评 | 描述 |
|------|------|
| **概念差距问题** | 为什么有概念差距？ |
| **解释问题** | 没有真正解释意识 |
| **空洞** | 似乎只是在重复问题 |

**相关词条**：知识论证、感质、物理主义

### 【先天物理主义】(A Priori Physicalism)

**定义**：认为所有真理（包括心理真理）可以先天从物理真理推导出来的物理主义版本。

**费曼解释**：
先天物理主义说：
- 如果你知道所有物理事实
- 你可以（在原则上）推导出所有心理事实
- 不需要后天观察

**核心主张**：

| 要素 | 描述 |
|------|------|
| **可推导性** | 心理真理可从物理真理推导 |
| **先天** | 推导是先天的（概念上必然） |
| **概念分析** | 心理概念可分析为物理概念 |

**vs 后天物理主义**：

| 问题 | 先天物理主义 | 后天物理主义 |
|------|--------------|--------------|
| 可推导性 | 可先天推导 | 需要后天发现 |
| 水=H₂O | 先天真理 | 后天必然 |
| 心理-物理 | 可分析 | 不可分析 |

**问题**：
- 概念分析似乎失败
- 心理概念不能分析为物理概念
- 知识论证挑战先天物理主义

**相关词条**：物理主义、后天必然性

### 【后天物理主义】(A Posteriori Physicalism)

**定义**：认为心理-物理同一需要后天发现的物理主义版本。

**费曼解释**：
后天物理主义说：
- "疼痛 = C纤维激活"像"水 = H₂O"
- 不能从概念分析得出
- 需要经验发现

**核心主张**：

| 要素 | 描述 |
|------|------|
| **后天同一** | 心理-物理同一需要经验发现 |
| **必然性** | 同一是必然的（一旦发现） |
| **概念独立性** | 心理概念 ≠ 物理概念 |

**与Kripke**：

- Kripke：后天必然性存在
- 水 = H₂O是必然的，但后天发现
- 心理-物理同一也应该是这样

**问题**：
- Kripke自己反对心理-物理同一
- 因为"疼痛"和"C纤维激活"的模态性质不同
- 疼痛必然是疼痛，C纤维激活不必然是疼痛

**相关词条**：物理主义、Kripke、必然性

### 【异常一元论】(Anomalous Monism)

**定义**：Davidson提出的认为心理事件是物理事件，但心理定律不存在的理论。

**费曼解释**：
异常一元论说：
- 每个心理事件都是物理事件（一元论）
- 但没有严格的心理定律（异常）
- 心理描述不遵循严格定律

**核心三原则**：

| 原则 | 描述 |
|------|------|
| **因果交互原则** | 心理事件与物理事件因果交互 |
| **因果法则原则** | 因果关系需要严格定律 |
| **异常原则** | 不存在严格的心理定律 |

**如何一致？**

- 心理事件是物理事件
- 作为物理事件，它们遵循物理定律
- 但作为心理事件，没有严格定律

**问题**：

| 问题 | 描述 |
|------|------|
| **副现象论威胁** | 心理描述似乎无因果力 |
| **法则性** | 心理科学可能吗？ |
| **解释** | 心理解释的本质是什么？ |

**代表人物**：
- **Donald Davidson**：异常一元论创始人

**经典文献**：
- Davidson, "Mental Events" (1970)

**相关词条**：心理因果、物理主义

### 【心身同一论题】(Mind-Body Identity Thesis)

**定义**：认为心理状态与物理状态同一的核心论题。

**费曼解释**：
心身同一论题说：
- 疼痛 = C纤维激活
- 不是"疼痛导致C纤维激活"
- 而是"它们是同一回事"

**与Kripke的挑战**：

Kripke论证：
1. 同一陈述如果是真的，就是必然真的
2. "疼痛 = C纤维激活"如果是真的，就是必然真的
3. 但"疼痛"和"C纤维激活"的模态性质不同
4. 疼痛必然是疼痛，C纤维激活不必然是疼痛
5. 因此，"疼痛 = C纤维激活"是假的

**物理主义者的回应**：

| 回应 | 策略 |
|------|------|
| **现象概念策略** | 概念不同，指称相同 |
| **后天必然** | 同一是后天必然 |
| **否认前提** | 模态直觉不可靠 |

**相关词条**：同一理论、物理主义、Kripke

### 【机器中的幽灵】(Ghost in the Machine)

**定义**：Ryle对笛卡尔二元论的批判性称呼。

**费曼解释**：
Ryle说：
- 笛卡尔把心灵看作"机器中的幽灵"
- 这是"范畴错误"
- 就像参观牛津后问"大学在哪里"（它不在任何具体建筑里）

**范畴错误**：

**定义**：把某类概念错误地归入另一类范畴。

**例子**：
- "大学"不是另一类建筑，而是建筑的集合
- "心灵"不是另一类实体，而是行为倾向的集合

**Ryle的批判**：

| 二元论观点 | Ryle的批判 |
|------------|------------|
| 心灵是内在实体 | 这是范畴错误 |
| 心灵与身体交互 | 交互是错误的比喻 |
| 私密心理状态 | 心理概念是公开的 |

**相关词条**：行为主义、二元论、Ryle

### 【因果封闭性】(Causal Closure of the Physical)

**定义**：每个物理事件都有充分的物理原因的原则。

**费曼解释**：
因果封闭性说：
- 如果一个台球移动
- 它移动的原因（物理上）是充分的
- 不需要"心灵"来解释

**核心主张**：

| 要素 | 描述 |
|------|------|
| **物理原因充分** | 每个物理事件有物理原因 |
| **不需要超物理** | 不需要非物理原因 |
| **能量守恒** | 符合能量守恒定律 |

**支持因果封闭的论证**：

| 论证 | 描述 |
|------|------|
| **科学成功** | 物理学不需要非物理原因 |
| **能量守恒** | 非物理因果会违反能量守恒 |
| **解释充分** | 物理原因解释充分 |

**对二元论的挑战**：

1. 如果物理世界因果封闭
2. 且心理有因果力
3. 那么心理必须是物理的
4. 否则违反封闭性

**二元论的回应**：

| 回应 | 策略 |
|------|------|
| **否认封闭** | 量子不确定性为非物理因果留下空间 |
| **非 overdetermination** | 心理和物理原因是同一原因 |
| **非竞争性** | 不同层面的描述 |

**当代发展 (2020-2026)**：
- **量子因果**：量子力学与因果封闭
- **涌现因果**：涌现属性的因果力
- **信息因果**：信息的因果角色

**相关词条**：心理因果、物理主义、二元论

### 【本体论依赖】(Ontological Dependence)

**定义**：一个实体的存在依赖于另一个实体的关系。

**费曼解释**：
本体论依赖说：
- 微笑依赖于脸

**在心身问题中的应用**：

| **涌现依赖** | 心灵从大脑涌现 |

**依赖 vs 同一**：

### 【解释鸿沟】(Explanatory Gap)

**定义**：物理事实与意识经验之间的解释鸿沟。

**费曼解释**：
解释鸿沟说：
- 我们可以解释水 = H₂O
- 但无法解释"为什么C纤维激活是疼痛的感觉"
- 物理事实和意识之间有"鸿沟"

**核心问题**：

| 问题 | 描述 |
|------|------|
| **为什么** | 为什么某些物理过程有意识？ |
| **必然性** | 为什么是必然的？（如果是） |
| **解释** | 物理解释能解释意识吗？ |

**与知识论证的区别**：

| 论证 | 焦点 |
|------|------|
| **知识论证** | 认识论：玛丽学到新东西 |
| **解释鸿沟** | 解释：物理不能解释意识 |

**物理主义的回应**：

| 回应 | 策略 |
|------|------|
| **鸿沟是认知的** | 我们的认知限制，不是本体论鸿沟 |
| **现象概念** | 概念鸿沟，不是事实鸿沟 |
| **接受鸿沟** | 但否认本体论后果 |

**代表人物**：
- **Joseph Levine**：解释鸿沟提出者

**经典文献**：
- Levine, "Materialism and Qualia: The Explanatory Gap" (1983)

**相关词条**：困难问题、知识论证、物理主义

### 【困难问题】(Hard Problem of Consciousness)

**定义**：Chalmers提出的关于为什么物理过程会有主观体验的问题。

**费曼解释**：
困难问题问：
- 为什么我们不是"哲学僵尸"？
- 为什么有"感觉"？
- 物理过程为什么会有"光亮"？

**简单问题 vs 困难问题**：

| 类型 | 问题 | 可解决性 |
|------|------|----------|
| **简单问题** | 感知、注意、记忆如何工作 | 科学可解 |
| **困难问题** | 为什么有主观体验 | 可能无法用现有科学解决 |

**困难问题的核心**：

**为什么和如何**：
- 我们可以解释大脑如何处理信息
- 但无法解释为什么这伴随着意识
- "为什么有主观性？"

**哲学僵尸**：

**定义**：行为与正常人相同但没有意识的存在。

**可能性**：
- 如果哲学僵尸可能存在
- 那意识不是物理过程的必然结果
- 这挑战物理主义

**Chalmers的立场**：

- 困难问题需要新理论
- 意识可能是根本的（像质量、电荷）
- 支持自然主义二元论或泛心论

**物理主义的回应**：

| 回应 | 策略 |
|------|------|
| **否认困难问题** | 困难问题是幻觉 |
| **现象概念** | 概念鸿沟，不是本体论鸿沟 |
| **未来科学** | 科学将解决困难问题 |

**当代发展 (2020-2026)**：
- **新困难问题**：更精细的表述
- **AI困难问题**：AI有意识吗？
- **神经困难问题**：神经科学的视角

**代表人物**：
- **David Chalmers**：困难问题提出者

**经典文献**：
- Chalmers, "Facing Up to the Problem of Consciousness" (1995)
- Chalmers, *The Conscious Mind* (1996)

**相关词条**：意识、感质、解释鸿沟

### 【心理因果】(Mental Causation)

**定义**：心理状态作为原因产生物理或心理效果的现象。

**费曼解释**：
心理因果问：
- 我的"想要举手"导致我举手吗？
- 心理状态真的有因果力吗？
- 还是只是物理过程在起作用？

**核心问题**：

| 问题 | 描述 |
|------|------|
| **心理因果存在** | 心理状态能导致物理事件吗？ |
| **排除问题** | 物理原因是否排除了心理原因？ |
| **过度决定** | 同一效果有两个原因？ |

**排除论证** (Kim)：

1. 物理世界因果封闭
2. 心理状态 M 导致物理效果 P
3. 但 P 已经有充分的物理原因 P*
4. 如果 M ≠ P*，则 P 被过度决定
5. 过度决定不是普遍现象
6. 因此，M = P*（心理状态是物理状态）

**对物理主义的挑战**：
- 如果心理状态是物理状态，那有因果力
- 但如果心理状态是独立属性，那因果力有问题

**当代发展 (2020-2026)**：
- **神经因果**：神经层面的因果
- **涌现因果**：涌现属性的因果力
- **信息因果**：信息的因果角色

**相关词条**：因果封闭、排除问题、副现象论

---

### 【排除论证】(Exclusion Argument)

**定义**：Jaegwon Kim提出的论证，认为非还原物理主义无法解释心理因果。

**费曼解释**：
排除论证说：
- 你的手臂移动有物理原因（神经激活）
- 如果心理状态是独立的，它只是"旁观者"
- 因此，要么心理=物理，要么心理无因果

**论证结构**：

1. **因果封闭**：每个物理事件有充分的物理原因
2. **因果排除**：没有事件被两个独立原因过度决定
3. **心理因果**：心理事件有因果力
4. **结论**：心理事件是物理事件

**对非还原物理主义的挑战**：

| 问题 | 描述 |
|------|------|
| **独立性** | 如果心理属性独立于物理，因果力在哪？ |
| **副现象论** | 非还原导致副现象论？ |
| **选择** | 要么还原，要么接受副现象论 |

**回应**：

| 回应 | 策略 |
|------|------|
| **否认排除** | 不同层面的因果不排除 |
| **接受副现象论** | 心理无因果，但直觉是错的 |
| **功能解释** | 心理因果是功能因果 |

**代表人物**：
- **Jaegwon Kim**：排除论证提出者

**经典文献**：
- Kim, *Mind in a Physical World* (1998)

**相关词条**：心理因果、因果封闭、非还原物理主义

---

### 【拉姆齐句】(Ramsey Sentence)

**定义**：Lewis用于形式化功能主义的方法。

**费曼解释**：
拉姆齐句方法：
- 用存在量词替换理论术语
- 保留理论的结构关系
- 用于定义心理状态

**例子**：

原始理论：信念 B + 欲望 D → 行为 A

拉姆齐句：∃X∃Y (X + Y → A)

含义：存在某些状态 X 和 Y，当它们同时出现时，导致行为 A

**在功能主义中的应用**：

1. 收集常识心理学的概括
2. 用拉姆齐句形式化
3. 用物理实现填充变量

**相关词条**：分析功能主义、功能主义

---

### 【类型同一 vs 个例同一】(Type vs Token Identity)

**定义**：两种同一性概念——类型同一指类别同一，个例同一指具体实例同一。

**费曼解释**：
- **类型同一**：所有疼痛 = 所有C纤维激活
- **个例同一**：我的这次疼痛 = 我这次C纤维激活

**比较**：

| 问题 | 类型同一 | 个例同一 |
|------|----------|----------|
| 范围 | 所有实例 | 单个实例 |
| 强度 | 更强 | 更弱 |
| 多重实现问题 | 面临挑战 | 避免 |

**在心身问题中**：

| 理论 | 立场 |
|------|------|
| **类型同一理论** | 疼痛（类型）= C纤维激活（类型） |
| **个例同一理论** | 我的疼痛（个例）= 我的神经激活（个例） |
| **功能主义** | 疼痛（类型）= 疼痛功能（多种实现） |

**相关词条**：同一理论、多重实现

---

### 【常识心理学】(Folk Psychology)

**定义**：普通人用来理解和预测他人行为的常识理论框架。

**费曼解释**：
常识心理学说：
- 我们用"信念"和"欲望"解释行为
- "他带了伞，因为他相信会下雨"
- 这是普通人的"心理学理论"

**核心概念**：

| 概念 | 描述 |
|------|------|
| **信念** | 认为某事为真 |
| **欲望** | 想要某事发生 |
| **意图** | 打算做某事 |
| **解释** | 用信念和欲望解释行为 |

**对常识心理学的态度**：

| 立场 | 态度 | 代表 |
|------|------|------|
| **实在论** | 常识心理学是真的 | Fodor |
| **取消主义** | 常识心理学是假的 | Churchlands |
| **工具主义** | 常识心理学是有用的工具 | Dennett |

**当代发展 (2020-2026)**：
- **神经常识心理学**：神经科学视角
- **AI常识**：AI的常识理解
- **跨文化**：不同文化的常识心理学

**相关词条**：取消主义、功能主义、信念

---

### 【随附性】(Supervenience)

**定义**：一种依赖关系——没有B的变化就没有A的变化。

**费曼解释**：
随附性说：
- 心理状态依赖于物理状态
- 没有物理变化，心理不会变化
- 就像图像依赖于像素

**形式定义**：

A随附于B = 没有B的变化就没有A的变化

**在心身问题中**：

| 类型 | 描述 |
|------|------|
| **弱随附性** | 同一物理基础 → 同一心理状态（在同一世界） |
| **强随附性** | 同一物理基础 → 同一心理状态（在所有可能世界） |
| **全局随附性** | 物理同一的世界 → 心理同一 |

**随附性 vs 还原**：

| 问题 | 随附性 | 还原 |
|------|--------|------|
| 依赖 | 有依赖关系 | 可分析为更基本 |
| 必然性 | 可能是偶然的 | 通常是必然的 |
| 解释 | 不提供解释 | 提供解释 |

**问题**：
- 随附性太弱——不排除副现象论
- 随附性不能解释"为什么"

**当代发展 (2020-2026)**：
- **因果随附性**：随附性与因果
- **规范随附性**：道德与自然随附
- **AI随附性**：AI状态随附于物理

**相关词条**：物理主义、非还原物理主义、依赖

---

### 【心理 anomalism】(Psychological Anomalousness)

**定义**：Davidson的原则——不存在严格的心理定律。

**费曼解释**：
心理 anomalism 说：
- 物理定律是严格的
- 心理"定律"不是严格的
- 心理领域是"异常的"

**支持理由**：

| 理由 | 描述 |
|------|------|
| **规范性** | 心理受规范约束（理性） |
| **整体论** | 信念是整体性的 |
| **解释** | 心理解释是诠释性的 |

**与异常一元论的关系**：
- 这是异常一元论的核心原则之一
- 解释了为什么没有心理科学（严格意义上）

**相关词条**：异常一元论、Davidson

---

### 【心理物理定律】(Psychophysical Laws)

**定义**：连接心理状态和物理状态的定律。

**费曼解释**：
心理物理定律说：
- 如果存在，它们会告诉我们：
- "当C纤维激活时，就有疼痛"
- 但这样的定律存在吗？

**争论**：

| 立场 | 态度 |
|------|------|
| **同一理论** | 存在（类型同一） |
| **功能主义** | 不需要（功能定义） |
| **取消主义** | 不存在（概念错误） |
| **异常一元论** | 不存在（心理异常） |

**相关词条**：同一理论、功能主义、异常一元论

---

### 【感受性】(Qualia/Subjective Experience)

**定义**：主观的、第一人称的意识体验。

**费曼解释**：
感受性是"感觉"：
- 红色的红色感
- 疼痛的痛感
- 咖啡的味道

这是"做...是什么感觉"。

**感受性的特征**：

| 特征 | 描述 |
|------|------|
| **主观性** | 只有从第一人称可及 |
| **私密性** | 无法直接传达 |
| **直接性** | 直接感知，不需推理 |
| **不可错性** | 不能对自己的感受性出错 |

**感受性问题**：

| 问题 | 描述 |
|------|------|
| **存在性** | 感受性真实存在吗？ |
| **不可知性** | 感受性如何被他人知道？ |
| **物理主义兼容** | 感受性与物理主义兼容吗？ |

**倒置光谱**：

**思想实验**：
- 你看红色，我看绿色
- 但我们行为相同
- 我们如何知道？

**感受性 = 行为吗？**
- 不，倒置光谱显示它们不同

**Dennett的批评**：

**核心主张**：
- "感受性"是一个混乱的概念
- 我们直觉中的"感受性"不存在
- 应该取消这个概念

**当代发展 (2020-2026)**：
- **神经感受性**：感受性的神经基础
- **AI感受性**：AI有感受性吗？
- **跨物种感受性**：动物的感受性

**相关词条**：意识、知识论证、困难问题

---

### 【倒置光谱】(Inverted Spectrum)

**定义**：不同个体可能有不同的颜色体验但行为相同的思想实验。

**费曼解释**：
倒置光谱说：
- 你看红色，我看绿色
- 但我们都说"这是红色"
- 我们如何知道？

**论证**：

1. 倒置光谱是可能的
2. 如果可能，行为主义/功能主义是错的
3. 因为行为/功能相同但体验不同

**对行为主义的挑战**：
- 行为相同，但心理不同
- 因此，心理不能定义为行为

**对功能主义的挑战**：
- 功能相同，但体验可能不同
- 功能主义能排除倒置光谱吗？

**功能主义的回应**：
- 如果倒置光谱可能，功能主义有问题
- 但也许倒置光谱不可能？
- 也许功能角色决定了体验？

**相关词条**：感受性、行为主义、功能主义

---

### 【缺席感受性】(Absent Qualia)

**定义**：功能相同但没有感受性的假设情况。

**费曼解释**：
缺席感受性说：
- 一个系统功能完全正常
- 但没有"感觉"
- 就像"哲学僵尸"

**与哲学僵尸的关系**：
- 哲学僵尸是缺席感受性的极端例子
- 行为完全相同，但没有意识

**对功能主义的挑战**：
- 如果功能相同但感受性不同
- 那感受性不能由功能定义
- 功能主义不完整

**相关词条**：感受性、哲学僵尸、功能主义
