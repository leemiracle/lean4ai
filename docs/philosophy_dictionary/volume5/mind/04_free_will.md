## A-D

### 【自由意志】(Free Will)

**定义**：行动者能够在多种可能行动中进行选择的能力。

**费曼解释**：
自由意志问：
- 你的行为是你"自由选择"的吗？
- 还是预先决定的？
- 你"可以"选择不同吗？

**自由意志的核心问题**：

| 问题 | 描述 |
|------|------|
| **自由问题** | 我们有自由意志吗？ |
| **兼容问题** | 自由意志与决定论兼容吗？ |
| **道德责任** | 没有自由意志还有道德责任吗？ |
| **意志自由** | 什么是"自由"的意志？ |

**主要立场**：

| 立场 | 核心主张 | 代表 |
|------|----------|------|
| **硬决定论** | 决定论为真，没有自由意志 | Holbach |
| **自由意志论** | 决定论假，有自由意志 | Kane |
| **相容论** | 决定论与自由意志兼容 | Frankfurt |
| **硬不相容论** | 无论决定论真假，都没有自由意志 | Pereboom |

**自由意志的要素**：

| 要素 | 描述 |
|------|------|
| **选择能力** | 能在选项中选择 |
| **控制** | 对行为有控制 |
| **替代可能性** | 可以选择其他 |
| **来源自我** | 行为来源于自我 |

**自由意志的重要性**：

| 领域 | 重要性 |
|------|--------|
| **道德责任** | 自由意志是道德责任的前提 |
| **法律** | 自由意志是刑法的基础 |
| **人生意义** | 自由意志关系到人生意义 |
| **社会制度** | 影响奖惩制度 |

**代表人物**：
- **Harry Frankfurt**：相容论
- **Robert Kane**：自由意志论
- **Peter van Inwagen**：自由意志论证
- **Galen Strawson**：自由意志怀疑论

**经典文献**：
- Frankfurt, "Alternate Possibilities and Moral Responsibility" (1969)
- Kane, *The Significance of Free Will* (1996)
- van Inwagen, *An Essay on Free Will* (1983)

**当代发展 (2020-2026)**：
- **神经科学与自由意志**：Libet实验的新解释
- **AI自由意志**：AI能有自由意志吗？
- **双过程理论**：系统1/2与自由意志
- **实践必要性**：即使没有自由意志，我们也需要假定有

**相关词条**：决定论、相容论、道德责任

### 【决定论】(Determinism)

**定义**：每个事件都由先前事件和自然定律必然决定的观点。

**费曼解释**：
决定论说：
- 如果你知道宇宙所有粒子位置和速度
- 加上所有自然定律
- 你可以预测未来所有事件
- 一切都是"注定"的

**决定论的类型**：

| 类型 | 描述 |
|------|------|
| **因果决定论** | 每个事件有因果上充分的原因 |
| **逻辑决定论** | 关于未来的命题已有真值 |
| **神学决定论** | 上帝预知一切 |
| **物理决定论** | 物理定律决定一切 |

**拉普拉斯妖**：

**定义**：一个假想的存在，知道所有粒子状态和定律，能预测未来。

**含义**：
- 如果拉普拉斯妖可能
- 那未来是确定的
- 自由意志可能不存在

**对自由意志的威胁**：

| 论证 | 描述 |
|------|------|
| **后果论证** | 如果决定论为真，我们无法做其他 |
| **来源论证** | 我们的欲望由我们无法控制的因素决定 |

**当代物理学**：

| 理论 | 对决定论的态度 |
|------|----------------|
| **经典物理** | 支持决定论 |
| **量子力学** | 可能挑战决定论（有争议） |
| **混沌理论** | 决定但不可预测 |

**代表人物**：
- **Pierre-Simon Laplace**：拉普拉斯妖
- **Ted Honderich**：决定论者

**经典文献**：
- Laplace, *A Philosophical Essay on Probabilities* (1814)

**相关词条**：自由意志、因果决定论、拉普拉斯妖

### 【相容论】(Compatibilism)

**定义**：认为自由意志与决定论兼容的立场。

**费曼解释**：
相容论说：
- 决定论可能是真的
- 但我们仍然有自由意志
- 因为"自由"不意味着"非决定"

**相容论的核心**：

| 要素 | 描述 |
|------|------|
| **重新定义自由** | 自由=无强制，不是非决定 |
| **来源条件** | 行为来源于自己的欲望 |
| **等级结构** | 二阶欲望更重要 |

**古典相容论**：

**霍布斯/休谟**：
- 自由 = 无外部障碍
- 如果你想做X，你能做X，你就是自由的
- 不需要"可以做其他"

**条件分析**：
- S自由做A = 如果S选择做A，S就能做A
- 不要求"可以选择其他"

**对古典相容论的批评**：

| 批评 | 描述 |
|------|------|
| **Frankfurt案例** | 某人无法做其他但仍有责任 |
| **操纵论证** | 被操纵的人满足条件但不自由 |
| **来源问题** | 欲望来源是什么？ |

**Frankfurt相容论**：

**核心主张**：
- 自由意志不要求替代可能性
- 关键是"意志的结构"
- 二阶欲望与一阶欲望一致

**Frankfurt案例**：

**情境**：
- Jones决定做X
- Black是一个操控者，如果Jones想不做X，Black会强迫他
- 但Jones自己决定做X，Black不需要干预

**分析**：
- Jones无法做其他（Black会阻止）
- 但Jones似乎有道德责任
- 因此，替代可能性不是必要的

**层级欲望理论**：

| 层级 | 描述 | 例子 |
|------|------|------|
| **一阶欲望** | 想要做某事 | 想抽烟 |
| **二阶欲望** | 想要有某种欲望 | 不想有想抽烟的欲望 |
| **二阶意志** | 想要哪种一阶欲望有效 | 想要不抽烟的欲望生效 |

**自由意志的条件**：
- 一阶欲望与二阶意志一致
- 即，你"想要"你所"想要想要的"

**不相容论的批评**：

| 批评 | 描述 |
|------|------|
| **来源问题** | 二阶欲望的来源是什么？ |
| **回溯问题** | 什么保证二阶欲望的自由？ |
| **太弱** | Frankfurt条件太弱 |

**当代发展 (2020-2026)**：
- **半相容论**：道德责任与决定论兼容，但自由意志不一定
- **源头相容论**：强调行为的源头
- **理由响应机制**：自由意志是理由响应机制

**代表人物**：
- **Harry Frankfurt**：层级欲望理论
- **Daniel Dennett**：相容论辩护
- **John Fischer**：半相容论

**经典文献**：
- Frankfurt, "Alternate Possibilities and Moral Responsibility" (1969)
- Frankfurt, "Freedom of the Will and the Concept of a Person" (1971)
- Dennett, *Elbow Room* (1984)

**相关词条**：自由意志、决定论、Frankfurt案例

### 【不相容论】(Incompatibilism)

**定义**：认为自由意志与决定论不兼容的立场。

**费曼解释**：
不相容论说：
- 如果决定论为真，就没有自由意志
- 决定论和自由意志不能同时为真

**不相容论的类型**：

| 类型 | 核心主张 |
|------|----------|
| **自由意志论** | 有自由意志，因此决定论假 |
| **硬决定论** | 决定论真，因此无自由意志 |
| **硬不相容论** | 无论决定论真假，都无自由意志 |

**支持不相容论的论证**：

| 论证 | 描述 |
|------|------|
| **后果论证** | 决定论→无法做其他→无自由意志 |
| **直接论证** | 决定论与自由意志直接矛盾 |
| **操纵论证** | 决定论世界的人像被操纵 |

**后果论证** (van Inwagen)：

1. 如果决定论为真，我们的行为是自然定律和遥远过去事件的后果
2. 我们无法改变自然定律或遥远过去
3. 因此，我们无法改变我们的行为
4. 因此，如果决定论为真，我们没有自由意志

**相关词条**：自由意志、相容论、决定论

### 【自由意志论】(Libertarianism)

**定义**：认为有自由意志，因此决定论为假的不相容论立场。

**费曼解释**：
自由意志论说：
- 我们确实有自由意志
- 因此决定论必须是假的
- 某些事件是非决定的

**核心主张**：

| 要素 | 描述 |
|------|------|
| **不相容** | 决定论与自由意志不兼容 |
| **自由存在** | 我们有自由意志 |
| **非决定论** | 因此决定论假 |

**自由意志论的类型**：

| 类型 | 核心主张 | 代表 |
|------|----------|------|
| **事件因果** | 自由行为有非决定的事件原因 | Kane |
| **行动者因果** | 行动者是行为的原始原因 | Chisholm |
| **非因果论** | 自由行为无原因 | Ginet |

**事件因果自由意志论** (Kane)：

**核心主张**：
- 自由行为是非决定的
- 但仍然有原因（事件原因）
- 原因不充分决定结果

**交替努力模型**：
- 当意志分裂时（想A又想B）
- 结果是非决定的
- 这是"自由选择"

**问题**：

| 问题 | 描述 |
|------|------|
| **运气问题** | 非决定选择似乎是运气 |
| **控制问题** | 非决定如何保证控制？ |
| **随机性** | 非决定不是自由，是随机 |

**行动者因果自由意志论**：

**核心主张**：
- 行动者是行为的"原始原因"
- 不是事件导致事件
- 而是"行动者导致"事件

**Chisholm的论证**：
- 因果链必须有个起点
- 如果所有都是事件因果，没有自由
- 因此，必须有行动者因果

**问题**：

| 问题 | 描述 |
|------|------|
| **神秘性** | 行动者因果是什么？ |
| **科学不兼容** | 与科学世界观不兼容？ |
| **循环论证** | 行动者因果预设了自由？ |

**代表人物**：
- **Robert Kane**：事件因果自由意志论
- **Roderick Chisholm**：行动者因果
- **Carl Ginet**：非因果论

**经典文献**：
- Kane, *The Significance of Free Will* (1996)
- Chisholm, "Human Freedom and the Self" (1964)

**相关词条**：不相容论、决定论、行动者因果

### 【硬决定论】(Hard Determinism)

**定义**：认为决定论为真且因此没有自由意志的立场。

**费曼解释**：
硬决定论说：
- 决定论是真的
- 决定论与自由意志不兼容
- 因此，没有自由意志

**核心主张**：

| 要素 | 描述 |
|------|------|
| **决定论真** | 决定论是真的 |
| **不相容** | 决定论与自由意志不兼容 |
| **无自由** | 因此没有自由意志 |

**支持论证**：

| 论证 | 描述 |
|------|------|
| **科学支持** | 科学支持决定论 |
| **后果论证** | 决定论→无自由意志 |
| **来源论证** | 我们的本质由我们无法控制的因素决定 |

**道德责任问题**：

**核心困难**：
- 如果没有自由意志
- 道德责任如何可能？
- 惩罚如何正当？

**硬决定论的回应**：

| 回应 | 描述 |
|------|------|
| **取消责任** | 没有真正的道德责任 |
| **实用主义** | 责任是实用工具 |
| **向前看** | 惩罚是为了未来效果 |

**代表人物**：
- **Paul Holbach**：古典硬决定论
- **Galen Strawson**：当代硬决定论
- **Derk Pereboom**：硬不相容论

**经典文献**：
- Holbach, *The System of Nature* (1770)
- Strawson, "The Impossibility of Moral Responsibility" (1994)

**相关词条**：决定论、不相容论、道德责任

### 【硬不相容论】(Hard Incompatibilism)

**定义**：Pereboom提出的立场，认为无论决定论真假都没有自由意志。

**费曼解释**：
硬不相容论说：
- 不只是决定论与自由意志不兼容
- 非决定论也不给我们自由意志
- 因此，无论如何都没有自由意志

**核心主张**：

| 要素 | 描述 |
|------|------|
| **相容论失败** | 相容论不能拯救自由意志 |
| **自由意志论失败** | 非决定论不能给我们自由 |
| **无自由** | 无论如何都没有自由意志 |

**反对自由意志论的论证**：

| 论证 | 描述 |
|------|------|
| **运气论证** | 非决定选择只是运气 |
| **控制论证** | 非决定意味着失去控制 |
| **随机性** | 随机不是自由 |

**道德责任**：

**核心主张**：
- 没有自由意志，就没有基本道德责任
- 但我们可以保留"向前看"的责任概念

**四阶段策略**：

| 阶段 | 描述 |
|------|------|
| 1. 相容论失败 | 相容论道德责任不成立 |
| 2. 自由意志论失败 | 非决定论不能帮助 |
| 3. 无道德责任 | 因此没有基本道德责任 |
| 4. 实用后果 | 但社会制度可以保留 |

**代表人物**：
- **Derk Pereboom**：硬不相容论提出者

**经典文献**：
- Pereboom, *Living Without Free Will* (2001)

**相关词条**：不相容论、硬决定论

### 【道德责任】(Moral Responsibility)

**定义**：行动者因其行为而值得赞扬或责备的状态。

**费曼解释**：
道德责任问：
- 你做坏事，你"应该负责"吗？
- 什么条件使你"应负责任"？
- 如果没有自由意志，还有责任吗？

**道德责任的条件**：

| 条件 | 描述 |
|------|------|
| **控制条件** | 对行为有控制 |
| **认知条件** | 知道行为的性质 |
| **自由条件** | 自由选择行为 |

**责任的类型**：

| 类型 | 描述 |
|------|------|
| **回顾性责任** | 对过去行为负责 |
| **前瞻性责任** | 对未来行为有责任 |
| **因果责任** | 作为原因负责 |
| **道德责任** | 值得赞扬/责备 |

**自由意志与责任**：

| 观点 | 描述 |
|------|------|
| **传统观点** | 自由意志是责任的必要条件 |
| **Frankfurt观点** | 替代可能性不是必要的 |
| **硬不相容论** | 没有真正的道德责任 |

**Strawson的反应态度理论**：

**核心主张**：
- 道德责任与我们的"反应态度"相连
- 怨恨、感激、宽恕等
- 这些态度是自然的，不可避免的

**代表人物**：
- **P.F. Strawson**：反应态度理论
- **Harry Frankfurt**：层级理论

**经典文献**：
- Strawson, "Freedom and Resentment" (1962)
- Frankfurt, "Alternate Possibilities and Moral Responsibility" (1969)

**相关词条**：自由意志、反应态度

### 【替代可能性】(Alternative Possibilities)

**定义**：行动者可以在不同行动之间选择的可能性。

**费曼解释**：
替代可能性问：
- "我可以做其他"是什么意思？
- 自由意志需要替代可能性吗？
- Frankfurt案例说明了什么？

**替代可能性的原则**：

**原则 (PAP)**：一个人对其行为有道德责任，仅当他可以做其他。

**Frankfurt对PAP的挑战**：

**Frankfurt案例**：
- Jones决定做X
- Black会阻止Jones不做X（如果Jones想不做X）
- 但Jones自己决定做X，Black不需要干预

**分析**：
- Jones无法做其他
- 但Jones有道德责任
- 因此PAP是错的

**对Frankfurt的回应**：

| 回应 | 描述 |
|------|------|
| **闪烁的自由** | 在某个时刻有替代可能性 |
| **辩证支持** | 替代可能性提供辩证支持 |
| **新PAP** | 修改PAP以避免反例 |

**相关词条**：Frankfurt案例、道德责任

### 【Frankfurt案例】(Frankfurt Cases)

**定义**：Harry Frankfurt提出的思想实验，挑战替代可能性原则。

**费曼解释**：
Frankfurt案例：
- Jones决定做X
- Black（操控者）会阻止Jones做其他
- 但Jones自己选择做X
- Black不需要干预

Jones可以做其他吗？不能。
Jones有责任吗？有。
因此，替代可能性不是必要的。

**案例结构**：

| 要素 | 描述 |
|------|------|
| **行动者** | Jones |
| **操控者** | Black |
| **机制** | 如果Jones想不做X，Black干预 |
| **实际** | Jones自己选择做X，Black不干预 |

**哲学意义**：

| 意义 | 描述 |
|------|------|
| **挑战PAP** | 替代可能性不是必要的 |
| **支持相容论** | 决定论不一定否定责任 |
| **强调源头** | 责任在于行为源头 |

**对Frankfurt案例的批评**：

| 批评 | 描述 |
|------|------|
| **闪烁自由** | 在某个时刻有替代可能性 |
| **残留直觉** | 我们的直觉可能不支持结论 |
| **案例失败** | 案例可能不成立 |

**代表人物**：
- **Harry Frankfurt**：案例提出者

**经典文献**：
- Frankfurt, "Alternate Possibilities and Moral Responsibility" (1969)

**相关词条**：替代可能性、相容论

### 【后果论证】(Consequence Argument)

**定义**：van Inwagen提出的反对相容论的论证。

**费曼解释**：
后果论证说：
- 如果决定论为真，我们的行为是自然定律和遥远过去的后果
- 我们无法改变自然定律或遥远过去
- 因此，我们无法改变我们的行为

**论证形式**：

1. 如果决定论为真，我们的行为是自然定律和遥远过去事件的后果
2. 我们无法改变自然定律
3. 我们无法改变遥远过去的事件
4. 因此，我们无法改变我们的行为（如果决定论为真）
5. 因此，如果决定论为真，我们没有自由意志

**规则β**：
- 如果p蕴含q，且没有人可以做任何事使得p为假，则没有人可以做任何事使得q为假

**对后果论证的回应**：

| 回应 | 描述 |
|------|------|
| **否认规则β** | 规则β不成立 |
| **重新定义"能"** | "能"有不同含义 |
| **条件分析** | "能做其他"是条件句 |

**代表人物**：
- **Peter van Inwagen**：论证提出者

**经典文献**：
- van Inwagen, *An Essay on Free Will* (1983)

**相关词条**：不相容论、决定论

### 【行动者因果】(Agent Causation)

**定义**：行动者作为行为的原始原因的观点。

**费曼解释**：
行动者因果说：
- 通常我们说是"事件导致事件"
- 但自由行动中，是"行动者导致事件"
- 行动者是原始原因

**vs 事件因果**：

| 类型 | 描述 |
|------|------|
| **事件因果** | 事件1导致事件2 |
| **行动者因果** | 行动者导致事件 |

**Chisholm的论证**：

1. 如果所有因果都是事件因果，那么我们的行为由先前事件决定
2. 如果由先前事件决定，我们没有自由意志
3. 但我们有自由意志
4. 因此，必须有行动者因果

**问题**：

| 问题 | 描述 |
|------|------|
| **神秘性** | 行动者因果是什么？ |
| **科学问题** | 与科学世界观兼容吗？ |
| **循环** | 预设了自由？ |

**代表人物**：
- **Roderick Chisholm**：行动者因果提出者
- **Timothy O'Connor**：当代发展

**经典文献**：
- Chisholm, "Human Freedom and the Self" (1964)

**相关词条**：自由意志论、因果

---

### 【源头内在于】(Sourcehood)

**定义**：行为来源于行动者自身的条件。

**费曼解释**：
源头内在于说：
- 自由行为必须"来源于"行动者
- 不是被外在因素决定
- 行动者是行为的"源头"

**核心主张**：

| 要素 | 描述 |
|------|------|
| **来源条件** | 行为来源于行动者 |
| **非外在决定** | 不被外在因素决定 |
| **自我决定** | 行动者决定自己的行为 |

**在相容论中的角色**：
- 源头相容论强调来源条件
- 行为来自自己的欲望/信念
- 而不是外在强制

**相关词条**：自由意志、相容论

---

### 【意志自由】(Freedom of the Will)

**定义**：意志本身的自由，而非行为的自由。

**费曼解释**：
意志自由问：
- 自由是关于"行为"还是"意志"？
- 我的"意志"自由吗？
- 还是只有行为自由？

**意志自由 vs 行为自由**：

| 类型 | 描述 |
|------|------|
| **意志自由** | 意志/选择本身自由 |
| **行为自由** | 行为本身自由 |

**Frankfurt的区分**：
- 自由行为：没有外在障碍
- 自由意志：意志结构健康（一阶和二阶欲望一致）

**相关词条**：自由意志、Frankfurt

---

### 【层级欲望】(Hierarchical Desires)

**定义**：Frankfurt提出的欲望层级结构。

**费曼解释**：
层级欲望说：
- 一阶欲望：想要做某事
- 二阶欲望：想要有某种欲望
- 二阶意志：想要哪种欲望生效

**例子**：

| 层级 | 例子 |
|------|------|
| **一阶欲望** | 想抽烟 |
| **二阶欲望** | 不想有想抽烟的欲望 |
| **二阶意志** | 想要不抽烟的欲望生效 |

**自由意志条件**：
- 一阶欲望与二阶意志一致
- 你"想要"你所"想要想要的"

**相关词条**：Frankfurt、自由意志

---

### 【意志无能】(Unwilling Addict)

**定义**：Frankfurt案例中的一类人，欲望与意志冲突。

**费曼解释**：
意志无能者：
- 想吸毒（一阶欲望）
- 不想有这个欲望（二阶欲望）
- 不想吸毒的欲望生效（二阶意志）
- 但一阶欲望与二阶意志冲突

**与意愿吸毒者的区别**：

| 类型 | 一阶欲望 | 二阶意志 |
|------|----------|----------|
| **意志无能者** | 想吸毒 | 不想吸毒 |
| **意愿吸毒者** | 想吸毒 | 想吸毒 |
| **不想吸毒者** | 不想吸毒 | 不想吸毒 |

**哲学意义**：
- 只有意愿者有自由意志
- 意志无能者没有自由意志（尽管行为相同）

**相关词条**：Frankfurt、层级欲望

---

### 【意愿吸毒者】(Willing Addict)

**定义**：Frankfurt案例中的一类人，欲望与意志一致。

**费曼解释**：
意愿吸毒者：
- 想吸毒（一阶欲望）
- 想有这个欲望（二阶欲望）
- 想吸毒的欲望生效（二阶意志）
- 一阶欲望与二阶意志一致

**自由意志**：
- 意愿者有自由意志
- 尽管行为可能是不好的
- 自由意志不要求行为正确

**相关词条**：Frankfurt、意志无能

---

### 【反应态度】(Reactive Attitudes)

**定义**：P.F. Strawson提出的对他人行为的态度。

**费曼解释**：
反应态度是：
- 怨恨、感激、愤怒、宽恕
- 我们对他人行为的自然反应
- 这些态度构成了道德责任

**类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **个人反应态度** | 对个人的反应 | 怨恨、感激 |
| **代理反应态度** | 代理他人反应 | 愤怒（为他人） |
| **自我反应态度** | 对自己的反应 | 内疚、骄傲 |

**免除条件**：

| 条件 | 描述 |
|------|------|
| **未成年** | 儿童不适用 |
| **精神疾病** | 精神疾病者不适用 |
| **胁迫** | 被胁迫者不适用 |

**哲学意义**：
- 道德责任不需要形而上学自由意志
- 而是根植于我们的人际态度
- 这些态度是不可消除的

**代表人物**：
- **P.F. Strawson**：反应态度理论

**经典文献**：
- Strawson, "Freedom and Resentment" (1962)

**相关词条**：道德责任、自由意志

---

### 【运气问题】(Luck Problem)

**定义**：非决定的自由选择似乎依赖于运气的问题。

**费曼解释**：
运气问题说：
- 如果选择是非决定的
- 那结果似乎是"随机的"
- 随机不是自由

**核心困难**：

| 困难 | 描述 |
|------|------|
| **控制问题** | 随机意味着失去控制 |
| **责任问题** | 随机选择不应负责 |
| **自我问题** | 随机选择不是"我"的选择 |

**自由意志论的回应**：

| 回应 | 描述 |
|------|------|
| **行动者因果** | 不是随机，是行动者导致 |
| **增强控制** | 非决定不意味着无控制 |
| **理由响应** | 选择仍然响应理由 |

**相关词条**：自由意志论、非决定论

---

### 【来源论证】(Source Argument）

**定义**：基于行为来源的反对自由意志的论证。

**费曼解释**：
来源论证说：
- 你的欲望和信念从哪里来？
- 从基因、环境、教育等
- 这些你都无法控制
- 因此，你的行为不是真正自由的

**Strawson的基本论证**：

1. 你做你所做是因为你是什么样的
2. 你是什么样的由你的基因和环境决定
3. 你无法控制你的基因和环境
4. 因此，你无法控制你是怎样的
5. 因此，你无法控制你做什么

**回应**：

| 回应 | 描述 |
|------|------|
| **渐进自由** | 自由可以渐进获得 |
| **自我塑造** | 我们可以塑造自己 |
| **实用主义** | 即使源头问题，责任仍实用 |

**代表人物**：
- **Galen Strawson**：来源论证

**经典文献**：
- Strawson, "The Impossibility of Moral Responsibility" (1994)

**相关词条**：自由意志、道德责任

---

### 【闪烁的自由】(Flicker of Freedom)

**定义**：对Frankfurt案例的回应，认为仍有微弱的替代可能性。

**费曼解释**：
闪烁的自由说：
- 即使在Frankfurt案例中
- Jones仍有某种替代可能性
- 比如"选择做X"vs"被迫做X"

**核心主张**：

| 主张 | 描述 |
|------|------|
| **残留替代** | 仍有某种替代可能性 |
| **微弱但存在** | 虽然微弱，但存在 |
| **足以支持PAP** | 足以支持替代可能性原则 |

**对Frankfurt的反驳**：
- Frankfurt案例没有完全消除替代可能性
- 因此不能反驳PAP

**相关词条**：Frankfurt案例、替代可能性

---

### 【半相容论】(Semi-Compatibilism)

**定义**：Fischer提出的立场，认为道德责任与决定论兼容，但自由意志不一定。

**费曼解释**：
半相容论说：
- 道德责任与决定论兼容（这是主要问题）
- 自由意志是否与决定论兼容是次要的
- 重点是责任，不是自由

**核心主张**：

| 要素 | 描述 |
|------|------|
| **责任兼容** | 道德责任与决定论兼容 |
| **自由次要** | 自由意志是次要问题 |
| **Frankfurt支持** | Frankfurt案例支持责任兼容 |

**理由响应机制**：

**核心概念**：
- 自由意志/道德责任需要"理由响应机制"
- 即，能够对理由做出反应
- 这与决定论兼容

**代表人物**：
- **John Martin Fischer**：半相容论提出者

**经典文献**：
- Fischer, *The Metaphysics of Free Will* (1994)

**相关词条**：相容论、道德责任

---

### 【理由响应】(Reasons-Responsiveness)

**定义**：行动者能够根据理由调整行为的能力。

**费曼解释**：
理由响应说：
- 自由行动者能够对理由做出反应
- 如果有足够的理由做其他，你会做其他
- 这是自由的关键

**理由响应的类型**：

| 类型 | 描述 |
|------|------|
| **强响应** | 有理由就改变行为 |
| **弱响应** | 某些理由导致改变 |
| **无响应** | 无论如何都不改变 |

**在自由意志中的角色**：
- 理由响应是自由意志的关键
- 与决定论兼容
- 解释了道德责任

**相关词条**：半相容论、Fischer

---

### 【Libet实验】(Libet Experiment)

**定义**：Benjamin Libet的神经科学实验，似乎挑战自由意志。

**费曼解释**：
Libet实验：
- 让被试自由选择何时动手指
- 测量大脑准备电位（RP）
- RP在被试报告"决定"之前数百毫秒出现

**结果**：
- 大脑活动先于有意识决定
- 似乎"决定"在意识之前就已经发生
- 挑战自由意志？

**对Libet实验的解释**：

| 解释 | 描述 |
|------|------|
| **挑战自由意志** | 决定是无意识的 |
| **不挑战** | 准备电位不是决定 |
| **否决权** | 意识可以"否决"无意识倾向 |

**当代发展 (2020-2026)**：
- **新实验**：更精细的神经成像
- **新解释**：对实验结果的哲学分析
- **意识时间**：意识的时间问题

**代表人物**：
- **Benjamin Libet**：实验设计者

**经典文献**：
- Libet, "Unconscious Cerebral Initiative and the Role of Conscious Will" (1983)

**相关词条**：自由意志、神经科学

---

### 【否决假说】(Veto Hypothesis）

**定义**：对Libet实验的解释，认为意识可以"否决"无意识倾向。

**费曼解释**：
否决假说说：
- 虽然无意识先于有意识决定
- 但有意识可以"否决"无意识倾向
- 这是自由意志的位置

**核心主张**：

| 要素 | 描述 |
|------|------|
| **无意识启动** | 行为由无意识启动 |
| **意识否决** | 意识可以阻止 |
| **自由在于否决** | 自由意志在于否决能力 |

**批评**：

| 批评 | 描述 |
|------|------|
| **否决也是无意识** | 否决也可能是无意识的 |
| **时间太短** | 否决时间窗口太短 |
| **支持不足** | 实验支持不足 |

**相关词条**：Libet实验、自由意志

---

### 【深层自我】(Deep Self）

**定义**：Watson和Wolf提出的自由意志概念，强调与深层自我一致。

**费曼解释**：
深层自我说：
- 自由行为来自"深层自我"
- 不是表面的、外在影响的欲望
- 而是真正代表"你是谁"的欲望

**深层自我的特征**：

| 特征 | 描述 |
|------|------|
| **价值核心** | 代表核心价值观 |
| **稳定性** | 相对稳定 |
| **反思性** | 经过反思 |
| **认同** | 行动者认同 |

**在自由意志中的角色**：
- 来自深层自我的行为是自由的
- 被外在影响控制的行为不自由

**相关词条**：自由意志、相容论

---

### 【意志 vs 欲望】(Will vs Desire)

**定义**：意志与欲望的区别。

**费曼解释**：
意志vs欲望：
- 欲望：想要某物
- 意志：决定去做某事
- 欲望可能很多，意志是单一的决定

**区别**：

| 概念 | 描述 |
|------|------|
| **欲望** | 想要的状态 |
| **意志** | 决定的状态 |

**Frankfurt的贡献**：
- 区分一阶欲望和二阶意志
- 意志是"想让哪个欲望生效"

**相关词条**：层级欲望、自由意志

---

### 【非因果自由意志论】(Non-Causal Libertarianism）

**定义**：认为自由行为没有原因的自由意志论版本。

**费曼解释**：
非因果自由意志论说：
- 自由行为没有原因
- 既不是事件原因，也不是行动者原因
- 就是"无原因"的选择

**核心主张**：

| 要素 | 描述 |
|------|------|
| **无原因** | 自由行为无原因 |
| **非随机** | 但不是随机的 |
| **行动者控制** | 行动者仍有控制 |

**问题**：

| 问题 | 描述 |
|------|------|
| **随机性** | 无原因似乎等于随机 |
| **控制** | 如何控制无原因的行为？ |
| **神秘** | 这是神秘的 |

**代表人物**：
- **Carl Ginet**：非因果自由意志论

**经典文献**：
- Ginet, *On Action* (1990)

**相关词条**：自由意志论、行动者因果
