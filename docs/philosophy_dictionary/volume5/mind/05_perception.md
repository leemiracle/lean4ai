## A-D

### 【感知】(Perception)

**词源**：拉丁语 *perceptio* (感知)

**定义**：通过感官获取关于外部世界信息的过程。

**费曼解释**：
感知是：
- 你看到苹果
- 你听到音乐
- 你感到温度
- 通过感官获取世界信息

**感知的核心问题**：

| 问题 | 描述 |
|------|------|
| **对象问题** | 我们感知到的是什么？ |
| **直接性** | 我们直接感知世界吗？ |
| **知识问题** | 感知如何提供知识？ |
| **错觉** | 错觉如何可能？ |

**感知理论**：

| 理论 | 核心主张 | 代表 |
|------|----------|------|
| **直接实在论** | 直接感知外部对象 | Reid |
| **表征主义** | 感知通过表征 | Peacocke |
| **感觉材料论** | 感知到感觉材料 | Russell |
| **析取论** | 真实感知和错觉不同 | Snowdon |
| **具身感知** | 感知依赖身体 | Noë |

**感知的要素**：

| 要素 | 描述 |
|------|------|
| **感官** | 眼、耳、鼻、舌、身 |
| **刺激** | 外部物理刺激 |
| **神经处理** | 大脑处理 |
| **经验** | 主观体验 |

**感知 vs 知觉**：

| 概念 | 描述 |
|------|------|
| **感知 (Sensation)** | 原始感官输入 |
| **知觉 (Perception)** | 组织和解释的感知 |

**代表人物**：
- **Thomas Reid**：直接实在论
- **Austin**：感知哲学
- **Alva Noë**：具身感知

**经典文献**：
- Reid, *An Inquiry into the Human Mind* (1764)
- Austin, *Sense and Sensibilia* (1962)
- Noë, *Action in Perception* (2004)

**当代发展 (2020-2026)**：
- **预测处理**：感知作为预测
- **主动感知**：感知与行动
- **社会感知**：感知他人
- **AI感知**：机器感知

**相关词条**：直接实在论、表征主义、感觉材料

### 【直接实在论】(Direct Realism)

**定义**：认为我们直接感知外部对象的感知理论。

**费曼解释**：
直接实在论说：
- 你看到苹果
- 你直接看到苹果（不是苹果的图像）
- 感知直接连接世界

**核心主张**：

| 要素 | 描述 |
|------|------|
| **直接性** | 直接感知对象 |
| **对象是外部的** | 感知的是外部对象 |
| **无需中介** | 不需要表征或感觉材料 |

**支持论证**：

| 论证 | 描述 |
|------|------|
| **常识论证** | 这是我们普通人的看法 |
| **科学论证** | 科学假设直接感知 |
| **简单性** | 最简单的理论 |

**对直接实在论的挑战**：

| 挑战 | 描述 |
|------|------|
| **错觉** | 错觉时没有直接感知对象 |
| **幻觉** | 幻觉时没有对象 |
| **时间延迟** | 光传播需要时间 |
| **神经中介** | 大脑处理是中介 |

**Reid的回应**：

**感觉 vs 知觉**：
- 感觉：主观体验
- 知觉：对对象的信念
- 知觉直接指向对象

**代表人物**：
- **Thomas Reid**：直接实在论创始人
- **John McDowell**：当代直接实在论

**经典文献**：
- Reid, *An Inquiry into the Human Mind* (1764)

**相关词条**：感知、表征主义

### 【表征主义】(Representationalism/Intentionalism)

**定义**：认为感知通过表征来感知对象的感知理论。

**费曼解释**：
表征主义说：
- 你看到苹果
- 你的感知"表征"苹果
- 就像图片表征事物

**核心主张**：

| 要素 | 描述 |
|------|------|
| **表征** | 感知是表征状态 |
| **意向性** | 感知有内容 |
| **准确条件** | 感知有准确条件 |

**表征主义的类型**：

| 类型 | 描述 |
|------|------|
| **强表征主义** | 感知经验完全是表征 |
| **弱表征主义** | 感知主要是表征，但不止于此 |
| **现象表征主义** | 现象特征也是表征 |

**支持论证**：

| 论证 | 描述 |
|------|------|
| **错觉/幻觉统一** | 表征主义统一解释错觉和真实感知 |
| **透明性** | 感知经验"透明"指向对象 |
| **内容理论** | 与意向性理论一致 |

**透明性论证**：

**Moore/Gilbert Harman**：
- 当你看一个苹果
- 你只注意苹果，不是你的经验本身
- 经验是"透明"的

**对表征主义的批评**：

| 批评 | 描述 |
|------|------|
| **感质问题** | 表征主义能否解释感质？ |
| **倒置光谱** | 同一表征可能有不同感质 |
| **内容确定** | 感知内容如何确定？ |

**代表人物**：
- **Fred Dretske**：感知表征主义
- **Michael Tye**：现象表征主义
- **Gilbert Harman**：透明性论证

**经典文献**：
- Dretske, *Knowledge and the Flow of Information* (1981)
- Tye, *Ten Problems of Consciousness* (1995)

**相关词条**：感知、意向性、感质

### 【感觉材料】(Sense Data)

**定义**：感知的直接对象，心理实体。

**费曼解释**：
感觉材料说：
- 你看苹果
- 你直接感知的不是苹果
- 而是"苹果的感觉材料"
- 这是心理的、私密的

**核心主张**：

| 要素 | 描述 |
|------|------|
| **直接对象** | 感觉材料是感知的直接对象 |
| **心理实体** | 感觉材料在心中 |
| **私密性** | 只有感知者能接触 |
| **无误性** | 对感觉材料不会出错 |

**支持论证**：

| 论证 | 描述 |
|------|------|
| **错觉论证** | 错觉时仍有"某物"被感知 |
| **时间延迟** | 光传播需要时间，所以不能直接感知对象 |
| **幻觉论证** | 幻觉时有感觉材料但没有对象 |

**时间延迟论证**：

1. 光从星星到地球需要时间
2. 我现在看到的是"那时的"星星
3. 我不可能直接感知"现在的"星星
4. 因此，我感知的是感觉材料

**对感觉材料的批评**：

| 批评 | 描述 |
|------|------|
| **Austin批评** | 感觉材料理论基于错误前提 |
| **本体论膨胀** | 引入不必要的实体 |
| **解释力弱** | 不能解释感知 |

**Austin的批评**：

**核心主张**：
- 感觉材料理论依赖"论证"
- 但这些论证是薄弱的
- 普通语言分析显示问题

**代表人物**：
- **Bertrand Russell**：感觉材料论
- **G.E. Moore**：感觉材料论
- **H.H. Price**：感觉材料论
- **J.L. Austin**：批评者

**经典文献**：
- Russell, *The Problems of Philosophy* (1912)
- Austin, *Sense and Sensibilia* (1962)

**相关词条**：感知、直接实在论

### 【错觉】(Illusion)

**定义**：感知对象存在但被错误感知的情况。

**费曼解释**：
错觉是：
- 筷子在水中看起来弯曲
- 镜子中的像是反的
- 远处的火车看起来很小

对象存在，但感知不正确。

**错觉的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **光学错觉** | 光学原因的错觉 | 水中筷子弯曲 |
| **大小错觉** | 大小误判 | Muller-Lyer错觉 |
| **运动错觉** | 运动误判 | 旋转蛇错觉 |
| **颜色错觉** | 颜色误判 | 同色异彩 |

**哲学意义**：

| 意义 | 描述 |
|------|------|
| **挑战直接实在论** | 错觉似乎需要感觉材料或表征 |
| **支持析取论** | 真实感知和错觉本质不同 |
| **感知理论测试** | 测试感知理论的案例 |

**相关词条**：感知、幻觉、析取论

### 【幻觉】(Hallucination)

**定义**：没有外部对象时仍有的感知体验。

**费曼解释**：
幻觉是：
- 沙漠中的海市蜃楼（但没有水）
- 精神分裂症的幻听（没有声音）
- 药物导致的幻觉（没有对象）

没有对象，但有体验。

**幻觉的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **病理幻觉** | 疾病导致 | 精神分裂症幻听 |
| **药物幻觉** | 药物导致 | LSD幻觉 |
| **感觉剥夺** | 感觉剥夺导致 | 孤独实验 |
| **正常幻觉** | 正常情况 | 入睡前幻觉 |

**哲学意义**：

| 意义 | 描述 |
|------|------|
| **共同因素假设** | 幻觉和真实感知有共同因素 |
| **挑战直接实在论** | 直接实在论难以解释幻觉 |
| **支持表征主义** | 表征主义统一解释 |

**Macbeth的匕首**：

**情境**：
- Macbeth看到一把匕首
- 但没有匕首
- 这是幻觉

**分析**：
- 如果直接实在论是对的，他应该没有感知
- 但他有感知体验
- 因此需要感觉材料或表征

**相关词条**：错觉、感知、共同因素

### 【析取论】(Disjunctivism)

**定义**：认为真实感知和错觉/幻觉是根本不同类型的状态的理论。

**费曼解释**：
析取论说：
- 真实感知：看到苹果
- 错觉：看起来像苹果但不是
- 这两种状态根本不同，不是同一类型

**核心主张**：

| 要素 | 描述 |
|------|------|
| **非共同因素** | 真实感知和错觉没有共同因素 |
| **析取性质** | 状态类型是析取的 |
| **对象依赖** | 真实感知依赖外部对象 |

**vs 共同因素假设**：

| 问题 | 共同因素 | 析取论 |
|------|----------|--------|
| 共同因素 | 有 | 无 |
| 状态类型 | 相同 | 不同 |
| 对象依赖 | 不依赖 | 依赖 |

**支持论证**：

| 论证 | 描述 |
|------|------|
| **差异论证** | 真实感知和错觉有根本差异 |
| **直觉论证** | 我们直觉上认为它们不同 |
| **反驳怀疑论** | 有助于反驳怀疑论 |

**对析取论的批评**：

| 批评 | 描述 |
|------|------|
| **不可区分性** | 从内在看，它们不可区分 |
| **共同因素** | 似乎有共同因素 |
| **理论负担** | 需要更多解释 |

**代表人物**：
- **John McDowell**：析取论
- **M.G.F. Martin**：当代析取论
- **Paul Snowdon**：析取论

**经典文献**：
- McDowell, "Singular Thought and the Extent of Inner Space" (1986)
- Snowdon, "Perception, Vision and Causation" (1980)

**相关词条**：感知、共同因素、幻觉

### 【共同因素假设】(Common Factor Assumption)

**定义**：真实感知和错觉/幻觉有共同的心理因素。

**费曼解释**：
共同因素假设说：
- 你看苹果（真实）
- 你幻觉苹果（错觉）
- 这两种状态有共同的心理因素

**核心主张**：

| 要素 | 描述 |
|------|------|
| **共同因素** | 有共同的心理因素 |
| **经验相同** | 内在经验相同 |
| **外在差异** | 差异在外在因素 |

**支持论证**：

| 论证 | 描述 |
|------|------|
| **不可区分性** | 从内在不可区分 |
| **因果解释** | 共同因素解释相似性 |
| **神经科学** | 神经基础相似 |

**析取论的反对**：

**核心批评**：
- 如果析取论是对的
- 那共同因素假设是错的
- 真实感知和错觉根本不同

**相关词条**：析取论、幻觉

### 【感觉与知觉】(Sensation vs Perception)

**定义**：原始感官输入与组织解释后的感知的区别。

**费曼解释**：
感觉vs知觉：
- 感觉：原始的光、声刺激
- 知觉：识别为"苹果"、"音乐"
- 感觉是原始的，知觉是组织的

**感觉的特征**：

| 特征 | 描述 |
|------|------|
| **原始性** | 未经组织的 |
| **感官特异** | 特定感官的 |
| **自下而上** | 从感官开始 |

**知觉的特征**：

| 特征 | 描述 |
|------|------|
| **组织性** | 已组织的 |
| **对象识别** | 识别对象 |
| **自上而下** | 概念参与 |

**哲学意义**：
- 区分有助于理解感知
- 但感觉和知觉的边界有争议
- 有人认为不能清晰区分

**相关词条**：感知

### 【感知的内容】(Content of Perception)

**定义**：感知所呈现的世界信息。

**费曼解释**：
感知内容是：
- 你看到"红色的苹果"
- 这是感知的"内容"
- 它呈现了世界的某些方面

**内容的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **命题内容** | 可真可假 | "苹果是红的" |
| **对象内容** | 关于对象 | 关于这个苹果 |
| **概念内容** | 需要概念 | "苹果" |
| **非概念内容** | 不需要概念 | 原始色块 |

**概念内容 vs 非概念内容**：

| 问题 | 概念内容 | 非概念内容 |
|------|----------|------------|
| 需要概念？ | 是 | 否 |
| 动物有？ | 有争议 | 是 |
| 精细度 | 较粗 | 很精细 |

**非概念内容的论证**：

**Richness论证**：
- 感知经验非常丰富
- 我们没有那么多概念
- 因此存在非概念内容

**相关词条**：感知、概念

---

### 【非概念内容】(Non-Conceptual Content)

**定义**：不需要概念就可以拥有的感知内容。

**费曼解释**：
非概念内容说：
- 你看到精细的颜色渐变
- 你没有那么多颜色概念
- 但你仍然有这些内容

**支持论证**：

| 论证 | 描述 |
|------|------|
| **丰富性论证** | 感知太丰富，概念不够 |
| **动物论证** | 动物有感知但没有概念 |
| **婴儿论证** | 婴儿有感知但没有概念 |

**对非概念内容的批评**：

| 批评 | 描述 |
|------|------|
| **概念能力** | 也许我们有隐含的概念能力 |
| **内容概念** | 也许所有内容都需要概念 |
| **统一问题** | 非概念内容如何统一？ |

**代表人物**：
- **Christopher Peacocke**：非概念内容理论
- **John McDowell**：批评者

**经典文献**：
- Peacocke, "Nonconceptual Content" (1998)

**相关词条**：感知内容、概念

### 【感知的因果理论】(Causal Theory of Perception)

**定义**：感知需要对象因果作用于感官的理论。

**费曼解释**：
因果理论说：
- 你看到苹果
- 苹果必须因果作用于你的眼睛
- 光从苹果到你的眼睛
- 这是感知的必要条件

**Grice的分析**：

**核心主张**：
- S感知O = O因果导致S的感知经验
- 需要适当的因果链

**问题**：

| 问题 | 描述 |
|------|------|
| **假因果链** | 适当的因果链是什么？ |
| **因果过剩** | 如何排除不适当的因果？ |
| **幻觉** | 幻觉如何处理？ |

**相关词条**：感知、因果

### 【感知的可错性】(Fallibility of Perception)

**定义**：感知可能出错的事实。

**费曼解释**：
感知可错说：
- 你可能看错
- 你可能听错
- 感知不是绝对可靠的

**可错的类型**：

| 类型 | 描述 | 例子 |
|------|------|------|
| **错觉** | 对象存在但误感知 | 筷子弯曲 |
| **幻觉** | 无对象但感知 | 幻听 |
| **误认** | 误认对象 | 认错人 |
| **忽视** | 忽视对象 | 注意盲点 |

**哲学意义**：

| 意义 | 描述 |
|------|------|
| **知识问题** | 感知如何提供知识？ |
| **怀疑论** | 为怀疑论提供材料 |
| **感知理论** | 感知理论必须处理可错性 |

**相关词条**：错觉、幻觉、感知

### 【感知的论证】(Argument from Perception)

**定义**：从感知现象推出的哲学论证。

**费曼解释**：
感知论证是：
- 从感知的事实出发
- 推出某些哲学结论
- 如：存在外部世界

**例子**：

| 论证 | 结论 |
|------|------|
| **Moore的手论证** | 存在外部世界 |
| **错觉论证** | 需要感觉材料 |
| **幻觉论证** | 支持表征主义 |

**Moore的手论证**：

1. 这是一只手（举起手）
2. 这是另一只手（举起另一只手）
3. 因此，存在外部事物

**争议**：
- Moore的论证是否"偷运"了外部世界假设？
- 这是常识论证还是循环论证？

**相关词条**：感知、Moore

---

### 【Moore的手论证】(Moore's Proof)

**定义**：G.E. Moore为外部世界存在提供的证明。

**费曼解释**：
Moore说：
- "这是一只手"
- "这是另一只手"
- "因此，存在外部事物"

**论证结构**：

1. 前提1：这是一只手（举起右手）
2. 前提2：这是另一只手（举起左手）
3. 结论：存在至少两个外部事物

**对Moore的批评**：

| 批评 | 描述 |
|------|------|
| **循环** | 前提已经假设外部世界 |
| **偷运结论** | 结论已经被前提包含 |
| **没有回答怀疑论** | 没有回应怀疑论的挑战 |

**Moore的辩护**：

- 这是一个有效的论证
- 前提是我确实知道的
- 怀疑论的结论比常识更可疑

**代表人物**：
- **G.E. Moore**：常识哲学

**经典文献**：
- Moore, "Proof of an External World" (1939)

**相关词条**：感知、常识哲学

### 【感知经验】(Perceptual Experience)

**定义**：感知的主观体验方面。

**费曼解释**：
感知经验是：
