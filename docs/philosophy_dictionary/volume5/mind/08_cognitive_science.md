### 【认知科学】(Cognitive Science)

**定义**：研究心智和智能的跨学科领域。

**费曼解释**：
认知科学是：
- 心理学 + 计算机科学 + 神经科学
- 哲学 + 语言学 + 人类学
- 研究心智如何工作

**核心学科**：

| 学科 | 视角 |
|------|------|
| **心理学** | 行为和认知过程 |
| **计算机科学** | 计算模型 |
| **神经科学** | 神经基础 |
| **哲学** | 概念问题 |
| **语言学** | 语言认知 |
| **人类学** | 文化认知 |

**认知科学的历史**：

| 时期 | 发展 |
|------|------|
| **1950s** | 认知革命 |
| **1960s** | 信息处理模型 |
| **1980s** | 联结主义兴起 |
| **2000s** | 神经科学整合 |
| **2010s** | 深度学习革命 |
| **2020s** | 4E认知发展 |

**核心假设**：

| 假设 | 描述 |
|------|------|
| **计算假设** | 心智是计算系统 |
| **表征假设** | 心智使用表征 |
| **功能主义** | 功能定义心理状态 |

**研究主题**：

| 主题 | 描述 |
|------|------|
| **感知** | 如何感知世界 |
| **注意** | 如何选择信息 |
| **记忆** | 如何存储信息 |
| **语言** | 如何处理语言 |
| **推理** | 如何推理 |
| **决策** | 如何决策 |

**代表人物**：
- **George Miller**：认知科学奠基人
- **Noam Chomsky**：语言学革命
- **Herbert Simon**：认知架构
- **David Marr**：视觉计算理论

**经典文献**：
- Miller, "The Magical Number Seven" (1956)
- Chomsky, *Syntactic Structures* (1957)
- Marr, *Vision* (1982)

**当代发展 (2020-2026)**：
- **预测处理**：认知作为预测
- **具身认知**：身体在认知中的角色
- **AI整合**：AI与认知科学

### 【Marr的层次】(Marr's Levels)

**定义**：David Marr提出的分析信息处理系统的三个层次。

**费曼解释**：
Marr的层次说：
- 要理解一个系统，需要从三个层次分析
- 计算理论：做什么？
- 算法：怎么做？
- 实现：用什么？

**三个层次**：

| 层次 | 问题 | 描述 |
|------|------|------|
| **计算理论** | 做什么？ | 计算目标是什么 |
| **表征与算法** | 怎么做？ | 如何实现 |
| **硬件实现** | 用什么？ | 物理实现 |

**例子：收银机**：

| 层次 | 描述 |
|------|------|
| **计算理论** | 加法（目标） |
| **算法** | 十进制加法（方法） |
| **实现** | 齿轮/电子（物理） |

**哲学意义**：

| 意义 | 描述 |
|------|------|
| **多层次分析** | 需要多层次分析 |
| **独立性** | 层次相对独立 |
| **科学方法** | 提供科学方法 |

**代表人物**：
- **David Marr**：视觉计算理论

**经典文献**：
- Marr, *Vision* (1982)

**相关词条**：认知科学、视觉

| **意识** | 无意识 | 有意识 |
| **努力** | 无需努力 | 需要努力 |
| **进化** | 古老 | 较新 |

**例子**：

| 任务 | 系统1 | 系统2 |
|------|-------|-------|
| **1+1=?** | ✓ | |
| **27×43=?** | | ✓ |
| **直觉反应** | ✓ | |
| **逻辑推理** | | ✓ |

**哲学意义**：

| 意义 | 描述 |
|------|------|
| **理性问题** | 人类理性的局限 |
| **偏见** | 认知偏见的来源 |
| **自由意志** | 与自由意志的关系 |

### 【4E认知】(4E Cognition)

**定义**：认知是具身的、嵌入的、延伸的、生成的。

**费曼解释**：
4E认知说：
- Embodied（具身）
- Embedded（嵌入）
- Extended（延伸）
- Enacted（生成）

**四个E**：

| E | 描述 |
|---|------|
| **Embodied** | 认知依赖身体 |
| **Embedded** | 认知嵌入环境 |
| **Extended** | 认知延伸到环境 |
| **Enacted** | 认知通过行动生成 |

**具身 (Embodied)**：
- 认知依赖身体
- 身体塑造认知
- 身体不是"容器"

**嵌入 (Embedded)**：
- 认知嵌入环境
- 环境是认知的一部分
- 不是孤立的

**延伸 (Extended)**：
- 认知延伸到环境
- 工具是认知的一部分
- 如：笔记本是"记忆"

**生成 (Enacted)**：
- 认知通过行动生成
- 感知者参与构造感知
- 不是被动接收

**哲学意义**：

| 意义 | 描述 |
|------|------|
| **挑战计算主义** | 挑战传统计算主义 |
| **扩展认知边界** | 扩展认知的边界 |
| **跨学科整合** | 整合多学科视角 |

**代表人物**：
- **Andy Clark**：延伸心灵
- **Evan Thompson**：生成认知
- **Alva Noë**：具身感知

**经典文献**：
- Clark & Chalmers, "The Extended Mind" (1998)
- Varela et al., *The Embodied Mind* (1991)

**相关词条**：具身认知、延伸心灵

### 【延伸心灵】(Extended Mind)

**定义**：Clark和Chalmers提出的认知延伸到环境中的理论。

**费曼解释**：
延伸心灵说：
- 认知不只是在大脑中
- 可以延伸到环境中
- 如笔记本是"记忆"的一部分

**Otto案例**：

**情境**：
- Otto有记忆障碍
- 用笔记本记东西
- 笔记本功能等同记忆
- 因此，笔记本是"心灵"的一部分

**Parity原则**：
- 如果外部过程完成同样功能
- 且内部完成会被认为是认知
- 那外部过程也应被认为是认知

**支持论证**：

| 论证 | 描述 |
|------|------|
| **功能等同** | 外部工具功能等同内部过程 |
| **认知整合** | 外部工具被整合到认知中 |
| **动态交互** | 与外部工具动态交互 |

**对延伸心灵的批评**：

| 批评 | 描述 |
|------|------|
| **认知膨胀** | 把什么都算认知？ |
| **个体边界** | 个体边界在哪里？ |
| **神经依赖** | 认知最终依赖大脑 |

**代表人物**：
- **Andy Clark**：延伸心灵提出者
- **David Chalmers**：共同提出者

**经典文献**：
- Clark & Chalmers, "The Extended Mind" (1998)

**相关词条**：4E认知、外在主义

### 【预测处理】(Predictive Processing)

**定义**：认为认知是预测过程的当代理论。

**费曼解释**：
预测处理说：
- 大脑不断预测感官输入
- 感知是预测和实际输入的比较
- 预测误差驱动更新

**核心机制**：

| 机制 | 描述 |
|------|------|
| **预测** | 大脑预测感官输入 |
| **比较** | 比较预测和实际输入 |
| **误差** | 计算预测误差 |
| **更新** | 预测误差驱动更新 |

**与传统观点的区别**：

| 问题 | 传统观点 | 预测处理 |
|------|----------|----------|
| 感知方向 | 自下而上 | 自上而下 |
| 大脑功能 | 处理输入 | 预测输入 |
| 意识 | 处理结果 | 预测 |

**自由能原理**：

**Friston的理论**：
- 大脑最小化自由能
- 自由能≈预测误差
- 最小化预测误差

**哲学意义**：

| 意义 | 描述 |
|------|------|
| **统一理论** | 统一感知、行动、学习 |
| **具身兼容** | 与具身认知兼容 |
| **意识理论** | 意识的新解释 |

**代表人物**：
- **Karl Friston**：自由能原理
- **Andy Clark**：预测处理

**经典文献**：
- Friston, "The Free-Energy Principle" (2010)
- Clark, *Surfing Uncertainty* (2016)

**相关词条**：认知科学、贝叶斯大脑

### 【贝叶斯大脑】(Bayesian Brain)

**定义**：认为大脑是贝叶斯推理机器的理论。

**费曼解释**：
贝叶斯大脑说：
- 大脑进行贝叶斯推理
- 结合先验和证据
- 更新信念

**贝叶斯定理**：

P(H|E) = P(E|H) × P(H) / P(E)

- P(H|E)：后验概率
- P(E|H)：似然
- P(H)：先验
- P(E)：证据

**核心主张**：

| 要素 | 描述 |
|------|------|
| **先验** | 大脑有先验信念 |
| **证据** | 感官提供证据 |
| **后验** | 更新后的信念 |

**应用领域**：

| 领域 | 应用 |
|------|------|
| **感知** | 感知的贝叶斯解释 |
| **学习** | 学习的贝叶斯解释 |
| **决策** | 决策的贝叶斯解释 |

**相关词条**：预测处理、认知科学

### 【模块性】(Modularity)

**定义**：认为心智由专门模块组成的观点。

**费曼解释**：
模块性说：
- 心智不是统一的
- 而是由专门模块组成
- 每个模块专门处理某种任务

**Fodor的模块特征**：

| 特征 | 描述 |
|------|------|
| **领域特异性** | 只处理特定领域信息 |
| **强制性** | 自动运行 |
| **有限可及** | 内部过程不可及 |
| **快速** | 快速处理 |
| **信息封装** | 封装的 |
| **固定神经结构** | 固定神经基础 |

**模块的例子**：

| 模块 | 功能 |
|------|------|
| **面孔识别** | 识别面孔 |
| **语言模块** | 处理语言 |
| **心理理论** | 理解他人心智 |

**对模块性的批评**：

| 批评 | 描述 |
|------|------|
| **过度专门化** | 过度专门化？ |
| **交互** | 模块如何交互？ |
| **神经证据** | 神经证据不支持？ |

**代表人物**：
- **Jerry Fodor**：模块性理论
- **Steven Pinker**：模块性支持者

**经典文献**：
- Fodor, *The Modularity of Mind* (1983)

**相关词条**：认知科学、进化心理学

### 【先天 vs 后天】(Nature vs Nurture)

**定义**：认知能力是先天的还是后天获得的争论。

**费曼解释**：
先天vs后天问：
- 语言是先天的还是学习的？
- 智力是遗传的还是环境的？
- 先天和后天如何交互？

**立场**：

| 立场 | 描述 |
|------|------|
| **先天论** | 主要由先天决定 |
| **经验论** | 主要由经验决定 |
| **交互论** | 先天和后天交互 |

**语言先天论**：

**Chomsky的观点**：
- 儿童快速学会语言
- 输入不足但能学会
- 因此有先天语言能力

**通用语法**：
- 所有语言共享的结构
- 先天的语言知识

**对先天论的批评**：

| 批评 | 描述 |
|------|------|
| **统计学习** | 也许可以通过统计学习 |
| **文化差异** | 文化影响很大 |
| **神经可塑性** | 大脑可塑性强 |

**代表人物**：
- **Noam Chomsky**：语言先天论
- **Steven Pinker**：先天论支持者

**相关词条**：语言、模块性

---

### 【语言天赋论】(Nativism about Language)

**定义**：认为语言能力是先天的理论。

**费曼解释**：
语言天赋论说：
- 婴儿生来就有语言能力
- 环境只是触发
- 不是"白板"

**Chomsky的论证**：

| 论证 | 描述 |
|------|------|
| **刺激贫乏** | 语言输入不足以学习 |
| **普遍性** | 所有儿童都能学会 |
| **速度** | 快速学会 |
| **错误类型** | 犯特定类型的错误 |

**通用语法**：

| 要素 | 描述 |
|------|------|
| **普遍原则** | 所有语言共享的原则 |
| **参数** | 语言间变化的参数 |
| **先天** | 先天的语言知识 |

**对语言天赋论的批评**：

| 批评 | 描述 |
|------|------|
| **统计学习** | 统计学习可能足够 |
| **社会交互** | 社会交互的作用 |
| **神经证据** | 神经证据不支持 |

**代表人物**：
- **Noam Chomsky**：语言天赋论创始人

**经典文献**：
- Chomsky, *Syntactic Structures* (1957)

**相关词条**：先天vs后天、语言

### 【认知发展】(Cognitive Development)

**定义**：认知能力在个体发展中的变化。

**费曼解释**：
认知发展是：
- 儿童如何学会思考
- 认知能力如何发展
- 从简单到复杂

**Piaget的阶段理论**：

| 阶段 | 年龄 | 特征 |
|------|------|------|
| **感知运动** | 0-2岁 | 感觉和动作 |
| **前运算** | 2-7岁 | 符号思维 |
| **具体运算** | 7-11岁 | 逻辑思维 |
| **形式运算** | 11岁+ | 抽象思维 |

**核心概念**：

| 概念 | 描述 |
|------|------|
| **图式** | 认知结构 |
| **同化** | 整合新信息到图式 |
| **顺应** | 修改图式适应新信息 |

**对Piaget的批评**：

| 批评 | 描述 |
|------|------|
| **年龄不准确** | 发展可能更早 |
| **阶段太刚性** | 发展可能更连续 |
| **忽视社会** | 忽视社会因素 |

**Vygotsky的社会文化理论**：

**核心主张**：
- 认知发展依赖社会交互
- 语言是认知发展的工具
- 最近发展区

**代表人物**：
- **Jean Piaget**：认知发展阶段论
- **Lev Vygotsky**：社会文化理论

**经典文献**：
- Piaget, *The Psychology of the Child* (1966)
- Vygotsky, *Mind in Society* (1978)

**相关词条**：认知科学、发展心理学

---

### 【心理理论】(Theory of Mind)

**定义**：理解他人有心理状态的能力。

**费曼解释**：
心理理论是：
- 理解他人有信念、欲望
- 理解他人可能和自己不同
- "读心"能力

**发展时间线**：

| 年龄 | 能力 |
|------|------|
| **6个月** | 区分有生命和无生命 |
| **18个月** | 理解欲望 |
| **3-4岁** | 错误信念任务 |
| **5-6岁** | 二阶错误信念 |

**错误信念任务**：

**Sally-Anne任务**：
- Sally把球放在篮子里
- Sally离开
- Anne把球移到盒子里
- Sally回来会找哪里？

**结果**：
- 3岁儿童：说盒子（实际位置）
- 4岁+儿童：说篮子（Sally的信念）

**心理理论的解释**：

| 理论 | 描述 |
|------|------|
| **理论论** | 儿童发展理论 |
| **模拟论** | 模拟他人心理 |
| **模块论** | 先天模块 |

**自闭症与心理理论**：
- 自闭症患者心理理论受损
- 难以理解他人心理
- 支持心理理论是专门模块

**代表人物**：
- **Simon Baron-Cohen**：自闭症研究
- **Alan Leslie**：假装游戏研究

**经典文献**：
- Baron-Cohen et al., "Does the autistic child have a 'theory of mind'?" (1985)

**相关词条**：社会认知、模块性

---

### 【进化心理学】(Evolutionary Psychology)

**定义**：用进化论解释心理特征的学科。

**费曼解释**：
进化心理学说：
- 心智是进化的产物
- 心理特征有进化功能
- 用进化解释心理

**核心主张**：

| 要素 | 描述 |
|------|------|
| **进化产物** | 心智是进化的产物 |
| **适应性** | 心理特征是适应性的 |
| **石器心智** | 我们的心智适应石器时代 |

**研究主题**：

| 主题 | 进化解释 |
|------|----------|
| **配偶选择** | 繁殖成功的策略 |
| **亲代投资** | 投资于后代 |
| **利他主义** | 亲属选择、互惠利他 |
| **恐惧** | 避免危险 |

**对进化心理学的批评**：

| 批评 | 描述 |
|------|------|
| **适应性故事** | 容易编造适应性故事 |
| **基因决定论** | 基因决定论倾向 |
| **忽视文化** | 忽视文化影响 |

**代表人物**：
- **Leda Cosmides**：进化心理学创始人
- **John Tooby**：进化心理学创始人
- **Steven Pinker**：进化心理学支持者

**经典文献**：
- Cosmides & Tooby, "Evolutionary Psychology" (1997)

**相关词条**：模块性、先天vs后天

### 【认知偏见】(Cognitive Biases)

**定义**：系统性的认知错误或偏差。

**费曼解释**：
认知偏见是：
- 我们思考的系统性错误
- 不是随机错误
- 有规律可循

**常见偏见**：

| 偏见 | 描述 |
|------|------|
| **确认偏见** | 寻找确认已有信念的证据 |
| **可得性偏见** | 高估容易想到的事情 |
| **锚定效应** | 受初始信息影响 |
| **损失厌恶** | 损失比收益更重要 |
| **后见之明** | 事后觉得"早就知道" |

**偏见的解释**：

| 解释 | 描述 |
|------|------|
| **启发式** | 偏见是启发式的副作用 |
| **进化** | 偏见曾是适应性的 |
| **有限理性** | 有限理性的结果 |

**哲学意义**：

| 意义 | 描述 |
|------|------|
| **理性问题** | 人类理性的局限 |
| **知识问题** | 偏见如何影响知识 |
| **决策** | 如何做出更好决策 |

**代表人物**：
- **Daniel Kahneman**：偏见研究
- **Amos Tversky**：偏见研究

**经典文献**：
- Kahneman & Tversky, *Judgment under Uncertainty* (1982)

**相关词条**：双系统理论、推理

---

### 【启发式】(Heuristics)

**定义**：简化的决策规则。

**费曼解释**：
启发式是：
- 快速决策的捷径
- 不总是最优
- 但通常足够好

**常见启发式**：

| 启发式 | 描述 |
|--------|------|
| **可得性** | 用容易想到的判断频率 |
| **代表性** | 用相似性判断概率 |
| **锚定** | 用初始值作为起点 |

**启发式 vs 算法**：

| 特征 | 启发式 | 算法 |
|------|--------|------|
| **速度** | 快 | 慢 |
| **准确性** | 有时错误 | 准确 |
| **计算成本** | 低 | 高 |

**Gigerenzer的观点**：

**核心主张**：
- 启发式不是"次优"
- 而是"快且节俭"
- 在某些环境中优于复杂算法

**代表人物**：
- **Daniel Kahneman**：启发式研究
- **Gerd Gigerenzer**：快且节俭启发式

**经典文献**：
- Gigerenzer et al., *Simple Heuristics That Make Us Smart* (1999)

**相关词条**：认知偏见、决策

---

### 【有限理性】(Bounded Rationality)

**定义**：考虑到认知限制的理性概念。

**费曼解释**：
有限理性说：
- 人类不是完全理性的
- 认知资源有限
- 追求"满意"而非"最优"

**vs 完全理性**：

| 问题 | 完全理性 | 有限理性 |
|------|----------|----------|
| 信息 | 完全信息 | 有限信息 |
| 计算 | 无限计算 | 有限计算 |
| 目标 | 最优化 | 满意化 |

**Herbert Simon的贡献**：

**核心概念**：
- 有限理性
- 满意化
- 启发式

**代表人物**：
- **Herbert Simon**：有限理性提出者

**经典文献**：
- Simon, *Models of Man* (1957)

**相关词条**：决策、启发式

---

### 【动力系统理论】(Dynamical Systems Theory)

**定义**：用动力系统数学描述认知的理论。

**费曼解释**：
动力系统理论说：
- 认知是动态过程
- 用微分方程描述
- 强调时间性和连续性

**核心概念**：

| 概念 | 描述 |
|------|------|
| **状态空间** | 所有可能状态的集合 |
| **轨迹** | 状态在时间中的变化 |
| **吸引子** | 系统趋向的状态 |
| **相变** | 系统的突然变化 |

**对计算主义的挑战**：

| 问题 | 计算主义 | 动力系统 |
|------|----------|----------|
| 表征 | 需要 | 不需要 |
| 离散 | 离散状态 | 连续状态 |
| 时间 | 静态 | 动态 |

**代表人物**：
- **Tim van Gelder**：动力系统倡导者
- **Esther Thelen**：发展动力系统

**经典文献**：
- van Gelder, "What Might Cognition Be, If Not Computation?" (1995)

**相关词条**：认知科学、4E认知

---

### 【认知语言学】(Cognitive Linguistics）

**定义**：基于认知的语言学理论。

**费曼解释**：
认知语言学说：
- 语言不是独立模块
- 基于一般认知能力
- 概念隐喻是核心

**核心主张**：

| 要素 | 描述 |
|------|------|
| **非模块** | 语言不是独立模块 |
| **基于认知** | 基于一般认知能力 |
| **隐喻核心** | 概念隐喻是核心 |

**概念隐喻**：

| 隐喻 | 描述 |
|------|------|
| **时间是金钱** | "花时间"、"省时间" |
| **争论是战争** | "攻击观点"、"防御" |
| **爱是旅程** | "关系到了十字路口" |

**代表人物**：
- **George Lakoff**：认知语言学创始人
- **Mark Johnson**：概念隐喻

**经典文献**：
- Lakoff & Johnson, *Metaphors We Live By* (1980)

**相关词条**：语言、具身认知

### 【分布式认知】(Distributed Cognition)

**定义**：认知分布在人、工具和环境中的观点。

**费曼解释**：
分布式认知说：
- 认知不只是"在头脑中"
- 分布在人与人之间
- 分布在人与工具之间

**核心主张**：

| 要素 | 描述 |
|------|------|
| **分布** | 认知分布在多个主体 |
| **工具** | 工具是认知的一部分 |
| **协作** | 协作认知 |

**例子**：

| 例子 | 描述 |
|------|------|
| **驾驶舱** | 飞行员、仪表、程序共同认知 |
| **科学团队** | 团队协作认知 |
| **网络搜索** | 人+搜索引擎共同认知 |

**代表人物**：
- **Ed Hutchins**：分布式认知

**经典文献**：
- Hutchins, *Cognition in the Wild* (1995)

**相关词条**：延伸心灵、4E认知
