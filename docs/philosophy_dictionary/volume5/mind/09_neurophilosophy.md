### 【神经哲学】(Neurophilosophy)

**定义**：结合神经科学和哲学研究心智的跨学科领域。

**费曼解释**：
神经哲学是：
- 哲学问题 + 神经科学方法
- 用神经科学回答哲学问题
- 如：意识、自由意志、道德

**核心问题**：

| 问题 | 描述 |
|------|------|
| **心身问题** | 心灵与大脑的关系？ |
| **意识** | 意识的神经基础？ |
| **自由意志** | 自由意志的神经基础？ |
| **道德** | 道德的神经基础？ |

**方法**：

| 方法 | 描述 |
|------|------|
| **概念分析** | 哲学概念分析 |
| **神经科学** | 神经科学实验 |
| **整合** | 两者整合 |

**与心灵哲学的关系**：

| 关系 | 描述 |
|------|------|
| **互补** | 哲学和神经科学互补 |
| **经验约束** | 神经科学约束哲学 |
| **概念澄清** | 哲学澄清神经科学概念 |

**代表人物**：
- **Patricia Churchland**：神经哲学创始人
- **Paul Churchland**：取消主义
- **Antonio Damasio**：情绪与意识
- **Joseph LeDoux**：情绪神经科学

**经典文献**：
- P. Churchland, *Neurophilosophy* (1986)
- Damasio, *Descartes' Error* (1994)

**当代发展 (2020-2026)**：
- **神经伦理学**：道德的神经基础
- **神经美学**：美学的神经基础
- **神经法律**：法律的神经基础

### 【意识的神经相关物】(Neural Correlates of Consciousness, NCC)

**定义**：与特定意识状态最密切相关的神经事件和结构。

**费曼解释**：
NCC问：
- 当你看到红色时，大脑发生了什么？
- 哪些神经活动与"红色感"相关？
- 找到这些，我们就找到了NCC

**NCC的定义**：

| 要素 | 描述 |
|------|------|
| **最小充分** | 足以产生意识的最低神经机制 |
| **直接相关** | 与意识状态直接相关 |
| **可识别** | 可以通过实验识别 |

**研究方法**：

| 方法 | 描述 |
|------|------|
| **对比分析** | 比较有意识vs无意识条件 |
| **报告范式** | 让被试报告意识内容 |
| **脑成像** | fMRI、EEG等 |

**发现的NCC候选**：

| 候选区域 | 相关意识 | 证据 |
|----------|----------|------|
| **前额叶** | 取用意识 | 工作记忆 |
| **后皮层** | 视觉意识 | 视觉感知 |
| **丘脑** | 一般意识 | 意识调节 |
| **CL意识层** | 整合意识 | 整合信息 |

**意识的"标志"**：

| 标志 | 描述 |
|------|------|
| **P3b波** | 意识决策后300ms |
| **伽马同步** | 40Hz振荡 |
| **全脑整合** | 长程连接 |

**NCC的问题**：

| 问题 | 描述 |
|------|------|
| **相关性vs因果性** | NCC是原因还是结果？ |
| **充分性** | NCC足以产生意识吗？ |
| **普遍性** | NCC是普遍的吗？ |
| **简单问题vs困难问题** | NCC只解决简单问题？ |

**代表人物**：
- **Christof Koch**：NCC研究先驱
- **Francis Crick**：共同提出者
- **Stanislas Dehaene**：全局工作空间

**经典文献**：

### 【全局工作空间理论】(Global Workspace Theory, GWT)

**定义**：Baars和Dehaene提出的意识理论，认为意识是全局信息广播。

**费曼解释**：
全局工作空间理论说：
- 大脑有很多专门模块（无意识）
- 当信息被"广播"到全局工作空间
- 它就变成"有意识的"

就像剧院舞台上的聚光灯。

**核心要素**：

| 要素 | 描述 |
|------|------|
| **专门模块** | 无意识的专门处理器 |
| **全局工作空间** | 信息广播的中心 |
| **注意** | 选择信息进入工作空间 |
| **广播** | 信息被广播到所有模块 |

**神经基础**：

| 结构 | 功能 |
|------|------|
| **前额叶** | 全局工作空间 |
| **顶叶** | 注意控制 |
| **丘脑** | 信息整合 |
| **长程连接** | 全局广播 |

**关键预测**：

| 预测 | 描述 |
|------|------|
| **容量限制** | 有意识处理容量有限 |
| **全或无** | 信息要么完全有意识，要么完全无 |
| **广播** | 有意识信息被全局广播 |
| **报告** | 有意识信息可报告 |

**支持证据**：

| 证据 | 描述 |
|------|------|
| **注意瞬脱** | 注意资源有限 |
| **掩蔽效应** | 信息可能无意识 |
| **神经证据** | 前额叶激活与意识相关 |

**代表人物**：
- **Bernard Baars**：理论创始人
- **Stanislas Dehaene**：神经版本

**经典文献**：
- Baars, *A Cognitive Theory of Consciousness* (1988)
- Dehaene, *Consciousness and the Brain* (2014)

**相关词条**：NCC、意识

### 【整合信息理论】(Integrated Information Theory, IIT)

**定义**：Tononi提出的意识理论，认为意识等于整合信息（Φ）。

**费曼解释**：
整合信息理论说：
- 意识 = 信息整合程度
- 越整合的系统，越有意识
- Φ（Phi）衡量整合程度

**核心公理**：

| 公理 | 描述 |
|------|------|
| **内在存在** | 意识从内在存在 |
| **构成** | 意识有结构 |
| **信息** | 意识是信息 |
| **整合** | 意识是整合的 |
| **排他** | 意识是唯一的 |

**Φ（Phi）的计算**：

**简化版**：
- Φ = 信息量 - 部分信息之和
- 如果系统可分割，Φ低
- 如果系统不可分割，Φ高

**直觉**：
- Φ高：大脑（高度整合）
- Φ低：硬盘（信息多但不整合）
- Φ零：随机噪声

**IIT的应用**：

| 应用 | 描述 |
|------|------|
| **意识测量** | 用Φ测量意识水平 |
| **临床诊断** | 评估意识障碍 |
| **AI意识** | 评估AI的意识 |

**对IIT的批评**：

| 批评 | 描述 |
|------|------|
| **计算困难** | Φ难以计算 |
| **反直觉结果** | 某些简单系统有高Φ？ |
| **意识定义** | 是否正确捕获意识？ |
| **困难问题** | 是否解决困难问题？ |

**代表人物**：
- **Giulio Tononi**：理论创始人
- **Christof Koch**：支持者

**经典文献**：
- Tononi, "Consciousness as Integrated Information" (2008)
- Tononi, *Phi: A Voyage from the Brain to the Soul* (2012)

**相关词条**：意识、NCC

### 【取消主义】(Eliminativism）

**定义**：认为"信念"、"欲望"等常识心理学概念是错误的理论术语，应该被取消的哲学立场。

**费曼解释**：
取消主义说：
- "信念"、"欲望"像"燃素"、"以太"
- 是错误的理论概念
- 神经科学将取代常识心理学

**核心主张**：

| 要素 | 描述 |
|------|------|
| **常识心理学是理论** | 民间心理学是经验理论 |
| **常识心理学是错的** | 这个理论是错误的 |
| **应该取消** | 应该取消其概念 |
| **神经科学替代** | 神经科学将取代它 |

**取消主义 vs 还原主义**：

| 问题 | 还原主义 | 取消主义 |
|------|----------|----------|
| 常识心理学 | 真的（可还原） | 假的（应取消） |
| "信念" | 真实存在 | 不存在 |
| 未来 | 保留概念 | 取消概念 |

**支持取消主义的论证**：

| 论证 | 描述 |
|------|------|
| **理论失败** | 常识心理学在解释精神疾病等方面失败 |
| **科学发展** | 科学史上有概念被取消 |
| **神经科学进展** | 神经科学提供更好理论 |

**对取消主义的批评**：

| 批评 | 描述 |
|------|------|
| **自我反驳** | 取消主义者"相信"取消主义吗？ |
| **直觉问题** | 我们确实有信念！ |
| **实践问题** | 取消后如何理解人类行为？ |

**代表人物**：
- **Paul Churchland**：取消主义创始人
- **Patricia Churchland**：神经哲学

**经典文献**：
- P. Churchland, "Eliminative Materialism" (1981)
- Churchlands, *Matter and Consciousness* (1984)

**相关词条**：神经哲学、常识心理学

### 【自由意志的神经科学】(Neuroscience of Free Will)

**定义**：研究自由意志的神经基础。

**费曼解释**：
自由意志的神经科学问：
- 大脑如何做出决定？
- 有意识决定在神经层面如何发生？
- 我们有神经自由意志吗？

**Libet实验**：

**方法**：
- 让被试自由选择何时动手指
- 测量大脑准备电位（RP）
- RP在被试报告"决定"之前数百毫秒出现

**结果**：
- 大脑活动先于有意识决定
- 似乎"决定"在意识之前就已经发生

**对自由意志的挑战？**

| 解释 | 描述 |
|------|------|
| **挑战** | 决定是无意识的 |
| **不挑战** | 准备电位不是决定 |
| **否决权** | 意识可以"否决"无意识倾向 |

**后续研究**：

| 研究 | 发现 |
|------|------|
| **Soon et al.** | 决定可提前数秒预测 |
| **fMRI研究** | 更早的神经活动 |
| **批评** | 实验设计的局限性 |

**哲学意义**：

| 意义 | 描述 |
|------|------|
| **挑战直觉** | 挑战我们的直觉 |
| **相容论** | 与相容论兼容？ |
| **责任** | 对道德责任的影响？ |

**代表人物**：
- **Benjamin Libet**：实验设计者
- **Patrick Haggard**：意志神经科学

**经典文献**：
- Libet, "Unconscious Cerebral Initiative" (1983)
- Soon et al., "Unconscious determinants of free decisions" (2008)

**相关词条**：自由意志、Libet实验

- 看见大脑结构
- 测量大脑活动
- 研究大脑功能

**主要技术**：

| 技术 | 描述 | 测量 |
|------|------|------|
| **fMRI** | 功能磁共振 | 血氧水平 |
| **EEG** | 脑电图 | 电活动 |
| **MEG** | 脑磁图 | 磁活动 |
| **PET** | 正电子发射断层扫描 | 代谢 |
| **CT** | 计算机断层扫描 | 结构 |

**fMRI原理**：

| 要素 | 描述 |
|------|------|
| **BOLD信号** | 血氧水平依赖信号 |
| **神经活动** | 神经活动增加血流 |
| **空间分辨率** | 较好 |
| **时间分辨率** | 较差（秒级） |

**EEG原理**：

| 要素 | 描述 |
|------|------|
| **电活动** | 测量头皮电活动 |
| **时间分辨率** | 极好（毫秒级） |
| **空间分辨率** | 较差 |

**哲学问题**：

| 问题 | 描述 |
|------|------|
| **相关性vs因果性** | 影像只显示相关性 |
| **反向推理** | 从激活推断心理状态？ |
| **多重比较** | 统计问题 |

**相关词条**：神经科学、NCC

| 意义 | 描述 |

| **道德地位** | 无意识者的道德地位？ |

**相关词条**：意识、NCC

| 意义 | 描述 |

| **自我** | 裂脑患者的自我？ |
| **心灵哲学** | 对心灵哲学的启示 |

**代表人物**：
- **Roger Sperry**：裂脑研究先驱
- **Michael Gazzaniga**：后续研究者

**经典文献**：
- Sperry, "Lateral Specialization in the Surgically Separated Hemispheres" (1974)

**相关词条**：意识、自我
