## 📝 词条格式说明

每个词条包含：

```
【词条名】(英文名)
[词源/背景]
■ 定义：学术标准定义
■ 费曼解释：通俗易懂的解释
■ 核心要素：关键组成部分（3-5个）
■ 理论流派：不同理论立场
■ 代表人物：重要哲学家及其观点
■ 经典文献：核心文本
■ 当代发展（2020-2026）：最新研究
■ 跨学科联系：与其他学科的关系
■ 中国视角：中国学者的贡献
■ 争议问题：核心争论
■ 相关词条：交叉引用
```
