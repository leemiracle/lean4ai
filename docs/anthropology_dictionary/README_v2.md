## 🌍 人类学的独特性

### 四领域方法 (Four-Field Approach)
美国人类学传统包含四大分支：
1. **社会文化人类学**：研究当代文化和社会
2. **生物/体质人类学**：研究人类生物性和进化
3. **考古学**：研究过去人类通过物质遗存
4. **语言人类学**：研究语言与社会文化

### 核心方法
- **田野工作** (Fieldwork)：长期参与观察
- **民族志** (Ethnography)：深度描述
- **比较方法** (Comparative Method)：跨文化比较
- **整体观** (Holism)：综合视角

### 核心价值观
- **文化相对主义** (Cultural Relativism)：避免民族中心主义
- **反身性** (Reflexivity)：反思研究者立场
- **伦理责任** (Ethics)：尊重研究对象
- **社会正义** (Social Justice)：为边缘群体发声
