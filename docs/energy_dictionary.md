# Part 1: 基础概念 FUNDAMENTALS

## 1.1 能源分类 Energy Classification

**定义 Definition**:
按照来源、可再生性、使用方式等维度对能源进行的系统性分类。

---

### 1.1.1 按来源分类（Classification by Source）

| 类型 Type | 定义 Definition | 例子 Examples | 全球占比(2024) |
|-----------|----------------|---------------|----------------|
| **一次能源 Primary Energy** | 直接从自然界获取，未经转换 | 煤炭、原油、天然气、太阳能、风能、水能 | 100% (基准) |
| **二次能源 Secondary Energy** | 由一次能源转换而来 | 电力、汽油、柴油、氢气 | - |

**费曼解释 Feynman Explanation**:
```
想象能源就像食材：
- 一次能源 = 原材料（小麦、蔬菜、肉类）
  直接从大自然获得，没经过加工
  
- 二次能源 = 加工食品（面包、罐头、熟食）
  用原材料加工而成，更方便使用
  
为什么需要二次能源？
- 小麦 → 面包（更好吃、更易消化）
- 煤炭 → 电力（更清洁、更易传输）
- 天然气 → 氢气（更环保、应用更广）
```

---

### 1.1.2 按可再生性分类（Classification by Renewability）

| 类型 Type | 定义 | 主要能源 | 特点 |
|-----------|------|----------|------|
| **可再生能源 Renewable** | 可在短期内自然补充 | 太阳能、风能、水能、生物质能 | 清洁、可持续、但间歇性 |
| **非可再生能源 Non-renewable** | 形成需要数百万年 | 煤炭、石油、天然气、核能（铀） | 储量有限、但稳定可靠 |
| **清洁能源 Clean Energy** | 低/零碳排放 | 可再生能源 + 核能 | 环境友好 |

**费曼解释**:
```
可再生能源就像"工资"：
- 每个月都会发（太阳每天升起）
- 用完了下月还有（风吹不息）
- 但不是想什么时候用就什么时候用
  （太阳晚上不工作，风小时发电少）

非可再生能源就像"存款"：
- 地球存了3亿年的"积蓄"
- 我们200年就快用完了
- 用完就没有了（或者要等几百万年）
- 但想什么时候用就什么时候用
  （随时可以烧煤发电）
```

**全球能源结构演变 Global Energy Mix Evolution**:

| 2010 | 29.6% | 32.3% | 22.0% | 5.0% | 6.2% | 4.9% |

| 2024 | 26.5% | 29.8% | 23.5% | 4.2% | 6.8% | 9.2% |

**费曼解释**:
```
能量单位就像"货币"：
- 焦耳(J) = 基本单位（像"1分钱"）
- 千瓦时(kWh) = 电费单上的单位（像"1元"）
- 油当量(toe) = 国际贸易单位（像"1美元"）

为什么要换算？
就像用不同货币买东西：
- 中国用电 → 千瓦时
- 美国用气 → BTU
- 国际贸易 → 油当量(toe)

换算率：
1桶原油 ≈ 42加仑 ≈ 159升
      ≈ 5.8×10⁶ BTU
      ≈ 1,700 kWh
      ≈ 0.136 toe
```

## 1.3 能源效率 Energy Efficiency

### 1.3.1 效率定义

**热效率 Thermal Efficiency (η)**:
```
η = 有用能量输出 / 能量输入 × 100%
```

**卡诺效率 Carnot Efficiency (理想最大效率)**:
```
η_carnot = 1 - T_cold / T_hot
```
- T_hot: 热源温度（开尔文）
- T_cold: 冷源温度（开尔文）

### 1.3.2 各类发电效率对比

| 发电技术 | 热效率范围 | 最佳效率 | 备注 |
|----------|-----------|----------|------|
| **超超临界燃煤 Ultra-supercritical Coal** | 42-47% | 47% | 600°C+/30MPa+ |
| **超临界燃煤 Supercritical Coal** | 38-42% | 42% | 540°C/24MPa |
| **亚临界燃煤 Subcritical Coal** | 33-38% | 38% | 传统技术 |
| **天然气联合循环 NGCC** | 55-63% | 63% | GE 9HA, Siemens SGT5-8000H |
| **燃气轮机简单循环 Gas Turbine SC** | 33-42% | 42% | 峰值电站 |
| **柴油发电机 Diesel Generator** | 35-45% | 45% | 分布式电源 |
| **燃料电池 Fuel Cell (PEM)** | 50-60% | 60% | 氢燃料电池 |
| **燃料电池 Fuel Cell (SOFC)** | 55-65% | 65% | 固体氧化物 |
| **太阳能光伏 Solar PV** | 15-25% | 26.7% (单晶硅实验室) | 量产效率18-22% |
| **风力发电 Wind Turbine** | 35-50% | 50% | Betz极限59.3% |

**费曼解释**:
```
效率就像"转化率"：
- 煤电站烧100份煤 → 产生40份电 + 60份废热
  效率 = 40%

为什么不能100%？
- 热力学第二定律：热量不能完全转化为功
- 就像煮饭：不可能所有热量都进入米饭
  一定有热量散失到空气中

卡诺极限（理想情况）：
假设：
- 煤燃烧温度 T_hot = 600°C = 873K
- 冷却水温度 T_cold = 25°C = 298K
- 最大理论效率 = 1 - 298/873 = 65.9%

实际效率低于理论值：
- 热损失（烟囱、散热）
- 机械摩擦
- 电机损耗
```

**热泵的"超100%效率"解释**:
```
热泵COP (Coefficient of Performance) 可以>100%

这不是违反能量守恒！

热泵工作原理：
- 消耗1份电 → 从室外"搬运"3份热 → 室内得到4份热
- COP = 4份输出 / 1份输入 = 400%

类比：
- 电加热器 = 用电"制造"热量（1:1）
- 热泵 = 用电"搬运"热量（1:3-1:5）
- 就像：自己做饭 vs 叫外卖
  热泵不"制造"热量，只是"搬运"已有的热量
```

# Part 2: 化石能源 FOSSIL FUELS

## 2.1 石油 Petroleum

**定义 Definition**:
一种复杂的碳氢化合物混合物，主要由古代海洋生物遗骸经数百万年高温高压形成。

---

### 2.1.1 石油形成与勘探

**形成条件 Formation Conditions**:
| 要素 | 条件 | 时间 |
|------|------|------|
| **物质来源 Source Material** | 海洋浮游生物、藻类 | - |
| **沉积环境 Depositional Environment** | 海洋、湖泊缺氧环境 | - |
| **埋藏深度 Burial Depth** | 2,000-4,000米（生油窗） | 数百万年 |
| **温度 Temperature** | 60-120°C（石油窗） | - |
| **压力 Pressure** | 数百大气压 | - |
| **时间 Time** | 10-100百万年 | - |

**费曼解释**:
```
石油形成就像"高压锅慢炖"：

1. 原料：古代海洋生物（浮游生物、藻类）
   就像放入蔬菜和肉类

2. 环境：海底缺氧 → 防止完全分解
   就像密封锅盖 → 保留营养

3. 过程：沉积物层层覆盖
   每层沉积物 = 1锅盖的压力
   1000米沉积物 = 1000个锅盖！

4. 烹饪时间：几千万到上亿年
   地球是超级慢的高压锅

5. 结果：生物有机质 → 干酪根 → 石油
   就像：蔬菜 → 浓缩汤汁

关键：生油窗（Oil Window）
- 太浅(<1500m)：温度不够，生不成油
- 太深(>4500m)：温度太高，油裂解成气
- 刚好(2000-4000m)：温度60-120°C，最佳生油
```

|-------------|-------------|-----------|--------------|
| **常压蒸馏 Atmospheric Distillation** | 初步分离 | 原油 | LPG, 汽油, 煤油, 柴油, 渣油 |
| **减压蒸馏 Vacuum Distillation** | 深度分离重油 | 常压渣油 | VGO, 沥青 |
| **催化裂化 FCC** | 重油→轻油 | VGO | 汽油, 柴油, LPG |
| **加氢裂化 Hydrocracking** | 重油→轻油+脱硫 | VGO | 柴油, 航空煤油 |
| **催化重整 Catalytic Reforming** | 提高辛烷值 | 石脑油 | 高辛烷值汽油, 芳烃 |
| **烷基化 Alkylation** | 提高辛烷值 | 异丁烷+烯烃 | 高辛烷值汽油 |
| **异构化 Isomerization** | 提高辛烷值 | 正构烷烃 | 异构烷烃 |
| **焦化 Coking** | 渣油转化 | 渣油 | 汽油, 柴油, 石油焦 |
| **加氢处理 Hydrotreating** | 脱硫/脱氮 | 馏分油 | 清洁燃料 |
| **延迟焦化 Delayed Coking** | 渣油转化 | 渣油 | 汽油, 柴油, 焦炭 |

**费曼解释**:
```
炼油就像"分馏塔中的电梯游戏"：

想象一个超高的建筑（分馏塔，50米高）：
- 底层最热（400°C）
- 顶层最凉（20°C）

原油从中间进入 → 被加热 → 各种分子乘"电梯"上升

不同分子的"楼层"：
- 气体(LPG)：太轻，飞到顶层
  分子量最小，沸点最低

- 汽油：住在高层（30-200°C）
  小分子，开车最喜欢

- 柴油：住在中层（200-350°C）
  中等分子，卡车用

- 重油/渣油：住在底层（>350°C）
  大分子，太重上不去

为什么需要二次加工？
- 原油含30%汽油 → 我们需要50%
- 解决方案：把"底层住户"拆分
  - 催化裂化：把大分子"砸碎"
  - 加氢处理：加氢去除杂质
  - 重整：重新排列分子结构
```

### 2.1.5 非常规石油 Unconventional Oil

**页岩油 Shale Oil / 致密油 Tight Oil**:

| 项目 | 参数 |
|------|------|
| **定义** | 储存在低渗透率页岩中的原油 |
| **开采技术** | 水平钻井 + 水力压裂 ( fracking ) |
| **主要产地** | 美国Permian, Bakken, Eagle Ford |
| **美国产量 (2024)** | ~9.0百万桶/日 |
| **成本范围** | $35-65/桶 |
| **技术可采储量** | 全球约3,500亿桶 |

**费曼解释页岩油开采**:
```
页岩油开采 = 水平钻 + 压裂

传统油田：
- 石油存在"洞穴"里，像地下湖
- 打一口井，石油自己流出来

页岩油：
- 石油锁在"岩石孔隙"里，像海绵吸水
- 岩石太致密，石油流不出来

解决方案：
1. 水平钻井 Horizontal Drilling
   - 先垂直打下去 → 转弯水平延伸2-3公里
   - 增加接触面积

2. 水力压裂 Hydraulic Fracturing
   - 注入高压水+砂+化学剂
   - 压裂岩石 → 创造裂缝
   - 砂子支撑裂缝 → 石油流出

就像：
- 传统油田 = 用吸管喝饮料
- 页岩油 = 先把杯子砸碎，再喝
```

**油砂 Oil Sands / 沥青砂 Tar Sands**:

| 项目 | 参数 |
|------|------|
| **定义** | 砂、粘土、水、沥青混合物 |
| **主要产地** | 加拿大Alberta, 委内瑞拉Orinoco |
| **开采方式** | 露天开采 (浅层) / SAGD (深层) |
| **加拿大产量 (2024)** | ~3.3百万桶/日 |
| **成本范围** | $45-85/桶 |
| **技术可采储量** | 加拿大约1,700亿桶 |

**SAGD技术 (Steam Assisted Gravity Drainage)**:
```
油砂开采 (深层)：

地面
  ↓
  ↓   ┌──────────┐ 上井：注蒸汽
  ↓   │   ↘      │
  ↓   │    蒸汽腔 │
  ↓   │   ↙      │
  ↓   └──────────┘ 下井：采集沥青
  ↓
油砂层

原理：
- 蒸汽加热 → 沥青变稀
- 重力作用 → 流向下井
- 泵送到地面 → 分离处理
```

## 2.2 天然气 Natural Gas

**定义 Definition**:
主要由甲烷(CH₄)组成的气态化石燃料，通常与石油伴生或独立存在于气田中。

### 2.2.1 天然气组成与分类

**典型组成 Typical Composition**:

| 组分 Component | 化学式 | 含量范围 | 作用 |
|---------------|--------|----------|------|
| **甲烷 Methane** | CH₄ | 70-90% | 主要可燃成分 |

**费曼解释**:
```
天然气形成就像"高压发酵"：

1. 原料：古代生物残骸（和石油一样）
   
2. 环境差异：
   石油：温度60-120°C（生油窗）
   天然气：温度>120°C（生气窗）
   
   就像煮饭：
   - 60-80°C：米变成饭（石油）
   - >120°C：饭烧焦成炭+气体（天然气）
   
3. 天然气来源：
   a) 热裂解：石油继续加热 → 天然气
   b) 煤层气：煤炭生成时释放的气体
   c) 生物气：细菌分解有机物（浅层）
   
4. 常规 vs 非常规：
   常规：储存在多孔岩石中（像海绵吸水）
   非常规：储存在致密岩石中（像混凝土中的气泡）
     - 页岩气：在页岩中
     - 致密气：在致密砂岩中
     - 煤层气：在煤层中
     
   开采非常规气需要"压裂"：
   把岩石"震碎"，让气体释放出来
```

### 2.2.3 天然气处理与运输

**处理流程 Processing Flow**:

```
原料气 (Raw Gas)
    ↓
┌────────────────────┐
│ 脱酸 (Acid Gas Removal) │ ← 胺液吸收、物理溶剂
│ 去除CO₂, H₂S           │
└────────────────────┘
    ↓
┌────────────────────┐
│ 脱水 (Dehydration)     │ ← 乙二醇吸收、分子筛
│ 去除H₂O (防冰堵)      │
└────────────────────┘
    ↓
┌────────────────────┐
│ NGL回收 (NGL Recovery) │ ← 低温分离、透平膨胀
│ 回收C₂+ (乙烷、丙烷等) │
└────────────────────┘
    ↓
商品天然气 (Pipeline Quality Gas)
```

**运输方式 Transportation Methods**:

| 方式 Method | 适用场景 | 优缺点 |
|------------|----------|--------|

## 2.3 煤炭 Coal

**定义 Definition**:
由古代植物遗骸在地质作用下，经数千万年到数亿年的生物化学、物理化学作用形成的固态可燃矿物。

---

### 2.3.1 煤炭形成与分类

**成煤过程 Coal Formation Process**:

```
植物遗骸 (Plant Remains)
    ↓ 泥炭化作用 (Peatification)
    ↓ 数千年
泥炭 (Peat) [水分>75%, 碳含量~60%]
    ↓ 成岩作用 (Diagenesis)
    ↓ 埋藏、压实
褐煤 (Lignite) [水分30-40%, 碳含量65-75%]
    ↓ 变质作用 (Metamorphism)
    ↓ 温度压力升高
    ├─→ 亚烟煤 (Sub-bituminous) [水分15-30%, 碳含量75-85%]
    │
    ├─→ 烟煤 (Bituminous) [水分5-15%, 碳含量85-92%]
    │
    └─→ 无烟煤 (Anthracite) [水分<5%, 碳含量>92%]
```

**费曼解释**:
```
煤炭形成就像"压缩+烘烤"：

1. 原料：远古森林（石炭纪，3亿年前）
   当时地球温暖湿润，植物茂盛

2. 第一步：泥炭化（像堆肥）
   植物倒下 → 沼泽覆盖 → 缺氧分解
   水分75%+，像一堆烂泥

3. 第二步：埋藏压实（像压面饼）
   沉积物覆盖 → 越来越厚
   水分被挤出 → 体积缩小10倍以上

4. 第三步：变质"烘烤"（像烤面包）
   地热+压力 → 化学变化
   - 温度越高，碳含量越高
   - 时间越长，煤质越好
   
   类比：
   - 褐煤：烤了1小时（还软、水分多）
   - 烟煤：烤了5小时（硬度适中、热值高）
   - 无烟煤：烤了10小时（又硬又干、含碳最高）

关键：需要几千万到上亿年！
```

**费曼解释**:
```
光伏电池就像"三明治"结构：

从下到上：
1. 减反射层（玻璃）
   - 像镜子，反射阳光
   - 保护下层结构

2. 发射极（硅片)
   - 像海绵吸水
   - N型硅：电子贫乏
   - P型硅：电子富余
3. P-n结（夹层)
   - 电子想"过马路"
   - 从N区→ P区 → 形成电流
4. 背表面（铝板）
   - 像高速公路
   - 电子在此"行驶"
   - 汇集到负极输出
5. 透明电极（新型）
   - 正面收集电子
   - 替代金属栅线

   - 像透明导电玻璃

为什么效率不是100%？
- 波长匹配问题
  太阳光有各种波长
  硅只吸收部分波长（300-1100nm）
- 光学损失
  反射、遮挡
- 复合损失
  电池有电阻、连接损失
- 热损失
  部分光能变成热量
```

```

```

1. 收集：把秸秆打包
2. 运输：送到生物质电厂
3. 预处理：粉碎/烘干
4. 转化：
   a) 直接燃烧：锅炉烧蒸汽发电（效率20-25%）
   b) 气化发酵：细菌分解 → 沼气（效率40-60%）

| 技术 Technology | 过程 Process | 产品 Product | 效率% | 应用 Application |

| **水热液化 Hydrothermal Liquefaction** | 高压+热水→液化 | 液体燃料 | 40-60 | 实验阶段 |

## 3.5 地热能 Geothermal Energy
**定义 Definition**:

| **高温地热 High-temp** | >150°C | 干蒸汽/过热水 | 蒸汽发电 |
| **干热岩 Hot Dry Rock** | >200°C | 无流体 | 增强型地热系统(EGS) |

| 技术 Technology | 原理 Principle | 适用条件 Conditions | 效率% |
|---------------|-------------|---------------------|--------|
| **干蒸汽 Dry Steam** | 蒸汽直接推动汽轮机 | 蒸汽温度>150°C | 15-20 |
| **闪蒸汽 Flash Steam** | 过热水减压闪蒸成蒸汽 | 过热水资源 | 10-15 |

**增强型地热系统 Enhanced Geothermal Systems (EGS)**:
```
原理：在干热岩中通过水力压裂创造人工储层
流程：
1. 鋻井到干热岩层
2. 注入高压水， 压裂岩石
3. 形成裂隙网络
4. 循环注水， 带出热量
5. 地面发电
优势：资源量巨大，不受地理位置限制
挑战: 成本高、可能诱发地震
```
**费曼解释**:
```
地热能就像"地球的热水袋"：

地球内部：
- 地核温度：6000°C（像太阳表面）
- 越往外温度越低
- 地下几公里处：100-300°C

- 地下水被加热 → 热水/蒸汽

地热发电就像"温泉发电站"：
1. 找到地下"热水袋"
2. 打井下去
3. 抽出热水/蒸汽
4. 蒸汽推动汽轮机
5. 发电

高温地热：
- 像冰岛、新西兰
- 蒸汽直接从地下冒出来
- 推动汽轮机发电

- 效率最高(15-20%)

中低温地热：
- 更常见、但温度较低
- 用"热交换器"（双循环）
- 用低沸点工质（如异丁烷）
- 工质在地下加热 → 推动汽轮机

干热岩：
- 岩石很热，但没水
- 像一个干烧锅
- 解决：注入水，人工制造"温泉"
- 这就是增强型地热系统(EGS)
- 潜力巨大（全球资源够用几千年）
```

**费曼解释**:
```
海洋能就像"大海的呼吸"：

潮汐能：
- 月球引力 → 海水涨落
- 每天2次涨潮 + 2次落潮
- 可预测（不像风/太阳不可预测）
- 想象：水库放水发电
  但这个"水库"每天自动灌水/排水
  
波浪能：
- 风吹海面 → 波浪
- 波浪有能量（见过大浪吧）
- 发电方式：
  浮标随波浪上下运动 → 带动发电机
  像一个不倒翁在波浪上摇摆
  
潮流能：
- 海底也有"风"（洋流）
  潮汐/风力/密度差驱动
- 水下安装"风车"
  
温差能：
- 海面：25°C（被太阳晒热）
- 深海：4°C（冰冷）
- 温差25°C → 用热机发电
- 效率很低，但非常稳定
  
盐差能：
- 河口： 淡水
- 海洋： 咸水
- 盐度差产生渗透压
- 推动涡轮发电
- 还在实验阶段
```

**Part 3 Summary**:
- 太阳能: 光伏(电池/组件/系统)+ 光热(热水/采暖)
- 风能: 陆上(70%) + 海上(30%)，单机2-8 MW主流
- 水能: 常规水电 + 抽水蓄能(削峰填谷)
- 生物质能: 直接燃烧/气化/发酵(沼气/乙醇)
- 地热能: 高温蒸汽 + 中低温双循环 + EGS(干热岩)
- 海洋能: 潮汐(成熟) + 波浪(初期) + 其他(实验)
