## 📐 一、数论基础

### 1. 自然数 (Natural Numbers)
**费曼解释**：你从小就会数数的那些数：0, 1, 2, 3, 4, 5... 就像数苹果、数糖果一样。

**Peano公理**（数学基础）：
```lean
-- 在Lean中定义
inductive Nat where
  | zero : Nat          -- 有个起点0
  | succ : Nat → Nat    -- 每个数后面有个"后继"
```

**重要性质**：
- 加法交换律：a + b = b + a
- 加法结合律：(a + b) + c = a + (b + c)
- 乘法分配律：a × (b + c) = a × b + a × c

**应用**：计数、排列组合、算法分析

---

### 2. 整数 (Integers)
**费曼解释**：自然数加上负数。想象温度计，0度以上是正，以下是负。

**表示**：ℤ = {..., -2, -1, 0, 1, 2, ...}

**结构**：
- 构成一个**环**（Ring）
- 可以加减乘，但不能总除（3÷2不是整数）
- 有序的（可以比较大小）

**关键定理**：
- **除法算法**：对任意a, b（b≠0），存在唯一q, r使得 a = bq + r，其中 0 ≤ r < |b|
- **算术基本定理**：每个整数都能唯一分解为质数乘积（忽略符号和顺序）

---

### 3. 有理数 (Rational Numbers)
**费曼解释**：能写成"分数"形式的数。比如半个披萨是1/2，四分之三个小时是3/4。

**表示**：ℚ = { p/q | p, q ∈ ℤ, q ≠ 0 }

**特点**：
- 构成一个**域**（Field）- 能加减乘除
- 稠密的：任意两个有理数之间还有有理数
- 可数的：能和自然数一一对应

**实现**：
```lean
-- Lean中的有理数
def Rational := { p : ℤ × ℤ // p.2 ≠ 0 }
```

**应用**：精确计算、分数运算

---

### 4. 质数 (Prime Numbers)
**费曼解释**：只能被1和自己整除的大于1的自然数。2, 3, 5, 7, 11, 13, 17, 19, 23...

**重要性**：
- **算术基本定理**：质数是自然数的"原子"
- 密码学（RSA加密）
- 算法研究

**关键定理**：
- **欧几里得定理**：质数有无穷多个
- **质数定理**：π(n) ~ n/ln(n)，其中π(n)是≤n的质数个数
- **哥德巴赫猜想**（未解决）：每个>2的偶数都能写成两个质数之和

**最新进展**（2024-2025）：
- 张益唐的工作被改进：存在无穷多对质数，其差小于246（已优化至更小）
- 量子算法对质因数分解的影响

---

### 5. 最大公约数与最小公倍数 (GCD & LCM)
**费曼解释**：
- GCD：两个数都能整除的最大数。比如GCD(12, 18) = 6
- LCM：能被两个数整除的最小数。比如LCM(12, 18) = 36

**欧几里得算法**：
```
gcd(a, b) = gcd(b, a mod b)  当 b ≠ 0
gcd(a, 0) = a
```

**关键性质**：
- **贝祖等式**：gcd(a, b)存在整数x, y使得 gcd(a,b) = ax + by
- **关系**：gcd(a,b) × lcm(a,b) = |a × b|

**应用**：分数化简、密码学、多项式

---

### 6. 模运算 (Modular Arithmetic)
**费曼解释**：像钟表一样，12点后面是1点，不是13点。就是"绕圈圈"的算术。

**记号**：a ≡ b (mod n) 读作"a与b模n同余"

**例子**：
- 7 ≡ 2 (mod 5) 因为 7 - 2 = 5 能被5整除
- 下午3点 + 5小时 = 晚上8点 → 15 + 5 ≡ 20 (mod 12) ≡ 8 (mod 12)

**重要定理**：
- **费马小定理**：如果p是质数且gcd(a,p)=1，则 a^(p-1) ≡ 1 (mod p)
- **中国剩余定理**：如果gcd(m,n)=1，则 x ≡ a (mod m) 和 x ≡ b (mod n) 有唯一解 x (mod mn)

**应用**：密码学、哈希函数、校验码

---

### 7. 同余与剩余类
**费曼解释**：把所有"余数相同"的数归为一类。模5的话，{0, 5, 10, 15...}都是"0类"。

**结构**：
- ℤ/nℤ = { [0], [1], [2], ..., [n-1] }
- 当n是质数时，ℤ/pℤ 是一个**有限域**

**应用**：
- RSA加密：ℤ/nℤ 上的运算
- 纠错码
- 随机数生成

## 📊 二、代数基础

### 1. 群 (Group)
**费曼解释**：一个集合加上一种运算，满足：
1. 有个"单位元"（加法的0，乘法的1）
2. 每个元素有"逆元"（相反数或倒数）
3. 结合律：(a∘b)∘c = a∘(b∘c)

**例子**：
- 整数加法群 (ℤ, +)：单位元0，逆元-a
- 非零有理数乘法群 (ℚ\{0}, ×)：单位元1，逆元1/a
- 对称群 S₃：三角形的6种对称

**形式化**：
```lean
class Group (G : Type) where
  mul : G → G → G
  one : G
  inv : G → G
  mul_assoc : ∀ a b c, (a * b) * c = a * (b * c)
  one_mul : ∀ a, 1 * a = a
  mul_one : ∀ a, a * 1 = a
  inv_mul_cancel : ∀ a, a⁻¹ * a = 1
```

**应用**：对称性、密码学、物理（粒子物理）

---

### 2. 环 (Ring)
**费曼解释**：一个集合有两种运算（加法和乘法），像整数一样能加减乘（不要求除）。

**公理**：
1. (R, +) 是阿贝尔群
2. 乘法满足结合律
3. 乘法对加法满足分配律

**例子**：
- 整数 ℤ
- 多项式 ℤ[x]
- 矩阵 M_n(ℤ)
- 模n整数 ℤ/nℤ

**特殊类型**：
- **交换环**：乘法可交换（ab = ba）
- **整环**：无零因子的交换环（ab=0 ⟹ a=0 或 b=0）
- **域**：每个非零元素有乘法逆元

---

### 3. 域 (Field)
**费曼解释**：一个集合能加减乘除（除0以外）。像有理数、实数、复数。

**例子**：
- 有理数 ℚ
- 实数 ℝ
- 复数 ℂ
- 有限域 ℤ/pℤ（p是质数）

**性质**：
- 构成一个环
- 每个非零元素有逆元
- 特征为0或质数p

**应用**：
- 线性代数（向量空间在域上定义）
- 密码学（有限域）
- 编码理论

---

### 4. 向量空间 (Vector Space)
**费曼解释**：一组东西能"相加"和"拉伸"（数乘）。像平面上的箭头。

**定义**：V是域F上的向量空间，如果：
1. V是加法群
2. 可以用F的元素"缩放"V的元素
3. 满足分配律等公理

**例子**：
- ℝⁿ：n维实向量空间
- 多项式空间 F[x]
- 函数空间 C[0,1]

**关键概念**：
- **线性组合**：c₁v₁ + c₂v₂ + ... + cₙvₙ
- **线性无关**：只有c₁=c₂=...=0时线性组合为0
- **基**：线性无关且生成整个空间的向量组
- **维数**：基的元素个数

## 🎯 三、逻辑基础

### 1. 命题逻辑 (Propositional Logic)
**费曼解释**：研究"真"和"假"的推理规则。像判断"如果下雨，那么地会湿"。

**基本连接词**：
- ∧ (与/且)：p ∧ q 为真当且仅当 p和q都为真
- ∨ (或)：p ∨ q 为真当且仅当 p或q至少一个为真
- → (蕴含)：p → q 为假当且仅当 p为真且q为假
- ¬ (非)：¬p 为真当且仅当 p为假

**重言式**：
- p ∨ ¬p (排中律)
- p → p (同一律)
- ¬(p ∧ ¬p) (矛盾律)

**Lean实现**：
```lean
example (p q : Prop) : p ∧ q → q ∧ p := by
  intro ⟨hp, hq⟩
  constructor
  · exact hq
  · exact hp
```

---

### 2. 谓词逻辑 (Predicate Logic)
**费曼解释**：在命题逻辑基础上，允许谈论"所有"和"存在"。

**量词**：
- ∀ (全称量词)：∀x, P(x) 表示"对所有x，P(x)成立"
- ∃ (存在量词)：∃x, P(x) 表示"存在一个x使得P(x)成立"

**例子**：
- ∀n ∈ ℕ, n ≥ 0 (所有自然数都≥0)
- ∃n ∈ ℕ, n是质数 ∧ n > 100 (存在大于100的质数)

**推理规则**：
- 全称实例化：从 ∀x, P(x) 推出 P(a)
- 存在实例化：从 ∃x, P(x) 可以假设P(c)成立（c是新的常量）

---

### 3. 证明方法
**费曼解释**：构建可靠论证的技术。

**直接证明**：
```
要证：如果A，那么B
证明：假设A成立，推导...，得出B
```

**反证法**：
```
要证：A成立
证明：假设¬A，推导出矛盾，因此A成立
```

**数学归纳法**：
```
要证：∀n ∈ ℕ, P(n)
步骤：
1. 基础：证明P(0)
2. 归纳：证明P(k) → P(k+1)
```

**构造性证明**：
```
要证：∃x, P(x)
证明：明确给出c，验证P(c)
```

**例子**：
```lean
-- 归纳法示例
example : ∀ n : Nat, 0 + n = n := by
  intro n
  induction n with
  | zero => rfl
  | succ n ih => 
    simp [Nat.add_succ]
    rw [ih]
```

## 🔧 四、集合论基础

### 1. 集合 (Set)
**费曼解释**：一堆东西的集合。像一袋苹果、一个班级的学生。

**表示**：
- 列举法：{1, 2, 3, 4, 5}
- 描述法：{x | x是偶数}
- 区间：(0, 1), [a, b]

**基本运算**：
- 并集：A ∪ B = {x | x ∈ A 或 x ∈ B}
- 交集：A ∩ B = {x | x ∈ A 且 x ∈ B}
- 补集：Aᶜ = {x | x ∉ A}
- 差集：A \ B = A ∩ Bᶜ

**De Morgan定律**：
- (A ∪ B)ᶜ = Aᶜ ∩ Bᶜ
- (A ∩ B)ᶜ = Aᶜ ∪ Bᶜ

---

### 2. 关系 (Relation)
**费曼解释**：集合元素之间的"联系"。比如"小于"、"等于"、"是亲戚"。

**定义**：集合A上的关系R是A × A的子集

**等价关系**（满足三性质）：
1. 自反性：aRa
2. 对称性：aRb ⟹ bRa
3. 传递性：aRb ∧ bRc ⟹ aRc

**例子**：
- "="（等于）
- "≡ (mod n)"（模n同余）
- "~"（相似三角形）

**偏序关系**（满足三性质）：
1. 自反性：a ≤ a
2. 反对称性：a ≤ b ∧ b ≤ a ⟹ a = b
3. 传递性：a ≤ b ∧ b ≤ c ⟹ a ≤ c

---

### 3. 函数 (Function)
**费曼解释**：把一个东西"映射"到另一个东西的规则。像自动售货机：投币→出货。

**定义**：f: A → B 是一个规则，给每个a ∈ A分配唯一的f(a) ∈ B

**类型**：
- **单射**：不同输入→不同输出 (f(a)=f(b) ⟹ a=b)
- **满射**：每个输出都有对应输入 (∀b ∃a, f(a)=b)
- **双射**：既单又满，一一对应

**特殊函数**：
- 恒等函数：id(x) = x
- 常函数：f(x) = c
- 反函数：f⁻¹使得 f⁻¹(f(x)) = x

---

### 4. 基数 (Cardinality)
**费曼解释**：集合"大小"的度量。

**有限集**：{1,2,3}的基数是3

**无限集的惊奇**：
- ℕ和ℤ有相同基数（都是"可数"的）
- ℕ和ℚ有相同基数（有理数可数）
- ℝ的基数比ℕ大（实数不可数）

**记号**：
- |A| = |B|：存在双射A → B
- |A| ≤ |B|：存在单射A → B
- |ℕ| = ℵ₀ (阿列夫零)
- |ℝ| = 𝔠 (连续统)

**康托尔定理**：|A| < |P(A)|（幂集基数更大）

## 🎓 五、证明技术进阶

### 1. 数学归纳法变体
**强归纳法**：
```
要证：∀n, P(n)
证明：
1. 基础：P(0)
2. 归纳：假设∀k < n, P(k)，证明P(n)
```

**结构归纳法**（针对递归结构）：
```lean
example (α : Type) (P : List α → Prop)
    (base : P [])
    (step : ∀ x xs, P xs → P (x :: xs)) :
    ∀ xs, P xs := by
  intro xs
  induction xs with
  | nil => exact base
  | cons x xs ih => exact step x xs ih
```

**良基归纳法**（针对任意良基关系）

### 2. 反例构造
**费曼解释**：找一个例子证明命题不成立。

**例子**：
- 命题："所有连续函数都可导"
- 反例：f(x) = |x| 在x=0不可导

**技术**：
- 找边界情况
- 考虑极端情况

---

### 3. 不动点定理
**Brouwer不动点定理**：连续映射f: D → D（D是闭球）必有不动点x使得f(x)=x

**Banach不动点定理**：完备度量空间上的压缩映射有唯一不动点

**应用**：
- 微分方程解的存在性
- 经济学均衡
- 递归函数定义
