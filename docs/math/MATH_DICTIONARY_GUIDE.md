## 📋 内容规范

### 条目标准格式

每个数学概念/定理包含：

```markdown
#### 定义/定理 X.Y.Z (中文名)
**Definition/Theorem X.Y.Z (English Name)**

[严格定义/陈述]

**性质/推论**:
1. ...
2. ...

**例子**:
- ...

*证明*. [证明思路或完整证明] □

**历史注记**: 
[发展时间线、关键人物]

**跨领域联系**: 
[与其他分支的关系]

**应用**:
[实际应用]

**参考**: 
[文献编号]

**Lean4实现** (如适用):
```lean
[代码]
```
```

### 符号约定

| 符号 | 含义 | 示例 |
|------|------|------|
| ∀ | 全称量词 | ∀x P(x) |
| ∃ | 存在量词 | ∃x P(x) |
| ⟺ | 当且仅当 | P ⟺ Q |
| ⟹ | 蕴含 | P ⟹ Q |
| ∈ | 属于 | x ∈ S |
| ⊆ | 子集 | A ⊆ B |
| ⊨ | 满足/语义蕴含 | M ⊨ φ |
| ⊢ | 语法可证 | Γ ⊢ φ |
| ≅ | 同构 | G ≅ H |
| ≡ | 等价/同余 | a ≡ b (mod n) |
| ↦ | 映射到 | x ↦ f(x) |
| → | 函数/蕴含 | f: A → B |
| ⇒ | 推出 | A ⇒ B |
| □ | 证毕 | - |

### 编号系统

- **定义**: 罗马数字+节号+序号 (如 I.1.1)
- **定理**: 卷号+节号+序号 (如 1.1)
- **例**: 阿拉伯数字
- **注**: 文字说明

## 🎓 使用场景

### 场景1: 学习新概念
1. 在框架文件中找到对应的MSC分类
2. 打开相应的卷和部分
3. 阅读定义和例子
4. 查看相关定理和应用
5. 参考推荐文献深入学习

### 场景2: 查询定理
1. 通过索引或关键词找到定理
2. 查看完整陈述和条件
3. 阅读证明思路
4. 查看推论和应用
5. 追溯原始文献

### 场景3: 跨领域研究
1. 找到核心概念的条目
2. 查看"跨领域联系"部分
3. 了解在其他分支的类比和推广
4. 找到共同的理论基础

### 场景4: Lean4形式化
1. 查看概念的严格定义
2. 阅读Lean4实现代码
3. 理解形式化的难点
4. 参考Mathlib4的相关模块

### 场景5: 论文写作
1. 查找标准术语和符号
2. 引用经典定理的正确表述
3. 查找原始文献引用
4. 确保定义的准确性
