## 📐 一、向量空间深入

### 1. 向量空间回顾与推广
**费曼解释**：向量空间是能"相加"和"数乘"的集合。箭头是直观例子，但多项式、函数也可以是向量。

**公理体系**（域F上的向量空间V）：
1. **加法群**：(V, +) 是阿贝尔群
2. **数乘**：F × V → V
3. **分配律**：α(u + v) = αu + αv, (α + β)v = αv + βv
4. **结合律**：α(βv) = (αβ)v
5. **单位元**：1·v = v

**例子**：
- ℝⁿ：n维实向量
- F[x]：域F上的多项式
- C[a,b]：连续函数空间
- M_m,n(F)：m×n矩阵

**Lean实现**：
```lean
class VectorSpace (F V : Type) [Field F] extends AddCommGroup V where
  smul : F → V → V
  one_smul : ∀ v, (1 : F) • v = v
  mul_smul : ∀ (a b : F) (v : V), (a * b) • v = a • b • v
  smul_add : ∀ (a : F) (u v : V), a • (u + v) = a • u + a • v
  add_smul : ∀ (a b : F) (v : V), (a + b) • v = a • v + b • v
```

---

### 2. 子空间与商空间

#### 子空间 (Subspace)
**费曼解释**：向量空间中的"平面"。满足封闭性：相加、数乘还在里面。

**判定**：W ⊆ V 是子空间 ⟺
1. 0 ∈ W
2. u, v ∈ W ⟹ u + v ∈ W
3. u ∈ W, α ∈ F ⟹ αu ∈ W

**例子**：
- 平面过原点的直线（ℝ²的子空间）
- 齐次线性方程的解空间
- 对称矩阵集合

**生成子空间**：
```
Span{v₁, v₂, ..., v_n} = {α₁v₁ + α₂v₂ + ... + α_nv_n | α_i ∈ F}
```

#### 商空间 (Quotient Space)
**费曼解释**：把子空间"压缩"成一个点，剩下的空间。

**定义**：V/W = {v + W | v ∈ V}

**性质**：
- dim(V/W) = dim V - dim W
- 自然映射 π: V → V/W

---

### 3. 基与维数 (Basis and Dimension)
**费曼解释**：基是"坐标系的轴"。有了基，任何向量都能用坐标表示。

**线性无关**：
```
c₁v₁ + c₂v₂ + ... + c_nv_n = 0 ⟹ c₁ = c₂ = ... = c_n = 0
```

**生成集**：
```
每个v ∈ V 都能写成 v = c₁v₁ + c₂v₂ + ... + c_nv_n
```

**基**：线性无关的生成集

**维数**：基的元素个数（良定义）

**重要定理**：
- **基存在定理**：每个向量空间都有基（需要选择公理）
- **维数定理**：dim(U + W) = dim U + dim W - dim(U ∩ W)
- **基扩充定理**：线性无关集可扩充为基

**Lean实现**：
```lean
def IsBasis (F V : Type) [Field F] [VectorSpace F V] 
    (B : Set V) : Prop :=
  LinearIndependent F B ∧ Span F B = ⊤

def dimension (F V : Type) [Field F] [VectorSpace F V] : Cardinal :=
  Cardinal.mk (basis F V)
```

## 🔢 二、线性映射与矩阵

### 1. 线性映射 (Linear Map)
**费曼解释**：保持"线性结构"的函数。把直线映射成直线，保持平行。

**定义**：T: V → W 满足：
1. T(u + v) = T(u) + T(v)
2. T(αv) = αT(v)

**例子**：
- 旋转、缩放、投影
- 微分算子：D(f) = f'
- 积分算子：T(f)(x) = ∫₀ˣ f(t) dt

**核与像**：
- **核**：Ker(T) = {v ∈ V | T(v) = 0}
- **像**：Im(T) = {T(v) | v ∈ V}

**秩-零度定理**：
```
dim V = dim Ker(T) + dim Im(T)
```

---

### 2. 矩阵表示
**费曼解释**：线性映射的"数字表示"。选定基后，线性映射变成矩阵乘法。

**表示定理**：
给定V的基{e₁,...,e_n}和W的基{f₁,...,f_m}，线性映射T: V → W 对应矩阵A，满足：
```
T(e_j) = Σ_i A_ij f_i
```

**矩阵乘法**：对应线性映射的复合

**坐标变换**：
- 同一线性映射在不同基下有不同矩阵
- A 和 B 是相似矩阵 ⟺ A = P⁻¹BP

**例子**：
```lean
-- 旋转矩阵（90度）
def rotate90 : Matrix (Fin 2) (Fin 2) ℝ :=
  !![0, -1;
      1,  0]

-- 应用到向量
example : rotate90.mulVec ![1, 0] = ![0, 1] := by
  simp [Matrix.mulVec, rotate90]
  decider
```

---

### 3. 矩阵运算与性质

#### 基本运算
- 加法：(A + B)_ij = A_ij + B_ij
- 数乘：(αA)_ij = αA_ij
- 乘法：(AB)_ij = Σ_k A_ik B_kj
- 转置：(A^T)_ij = A_ji

#### 行列式 (Determinant)
**费曼解释**：矩阵"缩放体积"的倍数。正负号表示是否翻转。

**性质**：
- det(AB) = det(A)det(B)
- det(A^T) = det(A)
- det(A⁻¹) = 1/det(A)
- 可逆 ⟺ det ≠ 0

**几何意义**：
- |det A|：线性变换的体积倍数
- sign(det A)：保持/翻转方向

**计算方法**：
- 展开定理（Laplace展开）
- 初等变换
- 特征值乘积

#### 迹 (Trace)
**费曼解释**：特征值的和，或者对角线元素之和。

**性质**：
- tr(A + B) = tr(A) + tr(B)
- tr(αA) = αtr(A)
- tr(AB) = tr(BA)
- tr(A^T) = tr(A)

**应用**：矩阵的"大小"度量

---

### 4. 矩阵分解

#### LU分解
**费曼解释**：把矩阵拆成"上三角"和"下三角"的乘积。

```
A = LU
L：下三角矩阵（主对角线为1）
U：上三角矩阵
```

**应用**：解线性方程组、求行列式

#### QR分解
**费曼解释**：把矩阵拆成"正交"和"上三角"。

```
A = QR
Q：正交矩阵（Q^T Q = I）
R：上三角矩阵
```

**应用**：最小二乘、特征值计算

#### 奇异值分解 (SVD)
**费曼解释**：最强的矩阵分解，把任意矩阵分解成三个特殊矩阵的乘积。

```
A = U Σ V^T
U, V：正交矩阵
Σ：对角矩阵（奇异值）
```

**几何意义**：任何线性变换都能分解为"旋转-拉伸-旋转"

**应用**：
- 数据压缩
- 降维（PCA）
- 推荐系统
- 图像处理

#### 特征值分解
**费曼解释**：把矩阵分解为"特征向量和特征值"。

```
A = PDP^(-1)
P：特征向量组成的矩阵
D：特征值组成的对角矩阵
```

**条件**：矩阵可对角化（有足够多的线性无关特征向量）

## ⭐ 三、特征值与特征向量

### 1. 定义与几何意义
**费曼解释**：特征向量是变换后"方向不变"的向量，只是拉长或缩短。拉长/缩短的倍数就是特征值。

**定义**：
```
Av = λv, v ≠ 0
v：特征向量
λ：特征值
```

**几何理解**：
- 大多数向量被旋转
- 特征向量只被缩放，不旋转
- 特征值是缩放倍数

**求解**：
1. 特征方程：det(A - λI) = 0
2. 求出特征值λ
3. 对每个λ，解(A - λI)v = 0得到特征向量

**例子**：
```
A = [[2, 1],
     [1, 2]]

det(A - λI) = (2-λ)² - 1 = 0
λ₁ = 3, λ₂ = 1

λ = 3: v₁ = [1, 1]^T
λ = 1: v₂ = [1, -1]^T
```

---

### 2. 特征值的性质

#### 基本性质
- 特征值之和 = 迹：Σ λ_i = tr(A)
- 特征值之积 = 行列式：Π λ_i = det(A)
- 特征值个数 ≤ n（n×n矩阵）
- 实对称矩阵的特征值都是实数

#### 谱定理
**费曼解释**：对称矩阵特别好——可以完全用特征值和特征向量描述。

**实对称矩阵**：
```
A = A^T ⟹ 存在正交矩阵Q使得 Q^T A Q = D（对角）
```

**推论**：
- 特征值都是实数
- 不同特征值的特征向量正交
- 可以正交对角化

**应用**：
- 主成分分析（PCA）
- 二次型标准化
- 量子力学

---

### 3. 对角化
**费曼解释**：找到一组"最好的坐标轴"，让矩阵变得简单（对角）。

**可对角化条件**：
```
A可对角化 ⟺ A有n个线性无关的特征向量
⟺ 几何重数 = 代数重数（对每个特征值）
```

**对角化过程**：
```
A = PDP^(-1)
P：特征向量矩阵
D：特征值对角矩阵
```

**好处**：
- 矩阵幂运算：A^k = PD^k P^(-1)
- 矩阵函数：exp(A) = P exp(D) P^(-1)
- 微分方程求解

**Jordan标准型**：不可对角化时的"最佳"形式

---

### 4. 应用

#### 微分方程
```
dx/dt = Ax
解：x(t) = exp(At) x(0)
```

#### 图论
- 图的拉普拉斯矩阵
- 谱聚类

#### 机器学习
- PCA（主成分分析）
- 协方差矩阵特征值
- 降维

#### 量子力学
- 观测量 = 厄米算符
- 本征值 = 测量值
- 本征态 = 状态

---

## 📏 四、内积空间

### 1. 内积定义
**费曼解释**：给向量空间加上"角度"和"长度"的概念。像点积的推广。

**定义**：向量空间V上的内积 ⟨·,·⟩: V × V → F 满足：
1. 线性性：⟨αu + v, w⟩ = α⟨u, w⟩ + ⟨v, w⟩
2. 对称性（实）/ 共轭对称（复）：⟨u, v⟩ = ⟨v, u⟩̄
3. 正定性：⟨v, v⟩ ≥ 0，等号当且仅当v = 0

**例子**：
- ℝ⁸：⟨x, y⟩ = Σ x_i y_i（标准内积）
- C[a,b]：⟨f, g⟩ = ∫_a^b f(x)g(x) dx
- 复向量：⟨x, y⟩ = Σ x_i ȳ_i

**诱导的范数**：
```
‖v‖ = √⟨v, v⟩
```

---

### 2. 正交性
**费曼解释**：两个向量"垂直"。内积为0。

**定义**：u ⊥ v ⟺ ⟨u, v⟩ = 0

**正交基**：基向量两两正交且单位长度

**正交投影**：
```
proj_u(v) = (⟨v, u⟩ / ⟨u, u⟩) u
```

**Gram-Schmidt正交化**：
```
把任意基转化为正交基
u₁ = v₁
u₂ = v₂ - proj_{u₁}(v₂)
u₃ = v₃ - proj_{u₁}(v₃) - proj_{u₂}(v₃)
...
```

**应用**：
- 最小二乘法
- QR分解
- 信号处理

---

### 3. 正交补与投影

#### 正交补
**费曼解释**：所有"垂直"于某个子空间的向量。

**定义**：
```
W^⊥ = {v ∈ V | ⟨v, w⟩ = 0, ∀ w ∈ W}
```

**性质**：
- V = W ⊕ W^⊥
- (W^⊥)^⊥ = W

#### 正交投影算子
**费曼解释**：把向量"影子"投射到子空间上。

**定义**：P: V → W 满足：
1. P² = P（幂等）
2. P^* = P（自伴）

**矩阵表示**：
```
P = A(A^T A)^(-1) A^T
其中A的列是W的基
```

---

### 4. 伴随算子
**费曼解释**：线性映射的"共轭转置"推广。

**定义**：T*: W → V 满足：
```
⟨Tv, w⟩ = ⟨v, T*w⟩, ∀ v ∈ V, w ∈ W
```

**性质**：
- (T*)* = T
- (ST)* = T*S*
- Ker(T*) = Im(T)^⊥

**自伴算子**：T = T*
**正规算子**：TT* = T*T

**谱定理推广**：
- 自伴算子 ⟹ 可正交对角化
- 正规算子 ⟹ 可对角化

## 🏛️ 五、抽象代数深化

### 1. 群论深入

#### 群的同态与同构
**费曼解释**：同态是"保持群结构"的映射。同构是"完全一样"的群。

**同态**：φ: G → H 满足 φ(ab) = φ(a)φ(b)

**核与像**：
- Ker(φ) = {g ∈ G | φ(g) = e_H}
- Im(φ) = {φ(g) | g ∈ G}

**第一同构定理**：
```
G/Ker(φ) ≅ Im(φ)
```

**例子**：
- 指数映射：exp: (ℝ, +) → (ℝ^+, ×) 是同态
- 行列式：det: GL_n → F^* 是同态

#### 群作用
**费曼解释**：群"作用"在集合上，像对称群作用在几何图形上。

**定义**：群G在集合X上的作用是一个映射 G × X → X，满足：
1. e·x = x
2. (gh)·x = g·(h·x)

**轨道与稳定子**：
- 轨道：Orb(x) = {g·x | g ∈ G}
- 稳定子：Stab(x) = {g ∈ G | g·x = x}
- **轨道-稳定子定理**：|Orb(x)| = |G| / |Stab(x)|

**应用**：
- 计数问题（Burnside引理）
- 对称性分析
- 分子对称性

#### Sylow定理
**费曼解释**：有限群的"子群"结构定理。

**Sylow p-子群**：阶为p^k的子群，其中p^k || |G|

**三个定理**：
1. 存在性：Sylow p-子群存在
2. 共轭性：所有Sylow p-子群共轭
3. 计数：Sylow p-子群个数 ≡ 1 (mod p)

**应用**：证明群不可解、分类有限群

---

### 2. 环论深入

#### 理想 (Ideal)
**费曼解释**：环的"正规子群"，可以做商。

**定义**：I ⊆ R 是理想，如果：
1. (I, +) 是子群
2. r ∈ R, i ∈ I ⟹ ri ∈ I, ir ∈ I

**例子**：
- nℤ 是 ℤ 的理想
- (x) 是 F[x] 的理想（由x生成）

**主理想**：由一个元素生成的理想
```
(a) = {ra | r ∈ R}
```

**素理想与极大理想**：
- P是素理想 ⟺ ab ∈ P ⟹ a ∈ P 或 b ∈ P
- M是极大理想 ⟺ R/M 是域

#### 多项式环
**费曼解释**：多项式的"代数"，系数在环中。

**性质**：
- F[x] 是主理想整环（PID）
- F[x] 的理想都是主理想
- 除法算式：f = qg + r，deg r < deg g

**应用**：
- 代数几何
- 编码理论
- 密码学

---

### 3. 域论与扩张

#### 域扩张 (Field Extension)
**费曼解释**：从"小域"构造"大域"。像从ℝ构造ℂ。

**定义**：K/F 是域扩张，F ⊆ K

**次数**：[K : F] = K作为F-向量空间的维数

**塔定理**：
```
F ⊆ L ⊆ K ⟹ [K : F] = [K : L] [L : F]
```

**例子**：
- [ℂ : ℝ] = 2
- [ℚ(√2) : ℚ] = 2
- [ℚ(√2, √3) : ℚ] = 4

#### 代数扩张
**费曼解释**：添加"代数元素"（满足多项式方程的元素）。

**代数元素**：α ∈ K 是F上的代数元素 ⟺ 存在非零f(x) ∈ F[x]使得f(α) = 0

**最小多项式**：次数最低的首一多项式

**例子**：
- √2 在 ℚ 上代数，最小多项式 x² - 2
- π 在 ℚ 上超越（不代数）

**代数闭包**：包含所有代数元素的扩张

---

### 4. 伽罗瓦理论

#### 伽罗瓦群
**费曼解释**：域扩张的"对称性"——保持小域不变的自同构。

**定义**：
```
Gal(K/F) = {σ ∈ Aut(K) | σ|_F = id}
```

**基本定理**：
```
域扩张 K/F 的中间域 ⟷ Gal(K/F) 的子群
```

**例子**：
- Gal(ℂ/ℝ) ≅ ℤ/2ℤ
- Gal(ℚ(√2)/ℚ) ≅ ℤ/2ℤ

#### 根式可解
**费曼解释**：多项式方程能否用根式表示解。

**判定**：
```
多项式根式可解 ⟺ Galois群可解
```

**应用**：
- 证明五次及以上方程无一般根式解
- 古典作图问题（三等分角、倍立方）

**历史**：
- Abel (1824): 五次方程无一般解
- Galois (1832): 完整理论

---

### 5. 模论初步

#### 模 (Module)
**费曼解释**：向量空间的推广——把"域"换成"环"。

**定义**：环R上的模M满足（类似向量空间公理）：
1. (M, +) 是阿贝尔群
2. R × M → M (标量乘法)
3. 类似向量空间的公理

**关键区别**：
- 向量空间：域上，有基，维数良定义
- 模：环上，可能没有基

**例子**：
- 向量空间是域上的模
- ℤ-模 = 阿贝尔群
- F[x]-模 = 向量空间 + 线性算子

#### 自由模
**费曼解释**：有"基"的模，最像向量空间。

**定义**：M是自由模 ⟺ M ≅ R^n

**投射模**：自由模的"直和分量"

## 🎭 六、表示论

### 1. 群表示
**费曼解释**：用矩阵表示群元素。把抽象群"具体化"。

**定义**：群G在向量空间V上的表示是同态 ρ: G → GL(V)

**例子**：
- 平凡表示：ρ(g) = I
- 置换表示：置换矩阵
- 正则表示：群作用在自己上

**不可约表示**：没有非平凡的不变子空间

**Maschke定理**：有限群在特征不整除|G|的域上，表示可分解为不可约表示的直和

---

### 2. 特征标理论
**费曼解释**：表示的"指纹"——特征标唯一确定表示。

**定义**：χ(g) = tr(ρ(g))

**性质**：
- χ(e) = dim V（表示的维数）
- χ(hgh^(-1)) = χ(g)（类函数）
- 不可约表示的特征标正交

**特征标表**：群的"DNA"

**应用**：
- 分子光谱（化学）
- 粒子分类（物理）
- 群的结构分析

## 🚀 七、现代发展（2020-2025）

### 1. 量子计算与线性代数
**费曼解释**：量子比特用向量表示，量子门用矩阵表示。

**量子态**：单位向量 |ψ⟩ ∈ ℂ^(2^n)

**量子门**：酉矩阵 U ∈ U(2^n)

**关键概念**：
- 张量积：|ψ⟩ ⊗ |φ⟩
- 酉变换
- 测量：投影算符

**应用**：
- Shor算法（质因数分解）
- Grover算法（搜索）
