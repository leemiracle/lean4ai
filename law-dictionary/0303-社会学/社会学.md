# 社会学（Sociology）

**学科代码**：0303  
**门类**：法学（03）

---

## 📌 学科简介

社会学研究人类社会行为、社会关系、社会结构和社会变迁。它关注群体、组织、制度如何影响个人，以及社会问题如何产生和解决。

---

## 🎯 核心概念

### 一、社会学基础

#### 1. 社会（Society）

**费曼解释**：
社会就像一张巨大的"关系网"，每个人都是网上的一个节点。我们通过家庭、学校、工作、朋友圈等关系连接在一起。社会不只是人的集合，而是人与人之间的各种关系。

**核心要素**：
- 人口
- 文化
- 制度
- 组织

**应用场景**：理解社会现象

---

#### 2. 社会化（Socialization）

**费曼解释**：
社会化就是"学会做人"的过程。从婴儿到成年，我们学习社会的规矩、价值观、行为方式。就像新员工入职培训，学习公司的文化和规则。

**类型**：
| 阶段 | 内容 |
|------|------|
| 初级社会化 | 儿童时期学习基本规范 |
| 次级社会化 | 成年后学习专业技能 |
| 再社会化 | 改变原有观念行为 |

**核心要点**：
- 家庭是第一社会化场所
- 学校、媒体也起重要作用
- 终身进行

**应用场景**：教育、家庭研究

---

#### 3. 社会结构（Social Structure）

**费曼解释**：
社会结构是社会的"骨架"，包括各种制度、组织、群体。就像建筑的框架，决定了整个社会的形态和功能。

**主要组成**：
- 社会群体（家庭、朋友圈）
- 社会组织（学校、公司）
- 社会制度（婚姻、教育）
- 社会阶层

**核心要点**：
- 相对稳定
- 制约个人行为
- 影响资源分配

**应用场景**：分析社会问题、社会政策

---

### 二、社会问题

#### 4. 社会分层（Social Stratification）

**费曼解释**：
社会分层就是社会的"等级结构"。人们按照收入、职业、教育等被分成不同的层次。就像一栋楼有不同楼层，有的人住在高层，有的人住在底层。

**主要维度**：
| 维度 | 内容 |
|------|------|
| 经济 | 收入、财富 |
| 职业 | 社会地位 |
| 教育 | 学历水平 |
| 权力 | 影响力 |

**核心要点**：
- 不平等是普遍现象
- 影响机会和资源
- 可以流动但有限

**应用场景**：理解贫富差距、社会流动

---

#### 5. 社会流动（Social Mobility）

**费曼解释**：
社会流动就是"换个层次"。向上流动是升职加薪、提高地位；向下流动是失业、地位下降。代际流动是比较子女和父母的阶层变化。

**类型**：
| 类型 | 说明 |
|------|------|
| 垂直流动 | 上下阶层变化 |
| 水平流动 | 同阶层位置变化 |
| 代际流动 | 与父母阶层比较 |

**核心要点**：
- 受教育影响
- 机会公平很重要
- 社会开放程度

**应用场景**：教育政策、就业政策

---

#### 6. 贫困（Poverty）

**费曼解释**：
贫困是"不够用"的状态。绝对贫困是连基本生活都难以维持，相对贫困是与周围人相比差距太大。贫困不只是缺钱，还包括机会、能力、权利的缺失。

**类型**：
| 类型 | 标准 |
|------|------|
| 绝对贫困 | 收入低于基本生活线 |
| 相对贫困 | 收入低于社会平均一定比例 |
| 多维贫困 | 多个维度（健康、教育等） |

**核心要点**：
- 收入贫困 vs 多维贫困
- 贫困陷阱
- 扶贫政策

**应用场景**：社会救助、扶贫开发

---

### 三、社会研究方法

#### 7. 社会调查（Social Survey）

**费曼解释**：
社会调查就是用科学方法"收集信息"。通过问卷、访谈、观察等方式，了解人们的行为、态度、状况。就像做人口普查，挨家挨户询问登记。

**主要方法**：
| 方法 | 特点 |
|------|------|
| 问卷调查 | 样本量大、标准化 |
| 深度访谈 | 深入、灵活 |
| 参与观察 | 亲身体验 |
- 抽样方法
- 问卷设计
- 数据分析

**应用场景**：社会研究、市场调研

---

#### 8. 社会指标（Social Indicators）

**费曼解释**：
社会指标是衡量社会发展状况的"仪表盘"。就像汽车的速度表、油量表告诉你车况，社会指标告诉你社会的健康状况。

**常见指标**：
| 指标 | 衡量内容 |
|------|---------|
| 基尼系数 | 收入不平等 |
| 恩格尔系数 | 生活水平 |
| 人类发展指数 | 综合发展水平 |

**核心要点**：
- 客观 vs 主观指标
- 单一 vs 综合指标
- 跨国比较

**应用场景**：政策评估、国际比较
