# 经济法学（Economic Law）

**学科代码**：030107  
**一级学科**：法学（0301）

---

## 📌 学科简介

经济法学研究国家对经济活动的干预、管理和协调的法律规范。它涉及市场规制、宏观调控、消费者保护等领域。

---

## 🎯 核心概念

### 一、经济法基础

#### 1. 经济法（Economic Law）

**费曼解释**：
经济法是"管经济的法"——国家用法律手段调节经济活动。既不是完全自由放任，也不是完全计划控制。

**调整对象**：
| 对象 | 内容 |
|------|------|
| 市场规制关系 | 维护市场秩序 |
| 宏观调控关系 | 调节经济运行 |
| 社会保障关系 | 保障民生 |

**核心要点**：
- 国家干预
- 社会本位
- 实质公平

**应用场景**：市场监管、宏观调控

---

#### 2. 市场主体（Market Subjects）

**费曼解释**：
市场主体是"参与经济游戏的人"——企业、消费者、政府等。每种主体有不同的权利义务。

**类型**：
| 主体 | 地位 |
|------|------|
| 经营者 | 提供商品服务 |
| 消费者 | 购买使用 |
| 监管者 | 维护秩序 |

**应用场景**：市场准入、经营监管

---

### 二、市场规制法

#### 3. 反垄断法（Anti-Monopoly Law）

**费曼解释**：
反垄断法是"反大鱼吃小鱼"的法——防止企业垄断市场，保护公平竞争。不能因为你强就可以欺负别人。

**规制对象**：
| 类型 | 内容 |
|------|------|
| 垄断协议 | 价格联盟、分割市场 |
| 滥用市场支配地位 | 不公平高价、搭售 |
| 经营者集中 | 并购审查 |

**核心要点**：
- 保护竞争
- 规制垄断
- 效率与公平平衡

**应用场景**：平台经济、大型并购

---

#### 4. 反不正当竞争法（Anti-Unfair Competition Law）

**费曼解释**：
反不正当竞争法是"反作弊"的法——不许用欺骗、诋毁等手段竞争。就像体育比赛不能服用兴奋剂。

**不正当竞争行为**：
| 行为 | 内容 |
|------|------|
| 商业混淆 | 冒充他人 |
| 商业贿赂 | 回扣、行贿 |
| 虚假宣传 | 夸大宣传 |
| 侵犯商业秘密 | 窃取机密 |
| 诋毁商誉 | 恶意中伤 |

**应用场景**：市场竞争

---

#### 5. 消费者权益保护法（Consumer Protection Law）

**费曼解释**：
消费者权益保护法是"保护买家"的法——消费者是弱势群体，需要特殊保护。

**消费者权利**：
| 权利 | 内容 |
|------|------|
| 安全权 | 人身财产安全 |
| 知情权 | 了解商品信息 |
| 选择权 | 自主选择 |
| 公平交易权 | 公平合理 |
| 求偿权 | 受损获赔 |

**核心要点**：
- 倾斜保护
- 惩罚性赔偿
- 举证责任倒置

**应用场景**：网购、食品安全

---

### 三、宏观调控法

#### 6. 财政法（Fiscal Law）

**费曼解释**：
财政法是"管钱袋子"的法——政府的钱从哪来（税收）、到哪去（支出），都要依法进行。

**内容**：
| 内容 | 说明 |
|------|------|
| 预算法 | 政府预算管理 |
| 税法 | 税收征管 |
| 政府采购 | 政府买东西 |

**核心要点**：
- 预算法定
- 税收法定
- 公开透明

**应用场景**：政府预算、税收

---

#### 7. 金融法（Financial Law）

**费曼解释**：
金融法是"管钱流通"的法——银行、证券、保险等金融活动要依法进行，防范风险。

**内容**：
| 领域 | 内容 |
|------|------|
| 银行法 | 银行业监管 |
| 证券法 | 证券市场监管 |
| 保险法 | 保险业监管 |

**核心要点**：
- 防范风险
- 保护投资者
- 维护稳定

**应用场景**：银行、股票、保险
