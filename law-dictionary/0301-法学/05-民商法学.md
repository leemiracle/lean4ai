# 民商法学（Civil and Commercial Law）

**学科代码**：030105  
**一级学科**：法学（0301）

---

## 📌 学科简介

民商法学研究民事和商事关系的法律规范。民法调整平等主体之间的财产关系和人身关系，商法调整商事主体之间的商事关系。

---

## 🎯 核心概念

### 一、民法基础

#### 1. 民法（Civil Law）

**费曼解释**：
民法是"老百姓的法律"——管日常生活中的各种关系。买卖东西、租房、结婚离婚、遗产继承……都归民法管。

**核心原则**：
| 原则 | 说明 |
|------|------|
| 平等原则 | 地位平等 |
| 自愿原则 | 意思自治 |
| 公平原则 | 公平合理 |
| 诚信原则 | 诚实守信 |
| 公序良俗 | 遵守公共秩序 |

**应用场景**：日常民事活动

---

#### 2. 民事法律行为（Civil Legal Act）

**费曼解释**：
民事法律行为是"有法律效果的行为"。比如签合同——签了就有法律约束力，不履行要承担责任。

**有效要件**：
| 要件 | 内容 |
|------|------|
| 行为人具有民事行为能力 | 有资格 |
| 意思表示真实 | 是真实意愿 |
| 不违反法律、行政法规的强制性规定 | 合法 |
| 不违背公序良俗 | 合道德 |

**核心要点**：
- 意思表示为核心
- 合法性
- 行为能力

**应用场景**：合同、遗嘱

---

#### 3. 民事权利（Civil Rights）

**费曼解释**：
民事权利是"法律给你的权力"。你有权要求别人做什么（债权），或者禁止别人做什么（物权）。

**主要类型**：
| 类型 | 内容 |
|------|------|
| 人身权 | 姓名、肖像、名誉等 |
| 财产权 | 物权、债权 |
| 知识产权 | 专利、商标、著作权 |

**应用场景**：维权

---

### 二、物权法

#### 4. 物权（Property Rights）

**费曼解释**：
物权是对物品的"支配权"。你的手机，你有权占有、使用、处分。别人不能随便拿走。

**类型**：
| 类型 | 内容 |
|------|------|
| 所有权 | 完全支配 |
| 用益物权 | 使用收益 |
| 担保物权 | 担保债权 |

**核心要点**：
- 支配性
- 排他性
- 绝对性

**应用场景**：买房、担保

---

#### 5. 所有权（Ownership）

**费曼解释**：
所有权是最完整的物权——对物品的"完全控制"。你可以占有、使用、收益、处分。

**权能**：
| 权能 | 内容 |
|------|------|
| 占有 | 实际控制 |
| 使用 | 利用物品 |
| 收益 | 获得收益 |
| 处分 | 决定物品命运 |

**应用场景**：财产权

---

### 三、合同法

#### 6. 合同（Contract）

**费曼解释**：
合同是"双方的约定"。你说买、我说卖，双方意思表示一致，合同就成立了。签了合同就要履行，不履行要赔钱。

**成立要件**：
- 要约（一方提出）
- 承诺（另一方同意）
- 意思表示一致

**核心要点**：
- 意思自治
- 约束力
- 违约责任

**应用场景**：买卖、租赁、服务

---

#### 7. 违约责任（Liability for Breach）

**费曼解释**：
违约责任是"说话不算话的代价"。签了合同不履行，就要承担法律后果。

**责任形式**：
| 形式 | 内容 |
|------|------|
| 继续履行 | 强制履行 |
- 赔偿损失
- 支付违约金
- 定金罚则

**应用场景**：合同纠纷

---

### 四、商法

#### 8. 公司（Company）

**费曼解释**：
公司是"人造的人"——法律上独立的主体，可以拥有财产、签订合同、承担责任。股东只承担有限责任。

**类型**：
| 类型 | 特点 |
|------|------|
| 有限责任公司 | 人合性为主 |
- 股份有限公司
- 股东承担有限责任
- 独立法人

**应用场景**：公司治理、股权
