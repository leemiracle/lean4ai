# 宪法学（Constitutional Law）

**学科代码**：030103  
**一级学科**：法学（0301）

---

## 📌 学科简介

宪法学研究宪法的基本理论、宪法制度和宪法实施。宪法是国家的根本大法，具有最高的法律效力，是其他法律的立法基础。

---

## 🎯 核心概念

### 一、宪法基础

#### 1. 宪法（Constitution）

**费曼解释**：
宪法是国家的"根本大法"，就像一个国家的"总章程"。其他法律（刑法、民法等）都不能违反宪法，就像学校规章制度不能违反教育法一样。

**核心特征**：
| 特征 | 说明 |
|------|------|
| 最高性 | 效力最高 |
| 根本性 | 规定根本制度 |
| 稳定性 | 修改程序严格 |
| 纲领性 | 指导国家发展 |

**核心要点**：
- 国家根本大法
- 其他法律的母法
- 保障公民基本权利

**应用场景**：理解法律体系

---

#### 2. 宪法效力（Constitutional Effect）

**费曼解释**：
宪法效力就是宪法的"威力"——所有法律、法规、规章都必须符合宪法，违反宪法的无效。就像班级规定不能违反校规一样。

**核心要点**：
- 最高法律效力
- 立法依据
- 违宪无效

**应用场景**：立法、司法

---

### 二、国家制度

#### 3. 国体（State Nature）

**费曼解释**：
国体是国家的"性质"——国家权力属于谁。中国是人民民主专政的社会主义国家，国家一切权力属于人民。

**核心内容**：
- 人民民主专政
- 工农联盟为基础
- 人民是国家主人

**应用场景**：理解国家性质

---

#### 4. 政体（Political System）

**费曼解释**：
政体是国家"怎么组织"——权力如何运行。中国的政体是人民代表大会制度，人民通过选举代表管理国家。

**核心内容**：
| 内容 | 说明 |
|------|------|
| 人民代表大会制度 | 根本政治制度 |
| 民主集中制 | 组织原则 |
| 一府一委两院 | 行政、监察、司法 |

**核心要点**：
- 人民当家作主
- 议行合一
- 民主集中

**应用场景**：理解国家机构

---

#### 5. 国家机构（State Organs）

**费曼解释**：
国家机构是行使国家权力的机关体系。就像一个公司的组织架构，有决策层、执行层、监督层。

**主要机构**：
| 机构 | 职能 |
|------|------|
| 全国人大 | 最高权力机关、立法 |
| 国务院 | 最高行政机关 |
| 监察委员会 | 监察机关 |
| 人民法院 | 审判机关 |
| 人民检察院 | 法律监督机关 |

**核心要点**：
- 分工负责
- 相互配合
- 相互制约

**应用场景**：理解国家治理

---

### 三、公民基本权利

#### 6. 公民基本权利（Basic Rights）

**费曼解释**：
基本权利是宪法保障的"最低保障"——每个公民都享有的最重要的权利。这些权利不能被剥夺。

**主要权利**：
| 类型 | 具体权利 |
|------|---------|
| 平等权 | 法律面前人人平等 |
| 政治权利 | 选举权、被选举权 |
| 人身自由 | 人身、住宅不受侵犯 |
| 社会经济权利 | 劳动权、受教育权 |
| 文化权利 | 言论、出版自由 |

**核心要点**：
- 不可剥夺
- 国家保护义务
- 权利与义务统一

**应用场景**：维权、诉讼

---

#### 7. 平等权（Right to Equality）

**费曼解释**：
平等权是"法律面前人人平等"——同样的情况同样对待，不同的情况不同对待，不能歧视。

**核心内容**：
- 法律适用平等
- 权利能力平等
- 不允许特权

**应用场景**：反歧视、平等保护

---

### 四、宪法实施

#### 8. 宪法监督（Constitutional Supervision）

**费曼解释**：
宪法监督是检查法律、法规是否违反宪法。就像质检员检查产品是否合格，宪法监督检查法律是否"合宪"。

**核心机制**：
| 机制 | 内容 |
|------|------|
| 合宪性审查 | 审查法律是否合宪 |
| 备案审查 | 法规备案、审查 |
| 宪法解释 | 解释宪法含义 |

**核心要点**：
- 全国人大及其常委会监督
- 法规备案审查
- 违宪纠正

**应用场景**：立法、执法
