# 国际法学（International Law）

**学科代码**：030109  
**一级学科**：法学（0301）

---

## 📌 学科简介

国际法学研究调整国家间关系的法律规范。它包括国际公法、国际私法、国际经济法等分支。

---

## 🎯 核心概念

### 一、国际公法

#### 1. 国际法（International Law）

**费曼解释**：
国际法是"国家之间玩游戏的规则"——国家之间怎么相处、怎么解决争端。没有世界警察，主要靠国家自觉遵守。

**特点**：
| 特点 | 说明 |
|------|------|
| 主体主要是国家 | 不是个人 |
| 没有强制执行机构 | 靠自觉+国际压力 |
| 渊源多样 | 条约、习惯等 |

**核心要点**：
- 国家主权
- 国家同意
- 软法特征

**应用场景**：外交、国际组织

---

#### 2. 国家主权（State Sovereignty）

**费曼解释**：
国家主权是"我说了算"——一个国家在其领土范围内有最高权力，外国不能干涉。

**内容**：
| 内容 | 说明 |
|------|------|
| 对内最高权 | 统治本国 |
| 对外独立权 | 不受外国支配 |
| 自卫权 | 抵抗侵略 |

**核心要点**：
- 不干涉内政
- 平等互利
- 领土完整

**应用场景**：外交关系

---

#### 3. 条约（Treaty）

**费曼解释**：
条约是"国家间的合同"——国家之间签订的书面协议，要遵守。

**类型**：
| 类型 | 说明 |
|------|------|
| 双边条约 | 两国之间 |
| 多边条约 | 多国之间 |
- 约定必须遵守
- 对缔约国有约束力
- 不得保留条款

**应用场景**：国际协定

---

### 二、国际私法

#### 4. 国际私法（Private International Law）

**费曼解释**：
国际私法是"跨国私事适用哪国法"的问题。比如中国人跟美国人在日本结婚，该用哪国法律？

**核心问题**：
| 问题 | 内容 |
|------|------|
| 管辖权 | 哪国法院管 |
| 法律适用 | 用哪国法 |
| 判决承认与执行 | 判决能否跨国执行 |

**应用场景**：跨国婚姻、继承、合同

---

### 三、国际经济法

#### 5. 国际贸易法（International Trade Law）

**费曼解释**：
国际贸易法是"跨国买卖的规则"——国际货物买卖、支付、运输等。

**内容**：
| 内容 | 说明 |
|------|------|
| 国际货物买卖 | 跨国买卖合同 |
| 国际支付 | 信用证、托收等 |
| 国际运输 | 海运、空运规则 |

**核心要点**：
- 国际条约优先
- 国际惯例
- 当事人意思自治

**应用场景**：国际贸易

---

#### 6. 世界贸易组织（WTO）

**费曼解释**：
WTO是"国际贸易俱乐部"——成员国约定贸易规则，有争端可以投诉。

**原则**：
| 原则 | 内容 |
|------|------|
| 最惠国待遇 | 平等对待所有成员 |
| 国民待遇 | 对外国商品等同国内 |
| 关税减让 | 降低关税壁垒 |

**应用场景**：国际贸易争端
