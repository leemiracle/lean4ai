# 刑法学（Criminal Law）

**学科代码**：030104  
**一级学科**：法学（0301）

---

## 📌 学科简介

刑法学研究犯罪与刑罚的法律规范。它规定什么是犯罪、犯罪应受什么处罚。刑法是保护社会秩序、保障公民权利的重要法律部门。

---

## 🎯 核心概念

### 一、犯罪论

#### 1. 犯罪（Crime）

**费曼解释**：
犯罪是"严重违法"的行为——不只是违反法律，而且危害性大到必须用刑罚来惩罚。就像闯红灯是违法，但杀人放火才是犯罪。

**犯罪特征**：
| 特征 | 说明 |
|------|------|
| 社会危害性 | 对社会造成危害 |
| 刑事违法性 | 违反刑法规定 |
| 应受刑罚性 | 应当受刑罚处罚 |

**核心要点**：
- 三性统一
- 法定原则
- 程序保障

**应用场景**：刑事司法

---

#### 2. 犯罪构成（Constituent Elements of Crime）

**费曼解释**：
犯罪构成是判断一个行为是否构成犯罪的"清单"。就像购物清单一样，必须所有项目都满足才能构成犯罪。

**四个要件**：
| 要件 | 内容 |
|------|------|
| 犯罪客体 | 被侵害的法益 |
| 犯罪客观方面 | 危害行为、结果 |
| 犯罪主体 | 实施犯罪的人 |
| 犯罪主观方面 | 故意或过失 |

**核心要点**：
- 四要件说
- 缺一不可
- 主客观统一

**应用场景**：定罪量刑

---

#### 3. 故意犯罪（Intentional Crime）

**费曼解释**：
故意犯罪是"明知故犯"——明知自己的行为会发生危害结果，还希望或放任它发生。

**类型**：
| 类型 | 心理状态 |
|------|---------|
| 直接故意 | 希望+追求 |
| 间接故意 | 放任+不反对 |

**例子**：
- 直接故意：甲想杀乙，开枪打死乙
- 间接故意：甲放火烧仓库，明知可能烧死看门人，仍然放火

**应用场景**：区分故意与过失

---

#### 4. 过失犯罪（Negligent Crime）

**费曼解释**：
过失犯罪是"应当预见没有预见"或"预见但轻信能避免"——不是故意的，但因为疏忽大意或过于自信导致危害结果。

**类型**：
| 类型 | 心理状态 |
|------|---------|
| 疏忽大意过失 | 应预见未预见 |
| 过于自信过失 | 预见但轻信避免 |

**例子**：
- 疏忽大意：司机开车看手机，撞死行人
- 过于自信：司机酒后驾车，认为自己酒量好，结果撞人

**应用场景**：交通肇事、医疗事故

---

### 二、刑罚论

#### 5. 刑罚（Punishment）

**费曼解释**：
刑罚是犯罪要付出的"代价"——国家强制犯罪人承受的痛苦。刑罚的目的不只是惩罚，还包括教育、预防犯罪。

**刑罚目的**：
| 目的 | 内容 |
|------|------|
| 惩罚 | 报应犯罪 |
| 教育 | 改造罪犯 |
| 预防 | 防止再犯 |

**核心要点**：
- 罪刑法定
- 罪责刑相适应
- 人道主义

**应用场景**：量刑

---

#### 6. 主刑与附加刑（Principal and Supplementary Punishments）

**费曼解释**：
主刑是"主要处罚"，附加刑是"附加处罚"，可以单独或附加适用。

**主刑**：
| 刑罚 | 内容 |
|------|------|
| 管制 | 不关押，限制自由 |
| 拘役 | 短期剥夺自由（1-6个月） |
| 有期徒刑 | 中长期剥夺自由（6个月-15年） |
| 无期徒刑 | 终身剥夺自由 |
| 死刑 | 剥夺生命 |

**附加刑**：
| 刑罚 | 内容 |
|------|------|
| 罚金 | 强制缴纳金钱 |
| 剥夺政治权利 | 限制政治权利 |
| 没收财产 | 没收财产 |
| 驱逐出境 | 针对外国人 |

**应用场景**：判决

---

### 三、常见犯罪

#### 7. 故意杀人罪（Intentional Homicide）

**费曼解释**：
故意杀人是最严重的犯罪之一——故意非法剥夺他人生命。

**构成要件**：
- 客体：他人的生命权
- 客观方面：非法剥夺他人生命
- 主体：一般主体
- 主观方面：故意

**量刑**：
- 基本刑：死刑、无期徒刑或10年以上有期徒刑
- 情节较轻：3-10年有期徒刑

**应用场景**：命案

---

#### 8. 盗窃罪（Theft）

**费曼解释**：
盗窃是"偷东西"——秘密窃取他人财物。不是抢劫（暴力）、不是诈骗（欺骗），而是悄悄拿走。

**构成要件**：
- 客体：公私财物所有权
- 客观方面：秘密窃取
- 主体：一般主体
- 主观方面：故意+非法占有目的

**量刑标准**：
| 数额 | 量刑 |
|------|------|
| 数额较大（1000-3000元以上） | 3年以下 |
| 数额巨大（3-10万元以上） | 3-10年 |
| 数额特别巨大（30-50万元以上） | 10年以上或无期 |

**应用场景**：财产犯罪
