# 中国法学知识词典

## 📚 概述

本词典基于中国专业学科分类标准，对法学知识进行全面、系统、精准的分类和解释。采用费曼学习法，用通俗易懂的语言解释专业法律概念。

## 🎯 特点

- **最新性**：基于2024年最新法律法规和学科发展
- **全面性**：覆盖法学所有二级学科和主要研究方向
- **准确性**：严格遵循中国法学学科分类标准
- **易懂性**：费曼学习法解释，深入浅出
