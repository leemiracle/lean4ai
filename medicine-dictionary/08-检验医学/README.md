# 检验医学

> 医学二级学科代码：32080

---

## 检验医学概览

**费曼解释**：
检验医学是**通过化验发现疾病**。抽血、验尿、查大便，从体液中发现异常，帮助诊断疾病。

---

## 第一编：血液检验

### 一、血常规

#### 1. 血常规 (Complete Blood Count)

**费曼解释**：
血常规是最基本的血液检查。看红细胞（贫血？）、白细胞（感染？）、血小板（止血功能？）。

**主要指标及正常值**：
| 指标 | 正常值 | 异常意义 |
|------|--------|----------|
| RBC | 4.0-5.5×10¹²/L | ↓贫血 |
| Hb | 120-160 g/L | ↓贫血 |
| WBC | 4-10×10⁹/L | ↑感染 ↓免疫低下 |
| PLT | 100-300×10⁹/L | ↓出血风险 |
| 中性粒细胞% | 50-70% | ↑细菌感染 |
| 淋巴细胞% | 20-40% | ↑病毒感染 |

## 第二编：生化检验

### 二、肝功能

#### 2. 肝功能检查 (Liver Function Test)

**费曼解释**：
检查肝脏工作得好不好。转氨酶升高说明肝细胞损伤，胆红素升高说明胆汁排泄有问题。

**主要指标**：
| 指标 | 正常值 | 异常意义 |
|------|--------|----------|
| ALT | <40 U/L | ↑肝损伤 |
| AST | <40 U/L | ↑肝损伤、心梗 |
| 总胆红素 | <21 μmol/L | ↑黄疸 |
| 白蛋白 | 35-55 g/L | ↓肝硬化 |

---

### 三、肾功能

#### 3. 肾功能检查 (Renal Function Test)

**费曼解释**：
检查肾脏过滤能力。肌酐和尿素氮是废物，肾功能不好时这些废物在血液中堆积。

**主要指标**：
| 指标 | 正常值 | 异常意义 |
|------|--------|----------|
| 肌酐 (Cr) | 44-133 μmol/L | ↑肾功能下降 |
| 尿素氮 (BUN) | 2.9-8.2 mmol/L | ↑肾功能下降 |
| 尿酸 (UA) | 150-420 μmol/L | ↑痛风 |
| GFR | >90 ml/min | ↓肾功能下降 |

---

### 四、血糖

#### 4. 血糖检查 (Blood Glucose Test)

**费曼解释**：
检查血糖水平。空腹血糖≥7.0 mmol/L 或餐后2小时血糖≥11.1 mmol/L，可能是糖尿病。

**指标及正常值**：
| 指标 | 正常值 | 糖尿病标准 |
|------|--------|------------|
| 空腹血糖 | 3.9-6.1 mmol/L | ≥7.0 mmol/L |
| 餐后2h血糖 | <7.8 mmol/L | ≥11.1 mmol/L |
| HbA1c | <6.0% | ≥6.5% |

---

### 五、血脂

#### 5. 血脂检查 (Lipid Profile)

**费曼解释**：
检查血液中的脂肪。胆固醇和甘油三酯太高，容易得心血管病。

**主要指标**：
| 指标 | 正常值 | 异常意义 |
|------|--------|----------|
| 总胆固醇 (TC) | <5.2 mmol/L | ↑动脉硬化 |
| 甘油三酯 (TG) | <1.7 mmol/L | ↑心血管风险 |
| LDL-C | <3.4 mmol/L | ↑心血管风险 |
| HDL-C | >1.0 mmol/L | ↓心血管风险（好胆固醇） |

---

## 第三编：尿液检验

### 六、尿常规

#### 6. 尿常规 (Urinalysis)

**费曼解释**：
检查尿液。看有没有蛋白（肾病？）、糖（糖尿病？）、红细胞（血尿？）、白细胞（尿路感染？）。

**主要指标**：
| 指标 | 正常 | 异常意义 |
|------|------|----------|
| 尿蛋白 | 阴性 | 阳性：肾病 |
| 尿糖 | 阴性 | 阳性：糖尿病 |
| 尿红细胞 | <3/HP | 增多：血尿 |
| 尿白细胞 | <5/HP | 增多：尿路感染 |

---

## 第四编：其他检验

### 七、凝血功能

#### 7. 凝血功能检查 (Coagulation Test)

**主要指标**：
| 指标 | 意义 |
|------|------|
| PT | 外源性凝血途径 |
| APTT | 内源性凝血途径 |
| INR | 华法林抗凝监测 |

---

### 八、甲状腺功能

#### 8. 甲状腺功能检查 (Thyroid Function Test)

**主要指标**：
| 指标 | 正常值 | 异常意义 |
|------|--------|----------|
| TSH | 0.27-4.2 mIU/L | ↓甲亢 ↑甲减 |
| FT3 | 3.1-6.8 pmol/L | ↑甲亢 |
| FT4 | 12-22 pmol/L | ↑甲亢 ↓甲减 |

---

### 九、肿瘤标志物

#### 9. 肿瘤标志物 (Tumor Markers)

**费曼解释**：
肿瘤细胞产生的特殊物质。升高可能提示肿瘤，但不是绝对的。

**常见标志物**：
| 标志物 | 相关肿瘤 |
|--------|----------|
| AFP | 肝癌 |
| CEA | 消化道肿瘤 |
| CA125 | 卵巢癌 |
| PSA | 前列腺癌 |
| CA19-9 | 胰腺癌 |
