# 医学影像学

> 医学二级学科代码：32070

---

## 医学影像学概览

**费曼解释**：
医学影像学是**用各种技术给身体拍照**。让医生不用开刀就能看到身体内部的情况，是现代医学的"眼睛"。

**核心目的**：
- 发现病变
- 确定病变位置、大小、性质
- 指导治疗
- 评估疗效

---

## 第一编：X线成像

### 一、X线基础

#### 1. X线原理

**费曼解释**：X线是一种**高能射线**，能穿透人体。不同组织阻挡X线的能力不同，在胶片上形成黑白对比的图像。

**学术定义**：X线是波长很短的电磁波，具有很强的穿透力。

**X线特性**：
1. **穿透性**：能穿透人体
2. **荧光效应**：能使荧光物质发光
3. **感光效应**：能使胶片感光
4. **电离效应**：能使物质电离（生物效应基础）

**影像形成原理**：
- 骨骼：阻挡X线多，显示白色
- 软组织：中等阻挡，显示灰色
| 空气 | 阻挡少，显示黑色 |

---

#### 2. X线检查方法

**普通X线摄影**：
| 检查 | 用途 |
|------|------|
| 胸部X线 | 肺炎、结核、肿瘤、心脏增大 |
| 骨骼X线 | 骨折、脱位、骨肿瘤 |
| 腹部平片 | 肠梗阻、泌尿系结石 |
| 乳腺钼靶 | 乳腺癌筛查 |

**造影检查**：
- **定义**：引入对比剂，增加组织对比度
- **常用对比剂**：
  - 阳性对比剂：钡剂（消化道）、碘剂（血管、尿路）
  - 阴性对比剂：空气、二氧化碳

**常见造影检查**：
| 检查 | 对比剂 | 用途 |
|------|--------|------|
| 上消化道造影 | 钡剂 | 胃溃疡、胃癌 |
| 钡灌肠 | 钡剂 | 结肠病变 |
| 静脉肾盂造影 | 碘剂 | 尿路病变 |
| 血管造影 | 碘剂 | 血管病变 |

---

### 二、X线应用

#### 3. 胸部X线

**正常胸部X线表现**：
- 肺野：透亮（黑色）
- 心脏：密度高（白色），心胸比<0.5
- 肋骨、锁骨、肩胛骨：白色
- 膈肌：弧形白色线

**常见异常**：
| 病变 | X线表现 |
|------|----------|
| 肺炎 | 肺野片状密度增高影 |
| 肺结核 | 上肺斑片状、条索状影 |
| 肺癌 | 肿块影、分叶状 |
| 胸腔积液 | 肋膈角消失、外高内低弧形影 |
| 气胸 | 无肺纹理的透亮区 |
| 心脏增大 | 心胸比>0.5 |

---

#### 4. 骨骼X线

**骨折X线表现**：
- 骨折线：透亮线
- 骨折移位：断端错位
| 骨痂形成 | 愈合表现 |

**骨折类型**：
| 类型 | 特点 |
|------|------|
| 横形骨折 | 骨折线横行 |
| 斜形骨折 | 骨折线斜行 |
| 螺旋形骨折 | 骨折线螺旋状 |
| 粉碎性骨折 | 多个骨折碎片 |
| 压缩性骨折 | 椎体变扁 |
| 青枝骨折 | 儿童特有，不完全骨折 |

**骨肿瘤X线表现**：
- 良性：边界清楚、无骨膜反应
- 恶性：边界不清、骨膜反应（日光放射状、葱皮样）

## 第二编：CT成像

### 三、CT基础

#### 5. CT原理

**费曼解释**：CT是**3D的X线**。X线球管围绕身体旋转，从多个角度拍摄，电脑合成横断面图像。像把面包切成片，每一片都能看清楚。

**学术定义**：CT是利用X线束对人体进行扫描，经计算机处理后获得断层图像的成像技术。

**CT值（HU）**：
| 组织 | CT值(HU) |
|------|----------|
| 骨骼 | +1000 |
| 软组织 | +40-80 |
| 水 | 0 |
| 脂肪 | -100 |
| 空气 | -1000 |

**窗宽与窗位**：
- 窗宽：显示的CT值范围
- 窗位：显示的中心CT值
- 目的：突出显示不同组织

---

#### 6. CT检查方法

**平扫**：
- 不用对比剂
- 基础检查

**增强扫描**：
- 静脉注射碘对比剂
- 显示血管和病变的血供特点
- 鉴别良恶性病变

**CTA（CT血管造影）**：
- 显示血管
- 用于：冠脉CTA、脑血管CTA、肺动脉CTA

**CT灌注成像**：
- 评估组织血流灌注
- 用于：脑梗死、肿瘤

---

### 四、CT应用

#### 7. 头颅CT

**适应症**：
- 脑出血（首选）
| 脑梗死 |
| 脑肿瘤 |
| 颅脑外伤 |

**常见病变CT表现**：
| 病变 | CT表现 |
|------|--------|
| 脑出血 | 高密度（白色），边界清楚 |
| 脑梗死 | 低密度（黑色），按血管分布 |
| 脑肿瘤 | 肿块、占位效应、周围水肿 |
| 硬膜外血肿 | 凸透镜形高密度 |
| 硬膜下血肿 | 新月形高密度 |
| 蛛网膜下腔出血 | 脑池、脑沟高密度 |

---

#### 8. 胸部CT

**适应症**：
- 肺结节
| 肺癌 |
| 肺炎 |
| 肺结核 |
| 胸腔积液 |
| 纵隔病变 |

**肺结节分类**：
| 类型 | 特点 | 恶性可能 |
|------|------|----------|
| 实性结节 | 均匀软组织密度 | 中等 |
| 磨玻璃结节 | 模糊增高影 | 较高 |
| 混合结节 | 实性+磨玻璃 | 最高 |

**肺癌CT征象**：
- 分叶状
| 毛刺征 |
| 胸膜牵拉 |
| 含气支气管征 |
| 空泡征 |

---

#### 9. 腹部CT

**适应症**：
- 肝脏病变（肝癌、血管瘤）
| 胰腺病变 |
| 脾脏病变 |
| 肾脏病变 |
| 腹部肿块 |
| 腹部外伤 |

**肝脏病变CT表现**：
| 病变 | CT表现 |
|------|--------|
| 肝癌 | 动脉期强化、门脉期廓清（快进快出） |
| 肝血管瘤 | 向心性填充强化 |
| 肝囊肿 | 水样密度、无强化 |
| 肝脓肿 | 环形强化 |

## 第三编：MRI成像

### 五、MRI基础

#### 10. MRI原理

**费曼解释**：MRI用**强磁场和无线电波**成像。把身体放进大磁铁里，体内的氢原子核被磁化，发出信号，电脑把信号变成图像。没有辐射！

**学术定义**：MRI利用原子核在磁场中发生共振产生的信号进行成像。

**MRI的T1和T2**：
| 加权 | 特点 | 组织特点 |
|------|------|----------|
| T1WI | 解剖结构 | 脂肪白、水黑、肌肉灰 |
| T2WI | 病变检测 | 脂肪白、水白 |

**常见组织信号**：
| 组织 | T1WI | T2WI |
|------|------|------|
| 脂肪 | 高（白）| 高（白）|
| 水/脑脊液 | 低（黑）| 高（白）|
| 骨皮质 | 低（黑）| 低（黑）|
| 骨髓 | 高（白）| 中（灰）|
| 肌肉 | 中（灰）| 中（灰）|

**MRI禁忌症**：
- 心脏起搏器
| 人工耳蜗 |
| 金属异物（眼球、脑内）|
| 幽闭恐惧症 |

---

#### 11. MRI检查方法

**常规序列**：
- T1WI
- T2WI
- 脂肪抑制序列

**增强扫描**：
- 对比剂：钆（Gd）
- 用于：肿瘤、炎症

**特殊序列**：
| 序列 | 用途 |
|------|------|
| DWI（弥散加权）| 脑梗死早期诊断 |
| PWI（灌注加权）| 脑缺血 |
| MRA（磁共振血管成像）| 血管病变 |
| MRCP（磁共振胰胆管成像）| 胆管、胰管 |
| MRS（磁共振波谱）| 肿瘤定性 |

---

### 六、MRI应用

#### 12. 中枢神经系统MRI

**优势**：
- 无辐射
- 软组织分辨率高
| 多平面成像 |

**适应症**：
- 脑肿瘤
| 脑梗死 |
| 脑出血（亚急性、慢性）|
| 多发性硬化 |
| 脑炎 |
| 脊髓病变 |

**常见病变MRI表现**：
| 病变 | MRI表现 |
|------|----------|
| 脑梗死（急性）| DWI高信号、T2高信号 |
| 脑肿瘤 | 占位效应、周围水肿、异常强化 |
| 多发性硬化 | 脑室旁白质T2高信号 |
| 脊髓肿瘤 | 脊髓增粗、异常信号 |

---

#### 13. 骨关节MRI

**适应症**：
- 膝关节损伤（半月板、韧带）
| 肩关节损伤 |
| 脊柱病变（椎间盘突出）|
| 骨肿瘤 |
| 骨髓病变 |

**椎间盘突出MRI表现**：
- 椎间盘向后突出
| 压迫脊髓或神经根 |
| T2WI信号减低（退变）|

## 第四编：超声成像

### 七、超声基础

#### 14. 超声原理

**费曼解释**：超声用**高频声波**成像。声波遇到组织反射回来，形成图像。像蝙蝠用声波"看"东西。

**学术定义**：超声是利用超声波在人体内传播时的反射特性进行成像的技术。

**超声特点**：
- 无辐射、安全
| 实时成像 |
| 可重复检查 |
| 价格低廉 |

**局限性**：
- 受气体干扰（肺部、肠道）
| 受骨骼干扰 |
| 受体型影响（肥胖）|

---

#### 15. 超声检查方法

**常规超声**：
- B型超声（灰阶超声）
| 彩色多普勒（血流）|
| 频谱多普勒（血流速度）|

**特殊超声**：
| 技术 | 用途 |
|------|------|
| 三维超声 | 胎儿畸形筛查 |
| 超声造影 | 肝脏肿瘤鉴别 |
| 弹性成像 | 肝纤维化评估 |
| 超声内镜 | 胰腺、消化道 |

---

### 八、超声应用

#### 16. 腹部超声

**适应症**：
- 肝脏病变
| 胆囊病变（结石、息肉）|
| 胰腺病变 |
| 脾脏病变 |
| 肾脏病变 |

**常见病变超声表现**：

|------|----------|

| 胆囊结石 | 强回声、后方声影、可移动 |
| 肾结石 | 强回声、后方声影 |
| 肾囊肿 | 无回声、后壁增强 |

#### 17. 心脏超声

**费曼解释**：心脏超声是**给心脏做视频**。能看到心脏的结构、跳动、血流。

**检查内容**：

| 左房内径(LA) | <40 mm |
| 室间隔厚度(IVS) | 7-11 mm |

#### 18. 甲状腺超声

**适应症**：

## 第五编：核医学

### 九、核医学基础

#### 21. 核医学原理

**费曼解释**：核医学是**把放射性物质打进身体**，这些物质会跑到特定器官，仪器检测它们的位置和数量，反映器官的功能。

**学术定义**：核医学是利用放射性核素进行诊断和治疗的医学学科。

**特点**：
- 反映功能代谢
| 灵敏度高 |
| 可全身成像 |
| 辐射剂量较低 |

---

#### 22. PET-CT

**费曼解释**：PET-CT是**代谢+解剖的融合图像**。癌细胞代谢活跃，会大量吸收示踪剂，在图像上特别亮。

**原理**：
- 注射FDG（类似葡萄糖）
| 癌细胞摄取FDG多 |
| PET显示高代谢 |
| CT显示解剖位置 |
| 融合图像定位病灶 |

**适应症**：
- 肿瘤诊断和分期
| 肿瘤疗效评估 |
| 寻找原发灶 |
| 肿瘤复发监测 |

**SUV值**（标准摄取值）：
- 良性病变：SUV<2.5
| 恶性病变：SUV>2.5 |

---

#### 23. 骨扫描

**费曼解释**：骨扫描是**给骨头照相**。放射性药物跑到骨骼，有病变的地方吸收特别多，显示为"热点"。

**适应症**：
- 骨转移瘤
| 原发性骨肿瘤 |
| 骨折 |
| 骨髓炎 |
| 骨代谢疾病 |

**特点**：
- 可全身成像
| 发现早期骨病变 |
| 敏感性高 |

---

## 第六编：介入放射学

### 十、介入放射学

#### 24. 介入放射学概述

**费曼解释**：介入放射学是**边看边做**。在影像设备引导下，用细管子（导管）进入血管或器官，进行诊断和治疗。不用开大刀。

**学术定义**：在影像设备引导下，经皮穿刺或通过人体生理腔道，将特制的导管或器械送至病变部位进行诊断和治疗的技术。

**特点**：
- 微创
| 恢复快 |
| 可重复 |
| 并发症少 |

---

#### 25. 血管介入

**经皮冠状动脉介入(PCI)**：
- 适应症：冠心病
| 方法：穿刺股动脉或桡动脉，送入导管到冠状动脉，球囊扩张+支架植入 |

**动脉栓塞术**：
- 适应症：止血、肿瘤治疗
| 方法：经导管注入栓塞材料，阻断血流 |

**血管成形术**：
- 适应症：血管狭窄
| 方法：球囊扩张、支架植入 |

**动脉溶栓**：
- 适应症：急性脑梗死
| 方法：经导管注入溶栓药物 |

---

#### 26. 非血管介入

**穿刺活检**：
- 适应症：肿瘤定性
| 方法：影像引导下穿刺取组织 |

**引流术**：
- 适应症：脓肿、积液
| 方法：影像引导下置入引流管 |

**射频消融**：
- 适应症：肝癌、肺癌
| 方法：影像引导下穿刺，加热杀死肿瘤 |
