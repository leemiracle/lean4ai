> 医学二级学科代码：32070

---

## 医学影像学概览

**费曼解释**：
医学影像学是**用各种技术给身体拍照**。看清楚身体内部的病变，帮助诊断疾病。

---

## 第一编：X线

### 一、X线检查

#### 1. X线 (X-ray)

**费曼解释**：
X线是**能穿透身体的射线**。骨头挡住X线，显示白色；肺部空气让X线通过，显示黑色。

**应用**：
| 部位 | 用途 |
|------|------|
| 胸部 | 肺炎、结核、肿瘤 |
| 骨骼 | 骨折、关节脱位 |
| 腹部 | 肠梗阻、结石 |

**优点**：便宜、快速
**缺点**：有辐射、分辨率有限

---

## 第二编：CT

### 二、CT扫描

#### 2. CT (Computed Tomography)

**费曼解释**：
CT是**3D的X线**。从多个角度拍摄，电脑合成横断面图像。看得更清楚。

**应用**：
| 部位 | 用途 |
|------|------|
| 头颅 | 脑出血、肿瘤 |
| 胸部 | 肺结节、肺癌筛查 |
| 腹部 | 肝癌、胰腺炎 |

**优点**：分辨率高
**缺点**：辐射较大

---

## 第三编：MRI

### 三、磁共振成像

#### 3. MRI (Magnetic Resonance Imaging)

**费曼解释**：
MRI用**磁场和无线电波**成像，没有辐射。看软组织特别清楚。

**应用**：
| 部位 | 用途 |
|------|------|
| 大脑 | 脑肿瘤、脑梗死 |
| 脊柱 | 椎间盘突出 |
| 关节 | 韧带损伤 |

**优点**：无辐射、软组织分辨率高
**缺点**：贵、时间长、有金属不能做

---

## 第四编：超声

### 四、超声检查

#### 4. 超声 (Ultrasound)

**费曼解释**：
超声用**声波反射成像**。没有辐射，安全。看胎儿最常用。

**应用**：
| 部位 | 用途 |
|------|------|
| 腹部 | 肝胆胰脾、肾 |
| 妇产 | 胎儿检查 |
| 心脏 | 心脏结构、功能 |
| 血管 | 血管狭窄、血栓 |

**优点**：无辐射、实时、便宜
**缺点**：受气体和骨骼干扰

---

## 第五编：核医学

### 五、核医学检查

#### 5. PET-CT

**费曼解释**：
PET-CT是**代谢+解剖图像融合**。注射示踪剂，显示代谢活跃的部位，找肿瘤转移灶最有效。

**应用**：
- 肿瘤诊断和分期
- 肿瘤疗效评估
- 寻找原发灶
