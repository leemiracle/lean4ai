# 医学心理学

> 医学交叉学科

---

## 医学心理学概览

**费曼解释**：
医学心理学研究**心理和身体健康的关系**。心情影响身体，身体影响心情。治疗疾病要身心兼顾。

---

## 第一编：心身疾病

### 一、心身疾病

#### 1. 心身疾病 (Psychosomatic Disease)

**费曼解释**：
心身疾病是**心理因素引起的身体疾病**。压力大导致胃溃疡、高血压、哮喘加重。

**学术定义**：心理社会因素在发病、发展过程中起重要作用的躯体疾病。

**常见心身疾病**：
| 疾病 | 心理因素 |
|------|----------|
| 高血压 | 长期紧张、焦虑 |
| 冠心病 | A型性格、压力 |
| 胃溃疡 | 焦虑、紧张 |
| 哮喘 | 情绪波动 |
| 偏头痛 | 压力、紧张 |

---

## 第二编：心理干预

### 二、心理治疗

#### 2. 心理治疗 (Psychotherapy)

**费曼解释**：
心理治疗是**用谈话治疗心理问题**。改变不良认知，学习应对技能，缓解情绪问题。

**常用方法**：
| 方法 | 说明 |
|------|------|
| 认知行为治疗 | 改变不良认知和行为 |
| 支持性心理治疗 | 倾听、支持、鼓励 |
| 催眠治疗 | 诱导催眠状态 |

---

### 三、健康行为

#### 3. 健康行为 (Health Behavior)

**费曼解释**：
健康行为是**对健康有利的行为**。合理膳食、适量运动、戒烟限酒、心理平衡。

**四大基石**：
1. 合理膳食
2. 适量运动
3. 戒烟限酒
4. 心理平衡
