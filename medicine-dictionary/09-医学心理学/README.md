# 医学心理学

> 医学交叉学科

---

## 医学心理学概览

**费曼解释**：
医学心理学研究**心理和身体健康的关系**。心情影响身体，身体影响心情。心理因素可以导致疾病，疾病也会影响心理。

**核心观点**：
- 心身一体：心理和身体不可分割
- 生物-心理-社会医学模式：疾病是多种因素的结果
- 整体观：治疗疾病要考虑心理社会因素

---

## 第一编：医学心理学基础

### 一、心理学基础概念

#### 1. 心理现象 (Psychological Phenomena)

**费曼解释**：
心理现象包括**心理过程**和**个性心理**。
- 心理过程：认知、情绪、意志
- 个性心理：能力、气质、性格

**结构**：
```
心理现象
├── 心理过程
│   ├── 认知过程（感觉、知觉、记忆、思维、想象）
│   ├── 情绪情感过程
│   └── 意志过程
└── 个性心理
    ├── 个性倾向性（需要、动机、兴趣、信念）
    └── 个性心理特征（能力、气质、性格）
```

---

#### 2. 认知过程 (Cognitive Process)

**费曼解释**：认知过程是**认识世界的过程**。

**感觉**：
- 定义：直接作用于感觉器官的客观事物的个别属性的反映
- 分类：视觉、听觉、嗅觉、味觉、触觉等

**知觉**：
- 定义：直接作用于感觉器官的客观事物的整体属性的反映
- 特性：整体性、选择性、理解性、恒常性

**记忆**：
- 定义：对过去经验的反映
- 过程：识记→保持→再认/回忆
- 分类：瞬时记忆（<1秒）、短时记忆（1分钟内）、长时记忆

**思维**：
- 定义：对客观事物间接、概括的反映
- 特征：间接性、概括性

**想象**：
- 定义：对已有表象进行加工改造而形成新形象的过程

---

#### 3. 情绪与情感 (Emotion and Affect)

**费曼解释**：情绪是**对客观事物的态度体验**。

**情绪与情感的区别**：
| 特征 | 情绪 | 情感 |
|------|------|------|
| 稳定性 | 情境性、易变 | 稳定、持久 |
| 深度 | 表层 | 深层 |
| 表现 | 外显 | 内隐 |
| 例子 | 愤怒、恐惧 | 爱国、责任感 |

**基本情绪**：
- 快乐
- 愤怒
- 悲哀
- 恐惧

**情绪的功能**：
1. 适应功能
2. 动机功能
3. 组织功能
4. 信号功能

**情绪与健康**：
- 积极情绪：增强免疫力
- 消极情绪：降低免疫力，导致疾病

---

#### 4. 意志过程 (Volition Process)

**费曼解释**：意志是**自觉确定目标、克服困难、实现目标的心理过程**。

**意志的特征**：
1. 明确的目的性
2. 与克服困难相联系
3. 以随意运动为基础

**意志的品质**：
| 品质 | 含义 |
|------|------|
| 自觉性 | 能主动支配自己的行动 |
| 果断性 | 善于明辨是非，迅速决策 |
| 坚韧性 | 坚持不懈，百折不挠 |
| 自制力 | 控制自己的情绪和言行 |

---

#### 5. 个性心理 (Personality)

**费曼解释**：个性是一个人**独特的、稳定的心理特征总和**。

**能力**：
- 定义：顺利完成某种活动所必需的心理特征
- 分类：一般能力（智力）、特殊能力
- 智商(IQ)：= 智力年龄 / 实际年龄 × 100
- 正态分布：IQ 90-110为正常

**气质**：
- 定义：与生俱来的心理活动动力特征
- 四种类型：
| 类型 | 特点 | 适合职业 |
|------|------|----------|
| 多血质 | 灵活、活泼 | 记者、演员 |
| 胆汁质 | 冲动、热情 | 运动员、探险家 |
| 黏液质 | 安静、稳重 | 会计、管理者 |
| 抑郁质 | 敏感、内向 | 科学家、艺术家 |

**性格**：
- 定义：对现实稳定的态度和习惯化的行为方式
- 特征：态度特征、意志特征、情绪特征、理智特征
- 可塑性强，可以改变

---

### 二、应激与应对

#### 6. 应激 (Stress)

**费曼解释**：应激是**身体对压力的反应**。适度的压力让人进步，过度的压力让人崩溃。

**学术定义**：个体在觉察到需求与能力不平衡时产生的身心紧张状态。

**应激源分类**：
| 类型 | 例子 |
|------|------|
| 躯体性 | 疾病、疼痛、手术 |
| 心理性 | 焦虑、恐惧、挫折 |
| 社会性 | 失业、离婚、亲人死亡 |
| 文化性 | 移民、语言障碍 |

**应激反应**：
1. **生理反应**：心率加快、血压升高、血糖升高
2. **心理反应**：焦虑、恐惧、愤怒、抑郁
3. **行为反应**：逃避、攻击、依赖

**应激与健康**：
- 急性应激：短暂，可恢复
- 慢性应激：长期，导致疾病（高血压、冠心病、溃疡）

---

#### 7. 应对方式 (Coping Strategies)

**费曼解释**：应对是**处理压力的方法**。有的方法健康，有的方法有害。

**应对分类**：
| 类型 | 方式 | 例子 |
|------|------|------|
| 问题导向应对 | 解决问题 | 制定计划、寻求帮助 |
| 情绪导向应对 | 调节情绪 | 运动、倾诉、放松 |

**积极应对**：
- 寻求社会支持
- 积极重新评估
- 问题解决

**消极应对**：
- 回避
| 否认
- 酗酒、吸烟
- 攻击

---

### 三、心身关系

#### 8. 心身医学 (Psychosomatic Medicine)

**费曼解释**：心身医学研究**心理因素如何影响身体健康**。

**基本观点**：
1. 心理因素参与疾病的发生发展
2. 躯体疾病会引起心理反应
3. 心理干预可以辅助治疗躯体疾病

**心理生理机制**：
```
心理应激
    ↓
下丘脑激活
    ↓
├→ 交感神经激活 → 儿茶酚胺↑ → 心率↑、血压↑
├→ HPA轴激活 → 皮质醇↑ → 免疫抑制
└→ 免疫系统变化 → 易感染、肿瘤风险↑
```

## 第二编：心身疾病

### 四、心身疾病概述

#### 9. 心身疾病 (Psychosomatic Disease)

**费曼解释**：心身疾病是**心理因素引起的身体疾病**。

**学术定义**：心理社会因素在发病、发展过程中起重要作用的躯体器质性疾病和功能性障碍。

**特点**：
1. 有明确的躯体症状
2. 心理社会因素是重要病因
3. 情绪影响症状波动
4. 纯生物治疗效果不佳

**分类**：
| 系统 | 常见心身疾病 |
|------|--------------|
| 心血管系统 | 高血压、冠心病、心律失常 |
| 消化系统 | 消化性溃疡、肠易激综合征 |
| 呼吸系统 | 支气管哮喘 |
| 内分泌系统 | 甲状腺功能亢进、糖尿病 |
| 皮肤系统 | 神经性皮炎、荨麻疹 |
| 神经系统 | 偏头痛、紧张性头痛 |
| 泌尿生殖系统 | 性功能障碍 |

---

#### 10. 常见心身疾病

**原发性高血压**：
- 心理因素：长期紧张、焦虑、A型性格
- 发病机制：交感神经持续兴奋，血管收缩
- 心理干预：放松训练、生物反馈

**冠心病**：
- 心理因素：A型性格（竞争、急躁、敌意）
- 心理干预：行为矫正、压力管理

**消化性溃疡**：
- 心理因素：焦虑、紧张、压抑
- 发病机制：胃酸分泌增加，黏膜血流量减少
- 心理干预：减压、放松

**支气管哮喘**：
- 心理因素：情绪波动可诱发
- 心理干预：情绪管理、放松训练

**糖尿病**：
- 心理因素：应激影响血糖控制
- 心理干预：压力管理、健康教育

---

### 五、心身疾病的诊断与治疗

#### 11. 心身疾病的诊断

**诊断标准**：
1. 有明确的躯体症状或病理改变
2. 心理社会因素与疾病发生发展有关
3. 排除单纯的躯体疾病或精神障碍

**评估方法**：
- 心理测验（SAS、SDS、SCL-90）
- 生活事件量表
- 社会支持评定量表
- 人格测验（MMPI、EPQ）

---

#### 12. 心身疾病的治疗

**综合治疗原则**：
1. 躯体治疗（药物、手术）
2. 心理治疗
3. 行为干预
4. 社会支持

**常用心理治疗方法**：
| 方法 | 适应症 |
|------|--------|
| 支持性心理治疗 | 各种心身疾病 |
| 认知行为治疗 | 高血压、溃疡 |
| 生物反馈 | 高血压、偏头痛 |
| 放松训练 | 焦虑相关心身疾病 |
| 催眠治疗 | 疼痛、哮喘 |

## 第三编：患者心理

### 六、患者角色

#### 13. 患者角色 (Patient Role)

**费曼解释**：患者角色是社会给病人的**特殊身份**，有权利也有义务。

**患者角色的特征**：
1. 免除正常的社会责任
2. 不需要对自己的疾病负责
3. 有义务寻求医疗帮助
4. 有义务配合治疗、努力康复

**患者角色适应不良**：
| 类型 | 表现 |
|------|------|
| 角色缺如 | 否认疾病，拒绝患者身份 |
| 角色冲突 | 患者角色与其他角色冲突 |
| 角色减退 | 忽视疾病，继续工作 |
| 角色强化 | 疾病治愈后仍不想脱离患者角色 |

---

#### 14. 患者心理需要

**基本需要**：
1. **安全需要**：希望得到安全的诊疗
2. **被尊重的需要**：希望得到尊重和理解
3. **信息需要**：希望了解病情和治疗方案
4. **被接纳的需要**：希望被医务人员和家属接纳
5. **康复需要**：希望尽快康复

---

#### 15. 患者的情绪反应

**常见情绪反应**：
| 情绪 | 原因 | 处理 |
|------|------|------|
| 焦虑 | 不确定性、疾病威胁 | 提供信息、支持 |
| 恐惧 | 未知、疼痛、死亡 | 解释、陪伴 |
| 抑郁 | 疾病负担、预后差 | 心理干预、必要时药物 |
| 愤怒 | 不满、挫折 | 理解、倾听 |
| 孤独 | 住院、隔离 | 陪伴、沟通 |

---

#### 16. 慢性病患者心理

**慢性病心理特点**：
1. 长期痛苦
2. 社会功能下降
3. 经济负担
4. 人际关系变化

**心理分期**：
| 阶段 | 心理反应 |
|------|----------|
| 否认期 | 不相信、否认 |
| 愤怒期 | 怨天尤人 |
| 协议期 | 试图与疾病和解 |
| 抑郁期 | 悲观、绝望 |
| 接受期 | 接受现实，积极配合 |

---

#### 17. 临终患者心理

**库伯勒-罗斯的五阶段**：
1. **否认期**："这不可能是我"
2. **愤怒期**："为什么是我？"
3. **协议期**："让我活到..."
4. **抑郁期**：悲伤、沉默
5. **接受期**：平静面对

**临终关怀原则**：
- 减轻痛苦
- 维护尊严
- 满足愿望
- 陪伴支持

---

## 第四编：心理干预

### 七、心理治疗

#### 18. 心理治疗概述

**费曼解释**：心理治疗是**用心理学方法治疗心理问题**。像心灵的治疗师，用谈话、行为训练等方法帮助患者。

**学术定义**：治疗者运用心理学的理论和技术，通过言语和非言语的沟通方式，对患者施加影响，以消除或减轻症状、改善心理状态的治疗方法。

**基本要素**：
1. 治疗者（受过专业训练）
2. 患者（有求助动机）
3. 治疗理论和技术
4. 治疗关系

**适应症**：
- 神经症
- 心身疾病
- 适应障碍
- 人格障碍
- 情感障碍
- 创伤后应激障碍

---

#### 19. 支持性心理治疗

**费曼解释**：支持性心理治疗是**最基本的心理治疗**，通过倾听、安慰、鼓励、指导来帮助患者。

**基本技术**：
1. **倾听**：专心听患者诉说
2. **共情**：理解患者的感受
3. **解释**：帮助患者理解问题
4. **指导**：给予适当的建议
5. **鼓励**：增强患者信心
6. **安慰**：减轻患者痛苦

**原则**：
- 不评判
- 不批评
- 不强迫
- 尊重患者

---

#### 20. 认知行为治疗 (CBT)

**费曼解释**：认知行为治疗是**改变想法、改变行为**。错误的认知导致情绪问题，纠正认知就能改善情绪。

**核心理论**：
```
事件 → 认知 → 情绪 → 行为
       ↑
    可以改变
```

**认知歪曲类型**：
| 类型 | 例子 |
|------|------|
| 非黑即白 | "不完美就是失败" |
| 过度概括 | "一次失败意味着总是失败" |
| 灾难化 | "这简直是灾难" |
| 情绪化推理 | "我觉得可怕，所以真的很可怕" |
| 应该陈述 | "我应该做到完美" |

**技术**：
1. 识别负性自动思维
2. 挑战不合理信念
3. 行为实验
4. 家庭作业

**适应症**：抑郁症、焦虑症、恐惧症、强迫症

---

#### 21. 精神分析治疗

**费曼解释**：精神分析治疗是**挖掘潜意识里的冲突**。很多问题源于童年的创伤，被压抑到潜意识里。

**核心概念**：
1. **潜意识**：意识不到的心理活动
2. **防御机制**：保护自己免受焦虑
3. **移情**：把对重要他人的感情转移到治疗者身上

**常用技术**：
- 自由联想
- 梦的分析
- 移情分析

---

#### 22. 人本主义治疗

**费曼解释**：人本主义治疗相信**人有自我成长的能力**，治疗者提供一个温暖、接纳的环境，让患者自己找到答案。

**代表人物**：罗杰斯（Carl Rogers）

**核心条件**：
1. **真诚一致**：治疗者真实、诚实
2. **无条件积极关注**：接纳患者，不评判
3. **共情理解**：准确理解患者的感受

**方法**：以来访者为中心，让来访者自己探索

---

#### 23. 行为治疗

**费曼解释**：行为治疗是**用学习原理改变不良行为**。行为是学来的，也可以重新学。

**基本原理**：
1. **经典条件反射**：巴甫洛夫的狗
2. **操作性条件反射**：斯金纳的老鼠
3. **观察学习**：班杜拉的社会学习理论

**常用技术**：
| 技术 | 原理 | 应用 |
|------|------|------|
| 系统脱敏 | 逐渐暴露 | 恐惧症 |
| 冲击疗法 | 强烈暴露 | 恐惧症 |
| 厌恶疗法 | 惩罚不良行为 | 酒精依赖 |
| 正强化 | 奖励良好行为 | 各种行为问题 |
| 消退 | 不再强化 | 不良习惯 |

---

#### 24. 家庭治疗

**费曼解释**：家庭治疗认为**问题不只在个人，而在家庭关系**。改变家庭关系，问题就解决了。

**核心观点**：
1. 家庭是一个系统
2. 一个人的问题是家庭问题的表现
3. 改变家庭互动模式

**适应症**：
- 青少年问题
| 婚姻问题
- 精神疾病的家庭干预

---

#### 25. 团体治疗

**费曼解释**：团体治疗是**一群人在治疗师带领下的治疗**。成员之间互相支持、互相学习。

**治疗因素**：
1. 团体凝聚力
2. 普适性（不是我一个人这样）
3. 人际学习
4. 利他（帮助别人也有助于自己）
5. 宣泄

**适应症**：
- 人际关系问题
- 适应障碍
- 成瘾问题

---

### 八、心理评估

#### 26. 心理测验

**费曼解释**：心理测验是**用标准化工具测量心理特征**。像量身高体重一样，心理特征也可以量化。

**测验分类**：
| 类型 | 例子 |
|------|------|
| 智力测验 | WAIS、WISC |
| 人格测验 | MMPI、EPQ、16PF |
| 神经心理测验 | HRB、MMSE |
| 临床评定量表 | SAS、SDS、SCL-90 |

---

#### 27. 常用心理评定量表

**焦虑自评量表 (SAS)**：
- 评估焦虑程度
- <50分：正常
- 50-59分：轻度焦虑
- 60-69分：中度焦虑
- ≥70分：重度焦虑

**抑郁自评量表 (SDS)**：
- 评估抑郁程度
- <53分：正常
- 53-62分：轻度抑郁
- 63-72分：中度抑郁
- >72分：重度抑郁

**症状自评量表 (SCL-90)**：
- 评估整体心理健康状况
- 包含9个因子：躯体化、强迫、人际敏感、抑郁、焦虑、敌对、恐怖、偏执、精神病性

**生活事件量表 (LES)**：
- 评估生活事件压力
- 帮助了解患者的生活压力

## 第五编：特定领域医学心理学

### 九、疼痛心理学

#### 28. 疼痛的心理学

**费曼解释**：疼痛不只是身体的感觉，**心理因素也会影响疼痛**。

**疼痛的心理因素**：
1. 注意：越关注越痛
2. 情绪：焦虑、抑郁加重疼痛
3. 意义：认为疼痛有意义，痛感减轻
4. 既往经验：有过疼痛经历，痛感可能增强或减弱
5. 文化背景：不同文化对疼痛的表达不同

**慢性疼痛的心理干预**：
- 认知行为治疗
- 放松训练
- 生物反馈
| 催眠治疗

---

### 十、睡眠心理学

#### 29. 睡眠障碍

**费曼解释**：睡眠障碍和心理因素密切相关。**焦虑让人睡不着，睡不着又让人更焦虑**。

**失眠的心理因素**：
- 焦虑、抑郁
- 过度担心睡眠
- 不良睡眠习惯
- 条件性失眠（床=清醒）

**失眠的心理治疗**：
1. 睡眠卫生教育
2. 刺激控制疗法（床只用于睡眠）
3. 睡眠限制疗法
4. 认知治疗（纠正对睡眠的错误认知）
5. 放松训练

---

### 十一、健康行为

#### 30. 健康行为

**费曼解释**：健康行为是**有利于健康的行为**。有些行为损害健康，有些行为促进健康。

**健康行为**：
- 合理膳食
- 适量运动
- 充足睡眠
- 定期体检
- 遵医嘱用药

**危害健康行为**：
- 吸烟
- 过度饮酒
- 缺乏运动
- 不健康饮食
| 不遵医嘱

**行为改变理论**：
- 知-信-行模式（KAP）
- 健康信念模型（HBM）
- 行为改变阶段理论

**行为改变阶段**：
| 阶段 | 特点 |
|------|------|
| 前意向阶段 | 没有改变打算 |
| 意向阶段 | 有改变打算，但未行动 |
| 准备阶段 | 准备采取行动 |
| 行动阶段 | 正在改变行为 |
| 维持阶段 | 保持改变6个月以上 |

---

## 第六编：医患关系

### 十二、医患沟通

#### 31. 医患关系模式

**费曼解释**：医患关系是医生和病人之间的关系，有不同模式。

**三种模式**：
| 模式 | 特点 | 适用 |
|------|------|------|
| 主动-被动 | 医生主导，病人被动 | 昏迷、严重疾病 |
| 指导-合作 | 医生指导，病人配合 | 急性疾病 |
| 共同参与 | 医患共同决策 | 慢性疾病 |

**理想模式**：共同参与模式

---

#### 32. 医患沟通技巧

**基本原则**：
1. 尊重患者
2. 真诚倾听
3. 共情理解
4. 清晰表达

**沟通技巧**：
| 技巧 | 说明 |
|------|------|
| 开放式提问 | "您感觉怎么样？" |
| 闭式提问 | "您有发烧吗？" |
| 反馈 | 重复患者的话，确认理解 |
| 沉默 | 给患者思考时间 |
| 共情 | "我能理解您的感受" |

**告知坏消息**（SPIKES模型）：
1. S（Setting）：设置环境
2. P（Perception）：了解患者认知
3. I（Invitation）：询问患者想知道多少
4. K（Knowledge）：告知信息
5. E（Empathy）：共情回应
6. S（Summary）：总结下一步计划

---

## 第七编：特殊人群心理

### 十三、儿童心理

#### 33. 儿童心理发展

**费曼解释**：儿童的心理发展有规律，不同年龄有不同的特点。

**发展阶段**（皮亚杰）：
| 阶段 | 年龄 | 特点 |
|------|------|------|
| 感知运动期 | 0-2岁 | 通过感知和动作认识世界 |
| 前运算期 | 2-7岁 | 自我中心，思维不可逆 |
| 具体运算期 | 7-11岁 | 逻辑思维，但需具体事物 |
| 形式运算期 | >11岁 | 抽象思维 |

**儿童患病心理**：
- 分离焦虑
- 退行行为
| 恐惧医疗操作

---

### 十四、老年心理

#### 34. 老年心理特点

**费曼解释**：老年人的心理有特殊性，需要特别关注。

**老年心理特点**：
1. 认知功能下降（记忆力、注意力）
2. 情绪变化（孤独、抑郁）
3. 人格变化（固执、保守）
4. 社会角色变化（退休、丧偶）

**常见老年心理问题**：
| 问题 | 表现 |
|------|------|
| 抑郁 | 情绪低落、兴趣减退 |
| 焦虑 | 过度担心健康 |
| 痴呆 | 记忆力下降、认知障碍 |
| 失眠 | 入睡困难、早醒 |
