## 医学是什么？（费曼式解释）

**想象你的身体是一台超级精密的机器：**

- 这台机器怎么组成的？→ 解剖学
- 这台机器怎么运转的？→ 生理学
- 这台机器哪里坏了？→ 病理学
- 怎么修这台机器？→ 临床医学
- 怎么防止机器坏？→ 预防医学
- 用什么材料修？→ 药学

**医学就是研究人体健康和疾病的科学：**
1. 人体结构功能是什么？（基础医学）
2. 疾病怎么诊断治疗？（临床医学）
3. 如何预防疾病？（预防医学）
4. 药物怎么起作用？（药学）
