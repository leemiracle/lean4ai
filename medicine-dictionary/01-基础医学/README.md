> 医学二级学科代码：32010

## 基础医学概览

**费曼解释**：
基础医学是**医学的地基**。就像盖房子要先打地基，学医要先学基础医学：
- 解剖学：身体长什么样？
- 生理学：身体怎么工作？
- 病理学：病的时候身体怎么变？
- 药理学：药怎么起作用？

---

## 第一编：人体解剖学

### 一、运动系统

#### 1. 骨骼 (Skeleton)

**费曼解释**：
骨骼是身体的**钢筋框架**。206块骨头搭成一个人体支架，保护内脏（头骨保护脑，肋骨保护心肺），支撑身体，还能造血（骨髓）。

**学术定义**：骨骼是由骨组织构成的器官系统，具有支持、保护、运动和造血功能。

**关键知识点**：
- 成人骨骼：206块
- 分类：长骨、短骨、扁骨、不规则骨
- 骨的成分：有机质（胶原蛋白）+ 无机质（钙磷）

**常见问题**：
- 骨质疏松：骨头变脆，容易骨折
- 骨折：骨头断裂

**相关概念**：关节、骨髓、骨骼肌

---

#### 2. 关节 (Joint)

**费曼解释**：
关节是**骨头的连接点**，让骨头能动。就像门的铰链，让门能开能关。关节里有滑液（润滑油），还有软骨（缓冲垫）。

**学术定义**：关节是骨与骨之间的连接结构，允许骨骼运动。

**分类**：
| 类型 | 活动度 | 例子 |
|------|--------|------|
| 不动关节 | 不能动 | 颅骨缝 |
| 微动关节 | 稍能动 | 耻骨联合 |
| 可动关节 | 活动自如 | 膝关节、髋关节 |

**常见问题**：
- 关节炎：关节发炎，疼痛
- 脱臼：骨头从关节里出来

---

#### 3. 肌肉 (Muscle)

**费曼解释**：
肌肉是身体的**发动机**。想抬手，大脑发令，肌肉收缩，拉动骨头，手就抬起来了。

人体有600多块肌肉，占体重约40%。

**学术定义**：肌肉是由肌纤维组成的收缩性组织，负责身体运动。

**三种肌肉**：
| 类型 | 特点 | 分布 |
|------|------|------|
| 骨骼肌 | 随意肌，可控制 | 附着在骨骼上 |
| 心肌 | 不随意肌，自动跳动 | 心脏 |
| 平滑肌 | 不随意肌 | 血管、胃肠 |

**常见问题**：
- 肌肉拉伤：肌肉纤维撕裂
- 肌无力：肌肉无力

---

### 二、循环系统

#### 4. 心脏 (Heart)

**费曼解释**：
心脏是身体的**水泵**。它不停地跳动（每分钟60-100次），把血液泵到全身，输送氧气和营养。拳头大小，一天跳动约10万次。

**学术定义**：心脏是循环系统的中心器官，通过节律性收缩推动血液流动。

**结构**：
- 4个腔：左心房、左心室、右心房、右心室
- 4个瓣：二尖瓣、三尖瓣、主动脉瓣、肺动脉瓣
- 3层壁：心内膜、心肌层、心外膜

**血液循环路线**：
```
右心室 → 肺动脉 → 肺（换气）→ 肺静脉 → 左心房 → 左心室 → 主动脉 → 全身 → 上下腔静脉 → 右心房
```

**常见问题**：
- 心肌梗死：心脏血管堵塞
- 心力衰竭：心脏泵血功能下降

---

#### 5. 血管 (Blood Vessel)

**费曼解释**：
血管是身体的**管道系统**。动脉像主管道，把血液从心脏送出去；静脉像回水管，把血液收回心脏；毛细血管像细管，负责和组织交换物质。

**学术定义**：血管是血液流通的管道系统。

**三种血管**：
| 类型 | 特点 | 功能 |
|------|------|------|
| 动脉 | 壁厚、弹性大 | 输送血液到组织 |
| 静脉 | 壁薄、有瓣膜 | 回收血液到心脏 |
| 毛细血管 | 壁极薄 | 物质交换 |

**常见问题**：
- 动脉硬化：血管变硬变窄
- 静脉曲张：静脉扩张迂曲

---

#### 6. 血液 (Blood)

**费曼解释**：
血液是身体的**运输队**。红细胞运氧气，白细胞打细菌，血小板补伤口，血浆运营养和废物。

**学术定义**：血液是在心血管系统中循环的液体组织。

**成分**：
| 成分 | 功能 | 正常值 |
|------|------|--------|
| 红细胞 | 运输氧气 | 男4.0-5.5×10¹²/L |
| 白细胞 | 免疫防御 | 4-10×10⁹/L |
| 血小板 | 止血 | 100-300×10⁹/L |
| 血浆 | 运输物质 | 占血液55% |

**血型**：
- ABO血型：A、B、AB、O
- Rh血型：Rh阳性、Rh阴性

---

### 三、呼吸系统

#### 7. 肺 (Lung)

**费曼解释**：
肺是身体的**气体交换站**。吸气，氧气进入肺；呼气，二氧化碳排出体外。肺有3亿个肺泡，展开面积相当于一个网球场。

**学术定义**：肺是呼吸系统的主要器官，负责气体交换。

**结构**：
- 左肺：2叶
- 右肺：3叶
- 肺泡：气体交换的基本单位

**呼吸过程**：
```
空气 → 鼻/口 → 咽 → 喉 → 气管 → 支气管 → 肺泡 ←→ 毛细血管
```

**常见问题**：
- 肺炎：肺部感染
- 哮喘：气道痉挛
- 肺癌：肺部恶性肿瘤

---

### 四、消化系统

#### 8. 胃 (Stomach)

**费曼解释**：
胃是身体的**食物加工厂**。食物进入胃，胃分泌胃酸和消化酶，把食物搅碎成糊状（食糜），然后送到小肠。

**学术定义**：胃是消化道扩大的囊状器官，具有储存食物、分泌胃液、消化食物的功能。

**结构**：
- 贲门：入口
- 胃底、胃体、胃窦
- 幽门：出口
- 胃腺：分泌胃酸、胃蛋白酶

**胃酸的作用**：
1. 激活胃蛋白酶
2. 杀灭细菌
3. 促进铁、钙吸收

**常见问题**：
- 胃炎：胃黏膜炎症
- 胃溃疡：胃黏膜缺损
- 胃癌：胃部恶性肿瘤

---

#### 9. 肝脏 (Liver)

**费曼解释**：
肝脏是身体的**化工厂**。500多种化学反应在这里进行：
- 解毒：把有毒物质变成无毒
- 合成：制造蛋白质、胆固醇、胆汁
- 储存：储存糖原、维生素、铁

**学术定义**：肝脏是人体最大的实质性器官，具有代谢、解毒、合成、分泌等多种功能。

**功能**：
| 功能 | 说明 |
|------|------|
| 代谢 | 糖、脂肪、蛋白质代谢 |
| 解毒 | 药物、毒素的分解 |
| 合成 | 白蛋白、凝血因子 |
| 分泌 | 胆汁（帮助消化脂肪） |
| 储存 | 糖原、维生素 |

**常见问题**：
- 肝炎：肝脏炎症
- 肝硬化：肝脏纤维化
- 脂肪肝：肝内脂肪堆积

---

#### 10. 小肠 (Small Intestine)

**费曼解释**：
小肠是身体的**营养吸收站**。食物在这里被彻底消化，营养被吸收到血液。小肠有5-7米长，内壁有无数绒毛，大大增加吸收面积。

**学术定义**：小肠是消化道中最长的部分，是消化和吸收的主要场所。

**分部**：
- 十二指肠：25cm，接收胆汁和胰液
- 空肠：2.5m，主要消化吸收
- 回肠：3.5m，吸收维生素B12、胆盐

**常见问题**：
- 肠炎：小肠炎症
- 肠梗阻：肠道阻塞

---

### 五、泌尿系统

#### 11. 肾脏 (Kidney)

**费曼解释**：
肾脏是身体的**净水器**。每天过滤180升血液，把废物和多余水分变成尿液排出，保留有用的东西。

**学术定义**：肾脏是泌尿系统的主要器官，具有过滤血液、生成尿液、调节水电解质平衡的功能。

**功能**：
1. 过滤：清除代谢废物
2. 调节：维持水电解质平衡
3. 内分泌：分泌肾素、促红细胞生成素

**肾脏的过滤单位**：
- 肾单位：约100万个
- 肾小球：过滤
- 肾小管：重吸收、分泌

**常见问题**：
- 肾炎：肾脏炎症
- 肾结石：肾脏内形成结石
- 肾衰竭：肾功能丧失

---

### 六、神经系统

#### 12. 大脑 (Brain)

**费曼解释**：
大脑是身体的**总指挥部**。140亿个神经元，指挥思考、运动、感觉、情感。重约1.4公斤，却消耗全身20%的氧气。

**学术定义**：大脑是中枢神经系统的主要部分，是思维、意识、运动的最高级中枢。

**主要结构**：
| 部位 | 功能 |
|------|------|
| 大脑皮层 | 思考、感觉、运动 |
| 小脑 | 平衡、协调运动 |
| 脑干 | 呼吸、心跳等生命中枢 |
| 丘脑 | 感觉中继站 |
| 下丘脑 | 调节体温、饥饿、口渴 |

**大脑分区**：
- 额叶：思考、运动
- 顶叶：感觉
- 颞叶：听觉、记忆
- 枕叶：视觉

**常见问题**：
- 脑卒中（中风）：脑血管堵塞或破裂
- 脑肿瘤：脑部肿瘤
- 阿尔茨海默病：老年痴呆

---

#### 13. 脊髓 (Spinal Cord)

**费曼解释**：
脊髓是身体的**信息高速公路**。大脑发出的命令通过脊髓传到身体各处，身体的感觉也通过脊髓传回大脑。

**学术定义**：脊髓是中枢神经系统的低级部分，是大脑和周围神经之间的传导通路。

**功能**：
1. 传导：上下传导神经冲动
2. 反射：完成简单的反射活动

**常见问题**：
- 脊髓损伤：导致瘫痪
- 脊髓炎：脊髓炎症

---

#### 14. 神经元 (Neuron)

**费曼解释**：
神经元是神经系统的**基本单位**。像一个信息收发站：树突接收信息，细胞体处理信息，轴突发送信息。

**学术定义**：神经元是神经系统的基本结构和功能单位，具有接受、整合、传导信息的能力。

**结构**：
- 细胞体：代谢中心
- 树突：接收信号
- 轴突：传导信号
- 突触：神经元之间的连接点

---

### 七、内分泌系统

#### 15. 激素 (Hormone)

**费曼解释**：
激素是身体的**化学信使**。内分泌腺（如甲状腺、胰腺）分泌激素到血液，激素随血流到达靶器官，调节各种功能。

**学术定义**：激素是由内分泌腺或细胞分泌的高效能生物活性物质，通过血液或淋巴运输到靶细胞发挥作用。

**主要激素**：
| 激素 | 来源 | 功能 |
|------|------|------|
| 胰岛素 | 胰腺 | 降低血糖 |
| 甲状腺素 | 甲状腺 | 调节代谢 |
| 肾上腺素 | 肾上腺 | 应激反应 |
| 生长激素 | 垂体 | 促进生长 |
| 性激素 | 性腺 | 性征发育 |

**常见问题**：
- 糖尿病：胰岛素缺乏或抵抗
- 甲状腺功能亢进：甲状腺素过多
- 甲状腺功能减退：甲状腺素过少

---

### 八、免疫系统

#### 16. 免疫系统 (Immune System)

**费曼解释**：
免疫系统是身体的**国防军**。识别敌我，消灭入侵的细菌、病毒，清除体内的异常细胞（如癌细胞）。

**学术定义**：免疫系统是机体识别和排除抗原性异物、维持自身稳定的防御系统。

**组成**：
| 组成 | 作用 |
|------|------|
| 皮肤黏膜 | 第一道防线 |
| 吞噬细胞 | 吞噬病原体 |
| 抗体 | 特异性结合抗原 |
| T细胞 | 细胞免疫 |
| B细胞 | 产生抗体 |

**两种免疫**：
- **先天免疫**：生来就有，非特异性
- **适应性免疫**：后天获得，特异性

**常见问题**：
- 免疫缺陷：免疫力低下
- 自身免疫病：免疫系统攻击自身
- 过敏：免疫反应过度

---

#### 17. 抗原与抗体 (Antigen and Antibody)

**费曼解释**：
- **抗原**：敌军的制服——任何能引起免疫反应的东西（细菌、病毒、花粉）
- **抗体**：我军的识别器——专门识别和结合特定抗原的蛋白质

**学术定义**：
- 抗原：能诱导免疫应答并与相应抗体或致敏淋巴细胞发生特异性反应的物质
- 抗体：B细胞受抗原刺激后分化为浆细胞产生的免疫球蛋白

**抗体类型**：
| 类型 | 功能 |
|------|------|
| IgG | 最主要抗体，中和毒素 |
| IgM | 最早出现，急性感染指标 |
| IgA | 黏膜免疫 |
| IgE | 过敏反应 |
| IgD | B细胞表面受体 |

---

### 九、生殖系统

#### 18. 生殖系统 (Reproductive System)

**费曼解释**：
生殖系统负责**繁衍后代**。男性产生精子，女性产生卵子，精卵结合形成受精卵，发育成新个体。

**男性生殖系统**：
- 睾丸：产生精子、分泌雄激素
- 附睾：储存和成熟精子
- 输精管：输送精子
- 前列腺：分泌前列腺液

**女性生殖系统**：
- 卵巢：产生卵子、分泌雌激素和孕激素
- 输卵管：输送卵子、受精场所
- 子宫：孕育胎儿
- 阴道：分娩通道

**常见问题**：
- 不孕不育：生殖功能障碍
- 前列腺增生：前列腺良性增大
- 卵巢囊肿：卵巢内囊性肿物

## 第二编：生理学

### 一、细胞生理

#### 19. 细胞膜 (Cell Membrane)

**费曼解释**：
细胞膜是细胞的**城墙**。控制什么东西能进细胞，什么东西能出细胞。

**学术定义**：细胞膜是包围细胞的外层结构，由脂质双分子层和蛋白质组成，具有选择性通透性。

**功能**：
1. 保护细胞
2. 物质运输
3. 信号传递

---

#### 20. 细胞呼吸 (Cellular Respiration)

**费曼解释**：
细胞呼吸是细胞的**燃烧过程**。葡萄糖+氧气→能量（ATP）+二氧化碳+水。就像汽车烧汽油产生动力。

**学术定义**：细胞呼吸是细胞在氧气参与下分解有机物、释放能量的过程。

**过程**：
```
葡萄糖 → 糖酵解 → 三羧酸循环 → 电子传递链 → ATP
```

---

### 二、神经生理

#### 21. 动作电位 (Action Potential)

**费曼解释**：
动作电位是神经信号的**电脉冲**。神经细胞膜上的离子快速流动，产生电信号，像电线上的电流一样传导。

**学术定义**：动作电位是可兴奋细胞受到刺激时产生的快速、可传播的膜电位变化。

**过程**：
1. 静息电位：-70mV
2. 去极化：Na⁺内流，电位上升
3. 复极化：K⁺外流，电位下降
4. 恢复静息

---

#### 22. 突触传递 (Synaptic Transmission)

**费曼解释**：
突触传递是神经元之间的**接力棒传递**。一个神经元释放化学物质（神经递质），另一个神经元接收，信号就这样传递下去。

**学术定义**：突触传递是神经冲动通过突触从一个神经元传递到另一个神经元的过程。

**过程**：
1. 动作电位到达突触前膜
2. 神经递质释放
3. 神经递质与突触后膜受体结合
4. 产生突触后电位

---

### 三、心血管生理

#### 23. 血压 (Blood Pressure)

**费曼解释**：
血压是血液对血管壁的**压力**。就像水管里的水压。心脏收缩时压力高（收缩压），心脏舒张时压力低（舒张压）。

**学术定义**：血压是血液在血管内流动时对血管壁的侧压力。

**正常值**：
| 指标 | 正常值 | 高血压标准 |
|------|--------|------------|
| 收缩压 | 90-139 mmHg | ≥140 mmHg |
| 舒张压 | 60-89 mmHg | ≥90 mmHg |

**影响因素**：
- 心输出量
- 外周阻力
- 血容量
- 血管弹性

---

#### 24. 心电图 (Electrocardiogram, ECG)

**费曼解释**：
心电图是心脏电活动的**记录图**。心脏跳动产生微弱电流，在皮肤上测量并记录下来，形成PQRST波形。

**学术定义**：心电图是心脏兴奋过程中的电变化在体表的记录。

**波形意义**：
| 波形 | 意义 |
|------|------|
| P波 | 心房除极 |
| QRS波群 | 心室除极 |
| T波 | 心室复极 |

---

### 四、呼吸生理

#### 25. 肺活量 (Vital Capacity)

**费曼解释**：
肺活量是**最大呼吸量**。用力吸气吸到不能再吸，然后用力呼气呼到不能再呼，呼出的气体总量就是肺活量。

**学术定义**：肺活量是最大吸气后所能呼出的最大气量。

**正常值**：
- 男性：约3500ml
- 女性：约2500ml

---

#### 26. 血氧饱和度 (Oxygen Saturation)

**费曼解释**：
血氧饱和度是**血红蛋白携带氧气的程度**。100%表示血红蛋白全部带氧，低于90%就是缺氧。

**学术定义**：血氧饱和度是血红蛋白与氧结合的程度，即血氧含量与血氧容量的百分比。

**正常值**：95-100%

---

### 五、肾脏生理

#### 27. 肾小球滤过率 (Glomerular Filtration Rate, GFR)

**费曼解释**：
肾小球滤过率是**肾脏过滤能力的指标**。每分钟肾脏能过滤多少毫升血液。数值越低，肾功能越差。

**学术定义**：肾小球滤过率是单位时间内两肾生成的原尿量。

**正常值**：
- 男性：120±25 ml/min
- 女性：95±20 ml/min

**分期**（慢性肾脏病）：
| 分期 | GFR (ml/min) | 肾功能 |
|------|--------------|--------|
| 1期 | ≥90 | 正常 |
| 2期 | 60-89 | 轻度下降 |
| 3期 | 30-59 | 中度下降 |
| 4期 | 15-29 | 重度下降 |
| 5期 | <15 | 肾衰竭 |

---

## 第三编：病理学

### 一、细胞损伤

#### 28. 细胞死亡 (Cell Death)

**费曼解释**：
细胞死亡有两种方式：
- **坏死**：意外死亡，细胞破裂，引起炎症
- **凋亡**：程序性死亡，细胞安静地"自杀"，不影响周围

**学术定义**：
- 坏死：病理性的、被动的细胞死亡
- 凋亡：生理性的、主动的程序性细胞死亡

**比较**：
| 特征 | 坏死 | 凋亡 |
|------|------|------|
| 原因 | 损伤 | 程序 |
| 过程 | 细胞肿胀破裂 | 细胞收缩碎裂 |
| 炎症 | 有 | 无 |
| 意义 | 病理 | 生理/病理 |

---

#### 29. 炎症 (Inflammation)

**费曼解释**：
炎症是身体的**防御反应**。细菌入侵或组织损伤时，身体派"救援队"（白细胞、抗体等）到现场，红、肿、热、痛是战斗的痕迹。

**学术定义**：炎症是具有血管系统的活体组织对致炎因子所致损害的防御反应。

**五大表现**：
1. 红（充血）
2. 肿（渗出）
3. 热（代谢增强）
4. 痛（介质刺激）
5. 功能障碍

**分类**：
- 急性炎症：起病急，病程短
- 慢性炎症：起病慢，病程长

---

### 二、肿瘤

#### 30. 肿瘤 (Tumor)

**费曼解释**：
肿瘤是细胞的**失控增殖**。正常细胞有序生长，肿瘤细胞疯狂分裂，形成肿块。

**学术定义**：肿瘤是机体在各种致瘤因子作用下，局部组织细胞增生所形成的新生物。

**良性 vs 恶性**：
| 特征 | 良性肿瘤 | 恶性肿瘤 |
|------|----------|----------|
| 生长速度 | 慢 | 快 |
| 生长方式 | 膨胀性 | 浸润性 |
| 转移 | 无 | 有 |
| 复发 | 少 | 多 |
| 危害 | 较小 | 大 |

**肿瘤命名**：
- 良性：组织来源+瘤（脂肪瘤、纤维瘤）
- 恶性：组织来源+癌/肉瘤（肺癌、骨肉瘤）

---

#### 31. 癌症 (Cancer)

**费曼解释**：
癌症是**恶性肿瘤的统称**。癌细胞不仅疯狂生长，还会转移到身体其他部位，破坏器官功能。

**学术定义**：癌症是起源于上皮组织的恶性肿瘤。

**癌症的特征**：
1. 无限增殖
2. 逃避凋亡
3. 血管生成
4. 侵袭转移
5. 逃避免疫

**常见癌症**：
| 癌症 | 高发人群 | 主要症状 |
|------|----------|----------|
| 肺癌 | 吸烟者 | 咳嗽、咯血 |
| 胃癌 | 中老年人 | 上腹痛、消瘦 |
| 肝癌 | 乙肝/肝硬化 | 肝区痛、黄疸 |
| 结直肠癌 | 中老年人 | 便血、排便习惯改变 |
| 乳腺癌 | 女性 | 乳房肿块 |

---

## 第四编：微生物学

### 一、细菌

#### 32. 细菌 (Bacteria)

**费曼解释**：
细菌是**单细胞微生物**。很多细菌是有益的（肠道益生菌），但有些会引起疾病（肺炎链球菌引起肺炎）。

**学术定义**：细菌是一类原核细胞型微生物，有细胞壁、细胞膜、细胞质，无核膜和核仁。

**细菌结构**：
- 细胞壁：保护
- 细胞膜：物质交换
- 细胞质：代谢场所
- 核质：遗传物质

**致病菌举例**：
| 细菌 | 引起疾病 |
|------|----------|
| 金黄色葡萄球菌 | 皮肤感染 |
| 大肠杆菌 | 肠道感染 |
| 结核杆菌 | 结核病 |
| 链球菌 | 扁桃体炎 |

---

### 二、病毒

#### 33. 病毒 (Virus)

**费曼解释**：
病毒是**最简单的生命形式**。只有遗传物质和蛋白质外壳，必须寄生在细胞里才能繁殖。

**学术定义**：病毒是一类非细胞型微生物，只含有一种核酸（DNA或RNA），必须在活细胞内寄生。

**病毒结构**：
- 核心：核酸（DNA或RNA）
- 衣壳：蛋白质
- 包膜：部分病毒有

**常见病毒**：
| 病毒 | 引起疾病 |
|------|----------|
| 流感病毒 | 流行性感冒 |
| 新冠病毒 | 新型冠状病毒肺炎 |
| 乙肝病毒 | 乙型肝炎 |
| HIV | 艾滋病 |

---

#### 34. 抗生素 (Antibiotic)

**费曼解释**：
抗生素是**杀细菌的药物**。青霉素是第一个抗生素，发现于1928年。注意：抗生素只杀细菌，不杀病毒！

**学术定义**：抗生素是由微生物产生的、能抑制或杀灭其他微生物的化学物质。

**分类**：
| 类型 | 作用机制 | 代表药物 |
|------|----------|----------|
| β-内酰胺类 | 抑制细胞壁合成 | 青霉素、头孢 |
| 大环内酯类 | 抑制蛋白质合成 | 红霉素、阿奇霉素 |
| 喹诺酮类 | 抑制DNA合成 | 左氧氟沙星 |

**注意**：滥用抗生素导致耐药菌
