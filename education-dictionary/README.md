## 三、使用说明

### 费曼学习法特色

每个概念采用**费曼学习法**进行解释：

1. **【概念】**：术语的精确定义
2. **【大白话】**：用日常语言解释（像给12岁孩子讲）
3. **【举个例子】**：具体的应用场景
4. **【核心要点】**：最关键的记忆点

### 词典特点

- **新**：纳入2020-2024年最新概念（如ChatGPT教育应用、双减政策等）
- **全**：覆盖教育学全部14个二级学科
- **准**：严格依据国家标准和专业教材
