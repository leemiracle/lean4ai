# 数量经济学（Quantitative Economics）

**学科代码**：020209  
**学科门类**：应用经济学

## 📌 学科简介

数量经济学运用数学和统计方法研究经济问题，包括数理经济学、计量经济学、经济预测等内容。它为经济学提供精确的分析工具和实证方法。

## 🎯 核心概念

### 一、数理经济学基础

#### 1. 最优化方法（Optimization Methods）

**费曼解释**：
经济学核心问题：在约束条件下，追求最优。

**无约束优化**：
- 求f(x)的最大/最小值
- 方法：求导=0

**有约束优化**：
- 在g(x)≤0条件下，最大化f(x)
- 方法：拉格朗日乘数法

**例子**：
- 消费者：收入约束下效用最大化
- 企业：成本约束下利润最大化

**核心要点**：
- 一阶条件、二阶条件
- 库恩-塔克条件
- 凸优化

**应用场景**：经济模型构建、决策分析

---

#### 2. 博弈论基础（Game Theory Basics）

**费曼解释**：
你的最优选择取决于对手的选择——**博弈**。

**要素**：
- 参与者
- 策略
- 收益

**均衡**：
- 纳什均衡：没人愿意单方面改变策略

**经典博弈**：
| 博弈 | 特点 |
|------|------|
| 囚徒困境 | 个人理性导致集体非理性 |
| 智猪博弈 | 强弱不对等 |
| 斗鸡博弈 | 谁先退让谁输 |

**核心要点**：
- 纯策略 vs 混合策略
- 重复博弈
- 不完全信息博弈

**应用场景**：产业组织、机制设计

---

### 二、计量经济学

#### 3. 经典线性回归模型（Classical Linear Regression）

**费曼解释**：
用数据找变量之间的关系。

**模型**：y = Xβ + ε

**经典假设**：
1. 线性关系
2. 误差均值为0
3. 同方差
4. 无自相关
5. 自变量与误差不相关
6. 误差正态分布

**OLS估计**：β̂ = (X'X)⁻¹X'y

**性质**：
- 无偏性
- 有效性（BLUE）
- 一致性

**核心要点**：
- 高斯-马尔可夫定理
- 假设检验
- 预测

**应用场景**：经济分析、政策评估

---

#### 4. 内生性与工具变量（Endogeneity and IV）

**费曼解释**：
回归发现问题：x和y可能互为因果，或者有遗漏变量——**内生性**。

**后果**：OLS估计有偏。

**解决方法**：工具变量（IV）

找一个变量z：
- 与x相关（相关性）
- 与y无关（排他性）

**例子**：研究教育对收入的影响
- 问题：能力影响教育和收入（遗漏变量）
- 工具变量：出生季度（影响受教育年限，但不直接影响收入）

**核心要点**：
- 内生性检验
- 弱工具变量
- 两阶段最小二乘（2SLS）

**应用场景**：因果推断、政策评估

---

#### 5. 面板数据模型（Panel Data Models）

**费曼解释**：
同时有时间维度和个体维度的数据。

**模型类型**：
| 模型 | 特点 |
|------|------|
| 混合OLS | 忽略个体差异 |
| 固定效应 | 控制个体异质性 |
| 随机效应 | 假设个体差异随机 |

**选择**：豪斯曼检验

**优势**：
- 控制不可观测的个体特征
- 增加样本量
- 分析动态变化

**核心要点**：
- 个体效应
- 时间效应
- 动态面板

**应用场景**：政策评估、企业研究

---

#### 6. 因果推断（Causal Inference）

**费曼解释**：
相关性≠因果性。如何识别因果？

**方法**：

| 方法 | 原理 |
|------|------|
| 随机对照试验（RCT） | 随机分组 |
| 双重差分（DID） | 政策前后+有无政策对比 |
| 断点回归（RDD） | 政策门槛附近对比 |
| 倾向得分匹配（PSM） | 配对相似个体 |

**例子**（DID）：
- 实验组：实施新政策的地区
- 对照组：未实施的地区
- 政策效应 = (实验组后-实验组前) - (对照组后-对照组前)

**核心要点**：
- 反事实框架
- 识别策略
- 稳健性检验

**应用场景**：政策评估、项目评价

---

### 三、时间序列计量

#### 7. 单位根与协整（Unit Root and Cointegration）

**费曼解释**：

**单位根**：时间序列不平稳（随机游走）

**检验**：ADF检验、PP检验

**协整**：两个非平稳序列的线性组合平稳

**例子**：
- 消费和收入都是非平稳的
- 但消费-收入的线性组合可能是平稳的
- 说明存在长期均衡关系

**误差修正模型（ECM）**：
- 短期波动 + 长期均衡调整

**核心要点**：
- 伪回归
- 协整检验
- 误差修正

**应用场景**：宏观经济分析

---

#### 8. VAR模型（Vector Autoregression）

**费曼解释**：
多个变量相互影响，谁影响谁？

**VAR**：每个变量都是自身和其他变量滞后值的函数

**例子**：
GDP增长 = f(GDP滞后, 通胀滞后, 利率滞后)
通胀 = f(GDP滞后, 通胀滞后, 利率滞后)

**分析工具**：
- 脉冲响应：一个冲击的动态影响
- 方差分解：每个变量的贡献

**核心要点**：
- 滞后阶数选择
- 脉冲响应函数
- 格兰杰因果检验

**应用场景**：宏观政策分析、预测

---

### 四、预测方法

#### 9. 经济预测方法（Economic Forecasting Methods）

**费曼解释**：
预测未来经济走势。

**方法**：
| 方法 | 特点 |
|------|------|
| 时间序列 | ARIMA、指数平滑 |
| 计量模型 | 结构模型、VAR |
| 机器学习 | 神经网络、随机森林 |
| 景气分析 | 先行指标 |
| 组合预测 | 多种方法结合 |

**评价**：
- 均方误差（MSE）
- 均方根误差（RMSE）
- 平均绝对百分比误差（MAPE）

**注意**：
- 预测误差不可避免
- 区间预测优于点预测
- 情景分析

**核心要点**：
- 样本外预测
- 预测评估
- 不确定性

**应用场景**：政策规划、企业决策

---

### 五、计算经济学

#### 10. 数值方法（Numerical Methods）

**费曼解释**：
很多经济问题没有解析解，需要用计算机数值求解。

**方法**：
| 方法 | 应用 |
|------|------|
| 迭代法 | 求解方程 |
| 蒙特卡洛模拟 | 积分、风险管理 |
| 动态规划 | 最优决策 |
| 值函数迭代 | 动态模型求解 |

**例子**：
- 资产定价模型
- 动态随机一般均衡（DSGE）
- 拍卖设计

**核心要点**：
- 收敛性
- 精度与效率
- 编程实现

**应用场景**：复杂模型求解、模拟分析

---

#### 11. 机器学习与经济学（Machine Learning in Economics）

**费曼解释**：
机器学习方法越来越多用于经济分析。

**应用**：
| 应用 | 方法 |
|------|------|
| 预测 | 随机森林、神经网络 |
| 分类 | 逻辑回归、SVM |
| 变量选择 | LASSO、弹性网络 |
| 因果推断 | 双重机器学习 |
| 文本分析 | NLP、情感分析 |

**注意**：
- ML擅长预测，不擅长因果
- 需要结合经济学理论
- 可解释性问题

**核心要点**：
- 预测vs因果
- 交叉验证
- 正则化

**应用场景**：大数据分析、预测建模
