# 天文学（0704）知识词典

## 学科概述
天文学是研究宇宙中天体和宇宙结构的科学，包括太阳、行星、恒星、星系以及宇宙的起源和演化。它是人类最古老的自然科学之一，用观测和理论研究我们在宇宙中的位置。

## 一级学科分类
根据《学位授予和人才培养学科目录（2020年）》，天文学下设：

- 070401 天体物理
- 070402 天体测量与天体力学
- 070403 天文技术与方法

## 核心知识领域

### 一、天体物理（070401）

#### 1. 太阳物理
- **太阳结构**
  - 费曼解释：太阳像一个巨大的洋葱，从内到外：核心（核聚变，1500万度）、辐射层（能量以光子形式传递）、对流层（热气上升冷气下沉）、光球（我们看到的光来自这里）、色球、日冕。
  
- **太阳活动**
  - 费曼解释：太阳不是静止的。太阳黑子是较暗较冷的区域（磁场强），耀斑是剧烈的能量释放，日冕物质抛射向太空喷射带电粒子。这些活动有11年周期。
  
- **太阳风**
  - 费曼解释：太阳持续向外喷发带电粒子流，速度400-800 km/s。太阳风撞击地球磁场产生极光，也会影响卫星和通信。太阳风一直吹到太阳系边缘。

#### 2. 恒星物理
- **恒星形成**
  - 费曼解释：星际气体和尘埃在引力作用下坍缩。温度升高，最终点燃氢聚变，恒星诞生。就像把雪球捏紧会变热，只不过规模大得多。原恒星被周围物质包裹，逐渐露出真容。
  
- **恒星结构**
  - 费曼解释：恒星是引力和压力的平衡。引力要把物质拉向中心，热压力和辐射压力要向外膨胀。核心进行核聚变，产生能量对抗引力。能量从核心传输到表面，辐射出去。
  
- **恒星演化**
  - 费曼解释：恒星的生命历程取决于质量。小质量恒星（<0.5太阳质量）平静燃烧千亿年。太阳质量恒星：主序→红巨星→白矮星。大质量恒星：超新星爆发→中子星或黑洞。
  
- **恒星死亡**
  - 费曼解释：燃料耗尽后，引力获胜。太阳会抛掉外层成为行星状星云，核心留下白矮星（地球大小，太阳质量）。大质量恒星爆炸（超新星），核心塌缩成中子星（城市大小）或黑洞。

#### 3. 行星科学
- **太阳系行星**
  - 费曼解释：8大行星分两类。类地行星（水、金、地、火）：岩石金属，小而密。类木行星（木、土、天、海）：气体巨行星，大而轻。冥王星被降级为矮行星。
  
- **系外行星**
  - 费曼解释：围绕其他恒星的行星。已发现5000多颗。用凌星法（行星挡住恒星，光度微降）和视向速度法（恒星被行星拉扯，光谱红移蓝移）探测。寻找宜居行星是目标。
  
- **行星大气**
  - 费曼解释：行星大气的成分和性质。金星大气96%是CO₂，温室效应极强（460°C）。火星大气稀薄。木星大气有风暴（大红斑）。大气层决定行星的气候和宜居性。

#### 4. 星系天文
- **银河系**
  - 费曼解释：我们的家园。棒旋星系，直径10万光年，包含2000-4000亿颗恒星。太阳在猎户臂上，距中心2.6万光年，绕中心一圈要2.3亿年。中心有个超大质量黑洞（400万太阳质量）。
  
- **星系分类**
  - 费曼解释：哈勃分类法。椭圆星系（老恒星，无新恒星形成）、旋涡星系（旋臂，活跃的恒星形成）、不规则星系（形状不规则）。我们的银河系是棒旋星系。
  
- **星系演化**
  - 费曼解释：星系会合并和相互作用。银河系正在吞噬一些小星系，将来会与仙女座星系相撞（40亿年后）。星系的形状和恒星形成历史反映其演化历程。
  
- **活动星系**
  - 费曼解释：中心黑洞正在吞噬物质，释放巨大能量。类星体是最亮的活动星系核，远在几十亿光年外，却比近处的星系还亮。能量来源是黑洞吸积。

#### 5. 宇宙学
- **宇宙大爆炸**
  - 费曼解释：宇宙起源于138亿年前的一次"爆炸"。不是在空间中爆炸，而是空间本身在膨胀。早期宇宙又热又密，随膨胀冷却，形成基本粒子、原子核、原子，最后恒星和星系。
  
- **宇宙膨胀**
  - 费曼解释：哈勃发现遥远的星系在远离我们，越远跑得越快。不是星系在运动，而是空间在膨胀。把宇宙想象成气球，星系是气球上的点，吹大气球，点与点距离增加。
  
- **暗物质**
  - 费曼解释：星系旋转速度告诉我们，可见物质不足以提供足够的引力。必须存在看不见的物质——暗物质。占宇宙物质的85%，但不发光，不知是什么。寻找暗物质粒子是热点。
  
- **暗能量**
  - 费曼解释：宇宙膨胀在加速！某种神秘的力量在推动。叫暗能量，占宇宙的68%。可能是真空的能量，但理论预测值是观测值的10¹²⁰倍——物理学最大的危机之一。
  
- **宇宙微波背景**
  - 费曼解释：大爆炸留下的余晖。宇宙早期是等离子体，光子被散射。38万年后，宇宙冷却到3000K，电子与原子核结合，光子自由飞行。这些光子今天温度降到2.7K，充满全宇宙。

#### 6. 高能天体物理
- **脉冲星**
  - 费曼解释：旋转的中子星，发出周期性的脉冲信号。像灯塔的光束，每转一圈扫过地球一次。最快的毫秒脉冲星每秒转700多圈。脉冲星是精确的时钟，可用于探测引力波。
  
- **黑洞**
  - 费曼解释：引力强到连光都逃不掉。事件视界是"不归点"。黑洞不发光，但正在吞噬物质时会发出X射线。超大质量黑洞在星系中心，恒星级黑洞是大质量恒星的残骸。
  
- **伽马射线暴**
  - 费曼解释：宇宙中最猛烈的爆炸。短时间的伽马射线爆发，能量相当于太阳一生的能量在几秒内释放。可能是中子星合并或大质量恒星塌缩。2017年探测到中子星合并的引力波和伽马暴。
  
- **引力波**
  - 费曼解释：时空的涟漪。加速的质量会产生引力波，以光速传播。LIGO在2015年首次探测到，来自两个黑洞合并。引力波天文学打开了一扇观测宇宙的新窗户。

### 二、天体测量与天体力学（070402）

#### 1. 天体测量学
- **天球坐标系**
  - 费曼解释：想象地球在天球中心，天球是无限远的球面。赤道坐标系用赤经和赤纬（类似地球的经纬度）定位天体。不同坐标系用于不同目的。
  
- **恒星位置测量**
  - 费曼解释：精确测量恒星的方向。Gaia卫星测量了10亿颗恒星的位置、距离和运动，精度达到微角秒（百万分之一角秒）。这相当于从地球测量月球上硬币的大小。
  
- **距离测量**
  - 费曼解释：三角视差法——地球绕太阳，近处恒星相对远处背景移动，测量角度变化算距离。适用于几百光年内。更远用标准烛光（造父变星、Ia型超新星）。
  
- **自行运动**
  - 费曼解释：恒星在天空中缓慢移动。测量两个时刻的位置，差值就是自行。大多很小，但最近的恒星明显。自行告诉我们恒星在银河系中的运动。

#### 2. 天体力学
- **开普勒定律**
  - 费曼解释：第一定律：行星轨道是椭圆，太阳在焦点；第二定律：行星与太阳的连线在相等时间内扫过相等面积（角动量守恒）；第三定律：轨道周期的平方与半长轴的立方成正比。
  
- **轨道摄动**
  - 费曼解释：真实轨道不是完美的椭圆。其他行星的引力、相对论效应、非球形天体的引力，都会使轨道偏离开普勒轨道。精确的轨道计算要考虑这些摄动。
  
- **三体问题**
  - 费曼解释：三个天体在引力作用下的运动。一般没有解析解，只能数值积分。混沌现象——初值的微小差异导致长期行为的巨大差异。天气预报难，三体问题更难。
  
- **人造卫星轨道**
  - 费曼解释：人造卫星的轨道设计和维护。近地轨道（LEO）、地球同步轨道（GEO）、太阳同步轨道。轨道选择取决于任务需求。大气阻力、地球扁率、日月引力都会影响轨道。

#### 3. 时间与历法
- **时间系统**
  - 费曼解释：世界时（UT）基于地球自转，原子时（TAI）基于原子钟，协调世界时（UTC）是 compromises。地球自转越来越慢（潮汐摩擦），需要不定期加闰秒。
  
- **历法**
  - 费曼解释：阳历（公历）基于地球绑太阳公转周期，阴历基于月球盈亏周期。农历是阴阳合历。置闰规则保证历法与季节同步。
  
- **天文时间**
  - 费曼解释：儒略日是从公元前4713年开始连续计数的天数，方便计算时间间隔。贝塞尔年用于定义历元。

#### 4. 天文参考系
- **国际天球参考系（ICRS）**
  - 费曼解释：用遥远类星体的位置定义的惯性参考系。类星体太远，自行可以忽略，位置基本固定。这是天体测量的基准。
  
- **岁差与章动**
  - 费曼解释：地球像陀螺一样进动，自转轴方向缓慢改变（岁差，周期26000年），还有小幅摆动（章动）。这使得天球坐标系随时间变化，需要修正。

### 三、天文技术与方法（070403）

#### 1. 光学望远镜
- **折射望远镜**
  - 费曼解释：用透镜聚光。物镜把来自天体的平行光聚焦，目镜放大成像。简单但有色差（不同颜色焦点不同），大口径透镜难制造。
  
- **反射望远镜**
  - 费曼解释：用凹面镜聚光。牛顿式用平面镜把焦点引到侧面，卡塞格林式用凸面镜把焦点引到后方。大型望远镜都是反射式，没有色差，镜面可以很大。
  
- **望远镜参数**
  - 费曼解释：口径决定聚光能力（收集的光量）和分辨率（能分辨的细节）。焦距决定视场和放大率。相对口径（焦比）F/D影响像质和曝光时间。
  
- **自适应光学**
  - 费曼解释：大气湍流使星像抖动模糊。自适应光学用可变形镜实时校正波前畸变，抵消大气影响。需要亮参考星或激光导星。图像可以接近衍射极限。

#### 2. 射电天文
- **射电望远镜**
  - 费曼解释：接收天体发出的无线电波。基本结构是天线+接收机。单天线分辨率低（波长长），用干涉仪提高分辨率。FAST是最大的单天线射电望远镜（500米）。
  
- **射电干涉仪**
  - 费曼解释：多个天线相隔很远，接收的信号相干叠加，等效于一个口径等于天线间距的望远镜。VLBI（甚长基线干涉）可以跨洲际，分辨率达微角秒。
  
- **射电观测对象**
  - 费曼解释：中性氢21cm线（绘制银河系结构），脉冲星，分子谱线（星际化学），射电星系，宇宙微波背景。射电波段不受天气和白天影响。

#### 3. 空间天文
- **空间望远镜**
  - 费曼解释：在大气层外观测，避免大气干扰。哈勃太空望远镜（光学）、詹姆斯·韦伯太空望远镜（红外）、钱德拉X射线天文台、费米伽马射线望远镜。
  
- **多波段天文**
  - 费曼解释：不同波段的辐射来自不同的物理过程。射电（冷气体、电子）、红外（尘埃、恒星形成）、光学（恒星）、紫外（热恒星）、X射线（高温气体、黑洞）、伽马射线（高能过程）。
  
- **太阳系探测**
  - 费曼解释：探测器飞掠、环绕、着陆、巡视。旅行者号飞出太阳系，好奇号在火星上行驶，新视野号飞掠冥王星。直接探测太阳系天体。

#### 4. 探测器技术
- **CCD与CMOS**
  - 费曼解释：把光转换为电信号。CCD（电荷耦合器件）量子效率高（可达90%+），线性好，是天文探测的主力。CMOS更便宜，在普及。
  
- **光谱仪**
  - 费曼解释：把光分解成光谱。棱镜或光栅分光。光谱分辨率R=λ/Δλ，高分辨率光谱揭示谱线细节，测量化学成分、温度、速度。
  
- **光度计**
  - 费曼解释：测量天体亮度。测光系统（UBVRI等）定义标准波段。光度测量用于变星观测、系外行星探测。

#### 5. 数据处理
- **图像处理**
  - 费曼解释：从原始数据提取科学信息。本底扣除、暗流校正、平场校正、去宇宙线、图像配准、叠加。把多次曝光的图像合并，提高信噪比。
  
- **光谱分析**
  - 费曼解释：从光谱提取信息。波长定标、流量定标、连续谱拟合、谱线识别、谱线拟合。测量谱线的波长（红移/速度）、等值宽度（丰度）、轮廓（物理条件）。
  
- **天文信息学**
  - 费曼解释：处理海量的天文数据。LSST每晚产生15TB数据。机器学习用于分类星系、识别变星、发现异常天体。虚拟天文台整合全球数据。

## 重要天文发现

### 古代
- 公元前2000年：巴比伦人记录日月食
- 公元前300年：阿里斯塔克提出日心说
- 公元150年：托勒密地心说

### 近代
- 1543年：哥白尼《天体运行论》
- 1609年：开普勒行星运动定律
- 1610年：伽利略用望远镜观测
- 1687年：牛顿万有引力定律

### 现代
- 1920s：哈勃发现宇宙膨胀
- 1965年：彭齐亚斯和威尔逊发现宇宙微波背景
- 1990s：发现暗能量、系外行星
- 2015年：LIGO探测到引力波
- 2019年：EHT拍摄黑洞照片

## 重要天文常数

- 天文单位（AU）：1.496×10⁸ km（地球到太阳平均距离）
- 光年（ly）：9.461×10¹² km（光走一年的距离）
- 秒差距（pc）：3.086×10¹³ km（视差1角秒的距离）
- 哈勃常数H₀：70 km/s/Mpc（宇宙膨胀率）
- 宇宙年龄：138亿年
- 可观测宇宙半径：465亿光年
