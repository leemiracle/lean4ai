## 📊 核心知识体系

### 军事思想体系
```
军事思想
├─ 古代军事思想（孙武、克劳塞维茨等）
├─ 近代军事思想（毛泽东军事思想等）
├─ 现代军事思想（信息化战争理论等）
└─ 未来战争理论（智能化战争等）
```

### 战争层次
```
战争层次
├─ 战略层（国家层面，决定打不打）
├─ 战役层（战区层面，决定怎么打）
└─ 战术层（部队层面，决定具体打法）
```

### 军事力量构成
```
军事力量
├─ 作战力量（陆海空天电网）
├─ 保障力量（后勤、装备、技术）
├─ 支援力量（情报、通信、气象）
└─ 动员力量（预备役、民兵）
```

### 现代战争形态
```
战争形态演进
├─ 冷兵器战争 → 热兵器战争
├─ 机械化战争 → 信息化战争
└─ 正迈向智能化战争
```
