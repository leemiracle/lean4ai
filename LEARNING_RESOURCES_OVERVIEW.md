## 五、核心要点

### 5.1 理解LLM（3点）

```
1. 本质：训练数据压缩 + 统计模式
2. 可靠性：常识★★★ | 专业★★☆ | 时效★☆☆
3. 边界：擅长语言/代码/创意，不擅长最新/推理/实操
```

### 5.2 Prompt设计（3点）

```
1. 公式：角色 + 任务 + 背景 + 约束 + 格式
2. 原则：清晰 + 具体 + 结构化 + 渐进 + 可验证
3. 技巧：角色设定 | 思维链 | Few-shot | 结构化
```

### 5.3 职业应用（3点）

```
1. 高价值：重复工作 | 创意发散 | 学习加速
2. 需谨慎：专业判断 | 精确数据 | 敏感信息
3. 不适合：实时操作 | 物理任务 | 最终决策
```

### 5.4 问题解决（3点）

```
1. 诊断：理解 → 信息 → 格式 → 深度 → 边界
2. 编程：明确栈、给代码、说错误
3. 数学：定水平、要求步骤、给例子
```
