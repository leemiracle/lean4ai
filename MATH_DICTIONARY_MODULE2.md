## 🌊 一、极限与连续性

### 1. 序列的极限 (Limit of Sequence)
**费曼解释**：想象你在无限接近一个目标，但永远不会完全到达。序列的极限就是这个"目标值"。

**ε-N定义**：
```
lim (n→∞) a_n = L 意味着：
对任意 ε > 0，存在 N 使得当 n > N 时，|a_n - L| < ε
```

**直观理解**：
- 给我任意小的误差要求（ε）
- 我总能找到一项N，从那项开始，所有项都在误差范围内
- 像射击：靶心是L，随着次数增加，越来越接近靶心

**例子**：
- lim (n→∞) 1/n = 0 （倒数越来越小）
- lim (n→∞) (1 + 1/n)^n = e （自然常数）

**重要定理**：
- **夹逼定理**：如果 a_n ≤ b_n ≤ c_n 且 lim a_n = lim c_n = L，则 lim b_n = L
- **单调收敛定理**：有界的单调序列必收敛

**Lean实现**：
```lean
-- ε-N定义
def ConvergesTo (a : ℕ → ℝ) (L : ℝ) : Prop :=
  ∀ ε > 0, ∃ N, ∀ n > N, |a n - L| < ε

-- 例子
example : ConvergesTo (fun n => 1 / (n + 1)) 0 := by
  intro ε hε
  -- 找到N使得1/(N+1) < ε
  use Nat.ceil (1/ε)
  intro n hn
  simp
  rw [div_le_iff]
  sorry
```

### 2. 函数的极限 (Limit of Function)
**费曼解释**：当输入无限接近某个点时，输出接近什么值？

**ε-δ定义**：
```
lim (x→a) f(x) = L 意味着：
对任意 ε > 0，存在 δ > 0 使得当 0 < |x - a| < δ 时，|f(x) - L| < ε
```

**直观理解**：
- ε是输出的误差，δ是输入的误差
- 只要输入足够接近a，输出就足够接近L
- 但不关心f(a)本身是什么

**重要极限**：
- lim (x→0) sin(x)/x = 1
- lim (x→0) (1 + x)^(1/x) = e
- lim (x→∞) (1 + 1/x)^x = e

**单侧极限**：
- 左极限：x → a⁻ (从左边趋近)
- 右极限：x → a⁺ (从右边趋近)
- 极限存在 ⟺ 左右极限都存在且相等

---

### 3. 连续性 (Continuity)
**费曼解释**：函数图像能"一笔画"不断开。没有跳跃，没有断点。

**定义**：f在a点连续 ⟺ lim (x→a) f(x) = f(a)

**三种等价定义**：
1. ε-δ定义
2. 极限定义（如上）
3. 序列定义：对任意序列 x_n → a，有 f(x_n) → f(a)

**连续函数的例子**：
- 多项式函数（处处连续）
- sin(x), cos(x), e^x（处处连续）
- 1/x（在x=0不连续）

**间断类型**：
- **可去间断**：极限存在但不等于函数值
- **跳跃间断**：左右极限存在但不相等
- **无穷间断**：极限为无穷
- **振荡间断**：极限不存在

**重要定理**：
- **介值定理**：连续函数在闭区间上取遍中间值
- **最值定理**：连续函数在闭区间上必有最大值和最小值
- **一致连续**：δ只依赖于ε，不依赖于点的位置

---

### 4. 一致连续 (Uniform Continuity)
**费曼解释**：在区间任何地方，相同的"输入误差"都能保证相同的"输出误差"。不像普通连续，δ可能随点变化。

**定义**：
```
f在集合S上一致连续 ⟺
∀ ε > 0, ∃ δ > 0, ∀ x, y ∈ S, |x - y| < δ → |f(x) - f(y)| < ε
```

**对比**：
- 连续：δ可能依赖于ε和点a
- 一致连续：δ只依赖于ε

**重要定理**：
- **Cantor定理**：闭区间上的连续函数必一致连续

**例子**：
- f(x) = x² 在[0,1]上一致连续，在[0,∞)上不一致连续
- f(x) = sin(x) 在ℝ上一致连续

## 📈 二、导数与微分

### 1. 导数的定义 (Derivative)
**费曼解释**：函数在某点的"瞬间变化率"。像速度是位移对时间的导数。

**定义**：
```
f'(a) = lim (h→0) [f(a + h) - f(a)] / h
```

**几何意义**：切线的斜率

**物理意义**：
- 位移的导数 = 速度
- 速度的导数 = 加速度
- 加速度的导数 = 急动度

**基本求导法则**：
```
(c)' = 0                        (常数)
(x^n)' = nx^(n-1)               (幂函数)
(e^x)' = e^x                    (指数函数)
(ln x)' = 1/x                   (对数函数)
(sin x)' = cos x                (正弦)
(cos x)' = -sin x               (余弦)
```

**运算法则**：
```
(f + g)' = f' + g'              (和法则)
(f · g)' = f'g + fg'            (积法则)
(f / g)' = (f'g - fg') / g²     (商法则)
(f ∘ g)' = (f' ∘ g) · g'        (链式法则)
```

**Lean实现**：
```lean
-- 导数定义
def HasDerivAt (f : ℝ → ℝ) (f' : ℝ) (a : ℝ) : Prop :=
  Tendsto (fun h => (f (a + h) - f a) / h) (𝓝 0) (𝓝 f')

-- 基本定理
theorem deriv_add (f g : ℝ → ℝ) (a : ℝ) 
    (hf : HasDerivAt f f' a) (hg : HasDerivAt g g' a) :
    HasDerivAt (f + g) (f' + g') a := by
  -- 证明省略
  sorry
```

### 2. 高阶导数 (Higher-Order Derivatives)
**费曼解释**：导数的导数。二阶导数告诉你"变化的变化"。

**记号**：
- f''(x), f^(2)(x), d²f/dx²
- f^(n)(x) 表示n阶导数

**例子**：
```
f(x) = x³
f'(x) = 3x²
f''(x) = 6x
f'''(x) = 6
f^(4)(x) = 0
```

**应用**：
- 凸凹性：f'' > 0 凹（凸向上），f'' < 0 凸（凸向下）
- 拐点：f'' 变号的点
- 泰勒展开

---

### 3. 微分中值定理 (Mean Value Theorem)
**费曼解释**：如果你从A开车到B，平均速度80，那至少有一个时刻你的瞬时速度是80。

**Rolle定理**：f在[a,b]连续，(a,b)可导，f(a)=f(b) ⟹ ∃c∈(a,b)使得 f'(c)=0

**Lagrange中值定理**：
```
f在[a,b]连续，(a,b)可导 ⟹
∃c∈(a,b)使得 f'(c) = [f(b) - f(a)] / (b - a)
```

**Cauchy中值定理**：
```
f, g在[a,b]连续，(a,b)可导 ⟹
∃c∈(a,b)使得 [f(b)-f(a)]·g'(c) = [g(b)-g(a)]·f'(c)
```

**应用**：
- 证明函数单调性
- 证明不等式
- 洛必达法则的基础

---

### 4. 泰勒公式 (Taylor's Theorem)
**费曼解释**：用多项式近似任意函数，项数越多越精确。像用切线（一阶）近似，用抛物线（二阶）更精确。

**公式**：
```
f(x) = f(a) + f'(a)(x-a) + f''(a)(x-a)²/2! + ... + f^(n)(a)(x-a)^n/n! + R_n
```

**余项**：
- **Lagrange余项**：R_n = f^(n+1)(ξ)(x-a)^(n+1)/(n+1)!，ξ介于a和x之间
- **Peano余项**：R_n = o((x-a)^n) 当 x → a

**重要展开**：
```
e^x = 1 + x + x²/2! + x³/3! + ...
sin x = x - x³/3! + x⁵/5! - ...
cos x = 1 - x²/2! + x⁴/4! - ...
ln(1+x) = x - x²/2 + x³/3 - ...  (|x| < 1)
(1+x)^α = 1 + αx + α(α-1)x²/2! + ...  (|x| < 1)
```

**应用**：
- 函数近似计算
- 极限计算
- 物理模型简化

---

### 5. 洛必达法则 (L'Hôpital's Rule)
**费曼解释**：遇到0/0或∞/∞型的不定式，可以分子分母分别求导再求极限。

**定理**：
```
如果 lim (x→a) f(x)/g(x) 是 0/0 或 ∞/∞ 型，
且 lim (x→a) f'(x)/g'(x) 存在或为∞，
则 lim (x→a) f(x)/g(x) = lim (x→a) f'(x)/g'(x)
```

**例子**：
```
lim (x→0) sin x / x
= lim (x→0) cos x / 1  （洛必达）
= 1

lim (x→∞) x / e^x
= lim (x→∞) 1 / e^x  （洛必达）
= 0
```

**注意事项**：
- 必须是不定式
- 可以重复使用
- 不是万能的（有时反而复杂）

## 🔢 三、积分学

### 1. 不定积分 (Indefinite Integral)
**费曼解释**：导数的"逆运算"。已知速度求位移，已知斜率求函数。

**定义**：
```
∫ f(x) dx = F(x) + C
其中 F'(x) = f(x)，C是任意常数
```

**基本积分表**：
```
∫ x^n dx = x^(n+1)/(n+1) + C    (n ≠ -1)
∫ 1/x dx = ln|x| + C
∫ e^x dx = e^x + C
∫ sin x dx = -cos x + C
∫ cos x dx = sin x + C
```

**积分技巧**：
- 换元积分法
- 分部积分法
- 部分分式分解
- 三角替换

---

### 2. 定积分 (Definite Integral)
**费曼解释**：计算曲线下的"面积"。把区间分成很多小段，累加矩形面积，取极限。

**黎曼积分定义**：
```
∫[a,b] f(x) dx = lim (n→∞) Σ f(x_i*) · Δx
```

**黎曼和**：
- 左端点、右端点、中点
- Darboux上和、下和
- 积分存在 ⟺ 上积分 = 下积分

**牛顿-莱布尼茨公式**：
```
∫[a,b] f(x) dx = F(b) - F(a)
其中 F'(x) = f(x)
```

**性质**：
```
∫[a,a] f(x) dx = 0
∫[b,a] f(x) dx = -∫[a,b] f(x) dx
∫[a,c] f(x) dx = ∫[a,b] f(x) dx + ∫[b,c] f(x) dx
∫[a,b] [f(x) + g(x)] dx = ∫[a,b] f(x) dx + ∫[a,b] g(x) dx
```

**几何意义**：曲线与x轴围成的有向面积

---

### 3. 微积分基本定理 (Fundamental Theorem of Calculus)
**费曼解释**：连接微分和积分的桥梁。积分是微分的"逆运算"。

**第一基本定理**：
```
设 f 在 [a,b] 上连续，定义 F(x) = ∫[a,x] f(t) dt
则 F 在 [a,b] 上可导，且 F'(x) = f(x)
```

**第二基本定理**：
```
设 F'(x) = f(x)，f 在 [a,b] 上连续
则 ∫[a,b] f(x) dx = F(b) - F(a)
```

**意义**：
- 微分和积分互为逆运算
- 把定积分计算转化为不定积分
- 是整个微积分的核心

**Lean实现**：
```lean
-- 微积分基本定理
theorem fundamental_theorem_of_calculus 
    (f : ℝ → ℝ) (a b : ℝ) (hf : Continuous f) :
    (∫ x in a..b, f x) = (F b - F a) := by
  sorry

-- 其中 F 是 f 的原函数
```

---

### 4. 积分技巧

#### 换元积分法 (Substitution)
```
∫ f(g(x)) · g'(x) dx = ∫ f(u) du  （令 u = g(x)）
```

**例子**：
```
∫ 2x · e^(x²) dx
令 u = x², du = 2x dx
= ∫ e^u du
= e^u + C
= e^(x²) + C
```

#### 分部积分法 (Integration by Parts)
```
∫ u dv = uv - ∫ v du
```

**例子**：
```
∫ x · e^x dx
令 u = x, dv = e^x dx
= x · e^x - ∫ e^x dx
= x · e^x - e^x + C
= e^x(x - 1) + C
```

#### 三角替换
```
√(a² - x²) → x = a sin θ
√(a² + x²) → x = a tan θ
√(x² - a²) → x = a sec θ
```

---

### 5. 广义积分 (Improper Integral)
**费曼解释**：积分区间无限或函数有瑕点（趋于无穷）的积分。

**类型**：
1. **无穷区间**：∫[a,∞) f(x) dx = lim (b→∞) ∫[a,b] f(x) dx
2. **无界函数**：∫[a,b] f(x) dx = lim (ε→0⁺) ∫[a+ε,b] f(x) dx

**收敛判别法**：
- **比较判别法**：0 ≤ f ≤ g，∫ g 收敛 ⟹ ∫ f 收敛
- **极限判别法**：lim (x→∞) x^p·f(x) = L，p>1时收敛，p≤1时发散

**重要积分**：
```
∫[0,∞) e^(-x) dx = 1  （收敛）
∫[1,∞) 1/x dx = ∞     （发散）
∫[1,∞) 1/x² dx = 1    （收敛）
```

**Gamma函数**：
```
Γ(n) = ∫[0,∞) x^(n-1) e^(-x) dx
Γ(n+1) = n·Γ(n)
Γ(n+1) = n! （n是正整数）
```

## 📊 四、级数理论

### 1. 数项级数 (Series)
**费曼解释**：无限多个数相加。如果"加起来"趋于一个有限值，就说是收敛的。

**定义**：
```
Σ a_n 收敛 ⟺ lim (n→∞) S_n 存在
其中 S_n = a_1 + a_2 + ... + a_n 是部分和
```

**基本性质**：
- 收敛级数的和 = lim (n→∞) S_n
- 收敛 ⟹ lim (n→∞) a_n = 0（必要条件）
- 逆命题不成立：调和级数 Σ 1/n 发散

---

### 2. 收敛判别法

#### 正项级数

**比较判别法**：
```
如果 0 ≤ a_n ≤ b_n 且 Σ b_n 收敛，则 Σ a_n 收敛
```

**比值判别法（D'Alembert）**：
```
如果 lim (n→∞) |a_(n+1)/a_n| = L
- L < 1：绝对收敛
- L > 1：发散
- L = 1：不确定
```

**根值判别法（Cauchy）**：
```
如果 lim (n→∞) ⁿ√|a_n| = L
- L < 1：绝对收敛
- L > 1：发散
- L = 1：不确定
```

**积分判别法**：
```
如果 f(x) ≥ 0 递减，则 Σ f(n) 和 ∫[1,∞) f(x) dx 同敛散
```

#### 交错级数

**Leibniz判别法**：
```
Σ (-1)^n a_n 收敛，如果：
1. a_n ≥ 0 且递减
2. lim a_n = 0
```

---

### 3. 绝对收敛与条件收敛
**费曼解释**：
- 绝对收敛：|a_n|的级数收敛（"很强"的收敛）
- 条件收敛：级数收敛但|a_n|发散（"勉强"收敛）

**定义**：
```
Σ a_n 绝对收敛 ⟺ Σ |a_n| 收敛
Σ a_n 条件收敛 ⟺ Σ a_n 收敛但 Σ |a_n| 发散
```

**重要性质**：
- 绝对收敛 ⟹ 收敛
- 绝对收敛级数可以任意重排，和不变
- 条件收敛级数重排可能改变和（Riemann重排定理）

**例子**：
```
Σ (-1)^n / n  条件收敛
Σ (-1)^n / n² 绝对收敛
```

---

### 4. 幂级数 (Power Series)
**费曼解释**：用无穷多项多项式表示函数，项数越多越精确。

**形式**：Σ a_n (x - c)^n = a_₀ + a₁(x-c) + a₂(x-c)² + ...

**收敛半径**：
```
R = 1 / limsup (n→∞) ⁿ√|a_n|
在 |x - c| < R 内绝对收敛
在 |x - c| > R 发散
在 |x - c| = R 不确定
```

**重要幂级数**：
```
1/(1-x) = Σ x^n              (|x| < 1)
e^x = Σ x^n/n!               (所有x)
sin x = Σ (-1)^n x^(2n+1)/(2n+1)!  (所有x)
cos x = Σ (-1)^n x^(2n)/(2n)!      (所有x)
```

**应用**：
- 函数表示
- 近似计算
- 微分方程求解

---

### 5. Fourier级数 (Fourier Series)
**费曼解释**：用三角函数的"波"来合成任何周期函数。像声音分解成不同频率。

**形式**：
```
f(x) = a₀/2 + Σ [a_n cos(nx) + b_n sin(nx)]
```

**系数**：
```
a₀ = (1/π) ∫[-π,π] f(x) dx
a_n = (1/π) ∫[-π,π] f(x) cos(nx) dx
b_n = (1/π) ∫[-π,π] f(x) sin(nx) dx
```

**收敛定理**：
- Dirichlet条件：分段连续且分段单调 ⟹ Fourier级数收敛到 [f(x⁺) + f(x⁻)]/2

**应用**：
- 信号处理
- 图像压缩
- 偏微分方程

---

## 🌐 五、多元微积分

### 1. 多元函数的极限与连续
**费曼解释**：函数有多个输入变量。比如温度依赖于位置(x, y, z)和时间t。

**极限**：
```
lim (x,y)→(a,b) f(x,y) = L ⟺
对任意路径趋近(a,b)，f(x,y)都趋近L
```

**连续性**：
```
f在(a,b)连续 ⟺ lim (x,y)→(a,b) f(x,y) = f(a,b)
```

**注意**：
- 多元函数的极限必须沿所有路径都趋近相同值
- 路径无关性是关键

---

### 2. 偏导数 (Partial Derivative)
**费曼解释**：固定其他变量，只对一个变量求导。像只看东西方向的变化，不看南北方向。

**定义**：
```
∂f/∂x = lim (h→0) [f(x+h, y) - f(x,y)] / h
∂f/∂y = lim (h→0) [f(x, y+h) - f(x,y)] / h
```

**高阶偏导**：
```
∂²f/∂x², ∂²f/∂y², ∂²f/∂x∂y, ∂²f/∂y∂x
```

**混合偏导定理**：
```
如果 ∂²f/∂x∂y 和 ∂²f/∂y∂x 连续，则它们相等
```

**梯度**：
```
∇f = (∂f/∂x, ∂f/∂y, ∂f/∂z)
```

---

### 3. 方向导数与梯度 (Directional Derivative)
**费曼解释**：沿着某个方向的变化率。梯度指向增长最快的方向。

**方向导数**：
```
D_u f = ∇f · u  （u是单位向量）
```

**梯度性质**：
- 梯度方向：增长最快
- 梯度模：最大增长率
- 梯度垂直于等值面

**应用**：
- 优化算法（梯度下降）
- 机器学习（反向传播）
- 物理学（势能梯度）

---

### 4. 多重积分 (Multiple Integrals)
**费曼解释**：在二维、三维空间上"积分"。比如计算曲面下的体积。

**二重积分**：
```
∬_D f(x,y) dA = ∫∫ f(x,y) dx dy
```

**计算方法**：
- 累次积分（Fubini定理）
- 极坐标变换
- 柱坐标、球坐标

**变量替换**：
```
∬ f(x,y) dx dy = ∬ f(u,v) |J| du dv
其中 J = ∂(x,y)/∂(u,v) 是雅可比行列式
```

**三重积分**：
```
∭_V f(x,y,z) dV
```

**应用**：
- 体积计算
- 质心、转动惯量
- 概率计算

---

### 5. 曲线与曲面积分

#### 曲线积分
**第一类**（对弧长）：
```
∫_C f ds = ∫ f(r(t)) |r'(t)| dt
```

**第二类**（对坐标）：
```
∫_C P dx + Q dy = ∫ [P x'(t) + Q y'(t)] dt
```

**Green公式**：
```
∫_C P dx + Q dy = ∬_D (∂Q/∂x - ∂P/∂y) dx dy
```

#### 曲面积分
**第一类**（对面积）：
```
∬_Σ f dS
```

**第二类**（对坐标）：
```
∬_Σ P dy dz + Q dz dx + R dx dy
```

**Gauss公式**：
```
∬_Σ F · n dS = ∭_V ∇·F dV
```

**Stokes公式**：
```
∮_C F · dr = ∬_Σ (∇×F) · n dS
```

## 📐 六、实分析基础

### 1. 实数完备性 (Completeness of ℝ)
**费曼解释**：实数轴上没有"洞"。有理数有"洞"（√2不是有理数），但实数填满了。

**等价表述**：
1. **确界原理**：有上界的集合必有最小上界
2. **单调收敛定理**：有界单调序列必收敛
3. **闭区间套定理**：嵌套闭区间长度趋于0，有唯一交点
4. **Bolzano-Weierstrass定理**：有界序列必有收敛子列
5. **Cauchy完备性**：Cauchy序列必收敛

**应用**：证明中值定理、极值定理等

---

### 2. 度量空间 (Metric Space)
**费曼解释**：一个集合加上"距离"的概念。满足距离的公理。

**定义**：集合X上的度量d满足：
1. d(x, y) ≥ 0，等号当且仅当 x = y
2. d(x, y) = d(y, x) （对称性）
3. d(x, z) ≤ d(x, y) + d(y, z) （三角不等式）

**例子**：
- ℝ^n 欧氏距离：d(x, y) = √Σ(x_i - y_i)²
- 离散度量：d(x, y) = 0 若x=y，否则1
- 函数空间：d(f, g) = sup |f(x) - g(x)|

**概念**：
- 开集、闭集
- 收敛、连续
- 完备性、紧性

---

### 3. 点集拓扑基础
**费曼解释**：研究"接近"、"邻域"、"连续"的抽象理论。

**开集**：包含每个点的某个邻域

**闭集**：补集是开集

**紧集**：任何开覆盖都有有限子覆盖

**重要定理**：
- Heine-Borel：ℝ^n 中的紧集 ⟺ 有界闭集
- 连续函数在紧集上达到最值

---

## 🌀 七、复分析基础

### 1. 复数与复平面
**费曼解释**：实数的扩展，可以开负数的平方根。i² = -1。

**表示**：
- 代数形式：z = a + bi
- 极坐标：z = r(cos θ + i sin θ) = re^(iθ)
- 欧拉公式：e^(iθ) = cos θ + i sin θ

**运算**：
- 加法：(a+bi) + (c+di) = (a+c) + (b+d)i
- 乘法：(a+bi)(c+di) = (ac-bd) + (ad+bc)i
- 共轭：z̄ = a - bi
- 模：|z| = √(a² + b²)

---

### 2. 复函数与解析性
**费曼解释**：复平面上的函数。解析函数是"特别好"的复函数——可导。

**复导数**：
```
f'(z) = lim (h→0) [f(z+h) - f(z)] / h
```

**解析函数**：在邻域内可导的函数

**Cauchy-Riemann方程**：
```
f(z) = u(x,y) + iv(x,y) 解析 ⟺
∂u/∂x = ∂v/∂y, ∂u/∂y = -∂v/∂x
```

**解析函数的性质**：
- 无穷次可导
- 可展开为幂级数
- 保存角度（保角映射）

---

### 3. Cauchy积分定理与公式
**费曼解释**：解析函数的积分很"神奇"——只依赖于起点和终点，不依赖于路径。

**Cauchy积分定理**：
```
f 解析 ⟹ ∮_C f(z) dz = 0
```

**Cauchy积分公式**：
```
f(z) = (1/2πi) ∮_C [f(ζ)/(ζ-z)] dζ
```

**应用**：
- 计算积分
- 证明解析函数性质
- 留数定理的基础

---

### 4. 留数定理 (Residue Theorem)
**费曼解释**：计算复积分的强大工具。只需求出"留数"就能算积分。

**留数**：f在奇点z₀的洛朗展开中 (z-z₀)^(-1) 项的系数

**留数定理**：
```
∮_C f(z) dz = 2πi Σ Res[f, z_k]
其中 z_k 是 C 内的奇点
```

**计算留数**：
```
单极点：Res[f, z₀] = lim (z→z₀) (z-z₀)f(z)
n阶极点：Res[f, z₀] = (1/(n-1)!) lim (z→z₀) d^(n-1)/dz^(n-1) [(z-z₀)^n f(z)]
```

**应用**：计算实积分

**关键概念**：

- Clarke梯度

**应用**：

- 控制理论

### 2. 分数阶微积分
**费曼解释**：微分和积分的"分数次"推广。可以有1/2阶导数、√2阶导数。

**定义**：
- Riemann-Liouville分数阶积分
- Caputo分数阶导数

**应用**：
- 异常扩散
- 粘弹性材料
- 信号处理

---

### 3. 变分分析与优化

**应用**：
- 最优控制
- 机器学习（损失函数优化）
- 物理学（最小作用量原理）
