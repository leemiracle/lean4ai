# 生物信息学 Bioinformatics

> 用计算机科学和统计学的方法，理解生命的密码

## 什么是生物信息学？（费曼学习法）

想象你有一本用ATCG四个字母写的、长达30亿字符的书（人类基因组）。这书没有标点、没有章节、没有目录。生物信息学就是开发工具来"读懂"这本书的学科。

**核心公式：**
```
生物信息学 = 生物学问题 + 计算机工具 + 统计方法
```

**类比理解：**
- 生物学提供"问题"（这个基因是干嘛的？）
- 计算机科学提供"工具"（如何在1秒内搜索30亿字符）
- 统计学提供"判断标准"（这个结果是真实的还是噪音？）

## 第一部分：序列分析

### 1.1 基因组是什么？

**费曼解释：**
把DNA想象成一卷录像带。这卷带子上有三种内容：
1. **编码区（基因）**：真正"播放"的部分，会被翻译成蛋白质
2. **调控区**：像遥控器，决定什么时候播放、播放多少
3. **非编码区**：曾经被认为是"垃圾"，现在发现也有很多功能

**为什么需要计算机？**
- 人类基因组：3,000,000,000 碱基对
- 如果每秒读1个，需要95年
- 计算机可以在几小时内完成

### 1.2 序列比对（Sequence Alignment）

**问题：** 如何判断两个基因是否相关？

**费曼解释：**
想象你在比较两篇手抄的古老文献。抄写员可能：
- 漏字（缺失 deletion）
- 多写字（插入 insertion）
- 写错字（替换 substitution）

序列比对就是找出两段文字的"最佳对应方式"。

**经典算法：**

```
Needleman-Wunsch（全局比对）
├── 比较两个完整序列
└── 时间复杂度：O(mn)

Smith-Waterman（局部比对）
├── 找最相似的片段
└── 用于在数据库中搜索相似序列
```

**BLAST工具：**
```
查询序列：ATCGATCGATCG
         ↓↓↓↓
数据库：  [几十亿条序列]
         ↓↓↓↓
结果：    找到100个相似序列，最相似的E-value = 1e-50
```

E-value越小，结果越可信。E-value = 1e-50 意思是"随机出现这种匹配的概率是10^-50"

### 1.3 基因预测

**问题：** 给你一段DNA序列，哪里是基因？

**费曼解释：**
想象你在没有标点的古文中找句子。你需要识别：
- "起始信号"（相当于句首大写）
- "终止信号"（相当于句号）
- "语法规则"（三联体密码子）

**两种策略：**
```
1. 从头预测（Ab initio）
   ├── 只看DNA序列本身
   ├── 识别启动子、起始密码子等特征
   └── 工具：GeneMark、Augustus

2. 基于证据（Evidence-based）
   ├── 看RNA-seq数据（实际转录的证据）
   ├── 看蛋白质同源性
   └── 更准确，但需要更多数据
```

### 1.4 蛋白质组分析

**中心法则：**
```
DNA → RNA → 蛋白质 → 功能
 ↓     ↓       ↓
基因组 转录组  蛋白质组
```

**蛋白质序列分析：**
- **结构域预测**：蛋白质的"功能模块"
- **二级结构**：α螺旋、β折叠
- **翻译后修饰**：磷酸化、糖基化等

## 第二部分：结构生物信息学

### 2.1 蛋白质折叠问题

**费曼解释：**
蛋白质像一根绳子，会自动折叠成特定的3D形状。这个形状决定了它的功能。
- 序列 = 绳子的编织方式
- 结构 = 绳子最终盘成的形状
- 功能 = 这个形状能做什么

**为什么折叠很重要？**
```
错误折叠 → 淀粉样蛋白 → 阿尔茨海默症
错误折叠 → 错误的受体结合 → 癌症
正确折叠 → 酶活性 → 正常代谢
```

### 2.2 AlphaFold2 革命

**历史：**
```
1972年：Anfinsen提出"序列决定结构"
1994年：CASP竞赛开始（蛋白质结构预测奥林匹克）
2018年：AlphaFold初版
2020年：AlphaFold2达到实验精度！
2021年：AlphaFold蛋白质结构数据库发布
2022年：2亿+蛋白质结构预测完成
```

**AlphaFold2的突破：**
```
传统方法：
实验测定（X射线晶体学、冷冻电镜）
├── 时间：数月到数年
├── 成本：数万到数十万美元
└── 技术难度：高

AlphaFold2：
├── 时间：分钟到小时
├── 成本：计算资源费用
└── 准确度：中位数GDT-TS > 92（接近实验精度）
```

**技术原理（简化版）：**
```
输入：氨基酸序列
  ↓
步骤1：搜索进化相关序列（MSA）
  ↓
步骤2：构建成对表示
  ↓
步骤3：Evoformer迭代（注意力机制）
  ↓
步骤4：结构模块生成3D坐标
  ↓
输出：预测的3D结构 + 置信度
```

### 2.3 结构分析工具

```
PyMOL        可视化和分析蛋白质结构
ChimeraX     UCSF开发的结构可视化
DALI         结构比对
PDB          蛋白质结构数据库
```

## 第三部分：比较基因组学

### 3.1 核心思想

**费曼解释：**
想象你在比较不同版本的同一本书。通过比较，你能发现：
- 哪些章节从未改变（重要、不能改）
- 哪些章节差异很大（不那么重要）
- 哪些是新增的章节（新功能）

**比较的内容：**
```
同一物种不同个体：
├── SNP（单核苷酸多态性）
├── CNV（拷贝数变异）
└── 结构变异

不同物种：
├── 同源基因
├── 基因家族扩张/收缩
└── 新基因的起源
```

### 3.2 系统发育分析

**构建进化树：**
```
方法：
├── 最大似然法（Maximum Likelihood）
├── 贝叶斯推断（Bayesian Inference）
└── 邻接法（Neighbor Joining）

工具：
├── RAxML、IQ-TREE（建树）
├── BEAST（时间校准）
└── FigTree、iTOL（可视化）
```

### 3.3 正选择分析

**问题：** 哪些基因在快速进化？

**dN/dS比值：**
```
dN = 非同义替换率（改变氨基酸）
dS = 同义替换率（不改变氨基酸）

dN/dS < 1：纯化选择（不能变）
dN/dS = 1：中性进化
dN/dS > 1：正选择（快速进化！）
```

**工具：** PAML、HyPhy、CodeML

## 第四部分：系统生物学

### 4.1 从还原论到整体论

**费曼解释：**
传统生物学像拆钟表，研究每个齿轮。系统生物学是研究齿轮之间如何协作让钟表走动。

### 4.2 网络分析

**生物网络类型：**
```
蛋白质-蛋白质相互作用网络（PPI）
├── 节点：蛋白质
└── 边：相互作用

基因调控网络（GRN）
├── 节点：基因/转录因子
└── 边：调控关系

代谢网络
├── 节点：代谢物
└── 边：酶促反应
```

**关键概念：**
```
Hub节点    度数很高的节点，通常是必需基因
模块       功能相关的节点群
Motif      小的、重复出现的网络模式
Robustness 网络对扰动的抵抗能力
```

### 4.3 多组学整合

```
基因组    →  能做什么（潜力）
转录组    →  正在表达什么（当前状态）
蛋白质组  →  实际执行者
代谢组    →  最终产物
        ↓
     整合分析
        ↓
系统层面的理解
```

## 第六部分：单细胞分析

### 6.1 为什么需要单细胞？

**费曼解释：**
传统测序像把一袋水果打成奶昔再分析。你知道有什么营养，但不知道哪个苹果是坏的。
单细胞测序是逐个检查每个水果。

**技术原理：**
```
组织 → 单细胞悬浊液 → 微流控分离 → 单细胞条形码 → 
建库测序 → 生物信息分析
```

### 6.2 单细胞RNA测序（scRNA-seq）

**分析流程：**
```
1. 质控（QC）
   ├── 过滤低质量细胞
   ├── 过滤双细胞
   └── 标准化

2. 降维
   ├── PCA（线性降维）
   ├── t-SNE（非线性）
   └── UMAP（最常用）

3. 聚类
   └── 识别细胞类型

4. 差异表达
   └── 找标记基因

5. 注释
   └── 给聚类命名
```

**主流工具：**
```python
# Seurat (R)
library(Seurat)
pbmc <- Read10X(data.dir = "filtered_gene_bc_matrices/")
pbmc <- CreateSeuratObject(pbmc)
pbmc <- NormalizeData(pbmc)
pbmc <- FindVariableFeatures(pbmc)
pbmc <- RunPCA(pbmc)
pbmc <- FindNeighbors(pbmc)
pbmc <- FindClusters(pbmc)
pbmc <- RunUMAP(pbmc, dims = 1:10)

# Scanpy (Python)
import scanpy as sc
adata = sc.read_10x_mtx('filtered_gene_bc_matrices/')
sc.pp.filter_cells(adata, min_genes=200)
sc.pp.normalize_total(adata)
sc.pp.highly_variable_genes(adata)
sc.tl.pca(adata)
sc.pp.neighbors(adata)
sc.tl.umap(adata)
sc.tl.leiden(adata)
```

## 第七部分：空间转录组

### 7.1 为什么需要空间信息？

**费曼解释：**
知道一个房子里有厨房、卧室、卫生间是不够的。你需要知道它们的位置关系。空间转录组就是给基因表达添加"GPS坐标"。

### 7.2 主要技术

```
基于芯片：
├── 10x Visium      最常用，分辨率55μm
├── Visium HD       高分辨率，2μm
└── NanoString GeoMx

基于测序：
├── Slide-seq       分辨率10μm
├── Stereo-seq      亚细胞分辨率
└── MERFISH         单分子，1000+基因

基于成像：
├── seqFISH+        10000+基因
├── osmFISH
└── ISS
```

### 7.3 分析方法

```python
# Squidpy: Python空间转录组工具
import squidpy as sq

# 读取数据
adata = sq.datasets.visium_fluo_adata()

# 空间邻居图
sq.gr.spatial_neighbors(adata, n_neighs=6)

# 空间基因表达模式
sq.gr.spatial_autocorr(adata, mode="moran")

# 空间域识别
sq.gr.ligrec(adata, n_perms=100)  # 配体-受体分析

# 可视化
sq.pl.spatial_scatter(adata, color="cluster")
```
