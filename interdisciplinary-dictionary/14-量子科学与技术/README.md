## 量子力学基础

### 叠加态：同时存在的可能性

你在选择午餐——汉堡还是沙拉？经典世界只能选一个。量子世界可以处于"吃汉堡"和"吃沙拉"的**叠加状态**——同时做两件事！

```
经典比特：  0 ──── 或 ──── 1

量子比特：  |ψ⟩ = α|0⟩ + β|1⟩
            （同时是0和1）
```

**为什么叠加重要？**

- n个经典比特 = 1个状态
- n个量子比特 = 2^n 个状态同时存在

50个量子比特可同时表示约1千万亿个状态！

### 纠缠态：鬼魅般的超距作用

你有一双魔法手套，一只寄给北京，一只留在纽约。当你打开发现是左手时，**瞬间**就知道那边是右手。

量子纠缠更神奇：测量之前，**两只手套都还没决定**是左是右。是你打开盒子的行为，让两只手套同时"坍缩"成配对状态。

```
EPR对：|Φ⁺⟩ = (|00⟩ + |11⟩)/√2

测量第一个：得到0 → 第二个必然是0
            得到1 → 第二个必然是1
```

### 测量与坍缩

一个旋转的硬币，你问"是正面吗？"——这一问，硬币停止旋转，给你答案。问题强迫硬币做出选择。

```
测量前：|ψ⟩ = α|0⟩ + β|1⟩

测量后：以概率|α|²得到|0⟩，态坍缩为|0⟩
        以概率|β|²得到|1⟩，态坍缩为|1⟩
```

**海森堡不确定性：** 位置测量越准，动量测量越不准。Δx·Δp ≥ ℏ/2

## 量子计算

### 量子比特（Qubit）

**布洛赫球表示：**

```
|ψ⟩ = cos(θ/2)|0⟩ + e^(iφ)sin(θ/2)|1⟩

θ: 与z轴夹角（0到π）
φ: xy平面旋转角（0到2π）
```

**物理实现：**

| 技术路线 | 载体 | 优点 | 挑战 |
|----------|------|------|------|
| 超导 | 约瑟夫森结 | 可扩展、快 | 需极低温 |
| 离子阱 | 囚禁离子 | 高保真 | 扩展难 |
| 光量子 | 光子偏振 | 室温 | 存储难 |
| 半导体 | 量子点 | 兼容工艺 | 退相干 |

### 量子门

**单量子比特门：**

```
X门（量子NOT）：  X|0⟩ = |1⟩, X|1⟩ = |0⟩

H门（创造叠加）：H|0⟩ = (|0⟩+|1⟩)/√2
```

**双量子比特门：**

```
CNOT门：控制位为1 → 目标位翻转
|00⟩→|00⟩, |01⟩→|01⟩, |10⟩→|11⟩, |11⟩→|10⟩
```

**通用门集：** H门 + T门 + CNOT门可构建任意量子计算。

### 量子算法

| 问题 | 经典复杂度 | 量子复杂度 | 加速 |
|------|-----------|-----------|------|
| 搜索N项 | O(N) | O(√N) | 平方 |
| 大数分解 | 指数 | 多项式 | 指数 |
| 量子模拟 | O(e^n) | O(n^k) | 指数 |

**Shor算法（大数分解）：**

分解2048位数字：经典超算约10亿年，量子计算机理论上约几小时。威胁RSA加密！

**Grover算法（搜索）：**

在1000本书中找一本：经典方法平均500次，量子方法约32次（√1000）。

## 量子通信

### 量子密钥分发（QKD）

**核心思想：** 窃听者的测量会扰动量子态，因此可被检测。

**BB84协议：**

```
1. Alice随机选比特值(0/1)和基(+基/×基)，发送量子态
2. Bob随机选测量基，记录结果
3. 公布使用的基（不公布结果）
4. 保留基相同的比特（约50%）
5. 公布部分检测窃听
6. 误码率低于阈值，剩余为密钥
```

**安全性来源：**

- 不可克隆定理：无法复制未知量子态
- 测量干扰：窃听引入错误

### 量子隐形传态

**传信息，不是传物质！**

```
步骤：
1. Alice对待传态和她的一半纠缠对做Bell测量
2. 通过经典信道告诉Bob结果（2比特）
3. Bob根据结果操作，得到原始态

注意：原态被销毁，不违反信息守恒
```

2017年中国"墨子号"卫星实现千公里级隐形传态。

## 量子材料

### 主要类型

**1. 超导体**

```
特性：零电阻、完全抗磁性
高温超导体：Tc > 77K（液氮温度）
例：YBCO，Tc ≈ 93K
```

**2. 拓扑材料**

```
拓扑绝缘体：内部绝缘，表面导电
- 表面态受拓扑保护
- 不受杂质散射影响

拓扑超导体：可能存在马约拉纳费米子
```

**3. 二维材料**

```
石墨烯：单层碳原子，高导电、高强度
TMDs：可调带隙，谷电子学
```

**4. 量子自旋液体**

自旋永不冻结，高度纠缠，可能存在任意子。

## 量子模拟

### 为什么需要？

模拟50量子比特系统需要约1千万亿个复数，经典计算机存不下。

**费曼愿景（1982）：** "用量子机器模拟量子系统。"

### 方法

**数字模拟：** 用量子门构建目标系统演化

```
U(t) = e^(-iHt/ℏ)
分解：Trotter公式 e^(A+B) ≈ e^A·e^B
```

**模拟模拟：** 用可控系统类比目标系统

- 冷原子模拟凝聚态
- 离子阱模拟自旋模型

### 应用

- 高温超导机制
- 量子相变
- 分子能量计算
- 药物设计
