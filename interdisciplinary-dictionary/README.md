# 交叉学科知识词典

## 项目说明
本项目基于中国教育部《学位授予和人才培养学科目录（2020年）》新增的**交叉学科门类（14）**，并结合当代重要的交叉学科领域进行全面整理，采用费曼学习法进行通俗解释。

## 什么是交叉学科？

**费曼解释**：交叉学科就像两个不同领域的人结婚生的孩子——继承了父母的特点，又有自己独特的性格。比如生物化学，既懂生物又懂化学，用化学的方法研究生命现象。现代重大突破大多发生在学科交叉的地方。

## 🔬 交叉学科的特点

### 1. 问题驱动
**费曼解释**：传统学科像是按工具分类（锤子、锯子、螺丝刀），交叉学科是按要解决的问题分类（要建房子，用什么工具都行）。比如治疗癌症，需要生物学、化学、物理学、计算机科学的共同努力。

### 2. 方法融合
**费曼解释**：每个学科有自己的"独门绝技"。交叉学科就是把这些绝技组合起来。比如用物理学的方法研究大脑（神经物理），用计算机模拟社会（计算社会科学）。

### 3. 创新涌现
**费曼解释**：创新往往发生在边界。就像两个国家的边境城市，文化交融，产生独特的风貌。DNA双螺旋结构的发现，就是物理学家（威尔金斯、克里克）和生物学家（沃森）合作的结果。

## 🎯 典型交叉学科案例

### 案例1：生物信息学
- **交叉来源**：生物学 + 计算机科学 + 统计学
- **要解决的问题**：人类基因组有30亿碱基对，如何存储、分析、理解？
- **核心方法**：算法、数据库、机器学习、统计模型
- **费曼解释**：生物学产生了海量数据，计算机提供存储和分析工具，统计学帮助从噪声中找规律。三个缺一不可。

### 案例2：神经科学
- **交叉来源**：生物学 + 医学 + 心理学 + 计算机科学 + 物理学
- **要解决的问题**：大脑如何产生意识、记忆、情感？
- **核心方法**：电生理记录、脑成像、计算机模拟、基因编辑
- **费曼解释**：大脑是最复杂的系统，需要从分子（生物学）、细胞（医学）、行为（心理学）、网络（计算机）、物理原理（物理学）多个层面同时研究。

### 案例3：人工智能
- **交叉来源**：计算机科学 + 数学 + 统计学 + 神经科学 + 认知科学
- **要解决的问题**：如何让机器具有智能？
- **核心方法**：机器学习、深度学习、强化学习、符号推理
- **费曼解释**：人工智能想要模仿人类智能，就要了解人类如何思考（认知科学、神经科学），用数学描述（数学、统计），用计算机实现（计算机科学）。
