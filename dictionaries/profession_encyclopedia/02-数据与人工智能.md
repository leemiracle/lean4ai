# 数据与人工智能

## 模块概述
本模块收录75个数据与人工智能相关职业，涵盖数据分析、机器学习、深度学习、AI应用等方向。

## 2.1 数据分析类（15个职业）

### 2.1.1 数据分析师

**【费曼学习法解释】**
数据分析师就像"数据侦探"。公司每天产生大量数据，数据分析师从数据中找线索、找规律、找问题，告诉老板发生了什么、为什么发生、未来会怎样。

**【核心技能】**
- 必备：Excel高级应用、SQL（增删改查、复杂查询）
- 编程：Python或R（Pandas、NumPy、Matplotlib）
- 可视化：Tableau、Power BI、FineBI
- 统计：描述统计、假设检验、回归分析
- 工具：Google Analytics、Mixpanel、神策
- 业务：理解业务指标（DAU、GMV、留存、转化等）

**【工作内容】**
- 从数据库提取数据（SQL）
- 清洗和整理数据
- 数据分析和挖掘
- 数据可视化（报表、仪表盘）
- 撰写数据分析报告
- 给业务部门提供数据支持

**【学历要求】** 本科及以上（统计学、数学、计算机、经济等相关专业）

**【薪资范围】**
- 初级（1-3年）：15-25万/年
- 中级（3-5年）：25-45万/年
- 高级（5-10年）：45-80万/年
- 专家级（10年+）：80-150万/年

**【发展路径】**
数据分析师 → 高级数据分析师 → 数据科学家/数据团队负责人

**【细分方向】**
- 用户数据分析师
- 商业数据分析师
- 运营数据分析师
- 产品数据分析师

**【职业前景】** ⭐⭐⭐⭐⭐（几乎所有行业都需要，门槛适中，发展空间大）

### 2.1.2 商业分析师

**【费曼学习法解释】**
商业分析师用数据解决商业问题。不只是看数据，更要理解业务，把数据变成商业洞察和建议，帮公司做决策。

**【核心技能】**
- 数据分析：SQL、Excel、Python/R
- 业务理解：商业模式、指标体系
- 报告撰写：商业报告、PPT
- 沟通能力：向业务方解释数据
- 建模能力：商业模型、预测模型
- 行业知识：了解所在行业

**【工作内容】**
- 分析业务数据和指标
- 发现业务问题和机会
- 建立商业模型和预测
- 撰写商业分析报告
- 向业务部门提供建议
- 参与商业决策

**【学历要求】** 本科及以上（商科、经济、统计、计算机等相关专业）

**【薪资范围】**
- 初级（1-3年）：20-35万/年
- 中级（3-5年）：35-55万/年
- 高级（5-10年）：55-100万/年
- 专家级（10年+）：100-200万/年

**【发展路径】**
商业分析师 → 高级商业分析师 → 商业分析总监 → CMO/COO

**【细分方向】**
- 零售商业分析
- 电商商业分析
- 金融商业分析
- SaaS商业分析

**【职业前景】** ⭐⭐⭐⭐（战略价值高，适合有商业头脑的人）

**【细分方向】**

### 2.2.3 数据仓库工程师

**【费曼学习法解释】**
数据仓库工程师设计"数据仓库"。数据仓库就像公司的"数据中心"，把各种数据集中存储、整理好，方便分析师和业务人员使用。

**【核心技能】**
- 数据仓库建模：星型模型、雪花模型、维度建模
- SQL：复杂查询、存储过程
- ETL：ETL工具（Informatica、DataStage、Talend）
