## 🎯 职业分类标准

### 按技术难度
- **入门级**（0-1年经验）
- **初级**（1-3年经验）
- **中级**（3-5年经验）
- **高级**（5-10年经验）
- **专家级**（10年+经验）

### 按学历要求
- **不限/高中** - 基础服务、操作类
- **大专** - 技术工种、销售类
- **本科** - 专业技术、管理类
- **硕士** - 研发、高端技术
- **博士** - 科研、学术

### 按工作性质
- **全职** - 标准工作制
- **兼职** - 灵活时间
- **合同工** - 项目制
- **自由职业** - 自主接单
- **远程办公** - 可异地工作

## ⚠️ 重要提示

### 关于薪资
- 薪资范围为参考值，实际情况因公司、地区、个人能力而异
- 技术岗位薪资波动较大
- 高薪岗位通常伴随高压力和高要求

### 关于职业发展
- 技术路线：初级→中级→高级→专家→首席
- 管理路线：专员→主管→经理→总监→VP
- 跨界路线：单领域→跨领域→复合型人才→创业

### 关于技能要求
- 硬技能：专业技能、工具使用、证书资质
- 软技能：沟通、协作、学习、抗压
- 通用技能：数据分析、项目管理、数字化能力

### 关于学历
- 学历是门槛，不是天花板
- 能力和经验更重要
- 持续学习是关键
