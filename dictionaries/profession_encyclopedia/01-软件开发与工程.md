## 1.1 前端开发（12个职业）

### 1.1.1 前端开发工程师（Web Frontend）

**【费曼学习法解释】**
前端开发工程师就像网站的"装修师傅"。后端工程师把房子的框架（服务器和数据库）建好了，前端工程师负责装修房子（网页界面），让用户打开网站时看到的界面漂亮、好用、操作顺畅。

**【核心技能】**
- 必备：HTML5、CSS3、JavaScript（ES6+）
- 框架：至少精通1-2个主流框架（React/Vue/Angular）
- 工具：Webpack、Vite、Git
- 进阶：TypeScript、Node.js基础
- 测试：Jest、Cypress

**【工作内容】**
- 根据UI设计图开发网页界面
- 实现页面交互功能和动画效果
- 优化页面加载速度和性能
- 解决浏览器兼容性问题
- 与后端对接API接口
- 编写单元测试和集成测试

**【学历要求】** 本科及以上（计算机、软件工程相关专业）

**【薪资范围】**
- 初级（1-3年）：15-25万/年
- 中级（3-5年）：25-45万/年
- 高级（5-10年）：45-80万/年
- 专家级（10年+）：80-150万/年

**【发展路径】**
前端开发 → 高级前端 → 前端架构师 / 技术专家
或：前端开发 → 全栈工程师 → 架构师

**【细分方向】**
- Web应用前端
- 移动端H5
- 管理后台前端
- 可视化大屏前端

**【职业前景】** ⭐⭐⭐⭐⭐（需求持续增长，AI辅助开发降低门槛）

### 1.1.2 移动端开发工程师（iOS/Android）

**【费曼学习法解释】**
移动端开发工程师专门开发手机APP。就像建筑师建房子，有人建高楼大厦（电脑网站），有人建小别墅（手机APP），移动端开发工程师就是建手机APP的专家。

**【核心技能】**
- iOS方向：
  - 编程语言：Swift、Objective-C
  - 开发工具：Xcode
  - 框架：UIKit、SwiftUI
  - 开源库：AFNetworking、Alamofire
- Android方向：
  - 编程语言：Kotlin（主流）、Java
  - 开发工具：Android Studio
  - 框架：Jetpack Compose（新）、AndroidX
  - 开源库：Retrofit、Glide
- 共通：
  - RESTful API对接
  - 本地存储
  - 性能优化
  - 热更新技术

**【工作内容】**
- 开发iOS或Android原生APP
- 实现UI界面和交互逻辑
- 集成第三方SDK（支付、地图、推送等）
- 优化APP性能和用户体验
- 适配不同设备和系统版本
- 处理上线发布和版本更新

**【学历要求】** 本科及以上（计算机相关专业）

**【薪资范围】**
- 初级（1-3年）：18-30万/年
- 中级（3-5年）：30-50万/年
- 高级（5-10年）：50-90万/年
- 专家级（10年+）：90-180万/年

**【发展路径】**
移动端开发 → 高级移动端开发 → 移动端架构师
或：移动端开发 → 跨平台开发 → 技术专家

**【细分方向】**
- iOS开发
- Android开发
- 移动安全开发
- 移动性能优化

**【职业前景】** ⭐⭐⭐⭐（跨平台开发增长，原生开发需求稳定）

- 集成原生功能和SDK

### 1.1.5 前端架构师

**【费曼学习法解释】**
前端架构师是前端团队的"总设计师"。普通前端工程师负责做页面，前端架构师决定整个团队用什么技术、怎么组织代码、怎么保证质量，就像建筑总设计师设计整个城市的布局。

**【核心技能】**
- 深度掌握：至少精通2-3个主流框架
- 架构设计：微前端、组件化、模块化
- 性能优化：前端性能优化最佳实践
- 工程化：构建工具、CI/CD、自动化
- 监控：前端监控、日志、错误追踪
- 团队管理：技术选型、规范制定、团队培养

**【工作内容】**
- 制定前端技术选型和架构方案
- 设计前端工程化体系
- 制定代码规范和最佳实践
- 解决技术难题和性能瓶颈
- 指导团队技术成长
- 评审技术方案和代码

**【学历要求】** 本科及以上（计算机相关专业），5年以上经验

**【薪资范围】**
- 高级（5-8年）：50-100万/年
- 专家（8-10年）：100-150万/年
- 首席（10年+）：150-300万/年

**【发展路径】**
前端架构师 → 首席架构师 → CTO
或：前端架构师 → 技术总监 → CTO

**【细分方向】**
- Web前端架构
- 移动端架构
- 全栈架构

**【职业前景】** ⭐⭐⭐⭐⭐（高级岗位，稀缺性高，薪资天花板高）

### 1.1.6 Web全栈工程师

**【费曼学习法解释】**
全栈工程师就是"全能工程师"。既能做前端（用户看到的界面），也能做后端（服务器处理逻辑），还能管数据库（数据存储）。就像一个建筑师既能设计外观，也能设计结构，还能做室内装修。

**【核心技能】**
- 前端：至少精通1-2个框架（React/Vue/Angular）
- 后端：至少精通1-2门语言（Node.js/Java/Python/Go）
- 数据库：关系型（MySQL/PostgreSQL）和非关系型（MongoDB/Redis）
- 运维基础：Linux、Docker、部署
- API设计：RESTful、GraphQL
- 测试：单元测试、集成测试

**【工作内容】**
- 独立完成从前端到后端的开发
- 设计数据库和API接口
- 实现业务逻辑和功能
- 部署和维护应用
- 性能优化和bug修复
- 快速迭代产品功能

**【学历要求】** 本科及以上（计算机相关专业）

**【薪资范围】**
- 初级（1-3年）：20-35万/年
- 中级（3-5年）：35-55万/年
- 高级（5-10年）：55-90万/年
- 专家级（10年+）：90-180万/年

**【发展路径】**
全栈工程师 → 高级全栈工程师 → 架构师
或：全栈工程师 → 技术经理 → CTO

**【细分方向】**
- Web全栈
- 移动全栈
- 云全栈

**【职业前景】** ⭐⭐⭐⭐⭐（综合能力强，小公司需求大，薪资高）
