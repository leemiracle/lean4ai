# 知识词典系统

本项目汇集了各类知识词典，按照**专业学科字典**、**行业知识字典**和**产品知识字典**三大分类组织。

## 📚 一、专业学科字典 (Academic Discipline Dictionaries)

按学科代码和学科名称组织，涵盖所有学术领域的专业术语、概念和理论体系。

### 01. 哲学 (Philosophy)
- **目录**: `01-哲学/`
- **内容**: 马克思主义哲学、中国哲学、西方哲学、逻辑学、伦理学、美学、宗教学、科学技术哲学等

### 02. 经济学 (Economics)
- **目录**: `02-经济学/`
- **内容**: 理论经济学、应用经济学、新兴经济领域等

### 03. 法学 (Law)
- **目录**: `03-法学/`
- **内容**: 法理学、宪法学、行政法学、民商法学、刑法学、经济法学、诉讼法学、国际法学等

### 05. 文学 (Literature)
- **目录**: `05-文学/`
- **内容**: 文学理论、中国古代文学、中国现当代文学、外国语言文学等

### 08. 艺术学 (Art)
- **目录**: `08-艺术学/`
- **内容**: 艺术学理论、音乐学、舞蹈学、戏剧戏曲学、影视学、美术学、设计学、新媒体艺术等

### 11. 管理学 (Management)
- **目录**: `11-管理学/`
- **内容**: 管理基础与理论、管理科学与工程、会计与财务管理等

### 12. 理学 (Science)
- **目录**: `12-理学/`
- **内容**: 数学、物理学、化学、天文学、地理学、生物学等

### 13. 工学 (Engineering)
- **目录**: `13-工学/`
- **内容**: 机械工程、电气工程、计算机科学与技术、土木工程等

### 14. 跨学科 (Interdisciplinary)
- **目录**: `14-跨学科/`
- **内容**: 集成电路、人工智能、纳米科学、量子科学等新兴交叉学科

### 15. 军事学 (Military)
- **目录**: `15-军事学/`
- **内容**: 军事思想、战略学、战役学、战术学、军队指挥学等

### 16. 人类学 (Anthropology)
- **目录**: `16-人类学/`
- **内容**: 社会文化人类学等

### 4. 农业
