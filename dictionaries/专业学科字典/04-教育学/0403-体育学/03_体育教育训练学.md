#### 2. 体育教学目标 (PE Teaching Objectives/体育教学目标)

**白话解释**:  
上体育课要达到什么目的。

**目标体系**:
```
体育教学目标
├─ 运动参与目标（愿意参与）
├─ 运动技能目标（学会技能）
├─ 身体健康目标（增强体质）
├─ 心理健康目标（心理素质）
└─ 社会适应目标（合作精神）
```

**目标层次**:
| 层次 | 说明 | 举例 |
|------|------|------|
| 课程目标 | 整个课程的目标 | 掌握2-3项运动技能 |
| 学段目标 | 某学段的目标 | 初中阶段目标 |
| 单元目标 | 一个单元的目标 | 篮球单元目标 |
| 课时目标 | 一节课的目标 | 学会胸前传球 |

**目标要求**:
- 具体明确
- 可测量
- 可达成
- 有挑战性

#### 3. 体育教学原则 (PE Teaching Principles/体育教学原则)

**白话解释**:  
上体育课要遵循的基本规则。

**主要原则**:

1. **身心全面发展原则**
   - 身体与心理协调发展
   - 全面锻炼、全面发展

2. **直观性原则**
   - 动作示范要直观
   - 利用图示、视频

3. **循序渐进原则**
   - 由易到难、由简到繁
   - 逐步提高要求

4. **因材施教原则**
   - 根据学生特点教学
   - 区别对待不同水平

5. **巩固提高原则**
   - 反复练习
   - 巩固已学技能

6. **安全性原则**
   - 确保运动安全
   - 预防运动损伤

**内容选择原则**:
1. 科学性 - 符合学生发展特点

上体育课用的方法和手段。

**常用方法**:

| 类型 | 方法 | 适用情境 |

#### 6. 讲解与示范 (Explanation and Demonstration/讲解与示范)

**白话解释**:  
老师怎么讲、怎么做动作给学生看。

**讲解要求**:
```
1. 准确 - 动作要领准确
2. 简洁 - 语言精炼
3. 生动 - 通俗易懂
4. 重点 - 突出关键
```

**示范要求**:
1. **示范位置** - 让学生都能看清
2. **示范速度** - 先慢后快
3. **示范次数** - 适当重复
4. **示范方向** - 多角度展示

**示范类型**:
| 类型 | 说明 | 举例 |
|------|------|------|
| 完整示范 | 展示完整动作 | 初学时展示 |
| 分解示范 | 展示部分动作 | 学习复杂动作 |
| 正误对比 | 对比正确与错误 | 纠正动作时 |
| 慢速示范 | 慢动作展示 | 学习新动作时 |

---

#### 7. 练习法 (Practice Methods/练习法)

**白话解释**:  
让学生练习的方法。

**主要类型**:

1. **完整练习法**
   - 完整练习整个动作
   - 适用于简单动作

2. **分解练习法**
   - 把动作分成部分练习
   - 适用于复杂动作
   - 形式：单纯分解、递进分解

3. **重复练习法**
   - 重复练习同一动作
   - 用于巩固技能

4. **变换练习法**
   - 改变练习条件
   - 提高适应能力

5. **循环练习法**
   - 按顺序练习多个内容
   - 提高练习密度

---

#### 8. 体育课组织 (PE Class Organization/体育课组织)

**白话解释**:  
怎么组织学生上课。

**体育课结构**:
```
体育课
├─ 准备部分（10-15分钟）
│   ├─ 课堂常规
│   └─ 准备活动
├─ 基本部分（25-30分钟）
│   ├─ 新授内容
│   └─ 复习内容
└─ 结束部分（5-10分钟）
    ├─ 整理活动
    └─ 小结讲评
```

**队形组织**:
| 队形 | 适用情境 |
|------|----------|
| 四列横队 | 讲解、示范 |
| 圆形队 | 游戏、讨论 |
| 散点式 | 自由练习 |
| 分组式 | 分组活动 |

**练习密度**:
- 定义：学生实际练习时间占课时的比例
- 标准：30-50%为宜
- 提高：减少等待时间、增加器材

#### 11. 竞技能力 (Competitive Ability/竞技能力)

**白话解释**:  
运动员比赛的能力，是多种因素的综合。

**构成要素**:
```
竞技能力
├─ 体能
│   ├─ 力量
│   ├─ 速度
│   ├─ 耐力
│   ├─ 柔韧
│   └─ 灵敏
├─ 技能
│   ├─ 技术合理性
│   └─ 技术稳定性
├─ 战术能力
│   ├─ 战术意识
│   └─ 战术运用

│   ├─ 意志品质
│   └─ 心理稳定性

    └─ 临场应变
```

**项群训练理论**:
- 按竞技能力主导因素分类
- 体能主导类、技能主导类、技心能主导类

---

#### 12. 运动训练原则 (Training Principles/运动训练原则)

**白话解释**:  
训练运动员要遵循的基本规则。

**主要原则**:

1. **竞技需要原则**
   - 根据比赛需要训练
   - 训练与比赛相结合

2. **动机激励原则**
   - 激发训练动机
   - 内在动机与外在动机

3. **有效控制原则**
   - 科学控制训练过程
   - 及时调整训练计划

4. **系统训练原则**
   - 长期系统训练
   - 循序渐进

5. **周期安排原则**
   - 周期性安排训练
   - 准备期、比赛期、恢复期

6. **区别对待原则**
   - 因材施教
   - 针对个体特点训练

---

#### 13. 运动训练负荷 (Training Load/运动训练负荷)

**白话解释**:  
训练的量和强度，训练"有多累"。

**负荷构成**:
```
训练负荷
├─ 负荷量（练多少）
│   ├─ 练习次数
│   ├─ 练习时间
│   ├─ 练习距离
│   └─ 练习重量
└─ 负荷强度（练多累）
    ├─ 速度
    ├─ 高度
    ├─ 远度
    └─ 重量百分比
```

**负荷控制**:
1. **循序渐进** - 逐步增加负荷
2. **波浪式** - 高低交替
3. **跳跃式** - 适合高水平运动员

**负荷监控**:
| 指标 | 说明 |
|------|------|
| 心率 | 运动强度指标 |
| RPE | 主观疲劳度 |
| 血乳酸 | 无氧代谢指标 |
| 肌酸激酶 | 肌肉损伤指标 |

### 五、训练方法

#### 14. 基本训练方法 (Basic Training Methods/基本训练方法)

**白话解释**:  
常用的训练方法有哪些。

**主要方法**:

1. **重复训练法**
   - 重复练习同一内容
   - 间歇充分恢复
   - 发展速度、力量

2. **间歇训练法**
   - 练习间不完全恢复
   - 发展耐力
   - 严格控制间歇时间

3. **持续训练法**
   - 长时间持续运动
   - 发展有氧耐力
   - 强度较低

4. **变换训练法**
   - 改变练习条件
   - 提高适应能力
   - 如变换场地、变换强度

5. **循环训练法**
   - 多站点轮换练习
   - 提高全面体能
   - 适合集体训练

6. **比赛训练法**
   - 通过比赛形式训练
   - 提高实战能力

---

#### 15. 体能训练 (Physical Fitness Training/体能训练)

**白话解释**:  
训练身体的基本素质。

**五大素质**:

| 素质 | 说明 | 训练方法 |
|------|------|----------|
| 力量 | 肌肉收缩能力 | 抗阻训练 |
| 速度 | 快速移动能力 | 短距离冲刺 |
| 耐力 | 持续运动能力 | 长跑、间歇跑 |
| 柔韧 | 关节活动范围 | 拉伸训练 |
| 灵敏 | 快速变向能力 | 敏捷性训练 |

**力量训练**:
```
力量训练
├─ 最大力量（1RM）
├─ 快速力量（爆发力）
└─ 力量耐力（多次重复）
```

**耐力训练**:
- 有氧耐力：长时间中等强度
- 无氧耐力：间歇训练、乳酸训练

---

#### 16. 技术训练 (Technical Training/技术训练)

**白话解释**:  
训练运动技术，把动作做好。

**技术训练阶段**:
```
1. 泛化阶段
   - 建立基本动作模式
   - 动作不协调

2. 分化阶段
   - 动作逐渐协调
   - 错误减少

3. 巩固阶段
   - 动作定型
   - 较为稳定

4. 自动化阶段
   - 动作自动化
   - 可以专注于战术
```

**技术训练方法**:
1. **分解训练** - 分解动作练习
2. **完整训练** - 完整动作练习
3. **模拟训练** - 模拟比赛条件
4. **视频分析** - 观看分析动作

**技术诊断**:
- 观察分析
- 视频分析
- 仪器测量

---

#### 17. 战术训练 (Tactical Training/战术训练)

**白话解释**:  
训练比赛中的策略和配合。

**战术分类**:
```
战术
├─ 个人战术
│   └─ 个人技术选择
├─ 小组战术
│   └─ 2-3人配合
└─ 全队战术
    └─ 整体战术配合
```

**战术训练内容**:
1. **战术知识** - 了解战术原理
2. **战术意识** - 比赛中的判断
3. **战术配合** - 与队友配合
4. **战术应变** - 根据比赛调整

**训练方法**:
- 战术讲解
- 模拟演练
- 教学比赛
- 视频分析
- 实战总结

### 七、运动竞赛组织

#### 21. 运动竞赛 (Sports Competition/运动竞赛)

**白话解释**:  
体育比赛活动的总称。

**竞赛分类**:
```
按规模分
├─ 综合性运动会（奥运会、全运会）
├─ 单项锦标赛
└─ 友谊赛、邀请赛

按性质分
├─ 锦标赛（争夺名次）
├─ 联赛（积分排名）
├─ 杯赛（淘汰制）
└─ 表演赛（展示交流）
```

**竞赛制度**:
| 制度 | 说明 | 优缺点 |
|------|------|--------|
| 循环制 | 每队互相对抗 | 公平但场次多 |
| 淘汰制 | 输者淘汰 | 刺激但偶然性大 |
| 混合制 | 结合循环与淘汰 | 综合优点 |

---

#### 22. 赛前准备与临场指挥 (Competition Preparation/赛前准备与临场指挥)

**白话解释**:  
比赛前怎么准备，比赛中教练怎么指挥。

**赛前准备**:
