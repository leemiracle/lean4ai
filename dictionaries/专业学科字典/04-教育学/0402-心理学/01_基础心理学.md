## 📖 第一章：心理学概述

---

### 一、心理学基础概念

#### 1. 心理学 (Psychology/心理学)

**白话解释**:  
研究人的心理活动和行为规律的科学，就是研究人怎么想、为什么这么做。

**生活例子**:  
- 为什么人会有情绪？→ 情绪心理学
- 为什么广告能说服人？→ 消费心理学
- 为什么人会对某些事物恐惧？→ 心理障碍研究

**学术定义**:  
心理学是研究心理活动和行为规律的科学，它既研究动物的心理，也研究人的心理，以人的心理为主要研究对象。

**研究内容**:
```
心理学
├─ 心理过程
│   ├─ 认知过程（感觉、知觉、记忆、思维）
│   ├─ 情绪过程（喜怒哀乐）
│   └─ 意志过程（决定、行动）
├─ 个性心理
│   ├─ 个性倾向性（需要、动机、兴趣）
│   └─ 个性心理特征（能力、气质、性格）
└─ 心理状态
    ├─ 意识状态
    └─ 无意识状态
```

**费曼记忆**:  
心理学 = 心理过程 + 个性心理 + 行为规律

---

#### 2. 心理学研究方法 (Research Methods/研究方法)

**白话解释**:  
心理学家用什么方法来研究心理现象。

**主要方法**:

| 方法 | 说明 | 举例 |
|------|------|------|
| 观察法 | 在自然情境中观察行为 | 观察儿童游戏行为 |
| 实验法 | 控制变量研究因果关系 | 研究睡眠对记忆的影响 |
| 调查法 | 问卷、访谈收集数据 | 调查大学生压力状况 |
| 测验法 | 标准化工具测量心理特征 | 智力测试、性格测试 |
| 个案法 | 深入研究个别案例 | 研究特殊脑损伤患者 |

**实验设计要素**:
1. **自变量** - 研究者操纵的变量
2. **因变量** - 观测的结果变量
3. **控制变量** - 保持恒定的变量

---

#### 3. 神经系统基础 (Nervous System/神经系统)

**白话解释**:  
心理活动的物质基础，脑子是怎么工作的。

**基本结构**:
```
神经系统
├─ 中枢神经系统
│   ├─ 脑（大脑、小脑、脑干）
│   └─ 脊髓
└─ 周围神经系统
    ├─ 躯体神经系统
    └─ 自主神经系统
```

**大脑功能分区**:
| 脑区 | 主要功能 | 损伤后果 |
|------|----------|----------|
| 额叶 | 思维、计划、运动 | 执行功能障碍 |
| 顶叶 | 感觉整合、空间 | 忽视症状 |
| 颞叶 | 听觉、语言、记忆 | 听觉障碍 |
| 枕叶 | 视觉处理 | 视觉障碍 |

**神经元**:
- 基本结构：细胞体、树突、轴突
- 信息传递：电信号→化学信号→电信号

## 📖 第二章：感觉与知觉

---

### 二、感觉

#### 4. 感觉 (Sensation/感觉)

**白话解释**:  
眼睛、耳朵、鼻子等感觉器官接收外界刺激产生的最基本的心理活动。

**生活例子**:  
- 眼睛看到红色 → 视觉
- 耳朵听到铃声 → 听觉
- 舌头尝到甜味 → 味觉

**学术定义**:  
感觉是人脑对直接作用于感觉器官的客观事物的个别属性的反映。

**感觉类型**:
```
外部感觉
├─ 视觉（眼睛）
├─ 听觉（耳朵）
├─ 嗅觉（鼻子）
├─ 味觉（舌头）
└─ 皮肤觉（皮肤）

内部感觉
├─ 运动觉（肌肉、关节）
├─ 平衡觉（前庭）
└─ 机体觉（内脏）
```

**感觉阈限**:
- **绝对阈限** - 刚能引起感觉的最小刺激量
- **差别阈限** - 刚能引起差别感觉的最小变化量

---

#### 5. 视觉 (Vision/视觉)

**白话解释**:  
眼睛看到东西，识别颜色、形状等。

**视觉过程**:
```
光线 → 角膜 → 瞳孔 → 晶状体 → 视网膜 → 视神经 → 大脑视觉区
```

**基本特性**:
1. **色调** - 颜色种类（红、绿、蓝等）
2. **明度** - 颜色明暗程度
3. **饱和度** - 颜色纯度

**颜色理论**:
- **三色说** - 红、绿、蓝三种视锥细胞
- **对立过程说** - 黑-白、红-绿、黄-蓝三对对立

**视觉现象**:
1. **颜色混合** - 红+绿=黄
2. **颜色对比** - 背景色影响感知
3. **后像** - 盯着看红旗后看白墙出现绿色

---

#### 6. 听觉 (Hearing/听觉)

**白话解释**:  
耳朵听到声音，分辨音调、音量等。

**声音特性**:
| 物理特性 | 心理体验 | 说明 |
|----------|----------|------|
| 频率 | 音调 | 频率高→音调高 |
| 振幅 | 响度 | 振幅大→声音响 |
| 波形 | 音色 | 不同乐器音色不同 |

**听觉过程**:
```
声波 → 外耳道 → 鼓膜 → 听小骨 → 耳蜗 → 听神经 → 听觉皮层
```

**听觉理论**:
1. **频率理论** - 基底膜振动频率与声波一致
2. **位置理论** - 不同频率刺激耳蜗不同位置

---

### 三、知觉

#### 7. 知觉 (Perception/知觉)

**白话解释**:  
大脑把感觉信息整合起来，认出是什么东西。

**生活例子**:  
- 看到红色圆形的物体 → 认出是苹果
- 听到"汪汪"的声音 → 认出是狗叫

**学术定义**:  
知觉是人脑对直接作用于感觉器官的客观事物的整体属性的反映。

**感觉 vs 知觉**:
```
感觉：个别属性（红色、圆形、甜味）
知觉：整体属性（这是一个苹果）
```

**知觉特性**:
1. **选择性** - 从背景中选出对象
2. **整体性** - 把部分整合成整体
3. **理解性** - 用已有经验解释
4. **恒常性** - 条件变化知觉稳定

---

#### 8. 知觉恒常性 (Perceptual Constancy/知觉恒常性)

**白话解释**:  
东西变远了变小，但我们知道它实际没变小。

**类型**:
| 恒常性 | 说明 | 例子 |
|--------|------|------|
| 大小恒常性 | 距离变化，感知大小不变 | 远处的车看起来小，但知道它实际很大 |
| 形状恒常性 | 角度变化，感知形状不变 | 从侧面看书，仍是长方形 |
| 颜色恒常性 | 光线变化，感知颜色不变 | 黄光下白纸仍是白色 |
| 明度恒常性 | 光照变化，感知明度不变 | 阴影中的煤炭仍是黑的 |

### 四、记忆系统

#### 9. 记忆 (Memory/记忆)

**白话解释**:  
记住东西的能力，包括记住、保持、回忆出来。

**生活例子**:  
- 记住朋友的电话号码
- 考试时想起学过的知识
- 童年的美好回忆

**学术定义**:  
记忆是人脑对过去经验的保持和再现，包括识记、保持、再认或回忆三个基本环节。

**记忆过程**:
```
编码（输入）→ 存储（保存）→ 提取（输出）
```

**记忆类型**:
| 分类标准 | 类型 | 特点 |
|----------|------|------|
| 保持时间 | 感觉记忆、短时记忆、长时记忆 | 时间不同 |
| 内容特点 | 陈述性记忆、程序性记忆 | 内容不同 |
| 意识参与 | 内隐记忆、外显记忆 | 意识参与不同 |

---

#### 10. 三级记忆系统 (Three-Stage Memory/三级记忆系统)

**白话解释**:  
记忆分三个阶段：一闪而过的、暂时记住的、长期记住的。

**系统结构**:
```
感觉记忆（0.25-2秒）
    ↓ 注意
短时记忆（5秒-1分钟）
    ↓ 复述
长时记忆（永久）
```

**各阶段特点**:
| 阶段 | 保持时间 | 容量 | 功能 |
|------|----------|------|------|
| 感觉记忆 | 0.25-2秒 | 大 | 瞬间登记 |
| 短时记忆 | 1分钟内 | 7±2组块 | 信息加工 |
| 长时记忆 | 永久 | 无限 | 永久存储 |

**工作记忆**:
- 短时记忆的扩展
- 包括：中央执行系统、语音回路、视觉空间画板、情景缓冲器
- 功能：暂时存储和加工信息

---

#### 11. 遗忘 (Forgetting/遗忘)

**白话解释**:  
记住的东西想不起来了。

**遗忘曲线** (艾宾浩斯):
```
时间      保持量
刚记完     100%
20分钟     58%
1小时      44%
1天        33%
6天        25%
31天       21%
```

**遗忘原因**:
1. **衰退说** - 记忆痕迹随时间消退
2. **干扰说** - 新旧信息相互干扰
3. **压抑说** - 痛苦记忆被压抑
4. **提取失败说** - 线索不足导致提取失败

**减少遗忘的方法**:
- 及时复习
- 分散复习
- 多种编码方式
- 深度加工

### 五、思维

#### 12. 思维 (Thinking/思维)

**白话解释**:  
动脑子想事情，解决问题、做决定。

**生活例子**:  
- 解数学题 → 问题解决思维
- 制定旅行计划 → 计划决策思维
- 创作一幅画 → 创造性思维

**学术定义**:  
思维是人脑借助语言实现的，对客观事物的本质属性和规律性联系的间接、概括的反映。

**思维特征**:
1. **间接性** - 借助媒介认识事物
2. **概括性** - 抽取共同本质特征
3. **问题性** - 围绕问题展开

**思维类型**:
```
思维
├─ 直观动作思维（动手思考）
├─ 具体形象思维（形象思考）
├─ 抽象逻辑思维（概念思考）
├─ 聚合思维（求同）
├─ 发散思维（求异）
└─ 创造性思维（创新）
```

---

#### 13. 概念 (Concept/概念)

**白话解释**:  
对事物本质的认识，比如"鸟"是有羽毛、会下蛋的动物。

**学术定义**:  
概念是人脑对客观事物本质属性的反映，是思维的基本形式。

**概念结构**:
```
概念
├─ 内涵（本质属性）
└─ 外延（适用范围）
```

**概念形成**:
1. **假设检验说** - 形成假设，检验修正
2. **样例学习说** - 通过典型样例学习

**人工概念实验** (布鲁纳):
- 通过卡片分类实验研究概念形成
- 发现策略影响概念学习效率

---

#### 14. 问题解决 (Problem Solving/问题解决)

**白话解释**:  
想办法解决遇到的难题。

**问题解决阶段**:
```
1. 发现问题
2. 分析问题
3. 提出假设
4. 检验假设
```

**问题解决策略**:
| 策略 | 说明 | 举例 |
|------|------|------|
| 算法 | 尝试所有可能 | 逐个试密码 |
| 启发法 | 经验法则 | 从简单开始 |
| 手段-目的分析 | 缩小差距 | 先解决子问题 |
| 逆向搜索 | 从目标倒推 | 数学证明 |

**影响因素**:
1. **问题表征方式**
2. **定势** - 固有思维模式
3. **功能固着** - 只看到常规用途
4. **动机水平**
5. **知识经验**

---

#### 15. 创造性思维 (Creative Thinking/创造性思维)

**白话解释**:  
想出新颖独特的想法。

**特征**:
1. **流畅性** - 想法多
2. **变通性** - 角度灵活
3. **独创性** - 与众不同

**创造性思维过程** (沃拉斯):
```
准备期 → 酝酿期 → 豁朗期（灵感）→ 验证期
```

**培养方法**:
1. 头脑风暴
2. 自由联想
3. 打破定势
4. 营造宽松环境

### 六、语言

#### 16. 语言与思维 (Language and Thinking/语言与思维)

**白话解释**:  
语言是思维的工具，没有语言很难进行复杂思考。

**关系**:
- 语言是思维的载体
- 思维影响语言表达
- 语言影响思维方式

**萨皮尔-沃尔夫假说**:
- 语言决定思维方式
- 不同语言的人思维不同

**语言结构**:
```
音位（声音）
    ↓
语素（意义单位）
    ↓
词（独立运用）
    ↓
句子（表达完整意思）
```

### 七、意识与注意

#### 17. 意识 (Consciousness/意识)

**白话解释**:  
清醒时知道自己在做什么、想什么的状态。

**意识状态**:
```
意识状态
├─ 清醒意识（正常状态）
├─ 睡眠（做梦）
├─ 催眠（特殊意识状态）
├─ 冥想（专注状态）
└─ 药物影响状态
```

**意识功能**:
1. 监视内部和外部环境
2. 控制心理活动和行为

**无意识**:
- 不能直接觉察的心理活动
- 包括：阈下知觉、内隐记忆、自动化行为

---

#### 18. 注意 (Attention/注意)

**白话解释**:  
心思集中在某件事上，其他事暂时不管。

**学术定义**:  
注意是心理活动对一定对象的指向和集中。

**注意特性**:
1. **指向性** - 选择某个对象
2. **集中性** - 聚焦在该对象

**注意类型**:
| 类型 | 特点 | 举例 |
|------|------|------|
| 无意注意 | 无预定目的 | 突然听到响声 |
| 有意注意 | 有预定目的、需意志 | 上课专心听讲 |
| 有意后注意 | 目的性、不需意志 | 熟练阅读 |

**注意品质**:
1. **广度** - 同时能注意多少
2. **稳定性** - 能保持多久
3. **分配** - 同时做几件事
4. **转移** - 从一个转向另一个

---

#### 19. 睡眠与梦 (Sleep and Dream/睡眠与梦)

**白话解释**:  
人为什么要睡觉，为什么会做梦。

**睡眠周期**:
```
清醒 → N1 → N2 → N3 → REM → N2 → ...
(约90分钟一个周期，每夜4-6个周期)
```

**睡眠阶段**:
| 阶段 | 特点 | 脑波 |
|------|------|------|
| N1 | 浅睡，易醒 | α波→θ波 |
| N2 | 睡眠锭出现 | θ波+睡眠锭 |
| N3 | 深睡，难唤醒 | δ波（慢波） |
| REM | 快速眼动，做梦 | 类似清醒 |

**梦的功能**:
1. 处理情绪
2. 整合记忆
3. 问题解决

## 💡 实务应用场景

### 场景1: 提高学习效率
- **问题**: 如何记住更多知识？
- **基础心理学回答**: 
  - 利用艾宾浩斯遗忘曲线及时复习
  - 深度加工信息，而非死记硬背
  - 分散复习优于集中复习
  - 多种编码方式（视觉+听觉）

### 场景2: 解决问题卡壳
- **问题**: 遇到难题想不通怎么办？
- **基础心理学回答**: 
  - 酝酿效应：暂时放下，休息一下
  - 打破定势：换个角度思考
  - 逆向思维：从目标倒推

### 场景3: 提高注意力
- **问题**: 上课总是走神怎么办？
- **基础心理学回答**: 
  - 培养有意注意
  - 排除干扰因素
  - 设定明确目标
  - 适当休息，避免疲劳
