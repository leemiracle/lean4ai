### 一、营养物质概述

#### 1. 营养物质 (Nutrients/营养物质)

**白话解释**:  
动物"吃饭"获得的营养成分，用来长肉、产奶、产蛋、维持生命。

**生活例子**:  
- 蛋白质长肌肉
- 淀粉提供能量
- 钙质强壮骨骼

**学术定义**:  
动物从饲料中获得，用于维持生命、生长、生产（肉、蛋、奶、毛等）的化学物质。

**营养物质分类**:
```
营养物质分类（六大类）
├── 水分
├── 蛋白质
├── 脂肪
├── 碳水化合物
├── 矿物质
│   ├── 常量元素：Ca、P、Na、Cl、K、Mg、S
│   └── 微量元素：Fe、Cu、Zn、Mn、Co、I、Se
└── 维生素
    ├── 脂溶性：A、D、E、K
    └── 水溶性：B族、C
```

**费曼记忆**:  
营养六要素 = 水 + 蛋 + 脂 + 碳 + 矿 + 维

---

#### 2. 能量 (Energy/能量)

**白话解释**:  
动物做任何事都需要"能量"，像汽车需要汽油。

**生活例子**:  
- 动物活动、呼吸消耗能量
- 长肉、产奶需要能量

**学术定义**:  
营养物质氧化释放的化学能，是动物生命活动的基础。

**能量体系（以猪为例）**:
```
能量分类
总能量(GE)
  ├─ 消化能(DE) = GE - 粪能
  │   ├─ 代谢能(ME) = DE - 尿能 - 气体能
  │   │   └─ 净能(NE) = ME - 热增耗
  │   │       ├─ 维持净能(NEm)
  │   │       └─ 生产净能(NEp)
```

**能量单位**:
- 焦耳(J)、千焦(kJ)、兆焦(MJ)
- 卡(cal)、千卡(kcal)、兆卡(Mcal)
- 1 cal = 4.184 J

**各类动物能量需要**:
| 动物 | 能量体系 | 单位 |
|------|---------|------|
| 猪 | 消化能(DE) | MJ/kg |
| 家禽 | 代谢能(ME) | MJ/kg |
| 牛 | 净能(NE) | MJ/kg |

### 二、蛋白质与氨基酸

#### 3. 蛋白质 (Protein/蛋白质)

**白话解释**:  
蛋白质是"长肉的原料"，构成肌肉、毛、皮、蹄等。

**生活例子**:  
- 鱼粉、豆粕蛋白含量高
- 动物生长快需要高蛋白

**学术定义**:  
由氨基酸构成的高分子化合物，是动物体组织的主要成分。

**蛋白质组成**:
```
蛋白质组成
├── 氨基酸（20种）
│   ├── 必需氨基酸（EAA）
│   │   └── 动物不能合成，必须从饲料获得
│   │       猪：Lys、Met、Trp、Thr、Ile、Leu、Val、Phe、Arg、His
│   │       鸡：+Gly、Cys、Pro（共13种）
│   └── 非必需氨基酸（NEAA）
│       └── 动物可以合成
└── 粗蛋白(CP)
    └── N × 6.25（平均含氮量16%）
```

**粗蛋白测定**:
```
粗蛋白(CP) = 氮含量(N) × 6.25

例：豆粕含氮7.0%
CP = 7.0 × 6.25 = 43.75%
```

---

#### 4. 必需氨基酸 (Essential Amino Acids/必需氨基酸)

**白话解释**:  
动物自己造不出来，必须"吃进去"的氨基酸。

**生活例子**:  
- 猪容易缺赖氨酸
- 鸡容易缺蛋氨酸

**学术定义**:  
动物体内不能合成或合成速度不能满足需要，必须从饲料中获得的氨基酸。

**限制性氨基酸**:
```
限制性氨基酸顺序
├── 猪
│   ├── 第一限制：赖氨酸(Lys)
│   ├── 第二限制：蛋氨酸(Met)
│   └── 第三限制：色氨酸(Trp)
└── 鸡
    ├── 第一限制：蛋氨酸(Met)
    ├── 第二限制：赖氨酸(Lys)
    └── 第三限制：色氨酸(Trp)
```

**理想氨基酸模式**:
```
理想蛋白：各种必需氨基酸的比例平衡

以赖氨酸为100：
赖氨酸 100
蛋+胱  60
苏氨酸 65
色氨酸 18
异亮氨酸 60
亮氨酸 100
```

**氨基酸平衡的重要性**:
```
木桶理论：
蛋白质合成 = 最低氨基酸的含量

缺乏一种必需氨基酸 → 其他氨基酸无法利用
```

### 三、碳水化合物

#### 5. 碳水化合物 (Carbohydrates/碳水化合物)

**白话解释**:  
植物的主要成分，是动物最主要的能量来源。

**生活例子**:  
- 玉米、小麦主要成分是淀粉
- 粗饲料含有大量纤维素

**学术定义**:  
由碳、氢、氧组成的多羟基化合物，包括单糖、双糖、多糖等。

**碳水化合物分类**:
```
碳水化合物分类
├── 单糖
│   └── 葡萄糖、果糖
├── 双糖
│   ├── 蔗糖
│   ├── 乳糖
│   └── 麦芽糖
└── 多糖
    ├── 淀粉（可消化）
    │   └── 玉米、小麦的主要成分
    └── 粗纤维（难消化）
        ├── 纤维素
        ├── 半纤维素
        └── 木质素
```

**单胃动物vs反刍动物**:
| 特性 | 单胃动物（猪、鸡） | 反刍动物（牛、羊） |
|------|------------------|------------------|
| 淀粉消化 | 好 | 好 |
| 粗纤维消化 | 差 | 好（瘤胃微生物发酵） |
| 粗饲料利用 | 低 | 高 |

---

#### 6. 粗纤维 (Crude Fiber/粗纤维)

**白话解释**:  
植物细胞壁的成分，猪鸡消化不了，牛羊可以。

**生活例子**:  
- 稻草、玉米秸秆粗纤维含量高
- 仔猪料要低纤维

**学术定义**:  
饲料中不溶于稀酸、稀碱的有机物质，包括纤维素、半纤维素和木质素。

**粗纤维的作用**:
```
单胃动物：
├── 适量：促进肠道蠕动
├── 过多：影响养分消化、降低采食量
└── 一般限制<5-8%

反刍动物：
├── 能量来源（VFA）
├── 维持瘤胃功能
├── 维持乳脂率
└── 必须保证NDF>25%
```

**中性洗涤纤维(NDF)与酸性洗涤纤维(ADF)**:
```
细胞壁成分
├── NDF（中性洗涤纤维）
│   └── 纤维素 + 半纤维素 + 木质素
└── ADF（酸性洗涤纤维）
    └── 纤维素 + 木质素

NDF > ADF
NDF代表总纤维
ADF代表难消化纤维
```

### 四、脂肪

#### 7. 脂肪与脂肪酸 (Fat and Fatty Acids/脂肪与脂肪酸)

**白话解释**:  
脂肪是"浓缩能量"，能量是碳水化合物的2.25倍。

**生活例子**:  
- 豆油、猪油是脂肪
- 添加脂肪可提高饲料能量

**学术定义**:  
由甘油和脂肪酸组成的酯类化合物，是高能量营养物质。

**脂肪的能量**:
```
脂肪能量
├── 碳水化合物：17.5 MJ/kg
├── 蛋白质：23.6 MJ/kg
└── 脂肪：39.3 MJ/kg

脂肪能量 = 碳水化合物 × 2.25
```

**脂肪酸分类**:
```
脂肪酸分类
├── 饱和脂肪酸
│   └── 无双键：硬脂酸、软脂酸
├── 不饱和脂肪酸
│   ├── 单不饱和：油酸(18:1)
│   └── 多不饱和：亚油酸(18:2)、亚麻酸(18:3)
└── 必需脂肪酸(EFA)
    ├── 亚油酸(18:2 n-6)
    ├── α-亚麻酸(18:3 n-3)
    └── 花生四烯酸(20:4 n-6)
```

**脂肪的作用**:
1. **高能量来源**
2. **必需脂肪酸来源**
3. **脂溶性维生素载体**
4. **改善饲料适口性**

### 五、矿物质

#### 8. 常量元素 (Macro Minerals/常量元素)

**白话解释**:  
动物需要量大的矿物质，主要是钙、磷、钠、氯等。

**生活例子**:  
- 骨粉、石粉补充钙磷
- 食盐补充钠和氯

**学术定义**:  
动物需要量大于100mg/kg体重的矿物质元素。

**主要常量元素**:
```
常量元素（7种）
├── 钙(Ca)
│   ├── 骨骼、牙齿的主要成分
│   ├── 血液凝固
│   ├── 神经肌肉传导
│   └── 缺乏：佝偻病、软骨症
├── 磷(P)
│   ├── 骨骼成分
│   ├── ATP、核酸成分
│   ├── 缺乏：异嗜癖
│   └── Ca:P = 1-2:1
├── 钠(Na)、氯(Cl)
│   ├── 食盐(NaCl)补充
│   ├── 维持渗透压
│   └── 缺乏：食欲下降
├── 钾(K)
│   └── 维持细胞内渗透压
├── 镁(Mg)
│   └── 骨骼、酶活化
└── 硫(S)
    └── 含硫氨基酸、B族维生素成分
```

**钙磷来源**:
| 来源 | Ca(%) | P(%) |
|------|-------|------|
| 石粉 | 38 | 0 |
| 骨粉 | 30 | 14 |
| 磷酸氢钙 | 22 | 18 |
| 贝壳粉 | 33 | 0 |

---

#### 9. 微量元素 (Trace Minerals/微量元素)

**白话解释**:  
需要量小但不可缺少的矿物质，像铁、锌、铜等。

**生活例子**:  
- 缺铁贫血
- 缺锌皮肤问题

**学术定义**:  
动物需要量小于100mg/kg体重的矿物质元素。

**主要微量元素**:
```
微量元素（15+种）
├── 铁(Fe)
│   ├── 血红蛋白成分
│   └── 缺乏：贫血
├── 铜(Cu)
│   ├── 血红蛋白合成
│   ├── 被毛色素
│   └── 缺乏：贫血、被毛褪色
├── 锌(Zn)
│   ├── 多种酶成分
│   ├── 皮肤健康
│   └── 缺乏：皮肤角化不全
├── 锰(Mn)
│   ├── 骨骼发育
│   └── 缺乏：滑腱症
├── 碘(I)
│   ├── 甲状腺素成分
│   └── 缺乏：甲状腺肿
├── 硒(Se)
│   ├── 抗氧化（谷胱甘肽过氧化物酶）
│   ├── 缺乏：白肌病
│   └── 中毒：掉毛、蹄壳脱落
└── 钴(Co)
    └── 维生素B12成分
```

**微量元素添加剂形式**:
| 元素 | 常用形式 |
|------|---------|
| 铁 | 硫酸亚铁、富马酸铁 |
| 铜 | 硫酸铜 |
| 锌 | 硫酸锌、氧化锌 |
| 锰 | 硫酸锰 |
| 硒 | 亚硒酸钠 |
| 碘 | 碘化钾 |

### 六、维生素

#### 10. 脂溶性维生素 (Fat-soluble Vitamins/脂溶性维生素)

**白话解释**:  
溶于脂肪的维生素，包括A、D、E、K。

**生活例子**:  
- 鱼肝油含维生素A、D
- 维生素E抗氧化

**学术定义**:  
溶于脂溶性溶剂的维生素，可在体内储存。

**脂溶性维生素功能**:
```
脂溶性维生素（4种）
├── 维生素A
│   ├── 视觉（视紫红质）
│   ├── 上皮组织健康
│   ├── 免疫功能
│   └── 缺乏：夜盲症、干眼病
├── 维生素D
│   ├── 钙磷吸收
│   ├── 骨骼发育
│   └── 缺乏：佝偻病
├── 维生素E
│   ├── 抗氧化
│   ├── 生殖功能
│   └── 缺乏：白肌病、繁殖障碍
└── 维生素K
    ├── 血液凝固
    └── 缺乏：出血不止
```

**维生素单位**:
- 维生素A：IU（国际单位）= 0.3μg视黄醇
- 维生素D：IU = 0.025μg胆钙化醇
- 维生素E：IU = 1mg DL-α-生育酚乙酸酯

---

#### 11. 水溶性维生素 (Water-soluble Vitamins/水溶性维生素)

**白话解释**:  
溶于水的维生素，包括B族维生素和维生素C。

**生活例子**:  
- 缺B₁引起脚气病
- 反刍动物瘤胃微生物可合成B族维生素

**学术定义**:  
溶于水的维生素，体内不能储存，需每日供给。

**水溶性维生素功能**:
```
水溶性维生素
├── 维生素B族
│   ├── B₁(硫胺素)：能量代谢
│   ├── B₂(核黄素)：氧化还原反应
│   ├── B₃(烟酸)：NAD、NADP成分
│   ├── B₅(泛酸)：CoA成分
│   ├── B₆(吡哆醇)：氨基酸代谢
│   ├── B₇(生物素)：羧化酶辅酶
│   ├── B₉(叶酸)：一碳单位转移
│   ├── B₁₂(钴胺素)：造血、神经
│   │   └── 缺乏：贫血、神经症状
│   └── 胆碱：卵磷脂成分
└── 维生素C(抗坏血酸)
    ├── 抗氧化
    ├── 胶原蛋白合成
    └── 大多数动物可合成
```

**B族维生素特点**:
1. **辅酶作用** - 参与代谢
2. **缺乏症** - 皮肤、神经、消化问题
3. **反刍动物** - 瘤胃微生物可合成
4. **单胃动物** - 必须从饲料获得

### 一、能量饲料

#### 12. 谷实类饲料 (Cereal Grains/谷实类)

**白话解释**:  
玉米、小麦等粮食，能量高、适口性好。

**生活例子**:  
- 玉米是"饲料之王"
- 小麦可替代部分玉米

**学术定义**:  
干物质中粗纤维<18%、粗蛋白<20%的饲料，主要提供能量。

**主要谷实类饲料**:
```
谷实类饲料
├── 玉米（饲料之王）
│   ├── 优点：能量高、适口性好
│   ├── 缺点：蛋白低、赖氨酸少
│   └── 用量：50-70%
├── 小麦
│   ├── 优点：蛋白较高
│   ├── 缺点：含NSP、粘性
│   └── 用量：<30%
├── 稻谷
│   ├── 优点：适口性好
│   ├── 缺点：粗纤维高
│   └── 用量：<30%
└── 高粱
    ├── 优点：抗逆性强
    ├── 缺点：含单宁、适口性差
    └── 用量：<20%
```

**玉米营养成分**（干物质基础）:
| 成分 | 含量(%) |
|------|--------|
| 干物质 | 86-88 |
| 粗蛋白 | 8-9 |
| 粗脂肪 | 3.5-4 |
| 粗纤维 | 2-3 |
| 消化能(猪) | 14-15 MJ/kg |

---

#### 13. 糠麸类饲料 (Bran/糠麸类)

**白话解释**:  
粮食加工的副产品，营养价值比原粮低。

**生活例子**:  
- 麦麸是面粉厂副产品
- 米糠是碾米厂副产品

**学术定义**:  
谷实类加工的副产品，包括麦麸、米糠等。

**主要糠麸类饲料**:
```
糠麸类饲料
├── 小麦麸
│   ├── 成分：蛋白15%、纤维9%
│   ├── 特点：B族维生素丰富、轻泻
│   └── 用量：10-25%
├── 米糠
│   ├── 成分：蛋白13%、脂肪15%
│   ├── 特点：脂肪高、易氧化酸败
│   └── 用量：<15%
└── 次粉
    ├── 成分：介于小麦和麦麸之间
    └── 用量：10-20%
```

### 二、蛋白质饲料

#### 14. 植物性蛋白饲料 (Plant Protein Feeds/植物蛋白)

**白话解释**:  
植物来源的高蛋白饲料，主要是饼粕类。

**生活例子**:  
- 豆粕是养猪主要蛋白来源
- 菜籽粕、棉籽粕也是常用蛋白饲料

**学术定义**:  
干物质中粗蛋白≥20%、粗纤维<18%的植物性饲料。

**主要植物蛋白饲料**:
```
植物蛋白饲料
├── 大豆粕（豆粕）
│   ├── 蛋白：43-48%
│   ├── 特点：蛋白品质好、赖氨酸高
│   ├── 缺点：含抗营养因子（需加工）
│   └── 用量：15-30%
├── 菜籽粕
│   ├── 蛋白：35-38%
│   ├── 缺点：含硫葡萄糖苷、芥子碱
│   └── 用量：<10%
├── 棉籽粕
│   ├── 蛋白：40-44%
│   ├── 缺点：含棉酚
│   └── 用量：<10%
└── 花生粕
    ├── 蛋白：45-50%
    ├── 缺点：易发霉（黄曲霉毒素）
    └── 用量：<10%
```

**豆粕营养特点**:
- 蛋白含量43-48%
- 赖氨酸含量高（2.5-2.8%）
- 氨基酸平衡好
- 适口性好
- 是最优质的植物蛋白源

---

#### 15. 动物性蛋白饲料 (Animal Protein Feeds/动物蛋白)

**白话解释**:  
动物来源的高蛋白饲料，蛋白品质好。

**生活例子**:  
- 鱼粉蛋白含量高、品质好
- 肉骨粉、血粉也是动物蛋白

**学术定义**:  
来源于动物的蛋白质饲料，蛋白含量高、氨基酸平衡好。

**主要动物蛋白饲料**:
```
动物蛋白饲料
├── 鱼粉
│   ├── 蛋白：60-72%
│   ├── 特点：蛋白品质最好、未知生长因子
│   ├── 缺点：价格高、可能含组胺
│   └── 用量：2-8%
├── 肉骨粉
│   ├── 蛋白：45-55%
│   ├── 特点：钙磷丰富
│   └── 用量：<5%
├── 血粉
│   ├── 蛋白：80-90%
│   ├── 缺点：异亮氨酸低、消化率低
│   └── 用量：<3%
└── 羽毛粉
    ├── 蛋白：80-85%
    ├── 缺点：赖氨酸、蛋氨酸低
    └── 用量：<3%
```

---

### 三、粗饲料

#### 16. 粗饲料 (Roughage/粗饲料)

**白话解释**:  
粗纤维含量高的饲料，主要给牛羊吃。

**生活例子**:  
- 干草、秸秆是粗饲料
- 青贮玉米是优质粗饲料

**学术定义**:  
干物质中粗纤维≥18%的饲料，体积大、能量低。

**主要粗饲料**:
```
粗饲料
├── 干草
│   ├── 羊草干草
│   ├── 苜蓿干草（优质）
│   └── 杂草干草
├── 秸秆
│   ├── 玉米秸秆
│   ├── 小麦秸秆
│   └── 稻草
├── 青贮饲料
│   ├── 玉米青贮
│   ├── 苜蓿青贮
│   └── 秸秆青贮
└── 其他
    ├── 糟渣类
    └── 糖蜜
```

**粗饲料品质评价**:
| 指标 | 优质 | 中等 | 差 |
|------|------|------|-----|
| CP(%) | >15 | 10-15 | <10 |
| NDF(%) | <50 | 50-60 | >60 |
| ADF(%) | <35 | 35-45 | >45 |
| 消化率(%) | >65 | 55-65 | <55 |

### 四、饲养标准与配方设计

#### 17. 饲养标准 (Feeding Standards/饲养标准)

**白话解释**:  
动物每天需要多少营养的"参考书"。

**生活例子**:  
- NRC标准（美国）
- 中国猪饲养标准

**学术定义**:  
根据动物不同生理阶段和生产水平，规定的各种营养物质需要量。

**常用饲养标准**:
```
饲养标准
├── NRC（美国国家研究委员会）
│   ├── 《猪营养需要》
│   ├── 《家禽营养需要》
│   └── 《奶牛营养需要》
├── ARC（英国农业研究委员会）
├── INRA（法国）
└── 中国饲养标准
    ├── NY/T 65-2004《猪饲养标准》
    └── NY/T 33-2004《鸡饲养标准》
```

**饲养标准内容**:
1. **维持需要** - 维持生命活动
2. **生产需要** - 生长、产奶、产蛋、繁殖
3. **总需要量** = 维持 + 生产

---

#### 18. 饲料配方设计 (Feed Formulation/配方设计)

**白话解释**:  
按照营养需要，计算各种饲料用多少。

**生活例子**:  
- 猪饲料配方：玉米60%+豆粕25%+麸皮10%+预混料5%
- 配方要满足营养需要，还要便宜

**学术定义**:  
根据动物营养需要和饲料营养成分，计算各种饲料原料的配合比例。

**配方设计原则**:
```
配方设计原则
├── 营养性原则
│   ├── 满足营养需要
│   ├── 氨基酸平衡
│   └── 钙磷平衡
├── 经济性原则
│   └── 成本最低
├── 适口性原则
│   └── 动物爱吃
├── 安全性原则
│   └── 无毒害、符合法规
└── 实用性原则
    └── 原料可获得、易加工
```

**配方设计方法**:
```
配方设计方法
├── 试差法
│   ├── 先拟定初步配方
│   ├── 计算营养成分
│   └── 调整至符合标准
├── 方形法（对角线法）
│   └── 适用于两种原料
├── 线性规划法（计算机）
│   ├── 目标：成本最低
│   └── 约束：营养需要
└── 多目标规划法
    └── 考虑多个目标
```

---

#### 19. 配合饲料 (Formula Feed/配合饲料)

**白话解释**:  
按配方混合好的饲料，直接喂动物。

**生活例子**:  
- 全价料：营养全面，直接喂
- 浓缩料：加玉米后喂

**学术定义**:  
根据动物营养需要，将多种饲料原料按一定比例配合而成的饲料。

**配合饲料分类**:
```
配合饲料类型
├── 全价配合饲料
│   └── 营养全面，可直接饲喂
├── 浓缩饲料
│   ├── 蛋白+矿物质+维生素
│   └── 需加能量饲料
├── 精料补充料
│   └── 补充粗饲料不足
├── 添加剂预混料
│   ├── 维生素+微量元素+药物
│   └── 用量0.5-5%
└── 代乳料
    └── 人工乳
```

**配合饲料形态**:
```
配合饲料形态
├── 粉料
│   ├── 加工简单
│   └── 易挑食、浪费
├── 颗粒料
│   ├── 适口性好、浪费少
│   └── 加工成本高
├── 膨化料
│   ├── 消化率高
│   └── 成本最高
└── 液体饲料
    └── 代乳料、糖蜜
```

#### 20. 饲料添加剂 (Feed Additives/饲料添加剂)

**白话解释**:  
饲料里加的"调料"和"保健品"，量少但作用大。

**生活例子**:  
- 抗生素促生长（现在限制使用）
- 酶制剂帮助消化
- 益生菌调理肠道

**学术定义**:  
在饲料加工、贮存、使用过程中添加的少量或微量物质。

**添加剂分类**:
```
饲料添加剂分类
├── 营养性添加剂
│   ├── 维生素
│   ├── 微量元素
│   ├── 氨基酸
│   └── 非蛋白氮（反刍动物）
├── 非营养性添加剂
│   ├── 生长促进剂
│   │   ├── 抗生素（限用）
│   │   ├── 益生菌
│   │   ├── 益生元
│   │   ├── 酶制剂
│   │   └── 植物提取物
│   ├── 驱虫保健剂
│   ├── 饲料保存剂
│   │   ├── 抗氧化剂
│   │   └── 防霉剂
│   └── 其他
│       ├── 调味剂
│       ├── 着色剂
│       └── 粘结剂
└── 药物饲料添加剂
    └── 需按兽药管理
```

**常用添加剂**:
| 类型 | 举例 | 作用 |
|------|------|------|
| 氨基酸 | 赖氨酸、蛋氨酸 | 补充限制性氨基酸 |
| 酶制剂 | 植酸酶、木聚糖酶 | 提高消化率 |
| 益生菌 | 枯草芽孢杆菌、乳酸菌 | 调理肠道 |
| 酸化剂 | 柠檬酸、延胡索酸 | 降低胃pH |
| 抗氧化剂 | 乙氧喹、BHT | 防止氧化 |

**添加剂使用原则**:
1. **安全性** - 对动物和人体无害
2. **有效性** - 确有作用
3. **合法性** - 符合法规
4. **经济性** - 成本效益合理

## 🎯 重点概念速记

### 必背概念（20个）

1. **营养物质六要素** - 水、蛋、脂、碳、矿、维
2. **粗蛋白(CP)** - N × 6.25
3. **必需氨基酸** - 动物不能合成的氨基酸
4. **限制性氨基酸** - 缺乏最严重的必需氨基酸
5. **粗纤维** - 不溶于稀酸稀碱的有机物
6. **NDF/ADF** - 中性/酸性洗涤纤维
7. **脂肪能量** - 碳水化合物的2.25倍
8. **必需脂肪酸** - 亚油酸、亚麻酸、花生四烯酸
9. **常量元素** - Ca、P、Na、Cl、K、Mg、S
10. **微量元素** - Fe、Cu、Zn、Mn、I、Se等
11. **脂溶性维生素** - A、D、E、K
12. **水溶性维生素** - B族、C
13. **能量饲料** - CP<20%、CF<18%
14. **蛋白饲料** - CP≥20%、CF<18%
15. **粗饲料** - CF≥18%
16. **饲养标准** - 营养需要量规定
17. **全价饲料** - 营养全面、可直接饲喂
18. **浓缩料** - 蛋白+矿+维，需加能量饲料
19. **预混料** - 添加剂混合物，用量0.5-5%
20. **酶制剂** - 提高饲料消化率

## 💡 实务应用场景

### 场景1：设计仔猪饲料配方
- **问题**：如何设计20-50kg生长猪饲料配方？
- **动物营养与饲料解答**：
  1. 查饲养标准，确定营养需要
     - DE: 13.4 MJ/kg, CP: 16%, Lys: 0.85%
  2. 选择原料：玉米、豆粕、麦麸、预混料
  3. 计算配方（试差法或电脑软件）
     - 玉米65%、豆粕22%、麦麸10%、预混料3%
  4. 检查营养指标是否达标
  5. 计算成本

### 场景2：奶牛日粮配合
- **问题**：奶牛如何配制日粮？
- **动物营养与饲料解答**：
  1. 确定干物质采食量(DMI)
  2. 满足粗饲料需要(NDF>25%)
  3. 补充精料满足能量和蛋白
  4. 添加矿物质和维生素
  5. 注意瘤胃健康（粗精比）
