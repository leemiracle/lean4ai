### 一、必需元素概述

#### 1. 必需营养元素 (Essential Elements/必需元素)

**白话解释**:  
植物生长离不开的"营养品"，少了就不行。

**生活例子**:  
- 人需要吃饭、喝水、吃菜
- 植物需要氮、磷、钾等元素

**学术定义**:  
植物正常生长发育不可缺少的元素，缺乏时植物不能完成生活史。

**判断标准（Arnon标准）**:
1. **必要性** - 缺乏该元素，植物不能完成生活史
2. **专一性** - 该元素的功能不能被其他元素替代
3. **直接性** - 该元素直接参与植物代谢

**17种必需元素**:
```
植物必需元素（17种）
├── 大量元素（9种）
│   ├── 碳 C、氢 H、氧 O ← 来自CO₂、H₂O
│   ├── 氮 N、磷 P、钾 K ← 肥料三要素
│   └── 钙 Ca、镁 Mg、硫 S ← 中量元素
└── 微量元素（8种）
    ├── 铁 Fe、锰 Mn、锌 Zn
    ├── 铜 Cu、硼 B、钼 Mo
    └── 氯 Cl、镍 Ni

含量顺序：C > O > H > N > K > Ca > Mg > P > S > 微量元素
```

**费曼记忆**:  
必需元素 = 9大量 + 8微量 = 17种

---

#### 2. 营养元素分类 (Nutrient Classification/营养元素分类)

**白话解释**:  
把17种元素按含量多少和来源分类。

**生活例子**:  
- 碳、氢、氧来自空气和水
- 氮磷钾需要施肥补充

**学术定义**:  
根据元素在植物体内含量、生理功能、吸收形态进行的分类。

**按含量分类**:
| 分类 | 含量(占干重) | 元素 |
|------|-------------|------|
| 大量元素 | >0.1% | C、H、O、N、P、K、Ca、Mg、S |
| 微量元素 | <0.1% | Fe、Mn、Zn、Cu、B、Mo、Cl、Ni |

**按来源分类**:
```
营养元素来源
├── 来自空气和水
│   └── 碳(C)、氢(H)、氧(O)
├── 来自土壤（一般不缺）
│   └── 钙(Ca)、镁(Mg)、硫(S)、微量元素
└── 来自土壤（常需补充）
    └── 氮(N)、磷(P)、钾(K) ← 肥料三要素
```

**按生理功能分类**:
1. **结构元素** - C、H、O、N、S（构成有机物）
2. **酶活化剂** - K、Ca、Mg、Mn、Zn（调节酶活性）
3. **电子传递** - Fe、Cu、Mo（氧化还原）
4. **其他功能** - B、Cl、Ni

### 二、大量元素

#### 3. 氮素营养 (Nitrogen/氮)

**白话解释**:  
氮是植物的"长叶剂"，让叶子绿油油、植株长得快。

**生活例子**:  
- 氮肥施多了，庄稼疯长
- 缺氮时叶片发黄

**学术定义**:  
氮是构成蛋白质、核酸、叶绿素的重要元素，促进营养生长。

**氮的生理功能**:
```
氮的生理作用
├── 构成蛋白质（1/6的氮）
├── 构成核酸（DNA、RNA）
├── 构成叶绿素（光合作用）
├── 构成酶、激素
└── 参与维生素合成
```

**氮素形态**:
| 形态 | 特点 | 植物吸收 |
|------|------|---------|
| 铵态氮(NH₄⁺) | 带正电、易被吸附 | 直接吸收 |
| 硝态氮(NO₃⁻) | 带负电、易淋失 | 直接吸收 |
| 酰胺态氮(尿素) | 有机态、需转化 | 转化后吸收 |

**缺氮症状**:
1. **老叶先发黄** - 氮可移动，从老叶向新叶转移
2. **植株矮小** - 生长受抑
3. **分蘖/分枝减少**
4. **叶片薄、小**

**氮过剩症状**:
- 徒长、易倒伏
- 延迟成熟
- 抗病虫害能力下降

---

#### 4. 磷素营养 (Phosphorus/磷)

**白话解释**:  
磷是植物的"生根剂"，促进根系发育和开花结果。

**生活例子**:  
- 缺磷时叶片发紫
- 苗期施磷促进长根

**学术定义**:  
磷是ATP、核酸、磷脂的组成成分，参与能量代谢和遗传信息传递。

**磷的生理功能**:
```
磷的生理作用
├── 构成ATP（能量货币）
├── 构成核酸（DNA、RNA）
├── 构成磷脂（细胞膜）
├── 参与光合作用、呼吸作用
└── 促进根系发育、开花结实
```

**磷素形态**:
- 土壤中主要是H₂PO₄⁻和HPO₄²⁻
- pH 5-7时有效性最高

**缺磷症状**:
1. **叶片暗绿或紫红** - 花青素积累
2. **根系发育差** - 根少而短
3. **开花迟、成熟晚**
4. **籽粒不饱满**

**磷肥特点**:
- 移动性差（不要撒表面）
- 当季利用率低（15-25%）
- 后效长

---

#### 5. 钾素营养 (Potassium/钾)

**白话解释**:  
钾是植物的"健身剂"，增强抗性、提高品质。

**生活例子**:  
- 施钾肥的西瓜更甜
- 缺钾容易倒伏

**学术定义**:  
钾是60多种酶的活化剂，调节渗透压，增强植物抗逆性。

**钾的生理功能**:
```
钾的生理作用
├── 酶的活化剂（60+种酶）
├── 调节渗透压（气孔开闭）
├── 促进光合产物运输
├── 增强抗逆性（抗旱、抗寒、抗病）
└── 改善品质（增加糖分、蛋白质）
```

**钾肥类型**:
| 钾肥品种 | K₂O含量 | 特点 |
|---------|---------|------|
| 氯化钾 | 60% | 含氯，忌氯作物慎用 |
| 硫酸钾 | 50% | 不含氯，适合各种作物 |
| 硝酸钾 | 44% | 氮钾复合，水溶性好 |
| 草木灰 | 5-10% | 有机钾肥，碱性 |

**缺钾症状**:
1. **老叶叶缘焦枯** - 像被火烧过
2. **叶片卷曲、皱缩**
3. **茎秆软弱，易倒伏**
4. **品质下降**

---

#### 6. 中量元素 (Secondary Elements/中量元素)

**白话解释**:  
钙、镁、硫虽然需求量比氮磷钾少，但也很重要。

**生活例子**:  
- 缺钙导致番茄"脐腐病"
- 缺镁叶片叶脉间失绿

**学术定义**:  
植物需要量介于大量元素和微量元素之间的元素，包括钙、镁、硫。

**钙(Ca)的生理功能**:
1. **构成细胞壁** - 果胶酸钙
2. **稳定细胞膜**
3. **信号传递** - 钙信使
4. **缓解其他元素毒害**

**缺钙症状**:
- 新叶变形、叶尖粘在一起
- 根尖坏死
- 番茄脐腐病、大白菜干烧心

**镁(Mg)的生理功能**:
1. **叶绿素的核心** - 叶绿素分子的中心
2. **酶的活化剂** - RuBP羧化酶等
3. **参与ATP反应**

**缺镁症状**:
- 老叶叶脉间失绿（叶脉仍绿）
- 严重时出现坏死斑

**硫(S)的生理功能**:
1. **构成蛋白质** - 含硫氨基酸
2. **构成辅酶A、铁氧还蛋白**
3. **参与芥子油合成**（十字花科）

**缺硫症状**:
- 新叶发黄（类似缺氮，但新叶先表现）
- 植株矮小

---

### 三、微量元素

#### 7. 微量元素营养 (Micronutrients/微量元素)

**白话解释**:  
虽然需求量少，但缺一不可，像维生素一样。

**生活例子**:  
- 缺铁叶片发黄（叶脉间失绿）
- 缺硼"花而不实"

**学术定义**:  
植物需求量小于干重0.1%的必需元素，参与酶活性、电子传递等过程。

**主要微量元素功能**:
```
微量元素功能
├── 铁(Fe)
│   ├── 叶绿素合成必需
│   ├── 电子传递（细胞色素）
│   └── 缺素：新叶失绿
├── 锰(Mn)
│   ├── 光系统II的成分
│   ├── 酶活化剂
│   └── 缺素：新叶失绿、坏死斑
├── 锌(Zn)
│   ├── 生长素合成
│   ├── 多种酶的成分
│   └── 缺素：小叶病
├── 硼(B)
│   ├── 花粉管伸长
│   ├── 糖的运输
│   └── 缺素：花而不实、空心病
├── 铜(Cu)
│   ├── 氧化酶成分
│   ├── 光合作用
│   └── 缺素：新叶失绿、扭曲
└── 钼(Mo)
    ├── 硝酸还原酶成分
    ├── 固氮酶成分
    └── 缺素：老叶失绿、鞭尾病
```

**微量元素缺乏诊断**:
| 元素 | 移动性 | 缺素部位 | 典型症状 |
|------|--------|---------|---------|
| Fe | 难移动 | 新叶 | 叶脉间失绿 |
| Mn | 难移动 | 新叶 | 失绿、坏死斑 |
| Zn | 可移动 | 老叶/新叶 | 小叶、簇生 |
| B | 难移动 | 新叶 | 花而不实、空心 |
| Cu | 难移动 | 新叶 | 失绿、扭曲 |
| Mo | 可移动 | 老叶 | 失绿、鞭尾 |

### 一、根系吸收

#### 8. 养分吸收途径 (Nutrient Uptake Pathways/养分吸收途径)

**白话解释**:  
养分通过两种方式进入根部：走"快速通道"（质外体）或"安检通道"（共质体）。

**生活例子**:  
- 走高速公路（快，但可能被拦）
- 走小路（慢，但安全）

**学术定义**:  
养分从土壤进入植物体内的途径，包括质外体途径和共质体途径。

**两条途径**:
```
养分吸收途径
├── 质外体途径（Apoplast pathway）
│   ├── 走细胞壁和细胞间隙
│   ├── 速度快
│   └── 在内皮层被凯氏带阻挡 → 必须跨膜
└── 共质体途径（Symplast pathway）
    ├── 走细胞质，通过胞间连丝
    ├── 速度慢
    └── 有选择性
```

**凯氏带的作用**:
- 内皮层细胞的带状结构
- 阻挡质外体水流
- 强迫养分跨膜选择性吸收

---

#### 9. 离子跨膜运输 (Ion Transport Across Membrane/离子跨膜运输)

**白话解释**:  
离子要过"城门"（细胞膜）进入细胞，有被动和主动两种方式。

**生活例子**:  
- 顺流而下（被动运输）
- 逆流而上（主动运输，要消耗能量）

**学术定义**:  
离子通过细胞膜的方式，包括被动运输和主动运输。

**运输方式分类**:
```
跨膜运输
├── 被动运输（不耗能，顺浓度梯度）
│   ├── 简单扩散
│   ├── 离子通道
│   └── 协助扩散（载体蛋白）
└── 主动运输（耗能，逆浓度梯度）
    └── 离子泵（H⁺-ATPase、Ca²⁺-ATPase）
```

**离子通道特点**:
1. **速度快** - 是载体的1000倍
2. **门控性** - 受电压、配体调控
3. **选择性** - 对特定离子开放

**离子载体特点**:
1. **有选择性**
2. **可饱和** - 符合米氏方程
3. **竞争性抑制**

---

#### 10. 影响养分吸收的因素 (Factors Affecting Uptake/影响吸收的因素)

**白话解释**:  
植物"胃口"好不好，受很多因素影响。

**生活例子**:  
- 温度低，植物吸收慢
- 土壤太干，养分吸收不了

**学术定义**:  
影响根系吸收养分的内部和外部因素。

**主要影响因素**:
```
影响养分吸收的因素
├── 内部因素
│   ├── 根系形态（根长、根表面积）
│   ├── 植物种类和品种
│   ├── 生长阶段（需肥临界期）
│   └── 代谢强度
└── 外部因素
    ├── 温度 → 影响代谢和扩散
    ├── 水分 → 影响离子溶解和运输
    ├── 通气 → 影响根系呼吸
    ├── pH → 影响养分有效性
    ├── 离子相互作用（拮抗/协同）
    └── 土壤溶液浓度
```

**温度影响**:
- 最适温度：20-30℃
- 低温：吸收减慢
- 高温：根系受损

**pH影响**:
| pH范围 | 有效养分 |
|--------|---------|
| <5.5 | Fe、Mn、Zn、Cu有效性高，可能毒害 |
| 5.5-7.0 | 大多数养分有效性最佳 |
| >7.5 | P、Fe、Mn、Zn有效性降低 |

**离子相互作用**:
1. **拮抗作用**（竞争吸收）
   - NH₄⁺与K⁺
   - Ca²⁺与Mg²⁺
   - K⁺与Mg²⁺

2. **协同作用**（促进吸收）
   - NO₃⁻与K⁺
   - Ca²⁺与K⁺

### 二、养分运输与分配

#### 11. 养分运输 (Nutrient Transport/养分运输)

**白话解释**:  
养分从根运到叶子，再从叶子运到果实，需要"物流系统"。

**生活例子**:  
- 高速公路运货（木质部）
- 回程车送货（韧皮部）

**学术定义**:  
养分在植物体内的运输途径和方式，包括木质部运输和韧皮部运输。

**两条运输途径**:
```
养分运输途径
├── 木质部运输（xylem）
│   ├── 方向：根 → 地上部（单向）
│   ├── 动力：蒸腾拉力、根压
│   ├── 主要运输：水、无机离子
│   └── 速度：快
└── 韧皮部运输（phloem）
    ├── 方向：双向（源→库）
    ├── 动力：压力流学说
    ├── 主要运输：光合产物、有机氮磷
    └── 速度：慢
```

**养分再利用**:
```
养分移动性分类
├── 高度移动（N、P、K、Mg）
│   └── 缺素症状先出现在老叶
├── 中等移动（S、Mo、Zn）
│   └── 缺素症状新老叶都出现
└── 难移动（Ca、B、Fe、Mn、Cu）
    └── 缺素症状先出现在新叶
```

---

#### 12. 养分分配 (Nutrient Partitioning/养分分配)

**白话解释**:  
养分运到植物各部分，优先供给"最重要的器官"。

**生活例子**:  
- 开花结果时，养分优先运到果实
- 缺素时，老叶先牺牲

**学术定义**:  
养分在植物各器官之间的分配规律，遵循源库关系和优先级。

**分配优先级**:
```
养分分配优先级
├── 最高：生长点、幼叶、花果
├── 中等：功能叶
└── 最低：老叶、茎、根

分配原则：
1. 生长中心优先
2. 竞争力：新叶 > 老叶
3. 生殖器官 > 营养器官（后期）
```

**源库关系**:
- **源（Source）**: 输出养分的器官（成熟叶、根系）
- **库（Sink）**: 接收养分的器官（幼叶、花、果、根）

**应用**:
- 棉花打顶 → 源库关系改变 → 蕾铃获得更多养分
- 果树疏花疏果 → 减少库的数量 → 提高果实品质

### 三、营养诊断方法

#### 13. 营养诊断 (Nutrient Diagnosis/营养诊断)

**白话解释**:  
给植物"体检"，看它缺什么营养。

**生活例子**:  
- 看叶子颜色判断缺什么肥
- 测土配方施肥

**学术定义**:  
判断植物营养状况的方法，包括形态诊断、化学诊断、酶学诊断等。

**诊断方法**:
```
营养诊断方法
├── 形态诊断
│   ├── 看缺素症状
│   ├── 简单直观
│   └── 缺点：出现症状已晚
├── 化学诊断
│   ├── 土壤测试（测土）
│   ├── 植株分析（测植株）
│   └── 优点：量化准确
├── 酶学诊断
│   ├── 测酶活性
│   └── 优点：早期发现
└── 施肥诊断
    └── 通过施肥试验验证
```

**缺素症诊断表**:
| 症状部位 | 症状特征 | 可能缺乏元素 |
|---------|---------|------------|
| 老叶 | 失绿、坏死 | N、K、Mg、P、Zn、Mo |
| 新叶 | 失绿、变形 | Ca、B、Fe、Mn、Cu、S |
| 叶缘 | 焦枯 | K |
| 叶脉间 | 失绿 | Mg、Fe、Mn、Zn |
| 整株 | 紫红、矮小 | P |
| 顶芽 | 坏死 | Ca、B |

---

#### 14. 植株分析 (Plant Analysis/植株分析)

**白话解释**:  
化验植株的养分含量，判断营养状况。

**生活例子**:  
- 抽血化验身体营养状况
- 测叶片氮含量指导追肥

**学术定义**:  
通过分析植物组织的养分含量，诊断营养状况的方法。

**常用指标**:
1. **全量分析** - 测总养分含量
2. **速测** - 测组织汁液养分
3. **临界值法** - 低于某值即缺乏

**主要元素临界值**（参考）:
| 作物 | 部位 | N(%) | P(%) | K(%) |
|------|------|------|------|------|
| 水稻 | 叶片 | 2.5-4.0 | 0.2-0.4 | 1.5-3.0 |
| 小麦 | 叶片 | 3.0-4.5 | 0.2-0.5 | 1.5-3.0 |
| 玉米 | 穗位叶 | 2.8-4.0 | 0.25-0.5 | 1.5-2.5 |

**DRIS法**（综合诊断施肥法）:
- 考虑养分之间的平衡
- 比单一临界值更科学

### 四、施肥原理与方法

#### 15. 施肥的基本原理 (Fertilization Principles/施肥原理)

**白话解释**:  
施肥有原则，不能乱来。

**生活例子**:  
- 缺什么补什么
- 不能一次吃太饱

**学术定义**:  
指导科学施肥的基本原则，包括养分归还学说、最小养分律、报酬递减律等。

**四大施肥原理**:
```
施肥基本原理
├── 1. 养分归还学说（Liebig）
│   └── 每次收获都带走养分 → 必须归还
├── 2. 最小养分律（Liebig）
│   └── 产量受最低养分限制（木桶理论）
├── 3. 报酬递减律（Mitscherlich）
│   └── 施肥越多，增产越少
└── 4. 因子综合作用律
    └── 产量受多种因子影响
```

**最小养分律**:
```
木桶理论：
产量 = 最短缺养分的供给量

例如：
土壤缺氮 → 施钾无效
必须补足最缺的元素
```

**报酬递减律**:
```
施肥量与产量关系
产量
  ↑
  │      _________
  │    /
  │  /
  │/
  └────────────→ 施肥量

边际产量递减
```

#### 16. 配方施肥 (Formula Fertilization/配方施肥)

**白话解释**:  
测土配方，缺什么补什么，需要多少施多少。

**生活例子**:  
- 医生开药方，对症下药
- 测土后按配方施肥

**学术定义**:  
根据土壤供肥能力和作物需肥规律，确定施肥量、比例、时期和方法的技术。

**配方施肥方法**:
```
配方施肥流程
├── 第一步：测土
│   └── 测定土壤养分含量
├── 第二步：配方
│   ├── 确定目标产量
│   ├── 计算需肥量
│   └── 确定养分配比
└── 第三步：施肥
    ├── 确定施肥时期
    ├── 确定施肥方法
    └── 选择肥料品种
```

**施肥量计算公式**:
```
施肥量 = (目标产量需肥量 - 土壤供肥量) / 肥料利用率

目标产量需肥量 = 目标产量 × 单位产量吸肥量
土壤供肥量 = 土壤养分测定值 × 0.15 × 校正系数
```

**示例（水稻）**:
```
目标产量：600 kg/亩
需氮量：600 × 0.02 = 12 kg N/亩
土壤供氮：100 × 0.15 × 0.6 = 9 kg N/亩
需补氮：(12-9)/0.4 = 7.5 kg尿素/亩
```

#### 17. 施肥时期与方法 (Fertilization Timing and Methods/施肥时期与方法)

**白话解释**:  
什么时候施肥、怎么施肥，有讲究。

**生活例子**:  
- 底肥：播种前施
- 追肥：生长期间施
- 叶面喷肥：快速补肥

**学术定义**:  
根据作物需肥规律，确定最佳施肥时期和施肥方式的技术。

**施肥时期**:
```
施肥时期
├── 基肥（底肥）
│   ├── 播种或移栽前施
│   ├── 占总施肥量40-60%
│   └── 有机肥+缓释肥
├── 种肥
│   ├── 播种时施
│   ├── 少量速效肥
│   └── 促进幼苗生长
└── 追肥
    ├── 生长期施
    ├── 多次追肥
    └── 关键期重施
```

**作物营养关键期**:
| 关键期 | 特点 | 施肥重点 |
|--------|------|---------|
| 营养临界期 | 需求迫切、敏感 | 苗期补磷 |
| 最大效率期 | 肥效最高 | 穗肥、花肥 |

**施肥方法**:
```
施肥方法
├── 土壤施肥
│   ├── 撒施
│   ├── 条施/穴施
│   └── 深施
├── 叶面施肥
│   ├── 优点：快、省
│   ├── 缺点：量少、成本高
│   └── 适用：微量元素、生长后期
└── 灌溉施肥（水肥一体化）
    ├── 滴灌施肥
    └── 喷灌施肥
```

#### 18. 肥料类型 (Fertilizer Types/肥料类型)

**白话解释**:  
肥料有很多种，各有特点。

**生活例子**:  
- 尿素、二铵、氯化钾是化肥
- 农家肥是有机肥

**学术定义**:  
提供植物养分的物料，按来源分为有机肥、化肥、生物肥等。

**肥料分类**:
```
肥料分类
├── 化学肥料（无机肥）
│   ├── 氮肥：尿素、碳铵、硫铵
│   ├── 磷肥：过磷酸钙、磷酸二铵
│   ├── 钾肥：氯化钾、硫酸钾
│   └── 复合肥：氮磷钾复合肥
├── 有机肥料
│   ├── 农家肥（厩肥、堆肥）
│   ├── 秸秆还田
│   └── 绿肥
└── 生物肥料
    ├── 根瘤菌肥
    ├── 固氮菌肥
    └── 解磷解钾菌肥
```

**常见氮肥**:
| 氮肥 | 含N量 | 特点 | 适用性 |
|------|-------|------|--------|
| 尿素 | 46% | 中性、水溶、广泛使用 | 各种土壤作物 |
| 碳酸氢铵 | 17% | 易挥发、有氨味 | 水田 |
| 硫酸铵 | 21% | 生理酸性 | 忌氯作物 |
| 氯化铵 | 25% | 生理酸性、含氯 | 非忌氯作物 |

**复合肥**:
```
N-P₂O₅-K₂O含量表示
例如：15-15-15 表示含N 15%、P₂O₅ 15%、K₂O 15%

复合肥优点：养分全面、施用方便
复合肥缺点：比例固定、可能浪费
```

**有机肥优点**:
1. 提供全面养分
2. 改良土壤结构
3. 提高土壤肥力
4. 促进微生物活动

#### 19. 豆科植物固氮 (Legume Nitrogen Fixation/生物固氮)

**白话解释**:  
豆科植物能"自己造氮"，不需要施太多氮肥。

**生活例子**:  
- 种过大豆的地，下茬作物长得好
- 花生根部长瘤子（根瘤）

**学术定义**:  
豆科植物与根瘤菌共生，将空气中的N₂还原为NH₃的过程。

**固氮机理**:
```
根瘤固氮过程
├── 根瘤菌侵入根毛
├── 形成侵入线
├── 根瘤形成
├── 根瘤菌固氮
│   └── N₂ + 8H⁺ + 8e⁻ → 2NH₃ + H₂
└── NH₃转化为氨基酸
```

**固氮条件**:
1. **根瘤菌** - 必须有相应的根瘤菌
2. **能量** - 光合产物提供碳源和能量
3. **钼** - 固氮酶的成分
4. **缺氧环境** - 根瘤内有豆血红蛋白

**应用**:
- 种植豆科绿肥
- 豆科接种根瘤菌
- 轮作倒茬

#### 20. 缺素症的矫正 (Correction of Deficiency/缺素矫正)

**白话解释**:  
发现缺什么，赶紧补什么。

**生活例子**:  
- 缺氮追施尿素
- 缺铁喷施硫酸亚铁

**学术定义**:  
针对植物缺素症状，采取相应措施补充缺乏元素的技术。

**矫正方法**:
| 缺素元素 | 矫正方法 |
|---------|---------|
| N | 追施速效氮肥（尿素） |
| P | 追施速效磷肥、叶面喷施 |
| K | 追施钾肥、叶面喷施磷酸二氢钾 |
| Ca | 叶面喷施钙肥、土壤施石灰 |
| Mg | 叶面喷施硫酸镁、土壤施镁肥 |
| Fe | 叶面喷施螯合铁、硫酸亚铁 |
| B | 叶面喷施硼肥、基施硼砂 |
| Zn | 叶面喷施硫酸锌、基施锌肥 |

**叶面喷肥注意事项**:
1. **浓度适宜** - 过高会烧伤叶片
2. **时间选择** - 阴天或傍晚
3. **喷施部位** - 叶背吸收更好
4. **次数** - 一般2-3次

**常用叶面肥浓度**:
| 肥料 | 喷施浓度 |
|------|---------|
| 尿素 | 0.5-1% |
| 磷酸二氢钾 | 0.2-0.3% |
| 硫酸亚铁 | 0.2-0.5% |
| 硫酸锌 | 0.1-0.2% |
| 硼砂 | 0.1-0.2% |

5. **磷的生理功能** - ATP、核酸、磷脂
6. **钾的生理功能** - 酶活化、渗透调节、增强抗性
7. **缺氮症状** - 老叶发黄
8. **缺磷症状** - 叶片紫红、根系差

10. **中量元素** - 钙、镁、硫
11. **微量元素** - 铁、锰、锌、铜、硼、钼、氯、镍
12. **质外体途径** - 走细胞壁
13. **共质体途径** - 走细胞质
14. **主动运输** - 逆浓度梯度，耗能

## 💡 实务应用场景

### 场景1：诊断水稻缺素
- **问题**：水稻叶片发黄，如何判断缺什么？
- **植物营养学解答**：
  1. 老叶发黄 → 可能缺氮
  2. 叶片紫红 → 可能缺磷
  3. 叶缘焦枯 → 可能缺钾
  4. 结合土壤测试确认
  5. 对症施肥矫正

### 场景2：果树施肥
- **问题**：苹果树全年如何施肥？
- **植物营养学解答**：
  1. 秋施基肥（有机肥+复合肥）
  2. 春季追肥（花前肥，氮为主）
  3. 果实膨大期追肥（钾为主）
  4. 叶面喷肥（钙、硼等微量元素）
  5. 根据产量调整施肥量
