### 一、植物病害基础

#### 1. 植物病害 (Plant Disease/植物病害)

**白话解释**:  
庄稼"生病"了，不正常了，长不好了。

**生活例子**:  
- 水稻得稻瘟病，叶子长斑、谷粒不饱满
- 小麦得锈病，叶子上像生锈一样

**学术定义**:  
植物受生物或非生物因子影响，正常生理功能受到干扰，导致生长受阻、产量下降、品质变劣的现象。

**病害类型**:
| 类型 | 病原 | 例子 |
|------|------|------|
| 侵染性病害 | 病原物（真菌、细菌、病毒等） | 稻瘟病、小麦锈病 |
| 非侵染性病害 | 环境因素（缺素、温度等） | 缺氮黄化、冻害 |

**病害症状**:
1. **病状** - 植物表现
   - 斑点、腐烂、枯萎、畸形、变色等
   
2. **病征** - 病原物表现
   - 霉状物、粉状物、锈状物、菌脓等

**费曼记忆**:  
病害 = 病原 + 寄主 + 环境（病害三角）

---

#### 2. 植物病害三角 (Disease Triangle/病害三角)

**白话解释**:  
病害发生需要三个条件同时具备：有病原、有感病寄主、有适宜环境。

**生活例子**:  
- 有稻瘟病菌，但水稻抗病，病害不重
- 有感病品种，但天气干燥，病害也轻

**学术定义**:  
植物侵染性病害发生必须具备病原物、寄主植物、环境条件三个基本要素的相互作用。

**图示**:
```
          病原物
            /\
           /  \
          /    \
         / 病害 \
        /        \
       /__________\
   寄主植物      环境条件
```

**病害三角的含义**:
1. **病原物** - 引起病害的原因
2. **寄主植物** - 被侵害的对象
3. **环境条件** - 影响病害发生的条件

**防治启示**:
- 消灭病原（灭菌）
- 种植抗病品种（改造寄主）
- 改善环境（栽培管理）

---

#### 3. 病原物 (Pathogen/病原物)

**白话解释**:  
让庄稼生病的"坏东西"。

**生活例子**:  
- 真菌像小蘑菇一样，长在植物上
- 细菌是肉眼看不见的微小生物

**学术定义**:  
引起植物病害的生物，包括真菌、细菌、病毒、线虫、寄生性种子植物等。

**主要病原物类型**:
| 类型 | 特点 | 所致病害 | 传播方式 |
|------|------|----------|----------|
| 真菌 | 有菌丝、孢子 | 稻瘟病、锈病 | 气流、水、种子 |
| 细菌 | 单细胞、无真核 | 水稻白叶枯病 | 雨水、昆虫、伤口 |
| 病毒 | 极小、无细胞结构 | 水稻条纹叶枯病 | 昆虫、嫁接 |
| 线虫 | 微小线形动物 | 大豆胞囊线虫病 | 土壤、水流 |
| 植原体 | 无细胞壁细菌 | 枣疯病 | 昆虫 |

**真菌病害**:
- 占植物病害的80%以上
- 特征：有霉状物、粉状物、小黑点
- 例子：稻瘟病、小麦锈病、玉米大小斑病

**细菌病害**:
- 特征：水渍状、菌脓
- 例子：水稻白叶枯病、白菜软腐病

**病毒病害**:
- 特征：花叶、矮缩、畸形
- 例子：水稻条纹叶枯病、烟草花叶病

---

#### 4. 侵染过程 (Infection Process/侵染过程)

**白话解释**:  
病原物从接触植物到发病的全过程。

**学术定义**:  
病原物从接触寄主植物，经侵入、潜育到发病的连续过程。

**侵染过程阶段**:
```
1. 接触期
   └── 病原物到达植物表面

2. 侵入期
   └── 病原物进入植物体内
   ├── 直接侵入（穿透表皮）
   ├── 自然孔口（气孔、水孔）
   └── 伤口侵入

3. 潜育期
   └── 病原物在体内扩展
   └── 从侵入到症状出现

4. 发病期
   └── 症状表现出来
   └── 产生繁殖体
```

**影响侵染的因素**:
1. **病原物数量** - 越多越易发病
2. **寄主抗性** - 抗病品种难侵入
3. **环境条件** - 湿度最关键

---

#### 5. 病害循环 (Disease Cycle/病害循环)

**白话解释**:  
病害从发生到再次发生的循环过程。

**学术定义**:  
病原物从一个生长季节开始侵染，到下一个生长季节再次侵染的全过程。

**病害循环环节**:
```
┌─────────────────────────────────────┐
│                                     │
│   越冬/越夏 ←───── 传播 ─────→ 侵染 │
│      ↑                           │  │
│      │                           ↓  │
│   初侵染源 ←─── 潜育发病 ←───── 侵入 │
│                                     │
└─────────────────────────────────────┘
```

**各环节说明**:
| 环节 | 内容 |
|------|------|
| 越冬/越夏 | 病原物度过不利环境 |
| 初侵染 | 生长季第一次侵染 |
| 再侵染 | 同一生长季多次侵染 |
| 传播 | 病原物从一处到另一处 |

**越冬场所**:
- 种子、苗木
- 病株残体
- 土壤
- 野生寄主
- 昆虫

**传播方式**:
| 方式 | 特点 | 例子 |
|------|------|------|
| 气流传播 | 距离远 | 小麦锈病 |
| 雨水传播 | 距离近 | 水稻白叶枯病 |
| 昆虫传播 | 病毒病主要方式 | 水稻条纹叶枯病 |
| 种子传播 | 随种子调运远距离 | 小麦腥黑穗病 |
| 土壤传播 | 局部发生 | 多种土传病害 |

### 二、主要作物病害

#### 6. 稻瘟病 (Rice Blast/稻瘟病)

**白话解释**:  
水稻的"头号杀手"，严重时能让水稻绝收。

**生活例子**:  
- 水稻叶片上出现梭形褐斑
- 穗颈变黑，造成白穗

**学术定义**:  
由稻瘟病菌（Magnaporthe oryzae）引起的水稻真菌性病害，是世界范围内最重要的水稻病害之一。

**症状类型**:
| 类型 | 危害部位 | 症状特点 |
|------|----------|----------|
| 苗瘟 | 秧苗 | 变褐枯死 |
| 叶瘟 | 叶片 | 梭形褐斑 |
| 节瘟 | 茎节 | 黑斑、易折断 |
| 穗颈瘟 | 穗颈 | 变黑、白穗 |
| 粒瘟 | 谷粒 | 褐斑 |

**发病条件**:
1. **温度** - 25-28℃最适
2. **湿度** - 高湿有利发病
3. **品种** - 不同品种抗性差异大
4. **氮肥** - 偏施氮肥加重病害

**防治方法**:
```
1. 农业防治
   ├── 选用抗病品种
   ├── 合理施肥（不过量施氮）
   └── 科学管水（浅水勤灌）

2. 化学防治
   ├── 种子处理（浸种消毒）
   └── 发病初期喷药（三环唑等）
```

---

#### 7. 小麦锈病 (Wheat Rust/小麦锈病)

**白话解释**:  
小麦叶子上长"铁锈"，影响光合作用。

**生活例子**:  
- 条锈病：叶子上的黄条纹
- 叶锈病：叶子上的橙红色斑点

**学术定义**:  
由锈菌引起的小麦真菌性病害，包括条锈病、叶锈病、秆锈病三种。

**三种锈病对比**:
| 类型 | 病原 | 症状特点 | 发生特点 |
|------|------|----------|----------|
| 条锈病 | 条形柄锈菌 | 黄条纹、排列成行 | 发生早、危害重 |
| 叶锈病 | 小麦隐匿柄锈菌 | 橙红圆形斑点 | 发生较晚 |
| 秆锈病 | 禾柄锈菌 | 茎秆上大红斑 | 发生最晚 |

**条锈病流行规律**:
```
西北越夏区（夏季）→ 秋季气流传播 → 华北冬麦区（冬季）→ 春季流行
```

**防治方法**:
1. **品种布局** - 种植抗病品种
2. **药剂拌种** - 播种前用三唑类拌种
3. **秋苗防治** - 早春喷药预防
4. **流行期防治** - 发现中心病株及时防治

### 一、昆虫基础知识

#### 8. 昆虫 (Insect/昆虫)

**白话解释**:  
六条腿、身体分节的小动物，有些吃庄稼，是害虫。

**生活例子**:  
- 蝗虫 - 大面积吃庄稼
- 蚜虫 - 吸食植物汁液
- 蚊子 - 叮人吸血

**学术定义**:  
节肢动物门昆虫纲动物，身体分为头、胸、腹三部分，有三对足，通常有两对翅。

**昆虫基本特征**:
```
        头部
         ├── 触角（感觉）
         ├── 复眼（视觉）
         └── 口器（取食）
         
        胸部
         ├── 三节
         ├── 三对足
         └── 通常两对翅
         
        腹部
         ├── 多节
         └── 生殖、消化系统
```

**昆虫口器类型**:
| 类型 | 特点 | 代表害虫 | 危害方式 |
|------|------|----------|----------|
| 咀嚼式 | 像牙齿，啃食 | 蝗虫、螟虫 | 吃叶子、蛀茎 |
| 刺吸式 | 像针管，吸汁 | 蚜虫、飞虱 | 吸汁液、传毒 |
| 舐吸式 | 舔食 | 家蝇 | 污染食物 |
| 虹吸式 | 像吸管 | 蝴蝶 | 吸花蜜 |

---

#### 9. 害虫 (Pest/害虫)

**白话解释**:  
吃庄稼、传播疾病的"坏虫子"。

**生活例子**:  
- 水稻螟虫蛀茎，造成"枯心"
- 玉米螟蛀穗，影响产量

**学术定义**:  
对人类生产、生活造成危害的昆虫，包括农业害虫、卫生害虫等。

**害虫分类**:
| 分类标准 | 类型 | 例子 |
|----------|------|------|
| 取食方式 | 咀嚼式 | 蝗虫、玉米螟 |
|          | 刺吸式 | 蚜虫、飞虱 |
| 取食范围 | 单食性 | 三化螟（只吃水稻） |
|          | 多食性 | 粘虫（吃多种作物） |
| 危害部位 | 食叶类 | 粘虫、稻纵卷叶螟 |
|          | 蛀茎类 | 二化螟、玉米螟 |
|          | 食根类 | 蛴螬、金针虫 |
|          | 食果类 | 桃小食心虫 |

---

#### 10. 昆虫变态 (Metamorphosis/变态)

**白话解释**:  
虫子从小到大"变身"的过程。

**生活例子**:  
- 毛毛虫变成蝴蝶
- 蛆变成苍蝇

**学术定义**:  
昆虫从卵到成虫的生长发育过程中，形态、结构、生活习性发生显著变化的现象。

**变态类型**:
| 类型 | 过程 | 例子 |
|------|------|------|
| 完全变态 | 卵→幼虫→蛹→成虫 | 蝴蝶、蝇、甲虫 |
| 不完全变态 | 卵→若虫→成虫 | 蝗虫、蚜虫、飞虱 |

**完全变态各虫态特点**:
```
卵
└── 不活动，保护胚胎

幼虫
├── 活动取食期
├── 危害作物的主要虫态
└── 需蜕皮多次（龄期）

蛹
├── 不食不动
├── 内部重组
└── 是防治关键期

成虫
├── 繁殖期
├── 可能危害（取食）
└── 防治重点是阻止产卵
```

**不完全变态**:
```
卵 → 若虫 → 成虫
     ↑
  若虫像小成虫
  逐渐长大
```

---

#### 11. 害虫生活史 (Life History/生活史)

**白话解释**:  
害虫一年怎么过，生几代。

**学术定义**:  
害虫从卵开始到成虫繁殖后代的全过程，包括年生活史和世代。

**关键概念**:
1. **世代 (Generation)**: 从卵到成虫产生下一代的过程
2. **年生活史**: 一年内各世代的发生规律
3. **越冬**: 冬季停止发育
4. **越夏**: 夏季停止发育

**世代命名**:
```
越冬代 → 第1代 → 第2代 → ... → 越冬代
  ↑                                  ↑
去年越冬虫                        今年越冬虫
```

**发生代数**:
| 害虫 | 年发生代数 | 越冬虫态 |
|------|------------|----------|
| 二化螟 | 2-3代 | 幼虫 |
| 粘虫 | 2-5代 | 成虫/蛹 |
| 蚜虫 | 10-30代 | 卵/成蚜 |

### 二、主要害虫

#### 12. 水稻螟虫 (Rice Stem Borer/水稻螟虫)

**白话解释**:  
蛀食水稻茎秆的虫子，造成"枯心""白穗"。

**生活例子**:  
- 稻田里发现"枯心苗" - 螟虫危害
- 抽穗后发现"白穗" - 螟虫危害

**学术定义**:  
危害水稻的螟虫总称，主要包括二化螟、三化螟、大螟等，属鳞翅目螟蛾科。

**三种螟虫对比**:
| 特征 | 二化螟 | 三化螟 | 大螟 |
|------|--------|--------|------|
| 寄主 | 水稻为主，也吃玉米 | 仅水稻 | 水稻、玉米、甘蔗 |
| 越冬 | 幼虫在稻桩 | 幼虫在稻桩 | 幼虫在稻桩 |
| 危害状 | 枯鞘、枯心、白穗 | 枯心、白穗 | 枯心、白穗 |
| 世代数 | 2-3代/年 | 3-4代/年 | 3-4代/年 |

**危害特点**:
```
苗期受害 → 枯鞘（叶鞘变黄）→ 枯心（心叶枯死）
孕穗期受害 → 枯孕穗
抽穗后受害 → 白穗（谷粒不实）
```

**防治方法**:
1. **农业防治**:
   - 处理稻桩，减少越冬虫源
   - 适时播种，避害
   
2. **物理防治**:
   - 灯光诱杀成虫
   
3. **化学防治**:
   - 卵孵化高峰期施药
   - 使用氯虫苯甲酰胺等药剂

---

#### 13. 蚜虫 (Aphid/蚜虫)

**白话解释**:  
小小的绿虫子，趴在嫩叶、嫩梢上吸汁。

**生活例子**:  
- 小麦穗上有绿色小虫
- 蔬菜嫩叶卷曲，背面有蚜虫

**学术定义**:  
半翅目蚜科昆虫的统称，以刺吸式口器吸食植物汁液，繁殖力强。

**蚜虫特点**:
1. **繁殖快** - 一年10-30代
2. **孤雌生殖** - 不交配就能繁殖
3. **传播病毒** - 是病毒病主要传播者

**危害方式**:
```
直接危害
├── 吸食汁液 → 植物失水、卷叶
├── 分泌蜜露 → 煤污病
└── 刺激畸形 → 瘤蚜

间接危害
└── 传播病毒 → 病毒病
```

**主要蚜虫种类**:
| 蚜虫 | 寄主 | 危害 |
|------|------|------|
| 麦长管蚜 | 小麦 | 穗部吸汁，传播病毒 |
| 桃蚜 | 桃、烟草、蔬菜 | 多食性，传毒 |
| 棉蚜 | 棉花、瓜类 | 卷叶 |
| 豆蚜 | 豆类 | 嫩梢危害 |

**防治方法**:
1. **天敌利用** - 瓢虫、草蛉吃蚜虫
2. **黄板诱杀** - 利用蚜虫趋黄性
3. **药剂防治** - 吡虫啉、噻虫嗪等

---

#### 14. 稻飞虱 (Rice Planthopper/稻飞虱)

**白话解释**:  
会飞的小虫子，群集在水稻基部吸汁，严重时造成"虱烧"。

**学术定义**:  
半翅目飞虱科害虫，主要包括褐飞虱、白背飞虱、灰飞虱，是水稻重要害虫。

**三种飞虱对比**:
| 特征 | 褐飞虱 | 白背飞虱 | 灰飞虱 |
|------|--------|----------|--------|
| 体色 | 褐色 | 黄白相间 | 灰色 |
| 越冬 | 不能在北方越冬 | 迁飞 | 能越冬 |
| 主要危害 | 抽穗后 | 分蘖期 | 苗期 |
| 传毒 | × | × | 水稻条纹病毒 |

**危害特点**:
```
群集危害
├── 吸汁 → 叶片发黄
├── 产卵 → 伤口感染
├── 分泌蜜露 → 煤污病
└── 大发生 → "虱烧"（成片枯死）
```

**防治方法**:
1. **农业防治**:
   - 种植抗虫品种
   - 合理施肥
   
2. **化学防治**:
   - 防治适期：低龄若虫高峰期
   - 常用药剂：噻嗪酮、吡蚜酮

### 一、农药基础

#### 15. 农药 (Pesticide/农药)

**白话解释**:  
用来杀虫、杀菌、除草的"药"。

**生活例子**:  
- 杀虫剂打虫子
- 除草剂除草
- 杀菌剂治病

**学术定义**:  
用于预防、消灭或控制危害农业、林业的病、虫、草和其他有害生物，以及调节植物、昆虫生长的化学物质或生物制剂。

**农药分类**:
| 分类标准 | 类型 | 例子 |
|----------|------|------|
| 防治对象 | 杀虫剂 | 吡虫啉、氯虫苯甲酰胺 |
|          | 杀菌剂 | 三环唑、多菌灵 |
|          | 除草剂 | 草甘膦、莠去津 |
|          | 杀螨剂 | 阿维菌素 |
|          | 植物生长调节剂 | 赤霉素 |
| 来源 | 化学合成农药 | 有机磷、菊酯类 |
|      | 生物农药 | 苏云金杆菌、阿维菌素 |
| 作用方式 | 触杀剂 | 大多数杀虫剂 |
|          | 内吸剂 | 吡虫啉 |
|          | 胃毒剂 | 敌百虫 |

---

#### 16. 农药剂型 (Pesticide Formulation/农药剂型)

**白话解释**:  
农药加工成什么样子，方便使用。

**学术定义**:  
农药原药经加工处理后的形态，便于使用、提高效果。

**常见剂型**:
| 剂型 | 特点 | 使用方式 |
|------|------|----------|
| 乳油(EC) | 油状液体，需加水稀释 | 喷雾 |
| 可湿性粉剂(WP) | 粉末，加水成悬浮液 | 喷雾 |
| 悬浮剂(SC) | 稳定悬浮液 | 喷雾 |
| 水分散粒剂(WDG) | 颗粒，遇水分散 | 喷雾 |
| 颗粒剂(GR) | 颗粒状 | 撒施 |
| 粉剂(DP) | 细粉 | 喷粉 |
| 种子处理剂 | 处理种子用 | 拌种 |

**剂型选择原则**:
- 水基化剂型更环保（悬浮剂、水剂）
- 颗粒剂适合土壤处理
- 种子处理剂预防苗期病虫害

---

#### 17. 农药安全使用 (Safe Use of Pesticide/农药安全使用)

**白话解释**:  
农药有毒，要小心用，别把自己毒到，也别污染环境。

**学术定义**:  
按照农药标签和相关规定，科学合理使用农药，保障人畜安全和环境健康。

**农药毒性分级**:
| 毒性级别 | LD50（大鼠口服） | 标识颜色 |
|----------|------------------|----------|
| 剧毒 | <5mg/kg | 红色 |
| 高毒 | 5-50mg/kg | 红色 |
| 中等毒 | 50-500mg/kg | 黄色 |
| 低毒 | 500-5000mg/kg | 蓝色 |
| 微毒 | >5000mg/kg | 绿色 |

**安全间隔期 (PHI)**:
- 定义：最后一次施药到收获的天数
- 作用：保证农产品农药残留不超标

**安全使用要点**:
```
1. 防护措施
   ├── 穿防护服
   ├── 戴口罩、手套
   └── 避风施药

2. 科学用药
   ├── 对症下药
   ├── 适时用药
   ├── 适量用药
   └── 交替用药（延缓抗性）

3. 施药时间
   ├── 避开高温
   ├── 避开花期（保护蜜蜂）
   └── 3级风以下

4. 农药储存
   ├── 专库存放
   ├── 上锁保管
   └── 远离食品
```

#### 18. 农药抗药性 (Pesticide Resistance/抗药性)

**白话解释**:  
虫子打不死、病菌杀不掉，因为它们"进化"了。

**生活例子**:  
- 同样的药，以前有效现在无效
- 需要加大剂量才有效

**学术定义**:  
有害生物对农药敏感性降低的现象，由遗传变异和选择压力导致。

**抗药性产生原因**:
1. **长期单一用药** - 选择压力
2. **用药不当** - 剂量不足、次数过多
3. **遗传变异** - 有害生物适应

**延缓抗性的策略**:
```
1. 轮换用药
   └── 不同作用机理的药剂交替使用

2. 混合用药
   └── 两种药剂混用

3. 减少用药次数
   └── 综合防治

4. 选用高效药剂
   └── 一次解决，不留下抗性个体
```

### 二、主要农药类型

#### 19. 杀虫剂 (Insecticide/杀虫剂)

**白话解释**:  
专门杀虫子的药。

**学术定义**:  
用于防治害虫的农药，按作用方式分为触杀、胃毒、内吸、熏蒸等类型。

**主要杀虫剂类别**:
| 类别 | 代表药剂 | 作用机理 | 特点 |
|------|----------|----------|------|
| 有机磷 | 敌敌畏、毒死蜱 | 抑制乙酰胆碱酯酶 | 速效、毒性较高 |
| 拟除虫菊酯 | 氯氰菊酯 | 神经毒剂 | 速效、低毒 |
| 新烟碱类 | 吡虫啉、噻虫嗪 | 神经毒剂 | 内吸、对刺吸式害虫高效 |
| 酰胺类 | 氯虫苯甲酰胺 | 鱼尼丁受体 | 高效、低毒、持效期长 |
| 生物农药 | 苏云金杆菌、阿维菌素 | 多种 | 环保、对人畜安全 |

**选用原则**:
- 刺吸式害虫 → 内吸性杀虫剂（吡虫啉）
- 咀嚼式害虫 → 胃毒+触杀剂
- 钻蛀性害虫 → 内吸剂+颗粒剂

---

#### 20. 杀菌剂 (Fungicide/杀菌剂)

**白话解释**:  
专门治病的药。

**学术定义**:  
用于防治植物病害的农药，按作用方式分为保护性、治疗性、内吸性等。

**主要杀菌剂类别**:
| 类别 | 代表药剂 | 作用方式 | 特点 |
|------|----------|----------|------|
| 保护性 | 波尔多液、代森锰锌 | 预防 | 在发病前使用 |
| 治疗性 | 三环唑、多菌灵 | 治疗 | 可在发病后使用 |
| 内吸性 | 三唑类、甲氧基丙烯酸酯类 | 内吸 | 被植物吸收 |

**使用原则**:
```
保护性杀菌剂 ─── 发病前或发病初期使用
     ↓
治疗性杀菌剂 ─── 发病后使用
     ↓
内吸性杀菌剂 ─── 可被植物传导
```

---

#### 21. 除草剂 (Herbicide/除草剂)

**白话解释**:  
专门除草的药。

**学术定义**:  
用于防除杂草的农药，按作用方式分为选择性、灭生性等。

**除草剂分类**:
| 类型 | 特点 | 例子 | 适用 |
|------|------|------|------|
| 选择性 | 只杀杂草不伤作物 | 莠去津、苄嘧磺隆 | 农田 |
| 灭生性 | 杀死所有绿色植物 | 草甘膦、百草枯 | 免耕田、果园 |
| 触杀型 | 接触部位死亡 | 百草枯 | 地上杂草 |
| 内吸型 | 传导到全株 | 草甘膦 | 宿根杂草 |

**除草剂安全使用**:
1. **选择对路** - 作物对药剂敏感
2. **剂量准确** - 过量伤作物
3. **时期适宜** - 杂草2-4叶期最佳
4. **防止飘移** - 避免伤害邻近作物

#### 22. IPM (Integrated Pest Management/有害生物综合治理)

**白话解释**:  
不能光靠打药，要用多种方法一起对付病虫害。

**生活例子**:  
- 种抗病品种 + 清理病株 + 必要时打药
- 这就是综合治理

**学术定义**:  
综合考虑生态、经济、社会效益，协调运用多种防治措施，将有害生物控制在经济损害允许水平以下的管理策略。

**IPM原则**:
```
1. 生态原则
   └── 维护生态平衡

2. 经济原则
   └── 防治收益 > 防治成本

3. 综合原则
   └── 多种方法协调

4. 可持续原则
   └── 长期有效
```

**综合防治措施**:
| 防治方法 | 内容 | 特点 |
|----------|------|------|
| 植物检疫 | 防止有害生物传入 | 预防性 |
| 农业防治 | 栽培措施防病虫 | 基础性 |
| 物理防治 | 灯光、黄板、防虫网 | 无污染 |
| 生物防治 | 天敌、生物农药 | 环保 |
| 化学防治 | 农药防治 | 应急、高效 |

---

#### 23. 绿色防控 (Green Prevention and Control/绿色防控)

**白话解释**:  
用环保的方法防虫治病，少用或不用化学农药。

**学术定义**:  
以保护生态环境、保障农产品安全为目标，优先采用非化学防治措施，最大限度减少化学农药使用的防控策略。

**绿色防控技术**:
1. **生态调控**:
   - 合理轮作
   - 种植诱集植物
   - 建设生态廊道

2. **物理防治**:
   - 杀虫灯（诱杀成虫）
   - 黄板（诱杀蚜虫、飞虱）
   - 防虫网（阻隔害虫）
   - 性诱剂（诱杀害虫）

3. **生物防治**:
   - 释放天敌（赤眼蜂、瓢虫）
   - 使用生物农药（苏云金杆菌）
   - 保护自然天敌

4. **科学用药**:
   - 高效低毒农药
   - 精准施药
   - 减量用药

## 💡 实务应用场景

### 场景1：水稻病虫害防治
- **问题**：稻田有螟虫和稻飞虱，怎么防治？
- **植保解答**：
  1. 选用抗虫品种
  2. 处理稻桩，减少虫源
  3. 安装杀虫灯、性诱剂
  4. 监测虫情，适时用药
  5. 选用高效低毒药剂（氯虫苯甲酰胺、吡蚜酮）

### 场景2：蔬菜绿色防控
- **问题**：大棚蔬菜想减少农药使用，怎么办？
- **植保解答**：
  1. 悬挂黄板诱杀蚜虫、白粉虱
  2. 安装防虫网阻隔害虫
  3. 释放天敌（丽蚜小蜂防治白粉虱）
  4. 选用生物农药
  5. 必要时使用高效低毒农药
