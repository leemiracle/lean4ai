### 一、水环境因子

#### 1. 水温 (Water Temperature/水温)

**白话解释**:  
水的温度，直接决定鱼能不能活、长得快不快。

**生活例子**:  
- 热带鱼怕冷
- 冬天鱼不食不长

**学术定义**:  
水体温度，是影响鱼类代谢、生长和繁殖的关键环境因子。

**温度对鱼类的影响**:
```
水温影响
├── 代谢
│   ├── 适温范围内：温度↑，代谢↑
│   └── 超过最适温：代谢下降
├── 生长
│   ├── 最适温：生长最快
│   ├── 低温：生长慢
│   └── 高温：食欲下降
├── 繁殖
│   ├── 需要一定积温
│   └── 温度刺激产卵
└── 溶氧
    └── 温度↑，溶氧↓
```

**养殖鱼类适温范围**:
| 鱼类 | 生存温度(℃) | 最适生长温度(℃) |
|------|------------|---------------|
| 鲤鱼 | 0-35 | 20-28 |
| 草鱼 | 0-38 | 22-28 |
| 鲫鱼 | 0-35 | 17-25 |
| 罗非鱼 | 12-40 | 25-32 |
| 虹鳟 | 0-25 | 12-18 |
| 南美白对虾 | 15-38 | 25-32 |

**费曼记忆**:  
温水鱼喜20-30℃，冷水鱼喜15-20℃

#### 2. 溶解氧 (Dissolved Oxygen/溶解氧)

**白话解释**:  
水里有多少氧气，鱼"呼吸"靠它。

**生活例子**:  
- 鱼浮头（缺氧）
- 清晨鱼塘鱼游到水面

**学术定义**:  
溶解在水中的分子氧，是鱼类呼吸必需的物质。

**溶氧来源与消耗**:
```
溶氧来源
├── 浮游植物光合作用（主要）
│   └── 白天产氧
├── 空气中氧气溶入
│   └── 风力增氧
├── 补水带入
└── 机械增氧

溶氧消耗
├── 鱼类呼吸
├── 浮游生物呼吸
├── 细菌分解有机物
├── 底泥耗氧
└── 夜间植物呼吸
```

**溶氧变化规律**:
```
溶氧日变化
├── 清晨（最低）
│   └── 日出前：1-3mg/L
├── 白天（上升）
│   └── 光合作用产氧
├── 下午（最高）
│   └── 15-18时：10-15mg/L
└── 夜间（下降）
    └── 无光合作用
```

**溶氧标准**:
| 溶氧(mg/L) | 鱼类反应 |
|-----------|---------|
| >5 | 正常 |
| 3-5 | 轻度浮头 |
| 2-3 | 严重浮头 |
| <1 | 窒息死亡 |

**增氧措施**:
```
增氧方法
├── 机械增氧
│   ├── 叶轮式增氧机
│   ├── 水车式增氧机
│   └── 微孔曝气
├── 生物增氧
│   └── 培育浮游植物
├── 化学增氧
│   └── 增氧剂（过碳酸钠）
└── 管理措施
    ├── 合理密度
    ├── 及时换水
    └── 清除过多淤泥
```

#### 3. pH值 (pH Value/酸碱度)

**白话解释**:  
水的酸碱性，影响鱼的生存和生长。

**生活例子**:  
- 酸雨影响鱼塘
- 生石灰调节酸碱度

**学术定义**:  
水体中氢离子浓度的负对数，反映水体酸碱程度。

**pH对鱼类的影响**:
```
pH影响
├── 鱼类生存
│   ├── 最适：6.5-8.5
│   ├── 过酸(<5)：死亡
│   └── 过碱(>10)：死亡
├── 氨氮毒性
│   └── pH↑，非离子氨↑（毒性↑）
├── 浮游生物
│   └── 影响藻类生长
└── 药物效果
    └── 影响药物活性
```

**pH调节方法**:
```
pH调节
├── pH过低（偏酸）
│   ├── 施生石灰
│   └── 换水
└── pH过高（偏碱）
    ├── 换水
    └── 施有机肥
```

---

#### 4. 氨氮 (Ammonia Nitrogen/氨氮)

**白话解释**:  
鱼粪便、残饵分解产生的有毒物质。

**生活例子**:  
- 鱼塘氨氮过高引起中毒
- 换水降低氨氮

**学术定义**:  
水体中以游离氨(NH₃)和铵离子(NH₄⁺)形式存在的氮，是鱼类代谢和有机物分解的产物。

**氨氮来源**:
```
氨氮来源
├── 鱼类排泄
│   ├── 鳃排泄氨
│   └── 粪便
├── 残饵分解
├── 有机物分解
└── 底泥释放
```

**氨氮毒性**:
```
氨氮毒性
├── 非离子氨(NH₃)有毒
│   └── 毒性：NH₃ >> NH₄⁺
├── 影响因素
│   ├── pH↑ → NH₃%↑ → 毒性↑
│   └── 温度↑ → NH₃%↑ → 毒性↑
└── 中毒症状
    ├── 鱼体色变深
    ├── 呼吸急促
    └── 死亡
```

**氨氮控制标准**:
| 指标 | 安全浓度(mg/L) |
|------|---------------|
| 总氨氮 | <0.5 |
| 非离子氨 | <0.02 |

**氨氮控制措施**:
```
氨氮控制
├── 合理投喂
│   └── 减少残饵
├── 增加换水
├── 生物处理
│   ├── 硝化细菌
│   └── 水生植物吸收
├── 吸附处理
│   └── 沸石粉
└── 控制密度
```

### 一、养殖鱼类分类

#### 5. 养殖鱼类分类 (Classification of Cultured Fish/养殖鱼类分类)

**白话解释**:  
按不同方式把养殖鱼类分类。

**生活例子**:  
- 淡水鱼vs海水鱼
- 草食性鱼vs肉食性鱼

**学术定义**:  
根据鱼类的生活习性、食性、栖息水层等特点进行的分类。

**按栖息环境分类**:
```
按栖息环境
├── 淡水鱼类
│   ├── 鲤、鲫、草鱼、鲢、鳙
│   └── 占养殖主体
├── 海水鱼类
│   ├── 大黄鱼、鲈鱼、石斑鱼
│   └── 网箱养殖
└── 咸淡水鱼类
    └── 罗非鱼、鲻鱼
```

**按食性分类**:
```
按食性分类
├── 草食性
│   ├── 草鱼、团头鲂
│   └── 以水草为食
├── 滤食性
│   ├── 鲢（滤食浮游植物）
│   ├── 鳙（滤食浮游动物）
│   └── 鳃耙过滤
├── 杂食性
│   ├── 鲤、鲫
│   └── 动植物都吃
└── 肉食性
    ├── 鳜鱼、鲈鱼、鲶鱼
    └── 以小鱼虾为食
```

**按栖息水层分类**:
```
按栖息水层
├── 上层鱼
│   └── 鲢（滤食浮游植物）
├── 中层鱼
│   └── 草鱼
└── 下层/底层鱼
    ├── 鲤、鲫
    └── 鳙（滤食浮游动物）
```

### 二、主要养殖鱼类

#### 6. "四大家鱼" (Four Major Carps/四大家鱼)

**白话解释**:  
中国传统养殖的主要鱼类：青、草、鲢、鳙。

**生活例子**:  
- 草鱼吃草
- 鲢鱼"白鲢"

**学术定义**:  
青鱼、草鱼、鲢鱼、鳙鱼四种主要淡水养殖鱼类的统称。

**四大家鱼特点**:
```
四大家鱼
├── 青鱼
│   ├── 食性：肉食性（螺蛳、蚌）
│   ├── 栖息：下层
│   └── 特点：生长快、体型大
├── 草鱼
│   ├── 食性：草食性
│   ├── 栖息：中层
│   └── 特点：食量大、生长快
├── 鲢鱼(白鲢)
│   ├── 食性：滤食浮游植物
│   ├── 栖息：上层
│   └── 特点：生长快、易养殖
└── 鳙鱼(花鲢)
    ├── 食性：滤食浮游动物
    ├── 栖息：中上层
    └── 特点：头大、味美
```

**多品种混养**:
```
混养原理
├── 食性互补
│   ├── 草鱼食草 + 粪便肥水
│   ├── 鲢鱼滤食浮游植物
│   └── 鳙鱼滤食浮游动物
├── 栖息水层互补
│   └── 上、中、下层利用
└── 配比
    ├── 草鱼为主：40-50%
    ├── 鲢鳙：30-40%
    └── 鲤鲫：10-20%
```

### 三、繁殖生物学

#### 8. 鱼类繁殖 (Fish Reproduction/鱼类繁殖)

**白话解释**:  
鱼"生孩子"，自然产卵或人工催产。

**生活例子**:  
- 春季鱼类自然繁殖
- 人工催产技术

**学术定义**:  
鱼类产生后代的过程，包括性腺发育、成熟、产卵和受精。

**鱼类性腺发育分期**:
```
性腺发育分期（6期）
├── 第I期：性腺原始期
├── 第II期：性腺发育期
├── 第III期：性腺生长Ⅰ期
├── 第IV期：性腺生长Ⅱ期（成熟前期）
├── 第V期：性腺成熟期
└── 第VI期：产卵后恢复期
```

**产卵类型**:
```
产卵类型
├── 一次性产卵
│   ├── 鲤、鲫
│   └── 一年产卵一次
└── 多次产卵
    ├── 罗非鱼
    └── 一年多次产卵
```

**繁殖季节**:
| 鱼类 | 繁殖季节 | 水温(℃) |
|------|---------|---------|
| 鲤鲫 | 4-6月 | 18-25 |
| 草鱼 | 5-6月 | 20-26 |
| 鲢鳙 | 5-6月 | 20-28 |
| 罗非鱼 | 5-10月 | 22-32 |

#### 9. 人工催产 (Artificial Propagation/人工催产)

**白话解释**:  
给鱼打针"催生"，让鱼产卵。

**生活例子**:  
- 注射激素
- 人工授精

**学术定义**:  
通过注射激素，促进鱼类性腺成熟和产卵的技术。

**催产剂种类**:
```
催产剂类型
├── PG（脑垂体）
│   └── 天然激素来源
├── HCG（绒毛膜促性腺激素）
│   └── 促进排卵
└── LRH-A（促黄体素释放激素类似物）
    └── 促进性腺发育
```

**催产程序**:
```
人工催产流程
├── 选择亲鱼
│   ├── 性腺发育良好（第IV期末）
│   ├── 体质健壮
│   └── 无病无伤
├── 注射催产剂
│   ├── 一次注射或两次注射
│   ├── 胸鳍基部注射
│   └── 剂量依种类、体重定
├── 效应时间
│   └── 注射后到产卵的时间
│       ├── 一次注射：10-16h
│       └── 两次注射：6-10h
├── 自然产卵
│   └── 亲鱼自行产卵、受精
└── 人工授精
    ├── 干法授精
    ├── 半干法授精
    └── 湿法授精
```

**人工授精操作**:
```
干法授精步骤
├── 1. 捕起成熟亲鱼
├── 2. 擦干鱼体
├── 3. 挤卵入干盆
├── 4. 挤精液于卵上
├── 5. 搅拌均匀（1-2分钟）
├── 6. 加水搅拌
├── 7. 清洗多余精液
└── 8. 移入孵化器
```

#### 10. 孵化技术 (Incubation Technology/孵化技术)

**白话解释**:  
鱼卵孵化成小鱼苗的技术。

**生活例子**:  
- 孵化环道
- 流水孵化

在适宜条件下，使受精卵发育成鱼苗的技术过程。

**孵化方式**:
```
孵化方式
├── 静水孵化
│   ├── 网箱孵化
│   └── 水泥池孵化
├── 流水孵化（最常用）
│   ├── 孵化环道
│   ├── 孵化桶
│   └── 孵化槽

```

**孵化条件**:
```
孵化环境条件

│   ├── 过低：孵化时间长
│   └── 过高：畸形多
├── 溶氧
│   └── >5mg/L
├── 水质
│   ├── 清洁
│   └── 无污染
└── 流速
    ├── 适中
    └── 不冲走鱼卵
```

**孵化时间**:
| 鱼类 | 孵化水温(℃) | 孵化时间(h) |
|------|------------|------------|

| 草鱼 | 22-24 | 30-32 |
| 鲢鱼 | 22-24 | 28-30 |

#### 12. 鱼种放养 (Fish Stocking/鱼种放养)

**白话解释**:  
把鱼苗放进池塘开始养殖。

**生活例子**:  
- 春季放养
- 合理密度

**学术定义**:  
将鱼种按一定密度和配比放入养殖水体的过程。

**放养时间**:
```
放养时间
├── 春放（最佳）
│   └── 水温稳定在10℃以上
├── 秋放
│   └── 10-11月
└── 冬放
    └── 南方地区
```

**放养密度**:
```
放养密度确定因素
├── 池塘条件
│   ├── 水源好 → 密度高
│   └── 设备好 → 密度高
├── 鱼种规格
│   └── 规格大 → 密度低
├── 出塘规格
│   └── 出塘大 → 密度低
├── 技术水平
│   └── 技术高 → 密度高
└── 经济效益
    └── 综合考量
```

**放养模式示例**（亩产1000kg）:
```
高产模式
├── 草鱼（250g/尾）200尾
├── 鲢鱼（100g/尾）150尾
├── 鳙鱼（100g/尾）50尾
├── 鲤鱼（50g/尾）100尾
└── 鲫鱼（25g/尾）200尾
```

#### 13. 投喂技术 (Feeding Technology/投喂技术)

**白话解释**:  
怎么给鱼喂食。

**生活例子**:  
- 定时定量
- 投喂配合饲料

**学术定义**:  
科学投喂饲料，提高饲料利用率和养殖效益的技术。

**投喂原则**:
```
投喂"四定"原则
├── 定时
│   └── 每天2-4次，固定时间
├── 定位
│   └── 固定食场
├── 定质
│   └── 饲料新鲜、营养全面
└── 定量
    └── 日投喂量为鱼体重2-5%
```

**投喂量确定**:
```
投喂量 = 鱼体重 × 日投喂率

日投喂率
├── 水温20℃以下：1-2%
├── 水温20-25℃：2-3%
├── 水温25-30℃：3-5%
└── 水温30℃以上：2-3%
```

**饲料种类**:
```
饲料类型
├── 天然饲料
│   ├── 芜萍、浮萍（草鱼）
│   ├── 螺蛳、蚌（青鱼）
│   └── 浮游生物（鲢鳙）
├── 人工饲料
│   ├── 粉料
│   ├── 颗粒料
│   └── 膨化料
└── 配合饲料
    ├── 全价配合饲料
    └── 营养全面
```

### 五、常见鱼病

#### 14. 细菌性鱼病 (Bacterial Fish Diseases/细菌性鱼病)

**白话解释**:  
细菌引起的鱼病。

**生活例子**:  
- 烂鳃病
- 肠炎病

**学术定义**:  
由细菌感染引起的鱼类疾病。

**主要细菌病**:
```
细菌性鱼病
├── 细菌性烂鳃病
│   ├── 病原：柱状黄杆菌
│   ├── 症状：鳃丝腐烂、颜色异常
│   └── 防治：漂白粉、二氧化氯
├── 细菌性肠炎病
│   ├── 病原：点状气单胞菌
│   ├── 症状：肛门红肿、肠壁充血
│   └── 防治：大蒜素、土霉素
└── 赤皮病
    ├── 病原：荧光假单胞菌
    ├── 症状：体表出血、鳞片脱落
    └── 防治：漂白粉、强氯精
```

---

#### 15. 寄生虫性鱼病 (Parasitic Fish Diseases/寄生虫性鱼病)

**白话解释**:  
虫子寄生在鱼体上引起的病。

**生活例子**:  
- 车轮虫病
- 指环虫病

**学术定义**:  
由寄生虫寄生引起的鱼类疾病。

**主要寄生虫病**:
```
寄生虫性鱼病
├── 车轮虫病
│   ├── 病原：车轮虫
│   ├── 部位：体表、鳃
│   ├── 症状：粘液增多、呼吸困难
│   └── 防治：硫酸铜+硫酸亚铁
├── 指环虫病
│   ├── 病原：指环虫
│   ├── 部位：鳃
│   ├── 症状：鳃丝粘液多、贫血
│   └── 防治：晶体敌百虫
├── 小瓜虫病（白点病）
│   ├── 病原：多子小瓜虫
│   ├── 症状：体表白点
│   └── 防治：辣椒粉+生姜
└── 锚头蚤病
    ├── 病原：锚头蚤
    ├── 症状：虫体寄生体表
    └── 防治：晶体敌百虫
```

## 🎯 重点概念速记

### 必背概念（16个）

1. **水温** - 影响鱼类代谢、生长
2. **溶解氧** - 鱼类呼吸必需，>5mg/L
3. **浮头** - 缺氧时鱼类游向水面
4. **pH值** - 最适6.5-8.5
5. **氨氮** - 有毒物质，非离子氨毒性大
6. **四大家鱼** - 青、草、鲢、鳙
7. **混养原理** - 食性、水层互补
8. **滤食性鱼类** - 鲢鳙，过滤浮游生物
9. **人工催产** - 注射激素促进产卵
10. **催产剂** - PG、HCG、LRH-A
11. **人工授精** - 干法、半干法、湿法
12. **孵化条件** - 水温22-26℃，溶氧>5mg/L
13. **投喂"四定"** - 定时、定位、定质、定量
14. **日投喂率** - 占鱼体重2-5%
15. **细菌性烂鳃病** - 柱状黄杆菌引起
16. **综合防治** - 预防为主，多种措施结合

## 💡 实务应用场景

### 场景1：池塘养鱼高产技术
- **问题**：如何实现池塘养鱼高产？
- **水产养殖解答**：
  1. 池塘条件：5-10亩，水深2m以上
  2. 彻底清塘消毒
  3. 多品种混养：草鱼为主+鲢鳙+鲤鲫
  4. 合理密度：总密度1000-1500尾/亩
  5. 科学投喂：四定原则，配合饲料
  6. 水质调控：定期换水，增氧
  7. 鱼病预防：定期消毒，及时发现治疗

### 场景2：人工繁殖草鱼
- **问题**：如何进行草鱼人工繁殖？
- **水产养殖解答**：
  1. 选择成熟亲鱼：5龄以上、体重5kg以上
  2. 注射催产剂：LRH-A + DOM
  3. 效应时间：10-14小时
  4. 人工授精：干法授精
  5. 孵化：流水孵化环道
  6. 孵化时间：30-32小时（22-24℃）
  7. 鱼苗培育：豆浆培育
