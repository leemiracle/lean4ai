## 📖 第一章：运筹学与优化

---

### 一、线性规划

#### 1. 线性规划 (Linear Programming/线性规划)

**白话解释**:  
在有限资源下，找到最优分配方案。

**生活例子**:  
- 工厂生产两种产品，机器时间有限，如何安排产量利润最大？
- 投资组合：资金有限，如何分配收益最高？

**数学形式**:
```
目标函数: max Z = c₁x₁ + c₂x₂
约束条件: a₁₁x₁ + a₁₂x₂ ≤ b₁
         a₂₁x₁ + a₂₂x₂ ≤ b₂
         x₁, x₂ ≥ 0
```

**关键要素**:
1. **决策变量** - 需要决定的量
2. **目标函数** - 要优化的目标
3. **约束条件** - 限制条件
4. **非负约束** - 变量不能为负

**费曼记忆**:  
线性规划 = 目标 + 约束 + 最优化

---

#### 2. 单纯形法 (Simplex Method/单纯形法)

**白话解释**:  
一种逐步找到最优解的计算方法。

**核心思想**:
- 从可行域的一个顶点出发
- 沿着边移动到更优的顶点
- 直到找到最优解

**步骤**:
1. 将问题转化为标准形式
2. 确定初始基可行解
3. 进行迭代计算
4. 判断是否达到最优

**应用场景**:
- 生产计划优化
- 运输问题
- 人力资源配置

---

#### 3. 对偶理论 (Duality Theory/对偶理论)

**白话解释**:  
每个线性规划问题都有对应的"影子问题"。

**原问题 vs 对偶问题**:
| 原问题 | 对偶问题 |
|--------|----------|
| 最大化 | 最小化 |
| n个变量 | n个约束 |
| m个约束 | m个变量 |

**经济学意义**:
- 对偶变量 = 影子价格
- 反映资源的边际价值

**对偶定理**:
- 弱对偶定理
- 强对偶定理
- 互补松弛定理

---

#### 4. 敏感性分析 (Sensitivity Analysis/敏感性分析)

**白话解释**:  
参数变化时，最优解会不会变？

**分析内容**:
1. **目标函数系数变化**
2. **约束右边值变化**
3. **添加新变量**
4. **添加新约束**

**应用价值**:
- 了解解的稳定性
- 识别关键参数
- 指导决策调整

---

### 二、整数规划

#### 5. 整数规划 (Integer Programming/整数规划)

**白话解释**:  
变量必须是整数的优化问题。

**生活例子**:
- 生产多少台机器？（不能生产半台）
- 选哪些项目投资？（投或不投）

**分类**:
1. **纯整数规划** - 所有变量为整数
2. **混合整数规划** - 部分变量为整数
3. **0-1规划** - 变量只能取0或1

**求解方法**:
- 分支定界法
- 割平面法
- 启发式算法

---

#### 6. 分支定界法 (Branch and Bound/分支定界法)

**白话解释**:  
通过分支和剪枝，逐步缩小搜索范围。

**核心步骤**:
1. **松弛** - 先解连续问题
2. **分支** - 将问题分解为子问题
3. **定界** - 确定上下界
4. **剪枝** - 删除不可能最优的分支

**费曼记忆**:  
分支定界 = 分解 + 估计 + 排除

---

### 三、图论与网络

#### 7. 最短路径 (Shortest Path/最短路径)

**白话解释**:  
从A到B，走哪条路最近？

**应用场景**:
- 导航路线规划
- 物流配送路径
- 网络数据传输

**经典算法**:
1. **Dijkstra算法** - 正权图
2. **Bellman-Ford算法** - 可负权
3. **Floyd算法** - 全点对最短路径

---

#### 8. 最大流 (Maximum Flow/最大流)

**白话解释**:  
网络中最多能通过多少流量？

**关键概念**:
1. **容量** - 边的最大流量
2. **流量** - 实际通过量
3. **残留容量** - 剩余可用的容量
4. **增广路径** - 能增加流量的路径

**Ford-Fulkerson算法**:
1. 找增广路径
2. 增加流量
3. 更新残留网络
4. 重复直到没有增广路径

**应用**:
- 交通流量优化
- 通信网络设计
- 供应链管理

---

#### 9. 最小生成树 (Minimum Spanning Tree/最小生成树)

**白话解释**:  
用最少的边连接所有点，总长度最短。

**经典算法**:
1. **Kruskal算法** - 按边排序，贪心选择
2. **Prim算法** - 从一点扩展

**应用场景**:
- 网络布线设计
- 管道铺设
- 电路设计

## 📖 第二章：预测与决策

---

### 一、预测方法

#### 10. 时间序列分析 (Time Series Analysis/时间序列分析)

**白话解释**:  
根据历史数据预测未来。

**基本组成**:
1. **趋势** - 长期上升或下降
2. **季节性** - 周期性波动
3. **循环** - 非固定周期波动
4. **随机** - 不规则变动

**常用方法**:
- 移动平均法
- 指数平滑法
- ARIMA模型
- 季节性分解

---

#### 11. 回归分析 (Regression Analysis/回归分析)

**白话解释**:  
找到变量之间的关系，用来自变量预测因变量。

**类型**:
1. **简单线性回归** - 一个自变量
2. **多元线性回归** - 多个自变量
3. **非线性回归** - 曲线关系
4. **逻辑回归** - 分类问题

**模型评估**:
- R² (决定系数)
- 均方误差 (MSE)
- F检验
- t检验

---

### 二、决策分析

#### 12. 决策树 (Decision Tree/决策树)

**白话解释**:  
把决策过程画成树形结构，选择最优路径。

**组成要素**:
1. **决策节点** - 方形，需要选择
2. **机会节点** - 圆形，不确定结果
3. **结果节点** - 终点，收益值
4. **分支** - 不同选择或结果

**决策准则**:
- 期望值最大化
- 效用最大化
- 风险规避

---

#### 13. 博弈论 (Game Theory/博弈论)

**白话解释**:  
你的决策影响别人，别人的决策也影响你。

**基本概念**:
1. **博弈方** - 参与者
2. **策略** - 可选择的行动
3. **支付** - 收益或损失
4. **信息** - 对局势的了解程度

**经典案例**:
- **囚徒困境** - 个人理性导致集体非理性
- **智猪博弈** - 大猪小猪的责任分配
- **性别战** - 协调问题

**纳什均衡**:
每个博弈方的策略都是对其他方策略的最佳反应。

---

#### 14. 贝叶斯决策 (Bayesian Decision/贝叶斯决策)

**白话解释**:  
利用新信息更新对不确定性的判断。

**贝叶斯公式**:
```
P(A|B) = P(B|A) × P(A) / P(B)
```

**应用步骤**:
1. 确定先验概率
2. 收集新信息
3. 计算后验概率
4. 基于后验概率决策

## 📖 第三章：项目管理与质量控制

---

### 一、项目管理

#### 15. 关键路径法 (Critical Path Method/CPM)

**白话解释**:  
找出项目中最耗时的任务链，决定项目最短完成时间。

**核心概念**:
1. **最早开始时间 (ES)**
2. **最早完成时间 (EF)**
3. **最晚开始时间 (LS)**
4. **最晚完成时间 (LF)**
5. **总时差** = LS - ES

**关键路径**: 时差为零的任务链

---

#### 16. PERT技术 (Program Evaluation Review Technique)

**白话解释**:  
考虑任务时间的不确定性，估算项目完成概率。

**三点估计**:
- **乐观时间 (a)** - 一切顺利
- **最可能时间 (m)** - 正常情况
- **悲观时间 (b)** - 最坏情况

**期望时间**:
```
t = (a + 4m + b) / 6
```

---

### 二、质量控制

#### 17. 六西格玛 (Six Sigma/六西格玛)

**白话解释**:  
追求极低缺陷率的质量管理方法。

**DMAIC流程**:
1. **Define** - 定义问题
2. **Measure** - 测量现状
3. **Analyze** - 分析原因
4. **Improve** - 改进方案
5. **Control** - 控制维持

**质量水平**:
- 3σ → 6.68%缺陷率
- 6σ → 0.00034%缺陷率

---

#### 18. 统计过程控制 (Statistical Process Control/SPC)

**白话解释**:  
用统计方法监控生产过程，及时发现问题。

**核心工具**:
1. **控制图** - 监控过程稳定性
2. **过程能力指数 (Cp, Cpk)**
3. **因果图** - 分析问题原因
4. **直方图** - 数据分布

**判异准则**:
- 点出界
- 连续7点在中心线一侧
- 连续7点上升或下降

## 📖 第四章：库存与供应链

---

#### 19. EOQ模型 (Economic Order Quantity/经济订货批量)

**白话解释**:  
订多少货，总成本最低？

**基本假设**:
- 需求恒定
- 瞬间补货
- 不允许缺货

**公式**:
```
Q* = √(2DS/H)
```
- D = 年需求量
- S = 每次订货成本
- H = 单位年存储成本

---

#### 20. 牛鞭效应 (Bullwhip Effect/牛鞭效应)

**白话解释**:  
需求波动从下游向上游逐级放大。

**原因**:
1. 需求预测更新
2. 批量订货
3. 价格波动
4. 短缺博弈

**解决措施**:
- 信息共享
- 减少批量
- 稳定价格
- VMI供应商管理库存

---

## 📖 第五章：排队论与模拟

---

#### 21. 排队论 (Queuing Theory/排队论)

**白话解释**:  
研究等待时间和服务效率。

**基本参数**:
1. **λ** - 到达率
2. **μ** - 服务率
3. **ρ** - 服务强度 = λ/μ
4. **L** - 系统中平均人数
5. **W** - 系统中平均等待时间

**Little公式**:
```
L = λ × W
```

---

#### 22. 蒙特卡洛模拟 (Monte Carlo Simulation/蒙特卡洛模拟)

**白话解释**:  
用随机数模拟复杂系统，估计结果。

**基本步骤**:
1. 建立概率模型
2. 生成随机数
3. 进行大量模拟
4. 统计分析结果

**应用场景**:
- 风险评估
- 金融定价
- 项目估算
