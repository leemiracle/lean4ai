## 第一部分：公共管理基础

---

### 公共管理

**学科归属**：公共管理 > 公共管理基础  
**英文名称**：Public Management  
**关联概念**：[公共利益] [公共部门] [治理]

#### 核心定义（一句话）
公共部门为实现公共利益而进行的管理活动。

#### 费曼式解释（说人话）

**什么是公共管理？**

简单说：**政府和非营利组织怎么管理**。

**公共管理 vs 私人管理**：

| 公共管理 | 私人管理 |
|---------|---------|
| 目标：公共利益 | 目标：利润最大化 |
| 资金：税收、拨款 | 资金：投资、收入 |
| 决策：多元利益相关者 | 决策：股东主导 |
| 问责：公众、媒体、立法机构 | 问责：股东、董事会 |
| 效率要求：公平与效率平衡 | 效率要求：利润优先 |

**公共管理的范围**：
- 政府部门（中央、地方）
- 事业单位（学校、医院）
- 非营利组织（NGO）

**核心问题**：
- 如何提供公共产品和服务？
- 如何平衡效率和公平？
- 如何提高政府效率？

---

### 新公共管理

**学科归属**：公共管理 > 公共管理理论  
**英文名称**：New Public Management (NPM)  
**关联概念**：[效率] [市场机制] [私有化]

#### 核心定义（一句话）
20世纪80年代兴起的，将私营部门管理方法引入公共部门的改革运动。

#### 费曼式解释（说人话）

**什么是新公共管理？**

**核心思想**：**用企业管理的方式管政府**。

**主要特点**：

| 特点 | 内容 |
|------|------|
| 关注产出 | 不只是投入，更看结果 |
| 引入竞争 | 打破垄断，提高效率 |
| 绩效管理 | 设定目标，考核结果 |
| 分权化 | 给一线更多自主权 |
| 顾客导向 | 把公民当"顾客" |
| 私有化 | 能外包的尽量外包 |

**英国撒切尔改革**：
- 国企私有化
- 引入竞争机制
- 绩效考核

**批评**：
- 过度强调效率，忽视公平
- "顾客"不等于"公民"
- 可能削弱公共性

---

### 新公共服务

**学科归属**：公共管理 > 公共管理理论  
**提出者**：珍妮特·登哈特、罗伯特·登哈特  
**英文名称**：New Public Service (NPS)  
**关联概念**：[服务] [民主] [公民权]

#### 核心定义（一句话）
强调政府的服务角色和民主价值的公共管理理论。

#### 费曼式解释（说人话）

**新公共服务 vs 新公共管理**：

| 新公共管理 | 新公共服务 |
|-----------|-----------|
| 政府掌舵 | 政府服务 |
| 公民是顾客 | 公民是公民 |
| 效率优先 | 民主、公平、效率平衡 |
| 市场机制 | 协商、参与 |

**七条原则**：
1. 服务而非掌舵
2. 公共利益是目标
3. 战略地思考，民主地行动
4. 服务于公民而非顾客
5. 责任并不简单
6. 重视人而不只是生产率
7. 公民权和公共服务比企业家精神更重要

---

### 治理理论

**学科归属**：公共管理 > 治理理论  
**英文名称**：Governance Theory  
**关联概念**：[网络治理] [多中心治理] [协同治理]

#### 核心定义（一句话）
政府、市场、社会共同参与公共事务管理的模式和过程。

#### 费曼式解释（说人话）

**治理 vs 统治**：

| 统治 | 治理 |
|------|------|
| 政府单中心 | 多元主体 |
| 命令控制 | 协商合作 |
| 自上而下 | 网络化 |
| 政府主导 | 共同参与 |

**治理的主体**：
1. **政府**：规则制定、服务提供
2. **市场**：资源配置、效率驱动
3. **社会**：公众参与、监督

**治理的特点**：
- 多元主体
- 协商互动
- 网络结构
- 信任机制

**中国的实践**：
- 多元共治
- 基层治理创新
- 社会组织参与

## 第二部分：行政管理

---

### 行政管理

**学科归属**：公共管理 > 行政管理  
**英文名称**：Public Administration  
**关联概念**：[行政职能] [行政组织] [行政改革]

#### 核心定义（一句话）
国家行政机关依法管理国家和社会事务的活动。

#### 费曼式解释（说人话）

**什么是行政管理？**

简单说：**政府怎么运作**。

**行政管理的职能**：

| 职能 | 内容 |
|------|------|
| 计划 | 制定政策、规划 |
| 组织 | 建立机构、配置人员 |
| 人事 | 公务员管理 |
| 财务 | 预算、财政 |
| 协调 | 部门间协调 |
| 监督 | 监察、审计 |

**行政组织**：
- 中央政府
- 地方政府
- 政府部门
- 事业单位

**行政改革的目标**：
- 提高效率
- 增强服务
- 减少官僚主义
- 适应社会发展

---

### 行政改革

**学科归属**：公共管理 > 行政管理  
**英文名称**：Administrative Reform  
**关联概念**：[机构改革] [简政放权] [放管服]

#### 核心定义（一句话）
对行政体制、机构、职能、流程进行的系统性变革。

#### 费曼式解释（说人话）

**为什么要行政改革？**
- 机构臃肿
- 效率低下
- 职能交叉
- 不适应社会发展

**改革的方向**：

| 方向 | 内容 |
|------|------|
| 简政放权 | 减少审批、下放权力 |
| 机构整合 | 大部制改革 |
| 流程优化 | 一站式服务 |
| 数字化 | 互联网+政务服务 |
| 绩效管理 | 目标考核 |

**中国"放管服"改革**：
- **放**：简政放权
- **管**：放管结合
- **服**：优化服务

**改革的原则**：
- 问题导向
- 系统设计
- 逐步推进
- 法治保障

---

### 公务员制度

**学科归属**：公共管理 > 行政管理  
**英文名称**：Civil Service System  
**关联概念**：[考录] [职级] [绩效考核]

#### 核心定义（一句话）
对国家公职人员的管理制度体系。

#### 费曼式解释（说人话）

**什么是公务员？**
- 依法履行公职
- 纳入国家行政编制
- 由国家财政负担工资福利

**公务员制度的特点**：

| 特点 | 内容 |
|------|------|
| 公开考试 | 凡进必考 |
| 职务常任 | 不随政治变化 |
| 政治中立 | 依法行政 |
| 职级并行 | 职务与职级分离 |

**公务员管理环节**：
1. **考录**：公开招考、择优录取
2. **培训**：初任培训、在职培训
3. **考核**：年度考核、专项考核
4. **晋升**：竞争上岗、择优晋升
5. **监督**：纪律约束、问责机制
6. **退出**：退休、辞职、辞退

**公务员的义务**：
- 忠于宪法
- 依法履职
- 为人民服务
- 接受监督

## 第三部分：公共政策

---

### 公共政策

**学科归属**：公共管理 > 公共政策  
**英文名称**：Public Policy  
**关联概念**：[政策过程] [政策分析] [政策评估]

#### 核心定义（一句话）
政府为解决公共问题而制定的行动准则和方案。

#### 费曼式解释（说人话）

**什么是公共政策？**

简单说：**政府决定怎么解决社会问题**。

**公共政策的类型**：

| 类型 | 特点 | 例子 |
|------|------|------|
| 分配性政策 | 给予利益 | 补贴政策 |
| 再分配政策 | 利益调整 | 税收政策 |
| 管制性政策 | 限制行为 | 环保法规 |
| 自我管制政策 | 行业自律 | 行业标准 |

**政策的层次**：
1. 元政策：制定政策的政策
2. 总政策：基本路线、方针
3. 基本政策：具体领域的政策
4. 具体政策：操作性政策

---

### 政策过程

**学科归属**：公共管理 > 公共政策  
**英文名称**：Policy Process  
**关联概念**：[议程设置] [政策执行] [政策评估]

#### 核心定义（一句话）
公共政策从产生到终结的全过程。

#### 费曼式解释（说人话）

**政策过程的阶段**：

```
问题识别 → 议程设置 → 政策制定 → 政策执行 → 政策评估 → 政策终结
    ↑                                                        ↓
    ←←←←←←←←←←←← 反馈与调整 ←←←←←←←←←←←←←←←←←←←←←
```

**各阶段详解**：

| 阶段 | 问题 | 关键点 |
|------|------|--------|
| 问题识别 | 什么问题需要解决？ | 问题界定 |
| 议程设置 | 这个问题值得政府关注吗？ | 进入决策视野 |
| 政策制定 | 怎么解决？ | 方案设计、选择 |
| 政策执行 | 怎么落地？ | 组织实施 |
| 政策评估 | 效果如何？ | 效果评价 |
| 政策终结 | 还继续吗？ | 调整或终止 |

---

### 政策工具

**学科归属**：公共管理 > 公共政策  
**英文名称**：Policy Instruments  
**关联概念**：[管制] [激励] [信息]

#### 核心定义（一句话）
政府实施政策所采用的各种手段和方法。

#### 费曼式解释（说人话）

**政策工具的类型**：

| 类型 | 机制 | 例子 |
|------|------|------|
| 管制工具 | 强制 | 法律、法规、标准 |
| 经济工具 | 激励 | 税收、补贴、罚款 |
| 信息工具 | 说服 | 宣传、教育、信息公开 |
| 组织工具 | 直接提供 | 公共服务、国有企业 |
| 自愿工具 | 自主行动 | 自愿协议、认证 |

**选择政策工具考虑的因素**：
1. 政策目标
2. 资源约束
3. 政治可行性
4. 实施能力
5. 预期效果

**组合使用**：
- 单一工具往往不够
- 需要组合搭配
- 如：环保政策 = 法规 + 排污费 + 宣传

---

### 政策评估

**学科归属**：公共管理 > 公共政策  
**英文名称**：Policy Evaluation  
**关联概念**：[效果评估] [效率评估] [影响评估]

#### 核心定义（一句话）
系统评价政策效果和影响的活动。

#### 费曼式解释（说人话）

**评估什么？**

| 评估类型 | 问题 |
|----------|------|
| 效果评估 | 达到目标了吗？ |
| 效率评估 | 成本合理吗？ |
| 公平评估 | 公平吗？ |
| 影响评估 | 有什么副作用？ |
| 过程评估 | 实施过程怎样？ |

**评估的方法**：

| 方法 | 内容 |
|------|------|
| 前后对比 | 政策前后变化 |
| 有无对比 | 政策区与非政策区对比 |
| 成本效益分析 | 投入产出比 |
| 专家评估 | 专家打分 |
| 公众满意度调查 | 问卷、访谈 |

**评估的困难**：
- 因果关系难以确定
- 长期效果难以测量
- 多种因素影响
- 数据获取困难

## 第四部分：社会保障

---

### 社会保障

**学科归属**：公共管理 > 社会保障  
**英文名称**：Social Security  
**关联概念**：[社会保险] [社会救助] [社会福利]

#### 核心定义（一句话）
国家和社会通过立法对公民提供基本生活保障的制度体系。

#### 费曼式解释（说人话）

**什么是社会保障？**

简单说：**让老百姓有基本保障，不怕生老病死**。

**社会保障的三大支柱**：

| 支柱 | 内容 | 特点 |
|------|------|------|
| 社会保险 | 养老、医疗、失业等 | 强制参保、权利义务对应 |
| 社会救助 | 低保、特困救助 | 免费、托底 |
| 社会福利 | 老年人、残疾人、儿童福利 | 面向特殊群体 |

**中国的社会保障体系**：

```
社会保障
├── 社会保险
│   ├── 养老保险
│   ├── 医疗保险
│   ├── 失业保险
│   ├── 工伤保险
│   └── 生育保险
├── 社会救助
│   ├── 最低生活保障
│   ├── 特困人员供养
│   └── 临时救助
└── 社会福利
    ├── 老年人福利
    ├── 残疾人福利
    └── 儿童福利
```

---

### 社会保险

**学科归属**：公共管理 > 社会保障  
**英文名称**：Social Insurance  
**关联概念**：[养老保险] [医疗保险] [缴费]

#### 核心定义（一句话）
通过缴费形成的保障基金，对参保人提供风险保障的制度。

#### 费曼式解释（说人话）

**中国的五险**：

| 险种 | 保障什么 | 缴费比例（大致） |
|------|---------|-----------------|
| 养老保险 | 退休后养老金 | 单位16%+个人8% |
| 医疗保险 | 看病报销 | 单位9%+个人2% |
| 失业保险 | 失业补助 | 单位0.5%+个人0.5% |
| 工伤保险 | 工伤保障 | 单位缴纳 |
| 生育保险 | 生育保障 | 单位缴纳 |

**社会保险的特点**：
1. **强制性**：必须参保
2. **互助性**：大家互助共济
3. **权利义务对应**：缴费才能享受
4. **政府补贴**：财政支持

**社会保险基金**：
- 来源：缴费+财政补贴+投资收益
- 管理：专款专用
- 安全：保值增值
