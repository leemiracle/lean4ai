## 📖 第一章：社会保障基础理论

---

### 一、社会保障概述

#### 1. 社会保障 (Social Security/社会保障)

**白话解释**:  
国家建立的，保障基本生活的安全网。

**核心特征**:
1. **强制性** - 必须参加
2. **福利性** - 非营利
3. **社会性** - 全社会参与
4. **保障性** - 保障基本生活

**功能**:
1. **稳定器** - 维护社会稳定
2. **安全网** - 防范生活风险
3. **调节器** - 调节收入分配
4. **助推器** - 促进经济发展

**费曼记忆**:  
社会保障 = 国家主导 + 社会参与 + 保障基本生活

---

#### 2. 社会保障体系 (Social Security System/社会保障体系)

**白话解释**:  
各种社会保障项目组成的整体。

**中国社会保障体系**:
```
社会保险 (核心)
├─ 养老保险
├─ 医疗保险
├─ 失业保险
├─ 工伤保险
└─ 生育保险

社会救助 (兜底)
├─ 最低生活保障
├─ 特困人员供养
├─ 医疗救助
└─ 住房救助

社会福利 (提升)
├─ 老年人福利
├─ 残疾人福利
├─ 儿童福利
└─ 教育福利

社会优抚 (特殊)
├─ 军人抚恤
└─ 军人安置
```

---

#### 3. 社会保障原则 (Social Security Principles/社保原则)

**白话解释**:  
建立和运行社保要遵循的基本准则。

**基本原则**:
1. **公平原则** - 人人有保障
2. **与经济发展相适应** - 量力而行
3. **权利与义务相结合** - 先缴费后享受
4. **统筹兼顾** - 城乡统筹

**平衡关系**:
- 公平 vs 效率
- 权利 vs 义务
- 现收现付 vs 基金积累
- 国家责任 vs 个人责任

## 📖 第二章：社会保险

---

### 一、养老保险

#### 4. 养老保险 (Pension Insurance/养老保险)

**白话解释**:  
年轻时交钱，老了领钱。

**中国养老保险体系**:
```
第一支柱 - 基本养老保险
├─ 城镇职工养老保险
└─ 城乡居民养老保险

第二支柱 - 企业/职业年金
├─ 企业年金
└─ 职业年金

第三支柱 - 个人养老金
└─ 个人储蓄性养老保险
```

**缴费与待遇**:
- 职工养老：单位16%+个人8%
- 居民养老：自愿缴费+政府补贴
- 待遇：缴费满15年，退休后按月领取

---

#### 5. 养老保险制度 (Pension System/养老保险制度)

**白话解释**:  
养老保险的具体运作规则。

**计发办法**:
```
基础养老金 = (当地平均工资 + 本人指数化工资) / 2 × 缴费年限 × 1%

个人账户养老金 = 个人账户储存额 / 计发月数
```

**多缴多得机制**:
- 缴费年限越长，待遇越高
- 缴费基数越高，待遇越高

**退休年龄**:
- 男性：60岁
- 女干部：55岁
- 女工人：50岁
- 延迟退休政策推进中

---

### 二、医疗保险

#### 6. 医疗保险 (Medical Insurance/医疗保险)

**白话解释**:  
看病报销的制度。

**中国医保体系**:
```
职工基本医疗保险
├─ 个人账户 (门诊)
└─ 统筹基金 (住院)

城乡居民基本医疗保险
└─ 统一统筹

大病保险
└─ 超过一定费用二次报销

医疗救助
└─ 困难人群救助
```

**报销比例**:
- 起付线以下：自付
- 起付线-封顶线：按比例报销
- 封顶线以上：大病保险

---

#### 7. 医保支付方式 (Medical Payment Method/医保支付)

**白话解释**:  
医保怎么付钱给医院。

**主要方式**:
1. **按项目付费** - 做多少项付多少
2. **按病种付费 (DRG)** - 按病打包
3. **按人头付费** - 按人数打包
4. **总额预付** - 给总额控制

**改革方向**:
- DRG/DIP付费改革
- 控制医疗费用过快增长
- 提高医保基金效率

---

### 三、其他保险

#### 8. 失业保险 (Unemployment Insurance/失业保险)

**白话解释**:  
失业了暂时领点钱过渡。

**领取条件**:
1. 缴费满1年
2. 非因本人意愿中断就业
3. 已办理失业登记

**待遇标准**:
- 领取期限：最长24个月
- 标准：低于最低工资、高于低保
- 同时享受医保待遇

---

#### 9. 工伤保险 (Work Injury Insurance/工伤保险)

**白话解释**:  
工作中受伤有保障。

**保障范围**:
1. **工伤医疗** - 医疗费用报销
2. **工伤津贴** - 停工留薪期工资
3. **伤残待遇** - 伤残补助金
4. **工亡待遇** - 丧葬补助+抚恤金

**认定条件**:
- 工作时间和工作场所内，因工作原因受到事故伤害
- 工作时间前后，从事与工作有关的预备性或收尾性工作受到事故伤害
- 上下班途中受到非本人主要责任交通事故伤害

---

#### 10. 生育保险 (Maternity Insurance/生育保险)

**白话解释**:  
生孩子有补贴和报销。

**待遇内容**:
1. **生育医疗费** - 检查、分娩费用
2. **生育津贴** - 产假期间工资
3. **计划生育手术费**

**与职工医保合并**:
2019年起，生育保险和职工医保合并实施，仍保留生育保险待遇。

## 📖 第三章：社会救助

---

#### 11. 社会救助 (Social Assistance/社会救助)

**白话解释**:  
政府给困难人群发钱救急。

**特点**:
1. **兜底性** - 最后一道防线
2. **无偿性** - 不用还
3. **选择性** - 只给困难人群
4. **临时性** - 困难解除就停

**救助体系**:
```
最低生活保障 (低保)
└─ 收入低于标准的家庭

特困人员供养
├─ 无劳动能力
├─ 无生活来源
└─ 无法定赡养人

专项救助
├─ 医疗救助
├─ 教育救助
├─ 住房救助
└─ 就业救助

临时救助
└─ 突发困难
```

---

#### 12. 最低生活保障 (Minimum Living Security/低保)

**白话解释**:  
收入不够，政府补齐到最低标准。

**认定条件**:
1. 家庭人均收入低于当地低保标准
2. 家庭财产符合规定

**计算方法**:
```
低保金 = 低保标准 - 家庭人均收入
```

**动态管理**:
- 定期复核
- 收入增加调整
- 收入超标退出

---

#### 13. 医疗救助 (Medical Assistance/医疗救助)

**白话解释**:  
困难人群看病政府帮忙。

**救助对象**:
- 低保对象
- 特困人员
- 低收入家庭
- 因病致贫重病患者

**救助方式**:
1. **资助参保** - 代缴医保费
2. **门诊救助** - 门诊费用补助
3. **住院救助** - 住院费用补助

---

## 📖 第四章：社会福利

---

#### 14. 社会福利 (Social Welfare/社会福利)

**白话解释**:  
国家提供的，提升生活质量的公共服务。

**特点**:
1. **普惠性** - 广覆盖
2. **服务性** - 提供服务为主
3. **层次性** - 不同人群不同服务

**主要领域**:
- 老年人福利
- 儿童福利
- 残疾人福利
- 教育福利
- 住房福利

---

#### 15. 老年人福利 (Elderly Welfare/老年人福利)

**白话解释**:  
给老年人提供的服务和优待。

**主要内容**:
1. **养老机构** - 敬老院、养老院
2. **社区养老** - 日间照料、上门服务
3. **居家养老** - 适老化改造
4. **优待政策** - 免费乘车、景区优惠

**养老服务体系建设**:
```
9073格局:
- 90% 居家养老
- 7% 社区养老
- 3% 机构养老
```

---

#### 16. 残疾人福利 (Disability Welfare/残疾人福利)

**白话解释**:  
给残疾人提供的帮助和服务。

**主要措施**:
1. **两项补贴**:
   - 困难残疾人生活补贴
   - 重度残疾人护理补贴

2. **康复服务** - 康复训练、辅具适配
3. **就业扶持** - 按比例就业
4. **无障碍建设** - 公共设施无障碍

---

## 📖 第五章：社会保障基金管理

---

#### 17. 社保基金 (Social Security Fund/社保基金)

**白话解释**:  
社保的钱从哪来、怎么管。

**筹资模式**:
1. **现收现付制** - 当年收当年付
2. **完全积累制** - 个人账户积累
3. **部分积累制** - 统筹+个人账户

**中国养老保险筹资**:
```
社会统筹 (现收现付)
+ 
个人账户 (积累)
= 
统账结合
```

---

#### 18. 社保基金投资 (Social Security Fund Investment/社保基金投资)

**白话解释**:  
社保基金怎么保值增值。

**投资原则**:
1. **安全性** - 不能亏
2. **流动性** - 随时能取
3. **收益性** - 要赚钱

**投资范围**:
- 银行存款
- 国债
- 金融债
- 股票（比例受限）
- 基础设施

---

#### 19. 社保基金监管 (Social Security Fund Supervision/社保基金监管)

**白话解释**:  
监督社保基金不被人挪用。

**监管体系**:
1. **行政监管** - 人社、财政、审计
2. **社会监督** - 公众、媒体
3. **内部控制** - 制度约束

**监管重点**:
- 基金征收
- 基金支付
- 基金结余
- 基金投资

## 📖 第六章：社会保障改革与发展

---

#### 20. 人口老龄化挑战 (Aging Challenge/老龄化挑战)

**白话解释**:  
老人越来越多，年轻人不够养。

**老龄化指标**:
- 60岁以上人口占比超过10%
- 65岁以上人口占比超过7%

**中国老龄化特点**:
1. **规模大** - 老年人口世界第一
2. **速度快** - 老龄化速度加快
3. **未富先老** - 经济水平还不够高

**应对策略**:
- 延迟退休
- 发展第三支柱
- 促进生育
- 发展银发经济

---

#### 21. 多层次保障体系 (Multi-level Security System/多层次保障)

**白话解释**:  
政府、企业、个人一起养老。

**三支柱体系**:
| 支柱 | 主体 | 特点 |
|------|------|------|
| 第一支柱 | 国家 | 基本保障 |
| 第二支柱 | 企业 | 补充保障 |
| 第三支柱 | 个人 | 自愿储蓄 |

**发展目标**:
- 第一支柱保基本
- 第二支柱扩覆盖
- 第三支柱快发展

---

#### 22. 社保数字化 (Social Security Digitalization/社保数字化)

**白话解释**:  
用信息技术改进社保服务。

**应用场景**:
1. **网上办理** - 足不出户办业务
2. **移动支付** - 手机领养老金
3. **大数据监管** - 防冒领
4. **全国联网** - 异地结算

**发展方向**:
- 一网通办
- 异地通办
- 智能服务
