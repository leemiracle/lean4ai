## 📖 第一章：会计基础

---

### 一、会计基本概念

#### 1. 会计 (Accounting/会计)

**白话解释**:  
用数字记录企业的经济活动，告诉别人企业过得怎么样。

**核心功能**:
1. **确认** - 哪些需要记录
2. **计量** - 记录多少钱
3. **记录** - 怎么记账
4. **报告** - 怎么告诉别人

**会计分类**:
| 类型 | 对象 | 使用者 |
|------|------|--------|
| 财务会计 | 对外报告 | 投资者、债权人 |
| 管理会计 | 内部决策 | 管理层 |
| 成本会计 | 成本核算 | 成本控制 |
| 税务会计 | 纳税申报 | 税务机关 |

**费曼记忆**:  
会计 = 记录 + 计量 + 报告

---

#### 2. 会计假设 (Accounting Assumptions/会计假设)

**白话解释**:  
会计工作的前提条件，没有这些就没法记账。

**四大基本假设**:
1. **会计主体** - 只记企业的事，不管老板私事
2. **持续经营** - 假设企业会一直开下去
3. **会计分期** - 把时间切成段（月、季、年）
4. **货币计量** - 用钱来衡量一切

**实务意义**:
- 会计主体 → 独立核算
- 持续经营 → 折旧、摊销
- 会计分期 → 定期报表
- 货币计量 → 统一度量

---

#### 3. 会计恒等式 (Accounting Equation/会计恒等式)

**白话解释**:  
资产等于权益，这是会计的根本。

**基本公式**:
```
资产 = 负债 + 所有者权益
```

**扩展公式**:
```
资产 = 负债 + 所有者权益 + (收入 - 费用)
```

**理解方式**:
- 资产: 企业拥有的资源
- 负债: 借来的钱
- 所有者权益: 股东投的钱 + 赚的钱

---

#### 4. 复式记账法 (Double Entry Bookkeeping/复式记账法)

**白话解释**:  
每笔业务至少影响两个账户，保持平衡。

**记账规则**:
```
有借必有贷，借贷必相等
```

**借贷方向**:
| 账户类型 | 增加 | 减少 |
|----------|------|------|
| 资产 | 借 | 贷 |
| 负债 | 贷 | 借 |
| 所有者权益 | 贷 | 借 |
| 收入 | 贷 | 借 |
| 费用 | 借 | 贷 |

---

### 二、会计要素

#### 5. 资产 (Assets/资产)

**白话解释**:  
企业拥有的、能带来收益的资源。

**分类**:
1. **流动资产** - 一年内能变现
   - 货币资金
   - 应收账款
   - 存货

2. **非流动资产** - 长期使用
   - 固定资产
   - 无形资产
   - 长期投资

**确认条件**:
- 企业拥有或控制
- 能带来经济利益
- 能可靠计量

---

#### 6. 负债 (Liabilities/负债)

**白话解释**:  
企业欠别人的钱，需要还。

**分类**:
1. **流动负债** - 一年内要还
   - 短期借款
   - 应付账款
   - 预收账款

2. **非流动负债** - 一年后还
   - 长期借款
   - 应付债券

**实务提示**:  
负债不一定是坏事，合理负债可以放大收益。

---

#### 7. 所有者权益 (Owner's Equity/所有者权益)

**白话解释**:  
真正属于股东的钱。

**构成**:
1. **实收资本** - 股东投入
2. **资本公积** - 股本溢价等
3. **盈余公积** - 利润留存
4. **未分配利润** - 还没分的利润

**公式**:
```
所有者权益 = 资产 - 负债
```

---

#### 8. 收入 (Revenue/收入)

**白话解释**:  
企业卖东西赚的钱。

**确认条件**:
1. 商品所有权转移
2. 收入金额能可靠计量
3. 经济利益很可能流入
4. 相关成本能可靠计量

**分类**:
- 主营业务收入
- 其他业务收入

---

#### 9. 费用 (Expense/费用)

**白话解释**:  
为了赚钱花掉的钱。

**分类**:
1. **营业成本** - 卖出产品的成本
2. **期间费用**:
   - 销售费用
   - 管理费用
   - 财务费用

**费用vs成本**:
- 费用: 期间消耗
- 成本: 对象化到产品

---

#### 10. 利润 (Profit/利润)

**白话解释**:  
赚了多少钱。

**计算公式**:
```
营业利润 = 营业收入 - 营业成本 - 税金 - 期间费用
利润总额 = 营业利润 + 营业外收支
净利润 = 利润总额 - 所得税
```

**利润分配顺序**:
1. 弥补亏损
2. 提取法定盈余公积
3. 提取任意盈余公积
4. 分配股利

## 📖 第二章：财务报表

---

### 一、三大报表

#### 11. 资产负债表 (Balance Sheet/资产负债表)

**白话解释**:  
某一时点企业的"家底"。

**结构**:
```
资产             负债和所有者权益
────────────────────────────────
流动资产         流动负债
非流动资产       非流动负债
                 所有者权益
────────────────────────────────
资产合计 = 负债和所有者权益合计
```

**阅读重点**:
- 资产结构是否合理
- 负债水平是否过高
- 流动性是否充足

---

#### 12. 利润表 (Income Statement/利润表)

**白话解释**:  
一段时间企业赚了多少钱。

**结构**:
```
营业收入
  - 营业成本
  = 毛利
  - 期间费用
  = 营业利润
  + 营业外收支
  = 利润总额
  - 所得税
  = 净利润
```

**关键指标**:
- 毛利率 = 毛利/营业收入
- 净利率 = 净利润/营业收入

---

#### 13. 现金流量表 (Cash Flow Statement/现金流量表)

**白话解释**:  
钱从哪来，到哪去。

**三大活动**:
1. **经营活动** - 日常业务
   - 销售收到现金
   - 采购支付现金

2. **投资活动** - 买卖资产
   - 购买固定资产
   - 对外投资

3. **筹资活动** - 借钱还钱
   - 借款收到现金
   - 偿还债务

**关键指标**:
- 经营现金流净额
- 自由现金流

---

### 二、财务分析

#### 14. 偿债能力分析 (Solvency Analysis/偿债能力)

**白话解释**:  
企业能不能还得起债。

**短期偿债能力**:
| 指标 | 公式 | 标准 |
|------|------|------|
| 流动比率 | 流动资产/流动负债 | ≥2 |
| 速动比率 | (流动资产-存货)/流动负债 | ≥1 |
| 现金比率 | 现金/流动负债 | 越高越好 |

**长期偿债能力**:
| 指标 | 公式 | 意义 |
|------|------|------|
| 资产负债率 | 负债/资产 | 负债水平 |
| 产权比率 | 负债/所有者权益 | 财务杠杆 |

---

#### 15. 盈利能力分析 (Profitability Analysis/盈利能力)

**白话解释**:  
企业赚钱的能力怎么样。

**核心指标**:
| 指标 | 公式 | 意义 |
|------|------|------|
| 毛利率 | 毛利/收入 | 产品盈利能力 |
| 净利率 | 净利润/收入 | 最终盈利能力 |
| ROE | 净利润/净资产 | 股东回报率 |
| ROA | 净利润/总资产 | 资产回报率 |

**杜邦分析**:
```
ROE = 净利率 × 资产周转率 × 权益乘数
```

---

#### 16. 营运能力分析 (Operating Efficiency/营运能力)

**白话解释**:  
企业转得快不快。

**核心指标**:
| 指标 | 公式 | 意义 |
|------|------|------|
| 存货周转率 | 营业成本/平均存货 | 存货周转速度 |
| 应收账款周转率 | 营业收入/平均应收 | 收款速度 |
| 总资产周转率 | 营业收入/平均资产 | 资产利用效率 |

### 三、典型业务处理

#### 17. 折旧 (Depreciation/折旧)

**白话解释**:  
固定资产的价值慢慢转移到产品中。

**折旧方法**:
1. **直线法**: (原值 - 残值)/使用年限
2. **双倍余额递减法**: 年初净值 × 2/年限
3. **年数总和法**: (原值 - 残值) × 剩余年限/年数总和

**会计分录**:
```
借: 制造费用/管理费用
    贷: 累计折旧
```

---

#### 18. 坏账准备 (Bad Debt Provision/坏账准备)

**白话解释**:  
应收账款可能收不回来，先预估损失。

**计提方法**:
1. **余额百分比法**: 应收余额 × 估计比例
2. **账龄分析法**: 按账龄分别计提
3. **个别认定法**: 逐笔评估

**会计分录**:
```
借: 信用减值损失
    贷: 坏账准备
```

---

#### 19. 存货计价 (Inventory Valuation/存货计价)

**白话解释**:  
存货值多少钱，用哪种成本。

**计价方法**:
1. **先进先出法 (FIFO)** - 先买的先出
2. **加权平均法** - 平均成本
3. **个别计价法** - 逐个认定

**存货跌价准备**:
成本与可变现净值孰低

## 📖 第四章：成本会计

---

#### 20. 成本核算 (Cost Accounting/成本核算)

**白话解释**:  
算清楚产品成本多少钱。

**成本构成**:
```
产品成本 = 直接材料 + 直接人工 + 制造费用
```

**成本流程**:
```
材料 → 在产品 → 产成品 → 营业成本
```

**核算方法**:
1. **品种法** - 单步骤生产
2. **分批法** - 订单生产
3. **分步法** - 多步骤生产

---

#### 21. 本量利分析 (CVP Analysis/本量利分析)

**白话解释**:  
成本、销量、利润的关系。

**基本公式**:
```
利润 = (单价 - 单位变动成本) × 销量 - 固定成本
```

**盈亏平衡点**:
```
盈亏平衡销量 = 固定成本/(单价 - 单位变动成本)
```

**边际贡献**:
```
边际贡献 = 单价 - 单位变动成本
```

## 📖 第五章：会计规范

---

#### 22. 会计准则 (Accounting Standards/会计准则)

**白话解释**:  
记账的"规则手册"。

**中国会计准则体系**:
1. **基本准则** - 总体原则
2. **具体准则** - 具体业务
3. **应用指南** - 操作指引

**国际会计准则 (IFRS)**:
- 全球通用标准
- 中国准则持续趋同

---

#### 23. 会计信息质量要求

**八大要求**:
1. **可靠性** - 真实反映
2. **相关性** - 对决策有用
3. **可理解性** - 清晰易懂
4. **可比性** - 可以比较
5. **实质重于形式** - 看本质
6. **重要性** - 抓重点
7. **谨慎性** - 不高估资产
8. **及时性** - 及时报告
