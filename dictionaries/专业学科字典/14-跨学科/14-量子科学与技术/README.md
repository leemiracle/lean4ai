## 学科概述

### 什么是量子科学？

想象你有一枚神奇硬币，可以**同时**是正面和反面——直到你看它一眼，它才"决定"变成其中一面。这就是量子世界的神奇之处。

"量子"不是某种粒子，而是指能量的最小单位。微观世界里，能量是一份一份的，不是连续的。

## 量子力学基础

### 叠加态：同时存在的可能性

你在选择午餐——汉堡还是沙拉？经典世界只能选一个。量子世界可以处于两者的**叠加状态**——同时做两件事！

```
经典比特：  0 ──── 或 ──── 1

量子比特：  |ψ⟩ = α|0⟩ + β|1⟩
            （同时是0和1）
```

**数学表达：**
- |ψ⟩ 是量子态（狄拉克符号）
- α, β 是复数概率幅
- |α|² + |β|² = 1（概率守恒）

**为什么叠加重要？**

n个经典比特 = 1个状态，n个量子比特 = 2^n 个状态同时存在。50个量子比特可同时表示约1千万亿个状态！

### 纠缠态：鬼魅般的超距作用

你有一双魔法手套，一只寄给北京，一只留在纽约。打开发现是左手时，**瞬间**知道那边是右手。

量子纠缠更神奇：测量之前，**两只手套都还没决定**是左是右。测量让它们同时"坍缩"成配对状态。

```
EPR对：|Φ⁺⟩ = (|00⟩ + |11⟩)/√2

测量第一个：得到0 → 第二个必然是0
            得到1 → 第二个必然是1
```

**贝尔不等式验证：** 经典预测 S ≤ 2，量子预测 S ≤ 2√2，实验证实量子纠缠！

### 测量与坍缩

一个旋转的硬币，你问"是正面吗？"——这一问，硬币停止旋转。问题强迫硬币做出选择。

```
测量前：|ψ⟩ = α|0⟩ + β|1⟩

测量后：以概率|α|²得到|0⟩，态坍缩
        以概率|β|²得到|1⟩，态坍缩
```

**海森堡不确定性：** Δx · Δp ≥ ℏ/2，位置测量越准，动量测量越不准。这是自然的根本限制。

## 量子计算

### 量子比特（Qubit）

**经典比特 vs 量子比特：**

```
经典比特：开关 ─── [开] 或 [关]
                    1      0

量子比特：箭头 ─── 可指向任何方向
```

**布洛赫球表示：**

```
|ψ⟩ = cos(θ/2)|0⟩ + e^(iφ)sin(θ/2)|1⟩

θ: 与z轴夹角（0到π）
φ: xy平面旋转角（0到2π）
```

**物理实现：**

| 技术路线 | 载体 | 优点 | 挑战 |
|----------|------|------|------|
| 超导 | 约瑟夫森结 | 可扩展、快 | 需极低温 |
| 离子阱 | 囚禁离子 | 高保真 | 扩展难 |
| 光量子 | 光子偏振 | 室温 | 存储难 |
| 半导体 | 量子点 | 兼容工艺 | 退相干 |
| 中性原子 | 原子阵列 | 高度并行 | 控制复杂 |

### 量子门

**单量子比特门：**

```
X门（量子NOT）：X|0⟩=|1⟩, X|1⟩=|0⟩

H门（创造叠加）：
  H|0⟩ = (|0⟩+|1⟩)/√2
  H|1⟩ = (|0⟩-|1⟩)/√2

Z门（相位翻转）：Z|0⟩=|0⟩, Z|1⟩=-|1⟩
```

**双量子比特门：**

```
CNOT门：控制位为1 → 目标位翻转
|00⟩→|00⟩, |01⟩→|01⟩, |10⟩→|11⟩, |11⟩→|10⟩
```

**通用门集：** H门 + T门 + CNOT门可构建任意量子计算。

### 量子算法

| 问题 | 经典复杂度 | 量子复杂度 | 加速 |
|------|-----------|-----------|------|
| 搜索N项 | O(N) | O(√N) | 平方 |
| 大数分解 | 指数 | 多项式 | 指数 |
| 量子模拟 | O(e^n) | O(n^k) | 指数 |

**Shor算法（大数分解）：**

分解2048位数字：经典超算约10亿年，量子计算机理论上约几小时。威胁RSA加密！

**Grover算法（搜索）：**

在1000本书中找一本：经典方法平均500次，量子方法约32次（√1000）。

**量子机器学习：** HHL算法、量子主成分分析、变分量子算法（VQE、QAOA）。

## 量子通信

### 量子密钥分发（QKD）

**核心思想：** 窃听者的测量会扰动量子态，因此可被检测。

**BB84协议：**

```
1. Alice随机选比特值(0/1)和基(+基/×基)，发送量子态
2. Bob随机选测量基，记录结果
3. 公布使用的基（不公布结果）
4. 保留基相同的比特（约50%）
5. 公布部分检测窃听
6. 误码率低于阈值，剩余为密钥
```

**安全性来源：**

- 不可克隆定理：无法复制未知量子态
- 测量干扰：窃听引入错误
- 单光子特性：截获后无法重发

**实际系统：**

| 类型 | 距离 | 场景 |
|------|------|------|
| 光纤QKD | 500+ km | 城市间 |
| 自由空间 | 1000+ km | 卫星通信 |

### 量子隐形传态

**传信息，不是传物质！**

```
步骤：
1. Alice对待传态和她的一半纠缠对做Bell测量
2. 通过经典信道告诉Bob结果（2比特）
3. Bob根据结果操作，得到原始态

注意：原态被销毁，不违反信息守恒
```

**Bell态：**
```
|Φ⁺⟩ = (|00⟩ + |11⟩)/√2    |Φ⁻⟩ = (|00⟩ - |11⟩)/√2
|Ψ⁺⟩ = (|01⟩ + |10⟩)/√2    |Ψ⁻⟩ = (|01⟩ - |10⟩)/√2
```

2017年中国"墨子号"卫星实现千公里级隐形传态。

## 量子传感

### 量子精密测量

**精度极限：**

```
标准量子极限（SQL）：精度 ∝ 1/√N
海森堡极限（HL）：    精度 ∝ 1/N

使用纠缠粒子可达到海森堡极限，改进√N倍
```

### 主要技术

| 类型 | 量子系统 | 测量量 | 应用 |
|------|----------|--------|------|
| 原子钟 | 冷原子 | 时间/频率 | GPS、通信 |
| 量子磁力计 | NV色心 | 磁场 | 医疗、地质 |
| 量子重力计 | 冷原子 | 重力加速度 | 资源勘探 |

### 原子钟

利用原子能级跃迁的稳定频率作为时间标准。

```
铯原子钟：9,192,631,770 Hz（定义秒）
光钟精度：10^-18（3000年误差不到1秒）
```

应用：GPS定位、基础物理研究、金融时间戳。

### NV色心

金刚石中的氮-空位缺陷，室温下工作。

应用：磁场测量（检测神经元电活动）、纳米温度成像、细胞内传感。

## 量子材料

### 定义

展现出显著量子效应的材料，其性质无法用经典物理完全解释。

### 主要类型

**1. 超导体**

```
特性：零电阻、完全抗磁性
临界参数：温度Tc、磁场Hc、电流密度Jc
高温超导体：Tc > 77K（液氮温度）
例：YBCO（钇钡铜氧），Tc ≈ 93K
```

**2. 拓扑材料**

```
拓扑绝缘体：内部绝缘，表面导电
- 表面态受拓扑保护
- 不受杂质散射影响

拓扑超导体：可能存在马约拉纳费米子
可用于拓扑量子计算
```

**3. 二维材料**

```
石墨烯：单层碳原子，高导电、高强度
TMDs：可调带隙，谷电子学应用
```

**4. 量子自旋液体**

自旋永不冻结（即使在绝对零度），高度纠缠，可能存在任意子。

## 量子模拟

### 为什么需要？

模拟50量子比特系统需要约1千万亿个复数，经典计算机存不下！

**费曼愿景（1982）：** "用量子机器模拟量子系统。"

### 方法

**数字量子模拟：**

```
用量子门构建目标系统演化
U(t) = e^(-iHt/ℏ)
分解：Trotter公式 e^(A+B) ≈ e^A·e^B
```

**模拟量子模拟：**

- 冷原子模拟凝聚态系统
- 离子阱模拟自旋模型
- 超导电路模拟量子电动力学

### 应用

- 凝聚态：高温超导机制、量子相变、强关联系统
- 量子化学：分子能量、化学反应、药物设计
- 高能物理：格点规范场论
