## 学科概述

### 什么是遥感？

想象你在拍照。你举起相机，按下快门，照片就记录了你眼前的景象。遥感做的事情本质上一样——只是相机的位置在太空中，拍摄的对象是整个地球。

**遥感（Remote Sensing）** = 远距离感知

简单说，就是不接触物体，从远处"感知"它的信息。

### 四大学科的交叉融合

遥感是个"混血儿"，它的基因来自四个家族：

```
┌─────────────────────────────────────────────────────────────┐
│                        遥感科学与技术                         │
├──────────────┬──────────────┬──────────────┬────────────────┤
│    航天技术   │   电子技术    │   地理学     │   计算机科学   │
│              │              │              │                │
│ • 卫星轨道    │ • 传感器     │ • 地表特征   │ • 图像处理     │
│ • 火箭发射    │ • 信号处理   │ • 地理规律   │ • 数据存储     │
│ • 空间环境    │ • 电磁波     │ • 地图投影   │ • 人工智能     │
└──────────────┴──────────────┴──────────────┴────────────────┘
```

**为什么需要这四个？**

- **航天**：把相机送上天，站得高才能看得远
- **电子**：制造"眼睛"（传感器），接收和记录信号
- **地理**：理解拍到的内容，知道这是森林还是沙漠
- **计算机**：处理海量数据，从照片中提取有用信息

### 一个类比：卫星是太空中的超级相机

| 相机部件 | 遥感对应 | 作用 |
|---------|---------|------|
| 镜头 | 传感器光学系统 | 收集光线 |
| 感光元件 | 探测器 | 把光变成电信号 |
| 存储卡 | 卫星存储器 | 保存图像 |
| 三脚架 | 卫星平台 | 保持稳定 |
| Wi-Fi传输 | 数据下传链路 | 传回地面 |

## 遥感物理基础

### 核心原理：电磁波与物质的"对话"

想象每个物体都会"说话"——用电磁波说话。

**太阳发出光 → 照到地面 → 地面"回答"一部分光 → 卫星收到回答**

不同物体的"回答"不一样：
- 植物说："我喜欢吸收红光，反射绿光和近红外光"
- 水体说："我几乎吃掉所有光，很少反射"
- 沙漠说："我把大部分光都反射回去"

这种"回答方式"叫做**光谱特征**，就像每个物体的指纹。

### 电磁波谱：遥感的"语言字典"

```
波长从短到长：
┌────────┬────────┬────────┬────────┬────────┬────────┬────────┐
│  γ射线  │  X射线  │ 紫外线 │ 可见光 │ 红外线 │ 微波   │ 无线电 │
└────────┴────────┴────────┴────────┴────────┴────────┴────────┘
 人体有害  人体有害   遥感少用  眼睛可见 黑暗中也能 大气穿透 通信用
                            感知温度   感知     能力强
```

**遥感最常用的波段：**

1. **可见光（0.4-0.7μm）**：就是我们能看到的颜色
2. **近红外（0.7-1.3μm）**：植物的健康指标
3. **热红外（3-14μm）**：感知温度，森林火灾监测用
4. **微波（1mm-1m）**：能穿透云层，全天候工作

### 一个关键概念：大气窗口

地球被大气层包裹着。大气层像个滤网，有些波长的光被吸收了，有些能穿透。

**大气窗口** = 大气层"允许通过"的波段

就像你家的窗户，有些光能进来，有些被玻璃挡住了。遥感要选择在大气窗口内的波段，否则信号被大气"吃掉"了。

## 遥感平台

### 三层高度，三种视野

遥感平台就是"相机的位置"。位置越高，看得越广，但细节越少。

```
高度 ↑
 │      ┌─────────────────────────────────────┐
 │      │         卫星（400-36000km）          │
 │      │   视野：整个国家/洲                   │
 │      │   特点：周期性重复观测                │
 │      ├─────────────────────────────────────┤
 │      │         航空飞机（1-20km）           │
 │      │   视野：一个城市/县                   │
 │      │   特点：灵活、分辨率高                │
 │      ├─────────────────────────────────────┤
 │      │         无人机（0.1-5km）            │
 │      │   视野：一个小区/工地                │
 │      │   特点：超低空、超灵活、实时         │
 │      └─────────────────────────────────────┘
```

### 卫星遥感：太空中的守望者

**按轨道分类：**

| 轨道类型 | 高度 | 特点 | 应用 |
|---------|------|------|------|
| 低轨LEO | 200-2000km | 离地球近，分辨率高 | 高分辨率成像、侦察 |
| 中轨MEO | 2000-35786km | 覆盖范围大 | GPS导航 |
| 地球同步GEO | 35786km | 相对地面静止 | 气象卫星、通信 |

**著名遥感卫星：**

- **Landsat系列**（美国）：1972年至今，最长的地球观测记录
- **Sentinel系列**（欧盟）：免费开放数据
- **高分系列**（中国）：国产高分辨率卫星
- **WorldView**（商业）：分辨率可达0.3米

### 航空遥感：灵活的"中距离选手"

用飞机搭载传感器飞行：

**优点：**
- 时间灵活，想飞就飞
- 分辨率比卫星高
- 可以针对特定区域

**缺点：**
- 覆盖范围有限
- 受天气影响大
- 成本较高

### 无人机遥感：贴地飞行的"小精灵"

近年来发展最快的遥感平台：

**为什么无人机这么火？**
- 便宜：几万元就能入门
- 灵活：一个人就能操作
- 精细：厘米级分辨率
- 实时：现场就能看结果

**应用场景：**
- 农田精准管理
- 建筑工地监测
- 灾后快速评估
- 考古测绘

## 传感器技术

### 传感器：遥感的"眼睛"

传感器是把电磁波信号转换成可用数据的装置。理解传感器，就理解了遥感数据是怎么来的。

### 两大类型

**1. 被动式传感器**（依靠太阳光）

```
太阳 → 地面 → 传感器
      ↑反射
```

就像普通相机，需要光线才能拍照。

- 多光谱扫描仪
- 高光谱传感器
- 热红外传感器

**2. 主动式传感器**（自己发射信号）

```
传感器 → 发射信号 → 地面 → 接收反射信号 → 传感器
```

就像闪光灯或雷达，自己发光/发波。

- 激光雷达（LiDAR）
- 合成孔径雷达（SAR）

### 关键参数

**空间分辨率**：一个像素代表多大范围

```
空间分辨率：1米
意味着：图像中一个像素 = 地面1米×1米

分辨率越高 → 细节越清晰 → 但数据量越大
```

**光谱分辨率**：能区分多少种颜色

```
普通相机：3个波段（红、绿、蓝）
多光谱：4-20个波段
高光谱：200+个波段

光谱越细 → 能识别的物质越多
```

**时间分辨率**：多久重访一次

```
地球同步卫星：持续观测（每15分钟）
极轨卫星：每天2次
商业卫星：几天一次
```

### 几种重要传感器详解

**多光谱传感器**
- 通道：4-10个
- 每个通道对应一个波段
- 像是用不同颜色的眼镜看世界

**高光谱传感器**
- 通道：200+
- 可以识别具体的化学成分
- 像是能看到颜色的DNA

**热红外传感器**
- 感知温度分布
- 用于城市热岛、森林火灾、火山监测

**合成孔径雷达（SAR）**
- 不怕云雨，全天候工作
- 可以"看到"地面微小变形（毫米级）
- 地震监测、地面沉降

**激光雷达（LiDAR）**
- 主动发射激光脉冲
- 直接测量距离，生成三维模型
- 地形测绘、森林结构、自动驾驶

## 图像处理

### 从原始数据到可用信息

传感器传回来的不是漂亮的照片，而是数字矩阵。图像处理就是把这些数字变成有意义的信息。

```
原始数据 → 预处理 → 增强 → 分类 → 提取信息
```

### 预处理：清洗数据

**辐射定标**
- 把数字值（DN）转换成物理量（反射率）
- 就像把相机的"亮度值"转换成"真实光照强度"

**大气校正**
- 去除大气干扰
- 获得地表真实反射率

**几何校正**
- 消除图像畸变
- 让图像和真实位置对应

**正射校正**
- 消除地形影响
- 让图像像平面地图一样可测量

### 图像增强：让重要信息更突出

**对比度拉伸**
- 增强图像对比度
- 让暗的更暗，亮的更亮

**假彩色合成**
- 人眼只能看三个波段（RGB）
- 选择不同波段组合，突出不同特征

```
真彩色：R=红光，G=绿光，B=蓝光 → 和眼睛看到的一样
假彩色：R=近红外，G=红光，B=绿光 → 植物显示为红色
```

**图像融合**
- 把高分辨率黑白图和低分辨率彩色图合并
- 得到既清晰又有颜色的图像

### 变化检测：找不同

对比不同时期的图像，发现变化：

- 城市扩张监测
- 森林砍伐追踪
- 洪水淹没范围
- 违章建筑发现

## 遥感解译

### 从"看到"到"看懂"

解译就是理解图像内容，回答"这是什么"的问题。

### 两种解译方式

**1. 目视解译**（人工识别）

依靠专业人员的经验和知识。

**解译标志：**
- 形状：圆形、方形、线形
- 大小：相对尺寸
- 色调/颜色：灰度、色彩
- 纹理：粗糙、平滑
- 模式：排列规律
- 阴影：立体感
- 位置/关联：和相关地物的关系

**2. 计算机自动解译**（分类算法）

让计算机自动识别地物类型。

### 图像分类方法

**监督分类**
- 人先告诉计算机样本
- 计算机学习后识别整幅图

```
1. 选择训练样本（这是森林，这是水）
2. 算法学习特征
3. 对整幅图进行分类
```

**非监督分类**
- 计算机自动分组
- 人再给每组起名字

**常用算法：**
- 最大似然法
- 支持向量机（SVM）
- 随机森林
- 深度学习（CNN）

### 精度验证

分类完要知道准不准：

**混淆矩阵**
```
                预测值
              森林  水体  城市
实际值 森林    85    5    10
       水体    3     92   5
       城市    7     3    90

总体精度 = (85+92+90)/300 = 89%
```

### 农业：从"看天吃饭"到"知天而作"

## 遥感大数据

### 遥感数据的"大"特点

**体量巨大**
- 一颗卫星每天产生TB级数据
- Landsat 50年积累的PB级数据

**类型多样**
- 多源：卫星、飞机、无人机
- 多分辨率：空间、光谱、时间
- 多格式：光学、雷达、激光

**价值密度低**
- 海量数据中，有价值的信息需要挖掘

### 遥感大数据技术栈

**数据存储**
- 云存储
- 分布式文件系统
- 数据立方（Data Cube）

**数据处理**
- 高性能计算
- GPU加速
- 云计算平台

**数据分析**
- 深度学习
- 时间序列分析
- 知识图谱

### Google Earth Engine：云端遥感

**革命性平台**
- 不需要下载数据
- 在云端处理
- 浏览器即可使用

**优势：**
- PB级数据在线
- 强大计算能力
- 免费使用（学术研究）

### 国产平台

- 航天宏图PIE-Engine
- 中科星图GEOVIS
- 阿里达摩院AI Earth
