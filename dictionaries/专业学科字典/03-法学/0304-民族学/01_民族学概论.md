### 一、民族的基本概念

#### 1. 民族 (Ethnic Group/民族)

**白话解释**:  
一群有共同语言、文化、历史记忆的人，觉得"我们是一家人"。

**生活例子**:  
- 汉族 - 说汉语，过春节
- 藏族 - 说藏语，过藏历新年
- 维吾尔族 - 说维吾尔语，过古尔邦节

**学术定义**:  
在历史上形成的具有共同语言、共同地域、共同经济生活以及表现于共同文化特点上的共同心理素质的人们共同体。

**关键要点**:
1. **共同语言** - 交流工具相同
2. **共同地域** - 生活区域相近
3. **共同经济生活** - 生产方式相似
4. **共同心理素质** - 文化认同一致

**费曼记忆**:  
民族 = 语言 + 地域 + 经济 + 心理

---

#### 2. 民族认同 (Ethnic Identity/民族认同)

**白话解释**:  
"我觉得我属于哪个民族"的心理归属感。

**生活例子**:  
- 小王身份证上写着"回族"，他觉得自己是回族人
- 小李从小在城市长大，父母是苗族，但她认同自己是汉族

**学术定义**:  
个体对自己所属民族群体的归属感和心理认同，包括对民族文化的接受和情感依附。

**认同层次**:
1. **认知层面** - 知道自己属于哪个民族
2. **情感层面** - 对本民族有感情
3. **行为层面** - 参与民族活动、传承民族文化

**影响因素**:
- 家庭教育
- 社会环境
- 文化接触
- 政策影响

**实务提示**:  
民族认同具有主观性，不能强制。

---

#### 3. 民族意识 (Ethnic Consciousness/民族意识)

**白话解释**:  
"我们是同一群人，跟别人不一样"的想法。

**生活例子**:  
- 蒙古族看到那达慕大会感到自豪
- 朝鲜族看到朝鲜族传统服装觉得亲切

**学术定义**:  
民族成员对自己民族归属、民族利益、民族特点的自我意识。

**表现形式**:
1. **归属意识** - 我是谁
2. **利益意识** - 我们的利益
3. **区别意识** - 我们和他们
4. **发展意识** - 我们的未来

**双重性**:
- 积极面：增强凝聚力，传承文化
- 消极面：可能导致民族冲突

**实务提示**:  
引导民族意识向积极方向发展。

---

#### 4. 民族特征 (Ethnic Characteristics/民族特征)

**白话解释**:  
一个民族区别于其他民族的特点。

**生活例子**:  
- 服装：蒙古袍、藏袍、维吾尔族花帽
- 饮食：蒙古族喝奶茶、藏族吃糌粑
- 节日：傣族泼水节、彝族火把节

**四大特征** (斯大林民族定义):
1. **共同语言** - 交流工具
2. **共同地域** - 生活空间
3. **共同经济生活** - 生产方式
4. **共同心理素质** - 文化认同

**中国实践**:
- 民族识别时综合考虑多种特征
- 不强制要求同时具备四大特征
- 尊重民族自我认同

**实务提示**:  
民族特征会随时代变化，不是一成不变的。

---

#### 5. 民族形成 (Ethnic Formation/民族形成)

**白话解释**:  
一个民族是怎么产生的。

**生活例子**:  
- 汉族经过几千年融合形成
- 满族从女真族演变而来

**形成途径**:
1. **氏族部落融合** - 古代民族形成
2. **地域融合** - 共同生活形成
3. **国家推动** - 政策引导形成
4. **分化重组** - 新民族诞生

**历史阶段**:
- 原始社会 → 氏族部落
- 奴隶社会 → 部落联盟
- 封建社会 → 民族初步形成
- 近现代 → 民族国家形成

**中国特点**:
- 多元一体格局
- 各民族长期交往交流交融

**实务提示**:  
民族形成是历史过程，不能人为强制。

### 二、民族学的基本理论

#### 6. 民族学 (Ethnology/民族学)

**白话解释**:  
专门研究各个民族的学科，研究他们的文化、历史、社会。

**生活例子**:  
- 民族学家去村里研究少数民族的风俗
- 记录濒临消失的民族语言

**学术定义**:  
研究民族的起源、发展、演变、分布、文化特点及相互关系的综合性学科。

**研究内容**:
1. **民族起源** - 民族从哪里来
2. **民族发展** - 民族怎么发展
3. **民族文化** - 各民族的文化特点
4. **民族关系** - 民族之间如何相处

**研究方法**:
- 田野调查 (到民族地区实地考察)
- 比较研究 (对比不同民族)
- 历史文献 (查阅史料)
- 跨学科研究 (结合考古、语言等)

**实务提示**:  
民族学研究要尊重民族感情，避免偏见。

---

#### 7. 田野调查 (Fieldwork/田野调查)

**白话解释**:  
到民族地区去实地看看，跟当地人聊天，观察他们的生活。

**生活例子**:  
- 民族学学生去云南少数民族村寨住一个月
- 记录老人的口述历史
- 参与当地节日活动

**学术定义**:  
研究者深入研究对象的生活环境，通过参与观察、访谈等方法收集第一手资料的研究方法。

**调查内容**:
1. **物质文化** - 房屋、服饰、工具
2. **精神文化** - 宗教、艺术、习俗
3. **社会结构** - 家庭、组织、制度
4. **语言文字** - 方言、文字、口头传说

**调查方法**:
- 参与观察 (融入当地生活)
- 深度访谈 (跟当地人聊天)
- 问卷调查 (收集数据)
- 影像记录 (拍照、录像)

**注意事项**:
- 尊重当地风俗
- 不干涉当地生活
- 保护隐私
- 征得同意

**实务提示**:  
田野调查是民族学的基本功，需要耐心和尊重。

---

#### 8. 文化相对主义 (Cultural Relativism/文化相对主义)

**白话解释**:  
不能用一个民族的标准去评价另一个民族，每个民族的文化都有自己的道理。

**生活例子**:  
- 不能说"吃牛肉是对的，吃猪肉是错的"
- 不能说"穿西装是文明的，穿民族服装是落后的"

**学术定义**:  
认为每种文化都有其独特价值和存在理由，应以该文化自身标准而非外部标准来理解和评价。

**核心观点**:
1. **文化平等** - 各民族文化平等
2. **避免偏见** - 不用自己标准评价他人
3. **理解背景** - 结合历史环境理解文化
4. **尊重差异** - 接受文化多样性

**理论意义**:
- 反对文化中心主义
- 促进跨文化理解
- 保护文化多样性

**中国实践**:
- 各民族一律平等
- 尊重各民族文化
- 不搞民族歧视

**实务提示**:  
理解不等于认同，但必须尊重。

---

#### 9. 民族中心主义 (Ethnocentrism/民族中心主义)

**白话解释**:  
觉得自己的民族最优秀，其他民族都不如自己。

**生活例子**:  
- "我们民族的饮食才是正宗的"
- "他们民族的习俗太落后了"

**学术定义**:  
以自己民族的文化标准去衡量和评价其他民族，认为本民族文化优于他民族的心理倾向。

**表现形式**:
1. **文化优越感** - 我们的文化最好
2. **文化偏见** - 对他民族文化有偏见
3. **文化歧视** - 看不起他民族文化
4. **文化排斥** - 拒绝接受他民族文化

**危害**:
- 导致民族冲突
- 破坏民族团结
- 阻碍文化交流
- 引发民族压迫

**克服方法**:
- 培养跨文化理解
- 学习文化相对主义
- 增进民族交流
- 树立平等观念

**实务提示**:  
民族中心主义是民族关系的大敌。

### 三、民族语言与文化

#### 10. 民族语言 (Ethnic Language/民族语言)

**白话解释**:  
一个民族使用的语言，是民族的重要标志。

**生活例子**:  
- 藏语 - 藏族使用
- 蒙古语 - 蒙古族使用
- 朝鲜语 - 朝鲜族使用

**学术定义**:  
特定民族群体共同使用、世代相传、具有独特语音、词汇、语法体系的语言系统。

**中国语言状况**:
- 56个民族
- 80多种语言 (部分民族使用相同语言)
- 29种文字

**语言分类**:
| 语系 | 民族 |
|------|------|
| 汉藏语系 | 汉族、藏族、壮族、苗族、瑶族等 |
| 阿尔泰语系 | 蒙古族、维吾尔族、哈萨克族、满族等 |
| 南亚语系 | 佤族、布朗族、德昂族等 |
| 印欧语系 | 塔吉克族、俄罗斯族等 |
| 南岛语系 | 高山族等 |

**语言功能**:
1. **交流工具** - 族内交流
2. **文化载体** - 传承文化
3. **认同标志** - 民族认同
4. **思维工具** - 认识世界

**保护政策**:
- 双语教育
- 濒危语言保护
- 语言文字规范化

**实务提示**:  
语言是民族的核心特征，要尊重和保护。

---

#### 11. 民族文化 (Ethnic Culture/民族文化)

**白话解释**:  
一个民族世代相传的生活方式、价值观念、风俗习惯等。

**生活例子**:  
- 蒙古族：住蒙古包，骑马，摔跤
- 傣族：泼水节，孔雀舞，竹楼
- 藏族：转经，哈达，酥油茶

**学术定义**:  
民族群体在长期历史发展中创造、积累、传承的物质财富和精神财富的总和。

**文化层次**:
```
民族文化
├─ 物质文化
│   ├─ 生产工具
│   ├─ 生活用品
│   └─ 服饰建筑
├─ 制度文化
│   ├─ 社会组织
│   ├─ 婚姻制度
│   └─ 礼仪规范
└─ 精神文化
    ├─ 宗教信仰
    ├─ 价值观念
    └─ 文学艺术
```

**文化特点**:
1. **民族性** - 各民族独特
2. **传承性** - 代代相传
3. **稳定性** - 相对稳定
4. **变异性** - 随时代变化

**保护传承**:
- 非物质文化遗产保护
- 文化产业发展
- 文化教育传承
- 文化数字化保存

**实务提示**:  
民族文化是民族的生命力，要保护更要创新。

---

#### 12. 民族习俗 (Ethnic Customs/民族习俗)

**白话解释**:  
一个民族约定俗成的习惯，包括婚丧嫁娶、饮食起居、节庆活动等。

**生活例子**:  
- 藏族献哈达表示敬意
- 蒙古族喝奶茶待客
- 维吾尔族歌舞迎宾

**学术定义**:  
民族群体在长期社会生活中形成并世代传承的风俗习惯和行为规范。

**习俗分类**:
1. **生活习俗**
   - 饮食：清真、素食、忌口
   - 服饰：蒙古袍、旗袍、藏袍
   - 居住：蒙古包、竹楼、窑洞

2. **人生礼仪**
   - 出生：满月、百日
   - 婚嫁：婚礼仪式
   - 丧葬：天葬、土葬、火葬

3. **节庆习俗**
   - 春节 (汉族)
   - 泼水节 (傣族)
   - 古尔邦节 (维吾尔族等)
   - 藏历新年 (藏族)

4. **禁忌习俗**
   - 饮食禁忌
   - 行为禁忌
   - 语言禁忌

**习俗功能**:
- 维系群体认同
- 规范社会行为
- 传承文化记忆
- 增强凝聚力

**实务提示**:  
尊重民族习俗，不随便评价或嘲笑。

### 四、民族社会与结构

#### 13. 民族社会 (Ethnic Society/民族社会)

**白话解释**:  
以民族为单位的社会群体，有自己的社会组织和社会关系。

**生活例子**:  
- 藏族村寨的互助传统
- 维吾尔族的社区生活
- 蒙古族的部落组织

**学术定义**:  
以民族为基本单位构成的社会群体，具有特定的社会组织结构、社会关系和社会制度。

**社会特点**:
1. **共同性** - 文化、语言相同
2. **稳定性** - 成员关系稳定
3. **封闭性** - 对外相对封闭
4. **传承性** - 代际传承

**社会组织形式**:
| 类型 | 特点 | 例子 |
|------|------|------|
| 氏族 | 血缘关系 | 部落社会 |
| 村寨 | 地缘关系 | 少数民族村寨 |
| 宗教 | 宗教信仰 | 清真寺社区 |
| 现代 | 法律关系 | 城市民族社区 |

**社会变迁**:
- 传统社会 → 现代社会
- 封闭社会 → 开放社会
- 农业社会 → 工业/信息社会

**实务提示**:  
民族社会在现代化进程中面临转型挑战。

---

#### 14. 民族社区 (Ethnic Community/民族社区)

**白话解释**:  
民族成员聚居的区域，大家有共同文化和生活方式。

**生活例子**:  
- 新疆的维吾尔族聚居区
- 西藏的藏族村寨
- 城市的民族街区

**学术定义**:  
以特定民族成员为主体，在特定地域范围内形成的社会共同体，具有共同的民族文化认同和生活方式。

**社区类型**:
1. **传统社区** - 农村民族村寨
2. **城市社区** - 城市民族街区
3. **混合社区** - 多民族聚居区
4. **流动社区** - 外来民族群体

**社区功能**:
1. **居住功能** - 提供生活空间
2. **文化功能** - 传承民族文化
3. **社会功能** - 维系社会关系
4. **经济功能** - 发展民族经济

**社区建设**:
- 基础设施改善
- 文化设施建设
- 社区服务完善
- 民族团结促进

**实务提示**:  
城市民族社区面临文化传承和发展困境。

---

#### 15. 民族家庭 (Ethnic Family/民族家庭)

**白话解释**:  
民族成员组成的基本社会单位，有自己的家庭制度和习俗。

**生活例子**:  
- 藏族大家庭三代同堂
- 摩梭族的母系家庭
- 维吾尔族的核心家庭

**学术定义**:  
以婚姻、血缘或收养关系为基础，以民族文化为纽带的初级社会群体。

**家庭类型**:
1. **核心家庭** - 父母+子女
2. **主干家庭** - 祖父母+父母+子女
3. **联合家庭** - 兄弟姐妹+配偶+子女
4. **特殊家庭** - 摩梭族走婚家庭

**婚姻习俗**:
| 民族 | 婚姻形式 |
|------|----------|
| 汉族 | 一夫一妻 |
| 藏族 | 一夫一妻为主 |
| 摩梭族 | 走婚 |
| 部分民族 | 历史上有一夫多妻 |

**家庭功能**:
1. **生育功能** - 人口再生产
2. **经济功能** - 生产消费单位
3. **教育功能** - 文化传承
4. **情感功能** - 情感支持

**现代变迁**:
- 家庭规模小型化
- 婚姻观念现代化
- 性别平等提升
- 代际关系变化

**实务提示**:  
尊重民族家庭习俗，但不符合法律的要依法纠正。

---

#### 16. 民族社会组织 (Ethnic Social Organization/民族社会组织)

**白话解释**:  
民族社会中的各种组织形式，比如村寨组织、宗族组织、宗教组织等。

**生活例子**:  
- 彝族的"家支"组织
- 藏族的寺院组织
- 苗族的"议榔"组织

**学术定义**:  
民族社会内部形成的具有特定功能、规范和成员的组织体系，承担社会管理、文化传承等功能。

**组织类型**:
1. **血缘组织** - 宗族、家支、氏族
2. **地缘组织** - 村寨、部落
3. **宗教组织** - 寺院、教会
4. **现代组织** - 协会、合作社

**传统社会组织**:
| 组织类型 | 民族 | 功能 |
|----------|------|------|
| 家支 | 彝族 | 互助、调解纠纷 |
| 议榔 | 苗族 | 制定规约、维护秩序 |
| 寺院 | 藏族 | 宗教、教育、社会服务 |
| 比大 | 哈尼族 | 社区管理 |

**现代转型**:
- 传统组织功能弱化
- 现代组织逐步建立
- 村民自治制度推广
- 社会组织发展

**实务提示**:  
发挥传统组织积极作用，引导其与现代制度对接。

### 五、民族发展问题

#### 17. 民族发展 (Ethnic Development/民族发展)

**白话解释**:  
民族在经济、文化、社会各方面的进步和提高。

**生活例子**:  
- 民族地区脱贫致富
- 少数民族教育水平提高
- 民族文化走向世界

**学术定义**:  
民族群体在经济、政治、文化、社会等方面取得的全面进步，表现为生产力提高、文化繁荣、社会进步。

**发展内容**:
1. **经济发展** - 生产力提高，生活改善
2. **政治发展** - 民族平等，当家作主
3. **文化发展** - 文化繁荣，传承创新
4. **社会发展** - 教育医疗，社会保障
5. **生态发展** - 环境保护，可持续发展

**发展指标**:
| 指标 | 内容 |
|------|------|
| 经济 | 人均收入、产业结构 |
| 教育 | 识字率、受教育年限 |
| 健康 | 预期寿命、医疗条件 |
| 文化 | 文化设施、文化传承 |
| 政治 | 参政比例、民族干部 |

**发展挑战**:
1. **发展不平衡** - 区域差异大
2. **人才缺乏** - 高层次人才少
3. **文化传承** - 传统与现代冲突
4. **生态保护** - 发展与保护矛盾

**实务提示**:  
民族发展要尊重民族意愿，不能搞一刀切。

#### 18. 民族地区 (Ethnic Region/民族地区)

**白话解释**:  
少数民族聚居的地区，包括自治区、自治州、自治县等。

**生活例子**:  
- 西藏自治区
- 新疆维吾尔自治区
- 内蒙古自治区
- 凉山彝族自治州

**学术定义**:  
少数民族相对集中居住，依法实行民族区域自治的行政区域。

**地区类型**:
| 级别 | 数量 | 例子 |
|------|------|------|
| 自治区 | 5个 | 西藏、新疆、内蒙古、宁夏、广西 |
| 自治州 | 30个 | 凉山、甘孜、临夏等 |
| 自治县 | 120个 | 各省民族自治县 |
| 民族乡 | 1000+ | 各地民族乡 |

**地区特点**:
1. **地域广阔** - 占国土64%
2. **资源丰富** - 矿产、能源、旅游
3. **生态重要** - 生态屏障
4. **发展滞后** - 经济相对落后
5. **文化多样** - 多民族聚居

**发展策略**:
- 西部大开发
- 精准扶贫
- 乡村振兴
- 对口支援

**实务提示**:  
民族地区发展关系到国家统一和民族团结。

#### 19. 民族经济 (Ethnic Economy/民族经济)

**白话解释**:  
民族地区和民族群体的经济活动，包括传统经济和现代产业。

**生活例子**:  
- 蒙古族的畜牧业
- 维吾尔族的棉花种植
- 藏族的旅游业
- 苗族的手工艺品

**学术定义**:  
在特定民族地区或民族群体中形成的经济体系，包括生产方式、经济结构、经济关系等。

**传统经济**:
1. **游牧经济** - 蒙古族、哈萨克族
2. **农耕经济** - 壮族、傣族
3. **狩猎采集** - 鄂伦春族 (历史上)
4. **渔业经济** - 京族、赫哲族

**现代产业**:
| 产业类型 | 例子 |
|----------|------|
| 特色农业 | 棉花、水果、药材 |
| 畜牧业 | 牛羊肉、奶制品 |
| 旅游业 | 民族风情游、生态游 |
| 手工业 | 地毯、刺绣、银饰 |
| 能源产业 | 石油、天然气、水电 |

**发展模式**:
1. **特色产业** - 发挥资源优势
2. **文化旅游** - 文化资源变现
3. **生态经济** - 绿色发展
4. **跨境贸易** - 沿边优势

**发展挑战**:
- 产业结构单一
- 市场化程度低
- 人才资金缺乏
- 生态压力大

**实务提示**:  
发展民族经济要兼顾经济效益和生态保护。

#### 20. 民族教育 (Ethnic Education/民族教育)

**白话解释**:  
针对少数民族的教育，包括双语教育、民族学校、民族班等。

**生活例子**:  
- 藏族学生学藏语和汉语
- 新疆的双语学校
- 内地新疆班、西藏班

**学术定义**:  
针对少数民族特点和需求，在民族地区或面向少数民族学生开展的教育活动，包括双语教育、民族文化教育等。

**教育形式**:
1. **双语教育** - 民族语言+国家通用语言
2. **民族学校** - 民族中小学、民族高校
3. **民族班** - 内地民族班、预科班
4. **成人教育** - 农民工培训、扫盲教育

**教育政策**:
| 政策 | 内容 |
|------|------|
| 优惠政策 | 高考加分、定向招生 |
| 双语教育 | 保护母语、学习通用语言 |
| 寄宿制 | 解决偏远地区教育问题 |
| 师资培养 | 民族地区教师培训 |

**教育成就**:
- 适龄儿童入学率大幅提高
- 人均受教育年限增加
- 大学生比例上升
- 女性教育水平提升

**面临挑战**:
1. **师资不足** - 高水平教师缺乏
2. **质量不高** - 教育质量待提升
3. **流失严重** - 人才外流
4. **语言平衡** - 双语教育如何平衡

**实务提示**:  
教育是民族发展的根本，要优先保障。

#### 21. 民族人口 (Ethnic Population/民族人口)

**白话解释**:  
各民族的人口数量、分布、结构等情况。

**生活例子**:  
- 汉族人口最多，13亿+
- 壮族是人口最多的少数民族，1800万+
- 塔塔尔族人口最少，不到1万

**学术定义**:  
特定民族群体的人口数量、分布、结构及其变动规律，包括人口数量、性别年龄结构、人口流动等。

**中国民族人口 (2020年第七次人口普查)**:
| 排名 | 民族 | 人口 |
|------|------|------|

| 3 | 维吾尔族 | 1177万 |
| 4 | 回族 | 1137万 |
| 5 | 苗族 | 1106万 |

**人口特点**:
1. **汉族人口最多** - 占91.1%

**人口问题**:
- 人口分布不均
- 人口流动加剧
- 老龄化趋势
- 男女比例失衡

**人口政策**:
- 计划生育 (已放开)
- 鼓励生育 (部分地区)
- 人口迁移管理
- 户籍制度改革

**实务提示**:  
人口变化影响民族发展和民族关系。

#### 22. 民族流动 (Ethnic Migration/民族流动)

**白话解释**:  
少数民族成员从民族地区流动到其他地方，主要是进城务工。

**生活例子**:  
- 维吾尔族商人在全国各地开店
- 藏族学生在内地读书
- 少数民族农民工在城市工作

**学术定义**:  
民族成员在地域间的移动和重新分布，包括从农村到城市、从民族地区到内地、从国内到国外的人口流动。

**流动类型**:
1. **经济流动** - 务工经商
2. **教育流动** - 外出求学
3. **婚姻流动** - 婚嫁迁移
4. **政策流动** - 搬迁安置

**流动特点**:
| 特点 | 表现 |
|------|------|
| 规模大 | 数千万少数民族流动人口 |
| 方向明确 | 主要流向东部沿海城市 |
| 年轻化 | 以青壮年为主 |
| 季节性 | 部分呈季节性流动 |

**流动影响**:
**积极影响**:
- 促进经济发展
- 增进民族交流
- 提高收入水平
- 开阔眼界

**消极影响**:
- 文化适应困难
- 民族地区人才流失
- 城市管理压力
- 族群关系复杂

**应对措施**:
- 城市民族工作
- 语言文化培训
- 法律援助服务
- 社区融入支持

**实务提示**:  
民族流动是现代化进程的必然，要做好服务和管理。

- 各民族共同繁荣

#### 25. 民族冲突 (Ethnic Conflict/民族冲突)

**白话解释**:  
不同民族之间的矛盾激化，发生对抗甚至暴力事件。

**生活例子**:  
- 乌鲁木齐"7·5"事件 (2009年)
- 民族地区发生的群体性事件
- 民族之间的纠纷冲突

**学术定义**:  
不同民族群体之间因利益、文化、宗教、领土等问题引发的矛盾对抗，表现为语言冲突、群体冲突、武装冲突等形式。

**冲突类型**:
1. **利益冲突** - 资源争夺、就业竞争
2. **文化冲突** - 文化差异、宗教矛盾
3. **权力冲突** - 政治权力、自治要求
4. **认同冲突** - 身份认同、分离倾向

**冲突原因**:
| 原因类型 | 具体原因 |
|----------|----------|
| 经济 | 发展差距、利益分配 |
| 政治 | 权力分配、自治诉求 |
| 文化 | 文化差异、宗教矛盾 |
| 历史 | 历史积怨、历史记忆 |
| 外部 | 外部势力、国际影响 |

**预防措施**:
1. **经济发展** - 缩小差距，共同富裕
2. **政治参与** - 平等权利，当家作主
3. **文化交流** - 增进理解，消除偏见
4. **法治保障** - 公平正义，依法处理
5. **教育引导** - 团结教育，消除极端

**处置原则**:
- 依法处置
- 区别对待
- 教育为主
- 防止扩大

**实务提示**:  
民族冲突危害极大，要以预防为主，依法处置。

#### 26. 民族分裂主义 (Ethnic Separatism/民族分裂主义)

**白话解释**:  
想把民族地区从国家分裂出去的思想和行为。

**生活例子**:  
- "东突"分裂势力
- "藏独"分裂势力
- 宣扬民族独立的活动

**学术定义**:  
以民族自决为名，企图将特定民族地区从多民族国家分裂出去，建立独立民族国家的政治主张和行动。

**表现形式**:
1. **意识形态** - 宣扬民族独立
2. **政治活动** - 分裂组织、国际游说
3. **暴力恐怖** - 恐怖袭击、武装冲突
4. **外部勾连** - 勾结外部势力

**中国面临的分裂主义**:
| 分裂势力 | 目标地区 | 主要活动 |
|----------|----------|----------|
| 东突 | 新疆 | 恐怖袭击、分裂宣传 |
| 藏独 | 西藏 | 分裂活动、国际游说 |
| 台独 | 台湾 | 分裂政权、国际活动 |

**危害**:
1. **国家统一** - 威胁国家主权
2. **民族团结** - 破坏民族团结
3. **社会稳定** - 引发社会动荡
4. **经济发展** - 阻碍经济发展
5. **人民安全** - 威胁人民生命

**应对措施**:
- 依法打击
- 发展经济
- 改善民生
- 文化认同
- 国际合作

**实务提示**:  
民族分裂主义是国家安全的重大威胁。

### 六、民族研究方法

#### 27. 民族识别 (Ethnic Identification/民族识别)

**白话解释**:  
政府认定哪些群体是独立的民族，给他们正式的民族身份。

**生活例子**:  
- 新中国初期，有400多个群体申请民族身份
- 经过识别，确定56个民族
- 有些群体没有被认定为独立民族

**学术定义**:  
国家依法对国内各民族群体的民族成分进行认定和划分的过程，确定其是否构成独立民族。

**识别标准**:
1. **民族特征** - 语言、文化、历史
2. **民族意愿** - 群体自我认同
3. **历史渊源** - 历史发展脉络
4. **相近程度** - 与相近民族的比较

**识别过程** (1950-1979):
| 阶段 | 时间 | 成果 |
|------|------|------|
| 第一阶段 | 1950-1954 | 识别38个民族 |
| 第二阶段 | 1954-1965 | 识别16个民族 |
| 第三阶段 | 1965-1979 | 识别2个民族 |
| 总计 | | 56个民族 |

**识别原则**:
- 科学性与政策性结合
- 民族特征与民族意愿结合
- 历史与现实结合
- 尊重科学，尊重意愿

**遗留问题**:
- 未识别民族群体 (如穿青人、夏尔巴人)
- 民族成分变更
- 民族认同争议

**实务提示**:  
民族识别是历史性工作，要尊重科学和事实。

#### 28. 民族志 (Ethnography/民族志)

**白话解释**:  
详细记录一个民族的文化、社会、生活的文字或影像资料。

**生活例子**:  
- 《苗族社会调查报告》
- 《藏族文化志》
- 纪录片《舌尖上的中国》中的民族饮食

**学术定义**:  
通过田野调查等方法，对一个民族或社区的文化、社会、生活进行系统描述和记录的研究成果。

**民族志内容**:
1. **自然环境** - 地理位置、气候特点
2. **历史沿革** - 民族起源、历史变迁
3. **社会结构** - 家庭、宗族、社会组织
4. **经济生活** - 生产方式、经济活动
5. **物质文化** - 服饰、饮食、建筑
6. **精神文化** - 宗教、艺术、习俗
7. **语言文字** - 语言特点、文字系统

**民族志类型**:
| 类型 | 特点 |
|------|------|
| 传统民族志 | 全面系统描述 |
| 专题民族志 | 专注某一领域 |
| 影像民族志 | 以影像为主 |
| 数字民族志 | 数字化记录 |

**民族志价值**:
- 学术研究资料
- 文化保护记录
- 政策制定参考
- 文化传承载体

**写作原则**:
- 客观真实
- 深入细致
- 尊重文化
- 保护隐私

**实务提示**:  
民族志是民族学研究的核心成果形式。

#### 29. 跨文化比较 (Cross-cultural Comparison/跨文化比较)

**白话解释**:  
对比不同民族的文化，找出相同和不同的地方。

**生活例子**:  
- 对比汉族春节和藏族藏历新年
- 对比蒙古族和哈萨克族的游牧文化
- 对比不同民族的婚姻习俗

**学术定义**:  
对不同民族或文化的相同领域进行系统对比，发现文化差异和共性，揭示文化规律的研究方法。

**比较类型**:
1. **共时比较** - 同一时期不同民族
2. **历时比较** - 同一民族不同时期
3. **区域比较** - 不同区域民族
4. **专题比较** - 特定文化领域

**比较内容**:
| 比较领域 | 比较内容 |
|----------|----------|
| 家庭制度 | 婚姻形式、家庭结构 |
| 经济制度 | 生产方式、分配制度 |
| 政治制度 | 社会组织、权力结构 |
| 宗教信仰 | 信仰体系、仪式活动 |
| 艺术表现 | 音乐、舞蹈、绘画 |

**比较方法**:
1. **个案比较** - 选择典型案例
2. **统计比较** - 数据统计分析
3. **类型比较** - 按类型分类比较
4. **全球比较** - 全球范围比较

**注意事项**:
- 避免民族中心主义
- 结合历史背景
- 注意文化语境
- 避免简单类比

**实务提示**:  
跨文化比较要有理论指导，不能为比较而比较。

#### 30. 民族学理论 (Ethnological Theory/民族学理论)

**白话解释**:  
解释民族现象、民族发展规律的理论体系。

**生活例子**:  
- 解释为什么有些民族发展快，有些慢
- 解释民族冲突为什么发生
- 解释民族文化如何演变

**学术定义**:  
对民族现象、民族关系、民族发展等进行系统解释的理论体系，包括民族形成理论、民族关系理论、民族文化理论等。

**主要理论流派**:
1. **进化论学派**
   - 认为所有文化都经历相同发展阶段
   - 代表人物：泰勒、摩尔根

2. **传播论学派**
   - 认为文化传播是文化发展的主要动力
   - 代表人物：格雷布纳、施密特

3. **功能主义学派**
   - 认为文化各要素有其社会功能
   - 代表人物：马林诺夫斯基、拉德克利夫-布朗

4. **结构主义学派**
   - 认为文化有深层结构
   - 代表人物：列维-施特劳斯

5. **马克思主义民族理论**
   - 从经济基础和阶级关系分析民族问题
   - 中国民族学的主流理论

**中国民族学理论**:
- 马克思主义民族理论指导
- 中华民族共同体理论
- 民族区域自治理论
- 民族发展理论

**实务提示**:  
理论指导实践，要学习多种理论，但要立足中国实际。
