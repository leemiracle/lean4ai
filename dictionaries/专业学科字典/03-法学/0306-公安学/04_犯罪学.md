### 一、犯罪基本概念

#### 1. 犯罪 (Crime/犯罪)

**白话解释**:  
做了坏事,违反刑法,要受到刑事处罚。

**生活例子**:  
- 杀人
- 抢劫
- 盗窃
- 诈骗

**学术定义**:  
违反刑法规定,危害社会,依法应当受到刑罚处罚的行为。

**犯罪特征**:
1. **社会危害性** - 危害社会利益
2. **刑事违法性** - 违反刑法规定
3. **应受刑罚处罚性** - 应当受刑罚处罚

**犯罪构成**:
| 要件 | 内容 |
|------|------|
| 犯罪主体 | 实施犯罪的人 |
| 犯罪主观方面 | 故意或过失 |
| 犯罪客体 | 被侵害的法益 |
| 犯罪客观方面 | 危害行为和结果 |

**犯罪分类**:
- 按客体: 危害国家安全罪、危害公共安全罪等
- 按主体: 自然人犯罪、单位犯罪
- 按主观: 故意犯罪、过失犯罪

**实务提示**:  
犯罪是刑法规定的严重违法行为。

---

#### 2. 犯罪现象 (Criminal Phenomenon/犯罪现象)

**白话解释**:  
社会上存在的犯罪情况和表现。

**生活例子**:  
- 犯罪率上升或下降
- 某类犯罪增多
- 犯罪手段变化

**学术定义**:  
一定时期、一定地区内犯罪的整体状况、表现形态和发展变化规律。

**现象内容**:
1. **犯罪状况** - 犯罪的数量、类型、分布
2. **犯罪特点** - 犯罪的主要特征
3. **犯罪动态** - 犯罪的发展变化
4. **犯罪结构** - 各类犯罪的比例

**现象特征**:
| 特征 | 内容 |
|------|------|
| 客观性 | 客观存在 |
| 历史性 | 不同历史时期不同 |
| 社会性 | 受社会因素影响 |
| 区域性 | 不同地区不同 |

**研究内容**:
- 犯罪率
- 犯罪类型分布
- 犯罪人群特征
- 犯罪时间空间分布

**实务提示**:  
了解犯罪现象是犯罪预防的基础。

---

#### 3. 犯罪率 (Crime Rate/犯罪率)

**白话解释**:  
犯罪数量的比例。

**生活例子**:  
- 每万人中犯罪的人数
- 犯罪案件数与人口的比例

**学术定义**:  
一定时期内,一定地区中犯罪案件数或犯罪人数与人口总数的比率,是衡量犯罪状况的重要指标。

**计算方法**:
- **犯罪发案率** = 犯罪案件数 / 人口总数 × 100000
- **犯罪人率** = 犯罪人数 / 人口总数 × 100000

**犯罪率类型**:
| 类型 | 内容 |
|------|------|
| 总犯罪率 | 所有犯罪的比例 |
| 类型犯罪率 | 某类犯罪的比例 |
| 年龄犯罪率 | 某年龄段犯罪比例 |
| 地区犯罪率 | 某地区犯罪比例 |

**影响因素**:
- 经济发展水平
- 社会管理能力
- 文化道德水平
- 法律制度完善程度

**注意事项**:
- 犯罪黑数 (未发现的犯罪)
- 立案标准变化
- 统计口径差异

**实务提示**:  
犯罪率是衡量治安状况的重要指标。

---

#### 4. 犯罪黑数 (Dark Figure of Crime/犯罪黑数)

**白话解释**:  
实际发生了但没被发现的犯罪。

**生活例子**:  
- 没报警的盗窃
- 受害人不敢报案的性侵
- 没被发现的腐败

**学术定义**:  
实际发生但没有被公安机关发现、立案、统计的犯罪数量,是实际犯罪数与已知犯罪数的差额。

**黑数原因**:
1. **受害人不愿报案** - 怕报复、怕麻烦
2. **受害人不知情** - 不知道自己受害
3. **公安机关未发现** - 隐蔽性强
4. **立案标准问题** - 不够立案标准
5. **统计问题** - 统计遗漏

**黑数特点**:
| 犯罪类型 | 黑数大小 |
|------|----------|
| 暴力犯罪 | 较小 |
| 性犯罪 | 较大 |
| 腐败犯罪 | 较大 |
| 盗窃犯罪 | 中等 |

**估算方法**:
- 被害调查
- 自报调查
- 专家估算

**研究意义**:
- 了解真实犯罪状况
- 改进犯罪统计
- 优化犯罪对策

**实务提示**:  
犯罪黑数影响对犯罪状况的准确判断。

---

#### 5. 犯罪构成 (Constitutive Elements of Crime/犯罪构成)

**白话解释**:  
判断一个行为是不是犯罪的标准。

**生活例子**:  
- 有人实施了危害行为
- 主观上是故意的
- 侵犯了法律保护的利益
- 造成了危害结果

**学术定义**:  
刑法规定的,决定某一具体行为的社会危害性及其程度,而为该行为构成犯罪所必需的一切主观要件和客观要件的有机统一。

**构成要件**:
1. **犯罪主体** - 实施犯罪的人
   - 自然人
   - 单位

2. **犯罪主观方面** - 心理态度
   - 故意 (直接故意、间接故意)
   - 过失 (疏忽大意、过于自信)

3. **犯罪客体** - 被侵害的法益
   - 国家安全
   - 公共安全
   - 人身权利
   - 财产权利

4. **犯罪客观方面** - 危害行为和结果
   - 危害行为
   - 危害结果
   - 因果关系

**判断标准**:
| 情况 | 判断 |
|------|------|
| 四要件齐备 | 构成犯罪 |
| 缺一要件 | 不构成犯罪 |

**实务意义**:
- 区分罪与非罪
- 区分此罪与彼罪
- 决定刑罚轻重

**实务提示**:  
犯罪构成是定罪的基础。

### 二、犯罪学理论

#### 6. 犯罪学 (Criminology/犯罪学)

**白话解释**:  
研究犯罪为什么发生,怎么预防的学科。

**生活例子**:  
- 分析犯罪原因
- 研究犯罪规律
- 提出预防对策

**学术定义**:  
研究犯罪现象、犯罪原因、犯罪规律和犯罪对策的综合性学科,涉及社会学、心理学、法学等多个领域。

**研究内容**:
1. **犯罪现象** - 犯罪的状况和特点
2. **犯罪原因** - 为什么会犯罪
3. **犯罪规律** - 犯罪的变化规律
4. **犯罪对策** - 如何预防和控制犯罪

**研究方法**:
| 方法 | 内容 |
|------|------|
| 统计分析 | 犯罪数据分析 |
| 案例研究 | 典型案例剖析 |
| 比较研究 | 不同地区比较 |
| 实证调查 | 实地调查研究 |
| 理论研究 | 理论建构分析 |

**学科特点**:
- 多学科交叉
- 理论与实践结合
- 宏观与微观结合

**实务提示**:  
犯罪学为犯罪预防和控制提供理论指导。

---

#### 7. 犯罪原因 (Causes of Crime/犯罪原因)

**白话解释**:  
人为什么会犯罪。

**生活例子**:  
- 贫穷导致盗窃
- 心理问题导致暴力
- 不良环境影响

**学术定义**:  
导致犯罪行为发生的各种因素的总和,包括社会因素、心理因素、生物因素、文化因素等。

**原因分类**:
1. **社会原因**
   - 贫困
   - 失业
   - 教育缺失
   - 社会不公

2. **个体原因**
   - 心理异常
   - 道德缺失
   - 法律意识淡薄
   - 性格缺陷

3. **文化原因**
   - 暴力文化
   - 拜金主义
   - 道德沦丧

4. **情境原因**
   - 犯罪机会
   - 被害人因素
   - 环境诱导

**原因特点**:
| 特点 | 内容 |
|------|------|
| 综合性 | 多种因素综合作用 |
| 层次性 | 有主次之分 |
| 动态性 | 随时代变化 |
| 交互性 | 相互影响 |

**实务提示**:  
了解犯罪原因有助于预防犯罪。

---

#### 8. 犯罪社会学 (Criminal Sociology/犯罪社会学)

**白话解释**:  
从社会角度研究犯罪。

**生活例子**:  
- 社会环境对犯罪的影响
- 社会结构变化与犯罪的关系

**学术定义**:  
运用社会学的理论和方法研究犯罪现象、犯罪原因和犯罪规律的学科,是犯罪学的重要分支。

**主要理论**:
1. **社会结构理论**
   - 贫困与犯罪
   - 社会分层与犯罪
   - 失范理论

2. **社会化理论**
   - 家庭教育与犯罪
   - 学校教育与犯罪
   - 同伴影响

3. **社会控制理论**
   - 社会约束力减弱
   - 人际关系疏离
   - 社会纽带断裂

**研究重点**:
| 重点 | 内容 |
|------|------|
| 社会因素 | 贫困、失业、教育 |
| 社会环境 | 家庭、学校、社区 |
| 社会结构 | 分层、流动、变迁 |
| 社会控制 | 法律、道德、舆论 |

**中国实践**:
- 社会转型与犯罪
- 城市化与犯罪
- 社会治理与犯罪预防

**实务提示**:  
社会因素是影响犯罪的重要原因。

---

#### 9. 犯罪心理学 (Criminal Psychology/犯罪心理学)

**白话解释**:  
从心理角度研究犯罪。

**生活例子**:  
- 分析犯罪人的心理
- 研究犯罪动机
- 心理画像

**学术定义**:  
运用心理学的理论和方法研究犯罪人的心理活动、心理特征和犯罪行为的学科。

**研究内容**:
1. **犯罪心理结构** - 犯罪人的心理特征
2. **犯罪动机** - 为什么想犯罪
3. **犯罪人格** - 犯罪人的性格特征
4. **犯罪决策** - 如何决定犯罪

**心理类型**:
| 类型 | 特点 |
|------|------|
| 反社会人格 | 缺乏同情心、道德感 |
| 暴力倾向 | 易冲动、好攻击 |
| 成瘾性人格 | 易形成各种成瘾 |
| 低自我控制 | 冲动、缺乏规划 |

**心理原因**:
- 童年创伤
- 心理障碍
- 认知偏差
- 情绪失控

**实务提示**:  
了解犯罪心理有助于犯罪预防和矫正。

#### 10. 标签理论 (Labeling Theory/标签理论)

**白话解释**:  
被人贴上"坏人"标签后,真的会变成坏人。

**生活例子**:  
- 被贴上"小偷"标签后继续偷
- 被判刑后难以融入社会又犯罪

**学术定义**:  
认为犯罪是社会对某些行为的反应和定义,当一个人被贴上"犯罪人"标签后,可能形成犯罪认同,继续实施犯罪。

**核心观点**:
1. **犯罪是社会定义** - 犯罪是社会定义的
2. **标签效应** - 标签影响自我认同
3. **次级越轨** - 被标签后继续犯罪

**标签过程**:
| 阶段 | 内容 |
|------|------|
| 初级越轨 | 第一次违规 |
| 被发现 | 被社会发现 |
| 被标签 | 被贴上标签 |
| 接受标签 | 内化标签 |
| 次级越轨 | 继续犯罪 |

**启示意义**:
- 避免轻易贴标签
- 注重教育挽救
- 帮助融入社会
- 减少污名化

**实务提示**:  
标签效应提醒我们谨慎对待初犯和轻微犯罪。

#### 11. 紧张理论 (Strain Theory/紧张理论)

**白话解释**:  
想得到的东西得不到,压力太大就去犯罪。

**生活例子**:  
- 穷人想发财却没机会,就去抢劫
- 想成功却处处碰壁,就走歪门邪道

**学术定义**:  
认为犯罪是由于社会文化目标与合法手段之间的断裂造成的,当个人无法通过合法手段实现目标时,可能选择犯罪手段。

**理论内容**:
1. **文化目标** - 社会鼓励的目标 (财富、成功)
2. **制度手段** - 合法实现目标的途径
3. **目标与手段断裂** - 目标难以通过合法手段实现
4. **紧张压力** - 产生心理压力
5. **犯罪选择** - 选择犯罪手段

**适应方式**:
| 方式 | 内容 |
|------|------|
| 顺从 | 接受目标和手段 |
| 创新 | 接受目标,拒绝手段 (犯罪) |
| 形式主义 | 拒绝目标,接受手段 |
| 退缩 | 拒绝目标和手段 |
| 反叛 | 用新目标和新手段 |

**启示意义**:
- 创造公平机会
- 缩小贫富差距
- 提供合法途径
- 缓解社会压力

**实务提示**:  
紧张理论强调社会公平对预防犯罪的重要性。

### 三、常见犯罪类型

#### 12. 暴力犯罪 (Violent Crime/暴力犯罪)

**白话解释**:  
用暴力手段实施的犯罪。

**生活例子**:  
- 杀人
- 伤害
- 抢劫
- 强奸

**学术定义**:  
以暴力或暴力威胁为手段,侵害他人人身权利和财产权利的犯罪行为。

**犯罪类型**:
1. **杀人犯罪** - 故意杀人、过失致死
2. **伤害犯罪** - 故意伤害、过失伤害
3. **抢劫犯罪** - 抢劫、抢夺
4. **性犯罪** - 强奸、猥亵

**犯罪特点**:
| 特点 | 内容 |
|------|------|
| 暴力性 | 使用暴力手段 |
| 危害性 | 严重危害人身安全 |
| 突发性 | 常常突然发生 |
| 情感性 | 常与情感纠纷有关 |

**犯罪原因**:
- 情感纠纷
- 利益冲突
- 性格暴躁
- 心理障碍
- 酒精毒品

**预防措施**:
- 矛盾纠纷调解
- 危险物品管控
- 重点人员管控
- 社会心理服务

**实务提示**:  
暴力犯罪危害极大,要重点预防。

---

#### 13. 财产犯罪 (Property Crime/财产犯罪)

**白话解释**:  
冲着钱去的犯罪。

**生活例子**:  
- 盗窃
- 诈骗
- 抢劫
- 贪污

**学术定义**:  
以非法占有为目的,侵害公私财产权利的犯罪行为。

**犯罪类型**:
1. **盗窃犯罪** - 盗窃、入室盗窃
2. **诈骗犯罪** - 电信诈骗、网络诈骗
3. **抢劫犯罪** - 抢劫、抢夺
4. **职务侵占** - 贪污、挪用、侵占

**犯罪特点**:
| 特点 | 内容 |
|------|------|
| 谋利性 | 以获取财物为目的 |
| 隐蔽性 | 多数比较隐蔽 |
| 多发性 | 发案率高 |
| 多样性 | 手段多样 |

**犯罪原因**:
- 贫困
- 贪欲
- 机会
- 技能

**预防措施**:
- 技防设施
- 财物保管
- 防范意识
- 经济发展

**实务提示**:  
财产犯罪是最常见的犯罪类型。

---

#### 14. 毒品犯罪 (Drug Crime/毒品犯罪)

**白话解释**:  
跟毒品有关的犯罪。

**生活例子**:  
- 贩卖毒品
- 运输毒品
- 制造毒品
- 容留吸毒

**学术定义**:  
违反毒品管理法规,走私、贩卖、运输、制造、持有毒品等与毒品相关的犯罪行为。

**犯罪类型**:
1. **走私毒品** - 走私毒品入境
2. **贩卖毒品** - 出售毒品
3. **运输毒品** - 运送毒品
4. **制造毒品** - 制造毒品
5. **持有毒品** - 非法持有毒品
6. **容留吸毒** - 容留他人吸毒

**犯罪特点**:
| 特点 | 内容 |
|------|------|
| 组织性 | 多有组织 |
| 隐蔽性 | 手段隐蔽 |
| 暴力性 | 伴有暴力 |
| 国际性 | 跨国犯罪多 |

**社会危害**:
- 毁灭个人健康
- 破坏家庭幸福
- 危害社会治安
- 影响经济发展

**预防措施**:
- 严厉打击
- 缉毒执法
- 禁毒宣传
- 戒毒康复

**实务提示**:  
毒品犯罪危害极大,要严厉打击。

---

#### 15. 网络犯罪 (Cybercrime/网络犯罪)

**白话解释**:  
利用网络实施的犯罪。

**生活例子**:  
- 网络诈骗
- 网络盗窃
- 网络攻击
- 网络赌博

**学术定义**:  
利用计算机网络技术实施的,或者针对计算机网络实施的犯罪行为。

**犯罪类型**:
1. **网络诈骗** - 电信网络诈骗
2. **网络盗窃** - 盗窃网络财产
3. **网络攻击** - 黑客攻击
4. **网络赌博** - 网络赌博
5. **网络色情** - 传播淫秽信息
6. **侵犯公民信息** - 非法获取出售信息

**犯罪特点**:
| 特点 | 内容 |
|------|------|
| 技术性 | 使用网络技术 |
| 隐蔽性 | 身份隐匿 |
| 跨地域性 | 跨地区跨国 |
| 危害性 | 危害范围广 |

**犯罪手段**:
- 钓鱼网站
- 木马病毒
- 社会工程学
- 黑客攻击

**预防措施**:
- 网络安全防护
- 个人信息保护
- 防范意识教育
- 技术手段打击

**实务提示**:  
网络犯罪是新兴犯罪类型,要高度重视。

---

#### 16. 有组织犯罪 (Organized Crime/有组织犯罪)

**白话解释**:  
有组织的团伙犯罪。

**生活例子**:  
- 黑社会性质组织
- 犯罪集团
- 涉黑涉恶团伙

**学术定义**:  
三人以上,有组织、有计划地实施犯罪活动的犯罪形式,包括黑社会性质组织犯罪、犯罪集团犯罪等。

**组织类型**:
1. **黑社会性质组织** - 有严密组织、经济实力、暴力手段
2. **犯罪集团** - 三人以上固定犯罪组织
3. **涉恶团伙** - 组织性较弱的犯罪团伙

**组织特征**:
| 特征 | 内容 |
|------|------|
| 组织性 | 有组织结构 |
| 计划性 | 有计划实施犯罪 |
| 连续性 | 连续多次犯罪 |
- 暴力性 | 常伴暴力威胁 |
| 非法控制 | 形成非法控制 |

**主要犯罪**:
- 暴力犯罪
- 走私贩毒
- 非法经营
- 垄断行业

**打击措施**:
- 依法严惩
- 打掉组织
- 深挖保护伞
- 综合治理

**实务提示**:  
有组织犯罪危害大,要严厉打击。

---

#### 17. 职务犯罪 (Official Crime/职务犯罪)

**白话解释**:  
当官的或工作人员利用职务犯罪。

**生活例子**:  
- 贪污
- 受贿
- 滥用职权
- 玩忽职守

**学术定义**:  
国家工作人员利用职务便利实施的犯罪,或者因职务行为而构成的犯罪。

**犯罪类型**:
1. **贪污贿赂犯罪** - 贪污、受贿、行贿
2. **渎职犯罪** - 滥用职权、玩忽职守
3. **侵权犯罪** - 刑讯逼供、非法拘禁

**犯罪特点**:
| 特点 | 内容 |
|------|------|
| 主体特定 | 国家工作人员 |
| 职务关联 | 利用职务便利 |
| 隐蔽性 | 手段隐蔽 |
- 智能性 | 需要一定智能 |
| 危害性 | 严重危害社会 |

**犯罪原因**:
- 权力缺乏监督
- 利益诱惑
- 思想滑坡
- 制度漏洞

**预防措施**:
- 权力监督
- 制度建设
- 廉政教育
- 严厉惩处

**实务提示**:  
职务犯罪严重损害政府公信力。

---

#### 18. 未成年人犯罪 (Juvenile Delinquency/未成年人犯罪)

**白话解释**:  
未成年人(18岁以下)犯罪。

**生活例子**:  
- 青少年打架斗殴
- 未成年人盗窃
- 校园欺凌

**学术定义**:  
已满14周岁不满18周岁的人实施的危害社会、依法应当受到刑事处罚的行为。

**犯罪特点**:
| 特点 | 内容 |
|------|------|
| 年龄特征 | 14-18岁 |
| 团伙性 | 多团伙犯罪 |
| 模仿性 | 模仿成人犯罪 |
| 冲动性 | 常因冲动犯罪 |
| 可塑性 | 可教育矫正 |

**犯罪类型**:
- 暴力犯罪
- 财产犯罪
- 性犯罪
- 毒品犯罪

**犯罪原因**:
- 家庭因素 (缺爱、溺爱)
- 学校因素 (辍学、成绩差)
- 社会因素 (不良影响)
- 个人因素 (心理问题)

**预防措施**:
- 家庭教育
- 学校教育
- 社会关爱
- 司法保护

**司法原则**:
- 教育为主、惩罚为辅
- 不公开审理
- 轻缓处理
- 帮教挽救

**实务提示**:  
未成年人犯罪要注重教育挽救。

### 四、犯罪预防

#### 19. 犯罪预防 (Crime Prevention/犯罪预防)

**白话解释**:  
想办法不让犯罪发生。

**生活例子**:  
- 安装监控
- 加强巡逻
- 法制教育

**学术定义**:  
采取各种措施,消除犯罪原因,减少犯罪机会,防止犯罪发生的活动。

**预防类型**:
1. **社会预防** - 从社会层面预防
2. **情境预防** - 从环境层面预防
3. **被害人预防** - 从被害人角度预防
4. **司法预防** - 通过司法手段预防

**预防层次**:
| 层次 | 内容 |
|------|------|
| 一级预防 | 面向全体公众的预防 |
| 二级预防 | 面向高危人群的预防 |
| 三级预防 | 面向犯罪人的预防 |

**预防措施**:
- 社会治理
- 教育宣传
- 技术防范
- 司法打击

**预防原则**:
- 综合治理
- 预防为主
- 标本兼治
- 全民参与

**实务提示**:  
犯罪预防是降低犯罪率的根本途径。

---

#### 20. 社会预防 (Social Prevention/社会预防)

**白话解释**:  
从社会层面预防犯罪。

**生活例子**:  
- 发展经济
- 改善教育
- 完善社会保障

**学术定义**:  
通过改善社会环境、加强社会管理、促进社会公平等措施,从根源上预防犯罪的策略。

**预防内容**:
1. **经济发展** - 发展经济,减少贫困
2. **教育发展** - 普及教育,提高素质
3. **社会保障** - 完善保障,减少压力
4. **文化建设** - 发展文化,提升道德
5. **社会治理** - 加强治理,减少机会

**具体措施**:
| 领域 | 措施 |
|------|------|
| 经济 | 发展经济、创造就业 |
| 教育 | 普及教育、素质教育 |
| 社保 | 完善社保、兜底保障 |
| 文化 | 先进文化、道德建设 |
| 治理 | 基层治理、矛盾化解 |

**预防效果**:
- 减少犯罪原因
- 降低犯罪动机
- 消除犯罪土壤
- 构建和谐社会

**实务提示**:  
社会预防是治本之策。

---

#### 21. 情境预防 (Situational Prevention/情境预防)

**白话解释**:  
改变环境,减少犯罪机会。

**生活例子**:  
- 安装监控摄像头
- 增加照明
- 安装防盗门

**学术定义**:  
通过改变犯罪发生的环境和情境,增加犯罪难度和风险,减少犯罪机会的预防策略。

**预防原理**:
1. **增加犯罪难度** - 让犯罪更难实施
2. **增加犯罪风险** - 让犯罪更容易被发现
3. **减少犯罪收益** - 让犯罪无利可图
4. **消除犯罪借口** - 让犯罪没有理由

**具体措施**:
| 类型 | 措施 |
|------|------|
| 增加难度 | 防盗门、监控、围墙 |
| 增加风险 | 巡逻、监控、报警 |
| 减少收益 | 财物标记、快速反应 |
| 消除借口 | 宣传教育、明确规则 |

**适用场景**:
- 住宅小区
- 商业场所
- 公共交通
- 学校医院

**实务提示**:  
情境预防是有效的犯罪预防手段。

---

#### 22. 被害预防 (Victimological Prevention/被害预防)

**白话解释**:  
教人不要被犯罪侵害。

**生活例子**:  
- 不要深夜独自外出
- 保管好财物
- 不轻信陌生人

**学术定义**:  
从被害人角度,通过提高防范意识、减少被害风险,预防犯罪发生的策略。

**预防内容**:
1. **防范意识** - 提高安全防范意识
2. **防范知识** - 学习防范知识技能
3. **防范行为** - 采取防范行为措施
4. **减少风险** - 减少被害风险因素

**防范措施**:
| 场景 | 措施 |
|------|------|
| 财产安全 | 保管财物、防范盗窃诈骗 |
| 人身安全 | 避免危险场所、结伴出行 |
| 网络安全 | 保护信息、防范网络诈骗 |
| 交通安全 | 遵守规则、安全出行 |

**高危人群**:
- 老年人
- 未成年人
- 女性群体
- 外来人口

**实务提示**:  
被害预防是犯罪预防的重要方面。

---

#### 23. 刑罚预防 (Penal Prevention/刑罚预防)

**白话解释**:  
用刑罚来预防犯罪。

**生活例子**:  
- 严厉打击犯罪
- 判刑震慑
- 特殊预防

**学术定义**:  
通过刑罚的适用和执行,对犯罪人进行惩罚和矫正,对社会公众进行警示,从而预防犯罪的策略。

**预防类型**:
1. **一般预防** - 对社会公众的警示
2. **特殊预防** - 对犯罪人的预防

**一般预防**:
| 内容 | 具体表现 |
|------|----------|
| 威慑 | 刑罚威慑潜在犯罪人 |
| 教育 | 教育公民遵纪守法 |
| 安抚 | 安抚被害人和社会 |

**特殊预防**:
| 内容 | 具体表现 |
|------|----------|
| 惩罚 | 惩罚犯罪人 |
| 矫正 | 矫正犯罪人 |
| 改造 | 改造犯罪人 |
| 剥夺能力 | 剥夺再犯能力 |

**刑罚功能**:
- 惩罚功能
- 矫正功能
- 威慑功能
- 安抚功能

**实务提示**:  
刑罚是重要的犯罪预防手段。

## 📖 第四章：犯罪矫正

### 五、犯罪矫正

#### 24. 犯罪矫正 (Criminal Correction/犯罪矫正)

**白话解释**:  
改造犯罪人,让他回归社会。

**生活例子**:  
- 监狱改造
- 社区矫正
- 教育感化

**学术定义**:  
通过刑罚执行和各种教育改造措施,转变犯罪人的思想和行为,使其回归社会的活动。

**矫正目标**:
1. **惩罚犯罪** - 惩罚犯罪行为
2. **改造罪犯** - 改造犯罪人
3. **预防再犯** - 预防重新犯罪
4. **回归社会** - 帮助回归社会

**矫正原则**:
| 原则 | 内容 |
|------|------|
| 教育感化 | 以教育为主 |
| 劳动改造 | 通过劳动改造 |
| 惩罚与教育结合 | 惩罚中教育 |
| 因人施教 | 区别对待 |

**矫正方式**:
- 监狱矫正
- 社区矫正
- 教育矫正
- 劳动矫正

**实务提示**:  
矫正是预防再犯的重要手段。

---

#### 25. 监狱矫正 (Prison Correction/监狱矫正)

**白话解释**:  
在监狱里改造罪犯。

**生活例子**:  
- 坐牢
- 劳动改造
- 学习教育

**学术定义**:  
在监狱等监管场所内,对判处监禁刑的罪犯进行惩罚和改造的活动。

**矫正内容**:
1. **思想教育** - 思想政治教育
2. **文化教育** - 文化知识教育
3. **职业技能教育** - 职业技能培训
4. **劳动改造** - 劳动改造
5. **心理矫治** - 心理辅导治疗

**矫正制度**:
| 制度 | 内容 |
|------|------|
| 分押分管 | 分类关押管理 |
| 计分考核 | 考核表现 |
| 减刑假释 | 减刑假释制度 |
| 生活卫生 | 生活医疗保障 |

**矫正原则**:
- 惩罚与改造相结合
- 教育与劳动相结合
- 严格管理与文明管理相结合

**面临问题**:
- 监狱拥挤
- 矫正效果不理想
- 再犯率较高
- 回归社会困难

**实务提示**:  
监狱矫正是主要的矫正方式。

---

#### 26. 社区矫正 (Community Correction/社区矫正)

**白话解释**:  
不坐牢,在社区里接受矫正。

**生活例子**:  
- 缓刑人员在社区接受矫正
- 假释人员在社区接受监督
- 定期向司法所报告

**学术定义**:  
将被判处管制、宣告缓刑、假释、暂予监外执行的罪犯置于社区内,由专门机关在相关社会组织和志愿者协助下,进行矫正的活动。

**适用对象**:
1. **被判处管制的**
2. **被宣告缓刑的**
3. **被裁定假释的**
4. **被暂予监外执行的**

**矫正内容**:
| 内容 | 具体措施 |
|------|----------|
| 监督管理 | 定期报告、活动限制 |
| 教育矫正 | 法治教育、道德教育 |
| 心理矫正 | 心理辅导、心理治疗 |
| 社会适应性帮扶 | 就业帮助、生活帮助 |

**矫正措施**:
- 定期报告
- 限制活动
- 集中教育
- 社区服务
- 心理辅导

**优势特点**:
- 不脱离社会
- 家庭支持
- 社会资源
- 降低成本

**实务提示**:  
社区矫正有利于罪犯回归社会。

---

#### 27. 累犯 (Recidivism/累犯)

**白话解释**:  
再次犯罪。

**生活例子**:  
- 出狱后又犯罪
- 多次犯罪

**学术定义**:  
因犯罪受过刑事处罚,在刑罚执行完毕或赦免以后,在法定期限内又犯罪的人。

**累犯类型**:
1. **一般累犯** - 前后都是有期徒刑以上,5年内又犯罪
2. **特别累犯** - 前后都是危害国家安全犯罪、恐怖活动犯罪、黑社会性质组织犯罪

**累犯条件**:
| 条件 | 一般累犯 | 特别累犯 |
|------|----------|----------|
| 前罪 | 有期徒刑以上 | 特定犯罪 |
| 后罪 | 有期徒刑以上 | 同类犯罪 |
| 时间 | 5年内 | 无限制 |
| 年龄 | 18岁以上 | 无限制 |

**累犯后果**:
- 从重处罚
- 不得缓刑
- 不得假释

**累犯原因**:
- 改造不彻底
- 就业困难
- 社会歧视
- 旧习难改

**预防措施**:
- 提高矫正质量
- 帮助回归社会
- 消除社会歧视
- 加强后续帮教

**实务提示**:  
累犯是犯罪预防的重点。

---

#### 28. 再犯预防 (Recidivism Prevention/再犯预防)

**白话解释**:  
防止犯罪人重新犯罪。

**生活例子**:  
- 出狱后帮教
- 就业帮助
- 心理辅导

**学术定义**:  
采取各种措施,帮助刑满释放人员顺利回归社会,防止重新犯罪的活动。

**预防内容**:
1. **教育矫正** - 提高矫正质量
2. **技能培训** - 培养谋生技能
3. **就业帮助** - 帮助就业创业
4. **生活帮扶** - 解决生活困难
5. **心理辅导** - 心理疏导帮助

**预防措施**:
| 阶段 | 措施 |
|------|------|
| 监狱内 | 提高矫正质量 |
| 出狱前 | 做好出狱准备 |
| 出狱后 | 帮助融入社会 |
| 稳定期 | 持续关注帮教 |

**责任主体**:
- 司法行政机关
- 社会组织
- 家庭
- 社区

**预防难点**:
- 就业困难
- 社会歧视
- 家庭拒绝
- 旧环境诱惑

**实务提示**:  
再犯预防是社会治安的重要保障。
