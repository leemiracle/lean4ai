### 一、侦查基本概念

#### 1. 刑事侦查 (Criminal Investigation/刑事侦查)

**白话解释**:  
查案破案,抓住罪犯。

**生活例子**:  
- 现场勘查
- 侦查讯问
- 抓捕犯罪嫌疑人

**学术定义**:  
公安机关、人民检察院依法对刑事案件进行调查取证、查明案情的专门活动,是刑事诉讼的重要组成部分。

**侦查任务**:
1. **查明案情** - 查明案件事实真相
2. **收集证据** - 收集犯罪证据
3. **抓获犯罪嫌疑人** - 抓捕犯罪嫌疑人
4. **挽回损失** - 追缴赃款赃物

**侦查特点**:
| 特点 | 内容 |
|------|------|
| 法定性 | 依法进行侦查 |
| 强制性 | 有国家强制力 |
| 秘密性 | 侦查活动保密 |
| 及时性 | 必须及时进行 |
| 专门性 | 专门机关进行 |

**侦查主体**:
- 公安机关
- 人民检察院
- 国家安全机关
- 监狱 (狱内案件)

**实务提示**:  
侦查是刑事诉讼的重要环节。

---

#### 2. 侦查权 (Investigation Power/侦查权)

**白话解释**:  
查案的权力。

**生活例子**:  
- 立案权
- 侦查措施权
- 强制措施权

**学术定义**:  
国家法律赋予侦查机关对刑事案件进行调查取证、查明案情的权力,是国家权力的重要组成部分。

**权力内容**:
1. **立案权** - 决定是否立案
2. **调查权** - 调查案件事实
3. **证据收集权** - 收集固定证据
4. **强制措施权** - 采取强制措施
5. **技术侦查权** - 使用技术手段

**权力特征**:
| 特征 | 内容 |
|------|------|
| 法定性 | 法律授权 |
| 专属性 | 专门机关行使 |
| 强制性 | 有国家强制力 |
| 有限性 | 有边界限制 |

**权力边界**:
- 依法行使
- 程序正当
- 比例原则
- 人权保障

**权力监督**:
- 检察监督
- 法院审查
- 人大监督
- 社会监督

**实务提示**:  
侦查权关系公民权利,必须依法行使。

---

#### 3. 立案 (Filing a Case/立案)

**白话解释**:  
决定开始查案。

**生活例子**:  
- 被盗后报警,警察立案
- 发现犯罪线索,决定立案

**学术定义**:  
公安机关、人民检察院对报案、控告、举报、自首等材料进行审查后,认为有犯罪事实需要追究刑事责任,决定作为刑事案件进行侦查的诉讼活动。

**立案条件**:
1. **有犯罪事实** - 确实发生了犯罪
2. **需要追究刑事责任** - 不是轻罪或过时效
3. **属于本机关管辖** - 管辖权正确

**立案程序**:
| 程序 | 内容 |
|------|------|
| 接受 | 接受报案材料 |
| 审查 | 审查材料真实性 |
| 决定 | 决定是否立案 |
| 通知 | 通知报案人 |

**不立案情形**:
- 没有犯罪事实
- 情节显著轻微
- 已过追诉时效
- 经特赦令免除
- 犯罪嫌疑人死亡
- 其他依法不追究

**立案监督**:
- 检察院立案监督
- 报案人申请复议
- 报案人申请检察监督

**实务提示**:  
立案是侦查的起点,要严格把关。

---

#### 4. 侦查阶段 (Investigation Stage/侦查阶段)

**白话解释**:  
从立案到侦查终结这段时间。

**生活例子**:  
- 公安机关侦查案件的过程
- 收集证据、讯问犯罪嫌疑人

**学术定义**:  
从立案开始到侦查终结的整个诉讼阶段,是刑事诉讼的重要环节。

**阶段内容**:
1. **初查** - 初步调查核实
2. **立案** - 决定立案
3. **侦查** - 全面侦查
4. **侦查终结** - 完成侦查

**侦查期限**:
| 情况 | 期限 |
|------|------|
| 一般案件 | 2个月 |
| 案情复杂 | 延长1个月 |
| 重大复杂 | 延长2个月 |
| 特别重大复杂 | 延长2个月 |

**期限计算**:
- 从立案之日起算
- 不包括拘留、逮捕时间
- 羁押期限另算

**侦查终结**:
- 案情清楚
- 证据确实充分
- 法律手续完备
- 写出起诉意见书

**实务提示**:  
侦查要在法定期限内完成。

---

#### 5. 侦查原则 (Investigation Principles/侦查原则)

**白话解释**:  
查案要遵守的基本规矩。

**生活例子**:  
- 不能刑讯逼供
- 不能伪造证据
- 要保护人权

**学术定义**:  
侦查机关和侦查人员在侦查活动中必须遵循的基本准则和行为规范。

**基本原则**:
1. **依法侦查原则** - 严格依法进行
2. **客观全面原则** - 客观全面收集证据
3. **及时高效原则** - 及时开展侦查
4. **保障人权原则** - 尊重保障人权
5. **依靠群众原则** - 依靠群众支持
6. **保守秘密原则** - 保守侦查秘密

**原则要求**:
| 原则 | 具体要求 |
|------|----------|
| 依法侦查 | 按法律程序进行 |
| 客观全面 | 收集有罪无罪证据 |
| 及时高效 | 不拖延延误 |
| 保障人权 | 不刑讯逼供 |
| 依靠群众 | 发动群众支持 |
| 保守秘密 | 不泄露案情 |

**违反后果**:
- 证据被排除
- 案件被撤销
- 承担法律责任

**实务提示**:  
侦查原则是侦查工作的基本准则。

### 二、一般侦查措施

#### 6. 讯问 (Interrogation/讯问)

**白话解释**:  
审问犯罪嫌疑人。

**生活例子**:  
- 在看守所审问犯罪嫌疑人
- 询问作案经过
- 核实证据

**学术定义**:  
侦查人员依法对犯罪嫌疑人进行面对面提问,要求其如实回答案件有关问题的侦查行为。

**讯问主体**:
- 侦查人员 (不少于2人)

**讯问对象**:
- 犯罪嫌疑人

**讯问地点**:
| 对象 | 地点 |
|------|------|
| 未羁押 | 传唤到公安机关 |
| 已羁押 | 看守所 |

**讯问程序**:
1. **告知权利** - 告知享有的权利
2. **核对身份** - 核实身份信息
3. **提问** - 提问案件情况
4. **记录** - 制作讯问笔录
5. **核对签名** - 核对后签名

**禁止行为**:
- 刑讯逼供
- 威胁引诱
- 欺骗
- 非法拘禁

**权利保障**:
- 如实供述可以从宽
- 有权聘请律师
- 有权拒绝回答无关问题
- 有权核对笔录

**实务提示**:  
讯问必须合法,禁止刑讯逼供。

---

#### 7. 询问 (Inquiry/询问)

**白话解释**:  
找证人、被害人了解情况。

**生活例子**:  
- 询问目击证人
- 询问被害人
- 询问知情人员

**学术定义**:  
侦查人员依法向证人、被害人等了解案件情况的侦查行为。

**询问对象**:
1. **证人** - 了解案件情况的人
2. **被害人** - 犯罪受害人
3. **其他知情人员** - 其他了解情况的人

**询问地点**:
| 对象 | 地点 |
|------|------|
| 一般情况 | 证人所在地或公安机关 |
| 特殊情况 | 证人提出的地点 |

**询问程序**:
1. **告知义务** - 告知如实作证义务
2. **核对身份** - 核实身份信息
3. **提问** - 提问案件情况
4. **记录** - 制作询问笔录
5. **核对签名** - 核对后签名

**询问要求**:
- 个别进行
- 客观记录
- 不诱导提示
- 保护证人安全

**证人权利**:
- 作证受法律保护
- 有权获得补偿
- 特殊情况下可以保密

**实务提示**:  
询问要个别进行,保护证人安全。

---

#### 8. 勘验检查 (Examination/勘验检查)

**白话解释**:  
到现场查看,检查物证。

**生活例子**:  
- 命案现场勘查
- 提取指纹、DNA
- 拍照录像

**学术定义**:  
侦查人员对与犯罪有关的场所、物品、人身、尸体等进行勘查、检验的侦查行为。

**勘验类型**:
1. **现场勘验** - 勘查犯罪现场
2. **物证检验** - 检验物证
3. **尸体检验** - 检验尸体
4. **人身检查** - 检查人身

**勘验程序**:
| 程序 | 内容 |
|------|------|
| 保护现场 | 第一时间保护现场 |
| 勘验准备 | 准备勘验设备 |
| 实地勘验 | 到现场勘验 |
| 提取痕迹 | 提取指纹、DNA等 |
| 现场记录 | 拍照、录像、绘图 |
| 制作笔录 | 制作勘验笔录 |

**勘验内容**:
- 现场方位
- 现场状况
- 痕迹物品
- 尸体状况
- 其他情况

**勘验要求**:
- 及时勘验
- 全面细致
- 规范操作
- 客观记录

**实务提示**:  
勘验是获取证据的重要手段。

---

#### 9. 搜查 (Search/搜查)

**白话解释**:  
搜查场所、人身,找证据。

**生活例子**:  
- 搜查犯罪嫌疑人住所
- 搜查犯罪嫌疑人身体
- 搜查车辆、物品

**学术定义**:  
侦查人员对犯罪嫌疑人以及可能隐藏罪犯或者犯罪证据的人的身体、物品、住处和其他有关的地方进行搜索、检查的侦查行为。

**搜查对象**:
1. **人身** - 搜查身体
2. **物品** - 搜查携带物品
3. **住所** - 搜查住处
4. **其他场所** - 搜查其他有关场所

**搜查条件**:
- 收集犯罪证据
- 查获犯罪嫌疑人

**搜查程序**:
| 程序 | 内容 |
|------|------|
| 出示证件 | 出示搜查证 |
| 说明来意 | 说明搜查目的 |
| 实施搜查 | 进行搜查 |
| 提取证据 | 提取相关证据 |
| 制作笔录 | 制作搜查笔录 |

**见证人要求**:
- 应当有见证人在场
- 见证人签名

**特殊情况**:
- 紧急情况可以无证搜查
- 但事后要补办手续

**实务提示**:  
搜查必须出示搜查证(紧急情况除外)。

---

#### 10. 扣押 (Seizure/扣押)

**白话解释**:  
扣留证据物品。

**生活例子**:  
- 扣押作案工具
- 扣押赃款赃物
- 扣押文件资料

**学术定义**:  
侦查机关依法强行提取、扣留与案件有关的物品、文件的侦查行为。

**扣押对象**:
1. **物证** - 证明案件事实的物品
2. **书证** - 证明案件事实的文件
3. **赃物** - 犯罪所得
4. **作案工具** - 犯罪使用的工具

**扣押程序**:
| 程序 | 内容 |
|------|------|
| 出示证件 | 出示侦查证件 |
| 说明理由 | 说明扣押理由 |
| 清点登记 | 清点扣押物品 |
| 制作清单 | 制作扣押清单 |
| 签名确认 | 当事人签名确认 |

**扣押要求**:
- 与案件有关
- 当场清点
- 制作清单
- 妥善保管

**保管要求**:
- 专人保管
- 不得使用
- 不得损毁
- 案结后处理

**实务提示**:  
扣押物品要制作清单,妥善保管。

---

#### 11. 鉴定 (Expert Evaluation/鉴定)

**白话解释**:  
请专家鉴定证据。

**生活例子**:  
- DNA鉴定
- 指纹鉴定
- 法医鉴定

**学术定义**:  
侦查机关指派或聘请具有专门知识的人,对案件中某些专门性问题进行鉴别和判断的侦查行为。

**鉴定类型**:
1. **法医鉴定** - 死因、伤情鉴定
2. **痕迹鉴定** - 指纹、足迹鉴定
3. **DNA鉴定** - 生物物证鉴定
4. **毒物鉴定** - 毒品、毒物鉴定
5. **精神病鉴定** - 精神状态鉴定
6. **文件鉴定** - 笔迹、印章鉴定
7. **会计鉴定** - 财务会计鉴定

**鉴定程序**:
| 程序 | 内容 |
|------|------|
| 委托 | 委托鉴定机构 |
| 提供材料 | 提供检材样本 |
| 鉴定 | 进行鉴定 |
| 出具意见 | 出具鉴定意见 |
| 告知 | 告知当事人 |

**鉴定要求**:
- 有鉴定资质
- 客观公正
- 科学规范
- 出具书面意见

**当事人权利**:
- 申请补充鉴定
- 申请重新鉴定
- 查看鉴定意见

**实务提示**:  
鉴定意见是重要证据,要科学规范。

---

#### 12. 辨认 (Identification/辨认)

**白话解释**:  
让证人、被害人认人、认物。

**生活例子**:  
- 让被害人辨认犯罪嫌疑人
- 让证人辨认作案工具
- 辨认无名尸体

**学术定义**:  
侦查机关组织被害人、证人、犯罪嫌疑人对与犯罪有关的物品、文件、尸体、场所或者人员进行辨识、确认的侦查行为。

**辨认类型**:
1. **人身辨认** - 辨认犯罪嫌疑人
2. **物品辨认** - 辨认涉案物品
3. **尸体辨认** - 辨认尸体
4. **场所辨认** - 辨认犯罪场所

**辨认程序**:
| 程序 | 内容 |
|------|------|
| 准备 | 准备辨认对象 |
| 告知 | 告知辨认要求 |
| 辨认 | 进行辨认 |
| 记录 | 制作辨认笔录 |

**辨认要求**:
- 个别进行
- 混杂辨认
- 不暗示诱导
- 如实记录

**混杂辨认**:
- 辨认人时,至少7人混合
- 辨认照片时,至少10张混合
- 辨认物品时,至少5件混合

**实务提示**:  
辨认要混杂进行,不能暗示诱导。

### 三、特殊侦查措施

#### 13. 技术侦查 (Technical Investigation/技术侦查)

**白话解释**:  
用高科技手段侦查。

**生活例子**:  
- 监听电话
- 定位追踪
- 秘密拍照录像

**学术定义**:  
侦查机关在办理特定案件时,依法使用技术手段进行的专门调查工作,包括电子监听、通讯截获、秘密拍照录像等。

**适用范围**:
1. **危害国家安全犯罪**
2. **恐怖活动犯罪**
3. **黑社会性质组织犯罪**
4. **重大毒品犯罪**
5. **其他严重危害社会的犯罪**

**技术手段**:
| 手段 | 内容 |
|------|------|
| 监听 | 监听通讯 |
| 截获 | 截获通讯内容 |
| 定位 | 定位嫌疑人 |
| 秘密拍照 | 秘密拍照取证 |
| 秘密录像 | 秘密录像取证 |

**适用程序**:
1. **审批** - 经过严格审批
2. **期限** - 不超过3个月
3. **续期** - 可以续期
4. **保密** - 严格保密

**使用要求**:
- 经过严格审批
- 适用范围有限
- 期限限制
- 保密要求

**实务提示**:  
技术侦查涉及隐私,要严格依法。

---

#### 14. 秘密侦查 (Secret Investigation/秘密侦查)

**白话解释**:  
隐蔽身份进行侦查。

**生活例子**:  
- 卧底侦查
- 特情侦查
- 化装侦查

**学术定义**:  
侦查人员隐瞒真实身份,以其他身份为掩护,接近侦查对象,获取犯罪情报和证据的侦查方式。

**秘密侦查类型**:
1. **卧底侦查** - 长期潜伏侦查
2. **特情侦查** - 使用特情人员
3. **诱惑侦查** - 诱惑犯罪实施
4. **化装侦查** - 化装接近侦查

**适用范围**:
- 有组织犯罪
- 毒品犯罪
- 走私犯罪
- 贿赂犯罪

**使用原则**:
| 原则 | 内容 |
|------|------|
| 必要性 | 确有必要才能使用 |
| 合法性 | 不得诱发犯罪 |
| 适度性 | 适度使用 |
| 保密性 | 严格保密 |

**证据使用**:
- 转化为公开证据
- 保护侦查人员安全
- 保护特情人员安全

**实务提示**:  
秘密侦查要慎重使用,严格审批。

---

#### 15. 控制下交付 (Controlled Delivery/控制下交付)

**白话解释**:  
在监控下让交易进行,抓现行。

**生活例子**:  
- 毒品交易时抓捕
- 贿赂交易时抓捕
- 走私货物时抓捕

**学术定义**:  
侦查机关在发现毒品等违禁品或可疑物品后,在监控下允许其继续运输、交付,以查获更多犯罪嫌疑人或犯罪证据的侦查措施。

**适用范围**:
1. **毒品犯罪** - 毒品交易
2. **走私犯罪** - 走私货物
3. **非法买卖** - 非法交易

**操作程序**:
| 程序 | 内容 |
|------|------|
| 发现 | 发现违禁品 |
| 决定 | 决定控制下交付 |
| 监控 | 监控运输过程 |
| 抓捕 | 交付时抓捕 |
| 搜证 | 搜集证据 |

**注意事项**:
- 不得流失违禁品
- 保障公共安全
- 及时抓捕
- 固定证据

**风险控制**:
- 防止失控
- 防止伤人
- 及时收网

**实务提示**:  
控制下交付风险大,要周密部署。

---

## 📖 第三章：侦查程序

---

### 四、强制措施

#### 16. 拘传 (Summon by Warrant/拘传)

**白话解释**:  
强制带人到案。

**生活例子**:  
- 传唤不到案,强制拘传
- 拘传到公安机关讯问

**学术定义**:  
公安机关对未被羁押的犯罪嫌疑人,依法强制其到案接受讯问的强制措施。

**适用条件**:
- 经过合法传唤
- 无正当理由不到案

**拘传程序**:
1. **出示拘传证** - 出示拘传证
2. **强制到案** - 强制带到公安机关
3. **讯问** - 立即讯问
4. **送回或羁押** - 送回或变更措施

**拘传期限**:
| 情况 | 期限 |
|------|------|
| 一般情况 | 12小时 |
| 案情重大复杂 | 24小时 |

**拘传要求**:
- 保证犯罪嫌疑人饮食
- 保证必要休息时间
- 不得以连续拘传变相羁押

**实务提示**:  
拘传是强制到案的措施,不能变相羁押。

---

#### 17. 取保候审 (Bail/取保候审)

**白话解释**:  
交钱或找人担保,不关押。

**生活例子**:  
- 缴纳保证金取保
- 找保证人担保

**学术定义**:  
公安机关责令犯罪嫌疑人提出保证人或者缴纳保证金,保证不逃避侦查、随传随到,不予羁押的强制措施。

**适用条件**:
1. **可能判处管制、拘役或独立适用附加刑的**
2. **可能判处有期徒刑以上刑罚,采取取保候审不致发生社会危险性的**
3. **患有严重疾病、生活不能自理的**
4. **怀孕或正在哺乳自己婴儿的妇女**
5. **羁押期限届满,案件尚未办结的**

**保证方式**:
| 方式 | 内容 |
|------|------|
| 保证人 | 提供保证人担保 |
| 保证金 | 缴纳保证金担保 |

**保证金数额**:
- 起点1000元
- 根据案件情况确定

**遵守规定**:
- 未经批准不得离开所居住的市县
- 住址、工作单位联系方式变动,24小时内报告
- 传讯时及时到案
- 不得干扰证人作证
- 不得毁灭伪造证据

**实务提示**:  
取保候审是重要的非羁押措施。

---

#### 18. 监视居住 (Residential Surveillance/监视居住)

**白话解释**:  
限制活动范围,被监视。

**生活例子**:  
- 在指定住所居住
- 不得随意外出
- 被监视活动

**学术定义**:  
公安机关责令犯罪嫌疑人不得离开住处或指定住所,对其行动加以监视的强制措施。

**适用条件**:
1. **符合取保候审条件,但无法提供保证人或缴纳保证金的**
2. **患有严重疾病、生活不能自理的**
3. **怀孕或正在哺乳自己婴儿的妇女**
4. **系生活不能自理的人的唯一扶养人**
5. **羁押期限届满,案件尚未办结的**

**监视方式**:
| 方式 | 内容 |
|------|------|
| 住处监视 | 在本人住处监视 |
| 指定居所监视 | 在指定居所监视 |

**期限**:
- 最长6个月

**遵守规定**:
- 未经批准不得离开执行监视居住的处所
- 未经批准不得会见他人或通信
- 传讯时及时到案
- 不得干扰证人作证
- 不得毁灭伪造证据
- 将护照等出入境证件、身份证件、驾驶证件交执行机关保存

**实务提示**:  
监视居住是较严厉的非羁押措施。

---

#### 19. 刑事拘留 (Criminal Detention/刑事拘留)

**白话解释**:  
暂时关押起来。

**生活例子**:  
- 现行犯被拘留
- 重大嫌疑人被拘留

**学术定义**:  
公安机关对现行犯或重大嫌疑分子,在紧急情况下依法采取的暂时剥夺其人身自由的强制措施。

**适用条件**:
1. **正在预备犯罪、实行犯罪或犯罪后即时被发觉的**
2. **被害人或在场亲眼看见的人指认他犯罪的**
3. **在身边或住处发现有犯罪证据的**
4. **犯罪后企图自杀、逃跑或在逃的**
5. **有毁灭、伪造证据或串供可能的**
6. **不讲真实姓名、住址,身份不明的**
7. **有流窜作案、多次作案、结伙作案重大嫌疑的**

**拘留期限**:
| 情况 | 期限 |
|------|------|
| 一般情况 | 3日 |
| 延长1-4日 | 3+7=10日 |
| 流窜结伙多次 | 30日 |
| 检察院批准逮捕 | 7日 |
| 最长期限 | 37日 |

**拘留程序**:
1. **出示拘留证** - 出示拘留证
2. **宣布拘留** - 宣布拘留决定
3. **送看守所** - 送看守所羁押
4. **通知家属** - 24小时内通知家属
5. **讯问** - 24小时内讯问

**权利保障**:
- 及时讯问
- 通知家属
- 律师会见

**实务提示**:  
拘留是暂时羁押措施,要及时审查。

---

#### 20. 逮捕 (Arrest/逮捕)

**白话解释**:  
正式关押。

**生活例子**:  
- 经检察院批准逮捕
- 长期羁押

**学术定义**:  
经人民检察院批准或人民法院决定,由公安机关执行的对犯罪嫌疑人、被告人实行羁押,剥夺其人身自由的最严厉的强制措施。

**适用条件**:
1. **有证据证明有犯罪事实**
2. **可能判处徒刑以上刑罚**
3. **采取取保候审、监视居住等方法尚不足以防止发生社会危险性,而有逮捕必要的**

**批准机关**:
- 人民检察院批准
- 人民法院决定

**执行机关**:
- 公安机关执行

**逮捕期限**:
| 情况 | 期限 |
|------|------|
| 一般侦查 | 2个月 |
| 案情复杂 | 延长1个月 |
| 重大复杂 | 延长2个月 |
| 特别重大复杂 | 延长2个月 |

**逮捕程序**:
1. **提请批准** - 公安机关提请检察院批准
2. **审查批准** - 检察院审查批准
3. **执行逮捕** - 公安机关执行
4. **通知家属** - 24小时内通知
5. **讯问** - 24小时内讯问

**权利保障**:
- 及时讯问
- 通知家属
- 律师辩护

**实务提示**:  
逮捕是最严厉的强制措施,要严格审批。

---

## 📖 第四章：侦查终结

---

### 五、侦查终结

#### 21. 侦查终结 (Conclusion of Investigation/侦查终结)

**白话解释**:  
查完了,准备移送审查起诉。

**生活例子**:  
- 案情查清楚了
- 证据收集够了
- 写好起诉意见书

**学术定义**:  
侦查机关经过侦查,认为案件事实已经查清,证据确实充分,足以认定犯罪嫌疑人是否犯罪,决定结束侦查的诉讼活动。

**终结条件**:
1. **案件事实清楚** - 查清了案件事实
2. **证据确实充分** - 证据足以定案
3. **法律手续完备** - 法律手续齐全

**终结处理**:
| 情况 | 处理 |
|------|------|
| 犯罪事实清楚,证据充分 | 移送审查起诉 |
| 不构成犯罪 | 撤销案件 |
| 证据不足 | 继续侦查或撤销 |

**移送材料**:
- 起诉意见书
- 证据材料
- 法律手续
- 其他相关材料

**告知权利**:
- 告知犯罪嫌疑人侦查终结
- 告知可以委托辩护人

**实务提示**:  
侦查终结要符合法定条件。

---

#### 22. 起诉意见书 (Prosecution Recommendation/起诉意见书)

**白话解释**:  
公安机关写给检察院的报告,建议起诉。

**生活例子**:  
- 描述犯罪事实
- 列举证据
- 提出起诉建议

**学术定义**:  
公安机关侦查终结后,认为犯罪事实清楚,证据确实充分,依法应当追究刑事责任,向人民检察院移送审查起诉时制作的法律文书。

**主要内容**:
1. **犯罪嫌疑人基本情况** - 姓名、性别、年龄等
2. **案件来源和侦破经过** - 案件怎么来的,怎么破的
3. **犯罪事实** - 犯了什么罪,怎么犯的
4. **证据情况** - 有什么证据证明
5. **法律适用** - 触犯什么法律
6. **起诉建议** - 建议起诉

**写作要求**:
| 要求 | 内容 |
|------|------|
| 客观真实 | 实事求是 |
| 证据充分 | 证据支撑 |
| 逻辑清晰 | 条理清楚 |
| 文字规范 | 用词准确 |

**附送材料**:
- 证据目录
- 证据材料
- 法律手续
- 其他材料

**实务提示**:  
起诉意见书是重要的法律文书。

---

#### 23. 补充侦查 (Supplementary Investigation/补充侦查)

**白话解释**:  
检察院觉得证据不够,退回来让公安继续查。

**生活例子**:  
- 证据不足退回补充侦查
- 查清遗漏事实
- 补充相关证据

**学术定义**:  
人民检察院在审查起诉过程中,认为案件事实不清、证据不足或遗漏罪行,将案件退回公安机关进行补充调查的诉讼活动。

**退回情形**:
1. **事实不清** - 案件事实没查清
2. **证据不足** - 证据不够充分
3. **遗漏罪行** - 遗漏其他犯罪
4. **遗漏犯罪嫌疑人** - 遗漏其他犯罪嫌疑人

**退回程序**:
| 程序 | 内容 |
|------|------|
| 审查 | 检察院审查 |
| 决定 | 决定退回 |
| 通知 | 通知公安机关 |
| 补充 | 公安机关补充侦查 |
| 重报 | 重新移送 |

**补充侦查期限**:
- 1个月
- 以2次为限

**补充后处理**:
- 证据充分,重新移送
- 证据仍然不足,检察院作出不起诉决定

**实务提示**:  
补充侦查要针对问题补充,不能敷衍。

---

#### 24. 撤销案件 (Dismissal of Case/撤销案件)

**白话解释**:  
不查了,撤销案件。

**生活例子**:  
- 没有犯罪事实,撤销
- 不构成犯罪,撤销

**学术定义**:  
公安机关在侦查过程中,发现不应对犯罪嫌疑人追究刑事责任,依法撤销案件的诉讼活动。

**撤销情形**:
1. **没有犯罪事实** - 根本没发生犯罪
2. **情节显著轻微** - 情节太轻,不认为是犯罪
3. **已过追诉时效** - 超过追诉期限
4. **经特赦令免除** - 特赦免罪
5. **犯罪嫌疑人死亡** - 人死了
6. **其他依法不追究** - 其他情形

**撤销程序**:
| 程序 | 内容 |
|------|------|
| 发现 | 发现撤销情形 |
| 审查 | 审查是否符合条件 |
| 决定 | 决定撤销 |
| 释放 | 释放被羁押人 |
| 通知 | 通知相关人员 |

**后续处理**:
- 释放被羁押人
- 解除强制措施
- 发还扣押物品
- 通知报案人

**实务提示**:  
撤销案件要及时,保护当事人权利。

## 💡 实务应用场景

### 场景1: 立案审查
- **问题**: 接到报案后如何决定是否立案?
- **回答**: 
  - 审查报案材料
  - 判断是否有犯罪事实
  - 判断是否需要追究责任
  - 判断是否属于本机关管辖
  - 符合条件的立案

### 场景2: 现场勘查
- **问题**: 接到命案报案后如何现场勘查?
- **回答**: 
  1. 第一时间保护现场
  2. 立即到达现场
  3. 全面细致勘查
  4. 提取痕迹物证
  5. 拍照录像绘图
  6. 制作勘验笔录

### 场景3: 侦查讯问
- **问题**: 如何合法有效讯问犯罪嫌疑人?
- **回答**: 
  - 依法告知权利
  - 不得刑讯逼供
  - 客观全面记录
  - 核对后签名
  - 全程录音录像
