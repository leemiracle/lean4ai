## 📖 第一章：人口学基本概念

---

### 一、人口统计学基础

#### 1. 人口学 (Demography/人口学)

**白话解释**:  
研究人的数量、结构、分布和变化的学问。

**生活例子**:  
- 统计有多少人 → 人口普查
- 研究生多少孩子 → 生育研究
- 分析活多久 → 寿命研究
- 研究往哪搬 → 迁移研究

**学术定义**:  
研究人口数量、结构、分布、变动及其与社会经济发展相互关系的科学。

**研究对象**:
1. **人口数量** - 有多少人
2. **人口结构** - 男女老少比例
3. **人口分布** - 人在哪里
4. **人口变动** - 生死迁移

**研究方法**:
| 方法 | 说明 |
|------|------|
| 人口普查 | 全面调查 |
| 抽样调查 | 部分调查 |
| 人口登记 | 日常记录 |
| 统计分析 | 数据分析 |

**人口学分支**:
1. **人口统计学** - 数据统计
2. **人口经济学** - 人口与经济
3. **人口社会学** - 人口与社会
4. **人口地理学** - 人口分布

**实务提示**:  
人口是一切社会活动的基础。

---

#### 2. 人口普查 (Population Census/人口普查)

**白话解释**:  
国家数人头，看看有多少人、都是谁。

**生活例子**:  
- 每10年一次 → 2020年第七次普查
- 上门登记 → 普查员入户
- 填表 → 回答问题

**学术定义**:  
在国家统一规定的时间内，按照统一的方法、统一的项目、统一的调查表和统一的标准时点，对全国人口普遍地、逐户逐人地进行的一次性调查登记。

**中国人口普查历史**:
| 普查 | 年份 | 人口数 |
|------|------|--------|
| 第一次 | 1953年 | 6.0亿 |
| 第二次 | 1964年 | 7.2亿 |
| 第三次 | 1982年 | 10.3亿 |
| 第四次 | 1990年 | 11.6亿 |
| 第五次 | 2000年 | 12.7亿 |
| 第六次 | 2010年 | 13.4亿 |
| 第七次 | 2020年 | 14.1亿 |

**普查内容**:
1. **基本信息** - 姓名、性别、年龄
2. **户籍信息** - 户口所在地
3. **教育信息** - 学历
4. **就业信息** - 职业
5. **婚姻信息** - 婚姻状况
6. **住房信息** - 居住情况

**普查意义**:
- 掌握人口底数
- 制定政策依据
- 规划资源分配
- 研究发展趋势

**实务提示**:  
人口普查是最全面的人口统计方式。

---

#### 3. 人口结构 (Population Structure/人口结构)

**白话解释**:  
人口中不同类型人的比例。

**生活例子**:  
- 男女比例 → 性别结构
- 老少比例 → 年龄结构
- 城乡比例 → 城乡结构

**学术定义**:  
人口总体内部各种属性（性别、年龄、民族、文化、职业等）的人口比例关系。

**结构类型**:
1. **性别结构** - 男女比例
2. **年龄结构** - 年龄分布
3. **城乡结构** - 城乡比例
4. **教育结构** - 学历分布
5. **职业结构** - 职业分布

**年龄结构类型**:
| 类型 | 特点 | 人口增长 |
|------|------|----------|
| 年轻型 | 少年儿童多 | 增长快 |
| 成年型 | 劳动人口多 | 增长稳定 |
| 老年型 | 老年人多 | 增长慢或负增长 |

**中国人口结构特点**:
- 性别比偏高 (男多女少)
- 老龄化加速
- 劳动人口开始减少
- 城镇人口超过农村

**实务提示**:  
人口结构决定社会发展方向。

---

#### 4. 人口分布 (Population Distribution/人口分布)

**白话解释**:  
人在什么地方生活。

**生活例子**:  
- 中国东部人多 → 胡焕庸线东南
- 沿海地区人口密集
- 西部地区人口稀少

**学术定义**:  
一定时间内人口在地理空间的聚集、散布状况。

**分布特征**:
1. **不均衡性** - 有的地方多，有的地方少
2. **区域性** - 不同地区密度不同
3. **历史性** - 随时间变化

**影响分布的因素**:
| 因素 | 影响 |
|------|------|
| 自然因素 | 气候、地形、水源 |
| 经济因素 | 就业机会、收入水平 |
| 社会因素 | 文化、历史、政策 |
| 政治因素 | 行政区划、战争 |

**胡焕庸线**:
- 从黑龙江黑河到云南腾冲
- 东南半壁占43%国土、94%人口
- 西北半壁占57%国土、6%人口
- 体现中国人口分布不均

**中国人口分布**:
- 东部密集，西部稀少
- 沿海发达地区人口多
- 平原地区人口密集
- 高原、沙漠地区人口少

**实务提示**:  
人口分布影响区域发展。

---

#### 5. 人口密度 (Population Density/人口密度)

**白话解释**:  
每平方公里有多少人。

**生活例子**:  
- 北京：1300人/km²
- 上海：3800人/km²
- 西藏：3人/km²

**学术定义**:  
单位土地面积上居住的人口数，通常以每平方公里人口数表示。

**计算公式**:  
人口密度 = 总人口 ÷ 土地面积

**密度等级**:
| 密度 | 人数/km² | 类型 |
|------|----------|------|
| 极低 | <1 | 荒漠区 |
| 低 | 1-50 | 稀疏区 |
| 中 | 50-200 | 中等区 |
| 高 | 200-500 | 稠密区 |
| 极高 | >500 | 极密区 |

**中国人口密度**:
- 全国平均：147人/km²
- 最密：澳门、香港、上海
- 最稀：西藏、青海

**密度意义**:
- 反映人口拥挤程度
- 影响资源环境压力
- 影响经济发展方式

**实务提示**:  
人口密度是衡量人口分布的重要指标。

### 二、人口变动

#### 6. 出生率 (Birth Rate/出生率)

**白话解释**:  
一年中出生的婴儿占人口的比例。

**生活例子**:  
- 出生率10‰ → 每千人出生10个婴儿
- 中国2020年出生率8.52‰
- 越穷的国家出生率越高

**学术定义**:  
一定时期内（通常为一年）出生人数与同期平均人数之比，用千分比表示。

**计算公式**:  
出生率 = (年出生人数 ÷ 年平均人数) × 1000‰

**出生率水平**:
| 水平 | 数值 | 类型 |
|------|------|------|
| 高 | >30‰ | 高生育率 |
| 中 | 15-30‰ | 中等生育率 |
| 低 | <15‰ | 低生育率 |
| 超低 | <10‰ | 极低生育率 |

**中国出生率变化**:
| 年份 | 出生率 |
|------|--------|
| 1963年 | 43.37‰ |
| 1980年 | 18.21‰ |
| 2000年 | 14.03‰ |
| 2010年 | 11.90‰ |
| 2020年 | 8.52‰ |
| 2023年 | 6.39‰ |

**影响出生率的因素**:
1. **经济发展水平**
2. **受教育程度**
3. **计划生育政策**
4. **婚姻观念**
5. **生育意愿**

**实务提示**:  
出生率是衡量生育水平的重要指标。

---

#### 7. 死亡率 (Death Rate/死亡率)

**白话解释**:  
一年中死亡人数占人口的比例。

**生活例子**:  
- 死亡率7‰ → 每千人死亡7人
- 中国死亡率约7‰
- 发达国家死亡率反而高 (老龄化)

**学术定义**:  
一定时期内（通常为一年）死亡人数与同期平均人数之比，用千分比表示。

**计算公式**:  
死亡率 = (年死亡人数 ÷ 年平均人数) × 1000‰

**死亡率水平**:
| 水平 | 数值 | 类型 |
|------|------|------|
| 高 | >15‰ | 高死亡率 |
| 中 | 7-15‰ | 中等死亡率 |
| 低 | <7‰ | 低死亡率 |

**影响死亡率的因素**:
1. **医疗卫生水平**
2. **生活水平**
3. **人口年龄结构**
4. **环境污染**
5. **战争、灾害**

**特殊死亡率**:
- **婴儿死亡率** - 不满1岁死亡婴儿与活产婴儿之比
- **孕产妇死亡率** - 孕产妇死亡与活产数之比

**中国死亡率变化**:
- 1949年前：25‰以上
- 1980年：6.34‰
- 2020年：7.07‰
- 近年略有上升 (老龄化)

**实务提示**:  
死亡率反映社会发展水平。

---

#### 8. 自然增长率 (Natural Growth Rate/自然增长率)

**白话解释**:  
出生率减死亡率，就是人口自然增长的速度。

**生活例子**:  
- 出生率10‰，死亡率7‰ → 自然增长率3‰
- 正数 → 人口增加
- 负数 → 人口减少

**学术定义**:  
一定时期内（通常为一年）人口自然增加数（出生人数减死亡人数）与同期平均人数之比。

**计算公式**:  
自然增长率 = 出生率 - 死亡率

**增长类型**:
| 类型 | 数值 | 特点 |
|------|------|------|
| 正增长 | >0 | 人口增加 |
| 零增长 | =0 | 人口不变 |
| 负增长 | <0 | 人口减少 |

**中国自然增长率变化**:
| 年份 | 自然增长率 |
|------|------------|
| 1963年 | 33.33‰ |
| 1980年 | 11.87‰ |
| 2000年 | 7.58‰ |
| 2010年 | 4.79‰ |
| 2020年 | 1.45‰ |
| 2022年 | -0.85‰ (首次负增长) |

**人口转变理论**:
```
第一阶段：高出生、高死亡、低增长
    ↓
第二阶段：高出生、低死亡、高增长
    ↓
第三阶段：低出生、低死亡、低增长
    ↓
第四阶段：低出生、低死亡、零增长或负增长
```

**实务提示**:  
自然增长率是判断人口趋势的关键指标。

---

#### 9. 人口迁移 (Population Migration/人口迁移)

**白话解释**:  
人从一个地方搬到另一个地方长期居住。

**生活例子**:  
- 农村人进城 → 城乡迁移
- 北方人去南方 → 区域迁移
- 出国定居 → 国际迁移

**学术定义**:  
人口在地理空间上的位置改变，通常指跨越一定行政区界线、改变居住地的长期性移动。

**迁移类型**:
| 类型 | 说明 | 例子 |
|------|------|------|
| 国内迁移 | 国内流动 | 农民工进城 |
| 国际迁移 | 跨国流动 | 移民、留学 |
| 永久迁移 | 长期定居 | 移民他国 |
| 暂时迁移 | 短期流动 | 务工、求学 |

**迁移原因** (推拉理论):
1. **推力** - 原居地不利因素
   - 就业机会少
   - 收入低
   - 生活条件差

2. **拉力** - 目的地有利因素
   - 就业机会多
   - 收入高
   - 生活条件好

**中国人口流动特点**:
- 规模大：近3亿流动人口
- 方向：从农村到城市、从西部到东部
- 特点：务工经商为主
- 问题：户籍限制、子女教育

**迁移影响**:
- 改变人口分布
- 影响经济发展
- 带来社会问题
- 促进文化交流

**实务提示**:  
人口迁移是影响人口分布的重要因素。

---

#### 10. 净迁移率 (Net Migration Rate/净迁移率)

**白话解释**:  
迁入的人减去迁出的人的比例。

**生活例子**:  
- 北京每年净迁入几十万人
- 东北人口净流出
- 深圳净迁入人口多

**学术定义**:  
一定时期内（通常为一年）迁入人数与迁出人数之差与同期平均人数之比。

**计算公式**:  
净迁移率 = [(迁入人数 - 迁出人数) ÷ 平均人数] × 1000‰

**迁移平衡类型**:
| 类型 | 净迁移率 | 说明 |
|------|----------|------|
| 净迁入 | >0 | 人口流入 |
| 平衡 | ≈0 | 流入流出相当 |
| 净迁出 | <0 | 人口流出 |

**中国区域迁移特点**:
- 净迁入：广东、浙江、江苏、北京、上海
- 净迁出：河南、四川、安徽、湖南、黑龙江

**迁移影响**:
| 净迁入地区 | 净迁出地区 |
|------------|------------|
| 劳动力充足 | 劳动力流失 |
| 经济发展快 | 发展乏力 |
- 基础设施压力大 | 空心化严重 |

**实务提示**:  
净迁移率反映地区吸引力。

### 一、生育问题

#### 11. 生育率 (Fertility Rate/生育率)

**白话解释**:  
一个女人一生平均生几个孩子。

**生活例子**:  
- 生育率2.1 → 人口不增不减
- 生育率1.0 → 人口减少
- 中国生育率约1.3

**学术定义**:  
衡量生育水平的各种指标，最常用的是总和生育率。

**生育率指标**:
1. **一般生育率** = 出生人数 ÷ 育龄妇女数
2. **总和生育率 (TFR)** = 妇女一生平均生育子女数
3. **年龄别生育率** = 某年龄妇女生育率

**总和生育率水平**:
| TFR | 含义 |
|-----|------|
| >2.1 | 人口增长 |
| =2.1 | 更替水平 (人口稳定) |
| <2.1 | 人口减少 |
| <1.5 | 很低生育率 |
| <1.3 | 极低生育率 |

**中国生育率变化**:
| 年份 | 总和生育率 |
|------|------------|
| 1960年代 | 6.0左右 |
| 1980年 | 2.6 |
| 2000年 | 1.22 |
| 2010年 | 1.18 |
| 2020年 | 1.30 |
| 2022年 | 1.09 |

**低生育率原因**:
1. **经济压力** - 养孩子贵
2. **教育竞争** - 教育投入大
3. **职业发展** - 影响工作
4. **观念变化** - 不想生
5. **结婚晚** - 育龄缩短

**更替水平**:  
TFR=2.1，人口长期保持稳定不增不减。

**实务提示**:  
生育率决定未来人口趋势。

---

#### 12. 计划生育政策 (Family Planning Policy/计划生育政策)

**白话解释**:  
国家规定能生几个孩子。

**生活例子**:  
- 独生子女政策 → 只能生一个
- 二孩政策 → 可以生两个
- 三孩政策 → 可以生三个

**学术定义**:  
国家通过政策、法律手段对人口生育进行调节和控制的制度。

**中国计划生育历程**:
| 时间 | 政策 | 内容 |
|------|------|------|
| 1970年代 | 晚稀少 | 晚婚晚育、少生 |
| 1980-2015 | 独生子女 | 一对夫妇一个孩子 |
| 2016-2021 | 全面二孩 | 可以生两个 |
| 2021至今 | 三孩政策 | 可以生三个 |

**政策效果**:
| 成就 | 问题 |
|------|------|
| 少生4亿人 | 老龄化加速 |
| 促进经济发展 | 性别比失衡 |
| 提高妇女地位 | 劳动力减少 |
| 改善生活水平 | 独生子女问题 |

**配套措施**:
- 延长产假
- 生育补贴
- 托育服务
- 税收优惠
- 住房支持

**其他国家生育政策**:
- 日本、韩国：鼓励生育
- 法国、北欧：高福利鼓励
- 新加坡：从控制到鼓励

**实务提示**:  
人口政策需要根据发展阶段调整。

---

#### 13. 低生育率陷阱 (Low Fertility Trap/低生育率陷阱)

**白话解释**:  
生育率降到很低就很难回升。

**生活例子**:  
- 日本生育率长期低迷
- 韩国生育率全球最低
- 欧洲很多国家生育率低

**学术定义**:  
生育率一旦降到很低水平，就会产生自我强化的机制，很难再回升的现象。

**陷阱机制**:
1. **人口学机制** - 育龄妇女减少
2. **社会学机制** - 理想家庭规模变小
3. **经济学机制** - 生育成本上升

**低生育率后果**:
| 后果 | 影响 |
|------|------|
| 人口减少 | 劳动力不足 |
| 老龄化 | 养老压力 |
| 经济萎缩 | 增长乏力 |
| 社会保障 | 负担加重 |

**应对措施**:
1. **经济激励** - 生育补贴
2. **服务支持** - 托育服务
3. **时间保障** - 产假制度
4. **文化引导** - 宣传引导

**实务提示**:  
避免陷入低生育率陷阱至关重要。

### 二、老龄化问题

#### 14. 人口老龄化 (Population Aging/人口老龄化)

**白话解释**:  
老人越来越多，占的比例越来越大。

**生活例子**:  
- 60岁以上超过10% → 老龄化社会
- 65岁以上超过14% → 深度老龄化
- 中国正在快速老龄化

**学术定义**:  
老年人口在总人口中所占比例不断上升的过程。

**老龄化标准** (联合国):
| 65岁以上占比 | 类型 |
|--------------|------|
| <4% | 年轻型 |
| 4-7% | 成年型 |
| 7-14% | 老龄化社会 |
| 14-21% | 深度老龄化 |
| >21% | 超级老龄化 |

**中国老龄化进程**:
| 年份 | 65岁以上占比 |
|------|--------------|
| 2000年 | 7.0% |
| 2010年 | 8.9% |
| 2020年 | 13.5% |
| 2035年 | 20%+ (预计) |
| 2050年 | 25%+ (预计) |

**老龄化原因**:
1. **生育率下降** - 孩子少了
2. **寿命延长** - 老人活得久了
3. **医疗进步** - 更健康长寿

**老龄化影响**:
| 积极影响 | 消极影响 |
|----------|----------|
| 银发经济 | 劳动力减少 |
| 智慧积累 | 养老压力 |
| 社会稳定 | 医疗负担 |
| | 经济增长放缓 |

**应对老龄化**:
1. **延迟退休** - 延长工作年限
2. **完善养老** - 养老保障体系
3. **发展银发经济** - 老年产业
4. **鼓励生育** - 提高生育率

**实务提示**:  
老龄化是全球性趋势，需要积极应对。

#### 15. 抚养比 (Dependency Ratio/抚养比)

**白话解释**:  
需要养的人占能干活的人的比例。

**生活例子**:  
- 抚养比40% → 100个劳动人口养40个非劳动人口
- 老人多了 → 老年抚养比上升
- 孩子少了 → 少儿抚养比下降

**学术定义**:  
非劳动年龄人口与劳动年龄人口之比，表明每100名劳动年龄人口需要负担多少非劳动年龄人口。

**计算公式**:  
抚养比 = (少儿人口 + 老年人口) ÷ 劳动年龄人口 × 100%

**抚养比类型**:
| 类型 | 计算 |
|------|------|
| 总抚养比 | (0-14岁 + 65岁以上) ÷ 15-64岁 |
| 少儿抚养比 | 0-14岁 ÷ 15-64岁 |
| 老年抚养比 | 65岁以上 ÷ 15-64岁 |

**中国抚养比变化**:
| 年份 | 总抚养比 | 老年抚养比 |
|------|----------|------------|
| 1982年 | 62.6% | 8.0% |
| 2000年 | 42.6% | 9.9% |
| 2010年 | 34.2% | 11.9% |
| 2020年 | 45.9% | 19.7% |
| 2035年 | 50%+ (预计) | 30%+ (预计) |

**人口红利**:  
劳动人口多、抚养比低，有利于经济发展的时期。

**人口红利消失**:  
抚养比上升，劳动力减少，经济增长放缓。

**实务提示**:  
抚养比反映人口负担程度。

#### 16. 养老保险 (Pension Insurance/养老保险)

**白话解释**:  
工作时交钱，老了领钱的制度。

**生活例子**:  
- 每月交养老保险 → 退休后领养老金
- 城镇职工养老保险 → 上班族
- 城乡居民养老保险 → 农民、无业者

**学术定义**:  
国家和社会根据一定的法律和法规，为解决劳动者在达到国家规定的解除劳动义务的劳动年龄界限，或因年老丧失劳动能力退出劳动岗位后的基本生活而建立的一种社会保险制度。

**中国养老保险体系**:
```
三支柱体系
├─ 第一支柱：基本养老保险
│   ├─ 城镇职工基本养老保险
│   └─ 城乡居民基本养老保险
├─ 第二支柱：企业年金/职业年金
└─ 第三支柱：个人养老金
```

**养老金计算**:
- 基础养老金 + 个人账户养老金
- 多缴多得、长缴多得

**养老金压力**:
| 问题 | 原因 |
|------|------|
| 收不抵支 | 老人增多、缴费人少 |
| 地区不平衡 | 人口流动 |
| 替代率下降 | 退休金占工资比例下降 |

**延迟退休**:  
逐步延迟退休年龄，缓解养老金压力。

**实务提示**:  
养老保险是应对老龄化的重要保障。

### 三、其他人口问题

#### 17. 性别比失衡 (Sex Ratio Imbalance/性别比失衡)

**白话解释**:  
男多女少，比例不正常。

**生活例子**:  
- 正常：100个女婴对应103-107个男婴
- 中国：100个女婴对应111个男婴 (出生时)
- 三千万光棍 → 找不到老婆

**学术定义**:  
人口中男女人数不相等，特别是出生性别比偏离正常范围的现象。

**性别比指标**:
| 性别比 | 计算 |
|--------|------|
| 总性别比 | 男性人口 ÷ 女性人口 × 100 |
| 出生性别比 | 男婴数 ÷ 女婴数 × 100 |

**正常范围**:  
出生性别比：103-107

**中国性别比变化**:
| 年份 | 出生性别比 |
|------|------------|
| 1982年 | 108.5 |
| 1990年 | 111.3 |
| 2000年 | 116.9 |
| 2010年 | 118.1 |
| 2020年 | 111.3 |

**失衡原因**:
1. **传统观念** - 重男轻女
2. **性别选择** - B超、流产
3. **政策影响** - 独生子女政策
4. **经济因素** - 农村需要男劳动力

**失衡后果**:
| 后果 | 影响 |
|------|------|
| 婚姻挤压 | 男性难找对象 |
| 社会稳定 | 影响社会和谐 |
| 人口再生产 | 影响生育 |
| 性别歧视 | 加剧不平等 |

**治理措施**:
- 禁止非医学需要的性别鉴定
- 打击选择性别人工终止妊娠
- 宣传男女平等
- 完善社会保障

**实务提示**:  
性别比失衡影响社会稳定。

---

#### 18. 人口红利 (Demographic Dividend/人口红利)

**白话解释**:  
干活的人多、要养的人少，经济发展容易。

**生活例子**:  
- 中国改革开放后 → 人口红利期
- 劳动力充足 → 工厂不愁招人
- 抚养比低 → 储蓄率高

**学术定义**:  
当人口结构呈现"中间大、两头小"的特征，即劳动年龄人口占比高、抚养比低时，有利于经济发展的时期。

**红利阶段**:
```
人口红利期：抚养比 < 50%
    ├─ 第一阶段：少儿抚养比下降
    ├─ 第二阶段：劳动力充足
    └─ 第三阶段：老年抚养比上升
```

**中国人口红利**:
- 1980-2010年：人口红利期
- 2010年后：红利开始消失
- 2030年后：转为人口负债

**红利效应**:
| 效应 | 说明 |
|------|------|
| 劳动力充足 | 供给充足、成本低 |
| 储蓄率高 | 负担轻、储蓄多 |
| 投资增加 | 资金充裕 |
| 经济增长 | 增长快 |

**红利消失的影响**:
- 劳动力成本上升
- 经济增长放缓
- 养老压力增加
- 需要转型升级

**人才红利**:  
从人口数量红利转向人口质量红利。

**实务提示**:  
人口红利是有限的，要抓住机会发展。

#### 19. 人口政策 (Population Policy/人口政策)

**白话解释**:  
国家怎么管人口的规矩。

**生活例子**:  
- 计划生育 → 控制人口
- 鼓励生育 → 增加人口
- 户籍改革 → 管理人口

**学术定义**:  
国家为调节人口数量、提高人口质量、改善人口结构、引导人口分布而制定的行为准则。

**政策类型**:
| 类型 | 目的 | 例子 |
|------|------|------|
| 限制性政策 | 控制人口 | 独生子女政策 |
| 鼓励性政策 | 增加人口 | 生育补贴 |
| 中性政策 | 不干预 | 自由生育 |

**中国人口政策演变**:
| 时期 | 政策 | 目标 |
|------|------|------|
| 1950年代 | 鼓励生育 | 增加人口 |
| 1970-2015 | 计划生育 | 控制人口 |
| 2016至今 | 鼓励生育 | 应对低生育 |

**当前人口政策**:
1. **三孩政策** - 可以生三个
2. **配套措施** - 托育、教育、住房
3. **户籍改革** - 放宽落户
4. **人才引进** - 吸引人口

**政策效果评估**:
- 短期效果 vs 长期效果
- 直接效果 vs 间接效果
- 积极效果 vs 消极效果

**实务提示**:  
人口政策需要根据国情调整。

#### 20. 人口预测 (Population Projection/人口预测)

**白话解释**:  
估算未来会有多少人口。

**生活例子**:  
- 2030年中国人口多少？
- 2050年印度人口多少？
- 未来劳动力够不够？

**学术定义**:  
根据人口现状和发展规律，运用科学方法对未来人口数量、结构、分布进行估算。

**预测方法**:
1. **数学模型法** - 公式推算
2. **队列要素法** - 分年龄推算
3. **情景分析法** - 不同假设

**中国人口预测**:
| 年份 | 总人口 | 峰值 |
|------|--------|------|
| 2022年 | 14.1亿 | - |
| 2029年 | - | 14.4亿 (峰值) |
| 2050年 | 13.6亿 | - |
| 2100年 | 10亿左右 | - |

**预测不确定性**:
- 生育率变化
- 死亡率变化
- 迁移变化
- 政策变化

**预测用途**:
- 制定政策
- 规划资源
- 安排财政
- 预警风险

**实务提示**:  
人口预测为决策提供依据。
