## 📖 第一章：人类学基础

---

### 一、人类学的基本概念

#### 1. 人类学 (Anthropology/人类学)

**白话解释**:  
研究人的学问，包括人的生物性和文化性。

**生活例子**:  
- 研究北京人化石 → 古人类学
- 研究少数民族文化 → 文化人类学
- 研究人体结构 → 体质人类学

**学术定义**:  
研究人类的起源、进化、变异以及人类所创造的文化的科学。

**人类学分支**:
```
人类学
├─ 体质人类学 (研究人的生物性)
│   ├─ 古人类学 (人类起源)
│   ├─ 人体学 (人体形态)
│   └─ 灵长类学 (猿猴研究)
└─ 文化人类学 (研究人的文化)
    ├─ 民族学 (民族文化)
    ├─ 考古学 (古代文化)
    └─ 语言人类学 (语言文化)
```

**研究方法**:
| 方法 | 说明 |
|------|------|
| 田野调查 | 到现场长期观察 |
| 参与观察 | 参与当地生活 |
| 深度访谈 | 深入交谈 |
| 民族志 | 描述文化 |

**人类学特点**:
1. **整体性** - 把文化看作整体
2. **比较性** - 比较不同文化
3. **相对性** - 不用自己标准评价
4. **田野性** - 强调实地调查

**实务提示**:  
人类学帮助我们理解"他者"。

---

#### 2. 文化 (Culture/文化)

**白话解释**:  
一群人共同的活法，包括知识、信仰、艺术、道德、法律、风俗等。

**生活例子**:  
- 中国人用筷子 → 饮食文化
- 春节团圆 → 节日文化
- 孝顺父母 → 伦理文化

**学术定义**:  
人类在社会实践中创造的物质财富和精神财富的总和，是人类社会特有的现象。

**文化要素**:
1. **物质文化** - 工具、建筑、器物
2. **制度文化** - 规范、制度、组织
3. **精神文化** - 价值观、信仰、艺术

**文化特征**:
| 特征 | 说明 |
|------|------|
| 习得性 | 后天学习，非遗传 |
| 共享性 | 群体成员共享 |
| 象征性 | 有象征意义 |
| 整体性 | 各部分相互联系 |
| 变异性 | 不同群体不同 |
| 传承性 | 代代相传 |

**文化功能**:
1. **适应功能** - 帮助适应环境
2. **整合功能** - 整合社会
3. **导向功能** - 指导行为
4. **认同功能** - 提供身份认同

**文化类型**:
- 主流文化 vs 亚文化
- 传统文化 vs 现代文化
- 本土文化 vs 外来文化

**实务提示**:  
文化是人类学研究的核心概念。

---

#### 3. 文化相对主义 (Cultural Relativism/文化相对主义)

**白话解释**:  
不要用自己的标准评价别人的文化。

**生活例子**:  
- 部落人裸体不是落后 → 适应热带气候
- 吃虫子不是野蛮 → 营养丰富
- 一夫多妻不一定不好 → 特定环境下合理

**学术定义**:  
认为任何文化都有其存在的合理性，应该从该文化自身的角度去理解，而不应该用其他文化的标准去评判。

**核心观点**:
1. **没有优劣之分** - 文化没有高级低级
2. **情境理解** - 放在具体情境中理解
3. **避免民族中心主义** - 不以自我为中心

**民族中心主义**:  
用自己的文化标准去评价其他文化，认为自己的文化最好。

**相对主义的边界**:
| 支持 | 批评 |
|------|------|
| 尊重多样性 | 可能为恶习辩护 |
| 避免偏见 | 可能否定普世价值 |
| 促进理解 | 可能陷入道德相对主义 |

**平衡**:  
既要尊重文化多样性，也要坚持基本人权。

**实务提示**:  
理解文化相对主义是学人类学的第一步。

---

#### 4. 民族中心主义 (Ethnocentrism/民族中心主义)

**白话解释**:  
认为自己的文化最好，别人都不如自己。

**生活例子**:  
- 西方人认为西方文明最先进
- 中国人认为中华文明最优秀
- 都在用自己的标准看别人

**学术定义**:  
以自己所属群体的文化为标准，去评价其他群体的文化，认为自己群体的文化优于其他群体的倾向。

**表现形式**:
1. **自我中心** - 我们最好
2. **歧视他者** - 他们不好
3. **文化优越感** - 我们文明
4. **文化偏见** - 他们野蛮

**民族中心主义危害**:
| 危害 | 例子 |
|------|------|
| 文化歧视 | 种族歧视 |
| 冲突战争 | 文化冲突 |
| 沟通障碍 | 理解困难 |
| 合作失败 | 无法合作 |

**克服方法**:
1. **文化相对主义** - 相对地看问题
2. **跨文化理解** - 理解他者
3. **批判性反思** - 反思自己
4. **多元文化教育** - 多元教育

**实务提示**:  
民族中心主义是理解他者的最大障碍。

#### 5. 田野调查 (Fieldwork/田野调查)

**白话解释**:  
到研究对象生活的地方，跟他们一起生活，观察研究。

**生活例子**:  
- 人类学家住在部落里一年
- 跟着当地人一起干活、吃饭
- 记录他们的生活

**学术定义**:  
研究者深入研究对象所处的自然环境和社会环境，通过参与观察、访谈等方法收集资料的调查研究方法。

**田野调查特点**:
1. **长期性** - 通常一年以上
2. **参与性** - 参与当地生活
3. **整体性** - 全面了解文化
4. **主观性** - 研究者本身就是工具

**调查方法**:
| 方法 | 说明 |
|------|------|
| 参与观察 | 参与当地生活观察 |
| 深度访谈 | 深入交谈了解 |
| 生活史 | 记录个人生活经历 |
| 谱系法 | 记录亲属关系 |
| 收集文物 | 收集物质文化 |

**田野调查步骤**:
```
准备阶段
    ↓
进入田野
    ↓
建立关系
    ↓
收集资料
    ↓
分析资料
    ↓
撰写民族志
```

**田野伦理**:
1. **知情同意** - 告知研究目的
2. **匿名保护** - 保护被研究者
3. **互惠原则** - 给当地回馈
4. **尊重文化** - 尊重当地文化

**实务提示**:  
田野调查是人类学的基本功。

---

#### 6. 民族志 (Ethnography/民族志)

**白话解释**:  
详细记录一个民族文化的文字作品。

**生活例子**:  
- 《菊与刀》 → 描述日本文化
- 《江村经济》 → 描述中国农村
- 《纳文》 → 描述新几内亚部落

**学术定义**:  
基于田野调查，对特定文化进行全面、深入描述的文本。

**民族志内容**:
1. **环境背景** - 地理、历史
2. **社会结构** - 亲属、政治、经济
3. **文化习俗** - 信仰、仪式、艺术
4. **日常生活** - 吃穿住行
5. **变迁过程** - 文化变化

**优秀民族志标准**:
| 标准 | 要求 |
|------|------|
| 真实性 | 如实记录 |
| 整体性 | 全面描述 |
| 深刻性 | 深入分析 |
| 可读性 | 通俗易懂 |

**民族志写作**:
- 厚描述 (Thick Description) - 深入细致描述
- 主位视角 (Emic) - 当地人视角
- 客位视角 (Etic) - 研究者视角

**经典民族志**:
1. 马林诺夫斯基《西太平洋的航海者》
2. 米德《萨摩亚人的成年》
3. 费孝通《江村经济》
4. 格尔茨《深层的游戏》

**实务提示**:  
民族志是人类学研究的成果形式。

### 二、社会文化人类学

#### 7. 亲属制度 (Kinship System/亲属制度)

**白话解释**:  
人与人之间因为血缘或婚姻形成的关系网络。

**生活例子**:  
- 父母子女 → 血缘关系
- 夫妻 → 婚姻关系
- 堂兄弟 → 父系亲属
- 表兄弟 → 母系亲属

**学术定义**:  
社会根据血缘和婚姻关系，规定人们之间相互关系和义务的制度体系。

**亲属类型**:
1. **血缘亲属** - 有血缘关系
   - 父母、子女、兄弟姐妹

2. **姻缘亲属** - 婚姻形成
   - 配偶、配偶亲属

3. **拟制亲属** - 法律认定
   - 养父母、养子女

**亲属称谓制度**:
| 类型 | 特点 | 例子 |
|------|------|------|
| 爱斯基摩制 | 区分核心家庭 | 父母、叔伯分开 |
| 夏威夷制 | 不分父系母系 | 所有同辈叫兄弟姐妹 |
| 易洛魁制 | 分父系母系 | 父系兄弟叫父亲 |
| 苏丹制 | 详细区分 | 每种关系有称谓 |

**继嗣制度**:
1. **父系继嗣** - 沿父亲血脉
2. **母系继嗣** - 沿母亲血脉
3. **双系继嗣** - 父母双方
4. **双边继嗣** - 可选父系或母系

**中国亲属制度特点**:
- 父系为主
- 内外有别
- 辈分严格
- 称谓详细

**实务提示**:  
亲属制度是社会的基本组织形式。

---

#### 8. 婚姻制度 (Marriage System/婚姻制度)

**白话解释**:  
社会认可的男女结合方式。

**生活例子**:  
- 一夫一妻 → 现代社会主流
- 一夫多妻 → 古代、部分伊斯兰国家
- 走婚 → 摩梭人

**学术定义**:  
社会认可的、涉及两性结合的、相对稳定的社会关系制度。

**婚姻类型**:
1. **一夫一妻制** - 一男一女
2. **一夫多妻制** - 一男多女
3. **一妻多夫制** - 一女多男
4. **群婚** - 多对夫妻

**婚姻规则**:
| 规则 | 说明 | 例子 |
|------|------|------|
| 乱伦禁忌 | 禁止近亲通婚 | 不能兄妹结婚 |
| 内婚制 | 群体内通婚 | 种姓内婚 |
| 外婚制 | 群体外通婚 | 氏族外婚 |
| 优先婚 | 优先某类婚姻 | 交错从表婚 |

**婚姻形式**:
1. **掠夺婚** - 抢亲
2. **买卖婚** - 聘礼
3. **交换婚** - 换亲
4. **服役婚** - 劳动换婚姻
5. **自由婚** - 自由恋爱

**婚姻功能**:
- 合法的性关系
- 生育子女
- 经济合作
- 社会联盟

**实务提示**:  
婚姻制度因文化而异。

---

#### 9. 宗族 (Lineage/宗族)

**白话解释**:  
同一个祖先传下来的父系家族群体。

**生活例子**:  
- 张氏宗族 → 所有姓张的
- 修族谱 → 记录祖先
- 祠堂 → 祭祖的地方

**学术定义**:  
基于共同祖先认同、有明确世系关系的父系亲属群体。

**宗族要素**:
1. **共同祖先** - 认同一个祖先
2. **父系血缘** - 父系传承
3. **世系清晰** - 知道辈分
4. **组织严密** - 有组织结构

**宗族组织**:
```
宗族
├─ 族长 (领导)
├─ 族规 (规则)
├─ 祠堂 (活动场所)
├─ 族田 (公共财产)
└─ 族谱 (血缘记录)
```

**宗族功能**:
| 功能 | 说明 |
|------|------|
| 祭祀祖先 | 敬祖祭祖 |
| 互助合作 | 互相帮助 |
| 纠纷调解 | 解决矛盾 |
| 教育子弟 | 传承文化 |
| 救济贫困 | 帮助族人 |

**中国宗族特点**:
- 南方宗族势力强
- 农村宗族影响大
- 改革开放后宗族复兴

**宗族变迁**:
- 传统社会：宗族势力大
- 新中国后：宗族受冲击
- 改革开放：宗族复兴
- 现代社会：宗族影响减弱

**实务提示**:  
宗族是中国传统社会的重要组织。

#### 10. 社会组织 (Social Organization/社会组织)

**白话解释**:  
人为了某个目的组织起来的团体。

**生活例子**:  
- 宗族 → 血缘组织
- 村落 → 地缘组织
- 行会 → 业缘组织
- 秘密会社 → 特殊组织

**学术定义**:  
人们为了实现特定目标，按照一定规则组织起来的社会群体。

**组织类型**:
1. **血缘组织** - 基于血缘
   - 家族、宗族、氏族

2. **地缘组织** - 基于地域
   - 村落、同乡会

3. **业缘组织** - 基于职业
   - 行会、工会

4. **年龄级组织** - 基于年龄
   - 青年会、老人会

5. **秘密会社** - 非公开组织
   - 帮会、教门

**组织特征**:
| 特征 | 说明 |
|------|------|
| 目标性 | 有共同目标 |
| 结构性 | 有组织结构 |
| 规范性 | 有行为规范 |
| 持续性 | 相对稳定 |

**中国社会组织特点**:
- 关系本位
- 人情社会
- 差序格局

**实务提示**:  
社会组织是社会运行的重要载体。

### 一、民俗学基础

#### 11. 民俗学 (Folklore/民俗学)

**白话解释**:  
研究老百姓日常生活中的文化习俗。

**生活例子**:  
- 春节习俗 → 节日民俗
- 民间故事 → 口头民俗
- 剪纸 → 物质民俗
- 婚嫁习俗 → 人生民俗

**学术定义**:  
研究民间文化、习俗、传统及其传承规律的学科。

**民俗分类**:
1. **物质民俗** - 衣食住行
2. **社会民俗** - 社会组织
3. **精神民俗** - 信仰禁忌
4. **语言民俗** - 民间文学

**民俗特征**:
| 特征 | 说明 |
|------|------|
| 集体性 | 群体创造共享 |
| 传承性 | 代代相传 |
| 变异性 | 在传承中变化 |
| 地域性 | 不同地方不同 |
| 模式化 | 有固定模式 |

**民俗功能**:
1. **规范功能** - 规范行为
2. **教育功能** - 传播知识
3. **娱乐功能** - 提供娱乐
4. **认同功能** - 身份认同

**研究方法**:
- 田野调查
- 民俗志
- 比较研究

**实务提示**:  
民俗是民间文化的重要载体。

---

#### 12. 民间文学 (Folk Literature/民间文学)

**白话解释**:  
老百姓口口相传的文学作品。

**生活例子**:  
- 牛郎织女 → 民间故事
- 《梁祝》 → 民间传说
- 信天游 → 民间歌谣
- 绕口令 → 民间游戏

**学术定义**:  
由人民大众集体创作、口头传播的文学作品。

**民间文学类型**:
1. **神话** - 解释世界起源
   - 女娲补天、后羿射日

2. **传说** - 历史人物故事
   - 孟姜女哭长城

3. **故事** - 虚构的故事
   - 灰姑娘、白雪公主

4. **歌谣** - 民间歌曲
   - 民歌、童谣

5. **谚语** - 经验总结
   - 谚语、俗语

6. **谜语** - 智力游戏
   - 谜语、脑筋急转弯

**民间文学特征**:
| 特征 | 说明 |
|------|------|
| 口头性 | 口头创作传播 |
| 集体性 | 集体创作修改 |
| 变异性 | 在流传中变化 |
| 传统性 | 有传统母题 |

**民间文学价值**:
1. **文学价值** - 文学创作源泉
2. **史料价值** - 了解历史
3. **文化价值** - 文化传承
4. **教育价值** - 道德教育

**实务提示**:  
民间文学是民族文化的重要组成部分。

---

#### 13. 节日民俗 (Festival Folklore/节日民俗)

**白话解释**:  
特定日子里的传统活动。

**生活例子**:  
- 春节 → 放鞭炮、贴春联
- 中秋节 → 赏月、吃月饼
- 端午节 → 包粽子、赛龙舟

**学术定义**:  
在特定时间周期性举行的各种庆祝活动及其相关的习俗。

**中国传统节日**:
| 节日 | 时间 | 主要活动 |
|------|------|----------|
| 春节 | 农历正月初一 | 团圆、拜年 |
| 元宵节 | 正月十五 | 赏灯、吃元宵 |
| 清明节 | 公历4月5日左右 | 扫墓、踏青 |
| 端午节 | 五月初五 | 粽子、赛龙舟 |
| 七夕节 | 七月初七 | 牛郎织女 |
| 中秋节 | 八月十五 | 赏月、月饼 |
| 重阳节 | 九月初九 | 登高、敬老 |

**节日功能**:
1. **时间标记** - 划分时间
2. **社会整合** - 增强凝聚力
3. **文化传承** - 传递文化
4. **调节生活** - 张弛有度
5. **经济功能** - 促进消费

**节日变迁**:
- 传统节日淡化
- 西方节日流行
- 节日商业化
- 节日复兴

**少数民族节日**:
- 泼水节 (傣族)
- 火把节 (彝族)
- 那达慕 (蒙古族)
- 古尔邦节 (回族)

**实务提示**:  
节日是文化传承的重要载体。

#### 14. 人生礼仪 (Life Cycle Rituals/人生礼仪)

**白话解释**:  
人生重要节点的仪式活动。

**生活例子**:  
- 满月酒 → 出生礼仪
- 成人礼 → 成年礼仪
- 婚礼 → 结婚礼仪
- 葬礼 → 死亡礼仪

**学术定义**:  
围绕人的一生中重要转折点（出生、成年、结婚、死亡）举行的各种仪式和习俗。

**人生礼仪类型**:
1. **诞生礼** - 出生相关
   - 满月、百日、周岁

2. **成年礼** - 成年仪式
   - 冠礼、笄礼

3. **婚礼** - 结婚仪式
   - 订婚、迎娶、回门

4. **丧礼** - 死亡仪式
   - 守灵、出殡、祭祀

**礼仪功能**:
| 功能 | 说明 |
|------|------|
| 过渡功能 | 标志身份转变 |
| 整合功能 | 社会整合 |
| 教育功能 | 传递价值观 |
| 情感功能 | 表达情感 |

**通过礼仪** (范·根纳普):
1. **分离阶段** - 脱离原有身份
2. **阈限阶段** - 过渡状态
3. **聚合阶段** - 进入新身份

**中国人生礼仪特点**:
- 重视家族
- 强调孝道
- 礼仪繁琐
- 地域差异大

**实务提示**:  
人生礼仪是个人社会化的重要环节。

老百姓日常生活中的物质文化。

- 民居 → 四合院、窑洞
- 服饰 → 旗袍、藏袍
- 饮食 → 饺子、月饼
- 工艺 → 剪纸、刺绣

**学术定义**:  
人民在日常生活中创造和使用的物质文化产品及相关习俗。

**物质民俗分类**:
1. **服饰民俗** - 服装、饰品
2. **饮食民俗** - 食物、饮食习惯
3. **居住民俗** - 房屋、村落
4. **交通民俗** - 交通工具
5. **生产民俗** - 生产工具、方式

**中国民居类型**:
| 类型 | 地区 | 特点 |
|------|------|------|

**饮食民俗**:
- 八大菜系
- 南甜北咸
- 春节饺子
- 中秋月饼

**物质民俗价值**:
1. **实用价值** - 满足生活需要
2. **审美价值** - 体现美感
3. **文化价值** - 承载文化
4. **经济价值** - 旅游资源

物质民俗是最直观的文化表现。

### 二、民间信仰

#### 16. 民间信仰 (Folk Belief/民间信仰)

**白话解释**:  
老百姓自发的宗教信仰和迷信活动。

**生活例子**:  
- 烧香拜佛 → 佛教信仰
- 祭祖 → 祖先崇拜
- 风水 → 占卜信仰
- 本命年穿红 → 禁忌习俗

**学术定义**:  
民间自发形成的、非制度化的宗教信仰及相关实践活动。

**民间信仰内容**:
1. **自然崇拜** - 天地日月
2. **祖先崇拜** - 祭祀祖先
3. **神灵崇拜** - 各路神仙
4. **鬼魂崇拜** - 鬼魂信仰
5. **巫术占卜** - 预测未来

**中国民间信仰特点**:
| 特点 | 说明 |
|------|------|
| 多神崇拜 | 信仰多个神灵 |
| 实用主义 | 有求必应 |
| 模糊性 | 体系不严密 |
| 混合性 | 佛道儒混合 |

**民间信仰功能**:
1. **心理功能** - 提供心理安慰
2. **社会功能** - 社会整合
3. **道德功能** - 劝善惩恶
4. **文化功能** - 文化传承

**民间信仰与制度宗教**:
| 民间信仰 | 制度宗教 |
|----------|----------|
| 自发形成 | 有组织 |
| 无经典 | 有经典 |
| 无神职人员 | 有神职人员 |
| 灵活多样 | 相对统一 |

**实务提示**:  
民间信仰是理解民间文化的重要窗口。

#### 17. 禁忌 (Taboo/禁忌)

**白话解释**:  
不能做的事，做了会倒霉。

**生活例子**:  
- 不能打伞进屋 → 倒霉
- 不能送钟 → 不吉利
- 本命年穿红 → 辟邪
- 孕妇不能参加婚礼 → 冲撞

**学术定义**:  
社会对某些行为、言语、对象的禁止，违犯被认为会招致不幸或惩罚。

**禁忌类型**:
1. **行为禁忌** - 不能做的事
2. **言语禁忌** - 不能说的话
3. **饮食禁忌** - 不能吃的东西
4. **触犯禁忌** - 不能接触的事物

**禁忌分类**:
| 类型 | 例子 |
|------|------|
| 岁时禁忌 | 正月不剃头 |
| 人生禁忌 | 孕妇禁忌 |
| 行业禁忌 | 渔民禁忌 |
| 数字禁忌 | 4不吉利 |

**禁忌功能**:
1. **保护功能** - 保护个人
2. **规范功能** - 规范行为
3. **认同功能** - 群体认同
4. **心理功能** - 减少焦虑

**禁忌变迁**:
- 科学发展 → 禁忌减少
- 但仍有心理影响
- 新禁忌产生 (如网络禁忌)

**实务提示**:  
禁忌反映文化的深层心理。

#### 18. 巫术 (Witchcraft/巫术)

**白话解释**:  
用超自然力量影响现实的手段。

**生活例子**:  
- 算命 → 预测未来
- 跳大神 → 治病驱邪
- 画符 → 避邪祈福
- 诅咒 → 伤害他人

**学术定义**:  
人们企图借助超自然力量，通过特定仪式影响或控制客观事物的行为。

**巫术类型**:
1. **白巫术** - 善意的
   - 祈雨、治病

2. **黑巫术** - 恶意的
   - 诅咒、害人

3. **交感巫术** - 通过感应
   - 相似律、接触律

**弗雷泽的巫术原理**:
| 原理 | 说明 | 例子 |
|------|------|------|
| 相似律 | 相似产生相似 | 扎小人 |
| 接触律 | 接触过就有联系 | 用头发施法 |

**巫术功能**:
1. **心理功能** - 减少焦虑
2. **社会功能** - 社会控制
3. **医疗功能** - 心理治疗
4. **解释功能** - 解释不幸

**巫术与宗教区别**:
| 巫术 | 宗教 |
|------|------|
| 强制神灵 | 祈求神灵 |
| 技术性 | 信仰性 |
| 个人操作 | 集体仪式 |

**实务提示**:  
巫术是人类理解世界的一种方式。

#### 19. 文化变迁 (Cultural Change/文化变迁)

**白话解释**:  
文化随时间发生变化。

**生活例子**:  
- 传统文化减少 → 现代文化增多
- 西方文化影响 → 文化融合
- 互联网普及 → 新文化产生

**学术定义**:  
文化内容、结构、功能等发生的变化。

**变迁原因**:
1. **内部原因** - 文化内部发展
2. **外部原因** - 文化接触传播
3. **环境变化** - 适应新环境
4. **技术进步** - 技术推动

**变迁方式**:
| 方式 | 说明 | 例子 |
|------|------|------|
| 发明 | 创造新文化 | 新技术 |
| 发现 | 发现新事物 | 新大陆 |
| 传播 | 文化传播 | 佛教传入 |
| 涵化 | 文化融合 | 西学东渐 |

**传播方式**:
1. **迁移传播** - 人口迁移带来
2. **贸易传播** - 商业交流
3. **战争传播** - 征服带来
4. **媒体传播** - 大众传播

**中国近现代文化变迁**:
- 五四运动 → 文化启蒙
- 改革开放 → 文化开放
- 全球化 → 文化融合
- 互联网 → 文化多元化

**实务提示**:  
文化变迁是文化发展的常态。

---

#### 20. 涵化 (Acculturation/涵化)

**白话解释**:  
不同文化接触后互相影响、发生变化。

**生活例子**:  
- 中国人穿西装 → 西方文化影响
- 外国人学中文 → 中国文化传播
- 肯德基在中国 → 本土化

**学术定义**:  
不同文化群体在持续直接接触中，原有文化模式发生变化的过程。

**涵化类型**:
1. **同化** - 少数融入多数
2. **融合** - 相互融合
3. **分离** - 保持独立
4. **边缘化** - 两边都不属
5. **整合** - 保持认同同时融入

**涵化结果**:
| 结果 | 说明 |
|------|------|
| 文化借用 | 借用他文化元素 |
| 文化适应 | 调整适应新文化 |
| 文化抗拒 | 抵制外来文化 |
| 文化复兴 | 复兴传统文化 |

**中国涵化现象**:
- 西装、西餐 → 西方文化
- 圣诞节、情人节 → 西方节日
- 但也本土化 → 中国化

**全球化与文化多样性**:
- 同质化 vs 多样性
- 西方化 vs 本土化
- 需要平衡发展

**实务提示**:  
涵化是文化接触的必然结果。

## 💡 实务应用场景

### 场景1: 理解文化差异
- **问题**: 为什么有的民族吃虫子？
- **回答**: 
  1. 文化相对主义 - 不同文化有不同逻辑
  2. 适应环境 - 热带地区蛋白质来源
  3. 美味营养 - 不是所有文化都认为虫子恶心

### 场景2: 分析民俗现象
- **问题**: 为什么春节要贴春联？
- **回答**: 
  1. 驱邪祈福 - 驱除邪气、祈求福气
  2. 文化传承 - 代代相传的习俗
  3. 身份认同 - 中国人过中国节

### 场景3: 理解文化变迁
- **问题**: 传统文化会消失吗？
- **回答**: 
  1. 文化在变迁中发展
  2. 传统会以新形式存在
  3. 保护与创新并重
