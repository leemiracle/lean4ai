## 📖 第一章：社会学的基本概念

---

### 一、社会学的定义与对象

#### 1. 社会学 (Sociology/社会学)

**白话解释**:  
研究社会如何运转、人如何在群体中生活的一门学问。

**生活例子**:  
- 为什么城里人比乡下人更冷漠？
- 为什么穷人的孩子更容易穷？
- 为什么现在结婚的越来越少？

**学术定义**:  
研究人类社会行为、社会关系、社会结构、社会变迁及其规律的科学。

**研究对象**:
1. **社会行为** - 人的行为如何受社会影响
2. **社会关系** - 人与人之间的关系
3. **社会结构** - 社会的组织方式
4. **社会变迁** - 社会如何变化

**社会学特点**:
| 特点 | 说明 |
|------|------|
| 整体性 | 把社会看作整体 |
| 实证性 | 用事实说话 |
| 批判性 | 反思社会问题 |
| 应用性 | 解决实际问题 |

**社会学想象力**:  
把个人困扰放在社会大背景中理解的能力。

**实务提示**:  
社会学帮助我们理解"为什么"。

---

#### 2. 社会 (Society/社会)

**白话解释**:  
一群人长期生活在一起，形成了共同的文化和规则。

**生活例子**:  
- 中国社会 → 14亿人共同生活
- 网络社会 → 网民形成的虚拟社会
- 校园社会 → 学校里的小社会

**学术定义**:  
以共同的物质生产活动为基础而相互联系的人类生活共同体。

**社会要素**:
1. **人口** - 一定数量的人
2. **文化** - 共同的价值观念
3. **制度** - 行为规范体系
4. **组织** - 社会组织形式
5. **互动** - 人与人之间的交往

**社会特征**:
| 特征 | 说明 |
|------|------|
| 长期性 | 不是临时聚集 |
| 互动性 | 人与人互动 |
| 规范性 | 有规则约束 |
| 文化性 | 有共同文化 |
| 结构性 | 有组织结构 |

**社会类型**:
1. **原始社会** - 狩猎采集
2. **农业社会** - 农耕为主
3. **工业社会** - 工业生产
4. **信息社会** - 信息产业
5. **后工业社会** - 服务业为主

**实务提示**:  
社会是人组成的，没有人就没有社会。

---

#### 3. 社会结构 (Social Structure/社会结构)

**白话解释**:  
社会的"骨架"，各种社会关系的组织方式。

**生活例子**:  
- 家庭 → 社会的基本单位
- 学校 → 教育机构
- 公司 → 经济组织
- 政府 → 政治机构

**学术定义**:  
社会系统中各种社会地位、社会角色、社会群体、社会制度的相对稳定的排列组合。

**结构要素**:
1. **地位 (Status)** - 在社会中的位置
2. **角色 (Role)** - 对应地位的行为期待
3. **群体 (Group)** - 有共同特征的人群
4. **制度 (Institution)** - 稳定的行为规范

**地位类型**:
| 类型 | 特点 | 例子 |
|------|------|------|
| 先赋地位 | 天生的 | 性别、种族 |
| 自致地位 | 后天获得的 | 职业、学历 |

**角色冲突**:  
一个人扮演多个角色时产生的矛盾。

**结构功能**:  
社会结构维持社会运转的功能。

**实务提示**:  
社会结构决定人的行为方式。

---

#### 4. 社会行动 (Social Action/社会行动)

**白话解释**:  
考虑到别人会怎么反应，所以这么做。

**生活例子**:  
- 上课不说话 → 怕影响别人
- 穿正装面试 → 给考官好印象
- 帮老人让座 → 希望被认可

**学术定义**:  
行动者在行动时考虑到他人的行为，并以他人的行为为取向的行动。

**韦伯的社会行动类型**:
1. **目的理性** - 为达到目的
   - 赚钱所以工作

2. **价值理性** - 基于价值观
   - 信仰所以做善事

3. **情感行动** - 基于情感
   - 生气所以打人

4. **传统行动** - 基于习惯
   - 过年所以要回家

**行动vs行为**:
| 行动 | 行为 |
|------|------|
| 有意义 | 无意义 |
| 有意图 | 无意图 |
| 社会性 | 生理性 |

**实务提示**:  
社会学研究的不是行为，而是有意义的行动。

---

#### 5. 社会互动 (Social Interaction/社会互动)

**白话解释**:  
人与人之间有来有往的交往。

**生活例子**:  
- 聊天 → 语言互动
- 握手 → 肢体互动
- 点赞 → 网络互动
- 买卖 → 经济互动

**学术定义**:  
两个或两个以上的人之间相互影响、相互作用的行动过程。

**互动形式**:
1. **交换** - 互惠互利
2. **合作** - 共同完成目标
3. **竞争** - 争夺资源
4. **冲突** - 利益对立
5. **强制** - 一方压制另一方

**互动理论**:
| 理论 | 核心观点 |
|------|----------|
| 符号互动论 | 通过符号互动 |
| 拟剧论 | 人生如戏 |
| 交换论 | 互惠交换 |

**戈夫曼的拟剧论**:  
人生就像演戏，有前台后台，有角色扮演。

**实务提示**:  
互动是社会形成的基础。

---

### 二、社会化

#### 6. 社会化 (Socialization/社会化)

**白话解释**:  
从一个"自然人"变成"社会人"的过程。

**生活例子**:  
- 婴儿学说话 → 语言社会化
- 孩子上学 → 学校社会化
- 新人入职 → 职业社会化

**学术定义**:  
个体学习社会文化、掌握社会技能、形成社会人格，成为合格社会成员的过程。

**社会化内容**:
1. **学习文化** - 价值观、规范
2. **学习技能** - 生活、工作技能
3. **形成人格** - 性格、习惯
4. **扮演角色** - 社会角色

**社会化阶段**:
| 阶段 | 年龄 | 主要任务 |
|------|------|----------|
| 婴幼儿期 | 0-3岁 | 基本信任 |
| 儿童期 | 3-12岁 | 基本技能 |
| 青少年期 | 12-18岁 | 身份认同 |
| 成年期 | 18岁+ | 角色适应 |
| 老年期 | 退休后 | 角色转换 |

**社会化主体**:
1. **家庭** - 第一社会化机构
2. **学校** - 系统社会化
3. **同伴群体** - 平等互动
4. **大众传媒** - 信息传播
5. **工作单位** - 职业社会化

**再社会化**:  
重新学习新的价值观和行为方式。

**实务提示**:  
社会化是终身过程。

---

#### 7. 社会角色 (Social Role/社会角色)

**白话解释**:  
你在某个位置上应该怎么做。

**生活例子**:  
- 老师 → 教书育人
- 学生 → 认真学习
- 父母 → 养育子女
- 子女 → 孝敬父母

**学术定义**:  
与某种社会地位相联系的、被期待的行为模式。

**角色要素**:
1. **角色期待** - 别人希望你怎么做
2. **角色领悟** - 你自己怎么理解
3. **角色实践** - 你实际怎么做

**角色类型**:
| 类型 | 特点 | 例子 |
|------|------|------|
| 先赋角色 | 与生俱来 | 父母子女 |
| 自致角色 | 后天获得 | 职业角色 |
| 规定性角色 | 规定明确 | 法官、医生 |
| 开放性角色 | 规定模糊 | 朋友 |

**角色问题**:
1. **角色冲突** - 不同角色要求冲突
2. **角色紧张** - 角色要求太多
3. **角色不清** - 不知道怎么做
4. **角色失败** - 无法完成角色

**角色集**:  
一个人扮演的所有角色的总和。

**实务提示**:  
角色是连接个人与社会的桥梁。

---

#### 8. 社会群体 (Social Group/社会群体)

**白话解释**:  
一群人有共同特征，经常互动。

**生活例子**:  
- 家人 → 家庭群体
- 同学 → 同伴群体
- 老乡 → 地缘群体
- 同事 → 工作群体

**学术定义**:  
通过一定的社会关系结合起来进行共同活动的集体。

**群体特征**:
1. **一定数量** - 两人以上
2. **持续互动** - 长期交往
3. **共同规范** - 有规则
4. **群体认同** - 认同感
5. **结构分化** - 有分工

**群体类型**:
| 分类标准 | 类型 | 例子 |
|----------|------|------|
| 规模 | 大群体/小群体 | 国家/家庭 |
| 关系 | 首属群体/次属群体 | 家人/同事 |
| 组织 | 正式群体/非正式群体 | 公司/朋友圈 |
| 参照 | 内群体/外群体 | 我们/他们 |

**库利的首属群体**:  
面对面互动、情感深厚的群体，如家庭。

**参照群体**:  
作为评价标准和行为参照的群体。

**实务提示**:  
群体是个人社会化的重要场所。

---

### 三、社会分层

#### 9. 社会分层 (Social Stratification/社会分层)

**白话解释**:  
社会像蛋糕一样分层，有人在上层，有人在下层。

**生活例子**:  
- 富人 vs 穷人
- 有权人 vs 普通人
- 高学历 vs 低学历
- 城里人 vs 乡下人

**学术定义**:  
社会成员按照某种标准被划分为不同等级、阶层的现象。

**分层标准**:
1. **经济标准** - 收入、财富
2. **政治标准** - 权力
3. **社会标准** - 声望、地位
4. **文化标准** - 学历、品位

**分层维度**:
| 维度 | 内容 |
|------|------|
| 阶级 | 经济地位 |
| 身份 | 社会声望 |
| 权力 | 政治影响 |
| 职业 | 工作类型 |

**社会流动**:  
个人或群体在社会分层体系中位置的变化。

**流动类型**:
1. **向上流动** - 地位上升
2. **向下流动** - 地位下降
3. **代际流动** - 父子比较
4. **代内流动** - 自己一生变化

**实务提示**:  
社会分层是社会不平等的重要体现。

---

#### 10. 社会阶级 (Social Class/社会阶级)

**白话解释**:  
根据经济地位划分的社会群体。

**生活例子**:  
- 工人阶级 → 靠工资生活
- 中产阶级 → 有一定财产
- 上层阶级 → 掌握大量财富

**学术定义**:  
在生产关系中处于相同地位、具有共同利益的社会集团。

**马克思的阶级理论**:
1. **资产阶级** - 拥有生产资料
2. **无产阶级** - 出卖劳动力
3. **阶级斗争** - 阶级之间冲突

**韦伯的阶级理论**:
- **阶级** - 市场地位 (经济)
- **身份** - 社会声望 (社会)
- **政党** - 权力地位 (政治)

**中国当前阶级/阶层**:
| 阶层 | 特点 |
|------|------|
| 上层 | 大企业家、高级官员 |
| 中产阶级 | 专业人士、管理者 |
| 工人阶层 | 产业工人 |
| 农民阶层 | 务农者 |
| 新社会阶层 | 自由职业者、创业者 |

**阶级意识**:  
阶级成员对自己阶级地位的认同。

**实务提示**:  
阶级是社会分析的重要工具。

---

#### 11. 社会不平等 (Social Inequality/社会不平等)

**白话解释**:  
社会资源分配不公平，有的人多，有的人少。

**生活例子**:  
- 收入不平等 → 富人越来越富
- 教育不平等 → 重点学校学生条件好
- 健康不平等 → 富人寿命更长

**学术定义**:  
社会成员在占有社会资源、享有社会机会方面存在的差异。

**不平等类型**:
1. **经济不平等** - 收入、财富
2. **政治不平等** - 权力、影响
3. **社会不平等** - 声望、地位
4. **机会不平等** - 教育、发展

**基尼系数**:  
测量收入不平等的指标。
- 0 = 完全平等
- 1 = 完全不平等
- 0.4以上 = 警戒线

**不平等原因**:
| 理论 | 原因 |
|------|------|
| 功能主义 | 必需的不平等 |
| 冲突理论 | 阶级剥削 |
| 互动论 | 社会建构 |

**不平等后果**:
- 社会矛盾加剧
- 社会流动受阻
- 社会效率降低

**实务提示**:  
适度的差异激励发展，过度的差异危害稳定。

---

#### 12. 社会流动 (Social Mobility/社会流动)

**白话解释**:  
人在社会阶梯上的位置变动。

**生活例子**:  
- 农村孩子考上大学 → 向上流动
- 企业家破产 → 向下流动
- 子女比父母更成功 → 代际流动

**学术定义**:  
个人或群体在社会分层结构中位置的变化。

**流动类型**:
| 类型 | 说明 | 例子 |
|------|------|------|
| 垂直流动 | 上下移动 | 升职、降职 |
| 水平流动 | 同层移动 | 换工作 |
| 代际流动 | 两代人比较 | 子女vs父母 |
| 代内流动 | 一生变化 | 职业变化 |

**影响流动的因素**:
1. **个人因素** - 能力、努力
2. **家庭背景** - 父母地位
3. **教育机会** - 受教育程度
4. **社会制度** - 制度开放性
5. **经济发展** - 经济机会

**开放社会 vs 封闭社会**:
| 开放社会 | 封闭社会 |
|----------|----------|
| 流动机会多 | 流动机会少 |
| 能力重要 | 出身重要 |
| 印度种姓制 | 现代社会 |

**中国改革开放后**:
- 社会流动加速
- 机会增加
- 但阶层固化风险

**实务提示**:  
社会流动是衡量社会开放性的重要指标。

### 一、家庭制度

#### 13. 家庭 (Family/家庭)

**白话解释**:  
有血缘或婚姻关系的人生活在一起。

**生活例子**:  
- 父母和孩子 → 核心家庭
- 三代同堂 → 扩展家庭
- 夫妻二人 → 丁克家庭

**学术定义**:  
以婚姻、血缘或收养关系为基础的社会生活的基本单位。

**家庭功能**:
1. **生育功能** - 生育子女
2. **经济功能** - 生产、消费
3. **社会化功能** - 教育子女
4. **情感功能** - 情感支持
5. **赡养功能** - 赡养老人

**家庭类型**:
| 类型 | 特点 |
|------|------|
| 核心家庭 | 夫妻+未婚子女 |
| 主干家庭 | 两代+已婚子女 |
| 联合家庭 | 多对夫妻 |
| 单亲家庭 | 一方+子女 |
| 丁克家庭 | 夫妻不要孩子 |

**家庭变迁**:
- 核心家庭增多
- 家庭规模缩小
- 离婚率上升
- 晚婚晚育增多

**中国家庭特点**:
- 孝道文化
- 家族观念
- 计划生育影响
- 独生子女家庭

**实务提示**:  
家庭是社会的基本单位。

---

#### 14. 婚姻 (Marriage/婚姻)

**白话解释**:  
一男一女正式成为夫妻的关系。

**生活例子**:  
- 领结婚证 → 法律婚姻
- 办婚礼 → 社会认可
- 共同生活 → 事实婚姻

**学术定义**:  
社会认可的男女两性结合的合法形式，是建立家庭的基础。

**婚姻功能**:
1. **合法的性关系** - 社会认可
2. **生育子女** - 合法生育
3. **经济合作** - 共同生活
4. **情感支持** - 相互依靠

**婚姻类型**:
| 类型 | 特点 | 例子 |
|------|------|------|
| 一夫一妻制 | 一夫一妻 | 现代社会主流 |
| 一夫多妻制 | 一个丈夫多个妻子 | 古代社会 |
| 一妻多夫制 | 一个妻子多个丈夫 | 少数地区 |
| 群婚 | 多对夫妻 | 原始社会 |

**择偶标准**:
1. **同质婚** - 找相似的
2. **异质婚** - 找互补的
3. **梯度婚** - 男高女低

**婚姻问题**:
- 离婚率上升
- 晚婚现象普遍
- 不婚族增多
- 婚姻不稳定

**实务提示**:  
婚姻制度是社会稳定的基础。

### 二、教育制度

#### 15. 教育 (Education/教育)

**白话解释**:  
学习知识、技能和价值观的过程。

**生活例子**:  
- 上学 → 正式教育
- 培训 → 非正式教育
- 自学 → 自我教育

**学术定义**:  
社会有目的、有计划地培养人的社会活动。

**教育功能**:
1. **社会化功能** - 传授文化
2. **选择功能** - 选拔人才
3. **职业功能** - 职业准备
4. **整合功能** - 社会整合

**教育类型**:
| 类型 | 说明 |
|------|------|
| 正式教育 | 学校教育 |
| 非正式教育 | 日常学习 |
| 非正规教育 | 培训、进修 |

**教育不平等**:
1. **城乡差距** - 农村教育资源少
2. **阶层差距** - 富人孩子条件好
3. **区域差距** - 发达地区教育好
4. **性别差距** - 部分地区重男轻女

**教育功能主义 vs 冲突论**:
| 功能主义 | 冲突论 |
|----------|--------|
| 传播知识 | 再产不平等 |
| 选拔人才 | 维护阶级 |
| 促进流动 | 限制流动 |

**中国教育现状**:
- 义务教育普及
- 高等教育大众化
- 教育竞争激烈
- 减负政策推行

**实务提示**:  
教育是重要的社会流动渠道。

---

#### 16. 学校 (School/学校)

**白话解释**:  
专门教书育人的地方。

**生活例子**:  
- 小学、中学 → 基础教育
- 大学 → 高等教育
- 职业学校 → 职业教育

**学术定义**:  
专门从事教育活动的正式组织机构。

**学校功能**:
1. **教学功能** - 传授知识
2. **社会化功能** - 培养合格公民
3. **选择功能** - 分流、选拔
4. **保护功能** - 照顾未成年人

**学校作为组织**:
```
学校
├─ 正式结构
│   ├─ 行政系统
│   ├─ 教学系统
│   └─ 后勤系统
└─ 非正式结构
    ├─ 同伴群体
    ├─ 师生关系
    └─ 同事关系
```

**隐性课程**:  
学校未明文规定但实际传递的价值观念和行为规范。

**学校中的不平等**:
- 贴标签效应
- 自证预言
- 教师期待

**实务提示**:  
学校是社会化的重要场所。

---

### 三、其他社会制度

#### 17. 宗教 (Religion/宗教)

**白话解释**:  
信仰神灵、追求超越的一套信仰体系。

**生活例子**:  
- 佛教 → 供奉佛祖
- 基督教 → 信仰上帝
- 伊斯兰教 → 信仰真主
- 道教 → 修道成仙

**学术定义**:  
对超自然力量的信仰以及由此产生的仪式、组织和道德规范体系。

**宗教要素**:
1. **信仰** - 超自然观念
2. **仪式** - 宗教活动
3. **组织** - 宗教团体
4. **规范** - 宗教道德

**宗教功能**:
| 正功能 | 负功能 |
|--------|--------|
| 心理安慰 | 可能阻碍进步 |
| 社会整合 | 可能引发冲突 |
| 道德约束 | 可能压制个性 |
| 意义提供 | 可能迷信 |

**宗教类型**:
1. **多神教** - 多个神
2. **一神教** - 一个神
3. **泛神教** - 万物有灵
4. **无神论** - 不信神

**世俗化**:  
宗教对社会生活影响减弱的过程。

**中国宗教政策**:  
宗教信仰自由、政教分离。

**实务提示**:  
宗教是人类文化的重要组成部分。

---

#### 18. 经济制度 (Economic Institution/经济制度)

**白话解释**:  
社会怎么组织生产、分配和消费的规则。

**生活例子**:  
- 市场经济 → 供求决定价格
- 计划经济 → 政府分配资源
- 混合经济 → 市场+政府

**学术定义**:  
社会对生产、分配、交换、消费等经济活动进行组织和管理的制度体系。

**经济制度类型**:
| 类型 | 特点 | 例子 |
|------|------|------|
| 传统经济 | 习惯决定 | 原始社会 |
| 市场经济 | 市场调节 | 美国等 |
| 计划经济 | 政府计划 | 改革前中国 |
| 混合经济 | 市场+政府 | 现代中国 |

**经济全球化影响**:
- 市场扩大
- 竞争加剧
- 贫富分化
- 产业结构调整

**经济制度功能**:
1. **资源配置** - 怎么分配资源
2. **生产组织** - 怎么组织生产
3. **产品分配** - 怎么分配产品
4. **消费调节** - 怎么调节消费

**中国社会主义市场经济**:  
以公有制为主体、多种所有制共同发展。

**实务提示**:  
经济制度决定社会的基本运行方式。

---

#### 19. 政治制度 (Political Institution/政治制度)

**白话解释**:  
社会怎么管理公共事务、分配权力的规则。

**生活例子**:  
- 选举制度 → 怎么选领导人
- 议会制度 → 怎么立法
- 行政制度 → 怎么执行

**学术定义**:  
社会对政治权力进行组织、分配和运作的制度体系。

**政治制度功能**:
1. **权力分配** - 谁有权
2. **决策制定** - 怎么决策
3. **社会控制** - 维护秩序
4. **利益整合** - 整合利益

**政治制度类型**:
| 类型 | 特点 |
|------|------|
| 民主制 | 人民参与 |
| 威权制 | 权力集中 |
| 极权制 | 全面控制 |

**韦伯的合法性类型**:
1. **传统型** - 传统习惯
2. **魅力型** - 个人魅力
3. **法理型** - 法律制度

**政治社会化**:  
公民学习政治知识和政治态度的过程。

**实务提示**:  
政治制度决定社会的治理方式。

#### 20. 医疗制度 (Medical Institution/医疗制度)

**白话解释**:  
社会怎么提供医疗服务的制度。

**生活例子**:  
- 医保 → 医疗保险
- 公立医院 → 政府办医院
- 私立医院 → 私人办医院

**学术定义**:  
社会对医疗卫生服务进行组织、管理和提供的制度体系。

**医疗制度功能**:
1. **疾病治疗** - 治病救人
2. **疾病预防** - 预防保健
3. **健康维护** - 促进健康
4. **医疗公平** - 公平分配

**医疗制度类型**:
| 类型 | 特点 | 例子 |
|------|------|------|
| 国家医疗 | 政府提供 | 英国NHS |
| 社会保险 | 强制保险 | 德国 |
| 商业保险 | 自愿保险 | 美国 |
| 混合制度 | 多种结合 | 中国 |

**健康不平等**:
- 富人更健康
- 教育程度高的人更健康
- 城市居民更健康
- 社会经济地位影响健康

**中国医疗改革**:
- 全民医保
- 基本药物制度
- 公立医院改革
- 分级诊疗

**实务提示**:  
医疗制度关系到每个人的健康。

### 一、社会问题

#### 21. 社会问题 (Social Problem/社会问题)

**白话解释**:  
影响很多人、需要社会解决的问题。

**生活例子**:  
- 贫困 → 很多人没钱
- 犯罪 → 治安问题
- 污染 → 环境问题
- 失业 → 就业问题

**学术定义**:  
影响相当数量社会成员的正常生活、危害社会运行、需要社会力量解决的社会现象。

**社会问题特征**:
1. **普遍性** - 影响很多人
2. **危害性** - 有害于社会
3. **社会性** - 需要社会解决
4. **持久性** - 不是短期现象

**社会问题类型**:
| 类型 | 例子 |
|------|------|
| 经济问题 | 贫困、失业 |
| 政治问题 | 腐败、犯罪 |
| 社会问题 | 家庭破裂、老龄化 |
| 文化问题 | 价值混乱、道德滑坡 |
| 环境问题 | 污染、资源枯竭 |

**社会问题的社会建构**:  
社会问题是社会认定的，不同社会有不同的问题定义。

**中国当前主要社会问题**:
1. **贫富差距** - 收入不平等
2. **就业压力** - 大学生就业难
3. **住房问题** - 房价高
4. **养老问题** - 人口老龄化
5. **教育公平** - 教育资源不均

**实务提示**:  
社会问题需要全社会共同解决。

---

#### 22. 贫困 (Poverty/贫困)

**白话解释**:  
没有足够的钱维持基本生活。

**生活例子**:  
- 绝对贫困 → 没吃没穿
- 相对贫困 → 比别人穷
- 农村贫困 → 贫困地区
- 城市贫困 → 下岗工人

**学术定义**:  
个人或家庭缺乏获得维持基本生活所需的物质和社会资源的状态。

**贫困类型**:
1. **绝对贫困** - 低于基本生活线
2. **相对贫困** - 相对于社会平均水平
3. **长期贫困** - 长期处于贫困
4. **暂时贫困** - 短期陷入贫困

**贫困线**:  
划分贫困与非贫困的标准。

**贫困原因**:
| 理论 | 解释 |
|------|------|
| 个人因素论 | 个人能力不足 |
| 文化贫困论 | 贫困文化传承 |
| 结构因素论 | 社会结构不公 |
| 剥夺论 | 资源被剥夺 |

**中国脱贫攻坚**:
- 2020年完成脱贫攻坚
- 近1亿人脱贫
- 消除绝对贫困
- 转向乡村振兴

**实务提示**:  
贫困是社会不平等的重要表现。

---

#### 23. 犯罪 (Crime/犯罪)

**白话解释**:  
违反法律、要受惩罚的行为。

**生活例子**:  
- 偷盗 → 盗窃罪
- 打人 → 故意伤害罪
- 贪污 → 贪污罪
- 欺诈 → 诈骗罪

**学术定义**:  
违反刑法、应受刑罚处罚的危害社会的行为。

**犯罪类型**:
| 类型 | 例子 |
|------|------|
| 暴力犯罪 | 杀人、抢劫 |
| 财产犯罪 | 盗窃、诈骗 |
| 经济犯罪 | 贪污、走私 |
| 性犯罪 | 强奸、猥亵 |
| 毒品犯罪 | 贩毒、吸毒 |

**犯罪理论**:
1. **社会控制理论** - 社会控制弱导致犯罪
2. **紧张理论** - 目标手段差距导致犯罪
3. **标签理论** - 被贴标签导致犯罪
4. **学习理论** - 学会了犯罪

**犯罪预防**:
1. **情境预防** - 减少犯罪机会
2. **社会预防** - 消除犯罪原因
3. **发展预防** - 早期干预

**实务提示**:  
犯罪是社会问题的极端表现。

---

### 二、社会变迁

#### 24. 社会变迁 (Social Change/社会变迁)

**白话解释**:  
社会随时间推移发生变化。

**生活例子**:  
- 农业社会 → 工业社会
- 乡村社会 → 城市社会
- 传统社会 → 现代社会

**学术定义**:  
社会结构、社会制度、社会文化等发生的变化。

**变迁类型**:
1. **渐进变迁** - 缓慢变化
2. **激进变迁** - 快速变化
3. **进步变迁** - 向好的方向
4. **倒退变迁** - 向坏的方向

**变迁原因**:
| 原因 | 例子 |
|------|------|
| 技术进步 | 工业革命 |
| 经济发展 | 市场经济 |
| 文化传播 | 全球化 |
| 社会运动 | 改革开放 |
| 人口变化 | 人口老龄化 |

**现代化**:  
从传统社会向现代社会转变的过程。

**中国改革开放**:  
- 1978年开始
- 市场经济改革
- 经济快速发展
- 社会深刻变化

**实务提示**:  
社会变迁是历史的常态。

---

#### 25. 城市化 (Urbanization/城市化)

**白话解释**:  
农村人往城里跑，城市越来越多、越来越大。

**生活例子**:  
- 农民进城打工
- 小城市变成大城市
- 乡村变成城镇

**学术定义**:  
人口向城市集中、城市数量增加、城市规模扩大的过程。

**城市化表现**:
1. **人口城市化** - 人口向城市集中
2. **地域城市化** - 城市面积扩大
3. **生活方式城市化** - 城市生活方式普及

**城市化水平**:  
城市人口占总人口的比例。

**中国城市化进程**:
| 年份 | 城市化率 |
|------|----------|
| 1978年 | 17.9% |
| 2000年 | 36.2% |
| 2020年 | 63.9% |
| 2035年 | 75%+ (预计) |

**城市化问题**:
1. **城市病** - 交通拥堵、环境污染
2. **农民工问题** - 户籍、就业、子女教育
3. **城乡差距** - 城乡发展不平衡
4. **农村空心化** - 农村人口减少

**新型城镇化**:  
以人为核心的城镇化，注重质量。

**实务提示**:  
城市化是现代化的必然趋势。
