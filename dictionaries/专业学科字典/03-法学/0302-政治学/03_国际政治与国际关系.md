### 一、国际政治基本概念

#### 1. 国际政治 (International Politics/国际政治)

**白话解释**:  
国家之间打交道的过程，包括合作、竞争、冲突。

**生活例子**:  
- 美国制裁俄罗斯
- 中国"一带一路"倡议
- 联合国通过决议

**学术定义**:  
国际社会各行为主体之间政治关系的总和，主要是国家间围绕权力、利益和安全进行的互动。

**核心要素**:
1. **行为主体** - 国家、国际组织等
2. **互动关系** - 合作、竞争、冲突
3. **核心议题** - 权力、利益、安全
4. **制度框架** - 国际规则、国际法

**研究层次**:
| 层次 | 分析对象 | 例子 |
|------|----------|------|
| 个人层次 | 领导人决策 | 普京的决策风格 |
| 国家层次 | 国家属性 | 中国的发展模式 |
| 体系层次 | 国际结构 | 美中竞争格局 |

**与国内政治区别**:
| 国内政治 | 国际政治 |
|----------|----------|
| 有中央政府 | 无世界政府 |
| 有法律强制 | 依赖自愿遵守 |
| 等级秩序 | 无政府状态 |
| 权力集中 | 权力分散 |

**实务提示**:  
国际政治是无政府状态下的政治。

---

#### 2. 国际关系 (International Relations/国际关系)

**白话解释**:  
国家之间各种关系的总称，不只是政治关系。

**生活例子**:  
- 中美贸易关系
- 中国与东盟外交关系
- 中欧文化交流

**学术定义**:  
国际社会各行为主体之间各种关系的总和，包括政治、经济、文化、军事等关系。

**关系类型**:
1. **政治关系** - 外交、结盟
2. **经济关系** - 贸易、投资
3. **军事关系** - 军事合作、冲突
4. **文化关系** - 交流、传播

**研究范式**:
| 范式 | 核心观点 | 代表人物 |
|------|----------|----------|
| 现实主义 | 权力、利益 | 摩根索 |
| 自由主义 | 合作、制度 | 基欧汉 |
| 建构主义 | 观念、认同 | 温特 |

**实务提示**:  
国际关系比国际政治范围更广。

---

#### 3. 国际体系 (International System/国际体系)

**白话解释**:  
国际社会的"游戏规则"和权力结构。

**生活例子**:  
- 冷战时期的两极体系
- 冷战后的单极体系
- 当前的多极化趋势

**学术定义**:  
国际社会各行为主体按一定规则互动形成的整体结构和秩序。

**体系要素**:
1. **行为主体** - 国家、国际组织
2. **权力结构** - 单极、两极、多极
3. **互动规则** - 国际法、国际规范
4. **互动模式** - 合作、竞争、冲突

**体系类型**:
| 类型 | 特点 | 例子 |
|------|------|------|
| 单极体系 | 一个超级大国 | 冷战后美国主导 |
| 两极体系 | 两个超级大国 | 冷战美苏对峙 |
| 多极体系 | 多个大国并存 | 一战前欧洲 |

**体系变迁**:
```
威斯特伐利亚体系 (1648年)
    ↓
维也纳体系 (1815年)
    ↓
凡尔赛-华盛顿体系 (1919年)
    ↓
雅尔塔体系 (1945年)
    ↓
后冷战体系 (1991年至今)
```

**实务提示**:  
理解国际体系是分析国际政治的基础。

---

#### 4. 国际行为主体 (International Actor/国际行为主体)

**白话解释**:  
能在国际上独立行动的"玩家"。

**生活例子**:  
- 国家 → 中国、美国、俄罗斯
- 国际组织 → 联合国、欧盟
- 跨国公司 → 苹果、谷歌
- 非政府组织 → 绿色和平组织

**学术定义**:  
能够独立参与国际事务、对国际政治产生影响的行为体。

**主体类型**:
1. **国家行为主体**
   - 主权国家
   - 最主要的行为主体

2. **非国家行为主体**
   - 国际组织 (联合国、WTO)
   - 跨国公司 (世界500强)
   - 非政府组织 (NGO)
   - 恐怖组织

**国家行为主体特征**:
| 特征 | 说明 |
|------|------|
| 主权 | 对内最高、对外独立 |
| 领土 | 一定范围疆域 |
| 人口 | 一定数量居民 |
| 政府 | 管理机构 |

**非国家主体作用**:
- 国际组织：协调、规制
- 跨国公司：经济影响
- NGO：议题倡导

**实务提示**:  
国家仍然是最重要的国际行为主体。

### 二、国际政治理论

#### 5. 现实主义 (Realism/现实主义)

**白话解释**:  
国际政治就是"弱肉强食"，实力决定一切。

**生活例子**:  
- 大国欺负小国
- 弱国无外交
- 军事力量最重要

**学术定义**:  
强调国际政治的无政府状态、权力的中心地位和国家利益至上的一套理论。

**核心观点**:
1. **人性本恶** - 人追求权力
2. **无政府状态** - 没有世界政府
3. **权力中心** - 权力是核心变量
4. **国家利益** - 国家追求自身利益
5. **安全困境** - 追求安全引发冲突

**代表人物**:
| 学者 | 贡献 |
|------|------|
| 修昔底德 | 《伯罗奔尼撒战争史》 |
| 马基雅维利 | 《君主论》 |
| 霍布斯 | 自然状态论 |
| 摩根索 | 《国家间政治》 |
| 米尔斯海默 | 进攻性现实主义 |

**现实主义分支**:
1. **古典现实主义** - 人性本恶
2. **结构现实主义 (新现实主义)** - 体系结构决定
3. **进攻性现实主义** - 追求霸权
4. **防御性现实主义** - 维持均势

**修昔底德陷阱**:  
新兴大国崛起必然挑战守成大国，引发冲突。

**实务提示**:  
现实主义是理解国际政治的起点。

---

#### 6. 自由主义 (Liberalism/自由主义)

**白话解释**:  
国家可以通过合作实现共赢，不一定要打仗。

**生活例子**:  
- 欧盟国家合作
- WTO促进自由贸易
- 联合国维护和平

**学术定义**:  
强调国际合作、国际制度、民主国家间和平的理论。

**核心观点**:
1. **可以合作** - 合作是可能的
2. **制度重要** - 国际制度促进合作
3. **民主和平** - 民主国家不打仗
4. **相互依赖** - 经济依赖减少冲突
5. **多元主体** - 不只有国家

**代表人物**:
| 学者 | 贡献 |
|------|------|
| 康德 | 永久和平论 |
| 威尔逊 | 十四点原则 |
| 基欧汉 | 霸权之后 |
| 奈 | 软实力 |

**民主和平论**:  
民主国家之间很少发生战争。

**商业和平论**:  
贸易相互依赖减少冲突。

**制度和平论**:  
国际制度促进合作、减少冲突。

**与现实主义区别**:
| 现实主义 | 自由主义 |
|----------|----------|
| 冲突是常态 | 合作是可能的 |
| 权力至上 | 制度重要 |
| 国家中心 | 多元主体 |

**实务提示**:  
自由主义强调合作的可能性。

---

#### 7. 建构主义 (Constructivism/建构主义)

**白话解释**:  
国际政治是由人们的观念和认同"建构"出来的。

**生活例子**:  
- 欧盟国家认同"欧洲人"身份
- 冷战结束源于观念变化
- 中国"和平崛起"的叙事

**学术定义**:  
强调观念、认同、规范、文化等非物质因素对国际政治影响的理论。

**核心观点**:
1. **观念重要** - 观念塑造利益
2. **认同建构** - 认同决定行为
3. **规范影响** - 规范约束行为
4. **社会建构** - 现实是社会建构的
5. **可以改变** - 观念可以改变

**代表人物**:
| 学者 | 贡献 |
|------|------|
| 奥努夫 | 首次提出 |
| 温特 | 《国际政治的社会理论》 |
| 芬尼莫尔 | 国际规范 |

**关键概念**:
1. **认同 (Identity)** - 我是谁？
2. **规范 (Norm)** - 应该怎么做？
3. **观念 (Idea)** - 相信什么？
4. **文化 (Culture)** - 共同价值观

**温特的三个无政府文化**:
1. **霍布斯文化** - 敌对关系
2. **洛克文化** - 竞争关系
3. **康德文化** - 友谊关系

**实务提示**:  
建构主义强调"观念的力量"。

---

#### 8. 国际秩序 (International Order/国际秩序)

**白话解释**:  
国际社会比较稳定、有规则的状态。

**生活例子**:  
- 二战后联合国体系
- 冷战后美国主导秩序
- 当前的秩序变化

**学术定义**:  
国际体系中相对稳定的权力分配、制度安排和规范体系。

**秩序要素**:
1. **权力分配** - 谁强谁弱
2. **制度安排** - 规则制度
3. **规范体系** - 价值观念
4. **稳定预期** - 行为可预期

**秩序类型**:
| 类型 | 特点 | 例子 |
|------|------|------|
| 霸权秩序 | 一国主导 | 罗马治下的和平 |
| 均势秩序 | 大国平衡 | 维也纳体系 |
| 制度秩序 | 规则约束 | WTO秩序 |
| 集体安全 | 共同维护 | 联合国 |

**秩序转型**:
- 权力转移引发秩序变化
- 新兴大国挑战现有秩序
- 秩序重塑是历史常态

**中国主张**:  
构建人类命运共同体，推动国际秩序更加公正合理。

**实务提示**:  
国际秩序不是一成不变的。

### 三、国际安全

#### 9. 国家安全 (National Security/国家安全)

**白话解释**:  
国家没有危险，不受威胁。

**生活例子**:  
- 国防安全 → 没有外敌入侵
- 政治安全 → 政权稳定
- 经济安全 → 经济稳定发展
- 网络安全 → 网络不受攻击

**学术定义**:  
国家主权、领土完整、政治制度、经济发展、社会稳定等不受威胁和侵害的状态。

**安全类型**:
1. **传统安全**
   - 军事安全
   - 政治安全
   - 国土安全

2. **非传统安全**
   - 经济安全
   - 能源安全
   - 网络安全
   - 生态安全
   - 公共卫生安全

**中国总体国家安全观**:
```
总体国家安全观
├─ 政治安全 (根本)
├─ 经济安全 (基础)
├─ 军事、文化、社会安全 (保障)
└─ 国际安全 (依托)
```

**11个安全领域**:
1. 政治安全
2. 国土安全
3. 军事安全
4. 经济安全
5. 文化安全
6. 社会安全
7. 科技安全
8. 网络安全
9. 生态安全
10. 资源安全
11. 核安全

**实务提示**:  
国家安全是国家的核心利益。

#### 10. 安全困境 (Security Dilemma/安全困境)

**白话解释**:  
你想更安全，结果让对方更不安全，引发军备竞赛。

**生活例子**:  
- A国加强军备 → B国感到威胁 → B国也加强军备 → A国更加不安全
- 美苏军备竞赛

**学术定义**:  
一国为增强自身安全而采取的措施，反而降低了他国的安全，导致他国采取对应措施，最终各国安全都降低的困境。

**困境逻辑**:
```
A国增强军备
    ↓
B国感到威胁
    ↓
B国也增强军备
    ↓
A国更不安全
    ↓
循环往复
```

**困境原因**:
1. **无政府状态** - 没有世界政府
2. **不确定性** - 不确定对方意图
3. **最坏打算** - 宁可信其有
4. **缺乏信任** - 国家间不信任

**走出困境**:
1. **建立信任措施** - 增加透明度
2. **军控条约** - 限制军备
3. **安全合作** - 共同安全
4. **国际制度** - 规范约束

**实务提示**:  
安全困境是国际冲突的重要根源。

---

#### 11. 均势 (Balance of Power/均势)

**白话解释**:  
大国之间力量差不多，谁也不能轻易吃掉谁。

**生活例子**:  
- 一战前欧洲均势
- 冷战美苏均势
- 当前多极化趋势

**学术定义**:  
国际体系中主要国家力量相对均衡，没有一国能够主导他国的状态。

**均势类型**:
1. **简单均势** - 两大国对峙
2. **复杂均势** - 多大国并存
3. **整体均势** - 全球范围
4. **局部均势** - 地区范围

**均势策略**:
| 策略 | 做法 | 例子 |
|------|------|------|
| 结盟 | 与他国结盟 | 北约 |
| 扶持弱者 | 支持弱的一方 | 美国扶持欧洲 |
| 分而治之 | 分化对手 | 英国大陆政策 |
| 补偿原则 | 战后补偿 | 维也纳体系 |

**均势作用**:
- 维护和平
- 防止霸权
- 保持稳定

**均势困境**:
- 不稳定
- 可能失效
- 引发军备竞赛

**实务提示**:  
均势是维护国际稳定的重要机制。

---

#### 12. 霸权 (Hegemony/霸权)

**白话解释**:  
一个国家在国际上说了算。

**生活例子**:  
- 罗马帝国
- 大英帝国
- 二战后美国

**学术定义**:  
一个国家在国际体系中拥有压倒性优势，能够主导国际事务的状态。

**霸权特征**:
1. **军事优势** - 最强大的军事力量
2. **经济优势** - 最强大的经济体
3. **制度主导** - 制定国际规则
4. **文化影响** - 文化软实力

**霸权稳定论**:  
霸权国维护国际秩序，提供公共产品。

**霸权衰退**:
- 经济相对衰落
- 军事负担过重
- 新兴大国挑战
- 内部问题

**美国霸权**:
- 军事霸权 - 全球军事存在
- 金融霸权 - 美元霸权
- 科技霸权 - 科技领先
- 制度霸权 - 制定规则

**中国立场**:  
反对霸权主义，推动多极化。

**实务提示**:  
霸权不是永久的，会经历兴衰。

### 一、国际组织

#### 13. 联合国 (United Nations/联合国)

**白话解释**:  
最大的国际组织，协调各国关系、维护和平。

**生活例子**:  
- 安理会通过决议
- 维和部队执行任务
- 联合国大会讨论问题

**学术定义**:  
第二次世界大战后成立的，最具普遍性、代表性和权威性的政府间国际组织。

**联合国组成**:
```
联合国
├─ 大会 (General Assembly)
│   └─ 193个会员国
├─ 安全理事会 (Security Council)
│   ├─ 5个常任理事国
│   └─ 10个非常任理事国
├─ 经济及社会理事会 (ECOSOC)
├─ 托管理事会 (已停止运作)
├─ 国际法院 (ICJ)
└─ 秘书处
```

**主要机构职能**:
| 机构 | 职能 |
|------|------|
| 大会 | 讨论、建议 |
| 安理会 | 维护和平、强制执行 |
| 经社理事会 | 经济社会事务 |
| 国际法院 | 司法裁判 |
| 秘书处 | 日常事务 |

**安理会常任理事国 (五常)**:
- 中国
- 美国
- 俄罗斯
- 英国
- 法国

**安理会权力**:
- 一票否决权
- 强制执行决议
- 制裁措施
- 授权武力

**联合国作用**:
1. 维护国际和平与安全
2. 发展国际友好关系
3. 促进国际合作
4. 协调各国行动

**实务提示**:  
联合国是最重要的国际组织。

---

#### 14. 世界贸易组织 (World Trade Organization/WTO)

**白话解释**:  
管理全球贸易规则的组织。

**生活例子**:  
- 降低关税
- 解决贸易争端
- 制定贸易规则

**学术定义**:  
处理全球贸易规则的唯一全球性国际组织。

**WTO职能**:
1. **制定规则** - 贸易规则
2. **市场开放** - 降低贸易壁垒
3. **争端解决** - 解决贸易纠纷
4. **贸易审查** - 监督贸易政策

**基本原则**:
| 原则 | 内容 |
|------|------|
| 最惠国待遇 | 不歧视任何成员 |
| 国民待遇 | 外国商品与本国商品同等对待 |
| 关税减让 | 降低关税壁垒 |
| 透明度 | 政策公开透明 |

**争端解决机制**:
```
磋商 (60天)
    ↓
专家组审理
    ↓
上诉机构审查
    ↓
执行裁决
```

**中国与WTO**:
- 2001年加入WTO
- 最大货物贸易国
- 深度参与多边贸易

**实务提示**:  
WTO是全球贸易治理的核心。

---

#### 15. 国际货币基金组织 (International Monetary Fund/IMF)

**白话解释**:  
给有经济困难的国家借钱、救急的组织。

**生活例子**:  
- 希腊债务危机时IMF救助
- 阿根廷多次接受IMF贷款
- 提供经济政策建议

**学术定义**:  
促进国际货币合作、维护金融稳定、促进国际贸易的国际金融机构。

**IMF职能**:
1. **监督** - 监督各国经济政策
2. **贷款** - 向困难国家提供贷款
3. **技术援助** - 提供技术支持

**特别提款权 (SDR)**:
- IMF创设的国际储备资产
- 一篮子货币 (美元、欧元、人民币、日元、英镑)
- 人民币2016年加入SDR

**中国与IMF**:
- 第三大份额国
- 人民币纳入SDR
- 参与全球金融治理

**实务提示**:  
IMF是全球金融安全网的重要组成部分。

---

#### 16. 世界银行 (World Bank/世界银行)

**白话解释**:  
给发展中国家贷款搞建设的银行。

**生活例子**:  
- 给中国修路、建学校贷款
- 支持非洲减贫项目
- 提供技术援助

**学术定义**:  
向发展中国家提供中长期贷款和技术援助的国际金融机构。

**世界银行集团**:
```
世界银行集团
├─ 国际复兴开发银行 (IBRD)
├─ 国际开发协会 (IDA)
├─ 国际金融公司 (IFC)
├─ 多边投资担保机构 (MIGA)
└─ 国际投资争端解决中心 (ICSID)
```

**主要职能**:
1. **发展贷款** - 向发展中国家贷款
2. **技术援助** - 提供技术支持
3. **知识分享** - 传播发展经验

**中国与世界银行**:
- 曾经是最大借款国
- 现在向世界银行捐款
- 转型为合作伙伴

**实务提示**:  
世界银行是全球发展融资的重要来源。

---

### 二、全球治理

#### 17. 全球治理 (Global Governance/全球治理)

**白话解释**:  
全世界一起商量解决问题。

**生活例子**:  
- 气候变化 → 巴黎协定
- 疫情防控 → WHO协调
- 贸易规则 → WTO制定

**学术定义**:  
通过国际制度、国际规范、国际组织等机制，对全球性问题进行管理的活动和过程。

**治理主体**:
1. **国家** - 主权国家
2. **国际组织** - 联合国、WTO等
3. **跨国公司** - 跨国经济主体
4. **非政府组织** - NGO
5. **公民社会** - 公民参与

**治理领域**:
| 领域 | 治理机制 |
|------|----------|
| 贸易 | WTO |
| 金融 | IMF、世界银行 |
| 气候 | 联合国气候变化框架公约 |
| 安全 | 联合国安理会 |
| 卫生 | WHO |

**治理困境**:
1. **代表性问题** - 谁代表谁？
2. **效率问题** - 决策效率低
3. **执行问题** - 缺乏强制力
4. **公平问题** - 发展中国家话语权不足

**中国主张**:  
共商共建共享的全球治理观。

**实务提示**:  
全球治理需要各方共同参与。

---

#### 18. 区域一体化 (Regional Integration/区域一体化)

**白话解释**:  
一个地区的国家联合起来，像一家人一样。

**生活例子**:  
- 欧盟 - 欧洲一体化
- 东盟 - 东南亚一体化
- 非盟 - 非洲一体化

**学术定义**:  
地理上相邻或相近的国家，通过制度安排，逐步实现经济、政治、社会等领域的一体化。

**一体化层次**:
```
自由贸易区 → 关税同盟 → 共同市场 → 经济联盟 → 政治联盟
```

**欧盟一体化**:
| 阶段 | 成就 |
|------|------|
| 1957年 | 欧洲经济共同体 |
| 1993年 | 欧盟成立 |
| 2002年 | 欧元流通 |
| 2009年 | 《里斯本条约》 |

**东盟一体化**:
- 1967年成立
- 10个成员国
- 东盟共同体 (2015年)

**一体化动因**:
1. **经济利益** - 扩大市场
2. **政治合作** - 增强影响力
3. **安全保障** - 集体安全
4. **应对挑战** - 共同应对

**实务提示**:  
区域一体化是全球化的补充。

---

#### 19. 全球化 (Globalization/全球化)

**白话解释**:  
世界变成"地球村"，各国联系越来越紧密。

**生活例子**:  
- 中国制造卖到全世界
- 美国电影在中国上映
- 疫情全球传播
- 气候变化影响各国

**学术定义**:  
世界各国在经济、政治、文化等领域相互联系、相互依赖程度不断加深的过程。

**全球化表现**:
1. **经济全球化**
   - 贸易全球化
   - 金融全球化
   - 生产全球化

2. **政治全球化**
   - 国际合作
   - 全球治理

3. **文化全球化**
   - 文化传播
   - 价值共享

**全球化影响**:
| 积极 | 消极 |
|------|------|
| 促进发展 | 贫富分化 |
| 技术传播 | 文化冲击 |
| 交流合作 | 主权挑战 |

**逆全球化**:
- 贸易保护主义
- 民粹主义兴起
- 疫情加速逆全球化

**中国立场**:  
支持经济全球化，反对保护主义。

**实务提示**:  
全球化是不可逆转的历史趋势。

---

#### 20. 人类命运共同体 (Community with a Shared Future for Mankind/人类命运共同体)

**白话解释**:  
全人类命运相连，应该共同面对挑战。

**生活例子**:  
- 疫情不分国界，需要共同应对
- 气候变化影响全人类
- 贸易摩擦影响全球

**学术定义**:  
习近平主席提出的全球治理理念，强调各国相互依存、命运与共。

**核心内涵**:
1. **持久和平** - 各国和平共处
2. **普遍安全** - 共同应对威胁
3. **共同繁荣** - 实现共同发展
4. **开放包容** - 尊重多样性
5. **清洁美丽** - 保护生态环境

**建设路径**:
```
对话协商 → 建设持久和平的世界
共建共享 → 建设普遍安全的世界
合作共赢 → 建设共同繁荣的世界
交流互鉴 → 建设开放包容的世界
绿色低碳 → 建设清洁美丽的世界
```

**实践载体**:
- "一带一路"倡议
- 全球发展倡议
- 全球安全倡议
- 全球文明倡议

**国际意义**:
- 中国智慧
- 中国方案
- 回答"人类向何处去"

**实务提示**:  
人类命运共同体是中国外交的核心理念。
