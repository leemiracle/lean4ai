### 一、计算机组成

#### 1. 冯·诺依曼架构 (Von Neumann Architecture/冯诺依曼架构)

**白话解释**:  
现代计算机的基本结构，存储程序概念。

**五大部件**:
| 部件 | 功能 |
|------|------|
| 运算器 | 算术逻辑运算 |
| 控制器 | 指令执行控制 |
| 存储器 | 存储程序和数据 |
| 输入设备 | 数据输入 |
| 输出设备 | 结果输出 |

**核心思想**:
- 存储程序
- 二进制
- 顺序执行

**费曼记忆**:  
计算机 = 存储 + 运算 + 控制 + I/O

---

#### 2. CPU (Central Processing Unit/中央处理器)

**白话解释**:  
计算机的大脑，执行程序指令。

**CPU组成**:
1. **ALU**: 算术逻辑单元
2. **CU**: 控制单元
3. **寄存器**: 临时存储
4. **缓存**: 高速缓存

**性能指标**:
| 指标 | 含义 |
|------|------|
| 主频 | 时钟频率 |
| 核心数 | 并行处理能力 |
| 缓存 | 片上存储 |
| 指令集 | 支持的指令 |

**指令执行周期**:
取指 → 译码 → 执行 → 访存 → 写回

---

#### 3. 存储器 (Memory/存储器)

**白话解释**:  
存储程序和数据的设备。

**存储层次**:
| 层次 | 速度 | 容量 | 成本 |
|------|------|------|------|
| 寄存器 | 最快 | 最小 | 最高 |
| L1缓存 | 很快 | 很小 | 高 |
| L2/L3缓存 | 快 | 小 | 较高 |
| 主存(RAM) | 中 | 中 | 中 |
| 固态硬盘 | 较慢 | 大 | 较低 |
| 机械硬盘 | 慢 | 很大 | 低 |

**存储器特点**:
- 越快越贵
- 越快越小
- 层次化设计

---

#### 4. 操作系统 (Operating System/操作系统)

**白话解释**:  
管理计算机硬件和软件资源的程序。

**主要功能**:
| 功能 | 说明 |
|------|------|
| 进程管理 | CPU调度 |
| 内存管理 | 内存分配 |
| 文件系统 | 数据存储 |
| 设备管理 | 硬件驱动 |
| 用户接口 | 人机交互 |

**常见操作系统**:
- Windows
- Linux
- macOS
- Android
- iOS

### 二、程序设计

#### 5. 算法 (Algorithm/算法)

**白话解释**:  
解决问题的有限步骤序列。

**算法特性**:
1. **有穷性**: 有限步骤完成
2. **确定性**: 每步明确
3. **可行性**: 可以执行
4. **输入**: 0或多个输入
5. **输出**: 1或多个输出

**复杂度分析**:
| 复杂度 | 名称 | 例子 |
|--------|------|------|
| O(1) | 常数 | 数组访问 |
| O(log n) | 对数 | 二分查找 |
| O(n) | 线性 | 遍历 |
| O(n log n) | 线性对数 | 快速排序 |
| O(n²) | 平方 | 冒泡排序 |

---

#### 6. 数据结构 (Data Structure/数据结构)

**白话解释**:  
组织和存储数据的方式。

**基本数据结构**:
| 结构 | 特点 | 操作复杂度 |
|------|------|------------|
| 数组 | 连续存储 | O(1)访问 |
| 链表 | 指针连接 | O(1)插入 |
| 栈 | LIFO | O(1) |
| 队列 | FIFO | O(1) |
| 树 | 层次结构 | O(log n) |
| 图 | 网状结构 | 依赖算法 |
| 哈希表 | 键值对 | O(1)平均 |

**选择原则**:
根据操作特点选择合适的数据结构。

---

#### 7. 编程语言 (Programming Language/编程语言)

**白话解释**:  
人与计算机交流的语言。

**语言分类**:
| 类型 | 特点 | 例子 |
|------|------|------|
| 机器语言 | 0和1 | 二进制 |
| 汇编语言 | 助记符 | ASM |
| 高级语言 | 接近自然语言 | C, Java, Python |
| 脚本语言 | 解释执行 | Python, JavaScript |

**编程范式**:
- 面向过程
- 面向对象
- 函数式
- 声明式

---

#### 8. 编译与解释

**白话解释**:  
- 编译: 一次翻译，执行多次
- 解释: 边翻译边执行

**对比**:
| 特点 | 编译型 | 解释型 |
|------|--------|--------|
| 执行速度 | 快 | 慢 |
| 开发效率 | 低 | 高 |
| 跨平台 | 差 | 好 |
| 例子 | C, C++ | Python, JS |

**编译过程**:
```
源代码 → 词法分析 → 语法分析 → 语义分析 → 中间代码 → 优化 → 目标代码
```

### 四、网络技术

#### 11. 计算机网络 (Computer Network/计算机网络)

**白话解释**:  
连接多台计算机实现资源共享和通信。

**网络分类**:
| 类型 | 范围 | 例子 |
|------|------|------|
| LAN | 局域网 | 办公室网络 |
| WAN | 广域网 | 互联网 |
| MAN | 城域网 | 城市网络 |

**OSI七层模型**:
| 层次 | 名称 | 功能 |
|------|------|------|
| 7 | 应用层 | 应用程序 |
| 6 | 表示层 | 数据格式 |
| 5 | 会话层 | 会话管理 |
| 4 | 传输层 | 端到端传输 |
| 3 | 网络层 | 路由选择 |
| 2 | 数据链路层 | 帧传输 |
| 1 | 物理层 | 比特传输 |

---

#### 12. TCP/IP协议 (TCP/IP/传输控制协议/网际协议)

**白话解释**:  
互联网通信的协议标准。

**四层模型**:
| 层 | 协议 |
|----|------|
| 应用层 | HTTP, FTP, DNS, SMTP |
| 传输层 | TCP, UDP |
| 网络层 | IP, ICMP, ARP |
| 链路层 | Ethernet, WiFi |

**TCP三次握手**:
1. SYN
2. SYN+ACK
3. ACK

**TCP特点**: 可靠、有序、面向连接

---

### 七、前沿技术

#### 19. 云计算 (Cloud Computing/云计算)

**白话解释**:  
通过网络提供计算资源服务。

**服务模式**:
| 模式 | 内容 | 例子 |
|------|------|------|
| IaaS | 基础设施 | AWS EC2 |
| PaaS | 平台 | Heroku |

**特点**:
- 按需使用
- 弹性扩展
- 按量计费

---

#### 20. 区块链 (Blockchain/区块链)

**白话解释**:  
去中心化的分布式账本技术。

**核心特点**:
1. **去中心化**: 无中央机构
2. **不可篡改**: 数据难以修改
3. **透明公开**: 所有交易可见
4. **共识机制**: 节点达成共识

**应用**:
- 加密货币
- 智能合约
- 供应链追溯
- 数字身份
