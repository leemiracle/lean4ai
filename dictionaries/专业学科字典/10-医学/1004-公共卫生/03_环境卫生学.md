### 一、大气污染概述

#### 1. 大气污染 (Air Pollution/大气污染)

**白话解释**:  
空气中有害物质超标，影响健康和环境。

**生活例子**:  
- 雾霾天气：PM2.5超标，能见度下降
- 汽车尾气：交通拥堵时空气质量差
- 工厂排放：烟囱冒黑烟

**学术定义**:  
大气中污染物浓度达到有害程度，超过了环境质量标准和卫生标准，破坏生态系统和人类正常生存条件，对人、动植物和物体产生不良影响的现象。

**大气污染源分类**:
```
污染源分类
├── 按产生方式
│   ├── 自然源：火山、沙尘暴、森林火灾
│   └── 人为源：工业、交通、生活
│
├── 按排放形式
│   ├── 点源：工厂烟囱
│   ├── 线源：公路
│   └── 面源：居民区
│
└── 按空间分布
    ├── 局地污染：某工厂附近
    └── 区域污染：整个城市
```

**主要大气污染物**:
| 污染物 | 来源 | 危害 |
|--------|------|------|
| PM2.5 | 燃烧、扬尘 | 呼吸系统疾病 |
| SO₂ | 燃煤 | 呼吸道刺激 |
| NOx | 汽车尾气 | 呼吸道损伤 |
| CO | 不完全燃烧 | 缺氧 |
| O₃ | 光化学反应 | 呼吸道刺激 |
| 铅 | 汽油（无铅汽油已减少） | 神经系统损伤 |

**费曼记忆**:  
大气污染 = 脏空气 = 燃烧+尾气+工厂排放

**临床联系**:  
- 哮喘发作：污染天气加重
- 慢阻肺：长期暴露风险
- 肺癌：PM2.5是一类致癌物

---

#### 2. PM2.5 (Particulate Matter 2.5/PM2.5)

**白话解释**:  
空气中直径≤2.5微米的颗粒物，小到能进入肺泡甚至血液。

**生活例子**:  
- 雾霾天：PM2.5浓度高
- 柴油车冒黑烟：PM2.5主要来源
- 空气净化器：过滤PM2.5

**学术定义**:  
空气动力学直径≤2.5μm的颗粒物，又称细颗粒物，能够深入呼吸道和肺泡，对人体健康危害最大。

**PM2.5的特性**:
```
PM2.5特点
├── 来源
│   ├── 一次排放：燃烧直接产生
│   └── 二次生成：气态污染物转化
│
├── 成分
│   ├── 有机物：多环芳烃等
│   ├── 无机物：硫酸盐、硝酸盐
│   └── 重金属：铅、镉、砷等
│
└── 特性
    ├── 比表面积大：吸附有害物质
    ├── 停留时间长：数天至数周
    └── 传输距离远：可跨区域
```

**PM2.5健康危害**:
| 系统 | 危害 |
|------|------|
| 呼吸系统 | 哮喘、慢阻肺、肺癌 |
| 心血管系统 | 心肌梗死、心律失常 |
| 神经系统 | 认知功能下降 |
| 生殖系统 | 低出生体重 |

**PM2.5标准（中国）**:
- 年平均：≤35μg/m³
- 24小时平均：≤75μg/m³

**费曼记忆**:  
PM2.5 = 小颗粒，入肺入血，危害最大

**临床联系**:  
- 心肺疾病患者：减少污染天气外出
- 健康宣教：雾霾天戴口罩
- 疾病监测：污染高峰后就诊增加

---

#### 3. 光化学烟雾 (Photochemical Smog/光化学烟雾)

**白话解释**:  
阳光照射汽车尾气产生的"毒雾"，夏天中午最严重。

**生活例子**:  
- 洛杉矶光化学烟雾事件：历史上著名公害事件
- 夏天中午蓝天变成灰白色
- 刺眼、流泪、咳嗽

**学术定义**:  
大气中的氮氧化物和碳氢化合物等一次污染物，在阳光照射下发生光化学反应，生成臭氧、过氧乙酰硝酸酯（PAN）等二次污染物，形成的混合污染现象。

**光化学烟雾的形成条件**:
```
光化学烟雾形成条件
├── 前体物
│   ├── 氮氧化物（NOx）
│   └── 碳氢化合物（HC）
│
├── 气象条件
│   ├── 强烈阳光
│   ├── 高温
│   └── 低风速
│
└── 地形条件
    └── 盆地或谷地（不利扩散）
```

**光化学烟雾的成分**:
| 成分 | 特性 |
|------|------|
| 臭氧（O₃） | 主要成分，强氧化剂 |
| PAN | 强刺激物 |
| 醛类 | 甲醛、乙醛 |
| 硝酸酯 | 硝酸过氧乙酰酯 |

**光化学烟雾的危害**:
- **眼部**: 刺痛、流泪
- **呼吸道**: 咳嗽、呼吸困难
- **植物**: 叶片损伤

**费曼记忆**:  
光化学烟雾 = 阳光+尾气=毒雾

**临床联系**:  
- 夏季午后：空气质量最差
- 户外活动：避开污染高峰
- 眼部症状：可能因臭氧刺激

### 二、饮用水卫生

#### 4. 水污染 (Water Pollution/水污染)

**白话解释**:  
水里混入了有害物质，不能喝了。

**生活例子**:  
- 工厂排放：河流变黑发臭
- 农药流失：地下水污染
- 生活污水：细菌超标

**学术定义**:  
由于人类活动排放的污染物进入水体，使水和水体的物理、化学性质或生物群落组成发生变化，从而降低水体使用价值的现象。

**水污染源分类**:
```
水污染源
├── 点源污染
│   ├── 工业废水
│   └── 生活污水
│
├── 面源污染
│   ├── 农田径流
│   └── 城市径流
│
└── 内源污染
    └── 底泥释放
```

**主要水污染物**:
| 类型 | 污染物 | 来源 | 危害 |
|------|--------|------|------|
| 有机物 | BOD、COD | 生活污水 | 水体缺氧 |
| 重金属 | 铅、汞、镉 | 工业 | 慢性中毒 |
| 病原体 | 细菌、病毒 | 粪便 | 传染病 |
| 营养盐 | 氮、磷 | 农业、生活 | 富营养化 |
| 内分泌干扰物 | 农药、塑化剂 | 工业、生活 | 内分泌紊乱 |

**费曼记忆**:  
水污染 = 脏水 = 工业+农业+生活排放

**临床联系**:  
- 水源性传染病：霍乱、痢疾
- 重金属中毒：水俣病（汞）、痛痛病（镉）
- 肿瘤：砷污染与皮肤癌

---

#### 5. 饮用水标准 (Drinking Water Standards/饮用水标准)

**白话解释**:  
"好水"的标准——规定了水里有啥、不能有啥、含量多少。

**生活例子**:  
- 自来水检测：确保符合标准
- 瓶装水：标注符合国家标准
- 水质报告：每年发布的检测结果

**学术定义**:  
对饮用水中各种物质的最大允许浓度、物理化学指标和卫生要求所作的强制性规定。

**主要水质指标**:
```
饮用水标准主要指标
├── 微生物指标
│   ├── 总大肠菌群：不得检出
│   ├── 耐热大肠菌群：不得检出
│   └── 大肠埃希氏菌：不得检出
│
├── 毒理指标
│   ├── 砷：≤0.01mg/L
│   ├── 铅：≤0.01mg/L
│   ├── 氟化物：≤1.0mg/L
│   └── 硝酸盐：≤10mg/L
│
├── 感官性状指标
│   ├── 色度：≤15度
│   ├── 浑浊度：≤1NTU
│   ├── 臭和味：无异臭异味
│   └── 肉眼可见物：无
│
└── 一般化学指标
    ├── pH：6.5-8.5
    ├── 总硬度：≤450mg/L
    └── 溶解性总固体：≤1000mg/L
```

**生活饮用水卫生标准（GB5749-2022）**:
- 水质指标：97项
- 常规指标：43项
- 非常规指标：54项

**费曼记忆**:  
饮用水标准 = 安全水的"尺子"

**临床联系**:  
- 饮用水安全：预防水源性疾病
- 水质监测：及时发现污染
- 应急处理：污染事件响应

---

#### 6. 饮用水消毒 (Water Disinfection/饮用水消毒)

**白话解释**:  
杀灭水里的细菌和病毒，让水能安全饮用。

**生活例子**:  
- 自来水加氯消毒
- 煮沸水杀菌
- 野外用消毒片

**学术定义**:  
利用物理或化学方法杀灭水中病原微生物，使饮用水达到卫生标准的处理过程。

**常用消毒方法**:
| 方法 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| 氯消毒 | 氯氧化杀菌 | 成本低、持续消毒 | 产生消毒副产物 |
| 臭氧消毒 | 强氧化杀菌 | 效果好、无残留 | 成本高、无持续消毒 |
| 紫外线消毒 | UV破坏DNA | 无化学残留 | 无持续消毒 |
| 二氧化氯 | 氧化杀菌 | 效果好、副产物少 | 成本较高 |

**氯消毒的原理**:
```
氯消毒过程
├── Cl₂ + H₂O → HOCl + HCl
├── HOCl → H⁺ + OCl⁻
└── HOCl和OCl⁻杀菌

有效氯：HOCl和OCl⁻的总量
余氯：消毒后水中剩余的氯
├── 游离余氯：HOCl + OCl⁻
└── 化合余氯：与氨氮结合的氯
```

**消毒副产物**:
- 三卤甲烷（THMs）
- 卤乙酸（HAAs）
- 致癌风险

**费曼记忆**:  
消毒 = 杀菌，氯最常用，但有副产物

**临床联系**:  
- 旅行饮水安全：选择可靠水源
- 应急饮水：煮沸最简单有效
- 消毒副产物：长期风险需关注

### 三、土壤污染

#### 7. 土壤污染 (Soil Pollution/土壤污染)

**白话解释**:  
土地被有害物质"毒害"了，种出来的粮食不安全。

**生活例子**:  
- 工业废渣倾倒：重金属污染土壤
- 农药过度使用：土壤残留
- 电子垃圾：铅、汞污染

**学术定义**:  
由于人类活动产生的有害物质进入土壤，积累到一定程度，超过土壤自净能力，引起土壤组成、结构和功能变化，并通过食物链影响人体健康的现象。

**土壤污染源**:
```
土壤污染来源
├── 工业污染
│   ├── 工业废水
│   ├── 工业废渣
│   └── 大气沉降
│
├── 农业污染
│   ├── 农药
│   ├── 化肥
│   └── 畜禽粪便
│
└── 生活污染
    ├── 生活垃圾
    └── 生活污水
```

**主要土壤污染物**:
| 类型 | 污染物 | 来源 | 危害途径 |
|------|--------|------|----------|
| 重金属 | 铅、镉、汞、砷 | 工业、采矿 | 食物链富集 |
| 有机污染物 | 农药、多氯联苯 | 农业、工业 | 作物吸收 |
| 病原体 | 细菌、病毒、寄生虫 | 粪便 | 直接接触 |
| 放射性物质 | 铀、钚 | 核工业 | 辐射 |

**费曼记忆**:  
土壤污染 = 毒土 = 工业+农业+生活污染

**临床联系**:  
- 痛痛病：镉污染，骨痛、骨折
- 水俣病：汞污染，神经系统损伤
- 慢性中毒：重金属长期暴露

---

#### 8. 重金属污染 (Heavy Metal Pollution/重金属污染)

**白话解释**:  
土壤中的"毒金属"——铅、汞、镉等，在食物链中富集，最终伤害人类。

**生活例子**:  
- 镉大米：湖南等地镉污染区大米
- 血铅超标：儿童居住在工业区
- 汞超标：食用受污染的鱼类

**学术定义**:  
重金属元素进入环境并积累到有害水平，通过生物富集和食物链传递，对人体健康造成危害的污染现象。

**主要重金属污染物**:
```
重金属污染
├── 铅（Pb）
│   ├── 来源：冶炼、电池、汽油
│   ├── 靶器官：神经系统（儿童敏感）
│   └── 表现：智力下降、贫血
│
├── 汞（Hg）
│   ├── 来源：冶炼、仪表、化工
│   ├── 靶器官：神经系统
│   └── 表现：水俣病
│
├── 镉（Cd）
│   ├── 来源：冶炼、电镀、电池
│   ├── 靶器官：肾脏、骨骼
│   └── 表现：痛痛病
│
└── 砷（As）
    ├── 来源：冶炼、农药、矿石
    ├── 靶器官：皮肤、肝脏
    └── 表现：皮肤癌、周围神经病
```

**重金属污染的特点**:
1. **隐蔽性**: 不易发现
2. **持久性**: 难以降解
3. **富集性**: 食物链放大
4. **不可逆性**: 难以修复

**费曼记忆**:  
重金属 = 隐形杀手，富集、持久、难治

**临床联系**:  
- 儿童血铅检测：居住在高风险区
- 镉暴露监测：肾功能检查
- 预防为主：远离污染源

---

#### 9. 土壤自净 (Soil Self-purification/土壤自净)

**白话解释**:  
土壤的"消化能力"——能分解污染物，但能力有限。

**生活例子**:  
- 落叶腐烂：土壤微生物分解
- 粪便分解：土壤净化
- 石油泄漏：土壤微生物降解

**学术定义**:  
土壤通过物理、化学和生物学作用，将污染物转化为无害或危害较小的物质，使土壤恢复到原有功能的能力。

**土壤自净的机制**:
```
土壤自净机制
├── 物理作用
│   ├── 稀释：与周围土壤混合
│   ├── 扩散：向四周扩散
│   └── 吸附：粘土矿物吸附
│
├── 化学作用
│   ├── 氧化还原：污染物价态改变
│   ├── 沉淀：形成不溶性化合物
│   └── 络合：与有机质结合
│
└── 生物作用（最重要）
    ├── 微生物分解：细菌、真菌
    ├── 植物吸收：植物修复
    └── 动物吞食：土壤动物
```

**影响土壤自净的因素**:
| 因素 | 影响 |
|------|------|
| 温度 | 温度适中，微生物活性高 |
| 湿度 | 湿度适中，有利微生物 |
| pH | 中性偏碱有利于降解 |
| 有机质 | 提供营养，促进降解 |
| 通气性 | 好氧微生物需要氧气 |

**土壤自净的限度**:
- 污染超过自净能力：造成积累
- 人工修复需要：超负荷污染

**费曼记忆**:  
土壤自净 = 土壤的"清洁工"，但能力有限

**临床联系**:  
- 污染风险评估：考虑土壤自净能力
- 修复技术选择：生物修复利用自净
- 环境容量：不能超过自净限度

### 四、室内空气污染

#### 10. 室内空气污染 (Indoor Air Pollution/室内空气污染)

**白话解释**:  
室内空气比室外还"脏"——装修材料、家具、烹饪都可能污染室内空气。

**生活例子**:  
- 新装修房子有味道：甲醛超标
- 厨房油烟：烹饪污染
- 办公室闷：二氧化碳浓度高

**学术定义**:  
室内空气中存在有害物质，浓度超过卫生标准，对人体健康产生不良影响的现象。

**室内空气污染来源**:
```
室内污染源
├── 建筑装修材料
│   ├── 甲醛：人造板、胶黏剂
│   ├── 苯系物：油漆、涂料
│   └── TVOC：各种有机物
│
├── 人类活动
│   ├── 烹饪油烟
│   ├── 吸烟
│   └── 呼吸代谢
│
├── 家具用品
│   ├── 家具释放
│   └── 清洁剂
│
└── 室外进入
    ├── 大气污染物
    └── 花粉、尘螨
```

**主要室内污染物**:
| 污染物 | 来源 | 危害 |
|--------|------|------|
| 甲醛 | 人造板、胶水 | 致癌、呼吸道刺激 |
| 苯 | 油漆、溶剂 | 白血病风险 |
| TVOC | 装修材料 | 头痛、过敏 |
| PM2.5 | 烹饪、吸烟 | 呼吸系统疾病 |
| CO₂ | 呼吸 | 头晕、嗜睡 |
| 氡 | 建材、土壤 | 肺癌 |

**费曼记忆**:  
室内污染 = 装修+生活+室外进入

**临床联系**:  
- 病态建筑综合征：头痛、乏力、恶心
- 儿童哮喘：室内过敏原
- 装修病：甲醛中毒

---

#### 11. 甲醛 (Formaldehyde/甲醛)

**白话解释**:  
新装修的"毒气"——来自板材和胶水，有刺激性气味。

**生活例子**:  
- 新家具味道大：甲醛释放
- 新车有异味：内饰材料释放
- 检测超标：需要通风

**学术定义**:  
一种无色、有强烈刺激性气味的气体，是最常见的室内空气污染物之一，被国际癌症研究机构（IARC）列为一类致癌物。

**甲醛的来源与特性**:
```
甲醛来源
├── 人造板材：刨花板、密度板
├── 胶黏剂：脲醛树脂
├── 涂料：部分油漆
└── 纺织品：防皱处理

甲醛特性
├── 释放期：3-15年
├── 溶解性：易溶于水
└── 温度影响：温度高释放快
```

**甲醛健康危害**:
| 浓度(mg/m³) | 症状 |
|-------------|------|
| 0.06-0.07 | 嗅觉敏感者可闻到 |
| 0.1 | 有刺激感 |
| 0.5 | 流泪、咳嗽 |
| 12-24 | 呼吸困难 |
| >60 | 肺水肿、死亡 |

**甲醛标准（室内空气）**:
- GB/T 18883-2022：≤0.08mg/m³
- GB 50325-2020：≤0.07mg/m³（I类民用建筑）

**费曼记忆**:  
甲醛 = 装修毒气，3-15年释放，一类致癌物

**临床联系**:  
- 装修后检测：入住前检测甲醛
- 儿童房间：特别注意低浓度危害
- 新生儿缺陷：孕期暴露风险

---

#### 12. 病态建筑综合征 (Sick Building Syndrome/SBS/病态建筑综合征)

**白话解释**:  
"建筑让人病"——在某些建筑里待着就难受，离开就好了。

**生活例子**:  
- 办公室里头痛、眼干
- 到了公司就觉得累
- 回家后症状消失

**学术定义**:  
在特定建筑物或非工业室内环境中，人员出现的急性健康不适症状，离开该环境后症状缓解或消失，但找不到明确的病因或污染物。

**SBS的主要症状**:
```
SBS症状分类
├── 黏膜刺激症状
│   ├── 眼干、眼痒
│   ├── 咽喉不适
│   └── 鼻塞、流涕
│
├── 神经系统症状
│   ├── 头痛
│   ├── 头晕
│   └── 注意力不集中
│
├── 呼吸系统症状
│   ├── 咳嗽
│   └── 胸闷
│
└── 皮肤症状
    ├── 皮肤干燥
    └── 皮疹
```

**SBS的原因**:
| 因素 | 具体内容 |
|------|----------|
| 物理因素 | 温度、湿度、光照 |
| 化学因素 | 甲醛、VOC、CO₂ |
| 生物因素 | 细菌、霉菌、尘螨 |
| 心理因素 | 工作压力、空调不适 |

**SBS的诊断标准**:
1. 症状在特定建筑中出现
2. 离开建筑后症状改善
3. 无其他明确疾病诊断
4. 建筑中多人有类似症状

**费曼记忆**:  
SBS = 建筑让人病，离开就好，找不到原因

**临床联系**:  
- 办公室人群：高发人群
- 诊断排除法：排除其他疾病
- 改善环境：通风、绿植、净化器

### 五、风险评估

#### 13. 环境健康风险评估 (Environmental Health Risk Assessment/环境健康风险评估)

**白话解释**:  
"算算有多大危险"——评估环境污染对健康的危害程度。

**生活例子**:  
- 垃圾焚烧厂选址：评估对居民健康风险
- 工业项目环评：健康风险评估
- 化学品管理：评估使用风险

**学术定义**:  
对环境有害因素暴露可能造成的人体健康危害进行定性定量评估的过程，是环境管理决策的科学基础。

**风险评估的四个步骤**:
```
风险评估流程
├── 1. 危害识别
│   └── 确定物质是否有害
│   └── 方法：流行病学、动物实验
│
├── 2. 剂量-效应评估
│   └── 确定剂量与健康效应的关系
│   └── 关键参数：阈值、NOAEL
│
├── 3. 暴露评估
│   └── 确定人群暴露的剂量
│   └── 途径：吸入、食入、皮肤
│
└── 4. 风险特征描述
    └── 综合以上信息，描述风险
    └── 结果：致癌风险、非致癌危害指数
```

**风险评估的输出**:
| 类型 | 指标 | 可接受标准 |
|------|------|------------|
| 致癌风险 | 终生致癌风险 | <10⁻⁶ |
| 非致癌风险 | 危害指数HI | <1 |

**费曼记忆**:  
风险评估 = 识别危害→确定关系→评估暴露→描述风险

**临床联系**:  
- 环境污染事件：健康风险评估
- 社区健康调查：风险评估指导
- 公共卫生政策：基于风险评估

---

#### 14. 剂量-效应关系 (Dose-Response Relationship/剂量-效应关系)

**白话解释**:  
"量变到质变"——暴露越多，效应越强。

**生活例子**:  
- 喝酒：喝得越多，越醉
- 辐射：剂量越大，损害越重
- 药物：剂量决定疗效和毒性

**学术定义**:  
化学物质或物理因素作用于生物体，其产生的生物学效应与接触剂量之间存在的关系。

**剂量-效应关系曲线**:
```
剂量-效应曲线类型
├── S形曲线
│   └── 低剂量无效应，高剂量效应增强
│
├── 直线关系
│   └── 剂量与效应成正比
│
└── 阈值效应
    └── 低于阈值无效应
```

**关键参数**:
| 参数 | 定义 |
|------|------|
| NOAEL | 未观察到有害效应的最高剂量 |
| LOAEL | 观察到有害效应的最低剂量 |
| RfD | 参考剂量（安全剂量） |
| LD₅₀ | 半数致死剂量 |

**阈值vs无阈值**:
- **有阈值物质**: 低于某剂量无效应（大多数非致癌物）
- **无阈值物质**: 任何剂量都可能有风险（致癌物）

**费曼记忆**:  
剂量-效应 = 暴露越多，效应越强

**临床联系**:  
- 药物剂量：疗效和毒性平衡
- 职业暴露：设定安全限值
- 风险评估：剂量-效应是基础

---

#### 15. 暴露评估 (Exposure Assessment/暴露评估)

**白话解释**:  
"算算接触了多少"——评估人群接触污染物的量和途径。

**生活例子**:  
- 职业暴露监测：工人接触化学物剂量
- 居民暴露评估：住在工厂附近的风险
- 食物链暴露：吃受污染食物的量

**学术定义**:  
定量或定性描述人群与环境有害因素接触的强度、频率、持续时间和途径的过程。

**暴露途径**:
```
暴露途径
├── 吸入暴露
│   └── 通过呼吸系统
│   └── 空气污染物
│
├── 摄入暴露
│   ├── 食入：食物、饮水
│   └── 皮肤吸收：接触污染物
│
└── 多途径暴露
    └── 同一物质可能通过多种途径
```

**暴露参数**:
| 参数 | 说明 |
|------|------|
| 暴露浓度 | 环境中污染物浓度 |
| 摄入率 | 呼吸量、饮水量 |
| 暴露频率 | 每年暴露天数 |
| 暴露持续时间 | 暴露年限 |
| 体重 | 影响剂量计算 |

**暴露剂量的计算**:
```
日均暴露剂量（ADD）公式
ADD = (C × IR × EF × ED) / (BW × AT)

├── C：污染物浓度
├── IR：摄入率
├── EF：暴露频率
├── ED：暴露持续时间
├── BW：体重
└── AT：平均时间
```

**费曼记忆**:  
暴露评估 = 算接触了多少、多久、通过什么途径

**临床联系**:  
- 职业病诊断：评估职业暴露
- 环境污染调查：评估居民暴露
- 风险计算：暴露是关键参数

---

#### 16. 生物标志物 (Biomarkers/生物标志物)

**白话解释**:  
"身体里的证据"——检测体内有害物质或其效应，证明暴露和危害。

**生活例子**:  
- 血铅检测：证明铅暴露
- 尿镉检测：反映镉负荷
- DNA加合物：证明致癌物暴露

**学术定义**:  
反映生物系统与环境因子相互作用的任何可测量的指标，用于评估暴露、效应和易感性。

**生物标志物的分类**:
```
生物标志物分类
├── 暴露标志物
│   ├── 内剂量：血铅、尿汞
│   └── 生物有效剂量：DNA加合物
│   └── 反映体内负荷
│
├── 效应标志物
│   ├── 早期生物效应：酶活性改变
│   ├── 结构/功能改变：肺功能下降
│   └── 反映健康损害
│
└── 易感性标志物
    ├── 基因多态性
    └── 反映个体差异
```

**常用生物标志物**:
| 暴露 | 标志物 | 生物材料 |
|------|--------|----------|
| 铅 | 血铅 | 全血 |
| 汞 | 尿汞、血汞 | 尿、血 |
| 镉 | 尿β2微球蛋白、尿镉 | 尿 |
| 苯 | 尿反-反-粘康酸 | 尿 |
| 多环芳烃 | 1-羟基芘 | 尿 |

**费曼记忆**:  
生物标志物 = 暴露的证据+效应的证据+易感性指标

**临床联系**:  
- 职业健康监护：定期检测生物标志物
- 中毒诊断：生物标志物确诊
- 流行病学研究：客观暴露评估

### 六、公害病

#### 17. 水俣病 (Minamata Disease/水俣病)

**白话解释**:  
日本历史上最严重的汞中毒事件——工厂排放甲基汞，污染鱼贝，人吃了中毒。

**生活例子**:  
- 日本熊本县水俣湾事件：1956年发现
- 吃受污染的鱼后神经系统受损
- 先天性水俣病：母亲吃鱼，婴儿中毒

**学术定义**:  
由甲基汞中毒引起的以神经系统损害为主的公害病，因1956年在日本熊本县水俣湾首次发现而得名。

**水俣病的病因**:
```
水俣病病因链
├── 污染源
│   └── 氮肥厂排放含汞废水
│
├── 甲基化
│   └── 汞在底泥中转化为甲基汞
│
├── 生物富集
│   └── 鱼贝类富集甲基汞
│
└── 人类摄入
    └── 食用受污染鱼贝
```

**水俣病的临床表现**:
```
水俣病症状
├── 感觉障碍
│   ├── 四肢麻木
│   └── 痛温觉减退
│
├── 运动障碍
│   ├── 共济失调
│   └── 步态不稳
│
├── 视力障碍
│   └── 视野缩小
│
├── 听力障碍
│   └── 听力下降
│
└── 语言障碍
    └── 语言不清
```

**水俣病的教训**:
- 环境保护的重要性
- 生物富集效应
- 先天性中毒的可能性

**费曼记忆**:  
水俣病 = 汞污染+鱼富集+神经系统损害

**临床联系**:  
- 工业污染防控
- 食品安全监管
- 先天性疾病的预防

#### 18. 痛痛病 (Itai-itai Disease/痛痛病)

**白话解释**:  
"痛得要命"的病——镉中毒导致骨软化，全身剧痛。

**生活例子**:  
- 日本富山县神通川流域事件
- 吃镉污染的大米
- 骨骼变形、剧痛

**学术定义**:  
由镉慢性中毒引起的以骨软化和骨质疏松为特征的公害病，因患者全身疼痛、发出"痛啊痛啊"的喊叫而得名。

**痛痛病的病因**:
```
痛痛病病因链
├── 污染源
│   └── 矿山排放含镉废水
│
├── 土壤污染
│   └── 镉污染稻田土壤
│
├── 作物吸收
│   └── 大米富集镉
│
└── 人类摄入
    └── 长期食用镉米
```

**痛痛病的临床表现**:
| 系统 | 症状 |
|------|------|
| 骨骼系统 | 骨痛、骨软化、骨折 |
| 泌尿系统 | 肾小管损伤、蛋白尿 |
| 消化系统 | 食欲减退、贫血 |
| 全身 | 消瘦、衰弱 |

**痛痛病的诊断标准**:
1. 长期居住在污染区
2. 肾小管功能障碍
3. 骨软化或骨质疏松
4. 排除其他骨病

**费曼记忆**:  
痛痛病 = 镉污染+大米+骨软化剧痛

**临床联系**:  
- 土壤污染治理
- 食品安全标准
- 重金属中毒诊断

## 🎯 重点概念速记

### 必背概念（18个）

1. **大气污染**: 空气有害物质超标
2. **PM2.5**: ≤2.5μm颗粒物，入肺入血
3. **光化学烟雾**: 阳光+尾气=毒雾
4. **水污染**: 有害物质进入水体
5. **饮用水标准**: 安全水的质量要求
6. **氯消毒**: 杀菌但产生副产物
7. **土壤污染**: 有害物质进入土壤
8. **重金属污染**: 铅汞镉砷，富集难降解
9. **土壤自净**: 土壤清洁能力有限
10. **室内空气污染**: 装修+生活+室外
11. **甲醛**: 装修毒气，3-15年释放
12. **SBS**: 建筑让人病，离开就好
13. **风险评估**: 识别-剂量-暴露-风险
14. **剂量-效应**: 暴露越多，效应越强
15. **暴露评估**: 算接触多少、多久
16. **生物标志物**: 暴露的证据
17. **水俣病**: 汞中毒，神经损害
18. **痛痛病**: 镉中毒，骨软化剧痛
