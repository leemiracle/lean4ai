## 📖 第一章：统计学基础概念

---

### 一、基本概念

#### 1. 总体与样本 (Population & Sample/总体与样本)

**白话解释**:  
- 总体：研究对象的"全部"（比如所有中国人）
- 样本：从总体中抽取的"代表"（比如1000个中国人）

**生活例子**:  
- 全国高血压调查：总体是全国人，样本是调查的那些人
- 民意调查：总体是所有选民，样本是被调查的选民
- 药物试验：总体是所有病人，样本是参加试验的病人

**学术定义**:  
- **总体**: 根据研究目的确定的同质研究对象的全体
- **样本**: 从总体中随机抽取的有代表性的部分个体

**统计推断的目的**:
```
统计推断流程
├── 目标：了解总体特征
├── 方法：从总体中抽取样本
├── 分析：计算样本统计量
└── 推断：用样本推断总体参数
```

**参数 vs 统计量**:
| 概念 | 定义 | 符号 |
|------|------|------|
| 参数 | 总体的数值特征 | μ（总体均数）、σ（总体标准差） |
| 统计量 | 样本的数值特征 | x̄（样本均数）、s（样本标准差） |

**费曼记忆**:  
总体=全部；样本=代表；参数是总体的，统计量是样本的

**临床联系**:  
- 临床试验：样本推断总体疗效
- 流行病学调查：样本推断人群患病率
- 质量控制：抽样检验产品质量

---

#### 2. 变量与数据类型 (Variables & Data Types/变量与数据类型)

**白话解释**:  
数据分"等级"——有的只能数个数，有的可以比大小，有的可以加减乘除。

**生活例子**:  
- 性别：男/女（无序分类）
- 病情：轻/中/重（有序分类）
- 血压值：120mmHg（连续数值）

**学术定义**:  
根据变量取值的性质和数学特性，将数据分为不同的类型，不同类型数据适用不同的统计方法。

**数据类型分类**:
```
数据类型
├── 计量资料（数值变量）
│   ├── 连续型：血压、身高、体重
│   └── 离散型：白细胞计数、呼吸次数
│   └── 可进行加减乘除运算
│
├── 计数资料（分类变量）
│   ├── 二项分类：治愈/未治愈、男/女
│   └── 多项分类：血型A/B/AB/O
│   └── 只能计数，不能计算
│
└── 等级资料（有序变量）
    ├── 有序分类：轻/中/重、优/良/中/差
    └── 有顺序，但不能计算差值
```

**不同数据类型的统计分析方法**:
| 数据类型 | 描述统计 | 假设检验 |
|----------|----------|----------|
| 计量资料 | 均数、标准差 | t检验、方差分析 |
| 计数资料 | 率、构成比 | 卡方检验 |
| 等级资料 | 中位数、秩次 | 秩和检验 |

**费曼记忆**:  
计量=数值；计数=分类；等级=有序分类

**临床联系**:  
- 根据数据类型选择统计方法
- 错误使用方法导致结论不可靠
- 数据转换可改变数据类型

---

#### 3. 抽样误差 (Sampling Error/抽样误差)

**白话解释**:  
"样本不完美代表总体"——样本统计量和总体参数之间的差异。

**生活例子**:  
- 民意调查误差：1000人的调查结果和实际投票结果不同
- 多次抽样：每次抽样的结果不完全一致
- 临床试验：不同样本的结果略有差异

**学术定义**:  
由抽样造成的样本统计量与总体参数之间的差异，是由个体变异和抽样方法造成的不可避免误差。

**抽样误差的特点**:
```
抽样误差
├── 来源：个体变异+抽样方法
├── 不可避免：任何抽样都有误差
├── 可控制：增加样本量可减小
└── 随机性：服从一定的分布规律
```

**标准误 vs 标准差**:
| 指标 | 定义 | 意义 |
|------|------|------|
| 标准差（SD） | 个体值与均数的离散程度 | 描述个体差异 |
| 标准误（SE） | 样本统计量与总体参数的差异 | 描述抽样误差 |

**标准误的计算**:
```
均数的标准误：SE = s/√n
  ├── s：样本标准差
  └── n：样本量

样本量越大，标准误越小
```

**费曼记忆**:  
抽样误差 = 样本和总体的差距，样本量越大误差越小

**临床联系**:  
- 置信区间：考虑抽样误差
- 样本量估计：控制抽样误差
- 结果解释：考虑误差范围

### 二、集中趋势

#### 4. 均数 (Mean/均数)

**白话解释**:  
"平均值"——所有数据加起来除以个数。

**生活例子**:  
- 班级平均分：所有人分数加起来除以人数
- 平均血压：一组人的血压平均值
- 平均身高：一群人的身高平均值

**学术定义**:  
所有观测值之和除以观测值个数，是描述数据集中趋势最常用的指标。

**均数的计算**:
```
算术均数公式
x̄ = (x₁ + x₂ + ... + xₙ)/n = Σx/n

例子：5个病人血压 120, 130, 125, 135, 140
x̄ = (120+130+125+135+140)/5 = 650/5 = 130 mmHg
```

**均数的特点**:
1. **利用所有数据**: 每个值都参与计算
2. **受极端值影响**: 离群值会拉偏均数
3. **适用于正态分布**: 偏态分布不适用

**均数的优缺点**:
| 优点 | 缺点 |
|------|------|
| 计算简单 | 受极端值影响大 |
| 利用所有信息 | 不适合偏态分布 |
| 数学性质好 | 不适合开口数据 |

**费曼记忆**:  
均数 = 总和÷个数

**临床联系**:  
- 实验室正常值：健康人均数±标准差
- 疗效评价：治疗前后均数比较
- 偏态分布用中位数

---

#### 5. 中位数 (Median/中位数)

**白话解释**:  
"中间那个数"——排序后，一半数据比它大，一半比它小。

**生活例子**:  
- 房价中位数：比平均房价更能反映真实水平
- 收入中位数：不受少数超高收入者影响
- 医院住院天数中位数：不受少数长期住院影响

**学术定义**:  
将观测值按大小排序后，位于中间位置的数值，反映数据的中间水平。

**中位数的计算**:
```
中位数计算
├── 奇数个数据：中间那个数
│   例子：3, 5, 7, 9, 11 → 中位数 = 7
│
└── 偶数个数据：中间两个数的均数
    例子：3, 5, 7, 9 → 中位数 = (5+7)/2 = 6
```

**中位数的特点**:
1. **不受极端值影响**: 稳健性好
2. **适用于偏态分布**: 任何分布都能用
3. **适用于开口数据**: ">"或"<"的数据也能用

**均数 vs 中位数比较**:
| 特点 | 均数 | 中位数 |
|------|------|--------|
| 受极端值影响 | 大 | 小 |
| 适用分布 | 正态分布 | 任何分布 |
| 信息利用 | 全部 | 位置信息 |
| 计算复杂度 | 简单 | 需排序 |

**费曼记忆**:  
中位数 = 排序后的中间值，不受极端值影响

**临床联系**:  
- 生存分析：中位生存期
- 偏态数据：住院天数、医疗费用
- 正常值范围：偏态分布用百分位数

---

#### 6. 众数 (Mode/众数)

**白话解释**:  
"出现最多的那个值"——最常见的数。

**生活例子**:  
- 最常见的鞋码：商家备货最多的尺码
- 最常见的年龄：班级里人数最多的年龄段
- 最常见的血型：人群中比例最高的血型

**学术定义**:  
一组数据中出现次数最多的观测值。

**众数的特点**:
```
众数类型
├── 单峰分布：一个众数
├── 双峰分布：两个众数
└── 多峰分布：多个众数
```

**众数的应用**:
- 描述分类数据的集中趋势
- 了解最常见的情况
- 辅助判断分布形态

**费曼记忆**:  
众数 = 出现次数最多的值

**临床联系**:  
- 流行病学：最常见的感染类型
- 分类变量描述：如血型分布
- 市场分析：最畅销的产品

### 三、离散趋势

#### 7. 标准差 (Standard Deviation/标准差)

**白话解释**:  
"数据有多分散"——数据离平均值有多远，标准差越大越分散。

**生活例子**:  
- 两个班平均分都是80分：A班60-100分都有，B班75-85分——A班标准差大
- 血压波动大：标准差大
- 血糖控制不稳定：标准差大

**学术定义**:  
描述数据离散程度的指标，是各观测值与均数离差平方和的均数的平方根。

**标准差的计算**:
```
标准差公式
s = √[Σ(x-x̄)²/(n-1)]

计算步骤：
1. 计算均数 x̄
2. 计算每个值与均数的差 (x-x̄)
3. 差值平方 (x-x̄)²
4. 求和 Σ(x-x̄)²
5. 除以(n-1)
6. 开平方根
```

**标准差的意义**:
```
正态分布中
├── x̄ ± 1s：包含约68%的数据
├── x̄ ± 1.96s：包含约95%的数据
└── x̄ ± 2.58s：包含约99%的数据
```

**方差 vs 标准差**:
| 指标 | 计算 | 单位 |
|------|------|------|
| 方差 | s² | 原始单位的平方 |
| 标准差 | √方差 | 与原始单位相同 |

**费曼记忆**:  
标准差 = 数据离平均值的平均距离

**临床联系**:  
- 参考值范围：均数±1.96标准差
- 血糖波动：评估血糖控制稳定性
- 质量控制：标准差反映检测稳定性

---

#### 8. 变异系数 (Coefficient of Variation/变异系数)

**白话解释**:  
"相对标准差"——标准差除以均数，便于比较不同单位的数据。

**生活例子**:  
- 比较身高和体重的变异：单位不同，用变异系数
- 比较不同实验室检测稳定性：均数不同，用变异系数

**学术定义**:  
标准差与均数的比值，通常用百分数表示，用于比较不同度量单位或均数相差较大的两组数据的变异程度。

**变异系数的计算**:
```
变异系数公式
CV = (s/x̄) × 100%

例子：
├── A组：均数100，标准差10 → CV = 10/100 × 100% = 10%
└── B组：均数50，标准差8 → CV = 8/50 × 100% = 16%

结论：B组变异程度更大
```

**变异系数的应用**:
1. 比较不同单位数据的变异
2. 比较均数相差较大的数据
3. 评估实验室检测精密度

**费曼记忆**:  
变异系数 = 标准差/均数，便于比较

**临床联系**:  
- 实验室质量控制：CV应<5%
- 比较不同检测方法稳定性
- 评估仪器精密度

---

#### 9. 四分位数间距 (Interquartile Range/四分位数间距)

**白话解释**:  
"中间50%数据的范围"——从第25百分位到第75百分位的距离。

**生活例子**:  
- 工资四分位数：25%的人低于这个数，75%的人高于这个数
- 考试成绩分布：中间50%学生的分数范围

**学术定义**:  
上四分位数（P₇₅）与下四分位数（P₂₅）之差，反映中间50%数据的离散程度，是稳健的离散程度指标。

**四分位数的定义**:
```
四分位数
├── Q₁ (P₂₅)：下四分位数，25%的数据小于此值
├── Q₂ (P₅₀)：中位数，50%的数据小于此值
├── Q₃ (P₇₅)：上四分位数，75%的数据小于此值
└── IQR = Q₃ - Q₁：四分位数间距
```

**箱线图的构成**:
```
箱线图
    |          ← 最大值（或Q₃+1.5×IQR）
    |
 ┌──┴──┐
 │     │    ← Q₇₅（上四分位数）
-┤     ├-   ← Q₅₀（中位数）
 │     │    ← Q₂₅（下四分位数）
 └──┬──┘
    |
    |          ← 最小值（或Q₁-1.5×IQR）

    ○          ← 离群值（超出范围）
```

**费曼记忆**:  
IQR = 中间50%数据的范围，不受极端值影响

**临床联系**:  
- 偏态数据描述：用中位数和IQR
- 识别离群值：超出Q₁-1.5×IQR或Q₃+1.5×IQR
- 数据可视化：箱线图展示分布

### 四、正态分布

#### 10. 正态分布 (Normal Distribution/正态分布)

**白话解释**:  
"钟形曲线"——中间多、两边少、对称分布，最常见的分布形式。

**生活例子**:  
- 身高分布：中等身高的人最多，特别高和特别矮的人较少
- 考试成绩分布：中等分数最多，高分和低分较少
- 血压分布：正常血压的人最多

**学术定义**:  
一种连续型概率分布，其曲线呈钟形，两头低、中间高、左右对称，是统计学中最重要的分布。

**正态分布的特征**:
```
正态分布特征
├── 对称性：以均数为中心对称
├── 钟形曲线：中间高、两端低
├── 均数=中位数=众数：三者相等
└── 由两个参数决定：均数μ和标准差σ
```

**正态分布的曲线下面积**:
```
曲线下面积分布
├── μ ± 1σ：68.27%
├── μ ± 1.96σ：95.00%
├── μ ± 2.58σ：99.00%
└── μ ± 3σ：99.73%
```

**标准正态分布**:
- μ = 0, σ = 1的正态分布
- 用Z值转换：Z = (x-μ)/σ
- Z值表可查任何概率

**费曼记忆**:  
正态分布 = 钟形曲线，68%-95%-99%规则

**临床联系**:  
- 参考值范围建立
- 统计方法选择依据
- 质量控制图

---

#### 11. 参考值范围 (Reference Range/参考值范围)

**白话解释**:  
"正常人的数值范围"——95%健康人检测值所在的区间。

**生活例子**:  
- 血糖正常值：3.9-6.1 mmol/L（95%健康人在这个范围）
- 白细胞计数：4-10×10⁹/L
- 血红蛋白：男120-160 g/L

**学术定义**:  
绝大多数"正常人"的某项指标数值范围，通常指95%的健康个体某项指标的分布范围。

**参考值范围的制定方法**:
```
参考值范围制定
├── 正态分布法
│   └── x̄ ± 1.96s（双侧95%）
│   └── x̄ + 1.64s（单侧上限）
│   └── x̄ - 1.64s（单侧下限）
│
└── 百分位数法
    └── P₂.₅ ~ P₉₇.₅（双侧95%）
    └── 适用于偏态分布
```

**单侧 vs 双侧参考值**:
| 类型 | 适用情况 | 计算（95%） |
|------|----------|-------------|
| 双侧 | 过高过低都异常 | P₂.₅ ~ P₉₇.₅ |
| 单侧上限 | 只有过高异常 | < P₉₅ |
| 单侧下限 | 只有过低异常 | > P₅ |

**费曼记忆**:  
参考值范围 = 95%正常人的数值范围

**临床联系**:  
- 实验室检查：判断是否异常
- 体检评估：参考值比较
- 疾病筛查：超出范围需进一步检查

### 五、假设检验原理

#### 12. 假设检验基本思想 (Hypothesis Testing/假设检验基本思想)

**白话解释**:  
"反证法判断差异"——先假设没差异，再看数据是否支持这个假设。

**生活例子**:  
- 新药是否有效？先假设无效，看数据能否推翻
- 男女身高是否有差异？先假设无差异，用数据检验
- 硬币是否公平？先假设公平，看投掷结果

**学术定义**:  
根据样本信息推断总体参数是否具有某种特征（如差异、关系）的统计推断方法，采用"小概率反证"的思想。

**假设检验的步骤**:
```
假设检验五步法
├── 1. 建立假设
│   ├── H₀（无效假设）：假设无差异/无效应
│   └── H₁（备择假设）：假设有差异/有效应
│
├── 2. 确定检验水准α
│   └── 通常α = 0.05
│
├── 3. 计算检验统计量
│   └── t值、Z值、χ²值等
│
├── 4. 确定P值
│   └── P值：假设H₀为真时，出现当前结果或更极端结果的概率
│
└── 5. 做出推断
    ├── P ≤ α：拒绝H₀，接受H₁（有统计学意义）
    └── P > α：不拒绝H₀（无统计学意义）
```

**P值的解读**:
| P值 | 统计学意义 | 实际解释 |
|-----|------------|----------|
| P ≤ 0.05 | 有统计学意义 | 差异不太可能是偶然的 |
| P ≤ 0.01 | 有高度统计学意义 | 差异很不可能偶然 |
| P > 0.05 | 无统计学意义 | 差异可能是偶然的 |

**费曼记忆**:  
假设检验 = 反证法+小概率原理，P<0.05拒绝H₀

**临床联系**:  
- 临床试验：判断疗效是否有统计学意义
- 流行病学：判断关联是否有统计学意义
- 注意：统计学显著≠临床意义

---

#### 13. I型错误与II型错误 (Type I & II Errors/型错误与型错误)

**白话解释**:  
- I型错误："假阳性"——实际没差异，却判断有差异
- II型错误："假阴性"——实际有差异，却判断没差异

**生活例子**:  
- 诊断试验：I型错误=误诊，II型错误=漏诊
- 安检：I型错误=误报，II型错误=漏报
- 法庭判决：I型错误=冤枉好人，II型错误=放走坏人

**学术定义**:  
- **I型错误（α错误）**: H₀为真时，错误拒绝H₀的概率
- **II型错误（β错误）**: H₁为真时，错误接受H₀的概率

**假设检验的四种结果**:
```
实际情况 vs 检验结论
                │  拒绝H₀   │ 不拒绝H₀
    ────────────┼───────────┼──────────
    H₀为真      │  I型错误  │  正确
    H₁为真      │   正确    │ II型错误
```

**检验效能（Power）**:
```
检验效能 = 1 - β
├── 含义：H₁为真时，正确拒绝H₀的概率
├── 通常要求：Power ≥ 0.8
└── 影响因素：样本量、效应大小、α水平
```

**α与β的关系**:
- α减小时，β增大
- 需要权衡两种错误
- 通常固定α=0.05，通过增加样本量减小β

**费曼记忆**:  
I型=假阳性；II型=假阴性；效能=1-β

**临床联系**:  
- 临床试验设计：平衡两类错误
- 诊断试验评估：灵敏度与特异度
- 样本量估计：基于检验效能

---

### 六、常用假设检验方法

#### 14. t检验 (t-test/t检验)

**白话解释**:  
"比较两组平均值"——看两组的平均值是否有显著差异。

**生活例子**:  
- 新药组vs对照组血压下降值比较
- 男性vs女性身高比较
- 治疗前后血压值比较

**学术定义**:  
用于比较两组均数差异是否具有统计学意义的假设检验方法，适用于正态分布、小样本的计量资料。

**t检验的类型**:
```
t检验分类
├── 单样本t检验
│   └── 比较样本均数与已知总体均数
│   └── 例子：某地区儿童身高是否高于全国平均水平
│
├── 配对t检验
│   └── 比较同一对象处理前后的差异
│   └── 例子：治疗前后血压比较
│
└── 两独立样本t检验
    └── 比较两组独立样本的均数
    └── 例子：新药组vs对照组疗效比较
```

**t检验的应用条件**:
1. **正态性**: 数据近似正态分布
2. **独立性**: 各观测值相互独立
3. **方差齐性**: 两组方差相近（两样本t检验）

**t检验的结果解读**:
| P值 | 结论 |
|-----|------|
| P ≤ 0.05 | 两组均数差异有统计学意义 |
| P > 0.05 | 两组均数差异无统计学意义 |

**费曼记忆**:  
t检验 = 比较两组均数，P<0.05有差异

**临床联系**:  
- 药物疗效比较
- 治疗前后效果评估
- 两组基线资料比较

---

#### 15. 卡方检验 (Chi-square Test/卡方检验)

**白话解释**:  
"比较两组率或比例"——看两组的分类比例是否有显著差异。

**生活例子**:  
- 治疗组治愈率80%，对照组60%——是否有差异？
- 吸烟组肺癌发病率是否高于非吸烟组？
- 男女性别比例是否偏离1:1？

**学术定义**:  
用于比较两个或多个样本率/构成比差异的假设检验方法，适用于计数资料（分类变量）。

**卡方检验的类型**:
```
卡方检验分类
├── 四格表卡方检验
│   └── 比较两组率
│   └── 2×2表格
│
├── 行×列表卡方检验
│   └── 比较多组率或构成比
│   └── R×C表格
│
└── 配对卡方检验
    └── 比较配对设计的率
    └── 如两种诊断方法比较
```

**四格表卡方检验示例**:
```
        治愈  未治愈  合计
治疗组   40     10     50
对照组   30     20     50
合计     40     30    100

χ² = Σ[(实际频数-理论频数)²/理论频数]
```

**卡方检验的应用条件**:
1. **样本量足够**: 理论频数≥5
2. **独立性**: 每个个体只出现一次
3. **计数资料**: 分类变量

**费曼记忆**:  
卡方检验 = 比较率或比例，计数资料用

**临床联系**:  
- 治疗效果比较（有效率）
- 危险因素分析（暴露率）
- 诊断试验评价（灵敏度、特异度）

---

#### 16. 方差分析 (ANOVA/方差分析)

**白话解释**:  
"比较三组或更多组的平均值"——t检验的升级版，可以同时比较多组。

**生活例子**:  
- 三种降压药哪一种效果最好？
- 四个地区的平均寿命是否有差异？
- 不同年龄组的血压水平比较

**学术定义**:  
用于比较两组及以上样本均数差异的假设检验方法，通过分解总变异来检验各组均数是否来自同一总体。

**方差分析的类型**:
```
方差分析分类
├── 完全随机设计方差分析
│   └── 比较多个独立组
│   └── 单因素分析
│
├── 随机区组设计方差分析
│   └── 考虑区组因素
│   └── 两因素分析
│
└── 重复测量方差分析
    └── 同一对象多次测量
    └── 时间因素分析
```

**方差分析的基本思想**:
```
总变异分解
总变异(SST) = 组间变异(SSB) + 组内变异(SSW)

├── 组间变异：各组均数与总均数的差异
│   └── 反映处理效应
│
└── 组内变异：组内个体与组均数的差异
    └── 反映随机误差
```

**方差分析的应用条件**:
1. **正态性**: 各组数据正态分布
2. **方差齐性**: 各组方差相等
3. **独立性**: 各观测值相互独立

**方差分析结果解读**:
- P ≤ 0.05：至少有两组均数不同
- 需要事后检验：确定哪两组有差异

**费曼记忆**:  
方差分析 = 多组均数比较，F检验

**临床联系**:  
- 多组治疗方案比较
- 多时间点测量分析
- 多因素效应分析

### 七、相关分析

#### 17. 相关分析 (Correlation Analysis/相关分析)

**白话解释**:  
"两个变量手拉手"——一个增加，另一个也增加（正相关）；一个增加，另一个减少（负相关）。

**生活例子**:  
- 身高和体重：正相关
- 吸烟量和肺功能：负相关
- 学习时间和成绩：正相关
- 年龄和肺活量：负相关

**学术定义**:  
研究两个变量之间线性关系的方向和密切程度的统计分析方法。

**相关系数**:
```
Pearson相关系数r
├── 范围：-1 ≤ r ≤ 1
├── r > 0：正相关
├── r < 0：负相关
└── r = 0：无线性相关

相关程度判断：
├── |r| < 0.3：弱相关
├── 0.3 ≤ |r| < 0.7：中等相关
└── |r| ≥ 0.7：强相关
```

**相关的类型**:
| 类型 | r值 | 散点图 |
|------|-----|--------|
| 完全正相关 | r = 1 | 直线，斜向上 |
| 正相关 | 0 < r < 1 | 散点向上趋势 |
| 无相关 | r = 0 | 散点无规律 |
| 负相关 | -1 < r < 0 | 散点向下趋势 |
| 完全负相关 | r = -1 | 直线，斜向下 |

**Pearson相关应用条件**:
1. 两个变量都是连续型
2. 近似正态分布
3. 线性关系
4. 无极端值

**费曼记忆**:  
相关系数r：-1到1，正正、负负、零无关

**临床联系**:  
- 病因研究：暴露与疾病的关联
- 指标关系：两个临床指标的相关性
- 诊断评估：两种检查方法的一致性

---

#### 18. 回归分析 (Regression Analysis/回归分析)

**白话解释**:  
"用一个变量预测另一个"——建立数学方程，用X预测Y。

**生活例子**:  
- 用身高预测体重
- 用血糖预测并发症风险
- 用年龄预测肺活量

**学术定义**:  
研究一个或多个自变量与因变量之间数量关系的统计分析方法，可用于预测和控制。

**简单线性回归**:
```
回归方程
Y = a + bX + ε

├── Y：因变量（被预测）
├── X：自变量（预测因子）
├── a：截距
├── b：回归系数（斜率）
└── ε：误差项

回归系数b的意义：
├── X每增加1个单位，Y平均变化b个单位
├── b > 0：正相关
└── b < 0：负相关
```

**回归与相关的区别**:
| 特征 | 相关 | 回归 |
|------|------|------|
| 目的 | 描述关系程度 | 建立预测方程 |
| 方向 | 对称（X↔Y） | 不对称（X→Y） |
| 应用 | 关联分析 | 预测分析 |
| 结果 | 相关系数r | 回归系数b |

**多元线性回归**:
```
多元回归方程
Y = b₀ + b₁X₁ + b₂X₂ + ... + bₚXₚ + ε

├── 可纳入多个自变量
├── 控制混杂因素
└── 分析各因素的独立作用
```

**费曼记忆**:  
回归 = 预测方程，Y = a + bX

**临床联系**:  
- 疾病风险预测模型
- 疗效预测
- 正常值建立
