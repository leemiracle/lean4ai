### 一、临床基础检验

#### 1. 血液常规检验 (Complete Blood Count/CBC/血液常规)

**白话解释**:  
抽一管血，数数红细胞、白细胞、血小板有多少，判断身体基本状况。

**生活例子**:  
- 发烧时验血看白细胞
- 体检时的血常规
- 贫血时看红细胞和血红蛋白

**学术定义**:  
对血液中红细胞、白细胞、血小板等有形成分进行计数和形态学分析的检验方法。

**关键要点**:
1. **检测项目**:
   | 项目 | 英文缩写 | 正常值（成人） | 临床意义 |
   |------|----------|---------------|----------|
   | 红细胞计数 | RBC | 男4.0-5.5×10¹²/L | 贫血、红细胞增多 |
   | 血红蛋白 | Hb | 男120-160g/L | 贫血诊断金标准 |
   | 白细胞计数 | WBC | 4-10×10⁹/L | 感染、白血病 |
   | 血小板计数 | PLT | 100-300×10⁹/L | 出血性疾病 |
   | 红细胞压积 | HCT | 男40-50% | 血液浓缩/稀释 |

2. **自动化仪器**:
   - 电阻抗法：细胞计数
   - 激光散射法：细胞分类
   - 荧光染色法：精确分类

3. **标本采集**:
   - 静脉血：EDTA-K2抗凝
   - 采血量：2mL
   - 时间：晨起空腹

4. **质量控制**:
   - 室内质控：每日质控
   - 室间质评：定期比对

**费曼记忆**:  
血常规 = 数"三系"（红细胞、白细胞、血小板）

**临床联系**:  
- 感染：WBC↑
- 贫血：RBC↓、Hb↓
- 白血病：WBC异常增高

---

#### 2. 尿液常规检验 (Urinalysis/尿液常规)

**白话解释**:  
验尿看有没有问题，能发现肾脏、泌尿系统的很多疾病。

**生活例子**:  
- 尿路感染时尿液浑浊
- 糖尿病时尿糖阳性
- 肾炎时尿中有蛋白

**学术定义**:  
对尿液的物理性状、化学成分和有形成分进行检查的常规检验方法。

**关键要点**:
1. **检查项目**:
   | 项目 | 正常值 | 异常意义 |
   |------|--------|----------|
   | 颜色 | 淡黄 | 血尿（红色）、黄疸（浓茶色） |
   | 透明度 | 清亮 | 浑浊（感染） |
   | 比重 | 1.015-1.025 | 肾浓缩功能 |
   | pH | 5.0-8.0 | 酸碱失衡 |
   | 蛋白 | 阴性 | 肾病 |
   | 葡萄糖 | 阴性 | 糖尿病 |
   | 红细胞 | 0-3/HP | 血尿 |
   | 白细胞 | 0-5/HP | 感染 |

2. **尿干化学检测**:
   - 试带法
   - 11项联检
   - 快速筛查

3. **尿沉渣镜检**:
   - 离心取沉渣
   - 显微镜检查
   - 发现细胞、管型、结晶

4. **标本要求**:
   - 清洁中段尿
   - 新鲜送检（2小时内）
   - 避免污染

**费曼记忆**:  
尿常规 = 看（颜色、透明度）+ 测（干化学）+ 查（显微镜）

**临床联系**:  
- 尿路感染：WBC↑、细菌+
- 肾病综合征：蛋白+++
- 糖尿病：尿糖+

---

#### 3. 粪便常规检验 (Stool Analysis/粪便常规)

**白话解释**:  
化验大便，看有没有出血、寄生虫等问题。

**生活例子**:  
- 消化道出血时大便发黑
- 寄生虫感染时大便有虫卵
- 肠炎时大便有白细胞

**学术定义**:  
对粪便的性状、有形成分及隐血进行检查的常规检验方法。

**关键要点**:
1. **检查内容**:
   | 项目 | 正常 | 异常意义 |
   |------|------|----------|
   | 颜色 | 黄褐色 | 黑便（上消化道出血） |
   | 性状 | 成形软便 | 稀便、粘液便 |
   | 红细胞 | 无 | 下消化道出血 |
   | 白细胞 | 偶见 | 肠炎 |
   | 寄生虫 | 无 | 寄生虫感染 |
   | 隐血 | 阴性 | 消化道出血 |

2. **隐血试验（FOBT）**:
   - 化学法：假阳性多
   - 免疫法：特异性高
   - 意义：筛查消化道肿瘤

3. **寄生虫检查**:
   - 直接涂片
   - 浓集法
   - 漂浮法

**费曼记忆**:  
粪便常规 = 看（性状）+ 查（显微镜）+ 验（隐血）

**临床联系**:  
- 消化道出血：隐血+
- 肠炎：WBC+
- 寄生虫病：虫卵+

### 二、生物化学检验

#### 4. 肝功能检验 (Liver Function Tests/肝功能)

**白话解释**:  
验血看肝脏"工作"好不好，肝脏不好时各项指标会异常。

**生活例子**:  
- 肝炎时转氨酶升高
- 饮酒后肝功能异常
- 体检时的肝功能检查

**学术定义**:  
通过检测血清中肝脏代谢相关酶类、蛋白质和胆红素等指标，评估肝脏功能状态的检验。

**关键要点**:
1. **主要指标**:
   | 项目 | 缩写 | 正常值 | 临床意义 |
   |------|------|--------|----------|
   | 谷丙转氨酶 | ALT | <40U/L | 肝细胞损伤 |
   | 谷草转氨酶 | AST | <40U/L | 肝/心肌损伤 |
   | 总胆红素 | TBIL | <21μmol/L | 黄疸 |
   | 直接胆红素 | DBIL | <8μmol/L | 梗阻性黄疸 |
   | 白蛋白 | ALB | 35-55g/L | 肝合成功能 |
   | 球蛋白 | GLB | 20-30g/L | 免疫/肝病 |

2. **肝功能评价**:
   - 合成功能：ALB、PT
   - 排泄功能：胆红素
   - 损伤指标：ALT、AST
   - 胆汁淤积：ALP、GGT

3. **临床解读**:
   - ALT>AST：肝炎
   - AST>ALT：肝硬化、酒精肝
   - ALB↓：慢性肝病

**费曼记忆**:  
肝功能 = "酶+蛋白+胆红素"

**临床联系**:  
- 急性肝炎：ALT↑↑
- 肝硬化：ALB↓、A/G倒置
- 梗阻性黄疸：DBIL↑↑

---

#### 5. 肾功能检验 (Renal Function Tests/肾功能)

**白话解释**:  
验血看肾脏"过滤"功能好不好。

**生活例子**:  
- 肾炎时尿素氮升高
- 尿毒症时肌酐很高
- 体检时的肾功能检查

**学术定义**:  
通过检测血清中代谢废物和肾脏相关指标，评估肾脏排泄和调节功能的检验。

**关键要点**:
1. **主要指标**:
   | 项目 | 缩写 | 正常值 | 临床意义 |
   |------|------|--------|----------|
   | 血尿素氮 | BUN | 2.9-8.2mmol/L | 肾小球滤过 |
   | 血肌酐 | Scr | 44-133μmol/L | 肾功能 |
   | 尿酸 | UA | 150-420μmol/L | 痛风 |
   | 胱抑素C | CysC | 0.51-1.09mg/L | 早期肾损伤 |
   | eGFR | - | >90mL/min | 肾小球滤过率 |

2. **肾功能分期**（CKD分期）:
   | 分期 | eGFR | 描述 |
   |------|------|------|
   | 1期 | ≥90 | 正常 |
   | 2期 | 60-89 | 轻度下降 |
   | 3期 | 30-59 | 中度下降 |
   | 4期 | 15-29 | 重度下降 |
   | 5期 | <15 | 肾衰竭 |

3. **肌酐清除率**:
   - 公式计算
   - 24小时尿测定
   - 反映GFR

**费曼记忆**:  
肾功能 = BUN + Scr（尿素氮+肌酐）

**临床联系**:  
- 急性肾衰：BUN↑↑、Scr↑
- 慢性肾病：分期管理
- 痛风：UA↑

---

#### 6. 血糖检验 (Blood Glucose Test/血糖)

**白话解释**:  
验血看糖分多少，判断有没有糖尿病。

**生活例子**:  
- 糖尿病患者每天测血糖
- 体检空腹血糖
- 怀疑低血糖时急诊测血糖

**学术定义**:  
检测血液中葡萄糖浓度，用于糖尿病诊断和监测的检验。

**关键要点**:
1. **血糖参考值**:
   | 项目 | 正常值 | 糖尿病诊断 |
   |------|--------|-----------|
   | 空腹血糖 | 3.9-6.1mmol/L | ≥7.0mmol/L |
   | 餐后2h血糖 | <7.8mmol/L | ≥11.1mmol/L |
   | 随机血糖 | - | ≥11.1mmol/L+症状 |
   | 糖化血红蛋白 | 4-6% | ≥6.5% |

2. **糖耐量试验（OGTT）**:
   - 空腹口服75g葡萄糖
   - 测0、30、60、120分钟血糖
   - 诊断糖尿病和糖耐量异常

3. **糖化血红蛋白**:
   - 反映近2-3月平均血糖
   - 糖尿病控制金标准
   - 不受短期血糖波动影响

4. **检测方法**:
   - 静脉血：生化仪检测
   - 指血：血糖仪快速检测
   - 连续监测：CGM

**费曼记忆**:  
血糖 = 空腹≥7 + 餐后≥11 = 糖尿病

**临床联系**:  
- 糖尿病诊断
- 血糖监测
- 低血糖诊断

### 三、免疫学检验

#### 7. 免疫功能检验 (Immune Function Tests/免疫功能)

**白话解释**:  
验血看免疫系统强不强，有没有免疫缺陷或自身免疫病。

**生活例子**:  
- 艾滋病检查CD4细胞
- 自身免疫病查自身抗体
- 免疫球蛋白检测

**学术定义**:  
检测免疫细胞、免疫分子等指标，评估机体免疫状态的检验。

**关键要点**:
1. **免疫球蛋白**:
   | 类型 | 正常值 | 临床意义 |
   |------|--------|----------|
   | IgG | 7-16g/L | 体液免疫主力 |
   | IgA | 0.7-4g/L | 黏膜免疫 |
   | IgM | 0.4-2.3g/L | 早期感染 |
   | IgE | <100IU/mL | 过敏 |

2. **淋巴细胞亚群**:
   - CD3⁺：总T细胞
   - CD4⁺：辅助T细胞
   - CD8⁺：杀伤T细胞
   - CD19⁺：B细胞
   - CD4/CD8比值：1.5-2.5

3. **补体检测**:
   - C3、C4
   - 降低：SLE、肾小球肾炎

4. **自身抗体**:
   - ANA：抗核抗体（自身免疫病筛查）
   - RF：类风湿因子
   - 抗dsDNA：SLE特异性

**费曼记忆**:  
免疫功能 = 免疫球蛋白 + 淋巴细胞 + 补体

**临床联系**:  
- 免疫缺陷病：Ig↓、CD4↓
- 自身免疫病：自身抗体+
- 过敏：IgE↑

#### 8. 肿瘤标志物检验 (Tumor Markers/肿瘤标志物)

**白话解释**:  
血液中的"报警信号"，升高可能提示肿瘤，但不一定就是癌症。

**生活例子**:  
- 体检时查CEA、AFP
- 肝癌筛查AFP
- 前列腺癌查PSA

**学术定义**:  
由肿瘤细胞产生或宿主对肿瘤反应产生的物质，可用于肿瘤筛查、诊断和监测。

**关键要点**:
1. **常见肿瘤标志物**:
   | 标志物 | 相关肿瘤 | 正常值 |
   |--------|----------|--------|
   | AFP | 肝癌 | <20ng/mL |
   | CEA | 结直肠癌 | <5ng/mL |
   | PSA | 前列腺癌 | <4ng/mL |
   | CA125 | 卵巢癌 | <35U/mL |
   | CA19-9 | 胰腺癌 | <37U/mL |
   | CA15-3 | 乳腺癌 | <25U/mL |

2. **临床应用**:
   - 筛查：高危人群
   - 诊断：辅助参考
   - 监测：疗效观察、复发监测

3. **注意事项**:
   - 升高≠癌症
   - 良性疾病也可升高
   - 需结合影像学、病理
   - 动态观察更可靠

**费曼记忆**:  
AFP=肝癌，PSA=前列腺，CEA=肠癌

**临床联系**:  
- 高危人群筛查
- 治疗效果监测
- 复发早期发现

### 四、X线技术

#### 9. X线成像原理 (X-ray Imaging/X线成像)

**白话解释**:  
用X射线"透视"人体，骨头显白、肺显黑，能看骨折和肺炎。

**生活例子**:  
- 骨折拍片
- 肺炎胸透
- 牙科拍片

**学术定义**:  
利用X射线穿透人体后，不同组织对X线吸收差异形成影像的成像技术。

**关键要点**:
1. **成像原理**:
   ```
   X射线源 → 穿透人体 → 不同组织吸收不同
        ↓
   形成对比 → 胶片/探测器成像
   ```

2. **影像特点（密度）**:
   | 组织 | 吸收程度 | 影像表现 | 例子 |
   |------|----------|----------|------|
   | 高密度 | 高 | 白色 | 骨、钙化 |
   | 中密度 | 中 | 灰色 | 软组织 |
   | 低密度 | 低 | 黑色 | 肺（含气） |
   | 脂肪 | 较低 | 灰黑色 | 皮下脂肪 |

3. **检查类型**:
   - 透视：实时动态
   - 摄片：静态图像
   - 造影：引入对比剂

4. **辐射防护**:
   - ALARA原则
   - 屏蔽防护
   - 距离防护

**费曼记忆**:  
X线 = 骨白肺黑软组织灰

**临床联系**:  
- 骨折诊断
- 肺炎筛查
- 消化道造影

---

#### 10. 数字X线摄影 (Digital Radiography/DR/数字X线)

**白话解释**:  
不用胶片的"数码X光"，图像可以存电脑、放大看。

**生活例子**:  
- 医院拍片后电脑上看
- 图像可以调节亮度对比度
- 远程会诊

**学术定义**:  
使用数字探测器替代传统胶片，将X线信号直接转换为数字图像的成像技术。

**关键要点**:
1. **技术类型**:
   - CR（计算机X线摄影）：IP板+扫描
   - DR（数字X线摄影）：平板探测器

2. **优势**:
   - 图像质量好
   - 可后处理（窗宽窗位）
   - 存储传输方便
   - 辐射剂量低

3. **后处理功能**:
   - 窗宽窗位调节
   - 放大、测量
   - 边缘增强
   - 黑白反转

**费曼记忆**:  
DR = "数码相机"版X光

**临床联系**:  
- 常规X线检查
- PACS系统
- 远程放射

### 五、CT技术

#### 11. 计算机断层扫描 (Computed Tomography/CT)

**白话解释**:  
X射线绕着身体转一圈，电脑把身体"切片"看，一层层看得更清楚。

**生活例子**:  
- 头部CT看脑出血
- 胸部CT看肺结节
- 腹部CT看肿瘤

**学术定义**:  
利用X射线束对人体某部位进行断面扫描，经计算机处理重建断面图像的成像技术。

**关键要点**:
1. **成像原理**:
   ```
   X线球管旋转 → 多角度扫描
        ↓
   探测器接收数据
        ↓
   计算机重建 → 断层图像
   ```

2. **CT值（HU）**:
   | 组织 | CT值 |
   |------|------|
   | 骨 | +1000 |
   | 软组织 | +40-80 |
   | 水 | 0 |
   | 脂肪 | -100 |
   | 空气 | -1000 |

3. **CT类型**:
   - 平扫：常规扫描
   - 增强扫描：注射造影剂
   - CTA：血管成像
   - 三维重建

4. **临床应用**:
   - 头颅：脑出血、肿瘤
   - 胸部：肺结节、肺炎
   - 腹部：肿瘤、结石

**费曼记忆**:  
CT = X线"切片"检查

**临床联系**:  
- 脑出血诊断
- 肺癌筛查
- 肿瘤分期

---

#### 12. 多层螺旋CT (Multi-slice CT/MSCT/多层螺旋CT)

**白话解释**:  
更快的CT，一次转圈能切很多层，检查速度快、图像更清晰。

**生活例子**:  
- 心脏CT检查冠状动脉
- 肺结节筛查
- 全身快速扫描

**学术定义**:  
具有多排探测器的螺旋CT，一次旋转可获取多层图像，扫描速度快、分辨率高。

**关键要点**:
1. **技术特点**:
   - 多排探测器（16、64、128、256层）
   - 螺旋扫描：连续采集
   - 扫描速度快：几秒完成

2. **优势**:
   - 时间分辨率高
   - 空间分辨率高
   - 后处理功能强大

3. **后处理技术**:
   - MPR：多平面重建
   - MIP：最大密度投影
   - VR：容积再现

### 六、MRI技术

#### 13. 磁共振成像 (Magnetic Resonance Imaging/MRI)

**白话解释**:  
用强磁场和无线电波让身体里的水分子"说话"，形成图像。

**生活例子**:  
- 脑部MRI看肿瘤
- 关节MRI看半月板
- 脊柱MRI看椎间盘

**学术定义**:  
利用原子核在强磁场中发生共振产生信号，经计算机重建形成图像的成像技术。

**关键要点**:
1. **成像原理**:
   ```
   强磁场 → 氢原子核排列
        ↓
   射频脉冲 → 共振
        ↓
   停止脉冲 → 释放信号
        ↓
   计算机重建 → 图像
   ```

2. **常用序列**:
   | 序列 | 特点 | 应用 |
   |------|------|------|
   | T1WI | 解剖结构清晰 | 脂肪亮、水暗 |
   | T2WI | 病变敏感 | 水亮、脂肪暗 |
   | FLAIR | 抑制水信号 | 脑部病变 |
   | DWI | 水分子弥散 | 早期脑梗 |

3. **MRI优势**:
   - 无电离辐射
   - 软组织对比度高
   - 多平面成像
   - 功能成像

4. **禁忌证**:
   - 心脏起搏器
   - 金属植入物
   - 幽闭恐惧症

**费曼记忆**:  
MRI = 磁场 + 水分子 = 无辐射软组织成像

**临床联系**:  
- 脑肿瘤诊断
- 椎间盘突出
- 关节损伤

### 十二、视力检查

#### 20. 视力检查 (Visual Acuity Test/视力检查)

**白话解释**:  
测眼睛能看多清楚，用视力表读"E"字。

**生活例子**:  
- 驾照体检测视力
- 学生体检看视力表
- 配眼镜前查视力

**学术定义**:  
检测眼睛分辨物体细节能力的检查，是视功能检查的基本项目。

**关键要点**:
1. **视力表**:
   - 国际标准视力表：1.0为正常
   - 对数视力表：5.0为正常
   - Snellen视力表：20/20

2. **检查方法**:
   - 远视力：5米距离
   - 近视力：30厘米
   - 单眼检查

3. **记录方法**:
   | 视力 | 记录 |
   |------|------|
   | 1.0及以上 | 正常 |
   | 0.1-0.9 | 轻度下降 |
   | <0.1 | 低视力 |
   | 指数/手动/光感 | 重度障碍 |

4. **矫正视力**:
   - 裸眼视力
   - 矫正视力（戴镜）

**费曼记忆**:  
视力1.0 = 正常

**临床联系**:  
- 屈光不正诊断
- 眼病筛查
- 驾驶员体检

#### 21. 验光技术 (Refraction/验光技术)

**白话解释**:  
测眼睛度数，配多少度的眼镜合适。

**生活例子**:  
- 配眼镜前验光
- 儿童散瞳验光
- 隐形眼镜验配

**学术定义**:  
检测眼睛屈光状态并确定矫正镜片度数的技术。

**关键要点**:
1. **屈光不正类型**:
   | 类型 | 成因 | 矫正 |
   |------|------|------|
   | 近视 | 眼轴过长 | 凹透镜（-） |
   | 远视 | 眼轴过短 | 凸透镜（+） |
   | 散光 | 角膜不规则 | 柱镜 |

2. **验光方法**:
   - 电脑验光：初步筛查
   - 检影验光：客观准确
   - 主觉验光：综合验光仪

3. **验光流程**:
   ```
   问诊 → 裸眼视力 → 电脑验光

   检影验光 → 主觉验光 → 试戴

   ```

4. **处方内容**:
   - 球镜（近视/远视度数）
   - 柱镜（散光度数）
   - 轴位（散光方向）
   - 瞳距

**费曼记忆**:  
验光 = 近视"-"、远视"+"、散光有轴位

**临床联系**:

### 十三、听力检查

#### 22. 纯音测听 (Pure Tone Audiometry/纯音测听)

**白话解释**:  
戴耳机听各种频率的声音，看能听到多小的声音。

**生活例子**:  
- 职业病体检查听力
- 耳鸣患者查听力
- 助听器验配前检查

**学术定义**:  
通过给予不同频率的纯音信号，测定各频率听阈的听力学检查方法。

**关键要点**:
1. **测试频率**:
   - 125、250、500、1000、2000、4000、8000Hz
   - 覆盖语音频率范围

2. **听力图**:
   | 听力损失程度 | 听阈（dB HL） |
   |--------------|---------------|
   | 正常 | ≤25 |
   | 轻度 | 26-40 |
   | 中度 | 41-55 |
   | 中重度 | 56-70 |
   | 重度 | 71-90 |
   | 极重度 | >90 |

3. **传导性vs感音神经性**:
   - 气导：耳道→中耳→内耳
   - 骨导：颅骨→内耳
   - 气骨导差>10dB：传导性聋

4. **临床意义**:
   - 诊断听力损失类型
   - 指导助听器验配
   - 职业性听力损害评估

**费曼记忆**:  
纯音测听 = 画听力图（正常<25dB）

**临床联系**:  
- 听力损失诊断
- 助听器验配
- 职业病鉴定

#### 23. 声导抗检查 (Tympanometry/声导抗检查)

**白话解释**:  
测中耳"鼓起来"的能力，看有没有中耳炎、积液。

**生活例子**:

通过改变外耳道压力，测量中耳声导抗变化，评估中耳功能的检查方法。

**关键要点**:
1. **鼓室图类型**:
   | 类型 | 峰值位置 | 意义 |
   |------|----------|------|
   | A型 | 0daPa | 正常 |
   | As型 | 0daPa，峰低 | 耳硬化症 |
   | Ad型 | 0daPa，峰高 | 听骨链中断 |
   | B型 | 无峰 | 中耳积液 |
   | C型 | 负压 | 咽鼓管功能障碍 |

2. **检查内容**:
   - 鼓室图
   - 静态声导抗

3. **临床意义**:
   - 中耳病变诊断
   - 咽鼓管功能评估
   - 面神经功能评估

**费曼记忆**:  
A型=正常，B型=积液，C型=负压

**临床联系**:  
- 中耳炎诊断
- 儿童听力筛查
- 咽鼓管功能评估

#### 24. 听性脑干反应 (Auditory Brainstem Response/ABR/听性脑干反应)

**白话解释**:  
给声音后记录脑子的电反应，测听力不需要患者配合。

**生活例子**:

给予短声刺激后，记录从耳蜗到脑干听觉通路电反应的客观听力检查方法。

**关键要点**:
1. **波形识别**:
   | 波形 | 来源 |

   | II波 | 耳蜗核 |
   | III波 | 上橄榄核 |
   | IV波 | 外侧丘系 |
   | V波 | 下丘（最稳定） |

2. **临床应用**:
   - 高危新生儿听力筛查

   - 听神经瘤诊断
   - 脑死亡判定

3. **特点**:
   - 客观检查
   - 不需配合
   - 可睡眠中检查

**费曼记忆**:  
ABR = 客观听力检查，V波最重要

**临床联系**:  
- 新生儿听力筛查
- 听神经瘤诊断
- 功能性聋鉴别

## 🎯 重点概念速记

### 必背概念（25个）

1. **血液常规**: RBC、WBC、PLT三系检查
2. **尿液常规**: 物理性状+干化学+沉渣镜检
3. **粪便常规**: 性状+镜检+隐血
4. **肝功能**: ALT、AST、胆红素、白蛋白
5. **肾功能**: BUN、Scr、eGFR
6. **血糖检验**: 空腹血糖≥7mmol/L诊断糖尿病
7. **免疫功能**: Ig、淋巴细胞亚群、补体
8. **肿瘤标志物**: AFP、CEA、PSA、CA系列
9. **X线成像**: 骨白肺黑软组织灰
10. **数字X线(DR)**: 数字化X线摄影
11. **CT**: X线断层扫描
12. **多层螺旋CT**: 快速多层扫描
13. **MRI**: 磁共振成像，无辐射
14. **超声成像**: 医用声纳
15. **核医学成像**: 放射性核素显像
16. **物理治疗(PT)**: 运动+手法+理疗
17. **运动疗法**: 科学的运动处方
18. **作业治疗(OT)**: ADL训练
19. **言语治疗(ST)**: 语言+吞咽康复
20. **视力检查**: 1.0为正常
21. **验光技术**: 测定屈光度数
22. **纯音测听**: <25dB为正常
23. **声导抗检查**: 中耳功能评估
24. **ABR**: 客观听力检查
25. **助听器**: 声音放大设备

## 💡 学习建议

### 记忆技巧
1. **理解原理**: 知道"为什么"才能记住
2. **联系临床**: 结合疾病理解检查意义
3. **归纳总结**: 正常值要重点记忆
4. **实践操作**: 多动手，熟悉仪器设备
5. **病例学习**: 通过典型病例理解

### 学习顺序建议
```
医学检验 → 理解疾病诊断依据
    ↓
医学影像 → 看懂各种片子
    ↓
康复治疗 → 理解功能恢复
    ↓
眼视光/听力 → 专科技术
```
