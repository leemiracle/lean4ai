### 一、病史采集

#### 1. 病史采集 (History Taking/病史采集)

**白话解释**:  
"问诊"——医生通过问问题了解病情的过程，是诊断的第一步。

**生活例子**:  
- 医生问："哪里不舒服？多久了？"
- 患者答："头痛3天了，还有点发烧。"
- 医生追问："头痛是持续的还是阵发的？"

**学术定义**:  
医生通过与患者或知情者交谈，了解疾病发生、发展、诊治经过及既往健康状况的的过程，是诊断疾病的基本方法。

**病史的内容（主诉+现病史+既往史等）**:
```
1. 一般项目：姓名、性别、年龄、职业等
2. 主诉：患者就诊最主要症状+持续时间
3. 现病史：本次疾病的发生、发展、诊治经过
   - 起病情况
   - 主要症状特点
   - 病情演变
   - 伴随症状
   - 诊治经过
   - 一般情况
4. 既往史：过去疾病、手术、外伤、过敏史
5. 个人史：生活习惯、职业、疫区接触史
6. 婚育史：婚姻、生育情况
7. 家族史：家族遗传病史
```

**问诊技巧**:
1. **开放式提问**: "您哪里不舒服？"
2. **封闭式提问**: "头痛是持续的吗？"
3. **追问**: "还有其他症状吗？"
4. **核实**: "您是说头痛3天了，对吗？"

**费曼记忆**:  
病史 = 主诉（一句话概括）+ 现病史（故事主线）+ 其他背景

**临床意义**:  
70%的诊断信息来自病史采集

---

#### 2. 主诉 (Chief Complaint/主诉)

**白话解释**:  
患者来看病的"最主要原因"，用一句话说清楚。

**生活例子**:  
- "头痛3天"
- "胸痛2小时"
- "发现血压高5年"

**学术定义**:  
患者就诊时最主要的症状或体征及其持续时间，是现病史的核心和引导诊断的关键。

**主诉的要求**:
1. **简明扼要**: 通常不超过20个字
2. **包含要素**: 症状/体征 + 持续时间
3. **按时间排序**: 多个症状时按时间先后
4. **避免诊断术语**: 用症状描述，不用诊断名词

**好的主诉示例**:
- ✅ "胸骨后疼痛2小时"
- ✅ "反复咳嗽、咳痰10年，加重1周"
- ✅ "发现空腹血糖升高3个月"

**不好的主诉示例**:
- ❌ "患者因感冒来就诊"（含诊断术语）
- ❌ "胸闷、气短、咳嗽、发热、头痛5天"（症状太多）
- ❌ "不舒服1个月"（症状不明确）

**费曼记忆**:  
主诉 = 症状 + 时间（一句话说清就诊原因）

### 二、体格检查

#### 3. 体格检查 (Physical Examination/体格检查)

**白话解释**:  
"查体"——医生用眼、手、耳、工具检查身体，发现异常。

**生活例子**:  
- 用听诊器听心肺
- 用手按压腹部
- 用血压计量血压
- 用手电筒看咽喉

**学术定义**:  
医生运用感官（眼、耳、鼻、手）或借助简单工具（听诊器、叩诊锤等）检查患者身体状况的方法。

**基本方法（视触叩听）**:
| 方法 | 内容 | 例子 |
|------|------|------|
| 视诊 | 观察外貌、皮肤、体位等 | 黄疸、皮疹 |
| 触诊 | 用手触摸、按压 | 肝脾肿大、压痛 |
| 叩诊 | 叩击身体，听声音 | 肺部浊音、腹水 |
| 听诊 | 听体内声音 | 心音、呼吸音、肠鸣音 |

**体格检查顺序**:
```
一般检查 → 生命体征（T、P、R、BP）
         → 头颈部
         → 胸部（胸廓、肺、心）
         → 腹部
         → 脊柱四肢
         → 神经系统
```

**生命体征**:
1. **体温（T）**: 正常36-37℃
2. **脉搏（P）**: 正常60-100次/分
3. **呼吸（R）**: 正常16-20次/分
4. **血压（BP）**: 正常收缩压<140mmHg，舒张压<90mmHg

**费曼记忆**:  
查体 = 看（视诊）+ 摸（触诊）+ 敲（叩诊）+ 听（听诊）

#### 4. 心脏听诊 (Cardiac Auscultation/心脏听诊)

**白话解释**:  
用听诊器听心脏跳动的声音，判断心脏是否正常。

**生活例子**:  
- "咚哒、咚哒"：正常心音（第一心音+第二心音）
- "吹风样杂音"：可能有瓣膜问题
- 心律不齐：心跳快慢不均

**学术定义**:  
使用听诊器在心脏瓣膜听诊区听取心音、额外心音、心脏杂音、心包摩擦音等的方法。

**心脏瓣膜听诊区**:
| 听诊区 | 位置 | 听诊瓣膜 |
|--------|------|----------|
| 二尖瓣区 | 心尖部（左锁骨中线第5肋间） | 二尖瓣 |
| 肺动脉瓣区 | 胸骨左缘第2肋间 | 肺动脉瓣 |
| 主动脉瓣区 | 胸骨右缘第2肋间 | 主动脉瓣 |
| 主动脉瓣第二听诊区 | 胸骨左缘第3-4肋间 | 主动脉瓣 |
| 三尖瓣区 | 胸骨左缘第4-5肋间 | 三尖瓣 |

**心音**:
- **第一心音（S1）**: 心室收缩开始，房室瓣关闭
  - 特点：低、长、"咚"
- **第二心音（S2）**: 心室舒张开始，半月瓣关闭
  - 特点：高、短、"哒"
- **第三心音（S3）**: 舒张早期，血流冲击心室壁
  - 意义：心功能不全
- **第四心音（S4）**: 舒张晚期，心房收缩
  - 意义：心室顺应性下降

**心脏杂音分级**:
| 分级 | 强度 | 特点 |
|------|------|------|
| I级 | 最轻 | 仔细听才能听到 |
| II级 | 轻度 | 容易听到 |
| III级 | 中度 | 明显但无震颤 |
| IV级 | 响亮 | 伴震颤 |
| V级 | 很响 | 听诊器稍离胸壁可听到 |
| VI级 | 最响 | 听诊器离开胸壁可听到 |

**费曼记忆**:  
心音 = "咚哒"（收缩-舒张）+ 杂音（异常血流声音）

**临床联系**:  
- 收缩期杂音：二尖瓣关闭不全、主动脉瓣狭窄
- 舒张期杂音：二尖瓣狭窄、主动脉瓣关闭不全

### 三、辅助检查

#### 5. 实验室检查 (Laboratory Tests/实验室检查)

**白话解释**:  
"化验"——通过检测血液、尿液、粪便等样本，了解身体状况。

**生活例子**:  
- 抽血查血常规：看白细胞、红细胞、血小板
- 查血糖：看有没有糖尿病
- 查肝功能：看肝脏是否正常

**学术定义**:  
通过物理、化学、生物学等方法，对患者的血液、体液、分泌物、排泄物等标本进行检测，为诊断和治疗提供依据。

**常用检查项目**:

**1. 血常规**:
| 项目 | 正常值 | 临床意义 |
|------|--------|----------|
| 红细胞（RBC） | 男4.0-5.5×10¹²/L | 贫血、红细胞增多症 |
| 血红蛋白（Hb） | 男120-160g/L | 贫血 |
| 白细胞（WBC） | 4-10×10⁹/L | 感染、白血病 |
| 血小板（PLT） | 100-300×10⁹/L | 出血性疾病 |
| 中性粒细胞% | 50-70% | 细菌感染 |
| 淋巴细胞% | 20-40% | 病毒感染 |

**2. 生化检查**:
| 项目 | 正常值 | 临床意义 |
|------|--------|----------|
| 空腹血糖（FPG） | 3.9-6.1mmol/L | 糖尿病 |
| 尿素氮（BUN） | 3.2-7.1mmol/L | 肾功能 |
| 肌酐（Cr） | 44-133μmol/L | 肾功能 |
| ALT | 0-40U/L | 肝功能 |
| AST | 0-40U/L | 肝功能/心肌损伤 |
| 总胆固醇（TC） | <5.18mmol/L | 血脂异常 |
| 甘油三酯（TG） | <1.70mmol/L | 血脂异常 |

**3. 凝血功能**:
| 项目 | 正常值 | 临床意义 |
|------|--------|----------|
| PT（凝血酶原时间） | 11-13秒 | 外源性凝血功能 |
| APTT | 25-37秒 | 内源性凝血功能 |
| INR | 0.8-1.2 | 华法林抗凝监测 |

**费曼记忆**:  
化验 = 看数字是否在正常范围内

---

#### 6. 心电图 (Electrocardiogram, ECG/心电图)

**白话解释**:  
记录心脏电活动的"波浪图"，判断心脏有没有问题。

**生活例子**:  
- 正常心电图：规律的P-QRS-T波
- 心肌梗死：ST段抬高
- 心律失常：波形不规则

**学术定义**:  
利用心电图机从体表记录心脏每一心动周期所产生电活动变化的曲线图形。

**心电图的基本波形**:
```
P波：心房除极
  ↓
PR间期：房室传导时间
  ↓
QRS波群：心室除极
  ↓
ST段：心室复极早期
  ↓
T波：心室复极
  ↓
QT间期：心室除极+复极总时间
```

**正常值**:
- 心率：60-100次/分
- PR间期：0.12-0.20秒
- QRS时限：0.06-0.10秒
- QT间期：0.32-0.44秒

**常见异常心电图**:
| 诊断 | 心电图特点 |
|------|------------|
| 窦性心动过速 | 心率>100次/分，P波正常 |
| 房颤 | P波消失，f波出现，R-R绝对不规则 |
| 室早 | 提前出现宽大畸形QRS波 |
| 心肌梗死 | ST段抬高或压低，病理性Q波 |
| 房室传导阻滞 | PR间期延长或P波后QRS脱落 |

**费曼记忆**:  
心电图 = P-QRS-T（心房→心室的电活动顺序）

**临床联系**:  
- 胸痛患者必查心电图
- 心律失常诊断的金标准

### 四、心血管系统疾病

#### 7. 高血压 (Hypertension/高血压)

**白话解释**:  
血压长期过高，像水管压力太大，容易把血管"撑坏"。

**生活例子**:  
- 很多中老年人"血压高"
- 吃降压药控制血压
- 高血压如果不控制，可能引起中风

**学术定义**:  
以体循环动脉血压持续升高为主要特征的临床综合征，可伴有心、脑、肾等器官功能损害。

**诊断标准**:
| 分类 | 收缩压（mmHg） | 舒张压（mmHg） |
|------|----------------|----------------|
| 正常血压 | <120 | <80 |
| 正常高值 | 120-139 | 80-89 |
| 高血压 | ≥140 | ≥90 |
| 1级高血压 | 140-159 | 90-99 |
| 2级高血压 | 160-179 | 100-109 |
| 3级高血压 | ≥180 | ≥110 |

**病因**:
1. **原发性高血压**（95%）: 病因不明
   - 遗传因素
   - 高盐饮食
   - 肥胖、缺乏运动
   - 精神紧张

2. **继发性高血压**（5%）: 有明确病因
   - 肾性高血压
   - 内分泌性高血压
   - 血管性高血压

**危害（靶器官损害）**:
```
心脏 → 左心室肥厚 → 心力衰竭
血管 → 动脉硬化 → 冠心病
脑   → 脑卒中 → 偏瘫
肾脏 → 肾小球硬化 → 肾功能衰竭
眼底 → 视网膜病变
```

**治疗**:
1. **非药物治疗**:
   - 低盐饮食（<6g/天）
   - 控制体重
   - 规律运动
   - 戒烟限酒

2. **药物治疗**:
   | 药物类别 | 代表药物 | 适应症 |
   |----------|----------|--------|
   | 钙通道阻滞剂（CCB） | 氨氯地平 | 老年高血压 |
   | ACEI | 卡托普利 | 伴糖尿病、心衰 |
   | ARB | 缬沙坦 | 不耐受ACEI者 |
   | β受体阻滞剂 | 美托洛尔 | 伴冠心病 |
   | 利尿剂 | 氢氯噻嗪 | 老年高血压 |

**费曼记忆**:  
高血压 = 隐形杀手（长期损害心脑肾）

**目标血压**: <140/90mmHg（一般人群），<130/80mmHg（糖尿病、肾病患者）

#### 8. 冠心病 (Coronary Heart Disease/冠心病)

**白话解释**:  
给心脏供血的"管道"（冠状动脉）堵了，心肌缺血缺氧。

**生活例子**:  
- 活动后胸痛：心绞痛
- 剧烈胸痛不缓解：心肌梗死
- 放支架：打通堵塞的血管

**学术定义**:  
冠状动脉粥样硬化使血管腔狭窄或阻塞，或（和）因冠状动脉痉挛导致心肌缺血缺氧或坏死的心脏病。

**冠心病的分类**:
1. **无症状性心肌缺血**
2. **心绞痛**
3. **心肌梗死**
4. **缺血性心肌病**
5. **猝死**

**心绞痛（稳定性心绞痛）**:
- **诱因**: 劳累、情绪激动、饱餐
- **部位**: 胸骨后
- **性质**: 压榨感、紧缩感
- **持续时间**: 3-5分钟
- **缓解方式**: 休息、含服硝酸甘油

**心肌梗死（急性心肌梗死）**:
- **症状**: 持续性剧烈胸痛>30分钟，休息和硝酸甘油不缓解
- **伴随症状**: 大汗、恶心、濒死感
- **心电图**: ST段抬高（STEMI）或压低（NSTEMI）
- **心肌酶**: 肌钙蛋白、CK-MB升高
- **治疗**: 
  - STEMI：急诊PCI（支架植入）或溶栓
  - NSTEMI：抗凝、抗血小板、必要时PCI

**冠心病的治疗**:
1. **药物治疗（ABCDE方案）**:
   - A：阿司匹林（抗血小板）
   - B：β受体阻滞剂（减慢心率）
   - C：降胆固醇（他汀类）
   - D：控制糖尿病
   - E：健康教育

2. **介入治疗**: 
   - PCI（经皮冠状动脉介入）
   - 支架植入

3. **外科治疗**: 
   - CABG（冠状动脉旁路移植术，"搭桥"）

**费曼记忆**:  
冠心病 = 冠状动脉堵了 → 胸痛 → 支架/搭桥

**危险因素**: 高血压、高血脂、糖尿病、吸烟、肥胖、家族史

### 五、呼吸系统疾病

#### 9. 肺炎 (Pneumonia/肺炎)

**白话解释**:  
肺部感染发炎，像肺泡里"进了脏东西"，引起发热、咳嗽、咳痰。

**生活例子**:  
- 感冒后持续高热、咳嗽：可能是肺炎
- 老人肺炎可能不发烧，只表现为精神差
- 儿童肺炎：发热、咳嗽、呼吸急促

**学术定义**:  
由细菌、病毒、真菌、寄生虫等病原体引起的肺部感染性疾病，表现为肺泡和肺间质的炎症。

**肺炎的分类**:
1. **按解剖分类**:
   - 大叶性肺炎：整叶肺实变
   - 小叶性肺炎：沿支气管分布
   - 间质性肺炎：肺间质炎症

2. **按病因分类**:
   - 细菌性肺炎（最常见）：肺炎链球菌
   - 病毒性肺炎：流感病毒、新冠
   - 非典型病原体：支原体、衣原体

3. **按患病环境分类**:
   - 社区获得性肺炎（CAP）
   - 医院获得性肺炎（HAP）

**临床表现**:
- **症状**: 发热、咳嗽、咳痰、胸痛、呼吸困难
- **体征**: 肺部啰音、叩诊浊音
- **影像学**: 胸片或CT可见片状、斑片状阴影

**诊断标准（CAP）**:
1. 新出现的咳嗽、咳痰或原有症状加重
2. 发热
3. 肺实变体征和（或）闻及湿啰音
4. WBC>10×10⁹/L或<4×10⁹/L
5. 胸部影像学新出现片状、斑片状阴影

（1+2+3+4中任一项 + 5）可诊断

**治疗**:
- **经验性抗生素**: 根据病原体推测选择
  - 社区获得性：青霉素类、头孢类、大环内酯类
  - 医院获得性：广谱抗生素

- **对症治疗**: 
  - 退热
  - 止咳祛痰
  - 氧疗（低氧血症时）

**费曼记忆**:  
肺炎 = 发热 + 咳嗽咳痰 + 肺部阴影 → 抗生素治疗

#### 10. 慢性阻塞性肺疾病 (COPD/慢阻肺)

**白话解释**:  
长期咳嗽、咳痰、气短，肺功能逐渐下降，多见于老烟民。

**生活例子**:  
- "老慢支"：长期咳嗽咳痰
- 爬楼梯气喘：肺功能下降
- 冬天容易发作：受凉后加重

**学术定义**:  
以持续气流受限为特征的慢性呼吸系统疾病，气流受限不完全可逆，呈进行性发展。

**病因**:
1. **吸烟**（最重要）: 80-90%患者有吸烟史
2. **职业粉尘和化学物质**
3. **空气污染**
4. **遗传因素**: α1-抗胰蛋白酶缺乏

**临床表现**:
| 症状 | 特点 |
|------|------|
| 慢性咳嗽 | 晨间咳嗽明显 |
| 咳痰 | 白色黏液或浆液泡沫痰 |
| 气短 | 进行性加重，活动后明显 |
| 喘息 | 部分患者有 |

**诊断（肺功能检查）**:
```
吸入支气管扩张剂后：
FEV₁/FVC < 70%

严重程度分级：
GOLD 1级（轻度）：FEV₁ ≥ 80%预计值
GOLD 2级（中度）：50% ≤ FEV₁ < 80%
GOLD 3级（重度）：30% ≤ FEV₁ < 50%
GOLD 4级（极重度）：FEV₁ < 30%
```

**治疗**:
1. **稳定期**:
   - 戒烟（最重要）
   - 支气管扩张剂（LAMA、LABA）
   - 吸入糖皮质激素（重症）
   - 肺康复训练

2. **急性加重期（AECOPD）**:
   - 控制性氧疗
   - 抗生素（有感染时）
   - 支气管扩张剂
   - 糖皮质激素

**并发症**: 
- 慢性呼吸衰竭
- 自发性气胸
- 慢性肺源性心脏病

**费曼记忆**:  
COPD = 吸烟 + 长期咳嗽咳痰气短 + 不可逆气流受限

### 六、消化系统疾病

#### 11. 胃溃疡 (Gastric Ulcer/胃溃疡)

**白话解释**:  
胃黏膜被"自己消化"形成溃疡，像胃壁上有个小伤口。

**生活例子**:  
- 饭后胃痛：胃溃疡特点
- 饥饿时胃痛：十二指肠溃疡特点
- 黑便：溃疡出血

**学术定义**:  
发生于胃的消化性溃疡，是胃酸和胃蛋白酶对胃黏膜的自身消化作用导致的溃疡性病变。

**胃溃疡 vs 十二指肠溃疡**:
| 特点 | 胃溃疡 | 十二指肠溃疡 |
|------|--------|--------------|
| 疼痛规律 | 餐后痛（进食-疼痛-缓解） | 空腹痛（疼痛-进食-缓解） |
| 疼痛部位 | 剑突下偏左 | 剑突下偏右 |
| 年龄 | 中老年多见 | 青壮年多见 |
| 癌变风险 | 可癌变（<1%） | 几乎不癌变 |

**病因**:
1. **幽门螺杆菌（Hp）感染**: 70-80%
2. **非甾体抗炎药（NSAIDs）**: 阿司匹林、布洛芬等
3. **胃酸分泌过多**
4. **其他**: 吸烟、饮酒、应激

**诊断**:
- **胃镜**: 金标准，可直接观察溃疡
- **Hp检测**: 呼气试验、快速尿素酶试验
- **上消化道造影**: 钡餐检查

**治疗**:
1. **根除幽门螺杆菌（四联疗法）**:
   ```
   PPI（质子泵抑制剂）+ 铋剂 + 两种抗生素
   疗程：10-14天
   ```

2. **抑酸治疗**:
   - PPI（奥美拉唑、雷贝拉唑）
   - H₂受体拮抗剂（雷尼替丁）

3. **黏膜保护**:
   - 硫糖铝、铋剂

**费曼记忆**:  
胃溃疡 = 胃痛 + 胃镜发现溃疡 + 抑酸+抗Hp治疗

**警示征象（需警惕胃癌）**:
- 年龄>45岁
- 体重下降
- 黑便
- 贫血
- 溃疡形态不规则

#### 12. 肝硬化 (Cirrhosis/肝硬化)

**白话解释**:  
肝脏反复损伤后形成疤痕，变硬、变形，功能逐渐丧失。

**生活例子**:  
- 长期饮酒：酒精性肝硬化
- 乙肝不治疗：乙肝后肝硬化
- 肝硬化晚期：腹水、呕血

**学术定义**:  
一种或多种病因长期作用于肝脏，导致肝细胞弥漫性变性、坏死、再生，继而引起纤维组织增生和肝细胞结节状再生，最终发展为肝小叶结构破坏和假小叶形成的慢性肝病。

**病因**:
1. **病毒性肝炎**: 乙肝最常见
2. **酒精**: 长期大量饮酒
3. **胆汁淤积**: 原发性胆汁性肝硬化
4. **药物或毒物**
5. **遗传代谢性疾病**: 血色病、Wilson病
6. **自身免疫性肝病**

**临床表现（代偿期→失代偿期）**:

**代偿期**: 症状轻微
- 乏力、食欲减退
- 肝脾轻度肿大

**失代偿期**: 出现并发症
| 症状 | 机制 |
|------|------|
| 腹水 | 门静脉高压、低蛋白血症 |
| 呕血/黑便 | 食管胃底静脉曲张破裂 |
| 肝性脑病 | 血氨升高 |
| 黄疸 | 肝功能损害 |
| 蜘蛛痣、肝掌 | 雌激素灭活障碍 |

**诊断**:
- **肝功能**: ALT、AST升高，白蛋白降低
- **影像学**: B超、CT、MRI显示肝脏形态改变
- **肝纤维化指标**: FIB-4、APRI
- **胃镜**: 发现食管胃底静脉曲张
- **肝穿刺活检**: 金标准

**Child-Pugh分级（评估预后）**:
| 指标 | 1分 | 2分 | 3分 |
|------|-----|-----|-----|
| 胆红素(μmol/L) | <34 | 34-51 | >51 |
| 白蛋白(g/L) | >35 | 28-35 | <28 |
| 腹水 | 无 | 少量 | 中大量 |
| 肝性脑病 | 无 | 1-2级 | 3-4级 |
| PT延长(秒) | <4 | 4-6 | >6 |

- A级（5-6分）：代偿期
- B级（7-9分）：失代偿期
- C级（10-15分）：预后差

**治疗**:
1. **病因治疗**: 抗病毒、戒酒
2. **保肝治疗**: 水飞蓟素、甘草酸制剂
3. **并发症处理**:
   - 腹水：利尿剂、放腹水
   - 上消化道出血：止血、内镜治疗
   - 肝性脑病：降血氨
4. **肝移植**: 终末期

**费曼记忆**:  
肝硬化 = 肝变硬 + 门静脉高压 + 肝功能减退

### 七、内分泌系统疾病

#### 13. 糖尿病 (Diabetes Mellitus/糖尿病)

**白话解释**:  
血糖长期过高，像血液里"糖太多"，损害全身器官。

**生活例子**:  
- "三多一少"：多饮、多尿、多食、体重下降
- 很多中老年人"血糖高"
- 打胰岛素控制血糖

**学术定义**:  
以高血糖为特征的代谢性疾病，由胰岛素分泌缺陷和（或）胰岛素作用缺陷引起。

**糖尿病的分型**:
| 分型 | 特点 |
|------|------|
| 1型糖尿病 | 胰岛素绝对缺乏，需胰岛素治疗 |
| 2型糖尿病 | 胰岛素抵抗为主，最常见（90%以上） |
| 妊娠期糖尿病 | 妊娠期间发现 |
| 特殊类型糖尿病 | 继发于其他疾病 |

**诊断标准**:
| 指标 | 正常 | 糖尿病前期 | 糖尿病 |
|------|------|------------|--------|
| 空腹血糖(mmol/L) | <6.1 | 6.1-6.9 | ≥7.0 |
| 餐后2h血糖(mmol/L) | <7.8 | 7.8-11.0 | ≥11.1 |
| HbA1c(%) | <5.7 | 5.7-6.4 | ≥6.5 |

**临床表现**:
- **典型症状**: 三多一少（多饮、多尿、多食、体重下降）
- **不典型症状**: 乏力、皮肤瘙痒、视物模糊
- **无症状**: 很多2型糖尿病早期无症状

**并发症**:

**1. 急性并发症**:
- 糖尿病酮症酸中毒（DKA）
- 高渗性高血糖状态（HHS）

**2. 慢性并发症**:
| 类型 | 表现 |
|------|------|
| 大血管病变 | 冠心病、脑卒中 |
| 微血管病变 | 糖尿病肾病、视网膜病变 |
| 神经病变 | 周围神经病变、自主神经病变 |
| 糖尿病足 | 足部溃疡、坏疽 |

**治疗（五驾马车）**:
1. **健康教育**
2. **医学营养治疗**:
   - 控制总热量
   - 合理分配碳水化合物、蛋白质、脂肪
   - 低GI饮食

3. **运动治疗**:
   - 每周150分钟中等强度运动

4. **药物治疗**:
   | 药物类别 | 代表药物 | 机制 |
   |----------|----------|------|
   | 双胍类 | 二甲双胍 | 改善胰岛素抵抗 |
   | 磺脲类 | 格列美脲 | 促进胰岛素分泌 |
   | SGLT-2抑制剂 | 达格列净 | 促进尿糖排泄 |
   | GLP-1受体激动剂 | 利拉鲁肽 | 促进胰岛素分泌 |
   | 胰岛素 | 各类胰岛素 | 补充胰岛素 |

5. **血糖监测**

**控制目标**:
- HbA1c：<7.0%（一般人群）
- 空腹血糖：4.4-7.0mmol/L
- 餐后2h血糖：<10.0mmol/L

**费曼记忆**:  
糖尿病 = 血糖高 + 三多一少 + 并发症（心肾眼神经足）

#### 14. 甲状腺功能亢进症 (Hyperthyroidism/甲亢)

**白话解释**:  
甲状腺激素太多，新陈代谢过快，像汽车"油门踩到底"。

**生活例子**:  
- 心慌、手抖、消瘦
- 脾气急躁、多汗
- 眼睛突出（部分患者）

**学术定义**:  
多种原因导致甲状腺腺体本身产生甲状腺激素过多，引起以神经、循环、消化等系统兴奋性增高和代谢亢进为主要表现的临床综合征。

**病因分类**:
1. **Graves病**（最常见，80%以上）: 自身免疫性疾病
2. **毒性多结节性甲状腺肿**
3. **甲状腺自主高功能腺瘤**
4. **甲状腺炎**
5. **碘甲亢**

**临床表现**:
| 系统 | 症状 |
|------|------|
| 全身 | 怕热、多汗、乏力、体重下降 |
| 心血管 | 心悸、心动过速、心律失常 |
| 消化 | 食欲亢进、大便次数增多 |
| 神经 | 烦躁、易激动、手抖 |
| 生殖 | 月经减少、闭经 |
| 眼部 | 突眼、眼睑挛缩 |

**诊断**:
- **甲状腺功能**: TSH↓，FT₃、FT₄↑
- **TRAb**: Graves病阳性
- **甲状腺摄¹³¹I率**: 摄取率增高
- **甲状腺超声**: 弥漫性肿大

**治疗**:
1. **抗甲状腺药物（ATD）**:
   - 甲巯咪唑（MMI）
   - 丙硫氧嘧啶（PTU）
   - 疗程：1.5-2年

2. **放射性碘治疗（¹³¹I）**:
   - 适用于药物无效、复发者
   - 可能导致甲减

3. **手术治疗**:
   - 甲状腺次全切除术
   - 适用于巨大甲状腺肿、怀疑恶变

4. **对症治疗**:
   - β受体阻滞剂（控制心率）

**费曼记忆**:  
甲亢 = 甲状腺激素过多 + 代谢亢进（心慌、消瘦、多汗、手抖）

**注意**: Graves病可伴突眼，需眼科评估

### 八、外科基础

#### 15. 围术期管理 (Perioperative Management/围术期管理)

**白话解释**:  
手术前、手术中、手术后的全程管理，确保手术安全。

**生活例子**:  
- 手术前禁食：防止麻醉时呕吐
- 手术后伤口换药：防止感染
- 手术后早下床：防止血栓

**学术定义**:  
从确定手术治疗起，到与本次手术有关的治疗结束为止的时期，包括术前、术中、术后三个阶段的管理。

**术前准备**:
1. **一般准备**:
   - 禁食6-8小时
   - 术前用药（镇静剂、抗生素）
   - 备皮、备血

2. **特殊准备**:
   - 高血压：血压控制在180/100mmHg以下
   - 糖尿病：血糖控制在8.3mmol/L以下
   - 心脏病：心功能评估
   - 抗凝药物：术前停用（如华法林术前5天停）

**术后处理**:
1. **生命体征监测**: 血压、心率、呼吸、体温
2. **引流管管理**: 观察引流量、颜色
3. **伤口护理**: 换药、拆线
4. **疼痛管理**: 镇痛药物
5. **早期活动**: 预防DVT
6. **营养支持**: 肠功能恢复后进食

**常见并发症及处理**:
| 并发症 | 表现 | 处理 |
|--------|------|------|
| 术后出血 | 引流管血性液体增多、血压下降 | 输血、手术止血 |
| 切口感染 | 红肿热痛、发热 | 抗生素、切开引流 |
| 肺部感染 | 发热、咳嗽咳痰 | 抗生素、雾化 |
| 下肢深静脉血栓 | 下肢肿胀 | 抗凝、溶栓 |

**费曼记忆**:  
围术期 = 术前准备 + 术中管理 + 术后监护

#### 16. 麻醉 (Anesthesia/麻醉)

**白话解释**:  
让身体"失去感觉"的技术，手术时不觉得疼痛。

**生活例子**:  
- 拔牙打麻药：局部麻醉
- 剖腹产半麻：椎管内麻醉
- 大手术全麻：全身麻醉

**学术定义**:  
使用药物或其他方法，使患者全身或局部暂时失去感觉，以达到手术或其他医疗操作时无痛的目的。

**麻醉的分类**:
1. **全身麻醉**:
   - 吸入麻醉
   - 静脉麻醉
   - 复合麻醉

2. **局部麻醉**:
   | 类型 | 适用范围 |
   |------|----------|
   | 表面麻醉 | 黏膜表面 |
   | 局部浸润麻醉 | 小手术 |
   | 区域阻滞 | 手术区域 |
   | 神经阻滞 | 肢体手术 |

3. **椎管内麻醉**:
   - 蛛网膜下腔阻滞（腰麻）
   - 硬膜外阻滞

**全身麻醉的过程**:
```
诱导期 → 麻醉药物使患者入睡
    ↓
维持期 → 维持麻醉深度
    ↓
苏醒期 → 停药后患者苏醒
```

**常用麻醉药物**:
| 类型 | 药物 | 作用 |
|------|------|------|
| 静脉麻醉药 | 丙泊酚 | 镇静催眠 |
| 吸入麻醉药 | 七氟烷 | 镇静催眠 |
| 阿片类镇痛药 | 芬太尼 | 镇痛 |
| 肌松药 | 罗库溴铵 | 肌肉松弛 |
| 局麻药 | 利多卡因 | 局部麻醉 |

**麻醉风险**:
- 过敏反应
- 呼吸抑制
- 循环抑制
- 恶心呕吐
- 术后认知功能障碍

**费曼记忆**:  
麻醉 = 让你不疼 + 让你不动 + 让你睡着（全麻时）

### 九、常见外科疾病

#### 17. 急性阑尾炎 (Acute Appendicitis/急性阑尾炎)

**白话解释**:  
阑尾发炎，是最常见的急腹症，需要手术切除。

**生活例子**:  
- 突然肚子痛，从肚脐周围转到右下腹
- 发热、恶心、呕吐
- 需要急诊手术

**学术定义**:  
阑尾的急性炎症，是外科急腹症中最常见的疾病，主要表现为转移性右下腹痛。

**临床表现**:
| 特点 | 描述 |
|------|------|
| 腹痛 | 转移性右下腹痛（70-80%）：上腹→脐周→右下腹 |
| 胃肠道症状 | 恶心、呕吐 |
| 发热 | 低至中度发热 |
| 腹部体征 | 右下腹压痛（麦氏点）、反跳痛、肌紧张 |

**麦氏点（McBurney点）**: 右髂前上棘与脐连线中外1/3交界处

**病理类型**:
1. 急性单纯性阑尾炎
2. 急性化脓性阑尾炎
3. 急性坏疽性阑尾炎（易穿孔）
4. 阑尾周围脓肿

**诊断**:
- **病史**: 转移性右下腹痛
- **体征**: 右下腹固定压痛
- **实验室**: 白细胞升高
- **影像学**: B超、CT

**治疗**:
- **手术切除**（阑尾切除术）: 绝大多数急性阑尾炎需手术
  - 开腹手术
  - 腹腔镜手术（微创）

- **非手术**: 单纯性早期、阑尾周围脓肿先保守

**费曼记忆**:  
急性阑尾炎 = 转移性右下腹痛 + 麦氏点压痛 → 手术切除

**并发症**: 穿孔、腹膜炎、阑尾周围脓肿

#### 18. 胆囊结石 (Cholecystolithiasis/胆囊结石)

**白话解释**:  
胆囊里长了"石头"，可能引起腹痛、发热。

**生活例子**:  
- 吃油腻后右上腹痛
- B超发现"胆结石"
- 腹腔镜手术取胆囊

**学术定义**:  
发生在胆囊内的结石，主要为胆固醇结石或以胆固醇为主的混合性结石，常与急性胆囊炎并存。

**危险因素（5F）**:
- Female（女性）
- Forty（40岁以上）
- Fat（肥胖）
- Fertile（多次生育）
- Family（家族史）

**临床表现**:
1. **无症状**: 50%以上无症状
2. **胆绞痛**: 右上腹阵发性绞痛，向右肩背部放射
3. **急性胆囊炎**: 发热、右上腹痛、Murphy征阳性

**诊断**:
- **B超**: 首选，可见结石回声和声影
- **CT/MRCP**: 进一步评估

**治疗**:
1. **无症状结石**: 观察，不需手术
2. **有症状结石**: 手术治疗
   - 腹腔镜胆囊切除术（LC）: 金标准
   - 开腹胆囊切除术

**手术适应症**:
- 有症状的胆囊结石
- 胆囊结石>2-3cm
- 合并胆囊息肉>1cm
- 胆囊壁钙化

**费曼记忆**:  
胆囊结石 = 右上腹痛 + B超发现结石 → 腹腔镜手术

**并发症**: 急性胆囊炎、胆管结石、胆源性胰腺炎

### 十、妇产科学

#### 19. 妊娠诊断 (Diagnosis of Pregnancy/妊娠诊断)

**白话解释**:  
判断有没有怀孕，通过月经停止、验孕棒、B超等方法。

**生活例子**:  
- 月经推迟一周：可能怀孕
- 验孕棒两条杠：阳性
- B超看到孕囊：确诊怀孕

**学术定义**:  
根据临床表现、实验室检查和影像学检查，确定妊娠存在及其状态的过程。

**早期妊娠的诊断**:
1. **临床表现**:
   - 停经（最早、最重要）
   - 早孕反应：恶心、呕吐（6周左右出现）
   - 乳房胀痛
   - 尿频

2. **实验室检查**:
   - HCG测定：尿HCG（验孕棒）、血HCG
   - 血HCG>5IU/L可诊断妊娠

3. **超声检查**:
   - 阴道B超：停经5周可见妊娠囊
   - 腹部B超：停经6周可见妊娠囊
   - 可见胎心搏动：确诊活胎

**中晚期妊娠的诊断**:
1. **子宫增大**: 根据宫高判断孕周
2. **胎动**: 18-20周自觉胎动
3. **胎心**: 18-20周可听到
4. **超声检查**: 可观察胎儿发育

**预产期计算（末次月经法）**:
```
预产期 = 末次月经月份 -3（或+9）
        日期 + 7

例：末次月经2025年6月1日
预产期：2026年3月8日
```

**费曼记忆**:  
妊娠诊断 = 停经 + HCG阳性 + B超见孕囊

#### 20. 分娩 (Delivery/分娩)

**白话解释**:  
宝宝从妈妈肚子里出来的过程，顺产或剖腹产。

**生活例子**:  
- 见红、破水：分娩先兆
- 规律宫缩：分娩开始
- 宫口开全：可以生了

**学术定义**:  
妊娠满28周及以上，胎儿及其附属物从母体娩出的过程。

**分娩的发动**:
- **先兆临产**:
  - 假宫缩：不规则、强度弱
  - 见红：阴道少量血性分泌物
  - 胎儿下降感

- **临产**: 规律宫缩（间隔5-6分钟，持续30秒以上）+ 宫颈管消失 + 宫口扩张

**产程分期**:
| 产程 | 定义 | 时间 |
|------|------|------|
| 第一产程 | 规律宫缩到宫口开全 | 初产妇11-12h，经产妇6-8h |
| 第二产程 | 宫口开全到胎儿娩出 | 初产妇1-2h，经产妇<1h |
| 第三产程 | 胎儿娩出到胎盘娩出 | 5-15分钟 |

**分娩方式**:
1. **自然分娩（顺产）**: 阴道分娩
   - 优点：恢复快、并发症少
   - 条件：产道、产力、胎儿正常

2. **剖宫产**: 手术切开子宫取出胎儿
   - 适应症：头盆不称、胎儿窘迫、产道异常等

3. **阴道助产**: 产钳、胎头吸引

**Apgar评分（新生儿评估）**:
| 指标 | 0分 | 1分 | 2分 |
|------|-----|-----|-----|
| 心率 | 无 | <100次/分 | ≥100次/分 |
| 呼吸 | 无 | 慢、不规则 | 哭声响 |
| 肌张力 | 松弛 | 四肢略屈曲 | 活动好 |
| 喉反射 | 无反应 | 皱眉 | 咳嗽、喷嚏 |
| 皮肤颜色 | 全身苍白/青紫 | 躯干红、四肢紫 | 全身红润 |

- 正常：8-10分
- 轻度窒息：4-7分
- 重度窒息：0-3分

**费曼记忆**:  
分娩 = 宫缩 + 宫口开 + 宝宝出来（三个产程）

### 十一、儿科学

#### 21. 小儿肺炎 (Pediatric Pneumonia/小儿肺炎)

**白话解释**:  
儿童肺部感染，发热、咳嗽、呼吸急促，是儿童常见病。

**生活例子**:  
- 宝宝发烧、咳嗽
- 呼吸急促、吃奶差
- 医生听诊"肺部有啰音"

**学术定义**:  
由不同病原体或其他因素所致的肺部炎症，以发热、咳嗽、气促、呼吸困难和肺部固定中细湿啰音为共同临床表现。

**分类**:
1. **按病理**:
   - 支气管肺炎（最常见）
   - 大叶性肺炎
   - 间质性肺炎

2. **按病因**:
   - 细菌性：肺炎链球菌、流感嗜血杆菌
   - 病毒性：RSV、流感病毒
   - 支原体肺炎

**临床表现**:
- **发热**: 不规则热或弛张热
- **咳嗽**: 初为干咳，后有痰
- **气促**: 呼吸频率增快
- **呼吸困难**: 鼻翼扇动、三凹征
- **肺部体征**: 固定中细湿啰音

**重症肺炎的表现**:
1. **呼吸衰竭**: 呼吸困难、发绀
2. **心力衰竭**: 心率快、肝大、水肿
3. **中毒性脑病**: 惊厥、昏迷
4. **DIC**: 出血

**诊断**:
- **症状**: 发热、咳嗽、气促
- **体征**: 肺部啰音
- **胸片**: 斑片状阴影

**治疗**:
1. **抗感染**:
   - 细菌性：抗生素
   - 病毒性：对症支持
   - 支原体：大环内酯类

2. **对症治疗**:
   - 退热
   - 止咳祛痰
   - 氧疗
   - 雾化吸入

**费曼记忆**:  
小儿肺炎 = 发热 + 咳嗽 + 气促 + 肺部啰音

#### 22. 小儿腹泻 (Pediatric Diarrhea/小儿腹泻)

**白话解释**:  
宝宝拉肚子，次数多、便便稀，容易脱水。

**生活例子**:  
- 一天拉好几次
- 大便像水一样
- 精神差、口渴

**学术定义**:  
由多病原、多因素引起的以大便次数增多和大便性状改变为特点的消化道综合征。

**分类**:
1. **按病程**:
   - 急性腹泻：<2周
   - 迁延性腹泻：2周-2个月
   - 慢性腹泻：>2个月

2. **按病因**:
   - 感染性：病毒（轮状病毒最常见）、细菌
   - 非感染性：喂养不当、过敏

**临床表现**:
1. **轻型**:
   - 腹泻<10次/天
   - 无脱水
   - 精神尚可

2. **重型**:
   - 腹泻>10次/天
   - 脱水表现
   - 电解质紊乱

**脱水程度评估**:
| 表现 | 轻度 | 中度 | 重度 |
|------|------|------|------|
| 失水量（占体重%） | 3-5% | 5-10% | >10% |
| 精神 | 稍差 | 烦躁或萎靡 | 昏睡/昏迷 |
| 皮肤弹性 | 正常 | 稍差 | 极差 |
| 前囟/眼窝 | 稍凹陷 | 凹陷 | 明显凹陷 |
| 尿量 | 略减少 | 明显减少 | 无尿 |

**治疗原则**:
1. **液体疗法**（核心）:
   - 口服补液盐（ORS）
   - 静脉补液（重度脱水）

2. **继续饮食**: 
   - 母乳喂养继续
   - 人工喂养可换去乳糖配方

3. **药物治疗**:
   - 微生态制剂（益生菌）
   - 肠黏膜保护剂（蒙脱石散）
   - 补锌

**费曼记忆**:  
小儿腹泻 = 拉肚子 + 脱水（关键在于补液）

## 🎯 重点概念速记

### 必背概念（22个）

1. **病史采集**: 问诊了解病情
2. **主诉**: 主要症状+时间
3. **体格检查**: 视触叩听
4. **心脏听诊**: 五个瓣膜听诊区
5. **心电图**: P-QRS-T波
6. **高血压**: 血压≥140/90mmHg
7. **冠心病**: 冠状动脉狭窄/堵塞
8. **心绞痛**: 劳累后胸痛
9. **心肌梗死**: 持续胸痛+ST改变
10. **肺炎**: 肺部感染
11. **COPD**: 慢性阻塞性肺疾病
12. **胃溃疡**: 餐后痛
13. **十二指肠溃疡**: 空腹痛
14. **肝硬化**: 肝纤维化+假小叶
15. **糖尿病**: 血糖升高+三多一少
16. **甲亢**: 甲状腺激素过多
17. **围术期**: 术前+术中+术后
18. **麻醉**: 消除疼痛感觉
19. **急性阑尾炎**: 转移性右下腹痛
20. **胆囊结石**: 右上腹痛+B超确诊
21. **妊娠诊断**: 停经+HCG+B超
22. **分娩**: 三个产程
