# 经济学知识词典

## 📚 简介

本词典基于**中国教育部《学位授予和人才培养学科目录（2020年）》**，对经济学门类（代码：02）的所有二级学科和主要研究方向进行全面系统的知识分类。

### 核心特点
- ✅ **最新标准**：依据2024年最新学科分类和前沿发展
- ✅ **全面覆盖**：包含所有二级学科、三级学科及重要知识点
- ✅ **费曼学习法**：每个概念都用通俗易懂的语言解释
- ✅ **模块化设计**：按学科分类，便于查找和学习

## 🎯 费曼学习法说明

每个概念的解释遵循费曼学习法原则：
1. **简单易懂**：用日常语言解释复杂概念
2. **类比举例**：通过生活中的例子帮助理解
3. **核心要点**：抓住概念的本质
4. **应用场景**：说明在什么情况下使用
