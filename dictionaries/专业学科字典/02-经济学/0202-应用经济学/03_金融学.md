### 一、货币理论

#### 1. 货币 (Money/货币)

**白话解释**:  
大家都认可的交易媒介，从贝壳到纸币到数字货币。

**货币定义**:  
固定充当一般等价物的特殊商品或符号。

**五大职能**:
| 职能 | 含义 | 例子 |
|------|------|------|
| 价值尺度 | 衡量商品价值 | 标价100元 |
| 流通手段 | 交易媒介 | 买东西付钱 |
| 贮藏手段 | 保存财富 | 存钱 |
| 支付手段 | 延期支付 | 工资、赊账 |
| 世界货币 | 国际结算 | 美元、欧元 |

**货币形态演变**:
```
实物货币
├─ 贝壳、牲畜
└─ 缺点：不耐久、难分割

金属货币
├─ 金银铜
└─ 缺点：携带不便

纸币
├─ 信用货币
└─ 国家信用背书

电子货币
├─ 银行卡
└─ 电子支付

数字货币
├─ 数字人民币
└─ 去中心化（比特币）
```

**费曼记忆**:  
货币 = 一般等价物，五大职能记心间

---

#### 2. 货币制度 (Monetary System/货币制度)

**白话解释**:  
国家关于货币的法律规定。

**主要内容**:
```
货币制度
├─ 货币材料（金、银、纸）
├─ 货币单位（元、角、分）
├─ 货币铸造/发行
└─ 货币支付能力
```

**货币制度类型**:
| 类型 | 特点 | 时代 |
|------|------|------|
| 银本位制 | 白银为本位货币 | 古代 |
| 金本位制 | 黄金为本位货币 | 19-20世纪 |
| 金汇兑本位制 | 与某国货币挂钩 | 二战后 |
| 纸币本位制 | 纸币为本位货币 | 现代 |

**人民币制度**:
- 发行机构：中国人民银行
- 单位：元
- 法偿能力：无限法偿
- 管理制度：有管理的浮动汇率

**费曼记忆**:  
货币制度 = 货币材料 + 发行规定 + 法偿能力

### 二、信用理论

#### 3. 信用 (Credit/信用)

**白话解释**:  
借东西的承诺，未来偿还。

**信用定义**:  
以偿还为条件的价值单方面转移。

**信用形式**:
| 形式 | 主体 | 例子 |
|------|------|------|
| 商业信用 | 企业之间 | 赊销、分期付款 |
| 银行信用 | 银行与企业/个人 | 贷款 |
| 国家信用 | 政府 | 国债 |
| 消费信用 | 商家/银行与消费者 | 信用卡、消费贷 |
| 国际信用 | 国家之间 | 政府间贷款 |

**信用工具**:
```
信用工具
├─ 短期
│   ├─ 商业票据
│   ├─ 银行承兑汇票
│   └─ 国库券
├─ 中期
│   ├─ 企业中期债券
│   └─ 中期票据
└─ 长期
    ├─ 国债
    ├─ 企业债
    └─ 公司债
```

**信用评级**:
| 等级 | 含义 | 风险 |
|------|------|------|
| AAA | 最高等级 | 最低 |
| AA | 高等级 | 很低 |
| A | 中上等级 | 低 |
| BBB | 中等级 | 中等 |
| BB以下 | 投机级 | 高 |

**费曼记忆**:  
信用 = 借贷关系，按期还本付息

---

#### 4. 利率 (Interest Rate/利率)

**白话解释**:  
借钱的成本，存钱的收益。

**利率定义**:  
利息与本金的比率。

**利率类型**:
| 类型 | 含义 |
|------|------|
| 名义利率 | 银行标明的利率 |
| 实际利率 | 名义利率减通胀率 |
| 固定利率 | 合同期内不变 |
| 浮动利率 | 随市场变化 |
| 基准利率 | 央行确定的利率 |

**实际利率计算**:
```
实际利率 = 名义利率 - 通胀率

例：
名义利率 = 5%
通胀率 = 3%
实际利率 = 5% - 3% = 2%
```

**利率的作用**:
```
利率上升
├─ 储蓄增加
├─ 投资减少
├─ 消费减少
└─ 经济降温

利率下降
├─ 储蓄减少
├─ 投资增加
├─ 消费增加
└─ 经济刺激
```

**中国利率体系**:
```
央行利率
├─ 存款准备金利率
├─ 再贷款利率
└─ 公开市场操作利率
    ↓
市场利率
├─ SHIBOR
├─ LPR
└─ 国债收益率
    ↓
商业银行利率
├─ 存款利率
└─ 贷款利率
```

**费曼记忆**:  
实际利率 = 名义利率 - 通胀率

### 一、货币市场

#### 5. 货币市场 (Money Market/货币市场)

**白话解释**:  
借还钱期限在一年以内的短期资金市场。

**特点**:
1. 期限短（一年以内）
2. 流动性强
3. 风险低
4. 收益低

**主要工具**:
| 工具 | 发行者 | 期限 |
|------|--------|------|
| 国库券 | 财政部 | 3-12个月 |
| 商业票据 | 企业 | 1-270天 |
| 银行承兑汇票 | 银行 | 30-180天 |
| 大额存单 | 银行 | 7天-1年 |
| 回购协议 | 金融机构 | 隔夜-1年 |

**银行间拆借市场**:
```
银行间拆借
├─ 银行之间互相借钱
├─ 调剂头寸
└─ SHIBOR（上海银行间同业拆放利率）
```

**费曼记忆**:  
货币市场 = 一年以内的短期资金市场

### 二、资本市场

#### 6. 股票市场 (Stock Market/股票市场)

**白话解释**:  
买卖股票的地方，公司融资、股民投资。

**股票定义**:  
公司所有权的凭证。

**股票分类**:
| 类型 | 特点 | 权利 |
|------|------|------|
| 普通股 | 最常见 | 投票权、分红权 |
| 优先股 | 优先分红 | 优先分红、无投票权 |
| A股 | 人民币计价 | 国内投资者 |
| B股 | 外币计价 | 外资 |
| H股 | 港币计价 | 香港上市 |

**股票价格决定**:
```
股票价格 = 预期股息 / 市场利率

例：
预期股息 = 2元/股
市场利率 = 5%
股价 = 2 / 0.05 = 40元
```

**股票指数**:
| 指数 | 市场 | 代表 |
|------|------|------|
| 上证指数 | 上海 | 沪市整体 |
| 深证成指 | 深圳 | 深市整体 |
| 创业板指 | 创业板 | 创新企业 |
| 科创50 | 科创板 | 科技企业 |
| 沪深300 | 沪深 | 大盘股 |

**中国股市结构**:
```
上海证券交易所
├─ 主板
└─ 科创板（科技创新）

深圳证券交易所
├─ 主板
├─ 创业板（成长型）
└─ 中小板（已并入主板）

北京证券交易所
└─ 新三板精选层
```

**费曼记忆**:  
股票 = 所有权凭证，价格由预期收益决定

#### 7. 债券市场 (Bond Market/债券市场)

**白话解释**:  
买卖债券的地方，借钱给别人赚利息。

**债券定义**:  
债务凭证，发行者承诺还本付息。

**债券要素**:
```
债券
├─ 面值（本金）
├─ 票面利率（年利息）
├─ 期限（到期日）
└─ 发行价格
```

**债券分类**:
| 类型 | 发行者 | 风险 | 收益 |
|------|--------|------|------|
| 国债 | 中央政府 | 最低 | 最低 |
| 地方债 | 地方政府 | 低 | 低 |
| 企业债 | 国有企业 | 中 | 中 |
| 公司债 | 上市公司 | 较高 | 较高 |

**债券定价**:
```
债券价格 = 未来现金流的现值之和

例：
面值100元，票面利率5%，期限3年，市场利率6%

第1年利息现值：5/(1+6%) = 4.72
第2年利息现值：5/(1+6%)² = 4.45
第3年利息+本金现值：105/(1+6%)³ = 88.16

债券价格 = 4.72 + 4.45 + 88.16 = 97.33元
```

**债券与股票比较**:
| 对比项 | 债券 | 股票 |
|--------|------|------|
| 性质 | 债权 | 所有权 |
| 收益 | 固定利息 | 不确定股息 |
| 风险 | 低 | 高 |
| 偿还 | 到期还本 | 不还本 |

**费曼记忆**:  
债券 = 债务凭证，价格与市场利率反向

### 三、金融衍生品

#### 8. 金融衍生品 (Financial Derivatives/金融衍生品)

**白话解释**:  
价值来源于其他资产的金融工具。

**主要类型**:
| 类型 | 含义 | 用途 |
|------|------|------|
| 远期 | 未来按约定价格交易 | 锁定价格 |
| 期货 | 标准化远期合约 | 套期保值、投机 |
| 期权 | 买卖权利 | 风险管理 |
| 互换 | 交换现金流 | 降低成本 |

**期货交易**:
```
期货合约
├─ 标准化合约
├─ 交易所交易
├─ 保证金制度
└─ 每日结算

例：
3个月后买入100吨铜，约定价格50,000元/吨
├─ 期货多头：买入方
└─ 期货空头：卖出方
```

**期权交易**:
```
看涨期权（买权）：有权买入
看跌期权（卖权）：有权卖出

例：
买入看涨期权
├─ 支付期权费
├─ 有权按约定价格买入
└─ 没有义务
```

**衍生品功能**:
1. **套期保值**: 规避价格风险
2. **价格发现**: 预测未来价格
3. **投机**: 赚取差价

**费曼记忆**:  
衍生品 = 期货 + 期权 + 互换，价值源于基础资产

#### 9. 中央银行 (Central Bank/中央银行)

银行的银行，发行货币、制定货币政策。

**三大职能**:
```
发行的银行

银行的银行

政府的银行
├─ 代理国库

```

**中国人民银行**:
- 成立：1948年
- 职责：制定执行货币政策、维护金融稳定、金融服务

**央行资产负债表**:
```
资产
├─ 外汇储备
├─ 对政府债权
├─ 对银行债权
└─ 其他资产

负债
├─ 流通中货币
├─ 银行存款
├─ 政府存款
└─ 资本金
```

**费曼记忆**:  
央行 = 发行的银行 + 银行的银行 + 政府的银行

#### 10. 货币政策 (Monetary Policy/货币政策)

**白话解释**:  
央行通过控制货币供应量调节经济。

**政策目标**:
1. 物价稳定
2. 充分就业
3. 经济增长
4. 国际收支平衡

**政策工具**:
| 工具 | 操作 | 效果 |
|------|------|------|
| 存款准备金率 | 调整比例 | 控制信贷规模 |
| 再贴现率 | 调整利率 | 影响银行借款成本 |
| 公开市场操作 | 买卖债券 | 调节基础货币 |
| 利率政策 | 调整利率 | 影响投资消费 |

**货币政策传导**:
```
央行操作
    ↓
基础货币变化
    ↓
货币供应量变化
    ↓
利率变化
    ↓
投资消费变化
    ↓
产出物价变化
```

**政策类型**:
```
扩张性货币政策
├─ 降低准备金率
├─ 降低再贴现率
├─ 买入债券
└─ 降低利率
    ↓
刺激经济

紧缩性货币政策
├─ 提高准备金率
├─ 提高再贴现率
├─ 卖出债券
└─ 提高利率
    ↓
抑制经济
```

**费曼记忆**:  
货币政策工具 = 准备金率 + 再贴现率 + 公开市场操作 + 利率

负债业务（资金来源）

信用风险

### 一、金融风险

#### 13. 金融风险 (Financial Risk/金融风险)

**白话解释**:  
金融市场可能发生的损失。

**风险类型**:
| 风险 | 含义 | 例子 |
|------|------|------|
| 信用风险 | 借款人违约 | 贷款不还 |
| 市场风险 | 价格波动 | 股市下跌 |
| 流动性风险 | 无法变现 | 银行挤兑 |
| 操作风险 | 人为失误 | 系统故障 |
| 法律风险 | 法律变化 | 政策变化 |

**系统性风险**:
```
单个机构风险
    ↓
传导扩散
    ↓
整个系统受影响
    ↓
金融危机
```

**费曼记忆**:  
金融风险 = 信用 + 市场 + 流动性 + 操作

---

#### 14. 金融危机 (Financial Crisis/金融危机)

**白话解释**:  
金融系统严重失灵，资产价格暴跌。

**典型类型**:
| 类型 | 特征 | 例子 |
|------|------|------|
| 银行危机 | 银行挤兑、倒闭 | 1929年大萧条 |
| 货币危机 | 货币大幅贬值 | 1997亚洲金融危机 |
| 债务危机 | 无法偿还债务 | 2010欧债危机 |

**2008年金融危机**:
```
次贷危机
├─ 房价下跌
├─ 次贷违约
└─ 衍生品放大
    ↓
雷曼兄弟倒闭
    ↓
全球金融海啸
    ↓
经济衰退
```

**危机应对**:
1. 央行注入流动性
2. 政府救助金融机构
3. 财政刺激
4. 金融监管改革

**费曼记忆**:  
金融危机 = 风险积累 → 触发 → 传导 → 爆发

### 二、金融监管体系

#### 15. 金融监管 (Financial Regulation/金融监管)

**白话解释**:  
政府管理金融机构和市场的活动。

**监管目标**:
1. 防范系统性风险
2. 保护消费者权益
3. 维护市场秩序
4. 促进金融发展

**中国金融监管架构**:
```
国务院金融稳定发展委员会
        ↓
┌───────┼───────┐
↓       ↓       ↓
央行    金融监管总局  证监会
（货币）（银行保险）（证券）
```

**监管内容**:
| 内容 | 说明 |
|------|------|
| 市场准入 | 机构设立审批 |
| 业务运营 | 合规检查 |
| 风险控制 | 资本充足率 |
| 市场退出 | 破产处置 |

**巴塞尔协议**:
```
巴塞尔协议III
├─ 资本充足率 ≥ 8%
├─ 核心一级资本充足率 ≥ 4.5%
├─ 杠杆率 ≤ 3%
└─ 流动性覆盖率 ≥ 100%
```

**费曼记忆**:  
金融监管 = 准入 + 运营 + 风控 + 退出

#### 16. 宏观审慎管理

**白话解释**:  
从整体上防范金融系统性风险。

**主要内容**:
```
宏观审慎政策
├─ 逆周期调节
│   ├─ 逆周期资本缓冲
│   └─ 动态拨备
├─ 系统重要性机构监管
│   ├─ 附加资本要求
│   └─ 恢复与处置计划
└─ 跨市场风险防范
    ├─ 金融控股公司监管
    └─ 关联交易限制
```

**中国实践**:
- 住房信贷政策调整
- 系统重要性银行名单
- 金融控股公司监管

**费曼记忆**:  
宏观审慎 = 防范系统性风险

### 一、汇率与国际收支

#### 17. 汇率 (Exchange Rate/汇率)

**白话解释**:  
两种货币之间的兑换比例。

**汇率标价**:
| 方法 | 表达 | 例子 |
|------|------|------|
| 直接标价 | 1外币=?本币 | 1美元=7人民币 |
| 间接标价 | 1本币=?外币 | 1人民币=0.14美元 |

**汇率制度**:
| 制度 | 特点 | 代表 |
|------|------|------|
| 固定汇率 | 盯住某货币 | 港币盯住美元 |
| 浮动汇率 | 市场决定 | 美元、欧元 |
| 管理浮动 | 有管理浮动 | 人民币 |

**汇率影响**:
```
本币贬值
├─ 出口增加
├─ 进口减少
├─ 通胀上升
└─ 外债负担加重

本币升值
├─ 出口减少
├─ 进口增加
├─ 通胀下降
└─ 外债负担减轻
```

**人民币汇率**:
- 制度：以市场供求为基础、参考一篮子货币、有管理的浮动
- 2023年：约7.1元/美元

**费曼记忆**:  
贬值利好出口，升值利好进口

---

#### 18. 国际收支 (Balance of Payments/国际收支)

**白话解释**:  
一个国家与其他国家经济往来的记录。

**账户结构**:
```
国际收支
├─ 经常账户
│   ├─ 货物贸易
│   ├─ 服务贸易
│   ├─ 收入
│   └─ 经常转移
├─ 资本账户
│   └─ 资本转移
└─ 金融账户
    ├─ 直接投资
    ├─ 证券投资
    └─ 其他投资
```

**国际收支平衡**:
```
顺差：收入 > 支出
├─ 外汇储备增加
└─ 本币升值压力

逆差：收入 < 支出
├─ 外汇储备减少
└─ 本币贬值压力
```

**中国外汇储备**:
- 2023年：约3.2万亿美元
- 世界第一

**费曼记忆**:  
国际收支 = 经常账户 + 资本账户 + 金融账户
