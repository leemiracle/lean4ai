### 一、商品与货币

#### 1. 商品 (Commodity/商品)

**白话解释**:  
用来交换的劳动产品，既有用又值钱。

**生活例子**:  
- 超市里的面包 → 有用（能吃）+ 值钱（有人买）
- 自己做的饭 → 有用但不是商品（不卖）
- 空气 → 有用但不是商品（不劳动就能获得）

**学术定义**:  
用于交换的劳动产品，具有使用价值和价值二重性。

**关键要点**:
1. **使用价值** - 满足人们某种需要的属性
2. **价值** - 凝结在商品中的无差别人类劳动
3. **交换价值** - 一种商品与另一种商品交换的比例
4. **二因素统一** - 使用价值是物质承担者，价值是本质属性

**费曼记忆**:  
商品 = 有用 + 有劳动 + 用来换

---

#### 2. 货币 (Money/货币)

**白话解释**:  
大家都认可的交易媒介，从贝壳到纸币到数字货币。

**生活例子**:  
- 古代：贝壳、铜钱
- 现代：纸币、硬币
- 当代：微信支付、支付宝、数字人民币

**学术定义**:  
固定充当一般等价物的特殊商品，是商品交换发展到一定阶段的产物。

**五大职能**:
| 职能 | 含义 | 例子 |
|------|------|------|
| 价值尺度 | 衡量商品价值 | 标价100元 |
| 流通手段 | 交易媒介 | 买面包付钱 |
| 贮藏手段 | 财富储存 | 存钱 |
| 支付手段 | 延期支付 | 信用卡、赊账 |
| 世界货币 | 国际结算 | 美元、欧元 |

**货币演变**:
```
实物货币（贝壳、牲畜）
    ↓
金属货币（金银铜）
    ↓
纸币（信用货币）
    ↓
电子货币（银行卡）
    ↓
数字货币（数字人民币、比特币）
```

**费曼记忆**:  
货币 = 一般等价物，五大职能记心间

---

#### 3. 价值规律 (Law of Value/价值规律)

**白话解释**:  
商品的价值由社会必要劳动时间决定，价格围绕价值上下波动。

**生活例子**:  
- 手机价格：成本2000元的手机，可能卖2500元，也可能降价到1800元
- 季节性波动：夏天买羽绒服便宜，冬天买贵

**学术定义**:  
商品的价值由生产该商品的社会必要劳动时间决定，商品交换以价值量为基础实行等价交换。

**表现形式**:
```
价格
  ↑
  │     ∧     ∧
  │    / \   / \
  │   /   \ /   \
  │──/─────V─────\──→ 时间
  │        价值线
```

**作用**:
1. **调节生产** - 价格高，生产者增加；价格低，生产者减少
2. **刺激技术进步** - 个别劳动时间 < 社会必要劳动时间 = 赚钱
3. **优胜劣汰** - 效率低的生产者被淘汰

**费曼记忆**:  
价值决定价格，价格围绕价值波动

### 二、剩余价值理论

#### 4. 剩余价值 (Surplus Value/剩余价值)

**白话解释**:  
工人创造的价值中，被老板无偿占有的那部分。

**生活例子**:  
- 工人一天创造500元价值，工资200元，剩余300元被老板拿走
- 这300元就是剩余价值（利润来源）

**学术定义**:  
雇佣工人在生产过程中创造的、被资本家无偿占有的、超过劳动力价值的那部分价值。

**公式**:
```
商品价值 = c（不变资本）+ v（可变资本）+ m（剩余价值）

c: 机器、原材料（价值转移，不创造新价值）
v: 工人工资（劳动力价值）
m: 剩余价值（利润来源）

例：生产一件衣服
- 布料成本：30元（c）
- 工人工资：20元（v）
- 售价：100元
- 剩余价值：100 - 30 - 20 = 50元（m）
```

**剩余价值率**:
```
m' = m/v × 100%

例：m=50元，v=20元
m' = 50/20 × 100% = 250%
（剥削程度：工人每得1元工资，老板拿走2.5元）
```

**费曼记忆**:  
剩余价值 = 创造价值 - 工资

---

#### 5. 资本 (Capital/资本)

**白话解释**:  
能生钱的钱，用来赚钱的资产。

**生活例子**:  
- 机器设备 → 能生产商品卖钱
- 厂房 → 能出租或生产
- 资金 → 能投资获得收益

**学术定义**:  
能够带来剩余价值的价值，体现资本家剥削雇佣工人的社会关系。

**资本分类**:
| 类型 | 含义 | 特点 |
|------|------|------|
| 不变资本(c) | 机器、原材料 | 价值转移，不增殖 |
| 可变资本(v) | 劳动力 | 创造新价值，增殖源泉 |
| 固定资本 | 机器、厂房 | 多次使用，逐渐转移 |
| 流动资本 | 原材料、工资 | 一次使用，全部转移 |

**资本循环**:
```
货币资本(G) → 生产资本(P) → 商品资本(W') → 货币资本(G')
    ↓              ↓              ↓              ↓
  买生产资料     生产过程        卖商品        收回更多货币
```

**费曼记忆**:  
资本的本质是带来剩余价值的价值

---

### 三、资本积累与经济危机

#### 6. 资本积累 (Capital Accumulation/资本积累)

**白话解释**:  
老板把赚的钱再投入生产，滚雪球一样越做越大。

**生活例子**:  
- 小店赚钱 → 开分店 → 开连锁 → 上市
- 利润再投资，规模越来越大

**学术定义**:  
剩余价值的资本化，即把剩余价值的一部分转化为资本。

**积累公式**:
```
剩余价值(m) = 消费基金 + 积累基金
                ↓           ↓
             个人消费    扩大再生产

例：赚100万
- 消费20万（买房买车）
- 积累80万（再投资扩大生产）
```

**后果**:
1. **资本有机构成提高** - 机器增多，工人相对减少
2. **相对过剩人口** - 失业人口增加
3. **贫富差距扩大** - 财富越来越集中

**费曼记忆**:  
资本积累 = 剩余价值资本化

---

#### 7. 经济危机 (Economic Crisis/经济危机)

**白话解释**:  
经济"生病"了，生产过剩卖不出去，企业倒闭，工人失业。

**生活例子**:  
- 1929年大萧条：牛奶倒进河里，产品销毁
- 2008年金融危机：银行倒闭，股市崩盘
- 2020年疫情危机：停工停产，经济衰退

**学术定义**:  
资本主义生产过程中周期性爆发的生产过剩危机，其实质是相对过剩。

**根本原因**:  
生产社会化与资本主义私有制之间的矛盾

**表现**:
1. **生产过剩** - 商品卖不出去
2. **企业倒闭** - 大量破产
3. **失业增加** - 工人失业
4. **股市崩盘** - 金融危机
5. **信用破坏** - 债务违约

**周期阶段**:
```
繁荣 → 危机 → 萧条 → 复苏 → 繁荣...
 ↑                              ↑
 └────── 周期循环 ──────────────┘
```

**费曼记忆**:  
经济危机 = 生产相对过剩（不是绝对过剩）

### 一、古典经济学

#### 8. 亚当·斯密 (Adam Smith/亚当·斯密)

**白话解释**:  
经济学鼻祖，"看不见的手"理论提出者。

**核心思想**:  
- 《国富论》(1776)：自由市场经济
- "看不见的手"：市场自动调节资源配置
- 分工提高效率

**生活例子**:  
- 市场买菜：没人指挥，但想买的人能买到，想卖的人能卖出
- 餐馆竞争：不好吃的倒闭，好吃的留下

**看不见的手**:
```
每个人追求自己的利益
        ↓
    市场竞争
        ↓
社会资源最优配置（好像有一只看不见的手在指挥）
```

**费曼记忆**:  
亚当·斯密 = 看不见的手 + 分工提高效率

---

#### 9. 大卫·李嘉图 (David Ricardo/大卫·李嘉图)

**白话解释**:  
比较优势理论提出者，自由贸易倡导者。

**核心思想**:  
- 比较优势：各国生产自己相对擅长的产品
- 自由贸易：各国都获益

**生活例子**:  
- 律师打字慢，但请秘书打字更划算（律师专注办案）
- 中国生产衣服，美国生产芯片，然后贸易交换

**比较优势表**:
| 国家 | 生产衣服成本 | 生产芯片成本 | 比较优势 |
|------|-------------|-------------|----------|
| 中国 | 10小时 | 100小时 | 衣服 |
| 美国 | 20小时 | 50小时 | 芯片 |

**结论**:  
即使美国两种都更强，但相对优势不同，贸易仍对双方有利。

**费曼记忆**:  
李嘉图 = 比较优势 + 自由贸易

---

### 二、马克思主义经济学

#### 10. 马克思 (Karl Marx/马克思)

**白话解释**:  
政治经济学批判者，《资本论》作者。

**核心贡献**:
1. **劳动价值论** - 价值来源于劳动
2. **剩余价值理论** - 揭示剥削本质
3. **经济危机理论** - 解释危机周期性
4. **资本积累理论** - 分析贫富分化

**《资本论》三卷**:
- 第一卷：资本的生产过程
- 第二卷：资本的流通过程
- 第三卷：资本主义生产的总过程

**费曼记忆**:  
马克思 = 劳动价值 + 剩余价值 + 资本积累

---

### 三、新古典经济学

#### 11. 马歇尔 (Alfred Marshall/马歇尔)

**白话解释**:  
微观经济学奠基人，供求理论集大成者。

**核心贡献**:
1. **供求均衡** - 供给曲线和需求曲线交点决定价格
2. **边际分析** - 边际效用递减
3. **弹性概念** - 价格弹性、收入弹性

**供求曲线**:
```
价格
  ↑
  │   S（供给）
  │    \
  │     \   E（均衡点）
  │      \ /
  │       X
  │      / \
  │     /   \
  │    D（需求）
  └────────────→ 数量
```

**费曼记忆**:  
马歇尔 = 供求均衡 + 边际分析

---

#### 12. 凯恩斯 (John Maynard Keynes/凯恩斯)

**白话解释**:  
宏观经济学创始人，政府干预经济理论的提出者。

**核心思想**:
1. **有效需求不足** - 经济危机的根源
2. **政府干预** - 财政政策、货币政策
3. **乘数效应** - 投资的放大作用

**生活例子**:  
- 经济不好时，政府修桥铺路（投资）
- 工人拿到工资消费
- 商家赚到钱再投资
- 一笔投资带动多倍经济活动

**凯恩斯公式**:
```
Y = C + I + G + (X - M)

Y: 国民收入
C: 消费
I: 投资
G: 政府支出
X: 出口
M: 进口
```

**乘数效应**:
```
投资100万
  ↓
第一轮：工人收入100万，消费80万
  ↓
第二轮：商家收入80万，消费64万
  ↓
第三轮：...
  ↓
总效应：100万 × 乘数（如5）= 500万
```

**费曼记忆**:  
凯恩斯 = 有效需求 + 政府干预 + 乘数效应

### 一、中国经济史

#### 13. 中国古代经济

**白话解释**:  
以农业为主，小农经济，重农抑商。

**主要特征**:
1. **小农经济** - 男耕女织，自给自足
2. **土地制度** - 土地私有，地主佃农
3. **重农抑商** - 农业为本，商业被压制
4. **朝贡贸易** - 周边国家进贡

**经济政策演变**:
| 朝代 | 主要政策 | 特点 |
|------|----------|------|
| 秦汉 | 重农抑商 | 统一货币、度量衡 |
| 唐宋 | 商业繁荣 | 丝绸之路、市舶司 |
| 明清 | 海禁政策 | 闭关锁国 |

**费曼记忆**:  
古代经济 = 小农经济 + 重农抑商

---

#### 14. 中国近代经济（1840-1949）

**白话解释**:  
被动开放，民族资本主义艰难发展。

**发展阶段**:
1. **洋务运动**(1861-1895) - 官办企业兴起
2. **民族资本主义**(1895-1919) - 民办企业发展
3. **黄金十年**(1927-1937) - 民族工业短暂繁荣
4. **抗战时期**(1937-1945) - 经济遭受破坏
5. **解放战争**(1946-1949) - 通货膨胀，经济崩溃

**费曼记忆**:  
近代经济 = 被动开放 + 民族资本主义艰难发展

---

#### 15. 中国现代经济（1949至今）

**白话解释**:  
从计划经济到市场经济，改革开放创造奇迹。

**发展阶段**:
| 时期 | 经济体制 | 特点 |
|------|----------|------|
| 1949-1978 | 计划经济 | 公有制、计划生产 |
| 1978-1992 | 改革开放初期 | 家庭联产承包、乡镇企业 |
| 1992-2012 | 市场经济建立 | 南巡讲话、加入WTO |
| 2012至今 | 新时代经济 | 高质量发展、数字经济 |

**改革开放成就**:
- GDP：1978年3679亿元 → 2023年126万亿元
- 人均GDP：1978年385元 → 2023年8.94万元
- 贫困人口：1978年7.7亿 → 2020年全面脱贫

**费曼记忆**:  
现代经济 = 计划经济 → 市场经济 → 高质量发展

### 一、微观经济学

#### 17. 需求与供给 (Demand and Supply/需求与供给)

**白话解释**:  
- 需求：消费者想买且买得起
- 供给：生产者愿意卖且能生产

**生活例子**:  
- 需求：你想要iPhone且有钱买
- 供给：苹果公司生产iPhone愿意卖

**需求定律**:  
价格上升，需求量下降；价格下降，需求量上升

**供给定律**:  
价格上升，供给量上升；价格下降，供给量下降

**均衡价格**:
```
价格
  ↑
  │     S（供给）
  │      \
  │       \  E（均衡点）
  │        \____
  │        /
  │   ____/（需求）
  │  D
  └────────────→ 数量
```

**费曼记忆**:  
价格上升 → 需求减、供给增

---

#### 18. 弹性 (Elasticity/弹性)

**白话解释**:  
一个变量变化引起另一个变量变化的敏感程度。

**生活例子**:  
- 大米涨价：还是要买（缺乏弹性）
- 奢侈品涨价：不买了（富有弹性）

**需求价格弹性**:
```
Ed = 需求量变化率 / 价格变化率

Ed > 1: 富有弹性（奢侈品）
Ed = 1: 单位弹性
Ed < 1: 缺乏弹性（必需品）
Ed = 0: 完全无弹性（救命的药）
Ed = ∞: 完全弹性
```

**弹性分类**:
| 类型 | 例子 | 特点 |
|------|------|------|
| 富有弹性 | 奢侈品、可替代品 | 涨价销量大降 |
| 缺乏弹性 | 必需品、水电 | 涨价销量小降 |
| 单位弹性 | 一般商品 | 涨价销量等比下降 |

**费曼记忆**:  
必需品缺乏弹性，奢侈品富有弹性

---

#### 19. 消费者理论 (Consumer Theory/消费者理论)

**白话解释**:  
消费者如何在有限收入下实现效用最大化。

**核心概念**:
1. **效用** - 满足程度
2. **边际效用** - 多消费一单位带来的额外满足
3. **边际效用递减** - 越多越不稀罕

**生活例子**:  
- 吃第一个包子：太香了（边际效用高）
- 吃第三个包子：还行（边际效用降低）
- 吃第五个包子：撑了（边际效用为负）

**消费者均衡**:
```
MUx/Px = MUy/Py

每花1元钱在不同商品上获得的边际效用相等
```

**无差异曲线**:
```
Y商品
  ↑
  │  ●
  │   ●
  │    ●  （同一条曲线上的点效用相同）
  │     ●
  │      ●
  └────────→ X商品
```

**费曼记忆**:  
边际效用递减，每元边际效用相等时均衡

---

#### 20. 生产者理论 (Producer Theory/生产者理论)

**白话解释**:  
企业如何在有限资源下实现利润最大化。

**核心概念**:
1. **生产函数** - 投入与产出的关系
2. **边际产量** - 多投入一单位要素带来的额外产出
3. **边际产量递减** - 单一要素增加，边际产量递减

**生活例子**:  
- 一个农民种地：效率高
- 三个农民种地：还可以
- 十个农民挤在一块地：互相妨碍

**成本分类**:
| 成本类型 | 含义 | 例子 |
|----------|------|------|
| 固定成本(FC) | 不随产量变化 | 厂房租金 |
| 可变成本(VC) | 随产量变化 | 原材料 |
| 总成本(TC) | FC + VC | - |
| 边际成本(MC) | 多生产一单位增加的成本 | - |
| 平均成本(AC) | TC/Q | - |

**利润最大化条件**:
```
MR = MC

边际收益 = 边际成本
```

**费曼记忆**:  
边际产量递减，利润最大化时边际收益=边际成本

---

#### 21. 市场结构 (Market Structure/市场结构)

**白话解释**:  
市场竞争程度的不同类型。

**四种市场结构**:
| 类型 | 企业数量 | 产品 | 进入壁垒 | 例子 |
|------|----------|------|----------|------|
| 完全竞争 | 无数 | 同质 | 无 | 农产品 |
| 垄断竞争 | 很多 | 差异化 | 低 | 餐饮、服装 |
| 寡头垄断 | 少数 | 同质或差异 | 高 | 通信、汽车 |
| 完全垄断 | 一个 | 独特 | 极高 | 自来水、电网 |

**完全竞争特点**:
1. 无数买卖双方
2. 产品同质
3. 信息完全
4. 自由进出

**完全垄断特点**:
1. 独家经营
2. 价格制定者
3. 高进入壁垒

**费曼记忆**:  
竞争程度：完全竞争 > 垄断竞争 > 寡头 > 垄断

### 二、宏观经济学

#### 22. 国民收入核算 (National Income Accounting/国民收入核算)

**白话解释**:  
如何统计一个国家的经济总量。

**核心指标**:
1. **GDP** - 国内生产总值
2. **GNP** - 国民生产总值
3. **NDP** - 国内生产净值
4. **NI** - 国民收入

**GDP三种核算方法**:
| 方法 | 公式 | 说明 |
|------|------|------|
| 生产法 | 各部门增加值之和 | 生产角度 |
| 收入法 | 工资+利润+利息+租金+折旧+间接税 | 分配角度 |
| 支出法 | C+I+G+(X-M) | 使用角度 |

**支出法详解**:
```
GDP = C + I + G + (X - M)

C: 消费（居民消费）
I: 投资（企业投资、库存增加）
G: 政府支出
X: 出口
M: 进口
```

**名义GDP vs 实际GDP**:
- 名义GDP：按当年价格计算
- 实际GDP：按基年价格计算（剔除通胀）

**费曼记忆**:  
GDP = 消费 + 投资 + 政府支出 + 净出口

#### 23. 通货膨胀 (Inflation/通货膨胀)

**白话解释**:  
钱不值钱了，物价普遍持续上涨。

**生活例子**:  
- 10年前100元能买一车菜
- 现在100元只能买几斤

**衡量指标**:
```
CPI（消费者物价指数）= （当期一篮子商品价格/基期价格）× 100

通胀率 = （本期CPI - 上期CPI）/ 上期CPI × 100%
```

**类型**:
| 类型 | 通胀率 | 影响 |
|------|--------|------|
| 温和通胀 | 3%以下 | 正常，有利于经济 |
| 中度通胀 | 3%-10% | 需关注 |
| 严重通胀 | 10%-100% | 经济混乱 |
| 恶性通胀 | 100%以上 | 经济崩溃 |

**成因**:
1. **需求拉动** - 需求过多，供不应求
2. **成本推动** - 原材料、工资上涨
3. **货币超发** - 钱印太多了

**费曼记忆**:  
通胀 = 物价普遍持续上涨

---

#### 24. 失业 (Unemployment/失业)

**白话解释**:  
想工作但找不到工作的人。

**失业率**:
```
失业率 = 失业人口 / 劳动力人口 × 100%

劳动力人口 = 就业人口 + 失业人口
```

**失业类型**:
| 类型 | 原因 | 特点 |
|------|------|------|
| 摩擦性失业 | 换工作间隙 | 正常，短期 |
| 结构性失业 | 技能不匹配 | 需要培训 |
| 周期性失业 | 经济衰退 | 需要刺激经济 |
| 季节性失业 | 季节变化 | 如农业、旅游 |

**自然失业率**:  
经济正常运转时的失业率（通常4%-6%）

**奥肯定律**:
```
GDP每增加2%，失业率下降约1%

经济增长 → 就业增加
```

**费曼记忆**:  
失业类型：摩擦、结构、周期、季节

---

#### 25. 财政政策与货币政策 (Fiscal & Monetary Policy)

**白话解释**:  
- 财政政策：政府用税收和支出调节经济
- 货币政策：央行用货币供应量调节经济

**财政政策工具**:
| 工具 | 经济衰退时 | 经济过热时 |
|------|------------|------------|
| 税收 | 减税 | 加税 |
| 政府支出 | 增加支出 | 减少支出 |
| 国债 | 增发国债 | 减少国债 |

**货币政策工具**:
| 工具 | 经济衰退时 | 经济过热时 |
|------|------------|------------|
| 存款准备金率 | 降低 | 提高 |
| 再贴现率 | 降低 | 提高 |
| 公开市场操作 | 买入债券 | 卖出债券 |
| 利率 | 降低 | 提高 |

**政策配合**:
```
经济衰退 → 扩张性财政 + 扩张性货币
经济过热 → 紧缩性财政 + 紧缩性货币
滞胀 → 紧缩性货币 + 扩张性财政
```

**费曼记忆**:  
财政看政府收支，货币看央行利率

#### 26. 经济增长理论 (Economic Growth Theory)

**白话解释**:  
研究经济长期增长的动力和规律。

**增长核算**:
```
Y = A × F(K, L)

Y: 产出
A: 技术进步（全要素生产率）
K: 资本
L: 劳动
```

**索洛增长模型**:
```
人均产出增长 = 技术进步 + 资本深化

技术进步：长期增长的根本动力
资本深化：资本积累带来的增长
```

**增长因素**:
1. **资本积累** - 投资增加
2. **劳动增长** - 人口增加、教育提高
3. **技术进步** - 创新驱动
4. **制度改进** - 市场化、法治化

**中国经济转型**:
```
要素驱动 → 投资驱动 → 创新驱动
（低成本）  （高投资）  （高质量）
```

**费曼记忆**:  
长期增长靠技术进步和制度创新

两种货币之间的兑换比例。

| 标价方法 | 表达 |
|----------|------|
| 直接标价 | 1外币 = ？本币 |
| 间接标价 | 1本币 = ？外币 |

**汇率制度**:
| 制度 | 特点 | 例子 |
|------|------|------|

**汇率影响**:
```
本币贬值 → 出口增加、进口减少
```

**费曼记忆**:  
贬值利好出口，升值利好进口

### 一、人口经济学

#### 31. 人口红利 (Demographic Dividend/人口红利)

**白话解释**:  
劳动年龄人口多，抚养比低，经济增长快。

**人口结构**:
```
        人口金字塔
        老年人口（65岁以上）
        ────────────────
        劳动年龄人口（15-64岁）← 人口红利期
        ────────────────
        少年儿童（0-14岁）
```

**中国人口变化**:
| 时期 | 特点 | 政策 |
|------|------|------|
| 1950-1970 | 高出生率 | - |
| 1970-2015 | 独生子女 | 计划生育 |
| 2016-2020 | 二孩 | 全面二孩 |
| 2021至今 | 三孩 | 三孩政策 |

**人口红利消失的影响**:
1. 劳动力成本上升
2. 养老负担加重
3. 消费结构变化

**费曼记忆**:  
人口红利 = 劳动力多 + 抚养比低

#### 32. 人口老龄化 (Population Aging/人口老龄化)

**白话解释**:  
老年人越来越多，年轻人相对减少。

**衡量标准**:
```
老龄化社会：65岁以上人口 ≥ 7%
深度老龄化：65岁以上人口 ≥ 14%
超级老龄化：65岁以上人口 ≥ 20%
```

**中国现状**:
- 65岁以上人口：2023年约14.3%（深度老龄化）
- 2035年预计超过20%（超级老龄化）

**应对措施**:
1. 延迟退休
2. 发展养老产业
3. 完善社保体系
4. 鼓励生育

**费曼记忆**:  
老龄化 = 老人多 + 年轻人少

### 三、环境经济学

#### 35. 外部性 (Externality/外部性)

**白话解释**:  
你的行为影响别人，但你不需要为此负责或获益。

**类型**:
| 类型 | 影响 | 例子 |
|------|------|------|
| 正外部性 | 他人受益 | 教育、打疫苗 |
| 负外部性 | 他人受损 | 污染、噪音 |

**负外部性问题**:
```
工厂排污
  ↓
社会承担成本（健康、环境）
  ↓
工厂不承担成本 → 生产过多
```

**解决方案**:
1. **庇古税** - 对污染者征税
2. **产权界定** - 科斯定理
3. **可交易许可证** - 碳交易
4. **政府管制** - 排放标准

**费曼记忆**:  
外部性 = 你的行为影响别人，但你不承担后果

#### 36. 碳交易 (Carbon Trading/碳交易)

- 首批纳入发电行业
- 逐步扩展到其他行业

**碳定价方式**:
| 方式 | 特点 |
|------|------|
| 碳税 | 直接给碳定价 |
| 碳交易 | 市场决定碳价 |

**费曼记忆**:  
碳交易 = 排放权商品化，多排多付钱

27. **资源稀缺性** - 资源有限
28. **可持续发展** - 代际公平
29. **外部性** - 行为影响他人但不承担后果

31. **凯恩斯** - 有效需求、政府干预
32. **马克思** - 劳动价值论、剩余价值

## 💡 实务应用场景

### 场景1：分析商品价格变化
- **问题**: 为什么猪肉价格忽高忽低？
- **分析**: 
  - 供给端：猪周期、疫情、养殖成本
  - 需求端：消费习惯、替代品
  - 弹性：猪肉需求缺乏弹性
  - 价值规律：价格围绕价值波动

### 场景2：企业定价策略
- **问题**: 新产品如何定价？
- **分析**:
  - 需求弹性：奢侈品富有弹性
  - 竞争结构：垄断市场定价权高
  - 成本结构：边际成本定价
  - 消费者心理：价格歧视策略

### 场景3：宏观经济形势判断
- **问题**: 如何判断经济形势？
- **分析**:
  - GDP增长率
  - 通胀率（CPI）
  - 失业率
  - 贸易顺差/逆差
  - 政策取向
