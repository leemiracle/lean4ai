### 一、古代农业文明

#### 1. 农业革命 (Agricultural Revolution/农业革命)

**白话解释**:  
人类从采集狩猎转向定居农业，是文明的起点。

**发生时间**:  
约公元前10000年-8000年

**主要内容**:
```
采集狩猎 → 定居农业
    ↓
驯化植物：小麦、水稻、玉米
    ↓
驯化动物：羊、牛、马
    ↓
人口增长、文明诞生
```

**农业革命的意义**:
| 方面 | 变化 |
|------|------|
| 生产方式 | 采集 → 种植 |
| 居住方式 | 游牧 → 定居 |
| 人口规模 | 小 → 大 |
| 社会组织 | 部落 → 国家 |
| 技术进步 | 简单 → 复杂 |

**独立起源中心**:
1. **西亚**: 小麦、大麦
2. **东亚**: 水稻、小米
3. **中美洲**: 玉米
4. **安第斯**: 马铃薯

**费曼记忆**:  
农业革命 = 采集狩猎 → 定居农业

---

#### 2. 古代中国经济 (Ancient Chinese Economy/古代中国经济)

**白话解释**:  
中国古代以农业为主的经济体系。

**基本特征**:
1. **小农经济**: 男耕女织、自给自足
2. **土地私有**: 可以买卖土地
3. **重农抑商**: 重视农业、压制商业
4. **朝贡贸易**: 周边国家进贡

**土地制度演变**:
```
井田制（西周）
├─ 公田 + 私田
└─ 土地不能买卖
    ↓
土地私有制（春秋战国）
├─ 土地可以买卖
└─ 地主佃农关系形成
    ↓
均田制（北魏-唐）
├─ 国家分配土地
└─ 限制土地兼并
    ↓
土地私有制（宋以后）
├─ 土地自由买卖
└─ 地主制发展
```

**古代中国经济成就**:
| 时期 | 成就 |
|------|------|
| 秦汉 | 统一货币、度量衡 |
| 唐宋 | 丝绸之路、商业繁荣 |
| 元明 | 郑和下西洋 |
| 清前期 | 人口突破4亿 |

**费曼记忆**:  
古代中国经济 = 小农经济 + 重农抑商

#### 3. 古代地中海经济

**白话解释**:  
古希腊、罗马以贸易和奴隶制为特色的经济。

**古希腊经济**:
- 城邦经济
- 海上贸易
- 货币经济发达
- 奴隶劳动

**古罗马经济**:
```
罗马经济
├─ 农业（意大利半岛）
├─ 贸易（地中海）
├─ 奴隶制
│   ├─ 战俘奴隶
│   └─ 债务奴隶
└─ 基础设施
    ├─ 道路网
    └─ 输水道
```

**罗马的衰落原因**:
1. 奴隶制效率低下
2. 蛮族入侵
3. 贸易衰退
4. 货币贬值

**费曼记忆**:  
古代地中海经济 = 贸易 + 奴隶制

### 一、欧洲中世纪经济

#### 4. 封建庄园经济 (Feudal Manor Economy/封建庄园经济)

**白话解释**:  
中世纪欧洲以庄园为单位的经济组织。

**基本特征**:
1. **自给自足**: 庄园内部生产所需
2. **农奴制**: 农奴依附土地
3. **等级制度**: 领主-封臣-农奴
4. **自然经济**: 商品交换少

**庄园结构**:
```
庄园
├─ 领主自营地
├─ 农奴份地
├─ 公共地
│   ├─ 草地
│   └─ 森林
└─ 作坊
    ├─ 磨坊
    ├─ 面包房
    └─ 铁匠铺
```

**农奴的义务**:
| 义务 | 内容 |
|------|------|
| 劳役 | 在领主土地上劳动 |
| 实物地租 | 交纳农产品 |
| 货币地租 | 后来出现 |
| 其他 | 使用磨坊费、结婚税等 |

**费曼记忆**:  
庄园经济 = 自给自足 + 农奴制

---

#### 5. 中世纪城市经济

**白话解释**:  
中世纪后期城市兴起，商业和手工业发展。

**城市兴起原因**:
1. 贸易复兴
2. 手工业发展
3. 庄园效率低下
4. 十字军东征刺激贸易

**行会制度**:
```
行会 (Guild)
├─ 手工业行会
│   ├─ 学徒
│   ├─ 帮工
│   └─ 师傅
└─ 商人行会
    └─ 控制贸易
```

**行会的功能**:
| 功能 | 内容 |
|------|------|
| 生产控制 | 限制产量、质量标准 |
| 价格控制 | 统一价格 |
| 培训学徒 | 技术传承 |
| 互助保障 | 会员福利 |

**城市的特点**:
- 相对独立
- 商业发达
- 自由民增多

**费曼记忆**:  
中世纪城市 = 贸易复兴 + 行会制度

### 二、中国封建经济

#### 6. 中国封建地主经济

**白话解释**:  
中国封建社会以地主土地所有制为核心的经济。

**基本特征**:
1. **土地私有**: 土地可以买卖
2. **地主佃农**: 租佃关系
3. **小农经济**: 家庭生产
4. **国家赋税**: 农业税收

**土地占有结构**:
```
土地占有
├─ 皇庄（皇室土地）
├─ 官田（国家土地）
└─ 民田（私人土地）
    ├─ 地主土地（占多数）
    └─ 自耕农土地（占少数）
```

**租佃关系**:
```
地主
├─ 提供土地
└─ 收取地租（通常50%）

佃农
├─ 提供劳动力
├─ 承担地租
└─ 交纳赋税
```

**土地兼并**:
```
自耕农
├─ 天灾
├─ 赋税重
└─ 兼并
    ↓
失去土地 → 成为佃农或流民
```

**费曼记忆**:  
中国封建经济 = 地主土地制 + 租佃关系

### 一、商业革命

#### 7. 商业革命 (Commercial Revolution/商业革命)

**白话解释**:  
15-18世纪欧洲商业的重大变革。

**主要内容**:
1. **地理大发现**: 新航路开辟
2. **贸易扩张**: 世界市场形成
3. **价格革命**: 物价大幅上涨
4. **商业组织创新**: 股份公司兴起

**新航路开辟**:
```
1492年 哥伦布发现美洲
1498年 达·伽马到达印度
1519年 麦哲伦环球航行
```

**价格革命**:
```
1500-1600年欧洲物价
├─ 西班牙：上涨4倍
├─ 英国：上涨2.5倍
└─ 原因：美洲白银流入

影响：
├─ 地主受损（固定收入贬值）
├─ 商人获利（价格上涨）
└─ 工人受损（工资跟不上物价）
```

**股份公司**:
```
荷兰东印度公司 (1602年)
├─ 发行股票
├─ 分散风险
└─ 大规模贸易

英国东印度公司 (1600年)
```

**费曼记忆**:  
商业革命 = 地理大发现 + 世界市场 + 价格革命

---

#### 8. 圈地运动 (Enclosure Movement/圈地运动)

**白话解释**:  
英国地主圈占公地和份地，发展养羊业。

**过程**:
```
15-19世纪
公地、份地
    ↓
被圈占
    ↓
养羊（羊毛纺织需要）
    ↓
农民失去土地
    ↓
成为雇佣工人
```

**影响**:
| 方面 | 影响 |
|------|------|
| 农业 | 资本主义农场兴起 |
| 工业 | 提供劳动力 |
| 城市 | 城市人口增加 |
| 社会 | 农民失去土地 |

**马克思的评价**:
> "资本来到世间，从头到脚，每个毛孔都滴着血和肮脏的东西。"

**费曼记忆**:  
圈地运动 = 羊吃人 = 原始积累

### 二、工业革命

#### 9. 第一次工业革命 (First Industrial Revolution/第一次工业革命)

**白话解释**:  
1760-1840年代，机器取代手工，工厂取代作坊。

**发生地点**: 英国

**核心技术**:
| 技术 | 发明者 | 时间 |
|------|--------|------|
| 珍妮纺纱机 | 哈格里夫斯 | 1765年 |
| 蒸汽机 | 瓦特 | 1769年改良 |
| 机车 | 史蒂芬逊 | 1814年 |

**工业革命进程**:
```
纺织业机械化
├─ 珍妮纺纱机
├─ 水力纺纱机
└─ 蒸汽动力

蒸汽机应用
├─ 纺织业
├─ 采矿业
└─ 交通运输

钢铁冶炼
├─ 焦炭炼铁
└─ 新式炼钢

交通运输
├─ 铁路
└─ 轮船
```

**社会影响**:
1. **生产力大增**: 财富增加
2. **城市化加速**: 人口向城市集中
3. **阶级分化**: 资产阶级 vs 无产阶级
4. **殖民扩张**: 寻求原料和市场

**费曼记忆**:  
第一次工业革命 = 蒸汽机 + 纺织业 + 铁路

---

#### 10. 第二次工业革命 (Second Industrial Revolution/第二次工业革命)

**白话解释**:  
1870-1914年，电力、内燃机、化学的革新。

**核心技术**:
| 领域 | 技术发明 |
|------|----------|
| 电力 | 发电机、电动机、电灯 |
| 内燃机 | 汽车、飞机 |
| 化学 | 合成染料、炸药、化肥 |
| 通信 | 电话、无线电 |
| 钢铁 | 新式炼钢法 |

**特点**:
```
第一次工业革命
├─ 轻工业为主
├─ 技术发明来自工匠
└─ 英国领先

第二次工业革命
├─ 重工业为主
├─ 科学与工业结合
└─ 多国并行
```

**垄断组织**:
```
托拉斯 (Trust)
卡特尔 (Cartel)
辛迪加 (Syndicate)
康采恩 (Konzern)
```

**费曼记忆**:  
第二次工业革命 = 电力 + 内燃机 + 化学

### 三、中国经济近代化

#### 11. 洋务运动 (Self-Strengthening Movement/洋务运动)

**白话解释**:  
1861-1895年，清朝学习西方技术求富求强。

**主要内容**:
1. **军事工业**: 制造枪炮舰船
2. **民用工业**: 轮船、电报、铁路
3. **教育改革**: 新式学堂

**主要企业**:
| 企业 | 性质 | 成立年份 |
|------|------|----------|
| 安庆内军械所 | 军事 | 1861 |
| 江南制造总局 | 军事 | 1865 |
| 轮船招商局 | 民用 | 1872 |
| 开平矿务局 | 民用 | 1878 |
| 汉阳铁厂 | 民用 | 1890 |

**失败原因**:
1. 只学技术不改制度
2. 官僚管理腐败
3. 资金不足
4. 甲午战争战败

**费曼记忆**:  
洋务运动 = 中体西用 = 只学技术不改制度

#### 12. 民族资本主义发展

**白话解释**:  
19世纪末至1949年，中国民办企业艰难发展。

**发展阶段**:
```
初步发展 (1895-1913)
├─ 甲午战争刺激
├─ 设厂自由
└─ 纺织、面粉业为主

短暂春天 (1914-1922)
├─ 一战期间
├─ 列强无暇东顾
└─ 民族工业发展

曲折发展 (1923-1937)
├─ 军阀混战
├─ 1929经济危机影响
└─ 黄金十年短暂繁荣

抗战及内战 (1937-1949)
├─ 抗战破坏
├─ 官僚资本挤压
└─ 通货膨胀崩溃
```

**特点**:
1. **轻工业为主**: 纺织、面粉
2. **分布集中**: 沿海大城市
3. **力量薄弱**: 资金少、技术落后
4. **双重压迫**: 外资 + 官僚资本

**费曼记忆**:  
民族资本主义 = 轻工业为主 + 分布集中 + 发展艰难

### 一、资本主义发展

#### 13. 1929年大萧条 (Great Depression/大萧条)

**白话解释**:  
1929-1933年资本主义世界最严重的经济危机。

**表现**:
```
1929年10月 纽约股市崩盘
        ↓
    经济危机爆发
        ↓
┌───────┴───────┐
│  银行倒闭      │
│  企业破产      │
│  工人失业      │
│  贸易萎缩      │
└───────────────┘
```

**数据**:
| 指标 | 美国 | 德国 | 英国 |
|------|------|------|------|
| 工业生产下降 | 47% | 40% | 23% |
| 失业率 | 25% | 30% | 15% |
| 股市下跌 | 89% | - | - |

**应对**:
| 国家 | 政策 | 代表 |
|------|------|------|
| 美国 | 罗斯福新政 | 凯恩斯主义 |
| 德国 | 纳粹经济政策 | 国家垄断资本主义 |
| 英国 | 放弃金本位 | 货币贬值 |

**历史意义**:
- 证明市场不是万能的
- 凯恩斯革命
- 国家干预经济成为常态

**费曼记忆**:  
大萧条 = 股市崩盘 + 失业率25% + 罗斯福新政

---

#### 14. 战后资本主义黄金时代

**白话解释**:  
1945-1973年，资本主义经济高速增长。

**特征**:
1. **经济增长快**: 年均增长5%
2. **福利国家**: 社会保障完善
3. **政府调节**: 凯恩斯主义
4. **贸易自由化**: 关贸总协定

**布雷顿森林体系**:
```
1944年建立
├─ 美元与黄金挂钩（1盎司=35美元）
├─ 各国货币与美元挂钩
└─ 固定汇率制

1971年崩溃
├─ 美国黄金储备不足
└─ 尼克松停止兑换黄金
```

**滞胀 (1970s)**:
```
经济增长停滞 + 通货膨胀
├─ 石油危机
├─ 凯恩斯主义失灵
└─ 新自由主义兴起
```

**费曼记忆**:  
战后黄金时代 = 高增长 + 福利国家 + 布雷顿森林体系

### 二、社会主义经济发展

#### 15. 苏联计划经济

**白话解释**:  
苏联实行的高度集中的计划经济体制。

**主要内容**:
1. **公有制**: 生产资料公有
2. **计划经济**: 国家统一计划
3. **按劳分配**: 根据劳动分配

**五年计划**:
```
第一个五年计划 (1928-1932)
├─ 优先发展重工业
├─ 农业集体化
└─ 奠定工业基础

第二个五年计划 (1933-1937)
└─ 工业化基本完成
```

**成就与问题**:
| 成就 | 问题 |
|------|------|
| 快速工业化 | 农业滞后 |
| 国防工业强 | 轻工业弱 |
| 就业充分 | 效率低下 |
| 教育普及 | 创新不足 |

**苏联解体原因**:
1. 经济效率低下
2. 体制僵化
3. 军备竞赛负担
4. 政治体制问题

**费曼记忆**:  
苏联模式 = 公有制 + 计划经济 + 重工业优先

4. 全球化机遇

### 一、经济全球化

#### 17. 经济全球化 (Economic Globalization/经济全球化)

**白话解释**:  
世界各国经济联系日益紧密。

**主要表现**:
1. **贸易全球化**: 贸易壁垒降低
2. **金融全球化**: 资本自由流动
3. **生产全球化**: 全球价值链
4. **信息全球化**: 互联网连接

**推动因素**:
```
技术进步
├─ 通信技术
├─ 交通运输
└─ 互联网

制度变革
├─ 关贸总协定 → WTO
├─ 自由贸易协定
└─ 金融自由化

企业战略
├─ 跨国公司
├─ 外包
└─ 全球供应链
```

**全球价值链**:
```
美国：设计、研发
  ↓
中国：制造、组装
  ↓
东南亚：零部件
  ↓
全球销售
```

**逆全球化趋势**:
- 贸易保护主义
- 产业链回流
- 地缘政治冲突

**费曼记忆**:  
全球化 = 贸易 + 金融 + 生产 + 信息全球化

#### 18. 数字经济兴起

**白话解释**:  
以数据为核心生产要素的新经济形态。

**发展历程**:
```
1990年代：互联网兴起
├─ .com泡沫
└─ 门户网站

2000年代：Web 2.0
├─ 社交网络
├─ 电子商务
└─ 移动互联网

2010年代：平台经济
├─ 智能手机普及
├─ 共享经济
└─ 大数据

2020年代：智能化
├─ 人工智能
├─ 数字货币
└─ 元宇宙
```

**数字经济的特征**:
| 特征 | 说明 |
|------|------|
| 数据要素 | 数据成为核心资源 |
| 平台经济 | 平台连接供需 |
| 网络效应 | 用户越多价值越大 |
| 边际成本递减 | 复制成本接近零 |
| 赢者通吃 | 强者愈强 |

**中国数字经济**:
- 规模：2023年约53万亿元
- 占GDP比重：约42%
- 代表企业：阿里、腾讯、字节跳动

**费曼记忆**:  
数字经济 = 数据要素 + 平台经济 + 网络效应
