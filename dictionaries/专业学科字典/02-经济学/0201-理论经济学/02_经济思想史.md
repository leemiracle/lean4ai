### 一、古代经济思想

#### 1. 古希腊经济思想 (Ancient Greek Economic Thought/古希腊经济思想)

**白话解释**:  
古希腊哲学家对财富、贸易、利息的思考。

**代表人物**:
- **柏拉图**: 理想国中的分工思想
- **亚里士多德**: 区分"经济"与"货殖"

**核心观点**:
| 思想家 | 主要观点 | 著作 |
|--------|----------|------|
| 柏拉图 | 社会分工、财产公有 | 《理想国》 |
| 亚里士多德 | 反对高利贷、自然经济 | 《政治学》 |

**亚里士多德的区分**:
```
经济 (Oikonomia)
├─ 管理 household
├─ 满足自然需要
└─ 正当的活动

货殖 (Chrematistics)
├─ 追求货币积累
├─ 超出自然需要
└─ 不正当的活动
```

**费曼记忆**:  
古希腊重视伦理，反对过度追求货币

#### 2. 中国古代经济思想 (Ancient Chinese Economic Thought/中国古代经济思想)

**白话解释**:  
诸子百家对经济问题的看法。

**主要流派**:
| 流派 | 代表人物 | 核心观点 |
|------|----------|----------|
| 儒家 | 孔子、孟子 | 重义轻利、仁政 |
| 法家 | 韩非、商鞅 | 重农抑商、富国强兵 |
| 道家 | 老子、庄子 | 无为而治、小国寡民 |
| 墨家 | 墨子 | 兼爱互利、节用 |

**重要思想**:
1. **重农抑商** - 农业为本，商业为末
2. **轻徭薄赋** - 减轻人民负担
3. **均贫富** - 防止贫富悬殊
4. **节用** - 崇尚节俭

**管仲的经济思想**:
```
"仓廪实而知礼节，衣食足而知荣辱"
├─ 先富民，后教民
├─ 轻重之术（调节市场）
└─ 官山海（盐铁专卖）
```

**费曼记忆**:  
中国古代重农抑商，强调富民、节用

### 二、重商主义

#### 3. 重商主义 (Mercantilism/重商主义)

**白话解释**:  
16-18世纪欧洲的经济学说，认为财富就是金银，要多出口少进口。

**核心观点**:
1. **财富观**: 金银即财富
2. **贸易观**: 贸易顺差是好事
3. **政策观**: 国家干预经济
4. **殖民观**: 殖民地提供原料

**政策主张**:
| 政策 | 内容 | 目的 |
|------|------|------|
| 关税保护 | 高关税限制进口 | 保护本国产业 |
| 出口奖励 | 补贴出口企业 | 增加出口 |
| 殖民扩张 | 建立殖民地 | 获取原料和市场 |
| 航海法案 | 限制外国船只 | 发展本国航运 |

**代表人物**:
- **托马斯·孟**: 《英国得自对外贸易的财富》
- **柯尔贝尔**: 法国财政大臣，重商主义实践者

**局限**:
- 零和博弈思维
- 忽视国内消费
- 导致殖民地压迫

**费曼记忆**:  
重商主义 = 多出口少进口 = 积累金银

---

#### 4. 重农学派 (Physiocracy/重农学派)

**白话解释**:  
18世纪法国的经济学派，认为只有农业才创造财富。

**核心观点**:
1. **自然秩序**: 经济有自然规律
2. **纯产品**: 农业创造的净产品
3. **自由放任**: 政府少干预

**代表人物**:
- **魁奈** (François Quesnay): 《经济表》

**经济表**:
```
土地所有者
    ↙      ↘
生产阶级 ←→ 不生产阶级
(农民)      (工商业者)

财富只在农业中产生
```

**贡献**:
1. 第一个系统性经济理论
2. 强调经济规律
3. 主张自由放任

**局限**:
- 认为只有农业创造价值
- 忽视工业和服务业

**费曼记忆**:  
重农学派 = 只有农业创造纯产品

### 一、英国古典经济学

#### 5. 亚当·斯密 (Adam Smith/亚当·斯密)

**白话解释**:  
经济学鼻祖，1776年发表《国富论》，提出"看不见的手"。

**核心思想**:
1. **劳动价值论**: 劳动是财富的源泉
2. **看不见的手**: 市场自动调节
3. **分工理论**: 分工提高效率
4. **自由贸易**: 反对贸易保护

**看不见的手**:
```
每个人追求自己的利益
        ↓
    通过市场交换
        ↓
    促进社会利益
        ↓
好像有一只看不见的手在指挥
```

**分工的例子**:
```
大头针制造：
一个人做：1天做20根
分工后：10人协作，1天做48000根
        人均4800根，效率提高240倍
```

**《国富论》主要内容**:
| 篇章 | 内容 |
|------|------|
| 第一篇 | 分工、货币、价格 |
| 第二篇 | 资本积累 |
| 第三篇 | 经济史 |
| 第四篇 | 经济学说史 |
| 第五篇 | 财政 |

**费曼记忆**:  
亚当·斯密 = 看不见的手 + 分工 + 自由贸易

---

#### 6. 大卫·李嘉图 (David Ricardo/大卫·李嘉图)

**白话解释**:  
比较优势理论提出者，发展了劳动价值论。

**核心贡献**:
1. **比较优势**: 各国生产相对擅长的产品
2. **劳动价值论**: 价值由劳动时间决定
3. **地租理论**: 级差地租
4. **工资理论**: 工资铁律

**比较优势**:
```
      葡萄酒  毛呢
英国  120小时 100小时
葡萄牙 80小时  90小时

英国：毛呢效率更高（相对优势）
葡萄牙：葡萄酒效率更高（相对优势）

贸易后：两国都受益
```

**工资铁律**:
```
工资 = 工人生存所需的生活资料价值
     ↑
工资高于此 → 人口增加 → 劳动力供给增加 → 工资下降
工资低于此 → 人口减少 → 劳动力供给减少 → 工资上升
```

**费曼记忆**:  
李嘉图 = 比较优势 + 劳动价值论

---

#### 7. 马尔萨斯 (Thomas Malthus/马尔萨斯)

**白话解释**:  
人口论提出者，担心人口增长超过粮食增长。

**人口论**:
```
人口增长：按几何级数 (2, 4, 8, 16...)
粮食增长：按算术级数 (1, 2, 3, 4...)

结果：人口必然超过粮食供给
```

**抑制人口的方式**:
| 类型 | 方式 |
|------|------|
| 积极抑制 | 战争、瘟疫、饥荒 |
| 预防抑制 | 晚婚、节育 |

**贡献**:
- 关注人口与资源的关系
- 引发对可持续发展的思考

**局限**:
- 低估技术进步
- 忽视人口转型

**费曼记忆**:  
马尔萨斯 = 人口增长快于粮食增长

### 二、马克思经济学

#### 8. 马克思 (Karl Marx/马克思)

**白话解释**:  
《资本论》作者，批判资本主义，创立科学社会主义。

**核心理论**:
1. **劳动价值论**: 价值由社会必要劳动时间决定
2. **剩余价值论**: 揭示剥削本质
3. **资本积累**: 资本家不断扩大再生产
4. **经济危机**: 生产过剩的周期性危机

**剩余价值生产**:
```
工作日 = 必要劳动时间 + 剩余劳动时间
         (创造工资)       (创造剩余价值)

例：工作8小时
- 4小时创造工资价值
- 4小时创造剩余价值

剩余价值率 = 4/4 = 100%
```

**《资本论》三卷**:
| 卷 | 内容 |
|----|------|
| 第一卷 | 资本的生产过程 |
| 第二卷 | 资本的流通过程 |
| 第三卷 | 资本主义生产的总过程 |

**历史意义**:
- 揭示资本主义运行规律
- 为社会主义运动提供理论

**费曼记忆**:  
马克思 = 劳动价值 + 剩余价值 + 资本积累 + 经济危机

### 一、边际革命

#### 9. 边际革命 (Marginal Revolution/边际革命)

**白话解释**:  
1870年代，三位经济学家同时提出边际效用理论。

**三位先驱**:
| 经济学家 | 国籍 | 贡献 |
|----------|------|------|
| 杰文斯 | 英国 | 边际效用论 |
| 门格尔 | 奥地利 | 主观价值论 |
| 瓦尔拉斯 | 法国 | 一般均衡论 |

**核心思想**:
```
价值不是由劳动决定
而是由边际效用决定

边际效用 = 多消费一单位商品带来的额外满足

边际效用递减：吃得越多，越不觉得好吃
```

**贡献**:
1. 主观价值论取代客观价值论
2. 边际分析方法
3. 数学化经济学

**费曼记忆**:  
边际革命 = 价值由边际效用决定

---

#### 10. 门格尔 (Carl Menger/门格尔)

**白话解释**:  
奥地利学派创始人，提出主观价值论。

**核心观点**:
1. **主观价值**: 价值因人而异
2. **边际效用**: 最后一个单位的效用
3. **归算理论**: 生产要素价值来自产品价值

**例子**:
```
沙漠中的水 vs 钻石
- 水的边际效用低（很常见）
- 钻石的边际效用高（很稀少）

所以钻石比水贵
```

**奥地利学派特点**:
- 强调个人主义
- 主观价值论
- 反对数学化

**费曼记忆**:  
门格尔 = 主观价值 + 边际效用

---

### 二、马歇尔经济学

#### 11. 马歇尔 (Alfred Marshall/马歇尔)

**白话解释**:  
微观经济学奠基人，供求理论集大成者。

**核心贡献**:
1. **供求分析**: 均衡价格理论
2. **弹性概念**: 价格弹性、收入弹性
3. **消费者剩余**: 消费者获得的额外利益
4. **短期与长期**: 区分时间维度

**供求均衡图**:
```
价格
  ↑
  │     S（供给）
  │      \
  │       \  E（均衡）
  │        \____
  │        /
  │   ____/（需求）
  │  D
  └────────────→ 数量
```

**消费者剩余**:
```
消费者剩余 = 愿意支付的价格 - 实际支付的价格

例：
- 你愿意花50元买一本书
- 实际只花了30元
- 消费者剩余 = 50 - 30 = 20元
```

**费曼记忆**:  
马歇尔 = 供求均衡 + 弹性 + 消费者剩余

### 一、凯恩斯革命

#### 12. 凯恩斯 (John Maynard Keynes/凯恩斯)

**白话解释**:  
宏观经济学创始人，主张政府干预经济。

**历史背景**:
- 1929年大萧条
- 古典经济学无法解释
- 需要新理论

**核心理论**:
1. **有效需求**: 需求决定供给
2. **乘数效应**: 投资的放大作用
3. **流动性偏好**: 人们持有货币的动机
4. **政府干预**: 财政政策和货币政策

**凯恩斯革命要点**:
```
古典经济学：
供给创造需求 (萨伊定律)
├─ 经济自动趋向充分就业
└─ 政府不用干预

凯恩斯经济学：
需求决定供给
├─ 可能存在长期失业
└─ 需要政府干预
```

**有效需求不足的原因**:
| 因素 | 影响 |
|------|------|
| 边际消费倾向递减 | 消费不足 |
| 资本边际效率递减 | 投资不足 |
| 流动性偏好 | 利率不能太低 |

**费曼记忆**:  
凯恩斯 = 有效需求 + 政府干预 + 乘数效应

---

#### 13. 乘数效应 (Multiplier Effect/乘数效应)

**白话解释**:  
一笔投资会带来多倍的经济增长。

**原理**:
```
政府投资100万
    ↓
工人拿到100万工资，消费80万
    ↓
商家拿到80万收入，消费64万
    ↓
...继续下去...
    ↓
总收入 = 100 + 80 + 64 + ... = 500万

乘数 = 500/100 = 5
```

**乘数公式**:
```
乘数 K = 1/(1-MPC) = 1/MPS

MPC: 边际消费倾向（每增加1元消费多少）
MPS: 边际储蓄倾向（每增加1元储蓄多少）

例：MPC = 0.8
乘数 K = 1/(1-0.8) = 5
```

**政策含义**:
- 政府投资可以放大效果
- 适合经济衰退时期

**费曼记忆**:  
乘数效应 = 投资1元创造多倍收入

### 一、货币主义

#### 15. 弗里德曼 (Milton Friedman/弗里德曼)

**白话解释**:  
货币主义代表，反对凯恩斯主义，强调货币供应的重要性。

**核心观点**:
1. **货币数量论**: 通货膨胀是货币现象
2. **自然失业率**: 失业不能太低
3. **单一规则**: 货币供应稳定增长
4. **自由市场**: 反对政府干预

**货币数量论**:
```
MV = PY

M: 货币供应量
V: 货币流通速度（稳定）
P: 价格水平
Y: 产出（长期由生产率决定）

M增加 → P增加 → 通货膨胀
```

**对菲利普斯曲线的批判**:
```
短期：失业与通胀有替代关系
长期：无替代关系（垂直的菲利普斯曲线）

政策含义：
政府不能长期用通胀换就业
```

**费曼记忆**:  
弗里德曼 = 通胀是货币现象 + 自由市场

---

### 二、新制度经济学

#### 16. 科斯 (Ronald Coase/科斯)

**白话解释**:  
新制度经济学创始人，提出交易成本和产权理论。

**核心概念**:
1. **交易成本**: 市场交易的费用
2. **科斯定理**: 产权清晰时市场有效
3. **企业性质**: 企业替代市场

**交易成本**:
```
交易成本包括：
├─ 搜寻成本：找买卖双方
├─ 谈判成本：讨价还价
├─ 签约成本：订立合同
├─ 监督成本：确保履约
└─ 执行成本：强制执行
```

**科斯定理**:
```
如果交易成本为零，
产权初始分配不影响效率

例子：
工厂排污影响居民
├─ 方案1：给工厂排污权
├─ 方案2：给居民清洁权
└─ 只要交易成本为零，结果一样
```

**费曼记忆**:  
科斯 = 交易成本 + 产权 + 科斯定理

---

#### 17. 诺斯 (Douglass North/诺斯)

**白话解释**:  
新经济史学家，研究制度对经济发展的影响。

**核心观点**:
1. **制度重要**: 制度决定经济绩效
2. **路径依赖**: 历史影响现在
3. **制度变迁**: 制度如何演变

**制度类型**:
| 类型 | 内容 | 例子 |
|------|------|------|
| 正式制度 | 法律、规章 | 宪法、公司法 |
| 非正式制度 | 习惯、文化 | 商业惯例、信任 |

**路径依赖**:
```
早期选择 → 后续发展路径
    ↓
难以改变

例子：
QWERTY键盘
├─ 不是最优布局
├─ 但人们已经习惯
└─ 很难换成更好的布局
```

**费曼记忆**:  
诺斯 = 制度重要 + 路径依赖

### 三、行为经济学

#### 18. 行为经济学 (Behavioral Economics/行为经济学)

**白话解释**:  
将心理学引入经济学，研究人们的实际决策行为。

**与传统经济学的区别**:
| 传统经济学 | 行为经济学 |
|------------|------------|
| 理性人 | 有限理性 |
| 追求效用最大化 | 受心理因素影响 |
| 偏好一致 | 偏好不稳定 |

**核心概念**:
1. **有限理性**: 人不能完全理性
2. **损失厌恶**: 损失的痛苦大于获得的快乐
3. **框架效应**: 表述方式影响决策
4. **锚定效应**: 初始信息影响判断

**损失厌恶**:
```
获得100元的快乐 < 失去100元的痛苦

人们更害怕损失
├─ 赚了100元：还行
└─ 亏了100元：很心疼
```

**代表人物**:
- **丹尼尔·卡尼曼**: 诺贝尔经济学奖
- **理查德·塞勒**: 诺贝尔经济学奖

**费曼记忆**:  
行为经济学 = 人不是完全理性的

#### 19. 前景理论 (Prospect Theory/前景理论)

**白话解释**:  
描述人们在风险下的决策行为。

**核心观点**:
1. **参考点**: 决策基于参考点
2. **损失厌恶**: 损失比获得更敏感
3. **风险态度**: 盈利时厌恶风险，亏损时追求风险

**价值函数图**:
```
价值
  ↑
  │     ○ 获得区域（凹函数）
  │    /
  │───/─────── 参考点
  │  /
  │ ○ 损失区域（凸函数，更陡）
  └────────→ 参考点
```

**例子**:
```
方案A：确定获得1000元
方案B：50%获得2000元，50%获得0

多数人选择A（获得时厌恶风险）

方案C：确定损失1000元
方案D：50%损失2000元，50%损失0

多数人选择D（损失时追求风险）
```

**费曼记忆**:  
前景理论 = 参考点 + 损失厌恶 + 风险态度反转

### 四、发展经济学

#### 20. 发展经济学 (Development Economics/发展经济学)

**白话解释**:  
研究发展中国家如何实现经济发展的学科。

**核心问题**:
1. 为什么有些国家贫穷？
2. 如何实现经济起飞？
3. 如何避免中等收入陷阱？

**早期理论**:
| 理论 | 提出者 | 核心观点 |
|------|--------|----------|
| 大推进理论 | 罗森斯坦-罗丹 | 全面投资 |
| 二元经济理论 | 刘易斯 | 劳动力转移 |
| 起飞理论 | 罗斯托 | 经济发展阶段 |

**刘易斯二元经济模型**:
```
传统农业部门
├─ 劳动力过剩
├─ 边际生产率为零
└─ 工资很低

    ↓ 劳动力转移

现代工业部门
├─ 吸收农村劳动力
├─ 利润积累
└─ 扩大再生产
```

**费曼记忆**:  
发展经济学 = 研究穷国如何变富

## 🎯 重点概念速记

### 必背概念（20个）

1. **重商主义** - 多出口少进口积累金银
2. **重农学派** - 只有农业创造纯产品
3. **看不见的手** - 市场自动调节
4. **比较优势** - 各国生产相对擅长的
5. **人口论** - 人口增长快于粮食增长
6. **剩余价值** - 被资本家占有的价值
7. **边际革命** - 价值由边际效用决定
8. **主观价值论** - 价值因人而异
9. **供求均衡** - 供给等于需求
10. **消费者剩余** - 愿付价格减实际价格
11. **有效需求** - 有支付能力的需求
12. **乘数效应** - 投资的放大作用
13. **货币数量论** - 通胀是货币现象
14. **交易成本** - 市场交易的费用
15. **科斯定理** - 产权清晰市场有效
16. **路径依赖** - 历史影响现在
17. **行为经济学** - 人不是完全理性
18. **损失厌恶** - 损失的痛苦更大
19. **前景理论** - 风险下的决策
20. **二元经济** - 传统与现代并存
