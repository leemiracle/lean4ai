## 📊 军工装备概述

### 一、军工装备定义

**军工装备**是指由军工企业研发生产的武器装备，是军队作战能力的物质基础。

**白话解释**: 
军队用的武器装备，都是由军工企业造出来的，包括飞机、坦克、军舰、导弹等，这些就是军工装备。

## 🔀 军民融合

### 一、军民融合内涵

**军民融合**是将军用技术与民用技术融合发展，实现资源共享、优势互补。

**白话解释**: 
军用技术和民用技术互通有无，军队能用民企的技术，民企也能用军队的技术，互相促进。

---

### 二、军民融合模式

#### 2.1 军转民

**典型军转民技术**:

| 军用技术 | 民用转化 | 应用 |
|---------|---------|------|
| **卫星导航** | 北斗导航 | 手机导航、汽车导航 |
| **无人机** | 民用无人机 | 航拍、测绘、植保 |
| **雷达技术** | 气象雷达 | 天气预报 |
| **核技术** | 核电 | 发电 |
| **材料技术** | 碳纤维 | 体育器材、汽车 |

---

#### 2.2 民参军

**民参军典型企业**:

| 企业 | 领域 | 产品 |
|-----|------|------|
| **光威复材** | 材料 | 碳纤维 |
| **高德红外** | 光电 | 红外探测器、导弹 |
| **光启技术** | 材料 | 超材料 |
| **振华科技** | 电子 | 军工元器件 |

---

#### 2.3 军民两用技术

| 技术领域 | 军用 | 民用 |
|---------|------|------|
| **无人机** | 侦察、打击 | 航拍、植保 |
| **卫星导航** | 军事导航 | 民用导航 |
| **通信技术** | 军事通信 | 民用通信 |
| **网络安全** | 网络战 | 企业安全 |

---

### 三、军民融合案例

#### 案例一：北斗导航

```
北斗导航
├─ 起源：军用卫星导航系统
├─ 军用：导弹导航、部队定位
├─ 民用：手机、汽车、测绘
├─ 产业规模：千亿级
└─ 意义：不依赖GPS，战略自主
```

---

#### 案例二：大疆无人机

```
大疆无人机
├─ 起源：民用消费级无人机
├─ 技术：飞行控制、图像传输
├─ 市场：全球民用市场龙头
├─ 军用：部分军用改型
└─ 意义：民用技术领先全球
```

---

#### 案例三：海康威视

```
海康威视
├─ 起源：中电科28所孵化
├─ 技术：军工电子技术
├─ 市场：全球安防第一
├─ 军用：军用安防产品
└─ 意义：军技民用成功案例
```
