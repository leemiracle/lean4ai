# 物理学术语中英文对照表

## 1. 经典力学 (Classical Mechanics)

### 基本概念

| 中文 | 英文 | 定义 |
|------|------|------|
| 质点 | mass point / particle | 忽略大小和形状，仅考虑质量的理想化物体 |
| 参考系 | reference frame | 用于描述物体运动的坐标系 |
| 惯性系 | inertial frame | 牛顿运动定律成立的参考系 |
| 非惯性系 | non-inertial frame | 相对惯性系做加速运动的参考系 |
| 位移 | displacement | 物体位置的变化量，是矢量 |
| 速度 | velocity | 位移对时间的变化率，是矢量 |
| 加速度 | acceleration | 速度对时间的变化率，是矢量 |
| 质量 | mass | 物体惯性大小的量度 |
| 力 | force | 物体间相互作用，改变物体运动状态的原因 |
| 动量 | momentum | 质量与速度的乘积，p = mv |
| 冲量 | impulse | 力与作用时间的乘积，I = Ft |
| 功 | work | 力与位移的标积，W = F·s |
| 功率 | power | 功对时间的变化率，P = dW/dt |
| 能量 | energy | 物体做功本领的量度 |
| 动能 | kinetic energy | 物体因运动而具有的能量，E_k = ½mv² |
| 势能 | potential energy | 物体因位置而具有的能量 |
| 机械能 | mechanical energy | 动能与势能的总和 |
| 转动惯量 | moment of inertia | 刚体转动惯性大小的量度，I = ∫r²dm |
| 角动量 | angular momentum | 描述物体转动状态的物理量，L = r × p |
| 力矩 | torque | 力使物体转动的效应，τ = r × F |
| 角速度 | angular velocity | 角位移对时间的变化率，ω = dθ/dt |
| 角加速度 | angular acceleration | 角速度对时间的变化率，α = dω/dt |

### 运动定律

| 中文 | 英文 | 定义 |
|------|------|------|
| 牛顿第一定律 | Newton's first law | 物体不受外力时保持静止或匀速直线运动 |
| 牛顿第二定律 | Newton's second law | F = ma，力等于质量乘加速度 |
| 牛顿第三定律 | Newton's third law | 作用力与反作用力大小相等、方向相反 |
| 万有引力定律 | law of universal gravitation | F = Gm₁m₂/r² |
| 胡克定律 | Hooke's law | 弹性限度内，F = -kx |
| 开普勒定律 | Kepler's laws | 行星运动三定律 |
| 动量守恒定律 | conservation of momentum | 系统不受外力时总动量保持不变 |
| 能量守恒定律 | conservation of energy | 能量不能创生或消灭，只能转移或转化 |
| 角动量守恒 | conservation of angular momentum | 系统不受外力矩时总角动量保持不变 |
| 机械能守恒 | conservation of mechanical energy | 只有保守力做功时机械能守恒 |

### 振动与波

| 中文 | 英文 | 定义 |
|------|------|------|
| 简谐振动 | simple harmonic motion | 恢复力与位移成正比的振动，x = Acos(ωt + φ) |
| 振幅 | amplitude | 振动物体离开平衡位置的最大距离 |
| 周期 | period | 完成一次完整振动所需时间 |
| 频率 | frequency | 单位时间内完成振动的次数 |
| 相位 | phase | 描述振动状态的量，φ = ωt + φ₀ |
| 阻尼振动 | damped vibration | 振幅随时间减小的振动 |
| 受迫振动 | forced vibration | 在周期性外力作用下的振动 |
| 共振 | resonance | 驱动力频率等于固有频率时振幅最大 |
| 波 | wave | 振动在介质中的传播 |
| 波长 | wavelength | 相邻两个同相点间的距离 |
| 波速 | wave velocity | 波传播的速度，v = λf |
| 横波 | transverse wave | 振动方向与传播方向垂直的波 |
| 纵波 | longitudinal wave | 振动方向与传播方向平行的波 |
| 干涉 | interference | 两列波叠加产生的强弱分布现象 |
| 衍射 | diffraction | 波绕过障碍物继续传播的现象 |
| 驻波 | standing wave | 两列振幅相同的相干波叠加形成的波 |
