### 一、影视的基本概念

#### 1. 电影 (Film/Cinema/电影)

**白话解释**:  
通过摄影机记录影像、放映机投射到银幕上的艺术。

**生活例子**:  
- 《阿凡达》 → 科幻片
- 《泰坦尼克号》 → 爱情片
- 《霸王别姬》 → 文艺片
- 《流浪地球》 → 科幻片

**学术定义**:  
电影是通过摄影机记录活动影像，经过剪辑、配音等技术处理，通过放映机投射到银幕上，配合声音，供观众欣赏的综合艺术。

**关键要点**:
1. **技术性** - 依赖摄影技术
2. **综合性** - 多种艺术综合
3. **商业性** - 商业产业
4. **大众性** - 面向大众
5. **现代性** - 现代艺术形式

**费曼记忆**:  
电影 = 摄影 + 剪辑 + 声音 + 银幕

---

#### 2. 电视 (Television/电视)

**白话解释**:  
通过电子技术传输图像和声音的媒介。

**电视节目类型**:
1. **电视剧**: 连续剧、系列剧
2. **综艺节目**: 娱乐节目
3. **新闻节目**: 新闻报道
4. **纪录片**: 记录片
5. **体育节目**: 体育赛事

**与电影区别**:
| 电影 | 电视 |
|------|------|
| 银幕大 | 屏幕小 |
| 影院观看 | 家庭观看 |
| 一次性观看 | 可重复播放 |
| 制作精良 | 制作相对简单 |
| 90-180分钟 | 时长灵活 |

**电视艺术特点**:
- **即时性**: 可以直播
- **日常性**: 日常生活伴侣
- **互动性**: 观众参与
- **系列性**: 系列节目

---

#### 3. 影视语言 (Film Language/影视语言)

**白话解释**:  
影视用来表达意义的特殊"语言"系统。

**基本元素**:

1. **镜头 (Shot)**:
   - 最小单位
   - 从开拍到停拍

2. **场景 (Scene)**:
   - 由若干镜头组成
   - 同一地点、时间

   - 由若干场景组成
   - 完整叙事单元
