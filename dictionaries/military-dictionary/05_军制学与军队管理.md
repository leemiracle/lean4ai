## 📖 第一章：军制学基础

### 一、军制基本概念

#### 1. 军制 (Military System/军制)

**白话解释**:  
军队的组织制度和编制，就是"军队怎么组织"。

**内容**:
1. **组织编制**: 军队结构
2. **领导体制**: 领导关系
3. **管理制度**: 管理规则
4. **保障制度**: 后勤保障

---

#### 2. 军队组织结构

**中国军队组织体系**:

```
中央军委
├─ 军委机关（15个职能部门）
│   ├─ 联合参谋部
│   ├─ 政治工作部
│   ├─ 后勤保障部
│   ├─ 装备发展部
│   └─ ...
├─ 战区（5个）
│   ├─ 东部战区
│   ├─ 南部战区
│   ├─ 西部战区
│   ├─ 北部战区
│   └─ 中部战区
├─ 军兵种
│   ├─ 陆军
│   ├─ 海军
│   ├─ 空军
│   ├─ 火箭军
│   ├─ 战略支援部队
│   └─ 联勤保障部队
└─ 武警部队
```

---

#### 3. 军种 (Service/军种)

**白话解释**:  
军队的大类分工。

**中国军种**:

| 军种 | 主要任务 | 主要装备 |
|------|----------|----------|
| 陆军 | 地面作战 | 坦克、火炮、直升机 |
| 海军 | 海上作战 | 航母、驱逐舰、潜艇 |
| 空军 | 空中作战 | 战斗机、轰炸机、运输机 |
| 火箭军 | 战略打击 | 战略导弹 |
| 战略支援部队 | 信息作战 | 卫星、网络 |
| 联勤保障部队 | 后勤保障 | 运输、医疗、维修 |

---

#### 4. 兵种 (Branch/兵种)

**白话解释**:  
军种内部的专业分工。

**陆军兵种**:
1. **步兵**: 徒步作战
2. **装甲兵**: 坦克装甲车辆
3. **炮兵**: 火炮火箭炮
4. **防空兵**: 防空导弹
5. **陆航**: 直升机
6. **工程兵**: 工程作业
7. **通信兵**: 通信保障
8. **防化兵**: 核生化防护

**海军兵种**:
1. **水面舰艇部队**: 水面舰艇
2. **潜艇部队**: 潜艇
3. **海军航空兵**: 海军飞机
4. **海军陆战队**: 两栖作战
5. **岸防部队**: 岸防导弹

**空军兵种**:
1. **航空兵**: 飞机
2. **地空导弹兵**: 防空导弹
3. **雷达兵**: 雷达探测
4. **空降兵**: 空降作战

---

#### 5. 军衔 (Military Rank/军衔)

**白话解释**:  
军人的等级称号。

**中国军衔（军官）**:

| 等级 | 军衔 | 对应职务 |
|------|------|----------|
| 将官 | 上将 | 战区司令、军种司令 |
| 将官 | 中将 | 副战区职 |
| 将官 | 少将 | 正军职、副军职 |
| 校官 | 大校 | 正师职 |
| 校官 | 上校 | 副师职、正团职 |
| 校官 | 中校 | 副团职 |
| 校官 | 少校 | 正营职 |
| 尉官 | 上尉 | 副营职 |
| 尉官 | 中尉 | 正连职 |
| 尉官 | 少尉 | 副连职 |

**士兵军衔**:
1. **士官**: 一级军士长到下士
2. **义务兵**: 上等兵、列兵
