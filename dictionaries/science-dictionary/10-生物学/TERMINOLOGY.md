# 生物学术语表 (Biology Terminology)

---

## 1. 细胞生物学 (Cell Biology)

| 中文 | 英文 | 定义 |
|------|------|------|
| 细胞 | Cell | 生命的基本结构和功能单位 |
| 原核细胞 | Prokaryotic Cell | 无细胞核和膜结合细胞器的细胞 |
| 真核细胞 | Eukaryotic Cell | 具有细胞核和膜结合细胞器的细胞 |
| 细胞膜 | Cell Membrane | 包围细胞的磷脂双分子层结构 |
| 细胞核 | Nucleus | 含有遗传信息的细胞器，真核细胞的控制中心 |
| 核仁 | Nucleolus | 细胞核内合成核糖体的结构 |
| 细胞质 | Cytoplasm | 细胞膜与细胞核之间的半流体物质 |
| 细胞器 | Organelle | 细胞内具有特定功能的膜结合结构 |
| 线粒体 | Mitochondrion | 进行有氧呼吸和产生ATP的细胞器 |
| 叶绿体 | Chloroplast | 进行光合作用的植物细胞器 |
| 内质网 | Endoplasmic Reticulum | 参与蛋白质和脂质合成的膜系统 |
| 高尔基体 | Golgi Apparatus | 加工、分类和包装蛋白质的细胞器 |
| 核糖体 | Ribosome | 合成蛋白质的细胞器 |
| 溶酶体 | Lysosome | 含有消化酶的细胞器，分解废物 |
| 液泡 | Vacuole | 储存物质的大型囊泡 |
| 中心体 | Centrosome | 含有中心粒，参与细胞分裂 |
| 细胞骨架 | Cytoskeleton | 维持细胞形态的蛋白质纤维网络 |
| 微管 | Microtubule | 细胞骨架成分，参与物质运输和细胞分裂 |
| 微丝 | Microfilament | 细胞骨架成分，主要由肌动蛋白组成 |
| 中间纤维 | Intermediate Filament | 细胞骨架成分，提供机械强度 |
| 细胞周期 | Cell Cycle | 细胞从一次分裂到下一次分裂的过程 |
| 有丝分裂 | Mitosis | 产生两个相同子细胞的核分裂过程 |
| 减数分裂 | Meiosis | 产生配子的分裂过程，染色体减半 |
| 细胞分化 | Cell Differentiation | 细胞获得特定功能的过程 |
| 细胞凋亡 | Apoptosis | 程序性细胞死亡 |
| 细胞坏死 | Necrosis | 非程序性的细胞死亡 |
| 胞吞作用 | Endocytosis | 细胞摄取外部物质的过程 |
| 胞吐作用 | Exocytosis | 细胞分泌物质到外部的过程 |
| 主动运输 | Active Transport | 消耗能量逆浓度梯度运输物质 |
| 被动运输 | Passive Transport | 不消耗能量顺浓度梯度运输物质 |

| 表观遗传学 | Epigenetics | 不改变DNA序列的遗传变化 |
