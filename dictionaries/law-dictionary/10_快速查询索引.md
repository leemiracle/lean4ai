## 📑 五、法律流程图解

### 1. 民事诉讼流程

```
起诉 → 立案 → 送达 → 开庭 → 判决 → 上诉(可选) → 生效 → 执行
  ↓                        ↓
不予受理              驳回起诉
```

### 2. 行政复议/诉讼流程

```
行政行为 → 不服 → 复议(60日内) → 不服 → 诉讼(15日内)
                 ↓                      ↓
              维持/撤销              判决
```

### 3. 刑事诉讼流程

```
立案 → 侦查 → 审查起诉 → 审判 → 判决 → 执行
       ↓         ↓         ↓
   强制措施   起诉/不起诉  上诉/抗诉
```

### 4. 劳动争议解决流程

```
协商 → 调解 → 仲裁(必经) → 诉讼
                ↓
           裁决
```
