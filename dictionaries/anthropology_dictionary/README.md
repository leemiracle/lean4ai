## 🌍 人类学的独特视角

**整体观 (Holism)**：理解人类必须考虑生物、文化、语言、历史的所有方面

**比较视角 (Comparative Method)**：通过跨文化比较发现人类社会的共性与差异

**田野工作 (Fieldwork)**：长期深入参与观察是人类学的核心方法

**文化相对主义 (Cultural Relativism)**：理解文化需从其自身逻辑出发

**反身性 (Reflexivity)**：研究者要反思自己的立场和偏见
