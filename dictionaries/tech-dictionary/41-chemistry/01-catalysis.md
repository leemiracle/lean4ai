## 多相催化

**英文名称**：Heterogeneous Catalysis

**定义**：催化剂为固态，反应物为气相或液相，在催化剂表面发生化学反应

**催化剂组成**：
- **活性组分**：金属（Pt、Pd、Ni）、金属氧化物（V₂O₅、MoO₃）、沸石（Y型沸石）
- **载体（Support）**：γ-Al₂O₃（比表面积200 m²/g）、SiO₂（惰性）、活性炭
- **助剂（Promoter）**：提高活性/选择性（如Pt-Re催化重整）

**关键参数**：
- **比表面积（SSA）**：>100 m²/g（沸石>500 m²/g）
- **孔径分布**：微孔（<2 nm）、介孔（2-50 nm）、大孔（>50 nm）
- **金属分散度（Dispersion）**：>50%（金属颗粒<2 nm）
- **活性**：TOF（转换频率，Turnover Frequency），s⁻¹
- **选择性（Selectivity）**：目标产物摩尔分数/总产物

**反应动力学**：
Langmuir-Hinshelwood（LH）机制：
1. 反应物A吸附：A(g) + * → A*（吸附）
2. 反应物B吸附：B(g) + * → B*
3. 表面反应：A* + B* → C* + D*
4. 产物脱附：C* → C(g) + *
   - D* → D(g) + *

费曼解释：
多相催化像"固体催化剂表面搭台"。气相/液相反应物像"演员"，催化剂表面像"舞台"，演员上台（吸附），在舞台上表演（表面反应），下台（脱附）。催化剂不消耗（像舞台重复用），但能加速反应（降低活化能，像"催化剂给演员加道具"，更容易表演）。

**工业应用**：
1. **合成氨**：Fe催化剂（Fe₃O₄ + K₂O + Al₂O₃ + CaO），400-500℃、15-30 MPa（Haber-Bosch）
2. **催化裂化（FCC）**：Y型沸石（Si/Al比5-50），480-520℃、0.1-0.3 MPa
3. **催化重整**：Pt-Re/Al₂O₃，480-520℃、0.5-2.5 MPa，生产高辛烷值汽油
4. **氧化反应**：
   - SO₂→SO₃（V₂O₅催化剂，硫酸生产）
   - NH₃→NO（Pt-Rh催化剂，HNO₃生产）
5. **加氢反应**：
   - CO+3H₂→CH₄+H₂O（Ni/Al₂O₃，甲烷化）
   - C₆H₆+3H₂→C₆H₁₂（Pt/Al₂O₃，环己烷）

**市场数据**：
- **2023**：全球催化剂市场规模$35B（炼油催化剂占40%、化工催化剂占30%）
- **2026**：预计$50B（环保催化剂增长10%）
- **主要厂商**：BASF（德国）、Johnson Matthey（英国）、Clariant（瑞士）、中国石化催化剂公司

**趋势（2026）**：
1. **单原子催化（Single-Atom Catalysis，SAC）**：金属原子分散在载体上（100%金属利用），降低成本
2. **沸石催化**：多级孔沸石（微孔+介孔），提高大分子转化
3. **光催化**：TiO₂光催化（UV光），分解有机物、制氢
4. **电催化**：CO₂电催化还原为CO、CH₄（可再生能源+电催化）
