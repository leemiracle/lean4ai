# 定量研究

## 概述
定量研究是通过数值化数据收集和分析,验证假设、测量变量、发现规律的研究方法。它关注"多少"、"程度"和"关系",强调客观性和可重复性。

## 核心特征

### 1. 客观性强
- 标准化流程
- 数值化测量
- 可重复验证

### 2. 样本量大
- 通常100人以上
- 代表性取样
- 统计推断

### 3. 数据形式
- 数值数据
- 统计图表
- 数学模型

## 主要方法

### 1. 问卷调查
- 大规模数据收集
- 结构化问题
- 统计分析

### 2. 实验研究
- 控制变量
- 因果推断
- A/B测试

### 3. 观察统计
- 行为计数
- 时间测量
- 频率分析

### 4. 二手数据分析
- 历史数据
- 公开数据集
- Meta分析

### 5. 纵向研究
- 时间序列
- 面板数据
- 趋势分析

## 研究设计

### 1. 描述性研究
```
目的: 描述现状和特征
方法: 问卷调查、观察
输出: 统计描述、分布图
```

### 2. 相关性研究
```
目的: 发现变量关系
方法: 相关分析、回归
输出: 相关系数、回归方程
```

### 3. 因果性研究
```
目的: 验证因果关系
方法: 实验设计、准实验
输出: 效应量、显著性
```

## 统计分析方法

### 1. 描述统计
- 集中趋势: 均值、中位数、众数
- 离散程度: 标准差、方差、极差
- 分布形态: 偏度、峰度

### 2. 推断统计
```
参数检验:
- t检验 (两组比较)
- 方差分析ANOVA (多组比较)
- 卡方检验 (分类变量)

非参数检验:
- Mann-Whitney U
- Kruskal-Wallis
- Wilcoxon符号秩
```

### 3. 相关分析
- Pearson相关
- Spearman相关
- 偏相关
- 典型相关

### 4. 回归分析
- 线性回归
- 逻辑回归
- 多元回归
- 层次回归

### 5. 高级方法
- 因子分析
- 聚类分析
- 结构方程模型SEM
- 多层线性模型HLM

## 样本量计算

### 公式
```
简单随机抽样:
n = (Z² × p × (1-p)) / e²

其中:
n = 样本量
Z = 置信水平Z值
p = 预期比例
e = 允许误差
```

### 常用标准
- 探索性研究: 30-100
- 因子分析: 5-10倍题目数
- 回归分析: 10-20倍自变量数
- 结构方程: 200-500

## 信效度检验

### 信度 (Reliability)
- Cronbach's α > 0.7
- 分半信度
- 重测信度

### 效度 (Validity)
- 内容效度
- 构念效度
- 效标效度

## 案例分析

### 案例: 电商用户满意度调研

**研究目标**
量化评估用户满意度及其影响因素

**研究设计**
- 样本: N=1000, 分层随机抽样
- 工具: 26项Likert量表问卷
- 方法: 结构方程模型

**关键指标**
```
整体满意度: 3.85/5
推荐意愿NPS: 42

影响因素权重:
- 商品质量 (β=0.42)
- 价格合理性 (β=0.28)
- 物流速度 (β=0.18)
- 客服质量 (β=0.12)
```

**实践建议**
1. 优先提升商品质量管控
2. 优化价格策略
3. 缩短物流时长
4. 加强客服培训

## 最佳实践

### 1. 问卷设计
- 题目简洁明确
- 避免引导性
- 合理长度(10-15分钟)
- 反向题检验

### 2. 抽样策略
- 明确总体范围
- 保证代表性
- 提高回收率
- 处理无应答

### 3. 数据质量
- 注意力检查题
- 一致性检验
- 异常值处理
- 缺失值分析

### 4. 结果解读
- 统计显著≠实际重要
- 相关≠因果
- 样本代表性限制
- 多重比较校正

## 常见误区

### ❌ 错误认知
- P值越小越重要
- 样本越大越好
- 统计能证明一切
- 相关就是因果

### ✅ 正确理解
- 考虑效应量大小
- 足够的统计功效即可
- 统计只是工具
- 需要理论支持因果
