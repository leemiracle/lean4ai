# 问卷数据分析

## 概述
问卷数据分析是将收集的问卷数据转化为有意义的洞察的过程,包括数据清洗、统计分析、结果解读和报告撰写等环节。

## 分析流程

```
数据收集
   ↓
数据清洗
   ↓
描述统计
   ↓
推断统计
   ↓
结果解读
   ↓
报告呈现
```

## 数据准备与清洗

### 1. 数据导入与检查

#### 导入数据
```
常用格式:
- Excel (.xlsx, .xls)
- CSV (.csv)
- SPSS (.sav)
- 问卷平台导出

工具:
- SPSS: 文件 → 打开 → 数据
- Excel: 数据 → 获取数据
- Python: pd.read_csv()
- R: read.csv()
```

#### 初步检查
```
□ 样本量是否符合预期?
□ 变量是否完整?
□ 数据类型是否正确?
□ 缺失值情况?
□ 异常值情况?
```

### 2. 数据清洗

#### 剔除无效样本
```
剔除标准:
1. 完成时间过短
   - 如: 50题问卷3分钟完成
   - 标准差法: < 平均值-2SD
   
2. 注意力检查题答错
   - 预设正确答案的题
   
3. 答案模式异常
   - 全选同一选项
   - 规律性作答(如A,B,C,D循环)
   
4. 逻辑矛盾
   - 年龄18岁,工作年限10年
   - 使用频率"从不",满意度"非常满意"
```

#### 处理缺失值
```
缺失类型:
- 完全随机缺失(MCAR)
- 随机缺失(MAR)
- 非随机缺失(MNAR)

处理方法:
1. 列表删除: 删除有缺失的整条记录
   → 样本量减少
   
2. 配对删除: 分析时只用有数据的记录
   → 不同分析样本量不同
   
3. 均值替代: 用变量均值填充
   → 方差低估
   
4. 回归插补: 根据其他变量预测
   → 需要模型假设
   
5. 多重插补: 多次插补,合并结果
   → 最优但复杂
```

#### 处理异常值
```
识别方法:
1. 箱线图: < Q1-1.5IQR 或 > Q3+1.5IQR
2. Z分数: |Z| > 3
3. 百分位: < P1 或 > P99

处理方法:
1. 核实数据(是否录入错误)
2. 删除(确认是错误)
3. 保留(是真实极端值)
4. 截断(Winsorize)
5. 转换(如对数转换)
```

#### 数据转换
```
1. 反向题重新计分
   原始: 1-2-3-4-5
   转换: 6-原始分
   
2. 变量重编码
   年龄: 连续 → 分组
   18-25, 26-35, 36-45, 46+
   
3. 计算新变量
   平均满意度 = (满意度1 + 满意度2 + ...)/n
```

### 3. 数据质量检查

#### 信度分析
```
Cronbach's α系数:
- α ≥ 0.9: 优秀
- 0.8 ≤ α < 0.9: 良好
- 0.7 ≤ α < 0.8: 可接受
- 0.6 ≤ α < 0.7: 勉强接受
- α < 0.6: 不可接受

SPSS操作:
分析 → 量表 → 可靠性分析
```

#### 效度分析
```
内容效度:
- 专家评审
- 内容覆盖率

构念效度:
- 探索性因子分析(EFA)
- 验证性因子分析(CFA)

KMO和Bartlett检验:
- KMO > 0.7: 适合因子分析
- Bartlett显著性p<0.05
```

## 描述统计

### 1. 单变量分析

#### 分类变量
```
统计量:
- 频次(Frequency)
- 百分比(Percentage)
- 有效百分比
- 累计百分比

可视化:
- 柱状图
- 饼图
```

示例:
```
Q: 您的性别是?

性别    频次   百分比
男      450    45%
女      540    54%
其他     10     1%
合计   1000   100%
```

#### 连续变量
```
集中趋势:
- 均值(Mean): 平均水平
- 中位数(Median): 中间值
- 众数(Mode): 最常见值

离散程度:
- 标准差(SD): 离散程度
- 方差(Variance)
- 极差(Range): 最大-最小
- 四分位距(IQR): Q3-Q1

分布形态:
- 偏度(Skewness): 对称性
- 峰度(Kurtosis): 尖峭性

可视化:
- 直方图
- 箱线图
```

示例:
```
Q: 满意度评分(1-10分)

N     = 1000
Mean  = 7.52
SD    = 1.83
Median= 8.00
Min   = 1
Max   = 10
```

#### 分类 × 分类
```
方法: 交叉表(Crosstab)
可视化: 堆叠柱状图、分组柱状图

示例:
          男    女   合计
满意      300   380   680
不满意    150   160   310
中立       50    60   110
合计      500   600  1100
```

#### 分类 × 连续
```
方法: 分组描述统计
可视化: 分组箱线图、均值图

示例:
      性别   N    Mean    SD
满意度 男    500  7.35   1.92
      女    600  7.66   1.75
```

#### 连续 × 连续
```
方法: 相关分析
可视化: 散点图

示例:
          满意度  推荐意愿
满意度    1.00    0.72
推荐意愿  0.72    1.00
```

## 推断统计

### 1. 差异检验

#### 两组比较: t检验
```
独立样本t检验:
- 比较两组独立样本均值
- 例: 男女满意度差异

前提假设:
- 正态分布
- 方差齐性

SPSS操作:
分析 → 比较均值 → 独立样本T检验

结果解读:
- Levene方差齐性检验
  - p>0.05: 方差齐,读第一行
  - p≤0.05: 方差不齐,读第二行
- t检验显著性
  - p<0.05: 差异显著
```

示例结果:
```
Levene检验 F=3.21, p=0.073 → 方差齐
t检验 t=2.15, df=998, p=0.032 → 差异显著

结论: 男女性满意度存在显著差异(p<0.05),
     女性满意度(M=7.66)显著高于男性(M=7.35)
```

#### 多组比较: 方差分析(ANOVA)
```
单因素方差分析:
- 比较三组及以上均值
- 例: 不同年龄组满意度差异

前提假设:
- 正态分布
- 方差齐性
- 独立性

SPSS操作:
分析 → 比较均值 → 单因素ANOVA

结果解读:
1. F检验: 总体差异
2. 事后检验: 具体哪两组不同
   - LSD: 灵敏但易误
   - Bonferroni: 保守校正
   - Tukey HSD: 常用平衡
```

示例结果:
```
ANOVA结果: F=8.52, p<0.001 → 至少两组差异显著

事后检验(Bonferroni):
18-25 vs 26-35: p=0.012 *
18-25 vs 36-45: p=0.001 **
26-35 vs 36-45: p=0.456

结论: 36-45岁组满意度显著高于18-25岁组
```

#### 非参数检验
```
适用场景:
- 数据非正态分布
- 等级数据
- 样本量小

方法:
- Mann-Whitney U: 两组比较(对应t检验)
- Kruskal-Wallis: 多组比较(对应ANOVA)
- Wilcoxon符号秩: 配对样本
```

### 2. 关系检验

#### 相关分析
```
Pearson相关:
- 连续变量
- 线性关系
- 正态分布

Spearman相关:
- 等级变量
- 非线性单调关系
- 非正态分布

相关系数解读:
| r | = 0.00-0.19: 很弱
| r | = 0.20-0.39: 弱
| r | = 0.40-0.59: 中等
| r | = 0.60-0.79: 强
| r | = 0.80-1.00: 很强

SPSS操作:
分析 → 相关 → 双变量
```

示例:
```
满意度与推荐意愿的相关

Pearson r = 0.72, p<0.001

结论: 满意度与推荐意愿呈强正相关(r=0.72, p<0.001)
```

#### 卡方检验
```
适用场景:
- 两个分类变量
- 检验独立性

χ² = Σ [(观测值-期望值)² / 期望值]

前提条件:
- 期望频次≥5
- 样本独立

SPSS操作:
分析 → 描述统计 → 交叉表 → 统计量 → 卡方
```

示例:
```
性别与满意度的关联

χ² = 12.35, df = 2, p = 0.002

结论: 性别与满意度存在显著关联(p<0.01)
```

### 3. 回归分析

#### 线性回归
```
用途:
- 连续因变量
- 预测和解释

方程: Y = b0 + b1X1 + b2X2 + ... + e

评估指标:
- R²: 解释方差比例
- 调整R²: 校正后解释力
- F检验: 模型整体显著性
- t检验: 系数显著性

前提假设:
- 线性关系
- 误差正态分布
- 同方差性
- 无多重共线性
- 无自相关

SPSS操作:
分析 → 回归 → 线性
```

示例结果:
```
因变量: 满意度
自变量: 功能性、易用性、美观性

模型汇总: R² = 0.45, 调整R² = 0.44
ANOVA: F = 267.5, p < 0.001

系数:
         B      β      p
功能性  0.35   0.32  <0.001  ***
易用性  0.42   0.38  <0.001  ***
美观性  0.18   0.16  <0.001  ***

结论: 功能性、易用性、美观性均显著影响满意度,
     易用性影响最大(β=0.38)
```

#### 逻辑回归
```
用途:
- 二分类因变量(是/否)
- 预测概率

方程: log(p/(1-p)) = b0 + b1X1 + ...

评估指标:
- Cox & Snell R²
- Nagelkerke R²
- 分类正确率
- ROC曲线/AUC

SPSS操作:
分析 → 回归 → 二元Logistic
```

示例:
```
因变量: 是否推荐(0/1)
自变量: 满意度

模型: χ² = 312.5, p < 0.001
Nagelkerke R² = 0.52
分类正确率 = 78.5%

系数:
        B     OR    95%CI       p
满意度 0.85   2.34  2.01-2.72  <0.001

结论: 满意度每提高1分,推荐概率增加2.34倍
```

### 4. 因子分析

#### 探索性因子分析(EFA)
```
用途:
- 降维
- 发现潜在结构

步骤:
1. 适用性检验
   - KMO > 0.7
   - Bartlett p < 0.05

2. 因子提取
   - 主成分分析(PCA)
   - 主轴因子法(FA)

3. 因子数量确定
   - 特征值>1
   - 碎石图

4. 因子旋转
   - 正交旋转(方差最大)
   - 斜交旋转(-promax)

SPSS操作:
分析 → 降维 → 因子分析
```

示例:
```
20个满意度题目的因子分析

KMO = 0.89 → 适合
Bartlett χ² = 3521.6, p < 0.001 → 适合

提取3个因子,累计解释方差62.5%

因子载荷矩阵:
            因子1  因子2  因子3
功能1        0.78   0.21   0.15
功能2        0.82   0.18   0.12
...
易用1        0.23   0.75   0.20
易用2        0.19   0.80   0.16
...
美观1        0.15   0.18   0.77
美观2        0.21   0.15   0.81
...

命名:
因子1: 功能性
因子2: 易用性
因子3: 美观性
```

## 常见错误

### ❌ 统计错误
- 忽视前提假设检验
- 多重比较未校正
- 混淆相关与因果
- 过度解读小样本
- p值滥用

### ❌ 解释错误
- 统计显著≠实际重要
- 相关≠因果
- 样本≠总体
- 平均值陷阱

### ❌ 呈现错误
- 图表误导
- 数据选择性地呈现
- 过度概括
- 忽视不确定性

## 案例分析

### 案例: 用户满意度调研分析

**数据概况**
- 有效样本: 1052份
- 变量: 26题(5点Likert量表)

**分析过程**

1. **数据清洗**
   - 剔除答题时间<3分钟(42份)
   - 剔除注意力题错误(28份)
   - 最终有效: 1052份(92.6%)

2. **信效度检验**
   - 总体Cronbach's α = 0.91(优秀)
   - KMO = 0.88,适合因子分析

3. **因子分析**
   - 提取5个因子,累计方差65%
   - 功能性、易用性、美观性、稳定性、性价比

4. **描述统计**
   - 整体满意度 M=3.85, SD=0.72
   - NPS = 42

5. **差异检验**
   - 性别: t检验无显著差异(p=0.23)
   - 年龄: ANOVA显著(F=5.21, p<0.01)
   - 36-45岁满意度最高

6. **回归分析**
   - R² = 0.52
   - 易用性(β=0.35)影响最大
   - 功能性(β=0.28)次之

**核心结论**
1. 整体满意度良好,有提升空间
2. 易用性是满意度关键驱动因素
3. 中年用户满意度最高
4. 性价比满意度最低,需重点关注
