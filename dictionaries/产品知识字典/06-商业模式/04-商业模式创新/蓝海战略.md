# 蓝海战略 (Blue Ocean Strategy)

## 定义

蓝海战略是由W. Chan Kim和Renée Mauborgne提出的战略框架,主张企业应该开创无人竞争的市场空间(蓝海),而非在拥挤的市场中竞争(红海)。

---

## 核心概念

### 红海 vs 蓝海

| 维度 | 红海 | 蓝海 |
|------|------|------|
| 市场空间 | 已知市场 | 未知市场 |
| 竞争状态 | 激烈竞争 | 无竞争 |
| 战略焦点 | 击败竞争对手 | 创造新需求 |
| 需求 | 存量博弈 | 创造增量 |
| 价值创新 | 权衡取舍 | 同时追求差异化和低成本 |
| 规则 | 遵循行业规则 | 打破行业规则 |

---

### 蓝海战略的核心: 价值创新

**价值创新 = 价值 + 创新**

```
        创新
          ↑
          │
    先有价值而无创新 → 价值改进
          │
          │
    先有创新而无价值 → 技术先驱
          │
          │
    价值+创新同时实现 → 价值创新 ← 蓝海
          │
          │
          └─────────────────→ 价值
```

**关键**:
- 同时追求差异化**和**低成本
- 打破"差异化=高成本"的权衡
- 创造新需求

## 蓝海战略框架

### 一、战略布局图

**定义**: 可视化企业的战略轮廓

**坐标轴**:
- X轴: 行业竞争要素
- Y轴: 各要素的投入/水平

**绘制方法**:
1. 识别行业竞争要素
2. 评估各竞争者在各要素上的水平
3. 连接各点形成价值曲线

**战略类型**:
```
红海战略: 与竞争对手相似的价值曲线
蓝海战略: 独特的价值曲线
```

---

### 二、四步动作框架

```
┌──────────────┬──────────────┐
│    消除       │     创造     │
│ (Eliminate)  │   (Create)  │
│              │              │
│ 行业中哪些   │ 哪些从未提供 │
│ 竞争要素应   │ 的要素应该   │
│ 该被消除?    │ 被创造?      │
├──────────────┼──────────────┤
│    减少       │     增加     │
│  (Reduce)    │   (Raise)   │
│              │              │
│ 哪些要素应   │ 哪些要素应   │
│ 该减少到     │ 该增加到     │
│ 行业标准以下?│ 行业标准以上?│
└──────────────┴──────────────┘
```

## 蓝海战略案例分析

### 案例1: 太阳马戏团(Cirque du Soleil)

**传统马戏团的价值曲线**:
```
竞争要素: 动物表演、明星演员、多个舞台、
         场内销售、刺激危险、幽默

问题:
- 动物保护争议
- 成本高昂
- 观众老龄化
```

**太阳马戏团的四步动作**:
```
消除:
- 动物表演(消除成本和争议)
- 明星演员(降低成本)
- 多个舞台(简化运营)

减少:
- 刺激危险(降低保险成本)

增加:
- 帐篷(独特氛围)
- 艺术/戏剧元素(提升体验)

创造:
- 主题表演(全新的观看体验)
- 音乐和舞蹈(多艺术融合)
- 精致环境(高端定位)
```

**结果**:
- 创造全新市场: 马戏+戏剧
- 高票价,高利润
- 全球成功

---

### 案例2: 黄尾葡萄酒(Yellow Tail)

**美国葡萄酒市场(红海)**:
```
竞争要素:
- 葡萄酒品质和陈年
- 葡萄酒复杂度
- 营销投入
- 声望和历史

问题:
- 入门门槛高
- 消费者困惑
- 增长缓慢
```

**黄尾的四步动作**:
```
消除:
- 陈年品质(不强调)
- 葡萄酒复杂术语
- 声望营销

减少:
- 葡萄酒种类(只做2种)
- 营销投入(口碑为主)

增加:
- 易饮性(甜度适中)
- 趣味性(品牌形象)
- 零售店陈列

创造:
- 时尚品牌形象(袋鼠标志)
- 简单选择(只有红白两款)
- 大众定位(非葡萄酒爱好者)
```

**价值曲线**:
```
传统葡萄酒: 复杂、高端、高门槛
黄尾: 简单、有趣、易选择
```

**结果**:
- 2年内成为美国进口葡萄酒第一品牌
- 无需广告,口碑传播
- 开创"轻松葡萄酒"品类

---

### 案例3: 西南航空

**传统航空(红海)**:
```
竞争要素:
- 餐饮服务
- 座位等级
- 航线网络
- 枢纽运营
- 行李转运
- 价格
```

**西南航空的四步动作**:
```
消除:
- 餐饮服务
- 座位等级(只有经济舱)
- 枢纽运营(点对点)
- 行李转运

减少:
- 航线网络(短途为主)
- 服务人员(自助值机)

增加:
- 准点率
- 友好服务
- 频繁班次

创造:
- 点对点短途航空
- 极简服务
- 低成本模式
```

**结果**:
- 创造短途廉价航空市场
- 连续40+年盈利
- 行业标杆

---

### 案例4: 任天堂Wii

**传统游戏机竞争(红海)**:
```
竞争要素:
- 图形性能
- 处理速度
- 游戏数量
- 硬件配置
- 价格(高)

目标用户: 硬核玩家
```

**Wii的四步动作**:
```
消除:
- 传统手柄复杂按钮

减少:
- 图形性能(不如竞品)
- 处理速度
- 游戏数量

增加:
- 体感交互
- 社交游戏
- 家庭娱乐

创造:
- 体感控制器
- 非玩家市场(老人、女性、儿童)
- 运动健身游戏
```

**价值曲线**:
```
PlayStation/Xbox: 高性能、高价格、硬核玩家
Wii: 交互创新、低价格、大众市场
```

**结果**:
- 开创体感游戏市场
- 吸引非传统玩家
- 销量超过竞品

---

### 案例5: iTunes/App Store

**传统音乐/软件销售(红海)**:
```
音乐:
- 实体CD销售
- 专辑为单位
- 高价格

软件:
- 实体安装包
- 高价格
- 版本升级收费
```

**苹果的四步动作**:
```
消除:
- 实体介质
- 专辑强制购买

减少:
- 价格(单曲$0.99)
- 使用门槛

增加:
- 便捷性(即时购买)
- 选择灵活性(单曲购买)
- 发现功能

创造:
- 单曲销售模式
- 数字分发平台
- App生态系统
```

**结果**:
- 颠覆音乐和软件行业
- 创造全新的分发模式
- 生态收入

---

### 案例6: 小米

**传统手机市场(红海)**:
```
竞争要素:
- 线下渠道
- 营销广告
- 高利润率
- 低频更新

问题:
- 价格高
- 渠道成本高
```

**小米的四步动作**:
```
消除:
- 线下渠道(线上直销)
- 大规模广告(口碑营销)

减少:
- 利润率(硬件微利)
- 中间环节

增加:
- 性价比
- 粉丝参与
- 系统更新频率

创造:
- 互联网手机模式
- 粉丝经济
- 生态链产品
- MIUI社区
```

**结果**:
- 3年成为中国第一
- 开创互联网手机模式
- 生态链成功

## 寻找蓝海的六大路径

### 路径1: 跨越替代性行业

**方法**: 关注客户的替代选择,而非直接竞争对手

**案例**:
- 太阳马戏团: 马戏 vs 剧院 vs 歌剧
- Netflix: 电影租赁 vs 电影院 vs 电视

**思考问题**:
```
客户在选择我们产品时,
还会考虑哪些替代品?
他们为什么选择替代品?
```

---

### 路径2: 跨越战略集团

**方法**: 关注不同战略定位的竞争者

**战略集团**:
- 奢侈品牌 vs 大众品牌
- 高端 vs 低端

**案例**:
- 丰田雷克萨斯: 奢侈品质+合理价格
- Target: 廉价商店+设计感

**思考问题**:
```
行业内有哪些战略集团?
客户在升级或降级时会考虑什么?
```

---

### 路径3: 跨越客户链

**方法**: 重新定义目标客户

**客户角色**:
- 购买者
- 使用者
- 影响者

**案例**:
- 胰岛素: 从医生决策转向患者友好
- 教育软件: 从学校决策转向家长需求

**思考问题**:
```
谁是我们真正的客户?
谁应该成为我们的客户?
```

---

### 路径4: 跨越互补产品和服务

**方法**: 关注客户使用产品前、中、后的需求

**案例**:
- 电影院: 提供保姆服务(观影前)
- 汽车经销商: 提供二手车回购(购车后)

**思考问题**:
```
客户使用产品前需要什么?
使用过程中需要什么?
使用后需要什么?
```

---

### 路径5: 跨越功能-情感导向

**方法**: 从功能导向转向情感导向,或反之

**案例**:
- 星巴克: 咖啡(功能)→ 第三空间(情感)
- 宜家: 家具(功能)→ DIY乐趣(情感)
- 沃尔玛: 情感品牌→ 功能性低价

**思考问题**:
```
行业是功能导向还是情感导向?
可以反过来重新定位吗?
```

---

### 路径6: 跨越时间

**方法**: 基于趋势预测未来市场

**案例**:
- 苹果: 预见数字音乐趋势,推出iPod
- 特斯拉: 预见电动车趋势

**思考问题**:
```
有哪些明显的趋势?
这些趋势将如何改变客户价值?
今天可以做哪些准备?
```

## 蓝海战略实施

### 第一阶段: 可视化战略

#### 1. 绘制当前战略布局
- 识别行业竞争要素
- 绘制自己和竞争对手的价值曲线

#### 2. 可视化替代战略
- 应用四步动作框架
- 绘制新价值曲线

#### 3. 选择新战略
- 评估可行性
- 选择最佳方案

---

### 第二阶段: 执行战略

#### 1. 重新定义目标客户
- 明确新的客户群体
- 理解其需求和痛点

#### 2. 调整价值主张
- 重新设计产品/服务
- 匹配新客户需求

#### 3. 调整盈利模式
- 设计可持续的商业模式
- 确保成本可控

#### 4. 克服组织障碍
- 获得领导支持
- 调整组织结构
- 建立执行团队

---

### 第三阶段: 持续创新

#### 1. 监测竞争态势
- 跟踪竞争者反应
- 关注新进入者

#### 2. 扩张蓝海
- 深化差异化
- 扩大市场份额

#### 3. 寻找新蓝海
- 当蓝海变红时
- 开拓新的市场空间

---

## 蓝海战略原则

### 原则1: 重建市场边界
- 避免传统竞争思维
- 寻找新的市场空间

### 原则2: 关注全局,而非数字
- 使用战略布局图
- 可视化战略

### 原则3: 超越现有需求
- 关注非客户
- 创造新需求

### 原则4: 遵循合理的战略顺序
```
买方效用 → 价格 → 成本 → 接受
   ↓        ↓      ↓       ↓
 有吸引力  可接受  能盈利  能实施
```

## 蓝海战略的误区

### 误区1: 蓝海等于创新
**澄清**: 蓝海强调价值创新,不是单纯的技术创新

### 误区2: 蓝海等于新市场
**澄清**: 蓝海可以通过重新定义现有市场创造

### 误区3: 蓝海没有竞争
**澄清**: 蓝海最终会被模仿,需要持续创新

### 误区4: 蓝海只适合创业公司
**澄清**: 大企业也可以创造蓝海
