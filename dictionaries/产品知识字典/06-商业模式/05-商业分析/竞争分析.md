# 竞争分析 (Competitive Analysis)

## 定义

竞争分析是系统性地识别、评估和监控竞争对手,理解竞争格局,制定有效竞争策略的过程。

---

## 第一步: 竞争对手识别

### 竞争对手分类

#### 1. 直接竞争者
**定义**: 提供相同或类似产品,服务相同客户群

**识别方法**:
```
- 搜索相同关键词
- 客户提及的替代选择
- 行业报告中的竞品
- 应用商店同类应用
```

**示例**:
```
微信的直接竞争者:
- QQ
- 钉钉
- 飞书
- WhatsApp(国际)
```

---

#### 2. 间接竞争者
**定义**: 解决相同问题但方式不同

**识别方法**:
```
- 客户预算分配
- 替代方案分析
- 跨界竞品发现
```

**示例**:
```
Netflix的间接竞争者:
- 电影院(线下娱乐)
- YouTube(免费视频)
- 游戏(其他娱乐)
- 书籍(其他休闲)
```

---

#### 3. 潜在进入者
**定义**: 可能进入市场的新竞争者

**识别来源**:
```
- 大型科技公司
- 相关行业企业
- 创业公司
- 国际企业本土化
```

---

### 竞争对手筛选

**优先级矩阵**:
```
         威胁程度
           高
            ↑
    高 │ A 重点监测 │ B 定期跟踪 │
市场      │           │           │
份额      ├───────────┼───────────┤
    低 │ C 持续观察 │ D 一般关注 │
        └───────────┴───────────→
            低
```

### SWOT分析

**应用场景**: 对主要竞争对手进行SWOT分析

```
┌──────────────────┬──────────────────┐
│     优势         │     劣势         │
│  (Strengths)     │  (Weaknesses)    │
├──────────────────┼──────────────────┤
│     机会         │     威胁         │
│ (Opportunities)  │  (Threats)       │
└──────────────────┴──────────────────┘
```

**示例**:
```
竞品A - 某SaaS公司

优势:
- 品牌知名度高
- 资金充裕
- 销售团队强

劣势:
- 产品体验一般
- 客户服务差
- 价格偏高

机会:
- 市场增长快
- 客户需求升级
- 新技术机会

威胁:
- 新兴竞品涌现
- 客户议价能力增强
- 技术变革
```

## 竞争策略制定

### 领导者策略
**适用**: 市场份额第一

**策略**:
- 扩大总市场
- 保护市场份额
- 扩大市场份额

---

### 挑战者策略
**适用**: 市场份额第二、第三

**策略**:
- 正面进攻
- 侧翼进攻
- 包围进攻
- 绕道进攻
- 游击进攻

---

### 跟随者策略
**适用**: 市场份额较小

**策略**:
- 克隆者
- 模仿者
- 改进者

---

### 利基者策略
**适用**: 专注细分市场

**策略**:
- 特定客户利基
- 地理利基
- 产品利基
- 服务利基

## 常见错误

### 错误1: 忽视间接竞争
**后果**: 被跨界颠覆

### 错误2: 过度关注竞品
**后果**: 丧失创新,跟随陷阱

### 错误3: 静态分析
**后果**: 忽视竞争动态变化

### 错误4: 只看表面数据
**后果**: 忽视深层战略意图

### 错误5: 分析不行动
**后果**: 浪费资源,无实际价值
