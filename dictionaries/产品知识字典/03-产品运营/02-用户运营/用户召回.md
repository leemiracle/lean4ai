# 用户召回 (User Recall)

## 定义
用户召回是针对已流失或即将流失的用户，通过Push、短信、邮件等触达方式，配合优惠激励和内容吸引，重新激活用户使用的运营策略。

## 流失用户定义

### 流失判定标准

```
流失用户 = 超过N天未活跃的用户

N的确定：
- 电商：90天未购买
- 社交：30天未登录
- 内容：14天未访问
- 游戏：7天未登录
- 工具：30天未使用
```

### 流失程度分级

| 等级 | 定义 | 召回难度 | 召回策略 |
|------|------|----------|----------|
| 预流失 | 活跃度下降明显 | 低 | 预警干预 |
| 轻度流失 | 刚过流失阈值 | 中 | 精准召回 |
| 中度流失 | 流失2-4周 | 较高 | 强力召回 |
| 重度流失 | 流失1个月以上 | 高 | 大规模触达 |
| 沉默流失 | 流失3个月以上 | 极高 | 品牌唤醒 |

### 流失率计算

```
流失率 = 流失用户数 / 期初用户数 × 100%

月流失率 = 本月流失用户 / 月初活跃用户
年流失率 = 年度流失用户 / 年初用户

健康指标：
- B2C产品：<5%/月
- B2B产品：<2%/月
- 游戏类：<10%/月
```

## 流失预警

### 预警指标体系

```
流失风险评分模型：

Risk_Score = Σ(w_i × indicator_i)

指标示例：
- 最近活跃天数（权重30%）
- 使用频次变化（权重25%）
- 核心行为完成率（权重20%）
- 付费行为变化（权重15%）
- 社交互动变化（权重10%）
```

### 预警分级

| 风险等级 | 风险分数 | 用户特征 | 干预时机 |
|----------|----------|----------|----------|
| 低风险 | 0-30 | 正常活跃 | 常规运营 |
| 中风险 | 30-60 | 活跃下降 | 关注+轻度干预 |
| 高风险 | 60-80 | 明显下降 | 立即干预 |
| 极高风险 | 80-100 | 即将流失 | 强力挽留 |

### 预警触发流程

```
数据监控 → 风险计算 → 分级预警 → 策略匹配 → 执行触达 → 效果追踪
```

## 召回策略

### 01 预流失用户召回

#### 触达方式
| 渠道 | 打开率 | 成本 | 适用场景 |
|------|--------|------|----------|
| Push | 5-15% | 低 | APP活跃用户 |
| 站内信 | 10-30% | 低 | 登录用户 |
| 短信 | 5-20% | 中 | 有手机号 |
| 邮件 | 15-25% | 低 | 商务用户 |
| 微信 | 20-40% | 中 | 微信生态 |

#### 触达内容

**个性化推荐：**
```
文案模板：
"{用户名}，好久不见！
根据您的喜好，为您推荐{个性化内容}
点击查看 → {链接}"
```

**利益刺激：**
```
文案模板：
"您有一张{金额}元优惠券即将过期
仅剩{天数}天，快来使用吧！
立即使用 → {链接}"
```

**社交牵引：**
```
文案模板：
"{好友名}等{N}位好友发布了新内容
快来看看吧！
查看详情 → {链接}"
```

### 02 轻度流失召回

#### 召回策略矩阵

| 召回方式 | 内容 | 激励 | 预期召回率 |
|----------|------|------|------------|
| 优惠券 | 回归礼包 | 5-20元券 | 10-20% |
| 积分提醒 | 积分过期提醒 | 积分加倍 | 8-15% |
| 内容推送 | 个性化内容 | - | 5-10% |
| 社交召回 | 好友动态 | - | 8-12% |

#### 召回文案模板

**电商类：**
```
{用户名}，我们想您了！
🎁 回归专属礼包已到账
💰 {N}元无门槛券 × {M}张
📦 {商品}降价了，曾关注的商品
点击领取 → {链接}

[退订]回T
```

**内容类：**
```
{用户名}，这些精彩内容您还没看：
🔥 {热门内容1}
⭐ {热门内容2}
📈 {热点话题}正在讨论
点击查看 → {链接}
```

**工具类：**
```
{用户名}，{N}天没见，您的{数据}：
📊 {数据摘要}
📈 {变化趋势}
💡 {功能更新}
回来查看 → {链接}
```

### 03 中度流失召回

#### 强力召回策略

**组合拳策略：**
```
第一轮（Day 1）：
- 短信 + Push
- 回归大礼包
- 强调损失

第二轮（Day 3）：
- 邮件 + 短信
- 限时活动
- 紧迫感

第三轮（Day 7）：
- 短信
- 问卷调研
- 改进反馈
```

**激励升级：**
```
常规用户：10元优惠券
流失用户：30元专属券
重度流失：50元回归礼包

召回激励 > 常规激励
```

### 04 重度流失召回

#### 大规模召回策略

**短信群发：**
```
目标：流失3-6个月用户
频次：每月1-2次
内容：重大更新、年度活动
```

**邮件营销：**
```
目标：有邮箱的流失用户
内容：
- 产品重大更新
- 年度报告
- 节日祝福
- 限时活动
```

**品牌广告：**
```
目标：沉默用户
方式：信息流广告、品牌广告
内容：品牌形象、产品升级
```

## 召回效果评估

### 核心指标

| 指标 | 计算公式 | 目标 |
|------|----------|------|
| 召回率 | 回归用户/触达用户 | >5% |
| 召回成本 | 召回总成本/回归用户 | <LTV/5 |
| 召回留存 | 召回用户7日留存 | >30% |
| 召回转化 | 召回用户付费转化 | >3% |
| ROI | (召回收益-成本)/成本 | >1 |

### 召回漏斗

```
触达用户 100%
   ↓ 打开率 20%
打开用户 20%
   ↓ 点击率 30%
点击用户 6%
   ↓ 回归率 50%
回归用户 3%
   ↓ 留存率 30%
留存用户 0.9%
```

### 召回ROI计算

```
召回ROI = (召回用户价值 - 召回成本) / 召回成本

召回用户价值 = 召回人数 × 后续LTV
召回成本 = 短信成本 + 优惠券成本 + 运营成本

示例：
触达10万流失用户
召回率5% = 5000人
人均LTV贡献50元
总价值 = 25万元
总成本 = 5万元
ROI = (25-5)/5 = 400%
```

## 召回最佳实践

### 1. 精准触达

**用户分层召回：**
```
按流失原因分层：
- 需求不满足 → 产品改进告知
- 价格敏感 → 优惠刺激
- 竞品吸引 → 差异化优势
- 使用障碍 → 功能引导
```

### 2. 个性化内容

**基于用户历史：**
```
- 曾购买商品：降价提醒
- 曾关注内容：更新通知
- 曾使用功能：功能推荐
- 社交关系：好友动态
```

### 3. 合理频次

**触达频次控制：**
```
预流失：每周1-2次
轻度流失：每两周1次
中度流失：每月1-2次
重度流失：季度1次

同一用户：
- 短信不超过3次/月
- Push不超过5次/周
```

### 4. 时机选择

**最佳触达时间：**
```
工作日：
- 上午 9:00-10:00
- 中午 12:00-13:00
- 晚上 20:00-22:00

周末：
- 上午 10:00-11:00
- 下午 15:00-17:00
- 晚上 20:00-22:00
```

### 5. A/B测试

**测试维度：**
```
- 文案风格（情感型vs利益型）
- 激励金额（5元vs10元vs20元）
- 触达渠道（Push vs短信vs邮件）
- 发送时间（早vs中vs晚）
```

```

**情感型：**
```

→ {链接}"

```

来看看吧 → {链接}"
```

## 流失原因分析

### 常见流失原因

| 原因类型 | 占比 | 召回策略 |
|----------|------|----------|
| 需求满足后离开 | 20-30% | 新需求激发 |
| 竞品转移 | 15-25% | 差异化召回 |
| 体验不佳 | 15-20% | 改进告知 |
| 遗忘 | 10-15% | 品牌唤醒 |
| 场景变化 | 10-15% | 新场景引导 |
| 其他 | 10-20% | 问卷调研 |

### 流失调研

**问卷设计：**
```
1. 您为什么不再使用我们的产品？
□ 找到更好的替代品
□ 不再需要这类产品
□ 使用体验不好
□ 忘记了
□ 其他____

2. 如果我们改进以下方面，您会回来吗？
□ 价格更优惠
□ 功能更完善
□ 体验更好
□ 内容更丰富
```

## 召回与留存的关系

```
召回成本 ≈ 5-10倍留存成本

投入优先级：
1. 防流失（预防）
2. 促活跃（维护）
3. 召回流失（挽回）

最佳策略：减少流失 > 召回流失
```

## 注意事项

### 1. 合规性
- 遵守短信/邮件法规
- 提供退订选项
- 保护用户隐私

### 2. 用户体验
- 避免过度打扰
- 内容有价值
- 尊重用户选择

### 3. 成本控制
- 监控召回ROI
- 控制激励成本
- 优化触达效率

### 4. 长期价值
- 关注召回用户留存
- 避免短期行为
- 建立持续关系
