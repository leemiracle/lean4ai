# 促销活动 (Promotional Activities)

## 定义
促销活动是通过价格优惠、赠品、折扣等方式，刺激用户购买行为的营销活动。促销是最直接有效的转化手段。

## 促销活动类型

### 按优惠形式分类

| 类型 | 形式 | 适用场景 | 效果 |
|------|------|----------|------|
| 满减 | 满X元减Y元 | 提升客单价 | 强 |
| 折扣 | X折优惠 | 清库存、引流 | 强 |
| 优惠券 | 代金券 | 拉新、促转化 | 强 |
| 直降 | 直接降价 | 核心商品促销 | 强 |
| 买赠 | 买X送Y | 新品推广 | 中 |
| 秒杀 | 限时特价 | 引流、造势 | 强 |
| 拼团 | 多人成团优惠 | 社交裂变 | 强 |
| 预售 | 提前预订优惠 | 大促预热 | 中 |

### 按时间节点分类

| 类型 | 时间 | 特点 |
|------|------|------|
| 日常促销 | 平日 | 常规转化 |
| 周末促销 | 周末 | 小高峰 |
| 月末促销 | 月末 | 冲业绩 |
| 节日促销 | 节日 | 流量高峰 |
| 周年庆 | 周年 | 品牌活动 |
| 大促 | 618/双11 | 年度高峰 |

## 促销策略设计

### 01 满减策略

#### 满减门槛设计
```
满减公式：

满减力度 = 优惠金额 / 门槛金额

建议力度：
- 日常促销：10-15%
- 中型促销：15-20%
- 大型促销：20-30%

示例：
满100减15（力度15%）
满200减40（力度20%）
满300减80（力度27%）
```

#### 满减阶梯设计
```
阶梯设计原则：
1. 第一阶梯：略高于平均客单价
2. 第二阶梯：提升30-50%
3. 第三阶梯：高客单用户

示例（平均客单价80元）：
满99减10
满199减30
满299减60
```

### 02 优惠券策略

#### 优惠券类型
| 类型 | 形式 | 适用场景 |
|------|------|----------|
| 无门槛券 | 无使用门槛 | 新用户、拉新 |
| 满减券 | 满X元可用 | 提升客单价 |
| 折扣券 | X折优惠 | 清库存 |
| 品类券 | 指定品类可用 | 品类推广 |
| 商品券 | 指定商品可用 | 单品推广 |

#### 优惠券设计要素
```
优惠券设计 = 面额 + 门槛 + 有效期 + 使用范围 + 发放量

设计原则：
1. 面额：有吸引力但可控
2. 门槛：略高于客单价
3. 有效期：7-30天（制造紧迫感）
4. 使用范围：明确清晰
5. 发放量：根据预算控制
```

#### 优惠券发放策略
```
发放渠道：
- 主动发放：Push、短信、站内信
- 任务获取：签到、分享、游戏
- 购买赠送：购后发券
- 社群发放：社群专属

发放节奏：
- 新用户：注册即发
- 流失用户：回归发券
- 高价值用户：专属大额券
- 日常：定时发券
```

### 03 秒杀策略

#### 秒杀设计要素
```
秒杀 = 限时 + 限量 + 特价 + 爆款

设计要点：
1. 时间：整点、固定时间
2. 数量：限量营造稀缺
3. 价格：明显低于平时
4. 商品：爆款、刚需
```

#### 秒杀节奏
```
日常秒杀：
- 每天固定时间（如10点、20点）
- 3-5款商品
- 每款限量100-500件

大促秒杀：
- 整点秒杀
- 10-20款商品
- 每款限量500-2000件
```

### 04 拼团策略

#### 拼团模式
| 类型 | 人数 | 价格 | 适用场景 |
|------|------|------|----------|
| 2人团 | 2人 | 9折 | 日常拼团 |
| 3人团 | 3人 | 8折 | 中等优惠 |
| 5人团 | 5人 | 7折 | 深度优惠 |
| 老带新团 | 1老+X新 | 专属价 | 拉新裂变 |

#### 拼团设计公式
```
拼团成功率 = 发起人数 × 分享率 × 参团率

优化方向：
1. 增加发起人数：降低门槛、优惠吸引
2. 提升分享率：优化分享文案、激励机制
3. 提高参团率：限时、价格优势
```

## 促销活动策划

### 活动策划流程

```
确定目标 → 选品策略 → 定价策略 → 活动设计 → 推广执行 → 效果评估
```

### 01 目标设定

#### 促销目标类型
| 目标 | KPI | 促销策略 |
|------|-----|----------|
| 提升GMV | 销售额 | 满减、折扣 |
| 提升订单量 | 订单数 | 秒杀、低价 |
| 提升客单价 | 客单价 | 满减、买赠 |
| 清库存 | 库存周转 | 折扣、特价 |
| 拉新用户 | 新增用户 | 新人专享 |
| 提升复购 | 复购率 | 回购券 |

### 02 选品策略

#### 促销商品分类
| 类型 | 特点 | 作用 |
|------|------|------|
| 引流款 | 低价、刚需 | 吸引流量 |
| 爆款 | 高转化、高销量 | 主力销售 |
| 利润款 | 高毛利 | 贡献利润 |
| 清仓款 | 库存积压 | 清理库存 |
| 新品 | 新上市 | 品尝推广 |

#### 选品原则
```
引流款：10-20% SKU，超低价，限量
爆款：30-40% SKU，主力销售
利润款：30-40% SKU，保证利润
清仓款：10-20% SKU，折扣清仓
```

### 03 定价策略

#### 促销定价方法
```
1. 成本定价：成本 × (1 + 目标毛利率)
2. 竞争定价：参考竞品价格
3. 心理定价：99元 vs 100元
4. 锚定定价：原价999，现价399
5. 组合定价：套餐更优惠
```

#### 价格弹性
```
需求价格弹性 = 需求变化率 / 价格变化率

弹性 > 1：敏感，适合降价促销
弹性 < 1：不敏感，降价效果有限
```

### 04 活动节奏

#### 促销周期设计
```
大型促销节奏：

预热期（前7-15天）：
- 发布活动预告
- 预售定金
- 加购收藏

爆发期（活动当天）：
- 全力推广
- 限时秒杀
- 紧迫营销

返场期（后1-3天）：
- 延续优惠
- 最后机会
- 为下次铺垫
```

| 指标 | 计算公式 | 说明 |
|------|----------|------|

## 促销效果评估

### 评估维度

```
1. 销售维度
   - GMV达成率
   - 订单量
   - 客单价变化

2. 用户维度
   - 参与人数
   - 新用户占比
   - 复购率

3. 商品维度
   - 爆款表现
   - 库存清理
   - 毛利率

4. 成本维度
   - 促销成本
   - ROI
```

### ROI计算

```
促销ROI = (促销GMV - 促销成本) / 促销成本 × 100%

其中：
促销成本 = 让利金额 + 推广成本 + 运营成本

例：
GMV：100万
让利：20万
推广：10万
运营：5万
总成本：35万
ROI = (100-35)/35 = 186%
```

## 促销注意事项

### 1. 价格体系
```
注意事项：
- 促销价格不能破坏日常价格体系
- 促销频率不能太高
- 避免用户养成等促销的习惯

建议：
- 日常促销：每周1-2次
- 大型促销：每月1次
- 全年大促：3-4次
```

### 2. 利润控制
```
利润 = GMV × 毛利率 - 促销成本 - 推广成本

控制要点：
- 设置促销预算上限
- 监控实时毛利率
- 及时调整策略
```

### 3. 用户体验
```
注意点：
- 规则简单清晰
- 购买流程顺畅
- 不强制搭售
- 售后服务保障
```

### 4. 防刷单
```
措施：
- 限制购买数量
- 实名认证
- 风控系统
- 人工审核
```

## 最佳实践

### 1. 目标导向
- 明确促销目标
- 选择合适策略
- 数据验证效果

### 2. 组合策略
- 多种促销方式组合
- 不同商品不同策略
- 分层用户差异促销

### 3. 节奏把控
- 合理安排促销频率
- 大促小促结合
- 预热、爆发、返场

### 4. 数据驱动
- 实时监控数据
- 及时调整策略
- 复盘沉淀经验

### 5. 用户体验
- 规则简单易懂
- 流程顺畅
- 售后有保障
