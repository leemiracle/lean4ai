## 核心模型

### AARRR漏斗模型
```
        ┌─────────┐
        │  获取   │ Acquisition - 用户获取
        │Activation│
        ├─────────┤
        │  激活   │ Activation - 首次体验
        │Retention │
        ├─────────┤
        │  留存   │ Retention - 用户留存
        │ Revenue │
        ├─────────┤
        │  变现   │ Revenue - 商业变现
        │ Referral│
        ├─────────┤
        │  推荐   │ Referral - 口碑传播
        └─────────┘
```

### 增长飞轮
```
           用户体验
              ↑
         ┌────┴────┐
         │         │
    用户增长 ←─→ 商业收入
         │         │
         └────┬────┘
              ↓
           数据洞察
```

### RFM用户分群
| 维度 | 含义 | 高价值用户特征 |
|------|------|----------------|
| R (Recency) | 最近一次消费 | 时间短 |
| F (Frequency) | 消费频率 | 频次高 |
| M (Monetary) | 消费金额 | 金额大 |

## 运营公式

### 用户增长公式
```
用户增长 = 新增用户 + 召回用户 - 流失用户
```

### 病毒系数
```
K = i × c
K: 病毒系数
i: 每个用户发出的邀请数
c: 受邀者转化率
K > 1 时实现病毒增长
```

### LTV计算
```
LTV = ARPU × 用户生命周期
LTV = (客单价 × 购买频次) × 留存时长
```

### ROI计算
```
ROI = (收益 - 成本) / 成本 × 100%
```
