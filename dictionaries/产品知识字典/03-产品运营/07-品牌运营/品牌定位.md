# 品牌定位 (Brand Positioning)

## 定义
品牌定位是在目标用户心智中占据独特位置的战略，明确品牌代表什么、为谁服务、与竞品有何不同。

## 品牌定位模型

### 3C分析模型

```
┌─────────────────────────────────────────────┐
│              品牌定位三角                     │
├─────────────┬─────────────┬─────────────────┤
│   Customer  │  Company    │   Competition   │
│   用户需求   │  企业能力    │    竞争环境      │
└─────────────┴─────────────┴─────────────────┘

定位 = 用户需求 ∩ 企业能力 ∩ 竞争空白
```

### 定位三要素

```
1. 目标用户 (Target Audience)
   - 为谁服务？
   - 用户画像是什么？

2. 价值主张 (Value Proposition)
   - 提供什么价值？
   - 解决什么问题？

3. 差异化 (Differentiation)
   - 与竞品有何不同？
   - 为什么选我们？
```

## 品牌定位方法

### 1. STP分析法

```
S (Segmentation) 市场细分
   ↓
T (Targeting) 目标市场选择
   ↓
P (Positioning) 品牌定位
```

#### 市场细分
```
细分维度：

1. 人口属性
   - 年龄、性别、收入、教育

2. 地理区域
   - 城市、地区、国家

3. 心理特征
   - 价值观、生活方式、个性

4. 行为特征
   - 使用场景、购买习惯、忠诚度
```

#### 目标市场选择
```
评估标准：

1. 市场规模
   - 市场容量
   - 增长潜力

2. 竞争程度
   - 竞争对手数量
   - 市场集中度

3. 企业能力
   - 资源匹配度
   - 核心竞争力

4. 盈利能力
   - 毛利率
   - 规模效应
```

### 2. 定位矩阵

```
              高端
               ↑
    象限2      │     象限1
   差异化聚焦  │    差异化领导
──────────────┼──────────────→ 差异化
    象限3      │     象限4
   成本聚焦    │    成本领导
               ↓
             低端
```

## 品牌定位策略

### 常见定位策略

| 策略 | 说明 | 示例 |
|------|------|------|
| 功能定位 | 强调功能优势 | "充电5分钟，通话2小时" |
| 情感定位 | 情感共鸣 | "Just Do It" |
| 价值定位 | 性价比优势 | "性价比之王" |
| 场景定位 | 使用场景 | "怕上火喝王老吉" |
| 人群定位 | 特定人群 | "专为年轻人设计" |
| 类别定位 | 开创新品类 | "不是所有牛奶都叫特仑苏" |
| 文化定位 | 文化认同 | "国货之光" |

### 定位差异化方向

```
1. 产品差异化
   - 功能、性能、设计

2. 服务差异化
   - 服务质量、响应速度

3. 渠道差异化
   - 销售渠道、触达方式

4. 形象差异化
   - 品牌形象、文化内涵

5. 价格差异化
   - 高端/性价比
```

## 品牌定位表达

### 品牌定位陈述

```
定位陈述模板：

对于{目标用户}，
{品牌名}是{品类}，
它能提供{核心利益}，
因为{信任理由}。

示例：
对于追求健康生活的都市白领，
元气森林是无糖气泡水，
它能提供好喝又无负担的饮品体验，
因为采用赤藓糖醇代糖，0糖0脂0卡。
```

### 品牌口号 (Slogan)

```
Slogan原则：

1. 简短有力
   - 不超过10个字

2. 易于记忆
   - 朗朗上口

3. 突出卖点
   - 核心价值

4. 情感共鸣
   - 触动用户

经典案例：
- 耐克：Just Do It
- 苹果：Think Different
- 阿里：让天下没有难做的生意
- 小米：为发烧而生
```

### 品牌故事

```
品牌故事要素：

1. 背景
   - 品牌起源

2. 问题
   - 用户痛点

3. 解决
   - 如何解决

4. 价值
   - 带来什么价值

5. 愿景
   - 未来目标
```

## 品牌视觉识别 (VI)

### VI系统组成

```
1. 基础系统
   - 品牌标志(Logo)
   - 品牌色彩
   - 品牌字体
   - 辅助图形

2. 应用系统
   - 名片
   - 信纸
   - 包装
   - 广告
   - 网站等
```

### Logo设计原则

```
1. 简洁
   - 易于识别
   - 易于记忆

2. 相关
   - 与品牌相关
   - 传达品牌理念

3. 独特
   - 与竞品区分
   - 有辨识度

4. 可延展
   - 各种场景适用
   - 不同尺寸清晰
```

## 品牌定位验证

### 验证方法

```
1. 用户调研
   - 品牌认知度
   - 品牌联想
   - 购买意愿

2. 竞品对比
   - 差异化程度
   - 独特性评估

3. 内部验证
   - 团队理解一致
   - 资源匹配度

4. 市场测试
   - 小范围测试
   - 收集反馈
```

### 品牌健康度指标

| 指标 | 说明 | 测量方法 |
|------|------|----------|
| 品牌知名度 | 知道品牌的比例 | 问卷调研 |
| 品牌认知度 | 了解品牌的程度 | 问卷调研 |
| 品牌美誉度 | 对品牌的好感 | 问卷调研 |
| 品牌忠诚度 | 重复购买意愿 | 复购率 |
| 品牌推荐度 | 愿意推荐的比例 | NPS |

## 品牌定位最佳实践

### 1. 聚焦核心
- 定位要清晰聚焦
- 不要什么都想代表
- 一句话能说清

### 2. 持续一致
- 长期坚持定位
- 所有触点一致
- 不轻易改变

### 3. 有据可依
- 基于用户需求
- 基于企业能力
- 有差异化空间

### 4. 可执行
- 能落地执行
- 资源匹配
- 团队认同

### 5. 定期评估
- 监控品牌健康度
- 根据市场调整
- 保持与时俱进
