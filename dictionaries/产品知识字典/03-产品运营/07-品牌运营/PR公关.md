# PR公关 (Public Relations)

## 定义
PR公关是通过媒体和公众沟通，建立和维护企业与公众之间良好关系的管理活动
目的是塑造企业形象、传播品牌价值、处理公众关系。

## PR公关类型

### 按目的分类

| 类型 | 目的 | 常见形式 |
|------|------|----------|
| 品牌公关 | 品牌形象塑造 | 品牌活动、媒体报道 |
| 产品公关 | 产品推广 | 新品发布、评测 |
| 危机公关 | 危机处理 | 危机声明、澄清 |
| 投资者关系 | 投资者沟通 | 财报、投资者会议 |
| 政府关系 | 政府沟通 | 政策解读、合规 |
| 员工关系 | 内部沟通 | 内刊、员工活动 |

### 按传播方式分类

| 类型 | 形式 | 特点 |
|------|------|------|
| 媒体公关 | 媒体报道、专访 | 权威性、公信力 |
| 活动公关 | 发布会、沙龙 | 体验感、互动性 |
| 内容公关 | 软文、案例 | 深度、持续性 |
| 社交公关 | 社交媒体运营 | 互动性、即时性 |

## 媒体关系

### 媒体类型

| 类型 | 媒体 | 特点 |
|------|------|------|
| 权威媒体 | 央视、人民日报 | 权威性高 |
| 财经媒体 | 财新、第一财经 | 专业性强 |
| 科技媒体 | 36氪、虎嗅 | 行业垂直 |
| 大众媒体 | 新浪、搜狐 | 覆盖面广 |
| 自媒体 | KOL、公众号 | 影响力大 |

### 媒体关系维护

```
维护方法：

1. 媒体库建设
   - 建立媒体名单
   - 记者信息档案
   - 定期更新

2. 日常沟通
   - 定期沟通
   - 信息分享
   - 关系维护

3. 价值提供
   - 独家新闻
   - 采访机会
   - 行业洞察

4. 尊重专业
   - 不干涉报道
   - 尊重独立性
   - 诚实沟通
```

### 媒体沟通技巧

```
沟通要点：

1. 准备充分
   - 了解记者背景
   - 准备核心信息
   - 预设问题答案

2. 信息清晰
   - 核心信息明确
   - 简洁有力
   - 有数据支撑

3. 态度真诚
   - 诚实回答
   - 不回避问题
   - 承认不知道

4. 控制节奏
   - 引导话题
   - 回到核心信息
   - 适时结束
```

## 新闻稿撰写

### 新闻稿结构

```
倒金字塔结构：

┌─────────────────────────────────┐
│ 标题：吸引眼球、概括要点          │
├─────────────────────────────────┤
│ 导语：5W1H（最核心信息）          │
├─────────────────────────────────┤
│ 主体：展开说明、补充细节          │
├─────────────────────────────────┤
│ 背景：相关背景信息               │
├─────────────────────────────────┤
│ 引用：高管/专家引言              │
└─────────────────────────────────┘
```

### 5W1H原则

```
Who：谁
What：什么
When：何时
Where：何地
Why：为什么
How：如何
```

### 新闻稿模板

```
━━─ 新闻稿模板 ─━

【标题】{核心事件} - {品牌/公司} {动作}

【导语】
{时间}，{公司}在{地点}宣布{核心事件}，
这是{背景说明}的重要举措。

【主体】
{详细说明事件内容、意义、影响}

【背景】
{相关背景信息}

【引用】
{公司高管}表示："{观点/态度}"

【关于公司】
{公司简介}

【联系方式】
联系人：{姓名}
电话：{电话}
邮箱：{邮箱}
```

## 新闻发布会

### 发布会类型

| 类型 | 场合 | 特点 |
|------|------|------|
| 新品发布会 | 新产品上市 | 场面大、关注高 |
| 战略发布会 | 重大战略 | 权威性、影响大 |
| 危机发布会 | 危机处理 | 及时性、针对性 |
| 媒体沟通会 | 日常沟通 | 小型、深度 |

### 发布会流程

```
筹备阶段：
1. 确定主题和时间
2. 确定嘉宾和媒体名单
3. 准备发布会物料
4. 场地布置和技术调试

发布会流程：
1. 签到（提前30分钟）
2. 开场（主持人）
3. 主题演讲（主讲人）
4. 产品演示/视频
5. 嘉宾分享
6. 媒体问答
7. 专访（可选）

后续跟进：
1. 发送新闻稿
2. 媒体报道跟进
3. 效果评估
```

### 媒体问答技巧

```
应对技巧：

1. 准备充分
   - 预设问题清单
   - 准备标准答案

2. 倾听完整
   - 听完问题再回答
   - 确认理解正确

3. 回答简洁
   - 直接回答
   - 不展开过多

4. 搭桥技巧
   - "更重要的是..."
   - "我想强调的是..."
   - 引导回到核心信息

5. 不确定时
   - "我需要核实后回答"
   - "会后我会跟进"
```

## 公关效果评估

### 评估指标

| 维度 | 指标 | 说明 |
|------|------|------|
| 曝光 | 报道数量、曝光量 | 媒体覆盖 |
| 质量 | 媒体级别、报道深度 | 内容质量 |
| 情感 | 正面/负面比例 | 舆情情感 |
| 互动 | 转发、评论、点赞 | 用户参与 |
| 转化 | 搜索量、访问量 | 行为转化 |

### 媒体价值评估

```
公关价值计算：

1. 广告价值等价 (AVE)
   AVE = 报道版面/时长 × 广告刊例价

2. 公关ROI
   ROI = (公关收益 - 公关成本) / 公关成本

3. 品牌价值贡献
   通过品牌调研评估公关对品牌价值的贡献
```

## PR公关最佳实践

### 1. 真实透明
- 诚实沟通
- 不隐瞒欺骗
- 及时回应

### 2. 价值导向
- 提供有价值的信息
- 不做无用功
- 媒体友好

### 3. 长期关系
- 建立长期媒体关系
- 日常维护
- 互利共赢

### 4. 专业执行
- 专业团队
- 规范流程
- 高效执行

### 5. 效果评估
- 设定评估指标
- 数据驱动
- 持续优化
