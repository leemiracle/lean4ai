# 危机公关 (Crisis PR)

## 定义
危机公关是在企业面临负面事件时，通过有效的沟通和管理
控制事态发展、减少负面影响、维护企业声誉的公关活动。

## 危机类型

### 按来源分类

| 类型 | 说明 | 示例 |
|------|------|------|
| 产品危机 | 产品质量问题 | 产品缺陷、安全隐患 |
| 服务危机 | 服务问题 | 客服不当、服务差 |
| 舆论危机 | 负面舆论 | 网友吐槽、媒体曝光 |
| 法律危机 | 法律纠纷 | 诉讼、违规 |
| 管理危机 | 内部问题 | 高管丑闻、员工事件 |
| 灾难危机 | 不可抗力 | 自然灾害、事故 |

### 按严重程度分类

| 级别 | 特征 | 响应时间 |
|------|------|----------|
| 一级危机 | 重大事件、广泛传播 | 1小时内响应 |
| 二级危机 | 较大影响、持续发酵 | 4小时内响应 |
| 三级危机 | 一般问题、局部传播 | 24小时内响应 |

## 危机生命周期

```
潜伏期 → 爆发期 → 扩散期 → 处理期 → 恢复期
   ↓         ↓         ↓         ↓         ↓
 风险积累   事件发生   舆论发酵   危机处理   形象恢复
```

### 各阶段应对

| 阶段 | 特征 | 应对策略 |
|------|------|----------|
| 潜伏期 | 问题尚未暴露 | 建立预警、日常监测 |
| 爆发期 | 事件刚刚发生 | 快速响应、控制传播 |
| 扩散期 | 舆论持续发酵 | 积极沟通、引导舆论 |
| 处理期 | 危机处理中 | 公布进展、解决问题 |
| 恢复期 | 危机逐渐平息 | 总结教训、形象修复 |

## 危机公关原则

### 5S原则

```
1. 速度第一 (Speed)
   - 黄金4小时
   - 快速响应
   - 抢占先机

2. 真诚沟通 (Sincerity)
   - 态度真诚
   - 信息真实
   - 勇于承担责任

3. 承担责任 (Shoulder)
   - 不推卸责任
   - 主动承担
   - 积极解决

4. 权威证实 (Standard)
   - 第三方背书
   - 权威机构证实
   - 数据支撑

5. 系统运行 (System)
   - 系统化应对
   - 团队协作
   - 统一口径
```

## 危机处理流程

### 标准流程

```
发现危机 → 快速评估 → 组建团队 → 制定方案 → 对外沟通 → 持续跟进 → 总结复盘
```

### 1. 发现危机

```
监测渠道：

1. 社交媒体
   - 微博热搜
   - 微信朋友圈
   - 抖音/快手

2. 媒体监测
   - 新闻网站
   - 自媒体

3. 用户反馈
   - 客服
   - 评论

4. 内部报告
   - 员工上报
```

### 2. 快速评估

```
评估维度：

1. 事件性质
   - 真实性
   - 严重性
   - 影响范围

2. 传播情况
   - 传播渠道
   - 传播速度
   - 舆论倾向

3. 影响评估
   - 对品牌的影响
   - 对业务的影响
   - 对股价的影响

评估表：
| 维度 | 评分(1-5) | 说明 |
|------|-----------|------|
| 严重性 | {分数} | {说明} |
| 传播度 | {分数} | {说明} |
| 影响度 | {分数} | {说明} |
| 总分 | {分数} | {评级} |
```

### 3. 组建团队

```
团队组成：

1. 决策层
   - CEO/高管
   - 最终决策

2. 公关团队
   - 公关负责人
   - 媒体对接

3. 法务团队
   - 法律顾问
   - 合规审核

4. 业务团队
   - 相关业务负责人
   - 提供专业支持

5. 客服团队
   - 客服负责人
   - 用户沟通
```

### 4. 制定方案

```
方案要素：

1. 事实梳理
   - 事件经过
   - 关键事实

2. 责任认定
   - 责任归属
   - 处理方式

3. 沟通策略
   - 对谁说
   - 说什么
   - 怎么说

4. 行动计划
   - 补救措施
   - 后续跟进

5. 口径统一
   - 官方口径
   - Q&A准备
```

### 5. 对外沟通

```
沟通原则：

1. 及时
   - 第一时间回应
   - 不让谣言传播

2. 真诚
   - 态度诚恳
   - 不回避问题

3. 负责
   - 承担责任
   - 提出解决方案

4. 透明
   - 公布事实
   - 持续更新进展
```

## 危机声明撰写

### 声明结构

```
1. 开头
   - 表明态度
   - 承认问题

2. 事实说明
   - 事件经过
   - 关键事实

3. 道歉/解释
   - 诚恳道歉
   - 必要解释

4. 处理措施
   - 已采取措施
   - 后续计划

5. 承诺
   - 改进承诺
   - 联系方式
```

### 声明模板

```
━━─ 危机声明模板 ─━

关于{事件}的声明

我们关注到{事件描述}，对此我们深表歉意。

经核实，{事实说明}。

对于此事，我们的态度是{态度}。

我们已采取以下措施：
1. {措施1}
2. {措施2}
3. {措施3}

后续我们将{后续计划}。

我们承诺{改进承诺}。

如有疑问，请联系{联系方式}。

{公司名称}
{日期}
```

### 声明注意事项

```
注意事项：

1. 态度真诚
   - 不要推卸责任
   - 不要狡辩

2. 语言得体
   - 不要激化矛盾
   - 不要阴阳怪气

3. 信息准确
   - 核实事实
   - 数据准确

4. 行动具体
   - 措施具体可执行
   - 时间节点明确

5. 避免雷区
   - 不说"无可奉告"
   - 不说"是个例"
   - 不说"友商操作"
```

## 危机中的媒体应对

### 媒体沟通原则

```
1. 统一口径
   - 指定发言人
   - 统一回应内容

2. 主动沟通
   - 主动联系媒体
   - 提供准确信息

3. 及时更新
   - 持续发布进展
   - 不让媒体猜测

4. 尊重媒体
   - 不拒绝采访
   - 不威胁媒体
```

### 媒体问答技巧

```
应对技巧：

1. 准备充分
   - 预设问题
   - 准备答案

2. 控制情绪
   - 保持冷静
   - 不与记者冲突

3. 回答策略
   - 说事实不说推测
   - 说态度不说细节
   - 说解决不说指责

4. 搭桥技巧
   - "更重要的是..."
   - "我们正在..."
   - 引导到正面信息

5. 不确定时
   - "我会核实后回复"
   - "目前我能确认的是..."
```

### 监测内容

```
1. 传播监测
   - 传播量
   - 传播速度
   - 主要传播节点

2. 情感监测
   - 正面/负面比例
   - 情感趋势

3. 话题监测
   - 核心话题
   - 衍生话题

4. 关键人物监测
   - KOL观点
   - 意见领袖态度
```

|------|------|------|
