# 电商运营 (E-commerce Operations)

## 定义
电商运营是在电商平台上进行商品管理、营销推广、客户服务等综合性运营活动
目的是提升销售额、优化用户体验、提高运营效率。

## 核心指标

### 销售指标

| 指标 | 计算公式 | 说明 |
|------|----------|------|
| GMV | 订单金额总和 | 商品交易总额 |
| 实际销售额 | GMV - 退款 | 实际收入 |
| 订单量 | 订单总数 | 销售规模 |
| 客单价 | GMV/订单量 | 平均订单金额 |
| 连带率 | 商品件数/订单量 | 附加销售 |

### 流量指标

| 指标 | 计算公式 | 说明 |
|------|----------|------|
| 访客数(UV) | 访问用户数 | 流量规模 |
| 浏览量(PV) | 页面浏览数 | 页面热度 |
| 人均浏览 | PV/UV | 浏览深度 |
| 跳出率 | 跳出UV/总UV | 页面吸引力 |

### 转化指标

| 指标 | 计算公式 | 说明 |
|------|----------|------|
| 转化率 | 下单UV/访客UV | 整体转化 |
| 加购率 | 加购UV/访客UV | 购买意向 |
| 收藏率 | 收藏UV/访客UV | 兴趣程度 |
| 支付率 | 支付UV/下单UV | 支付转化 |

### 用户指标

| 指标 | 计算公式 | 说明 |
|------|----------|------|
| 复购率 | 复购用户/购买用户 | 用户忠诚 |
| 留存率 | 留存用户/总用户 | 用户粘性 |
| LTV | 用户生命周期价值 | 长期价值 |

## 商品运营

### 选品策略

```
选品维度：

1. 市场需求
   - 搜索量
   - 销量趋势
   - 市场容量

2. 竞争分析
   - 竞品数量
   - 价格区间
   - 差异化空间

3. 利润空间
   - 毛利率
   - 周转率
   - 资金占用

4. 供应链
   - 货源稳定性
   - 质量可控
   - 成本优势
```

### 商品分类

```
商品矩阵：

1. 引流款 (10-20%)
   - 低价、刚需
   - 吸引流量
   - 不追求利润

2. 爆款 (30-40%)
   - 高转化、高销量
   - 主力销售
   - 合理利润

3. 利润款 (30-40%)
   - 高毛利
   - 贡献利润
   - 差异化产品

4. 形象款 (5-10%)
   - 高端产品
   - 品牌形象
   - 拉高定位
```

### 商品定价

```
定价策略：

1. 成本定价
   价格 = 成本 × (1 + 毛利率)

2. 竞争定价
   - 参考竞品价格
   - 略低/略高

3. 价值定价
   - 基于用户感知价值
   - 高价值高价格

4. 心理定价
   - 99元 vs 100元
   - 锚定效应

5. 组合定价
   - 套餐优惠
   - 满减活动
```

### 库存管理

```
库存指标：

1. 库存周转率
   周转率 = 销售成本 / 平均库存

2. 库存周转天数
   周转天数 = 365 / 周转率

3. 库存预警
   - 安全库存
   - 补货点
   - 最大库存

库存优化：
- ABC分类管理
- 动销分析
- 滞销处理
```

   - 效果复盘

## 客户运营

### 客户分层

```
RFM分层：

R (Recency) 最近购买时间
F (Frequency) 购买频次
M (Monetary) 购买金额

| 类型 | R | F | M | 策略 |
|------|---|---|---|------|
| 重要价值 | 高 | 高 | 高 | VIP服务 |
| 重要发展 | 高 | 低 | 高 | 提升频次 |
| 重要保持 | 低 | 高 | 高 | 召回激活 |
| 重要挽留 | 低 | 低 | 高 | 重点挽回 |
| 一般价值 | 高 | 高 | 低 | 提升客单价 |
| 一般发展 | 高 | 低 | 低 | 培养习惯 |
| 一般保持 | 低 | 高 | 低 | 维持关系 |
| 低价值 | 低 | 低 | 低 | 低成本维护 |
```

### 客户运营策略

```
1. 新客户运营
   - 新人优惠
   - 首单转化
   - 留存引导

2. 活跃客户运营
   - 会员权益
   - 积分激励
   - 专属优惠

3. 沉默客户运营
   - 召回触达
   - 优惠刺激
   - 内容吸引

4. 流失客户运营
   - 流失预警
   - 挽回策略
   - 原因分析
```

## 服务运营

### 客服管理

```
客服指标：

| 指标 | 目标 | 说明 |
|------|------|------|
| 响应时间 | <30秒 | 首次响应 |
| 解决率 | >90% | 一次解决 |
| 满意度 | >95% | 服务评价 |
| 转化率 | >30% | 咨询转化 |

客服技巧：
1. 快速响应
2. 专业解答
3. 态度友好
4. 主动推荐
```

### 售后管理

```
售后流程：

1. 退款/退货
   - 快速处理
   - 减少纠纷

2. 投诉处理
   - 及时响应
   - 合理赔偿

3. 评价管理
   - 引导好评
   - 处理差评

售后指标：
- 退款率 <5%
- 纠纷率 <1%
- 差评率 <1%
```

## 数据分析

### 日常监控

```
日报指标：
- GMV
- 订单量
- 访客数
- 转化率
- 客单价
- 退款率

周报分析：
- 周度趋势
- 活动效果
- 商品表现
- 用户分析

月报总结：
- 月度业绩
- 同比环比
- 问题分析
- 优化建议
```

### 分析方法

```
1. 漏斗分析
   访问 → 浏览 → 加购 → 下单 → 支付
   分析各环节转化率

2. 同期群分析
   按时间分组分析用户行为
   分析用户生命周期

3. 商品分析
   销量分析
   利润分析
   库存分析

4. 用户分析
   用户画像
   行为分析
   价值分析
```
