# 广告变现 (Ad Monetization)

## 定义
广告变现是通过在产品中展示广告获得收入的商业模式
是互联网产品最常见的变现方式之一。

## 广告类型

### 按形式分类

| 类型 | 形式 | eCPM范围 | 用户体验 |
|------|------|----------|----------|
| Banner | 横幅广告 | 5-30元 | 一般 |
| 信息流 | Feed流原生 | 20-100元 | 较好 |
| 开屏 | APP开屏 | 50-200元 | 较好 |
| 插屏 | 弹窗广告 | 30-80元 | 较差 |
| 激励视频 | 用户主动观看 | 100-300元 | 好 |
| 贴片 | 视频贴片 | 30-100元 | 一般 |
| 原生广告 | 融入内容 | 50-150元 | 好 |

### 按计费方式分类

| 类型 | 说明 | 计算方式 |
|------|------|----------|
| CPM | 千次展示成本 | 成本/展示×1000 |
| CPC | 单次点击成本 | 成本/点击 |
| CPA | 单次行动成本 | 成本/转化 |
| oCPM | 优化千次展示 | 系统优化 |
| oCPC | 优化点击 | 系统优化 |
| CPT | 按时长计费 | 成本/天数 |

## 广告收入计算

### 基础公式

```
广告收入 = 展示量 × 展示率 × 点击率 × CPC
        = 展示量 × eCPM / 1000

其中：
- 展示量：广告展示次数
- 展示率：广告展示/请求
- 点击率(CTR)：点击/展示
- CPC：单次点击价格
- eCPM：千次展示收益
```

### eCPM计算

```
eCPM = (广告收入 / 展示量) × 1000

展开：
eCPM = CTR × CPC × 1000

示例：
CTR = 2%
CPC = 1元

eCPM = 0.02 × 1 × 1000 = 20元
```

### 收入预测

```
月广告收入预测：

月收入 = DAU × 人均广告展示 × eCPM / 1000 × 30

示例：
DAU = 100万
人均展示 = 20次
eCPM = 30元

月收入 = 100万 × 20 × 30 / 1000 × 30 = 1800万元
```

## 广告系统

### 广告系统组成

```
┌─────────────────────────────────────────────┐
│              广告系统架构                     │
├─────────────┬─────────────┬─────────────────┤
│   广告主     │   广告平台   │    流量方       │
├─────────────┼─────────────┼─────────────────┤
│  投放系统    │   竞价系统   │    SDK集成      │
│  创意管理    │   定向系统   │    广告位管理   │
│  数据报表    │   结算系统   │    收益统计     │
└─────────────┴─────────────┴─────────────────┘
```

### 广告请求流程

```
用户访问 → 广告请求 → 竞价排序 → 广告返回 → 展示 → 点击 → 转化
   ↓          ↓          ↓          ↓        ↓       ↓       ↓
  触发      携带信息    多方竞价   最优广告   记录   记录   记录
```

### 竞价机制

```
常见竞价方式：

1. 一价竞价
   - 最高出价者获胜
   - 按出价支付

2. 二价竞价
   - 最高出价者获胜
   - 按第二高出价支付

3. 综合竞价
   综合得分 = 出价 × 质量分
   - 质量分考虑CTR、相关性等
```

## 广告位设计

### 广告位类型

| 位置 | 类型 | 效果 | 注意事项 |
|------|------|------|----------|
| 首页信息流 | 信息流 | 高 | 融入内容 |
| 列表页 | 信息流/插入 | 高 | 不打断浏览 |
| 详情页 | 原生/横幅 | 中 | 相关性强 |
| 搜索结果 | 搜索广告 | 高 | 相关性要求高 |
| 启动页 | 开屏 | 高 | 时长控制 |
| 视频流 | 贴片/信息流 | 高 | 不影响观看 |
| 游戏内 | 激励视频 | 高 | 用户主动 |

### 广告位设计原则

```
1. 原生融入
   - 与内容形式一致
   - 不突兀

2. 不打断体验
   - 不影响核心功能
   - 可关闭/跳过

3. 适当频次
   - 不过度展示
   - 控制展示间隔

4. 相关性
   - 广告与内容相关
   - 提升点击率

5. 用户体验
   - 不误导用户
   - 清晰标识广告
```

### 广告频次控制

```
频次控制策略：

1. 日频次上限
   - 每用户每天最多展示N次

2. 时间间隔
   - 两次广告间隔至少N秒

3. 会话频次
   - 每次会话最多展示N次

4. 场景控制
   - 核心流程少广告
   - 非核心可适当增加

建议频次：
- 信息流：每10-15条内容1条广告
- 激励视频：用户主动触发
- 开屏：每日1-3次
```

## 广告变现优化

### 提升展示量

```
优化方向：

1. 增加广告位
   - 合理位置增加
   - 不影响体验

2. 提升填充率
   - 多广告源
   - 瀑布流策略

3. 提升展示率
   - 优化加载速度
   - 预加载策略
```

### 提升点击率

```
优化方向：

1. 广告相关性
   - 精准定向
   - 内容匹配

2. 广告创意
   - 吸引人的创意
   - A/B测试

3. 广告位置
   - 高可见区域
   - 首屏位置

4. 广告形式
   - 原生广告
   - 视频广告
```

### 提升eCPM

```
优化方向：

1. 广告源优化
   - 高价值广告源
   - 多平台接入

2. 竞价优化
   - 实时竞价
   - 底价设置

3. 流量分层
   - 高价值流量高价卖
   - 低价值流量低价卖

4. 数据价值
   - 提供用户数据
   - 精准定向
```

## 广告变现注意事项

### 用户体验

```
注意事项：

1. 不过度商业化
   - 广告占比控制
   - 核心功能优先

2. 不误导用户
   - 明确广告标识
   - 不伪装成内容

3. 不强制观看
   - 可跳过
   - 用户选择权

4. 不影响性能
   - 加载优化
   - 不卡顿
```

### 合规要求

```
合规要点：

1. 广告标识
   - 明确标注"广告"
   - 符合广告法

2. 内容审核
   - 广告内容合规
   - 敏感词过滤

3. 用户隐私
   - 隐私政策
   - 数据合规

4. 平台规则
   - 遵守平台政策
   - 避免违规
```

## 广告变现数据

### 核心指标

| 指标 | 计算公式 | 说明 |
|------|----------|------|
| 展示量 | 广告展示次数 | 流量规模 |
| 填充率 | 返回广告/请求 | 广告填充 |
| 展示率 | 展示/返回 | 实际展示 |
| 点击率 | 点击/展示 | 广告吸引力 |
| eCPM | 收入/展示×1000 | 千次收益 |
| ARPU | 收入/用户 | 人均收入 |

### 数据分析

```
分析维度：

1. 时间维度
   - 日/周/月趋势
   - 时段分布

2. 广告位维度
   - 各广告位效果
   - 优化方向

3. 用户维度
   - 新老用户ARPU
   - 用户分层

4. 渠道维度
   - 各平台收益
   - 渠道优化
```
