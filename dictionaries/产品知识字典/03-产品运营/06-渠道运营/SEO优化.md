# SEO优化 (Search Engine Optimization)

## 定义
SEO (Search Engine Optimization) 是搜索引擎优化，通过优化网站内容和结构
提升在搜索引擎中的自然排名，从而获得更多免费流量。

## SEO核心要素

### 影响排名的因素

```
┌─────────────────────────────────────────────┐
│              SEO影响因素                      │
├─────────────┬─────────────┬─────────────────┤
│   站内优化   │   站外优化   │    技术优化      │
├─────────────┼─────────────┼─────────────────┤
│  内容质量    │   外链质量   │    网站速度      │
│  关键词     │   社交信号   │    移动适配      │
│  内链结构    │   品牌提及   │    结构化数据    │
│  用户体验    │            │    HTTPS        │
└─────────────┴─────────────┴─────────────────┘
```

### 权重分布（估算）

| 因素 | 权重 | 说明 |
|------|------|------|
| 内容相关性 | 25% | 内容与搜索意图匹配度 |
| 内容质量 | 20% | 内容深度、原创性 |
| 外链质量 | 20% | 外部链接的质量和数量 |
| 用户体验 | 15% | 停留时间、跳出率 |
| 技术因素 | 10% | 速度、移动端、安全 |
| 社交信号 | 5% | 社交媒体提及 |
| 品牌信号 | 5% | 品牌搜索量 |

## 关键词研究

### 关键词类型

| 类型 | 搜索意图 | 示例 |
|------|----------|------|
| 导航词 | 找特定网站 | "淘宝官网" |
| 信息词 | 获取信息 | "怎么做红烧肉" |
| 商业词 | 研究产品 | "iPhone 15评测" |
| 交易词 | 完成交易 | "iPhone 15价格" |

### 关键词挖掘

```
挖掘方法：

1. 搜索建议
   - Google/百度搜索建议
   - 相关搜索

2. 关键词工具
   - Google Keyword Planner
   - 百度指数
   - 5118
   - Ahrefs
   - SEMrush

3. 竞品分析
   - 分析竞品关键词
   - 查看竞品排名

4. 用户调研
   - 用户搜索词
   - 问答平台
```

### 关键词评估

```
评估维度：

1. 搜索量
   - 月搜索量
   - 趋势变化

2. 竞争度
   - 排名难度
   - 竞争对手实力

3. 相关性
   - 与业务相关
   - 与内容匹配

4. 转化价值
   - 搜索意图
   - 商业价值

关键词得分 = 搜索量 × 0.3 + 相关性 × 0.3 + 低竞争 × 0.2 + 转化价值 × 0.2
```

### 关键词布局

```
布局位置：

1. 标题 (Title)
   - 包含核心关键词
   - 长度：中文≤30字，英文≤60字符
   - 格式：核心词 - 品牌词

2. 描述 (Description)
   - 包含关键词变体
   - 长度：中文≤80字，英文≤160字符
   - 吸引点击

3. H1标签
   - 包含核心关键词
   - 每页只有一个H1

4. 正文
   - 关键词密度2-3%
   - 自然融入
   - 同义词使用

5. URL
   - 包含关键词
   - 简洁有意义

6. 图片Alt
   - 描述图片内容
   - 包含相关关键词
```

## 内容优化

### 内容质量标准

```
高质量内容特征：

1. 原创性
   - 独特观点/内容
   - 非抄袭/采集

2. 深度
   - 内容全面
   - 有独到见解

3. 时效性
   - 内容新鲜
   - 定期更新

4. 可读性
   - 结构清晰
   - 易于阅读

5. 价值性
   - 解决用户问题
   - 提供有用信息
```

### 内容结构

```
优化结构：

1. 标题
   - 包含关键词
   - 吸引点击

2. 开头
   - 100字内说明核心内容
   - 吸引用户继续阅读

3. 小标题
   - H2/H3分层
   - 包含相关关键词

4. 正文
   - 段落简短
   - 图文结合
   - 列表清晰

5. 内链
   - 相关文章链接
   - 锚文本自然

6. 总结
   - 核心观点总结
   - 行动召唤
```

### 内容长度

```
建议长度：

| 内容类型 | 建议字数 |
|----------|----------|
| 博客文章 | 1500-2500字 |
| 产品页 | 500-1000字 |
| 落地页 | 800-1500字 |
| 深度文章 | 3000字以上 |

注意：
- 长度不是唯一标准
- 质量比长度重要
- 满足用户需求即可
```

## 技术SEO

### 网站速度

```
速度优化：

1. 图片优化
   - 压缩图片
   - 使用WebP格式
   - 懒加载

2. 代码优化
   - 压缩CSS/JS
   - 合并文件
   - 延迟加载

3. 服务器优化
   - CDN加速
   - 缓存策略
   - Gzip压缩

速度标准：
- 首屏加载 < 2秒
- 完全加载 < 3秒
```

### 移动端优化

```
优化要点：

1. 响应式设计
   - 适配各尺寸
   - 移动优先

2. 移动体验
   - 字体大小合适
   - 按钮大小合适
   - 避免弹窗

3. AMP (可选)
   - 加速移动页面
   - 提升体验

4. 移动测试
   - Google移动测试工具
   - 确保无问题
```

### 网站结构

```
结构优化：

1. URL结构
   - 层级不超过3层
   - 包含关键词
   - 静态URL

2. 导航
   - 清晰的导航结构
   - 面包屑导航
   - 站点地图

3. 内链
   - 相关页面互相链接
   - 锚文本自然
   - 重要页面多内链

4. 结构化数据
   - Schema标记
   - 帮助搜索引擎理解
```

### 技术检查清单

```
□ 网站速度 < 3秒
□ 移动端友好
□ HTTPS安全证书
□ 无死链
□ 无重复内容
□ XML站点地图
□ Robots.txt配置
□ 结构化数据
□ 404页面友好
□ 规范化标签(Canonical)
```

## 外链建设

### 外链类型

| 类型 | 质量 | 获取难度 |
|------|------|----------|
| 自然外链 | 最高 | 难 |
| 友情链接 | 中等 | 中 |
| 内容营销 | 高 | 中 |
| 嘉宾博客 | 高 | 中 |
| 目录提交 | 低 | 易 |
| 论坛签名 | 低 | 易 |

### 高质量外链特征

```
高质量外链标准：

1. 相关性
   - 与网站主题相关

2. 权威性
   - 来源网站权重高
   - DA/PA值高

3. 自然性
   - 锚文本多样
   - 增长自然

4. 多样性
   - 来源网站多样
   - 类型多样
```

### 外链建设方法

```
建设策略：

1. 内容营销
   - 创建优质内容
   - 自然吸引外链

2. 嘉宾博客
   - 为其他网站写文章
   - 获得外链

3. 资源页链接
   - 寻找资源页
   - 请求添加链接

4. 破损链接建设
   - 找到破损链接
   - 提供替代内容

5. 竞品外链分析
   - 分析竞品外链来源
   - 获取相同外链
```

```
本地SEO要点：

4. 本地内容
   - 本地相关内容
   - 本地活动

5. 本地外链
   - 本地目录
   - 本地媒体
```

## SEO最佳实践

### 1. 内容为王
- 高质量内容是基础
- 满足用户搜索意图
- 持续产出优质内容

### 2. 技术保障
- 网站速度要快
- 移动端友好
- 技术SEO无问题

### 3. 用户体验
- 页面体验要好
- 降低跳出率
- 提升停留时间

### 4. 长期主义
- SEO是长期工作
- 避免黑帽手法
- 建立长期价值

### 5. 数据驱动
- 用数据指导优化
- 持续监控效果
- 不断迭代优化
