# SEM投放 (Search Engine Marketing)

## 定义
SEM (Search Engine Marketing) 是搜索引擎营销，主要是通过在搜索引擎上投放付费广告
获得搜索流量。常见形式是搜索广告（PPC）。

## SEM投放流程

```
账户搭建 → 关键词规划 → 创意撰写 → 投放设置 → 数据监控 → 优化迭代
```

## 账户结构

### 层级结构

```
账户 (Account)
  │
  ├── 推广计划 (Campaign)
  │     │
  │     ├── 推广单元 (Ad Group)
  │     │     │
  │     │     ├── 关键词 (Keyword)
  │     │     └── 创意 (Ad)
  │     │
  │     ├── 推广单元
  │     └── ...
  │
  └── 推广计划
```

### 账户搭建原则

```
搭建原则：

1. 结构清晰
   - 按产品/服务分计划
   - 按关键词主题分单元

2. 便于管理
   - 层级不超过3层
   - 单元关键词10-30个

3. 便于优化
   - 数据可追踪
   - 便于调整

示例结构：
账户
├── 品牌计划
│   ├── 品牌词单元
│   └── 竞品词单元
├── 产品计划
│   ├── 产品A单元
│   ├── 产品B单元
│   └── 产品C单元
└── 通用词计划
    ├── 行业词单元
    └── 长尾词单元
```

## 关键词规划

### 关键词类型

| 类型 | 特点 | 竞价策略 |
|------|------|----------|
| 品牌词 | 转化高、竞争小 | 必须投放、排名第一 |
| 产品词 | 转化中、竞争中 | 重点投放 |
| 通用词 | 转化低、竞争大 | 选择性投放 |
| 竞品词 | 转化中、竞争中 | 视情况投放 |
| 长尾词 | 转化高、量小 | 性价比高 |

### 关键词挖掘

```
挖掘方法：

1. 搜索词报告
   - 分析用户实际搜索词
   - 添加有效词
   - 否定无效词

2. 关键词规划师
   - Google Keyword Planner
   - 百度关键词规划师

3. 竞品分析
   - 分析竞品关键词
   - 借鉴学习

4. 下拉推荐
   - 搜索引擎推荐词
   - 相关搜索

5. 第三方工具
   - 5118
   - 站长工具
```

### 关键词匹配方式

| 匹配方式 | 触发条件 | 示例(购买"英语培训") |
|----------|----------|---------------------|
| 精确匹配 | 完全一致 | "英语培训" |
| 短语匹配 | 包含且顺序一致 | "北京英语培训" |
| 广泛匹配 | 相关即可 | "学英语" |
| 广泛匹配修饰 | 包含核心词 | "英语学习培训" |

### 关键词出价

```
出价策略：

1. 排名预估出价
   - 根据目标排名出价
   - 参考竞价排名工具

2. ROI导向出价
   最高出价 = 可接受CPA × 转化率
   例：CPA 100元，转化率 10%
   最高出价 = 100 × 10% = 10元

3. 竞价策略
   - 手动出价：精准控制
   - 自动出价：系统优化
   - 增强CPC：智能调整

出价建议：
- 品牌词：确保第一
- 产品词：前三位
- 通用词：前五位
- 长尾词：根据ROI决定
```

### 否定关键词

```
否定关键词作用：
- 避免无效点击
- 降低成本
- 提高精准度

添加来源：
1. 搜索词报告
   - 分析不相关搜索词
   - 添加为否定词

2. 业务判断
   - 不提供的服务
   - 不相关的词

示例：
业务：高端英语培训
否定词：免费、低价、自学
```

## 创意撰写

### 创意要素

```
创意组成：
- 标题：25-30字
- 描述1：35-80字
- 描述2：35-80字
- 显示URL：可见的网址
- 访问URL：实际跳转地址
```

### 创意撰写原则

```
1. 相关性
   - 与关键词相关
   - 与落地页相关

2. 吸引力
   - 突出卖点
   - 差异化

3. 行动召唤
   - 引导点击
   - 明确行动

4. 通顺性
   - 语句通顺
   - 逻辑清晰
```

### 创意模板

```
模板1（问题解决型）：
标题：{关键词}_专业解决方案
描述：困扰{问题}？我们提供{解决方案}，{卖点}。立即咨询，{优惠}。

模板2（优势突出型）：
标题：{关键词}_{核心优势}
描述：{产品}提供{服务}，{优势1}，{优势2}。{行动召唤}。

模板3（促销型）：
标题：{关键词}_限时{优惠}
描述：{产品}{优惠活动}，仅剩{数量}！{行动召唤}。

示例：
标题：英语培训_3个月突破口语
描述：专业外教一对一，3个月突破口语障碍。免费试听，不满意全额退款。立即预约！
```

### 创意优化

```
优化方法：

1. A/B测试
   - 多套创意轮播
   - 选择点击率高的

2. 动态创意
   - 通配符{关键词}
   - 自动插入搜索词

3. 高级样式
   - 附加链接
   - 附加信息
   - 图片/视频

4. 定期更新
   - 避免创意疲劳
   - 结合活动更新
```

## 投放设置

### 地域设置

```
设置原则：
- 业务覆盖范围
- 目标客户分布
- 预算分配

策略：
1. 全国投放
   - 统一出价
   
2. 分地域投放
   - 不同地域不同计划
   - 差异化出价

3. 地域定向
   - 重点城市加码
   - 其他城市保守
```

### 时间设置

```
设置策略：

1. 全天投放
   - 适合品牌词
   - 预算充足

2. 时段优化
   - 分析各时段转化
   - 高转化时段重点投放

3. 时段差异化
   - 高峰时段高出价
   - 低峰时段低出价

建议时段：
- B2B：9:00-18:00
- B2C：9:00-22:00
```

### 预算设置

```
预算分配：

1. 日预算
   日预算 = 月预算 / 30 × 1.1（留10%弹性）

2. 计划预算
   - 核心计划：50-60%
   - 重点计划：30-40%
   - 测试计划：10%

3. 预算调整
   - 根据消耗速度调整
   - 高转化计划加预算
```

## 数据监控

### 核心指标

| 指标 | 计算公式 | 说明 |
|------|----------|------|
| 展现量 | Impressions | 广告展示次数 |
| 点击量 | Clicks | 广告点击次数 |
| 点击率 | CTR = 点击/展现 | 创意吸引力 |
| CPC | 花费/点击 | 单次点击成本 |
| CPM | (花费/展现)×1000 | 千次展示成本 |
| 转化量 | Conversions | 完成目标次数 |
| 转化率 | 转化/点击 | 转化效果 |
| CPA | 花费/转化 | 单次转化成本 |
| ROI | (收入-成本)/成本 | 投资回报率 |

## 投放优化

### 优化方向

```
1. 展现量优化
   - 增加关键词
   - 提高出价
   - 扩大匹配方式

2. 点击率优化
   - 优化创意
   - 提高排名
   - 添加否定词

3. 转化率优化
   - 优化落地页
   - 精准关键词
   - 提升相关性

4. 成本优化
   - 降低CPC
   - 提高转化率
   - 否定无效词
```

### 关键词优化

```
优化动作：

1. 添加关键词
   - 搜索词报告中的有效词

2. 调整出价
   - 高转化词提高出价
   - 低转化词降低出价

3. 调整匹配方式
   - 精准化匹配
   - 控制流量质量

4. 添加否定词
   - 不相关搜索词
   - 降低无效消耗

5. 暂停关键词
   - 长期无转化词
   - 成本过高词
```

### 创意优化

```
优化方法：

1. A/B测试
   - 测试不同创意
   - 优胜劣汰

2. 提升相关性
   - 通配符使用
   - 与关键词匹配

3. 突出卖点
   - 差异化优势
   - 用户利益点

4. 行动召唤
   - 引导点击
   - 紧迫感营造
```

### 落地页优化

```
优化要点：

1. 相关性
   - 与广告创意一致
   - 满足用户预期

2. 加载速度
   - 3秒内打开
   - 移动端优化

3. 转化路径
   - 简化流程
   - 明确CTA

4. 信任建设
   - 用户见证
   - 资质展示

5. A/B测试
   - 测试不同版本
   - 提升转化率
```
