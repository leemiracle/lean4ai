## 持续学习

### 学习方法

#### 刻意练习
```markdown
步骤:
1. 设定目标
2. 专注练习
3. 获得反馈
4. 纠正改进
5. 重复练习

应用:
- 写PRD:每周写2个
- 原型设计:每周3个界面
- 数据分析:每周1个分析
```

#### 费曼学习法
```markdown
步骤:
1. 选择要学习的概念
2. 用简单语言解释
3. 发现知识漏洞
4. 回顾和简化

应用:
- 学习新知识后,讲给别人听
- 写文章输出
- 制作思维导图
```

#### 输出倒逼输入
```markdown
方式:
- 写文章
- 做分享
- 录视频
- 教别人

好处:
- 加深理解
- 建立影响力
- 获得反馈
```

```markdown
- 完成1篇文章阅读

```

#### 月计划
```markdown
- 读完1本书
- 完成1个学习模块
- 输出1篇深度文章
- 参加1次行业活动
```

#### 年计划
```markdown
- 掌握1-2项核心技能
- 读10-20本书
- 完成1个认证
- 建立个人影响力
```

### 资源筛选
```markdown
标准:
- 权威性: 谁做的?什么背景?
- 实用性: 能用上吗?
- 时效性: 是否过时?
- 评价: 别人怎么说?

建议:
- 看评价和推荐
- 先试听/试读
- 选择高质量资源
```

## 学习建议

### 避免误区
```markdown
误区1: 收藏≠学习
解决: 收藏后要及时学习

误区2: 只输入不输出
解决: 强制输出,倒逼输入

误区3: 贪多不求精
解决: 精选资源,深度学习

误区4: 学习不应用
解决: 学以致用,实践验证
```

### 提升效率
```markdown
1. 明确学习目标
2. 选择合适资源
3. 制定学习计划
4. 坚持执行
5. 定期复盘
6. 输出分享
```

### 保持动力
```markdown
1. 找到学习的意义
2. 设定小目标
3. 及时奖励自己
4. 找学习伙伴
5. 看到成长进步
6. 享受学习过程
```
