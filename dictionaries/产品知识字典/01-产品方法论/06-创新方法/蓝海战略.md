# 蓝海战略

## 一、定义

**蓝海战略**是由W. Chan Kim和Renée Mauborgne提出的战略框架，主张通过价值创新开创无人竞争的市场空间（蓝海），而非在拥挤的市场中竞争（红海）。

```
红海 vs 蓝海

红海战略              蓝海战略
─────────────────────────────────────
在已有市场空间竞争    创造无人竞争的市场空间
打败竞争对手          让竞争对手无关紧要
利用现有需求          创造和获取新需求
价值-成本权衡         打破价值-成本权衡
协调价值链活动        整个系统协同

红海：竞争激烈，利润微薄
蓝海：创新空间，利润丰厚
```

## 二、价值创新

### 2.1 核心概念

**价值创新**是蓝海战略的基石，同时追求差异化和低成本。

```
价值创新 = 价值 + 创新

        │
 价值   │   蓝海战略
        │   （价值+创新）
        │       ●
        │
────────┼──────────────→
        │           创新
        
传统思维：
- 高价值 → 高成本
- 低成本 → 低价值

蓝海思维：
- 重新定义价值
- 创造新的需求
- 同时实现差异化和低成本
```

### 2.2 价值创新的实现

```
价值创新实现路径

1. 重新定义市场边界
   不局限于现有市场定义
   
2. 关注非用户
   扩大市场边界

3. 重新排序行业战略要素
   考虑剔除、减少、增加、创造

4. 跨越时间看趋势
   预判未来趋势

示例：任天堂Wii
- 传统游戏机：追求高性能、高画质
- Wii：降低性能、增加体感互动
- 结果：吸引非游戏玩家，开创蓝海
```

## 三、四步动作框架

### 3.1 基本概念

**四步动作框架**(ERRC Grid)是蓝海战略的核心工具，通过四个维度重新定义产品价值。

```
四步动作框架

          消除 (Eliminate)
               ↑
               │  行业内哪些因素
               │  应该被消除？
               │
减少 ──────────┼────────── 增加
(Reduce)      │          (Raise)
行业内哪些    │          行业内哪些
因素应该      │          因素应该
降低到标准    │          提升到标准
以下？        │          以上？
               │
               ↓
          创造 (Create)
          哪些因素是行业
          从未提供的？
```

### 3.2 四步动作应用

```
案例：太阳马戏团

传统马戏团 → 太阳马戏团

消除(Eliminate):
- 动物表演（成本高、争议大）
- 明星演员（成本高）
- 多个舞台（体验差）

减少(Reduce):
- 刺激元素（减少小丑、惊险）
- 场地座位（专注体验）

增加(Raise):
- 帐篷的舒适性
- 艺术性和故事性

创造(Create):
- 主题和故事线
- 艺术音乐和舞蹈
- 高雅的观赏环境

结果：
- 创造全新表演形式
- 吸引成年观众（非传统马戏观众）
- 票价是传统马戏的数倍
```

### 3.3 四步动作框架案例

| 行业 | 消除 | 减少 | 增加 | 创造 |
|-----|------|------|------|------|
| 西南航空 | 座位等级| 餐饮服务| 频次| 点对点|
| 黄尾葡萄酒| 葡萄酒术语| 单宁和橡木| 甜美果香| 简单选择|
| 小米手机| 线下渠道| 营销费用| 性价比| 互联网模式|
| 戴森| 尘袋| 重量| 吸力| 无线设计|

## 四、战略布局图

### 4.1 基本概念

**战略布局图**是可视化工具，用于比较不同产品在各竞争要素上的表现。

```
战略布局图示例：美国葡萄酒行业

价值
  ↑
高 │            传统高端葡萄酒
  │         ╱──────╲
  │      ╱           ╲
  │   ╱                 ╲
中 │─────────────────────  黄尾
  │   ╱─────────────────────╲
  │
低 │╱                          ╲
  └────────────────────────────→ 要素
   价格 营销 品质 包装 口感 易选

传统葡萄酒：高价、重营销、复杂
黄尾葡萄酒：低价、简单、易选择
```

### 4.2 战略布局图应用

```
步骤1: 识别行业竞争要素
列出行业内竞争的关键因素

步骤2: 绘制行业战略曲线
标出行业平均水平

步骤3: 绘制竞争对手曲线
标出主要竞争对手

步骤4: 绘制你的战略曲线
用ERRC框架重新设计

步骤5: 对比分析
寻找差异化机会

好的蓝海战略曲线：
- 与行业曲线明显不同
- 聚焦重点价值点
- 满足用户真实需求
```

### 4.3 战略布局图案例

```
案例：经济型酒店 vs 传统酒店

价值
  ↑
高 │ 传统五星酒店
  │ ╱────────╲
  │           ╲
中 │            ╲ 经济型酒店
  │         ╱────╲──────╲
  │      ╱
低 │   ╱
  └───────────────────────→ 要素
   价格 餐饮 健身 客房服务 床品 位置

如家酒店（经济型）：
- 消除：餐饮、健身、奢华装修
- 减少：客房服务、房间面积
- 增加：床品质量、干净程度
- 创造：标准化、连锁化
```

## 五、蓝海战略制定流程

### 5.1 六步路径框架

```
蓝海战略六步路径

1. 重建市场边界
   ┌─────────────────┐
   │ 跨越替代行业    │
   │ 跨越战略集团    │
   │ 跨越买家链      │
   │ 跨越互补产品    │
   │ 跨越功能-情感   │
   │ 跨越时间        │
   └────────┬────────┘
            ↓
2. 关注全景而非数字
   ┌─────────────────┐
   │ 使用战略布局图  │
   │ 看全局而非细节  │
   └────────┬────────┘
            ↓
3. 超越现有需求
   ┌─────────────────┐
   │ 关注非用户      │
   │ 扩大市场边界    │
   └────────┬────────┘
            ↓
4. 遵循合理战略顺序
   ┌─────────────────┐
   │ 买方效用→价格  │
   │ →成本→接受     │
   └────────┬────────┘
            ↓
5. 克服关键组织障碍
   ┌─────────────────┐
   │ 认知、资源、    │
   │ 动力、政治      │
   └────────┬────────┘
            ↓
6. 将战略执行建成战略
   ┌─────────────────┐
   │ 公平流程        │
   │ 战略执行一致性  │
   └─────────────────┘
```

### 5.2 重建市场边界

```
重建市场边界的六条路径

路径1: 跨越替代行业
示例：咖啡馆 vs 茶馆 vs 果汁店

路径2: 跨越战略集团
示例：经济型车 vs 豪华车 → 智能汽车

路径3: 跨越买家链
示例：从采购者到最终用户

路径4: 跨越互补产品和服务
示例：电影+爆米花 → 影院体验

路径5: 跨越功能-情感导向
示例：功能型工具 → 情感化品牌

路径6: 跨越时间参与塑造的外部趋势
示例：疫情 → 远程办公工具
```

## 六、蓝海战略案例

### 6.1 案例：苹果iPhone

```
苹果iPhone蓝海战略

2007年手机市场：
- 诺基亚主导功能机
- 黑莓主导商务机
- 智能机体验差

苹果蓝海战略：

消除(Eliminate):
- 物理键盘
- 手写笔
- 复杂按钮

减少(Reduce):
- 机身厚度
- 说明书

增加(Raise):
- 屏幕尺寸
- 触控体验
- 设计美感

创造(Create):
- 多点触控
- App Store生态
- 手机+互联网+音乐一体化

战略布局图：
传统手机：按键多、屏幕小、功能单一
iPhone：触控大屏、简洁设计、生态系统

结果：
- 重新定义智能手机
- 创造全新市场
- 成为最赚钱手机
```

滴滴蓝海战略：

增加(Raise):

创造(Create):

- 创造共享经济

## 八、蓝海战略最佳实践

DO:
✓ 关注非用户而非竞争对手
✓ 同时追求差异化和低成本
✓ 用四步动作框架打破行业惯例
✓ 绘制战略布局图可视化差异
✓ 创造而非瓜分市场
✓ 理解用户真实需求

DON'T:
✗ 陷入红海竞争思维
✗ 只关注现有用户
✗ 在价值-成本之间做权衡
✗ 模仿竞争对手
✗ 忽视执行障碍
✗ 停留在战略层面

记住：蓝海战略的核心是价值创新——不是打败竞争对手，而是让竞争对手变得无关紧要。
