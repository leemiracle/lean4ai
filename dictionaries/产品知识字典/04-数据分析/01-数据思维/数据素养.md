# 数据素养培养

## 1. 定义

数据素养（Data Literacy）是指理解、创建和使用数据的能力，包括数据的采集、管理、分析、解释和传播等综合能力。

### 核心能力维度

```
数据素养 = 数据理解 + 数据分析 + 数据应用 + 数据伦理
```

## 2. 数据素养能力模型

### 2.1 五层能力模型

```
Level 5: 数据文化领导者
├─ 制定数据战略

└─ 建设数据文化

Level 4: 数据分析师
├─ 高级统计分析
├─ 预测建模
└─ 数据产品化

```

### 3.2 统计学基础

**描述性统计**

```python
import pandas as pd
import numpy as np

# 示例数据
data = pd.Series([23, 45, 67, 89, 34, 56, 78, 90, 12, 45])

# 集中趋势
mean = data.mean()      # 平均值
median = data.median()  # 中位数
mode = data.mode()      # 众数

# 离散程度
std = data.std()        # 标准差
var = data.var()        # 方差
range_val = data.max() - data.min()  # 极差
iqr = data.quantile(0.75) - data.quantile(0.25)  # 四分位距

# 分布形态
skewness = data.skew()  # 偏度
kurtosis = data.kurt()  # 峰度

print(f"""
均值: {mean:.2f}
中位数: {median:.2f}
标准差: {std:.2f}
偏度: {skewness:.2f}
""")
```

**假设检验**

```python
from scipy import stats

# t检验：比较两组均值是否有显著差异
group_a = [23, 45, 67, 89, 34]
group_b = [12, 34, 56, 78, 90]

t_stat, p_value = stats.ttest_ind(group_a, group_b)

print(f"t统计量: {t_stat:.2f}")
print(f"p值: {p_value:.4f}")

if p_value < 0.05:
    print("结论：两组存在显著差异（p<0.05）")
else:
    print("结论：两组无显著差异")
```

        experiment_group,
        COUNT(*) as n,
        SUM(converted) as conversions,
        AVG(converted) as conversion_rate,
        STDDEV(converted) as std
    FROM ab_test_results
    GROUP BY experiment_group
)

    n,

## 4. 数据分析方法

### 4.1 对比分析

**时间对比**
```sql
-- 同比、环比分析
SELECT 
    DATE_FORMAT(order_date, '%Y-%m') as month,
    SUM(amount) as current_amount,
    LAG(SUM(amount), 1) OVER (ORDER BY DATE_FORMAT(order_date, '%Y-%m')) as last_month,
    LAG(SUM(amount), 12) OVER (ORDER BY DATE_FORMAT(order_date, '%Y-%m')) as last_year,
    ROUND((SUM(amount) - LAG(SUM(amount), 1) OVER (ORDER BY DATE_FORMAT(order_date, '%Y-%m'))) 
        / LAG(SUM(amount), 1) OVER (ORDER BY DATE_FORMAT(order_date, '%Y-%m')) * 100, 2) as mom_growth,
    ROUND((SUM(amount) - LAG(SUM(amount), 12) OVER (ORDER BY DATE_FORMAT(order_date, '%Y-%m'))) 
        / LAG(SUM(amount), 12) OVER (ORDER BY DATE_FORMAT(order_date, '%Y-%m')) * 100, 2) as yoy_growth
FROM orders
WHERE order_date >= DATE_SUB(NOW(), INTERVAL 24 MONTH)
GROUP BY DATE_FORMAT(order_date, '%Y-%m')
ORDER BY month;
```

SELECT 
    date,
    daily_revenue,
    AVG(daily_revenue) OVER (
        ORDER BY date 
        ROWS BETWEEN 6 PRECEDING AND CURRENT ROW
    ) as ma_7,

### 4.3 归因分析

**贡献度分析**
```python
def contribution_analysis(current, previous):
    """
    计算各因素对整体变化的贡献度
    """
    change = current - previous
    contributions = {}
    
    for factor in current.index:
        factor_change = current[factor] - previous[factor]
        contributions[factor] = (factor_change / change) * 100
    
    return pd.Series(contributions).sort_values(ascending=False)

# 示例：GMV变化归因
current_gmv = pd.Series({
    '流量': 100000,
    '转化率': 0.05,
    '客单价': 200
})

previous_gmv = pd.Series({
    '流量': 80000,
    '转化率': 0.04,
    '客单价': 180
})

# 使用杜邦分析拆解
current_total = current_gmv['流量'] * current_gmv['转化率'] * current_gmv['客单价']
previous_total = previous_gmv['流量'] * previous_gmv['转化率'] * previous_gmv['客单价']

print(f"GMV变化：{current_total - previous_total}")
```

### 6.2 数据脱敏方法

```python
import hashlib
import pandas as pd

def data_masking(df):
    """
    数据脱敏处理
    """
    # 手机号脱敏
    df['phone_masked'] = df['phone'].apply(
        lambda x: str(x)[:3] + '****' + str(x)[-4:]
    )
    
    # 邮箱脱敏
    df['email_masked'] = df['email'].apply(
        lambda x: x.split('@')[0][:2] + '***@' + x.split('@')[1]
    )
    
    # ID哈希化
    df['user_id_hash'] = df['user_id'].apply(
        lambda x: hashlib.md5(str(x).encode()).hexdigest()[:16]
    )
    
    # 数值加噪
    df['age_noised'] = df['age'] + np.random.normal(0, 2, len(df))
    
    return df

# 使用示例
users = pd.DataFrame({
    'user_id': [1, 2, 3],
    'phone': ['13812345678', '13987654321', '15011112222'],
    'email': ['user1@example.com', 'user2@example.com', 'user3@example.com'],
    'age': [25, 30, 35]
})

masked_data = data_masking(users)
print(masked_data[['phone_masked', 'email_masked', 'user_id_hash']])
```
