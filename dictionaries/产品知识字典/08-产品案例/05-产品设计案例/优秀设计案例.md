## 二、Apple设计案例

### iOS设计系统

#### 设计原则
```
Apple设计三大原则：
1. 清晰（Clarity）
   ├─ 文字清晰易读
   ├─ 图标精确鲜明
   ├─ 装饰适度
   └─ 功能驱动设计

2. 遵从（Deference）
   ├─ 流畅的动态效果
   ├─ 界面清晰通透
   ├─ 内容优先
   └─ 交互自然

3. 深度（Depth）
   ├─ 视觉层次分明
   ├─ 操作反馈清晰
   ├─ 增强愉悦感
   └─ 空间感设计

设计哲学：
简单而不简陋
```

#### 设计细节

**1. 圆角设计**
```
iOS圆角系统：
├─ 图标圆角：22.37%连续曲率
├─ 卡片圆角：10-20px
├─ 按钮圆角：8-12px
├─ 输入框圆角：8-12px
└─ 弹窗圆角：14px

连续曲率圆角：
普通圆角：简单圆弧
Apple圆角：连续曲率曲线
优势：更柔和、更自然

设计理念：
细节中的精致
```

**2. 动效设计**
```
iOS动效原则：
1. 目的性
   └─ 动效服务于功能

2. 自然性
   └─ 符合物理规律

3. 一致性
   └─ 统一的动效语言

4. 适度性
   └─ 不喧宾夺主

典型动效：
├─ 弹簧动画：自然回弹
├─ 视差效果：层次感
├─ 转场动画：流畅切换
├─ 反馈动画：即时反馈
└─ 加载动画：等待体验

实现：
CAMediaTimingFunction + UIDynamics
```

**3. 空白运用**
```
留白（Negative Space）：
定义：元素之间的空白区域

Apple留白特点：
├─ 大量留白：呼吸感
├─ 节奏感：疏密有致
├─ 焦点突出：视觉聚焦
└─ 高级感：品质感

应用：
├─ 列表行高：44pt最小高度
├─ 内容边距：充足的空间
├─ 组件间距：清晰的分隔
└─ 页面留白：宽松舒适

价值：
留白是设计的一部分
```

## 八、设计方法论

```
案例：
├─ Apple HIG

├─ Airbnb DLS
└─ Stripe Design System
```

### 方法2：设计思维
```
Design Thinking流程：
1. 同理心（Empathize）
   └─ 理解用户

2. 定义（Define）
   └─ 定义问题

3. 构思（Ideate）
   └─ 创意发散

4. 原型（Prototype）
   └─ 快速原型

5. 测试（Test）
   └─ 用户测试

迭代：
持续迭代优化
```

### 方法3：极简设计
```
极简设计原则：
1. 删除非必要
   └─ 每个元素都有目的

2. 简化复杂
   └─ 降低认知负担

3. 留白运用
   └─ 呼吸感和高级感

4. 一致性
   └─ 统一的设计语言

5. 功能优先
   └─ 形式服务功能

代表：
Apple、Notion、Linear
```
