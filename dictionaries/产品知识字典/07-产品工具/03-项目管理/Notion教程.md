# Notion教程

## 概述
Notion是一款集笔记、知识库、任务管理、数据库于一体的协作工具,以模块化设计和强大灵活性著称,是产品经理个人知识管理和团队协作的利器。

## 核心功能

### 内容类型
- 页面(Page)
- 数据库(Database)
- 看板(Kanban)
- 日历(Calendar)
- 时间轴(Timeline)
- 画廊(Gallery)

### 基础模块(Block)
- 文本模块
- 标题模块
- 列表模块
- 媒体模块
- 嵌入模块
- 数据库模块

### 协作功能
- 实时协作
- 评论讨论
- 分享发布
- 模板库
- 团队空间

## 使用指南

### 页面创建
```
1. 创建新页面
   - 点击"New Page"
   - 选择模板或空白页
   - 添加图标和封面

2. 添加内容块
   - 输入/调用命令菜单
   - 选择内容类型
   - 编辑内容

3. 组织结构
   - 创建子页面
   - 拖拽调整顺序
   - 建立页面链接
```

### 数据库使用
```
数据库类型:
- Table: 表格视图
- Board: 看板视图
- List: 列表视图
- Calendar: 日历视图
- Gallery: 画廊视图
- Timeline: 时间轴视图

数据库属性:
- Select: 单选
- Multi-select: 多选
- Date: 日期
- Person: 人员
- Checkbox: 复选框
- Number: 数字
- URL: 链接
- Relation: 关联
- Rollup: 汇总
```

### 模板应用
```
使用模板:
1. 点击"Templates"
2. 选择类别
3. 预览并应用

创建模板:
1. 在数据库中创建模板按钮
2. 设计模板内容
3. 保存为可复用模板
```

## 常见问题

### Q: 如何与他人分享页面?
A: 点击右上角"Share",设置权限后复制分享链接

### Q: 数据库如何筛选和排序?
A: 点击视图上方的"Filter"或"Sort"添加筛选和排序规则

### Q: 如何导出Notion内容?
A: 点击右上角"...",选择"Export"导出为Markdown/PDF/HTML
