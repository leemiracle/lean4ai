# 飞书教程

## 概述
飞书是字节跳动推出的企业协作平台,整合即时沟通、日历、在线文档、视频会议、云盘等功能,为企业提供一站式办公解决方案。

## 核心功能

### 即时沟通
- 单聊/群聊
- 消息撤回
- 引用回复
- 表情回应
- 消息静音
- 置顶聊天

### 文档协作
- 在线文档
- 多人实时协作
- 评论互动
- 版本历史
- 模板库

### 日程管理
- 日历视图
- 会议预定
- 日程提醒
- 订阅日历
- 忙闲查询

### 视频会议
- 音视频通话
- 屏幕共享
- 会议录制
- 在线白板
- 妙记转写

### 应用中心
- OKR管理
- 审批流程
- 任务管理
- 表单收集
- 机器人

## 使用指南

### 消息沟通
```
基础操作:
- 发送消息: 输入后按Enter
- 换行: Shift+Enter
- @提醒: 输入@选择人员
- 表情: 点击表情图标或输入:
- 文件传输: 拖拽或点击附件

消息管理:
- 消息撤回: 右键选择撤回
- 消息引用: 右键选择引用
- 消息转发: 右键选择转发
- 消息收藏: 右键选择收藏
```

### 群组管理
```
创建群组:
1. 点击"+" -> 创建群聊
2. 选择群成员
3. 设置群名称
4. 上传群头像

群组设置:
- 群公告
- 群文件
- 群权限
- 水印设置
- 机器人管理
```

### 文档编辑
```
创建文档:
1. 文档 -> 新建文档
2. 选择模板或空白
3. 编辑内容

文档功能:
- 标题样式: # ## ###
- 列表: - 或 1.
- 任务: []
- 表格: 插入表格
- 图片: 拖拽上传
- 代码块: ```
```

### 日程管理
```
创建日程:
1. 日历 -> 创建日程
2. 填写日程信息
3. 添加参会人
4. 设置提醒时间

日程功能:
- 重复日程
- 日程订阅
- 忙闲查询
- 视频会议
```

## 实战案例

### 案例1:产品需求评审会议
```流程:
1. 提前准备
   - 创建需求文档
   - 分享文档链接到群里
   - @相关人员预览

2. 预定会议
   - 日历创建日程
   - 添加参会人员
   - 附上会议议题
   - 开启飞书会议

3. 会议进行
   - 投屏演示文档
   - 实时记录会议纪要
   - 讨论并更新文档

4. 会后跟进
   - 会议录制回放
   - 妙记转写整理
   - 分发待办任务
```

### 案例2:项目协作管理
```协作方式:
1. 创建项目群
   - 拉相关人员进群
   - 设置群公告说明项目
   - 添加项目机器人

2. 文档协作
   - 项目文档集中管理
   - 建立文档目录结构
   - 设置编辑/查看权限

3. 任务跟进
   - 创建任务列表
   - 分配负责人
   - 设置截止时间
   - 进度更新同步

4. 周报汇总
   - 每周在群里发送周报
   - 使用文档模板
   - @相关人确认
```

### 案例3:跨部门协作
```协作机制:
1. 建立协作群
   - 跨部门成员
   - 设置管理员
   - 明确协作规范

2. 信息同步
   - 重要信息群公告
   - 定期同步会议
   - 文档及时更新

3. 问题反馈
   - 使用飞书表单收集
   - 集中处理反馈
   - 进度及时同步
```

## 常见问题

### Q: 如何查看历史消息?
A: 使用搜索功能,或滚动加载历史消息

### Q: 会议录制如何保存?
A: 录制内容自动保存在云空间,可在妙记中查看

### Q: 如何管理文档权限?
A: 文档右上角分享设置,可设置查看/编辑/管理权限
