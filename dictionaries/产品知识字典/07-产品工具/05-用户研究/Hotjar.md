# Hotjar教程

## 概述
Hotjar是一款用户行为分析工具,提供热力图、录屏、反馈收集等功能,帮助产品经理深入了解用户在网站上的真实行为和体验。

## 核心功能

### 热力图(Heatmaps)
- 点击热力图
- 移动热力图
- 滚动热力图
- 自定义热力图

### 录屏(Sessions)
- 用户会话录制
- 回放分析
- 标签筛选
- 分享功能

### 反馈工具
- 反馈收集器
- 民意调查
- 即时反馈
- 问卷表单

### 分析工具
- 漏斗分析
- 表单分析
- 入站链接
- 设备分析

### 热力图设置
```
创建热力图:
1. 点击"Heatmaps"
2. 点击"New Heatmap"
3. 输入页面URL
4. 设置样本量
5. 开始收集

类型说明:
- Click Map: 点击位置和频率
- Move Map: 鼠标移动轨迹
- Scroll Map: 页面滚动深度
- Desktop/Mobile: 不同设备视图

分析要点:
- 点击集中区域
- 忽略区域
- 滚动流失点
- 不同设备差异
```

### 会话录屏
```
查看录屏:
1. 点击"Recordings"
2. 筛选条件设置
3. 选择会话观看
4. 分析用户行为

筛选维度:
- 页面URL
- 用户属性
- 设备类型
- 浏览器
- 地理位置
- 会话时长
- 活跃时间

分析要点:
- 用户浏览路径
- 点击交互行为
- 表单填写过程
- 困惑停留点
- 退出页面
```

### 反馈收集
```
创建反馈:
1. 点击"Feedback"
2. 点击"New Feedback"
3. 选择反馈类型
4. 设计问题内容
5. 设置触发条件
6. 发布反馈

反馈类型:
- 表情评分(1-5分)
- 单选/多选
- 文本输入
- 截图功能

触发条件:
- 时间延迟
- 滚动深度
- 退出意图
- 特定页面
```

## 实战案例

### 案例1:落地页优化
```
问题: 落地页转化率低

分析步骤:
1. 创建热力图
   - 分析点击分布
   - 查看滚动深度
   - 识别有效区域

2. 查看会话录屏
   - 观察用户行为
   - 发现困惑点
   - 找出流失原因

3. 收集反馈
   - 设置反馈调查
   - 询问未转化原因
   - 收集改进建议

分析发现:
- CTA按钮点击率低
- 首屏以下内容很少被看到
- 表单字段过多

优化方案:
- 优化CTA按钮样式和位置
- 精简首屏内容,突出价值
- 减少表单字段

验证效果:
- 转化率提升30%
- CTA点击率提升50%
```

### 案例2:导航优化
```
问题: 用户找不到需要的功能

分析步骤:
1. 导航点击热力图
   - 哪些导航项被点击
   - 点击顺序如何

2. 录屏分析
   - 用户如何寻找功能
   - 是否有多次点击
   - 停留在哪些页面

3. 民意调查
   - "您是否找到了需要的功能?"
   - "您希望在哪里找到这个功能?"

发现:
- 某功能被频繁搜索
- 导航层级过深
- 分类不够清晰

优化:
- 调整导航结构
- 添加快捷入口
- 优化搜索功能
```

### 案例3:表单优化
```
问题: 表单提交率低

分析工具:
1. 表单分析
   - 字段填写时间
   - 放弃字段
   - 错误字段

2. 录屏观察
   - 填写过程是否顺畅
   - 是否有反复修改
   - 在哪个字段放弃

3. 反馈收集
   - "表单是否容易填写?"
   - "有什么困难?"

发现:
- 某字段填写时间长
- 错误提示不清晰
- 必填项过多

优化:
- 简化表单字段
- 优化错误提示
- 添加字段说明
- 自动填充功能

效果:
- 提交率提升25%
- 填写时间减少40%
```

## 最佳实践

### 1. 热力图分析
```
分析要点:
- 关注点击密度高的区域
- 识别被忽略的内容
- 对比不同设备表现
- 结合转化数据分析
```

### 2. 录屏分析
```
观看技巧:
- 快速浏览(2x-4x速度)
- 关注异常行为
- 标记重要会话
- 分类归档发现

分析重点:
- 用户目标
- 完成的任务
- 遇到的障碍
- 放弃的原因
```

### 3. 反馈设计
```
设计原则:
- 问题简洁明确
- 选项数量适中
- 触发时机合适
- 不打扰用户体验

常见问题:
- "这个页面有帮助吗?"
- "您来这里的目的?"
- "您遇到了什么问题?"
- "如何改进?"
```

### 4. 数据整合
```
整合方式:
- 结合GA数据分析
- 对比转化漏斗
- 结合定性定量
- 多维度交叉分析
```

## 常见问题

### Q: 数据收集需要多长时间?
A: 建议至少收集1000-3000个会话,根据流量需要几天到几周

### Q: 如何保护用户隐私?
A: 可以设置排除敏感字段、IP匿名、遵守GDPR规定

### Q: 录屏会影响网站性能吗?
A: Hotjar脚本经过优化,对性能影响很小
