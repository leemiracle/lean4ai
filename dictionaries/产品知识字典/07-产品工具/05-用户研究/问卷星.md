# 问卷星教程

## 概述
问卷星是国内领先的在线问卷调研平台,提供问卷设计、数据收集、统计分析等功能,是产品经理进行用户调研和市场研究的重要工具。

## 核心功能

### 问卷设计
- 多种题型支持
- 逻辑跳转
- 关联显示
- 分页设置
- 主题美化

### 数据收集
- 多渠道投放
- 二维码分享
- 微信红包
- 样本服务
- 投票功能

### 数据分析
- 实时统计
- 图表可视化
- 交叉分析
- 数据导出
- 报告生成

## 使用指南

### 创建问卷
```
步骤:
1. 登录问卷星
2. 点击"创建问卷"
3. 选择问卷类型
   - 调查问卷
   - 考试卷
   - 报名表
   - 测评
4. 选择创建方式
   - 从模板创建
   - 从文本导入
   - 空白问卷
```

### 题型介绍
```
基础题型:
- 单选题: 多选一
- 多选题: 多选多
- 填空题: 文本输入
- 下拉框: 下拉选择
- 矩阵题: 表格形式
- 量表题: 评分量表
- 排序题: 拖拽排序
- 文件上传: 上传文件

高级题型:
- NPS题: 净推荐值
- 李克特量表: 态度测量
- 语义差异量表: 对比评价
```

### 逻辑设置
```
关联逻辑:
- 跳转逻辑: 根据答案跳转
- 显示逻辑: 根据答案显示
- 引用逻辑: 引用之前答案

设置方法:
1. 编辑题目
2. 点击"添加逻辑"
3. 设置条件和动作
4. 保存设置
```

### 问卷投放
```
投放方式:
1. 链接分享
   - 复制问卷链接
   - 嵌入网页代码

2. 二维码
   - 下载二维码图片
   - 打印张贴

3. 社交媒体
   - 微信分享
   - 微博分享

4. 红包激励
   - 设置红包金额
   - 随机/定额红包

5. 样本服务
   - 购买精准样本
   - 指定人群投放
```

## 实战案例

### 案例1:用户满意度调研
```
问卷设计:

1. 开场白
   问卷标题: 用户满意度调研
   问卷说明: 感谢您抽出时间参与调研...
   预计时间: 3-5分钟

2. 筛选题
   Q1. 您是否使用过我们的产品?
   ○ 是
   ○ 否(跳转结束)

3. 使用情况
   Q2. 您使用我们的产品多长时间了?
   ○ 不到1个月
   ○ 1-6个月
   ○ 6个月-1年
   ○ 1年以上

   Q3. 您使用产品的频率是?
   ○ 每天
   ○ 每周几次
   ○ 每月几次
   ○ 偶尔使用

4. 满意度评分
   Q4. 请对以下方面进行评分(1-5分)
   - 产品功能 [1] [2] [3] [4] [5]
   - 易用性 [1] [2] [3] [4] [5]
   - 稳定性 [1] [2] [3] [4] [5]
   - 客服支持 [1] [2] [3] [4] [5]

5. NPS题
   Q5. 您有多大可能向朋友推荐我们的产品?
   0-10分打分

6. 开放题
   Q6. 您觉得我们的产品还有哪些需要改进的地方?
   [文本框]

7. 人口统计学信息
   Q7. 您的年龄段?
   Q8. 您的职业?
   Q9. 您所在的城市?
```

## 最佳实践

### 1. 问卷设计原则
```
设计要点:
- 问题数量控制在15-25题
- 语言简洁易懂
- 避免引导性问题
- 选项互斥完整
- 合理使用逻辑跳转
```

### 2. 题目设计技巧
```
单选题:
- 选项要覆盖所有可能
- 添加"其他"选项
- 避免选项重叠

多选题:
- 说明可选择数量
- 选项不宜过多(5-8个)

量表题:
- 5分制或7分制
- 标注端点含义

开放题:
- 提示回答方向
- 设置字数限制
```

### 3. 提高回收率
```
方法:
- 标题吸引人
- 开场白清晰说明目的
- 控制问卷时长(5-10分钟)
- 提供小礼品或红包
- 多渠道投放
- 设置进度条
```

### 4. 数据分析
```
分析维度:
- 描述性统计: 频次、占比
- 交叉分析: 不同群体差异
- 相关性分析: 指标关联
- 文本分析: 开放题词云
```

## 常见问题

### Q: 如何避免刷票?
A: 设置IP限制、答题时间限制、添加验证码

### Q: 如何设置问卷截止时间?
A: 在问卷设置中开启截止时间,到时自动停止收集

### Q: 如何导出原始数据?
A: 数据分析页面点击导出,选择Excel或SPSS格式
