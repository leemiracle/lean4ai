# Figma教程

## 概述
Figma是一款基于云端的协作式设计工具,支持实时协作、原型设计和代码导出,是当前最受欢迎的UI设计工具之一。

## 核心功能

### 设计功能
- 矢量编辑
- 布局网格
- 自动布局(Auto Layout)
- 组件系统
- 样式管理

### 协作功能
- 实时多人编辑
- 评论标注
- 版本历史
- 团队库
- 权限管理

### 原型功能
- 页面连接
- 交互过渡
- 智能动画
- 固定滚动
- 条件交互

### 开发功能
- 代码查看
- CSS导出
- 资源导出
- 设计标注

## 使用指南

### 界面布局
```
左侧栏:
- 图层面板
- 资源面板
- 组件面板

右侧栏:
- 设计面板
- 原型面板
- 代码面板
```

### Auto Layout使用
```应用场景:
1. 按钮组件 - 文字自适应
2. 列表项 - 多内容适配
3. 卡片布局 - 灵活排版
4. 响应式设计

设置方式:
- 选中组件右键添加Auto Layout
- 或使用快捷键Shift+A
```

### 组件创建
```
创建组件:
1. 选中需要复用的元素
2. 右键选择Create Component
3. 或使用快捷键Ctrl+Alt+K

组件变体:
1. 创建多个组件状态
2. 选中后右键Combine as Variants
3. 设置不同属性值
```

### 样式管理
```
颜色样式:
- 品牌色
- 功能色
- 灰度色板

文字样式:
- 标题样式
- 正文样式
- 辅助样式

效果样式:
- 阴影效果
- 模糊效果
```

## 实战案例

### 案例1:设计系统搭建
```步骤:
1. 建立颜色规范
   - 定义主色、辅色
   - 设置灰度色阶
   - 添加功能色

2. 字体规范
   - 定义字体层级
   - 设置行高字距
   - 创建文字样式

3. 组件库
   - 基础组件(按钮、输入框)
   - 复合组件(卡片、列表)
   - 业务组件
```

### 案例2:移动端App设计
```流程:
1. 创建移动端画框(375x812)
2. 设计首页布局
3. 添加组件和样式
4. 设置原型交互
5. 预览和测试
```

```团队协作:

3. 分配编辑/查看权限
4. 实时协作编辑
5. 添加评论反馈
6. 版本管理
```

## 常见问题

### Q: 如何实现响应式设计?
A: 使用Auto Layout结合约束设置,实现不同尺寸适配

### Q: 如何导出开发资源?
A: 选中元素后右侧面板设置导出参数,或使用Export功能

### Q: 如何实现团队组件库共享?
A: 发布组件到团队库,团队成员即可使用共享组件
