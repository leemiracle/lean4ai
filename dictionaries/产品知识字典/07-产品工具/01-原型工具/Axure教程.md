# Axure教程

## 概述
Axure RP是一款专业的快速原型设计工具,支持创建高保真交互原型,是产品经理必备工具之一。

## 核心功能

### 基础组件
- 矩形、圆形、文本等基础形状
- 动态面板
- 中继器
- 母版
- 标记组件

### 交互功能
- 页面跳转
- 状态切换
- 条件判断
- 变量使用
- 函数应用

### 动效效果
- 移动、旋转、缩放
- 淡入淡出
- 滑动效果
- 动画缓动

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+N | 新建项目 |
| Ctrl+S | 保存项目 |
| Ctrl+Z | 撤销 |
| Ctrl+Y | 重做 |
| Ctrl+C | 复制 |
| Ctrl+V | 粘贴 |
| Ctrl+D | 快速复制 |
| Ctrl+G | 组合 |
| Ctrl+Shift+G | 取消组合 |
| Ctrl+R | 预览原型 |
| F5 | 生成HTML |
| Ctrl+; | 显示/隐藏网格 |
| Ctrl+' | 显示/隐藏参考线 |

## 使用指南

### 项目创建流程
1. **新建项目**
   - 选择合适的项目尺寸
   - 设置移动端/PC端适配

2. **页面规划**
   - 使用站点地图规划页面结构
   - 建立清晰的页面层级关系

3. **组件设计**
   - 使用元件库快速搭建界面
   - 设置组件样式和属性

4. **交互设置**
   - 添加交互事件
   - 设置触发条件和动作

5. **预览发布**
   - F5预览效果
   - 生成HTML分享

```
使用场景:
- 多状态切换(如Tab页签)

- 轮播图
```

### 中继器应用
```
应用场景:
- 列表数据展示

- 动态内容渲染
```

## 实战案例

### 案例1:登录页面原型
```步骤:
1. 创建登录页面
2. 添加输入框组件(用户名、密码)
3. 添加登录按钮
4. 设置交互:
   - 点击登录验证输入
   - 成功跳转首页
   - 失败提示错误信息
5. 添加记住密码复选框
6. 设置表单验证逻辑
```

### 案例2:购物车功能
```功能点:
- 商品数量增减
- 价格自动计算
- 全选/反选
- 删除商品
- 结算按钮状态
```

### 案例3:Tab切换
```实现方式:
1. 创建动态面板
2. 添加多个状态
3. 每个状态对应一个Tab内容
4. Tab按钮设置切换状态交互
```

```
页面命名: 模块_页面名称_状态
```
