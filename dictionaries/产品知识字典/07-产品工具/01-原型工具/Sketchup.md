# Sketch教程

## 概述
Sketch是一款专为UI/UX设计师打造的矢量图形设计工具,以其轻量、高效著称,是Mac平台上主流的设计工具之一。

## 核心功能

### 设计功能
- 矢量绘图
- 布局网格
- 符号系统
- 样式管理
- 画板设置

### 原型功能
- 页面链接
- 交互动画
- 固定元素
- 热点区域

### 协作功能
- Sketch Cloud
- 团队库
- 版本历史
- 评论反馈

### 开发支持
- 代码导出
- 资源切图
- 设计标注
- 插件扩展

## 使用指南

### 画板设置
```
常用尺寸:
- iPhone 14 Pro: 393x852
- iPhone SE: 375x667
- Android: 360x800
- iPad: 768x1024
- Web Desktop: 1440x900
- Web Mobile: 375x667
```

### 符号(Symbol)使用
```
创建符号:
1. 选中元素
2. 点击Create Symbol
3. 或使用快捷键Cmd+Option+K

符号覆盖:
- 文字覆盖
- 图片覆盖
- 样式覆盖
```

### 样式管理
```
共享样式:
- 文字样式
- 图层样式
- 颜色变量

使用方式:
1. 创建样式
2. 命名保存
3. 应用到图层
4. 统一更新
```

### 布局网格
```
设置方式:
1. View > Canvas > Layout Settings
2. 设置列数、间距
3. 设置边距
4. 可视化参考线
```

## 最佳实践

```
图层命名:
```

## 常见问题

### Q: 如何导出不同倍率切图?
A: 选中元素,在右侧面板设置Export,添加1x/2x/3x导出

### Q: 如何实现Symbol嵌套?
A: 创建子Symbol后,将其作为元素放入父Symbol中

### Q: Sketch文件如何给Windows用户查看?
A: 使用Sketch Cloud分享,或导出PDF/图片格式
