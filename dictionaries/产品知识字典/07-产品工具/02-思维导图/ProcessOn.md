# ProcessOn教程

## 概述
ProcessOn是一款在线作图工具,支持流程图、思维导图、原型图、UML图等多种图表类型,免费使用且无需安装,是产品经理常用的绘图工具。

## 使用指南

### 创建图表
```
1. 新建文件
   - 选择图表类型
   - 选择模板或空白
   - 进入编辑界面

2. 添加元素
   - 从左侧图形库拖拽
   - 双击编辑文字
   - 设置样式属性

3. 连接元素
   - 选择连接线工具
   - 拖动创建连线
   - 设置连接样式

4. 美化图表
   - 调整颜色样式
   - 对齐分布元素
   - 添加图标图片
```

### 流程图绘制
```
基本元素:
- 开始/结束: 椭圆
- 处理步骤: 矩形
- 判断条件: 菱形
- 流程方向: 箭头
- 数据存储: 平行四边形

绘制要点:
1. 明确流程起点和终点
2. 每个步骤清晰独立
3. 判断分支明确
4. 箭头方向正确
5. 避免交叉连线
```

### 思维导图绘制
```
操作方式:
1. 选择思维导图类型
2. 编辑中心主题
3. 按Enter添加同级
4. 按Tab添加下级
5. 拖拽调整位置
6. 设置主题样式

样式选择:
- 经典思维导图
- 右侧分布图
- 组织结构图
- 树状结构图
```

## 实战案例

### 案例1:业务流程图
```绘制步骤:
1. 梳理业务流程
   - 识别关键环节
   - 明确参与角色
   - 确定决策节点

2. 绘制流程图
   开始 -> 用户操作 -> 系统判断 -> 处理A -> 结束
                      ↓ No
                   处理B -> 结束

3. 完善细节
   - 添加泳道区分角色
   - 标注异常流程
   - 添加说明文字
```

### 案例2:产品架构图
```层次结构:
应用层: 用户界面/交互层
  ↓
业务层: 业务逻辑/规则层
  ↓
服务层: 公共服务/能力层
  ↓
数据层: 数据存储/访问层
```

### 案例3:用户旅程图
```维度划分:
时间轴: 认知 -> 考虑 -> 购买 -> 使用 -> 推荐
  ↓        ↓       ↓       ↓       ↓
用户行为: 浏览   对比   下单   使用   分享
  ↓        ↓       ↓       ↓       ↓
触点: 广告  评测   商城   产品   社交
  ↓        ↓       ↓       ↓       ↓
情绪: 好奇  纠结   期待   满意   兴奋
```

## 常见问题

### Q: 如何创建泳道图?
A: 在流程图模式下,从图形库拖拽泳道元素到画布,调整泳道数量和标签

### Q: 如何调整图形的连接点?
A: 选中图形后,拖动蓝色连接点到合适位置

### Q: 文件数量有限制吗?
A: 免费版有文件数量限制,可通过邀请好友或付费扩容
