# XMind教程

## 概述
XMind是一款专业的思维导图软件,帮助用户理清思路、整理信息、提高工作效率,是产品经理进行需求分析和方案梳理的常用工具。

## 核心功能

### 基础功能
- 多种结构图
- 主题样式
- 图标标记
- 备注说明
- 附件插入

### 高级功能
- 演示模式
- 头脑风暴
- 演讲笔记
- ZEN模式
- 合并导图

### 协作功能
- 文件共享
- 导出多种格式
- 在线协作
- 云端同步

## 使用指南

### 创建思维导图
```
1. 选择结构类型
   - 思维导图
   - 逻辑图
   - 组织结构图
   - 树状图
   - 鱼骨图
   - 矩阵图
   - 时间轴

2. 确定中心主题
   - 简洁明了
   - 突出核心

3. 添加分支内容
   - 层级分明
   - 逻辑清晰
   - 简练表达
```

### 主题操作
```
主题类型:
- 中心主题: 核心概念
- 主要主题: 一级分支
- 子主题: 次级分支
- 自由主题: 独立内容

主题编辑:
- 双击编辑文字
- 拖拽调整位置
- 右键更多操作
```

### 样式美化
```
主题样式:
- 选择内置主题
- 自定义配色
- 调整字体大小
- 设置线条样式

图标标记:
- 优先级图标
- 进度标记
- 星标旗帜
- 笑脸表情
- 箭头指向
```

## 最佳实践

### 1. 结构清晰
- 层级不超过5层
- 每个分支3-7个要点
- 同级内容相互独立
- 使用关键词而非句子

### 2. 视觉优化
- 使用颜色区分类别
- 添加图标增强记忆
- 保持风格统一
- 突出重点内容

### 3. 内容组织
- MECE原则(相互独立、完全穷尽)
- 先发散后收敛
- 从整体到细节
- 持续迭代优化

### 4. 协作分享
- 导出PDF/PNG分享
- 使用标记标注重点
- 添加备注说明细节
- 版本管理便于追溯

## 常见问题

### Q: 如何调整分支结构?
A: 选中主题,拖拽到目标位置即可调整结构

### Q: 如何设置主题之间的关联?
A: 选中两个主题,右键选择"添加关联线"

### Q: 如何快速美化思维导图?
A: 使用内置主题样式,或一键应用配色方案
