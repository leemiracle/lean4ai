# 幕布教程

## 概述
幕布是一款结合大纲笔记与思维导图的工具,支持结构化记录和一键转换为思维导图,适合产品经理整理思路和输出文档。

### 大纲功能
- 层级缩进
- 折叠展开
- 拖拽调整
- 快捷键操作
- 内容搜索

### 思维导图
- 一键转换
- 多种样式
- 导出分享
- 演示模式

### 协作功能
- 在线协作

### 其他功能

- 移动端同步

## 使用指南

### 创建文档
```
1. 新建文档
   - 点击"+"创建
   - 选择空白文档或模板
   - 输入文档标题

2. 添加内容
   - 使用Enter添加同级
   - 使用Tab缩进下级
   - 支持文字、图片、链接

3. 组织结构
   - 拖拽调整位置
   - 折叠/展开分支
   - 设置完成状态
```

### 格式设置
```
文字格式:
- 加粗: **文字**
- 斜体: *文字*
- 删除线: ~~文字~~
- 代码: `代码`

特殊标记:
- 完成标记: -[] 或 -[x]
- 标签: #标签名
- 提到: @用户名
```

### 思维导图模式
```
转换方式:
1. 点击右上角"思维导图"按钮
2. 或使用快捷键切换
3. 自动生成导图视图

导图样式:
- 右侧分布图
- 组织结构图
- 树状图
- 鱼骨图
```
