# Tableau教程

## 概述
Tableau是一款强大的数据可视化工具,支持多种数据源连接,通过拖拽操作快速创建交互式仪表板,帮助产品经理直观展示数据分析结果。

## 核心功能

### 数据连接
- 多数据源连接
- 数据融合
- 实时/提取模式
- 数据源管理

### 数据处理
- 数据类型设置
- 字段拆分合并
- 计算字段
- 参数使用

### 可视化
- 多种图表类型
- 交互式筛选
- 钻取分析
- 地理分析

### 仪表板
- 仪表板布局
- 交互联动
- 故事叙述
- 布局容器

## 使用指南

### 连接数据
```
步骤:
1. 打开Tableau Desktop
2. 在连接面板选择数据源类型
   - Excel/CSV文件
   - 数据库(MySQL, PostgreSQL等)
   - 云数据源
   - Web数据连接器

3. 配置连接参数
   - 文件路径
   - 服务器地址
   - 账号密码

4. 选择数据表
   - 单表
   - 多表关联
   - 自定义SQL
```

### 数据准备
```
数据类型:
- 字符串(Abc)
- 数字(#)
- 日期(日历图标)
- 布尔值(T/F)
- 地理角色(地球图标)

数据处理:
- 更改数据类型: 点击字段图标
- 拆分字段: 右键 -> 拆分
- 合并字段: 拖拽字段重叠
- 创建计算字段: 右键 -> 创建计算字段
```

### 创建视图
```
基本操作:
1. 拖拽维度到行/列
2. 拖拽度量到相应区域
3. 选择图表类型
4. 调整样式设置

图表类型:
- 条形图: 类别对比
- 折线图: 趋势分析
- 饼图: 占比展示
- 散点图: 相关分析
- 地图: 地理分析
- 热力图: 密度展示
```

### 使用筛选器
```
筛选类型:
- 维度筛选: 选择特定值
- 度量筛选: 数值范围
- 日期筛选: 时间范围
- Top N筛选: 前N个

创建筛选器:
1. 右键字段 -> 显示筛选器
2. 拖拽字段到筛选器区域
3. 设置筛选条件
```

## 实战案例

### 案例1:用户增长仪表板
```
数据准备:
- 用户表: user_id, register_date, channel, city
- 登录表: user_id, login_date

创建视图:

1. 日活趋势(折线图)
   - 列: login_date(天)
   - 行: COUNT(user_id)
   - 筛选: 最近30天

2. 新增用户趋势(折线图)
   - 列: register_date(天)
   - 行: COUNT(user_id)

3. 渠道分布(饼图)
   - 颜色: channel
   - 角度: COUNT(user_id)

4. 城市排行(条形图)
   - 行: city
   - 列: COUNT(user_id)
   - 筛选: Top 10

5. 留存率热力图
   - 行: 注册周
   - 列: 留存天数
   - 颜色: 留存率

仪表板组装:
- 拖拽各工作表到仪表板
- 设置筛选器联动
- 调整布局对齐
```

### 案例2:销售分析仪表板
```
数据源:
- 订单表: order_id, date, customer, product, amount, region

分析维度:
1. 销售总览
   - 总销售额
   - 订单数量
   - 平均客单价

2. 时间趋势
   - 月度销售趋势
   - 同比环比对比

3. 产品分析
   - 产品销售排行
   - 类别占比

4. 区域分析
   - 地图展示销售分布
   - 区域对比

计算字段示例:
// 环比增长率
(SUM([销售额]) - LOOKUP(SUM([销售额]), -1)) / LOOKUP(SUM([销售额]), -1)

// 同比增长率
(SUM([销售额]) - LOOKUP(SUM([销售额]), -12)) / LOOKUP(SUM([销售额]), -12)

// 累计销售额
RUNNING_SUM(SUM([销售额]))
```

## 最佳实践

### 1. 数据建模
```
建议:
- 合理设计数据源结构
- 建立字段命名规范
- 使用关系模型而非关联
- 预计算复杂指标
```

### 2. 可视化设计
```
设计原则:
- 简洁清晰,避免过度装饰
- 颜色使用有意义
- 重要信息突出显示
- 保持一致的设计风格

颜色建议:
- 分类数据: 使用对比色
- 连续数据: 使用渐变色
- 警示信息: 使用红色系
- 保持色盲友好
```

### 3. 性能优化
```
优化方法:
- 使用数据提取而非实时连接
- 减少高基数维度
- 优化计算字段
- 合理使用筛选器
- 减少视图复杂度
```

### 4. 仪表板布局
```
布局建议:
- 重要指标放在左上角
- 从概览到详情的布局
- 合理使用容器
- 响应式设计考虑
- 留白增强可读性
```

## 进阶技巧

### 计算字段
```
字符串函数:
LEFT([字段], 3)              取左边3个字符
RIGHT([字段], 3)             取右边3个字符
CONTAINS([字段], "关键词")    包含判断
REPLACE([字段], "a", "b")    替换

日期函数:
DATEPART('month', [日期])    提取月份
DATEADD('day', 7, [日期])    日期加7天
DATEDIFF('day', [开始], [结束])  日期差

逻辑函数:
IF [条件] THEN [值] ELSE [值] END
CASE [字段] WHEN 'A' THEN 1 WHEN 'B' THEN 2 END
IIF([条件], [真值], [假值])

聚合函数:
SUM([字段])
AVG([字段])
COUNT([字段])
COUNTD([字段])               去重计数
```

### 参数使用
```
创建参数:
1. 右键 -> 创建参数
2. 设置数据类型和允许值
3. 在计算字段中使用参数

应用场景:
- 动态切换指标
- 调整阈值
- 选择维度
```

### LOD表达式
```
FIXED: 固定维度聚合
{FIXED [维度]: SUM([度量])}

INCLUDE: 包含维度聚合
{INCLUDE [维度]: AVG([度量])}

EXCLUDE: 排除维度聚合
{EXCLUDE [维度]: SUM([度量])}
```

### 动作交互
```
交互类型:
- 筛选动作: 点击过滤其他视图
- 高亮动作: 点击高亮相关数据
- URL动作: 点击跳转链接
- 参数动作: 点击更新参数

设置方法:
仪表板 -> 仪表板操作 -> 添加操作
```

## 常见问题

### Q: 如何处理大数据量性能问题?
A: 使用数据提取、优化计算、减少视图复杂度、使用Context筛选器

### Q: 如何实现同比环比分析?
A: 使用LOOKUP函数或表计算,结合日期筛选器实现

### Q: 如何分享仪表板给他人?
A: 发布到Tableau Server/Online,或导出为图片/PDF
