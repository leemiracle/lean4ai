## 使用指南

### 常用函数

#### 数学统计函数
```excel
SUM(区域)              求和
AVERAGE(区域)          平均值
COUNT(区域)            计数
COUNTA(区域)           非空计数
MAX(区域)              最大值
MIN(区域)              最小值
ROUND(数值,小数位)     四舍五入
SUMIF(条件区域,条件,求和区域)    条件求和
SUMIFS(求和区域,条件区域1,条件1,...)    多条件求和
```

#### 文本函数
```excel
LEFT(文本,字符数)      从左侧取字符
RIGHT(文本,字符数)     从右侧取字符
MID(文本,开始位置,字符数)  取中间字符
LEN(文本)              文本长度
TRIM(文本)             去除空格
CONCATENATE(文本1,文本2,...)  连接文本
TEXT(数值,格式)        格式化文本
```

#### 逻辑函数
```excel
IF(条件,真值,假值)     条件判断
AND(条件1,条件2,...)   与运算
OR(条件1,条件2,...)    或运算
IFERROR(表达式,错误值) 错误处理
IFS(条件1,值1,条件2,值2,...)  多条件判断
```

#### 查找引用函数
```excel
VLOOKUP(查找值,区域,列号,匹配方式)  垂直查找
HLOOKUP(查找值,区域,行号,匹配方式)  水平查找
INDEX(区域,行号,列号)   返回引用值
MATCH(查找值,区域,匹配方式)   返回位置
INDIRECT(引用文本)      间接引用
```

#### 日期函数
```excel
TODAY()                当前日期
NOW()                  当前日期时间
YEAR(日期)             提取年份
MONTH(日期)            提取月份
DAY(日期)              提取日期
DATEDIF(开始日期,结束日期,单位)  日期差
```

## 实战案例

### 案例1:用户数据分析
```
数据示例:
用户ID | 注册日期 | 城市 | 活跃天数 | 订单数 | 消费金额

分析维度:
1. 用户增长趋势
   - 按日期透视统计新增用户数
   - 绘制折线图展示趋势

2. 用户分布分析
   - 按城市透视统计用户数
   - 绘制饼图展示占比

3. 用户活跃度分析
   - 按活跃天数区间分组
   - 统计各区间用户占比

4. 消费能力分析
   - 计算客单价
   - RFM模型分层

函数应用:
=COUNTIF(C:C,"北京")                    统计北京用户数
=SUMIF(C:C,"北京",F:F)                  北京用户总消费
=AVERAGEIF(C:C,"北京",F:F)              北京用户平均消费
=DATEDIF(B2,TODAY(),"D")                计算注册天数
```

### 案例2:销售数据报表
```
数据结构:
日期 | 产品 | 类别 | 销量 | 金额 | 渠道

报表制作:
1. 数据透视表
   - 行: 日期(按月分组)
   - 列: 产品类别
   - 值: 销量求和、金额求和

2. 关键指标计算
   =SUM(D:D)                            总销量
   =SUM(E:E)                            总金额
   =SUM(E:E)/SUM(D:D)                   平均单价
   =COUNTA(A:A)-1                       订单数

3. 环比增长计算
   =IF(上月销量=0,0,(本月销量-上月销量)/上月销量)

4. 同比增长计算
   =IF(去年同期销量=0,0,(今年销量-去年同期销量)/去年同期销量)

5. 可视化呈现
   - 月度销售趋势折线图
   - 产品类别占比饼图
   - 渠道对比柱状图
```

### 案例3:产品运营监控
```
监控指标:
DAU | MAU | 留存率 | 转化率 | ARPU | GMV

数据整理:
1. 数据清洗
   =TRIM(A2)                            去除空格
   =IFERROR(VALUE(A2),0)                错误值处理

2. 数据计算
   =DAU/MAU                             活跃度
   =新用户数/总访问数                    注册转化率
   =付费用户数/活跃用户数                付费转化率
   =总收入/活跃用户数                   ARPU值

3. 趋势分析
   - 使用折线图展示指标趋势
   - 添加趋势线预测
   - 设置条件格式突出异常值

4. 仪表盘制作
   - 关键指标卡片
   - 趋势图表
   - 环比/同比对比
   - 目标完成度
```

## 常见问题

### Q: VLOOKUP为什么返回#N/A?
A: 检查查找值是否存在、匹配方式是否正确、数据类型是否一致

### Q: 如何快速对比两列数据?
A: 使用条件格式或VLOOKUP/MATCH函数对比

### Q: 数据透视表如何刷新?
A: 右键数据透视表选择刷新,或设置自动刷新
