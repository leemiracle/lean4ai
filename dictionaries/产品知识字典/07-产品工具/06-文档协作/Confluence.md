# Confluence教程

## 概述
Confluence是Atlassian公司开发的企业Wiki和知识管理工具,与Jira深度集成,广泛用于团队文档协作、知识沉淀和项目管理。

## 核心功能

### 内容管理
- 页面创建与编辑
- 空间管理
- 模板应用
- 版本控制
- 回收站

### 协作功能
- 实时协作编辑
- 评论讨论
- @提及
- 点赞反应
- 任务分配

### 知识管理
- 搜索功能
- 标签管理
- 目录结构
- 页面层级
- 相关内容推荐

### 集成功能
- Jira集成
- 第三方应用集成
- API接口
- 宏扩展

## 使用指南

### 空间管理
```
创建空间:
1. 点击"Spaces" -> "Create space"
2. 选择空间类型:
   - 空白空间
   - 知识库
   - 团队空间
   - 项目空间
3. 输入空间名称和键值
4. 设置权限

空间结构:
- 首页
- 页面树
- 博客文章
- 空间设置

权限设置:
- 查看
- 添加
- 删除
- 导出
- 管理
```

### 页面编辑
```
创建页面:
1. 点击"Create"
2. 选择模板或空白页
3. 编辑内容
4. 保存或发布

内容元素:
- 标题(H1-H6)
- 段落
- 列表(有序/无序)
- 表格
- 图片
- 代码块
- 链接
- 宏

富文本编辑:
- 文字格式
- 对齐方式
- 颜色样式
- 缩进列表
```

### 模板使用
```
内置模板:
- 产品需求文档(PRD)
- 会议纪要
- 项目计划
- 决策记录
- 回顾总结

创建模板:
1. 空间工具 -> 模板
2. 创建新模板
3. 设计模板内容
4. 添加变量和占位符
5. 保存模板

使用模板:
1. 创建页面
2. 选择模板
3. 填充内容
```

### 宏(Macro)应用
```
常用宏:
- 目录(Table of Contents)
- 子页面显示(Children Display)
- 包含页面(Include)
- 任务报告(Task Report)
- Jira问题(Jira Issues)
- 状态(Status)
- 日期(Date)
- 信息面板(Info/Note/Warning)
- 代码块(Code Block)
- 画廊(Gallery)

插入宏:
1. 输入"/"
2. 搜索宏名称
3. 配置参数
4. 插入页面
```

## 最佳实践

### 1. 页面命名规范
```
命名格式:
[类型] 名称 - 版本/状态

示例:
- [PRD] 购物车功能 v1.0
- [会议] 产品周会 2024-01-01
- [设计] 首页改版方案
- [技术] API接口文档
```

### 2. 内容组织
```
组织原则:
- 清晰的层级结构
- 合理的页面分类
- 统一的模板应用
- 完善的标签系统

优化建议:
- 添加目录导航
- 使用锚点链接
- 建立页面索引
- 定期整理归档
```

### 3. 协作规范
```
协作建议:
- 及时回复评论
- 合理使用@提醒
- 任务明确分配
- 定期更新内容

权限管理:
- 敏感信息控制访问
- 合理设置编辑权限
- 定期审计权限
```

### 4. 文档维护
```
维护要点:
- 定期更新内容
- 清理过时文档
- 归档历史版本
- 保持文档准确性

版本管理:
- 重要修改添加说明
- 保留历史版本
- 使用标签标记版本
```

- 日期范围
- 标签搜索

CQL(Confluence Query Language):
space = "项目空间" AND label = "PRD"
creator = currentUser() AND created > "2024-01-01"

## 常见问题

### Q: 如何恢复删除的页面?
A: 空间工具 -> 回收站 -> 恢复页面

### Q: 如何限制页面访问权限?
A: 页面锁图标 -> 限制查看/编辑

### Q: 如何导出空间内容?
A: 空间工具 -> 导出 -> 选择格式
