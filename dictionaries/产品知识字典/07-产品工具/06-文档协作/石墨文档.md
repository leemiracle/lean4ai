# 石墨文档教程

## 概述
石墨文档是一款支持多人实时协作的在线文档工具,提供文档、表格、幻灯片等多种格式,支持实时编辑、评论和版本管理。

## 核心功能

### 文档编辑
- 富文本编辑
- Markdown支持
- 实时协作
- 评论功能
- 版本历史

### 表格功能
- 在线表格
- 公式计算
- 数据透视
- 图表生成
- 条件格式

### 协作功能
- 多人实时编辑
- 评论讨论
- @提醒
- 修订记录
- 权限管理

### 其他功能
- 幻灯片
- 思维导图
- 表单收集
- 模板库
- 第三方集成

## 使用指南

### 创建文档
```
步骤:
1. 点击"+"或"新建"
2. 选择文档类型:
   - 文档
   - 表格
   - 幻灯片
   - 思维导图
   - 表单
3. 选择模板或空白
4. 开始编辑

文档管理:
- 文件夹分类
- 收藏夹
- 最近使用
- 回收站
```

### 文档编辑
```
基础格式:
- 标题(H1-H6)
- 正文
- 引用
- 代码块
- 列表(有序/无序)
- 任务列表
- 表格
- 图片
- 链接
- 分割线

高级功能:
- 目录
- 页眉页脚
- 页码
- 批注
- 脚注
```

### 表格使用
```
基础操作:
- 行列添加删除
- 单元格合并
- 格式设置
- 条件格式

公式示例:
=SUM(A1:A10)              求和
=AVERAGE(A1:A10)          平均值
=COUNT(A1:A10)            计数
=MAX(A1:A10)              最大值
=MIN(A1:A10)              最小值
=IF(A1>10, "大", "小")     条件判断
=VLOOKUP(查找值,区域,列号,匹配方式)  查找

数据功能:
- 排序
- 筛选
- 数据验证
- 冻结行列
```

### 协作设置
```
分享设置:
1. 点击"分享"
2. 设置权限:
   - 可编辑
   - 可评论
   - 只读
3. 选择分享方式:
   - 链接分享
   - 指定成员
   - 公开分享
4. 设置密码(可选)

协作功能:
- 多人实时编辑
- 光标位置显示
- 评论讨论
- @提醒
- 修订记录
```

## 实战案例

### 案例1:会议纪要
```markdown
# 产品周会纪要

## 会议信息
- 时间: 2024-01-01 10:00-11:00
- 地点: 会议室A
- 参会人: @张三 @李四 @王五
- 记录人: 张三

## 会议议题
1. 本周工作汇报
2. 下周工作计划
3. 问题讨论

## 会议内容

### 一、本周工作汇报
#### 产品组
- 完成PRD评审
- 用户访谈5人
- 数据分析报告

#### 开发组
- 购物车功能开发完成
- 性能优化

#### 测试组
- 完成第一轮测试
- 发现Bug 10个,已修复8个

### 二、下周工作计划
| 模块 | 任务 | 负责人 | 时间 |
|------|------|--------|------|
| 产品 | 支付功能PRD | 张三 | 周三 |
| 开发 | 支付功能开发 | 李四 | 周五 |
| 测试 | 第二轮测试 | 王五 | 下周一 |

### 三、问题讨论
1. **性能问题**
   - 问题: 页面加载慢
   - 方案: 优化接口,减少请求
   - 负责人: @李四

2. **用户反馈**
   - 问题: 找不到某功能
   - 方案: 优化导航结构
   - 负责人: @张三

## 待办事项
- [ ] 张三: 完成支付功能PRD(周三)
- [ ] 李四: 性能优化(周四)
- [ ] 王五: 准备测试用例(周二)

## 下次会议
- 时间: 2024-01-08 10:00
- 议题: 支付功能评审
```

## 最佳实践

```
命名格式:

示例:

- [方案] 上线方案 - 最终版
```

## 常见问题

### Q: 如何恢复误删的内容?
A: 查看修订记录或回收站,选择历史版本恢复

### Q: 多人编辑冲突怎么办?
A: 石墨支持实时协作,自动合并,冲突部分会提示

### Q: 如何导出文档?
A: 点击菜单 -> 导出 -> 选择格式(PDF/Word/Excel)
