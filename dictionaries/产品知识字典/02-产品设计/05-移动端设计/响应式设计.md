# 响应式设计

## 定义

响应式设计（Responsive Design）是一种让网页和应用能够自适应不同屏幕尺寸和设备的设计方法，确保在任何设备上都能提供良好的用户体验。

## 响应式设计原则

### 1. 核心原则

```
响应式 = 流式布局 + 媒体查询 + 弹性图片

1. 流式布局 (Fluid Grids)
   - 使用相对单位（%、rem）
   - 灵活的网格系统

2. 媒体查询 (Media Queries)
   - 根据屏幕尺寸应用不同样式

3. 弹性图片 (Flexible Images)
   - 图片自适应容器
```

### 2. 设计理念

- **移动优先**：从小屏幕开始设计
- **渐进增强**：逐步添加大屏特性
- **内容优先**：根据设备调整内容展示

## 断点设计

### 1. 常用断点

| 断点 | 宽度 | 设备 |
|------|------|------|
| xs | < 576px | 手机 |
| sm | ≥ 576px | 大手机 |
| md | ≥ 768px | 平板 |
| lg | ≥ 992px | 小屏桌面 |
| xl | ≥ 1200px | 桌面 |
| xxl | ≥ 1400px | 大屏桌面 |

### 2. 断点选择

```css
/* 移动优先断点 */
/* 默认样式（移动端） */

@media (min-width: 576px) {
  /* 大手机 */
}

@media (min-width: 768px) {
  /* 平板 */
}

@media (min-width: 992px) {
  /* 桌面 */
}

@media (min-width: 1200px) {
  /* 大屏桌面 */
}
```

### 3. 断点策略

```
内容驱动断点：
1. 先设计内容
2. 调整到布局开始变形
3. 在变形点设置断点
4. 重复以上步骤
```

**栅格代码示例**
```css

  width: 100%;

  margin: 0 auto;
}

.col-12 { width: 100%; }

.col-4 { width: 33.33%; }
.col-3 { width: 25%; }

@media (max-width: 768px) {
  .col-4, .col-6 { width: 100%; }
}
```

## 响应式元素

### 1. 响应式图片

**流式图片**
```css
img {
  max-width: 100%;
  height: auto;
}
```

**响应式图片（srcset）**
```html
<img 
  src="image-800.jpg"
  srcset="image-400.jpg 400w,
          image-800.jpg 800w,
          image-1200.jpg 1200w"
  sizes="(max-width: 600px) 100vw,
         (max-width: 1200px) 50vw,
         33vw"
  alt="响应式图片">
```

**Picture元素**
```html
<picture>
  <source media="(min-width: 1200px)" srcset="large.jpg">
  <source media="(min-width: 768px)" srcset="medium.jpg">
  <img src="small.jpg" alt="响应式图片">
</picture>
```

### 2. 响应式文字

**流式字号**
```css
/* 使用 clamp() */
h1 {
  font-size: clamp(24px, 5vw, 48px);
}

/* 响应式字号 */
h1 {
  font-size: 24px;
}

@media (min-width: 768px) {
  h1 {
    font-size: 36px;
  }
}

@media (min-width: 1200px) {
  h1 {
    font-size: 48px;
  }
}
```

**响应式行高**
```css
p {
  line-height: 1.5;
}

@media (min-width: 768px) {
  p {
    line-height: 1.6;
  }
}
```

### 3. 响应式间距

**相对单位**
```css
.section {
  padding: 5%; /* 相对于父元素 */
}

.container {
  margin: 0 auto;
  padding: 0 1rem;
}

@media (min-width: 768px) {
  .container {
    padding: 0 2rem;
  }
}
```

## 响应式设计技巧

### 1. 隐藏/显示元素

```css
/* 移动端隐藏 */
.desktop-only {
  display: none;
}

@media (min-width: 768px) {
  .desktop-only {
    display: block;
  }
}

/* 桌面端隐藏 */
.mobile-only {
  display: block;
}

@media (min-width: 768px) {
  .mobile-only {
    display: none;
  }
}
```

### 2. 重新排序

```css
.container {
  display: flex;
  flex-direction: column;
}

.sidebar {
  order: 2;
}

.content {
  order: 1;
}

@media (min-width: 768px) {
  .container {
    flex-direction: row;
  }
  
  .sidebar {
    order: 1;
  }
  
  .content {
    order: 2;
  }
}
```

### 3. 调整尺寸

```css
.card {
  padding: 1rem;
  font-size: 14px;
}

@media (min-width: 768px) {
  .card {
    padding: 2rem;
    font-size: 16px;
  }
}
```

## 响应式设计最佳实践

### 1. 设计建议

- 移动优先设计
- 使用相对单位
- 控制断点数量
- 测试真实设备

### 2. 常见错误

- ❌ 断点过多
- ❌ 固定宽度元素
- ❌ 忽视内容优先级
- ❌ 只用模拟器测试

### 3. 检查清单

- [ ] 是否移动优先？
- [ ] 断点是否合理？
- [ ] 布局是否流式？
- [ ] 图片是否响应式？
- [ ] 文字是否可读？
- [ ] 是否测试了多种设备？
