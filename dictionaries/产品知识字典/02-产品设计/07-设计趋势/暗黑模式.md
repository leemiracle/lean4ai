# 暗黑模式 (Dark Mode)

## 定义

暗黑模式是一种使用深色背景和浅色文字的界面主题，可以减少屏幕亮度、降低眼睛疲劳，并在OLED屏幕上节省电量。

## 暗黑模式优势

### 1. 用户体验优势

| 优势 | 说明 |
|------|------|
| 减少眼睛疲劳 | 低亮度更舒适 |
| 节省电量 | OLED屏幕更明显 |
| 降低干扰 | 深色背景更专注 |
| 环境适应 | 适合暗光环境 |
| 个性化 | 用户偏好选择 |

### 2. 适用场景

- 夜间使用
- 长时间阅读
- 电影/游戏环境
- 低光环境
- OLED设备

## 暗黑模式设计原则

### 1. 核心原则

**不要简单反转颜色**
```
✗ 错误：
直接将白色背景改为黑色
将黑色文字改为白色

✓ 正确：
重新设计配色系统
调整对比度和层级
优化每个元素的颜色
```

**保持层级关系**
```
亮色模式层级：
┌─────────────────────────────────┐
│ 背景 #FFFFFF                    │ ← 最底层
├─────────────────────────────────┤
│ 卡片 #F5F5F5                    │ ← 第二层
├─────────────────────────────────┤
│ 弹窗 #FFFFFF + 阴影             │ ← 第三层
└─────────────────────────────────┘

暗黑模式层级：
┌─────────────────────────────────┐
│ 背景 #121212                    │ ← 最底层（最深）
├─────────────────────────────────┤
│ 卡片 #1E1E1E                    │ ← 第二层（稍浅）
├─────────────────────────────────┤
│ 弹窗 #2C2C2C                    │ ← 第三层（更浅）
└─────────────────────────────────┘
```

### 2. 对比度原则

**文字对比度**
```
亮色模式：
- 主文字：#262626 on #FFFFFF（对比度 13.5:1）
- 次文字：#595959 on #FFFFFF（对比度 7.0:1）

暗黑模式：
- 主文字：#FFFFFF on #121212（对比度 17.4:1）
- 次文字：#B3B3B3 on #121212（对比度 8.5:1）

注意：暗黑模式不要用纯白（#FFFFFF），
      使用稍暗的白色（#E0E0E0）减少刺眼
```

## 暗黑模式配色

### 1. 背景色系统

```
Material Design 暗黑模式背景色：
- Level 0（最底层）：#121212
- Level 1（1层提升）：rgba(255,255,255,0.05) → #1E1E1E
- Level 2（2层提升）：rgba(255,255,255,0.07) → #2C2C2C
- Level 3（3层提升）：rgba(255,255,255,0.08) → #373737
- Level 4（4层提升）：rgba(255,255,255,0.09) → #424242
```

**应用示例**
```
┌─────────────────────────────────┐
│ #121212 背景                    │
│ ┌─────────────────────────────┐│
│ │ #1E1E1E 卡片                ││
│ │ ┌─────────────────────────┐││
│ │ │ #2C2C2C 输入框          │││
│ │ └─────────────────────────┘││
│ └─────────────────────────────┘│
└─────────────────────────────────┘
```

   切换到专门设计的暗色图片

## 暗黑模式实现

### 1. CSS变量方案

```css
:root {
  /* 亮色模式变量 */
  --bg-primary: #FFFFFF;
  --bg-secondary: #F5F5F5;
  --text-primary: #262626;
  --text-secondary: #595959;
  --border-color: #E8E8E8;
}

[data-theme="dark"] {
  /* 暗黑模式变量 */
  --bg-primary: #121212;
  --bg-secondary: #1E1E1E;
  --text-primary: #FFFFFF;
  --text-secondary: #B3B3B3;
  --border-color: rgba(255, 255, 255, 0.12);
}

/* 使用变量 */
body {
  background-color: var(--bg-primary);
  color: var(--text-primary);
}
```

### 2. 媒体查询方案

```css
/* 跟随系统设置 */
@media (prefers-color-scheme: dark) {
  body {
    background-color: #121212;
    color: #FFFFFF;
  }
}

/* 结合变量使用 */
@media (prefers-color-scheme: dark) {
  :root {
    --bg-primary: #121212;
    --text-primary: #FFFFFF;
  }
}
```

### 3. JavaScript切换

```javascript
// 切换主题
function toggleTheme() {
  const html = document.documentElement;
  const currentTheme = html.getAttribute('data-theme');
  const newTheme = currentTheme === 'dark' ? 'light' : 'dark';
  html.setAttribute('data-theme', newTheme);
  localStorage.setItem('theme', newTheme);
}

// 初始化主题
function initTheme() {
  const savedTheme = localStorage.getItem('theme');
  const prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches;
  const theme = savedTheme || (prefersDark ? 'dark' : 'light');
  document.documentElement.setAttribute('data-theme', theme);
}

// 监听系统主题变化
window.matchMedia('(prefers-color-scheme: dark)')
  .addEventListener('change', (e) => {
    if (!localStorage.getItem('theme')) {
      document.documentElement.setAttribute(
        'data-theme',
        e.matches ? 'dark' : 'light'
      );
    }
  });
```

## 常见组件适配

### 1. 按钮适配

```css
/* 主按钮 */
.button-primary {
  /* 亮色模式 */
  background-color: #1890FF;
  color: #FFFFFF;
}

[data-theme="dark"] .button-primary {
  /* 暗黑模式 */
  background-color: #177DDC;
  color: #FFFFFF;
}

/* 次要按钮 */
.button-secondary {
  /* 亮色模式 */
  background-color: #FFFFFF;
  border: 1px solid #D9D9D9;
  color: #595959;
}

[data-theme="dark"] .button-secondary {
  /* 暗黑模式 */
  background-color: #1E1E1E;
  border: 1px solid rgba(255, 255, 255, 0.12);
  color: #B3B3B3;
}
```

```css

  /* 亮色模式 */

  color: #262626;
}

[data-theme="dark"] .input {
  /* 暗黑模式 */
  background-color: #1E1E1E;
  border: 1px solid rgba(255, 255, 255, 0.12);

```

### 3. 卡片适配

  background-color: #121212;

  background-color: #1E1E1E;
