# 原型工具

## 定义

原型工具是用于创建产品原型的软件，帮助设计师快速验证设计想法、展示交互效果、进行用户测试。

## 原型类型

### 1. 按保真度分类

| 类型 | 特点 | 适用阶段 |
|------|------|----------|
| 低保真 | 简单、快速 | 概念验证 |
| 中保真 | 有一定细节 | 方案讨论 |
| 高保真 | 接近真实 | 用户测试 |

### 2. 按功能分类

| 类型 | 代表工具 | 特点 |
|------|----------|------|
| 综合设计 | Figma | 设计+原型+协作 |
| 专业原型 | Axure | 高保真交互原型 |
| 动效原型 | Principle | 流畅动效 |
| 交互原型 | ProtoPie | 高级交互 |
| 国内工具 | 墨刀 | 简单易用 |

### 3. 动画原理

**状态驱动**
```
State A → Transition → State B

动画 = 起始状态 + 结束状态 + 过渡效果
```

**常用动画**
| 类型 | 说明 |
|------|------|
| Auto | 自动过渡 |
| Scroll | 滚动 |
| Drag | 拖拽 |
| Tap | 点击 |

**动画属性**
```
- 位置

```

```

## ProtoPie

### 1. ProtoPie简介

**特点**
- 高级交互原型
- 传感器支持
- 模块化交互
- 跨设备联动

**适用场景**
- 复杂交互设计
- 多设备联动
- 高保真原型

### 2. 交互概念

**三要素**
```
Trigger（触发）+ Response（响应）= Interaction（交互）

示例：
Tap（点击）+ Move（移动）= 元素跟随点击移动
```

### 3. 触发类型

| 触发 | 说明 |
|------|------|
| Tap | 点击 |
| Long Press | 长按 |
| Swipe | 滑动 |
| Drag | 拖拽 |
| Rotate | 旋转 |
| Pinch | 捏合 |

### 4. 响应类型

| 响应 | 说明 |
|------|------|
| Move | 移动 |
| Scale | 缩放 |
| Rotate | 旋转 |
| Opacity | 透明度 |
| Color | 颜色 |
| Text | 文本 |

### 5. 高级功能

**传感器**
```
- 罗盘

**跨设备联动**

## 原型设计最佳实践

### 1. 原型原则

- **目的明确**：知道要验证什么
- **保真度适当**：不过度追求高保真
- **快速迭代**：快速修改验证
- **用户测试**：让用户参与测试

### 2. 设计流程

```
1. 确定目标
   ↓
2. 选择工具
   ↓
3. 制作原型
   ↓
4. 用户测试
   ↓
5. 迭代优化
```

### 3. 常见错误

- ❌ 过度追求高保真
- ❌ 忽视用户测试
- ❌ 工具选择不当
- ❌ 交互过于复杂

### 4. 交付规范

```
原型交付内容：
1. 原型文件
2. 交互说明文档
3. 操作指引
4. 测试反馈记录
```
