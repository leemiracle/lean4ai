# 图标设计规范

## 定义

图标是简化、抽象化的视觉符号，用于快速传达概念、功能或信息。良好的图标设计能提升用户理解效率和界面美观度。

## 图标的作用

### 1. 功能价值

- **快速识别**：比文字更快传达信息
- **节省空间**：在小面积内传达含义
- **跨语言**：超越文字的语言障碍
- **视觉引导**：引导用户操作

### 2. 情感价值

- 增强品牌识别
- 提升界面美观度
- 创造情感连接

## 图标类型

### 1. 按风格分类

#### 线性图标 (Outlined/Linear)

**特点**
- 线条描绘轮廓
- 简洁、现代、轻盈

**适用场景**
- 界面导航
- 功能入口
- 轻量级界面

```
示例：
┌─────────────────────────────────┐
│  ○  🔍  ⚙️  📁                 │
│  首页 搜索 设置 文件            │
└─────────────────────────────────┘
```

#### 填充图标 (Filled/Solid)

**特点**
- 实心填充
- 突出、醒目、有重量感

**适用场景**
- 激活状态
- 重要功能
- 需要强调时

```
示例：
┌─────────────────────────────────┐
│  ●  🔍  ⚙️  📁                 │
│  首页 搜索 设置 文件            │
└─────────────────────────────────┘
```

#### 双色图标 (Two-tone)

**特点**
- 主色 + 辅助色
- 层次丰富
- 视觉趣味

**适用场景**
- 品牌图标
- 强调图标
- 插画风格

#### 圆角图标 vs 尖角图标

**圆角**
- 友好、柔和
- 适合消费级产品

**尖角**
- 专业、严谨
- 适合工具类产品

### 2. 按功能分类

| 类型 | 说明 | 示例 |
|------|------|------|
| 导航图标 | 页面导航 | 首页、分类、我的 |
| 操作图标 | 用户操作 | 添加、删除、编辑 |
| 状态图标 | 状态指示 | 成功、警告、错误 |
| 品牌图标 | 品牌识别 | Logo、吉祥物 |
| 装饰图标 | 视觉装饰 | 空状态插画 |

## 图标设计原则

### 1. 简洁性

**原则**
- 去除不必要的细节
- 保留核心识别特征
- 最小可识别形式

```
过度复杂 → 简洁有效

原始：🏠（有烟囱、窗户、门、屋顶）
简化：△ + □（保留屋顶+主体）

原始：⚙️（复杂齿轮）
简化：○ + ⚙（简化齿轮）
```

### 2. 一致性

**保持一致的设计语言**

| 属性 | 统一要求 |
|------|----------|
| 风格 | 线性或填充，不混用 |
| 圆角 | 统一圆角半径 |
| 线宽 | 统一线条粗细 |
| 间距 | 统一内部间距 |
| 尺寸 | 统一图标尺寸 |

```
✓ 一致：
┌─────────────────────────────────┐
│  🔍  ⚙️  📁  ❤️                │
│  （都是线性、2px线宽、4px圆角） │
└─────────────────────────────────┘

✗ 不一致：
┌─────────────────────────────────┐
│  🔍  ⚙️  📁  ❤️                │
│  （线宽、圆角、风格不统一）     │
└─────────────────────────────────┘
```

### 3. 可识别性

**原则**
- 图标含义清晰
- 用户能快速理解
- 避免歧义

**常见图标含义**
| 图标 | 通用含义 |
|------|----------|
| 🔍 | 搜索 |
| 🏠 | 首页 |
| ⚙️ | 设置 |
| 🛒 | 购物车 |
| ❤️ | 收藏/喜欢 |
| 🗑️ | 删除 |
| ✏️ | 编辑 |
| ➕ | 添加 |

### 4. 可扩展性

**多尺寸适配**
```
16px × 16px   最小尺寸
24px × 24px   常用尺寸
32px × 32px   大尺寸
48px × 48px   特大尺寸
```

## 图标设计规范

### 1. 尺寸系统

**图标画布**
```
┌─────────────────────────────────┐
│                                 │
│   ┌─────────────────────────┐   │
│   │                         │   │
│   │                         │   │
│   │      图标内容区域       │   │
│   │      (Safe Area)        │   │
│   │                         │   │
│   │                         │   │
│   └─────────────────────────┘   │
│                                 │
└─────────────────────────────────┘
  画布尺寸 (24×24)    安全区域 (20×20)

标准尺寸：24×24px
安全区域：20×20px（留2px边距）
```

**常用尺寸**
| 尺寸 | 用途 |
|------|------|
| 16px | 小图标、行内图标 |
| 20px | 列表图标 |
| 24px | 标准图标 |
| 32px | 大图标 |
| 48px | 特大图标、空状态 |

### 2. 栅格系统

**像素对齐**
```
✓ 正确：线条对齐像素网格
┌─────────────────────────────────┐
│  ├───┤                         │
│  2px                            │
│  线条清晰                       │
└─────────────────────────────────┘

✗ 错误：线条未对齐
┌─────────────────────────────────┐
│   ├───┤                        │
│  1.5px                          │
│  线条模糊                       │
└─────────────────────────────────┘
```

**关键线 (Keylines)**
```
圆形关键线：
    ┌───────────┐
    │     ○     │  直径: 20px
    └───────────┘

方形关键线：
    ┌───────────┐
    │   ┌───┐   │  边长: 18px
    │   │   │   │
    │   └───┘   │
    └───────────┘

横向关键线：
    ┌───────────┐
    │           │
    │  ───────  │  宽: 20px
    │           │
    └───────────┘

纵向关键线：
    ┌───────────┐
    │     │     │
    │     │     │  高: 20px
    │     │     │
    └───────────┘
```

### 3. 线条规范

**线宽**
```
常用线宽：
- 细线：1px / 1.5px
- 标准：2px
- 粗线：2.5px / 3px

推荐：24px图标使用2px线宽
```

**线条端点**
```
平头端点 (Butt Cap)：
─────────

圆头端点 (Round Cap)：
●────●

推荐：圆头端点更友好
```

**线条连接**
```
斜接 (Miter Join)：
┐

圆角 (Round Join)：
╮

推荐：圆角连接更柔和
```

### 4. 圆角规范

**统一圆角**
```
小圆角：2px
标准圆角：4px
大圆角：6px

示例：
┌──────────┐  无圆角
┌────────╮ │  4px圆角
╭────────╯ │  8px圆角
```

### 5. 视觉校正

**光学平衡**
```
问题：几何中心 vs 视觉中心

┌─────────────────────────────────┐
│  △   ◇   ○                     │
│  三角 菱形 圆形                 │
│  （几何中心对齐，视觉不齐）     │
└─────────────────────────────────┘

校正后：
┌─────────────────────────────────┐
│    △                            │
│  ◇  ○                           │
│  （视觉对齐）                    │
└─────────────────────────────────┘
```

## 图标设计流程

```
1. 需求分析
   ├── 确定图标用途
   ├── 明确使用场景
   └── 了解目标用户
        ↓
2. 概念草图
   ├── 收集参考
   ├── 快速草图
   └── 选择最佳方案
        ↓
3. 矢量绘制
   ├── 使用栅格对齐
   ├── 遵循规范
   └── 保持一致性
        ↓
4. 测试优化
   ├── 多尺寸测试
   ├── 可识别性测试
   └── 迭代优化
        ↓
5. 导出交付
   ├── 多格式导出
   ├── 命名规范
   └── 文档说明
```

## 图标使用规范

### 1. 图标 + 文字

**原则**
- 图标辅助文字
- 两者含义一致
- 保持对齐

```
✓ 图标 + 文字组合
┌─────────────────────────────────┐
│  🔍 搜索                        │
│  ⚙️ 设置                        │
│  📁 文件                        │
└─────────────────────────────────┘
```

### 2. 图标状态

| 状态 | 表现 |
|------|------|
| 默认 | 默认颜色 |
| 悬停 | 颜色加深/高亮 |
| 激活 | 填充样式/品牌色 |
| 禁用 | 灰色/半透明 |

```
默认：○ 激活：●
默认：☆ 激活：★
默认：♡ 激活：❤️
```

### 3. 图标颜色

```css
/* 图标颜色规范 */
--icon-primary: #262626;      /* 主要图标 */
--icon-secondary: #8c8c8c;    /* 次要图标 */
--icon-disabled: #bfbfbf;     /* 禁用图标 */
--icon-brand: #1890ff;        /* 品牌色图标 */
--icon-success: #52c41a;      /* 成功 */
--icon-warning: #faad14;      /* 警告 */
--icon-error: #f5222d;        /* 错误 */
```

### 4. 图标间距

```
图标与文字间距：
┌─────────────────────────────────┐
│  🔍│搜索                        │
│    └─ 8px间距                   │
└─────────────────────────────────┘

图标之间间距：
┌─────────────────────────────────┐
│  🔍    ⚙️    📁                │
│    └─ 16px间距                  │
└─────────────────────────────────┘
```

## 图标库管理

### 1. 命名规范

```
格式：[类型]-[名称]-[变体]

示例：

action-add         操作-添加
action-add-filled  操作-添加-填充
```

## 最佳实践

### 1. 设计建议

- 保持简洁
- 保持一致性
- 遵循用户认知习惯
- 提供文字辅助（必要时）

### 2. 常见错误

- ❌ 过于复杂
- ❌ 风格不统一
- ❌ 含义不清晰
- ❌ 尺寸不规范
- ❌ 颜色过多
