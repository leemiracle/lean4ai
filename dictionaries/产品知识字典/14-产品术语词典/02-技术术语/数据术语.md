# 数据术语

## A-D

| 术语 | 英文 | 定义 | 示例 |
|------|------|------|------|
| AB测试 | A/B Testing | 对照实验方法 | 通过A/B测试优化转化率 |
| 归因分析 | Attribution Analysis | 追踪用户来源 | 分析营销渠道归因 |
| 异常检测 | Anomaly Detection | 发现异常数据 | 建立异常检测系统 |
| API调用 | API Call | 接口请求次数 | API调用量达到100万次 |
| 平均值 | Average/Mean | 数据算术平均 | 计算用户平均年龄 |
| 基准 | Benchmark | 对比参照标准 | 与行业基准对比 |
| 大数据 | Big Data | 海量数据技术 | 应用大数据分析用户行为 |
| BI | Business Intelligence | 商业智能 | 搭建BI数据平台 |
| 桶/分桶 | Bucket | 数据分组 | 将用户分桶进行测试 |
| 看板 | Dashboard | 数据可视化面板 | 设计运营数据看板 |
| 数据清洗 | Data Cleaning | 清理脏数据 | 进行数据清洗保证质量 |
| 数据仓库 | Data Warehouse | 数据存储系统 | 搭建企业数据仓库 |
| 数据湖 | Data Lake | 原始数据存储 | 建立数据湖存储日志 |
| 数据挖掘 | Data Mining | 发现数据规律 | 通过数据挖掘发现用户模式 |
| 数据管道 | Data Pipeline | 数据处理流程 | 建立ETL数据管道 |
| 数据可视化 | Data Visualization | 图表展示数据 | 使用可视化呈现分析结果 |
| 脱敏 | Data Masking | 隐藏敏感信息 | 对手机号进行脱敏处理 |
| 维度 | Dimension | 分析角度 | 从时间维度分析趋势 |
| 下钻 | Drill-down | 深入分析细节 | 点击下钻查看明细数据 |
| 去重 | Deduplication | 去除重复 | 用户ID去重统计 |
