# 隐私保护

## 核心摘要
《个人信息保护法》是中国个人信息保护的基础法律，确立了"告知-同意"原则、最小必要原则，赋予用户个人信息权益，产品需建立全流程个人信息保护体系。

## 法律概述

### 基本信息
- **全称**：《中华人民共和国个人信息保护法》
- **通过时间**：2021年8月20日
- **实施时间**：2021年11月1日
- **立法机关**：全国人民代表大会常务委员会
- **监管机构**：国家网信办、工信部、公安部等

### 立法目的
1. 保护个人信息权益
2. 规范个人信息处理活动
3. 促进个人信息合理利用
4. 维护公民合法权益

### 适用范围
- **地域范围**：中国境内处理自然人个人信息
- **域外效力**：向境内提供产品或服务、分析境内自然人
- **主体范围**：所有处理个人信息的组织和个人
- **信息范围**：以电子或其他方式记录的与已识别或可识别自然人有关的各种信息

## 核心原则

### 基本原则
| 原则 | 定义 | 合规要求 |
|-----|------|---------|
| 合法、正当、必要 | 处理需合法正当必要 | 不得过度处理 |
| 最小必要 | 最小范围收集 | 只收集必要信息 |
| 目的限制 | 不得超出目的范围 | 明确告知目的 |
| 公开透明 | 公开处理规则 | 隐私政策公开 |
| 质量保证 | 保证信息准确完整 | 提供更正渠道 |
| 安全保障 | 保障信息安全 | 采取安全措施 |

### 最小必要原则详解
**收集最小化**
- 只收集实现功能所必需的信息
- 不得强制收集非必要信息
- 不得拒绝非必要信息用户使用基本功能

**使用最小化**
- 不得超出约定范围使用
- 不得擅自改变用途
- 确需改变应重新获得同意

**存储最小化**
- 超过存储期限应删除
- 实现目的后及时删除
- 用户注销后及时删除

## 个人信息权益

### 用户核心权益
| 权益 | 定义 | 产品实现 |
|-----|------|---------|
| 知情权 | 知悉信息处理情况 | 隐私政策、告知 |
| 决定权 | 自主决定是否同意 | 授权机制 |
| 查阅复制权 | 查阅、复制个人信息 | 个人信息导出 |
| 更正权 | 更正错误信息 | 信息修改功能 |
| 删除权 | 要求删除信息 | 删除功能 |
| 注销权 | 注销账号 | 账号注销 |
| 撤回同意权 | 撤回授权 | 权限管理 |
| 自动化决策拒绝权 | 拒绝自动化决策 | 关闭个性化推荐 |

### 敏感个人信息

#### 定义与范围
**定义**：一旦泄露或非法使用，容易导致自然人人格尊严受损或人身、财产安全受损的个人信息。

**范围**
| 类别 | 具体信息 |
|-----|---------|
| 生物识别 | 人脸、指纹、声纹、虹膜、基因 |
| 宗教信仰 | 宗教信仰信息 |
| 特定身份 | 残疾、疾病等特定身份 |
| 医疗健康 | 病历、健康检查、生育信息 |
| 金融账户 | 账户、密码、财产信息 |
| 行踪轨迹 | 行踪轨迹信息 |
| 不满14周岁 | 未成年人信息 |
| 其他 | 其他敏感信息 |

#### 敏感信息处理要求
1. **特定目的**：具有特定目的和充分必要性
2. **单独同意**：取得个人单独同意
3. **告知要求**：告知处理的必要性和影响
4. **未成年人**：取得监护人同意

## 处理规则

### "告知-同意"机制

#### 告知要求
| 告知项 | 具体内容 |
|--------|---------|
| 处理者信息 | 名称、联系方式 |
| 处理目的 | 收集、使用目的 |
| 处理方式 | 如何处理 |
| 信息种类 | 收集哪些信息 |
| 保存期限 | 保存多久 |
| 权利行使方式 | 如何行使权利 |
| 第三方 | 是否涉及第三方 |

#### 同意要求
1. **自愿明确**：不得强迫、欺诈、误导
2. **充分知情**：确保用户充分理解
3. **明确表示**：需用户主动勾选或点击
4. **单独同意**：敏感信息、跨境传输需单独同意
5. **书面同意**：法律法规要求书面同意的

#### 例外情形（无需同意）
1. 订立、履行合同所必需
2. 履行法定职责或义务所必需
3. 应对突发公共卫生事件
4. 紧急情况下保护自然人健康、财产安全
5. 为公共利益实施新闻报道、舆论监督
6. 在合理范围内处理已合法公开信息
7. 法律、行政法规规定的其他情形

### 个人信息处理者义务

#### 一般义务
1. **制定内部管理制度**
2. **对个人信息实行分类管理**
3. **采取加密、去标识化等安全技术措施**
4. **合理确定操作权限**
5. **制定应急预案**
6. **开展安全教育培训**
7. **制定个人信息安全事件应急预案**
8. **定期合规审计**

#### 大型互联网平台义务
1. **建立健全个人信息保护机制**
2. **制定平台规则**：明确产品/服务提供者义务
3. **对违规行为处置**：停止服务
4. **定期发布社会责任报告**

## 典型场景合规

### App个人信息收集

#### 基本要求
1. **首次打开**：弹出隐私政策
2. **逐项同意**：逐项获取同意
3. **系统权限**：申请时说明理由
4. **非必要信息**：不得拒绝使用基本功能

#### 禁止行为
1. 未经同意收集
2. 超范围收集
3. 强制收集非必要信息
4. 频繁索要权限
5. 欺骗误导用户
6. 未提供撤回同意途径

#### 常见问题
| 问题 | 合规要求 | 改进建议 |
|-----|---------|---------|
| 一揽子同意 | 逐项同意 | 分场景授权 |
| 默认勾选 | 不得默认勾选 | 默认不勾选 |
| 拒绝后无法使用 | 可使用基本功能 | 允许拒绝后使用 |
| 频繁弹窗 | 合理频次 | 减少打扰 |
| 注销困难 | 提供便捷注销 | 简化注销流程 |

### 个人信息共享

#### 共享类型
| 类型 | 定义 | 合规要求 |
|-----|------|---------|
| 第三方SDK | 集成第三方SDK | 在隐私政策中说明 |
| 数据合作 | 与合作伙伴共享 | 取得用户同意 |
| 集团共享 | 集团内共享 | 告知并获同意 |
| 业务外包 | 委托处理 | 签订协议、监督 |

#### 共享合规要点
1. **必要性评估**：共享是否必要
2. **告知用户**：在隐私政策中告知
3. **获得同意**：取得用户同意
4. **签署协议**：与接收方签署协议
5. **安全评估**：评估接收方安全能力
6. **记录留存**：留存共享记录

### 跨境传输

#### 传输条件
| 条件 | 具体要求 |
|-----|---------|
| 必要性 | 确需向境外提供 |
| 告知 | 向个人告知境外接收方信息 |
| 单独同意 | 取得个人单独同意 |
| 安全评估 | 通过安全评估或认证 |

#### 安全评估情形
1. 关键信息基础设施运营者
2. 处理100万人以上个人信息
3. 累计向境外提供10万人以上个人信息
4. 累计向境外提供1万人以上敏感个人信息
5. 国家网信部门规定的其他情形

#### 合规流程
```
必要性评估 → 告知用户 → 获得单独同意 → 安全评估/认证 → 签署协议 → 传输 → 记录留存
```

### 自动化决策

#### 定义
通过程序自动分析、评估个人行为习惯、兴趣爱好或经济、健康、信用状况，进行决策的活动。

#### 合规要求
1. **透明度**：保证决策透明度
2. **公平性**：不得歧视
3. **拒绝权**：提供拒绝途径
4. **说明义务**：对决策结果说明
5. **人工干预**：提供人工干预途径

#### 典型场景
| 场景 | 合规要点 |
|-----|---------|
| 个性化推荐 | 提供关闭选项 |
| 定价决策 | 不得不合理的差别待遇 |
| 信用评估 | 保证准确性、提供异议渠道 |
| 内容推荐 | 提供关闭推荐选项 |

## 隐私政策要求

### 隐私政策必备内容
1. **个人信息处理者信息**
   - 名称、地址、联系方式
   - 个人信息保护负责人联系方式

2. **个人信息处理规则**
   - 收集哪些信息
   - 收集目的
   - 使用方式
   - 存储期限
   - 共享情况

3. **用户权利及行使方式**
   - 如何查阅、复制
   - 如何更正、删除
   - 如何注销
   - 如何撤回同意

4. **第三方信息**
   - 集成的第三方SDK
   - 共享的合作伙伴

5. **其他**
   - 隐私政策更新方式
   - 争议解决方式

### 隐私政策展示要求
1. **易于访问**：显著位置展示
2. **易于阅读**：清晰、易懂
3. **及时更新**：变化时更新
4. **版本管理**：保留历史版本

### 个人信息保护影响评估（PIA）

#### 适用情形
1. 处理敏感个人信息
2. 利用个人信息自动化决策
3. 委托处理、共享、转让、公开披露
4. 向境外提供个人信息
5. 其他对个人权益有重大影响的处理活动

#### 评估内容
1. 个人信息处理目的、方式是否合法、正当、必要
2. 对个人权益的影响及安全风险
3. 所采取的保护措施是否合法、有效并与风险程度相适应
