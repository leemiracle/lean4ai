## 💼 军工商业模式

### 一、传统模式

#### 1.1 传统军工商业模式

```
传统模式
├─ 客户
│  └─ 单一客户（军方）
│
├─ 采购
│  └─ 计划采购、指令性任务
│
├─ 定价
│  └─ 成本加成定价
│
├─ 竞争
│  └─ 有限竞争、指定研制
│
└─ 交付
   └─ 多年研制、小批量生产
```

---

### 二、新兴模式

#### 2.1 军民融合模式

| 模式 | 说明 | 案例 |
|-----|------|------|
| **军转民** | 军用技术民用化 | 北斗导航、无人机 |
| **民参军** | 民营企业参与军工 | 高德红外、光威复材 |
| **军民两用** | 技术军民通用 | 通信、材料 |

---

#### 2.2 竞争性采购

| 采购方式 | 说明 | 适用场景 |
|---------|------|---------|
| **公开招标** | 公开竞争 | 通用产品、民品配套 |
| **邀请招标** | 邀请特定供应商 | 专业产品 |
| **竞争性谈判** | 多轮谈判 | 复杂装备 |
| **单一来源** | 唯一供应商 | 特殊装备、专利技术 |

**趋势**: 竞争性采购比例逐年提高。

---

#### 2.3 成本控制模式

```
成本控制措施
├─ 目标成本管理
│  └─ 设定目标成本，倒逼设计优化
│
├─ 价值工程
│  └─ 功能/成本优化
│
├─ 精益生产
│  └─ 消除浪费、提高效率
│
└─ 供应链优化
   └─ 集中采购、战略采购
```

## 🔄 商业模式创新

### 一、全生命周期服务

#### 1.1 模式说明

| 阶段 | 服务内容 |
|-----|---------|
| **研制阶段** | 研发设计、试验验证 |
| **生产阶段** | 制造、质量控制 |
| **服役阶段** | 维修保障、升级改造 |
| **退役阶段** | 回收处置、退役管理 |

---

#### 1.2 一体化保障

```
一体化保障模式
├─ 研制+生产+保障一体化
│  └─ 总体单位负责全生命周期
│
├─ 维修保障社会化
│  └─ 引入社会力量参与保障
│
└─ 售后服务外包
   └─ 非核心业务外包
```

---

### 二、平台化模式

#### 2.1 平台化产品

| 平台 | 基础型号 | 改型产品 |
|-----|---------|---------|
| **歼-10平台** | 歼-10A | 歼-10B、歼-10C |
| **运-8平台** | 运-8 | 预警机、反潜机、电子战机 |
| **054A平台** | 054A护卫舰 | 出口型、改进型 |

**白话解释平台化**:
一个基础型号，通过换装不同设备，改装成不同用途的装备，就像汽车平台一样，降低成本、缩短周期。

---

#### 2.2 模块化设计

```
模块化设计
├─ 动力模块（发动机模块化）
├─ 武器模块（武器模块化）
├─ 电子模块（航电、雷达模块化）
└─ 任务模块（任务载荷模块化）
```

**优势**: 
- 快速改型
- 降低成本
- 提高通用性
- 便于维护

## 📊 商业模式对比

### 传统模式 vs 新模式

| 维度 | 传统模式 | 新模式 |
|-----|---------|--------|
| **客户** | 单一（军方） | 多元（军方+外贸+民用） |
| **竞争** | 有限竞争 | 开放竞争 |
| **定价** | 成本加成 | 市场化定价、目标成本 |
| **产品** | 定制化 | 平台化、模块化 |
| **服务** | 产品交付 | 全生命周期服务 |
| **技术** | 封闭研发 | 开放创新、军民融合 |
