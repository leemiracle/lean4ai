# 视觉心理学

## 概述
视觉心理学研究人类如何感知和理解视觉信息，为界面设计、视觉传达和用户体验设计提供科学的理论基础。

## 核心理论

### 视觉感知原理

#### 格式塔原理（Gestalt Principles）
1. **接近律（Proximity）**
   - 距离相近的元素被视为一组
   - 空间关系影响分组

2. **相似律（Similarity）**
   - 相似的元素被视为一组
   - 形状、颜色、大小相似

3. **连续律（Continuity）**
   - 视线沿平滑路径移动
   - 倾向于看到连续的线条

4. **闭合律（Closure）**
   - 大脑自动补全不完整图形
   - 倾向于看到完整的形状

5. **图形-背景（Figure-Ground）**
   - 区分主体和背景
   - 注意力焦点

6. **共同命运（Common Fate）**
   - 同方向运动的元素被视为一组
   - 动画分组原理

#### 视觉层次
1. **F型扫描模式**
   - 水平扫描顶部
   - 垂直扫描左侧
   - 主要内容在左上

2. **Z型扫描模式**
   - 左上到右上
   - 斜向左下到右下
   - 适用于简单页面

3. **视觉权重**
   - 大小、颜色、对比度
   - 位置、留白、纹理

### 色彩心理学

#### 色彩的情感意义
1. **红色**
   - 情感：激情、紧迫、危险
   - 应用：促销、警告、重要按钮
   - 生理：提高心率、刺激食欲

2. **蓝色**
   - 情感：信任、专业、平静
   - 应用：企业、科技、金融
   - 生理：降低血压、放松

3. **绿色**
   - 情感：自然、成长、安全
   - 应用：环保、健康、成功
   - 生理：放松眼睛

4. **黄色**
   - 情感：乐观、注意、温暖
   - 应用：警告、优惠、活泼品牌
   - 注意：过多引起焦虑

5. **橙色**
   - 情感：活力、友好、创意
   - 应用：CTA按钮、年轻品牌
   - 平衡：红色和黄色的特点

6. **紫色**
   - 情感：奢华、神秘、创意
   - 应用：高端品牌、创意领域
   - 稀有性：自然界较少见

#### 色彩组合原则
1. **60-30-10规则**
   - 60%主色
   - 30%辅助色
   - 10%强调色

2. **对比度原则**
   - 确保可读性
   - WCAG标准
   - 色盲友好

3. **色彩和谐**
   - 互补色
   - 类似色
   - 三角配色

### 视觉注意力

#### 注意力分配
1. **自下而上注意**
   - 被视觉特征吸引
   - 颜色、运动、对比

2. **自上而下注意**
   - 基于目标和期望
   - 任务驱动的搜索

3. **非注意盲视**
   - 注意力聚焦时忽略其他
   - 变化盲视

#### 视觉引导技术
1. **视觉流**
   - 引导眼睛移动路径
   - 符合阅读习惯

2. **焦点设计**
   - 一个主要焦点
   - 通过对比创造

3. **视觉锚点**
   - 固定视线位置
   - 减少认知负担

## 案例分析

### 案例1：Apple官网设计
**心理学原理**：留白、视觉焦点、色彩心理学
**设计应用**：
- 大量留白突出产品
- 产品图片为视觉焦点
- 简洁的色彩方案
- Z型视觉流
**效果**：传达高端、简约的品牌形象

### 案例2：Google搜索首页
**心理学原理**：格式塔原理、注意力聚焦
**设计应用**：
- 极简设计减少干扰
- Logo和搜索框为焦点
- 两个按钮的对称布局
- 彩色Logo增加记忆点
**效果**：快速完成搜索任务，用户体验极佳

### 案例3：Pinterest瀑布流
**心理学原理**：接近律、连续律、视觉流
**设计应用**：
- 瀑布流布局
- 等宽不等高的卡片
- 无缝滚动
- 视觉连续性
**效果**：沉浸式浏览体验，延长停留时间

### 案例4：Spotify深色主题
**心理学原理**：色彩心理学、视觉疲劳
**设计应用**：
- 深色背景减少眼疲劳
- 绿色强调品牌色
- 高对比度文本
- 色彩编码功能
**效果**：适合长时间使用，品牌识别度高

### 案例5：Airbnb的色彩系统
**心理学原理**：色彩情感、品牌识别
**设计应用**：
- 温暖的品牌色（粉色系）
- 友好的插画风格
- 一致的色彩语言
- 情感化的视觉传达
**效果**：传达欢迎、归属的品牌理念

### 案例6：Duolingo的视觉设计
**心理学原理**：色彩情感、游戏化视觉
**设计应用**：
- 明亮的色彩方案
- 友好的吉祥物
- 进度条和徽章
- 游戏化的界面元素
**效果**：降低学习压力，增加趣味性

## 设计清单

### 格式塔原理检查
- [ ] 相关元素是否分组
- [ ] 是否有清晰的视觉层次
- [ ] 图形和背景是否区分
- [ ] 是否有不必要的闭合需求

### 色彩设计检查
- [ ] 色彩方案是否一致
- [ ] 对比度是否足够
- [ ] 是否考虑色盲用户
- [ ] 色彩是否传达正确情感

### 视觉层次检查
- [ ] 是否有明确的视觉焦点
- [ ] 视觉流是否清晰
- [ ] 留白是否合理
- [ ] 重要性是否通过视觉体现

### 可读性检查
- [ ] 字体大小是否合适
- [ ] 行距和字距是否舒适
- [ ] 文本对比度是否足够
- [ ] 段落长度是否合理
