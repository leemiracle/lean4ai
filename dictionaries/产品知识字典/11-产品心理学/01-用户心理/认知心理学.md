# 认知心理学

## 概述
认知心理学研究人类如何处理信息、形成认知和理解世界，为产品设计提供科学的理论基础。

## 核心理论

### 认知偏差
1. **确认偏差**
   - 人们倾向于寻找支持自己观点的信息
   - 在产品中表现为：用户更容易注意到符合预期的内容

2. **锚定效应**
   - 最初获得的信息会影响后续判断
   - 在定价、评估中起重要作用

3. **可得性偏差**
   - 容易想到的事物被认为更常见或重要
   - 影响用户的风险评估和决策

4. **框架效应**
   - 同一问题的不同表述方式会导致不同决策
   - "90%成功率" vs "10%失败率"

### 注意力机制
1. **选择性注意**
   - 用户只能关注有限的信息
   - 产品需要突出重要内容

2. **注意力经济**
   - 注意力是稀缺资源
   - 设计需要高效传达信息

3. **视觉注意力**
   - 眼动追踪显示的关注模式
   - F型阅读模式、Z型阅读模式

### 记忆系统
1. **工作记忆**
   - 容量有限（7±2法则）
   - 临时存储和处理信息

2. **长期记忆**
   - 知识的永久存储
   - 通过重复和联想强化

3. **情景记忆**
   - 特定事件和经历的记忆
   - 影响用户体验回忆

## 产品设计应用

### 信息架构设计
1. **认知负荷管理**
   - 分块呈现信息
   - 渐进式披露
   - 减少不必要的决策

2. **导航设计**
   - 清晰的层级结构
   - 一致的位置和标识
   - 面包屑导航

3. **搜索设计**
   - 智能联想
   - 搜索历史
   - 高级筛选

### 界面设计原则
1. **一致性原则**
   - 降低学习成本
   - 利用已有心智模型

2. **反馈机制**
   - 即时反馈
   - 明确的状态指示

3. **错误预防**
   - 约束用户行为
   - 友好的错误提示

### 内容呈现策略
1. **视觉层次**
   - 大小对比
   - 色彩对比
   - 空间布局

2. **信息优先级**
   - 核心信息优先
   - 相关信息聚合

3. **可读性优化**
   - 字体大小和行距
   - 段落长度控制
   - 留白设计

## 案例分析

### 案例1：Gmail的标签系统
**心理学原理**：工作记忆容量限制、分类认知
**设计应用**：
- 使用颜色和标签管理邮件分类
- 标签数量限制，避免认知过载
- 自定义标签满足个性化需求
**效果**：提高邮件管理效率，降低用户的认知负担

### 案例2：电商平台的筛选功能
**心理学原理**：选择性注意、决策简化
**设计应用**：
- 多维度筛选选项
- 实时结果更新
- 已选条件可视化
**效果**：帮助用户快速找到目标商品，提升转化率

### 案例3：Spotify的播放列表推荐
**心理学原理**：可得性偏差、习惯形成
**设计应用**：
- 基于听歌历史推荐
- 个性化播放列表
- 每周发现新音乐
**效果**：增加用户粘性，提升使用时长

### 案例4：Uber的ETA显示
**心理学原理**：不确定性厌恶、即时反馈
**设计应用**：
- 实时位置追踪
- 预计到达时间
- 司机信息展示
**效果**：降低用户等待焦虑，提升满意度

- [ ] 是否有不必要的元素
- [ ] 导航是否清晰
- [ ] 是否有渐进式披露

### 注意力引导检查
- [ ] 重要信息是否突出
- [ ] 是否有视觉层次
- [ ] 是否避免了注意力分散

### 记忆友好检查
- [ ] 是否有一致的设计模式
- [ ] 是否有清晰的标识
- [ ] 是否支持搜索和筛选
- [ ] 是否有历史记录功能
