# 行为心理学

## 概述
行为心理学研究人类行为的形成机制、改变规律，为产品设计提供行为引导的理论依据。

## 核心理论

### 行为模型

#### Fogg行为模型
**公式：B = MAT**
- **B (Behavior)** - 行为
- **M (Motivation)** - 动机
- **A (Ability)** - 能力
- **T (Trigger)** - 触发器

**三个要素必须同时满足才能产生行为**

1. **动机**
   - 追求快乐/避免痛苦
   - 追求希望/避免恐惧
   - 追求社会认同/避免拒绝

2. **能力**
   - 时间成本
   - 金钱成本
   - 体力成本
   - 脑力成本
   - 社会成本
   - 习惯偏离

3. **触发器**
   - 外部触发：通知、广告、推荐
   - 内部触发：情绪、需求、习惯

### 习惯养成模型

#### 习惯回路
1. **提示(Cue)**
   - 触发行为的环境或信号
   - 时间提示、地点提示、情绪提示

2. **惯性行为(Routine)**
   - 自动化的行为模式
   - 需要重复练习形成

3. **奖励(Reward)**
   - 行为的正向反馈
   - 强化习惯形成

#### 习惯形成的四个阶段
1. **触发期** - 外部提示触发行为
2. **行动期** - 开始执行行为
3. **多变奖励期** - 不可预测的奖励强化
4. **投入期** - 用户投入增加转换成本

### 上瘾模型

#### Hook模型（Nir Eyal）
1. **触发(Trigger)**
   - 外部触发：推送、广告、社交
   - 内部触发：无聊、孤独、焦虑

2. **行动(Action)**
   - 最小可行性行为
   - 降低行动门槛

3. **多变奖励(Variable Reward)**
   - 部落奖励：社交认同
   - 猎物奖励：资源、信息
   - 自我奖励：操控感、成就感

4. **投入(Investment)**
   - 增加用户资产
   - 提高转换成本
   - 为下次触发做准备

## 产品设计应用

### 行为引导设计
1. **降低行动门槛**
   - 简化流程
   - 提供默认选项
   - 自动化操作

2. **增强动机**
   - 游戏化元素
   - 社交压力
   - 即时反馈

3. **优化触发器**
   - 个性化推送
   - 合适的时间和地点
   - 情感化文案

### 习惯养成策略
1. **建立触发机制**
   - 定期提醒
   - 场景关联
   - 社交互动

2. **简化核心行为**
   - 一键操作
   - 快捷方式
   - 语音控制

3. **设计奖励系统**
   - 积分徽章
   - 进度条
   - 社交分享

### 行为改变技术
1. **小步快跑**
   - 循序渐进的目标
   - 庆祝小成就

2. **社会认同**
   - 显示他人行为
   - 团队竞赛

3. **承诺与一致性**
   - 公开承诺
   - 逐步升级

## 案例分析

### 案例1：Duolingo的语言学习
**心理学原理**：习惯养成、上瘾模型
**设计应用**：
- 每日学习提醒（触发）
- 5分钟短课程（降低门槛）
- 连续打卡奖励（多变奖励）
- 学习排行榜（社交压力）
- 金币和宝石（虚拟奖励）
**效果**：用户日活和留存率显著提升

### 案例2：微信朋友圈
**心理学原理**：多变奖励、社交认同
**设计应用**：
- 红点提示（触发）
- 下拉刷新（行动）
- 不可预测的内容（多变奖励）
- 点赞和评论（社交奖励）
- 发布内容（投入）
**效果**：用户每天多次打开，形成使用习惯

### 案例3：支付宝蚂蚁森林
**心理学原理**：行为塑造、社会认同
**设计应用**：
- 每日能量收集提示（触发）
- 点击收集（简单行动）
- 偷好友能量（游戏化奖励）
- 种树公益（自我奖励）
- 好友排行榜（社会认同）
**效果**：提升用户活跃度和粘性

### 案例4：Netflix的自动播放
**心理学原理**：最小阻力路径、默认效应
**设计应用**：
- 剧集结束自动播放下一集
- 跳过片头片尾
- 个性化推荐
- 继续观看功能
**效果**：延长观看时长，减少用户流失

### 案例5：Instagram的故事功能
**心理学原理**：稀缺性、FOMO心理
**设计应用**：
- 24小时后消失（稀缺性）
- 置顶显示（视觉提示）
- 即时互动（低门槛）
- 浏览记录（社交奖励）
**效果**：增加用户发布频率和互动
