## 📖 核心知识点

### 产品思维
```
用户思维 → 场景思维 → 价值思维 → 数据思维 → 系统思维
```

### 产品流程
```
需求 → 设计 → 开发 → 测试 → 上线 → 迭代
```

### 增长模型
```
AARRR: 获取 → 激活 → 留存 → 变现 → 推荐
```

### 数据指标
```
北极星指标 → OKR → AARRR → 业务指标 → 技术指标
```

## 🚀 学习路径

### 初级产品经理（0-1年）
```
Week 1-2: 产品思维 → 产品流程
Week 3-4: PRD撰写 → 原型设计
Week 5-6: 需求分析 → 竞品分析
Week 7-8: 敏捷开发 → 团队协作
```

### 中级产品经理（1-3年）
```
Month 1: 数据分析 → 指标体系
Month 2: 用户研究 → 可用性测试
Month 3: 增长黑客 → 运营策略
Month 4: 商业模式 → 定价策略
```

### 高级产品经理（3-5年）
```
Quarter 1: 产品战略 → 创新方法
Quarter 2: 行业洞察 → 商业创新
Quarter 3: 团队管理 → 人才培养
Quarter 4: 职业发展 → 个人品牌
```
