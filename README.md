## 📚 项目简介

**形式化验证 + AI辅助 + 知识词典 + 代码验证** 的完整解决方案

包含 **300万+字** 的知识词典系统，**806个知识文件**，整合 **8大开源项目**，覆盖 **11个专业学科**。

## 🚀 快速开始

### 知识词典用户

```bash
# 浏览词典
cd dictionaries/
├── 专业学科字典/      # 11个学科领域
├── 行业知识字典/      # 能源、制造、农业等
└── 产品知识字典/      # 软硬件产品知识
```

### 技术开发者

```bash
# 1. 环境检查
lean --version  # 4.28.0

# 2. 安装依赖
pip install lean-dojo

# 3. 运行示例
cd lean4-projects/YC-Killer-Lean4
python3 lean4_bridge.py
```
