## 📖 使用指南

### 费曼学习法说明

每个术语的解释遵循费曼学习法原则：

1. **简单定义**：用日常语言给出核心含义
2. **类比说明**：用熟悉的例子帮助理解
3. **应用场景**：说明该概念在何处使用
4. **相关概念**：关联其他哲学概念

### 符号说明

- 📌 核心概念（必须掌握）
- 🔍 扩展阅读
- 💡 实例说明
- ⚡ 易混淆辨析
- 🆕 新兴概念（2010年后）
