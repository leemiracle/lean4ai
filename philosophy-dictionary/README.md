## 哲学是什么？（费曼式解释）

**想象你是一个永远在问"为什么"的三岁小孩：**

- 为什么天是蓝的？→ 科学问题
- 天是真实存在的吗？→ 哲学问题
- 我怎么知道我看到的天是真的？→ 认识论
- 什么样的世界是好的世界？→ 价值论

**哲学就是系统地思考这些最根本的问题：**
1. 世界是什么？（本体论）
2. 我们能知道什么？（认识论）
3. 我们应该怎么做？（伦理学）
4. 什么是美的？（美学）
5. 如何正确推理？（逻辑学）
