# 农学知识词典

## 📚 概述

本词典基于**中国教育部《学位授予和人才培养学科目录（2020年）》**，对农学门类（代码：09）的所有一级学科和二级学科进行全面系统的知识分类。

农学是研究农业生产理论和技术的学科门类，涵盖作物生产、园艺生产、农业资源利用、植物保护、动物生产、兽医、林业、渔业等领域。
