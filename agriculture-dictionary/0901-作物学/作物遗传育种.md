# 作物遗传育种（Crop Genetics and Breeding）

**学科代码**：090102  
**一级学科**：作物学（0901）

---

## 📌 学科简介

作物遗传育种是研究作物遗传变异规律和品种改良方法的学科，是提高作物产量、品质和抗性的核心技术。它结合遗传学、分子生物学和生物统计学，创造和选育优良品种。

---

## 🎯 核心概念

### 一、遗传基础

#### 1. 基因型与表型（Genotype and Phenotype）

**费曼解释**：
基因型是写在DNA里的"配方"，表型是做出来的"成品"。同样的配方，不同厨师（环境）做出的菜可能不一样。作物也一样，基因决定潜力，环境影响表现。

**核心要点**：
- 表型(P) = 基因型(G) + 环境(E) + G×E
- 遗传力 = 遗传变异/总变异
- 基因型×环境互作

**应用场景**：品种评价、选择决策

---

#### 2. 杂种优势（Heterosis）

**费曼解释**：
两个不同品种杂交，后代表现比父母都好——这就是杂种优势。杂交玉米、杂交水稻之所以产量高，就是利用了这个原理。但杂种优势只能用一代，下一代性状就乱了。

**表现**：
| 方面 | 表现 |
|------|------|
| 生长势 | 更旺盛 |
| 产量 | 增产15-30% |
| 抗性 | 增强 |

**利用技术**：
- 雄性不育系（三系法）
- 光温敏不育系（两系法）
- 化学杀雄

**核心要点**：
- 一般配合力（GCA）
- 特殊配合力（SCA）
- 只能用F1代

**应用场景**：玉米、水稻、油菜、蔬菜

---

#### 3. 分子育种（Molecular Breeding）

**费曼解释**：
传统育种靠"看"——看长相、看产量。分子育种靠"查"——查DNA，在种子阶段就知道它有哪些好基因。就像用GPS导航比看路牌更精准。

**主要技术**：
| 技术 | 作用 |
|------|------|
| 分子标记辅助选择（MAS） | 追踪目标基因 |
| 基因编辑（CRISPR） | 精准修改基因 |
| 全基因组选择（GS） | 预测育种值 |
| 转基因 | 导入外源基因 |

**核心要点**：
- 分子标记（SSR、SNP）
- QTL定位
- 功能基因克隆

**应用场景**：现代育种核心技术

---

### 二、育种方法

#### 4. 杂交育种（Cross Breeding）

**费曼解释**：
选两个各有优点的品种杂交，在后代中挑选结合了双亲优点的新品种。就像一个高一个聪明，希望孩子既高又聪明。

**程序**：
```
选亲本 → 杂交 → F1 → 自交分离 → 选择(F2-F6) → 稳定品系 → 试验 → 审定
```

**亲本选配**：
- 性状互补
- 适应性广
- 亲缘关系适中

**核心要点**：
- 组合育种
- 超亲育种
- 回交育种

**应用场景**：所有作物

---

#### 5. 选择育种（Selection Breeding）

**费曼解释**：
从群体中挑选表现好的个体繁殖，一代代筛选。就像赛马选种，留下最好的配种。

**方法**：
| 方法 | 适用 |
|------|------|
| 混合选择 | 自花授粉作物 |
| 穗行选择 | 提高选择效率 |
- 选择强度
- 遗传进度
- 选择周期

**应用场景**：自花授粉作物改良

---

### 三、种子生产

#### 6. 良种繁育（Seed Production）

**费曼解释**：
品种育成后要大量繁殖供农民使用。但繁殖过程中要保证种子纯度，不能混杂退化。

**程序**：
```
育种家种子 → 原原种 → 原种 → 良种 → 大田生产
```

**质量控制**：
| 指标 | 要求 |
|------|------|
| 纯度 | ≥98% |
| 发芽率 | ≥85% |
| 水分 | ≤13% |
| 净度 | ≥98% |

**核心要点**：
- 防杂保纯
- 原种更新
- 种子检验

**应用场景**：种子企业、种业发展
