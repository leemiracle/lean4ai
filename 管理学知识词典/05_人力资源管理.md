### 组织行为学

**学科归属**：人力资源管理 > 组织行为学  
**英文名称**：Organizational Behavior (OB)  
**关联概念**：[个体行为] [群体行为] [组织文化]

#### 核心定义（一句话）
研究组织中人的行为及其规律的学科。

#### 费曼式解释（说人话）

**什么是组织行为学？**

简单说：**研究"人在组织里为什么这么干"**。

**三个层次**：

| 层次 | 研究什么 | 问题 |
|------|---------|------|
| 个体 | 个人的行为 | 为什么他努力工作？ |
| 群体 | 团队的行为 | 为什么这个团队效率低？ |
| 组织 | 整体的行为 | 为什么这家公司有创新文化？ |

**打个比方**：
- 组织就像一台机器
- 人是零件
- 组织行为学研究零件怎么配合、为什么有时配合不好

**为什么重要？**
- 管理者要管人
- 了解人才能管好人
- 提高组织效率

### 动机

**学科归属**：人力资源管理 > 组织行为学  
**英文名称**：Motivation  
**关联概念**：[激励] [需求] [目标]

#### 核心定义（一句话）
推动个体采取行动的内在驱动力。

#### 费曼式解释（说人话）

**什么是动机？**

简单说：**让人行动的原因**。

**动机的过程**：
```
未满足的需求 → 紧张感 → 驱动力 → 行为 → 目标 → 需求满足
```

**经典理论**：

| 理论 | 核心观点 | 应用 |
|------|---------|------|
| 马斯洛需求层次 | 需求从低到高 | 满足不同层次需求 |
| 赫茨伯格双因素 | 保健因素vs激励因素 | 保健防不满，激励促努力 |
| 期望理论 | 努力→绩效→奖励→目标 | 设定可达目标、公平奖励 |
| 公平理论 | 比较决定满意度 | 确保公平感 |

**打个比方**：
- 动机像汽车的引擎
- 推动人前进
- 不同引擎动力不同

### 马斯洛需求层次理论

**学科归属**：人力资源管理 > 激励理论  
**提出者**：亚伯拉罕·马斯洛（1943）  
**英文名称**：Maslow's Hierarchy of Needs  
**关联概念**：[动机] [自我实现]

#### 核心定义（一句话）
人类需求从低到高分为五个层次，低层次满足后追求高层次。

#### 费曼式解释（说人话）

**五个层次**（从低到高）：

```
        自我实现需求
           /\
        尊重需求
           /\
        社交需求
           /\
        安全需求
           /\
        生理需求
```

| 层次 | 需求 | 例子 | 工作中的体现 |
|------|------|------|-------------|
| 生理 | 吃喝住 | 基本生活保障 | 基本工资 |
| 安全 | 安全稳定 | 不被伤害 | 社保、稳定工作 |
| 社交 | 归属感 | 朋友、爱情 | 团队氛围 |
| 尊重 | 被认可 | 地位、成就 | 晋升、表扬 |
| 自我实现 | 实现潜能 | 发挥才能 | 挑战性工作 |

**应用**：
- 了解员工在哪个层次
- 提供相应的激励
- 不能跳过低层次需求

### 赫茨伯格双因素理论

**学科归属**：人力资源管理 > 激励理论  
**提出者**：弗雷德里克·赫茨伯格（1959）  
**英文名称**：Herzberg's Two-Factor Theory  
**关联概念**：[保健因素] [激励因素]

#### 核心定义（一句话）
影响工作满意度的因素分为保健因素和激励因素两类。

#### 费曼式解释（说人话）

**两种因素**：

| 类型 | 作用 | 例子 |
|------|------|------|
| 保健因素 | 没有→不满；有→不不满 | 工资、工作条件、政策 |
| 激励因素 | 有→满意；没有→不满意 | 成就、认可、成长机会 |

**关键点**：
- 保健因素：防止不满，但不会产生满意
- 激励因素：产生满意，激发积极性

**打个比方**：
- 保健因素像房子的地基——没有会塌，有也只是正常
- 激励因素像装修——没有也能住，有会更满意

**管理启示**：
1. 先做好保健因素（工资不能太低）
2. 再用激励因素（给予挑战、认可）

### 期望理论

**学科归属**：人力资源管理 > 激励理论  
**提出者**：维克多·弗鲁姆（1964）  
**英文名称**：Expectancy Theory  
**关联概念**：[期望] [效价] [工具性]

#### 核心定义（一句话）
人的动机取决于对努力-绩效-奖励关系的期望。

#### 费曼式解释（说人话）

**公式**：
```
动机 = 期望(E) × 工具性(I) × 效价(V)
```

**三个要素**：

| 要素 | 问题 | 含义 |
|------|------|------|
| 期望(E) | 努力能达标吗？ | 努力→绩效的概率 |
| 工具性(I) | 达标能得奖吗？ | 绩效→奖励的概率 |
| 效价(V) | 奖励我想要吗？ | 奖励的价值 |

**例子**：
- 老板说"销售第一名奖励10万"
- 你想：
  - 我努力能拿第一吗？（期望）
  - 拿了第一真给10万吗？（工具性）
  - 我想要10万吗？（效价）

**只要有一个是0，动机就是0**

**管理启示**：
1. 设定可达成的目标（提高期望）
2. 承诺必须兑现（提高工具性）
3. 给员工想要的奖励（提高效价）

### 公平理论

**学科归属**：人力资源管理 > 激励理论  
**提出者**：约翰·亚当斯（1963）  
**英文名称**：Equity Theory  
**关联概念**：[公平感] [比较]

#### 核心定义（一句话）
员工会将自己投入产出比与他人比较，不公平感会影响行为。

#### 费曼式解释（说人话）

**公平的公式**：
```
自己的产出/自己的投入 = 他人的产出/他人的投入
```

**比较对象**：
- 同事（内部比较）
- 行业同行（外部比较）
- 自己过去（时间比较）

**三种情况**：

| 情况 | 感受 | 行为 |
|------|------|------|
| 产出/投入 > 他人 | 奢侈感 | 可能更努力（内疚） |
| 产出/投入 = 他人 | 公平感 | 保持现状 |
| 产出/投入 < 他人 | 不公平感 | 减少投入、抱怨、离职 |

**例子**：
- 你工资1万，工作量10
- 同事工资1.2万，工作量10
- 你会觉得不公平

**管理启示**：
1. 薪酬要公平透明
2. 绩效考核要公正
3. 关注员工的公平感

### 群体动力学

**学科归属**：人力资源管理 > 组织行为学  
**英文名称**：Group Dynamics  
**关联概念**：[群体规范] [从众] [凝聚力]

#### 核心定义（一句话）
研究群体内部互动规律的理论。

#### 费曼式解释（说人话）

**什么是群体动力学？**

研究**群体内部发生什么**。

**群体的发展阶段**（塔克曼模型）：

| 阶段 | 特点 | 行为 |
|------|------|------|
| 形成期 | 不确定 | 试探、观察 |
| 风暴期 | 冲突 | 争夺角色、意见分歧 |
| 规范期 | 和谐 | 建立规则、合作 |
| 执行期 | 高效 | 目标一致、配合默契 |
| 解散期 | 结束 | 任务完成 |

**群体规范**：
- 群体成员共同遵守的行为准则
- 可以促进或阻碍绩效

**从众**：
- 跟随多数的行为
- 可能有"群体思维"风险

### 社会惰化

**学科归属**：人力资源管理 > 组织行为学  
**英文名称**：Social Loafing  
**关联概念**：[群体行为] [搭便车]

#### 核心定义（一句话）
个人在群体中工作时比单独工作时努力程度下降的现象。

#### 费曼式解释（说人话）

**什么是社会惰化？**

简单说：**人多的时候反而偷懒**。

**例子**：
- 一个人拔河：100%努力
- 10个人拔河：每个人可能只出70%力

**原因**：
1. 个人贡献难以识别
2. 责任分散
3. 觉得自己的努力不重要

**如何克服**：
1. 明确个人责任
2. 评估个人贡献
3. 增强群体认同
4. 让工作有意义

**打个比方**：
- 像大锅饭：干多干少一个样
- 人人都在"搭便车"

### 组织文化

**学科归属**：人力资源管理 > 组织行为学  
**英文名称**：Organizational Culture  
**关联概念**：[价值观] [规范] [符号]

#### 核心定义（一句话）
组织成员共同拥有的价值观、信念、规范和行为方式。

#### 费曼式解释（说人话）

**什么是组织文化？**

简单说：**"我们这里是怎么做事的"**。

**组织文化的层次**（洋葱模型）：

```
表层：可见物（着装、环境、口号）
      ↓
中层：行为规范（怎么做事情）
      ↓
深层：价值观（什么重要）
      ↓
核心：基本假设（不言而喻的信念）
```

**例子**：
- 谷歌：创新、开放
- 华为：狼性文化、奋斗
- 阿里巴巴：客户第一、拥抱变化

**功能**：
1. 身份认同
2. 行为引导
3. 稳定感
4. 竞争优势

**怎么塑造？**
- 创始人/领导者示范
- 招聘选择
- 社会化培训
- 奖惩制度

### 人力资源规划

**学科归属**：人力资源管理 > 人力资源规划  
**英文名称**：Human Resource Planning  
**关联概念**：[需求预测] [供给分析]

#### 核心定义（一句话）
预测组织未来人力需求与供给，制定人力资源平衡计划的过程。

#### 费曼式解释（说人话）

**什么是人力资源规划？**

简单说：**未来需要多少人？什么样的人？从哪里来？**

**三个步骤**：

1. **需求预测**
   - 业务发展需要多少人？
   - 需要什么技能？

2. **供给分析**
   - 现有多少人？
   - 未来会有多少（离职、退休）？

3. **平衡计划**
   - 缺人 → 招聘/培养
   - 多人 → 裁员/转岗

**分析方法**：

| 方法 | 说明 |
|------|------|
| 定性方法 | 管理者判断、德尔菲法 |
| 定量方法 | 趋势分析、比率分析 |

### 招聘

**学科归属**：人力资源管理 > 招聘与配置  
**英文名称**：Recruitment  
**关联概念**：[选拔] [人岗匹配]

#### 核心定义（一句话）
吸引和筛选适合岗位要求的人才的过程。

#### 费曼式解释（说人话）

**什么是招聘？**

简单说：**找到合适的人**。

**招聘流程**：

```
工作分析 → 制定招聘计划 → 发布招聘信息 → 筛选简历 → 
面试测试 → 背景调查 → 录用决策 → 入职
```

**招聘渠道**：

| 渠道 | 优点 | 缺点 |
|------|------|------|
| 内部招聘 | 成本低、熟悉组织 | 选择范围小 |
| 外部招聘 | 选择范围大 | 成本高、风险大 |
| 校园招聘 | 潜力大、可塑性强 | 需要培训 |
| 猎头 | 高端人才 | 费用高 |
| 社交媒体 | 覆盖广、成本低 | 筛选工作量大 |

**选拔方法**：
- 简历筛选
- 面试（结构化/非结构化）
- 测评（能力、性格）
- 情景模拟
- 背景调查

### 结构化面试

**学科归属**：人力资源管理 > 招聘  
**英文名称**：Structured Interview  
**关联概念**：[面试] [选拔]

#### 核心定义（一句话）
按照预先设计的结构和程序进行的面试。

#### 费曼式解释（说人话）

**什么是结构化面试？**

**结构化 vs 非结构化**：

| 结构化 | 非结构化 |
|--------|----------|
| 问题预先设计 | 随机提问 |
| 按顺序问 | 想问什么问什么 |
| 统一评分标准 | 主观评价 |
| 可信度高 | 可信度低 |

**常见问题类型**：
- **情境问题**：如果遇到X情况，你会怎么做？
- **行为问题**：请举例说明你曾经怎么处理X？
- **知识问题**：你对X了解多少？

**STAR法则**（行为面试）：
- **S**ituation：什么情境？
- **T**ask：什么任务？
- **A**ction：做了什么？
- **R**esult：结果怎样？

### 培训与开发

**学科归属**：人力资源管理 > 培训开发  
**英文名称**：Training and Development  
**关联概念**： [学习] [能力提升]

#### 核心定义（一句话）
提升员工能力和素质的有计划活动。

#### 费曼式解释（说人话）

**什么是培训与开发？**

**培训 vs 开发**：

| 培训 | 开发 |
|------|------|
| 关注当前工作 | 关注未来发展 |
| 短期 | 长期 |
| 技能导向 | 潜能导向 |

**培训流程**（ADDIE模型）：

| 阶段 | 内容 |
|------|------|
| 分析(A) | 确定培训需求 |
| 设计(D) | 制定培训计划 |
| 开发(D) | 开发培训内容 |
| 实施(I) | 执行培训 |
| 评估(E) | 评估培训效果 |

**培训方法**：

| 方法 | 优点 | 缺点 |
|------|------|------|
| 课堂培训 | 系统性强 | 互动少 |
| 在岗培训 | 实战性强 | 影响工作 |
| 导师制 | 个性化 | 资源有限 |
| e-learning | 灵活 | 缺乏互动 |

**培训效果评估**（柯氏四级）：
1. 反应层：学员满意吗？
2. 学习层：学到了吗？
3. 行为层：行为改变了吗？
4. 结果层：业绩提升了吗？

### 绩效管理

**学科归属**：人力资源管理 > 绩效管理  
**英文名称**：Performance Management  
**关联概念**：[KPI] [OKR] [考核]

#### 核心定义（一句话）
对员工工作表现进行持续管理的过程，包括目标设定、过程管理、考核反馈。

#### 费曼式解释（说人话）

**什么是绩效管理？**

| 绩效管理 | 绩效考核 |
|---------|---------|
| 全过程 | 一个环节 |
| 持续进行 | 定期进行 |
| 发展导向 | 评价导向 |
| 双向沟通 | 单向评价 |

**绩效管理流程**：

```
目标设定 → 过程辅导 → 绩效考核 → 反馈面谈 → 结果应用
    ↑                                              ↓
    ←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←
```

**考核方法**：
- KPI（关键绩效指标）
- OKR（目标与关键结果）
- 360度评估
- BSC（平衡计分卡）

### KPI

**学科归属**：人力资源管理 > 绩效管理  
**全称**：Key Performance Indicator  
**中文名称**：关键绩效指标  
**关联概念**：[绩效管理] [指标]

#### 核心定义（一句话）
衡量组织或个人绩效的关键量化指标。

#### 费曼式解释（说人话）

**什么是KPI？**

简单说：**用什么数字衡量做得好不好**。

**KPI的特点**：
- **关键**：最重要的，不是所有的
- **绩效**：衡量结果
- **指标**：可以量化

**KPI设计原则**（SMART）：

| 原则 | 含义 | 例子 |
|------|------|------|
| Specific | 具体的 | 不是"提高销售"，是"销售额100万" |
| Measurable | 可衡量 | 有数字 |
| Achievable | 可达成 | 跳一跳够得着 |
| Relevant | 相关的 | 与目标相关 |
| Time-bound | 有时限 | 年底前 |

**KPI的层级**：
- 公司KPI：营收、利润
- 部门KPI：销售部的销售额
- 个人KPI：个人的业绩目标

**问题**：
- 只看数字，忽视其他
- 可能导致短视行为

---

### OKR

**学科归属**：人力资源管理 > 绩效管理  
**全称**：Objectives and Key Results  
**中文名称**：目标与关键结果  
**发起公司**：英特尔，后由谷歌推广  
**关联概念**：[KPI] [目标管理]

#### 核心定义（一句话）
由目标(O)和关键结果(KR)组成的目标管理方法。

#### 费曼式解释（说人话）

**什么是OKR？**

**O（Objective）**：想达成什么目标？
**KR（Key Results）**：怎么知道达成了？

**例子**：
```
O：提升产品用户体验
KR1：用户满意度从80%提升到90%
KR2：日活跃用户增加20%
KR3：用户投诉率降低50%
```

**OKR vs KPI**：

| OKR | KPI |
|-----|-----|
| 挑战性目标 | 达成性目标 |
| 透明公开 | 可能保密 |
| 不直接与薪酬挂钩 | 通常与薪酬挂钩 |
| 季度/月度 | 年度为主 |

**OKR的特点**：
1. **挑战性**：60-70%达成率是正常的
2. **透明**：全公司可见
3. **对齐**：上下级目标对齐
4. **灵活**：季度调整

---

### 薪酬管理

**学科归属**：人力资源管理 > 薪酬管理  
**英文名称**：Compensation Management  
**关联概念**：[工资] [福利] [激励]

#### 核心定义（一句话）
设计和实施员工薪酬体系的管理活动。

#### 费曼式解释（说人话）

**什么是薪酬管理？**

简单说：**给员工发多少钱、怎么发**。

**薪酬的组成**：

```
总薪酬
├── 直接薪酬
│   ├── 基本工资（固定）
│   ├── 绩效奖金（浮动）
│   └── 长期激励（股权、期权）
└── 间接薪酬（福利）
    ├── 法定福利（五险一金）
    └── 企业福利（补充保险、假期）
```

**薪酬设计原则**：

| 原则 | 含义 |
|------|------|
| 内部公平 | 同等工作同等报酬 |
| 外部竞争 | 与市场相比有竞争力 |
| 个人激励 | 与绩效挂钩 |
| 成本可控 | 企业负担得起 |

**薪酬结构**：
- 高固定、低浮动：稳定，适合行政
- 低固定、高浮动：激励强，适合销售

---

### 薪酬宽带

**学科归属**：人力资源管理 > 薪酬管理  
**英文名称**：Broadbanding  
**关联概念**：[薪酬结构] [职级]

#### 核心定义（一句话）
将多个薪酬等级合并为少数几个宽范围的薪酬等级。

#### 费曼式解释（说人话）

**什么是薪酬宽带？**

**传统薪酬 vs 宽带薪酬**：

| 传统 | 宽带 |
|------|------|
| 等级多（20-30级） | 等级少（5-10级） |
| 级差小 | 级差大 |
| 晋升才涨薪 | 不晋升也能涨 |
| 僵化 | 灵活 |

**好处**：
1. 减少晋升压力
2. 鼓励横向发展
3. 增加灵活性

**问题**：
1. 需要更强的管理能力
2. 可能导致薪酬不公

---

### 员工关系

**学科归属**：人力资源管理 > 员工关系  
**英文名称**：Employee Relations  
**关联概念**：[劳动关系] [沟通] [满意度]

#### 核心定义（一句话）
管理组织与员工之间的关系，维护和谐劳动关系。

#### 费曼式解释（说人话）

**什么是员工关系？**

简单说：**让员工和企业关系融洽**。

**员工关系的内容**：

1. **劳动关系**
   - 劳动合同
   - 劳动法规
   - 劳动争议处理

2. **沟通机制**
   - 员工意见收集
   - 定期沟通
   - 信息公开

3. **员工关怀**
   - 员工帮助计划（EAP）
   - 困难帮扶
   - 健康管理

4. **离职管理**
   - 离职面谈
   - 离职分析
   - 人才储备

**员工满意度调查**：
- 定期调查
- 分析问题
- 改进措施

## 第三部分：领导力

---

### 领导力

**学科归属**：人力资源管理 > 领导力  
**英文名称**：Leadership  
**关联概念**：[影响力] [管理] [权力]

#### 核心定义（一句话）
影响他人自愿实现目标的能力。

#### 费曼式解释（说人话）

**什么是领导力？**

简单说：**让别人愿意跟你干**。

**领导 vs 管理**：

| 管理 | 领导 |
|------|------|
| 关注事 | 关注人 |
| 维持秩序 | 带来变革 |
| 依靠制度 | 依靠影响力 |
| 正确地做事 | 做正确的事 |

**领导力的来源**（权力类型）：

| 权力类型 | 来源 | 例子 |
|----------|------|------|
| 法定权力 | 职位 | 我是经理 |
| 奖赏权力 | 给奖励的能力 | 能发奖金 |
| 强制权力 | 惩罚的能力 | 能开除 |
| 专家权力 | 专业知识 | 技术专家 |
| 参照权力 | 人格魅力 | 大家佩服 |

**真正的领导力**：依靠专家权力和参照权力，而不是职位权力。

---

### 领导风格

**学科归属**：人力资源管理 > 领导力  
**英文名称**：Leadership Style  
**关联概念**：[情境领导] [变革型领导]

#### 核心定义（一句话）
领导者在影响下属时表现出的行为模式。

#### 费曼式解释（说人话）

**常见的领导风格**：

| 风格 | 特点 | 适合场景 |
|------|------|---------|
| 专制型 | 独裁决策 | 危机时刻 |
| 民主型 | 参与决策 | 创新团队 |
| 放任型 | 不干预 | 专家团队 |
| 交易型 | 奖惩交换 | 常规工作 |
| 变革型 | 激发变革 | 变革时期 |
| 服务型 | 服务下属 | 任何场景 |

**库尔特·勒温的三种风格**：

1. **专制型**：
   - 领导决定一切
   - 效率高但满意度低

2. **民主型**：
   - 员工参与决策
   - 满意度高但效率可能低

3. **放任型**：
   - 员工自主
   - 可能混乱

**没有最好的风格，只有最适合的风格**。

---

### 情境领导理论

**学科归属**：人力资源管理 > 领导力  
**提出者**：保罗·赫塞、肯·布兰查德  
**英文名称**：Situational Leadership Theory  
**关联概念**：[领导风格] [下属成熟度]

#### 核心定义（一句话）
根据下属的成熟度选择合适的领导风格。

#### 费曼式解释（说人话）

**什么是情境领导？**

核心思想：**不同情况用不同的领导方式**。

**四个象限**：

| 成熟度 | 特点 | 领导风格 |
|--------|------|---------|
| M1：无能力无意愿 | 不会也不愿 | 告知式（详细指示） |
| M2：无能力有意愿 | 不会但愿意 | 推销式（解释+支持） |
| M3：有能力无意愿 | 会但不愿 | 参与式（共同决策） |
| M4：有能力有意愿 | 会也愿意 | 授权式（放手） |

**举个例子**：
- 新员工：不会也不熟 → 详细指导
- 成长中：会一点但需要鼓励 → 支持+鼓励
- 成熟员工：会也愿意 → 放手让他干

**启示**：
- 没有放之四海而皆准的风格
- 领导者要能灵活切换
- 了解下属的状态

---

### 变革型领导

**学科归属**：人力资源管理 > 领导力  
**提出者**：伯恩斯、巴斯  
**英文名称**：Transformational Leadership  
**关联概念**：[变革] [激励]

#### 核心定义（一句话）
通过愿景和激励，引导下属超越自身利益的领导方式。

#### 费曼式解释（说人话）

**什么是变革型领导？**

**变革型 vs 交易型**：

| 交易型领导 | 变革型领导 |
|-----------|-----------|
| 你干活，我给钱 | 为愿景而奋斗 |
| 维持现状 | 推动变革 |
| 短期激励 | 长期影响 |
| 关注利益交换 | 关注价值认同 |

**变革型领导的四个要素**（4I）：

| 要素 | 内容 |
|------|------|
| 理想化影响(II) | 以身作则，成为榜样 |
| 鼓舞人心(IM) | 描绘愿景，激发热情 |
| 智力激发(IS) | 挑战思维，鼓励创新 |
| 个性化关怀(IC) | 关注个人需求和发展 |

**例子**：
- 乔布斯：用"改变世界"的愿景激励员工
- 马斯克：火星移民的愿景

---

### 服务型领导

**学科归属**：人力资源管理 > 领导力  
**提出者**：罗伯特·格林利夫  
**英文名称**：Servant Leadership  
**关联概念**：[领导力] [服务]

#### 核心定义（一句话）
以服务下属为核心的领导方式。

#### 费曼式解释（说人话）

**什么是服务型领导？**

**传统领导**：领导在上，下属在下
**服务型领导**：领导在下，托举下属

**核心思想**：
- 领导者的首要任务是服务
- 帮助下属成长和发展
- 权力来自服务，不是职位

**服务型领导的特征**：
1. 倾听
2. 同理心
3. 治愈
4. 觉察
5. 说服
6. 概念化
7. 远见
8. 管家精神
9. 承诺他人成长
10. 建立社群

**例子**：
- 西南航空创始人：服务员工，员工服务客户
